## ◆ 妊娠希望の27歳女性が麻しんワクチン接種を相談するシナリオ

260312

**問題**
27歳女性。近いうちに子供が欲しい。母子手帳で麻しんワクチン接種記録が1回。麻しんワクチンは入手不可だが、MRワクチン（麻しん風しん混合）の在庫はある。
薬剤師が行う説明として適切なのはどれか。2つ選べ。

**選択肢**

1. 麻しんワクチン接種歴が1回あるので、追加のワクチン接種は必要ありません。
​
2. マスクを着用し、手洗いをすれば、ワクチン接種は必要ありません。
​
3. 麻しんワクチンの代替として、麻しん風しん混合ワクチンを接種できます。
​
4. 妊娠の可能性がある場合は、麻しんワクチンの接種はできません。
​
5. 麻しんワクチンは通常2回接種します。

**正解と理由**
- `3. 麻しんワクチンの代替として、麻しん風しん混合ワクチンを接種できます。`
- `5. 麻しんワクチンは通常2回接種します。`
- **理由の構成**
	- 選択肢3：`MRワクチン`（麻しん＋風しん混合）は麻しんワクチンの代替として使用可能 → 正しい
	- 選択肢5：麻しんワクチンの標準は`2回接種`（1回では不十分）→ 正しい

**解法の手順（思考の流れ）**
- 選択肢1「1回あるから不要」→ 2回接種が標準。1回では抗体不十分な場合あり → 誤り
- 選択肢2「マスク・手洗いで十分」→ 麻しんは`空気感染`。マスク・手洗いでは防げない → 誤り
- 選択肢3「MRで代替」→ 正しい。むしろ風しんも同時に予防できて利点あり
- 選択肢4「妊娠の可能性→接種不可」→ 「妊娠中」は禁忌だが「妊娠希望（未妊娠）」なら接種可能。接種後約1ヶ月間は避妊 → 誤り（紛らわしい）
- 選択肢5「通常2回」→ 正しい

**他の選択肢が誤りな理由**
- 1. 1回接種では十分な免疫が得られない場合がある。2回接種が推奨
- 2. 麻しんは`空気感染`（飛沫核感染）。マスクでは防御不十分（N95が必要）
- 4. 「妊娠の可能性がある」≠「妊娠中」。未妊娠なら接種可能。ただし接種後は約1ヶ月間の避妊が必要

**キーポイント**
- 麻しん=`空気感染`。基本再生産数R₀=12〜18（感染力が極めて強い）
- `MRワクチン`（生ワクチン）：妊娠中は禁忌。接種後約1ヶ月間の避妊が必要
- 麻しんワクチンは`2回接種`が標準（1歳＋5〜6歳）

（一言メモ）
- 「妊娠の可能性がある→打てない」は紛らわしいが、「妊娠中」でなければ打てる。接種後の避妊期間が重要
- MRワクチンは麻しんの代替として認められている。風しんも同時に防げるメリット
