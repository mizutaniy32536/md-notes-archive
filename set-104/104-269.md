## ◆ イコサペント酸エチルを食直後に服用する理由を問うシナリオ

260310

**問題**
- イコサペント酸エチル粒状カプセルを食直後に服用する理由として、正しいのはどれか。1つ選べ。

**選択肢**

1. 胃腸障害を防ぐため
​
2. 肝臓での初回通過効果を高めるため
​
3. 腎排泄を促進するため
​
4. 血中濃度のピークを遅らせるため
​
5. 吸収を促進するため

**正解と理由**
- `5. 吸収を促進するため`
- **理由の構成**
	- EPAは`脂肪酸`様物質。食直後は食事刺激により`胆汁酸`分泌が促進→`ミセル`形成→EPA（脂溶性）の溶解性向上→吸収促進

**他の選択肢が誤りな理由**
- 1. 胃腸障害予防が主目的ではない
- 2. 初回通過効果を「高める」ことは治療上の目的にならない
- 3. 腎排泄は食事タイミングと無関係
- 4. 血中濃度ピークの遅延は目的ではなく、むしろ吸収向上が目的

**キーポイント**
- `脂溶性薬物`の食直後投与 → 胆汁酸によるミセル形成 → 吸収促進
- 同様のパターン: `インドメタシンファルネシル`（脂溶性プロドラッグ、食後BA増大）、`グリセオフルビン`

（一言メモ）
- 「脂溶性＋食直後」→「胆汁酸ミセルで吸収促進」は鉄板パターン
