## ◆ 予防接種法の定期接種（A類・B類、ワクチン種類）に関する正誤を問うシナリオ

260312

**問題**
予防接種法に定める予防接種に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**

1. インフルエンザは、個人の発症又はその重症化の防止に比重を置くため、B類疾病に分類されている。
​
2. 日本脳炎の予防接種には、トキソイド由来のワクチンが用いられる。
​
3. ポリオのワクチンは、ジフテリア、百日咳、破傷風のワクチンとともに4種混合ワクチンとして接種される。
​
4. 水痘に対して、ワクチンの任意接種が行われている。
​
5. 原虫感染症に対して、ワクチンの任意接種が行われている。

**正解と理由**
- `1. インフルエンザは、個人の発症又はその重症化の防止に比重を置くため、B類疾病に分類されている。`
- `3. ポリオのワクチンは、ジフテリア、百日咳、破傷風のワクチンとともに4種混合ワクチンとして接種される。`
- **理由の構成**
	- 選択肢1：`インフルエンザ`=`B類疾病`=個人予防・重症化防止目的 → 正しい
	- 選択肢3：`DPT-IPV`=4種混合（ジフテリア＋百日咳＋破傷風＋ポリオ）→ 正しい

**解法の手順（思考の流れ）**
- 選択肢1：インフルエンザ=B類 → 正しい
- 選択肢2：`日本脳炎`ワクチン=`不活化ワクチン`。`トキソイド`はジフテリア・破傷風 → 誤り
- 選択肢3：DPT-IPV=4種混合 → 正しい
- 選択肢4：`水痘`=2014年から`定期接種`（A類）。任意接種ではない → 誤り
- 選択肢5：`原虫感染症`のワクチン（マラリアRTS,S等）は日本では接種されていない → 誤り

**他の選択肢が誤りな理由**
- 2. 日本脳炎の予防接種には、トキソイド由来のワクチンが用いられる。
	`トキソイド`は毒素を無毒化したもの（ジフテリア・破傷風に使用）。日本脳炎ワクチンは`不活化ワクチン`

- 4. 水痘に対して、ワクチンの任意接種が行われている。
	`水痘`は2014年から`A類定期接種`。1歳と2歳に2回接種

- 5. 原虫感染症に対して、ワクチンの任意接種が行われている。
	日本では原虫感染症（マラリア等）に対するワクチン接種は行われていない

**キーポイント**
- `B類疾病`：インフルエンザ、高齢者肺炎球菌 → 個人予防目的
- `トキソイド`：毒素を無毒化したワクチン → ジフテリア、破傷風
- `DPT-IPV`の覚え方：D（ジフテリア）P（百日咳pertussis）T（破傷風tetanus）IPV（不活化ポリオ）
- `水痘`：2014年から定期接種化。よく「任意」とひっかける

**関連知識の比較**
- `トキソイド`：ジフテリア、破傷風（毒素→ホルマリン処理→無毒化）
- `不活化ワクチン`：日本脳炎、インフルエンザ、B型肝炎、肺炎球菌
- `生ワクチン`：MR、水痘、BCG、ロタウイルス
- 近年の定期接種化：水痘（2014年）、B型肝炎（2016年）、ロタウイルス（2020年）

（一言メモ）
- `トキソイド`=ジフテリア＋破傷風の2つだけ。日本脳炎と間違えやすい
- 「水痘=任意」はもう古い情報。2014年から定期接種（A類）
