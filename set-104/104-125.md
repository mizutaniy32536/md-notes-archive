## ◆ 母子感染の種類と予防法（垂直感染・HBV対策・CRS）を問うシナリオ

260312

**問題**
母子感染とその予防に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**

1. 水平感染とは、母親から胎児又は新生児に病原体が直接伝播する感染様式である。
​
2. ヒト免疫不全ウイルス（HIV）は、母子感染することはない。
​
3. HBs抗原陽性の母親から生まれてくる児には、出生後、抗HBs人免疫グロブリンとB型肝炎ワクチンを投与する必要がある。
​
4. 梅毒トレポネーマに対して、人工栄養哺育などにより母乳を介した感染を防ぐ対策がなされている。
​
5. 妊娠初期に妊婦が風しんに罹患すると、先天性風しん症候群を起こすことがある。

**正解と理由**
- `3. HBs抗原陽性の母親から生まれてくる児には、出生後、抗HBs人免疫グロブリンとB型肝炎ワクチンを投与する必要がある。`
- `5. 妊娠初期に妊婦が風しんに罹患すると、先天性風しん症候群を起こすことがある。`
- **理由の構成**
	- 選択肢3：`HBV`の母子感染予防=出生後に`抗HBs免疫グロブリン`（受動免疫）＋`B型肝炎ワクチン`（能動免疫）→ 正しい
	- 選択肢5：妊娠初期の`風しん`感染→`先天性風しん症候群`（CRS）→ 正しい

**解法の手順（思考の流れ）**
- 選択肢1：「`水平感染`=母親→胎児」→ これは`垂直感染`の定義。水平感染は個人間の飛沫・接触等 → 誤り
- 選択肢2：「HIV母子感染しない」→ 妊娠中・分娩時・授乳で母子感染する → 誤り
- 選択肢3：HBs抗原陽性母→ 出生後に`HBIG`＋`HBワクチン` → 正しい
- 選択肢4：「梅毒=母乳予防で人工栄養」→ 母乳で予防するのは`HTLV-1`。梅毒は`経胎盤感染`で`ペニシリン`治療 → 誤り
- 選択肢5：妊娠初期の風しん→CRS → 正しい

**他の選択肢が誤りな理由**
- 1. `水平感染`と`垂直感染`を逆にしたひっかけ。水平=個人間（飛沫、接触等）、垂直=母→子
- 2. HIVは経胎盤＋産道＋母乳の3経路で母子感染する
- 4. `梅毒`の母子感染は`経胎盤感染`（先天梅毒）。母乳感染ではないため人工栄養は無関係。母乳で予防するのは`HTLV-1`

**キーポイント**
- `HBV`母子感染予防：出生後12時間以内に`HBIG`（受動免疫）＋`HBワクチン`（能動免疫）の同時投与
- `垂直感染`（母→子）：経胎盤、産道、母乳。`水平感染`：飛沫、接触、食品等
- 先天性風しん症候群の三主徴：難聴、先天性心疾患、白内障

**関連知識の比較**

| 感染症 | 母子感染予防法 |
|---|---|
| `HBV` | `HBIG`＋`HBワクチン` |
| `HTLV-1` | `人工栄養`（母乳中止） |
| `HIV` | `ART`＋`帝王切開`＋`人工栄養` |
| `梅毒` | 妊婦の`ペニシリン`治療 |
| `風しん` | 妊娠前`MRワクチン`接種 |

（一言メモ）
- 水平vs垂直の入れ替えは毎回出る。「垂直=母→子（上から下）」「水平=横の人→人」で覚える
- HBV予防の「HBIG＋ワクチン」=受動免疫＋能動免疫のダブル防御
- 「梅毒=母乳」「HTLV-1=経胎盤」は逆にしたひっかけ。正しくは梅毒=経胎盤、HTLV-1=母乳
