## ◆ 一次性能動輸送担体を選ぶシナリオ

260310

**問題**
- 一次性能動輸送担体はどれか。1つ選べ。

**選択肢**

1. グルコーストランスポーター GLUT1
​
2. P-糖タンパク質 MDR1
​
3. 有機アニオントランスポーター OAT1
​
4. H⁺/ペプチド共輸送体 PEPT1
​
5. Na⁺/グルコース共輸送体 SGLT2

**正解と理由**
- `2. P-糖タンパク質 MDR1`
- **理由の構成**
	- `P-gp`（MDR1、ABCB1）は`ABCトランスポーター`ファミリーに属し、`ATP`の加水分解エネルギーを直接利用して物質を輸送する`一次性能動輸送`担体

**他の選択肢が誤りな理由**
- 1. `GLUT1` = `促進拡散`（濃度勾配に従う、エネルギー不要）
- 3. `OAT1` = `二次性能動輸送`（α-ケトグルタル酸の濃度勾配を利用）
- 4. `PEPT1` = `二次性能動輸送`（H⁺濃度勾配を利用）
- 5. `SGLT2` = `二次性能動輸送`（Na⁺濃度勾配を利用）

**キーポイント**
- 一次性能動輸送 = **ATP直接利用** = `ABCトランスポーター`（P-gp, BCRP, MRP）、Na⁺/K⁺-ATPase
- 二次性能動輸送 = **イオン濃度勾配を利用** = SGLT, PEPT1, OAT等

（一言メモ）
- 「一次性」→ATP直接、「二次性」→他のイオン勾配。ABCが付いたら一次性と即答
