## ◆ 地域の人口動態を調べる際に必要な指標を判断するシナリオ
260307

**問題**
- ある地域の1年間の人口動態を調べる際、必要でないのはどれか。1つ選べ。

**選択肢**

1. 出生数
​
2. 死亡数
​
3. 老年人口
​
4. 婚姻数
​
5. 離婚数

**正解と理由**
- `3. 老年人口`
- **理由の構成**
	- **問題の整理**：ある地域の1年間の`人口動態`を調べる際に必要でない指標を選択する問題
	- **薬学的知識・仕様・制約**：`人口動態統計`は1年間における人口の変動（フロー）を記録する統計であり、`出生`・`死亡`・`死産`・`婚姻`・`離婚`の5つの届出事象を対象とする。一方、`老年人口`（65歳以上人口）は特定時点における人口構成を示す`人口静態統計`（国勢調査）の指標であり、動態統計には含まれない
	- **選択理由**：`老年人口`はストック（蓄積量）データであり、1年間の変動を示すフローデータではないため、人口動態統計には含まれず、選択肢3が正解と判断される

**解法の手順（思考の流れ）**
- まず「人口動態」というキーワードから、1年間の人口変動（フローデータ）を扱う統計であることを想起する
- 人口動態統計の5大項目は`出生`・`死亡`・`死産`・`婚姻`・`離婚`であることを確認する
- 選択肢1（出生数）、2（死亡数）、4（婚姻数）、5（離婚数）はすべて人口動態統計の項目に該当すると判断する
- 選択肢3（老年人口）は特定時点での年齢構成を示す`人口静態`の指標であり、動態統計には含まれないと判断する
- これにより`選択肢3`（老年人口）が正解であると結論づけた

**他の選択肢が誤りな理由**
- 1. 出生数
	`出生数`は人口動態統計の5大項目の1つであり、1年間の人口変動を把握するために必要な指標である。出生の届出は`戸籍法`に基づき14日以内に行う義務がある

- 2. 死亡数
	`死亡数`は人口動態統計の5大項目の1つであり、1年間の人口変動を把握するために必要な指標である。死亡の届出は`戸籍法`に基づき7日以内に行う義務がある

- 4. 婚姻数
	`婚姻数`は人口動態統計の5大項目の1つであり、1年間の社会動態を把握するために必要な指標である

- 5. 離婚数
	`離婚数`は人口動態統計の5大項目の1つであり、1年間の社会動態を把握するために必要な指標である

**キーポイント**
- `人口動態統計`の5大項目は「`出生`・`死亡`・`死産`・`婚姻`・`離婚`」であり、すべて1年間のフロー（変動量）データである
- `人口静態統計`は特定時点の人口構成を示すストック（蓄積量）データであり、`国勢調査`が代表的な調査である
- 人口動態統計は`厚生労働省`が所管し、人口静態統計（国勢調査）は`総務省`が所管する
- `死産`も人口動態統計の項目に含まれることを見落としやすいため注意が必要

**関連知識の比較**
- `人口動態統計`と`人口静態統計`の違い：動態は「フロー」（1年間の変動）、静態は「ストック」（ある時点の蓄積量）を扱う
- `国勢調査`は5年ごとに実施され、大規模調査と簡易調査が交互に行われる。直近の大規模調査は2020年（令和2年）
- `老年人口`は65歳以上、`年少人口`は0〜14歳、`生産年齢人口`は15〜64歳と定義される
- `合計特殊出生率`は人口動態統計から算出される指標であり、一人の女性が生涯に産む子どもの数の推計値を示す

（一言メモ）
- `人口動態統計`の5項目は「出・死・死産・婚・離」と覚え、すべて届出に基づく`フローデータ`であることがポイント
- `人口静態`と`人口動態`の区別は「ある時点のスナップショットか、1年間の変動記録か」で判断する
- `国勢調査`は`総務省`所管で5年ごとに実施、人口動態統計は`厚生労働省`所管で毎年集計される
