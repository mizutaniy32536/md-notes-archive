## ◆ C型慢性肝炎の感染経路として可能性が低い生活習慣を選ぶシナリオ

260312

**問題**
60歳男性。ジェノタイプ2型のC型慢性肝炎と診断。この患者から聴取した生活習慣や過去の経験のうち、C型慢性肝炎に罹患した原因として可能性が低いのはどれか。2つ選べ。

**選択肢**

1. 鹿肉の生食
​
2. 入れ墨や不衛生なピアスの穴あけ処置
​
3. 20歳代での手術時の輸血
​
4. 海外旅行中の生水の摂取

**正解と理由**
- `1. 鹿肉の生食`
- `4. 海外旅行中の生水の摂取`
- **理由の構成**
	- **問題の整理**：`HCV`（C型肝炎ウイルス）の感染経路は主に`血液感染`。血液を介さない経路は感染リスクが低い
	- **薬学的知識・仕様・制約**：`HCV`は`血液感染`が主。性的接触での感染は稀。経口感染（食品・水）では感染しない
	- **選択理由**：鹿肉の生食（経口）と生水の摂取（経口）はHCV感染の経路ではない

**解法の手順（思考の流れ）**
- 選択肢1：鹿肉の生食=`E型肝炎`のリスク因子。HCVの感染経路ではない → 可能性低い
- 選択肢2：入れ墨・ピアス=`血液曝露`のリスク。HCV感染経路として重要 → 可能性高い
- 選択肢3：手術時の輸血（特に1992年以前）=HCV感染の主要経路 → 可能性高い
- 選択肢4：生水の摂取=`A型肝炎`・`E型肝炎`のリスク因子。HCVの感染経路ではない → 可能性低い

**他の選択肢が誤りな理由（本問では「可能性が高い選択肢」）**
- 2. 入れ墨・ピアス：不衛生な器具による`血液曝露`はHCV感染の重要なリスク因子
- 3. 20歳代の輸血：1992年以前の輸血にはHCVスクリーニングが不十分。60歳の患者が20歳代=40年前≒1980年代 → 高リスク

**キーポイント**
- `HCV`の感染経路：①`輸血`（1992年以前）②`針刺し`③`入れ墨・ピアス`④`薬物注射（針の共有）`
- HCVは`血液感染`。経口感染しない（食品・水では感染しない）
- `E型肝炎`：鹿肉・猪肉・豚レバーの生食で感染。HCVと混同注意
- `A型肝炎`：生水・生貝（牡蠣等）で経口感染。海外旅行で注意

**関連知識の比較**

| 肝炎 | 感染経路 | 生食/生水 |
|---|---|---|
| `A型` | 経口感染 | ○（生水・生貝） |
| `B型` | 血液・性・母子 | × |
| `C型` | **血液** | × |
| `E型` | 経口感染 | ○（鹿肉・豚レバー） |

（一言メモ）
- 鹿肉の生食=`E型肝炎`のキーワード。C型肝炎のひっかけで出る
- 生水=`A型肝炎`。C型ではない
- 「1992年以前の輸血」=HCV感染の典型リスク因子。年齢計算がポイント
