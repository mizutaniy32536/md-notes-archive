## ◆ 血液製剤による感染被害の救済制度に関するシナリオ
260308

**問題**
- 血液製剤による感染被害の救済制度に関する記述として、正しいのはどれか。2つ選べ。

**選択肢**

1. 適正な使用目的に従い適正に使用された場合の感染被害の救済を目的としている。
​
2. 給付金は、原因となった血液製剤の製造販売業者が全額負担する。
​
3. 医療費給付の対象は、入院を要する程度以上の感染被害である。
​
4. 葬祭料や遺族年金に関する給付項目は含まれていない。
​
5. 製造販売業者の過失が裁判によって証明された場合に救済の対象となる。

**正解と理由**
- `1, 3`
- **理由の構成**
	- **問題の整理**：血液製剤による感染被害の救済制度（感染等被害救済制度）の対象・費用負担・給付項目・申請手続きに関する正誤を判断する
	- **薬学的知識・仕様・制約**：感染等被害救済制度はPMDA（医薬品医療機器総合機構）が運営する`無過失責任型の救済制度`であり、適正使用下での感染被害を対象とする — 費用は`製造販売業者と国の共同負担` — 給付項目は`医療費・医療手当・障害年金・遺族年金・葬祭料`を含む
	- **選択理由**：選択肢1は「適正使用＋感染被害」という制度の対象条件と一致。選択肢3は「入院を要する程度以上」という医療費給付の要件と一致

**解法の手順（思考の流れ）**
- 感染等被害救済制度の3つの柱「対象条件・費用負担・給付項目」を軸に各選択肢を照合
- 選択肢1：「適正な使用目的＋適正に使用された場合の感染被害」→ 制度の対象条件そのものなので正しい
- 選択肢2：費用負担は「製造販売業者＋国の共同負担」→「全額製造販売業者負担」は誤り
- 選択肢3：医療費給付の要件は「入院を要する程度以上」→ 正しい
- 選択肢4：給付項目に遺族年金・葬祭料は含まれる → 「含まれていない」は誤り
- 選択肢5：無過失責任制度であるため過失の証明・訴訟は不要 → 誤り

**他の選択肢が誤りな理由**
- 2. 給付金は全額製造販売業者負担ではない
	感染等被害救済制度の費用は`製造販売業者と国の共同負担`とされており、製造販売業者のみが全額負担するわけではない — 製造販売業者が拠出金を納付し、不足分を国が補填する仕組み

- 4. 葬祭料・遺族年金は給付項目に含まれる
	給付項目は`医療費・医療手当・障害年金・遺族年金・葬祭料`の5種類があり、葬祭料と遺族年金はどちらも含まれる — 「含まれていない」とする記述は誤り

- 5. 無過失責任制度のため過失証明・訴訟は不要
	感染等被害救済制度は`無過失責任型`の行政的救済制度であり、製造販売業者の過失を裁判で証明する必要はない — PMDAに申請するだけで審査・給付が行われる

**キーポイント**
- 感染等被害救済制度は`PMDA`が運営する`無過失責任型`の救済制度
- 対象：`適正な使用目的`に従い`適正に使用`された場合の感染被害
- 費用負担：`製造販売業者＋国`の共同負担（全額製造販売業者負担ではない）
- 医療費給付の要件：`入院を要する程度以上`の感染被害
- 給付項目5種：`医療費・医療手当・障害年金・遺族年金・葬祭料`（葬祭料・遺族年金も含まれる）
- 申請方法：PMDAへの申請のみ — 訴訟・過失証明は不要

**関連知識の比較**
- 感染等被害救済制度：血液製剤による感染被害 — 無過失責任 — PMDAへ申請 — 給付5種
- 医薬品副作用被害救済制度：医薬品の副作用被害 — 無過失責任 — PMDAへ申請 — 給付6種（感染等被害救済制度の5種＋調剤医療費）
- 薬害訴訟（民事）：過失の証明が必要 — 裁判所へ提訴 — 上記救済制度とは別の法的手続き

（一言メモ）
- `無過失責任` = 過失証明不要・訴訟不要、これが感染等被害救済制度の最大の特徴
- 費用負担の`「製造販売業者＋国」`と、医療費給付要件の`「入院を要する程度以上」`はセットで暗記
- 給付項目に`遺族年金・葬祭料`が含まれる点を忘れずに（「含まれない」という引っかけが頻出）
