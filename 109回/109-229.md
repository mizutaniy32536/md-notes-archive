## ◆ レジオネラ肺炎のシナリオ（続）
260308

**シナリオ**
60代男性。温泉旅館に宿泊後、38.5℃の発熱、悪寒、全身倦怠感、咳嗽、喀痰が出現し近医を受診。胸部X線で肺炎像を認めた。尿中抗原検査陽性。当該施設の循環冷却水からも同一菌が検出された（レジオネラ肺炎）。

**問題**
- この男性患者が罹患した疾病に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**

1. 感染症法において三類感染症に分類される。
​
2. 高齢者や免疫力の低下した人がかかる日和見感染症である。
​
3. ビル空調機の循環冷却水や加湿器から発生するエアロゾルを吸入することで集団発生しやすい。
​
4. 罹患者の飛沫によって、直接他者へ感染する。
​
5. ワクチン接種により予防することができる。

**正解と理由**
- `2, 3`
- **理由の構成**
	- **問題の整理**：レジオネラ症の感染症法上の分類・感染経路・感染リスク・予防法に関する正誤を判断する
	- **薬学的知識・仕様・制約**：レジオネラ症は`四類感染症`（三類ではない） — 感染経路は`環境（水系エアロゾル）→人`であり、人から人への感染はない — 高齢者・喫煙者・免疫低下者が感染リスク高 — レジオネラワクチンは存在しない
	- **選択理由**：選択肢2は「高齢者や免疫力低下者に多い日和見感染症的性質」と一致。選択肢3は「循環冷却水・加湿器のエアロゾル吸入による集団発生」という感染経路と一致

**解法の手順（思考の流れ）**
- レジオネラ症の4つの特性「感染症法分類・感染経路・感染リスク群・予防法」を軸に各選択肢を照合
- 選択肢1：三類感染症 → `四類感染症`が正しい → 誤り
- 選択肢2：高齢者・免疫低下者に多い日和見感染症的性質 → 正しい
- 選択肢3：循環冷却水・加湿器のエアロゾル吸入で集団発生 → 正しい（環境→人の感染経路）
- 選択肢4：罹患者の飛沫で直接他者へ感染 → `人→人感染はない` → 誤り
- 選択肢5：ワクチン接種で予防 → `レジオネラワクチンは存在しない` → 誤り

**他の選択肢が誤りな理由**
- 1. レジオネラ症は四類感染症（三類ではない）
	感染症法における三類感染症はコレラ・赤痢・腸チフス等の経口感染する腸管系感染症が中心 — レジオネラ症は`四類感染症`に分類される — 四類は主に動物や環境由来の感染症

- 4. 人から人への感染はない
	レジオネラ・ニューモフィラの感染経路は`環境（水系）→人`のみ — 罹患者の飛沫は感染源にならない — この点が飛沫感染するインフルエンザや新型コロナウイルスとの大きな違い

- 5. レジオネラワクチンは存在しない
	現時点でレジオネラ症に対する認可ワクチンは存在しない — 予防は感染源（循環冷却水・給湯システム等）の適切な管理・消毒による環境対策が中心

**キーポイント**
- レジオネラ症の感染症法上の分類：`四類感染症`（三類ではない）
- 感染経路：`環境（水系エアロゾル）→人`のみ — `人→人感染はない`
- 感染リスク群：`高齢者・喫煙者・免疫低下者・糖尿病患者`
- 感染源：`ビル空調機の循環冷却水・加湿器・温泉・給湯システム`のエアロゾル
- 予防法：`環境管理・消毒`（ワクチンなし）

**関連知識の比較**
- レジオネラ症（四類）：水系エアロゾル→人 — 人→人感染なし — ワクチンなし
- インフルエンザ（五類）：飛沫感染・接触感染 — 人→人感染あり — ワクチンあり
- 三類感染症の例：コレラ・赤痢・腸チフス・パラチフス・腸管出血性大腸菌感染症

（一言メモ）
- `レジオネラ＝四類＝環境→人のみ＝人→人なし＝ワクチンなし`で一括整理
- 「飛沫感染する」「三類」「ワクチンあり」はすべて引っかけ選択肢のパターン
