# 109-139

一般環境大気測定局及び自動車排出ガス測定局における大気汚染物質の測定法 に関する記述のうち、正しいのはどれか。2つ選べ。
1 一酸化炭素は、照射した赤外線の吸収量に基づいて測定される。
2 二酸化窒素は、紫外線の照射によって励起した二酸化窒素分子が発する蛍光の
強度に基づいて測定される。
3 光化学オキシダントは、ザルツマン試薬との反応により生じる生成物の吸光度
に基づいて測定される。
4 二酸化硫黄は、エチレンとの反応により生じる近紫外線領域の発光の強度に基
づいて測定される。
5 浮遊粒子状物質は、ろ紙上に粒子を捕集して、b 線を照射し、その透過量に基
づいて測定される。

## 正解
**1 と 5**

## 解説

### 大気汚染物質の測定法まとめ
| 物質 | 測定法 |
|---|---|
| 一酸化炭素（CO） | 非分散型赤外線吸収法（NDIR） |
| 二酸化窒素（NO₂） | 化学発光法、ザルツマン試薬法 |
| 二酸化硫黄（SO₂） | 紫外線蛍光法 |
| 光化学オキシダント（Ox） | 中性ヨウ化カリウム溶液法、紫外線吸収法 |
| 浮遊粒子状物質（SPM） | β線吸収法 |

### 各選択肢の検証
1. ⭕ 一酸化炭素は赤外線吸収法（NDIR法）
2. ❌ 二酸化窒素は蛍光法ではない（SO₂が紫外線蛍光法）
3. ❌ 光化学オキシダントはザルツマン試薬ではない（NO₂がザルツマン試薬法）
4. ❌ 二酸化硫黄はエチレンと反応させない（紫外線蛍光法）
5. ⭕ 浮遊粒子状物質はβ線吸収法

### ポイント
- **ザルツマン試薬** = NO₂専用（赤紫色の吸光度測定）
- **紫外線蛍光法** = SO₂専用
- **β線吸収法** = SPM（粒子の量を測定）
