# 109-147

医療法に基づく医療計画策定において定めるべき事項はどれか。2つ選べ。
1 がん等の ₅ 疾病の治療又は予防に係る事業に関する事項
2 献血に関する住民への理解及び献血受入の円滑な実施に関する事項
3 地域医療に必要となる未承認薬の治験の推進に関する事項
4 医療従事者の確保に関する事項
5 患者申出療養等の評価療養の実施に関する事項

## 正解
**1 と 4**

## 解説

### 医療計画とは
都道府県が地域の医療提供体制を整備するために策定する計画（医療法30条の4）

### 医療計画で定めるべき主要事項

#### 1. 5疾病・5事業・在宅医療
- **5疾病**：
  1. がん
  2. 脳卒中
  3. 心筋梗塞等の心血管疾患
  4. 糖尿病
  5. 精神疾患

- **5事業**：
  1. 救急医療
  2. 災害医療
  3. へき地医療
  4. 周産期医療
  5. 小児医療

- **在宅医療**

#### 2. 医療従事者の確保
- 医師・看護師等の確保対策
- 地域偏在の解消

#### 3. 病床の整備
- 病床数の目標
- 地域医療構想

#### 4. 医療の安全確保

### 各選択肢の検証
1. ⭕ **5疾病の治療又は予防** → 医療計画の柱
2. ❌ 献血 → 「献血推進計画」の範囲（医療計画ではない）
3. ❌ 未承認薬の治験 → 医療計画の範囲外
4. ⭕ **医療従事者の確保** → 医療計画で定めるべき事項
5. ❌ 評価療養 → 保険制度の話、医療計画の範囲外

### ポイント
**医療計画 = 5疾病・5事業・在宅医療 + 医療従事者確保 + 病床整備**
