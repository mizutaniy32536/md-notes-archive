# 109-312

麻薬及び向精神薬取締法に基づく処方（ブプレノルフィン経皮吸収型製剤：第二種向精神薬）に関する記述として、正しいのはどれか。2つ選べ。

1. 調剤するには、薬局の許可のほかに麻薬小売業者の許可を要する。
2. 処方箋に、麻薬施用者の免許証の番号が記載されているか確認しなければならない。
3. 堅固な設備内に施錠して保管しなければならない。
4. 廃棄する際、法令に基づく許可や届出の必要はない。
5. 規定の数量以上の紛失等の事故が発生した場合、「向精神薬事故届」を届け出なければならない。

## 正解
**4. 廃棄する際、法令に基づく許可や届出の必要はない。**
**5. 規定の数量以上の紛失等の事故が発生した場合、「向精神薬事故届」を届け出なければならない。**

## 解説

ブプレノルフィン経皮吸収型製剤は**第二種向精神薬**です。麻薬と向精神薬では取扱いのルールが異なります。

| 選択肢 | 正誤 | 理由 |
|---|---|---|
| 1. 麻薬小売業者の許可 | ❌ | **麻薬**を調剤する場合は麻薬小売業者の許可が必要。**向精神薬**は薬局の許可のみで調剤可能 |
| 2. 麻薬施用者の免許証番号 | ❌ | **麻薬**処方箋には麻薬施用者免許証番号の記載が必要。**向精神薬**では不要 |
| 3. 堅固な設備内に施錠保管 | ❌ | **麻薬**は堅固な設備内に施錠保管が必要。**向精神薬**は施錠義務なし（盗難防止の配慮は必要） |
| 4. 廃棄に許可や届出不要 | ⭕ | **向精神薬**の廃棄には、麻薬のような厳格な手続き（立会い、届出）は不要 |
| 5. 向精神薬事故届 | ⭕ | 向精神薬の紛失・盗難等の事故が発生した場合、「向精神薬事故届」の提出が義務付けられている |

📌 つまずきポイント：麻薬と向精神薬の取扱いの違い

### 麻薬 vs 向精神薬の比較

| 項目 | 麻薬 | 向精神薬 |
|---|---|---|
| **免許・許可** | 麻薬小売業者の免許が必要 | 薬局の許可のみでOK |
| **処方箋の記載** | 麻薬施用者免許証番号が必要 | 不要 |
| **保管** | 堅固な設備内に施錠保管 | 施錠義務なし（盗難防止配慮は必要） |
| **廃棄** | 立会い・届出が必要 | 許可・届出不要 |
| **事故時** | 麻薬事故届 | 向精神薬事故届 |
| **譲渡** | 麻薬譲渡証が必要 | 不要 |

> 💡 **覚え方**
> - **麻薬**：厳格（免許、施錠、立会い、届出すべて必要）
> - **向精神薬**：比較的緩やか（薬局免許のみ、施錠不要、廃棄届出不要）
> - **共通点**：事故時は事故届が必要！
