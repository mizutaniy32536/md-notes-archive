# 109-150

パーソナリティの認知におけるハロー効果の傾向を示す説明として、適切なの はどれか。1つ選べ。
1 自分が考える自分の姿と他人に見える姿が一致している。
2 好ましい（好ましくない）特徴があると、その他の特徴も高く（低く）認識す
る。
3 最初に提示された情報が全体の印象を方向づける。
4 最後に得られた情報が全体の印象に影響を与える。
5 個人的経験から、Ａという特性があると必ずＢという特性があると認識する。

## 正解
**2. 好ましい（好ましくない）特徴があると、その他の特徴も高く（低く）認識する**

## 解説

### ハロー効果（halo effect）とは
**一部の目立つ特徴（良い or 悪い）が、他の特徴の評価も引きずる現象**

"halo" = 後光、光輪（天使の頭の上にある光の輪）

### 具体例
- 美人・イケメン → 性格も良さそう、仕事もできそう
- 有名大学出身 → 頭が良い、信頼できる
- 一度ミス → 他の仕事もダメそう、信頼できない
- 好きな芸能人 → 何をやっても素晴らしい

### 各選択肢の検証
1. ❌ **自分が考える自分の姿と他人に見える姿が一致** → ジョハリの窓の「開放の窓」
2. ⭕ **好ましい（好ましくない）特徴があると、その他の特徴も高く（低く）認識する** → ハロー効果
3. ❌ **最初に提示された情報が全体の印象を方向づける** → 初頭効果（primacy effect）
4. ❌ **最後に得られた情報が全体の印象に影響** → 親近効果・新近効果（recency effect）
5. ❌ **個人的経験から、Aという特性があると必ずBという特性がある** → ステレオタイプ

### 関連する認知バイアス

| 効果 | 説明 | 例 |
|---|---|---|
| **ハロー効果** | 一部の特徴が全体の評価に影響 | 美人→性格も良さそう |
| **初頭効果** | 最初の情報が全体の印象を決める | 第一印象が重要 |
| **親近効果** | 最後の情報が印象に残りやすい | 最後の発表者が有利 |
| **ステレオタイプ** | 特定の集団に対する固定観念 | ○○人は△△だ |

### 医療現場での応用
- 患者の第一印象（身なり、態度）で判断しない
- 一つの検査結果だけで全体を評価しない
- バイアスに気づき、客観的な評価を心がける
