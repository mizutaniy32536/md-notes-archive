## ◆ 血液製剤によるHIV感染事例に関する生物学的知識を問うシナリオ
260308

**問題**
- （物理・化学・生物） 血友病の治療のために投与された非加熱血液製剤により HIV 感染が生じた事例に関連する生物学的知識として、正しいのはどれか。2つ選べ。

**選択肢**

1. 遺伝性血友病は X 連鎖潜性（劣性）遺伝により生じるため、男性に発症しやすい。
​
2. 血漿中の HIV は加熱処理しなくても、孔径 0.22 nm のメンブレンろ過により除去できる。
​
3. 血液製剤に混入した HIV は、治療を受ける患者の CD8 陽性 T リンパ球に感染する。
​
4. HIV はレトロウイルスであり、プロウイルス化により感染宿主細胞内で長期間潜伏できる。
​
5. HIV 感染による症状として、花びらのように分かれた核を持つ特徴的な腫瘍化 T 細胞がみられるようになる。

**正解と理由**
- `1. 遺伝性血友病は X 連鎖潜性（劣性）遺伝` / `4. HIV はレトロウイルスであり、プロウイルス化により長期潜伏`
- **理由の構成**
	- **問題の整理**：血液製剤によるHIV感染事例に関連する「生物学的知識」として正しいものを2つ選ぶ
	- **薬学的知識・仕様・制約**：血友病の遺伝様式、HIVのウイルス学的性質（サイズ・標的細胞・複製機構）、HTLV-1との鑑別がポイント
	- **選択理由**：選択肢1は血友病の遺伝様式の正確な記述、選択肢4はレトロウイルスの基本特性として正しい

**解法の手順（思考の流れ）**
- 血友病はX連鎖劣性遺伝 → X染色体が1本の男性は保因者が存在せず、Xに異常があれば直接発症 → 選択肢1は正しいと判断
- HIVのサイズは約100nm → 0.22μm（220nm）のメンブレンろ過では通過する可能性があり除去不可 → 選択肢2は誤りと判断
- HIVの標的細胞はCD4陽性T細胞（ヘルパーT細胞） → 選択肢3のCD8陽性（キラーT細胞）は誤りと判断
- HIVはレトロウイルス → 逆転写酵素でRNAをDNAに変換後、宿主ゲノムに組み込みプロウイルス化 → 免疫系に攻撃されにくく長期潜伏が可能 → 選択肢4は正しいと判断
- 「花びら状の核」を持つ腫瘍化T細胞 → HTLV-1による成人T細胞白血病（ATL）の特徴 → HIVではない → 選択肢5は誤りと判断

**他の選択肢が誤りな理由**
- 2. 孔径 0.22 nm のメンブレンろ過
	HIVのサイズは約`100nm`（0.1μm）。問題文の「0.22nm」は明らかな誤記で実際は`0.22μm（220nm）`を指すと解釈されるが、いずれにせよHIVはこのサイズのフィルターを通過するため`ろ過除去は不可` — 非加熱製剤問題の根本は加熱処理による不活化が行われなかったことにある

- 3. CD8 陽性 T リンパ球に感染
	HIVが感染する標的細胞は`CD4陽性T細胞`（ヘルパーT細胞）— CD4分子がHIVの`gp120`と結合することで細胞内侵入が起こる — `CD8陽性T細胞`（キラーT細胞）はHIVの標的ではない

- 5. 花びらのように分かれた核を持つ腫瘍化T細胞
	「花びら状の核」を持つ腫瘍化T細胞は`HTLV-1`（ヒトT細胞白血病ウイルス1型）による`成人T細胞白血病（ATL）`の特徴的所見 — `HIV感染症`ではみられない — HIVとHTLV-1の混同は頻出の誤りパターン

**キーポイント**
- 血友病の遺伝様式：`X連鎖劣性遺伝` → X染色体が1本しかない男性に発症しやすい（女性は保因者）
- HIVの標的細胞：`CD4陽性T細胞`（ヘルパーT細胞）一択 — CD8との混同に注意
- HIVの複製機構：`レトロウイルス` → `逆転写酵素`でRNA→DNA → 宿主ゲノムに組み込み`プロウイルス`化 → 長期潜伏
- 花びら状核の腫瘍化T細胞：`HTLV-1`によるATLの特徴 — HIVではない

**関連知識の比較**
- `HIV`：RNAウイルス（レトロウイルス科） — 標的：CD4陽性T細胞 — 逆転写酵素を持つ — プロウイルス化により長期潜伏 — AIDS（後天性免疫不全症候群）の原因
- `HTLV-1`：RNAウイルス（レトロウイルス科） — 標的：CD4陽性T細胞 — 成人T細胞白血病（ATL）の原因 — 花びら状核の異型リンパ球が特徴
- 血友病：`血友病A`（第VIII因子欠乏）・`血友病B`（第IX因子欠乏）ともにX連鎖劣性遺伝 — 男性に発症しやすい

（一言メモ）
- `CD4`とCD8の区別は国試頻出 — `HIV → CD4`を確実に結びつける
- `HTLV-1`と`HIV`はいずれもレトロウイルスだが起こす疾患が異なる — ATLの花びら核はHTLV-1と記憶
