# 第98回 問52

**問題文**
曇点を有する界面活性剤はどれか｡1つ選べ。

**選択肢**
1. ステアリン酸ナトリウム
2. ベンザルコニウム塩化物
3. ポリオキシソルビタン脂肪酸エステル
4. レシチン
5. ラウリル硫酸ナトリウム

**解説**
解答　解なし
非イオン性界面活性剤の溶解度が減少し始める温度を曇点という。界面活性剤のうち、ポリオキシエチレンソルビタン脂肪酸エステルは非イオン性界面活性剤に該当するため曇点を有する。なお、ステアリン酸ナトリウム、ベンザルコニウム塩化物、レシチン、ラウリル硫酸ナトリウムはイオン性界面活性剤であり、クラフト点（溶解度が増加し始める温度）を有する。
選択肢3のポリオキシソルビタン脂肪酸エステルは、界面活性剤として実在しないため、本問は解なしとなる。
