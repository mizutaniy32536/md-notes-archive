# 第98回 問49

**問題文**
フルオロウラシルのプロドラッグはどれか。1つ選べ。

**選択肢**
1. ドキシフルリジン
2. エノシタビン
3. シスプラチン
4. イリノテカン
5. シクロホスファミド

**正解:** 1

**解説**
プロドラッグとは、化学構造の一部を修飾し、体内で酵素や化学反応により活性化合物（親薬物）に戻るように工夫されたものである。プロドラッグ化の目的には、副作用の軽減、作用の持続化、吸収性の改善など様々なものが挙げられる。
フルオロウラシルのプロドラッグには、ドキシフルリジン、カルモフールなどがある。ドキシフルリジンは、腫瘍組織へのターゲティングを目的としたプロドラッグであり、カルモフールは、作用持続化を目的としたプロドラッグである。
