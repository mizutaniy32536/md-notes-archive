# 第98回 問246

**問題文**
47歳女性。眼科外来にて、以下の薬剤が処方された。
スクリーンショット 2017-04-17 11.12.50.png
問246（薬理）
処方1及び処方2で期待される薬理作用として、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 98-246 を参照

**選択肢**
1. アドレナリンα1受容体を遮断して、ブドウ膜強膜流出経路からの眼房水の流出を促進する。
2. 炭酸脱水酵素を阻害して、眼房水の産生を抑制する。
3. アドレナリンβ2受容体を遮断して、眼房水の産生を抑制する。
4. コリンエステラーゼを阻害して毛様体筋を収縮させ、シュレム管からの眼房水の流出を促進する。
5. プロスタグランジンF2α受容体を刺激して、ブドウ膜強膜流出経路からの眼房水の流出を促進する。

**正解:** 2,5

**解説**
ラタノプロストは、プロスタグランジンF2α受容体を刺激して、ブドウ膜強膜流出経路からの眼房水の流出を促進することで眼内圧を低下させる。また、ブリンゾラミドは、炭酸脱水酵素を阻害し、眼房水の産生を抑制することで眼内圧を低下させる。
1　誤
アドレナリンα1受容体を遮断する緑内障治療薬は、ブナゾシンである。
2　正
ブリンゾラミドの薬理作用である。
3　誤
アドレナリンβ2受容体を遮断する緑内障治療薬は、チモロールなどである。
4　誤
コリンエステラーゼを阻害する緑内障治療薬は、ジスチグミンなどである。
5　正
ラタノプロストの薬理作用である。
