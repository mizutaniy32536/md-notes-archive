# 第98回 問296

**問題文**
35歳女性。体重52 kg。左前胸部の肋骨に沿って帯状にかゆみが発生し、発赤し、数日後に激しく痛み出した。帯状疱疹と診断され、神経ブロックで痛みをとりながら薬物治療を行った。しかし、皮疹が治った後も疼痛は3ヶ月以上続いた。
問296（病態・薬物治療）
この疾患の病態に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**
1. 両側性に症状がみられることが多い。
2. 抗水痘帯状疱疹ウイルス抗体価が高値でも帯状疱疹に罹患する。
3. 帯状疱疹では、その多くにウイルス血症を起こす。
4. ウイルス再感染が原因である。
5. 免疫力低下や過労が危険因子である。

**正解:** 2,5

**解説**
帯状疱疹は、水痘帯状疱疹ウイルス（VZV）の回帰感染が原因で片側性に、知覚神経に沿う帯状の紅斑や小水疱、神経性疼痛などが発症する感染症である。小児期に水痘に罹患することで、その後ウイルスが神経細胞に潜伏感染を続け、加齢や過労、免疫力が低下した際に帯状疱疹として回帰発症する。
1　誤
神経に沿って片側性に症状が現れるため、体の両側にまたがることはほとんどない。
2　正
成人のほとんどは、小児期に水痘に罹患することで、抗水痘帯状疱疹ウイルス抗体価は陽性となるが、免疫力低下に伴い潜伏感染していたウイルスが増殖し、帯状疱疹に罹患する。
3　誤
小児期のVZV初期感染時の水痘では、ウイルス血症が認められることが多いが、成人の帯状疱疹ではVZVが知覚神経に沿って局在しており、ウイルス血症を認めることは少ない。
4　誤
前記参照。
5　正
