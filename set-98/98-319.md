# 第98回 問319

**問題文**
在宅患者に対する薬剤師の業務について、以下の問いに答えよ。
問319（法規）
在宅医療・介護についての記述のうち、正しいのはどれか。2つ選べ。

**選択肢**
1. 在宅患者訪問薬剤管理指導は、介護保険法に基づく居宅サービスの1つである。
2. 介護支援専門員は、寝たきり状態の要介護者に対して必要な援助を行うことを業務としている。
3. 保険薬局の開設者は、居宅療養管理指導を行う指定居宅サービス事業者とみなされる。
4. 在宅患者に対する居宅療養管理指導の給付に当たって、保険薬局は、患者が介護保険の適用を受けていることを確認する。

**正解:** 3,4

**解説**
1　誤
健康保険法に基づく居宅サービスの1つである。
2　誤
介護支援専門員は、要介護者等からの相談に応じ、ケアプランを作成したり、行政及びサービス事業者との連絡調整を行う。寝たきり状態の要介護者に対して必要な援助を行うことを業務とするのは、介護福祉士や訪問介護員（ホームヘルパー）等である。
3　正
保険薬局の指定を受けた薬局は、居宅療養管理指導を行う指定居宅サービス業者とみなされる（みなし指定）。
4　正
保険薬局は、居宅療養管理指導を行うにあたっては、被保険者証の提示を求めるなどにより、患者が介護保険の適用を受けていることを確認しなければならない。
