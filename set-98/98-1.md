# 第98回 問1

**問題文**
双極子モーメントが最も大きい分子はどれか。1つ選べ。

**選択肢**
1. HF
2. HCl
3. HBr
4. HI
5. H2

**正解:** 1

**解説**
双極子モーメントとは、分子内部の電荷の偏りの程度のことであり、ベクトル量で表される。
ハロゲン水素化物では、電気陰性度が大きいハロゲンを含むものほど、電荷の偏りが大きくなるため、電気陰性度が大きくなる（HF＞HCl＞HBr＞HI）。また、H2のような等核二原子分子では、極性がなく、双極子モーメントは0である。よって、双極子モーメントが最も大きい分子は、HFである。
