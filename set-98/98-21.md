## ◆ 平成23年以前に国内でヒトへの感染報告がない感染症を選ぶシナリオ
260312

**問題**

**正解と理由**
- `5. 鳥インフルエンザ（H5N1）`
- **理由の構成**
	- 鳥インフルエンザH5N1は日本国内での`ヒトへの感染報告は平成23年以前にはない`

**解法の手順（思考の流れ）**
- 選択肢1：コレラ → 三類感染症、国内で年間数十件の報告あり → ✗
- 選択肢2：レジオネラ症 → 四類感染症、国内で年間数百件の報告あり → ✗
- 選択肢3：細菌性赤痢 → 三類感染症、国内報告あり（海外からの輸入例含む）→ ✗
- 選択肢4：つつが虫病 → 四類感染症、国内で年間数百件の報告あり → ✗
- 選択肢5：鳥インフルエンザ（H5N1）→ 二類感染症、日本国内でのヒト感染例はない → **○**

**キーポイント**
- `鳥インフルエンザH5N1` = `二類感染症`
- アジア・中東での感染例はあるが、日本国内でのヒト感染報告なし
- 鳥→ヒト感染は起こりうるが、ヒト→ヒト感染は限定的

**関連知識の比較**

| 感染症 | 分類 | 国内ヒト感染報告 |
|---|---|---|
| 鳥インフルエンザH5N1 | 二類 | **なし** |
| コレラ | 三類 | あり |
| 細菌性赤痢 | 三類 | あり |
| レジオネラ症 | 四類 | あり |
| つつが虫病 | 四類 | あり |

（一言メモ）
- 鳥インフルH5N1 = 国内ヒト感染例ゼロ。これは覚えておく
- H5N1は致死率60%超（海外）、だから二類に分類
