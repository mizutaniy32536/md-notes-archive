# 第98回 問150

**問題文**
薬剤師が患者に応対するときのコミュニケーション技法又は態度についての説明のうち、正しいのはどれか。2つ選べ。

**選択肢**
1. 開いた質問の利点として、必要な情報を限られた時間で効率よく得られることがあげられる。
2. 共感的態度とは、患者が感じていることを、薬剤師ができる限り同じように感じようとする態度である。
3. パターナリズムに基づく医療では、十分な説明を受けた患者の意思が医療の方針に反映される。
4. アサーションを用いた表現法では、患者の権利や主張を考慮に入れながら、薬剤師が主張したいことを伝える。

**正解:** 2,4

**解説**
1　誤
必要な情報を限られた時間で効率よく得られることができるのは、閉じた質問である。閉じた質問は、「はい」、「いいえ」など答えやすい質問であり、患者が答えやすいなどの利点がある。
2　正
共感的態度とは、薬剤師が患者の立場に立って同じように感じようとする態度である。
3　誤
パターナリズムに基づく医療では、医療従事者側が責任をもって患者の最善の利益の決定を行うため、患者の意思が医療の方針に反映されるわけではない。
4　正
アサーションを用いた表現法では、対立した意見を尊重しつつ、自分自身の考え方も主張する。
