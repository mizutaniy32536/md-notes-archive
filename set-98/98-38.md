# 98-38

**問題文**
セラトロダストの抗アレルギー作用の機序はどれか。1つ選べ。

**選択肢**
1. ロイコトリエン受容体遮断
2. トロンボキサンA2受容体遮断
3. シクロオキシゲナーゼ−2阻害
4. 5−リポキシゲナーゼ阻害
5. ホスホリパーゼA2阻害

**正解:** 2

5. 誤 ホスホリパーゼA2阻害ではない。
