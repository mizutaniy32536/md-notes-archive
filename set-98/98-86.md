# 第98回 問86

**問題文**
入院患者に麻薬を処方する場合､麻薬を含む院内処方せんに必ず記載しなければならないのはどれか｡1つ選べ。

**選択肢**
1. 患者の住所
2. 病院の所在地
3. 病院の名称
4. 麻薬施用者の免許証番号
5. 処方せんの使用期間

**正解:** 4

**解説**
麻薬処方せんには、医師法で規定されている記載事項（患者の氏名、年齢、薬名、分量、用法、用量、発行年月日、使用期間、病院若しくは診療所の名称及び所在地又は医師の住所、（医師の）記名押印又は署名）に加えて、患者の住所と麻薬施用者の免許証番号を記載しなければならない。
院内麻薬処方せんにおいては、患者の住所や処方せんの使用期間など一部の記載事項を省略することができるが、麻薬施用者の免許証番号は必ず記載しなければならない。
