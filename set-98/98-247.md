# 第98回 問247

**問題文**
47歳女性。眼科外来にて、以下の薬剤が処方された。
スクリーンショット 2017-04-17 11.12.50.png
問247（実務）
これらの処方薬を使用するにあたって薬剤師が患者に行うべき指導として、適切なのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 98-247 を参照

**選択肢**
1. 点眼前には石鹸などで十分に手を洗う。
2. 点眼液の容器の先を眼瞼に固定して点眼する。
3. 夕方の点眼時は、2剤の点眼間隔を1分間あける。
4. 点眼後はすぐに開眼し、眼球を上下左右に動かす。
5. 全身性の副作用を軽減するために、点眼後涙嚢部を圧迫する。

**正解:** 1,5

**解説**
1　正
点眼前には石鹸などで十分に手を洗い、清潔にしておく必要がある。
2　誤
点眼容器内の汚染を避けるため、点眼液の容器の先が眼瞼やまつ毛に触れないように点眼する。
3　誤
2種類以上の点眼剤を併用する場合、点眼間隔が短いと先に点眼した薬液が後に点眼した薬液によって洗い流され、十分な効果が得られない可能性がある。そのため、点眼間隔は5分以上あける必要がある。
4　誤
点眼後はしばらく眼を閉じ、涙嚢部を指で圧迫することで、点眼液が涙嚢部から鼻粘膜へ流れることを防ぐことで、全身性の副作用が現れることを予防する。
5　正
