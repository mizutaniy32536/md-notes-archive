# 第98回 問174

**問題文**
乳剤に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**
1. o/w型の乳剤は、電気伝導性を示さない。
2. 合一しても振とうすればもとの分散状態に戻る。
3. w/o/w型やo/w/o型などの多重乳剤がある。
4. 分散媒と分散相の密度差を小さくすると、乳剤の分散状態は安定化する。
5. w/o型の乳剤は、メチレンブルーを加えると全体が着色される。

**正解:** 3,4

**解説**
1　誤
分散相（外相）がw（水）であるo/w型の乳剤は、電気を通しやすいため、電気伝導性を示す。
2　誤
合一とは、クリーミングを起こした乳剤または凝集を起こした乳剤をそのまま放置すると分散媒（内相）の粒子同士が結合し大きな粒子になることである。合一した状態の乳剤は、振とうしてももとの分散状態には戻らず、再分散不可能である。
3　正
4　正
分散媒（内相）と分散相（外相）の密度差を小さくすると、分散媒（内相）が浮上及び沈降しにくくなるため、乳剤の分散状態は安定化する。
5　誤
メチレンブルーは水に溶けやすいため、w（水）相を着色する。そのため、w/o型の乳剤にメチレンブルーを加えると、分散媒（内相）は着色されるが、分散相（外相）は着色されないため、全体は着色されない。
