# 第98回 問23

**問題文**
シアン中毒の解毒剤として用いられる薬物はどれか。1つ選べ。

**選択肢**
1. チオ硫酸ナトリウム
2. プラリドキシムヨウ化物（PAM）
3. ナロキソン
4. デフェロキサミン
5. ジメルカプロール

**正解:** 1

**解説**
シアン中毒の解毒薬には、チオ硫酸ナトリウムや亜硝酸アミル、ヒドロキシコバラミンが用いられる。
1　正
2　誤
プラリドキシムヨウ化物（PAM）は、有機リン系化合物による中毒の解毒剤として用いられる。
3　誤
ナロキソンは、モルヒネによる急性中毒の解毒剤として用いられる。
4　誤
デフェロキサミンは、鉄による中毒の解毒剤として用いられる。
5　誤
ジメルカプロールは、ヒ素、水銀、鉛などによる中毒の解毒剤として用いられる。また、カドミウム、セレン、鉄による中毒では毒性が増すことがあるため使用できない。
