# 第98回 問78

**問題文**
後発医薬品を対象としていない制度はどれか｡1つ選べ。

**選択肢**
1. 再審査制度
2. 再評価制度
3. 副作用・感染症報告制度
4. 医薬品・医療機器等安全性情報報告制度
5. 医薬品副作用被害救済制度

**正解:** 1

**解説**
再審査制度は、新医薬品又は新医療機器に対して適用される制度であり、後発医薬品は対象外である。再評価制度、副作用・感染症報告制度、医薬品・医療機器等安全性情報報告制度、医薬品副作用被害救済制度に関しては、先発医薬品だけでなく後発医薬品も対象となる。
※法律改定により、現在では新医薬品又は新再生医療等製品が対象。
