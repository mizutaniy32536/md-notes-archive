# 第98回 問343

**問題文**
35歳男性。一般用医薬品を求めて来局した。胸焼けを訴えたので、商品A（ファモチジン製剤、第一類医薬品）又は商品B（制酸を目的とする製剤、第二類医薬品）を使用するのが良いと判断した。この時の対応として、正しいのはどれか。2つ選べ。

**選択肢**
1. 登録販売者が、書面を用いて情報提供し商品Aを販売した。
2. 薬剤師が、口頭で情報提供し商品Aを販売した。
3. 薬剤師が、書面を用いて情報提供し商品Bを販売した。
4. 登録販売者が、口頭で情報提供し商品Bを販売した。

**正解:** 3,4

**解説**
一般用医薬品の販売制限と販売時の情報提供義務について以下にまとめる。
スクリーンショット 2017-04-26 18.55.18.png
1　誤
登録販売者は、商品A（第一類医薬品）を販売することはできない。
2　誤
商品A（第一類医薬品）を販売する場合には、薬剤師が書面を用いて必要な情報を提供する必要がある。
3　正
4　正
