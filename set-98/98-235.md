# 第98回 問235

**問題文**
42歳女性。健康診断（人間ドック）で眼底出血を指摘され、薬局の薬剤師に相談した。
問235（衛生）
特定健康診査及び特定保健指導に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**
1. 特定健康診査は、メタボリックシンドロームに着目したものである。
2. 特定健康診査の結果、腹囲、血圧、脂質検査、血糖検査などの基本項目が、1つでも正常値から外れると、特定保健指導の対象者となる。
3. 特定保健指導には、リスクの程度に応じて、動機付け支援、積極的支援がある。
4. 特定健康診査は、三次予防に重点を置いている。

**正解:** 1,3

**解説**
1　正
特定健康診査は、メタボリックシンドロームに着目した健診であり、40〜74歳の医療保険加入者に対して、身体計測、血圧測定、血液検査等を行う。
2　誤
特定健康診査の結果から、生活習慣病の発症リスクが高く、生活習慣の改善による生活習慣病の予防効果が多く期待できる者に対して、生活習慣を見直すサポートをするシステムを特定保健指導という。腹囲が正常値から外れており、それに加えて、血圧、血糖値、脂質検査値などが正常値から外れた場合に特定保健指導の対象となる。
3　正
特定保健指導は、リスクの程度に応じて、動悸付け支援、積極的支援があり、よりリスクの高い者が積極的支援となる。
4　誤
特定健康診査は、生活習慣を改善することによって疾病の発生を未然に防ぐ一次予防及び早期発見・早期治療の二次予防に重点を置いている。
