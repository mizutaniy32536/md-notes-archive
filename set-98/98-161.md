# 98-161

**問題文**
抗アレルギー薬の薬物名及び主たる作用機序の組合せのうち、正しいのはどれか。2つ選べ。

**選択肢**
1. オザグレル　トロンボキサン合成酵素阻害
2. ザフィルルカスト　トロンボキサンA2受容体遮断
3. スプラタスト　IgE抗体産生抑制
4. ラマトロバン　ヒスタミンH1受容体遮断
5. オロパタジン　ロイコトリエン受容体遮断

**正解:** 1,3

**解説**
1. 正 オザグレルはトロンボキサン合成酵素を阻害する。
2. 誤 ザフィルルカストはロイコトリエン受容体拮抗薬である。
3. 正 スプラタストはIL-4産生抑制を介してIgE抗体産生を抑制する。
4. 誤 ラマトロバンはトロンボキサンA2/PGD2受容体系の遮断薬である。
5. 誤 オロパタジンはヒスタミンH1受容体遮断とメディエーター遊離抑制作用を持つ。

**キーポイント**
- オザグレルは合成酵素阻害、ザフィルルカストはLT受容体遮断。
- スプラタストはTh2サイトカイン抑制からIgE産生を抑える。
