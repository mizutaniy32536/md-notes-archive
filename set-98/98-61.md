# 第98回 問61

**問題文**
クッシング症候群の病態に関する記述のうち、誤っているのはどれか。1つ選べ。

**選択肢**
1. 満月様顔貌
2. 中心性肥満
3. 高カリウム血症
4. 耐糖能異常
5. 無月経

**正解:** 3

**解説**
クッシング症候群は、副腎皮質からコルチゾールやアルドステロン、アンドロゲンが過剰分泌される疾患群で、コルチゾール分泌過剰により中心性肥満（満月様顔貌、水牛様脂肪沈着）、アルドステロン分泌過剰により高ナトリウム血症、低カリウム血症、アンドロゲン分泌過剰により無月経などの症状が現れる。
