# 第98回 問17

**問題文**
毒素型食中毒を引き起こす菌はどれか。1つ選べ。

**選択肢**
1. Clostridium perfringens
2. Salmonella enterica serovar Enteritidis
3. Staphylococcus aureus
4. Vibrio parahaemolyticus
5. Campylobacter jejuni

**正解:** 3

**解説**
細菌による食中毒は、菌に感染することによって発症する感染型食中毒と、菌が産生する毒素を摂取することによって発症する毒素型食中毒の2つに大別される。
1　誤
ウェルシュ菌（Clostridium perfringens）は、感染型食中毒を引き起す。
2　誤
サルモネラ・エンテリティディス（Salmonella enterica serovar Enteritidis）は、感染型食中毒を引き起す。
3　正
黄色ブドウ球菌（Staphylococcus aureus）は、毒素型食中毒を引き起す。
4　誤
腸炎ビブリオ（Vibrio parahaemolyticus）は、感染型食中毒を引き起す。
5　誤
カンピロバクター（Campylobacter jejuni）は、感染型食中毒を引き起す。
