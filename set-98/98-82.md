# 第98回 問82

**問題文**
経口糖尿病用薬のうち、重度の腎障害患者に対して禁忌でないのはどれか｡1つ選べ。

**選択肢**
1. ボグリボース
2. メトホルミン塩酸塩
3. シタグリプチンリン酸塩水和物
4. グリメピリド
5. ピオグリタゾン塩酸塩

**正解:** 1

**解説**

注：シダクリプチンリン酸塩は、出題された当時は重度の腎障害患者に対して禁忌であったが、現在は「血液透析または腹膜透析を要する患者を含む重度腎機能障害患者」への投与禁忌が解除されている。
