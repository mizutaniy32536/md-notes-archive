# 第98回 問336

**問題文**
病院で働く職種と主な職務内容の組合せのうち、誤っているのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 98-336 を参照

**正解:** 4

**解説**
臨床工学技師は、生命維持管理装置（人工呼吸器、人工心肺装置、血液浄化装置など）の操作や保守点検を行う。なお、職務内容が「身体に障害のある者に対し、基本的動作能力の回復を図るため、治療体操等の運動や物理的手段を加える」である職種は、理学療法士である。
