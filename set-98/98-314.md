# 第98回 問314

**問題文**
45歳男性から、「特定健康診査を受けたところ、医師から尿酸値が高いと指摘された。食生活を改善するように言われたが、何に注意すればいいかわからない。」と薬局の薬剤師に相談があった。
問314（実務）
摂取を控えるよう指導すべき食品はどれか。最も適切なものを2つ選べ。

**選択肢**
1. ビール
2. 牛乳
3. ほうれん草
4. 鶏レバー
5. 鶏卵

**正解:** 1,4

**解説**
尿酸はプリン体の最終代謝物であるため、プリン体を多く含む食品（鶏レバー、イワシ干物、あんこうの肝など）の摂取は控えるよう指導するべきである。また、アルコール飲料は尿酸の排泄を低下させることで血清尿酸値を上昇させるため、アルコール飲料（ビールなど）の摂取も控えるよう指導する。
