# 第98回 問142

**問題文**
消費者への販売に当たり、医薬品医療機器法に基づく販売業の許可及び薬局開設の許可のいずれも必要としないのはどれか。1つ選べ。

**選択肢**
1. 高度管理医療機器
2. 特定保守管理医療機器
3. 指定医薬部外品（有効成分の名称・分量を表示すべきものとして厚生労働大臣が指定した医薬部外品）
4. 日本薬局方ブドウ酒
5. 人の身体の構造に影響を及ぼすことが目的とされている物であって、 機械器具等でないもの（医薬部外品及び化粧品を除く｡）

**正解:** 3

**解説**
1　許可が必要である
高度管理医療機器及び特定保守管理医療機器を消費者へ販売するに当たっては、販売業の許可が必要である（医薬品医療機器法第39条）。
2　許可が必要である
解説1参照。
3　許可を必要としない
指定医薬部外品を消費者へ販売するに当たっては、特別な許可は必要としない。
4　許可が必要である
日本薬局方ブドウ酒は、医薬品に該当するため、医薬品販売業の許可又は薬局開設の許可を受けなければ、販売することはできない（医薬品医療機器法第24条）。
5　許可が必要である
人の身体の構造に影響を及ぼすことが目的とされている物であって、 機械器具等でないものは、医薬品に該当するため、医薬品販売業の許可又は薬局開設の許可を受けなければ、販売することはできない（医薬品医療機器法第24条）。
