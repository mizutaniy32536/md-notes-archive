# 第98回 問172

**問題文**
体内動態が線形性を示す薬物Aは、肝代謝と腎排泄によって体内から消失し、正常時における肝代謝クリアランスは全身クリアランスの20％である。また、腎疾患時に薬物Aの肝代謝クリアランスは変化しないが、腎排泄クリアランスは糸球体ろ過速度（GFR）に比例して変化する。
薬物Aを投与中の患者において、GFRが正常時の25％に低下したとする。薬物Aの血中濃度時間曲線下面積（AUC）を腎機能正常時と同じにするには、投与量を腎機能正常時の何％に変更すればよいか。最も近い値を1つ選べ。

**選択肢**
1. 20％
2. 40％
3. 80％
4. 120％
5. 250％

**正解:** 2

**解説**
設問に「体内動態が線形性を示す薬物Aは、肝代謝と腎排泄によって体内から消失し、正常時における肝代謝クリアランスは全身クリアランスの20％である」とあることから薬物Aの正常時における腎排泄クリアランスは全身クリアランスの80％であるといえる。
CL肝（正常）＝0.2 CLtot（正常）　　　CL腎（正常）＝0.8 CLtot（正常）
CLtot（正常）：腎機能正常時の全身クリアランス
CL肝（正常）：腎機能正常時の肝クリアランス
CL腎（正常）：腎機能正常時の腎クリアランス
また、「腎疾患時に薬物Aの肝代謝クリアランスは変化しないが、腎排泄クリアランスは糸球体ろ過速度（GFR）に比例して変化する。」とあることから、GFRが正常時の25％に低下した場合の患者の全身クリアランスは、以下のように求めることができる。

CLtot（腎疾患）＝CL腎（腎疾患）＋CL肝（腎疾患）＝0.2 CLtot（正常）＋0.2 CLtot（正常）

CLtot（腎疾患）：腎疾患時の全身クリアランス
CL肝（腎疾患）：腎疾患時の肝クリアランス

①式より、血中濃度時間曲線下面積は体内薬物移行量に比例し、全身クリアランスに反比例することがわかる。また、体内薬物移行量は投与量に比例することから、血中濃度時間曲線下面積は投与量に比例することがわかる。
これらのことより、薬物Aの血中濃度時間曲線下面積（AUC）を腎機能正常時と同じにするには、投与量を腎機能正常時の40％に変更すればよい。
