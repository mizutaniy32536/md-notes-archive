# 第98回 問341

**問題文**
ペン型インスリン製剤を患者に交付した後の使用方法及び保管方法の記述のうち、適切なのはどれか。2つ選べ。

**選択肢**
1. 使用前には空打ちを行う。
2. 毎日同一部位に注射する。
3. 未使用の製剤は、凍結して保管する。
4. 未使用の製剤は、室温で保管する。
5. 使用開始後の製剤は、室温で保管する。

**正解:** 1,5

**解説**
1　正
ペン型インスリン製剤を使用する前に、インスリンが出ることの確認及び注射針内の空気を抜くために注射器を上に向け空打ちを行う必要がある。
2　誤
ペン型インスリン製剤を使用する際、毎日同一部分に注射せず、2〜3 cmずつ注射部分をずらす必要がある。
3　誤
未使用のペン型インスリン製剤は、2〜8℃（冷所）で保管する。
4　誤
解説4参照
5　正
