# 第98回 問238

**問題文**
50歳男性。近医にて高血圧、不眠症のため以下の薬剤が処方され来局した。
スクリーンショット 2017-04-14 12.52.43.png
問238（実務）
薬剤師が薬歴を確認したところ、他院より爪白癬のためイトラコナゾール錠が処方されており、服用中であった。そこで、今回の処方医に疑義照会を行い処方薬の変更を提案した。その内容として、最も適切なのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 98-238 を参照

**選択肢**
1. エナラプリルマレイン酸塩錠をテルミサルタン錠に変更
2. エナラプリルマレイン酸塩錠をニフェジピン錠に変更
3. トリアゾラム錠をゾルピデム錠に変更
4. トリアゾラム錠をゾピクロン錠に変更
5. トリアゾラム錠をジアゼパム錠に変更

**正解:** 3

**解説**
抗真菌薬であるイトラコナゾールは、CYP3A4阻害作用を有しており、併用する薬物との相互作用に注意が必要であり、処方薬のうち、トリアゾラム錠はCYP3A4によって代謝されるため、イトラコナゾールとの併用は禁忌である。
1　誤
エナラプリルマレイン酸塩錠は、CYP3A4によって代謝される報告はないため、テルミサルタン錠への変更は必要ないと考えられる。
2　誤
エナラプリルマレイン酸塩錠は、CYP3A4によって代謝される報告はないため、ニフェジピン錠への変更は必要ないと考えられる。
3　正
ゾルピデムは超短時間型非ベンゾジアゼピン系睡眠薬であり、CYP3A4によって代謝されるが、イトラコナゾールとの併用による薬物動態への影響は報告されていないため、トリアゾラム錠からゾルピデム錠への処方変更は適切である。
4　誤
ゾピクロンは超短時間型非ベンゾジアゼピン系睡眠薬であるが、CYP3A4によって代謝されるため、イトラコナゾール錠とは併用注意となっている。そのため、トリアゾラム錠からゾピクロン錠への処方変更は適切ではない。
5　誤
ジアゼパム錠は抗不安薬であり、睡眠薬ではないため、トリアゾラム錠からジアゼパム錠への変更は適切ではない。
