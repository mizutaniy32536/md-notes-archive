# 第98回 問303

**問題文**
68歳男性。身長160 cm、体重50 kg。2年前に大腸がんの手術を受けたが、再発を認めたため、以下の処方(FOLFIRI)にて治療を受けることになった。
スクリーンショット 2017-04-17 12.39.25.png
問303（病態・薬物治療）
この患者において、UDP−グルクロン酸転移酵素(UGT)の遺伝子多型検査をした結果、UGT1A1*6のホモ接合体であったため、FOLFOX6に処方を変更することにした。正しいのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 98-303 を参照

**選択肢**
1. グラニセトロン塩酸塩注射液をパロノセトロン塩酸塩注射液に変更
2. グラニセトロン塩酸塩注射液をアプレピタントカプセルに変更
3. イリノテカン塩塩酸塩水和物注射液をオキサリプラチン注射液に変更
4. イリノテカン塩塩酸塩水和物注射液をエピルビシン塩酸塩注射液に変更
5. フルオロウラシル注射液をシタラビン注射液に変更
6. フルオロウラシル注射液をメトトレキサート注射液に変更

**正解:** 3

**解説**
イリノテカンの活性代謝物であるSN−38は、グルクロン酸抱合を受け胆汁中に排泄される。UGT1A1*6のホモ接合体である場合、SN−38の排泄が低下して副作用が増強するおそれがある。そのため、この患者においてはイリノテカンをオキサリプラチンに変更したFOLFOX療法で治療を行う。
