# 第98回 問13

**問題文**
タンパク質の翻訳後修飾において、糖鎖による修飾を受けるアミノ酸残基はどれか。1つ選べ。

**選択肢**
1. L−アラニン
2. L−システイン
3. L−トリプトファン
4. L−アスパラギン
5. L−グルタミン酸

**正解:** 4

**解説**
タンパク質の翻訳後修飾において、糖鎖による修飾を受けるアミノ酸残基には、L−アスパラギン残基、L−トレオニン（L−スレオニン）残基、L−セリン残基などがあげられる。なお、糖鎖修飾により、L−アスパラギン残基のカルバモイル基には、N−アセチル−β−D−グルコサミンが結合し、L−トレオニン（L−スレオニン）残基、L−セリン残基の水酸基には、N−アセチル−β−D−ガラクトサミンが結合する。
