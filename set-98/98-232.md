## ◆ 任意接種のみの疾病を選ぶシナリオ
260312

**問題**
小児と一緒に薬局を訪れた母親から、薬剤師に対して予防接種に関する相談があった。
小児に予防接種を受けさせる場合に、接種時期に関わらず任意接種となることを説明すべき疾病はどれか。２つ選べ。

**選択肢**

1. ジフテリア
​
2. インフルエンザ
​
3. 流行性耳下腺炎
​
4. ポリオ

**正解と理由**
- `2. インフルエンザ`
- `3. 流行性耳下腺炎（おたふくかぜ）`

**解法の手順（思考の流れ）**
- 選択肢1：ジフテリア → 四種混合ワクチン（DPT-IPV）として`定期接種あり` → ✗
- 選択肢2：インフルエンザ → 小児には`任意接種のみ`（高齢者は定期接種）→ **○**
- 選択肢3：流行性耳下腺炎（おたふくかぜ）→ `任意接種のみ`（定期接種化されていない）→ **○**
- 選択肢4：ポリオ → 四種混合ワクチンとして`定期接種あり` → ✗

**キーポイント**
- 任意接種代表例（小児）：`インフルエンザ`、`おたふくかぜ`、水痘（2014年定期接種化）、ロタウイルス（2020年定期接種化）
- ジフテリア・ポリオ = 四種混合（DPT-IPV）として定期接種

**関連知識の比較**

| ワクチン | 分類（小児） |
|---|---|
| インフルエンザ | **任意接種** |
| おたふくかぜ | **任意接種** |
| ジフテリア | 定期接種（四種混合） |
| ポリオ | 定期接種（四種混合） |
| 水痘 | 定期接種（2014年〜） |

（一言メモ）
- インフルエンザ（小児）= 任意接種。定期は高齢者のみ！
- おたふくかぜは今も任意接種（2025年時点で定期化検討中）
