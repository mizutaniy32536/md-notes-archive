# 第98回 問4

**問題文**
最も強い陽イオン交換能をもつ樹脂の交換基はどれか。1つ選べ。

**選択肢**
1. 2級アミン
2. 3級アミン
3. スルホン酸
4. カルボン酸
5. 4級アンモニウム

**正解:** 3

**解説**
交換基として、スルホン酸及びカルボン酸は、マイナスに荷電するので、陽イオンを交換する側鎖になる。スルホン酸は強酸であり、カルボン酸は弱酸であることから、陽イオン交換能の大小は、スルホン酸＞カルボン酸となる。なお、交換基として、アミンはプラスに荷電するので、陰イオンを交換する側鎖になる。
