# 第98回 問5

**問題文**
「溶液は赤紫色を呈し、その硫酸酸性溶液に過酸化水素試液を加えるとき、泡だって脱色する」ことによって確認される化合物はどれか。1つ選べ。

**選択肢**
1. 過マンガン酸塩
2. 臭素酸塩
3. 第一鉄塩
4. 第二銅塩
5. ヨウ化物

**正解:** 1

**解説**
日本薬局方において、過マンガン酸塩は「溶液は赤紫色を呈し、その硫酸酸性溶液に過酸化水素試液を加えるとき、泡だって脱色する」と規定されていることから、確認される化合物は、過マンガン酸塩であるといえる。スクリーンショット 2017-04-17 16.12.37.png
1　正
2　誤
日本薬局方において、臭素酸塩は、「硝酸酸性溶液に硝酸銀試薬を加えるとき、白色の結晶性沈殿を生じ、加熱するとき、沈殿は溶ける。これに、亜硝酸ナトリウム試液を追加するとき、淡黄色の沈殿を生じる」と規定されている。
3　誤
日本薬局法において、第一鉄塩は、「弱酸性溶液にヘキサシアノ鉄（Ⅲ）酸カリウムを加えるとき、青色の沈殿を生じ、希塩酸を追加しても沈殿は溶けない」と規定されている。
4　誤
日本薬局方において、第二銅塩は、「ヘキサシアノ鉄（Ⅱ）酸カリウムを加えるとき、赤褐色の沈殿を生じ、この一部に希硝酸を追加しても沈殿は溶けない」と規定されている。
5　誤
日本薬局法において、ヨウ化物は、「硝酸銀試液を加えるとき、黄色い沈殿を生じる」と規定されている。
