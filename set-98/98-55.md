# 第98回 問55

**問題文**
日本薬局方で散剤に対して規定されている試験はどれか。1つ選べ。

**選択肢**
1. エンドトシキン試験法
2. 不溶性微粒子試験法
3. 微生物限度試験法
4. 重金属試験法
5. 溶出試験法

**正解:** 5

**解説**
散剤とは、「経口投与する粉末状の製剤」のことであり、日本薬局方散剤に規定されている試験法は、溶出試験及び製剤均一性試験法（分包品）である。
1　誤
エンドトキシン試験法は、注射剤、注射剤に添付された溶解液、及び水性注射剤の溶剤のうち、皮内、皮下及び筋肉内投与のみに用いるものを除くもの、透析用剤などに適用される。
2　誤
不溶性微粒子試験法は、注射剤及び添付された溶解液・腹膜透析用剤、点眼剤及び添付された溶解液などに適用される。
3　誤
微生物限度試験は、医薬品の原料や製剤が既定の微生物学的品質規格に適合するか否かを判断することを主目的としたものである。
4　誤
重金属試験は、エキス剤や流エキス剤などに適用される。
5　正
溶出試験法は、経口製剤について溶出試験規格に適合しているかどうかを判定するために行うものであり、錠剤や散剤以外にも、懸濁剤、シロップ剤（懸濁したもの）、経口ゼリーに適用される。
