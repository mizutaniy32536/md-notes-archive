## ◆ 予防接種に関する正しい記述を選ぶシナリオ
260312

**問題**
小児と一緒に薬局を訪れた母親から、薬剤師に対して予防接種に関する相談があった。
予防接種に関する記述のうち、正しいのはどれか。２つ選べ。

**選択肢**

1. 我が国においては、麻疹の感染は完全に抑えられているとはいえない。
​
2. 麻疹と風疹の混合ワクチンの接種時期は、生後12～24月未満と小学校就学前である。
​
3. 結核の予防接種は行われていない。
​
4. DPT混合ワクチンには、百日咳菌のトキソイドが含まれている。

**正解と理由**
- `1. 我が国においては、麻疹の感染は完全に抑えられているとはいえない`
- `2. 麻疹と風疹の混合ワクチンの接種時期は、生後12〜24月未満と小学校就学前`

**解法の手順（思考の流れ）**
- 選択肢1：麻疹は減少したが、輸入例等からの流行事例が今もある → **正しい**
- 選択肢2：MRワクチン = 1期（1歳〜2歳未満）、2期（小学校入学前1年間）→ **正しい**
- 選択肢3：BCGワクチンが結核の定期接種として行われている → **誤り**
- 選択肢4：DPT = ジフテリアトキソイド＋`破傷風トキソイド`＋`百日咳（不活化菌体）`。百日咳はトキソイドではなく不活化ワクチン → **誤り**

**キーポイント**
- MRワクチン接種時期：
  - 1期：`生後12月〜24月未満`
  - 2期：`5〜7歳未満（小学校就学前の1年間）`
- DPTワクチン成分：
  - `ジフテリア` = トキソイド（毒素無毒化）
  - `破傷風` = トキソイド（毒素無毒化）
  - `百日咳` = `不活化ワクチン`（トキソイドではない）

（一言メモ）
- 百日咳はトキソイドではなく不活化ワクチン！これはひっかけ頻出
- MRワクチンは1・2期の時期を正確に覚える
