# 第98回 問339

**問題文**
患者が入院時に持ち込んだ医薬品（持参薬）等に関する入院先の医療機関の薬剤師の対応として、適切なのはどれか。2つ選べ。

**選択肢**
1. 当該医療機関の採用薬ではなかったので、持参薬の入院中の服用状況を診療録に記載しなかった。
2. 同一銘柄の医薬品が採用されていなかったので、持参薬の情報に加えて院内で採用されている同種同効薬の情報を担当医に提供した。
3. 持参薬を粉砕し分包したので、調剤料を算定した。
4. 持参薬以外にサプリメントの使用について確認した。

**正解:** 2,4

**解説**
1　誤
診療録には、患者の治療状況を記載する必要があるため、持参薬が医療機関の採用薬でない場合でも、薬剤師は持参薬の入院中の服用状況を診療録に記載する必要がある。
2　正
持参薬が同一銘柄の医薬品が当該医療機関で採用されていない場合、担当医が今後の治療薬を選択するために、薬剤師は持参薬の情報に加えて院内で採用されている同種同効薬の情報を提供する必要がある。
3　誤
持参薬を粉砕し分包しても、調剤料を算定することはできない。
4　正
