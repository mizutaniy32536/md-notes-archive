# 第98回 問335

**問題文**
漢方薬に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**
1. 虚実や陰陽などの体質や病態に応じて使用される。
2. 漢方処方に用いられる生薬は、すべて日本薬局方に収載されている。
3. 西洋医学的には全く異なる疾患であっても、同一の漢方薬が処方されることがある。
4. 漢方薬と西洋薬の組合せに併用禁忌はない。

**正解:** 1,3

**解説**
1　正
漢方薬は、虚実や陰陽など患者の体質や病態に応じて使用される。
2　誤
漢方処方に用いられる主な生薬に関しては、日本薬局方に収載されているが、一部の生薬については未収載のものがある。
3　正
漢方薬は複数の成分を配合しており、また症状や患者の状態などを考慮して処方されるため、全く異なる疾患であっても、同一の漢方薬が処方されることがある。
4　誤
漢方薬と西洋薬の組み合わせに併用禁忌の組み合わせは存在する。例としては、小柴胡湯とインンターフェロン製剤などがある。
