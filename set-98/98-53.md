# 第98回 問53

**問題文**
薬物の結晶多形を検出できる方法はどれか｡1つ選べ。

**選択肢**
1. 空気透過法
2. Ⅹ線回折法
3. 旋光度法
4. 粘度測定法
5. 気体吸着法

**正解:** 2

**解説**
同一の化学組成を持ちながら、結晶内で原子または分子の配列が異なる結晶のことを結晶多形という。結晶多形では、密度、融点、溶解度、融解熱、化学ポテンシャルが異なるため、密度、融点、溶解度の測定を行うことにより、結晶多形を検出することができる。また、結晶多形は、X線回折法、熱分析、赤外（IR）スペクトル、固体NMRによって検出することも可能である。
