# 98-34

**問題文**
急性膵炎の治療に用いられるタンパク質分解酵素阻害薬はどれか。1つ選べ。

**選択肢**
1. プロパンテリン
2. ウルソデオキシコール酸
3. フロプロピオン
4. ニザチジン
5. ナファモスタット

**正解:** 5

**解説**
解答　5

急性膵炎の治療に用いられるタンパク質分解酵素阻害薬はナファモスタットである。
