# 第98回 問183

**問題文**
再生不良性貧血に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**
1. 発症の原因の1つにウイルス感染症がある。
2. 末梢血の血小板数が増加する。
3. 末梢血の網状赤血球数は正常である。
4. 血清鉄値が低下する。
5. エリスロポエチン産生が亢進する。

**正解:** 1,5

**解説**
1　正
再生不良性貧血は何らかの原因による骨髄障害で造血環境にトラブルが生じ、赤血球やその前駆細胞のみならず、白血球、血小板のいずれもが再生不良となる病態であり、汎血球減少症状を起こす。その原因には薬剤性、放射線、ウイルス感染症など様々な原因がある。
2　誤
末梢血の血小板数は減少する（解説1参照）。
3　誤
赤血球の前駆細胞である網状赤血球数は低下する（解説1参照）。
4　誤
汎血球減少により造血が低下することで鉄などの造血に必要な因子が消費されずに血清鉄値は増加する。また、貧血症状を代償するため腎でエリスロポエチン産生が亢進する。
5　正
解説4参照
