# 第98回 問337

**問題文**
法律に基づいた医薬品の管理及び取扱いに関する記述のうち、適切なのはどれか。1つ選べ。

**選択肢**
1. フェンタニルクエン酸塩注射液とメチルフェニデート塩酸塩錠を、鍵のかかる同一の保管庫で保管した。
2. 麻薬施用者が、疾患の治療を目的としてジアセチルモルヒネ散を施用した。
3. 1％のコデインリン酸塩水和物散を、麻薬保管庫に保管しなかった。
4. スキサメトニウム塩化物注射液を、劇薬用の冷所保管庫に保管した。
5. セレギリン塩酸塩錠を、他の医薬品と区別して覚せい剤保管庫に保管した。

**正解:** 3

**解説**
1　誤
麻薬は覚せい剤以外の医薬品と区別して、鍵のかかる堅固な施設に保管する必要がある。そのため、麻薬に該当するフェンタニルクエン酸塩注射液と向精神薬に該当するメチルフェニデート塩酸塩錠は、同一の保管庫に保管することはできない。
2　誤
ジアセチルモルヒネ（ヘロイン）は、麻薬及び向精神薬取締法によって、原則、施用することが禁止されている。そのため、麻薬施用者が、疾患の治療を目的としてジアセチルモルヒネ散を施用することはできない。
3　正
家庭麻薬（非麻薬）に該当する1％のコデインリン酸塩水和物散は、麻薬保管庫に保管する必要はない。
4　誤
毒薬に該当するスキサメトニウム塩化物注射液は、毒薬用の冷所保管庫に保管する必要がある。
5　誤
覚せい剤原料は、鍵のかかる施設に保管する必要はあるが、他の医薬品と区別して保管する必要はない。そのため、覚せい剤原料に該当するセレギリン塩酸塩錠は鍵のかかる保管庫に保管する必要はあるが、他の医薬品と区別して保管する必要はない。
