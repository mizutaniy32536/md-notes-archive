# 第98回 問156

**問題文**
抗てんかん薬に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**
1. カルバマゼピンは、電位依存性Na＋チャネルを遮断し、強直間代発作を抑制する。
2. バルプロ酸は、セロトニン5-HT1A受容体を遮断し、すべての型の全般発作を抑制する。
3. プリミドンはGABAトランスアミナーゼを阻害するので、フェノバルビタールとの併用で相乗効果が期待できる。
4. クロナゼパムは、ベンゾジアゼピン受容体を遮断し、複雑部分発作を抑制する。
5. エトスクシミドは、T型Ca2＋チャネルを遮断し、欠神発作を抑制する。

**正解:** 1,5

**解説**
1　正
カルバマゼピンは、電位依存性Na＋チャネルを遮断し、抗けいれん作用を示す。強直間代発作や精神運動発作（複雑部分発作）などに用いられる。
2　誤
バルプロ酸は、GABAトランスアミナーゼを阻害することで、脳内GABA濃度を増加させ抗けいれん作用を示す。すべての発作型に用いられる。
3　誤
プリミドンは、生体内でフェノバルビタールに変換され、GABAA-Cl－チャネル複合体のバルビツレート結合部に結合することで抗けいれん作用を示す。プリミドンは、フェノバルビタールと作用点が同じであるため、相加効果は期待できるが、相乗効果は期待できない。
4　誤
クロナゼパムは、ベンゾジアゼピン系薬であり、ベンゾジアゼピン受容体を刺激することで抗けいれん作用を示す。精神運動発作（複雑部分発作）などに用いられる。
5　正
エトスクシミドは、T型Ca2＋チャネルを遮断し、抗けいれん作用を示す。欠神発作に用いられる。
