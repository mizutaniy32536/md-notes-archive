# 98-63

**問題文**
アレルギー性鼻炎の治療薬として、正しいのはどれか。1つ選べ。

**選択肢**
1. シメチジン
2. タクロリムス水和物
3. クラリスロマイシン
4. レバミピド
5. セチリジン塩酸塩

**正解:** 5

**解説**
1. 誤 シメチジンはH2受容体遮断薬で、消化性潰瘍などに用いる。
2. 誤 タクロリムス水和物は免疫抑制薬である。
3. 誤 クラリスロマイシンはマクロライド系抗菌薬である。
4. 誤 レバミピドは胃粘膜保護薬である。
5. 正 セチリジン塩酸塩は第二世代抗ヒスタミン薬で、アレルギー性鼻炎に用いられる。

**キーポイント**
- アレルギー性鼻炎の基本薬は第二世代抗ヒスタミン薬や点鼻ステロイド。
- 他選択肢は適応がまったく別物なので、薬効分類で即切りできる。
