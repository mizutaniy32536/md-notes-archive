# 第98回 問187

**問題文**
50歳男性。徐々に筋力低下及び筋萎縮を認めた。検査の結果、萎縮は神経原性と判明した。以下の疾患のうち該当する疾患はどれか。1つ選べ。

**選択肢**
1. 多発性筋炎
2. 遠位型ミオパチー
3. 筋萎縮性側索硬化症
4. 筋強直性ジストロフィー
5. 進行性筋ジストロフィー

**正解:** 3

**解説**
筋萎縮の原因は、筋を支配している脊髄のような中枢神経や運動神経に異常が起こる神経原性（ニューロパチー）と、筋肉そのものに異常が起こる筋原性（ミオパチー）に大別される。神経原性のものには、筋萎縮性側索硬化症などがあり、筋原性のものには、多発性筋炎、遠位型ミオパチー、筋強直性ジストロフィー、進行性筋ジストロフィーなどがある。
