# 第98回 問64

**問題文**
関節リウマチに関する記述のうち、誤っているのはどれか。1つ選べ。

**選択肢**
1. 女性に多い。
2. 初期症状に、朝の手指のこわばりがある。
3. 関節の炎症は、対称性に起こる。
4. 主な関節病変は、滑膜炎である。
5. 腫瘍壊死因子α（TNF-α）の産生低下により発症する。

**正解:** 5

**解説**
関節リウマチは、Ⅲ型アレルギーが関与する関節滑膜の慢性的な炎症であり、女性に多く、関節の炎症は左右対称性に起こる。その症状は、関節変形を伴い、特に朝のこわばり（起床時に強く現れる関節のこわばり）が特徴的である。
また、関節リウマチ患者の滑膜炎症部位からはプロスタグランジンなどの炎症性ケミカルメディエーターや、TNF-α（腫瘍壊死因子α）、インターロイキンなどの炎症性サイトカインが分泌されており、関節リウマチの病態形成に関与している。
