# 98-57

**問題文**
以下の胃潰瘍治療薬のうち、高プロラクチン血症を起こす危険性のある薬物はどれか。1つ選べ。

**選択肢**
1. ミソプロストール
2. プログルミド
3. スルピリド
4. ピレンゼピン塩酸塩水和物
5. スクラルファート水和物

**正解:** 3

**正解と理由**
スルピリドはドパミンD2受容体遮断作用を示す。ドパミンはプロラクチン分泌を抑制しているため、その遮断により高プロラクチン血症を起こしうる。

**他の選択肢が誤りな理由**
1. ミソプロストールはPGE1誘導体であり、D2遮断による高プロラクチン血症とは結び付かない。
2. プログルミドはガストリン関連の薬であり、D2遮断作用はない。
4. ピレンゼピンはM1受容体遮断薬である。
5. スクラルファートは粘膜保護薬であり、ドパミン系への直接作用はない。
