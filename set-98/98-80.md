# 第98回 問80

**問題文**
医療を受ける患者の権利として認めるべきものはどれか｡1つ選べ。

**選択肢**
1. 別の患者の薬剤服用歴を閲覧する。
2. 希望する薬剤の処方を医師に強要する｡
3. 自動車運転を避けるべき旨の服薬指導に従わない。
4. 医学研究に参加することを拒否する｡
5. 受けた治療の自己負担金の支払いを拒否する｡

**正解:** 4

**解説**
医療を受ける患者の権利を述べているものとして、「リスボン宣言」がある。リスボン宣言の中には、患者の自己決定権などが明記されており、医学研究（治験）への参加は、患者の同意が必要であり、患者自身が参加することを拒否することが認められている。
