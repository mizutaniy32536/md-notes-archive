# 第98回 問321

**問題文**
ある病院が製薬企業からの依頼を受けて治験を実施することになった。
問321（法規）
治験に関して製薬企業が行った業務について、正しいのはどれか。1つ選べ。

**選択肢**
1. 第Ⅱ相試験の開始に先立ち、厚生労働大臣に治験の計画を届け出た。
2. 治験責任医師に治験実施計画書の作成を依頼した。
3. 治験薬管理者に治験薬概要書の作成を依頼した。
4. 治験施設支援機関（SMO）に治験協力者（CRC）の派遣を委託した。
5. モニタリング担当者が当該治験の監査を実施した。

**正解:** 1

**解説**
1　正
製薬企業は、第Ⅰ相、第Ⅱ相、第Ⅲ相それぞれの相において治験を開始する前にあらかじめ治験計画を厚生労働大臣に届け出なければならない。
2　誤
治験実施計画書は、製薬企業が作成しなければならない。
3　誤
治験薬概要書は、製薬企業が作成しなければならない。
4　誤
治験実施支援機関（SMO）は治験実施医療機関からの依頼に基づき、治験業務をサポートするための組織であり、治験実施体制の整備等を行うほか、治験協力者（CRC）を派遣する業務を行う。したがって、SMOにCRCの派遣を委託するのは、実施医療機関であり、製薬企業ではない。
5　誤
監査を担当する監査担当者は、医薬品の開発に係る部門及びモニタリングを担当する部門に所属してはならない。
