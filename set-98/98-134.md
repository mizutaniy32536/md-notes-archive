# 第98回 問134

**問題文**
電離放射線の人体影響に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**
1. 影響は、確定的影響と確率的影響とに分けることができる。
2. 確定的影響には、しきい値が存在しない。
3. 等価線量は、人体への被曝線量を評価するために用いられる。
4. 酸素効果とは、酸素の存在により放射線の影響が減弱されることである。
5. 脂肪組織は、骨髄組織と同程度の感受性を示す。

**正解:** 1,3

**解説**
1　正
電離放射線の人体影響は確定的影響（しきい値注1が存在し、しきい値以上の線量の被曝により影響が発生し、受ける線量が多くなるほど障害の大きさが増大する。）と確率的影響（しきい値が存在せず、受ける線量が多くなるほど障害の発生率が増大するが、障害の大きさは変化しない）とに分けることができる。
注1：影響があらわれる最低の線量
2　誤
解説1参照
3　正
人体への被曝線量を評価するために等価線量（放射線が人体を通過したときに与えるエネルギーに加え、放射線の種類よる人体への影響の違いを考慮した放射線量）や実効線量（各臓器、各組織の放射線の感受性を考慮した生体影響を表した放射線量）を用いる。
4　誤
酸素効果とは、酸素の存在により放射線の影響が増強される現象のことである。
5　誤
細胞分裂が盛んな細胞ほど、未分化である細胞ほど放射線感受性は高い。そのため、骨髄組織に比べ、脂肪組織の放射線感受性は低い。
