# 第98回 問45

**問題文**
ある薬物が併用薬物の体内動態を変動させる要因のうち、併用薬物の血中濃度を低下させるのはどれか。1つ選べ。

**選択肢**
1. 消化管吸収の促進
2. 肝代謝酵素の阻害
3. 胆汁排泄の阻害
4. 腎尿細管再吸収の阻害
5. 腎尿細管分泌の阻害

**正解:** 4

**解説**
薬物を併用した際、血中濃度が低下する要因には、腎尿細管再吸収の阻害、消化管吸収の阻害、血漿タンパク置換などが挙げられる。一方、薬物を併用した際、血中濃度が増加する要因には、消化管吸収の促進、肝代謝酵素の阻害、胆汁排泄の阻害、腎尿細管分泌の阻害などが挙げられる。
