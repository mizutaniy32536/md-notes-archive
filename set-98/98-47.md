# 第98回 問47

**問題文**
薬物動態に線形性が成り立っているとき､経口投与後の平均吸収時間を算出する式はどれか｡1つ選べ。ただし､経口投与後と静脈内投与後の平均滞留時間（MRT）を､それぞれMRTpoとMRTivとする｡

**選択肢**
1. MRTpo＋MRTiv
2. MRTpo－MRTiv
3. MRTiv－MRTpo
4. MRTpo／MRTiv
5. MRTiv／MRTpo

**正解:** 2

**解説**
経口投与後の平均吸収時間は、経口投与後の平均滞留時間（MRTpo）と静脈内投与後の平均滞留時間（MRTiv）の差より求めることができる。経口投与後の平均吸収時間
（MAT）を算出する式は、以下のように表すことができる。
MAT＝MRTpo－MRTiv
