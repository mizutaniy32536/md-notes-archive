# 第98回 問218

**問題文**
62歳女性。身長148 cm、体重68 kg。糖尿病のため、食事療法及び運動療法に加え、処方1による治療を受けていたが、効果不十分のため処方2による治療に変更された。

問218（実務）
リラグルチドによる治療に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 98-218 を参照

**選択肢**
1. 1型糖尿病に適用される薬剤である。
2. 単独投与による低血糖のリスクは低いが、スルホニルウレア剤を併用した場合は低血糖を引き起こしやすい。
3. 自己会合を起こしたリラグルチドが、投与部位から緩徐に吸収されるため、持続的な効果が期待できる。
4. 他の薬物療法で十分な効果が得られない場合に限り使用できる。

**正解:** 2,3

**解説**
1　誤
リラグルチドは、間接的にインスリン分泌を促進する薬剤であり、2型糖尿病の診断が確定した患者にのみ適用を考慮する。1型糖尿病患者には禁忌である。
2　正
本剤は、血糖依存的にインスリン分泌を促進して血糖を低下させるため、単独投与では低血糖を起こしにくい。しかし、インスリン製剤やスルホニルウレア剤と併用した場合は低血糖を引き起こしやすいため、併用する場合はインスリン製剤やスルホニルウレア剤の減量を検討する。
3　正
リラグルチドは、消化管ホルモンであるインクレチンの一つであるGLP−1（グルカゴン様ペプチド−1）の26位リジンにパルミチン酸を付加して血中アルブミンとの親和性を高めるとともに、自己会合で7量体を形成しやすくしたGLP−1受容体作動薬である。皮下注射で使用することで投与部位から徐々に吸収されるため、1日1回の投与で作用が持続する。
4　誤
本剤の適用は、あらかじめ糖尿病治療の基本である食事療法、運動療法を十分に行ったうえで効果が不十分な場合に限り考慮する。
