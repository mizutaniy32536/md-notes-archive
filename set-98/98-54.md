# 第98回 問54

**問題文**
水中油型の乳剤性基剤はどれか。1つ選べ。

**選択肢**
1. マクロゴール
2. 白色ワセリン
3. 精製ラノリン
4. 流動パラフィン
5. 親水クリーム

**正解:** 5

**解説**
水中油型（O/W型）の乳剤性基剤とは、外相（連続相）である水の中に、内相（不連続相）である油の粒子が分散している状態の基剤のことである。水中油型（O/W型）の乳剤性基剤の代表的なものとして、親水クリームがある。なお、マクロゴールは水性基剤、白色ワセリン、流動パラフィンは油脂性基剤、精製ラノリンは油中水型（W/O型）乳剤性基剤に該当する。
