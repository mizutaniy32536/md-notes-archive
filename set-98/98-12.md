# 第98回 問12

**問題文**
血小板の前駆細胞はどれか。1つ選べ。

**選択肢**
1. 赤芽球
2. 肥満細胞
3. 巨核球
4. マクロファージ
5. 形質細胞

**正解:** 3

**解説**
血小板は、骨髄の巨核球系前駆細胞から巨核球を経て産生される。
1　誤
赤芽球は、赤血球の前駆細胞である。
2　誤
肥満細胞は、ヒスタミンなどのケミカルメディエーターを遊離し、即時型（Ⅰ型）アレルギーに関与する細胞である。
3　正
4　誤
マクロファージは、異物や細菌などに対する免疫反応に関与する細胞である。
5　誤
形質細胞は、Bリンパ球（B細胞）から分化した抗体産生細胞であり、体液性免疫に関与する。
