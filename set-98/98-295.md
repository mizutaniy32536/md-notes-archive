# 第98回 問295

**問題文**
35歳男性。体重60 kg。庭で作業中、スズメバチに刺された。局所が腫れ上がるとともに、呼吸困難を感じ、さらに全身にじん麻疹が出現した。そこで救急外来を受診した。
問295（実務）
その後、入院加療にて改善した。今後、同様の反応を繰り返す危険性があるため、退院時にアドレナリンの自己注射薬が処方された。症状発現時にいつでも自己注射ができるよう、薬剤師が「患者向医薬品ガイド」を参考に服薬指導した。「患者向医薬品ガイド」に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**
1. 「くすりのしおり」の別称であり、同一のものである。
2. 当該医薬品の製造販売業者が作成している。
3. すべての医療用医薬品について作成されている。
4. 薬剤の効果に関する内容は記載されていない。
5. 医薬品医療機器総合機構が管理している医薬品医療機器情報提供ホームページから入手できる。

**正解:** 2,5

**解説**
「患者向け医薬品ガイド」は、当該医薬品の製造販売業者が作成しているもので、医療関係者向けに作成されている添付文書を基に、患者向けに分かりやすく作成した文書である。独立行政法人医薬品医療機器総合機構（PMDA）が管理している医薬品医療機器情報提供ホームページから入手が可能である。
1　誤
「くすりのしおり」は、くすりの適正使用協議会が作成したもので、患者へのインフォームドコンセントを実践するにあたり利用できる資料の一つである。「患者向け医薬品ガイド」とは異なるものである。
2　正
前記参照。
3　誤
すべての医薬品ではなく、特に注意喚起すべき適正使用に関する情報を有する医薬品について作成されている。
4　誤
「患者向け医薬品ガイド」には、薬剤の一般名、含有量、効能効果、使用前に確認すべきこと、用法用量、副作用、保存方法など様々な情報が記載されている。
5　正
前記参照。
