# 第98回 問113

**問題文**
コレステロールに関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**
1. コレステロール生合成に関与する肝臓の3-ヒドロキシ-3-メチルグルタリルCoA（HMG-CoA）還元酵素遺伝子は、コレステロールによって発現が抑制される。
2. 細胞内のコレステロールの量が増加すると、細胞表面にある低密度リポタンパク質（LDL）受容体の数が増加する。
3. 肝臓でコレステロールから生成した胆汁酸は、その大部分が糞便中へ排泄される。
4. 細胞膜のコレステロールは、その大部分が脂肪酸とのエステル型として存在する。
5. 精巣では、コレステロールからアンドロゲンが生合成される。

**正解:** 1,5

**解説**
1　正
3-ヒドロキシ-3-メチルグルタリルCoA（HMG-CoA）還元酵素は、コレステロールの合成経路においてHMG-CoAからメバロン酸の合成に関与する酵素であり、コレステロール合成の律速酵素である。コレステロール産生が進行することによってその遺伝子発現が抑制される。
2　誤
低密度リポタンパク質（LDL）受容体は、血液中から細胞内へのLDLの取り込みに関与する。細胞内のコレステロール量によってその発現が調節されており、細胞内のコレステロール量が減少することで、LDL受容体数が増加する。
3　誤
胆汁酸は胆汁の成分であり、十二指腸へ分泌されて脂質の消化や吸収に関与する。胆汁酸は一部糞便中へ排泄されるが、大部分は腸肝循環によって再利用される。
4　誤
細胞膜のコレステロールは、大部分が遊離型として存在している。なお、細胞内やリポタンパク質中では脂肪酸とのエステル型として存在する。
5　正
ステロイドホルモンであるアンドロゲン（男性ホルモン）は、精巣においてコレステロールから生合成される。
