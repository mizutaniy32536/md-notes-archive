# 110-139

環境汚染（大気汚染、水質汚濁、土壌汚染等）を防止するための法規制に関す る記述として、正しいのはどれか。2つ選べ。
1. 大気汚染防止法では、自動車などの移動発生源から排出されるばい煙や揮発性 有機化合物及び粉じんについて排出基準が定められている。
2. 水質汚濁防止法では、健康に係る有害物質及び生活環境に係る汚染状態につい て排水基準が定められている。
3. 都道府県は、国が定めた一般排出基準あるいは一律排水基準よりも厳しい「上 乗せ排出基準」あるいは「上乗せ排水基準」を定めることができる。
4. 土壌汚染対策法では、すべての特定有害物質について「土壌含有基準」が定め られている。
5. ダイオキシン類対策特別措置法では、大気、水質、土壌及び農作物を対象に環 境基準が定められている。

解答　２、３  
  
１　誤  
大気汚染防止法では、固定発生源（工場や事業場）から排出または飛散する大気汚染物質（ばい煙、揮発性有機化合物、粉じん）について排出基準が定められている。  
  
２　正  
水質汚濁防止法では、工場や事業場から排出される水質汚濁物質について排水基準が定められている。排水基準により規定される物質は、人の健康に係る被害を生じるおそれのある物質（有害物質）を含む排水に係る項目と水の汚染状態を示す項目（生活環境項目）に分類される。  
  
３　正  
国が定めた一般排出基準あるいは一般排水基準だけでは大気汚染あるいは水質汚濁の防止が不十分な地域において、都道府県は条例に基づきより厳しい基準である「上乗せ排出基準」あるいは「上乗せ排水基準」を定めることができる。  
  
４　誤  
土壌汚染対策法は、土壌や地下水に含まれる特定有害物質による人の健康被害を防止することを目的としており、特定有害物質は第一種特定有害物質（揮発性有機化合物）、第二種特定有害物質（重金属等）、第三種特定有害物質（農薬等）に分類される。土壌含有基準は、汚染された土壌の直接摂取による健康影響の観点から定められた基準であり、第二種有害物質（重金属等）に対して設定されおり、すべての特定有害物質について定められているわけではない。  
  
５　誤  
ダイオキシン類対策特別措置法では、大気、水質（水底の底質を含む）、土壌を対象に環境基準が定められている。
