# 110-318

（実務） 患者の操作等について薬剤師が確認するポイントとして適切なのはどれか。2つ 選べ。
1. 注入前に、ペンを激しく振ってから、薬液確認窓から薬液があるか確認した か。
2. ペンを注射部位に押し当てる前に、針キャップを外したか。
3. 注入前の作動確認として、安全カバーを指でペン本体に押し込み、動くことを 確認したか。
4. 手を怪我するなどにより、ペンを注射部位に深く押し込むことができなくなっ ていないか。
5. 注射部位に対して斜め ₄₅ 度の角度で押し当て注入したか。

## 正解
**2 と 4**

## 解説

### ペン型注射器（オートインジェクター）の正しい使い方

#### 各選択肢の検証
1. ❌ **激しく振ってはダメ**
   - タンパク質製剤は激しく振ると変性する
   - 気泡が入る
   - 混濁製剤は「ゆっくり転倒混和」が正しい

2. ⭕ **針キャップは外す**
   - 基本中の基本だが、確認必須

3. ❌ **安全カバーを指で押し込むのはNG**
   - 安全カバーは注射部位に押し当てると自動的に引っ込む仕組み
   - 指で押し込むと誤作動や針刺し事故のリスク
   - 正しい作動確認は「空打ちテスト（エアショット）」

4. ⭕ **押し込む力が十分か確認**
   - オートインジェクターは一定の力で押し込む必要がある
   - 手の怪我、握力低下、関節変形などで押し込めないケースを確認

5. ❌ **90度（垂直）が正しい**
   - 45度ではない
   - 垂直に押し当てることで安全カバーが正しく作動

### ポイント
- **混濁製剤の混和**: ゆっくり転倒混和（激しく振らない）
- **作動確認**: 空打ちテスト（安全カバーを指で押さない）
- **注射角度**: 90度（垂直）
