## ◆ 経口投与された薬物の吸収経路を選ぶシナリオ

260308

**問題**
- 経口投与された薬物が吸収される経路として、正しいのはどれか。1つ選べ。

**選択肢**

1. 小腸 → リンパ管 → 大腸 → 全身循環系
​
2. 小腸 → リンパ管 → 肝臓 → 全身循環系
​
3. 小腸 → 門脈 → 大腸 → 全身循環系
​
4. 小腸 → 門脈 → 肝臓 → 全身循環系
​
5. 小腸 → 胆管 → 大腸 → 全身循環系
​
6. 小腸 → 胆管 → 肝臓 → 全身循環系

**正解と理由**
- `4. 小腸 → 門脈 → 肝臓 → 全身循環系`
- **理由の構成**
	- **問題の整理**：経口投与された薬物の一般的な吸収経路を選ぶ問題
	- **薬学的知識・仕様・制約**：経口投与された薬物は`小腸`で吸収された後、`門脈`を経由して`肝臓`に到達し、肝臓での代謝（`初回通過効果`）を受けた後に全身循環系に入る。これが経口投与の最も一般的な吸収経路である
	- **選択理由**：消化管で吸収された薬物の血流経路として、小腸→門脈→肝臓→全身循環系のみが解剖学的に正しい経路であるため

**解法の手順（思考の流れ）**
- まず経口投与薬の吸収部位は主に`小腸`であると想起
- 小腸から吸収された薬物がどこへ向かうかを考え、消化管の静脈血は`門脈`に集まることを想起
- 門脈は`肝臓`に直接流入し、ここで初回通過効果を受けることを確認
- リンパ管経路は脂溶性の高い特殊な薬物に限られる例外であり、一般論として「門脈系」が正しいと結論づけた

**他の選択肢が誤りな理由**
- 1. 小腸 → リンパ管 → 大腸 → 全身循環系
	`リンパ管`経由の吸収は脂溶性が極めて高い薬物や高分子に限定される。リンパは大腸を経由せず、`胸管`→`左鎖骨下静脈`経由で全身循環に入る

- 2. 小腸 → リンパ管 → 肝臓 → 全身循環系
	`リンパ管`経由で吸収された薬物は門脈を通らないため肝臓を経由しない。リンパ系は`肝初回通過効果を回避`できるのが特徴

- 3. 小腸 → 門脈 → 大腸 → 全身循環系
	`門脈`は肝臓に直接流入する静脈系であり、大腸を経由しない。大腸は消化管の一部であって門脈の先にはない

- 5. 小腸 → 胆管 → 大腸 → 全身循環系
	`胆管`は胆汁を十二指腸に分泌するための管であり、薬物の吸収経路ではない。腸肝循環の一部として胆汁排泄された薬物が再吸収されることはあるが、吸収経路とは異なる

- 6. 小腸 → 胆管 → 肝臓 → 全身循環系
	`胆管`は肝臓から十二指腸への方向に流れるものであり、小腸から肝臓への輸送経路ではない

**キーポイント**
- 経口投与薬の基本経路は「小腸 → `門脈` → `肝臓`（初回通過効果）→ 全身循環系」
- `初回通過効果`が大きい薬物はバイオアベイラビリティが低下するため、舌下投与や直腸投与（下部直腸）で回避する場合がある
- `リンパ管`経由の吸収は例外的であり、初回通過効果を受けない利点がある

**関連知識の比較**
- `門脈系`：消化管（胃〜直腸上部）の静脈血を肝臓に集める。初回通過効果を受ける
- `リンパ系`：胸管→左鎖骨下静脈。初回通過効果を受けない。脂溶性高分子や脂質製剤で利用
- 直腸投与の場合、`下部直腸`からの吸収は下直腸静脈→内腸骨静脈経由で門脈を迂回し、初回通過効果を部分的に回避できる
- `腸肝循環`：胆汁排泄された薬物が腸管で再吸収→門脈→肝臓→再排泄のサイクル

（一言メモ）
- 経口投与の基本は「門脈→肝臓」のセット。`門脈`が出たら必ず`肝臓`が次に来ると覚える
- `リンパ管`が出題されたら「初回通過効果の回避」と「胸管→左鎖骨下静脈」を連想する
- `胆管`は吸収ではなく排泄の方向（肝臓→十二指腸）。混同注意
