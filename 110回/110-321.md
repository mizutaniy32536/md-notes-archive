# 110-321

（実務） 薬剤師が ₂ 回目の訪問時に患者宅にあるヘルパーの訪問記録から、薬がきちんと 飲めているが、ここ ₁ ケ月ほど軟便が続いているという情報を得た。この患者に合 わせた薬剤師の対応として最も適切なのはどれか。1つ選べ。
1. 水分摂取を控えるよう患者と息子に指導した。
2. 酸化マグネシウムの減量を主治医に提案した。
3. 市販の止瀉薬を購入し、服用するよう患者と息子に指導した。
4. ファモチジンの増量を主治医に提案した。
5. ガランタミンの増量を主治医に提案した。

## 正解
**2. 酸化マグネシウムの減量を主治医に提案した**

## 解説

### 問題のポイント
患者が「1ヶ月ほど軟便が続いている」→ 原因薬剤を探す

### 各選択肢の検証

1. ❌ **水分摂取を控える**のはNG
   - 軟便でも脱水予防は重要
   - 特に高齢者は水分摂取が必須

2. ⭕ **酸化マグネシウムの減量を主治医に提案**
   - 酸化マグネシウム = 便秘薬（緩下剤）
   - 腸管内に水分を引き込んで便を柔らかくする
   - 軟便が続く = 効きすぎている可能性大
   - **減量を提案するのが最も適切な対応**

3. ❌ **市販の止瀉薬を勝手に指導**するのはNG
   - 主治医に相談せずに止瀉薬を追加するのは不適切
   - まず原因薬剤の調整を考えるべき

4. ❌ **ファモチジン（H2ブロッカー）の増量**
   - 軟便と関係ない
   - 胃薬を増やしても軟便は改善しない

5. ❌ **ガランタミン（認知症薬）の増量**
   - 軟便と関係ない
   - むしろガランタミンの副作用に消化器症状があるので増量は逆効果

### 酸化マグネシウムの特徴
- **作用**: 浸透圧性下剤（腸管内に水分を引き込む）
- **適応**: 便秘、制酸
- **副作用**: 
  - 過量投与 → 軟便・下痢
  - 高マグネシウム血症（特に高齢者・腎機能低下者）
- **対応**: 症状に応じて減量または休薬

### 薬剤師の役割
1. 副作用の可能性を評価
2. **主治医に疑義照会・減量提案**
3. 服薬指導の見直し
4. 多職種連携（ヘルパーの記録を活用）
