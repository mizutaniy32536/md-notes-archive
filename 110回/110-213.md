# 110-213

（実務） 薬剤師が、処方 ₂ の薬剤の服用及び注意事項について患者に説明する内容とし て、適切なのはどれか。2つ選べ。
1. 錠剤が大きくて飲みにくい場合は、粉砕し水や微温湯に懸濁して服用してもよ いこと。
2. 糞便中に白いものがみられる可能性があること。
3. グレープフルーツジュースの飲用は避けること。
4. 症状が改善しても、再燃しないように服薬を継続する必要があること。
5. 処方 ₂ の薬剤の服用中は、大腸がん検査を避けること。
