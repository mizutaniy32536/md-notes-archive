# 110-153

外科的手術時に用いられる薬物に関する記述として、正しいのはどれか。2つ 選べ。
1. 血液／ガス分配係数の大きい吸入麻酔薬ほど、麻酔の導入は速い。
2. 最小肺胞濃度（MAC）の大きい吸入麻酔薬ほど、麻酔作用は強い。
3. デクスメデトミジンは、アドレナリン a2 受容体を刺激することで、鎮痛及び 鎮静作用を生じる。
4. チアミラールは、c︲アミノ酪酸 GABAA 受容体のバルビツール酸結合部位に結 合することで、意識消失を生じる。
5. ケタミンは、ヒスタミン H1 受容体を遮断することで、不動化（筋弛緩）を起 こす。
