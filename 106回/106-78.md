# 106-78

日本における社会保障制度に含まれないのはどれか。1つ選べ。

1. 生活保護
2. 児童福祉
3. 年金
4. 感染症予防
5. 学校教育

## 正解
**5. 学校教育**

## 解説

社会保障制度の4本柱：

- **社会保険**：年金、医療保険、介護保険、雇用保険、労災保険
- **社会福祉**：児童福祉、障害者福祉、老人福祉など
- **公的扶助**：生活保護
- **保健医療・公衆衛生**：感染症予防、母子保健、精神保健など

学校教育は憲法第26条（教育を受ける権利）に基づく教育制度であり、憲法第25条（生存権）に基づく社会保障制度には含まれない
