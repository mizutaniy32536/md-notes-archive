## ◆ 小児処方における薬剤用量の適正性を判断するシナリオ
260306

**問題**
- 新入薬剤師の導入研修で、以下の処方を題材として提示した。患者は、2歳女児。体重12 kg。

処方（1日3回 毎食後 4日分、各薬剤の1回量）:
テオフィリンDS 20% 1回0.90g / クラリスロマイシンDS 10% 1回0.60g / 耐性乳酸菌製剤散100% 1回0.28g / カルボシステインDS 50% 1回0.24g / クレマスチンDS 0.1% 1回2.40g

表1 各成分の1回適正使用量（成分量）: テオフィリン4mg/kg、クラリスロマイシン5mg/kg、耐性乳酸菌23mg/kg、カルボシステイン10mg/kg、クレマスチン16μg/kg

用量が適切でないため疑義照会すべき薬剤はどれか。2つ選べ。

**選択肢**

1. テオフィリンドライシロップ20%

​
4. カルボシステインドライシロップ50%

**正解と理由**
- `1. テオフィリンドライシロップ20%` および `5. クレマスチンドライシロップ0.1%`
- **理由の構成**
	- **問題の整理**：2歳女児（体重12kg）に対する処方において、各薬剤の1回処方量と規格濃度から成分量を算出し、体重あたりの適正使用量と比較して用量の妥当性を判断する問題
	- **薬学的知識・仕様・制約**：小児薬用量の計算では、製剤の`規格濃度`（DS 20%＝200mg/g等）を正しく把握し、処方量（g）×濃度で`成分量`を算出する。特に`クレマスチンDS 0.1%`のように0.1%（＝1mg/g＝1000μg/g）という低濃度製剤では、mg単位とμg単位の変換を正確に行う必要がある
	- **選択理由**：テオフィリンDS 20%は処方1回0.90g→成分量180mg、適正量は4mg/kg×12kg＝48mgであり、約3.75倍の過剰投与。クレマスチンDS 0.1%は処方1回2.40g→成分量2400μg、適正量は16μg/kg×12kg＝192μgであり、約12.5倍の過剰投与。いずれも`疑義照会`が必要な用量逸脱である

**解法の手順（思考の流れ）**
- まず各薬剤について「処方量（g）×規格濃度」で1回あたりの成分量を算出する
- 次に「適正使用量（成分量/kg）×体重12kg」で適正な1回成分量を算出する
- 両者を比較し、大きく乖離している薬剤を特定する
- `テオフィリン`：0.90g×200mg/g＝180mg vs 4×12＝48mg → 約3.75倍で過剰と判断
- `クラリスロマイシン`：0.60g×100mg/g＝60mg vs 5×12＝60mg → 一致、適正
- `耐性乳酸菌`：0.28g×1000mg/g＝280mg vs 23×12＝276mg → ほぼ一致、適正
- `カルボシステイン`：0.24g×500mg/g＝120mg vs 10×12＝120mg → 一致、適正
- `クレマスチン`：2.40g×1000μg/g＝2400μg vs 16×12＝192μg → 約12.5倍で過剰と判断
- これにより選択肢1と5が疑義照会すべき薬剤と結論づけた

**他の選択肢が誤りな理由**
- 2. クラリスロマイシンドライシロップ10%
	`クラリスロマイシンDS 10%`（100mg/g）の処方1回0.60g→成分量60mg。適正量＝`5mg/kg`×12kg＝60mgでちょうど一致しており、用量は適正である

- 3. 耐性乳酸菌製剤散100%
	`耐性乳酸菌`散100%（1000mg/g）の処方1回0.28g→成分量280mg。適正量＝`23mg/kg`×12kg＝276mgでほぼ一致しており、用量は適正である

- 4. カルボシステインドライシロップ50%
	`カルボシステインDS 50%`（500mg/g）の処方1回0.24g→成分量120mg。適正量＝`10mg/kg`×12kg＝120mgでちょうど一致しており、用量は適正である

**キーポイント**
- 小児薬用量の計算では、`規格濃度`（%表示）を正しくg単位の成分量に変換することが最重要ステップである
- `DS 0.1%`のような低濃度製剤では、0.1%＝1mg/g＝`1000μg/g`であり、mg⇔μg変換のミスが過量投与の見逃しにつながる
- `テオフィリン`は`治療域が狭い`薬物（TDM対象）であり、過量投与により痙攣・不整脈等の重篤な副作用が生じうるため、用量確認は特に慎重に行う

**関連知識の比較**
- `テオフィリン`はキサンチン誘導体で`PDE阻害`による気管支拡張作用を有し、`CYP1A2`で代謝される。小児では代謝能が未熟なため成人より`クリアランス`が低く、血中濃度が上昇しやすい
- `クラリスロマイシン`は`CYP3A4阻害`作用を有し、テオフィリンとの併用で`テオフィリン血中濃度`を上昇させる可能性がある。本処方のように両薬剤が併用される場合はTDMの必要性も考慮する
- `クレマスチン`は第一世代`抗ヒスタミン薬`で、`中枢抑制作用`（眠気等）が強い。小児では痙攣誘発のリスクもあるため過量投与には特に注意が必要である
- 小児用製剤の規格濃度は`DS 20%`、`DS 50%`、`散100%`など多様であり、処方監査時には必ず規格を確認して成分量ベースで用量を評価する

（一言メモ）
- 小児薬用量の処方監査では「処方量（g）×`規格濃度`」で`成分量`を算出してから体重あたりの適正量と比較する手順を徹底する
- `DS 0.1%`のような極低濃度製剤は「0.1%＝1mg/g＝1000μg/g」と3段階で単位変換を確認すると、`μg`⇔mg間の計算ミスを防げる
- `テオフィリン`は`TDM対象薬`であり、有効血中濃度域が5〜15μg/mLと狭いため、小児処方では体重あたりの用量確認が必須となる
