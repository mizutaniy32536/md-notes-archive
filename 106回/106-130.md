## ◆ 食の安全に関わる法制度の正誤を判断するシナリオ
260305

**問題**
「食の安全性」を確保するための法制度に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**

1. 食品安全基本法において、消費者は食の安全性に積極的役割を果たすことが求められている。
​
2. 食品衛生法において、食品に触れる器具、容器包装、洗浄剤、及び乳幼児が口にするおもちゃは規制・措置の対象とはならない。
​
3. HACCPとは、食品製造における最終製品の抜き取り検査による衛生管理の方法である。
​
4. 食品表示法において、厚生労働大臣により食品に関する表示の基準が定められている。
​
5. 食品表示法において、食物アレルギーの発症数が多い、あるいは症状が重篤となる7品目を特定原材料と定め、表示を義務付けている。

**正解と理由**
- `1` と `5`
- **理由の構成**
	- **問題の整理**：食品安全基本法、食品衛生法、HACCP、食品表示法に関する5つの記述から正しいものを2つ選ぶ
	- **薬学的知識・仕様・制約**：
		- `食品安全基本法` は消費者の責務として、食品の安全性確保に積極的な役割を果たすことを明記している（第9条）
		- `食品衛生法` は食品だけでなく、添加物・器具・容器包装・おもちゃも規制対象に含む
		- `HACCP` は製造工程全体を通じた重要管理点（CCP）の連続的監視による衛生管理手法であり、最終製品の抜き取り検査とは異なる
		- `食品表示法` の表示基準は内閣総理大臣が策定する（厚生労働大臣ではない）
		- `特定原材料` 7品目（えび・かに・小麦・そば・卵・乳・落花生）は表示が義務付けられている
	- **選択理由**：選択肢1は食品安全基本法の消費者の責務規定に合致し、選択肢5は特定原材料7品目の表示義務に合致するため、この2つが正しい

**解法の手順（思考の流れ）**
- まず各選択肢がどの法律に関する記述かを整理する
- 選択肢1：`食品安全基本法` の消費者の責務 → 第9条で積極的役割を明記しており正しいと判断
- 選択肢2：`食品衛生法` の規制対象 → 器具・容器包装・おもちゃも規制対象に含まれるため「対象とはならない」は誤り
- 選択肢3：`HACCP` の定義 → 工程管理であり抜き取り検査ではないため誤り
- 選択肢4：`食品表示法` の基準策定者 → 内閣総理大臣であり厚生労働大臣ではないため誤り
- 選択肢5：`食品表示法` の特定原材料 → 7品目の表示義務は正しい
- 以上から選択肢1と5が正しいと結論づけた

**他の選択肢が誤りな理由**
- 2. 食品衛生法において、食品に触れる器具、容器包装、洗浄剤、及び乳幼児が口にするおもちゃは規制・措置の対象とはならない。
	`食品衛生法` は食品そのものだけでなく、器具・容器包装・おもちゃなども規制対象として明記している
	「対象とはならない」という記述は事実に反する

- 3. HACCPとは、食品製造における最終製品の抜き取り検査による衛生管理の方法である。
	`HACCP` は Hazard Analysis and Critical Control Point の略で、製造工程中の重要管理点を連続的に監視する手法
	「最終製品の抜き取り検査」は従来型の衛生管理であり、HACCPとは本質的に異なる

- 4. 食品表示法において、厚生労働大臣により食品に関する表示の基準が定められている。
	`食品表示法` における表示基準の策定権限は `内閣総理大臣` にある
	厚生労働大臣ではなく、消費者庁を所管する内閣総理大臣が担当する

**キーポイント**
- `食品安全基本法` は国・地方公共団体・食品関連事業者・消費者それぞれの責務を明記している
- `HACCP` は工程管理（連続監視）と抜き取り検査（最終製品検査）の違いを正確に区別する必要がある
- `食品表示法` の基準策定者は内閣総理大臣であり、厚生労働大臣や農林水産大臣と混同しやすい
- `特定原材料` 7品目と特定原材料に準ずるもの（21品目、推奨表示）の区別が重要

**関連知識の比較**
- `特定原材料` 7品目（義務表示）：えび・かに・小麦・そば・卵・乳・落花生（くるみは2025年度から追加され8品目になる動きがある）
- 特定原材料に準ずるもの（推奨表示）：アーモンド・あわび・いか・いくら・オレンジ・カシューナッツ・キウイフルーツ・牛肉・ごま・さけ・さば・大豆・鶏肉・バナナ・豚肉・まつたけ・もも・やまいも・りんご・ゼラチンなど
- `食品安全基本法` のリスク分析：リスク評価（食品安全委員会）、`リスク管理`（厚生労働省・農林水産省・消費者庁）、リスクコミュニケーション（関係者間の情報共有）
- `食品衛生法` の規制対象は食品・添加物・器具・容器包装・おもちゃと幅広い

（一言メモ）
- `HACCP` は「工程管理」で「抜き取り検査」ではない → この対比は頻出の引っかけパターン
- `食品表示法` の基準策定者は `内閣総理大臣`（消費者庁所管）であり、厚労大臣でも農水大臣でもない
- `特定原材料` 7品目は「えかこそたにら」などの語呂合わせで覚えると効率的
