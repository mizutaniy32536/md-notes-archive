## ◆ 発がん物質の代謝活性化経路を問うシナリオ
260303

**問題**
- 発がん物質の代謝活性化に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**

1. o-トルイジンは、シトクロムP450によりヒドロキシルアミン体となり、さらにアセチル化反応により活性化される。
​
2. サイカシンは、ベンジルアルコール型代謝物のエステルに活性化される。
​
3. スチレンは、シトクロムP450によりスチレンオキシドに活性化される。
​
4. ビス（クロロメチル）エーテルは、グルタチオン抱合により活性化される。
​
5. ナイトロジェンマスタードは、酸化的代謝反応で生じるアルキルジアゾヒドロキシドに活性化される。

**正解と理由**
- `1. o-トルイジン` と `3. スチレン`
- **理由の構成**
	- **問題の整理**：発がん物質の代謝活性化に関する正しい記述を2つ選ぶ問題
	- **薬学的知識・仕様・制約**：`o-トルイジン`（芳香族アミン）はCYP450による `N-水酸化` → ヒドロキシルアミン体 → `アセチル化` で活性化。`スチレン` はCYP450による `エポキシ化` → スチレンオキシドで活性化
	- **選択理由**：選択肢2はサフロールの代謝経路との混同、選択肢4・5は直接発がん物質であり代謝活性化は不要

**解法の手順（思考の流れ）**
- まず選択肢4（ビス(クロロメチル)エーテル）と5（ナイトロジェンマスタード）が `直接発がん物質` であることを想起し、代謝不要と判断
- 残る1・2・3のうち、サイカシンの代謝経路が「ベンジルアルコール型」ではなく「β-グルコシダーゼによる加水分解」であることを確認
- 1と3が正しい記述であると結論づけた

**他の選択肢が誤りな理由**
- 2. サイカシンは、ベンジルアルコール型代謝物のエステルに活性化される。
	`サイカシン` の代謝経路は腸内細菌の `β-グルコシダーゼ` による加水分解 → メチルアゾキシメタノール → メチルカチオン。「ベンジルアルコール型代謝物のエステル」は `サフロール` の代謝経路

- 4. ビス（クロロメチル）エーテルは、グルタチオン抱合により活性化される。
	`ビス(クロロメチル)エーテル` は `直接発がん物質` であり、代謝活性化を必要としない。グルタチオン抱合は通常「解毒」経路

- 5. ナイトロジェンマスタードは、酸化的代謝反応で生じるアルキルジアゾヒドロキシドに活性化される。
	`ナイトロジェンマスタード` は `直接発がん物質`（アルキル化剤）であり代謝不要。「アルキルジアゾヒドロキシド」は `ジメチルニトロソアミン` の代謝経路

**キーポイント**
- `直接発がん物質`：代謝不要（ビス(クロロメチル)エーテル、`ナイトロジェンマスタード`、エチレンイミン）
- `間接発がん物質`：代謝活性化が必要（芳香族アミン、多環芳香族炭化水素、ニトロソアミン等）
- サイカシンとサフロールの代謝経路の区別が頻出ポイント
- `グルタチオン抱合` は基本的に解毒経路

**関連知識の比較**
- `ベンゾ[a]ピレン`：CYP1A1による `エポキシ化` → ジオールエポキシド（究極発がん物質）
- `アフラトキシンB1`：CYP3A4等による `エポキシ化` → エキソ-8,9-エポキシド
- `2-ナフチルアミン`：CYP1A2による `N-水酸化` → 膀胱がんの原因。o-トルイジンと同じ芳香族アミンの活性化経路
- `ジメチルニトロソアミン`：CYP2E1による `酸化` → アルキルジアゾヒドロキシド → メチルカチオン

（一言メモ）
- `サイカシン`（ソテツに含まれる配糖体）は腸内細菌の `β-グルコシダーゼ` で加水分解される。`サフロール` のベンジルアルコール型経路と混同しやすい
- 「直接か間接か」の判断が第一ステップ。`ビス(クロロメチル)エーテル` と `ナイトロジェンマスタード` は直接型の代表
- 芳香族アミンの活性化は「CYP450 → `N-水酸化` → アセチル化 → ニトレニウムイオン」の流れで覚える
