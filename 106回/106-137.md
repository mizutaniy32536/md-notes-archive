## ◆ 化学物質の環境内動態と生物濃縮を問うシナリオ
260303

**問題**
- 化学物質の環境内動態に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**

1. 生物濃縮の程度を示す指標である濃縮係数は、化学物質の環境中濃度と生体内濃度の差で表される。
​
2. 1-オクタノール／水分配係数が小さい化学物質ほど、生物濃縮されやすい。
​
3. 生物濃縮には、直接濃縮と間接濃縮があり、後者は食物連鎖によって引き起こされる。
​
4. 直鎖型アルキルベンゼンスルホン酸は、分岐型のものに比べ環境中の微生物による分解を受けにくい。
​
5. DDT及びその代謝物は、動物の脂肪組織に残留しやすい。

**正解と理由**
- `3. 生物濃縮には、直接濃縮と間接濃縮があり、後者は食物連鎖によって引き起こされる。` と `5. DDT及びその代謝物は、動物の脂肪組織に残留しやすい。`
- **理由の構成**
	- **問題の整理**：化学物質の環境内動態に関する正しい記述を2つ選ぶ問題
	- **薬学的知識・仕様・制約**：生物濃縮は直接濃縮（環境→生物）と `間接濃縮`（食物連鎖経由）に分類される。`DDT` は脂溶性が高く脂肪組織に蓄積しやすい
	- **選択理由**：選択肢1は「差」ではなく「比」、2は「小さい」ではなく「大きい」、4は「分解されにくい」ではなく「分解されやすい」と逆

**解法の手順（思考の流れ）**
- 選択肢1：濃縮係数の定義を確認し、「差」ではなく「`比`」（生体内濃度÷環境中濃度）であると判断
- 選択肢2：logKowが「小さい」ほど水溶性が高く、生物濃縮されにくい。「`大きい`」が正しいと判断
- 選択肢4：直鎖型（LAS）は「`ソフト型`」で分解されやすく、分岐型（ABS）が分解されにくいと判断
- 選択肢3・5が正しい記述であると結論づけた

**他の選択肢が誤りな理由**
- 1. 生物濃縮の程度を示す指標である濃縮係数は、化学物質の環境中濃度と生体内濃度の差で表される。
	`濃縮係数`（BCF）= 生体内濃度 ÷ 環境中濃度であり、「差」ではなく「`比`」（割り算）で表される

- 2. 1-オクタノール／水分配係数が小さい化学物質ほど、生物濃縮されやすい。
	`logKow` が大きい（脂溶性が高い）ほど生物濃縮されやすい。「小さい」は逆

- 4. 直鎖型アルキルベンゼンスルホン酸は、分岐型のものに比べ環境中の微生物による分解を受けにくい。
	`LAS`（直鎖型）は `ソフト型` で分解されやすい。`ABS`（分岐型）が `ハード型` で分解されにくく、環境問題の原因となった

**キーポイント**
- `濃縮係数`（BCF）= 生体内濃度 ÷ 環境中濃度（比であり差ではない）
- `logKow` が大きい → 脂溶性高い → 生物濃縮されやすい
- LAS（直鎖型）= `ソフト型`（分解されやすい）、ABS（分岐型）= `ハード型`（分解されにくい）
- `DDT` は脂溶性が高く脂肪組織に蓄積、有機塩素系殺虫剤の代表

**関連知識の比較**
- `生物濃縮` の直接濃縮はエラや体表からの取り込みであり、間接濃縮は `食物連鎖`（生態系の上位ほど高濃度に蓄積）
- `DDT` の代謝物DDE・DDAも残留性があり、特に `DDE` が蓄積性が高い。鳥類の卵殻菲薄化の原因としても知られる
- `POPs条約` で規制されるのはPCB・DDT・ダイオキシン類等であり、いずれも難分解性・高蓄積性の共通点がある

（一言メモ）
- 「`差` と `比`」の引っ掛けは出題頻度が高い。濃縮係数は「比」であり、100倍濃縮なら BCF = 100
- `LAS` と `ABS` の違いは「直鎖 = ソフト = 分解されやすい」「分岐 = ハード = 分解されにくい」で覚える
- `DDT` は `POPs条約` で製造・使用が原則禁止されているが、マラリア対策として一部途上国での使用は認められている
