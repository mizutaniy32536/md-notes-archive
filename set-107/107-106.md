# 107-106

**問題文**
ボルテゾミブは、プロテアソームのβ5サブユニットのN末端トレオニン残基と結合し複合体を形成することにより、プロテアソームの働きを阻害する。以下の記述のうち、正しいのはどれか。2つ選べ。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-106 を参照

**選択肢**
1. アで示した複素環はピリミジン環である。
2. イで示した不斉中心の立体配置はS 配置である。
3. ウで示したホウ素はルイス酸として働く。
4. エで示したホウ素の形式電荷は＋1である。
5. エで示したホウ素はsp3混成である。

**正解:** 3,5

**解説**
解答　３、５

１　誤
アで示した複素環は、ピラジン環である。

２　誤
イで示した不斉中心の立体配置は、R 配置である。

３　正
ウで示したホウ素は、空軌道を有し電子対を受け取ることができるため、ルイス酸として働く。

４　誤
ホウ素（又はホウ素のイオン）の結合の手の数は以下の通りである。エで示したホウ素は4本の結合を形成しているため、形式電荷は－1である。

５　正
エで示したホウ素は、sp3混成である。
