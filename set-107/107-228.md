# 107-228

70歳男性、インフルエンザB型。薬剤師が提案する内容として適切なのはどれか。2つ選べ。

1. レボフロキサシン水和物錠
2. インフルエンザワクチン接種
3. アセトアミノフェン静注液
4. ペラミビル水和物注射液
5. アマンタジン塩酸塩錠

## 正解
**3, 4**

## 解説
アセトアミノフェンは高熱・頭痛の対症療法。ペラミビルはA型・B型両方に有効なノイラミニダーゼ阻害薬注射剤。レボフロキサシンは抗菌薬でウイルスに無効。ワクチンは感染後の治療に不適。アマンタジンはA型専用でB型に無効。
