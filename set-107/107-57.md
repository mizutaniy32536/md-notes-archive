# 107-57

**問題文**
間質性肺炎の指標として、特異度が高いのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-57 を参照

**選択肢**
1. アミラーゼ（AMY）
2. 脳性ナトリウム利尿ペプチド（BNP）
3. クレアチンキナーゼ（CK）
4. グリコヘモグロビン（HbA1c）
5. シアル化糖鎖抗原（KL-6）

**正解:** 5

**解説**
解答　５

間質性肺炎は、肺間質（肺胞隔壁）に炎症や線維化病変を認める疾患である。本疾患では、肺間質の線維化により、Ⅱ型肺胞上皮細胞の過形成に伴うシアル化糖鎖抗原（KL-6）の分泌亢進を認める。

１　誤
アミラーゼ（AMY）は、膵液などに多く含まれる糖質分解酵素であり、急性膵炎などで高値を示す。

２　誤
脳性ナトリウム利尿ペプチド（BNP）は、主に心臓で産生・分泌される活性ペプチドであり、心不全などで高値を示す。

３　誤
クレアチンキナーゼ（CK）は、細胞におけるエネルギー代謝酵素であり、急性心筋梗塞や横紋筋融解症などの細胞障害を認める疾患で高値を示す。

４　誤
グリコヘモグロビン（HbA1c）は、過去1〜2ヶ月前の血糖との相関がある検査項目であり、糖尿病で高値を示す。

５　正
