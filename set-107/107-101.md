# 107-101

**問題文**
D−リボースの構造を正しく表しているFischer投影式はどれか。1つ選べ。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-101 を参照

**選択肢**
（選択肢は画像を参照）

**正解:** 1

**解説**
解答　１

Fischer投影式における左右の結合は紙面より手前、上下の結合は紙面より奥で表される（下図参照）。

破線−くさび形表記のD−リボースの3つのキラル炭素の絶対配置は（2R,3R,4R）である。この絶対配置と一致する構造がD−リボースとなるため、選択肢１が正答となる。

１　正
本構造は、D−リボースの構造であり、絶対配置は（2R,3R,4R）である。

２　誤
本構造は、D−アラビノースの構造であり、絶対配置は（2S,3R,4R）である。

３　誤
本構造は、D−キシロースの構造であり、絶対配置は（2R,3S,4R）である。

４　誤
本構造は、L−リボース構造であり、絶対配置は（2S,3S,4S）である。

５　誤
本構造は、L−キシロースの構造であり、絶対配置は（2S,3R,4S）である。
