# 107-316

85歳男性。肺がん、在宅緩和ケア。オピオイド処方開始。
処方1：オキシコドン徐放錠5mg 1回1錠（1日2回）
処方2：オキシコドン散2.5mg 疼痛時（1日4回まで）
処方3：酸化マグネシウム錠330mg 1回1錠（1日3回 毎食後）
処方4：ドンペリドン錠10mg 1回1錠（1日3回 毎食前）

患者家族への服薬指導として、適切なのはどれか。2つ選べ。

1. 痛みが強い時は、効果をあげるために処方1の薬剤をかみ砕いて服用してください。
2. 便に錠剤が混在している場合は、患者の健康状態に異常がないか確認してください。
3. 痛みが強い時は、処方2の薬剤を服用してから5時間以上あけてください。
4. 便秘になる可能性があるので、処方3が処方されています。
5. 吐き気がおきる可能性があるので、処方4が処方されています。

## 正解
**4, 5**

## 解説

- 選択肢1（誤）: 徐放錠をかみ砕くと急激に血中濃度が上昇し、過量投与のリスク。絶対に噛み砕かない
- 選択肢2（誤）: オキシコドン徐放錠はゴーストタブレットが出ることがある（殻が便に混在）が、これは正常。異常ではない
- 選択肢3（誤）: レスキュードーズ（処方2）の投与間隔は通常1時間以上。5時間以上は長すぎる
- 選択肢4（正）: オピオイドの主要な副作用として便秘がある。酸化マグネシウムは便秘対策
- 選択肢5（正）: オピオイドの副作用として悪心・嘔吐がある。ドンペリドンは制吐剤
