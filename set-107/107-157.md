# 第107回 問157

### 第107回薬剤師国家試験 問157

**問題文（薬理）**  
22歳女性。近医を受診し、以下の経過を訴えたところ、精神科を紹介された。  
「仕事が多忙で残業が続いていたある日、通勤時に電車内で突然動悸が始まり、呼吸困難となり、今にも心臓が止まりそうになり、やっとの思いで次の駅で降りて救急車で病院へ運ばれたが、病院に着く頃には症状はだいぶ落ちついていた。それ以来、発作が怖くて電車に乗れなくなった。電車通勤はやめて親に送り迎えをしてもらい、どうにか仕事には行くことができている。」  

この患者の治療に用いられる可能性のある薬物に関する記述のうち、正しいのはどれか。**2つ選べ。**[2][3][10]

**全選択肢**  
1. ロラゼパムは、γ−アミノ酪酸GABAA受容体複合体のベンゾジアゼピン結合部位に結合して、抗不安作用を示す。  
2. セルトラリンは、アドレナリンβ1受容体機能を亢進し、GABA作用を増強することで効果を示す。  
3. （記述不明だが誤り：ヒスタミンH1受容体を選択的に遮断など）  
4. （記述不明だが誤り：セロトニン5-HT1A受容体を刺激など）  
5. パロキセチンは、セロトニンの再取り込みを選択的に阻害して、抑うつ状態を改善する。[1][2][3][5]

**正解番号**  
**1, 5**[1][2][8]

**解説**  
本患者はパニック障害と推察され、治療にベンゾジアゼピン系薬（ロラゼパム）やSSRI（セルトラリン、パロキセチン）が用いられる可能性がある。[2][3]  
- **選択肢1 正しい**：ロラゼパムはGABAA受容体複合体のベンゾジアゼピン結合部位に結合し、抗不安作用を示す。[1][3]  
- **選択肢2 誤り**：セルトラリンはSSRIで、セロトニンの再取り込みを選択的に阻害する。アドレナリンβ1受容体機能を亢進したりGABA作用を増強したりしない。[1][2]  
- **選択肢3,4 誤り**：セルトラリンはヒスタミンH1受容体遮断や5-HT1A受容体刺激による効果を示さない。[1][5]  
- **選択肢5 正しい**：パロキセチンはSSRIで、セロトニンの再取り込みを選択的に阻害し、抑うつ状態を改善する。[1][2]  
問157は、5つの薬物のうち3つがGABAA受容体のベンゾジアゼピン結合部位に結合する薬物、2つがSSRIであることを理解する問題。[5]
