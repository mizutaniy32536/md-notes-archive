# 107-24

**問題文**
公共用水域の水質汚濁に関する「人の健康の保護に関する環境基準」において、基準値が「検出されないこと」と定められているのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-24 を参照

**選択肢**
1. カドミウム
2. ヒ素
3. アルキル水銀
4. トリクロロエチレン
5. 硝酸性窒素及び亜硝酸性窒素

**正解:** 3

**解説**
解答　３

公共用水域の水質汚濁に関する「人の健康の保護に関する環境基準」（環境基本法）において、基準値が「検出されないこと」と定められているのは、アルキル水銀、PCB、全シアンである。
