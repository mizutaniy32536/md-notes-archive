## ◆ 疫学調査における因果関係の判定基準を問うシナリオ

260311

**問題**
疫学調査の結果に基づいて因果関係を判定する際の基準に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**

1. 関連の一致性とは、対象とする要因と疾病に関して、調査の時期、場所、対象集団などを変えても同様の結果が得られることである。
​
2. 関連の強固性とは、疫学以外の実験的な研究で得られた知見や理論と疫学調査の結果が矛盾しないことである。
​
3. 関連の特異性とは、要因の曝露があると相対危険度やオッズ比が統計学的に有意に高くなることである。
​
4. 関連の時間性とは、疾病の発生以前に要因の曝露があることである。
​
5. 関連の整合性とは、要因の曝露があれば必ず疾病の発生があり、曝露がなければ疾病の発生がないことである。

**正解と理由**
- `1. 関連の一致性とは、対象とする要因と疾病に関して、調査の時期、場所、対象集団などを変えても同様の結果が得られることである。`
- `4. 関連の時間性とは、疾病の発生以前に要因の曝露があることである。`
- **理由の構成**
	- **問題の整理**：因果関係の判定基準（Hillの基準）について、各基準の定義が正しく述べられているかを問う問題
	- **薬学的知識・仕様・制約**：因果関係の判定には`Hillの基準`（9つ）が用いられる。主な基準は以下の通り
		- `一致性`（consistency）：異なる時期・場所・対象集団でも同様の結果が再現される
		- `強固性`（strength）：`相対危険度`や`オッズ比`が大きいこと（関連が強い）
		- `特異性`（specificity）：ある要因がある疾病に特異的に結びつくこと（1対1の対応関係）
		- `時間性`（temporality）：曝露が疾病発生よりも時間的に先行すること
		- `整合性`（coherence）：疫学以外の実験的研究で得られた知見や理論と矛盾しないこと
	- **選択理由**：選択肢1は`一致性`の定義を正しく述べており、選択肢4は`時間性`の定義を正しく述べている

**解法の手順（思考の流れ）**
- まず、Hillの因果関係判定基準の各用語の定義を想起する
- 選択肢1「一致性＝再現性」→ 異なる条件下で同様の結果 → 正しいと判断
- 選択肢2「強固性＝実験知見との矛盾なし」→ これは`整合性`の定義であり、`強固性`は関連の強さ（`相対危険度`の大きさ）を指す → 入れ替わっている → 誤り
- 選択肢3「特異性＝相対危険度やオッズ比が有意に高い」→ これは`強固性`の定義であり、`特異性`は要因と疾病の1対1関係 → 入れ替わっている → 誤り
- 選択肢4「時間性＝疾病発生以前に曝露」→ 時間的先行性の定義そのもの → 正しいと判断
- 選択肢5「整合性＝曝露があれば必ず疾病」→ これは`特異性`に近い記述であり、`整合性`は実験知見との矛盾のなさ → 誤り
- 選択肢2と3、3と5が定義を入れ替えるひっかけパターンと認識し、`1`と`4`を選択

**他の選択肢が誤りな理由**
- 2. 関連の強固性とは、疫学以外の実験的な研究で得られた知見や理論と疫学調査の結果が矛盾しないことである。
	これは`整合性`（coherence）の定義。`強固性`（strength）は要因と疾病の関連の強さ、すなわち`相対危険度`や`オッズ比`が大きいことを意味する

- 3. 関連の特異性とは、要因の曝露があると相対危険度やオッズ比が統計学的に有意に高くなることである。
	これは`強固性`（strength）の定義。`特異性`（specificity）はある要因がある特定の疾病にのみ関連すること、つまり1対1の対応関係を指す

- 5. 関連の整合性とは、要因の曝露があれば必ず疾病の発生があり、曝露がなければ疾病の発生がないことである。
	これは`特異性`（specificity）に近い記述。`整合性`（coherence）は疫学調査の結果が、疫学以外の実験的研究や既存の理論と矛盾しないことを意味する

**キーポイント**
- `Hillの基準`は9つあり、因果関係の判定に用いる。ただし`時間性`以外は必須条件ではない
- 国試では各基準の定義を入れ替えるひっかけが頻出。特に`強固性`と`整合性`、`特異性`と`強固性`の入れ替えに注意
- `時間性`は因果関係判定で唯一の必須条件とされる。曝露→疾病の時間的順序が逆転していれば因果関係は成立しない
- `一致性`は「再現性」と同義。異なる研究者、異なる集団、異なる時期で同じ結果が得られるか

**関連知識の比較**
- Hillの9基準の全体像：`強固性`、`一致性`、`特異性`、`時間性`、`生物学的勾配`（量-反応関係）、`妥当性`（もっともらしさ）、`整合性`、`実験的証拠`、`類似性`
- `相対危険度`（RR）はコホート研究で算出、`オッズ比`（OR）は症例対照研究で算出
- 疫学における因果関係の判定は、`Koch（コッホ）の4原則`（感染症）とは区別する
- `交絡因子`の存在は因果関係の判定を困難にするため、研究デザインで`ランダム化`や`層別化`、`マッチング`を行う

（一言メモ）
- `Hillの基準`で唯一の必須条件は`時間性`（曝露→疾病の時間順序）、他はすべて参考基準
- 国試頻出ひっかけ：`強固性`=関連の強さ、`整合性`=既存知見との矛盾なし、この2つの入れ替えに要注意
- `特異性`は現代疫学では重視されにくい基準。1つの要因が複数疾病のリスクになることは珍しくない（例：`喫煙`→肺がん、心疾患、COPD）
