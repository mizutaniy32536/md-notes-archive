# 107-55

吸収促進剤の添加により水溶性薬物の吸収を改善した薬剤はどれか。１つ選べ。

１ バラシクロビル錠
２ セフチゾキシムナトリウム坐剤
３ イトラコナゾール内用液
４ シクロスポリン内用液
５ カンデサルタン シレキセチル錠

**正解:** ２

**解説**
１　誤
バラシクロビル錠は、アシクロビルにバリンを修飾させることで、消化管上皮細胞のペプチドトランスポーターPEPT1からの吸収を改善した製剤である。
２　正
セフチゾキシムナトリウム坐剤は、吸収促進剤であるカプリン酸ナトリウムを添加することで、直腸からの吸収を改善した製剤である。
３　誤
イトラコナゾール内用液は、溶解補助剤であるヒドロキシプロピル−βシクロデキストリンを添加し溶解性を向上させることで、消化管からの吸収を改善した製剤である。
４　誤
シクロスポリン内用液は、消化管内において自発的にo/w型エマルションを形成することで溶解性を向上させることで、消化管からの吸収を改善した製剤である。
５　誤
カンデサルタン シレキセチル錠は、カンデサルタンの脂溶性を向上させることで、消化管からの吸収を改善した製剤である。
