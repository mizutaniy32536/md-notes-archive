# 107-182

フルチカゾンプロピオン酸エステルを有効成分とする懸濁性点鼻液に含まれる添加物に関する記述のうち、正しいのはどれか。２つ選べ。 〔添加物〕カルボキシビニルポリマー、L−アルギニン、ベンザルコニウム塩化物、エデト酸ナトリウム水和物、ポリソルベート80、濃グリセリン、塩化ナトリウム、水酸化ナトリウム、精製水

１ ポリソルベート80は、懸濁化剤として添加されている。
２ ベンザルコニウム塩化物は、有効成分を可溶化するために添加されている。
３ 塩化ナトリウムは、微生物の増殖を抑制するために添加されている。
４ カルボキシビニルポリマーは、有効成分の鼻腔内滞留性を高めるために添加されている。
５ エデト酸ナトリウム水和物は、無痛化剤として添加されている。

**正解:** １、４

**解説**
１　正
ポリソルベート80は、非イオン性界面活性剤であり、懸濁粒子の沈降などを抑えるための懸濁化剤として添加されている。
２　誤
ベンザルコニウム塩化物は、陽イオン性界面活性剤であり、微生物の増殖を抑制するための保存剤として添加されている。
３　誤
塩化ナトリウムは、体液との浸透圧を調整するための等張化剤として添加されている。
４　正
カルボキシビニルポリマーは、水溶性高分子であり、有効成分の鼻腔内滞留性を高めるための付着剤として添加されている。
５　誤
エデト酸ナトリウム水和物は、キレート剤であり、金属類を不活化するための安定化剤として添加されている。
