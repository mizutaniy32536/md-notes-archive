# 107-269

# 第107回薬剤師国家試験 問269

## 問題文

この患者が処方2の薬剤の服用を中止し、テオフィリンの血中濃度が15μg/mLに低下するまでに要する時間として最も近いのはどれか。1つ選べ。[2]

ただし、テオフィリンの血中動態は線形1-コンパートメントモデルに従うものとし、血中消失半減期は6.9時間とする。なお、ln2＝0.69、ln3＝1.10とする。[1]

## 選択肢

1. 8時間[1]
2. 10時間[1]
3. 12時間[1]
4. 14時間[1]
5. 16時間[1]

## 正解

**2. 10時間**[5][8]

## 解説

この問題は、テオフィリンの血中濃度が40μg/mLから15μg/mLに低下するまでの時間を計算する問題です。[2]

**計算方法：**

半減期が6.9時間であることから、以下のように考えます。[1]

- 6.9時間後：40 → 20μg/mL（半減）
- 13.8時間後：20 → 10μg/mL（さらに半減）

初期濃度40μg/mLから15μg/mLへの低下は、この中間に位置します。正確には、ln2＝0.69を用いた一次反応式により計算すると、消失半減期が7時間弱であることから、大体10時間程度で目標濃度に到達することが分かります。[1]
