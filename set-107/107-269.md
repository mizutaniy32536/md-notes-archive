## ◆ テオフィリン血中濃度が40→15µg/mLに低下する時間を計算するシナリオ

260310

**問題**
- テオフィリンの服用を中止し、血中濃度が40µg/mLから15µg/mLに低下するまでに要する時間として最も近いのはどれか。1つ選べ。線形1-コンパートメントモデル、t₁/₂=6.9時間、ln2=0.69、ln3=1.10。

**選択肢**

1. 8時間

​

**正解と理由**
- `2. 10時間`
- **理由の構成**
	- **計算過程**：
		- k = ln2/t₁/₂ = 0.69/6.9 = 0.1 h⁻¹
		- C(t)=C₀×e^(-kt) → 15=40×e^(-0.1t)
		- ln(15/40) = -0.1t → ln(3/8) = -0.1t
		- ln(3/8) = ln3-ln8 = ln3-3ln2 = 1.10-3×0.69 = 1.10-2.07 = -0.97
		- t = 0.97/0.1 = 9.7h ≈ **10時間**

**キーポイント**
- 1コンパートメントモデル消失式: C(t) = C₀ × e^(-kt)
- k = ln2/t₁/₂ の関係を確実に使いこなす
- ln(a/b) = lna - lnb, ln(2ⁿ) = n×ln2 の対数分解テクニック

（一言メモ）
- ln(3/8)の計算がポイント。8=2³なので ln8=3ln2 と分解。与えられたln2, ln3を使い切る
