# 107-52

全身作用を目的としたナファレリン酢酸塩水和物製剤の適用部位として、最も適切なのはどれか。１つ選べ。

１ 眼
２ 肺
３ 鼻
４ 皮膚
５ 耳

**正解:** ３

**解説**
全身作用を目的としたナファレリン酢酸塩水和物製剤の適用部位として、最も適切なのは鼻である。鼻腔粘膜上皮細胞の構造は薄く、バリアー能が低いため、物質透過性が高く、ペプチド製剤のような高分子の吸収にも期待できる。そのため、ナファレリンのようなペプチド製剤の吸収部位として適切である。
