# 107-65

**問題文**
原発性アルドステロン症の臨床所見として、正しいのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-65 を参照

**選択肢**
1. 低血圧症
2. 低ナトリウム血症
3. 低カリウム血症
4. インスリン分泌の増加
5. レニン活性の上昇

**正解:** 3

**解説**
解答　３

原発性アルドステロン症は、腺腫や過形成などにより、副腎皮質球状層からアルドステロンが過剰に分泌される病態である。本疾患では、アルドステロン過剰分泌に伴い、高血圧、高ナトリウム血症、低カリウム血症を認め、さらにアルドステロンによるフィードバックにより血漿レニン活性の低下を認める。また、低カリウム血症に伴うインスリン分泌の低下を生じ、二次性糖尿病を認めることもある。
