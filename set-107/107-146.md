# 107-146

医薬品である覚醒剤原料について、薬局における法令に基づく取扱いとして、正しいのはどれか。２つ選べ。

１ かぎをかけて薬品庫に保管する。
２ 麻薬と一緒に保管できる。
３ 薬局で調剤するためには、覚醒剤施用機関としての指定を受ける必要がある。
４ 処方箋に基づき調剤し、患者に譲渡することができる。
５ 使用期限が切れた調剤前のものを廃棄した場合、30日以内に都道府県知事に届け出る。

**正解:** １、４

**解説**
１　正
医薬品である覚醒剤原料は、かぎのかけた場所に保管しなければならない。
２　誤
医薬品である覚醒剤原料と麻薬は、一緒に保管することはできない。なお、麻薬と一緒に保管できるのは覚醒剤である。
３　誤
薬局は、医薬品である覚醒剤原料を調剤する際、特段の手続きは必要ない。また、覚醒剤施用機関は、覚醒剤の施用を行うことができるものとして、所在地の都道府県知事から指定を受けた病院又は診療所をいう。
４　正
薬局は、医薬品である覚醒剤原料を処方箋に基づき調剤し、患者に譲渡することができる。
５　誤
使用期限が切れた調剤前の医薬品である覚醒剤原料を廃棄する場合、都道府県知事に届け出て当該職員の立会のもとで行う。また、調剤された医薬品である覚醒剤原料を廃棄する場合、廃棄後30日以内に都道府県知事に届け出る。
