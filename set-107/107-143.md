# 107-143

**問題文**
医薬品の開発における臨床試験に関する記述のうち、正しいのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-143 を参照

**選択肢**
1. 第Ⅰ相試験の主な目的は、用法・用量を決定することである。
2. 前期第Ⅱ相試験の主な目的は、薬物動態試験の実施である。
3. 後期第Ⅱ相試験の主な目的は、臨床薬理試験の実施である。
4. 第Ⅲ相試験の主な目的は、検証的試験の実施である。
5. 第Ⅳ相試験の主な目的は、効能・効果を追加することである。

**正解:** 4

**解説**
解答　４

１　誤
第Ⅰ相試験の主な目的は、安全性、忍容性、薬物動態を検討することである。

２　誤
第Ⅱ相試験では、主に探索的試験が実施される。第Ⅱ相試験の前期では少数の患者を対象として有効性、安全性の検討を行う。また、第Ⅱ相試験の後期では用量反応に関しての試験を行い第Ⅲ相試験での用法・用量なども検討する。

３　誤
解説２参照

４　正
第Ⅲ相試験では、主に検証的試験が実施される。検証的試験では治療上の有効性等を証明・確認することである。

５　誤
第Ⅳ相試験の主な目的は、第Ⅰ相〜第Ⅲ相試験で得られなかった情報の収集や使用成績調査等の結果の検証、予想できない副作用等を確認することである。
