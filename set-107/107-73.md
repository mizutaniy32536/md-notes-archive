# 107-73

**問題文**
GVP省令に基づき、新医薬品の適正使用のための情報提供と副作用情報の把握のために、市販後のある一定期間、製造販売業者が行う調査はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-73 を参照

**選択肢**
1. 一般使用成績調査
2. 製造販売後臨床試験
3. 市販直後調査
4. 特定使用成績調査
5. 使用成績比較調査

**正解:** 3

**解説**
解答　３

GVP省令に基づき、新医薬品の適正使用のための情報提供と副作用情報の把握のために、市販後のある一定期間、製造販売業者が行う調査は市販直後調査である。
なお、使用成績調査はGPSP省令に基づいて製造販売業者等が行う調査であり、製造販売後臨床試験はGPSP省令及びGCP省令に基づいて製造販売業者等が行う調査である。
使用成績調査は、医療機関から収集した情報を用いて、診療において、医薬品の副作用による疾病等の種類別の発現状況並びに品質、有効性及び安全性に関する情報の検出又は確認のために行う調査であり、一般使用成績調査、特定使用成績調査、使用成績比較調査の三つに分類される。

１　誤
一般使用成績調査とは、使用成績調査の分類の一つであり、医薬品を使用する者の条件を定めることなく行う調査（使用成績比較調査を除く）である。したがって、一般使用成績調査はGPSP省令に基づいて行われる調査である。

４　誤
特定使用成績調査とは、使用成績調査の分類の一つであり、小児、高齢者、妊産婦、腎機能障害又は肝機能障害など医薬品を使用する者の条件を定めて行う調査（使用成績比較調査を除く）である。したがって、特定使用成績調査はGPSP省令に基づいて行われる調査である。

５　誤
使用成績比較調査とは、使用成績調査の分類の一つであり、特定の医薬品を使用する者の情報と当該医薬品を使用しない者の情報とを比較することによって行う調査である。したがって、使用成績比較調査はGPSP省令に基づいて行われる調査である。
