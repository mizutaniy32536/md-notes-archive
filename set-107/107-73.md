# 107-73

GVP省令に基づき、新医薬品の適正使用のための情報提供と副作用情報の把握のために、市販後のある一定期間、製造販売業者が行う調査はどれか。1つ選べ。

1. 一般使用成績調査
2. 製造販売後臨床試験
3. 市販直後調査
4. 特定使用成績調査
5. 使用成績比較調査

## 正解
**3. 市販直後調査**

## 解説
市販直後調査はGVP省令に基づき、新医薬品の販売開始後6ヶ月間、医療機関への情報提供と副作用情報の収集を行う。一般使用成績調査や特定使用成績調査はGPSP省令に基づく。
