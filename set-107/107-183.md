# 107-183

**問題文**
薬物とターゲティング技術に関する記述として、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-183 を参照

**選択肢**
1. レボドパは、主にP−糖タンパク質により選択的に脳内に取り込まれる。
2. サラゾスルファピリジンは、腸内細菌により5−アミノサリチル酸に変換される。
3. フルシトシンは、腫瘍細胞内の酵素により5−フルオロウラシルに変換される。
4. アルプロスタジルは、乳酸・グリコール酸共重合体マイクロスフェアを担体として病変部位にターゲティングされる。
5. ガラクトシル人血清アルブミンジエチレントリアミン五酢酸テクネチウムは、肝細胞膜上のアシアロ糖タンパク質受容体に強く結合する。

**正解:** 2,5

**解説**
解答　２、５

１　誤
レボドパは、主にアミノ酸トランスポーターにより選択的に血液脳関門を通過し、脳内に取り込まれる。

２　正
サラゾスルファピリジンは、腸内細菌により還元的に切断され、5−アミノサリチル酸とスルファピリジンに変換される。

３　誤
フルシトシンは、腫瘍細胞内ではなく、真菌細胞内の酵素により5−フルオロウラシルに変換される。

４　誤
アルプロスタジルは、リピドマイクロスフェアを担体として、病変部位に受動的にターゲティングされる。なお、乳酸・グリコール酸共重合体マイクロスフェアは、投与部位で徐々に加水分解してマイクロカプセルが崩壊することで、含有した成分を持続的に放出することを目的とした放出制御型の担体である。これを利用した代表的な製剤に、リュープロレリン酢酸塩を含有したリュープリン®︎などがある。

５　正
ガラクトシル人血清アルブミンジエチレントリアミン五酢酸テクネチウムは、ヒト肝細胞膜上のアシアロ糖タンパク質受容体（ASGP）に特異的に結合し、肝臓の機能及び形態の診断に利用される。
