# 107-80

**問題文**
学校薬剤師の設置が法律で義務付けられていないのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-80 を参照

**選択肢**
1. 幼稚園
2. 小・中学校
3. 高等学校
4. 大学
5. 特別支援学校

**正解:** 4

**解説**
解答　４

学校保健安全法で大学以外の学校（幼稚園、小学校、中学校、高等学校、特別支援学校等）には学校薬剤師の設置が義務付けられているが、大学は学校薬剤師の設置が義務付けられていない。
