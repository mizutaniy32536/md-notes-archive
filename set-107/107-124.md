# 第107回 問124

**問題文**
図は、我が国のリスク要因別の関連死亡者数を示したものである。リスク要因a〜cの組合せとして、正しいのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-124 を参照

**選択肢**
（選択肢は画像を参照）

**正解:** 3

**解説**
解答　３

図より、リスク要因aによる関連死亡原因は、循環器疾患、悪性新生物、呼吸器系疾患であることから、喫煙、高血圧、運動不足の中で、リスク要因aは喫煙が該当することが判断できる。続いて、リスク要因bによる関連死亡原因は、循環器疾患のみとなることから、リスク要因bは高血圧が該当することが判断できる。最後に、リスク要因cによる関連死亡原因は、循環器疾患、悪性新生物、糖尿病といった生活習慣病であることから、リスク要因cは運動不足が該当することが判断できる。
