# 107-293

# 第107回薬剤師国家試験 問293

## 問題文

43歳男性。基礎疾患はない。海外に単身赴任中。一時帰国した3日後の夕食時に体調がすぐれず、早めに就寝した。翌朝から39°Cの発熱と発疹を認め、近医を受診し、風しんと診断された。[2]

この患者への対応として、適切なのはどれか。**2つ選べ。**[2]

## 全選択肢

1. この患者に、療養中に他の医療機関を受診する際は、不安を与えないために風しん感染の情報を伝えないように勧める。[2]

2. この患者に、妻との濃厚接触を避けて療養するよう伝える。[2]

3. この患者の母は、風しんウイルス抗体を有していると考えられると伝える。[2]

4. この患者の妻は、今すぐ風しんワクチン接種を受ける必要があると伝える。[2]

5. この患者の子は、今すぐ3回目の風しんワクチン接種を受ける必要があると伝える。[2]

## 正解番号

**2、3**[3][4]

## 解説

**選択肢2（正解）**：風しんは感染力が強く、患者の妻など濃厚接触者への感染リスクがあります。患者に対して、妻との濃厚接触を避けるよう指導することは、感染拡大を防ぐための重要な対応です。[6]

**選択肢3（正解）**：患者の母親の年齢（問題文から推定される年代）を考えると、小児期に風しんに罹患した可能性が高く、そのため風しんウイルス抗体を有していると考えられます。[4]

**選択肢1（不適切）**：他の医療機関を受診する際に、感染情報を隠すことは、医療提供者が適切な感染対策を講じられなくなるため不適切です。むしろ、医療機関に対して風しん感染を明確に伝える必要があります。

**選択肢4（不適切）**：患者の妻が既に感染している可能性があり、また妊娠中でない限り、風しんワクチン接種は濃厚接触後ただちに行うことは推奨されていません。

**選択肢5（不適切）**：患者の子が既に予防接種を2回受けている場合、年齢的に3回目の接種は通常必要ではありません。[6]
