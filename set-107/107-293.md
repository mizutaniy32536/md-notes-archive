## ◆ 風しん患者の家族（妻・母・子）への適切な指導内容を問うシナリオ

260311

**問題**
（107-292の続き）43歳男性の風しん患者への対応として、適切なのはどれか。2つ選べ。
（家族：妻、母（罹患歴あり）、子（風しんワクチン2回接種済み））

**選択肢**

1. この患者に、療養中に他の医療機関を受診する際は、不安を与えないために風しん感染の情報を伝えないように勧める。
​
2. この患者に、妻との濃厚接触を避けて療養するよう伝える。
​
3. この患者の母は、風しんウイルス抗体を有していると考えられると伝える。
​
4. この患者の妻は、今すぐ風しんワクチン接種を受ける必要があると伝える。
​
5. この患者の子は、今すぐ3回目の風しんワクチン接種を受ける必要があると伝える。

**正解と理由**
- `2. この患者に、妻との濃厚接触を避けて療養するよう伝える。`
- `3. この患者の母は、風しんウイルス抗体を有していると考えられると伝える。`
- **理由の構成**
	- **問題の整理**：妻への感染予防（妊娠の可能性）と家族の免疫状態の正しい評価が求められる
	- **薬学的知識・仕様・制約**：風しんが妊婦に感染すると`先天性風しん症候群`リスク。母は罹患歴あり=`抗体保有`。子は2回接種済み=3回目不要
	- **選択理由**：妻への感染予防指導（2）と母の抗体保有の説明（3）が適切

**解法の手順（思考の流れ）**
- 選択肢1「風しん情報を伝えない」→ 医療機関への感染情報の提供は`感染対策上必須`。不安を与えないための隠蔽は不適切 → 誤り
- 選択肢2「妻との濃厚接触を避ける」→ 妻への感染=`先天性風しん症候群`のリスク。飛沫感染予防が重要 → 正しい
- 選択肢3「母は抗体保有」→ 母は罹患歴あり=`自然感染による終身免疫`を保有している可能性が高い → 正しい
- 選択肢4「妻は今すぐワクチン接種」→ `生ワクチン`（MR）は妊娠中禁忌・接種後2ヶ月は避妊必要。「今すぐ」接種は適切でない場合がある → 誤り
- 選択肢5「子は3回目のワクチン」→ 2回接種済み=`定期接種完了`。3回目の接種は不要 → 誤り

**他の選択肢が誤りな理由**
- 1. 風しん感染の情報を伝えないように勧める。
	医療機関を受診する際に感染症情報を伝えることは感染対策上必須。`感染症法`でも届出義務がある。「不安を与えない」のは理由にならない

- 4. 今すぐ風しんワクチン接種を受ける必要があると伝える。
	風しんワクチンは`生ワクチン`（MRワクチン）。妊娠中は禁忌。接種後約2ヶ月間の避妊が必要。「今すぐ」ではなく、時期を選んで接種（通常は産後）

- 5. 今すぐ3回目の風しんワクチン接種を受ける必要があると伝える。
	`定期接種`（1回目：1歳、2回目：就学前）の2回接種完了で十分な免疫が得られる。3回目の根拠なし

**キーポイント**
- `先天性風しん症候群`（CRS）：妊娠12週以前の感染で重篤。難聴・先天性心疾患・白内障の三主徴
- 自然感染後の免疫=`終身免疫`（ほぼ一生続く）。母の罹患歴=抗体保有
- `MRワクチン`（風しん成分）は`生ワクチン`→ 妊婦禁忌、接種後2ヶ月避妊
- 風しんワクチン定期接種は2回で完了（1歳・就学前第2期）

**関連知識の比較**
- `暴露後ワクチン接種`（ポスト暴露予防）：麻疹は暴露後72時間以内に有効。風しんは効果なし
- `暴露後免疫グロブリン`：風しん暴露後の妊婦に使用可能（ただし胎児への予防効果は確立せず）
- 風しん抗体検査：`HI価`（赤血球凝集阻止）≥8が感染防御と判定

（一言メモ）
- `生ワクチン`の妊婦禁忌は絶対。「今すぐワクチン」は産後まで待つのが基本
- 母の「罹患歴あり」=自然感染=抗体あり。予防接種歴のないおばあさん世代は実は免疫持ってる場合多い
- 先天性風しん症候群の予防=妊婦への感染を絶対に防ぐ。夫が感染したら妻への濃厚接触回避が最優先
