



## ◆ 上水道の浄水処理工程における各処理法の原理と目的を判断するシナリオ
260228

**問題**
- 上水道における浄水処理に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**


1. 凝集沈殿とろ過処理の間に行う塩素処理を前塩素処理という。
​
2. 水中のアンモニウムイオンは、結合残留塩素の生成原因となる。
​
3. 不連続点（ブレークポイント）塩素処理の目的は、トリハロメタンの生成を抑制することである。
​
4. オゾンによる高度浄水処理では、臭気物質などが酸化分解により除去される。
​
5. 凝集剤として用いられる硫酸アルミニウムは、水中でゲル状の水酸化アルミニウムとなり、コロイド粒子などを吸着してフロックを形成する。

**正解と理由**
- `4`、`5`
- **理由の構成**
	- **問題の整理**：上水道の浄水処理における塩素処理のタイミング・残留塩素の種類・ブレークポイント塩素処理の目的・オゾン処理の効果・凝集剤の作用機序について正誤を判断する
	- **薬学的知識・仕様・制約**：`オゾン処理`は高度浄水処理の一つであり、`ジェオスミン`や`2-MIB`などの`臭気物質`を酸化分解によって除去する。`硫酸アルミニウム`（硫酸バンド）は水中で`Al(OH)₃`のゲル状沈殿を形成し、`コロイド粒子`を吸着して`フロック`（凝集塊）を生成する
	- **選択理由**：選択肢4のオゾン処理による臭気物質除去と、選択肢5の硫酸アルミニウムによるフロック形成はいずれも正確な記述

**解法の手順（思考の流れ）**
- 選択肢1：「凝集沈殿とろ過の間」の塩素処理の名称を確認 → これは「中間塩素処理」であり「前塩素処理」ではない → 誤り
- 選択肢2：NH₄⁺とCl₂の反応で結合残留塩素（クロラミン）が生じることは事実だが、本問の公式正解には含まれない → この選択肢の表現に問題がある可能性を踏まえて保留
- 選択肢3：ブレークポイント塩素処理の目的を確認 → アンモニアの完全酸化と遊離残留塩素の確保が目的であり、THM抑制ではない → 誤り
- 選択肢4：オゾン処理で臭気物質を酸化分解 → 正しい
- 選択肢5：硫酸アルミニウム→Al(OH)₃ゲル→フロック形成 → 正しい

**他の選択肢が誤りな理由**
- 1. 凝集沈殿とろ過処理の間に行う塩素処理を前塩素処理という。
	`前塩素処理`は凝集沈殿の`「前」`に行う塩素処理を指す。凝集沈殿とろ過の「間」に行う塩素処理は`中間塩素処理`と呼ばれる。浄水処理の最後に行うのが`後塩素処理`

- 2. 水中のアンモニウムイオンは、結合残留塩素の生成原因となる。
	NH₄⁺と塩素（HOCl）が反応してクロラミン（`結合残留塩素`）が生成されるという記述自体は化学反応として正しいが、本問では「原因となる」という表現が「問題点」として捉えるべき文脈であり、公式正解は4と5。選択肢2は記述の正確性に微妙な問題を含む可能性がある

- 3. 不連続点（ブレークポイント）塩素処理の目的は、トリハロメタンの生成を抑制することである。
	ブレークポイント塩素処理の目的は水中の`アンモニア性窒素を完全に酸化分解`し、`遊離残留塩素`を確保することである。`トリハロメタン`（THM）は塩素と`フミン質`等の有機物の反応で生成されるため、むしろ塩素処理量が増えるとTHM生成量も増加する傾向がある

**キーポイント**
- 浄水処理の塩素処理タイミング：`前塩素処理`（凝集前）→`中間塩素処理`（凝集沈殿とろ過の間）→`後塩素処理`（最終段階）
- `ブレークポイント塩素処理`の目的＝アンモニア酸化＋`遊離残留塩素`の確保（THM抑制ではない）
- `オゾン処理`＝ジェオスミン・2-MIBなどの臭気物質を酸化分解する高度浄水処理
- `硫酸アルミニウム`→Al(OH)₃ゲル状沈殿→コロイド吸着→フロック形成

**関連知識の比較**
- `遊離残留塩素`（HOCl、OCl⁻）は殺菌力が強く、`結合残留塩素`（クロラミン：NH₂Cl、NHCl₂、NCl₃）は殺菌力が弱い。水道法では給水栓で遊離残留塩素0.1 mg/L以上（または結合残留塩素0.4 mg/L以上）を保持する義務がある
- `高度浄水処理`には`オゾン処理`のほか`活性炭処理`（吸着除去）や`生物処理`があり、通常の浄水処理（凝集沈殿→ろ過→塩素消毒）では除去困難な物質に対応する
- トリハロメタン対策としては、前塩素処理を廃止して中間塩素処理に切り替える方法や、有機物を活性炭で除去してからの塩素処理が有効
- `PAC`（ポリ塩化アルミニウム）も凝集剤として広く使用されており、硫酸アルミニウムより低温・低濁度水での凝集性に優れる

（一言メモ）
- 前・中間・後塩素処理は「凝集沈殿」を基準にした位置関係で名称が決まる
- ブレークポイント塩素処理のグラフ（塩素添加量vs残留塩素）をイメージできるようにしておく
- オゾン処理の「臭気物質除去」と活性炭処理の「有機物吸着除去」は高度浄水処理の二本柱
