# 107-192

**問題文**
漢方薬に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-192 を参照

**選択肢**
1. 大建中湯は、不眠症に使用される。
2. 五苓散は、浮腫に使用される。
3. 小青竜湯は、アレルギー性鼻炎に使用される。
4. 補中益気湯は、月経困難に使用される。
5. 抑肝散は、気管支ぜん息に使用される。

**正解:** 2,3

**解説**
解答　２、３

１　誤
大建中湯は、腹が冷えて痛み、腹部膨満感がある諸症状などに用いられる。

２　正
五苓散は、のどが乾いて尿量が少ない状態の嘔吐や腹痛、浮腫などに用いられる。

３　正
小青竜湯は、感冒や気管支ぜん息、アレルギー性鼻炎などによる水様の痰、水様鼻汁、鼻閉、くしゃみ、喘鳴、咳嗽などに用いられる。

４　誤
補中益気湯は、消化機能が衰えている虚弱体質者の病後の体力増強、食欲不振などに用いられる。

５　誤
抑肝散は、神経症、不眠症、小児夜なき、小児疳症などに用いられる。
