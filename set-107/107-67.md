# 107-67

**問題文**
がん終末期における呼吸困難に対する治療薬はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-67 を参照

**選択肢**
1. アセトアミノフェン
2. アドレナリン
3. スキサメトニウム
4. デキストロメトルファン
5. モルヒネ

**正解:** 5

**解説**
解答　５

がんの終末期には、腫瘍による気管の閉塞、胸膜転移、胸水貯留などにより、約半数の患者が呼吸困難を呈する。呼吸困難に対する対症治療として、第一選択にモルヒネが用いられる。なお、モルヒネが呼吸困難を改善できる機序として、①呼吸中枢の感受性を低下、②呼吸数を低下させることでの酸素消費量の低下、③中枢性の鎮咳作用および鎮静作用などが関与すると考えられている。
