# 107-88

病院を退院した患者が在宅療養となった。かかりつけ薬剤師がその患者に居宅療養管理指導を行うために、居宅サービス計画書の作成を依頼する職種はどれか。１つ選べ。

１ 医師
２ 介護支援専門員
３ 作業療法士
４ 訪問看護師
５ 訪問介護員

**正解:** ２

**解説**
かかりつけ薬剤師等が療養上適切な居宅療養管理指導等を提供するために必要があると認める場合には、居宅介護支援事業者等に対し、居宅サービス計画書の作成を依頼する必要がある。
また、当該依頼を受けた居宅介護支援事業者は、介護支援専門員に居宅サービス計画の作成に関する業務を担当させるものとする（指定居宅介護支援等の事業の人員及び運営に関する基準　第十三条一項）。
