# 107-187

**問題文**
ネフローゼ症候群で必ずみられるのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-187 を参照

**選択肢**
1. 体重減少
2. 血尿
3. タンパク尿
4. 低アルブミン血症
5. 低コレステロール血症

**正解:** 3,4

**解説**
解答　３、４

ネフローゼ症候群とは、高度のタンパク尿により低タンパク（低アルブミン）血症をきたす腎臓疾患群の総称であり、すべての場合にタンパク尿、低アルブミン血症を呈する。また、それに続いて浮腫、高コレステロール血症を合併する場合も多い。
