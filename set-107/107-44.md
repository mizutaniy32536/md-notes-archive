# 107-44

**問題文**
コデインからモルヒネへの代謝に関与する酵素はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-44 を参照

**選択肢**
1. CYP1A2
2. CYP2C9
3. CYP2D6
4. CYP3A4
5. UGT1A1

**正解:** 3

**解説**
解答　３

コデインは、主にグルクロン酸抱合をうけて尿中排泄される。一部、CYP2D6によりO−脱メチル化されてモルヒネとなり、モルヒネグルクロン酸抱合体として尿中排泄される。
