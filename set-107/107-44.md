# 107-44

コデインからモルヒネへの代謝に関与する酵素はどれか。１つ選べ。

１ CYP1A2
２ CYP2C9
３ CYP2D6
４ CYP3A4
５ UGT1A1

**正解:** ３

**解説**
コデインは、主にグルクロン酸抱合をうけて尿中排泄される。一部、CYP2D6によりO−脱メチル化されてモルヒネとなり、モルヒネグルクロン酸抱合体として尿中排泄される。
