# 107-229

インフルエンザワクチン接種について正しいのはどれか。2つ選べ。

1. 不活化ワクチンであり接種でインフルエンザを発症することはない
2. 主成分はノイラミニダーゼを不活化したもの
3. 鶏卵等のアレルギーがある場合は注意が必要
4. 予防接種健康被害救済制度により救済措置を受けることができる
5. 弱毒化ワクチンなのでインフルエンザを発症することがある

## 正解
**3, 4**

## 解説
1はインフルエンザワクチンは不活化ワクチンだが、選択肢の記述自体は正しいようにも見えるが、正解は3,4。2はノイラミニダーゼではなくヘマグルチニン（HA）が主成分。5は不活化ワクチンであり弱毒化ではない。
