## ◆ 介護保険制度の基本事項を判断する問題

260226

**問題**
介護保険制度に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**

1. 高齢者の介護を社会全体で支え合う仕組みとして導入された。
​
2. 財源は税金と40歳以上の国民が負担する保険料である。
​
3. 薬剤師は、実務経験無しで、介護支援専門員の資格を取得できる。
​
4. ケアプランは、かかりつけの医師が作成する。
​
5. 介護サービスを受けるために支給される金額は、年齢によって限度額が決まっている。

**正解と理由**
- `1. 高齢者の介護を社会全体で支え合う仕組みとして導入された。`
- `2. 財源は税金と40歳以上の国民が負担する保険料である。`
- **理由の構成**
	- **問題の整理**：介護保険制度に関する5つの記述から正しいものを2つ選ぶ
	- **薬学的知識・仕様・制約**：`介護保険法`は2000年施行、高齢者介護の社会化が目的、財源は公費（税金）50%＋保険料50%の構成
	- **選択理由**：選択肢1は介護保険制度の基本理念そのもの、選択肢2は財源構成として正しい

**解法の手順（思考の流れ）**
1. 選択肢1は介護保険制度の導入目的（介護の`社会化`）として正しいと判断
2. 選択肢2の財源構成を確認 → 公費50%＋保険料50%で、被保険者は`40歳以上` → 正しい
3. 選択肢3の`介護支援専門員`（ケアマネジャー）受験資格には`実務経験5年以上`が必要 → 誤り
4. 選択肢4のケアプラン作成者は`介護支援専門員`（ケアマネジャー）であり医師ではない → 誤り
5. 選択肢5の支給限度額は「年齢」ではなく「`要介護度`」で決まる → 誤り

**他の選択肢が誤りな理由**
- 3. 薬剤師は、実務経験無しで、介護支援専門員の資格を取得できる。
	`介護支援専門員`（ケアマネジャー）の受験資格には、国家資格に基づく業務の`実務経験5年以上`かつ`900日以上`の従事が必要
- 4. ケアプランは、かかりつけの医師が作成する。
	`ケアプラン`（介護サービス計画）を作成するのは`介護支援専門員`（ケアマネジャー）であり、`医師`ではない
- 5. 介護サービスを受けるために支給される金額は、年齢によって限度額が決まっている。
	支給限度額は「年齢」ではなく「`要介護度`」（要支援1・2、要介護1〜5）によって決まる

**キーポイント**
- 介護保険の財源：`公費`（税金）50%＋`保険料`50%
- ケアプラン作成者＝`介護支援専門員`（ケアマネジャー）であり、医師ではない
- 支給限度額は「`要介護度`」で決まり、「年齢」ではない

**関連知識の比較**
- `第1号被保険者`＝65歳以上（原因を問わず要介護認定で利用可能）、`第2号被保険者`＝40〜64歳（`特定疾病`に限り利用可能）
- `特定疾病`には`末期がん`、関節リウマチ、`ALS`、パーキンソン病関連疾患、初老期認知症などが含まれる
- 利用者負担は原則`1割`（一定所得以上は2割または3割）
- `居宅療養管理指導`は薬剤師が行える介護保険サービスの1つ
- 介護認定は市町村の`介護認定審査会`が行い、`要支援1・2`と`要介護1〜5`の7段階に分類される

（一言メモ）
- `介護支援専門員`の受験には国家資格＋`実務経験5年`（900日）が必要で、無資格・無経験では取得不可
- ケアプラン＝`ケアマネジャー`が作成、主治医意見書は必要だが医師がプランを作るわけではない
- 介護保険の支給限度額は「`要介護度`」で決定、年齢ではない点がひっかけ頻出
