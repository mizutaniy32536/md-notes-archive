# 107-312

24歳女性。病院の婦人科を受診後、以下の処方箋を持参し来局した。
処方：レボノルゲストレル錠1.5 mg　1錠　1回分（頓服）

この患者への薬剤師の説明として、適切なのはどれか。1つ選べ。

1. この薬は今すぐ服用してください。
2. この薬を服用しても、月経周期や出血の状況に影響はありません。
3. この薬を服用すると、性感染症の発症を予防できます。
4. この薬を服用すると、完全に妊娠を回避できます。
5. この薬は計画的な避妊にも用いられます。

## 正解
**1. この薬は今すぐ服用してください。**

## 解説

レボノルゲストレル（ノルレボ）は緊急避妊薬。性交後72時間以内にできるだけ早く服用する必要がある

- 選択肢2（誤）: 月経周期や出血状況に影響を与えうる
- 選択肢3（誤）: 性感染症の予防効果はない
- 選択肢4（誤）: 完全ではない（約85%の妊娠阻止率）
- 選択肢5（誤）: 緊急避妊専用。計画的な避妊には用いない
