# 107-175

**問題文**
フェニトイン100 mg錠を1回1錠、1日3回服用するところ、誤って1回1錠、1日2回しか服用していなかった。その時の患者のフェニトインの血中濃度は4 µg/mLであった。この患者が処方どおりに1日3回服用した場合のフェニトインの血中濃度（µg/mL）として、最も近い値はどれか。1つ選べ。ただし、フェニトインの血中濃度は定常状態における平均値であるものとし、フェニトインの体内からの消失はMichaelis−Menten式で表され、Michaelis定数を4 µg/mL、バイオアベイラビリティを100%とする。

**解説**
解答　２

設問に「フェニトインの体内からの消失はMichaelis−Menten式で表され」とあることから、Michaelis−Menten式（①式）を用いて、1日3回服用した場合のフェニトインの血中濃度（µg/mL）を算出する。

また、患者の血中濃度が定常状態に達している場合、消失速度と投与速度は等しくなり、②式で表すことができる。

設問において、「Michaelis定数を4 µg/mL、バイオアベイラビリティを100%」とあることから、Km＝4 µg/mL、F＝1となり、②式に代入することで③式となる。

設問において、「誤って、フェニトイン錠100 mgを1回1錠、1日2回で服用」とあることから、Dpo＝200 mg、τ＝1 dayとなり、③式に代入し、Vmaxを算出する。

次に、処方通りにフェニトイン錠100 mgを1日3回で服用した際の定常状態における平均血中濃度を求める。Dpo＝300 mg、τ＝1 day、Vmax 400 mg/dayを③式に代入する。

従って、この患者が処方通りに1日3回服用した場合のフェニトインの血中濃度は、12 µg/mLである。
