# 107-313

### 第107回薬剤師国家試験 問313

**問題文（全文）**  
24歳女性。病院の婦人科を受診後、以下の処方箋を持参し来局した。  
（注：処方箋内容は問312と共通で、レボノルゲストレル錠などの緊急避妊薬が処方された保険外診療のケース。[1][2][3]）  

**問313（法規・制度・倫理）**  
この診療は保険外診療である。この処方箋に記載されていなければならないのはどれか。**2つ選べ**。[1][2][3]

1. 患者の年齢  
2. 保険者番号  
3. 病院の所在地  
4. 専門医の資格  
5. 処方理由[1][2][3]

**正解番号**  
**1, 3**[1][2][3][5][7]

**解説**  
保険外診療の処方箋は、医師法に基づく記載事項（患者の氏名・年齢、薬名・分量・用法・用量、発行年月日、使用期間、病院等の名称及び所在地、記名押印又は署名）が必須である。[2]  
- **1（患者の年齢）**：必須事項。[1][2]  
- **2（保険者番号）**：保険外診療のため不要。[1]  
- **3（病院の所在地）**：病院等の名称及び所在地が必須。[1][2]  
- **4（専門医の資格）**：レボノルゲストレル錠の処方は専門医資格不要。[1]  
- **5（処方理由）**：記載不要。[1][2]  
よって、正解は1と3。[1][2][3]
