# 107-11

女性の月経周期において、排卵直前に分泌量が急激に増加してピークに達し、排卵の開始を誘発するホルモンはどれか。１つ選べ。

１ 黄体形成ホルモン（LH）
２ オキシトシン
３ プロラクチン
４ ヒト絨毛性性腺刺激ホルモン（hCG）
５ プロゲステロン

**正解:** １

**解説**
女性の月経周期において、下垂体前葉から分泌される卵胞刺激ホルモン（FSH）により、原始卵胞が成熟卵胞へ変化し、その後、黄体形成ホルモン（LH）により、成熟卵胞は黄体へ変化する。この黄体は、成熟卵胞から卵子が放出（排卵）されたものであり、黄体の生成を促す黄体形成ホルモンを、別名排卵ホルモンとよぶ。
