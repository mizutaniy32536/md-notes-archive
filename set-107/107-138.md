



## ◆ 天然および人工放射性核種の性質と体内動態に関する正誤を判断するシナリオ
260228

**問題**
- 天然及び人工放射性核種に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**


1. 食品から摂取される天然放射性核種の中で、最も量が多いのは⁴⁰Kである。
​
2. ⁴⁰Kは、α線を放出して崩壊する。
​
3. 体内に取り込まれた人工放射性核種¹³⁷Csは、生体内で筋肉（全身）に集積する。
​
4. 物理学的半減期が28.8年である⁹⁰Srの生物学的半減期を50年とすると、実効半減期は31.2年となる。
​
5. ²²⁶Raなどの天然放射性核種から放出されるα線は、β⁻線及びγ線と比較して、体内被曝による生体損傷が小さい。

**正解と理由**
- `1`、`3`
- **理由の構成**
	- **問題の整理**：天然放射性核種（⁴⁰K、²²⁶Ra）と人工放射性核種（¹³⁷Cs、⁹⁰Sr）の崩壊様式・体内分布・半減期計算・被曝影響について正誤を判断する
	- **薬学的知識・仕様・制約**：`⁴⁰K`は食品中に最も多く含まれる天然放射性核種であり、バナナやほうれん草などカリウムを多く含む食品に由来する。`¹³⁷Cs`はカリウムに化学的性質が類似し、体内では`筋肉`（全身）に広く分布する
	- **選択理由**：選択肢1は⁴⁰Kが食品由来天然放射性核種として最多であることが正しく、選択肢3は¹³⁷Csの全身（筋肉）集積が正しいため、この2つが正解

**解法の手順（思考の流れ）**
- 選択肢1：食品由来の天然放射性核種で最多＝`⁴⁰K` → 正しい
- 選択肢2：⁴⁰Kの崩壊様式を確認 → β⁻線とγ線（＋EC崩壊）であり、α線ではない → 誤り
- 選択肢3：¹³⁷Csの体内分布＝カリウム類似で全身（筋肉）に分布 → 正しい
- 選択肢4：実効半減期の計算を行い、31.2年が正しいか検証する → 誤り
- 選択肢5：α線の体内被曝影響を確認 → 線質係数20で最も大きい → 誤り

**他の選択肢が誤りな理由**
- 2. ⁴⁰Kは、α線を放出して崩壊する。
	`⁴⁰K`は`β⁻線`を放出して`⁴⁰Ca`に崩壊する経路（約89%）と、`EC崩壊`（電子捕獲）により`⁴⁰Ar`に崩壊する経路（約11%、γ線放出を伴う）がある。α線は放出しない

- 4. 物理学的半減期が28.8年である⁹⁰Srの生物学的半減期を50年とすると、実効半減期は31.2年となる。
	実効半減期の計算式は `1/T_eff = 1/T_phys + 1/T_bio` であるから、`T_eff = (28.8 × 50) ÷ (28.8 + 50) = 1440 ÷ 78.8 ≒ 18.3年`となる。31.2年は物理学的半減期と生物学的半減期を「足し算」した誤った計算

- 5. ²²⁶Raなどの天然放射性核種から放出されるα線は、β⁻線及びγ線と比較して、体内被曝による生体損傷が小さい。
	`α線`は透過力が弱く外部被曝では影響が小さいが、`体内被曝`では電離作用が極めて強く、`線質係数`（放射線荷重係数）は`20`であり、β⁻線やγ線（いずれも1）と比較して生体損傷は`最も大きい`

**キーポイント**
- 食品由来天然放射性核種で最多＝`⁴⁰K`（カリウム含有食品に広く存在）
- 人工放射性核種の体内集積部位：`¹³⁷Cs`→全身（筋肉）、`⁹⁰Sr`→骨、`¹³¹I`→甲状腺
- 実効半減期の公式：`1/T_eff = 1/T_phys + 1/T_bio`（逆数の足し算）
- α線は体内被曝で`最も損傷が大きい`（線質係数20）

**関連知識の比較**
- `⁹⁰Sr`はカルシウムに化学的性質が類似するため骨に集積し、骨髄への影響から白血病のリスクを高める。⁹⁰Srの娘核種`⁹⁰Y`もβ⁻線放出核種であり、体内で連続崩壊する
- `¹³¹I`は甲状腺に集積し、甲状腺がんの原因となる。物理学的半減期は約8日と短い。甲状腺への集積を防ぐためにヨウ化カリウム（安定ヨウ素剤）の予防投与が行われる
- 放射線荷重係数はα線＝20、中性子線＝5〜20（エネルギーにより異なる）、β線・γ線・X線＝1であり、等価線量（Sv）＝吸収線量（Gy）×放射線荷重係数で計算する
- 実効半減期は必ず物理学的半減期と生物学的半減期の`どちらよりも短くなる`ため、算出値が両方の半減期より大きい場合は計算ミスと判断できる

（一言メモ）
- 核種の集積部位は「化学的に類似する元素の代謝経路」で覚える（Cs→K→筋肉、Sr→Ca→骨、I→甲状腺）
- 実効半減期の公式は「逆数の和」であり、足し算ではない点がひっかけポイント
- α線は「外部被曝＝影響小、体内被曝＝影響大」の逆転現象を押さえておく
