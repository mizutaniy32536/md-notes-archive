# 107-66

**問題文**
活性型ビタミンD3外用薬が適用される皮膚疾患はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-66 を参照

**選択肢**
1. 接触性皮膚炎
2. 爪白癬
3. アトピー性皮膚炎
4. じん麻疹
5. 尋常性乾癬

**正解:** 5

**解説**
解答　５

乾癬には様々な病型があり、うち尋常性乾癬がそのほとんどをしめる。乾癬とは、銀白色の鱗屑を伴い、境界明瞭な盛り上がった紅斑が全身に認められる。大きさ、数、形は様々で、発疹が癒合し大きな病変を認めることもある。好発部位は慢性の機械的な刺激を受けやすい頭部、肘・膝、臀部、下腿伸側などであり、そのほとんどに痒みを伴う。慢性で軽快と悪化を繰り返すため一律な治療方針はなく、疾患の程度に応じ治療法を選択する。通常、副腎皮質ステロイド製剤の外用薬、活性型ビタミンD3外用薬が用いられ、他に内服薬として免疫抑制薬や紫外線療法などが行われる。
