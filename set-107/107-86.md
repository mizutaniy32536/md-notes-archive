# 107-86

風しんの流行の原因となる主要な感染経路はどれか。１つ選べ。

１ 食品媒介感染
２ 昆虫媒介感染
３ 飛沫感染
４ 空気感染
５ 垂直感染

**正解:** ３

**解説**
風しんは、風しんウイルスの飛沫感染により生じる急性の発疹性感染症である。したがって、風しんの流行の原因となる主要な感染経路は飛沫感染である。
なお、妊娠初期の女性が風しんウイルスに罹患すると、垂直感染により、胎児が先天性風しん症候群を引き起こすことがあるが、風しんの流行の原因とはならない。
