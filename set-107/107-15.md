## ◆ リケッチアが原因の感染症を選ぶシナリオ
260312

**問題**
リケッチアを病原体とする感染症はどれか。１つ選べ。

**選択肢**

1. 梅毒
​
2. 熱帯熱マラリア
​
3. オンコセルカ症（河川盲目症）
​
4. ツツガムシ病
​
5. 日本脳炎

**正解と理由**
- `4. ツツガムシ病`
- **理由の構成**
	- ツツガムシ病 = `Orientia tsutsugamushi`（ツツガムシリケッチア）が病原体

**解法の手順（思考の流れ）**
- 選択肢1：梅毒 → `梅毒トレポネーマ`（スピロヘータ、細菌）→ ✗
- 選択肢2：熱帯熱マラリア → `熱帯熱マラリア原虫`（原虫）→ ✗
- 選択肢3：オンコセルカ症 → `回旋糸状虫`（線虫）→ ✗
- 選択肢4：ツツガムシ病 → `リケッチア`（偏性細胞内寄生菌）→ **○**
- 選択肢5：日本脳炎 → `日本脳炎ウイルス`（フラビウイルス）→ ✗

**キーポイント**
- `リケッチア` = 偏性細胞内寄生菌（細菌とウイルスの中間的存在）
- ツツガムシ病 = ツツガムシ（ダニの一種）が媒介
- 99-233のダニ媒介感染症とも関連

**関連知識の比較**

| 感染症 | 病原体 | 分類 |
|---|---|---|
| ツツガムシ病 | **リケッチア** | 偏性細胞内寄生菌 |
| 梅毒 | 梅毒トレポネーマ | スピロヘータ（細菌） |
| マラリア | マラリア原虫 | 原虫 |
| 日本脳炎 | 日本脳炎ウイルス | ウイルス |

（一言メモ）
- リケッチアの感染症 = ツツガムシ病、日本紅斑熱、発疹チフス
- ツツガムシ病は四類感染症
