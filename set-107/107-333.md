# 107-333

### 第107回薬剤師国家試験 問333

**問題文**  
33歳男性。B型慢性肝炎治療中。交通事故で病院に搬送された。この患者の血液で汚染された物品等と、消毒に適用されるものの組合せとして、正しいのはどれか。**２つ選べ。**[1][2][4]

**選択肢**  
1. 医療用拡大鏡レンズ　クロルヘキシジングルコン酸塩[2][4]  
2. 手術用器具　グルタラール[2][4]  
3. 病室の床　次亜塩素酸ナトリウム[2][4]  
4. 処置室の流し台　ポビドンヨード[2][4]  
5. 搬送用ストレッチャー　ベンザルコニウム塩化物[2][4]

**正解番号**  
**2, 3**[2][4][6][8]

**解説**  
本患者はB型慢性肝炎治療中で、血液はB型肝炎ウイルス（HBV）に汚染されている可能性が高いため、消毒薬はHBVに有効な**高水準または中水準消毒薬**を選択する必要がある。[2][4][8]  

- **1誤**: クロルヘキシジングルコン酸塩は**低水準消毒薬**で、HBVに無効。医療用拡大鏡レンズには使用可能だが、血液汚染には不適切。[2][4]  
- **2正**: グルタラールは**高水準消毒薬**で、手術用器具に適用可能であり、HBVに有効。[2][4]  
- **3正**: 次亜塩素酸ナトリウムは**中水準消毒薬**で、病室の床に適用可能であり、HBVに有効。[2][4]  
- **4誤**: ポビドンヨードは主に生体表面消毒用で、物品（特に金属製の流し台）では腐食・着色の恐れがあり不適切。[2][4]  
- **5誤**: ベンザルコニウム塩化物は**低水準消毒薬**で、HBVに無効。[2][4]  

以上より、正解は2と3。[2][4][6]
