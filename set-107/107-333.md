# 107-333

33歳男性。B型慢性肝炎治療中。交通事故で病院に搬送された。この患者の血液で汚染された物品等と、消毒に適用されるものの組合せとして、正しいのはどれか。２つ選べ。

**正解:** ２、３

**解説**
本患者は、B型肝炎治療中であり、この患者の血液で汚染された物品等に対する消毒薬は、B型肝炎ウイルスに有効なものを選択する必要がある。B型肝炎ウイルスには、高水準消毒薬のグルタラールと中水準消毒薬の次亜塩素酸ナトリウムが推奨される。また、アルコールやポビドンヨードも有効とされている。
１　誤
クロルヘキシジングルコン酸塩は、低水準消毒薬であり、B型肝炎ウイルスには無効である。
２　正
グルタラールは、高水準消毒薬であり、B型肝炎ウイルスに有効であるため、手術用器具の消毒に適用される。
３　正
次亜塩素酸ナトリウムは、中水準消毒薬であり、B型肝炎ウイルスに有効であるため、病室の床の消毒に適用される。
４　誤
ポビドンヨードは、中水準消毒薬であり、B型肝炎ウイルスに有効とされているが、主に傷口や手術部位の皮膚の消毒に用いるため、処置室の流し台の消毒には適用されない。
５　誤
ベンザルコニウム塩化物は、低水準消毒薬であり、B型肝炎ウイルスには無効である。
