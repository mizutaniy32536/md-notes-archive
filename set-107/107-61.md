# 107-61

**問題文**
閃輝暗点を伴うことがある頭痛はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-61 を参照

**選択肢**
1. 片頭痛
2. 反復性緊張型頭痛
3. 慢性緊張型頭痛
4. 反復性群発頭痛
5. 慢性群発頭痛

**正解:** 1

**解説**
解答　１

頭痛とは、頭頸部に限局する痛みの自覚症状であり、脳腫瘍、髄膜炎、脳炎、クモ膜下出血や脳卒中など脳や頭部の疾患の症状として生じる二次性頭痛と、これらの疾患をもたずに、頭痛発作を繰り返す一次性頭痛に分類される。さらに一次性頭痛は、片頭痛、緊張型頭痛、群発頭痛およびその他の三叉神経・自律神経性頭痛、その他の一次性頭痛疾患に分類される。このうち、片頭痛は、脳血管の収縮および拡張が関与する頭痛であり、前駆症状として閃輝暗点を伴うことがある。
