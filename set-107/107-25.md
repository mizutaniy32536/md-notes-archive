# 107-25

医療機関により廃棄される未使用の注射針が該当する区分として、最も適切なのはどれか。１つ選べ。

１ 事業系一般廃棄物
２ 家庭系一般廃棄物
３ 特別管理一般廃棄物
４ 特別管理産業廃棄物
５ 非感染性廃棄物

**正解:** ４

**解説**
医療機関から廃棄される注射針は、産業廃棄物に該当する。さらに、産業廃棄物のうち、爆発性、毒性、感染性など人の健康または生活環境に被害を生じるおそれのあるものは特別管理産業廃棄物に分類される。未使用の注射針は、感染源の付着はないが、鋭利なものであるため感染性廃棄物と同等の取扱いをする。よって未使用の注射針が該当する区分として、「特別管理産業廃棄物」が該当する。
