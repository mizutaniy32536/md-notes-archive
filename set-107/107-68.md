# 107-68

組換え体医薬品でないのはどれか。１つ選べ。

１ アルテプラーゼ
２ カルペリチド
３ ソラフェニブ
４ ニボルマブ
５ レノグラスチム

**正解:** ３

**解説**
組換え体医薬品とは、遺伝子組換え技術や細胞培養技術を用いて製造したタンパク質を有効成分とする医薬品である。組換え体医薬品には、体内に不足した生理活性タンパク質を補う補充療法に用いられる医薬品と、疾患の原因となる物質の機能を阻害する働きをもつ抗体医薬品がある。
１　組換え体医薬品である
アルテプラーゼは、遺伝子組換え法により得たt−PA 遺伝子によりチャイニーズハムスター卵巣細胞で生産されるアミノ酸527個から成る糖タンパク質である。
２　組換え体医薬品である
カルペリチドは、28個のアミノ酸から成るα型ヒト心房性ナトリウム利尿ポリペプチドを遺伝子組換え法で製造したものである。
３　組換え体医薬品でない
ソラフェニブは、製造に際し、遺伝子組換え技術や細胞培養技術を使用してないため、組換え体医薬品ではない。
４　組換え体医薬品である
ニボルマブは、ヒトPD−1に対する遺伝子組換えヒトIgG4モノクローナル抗体であり、チャイニーズハムスター卵巣細胞により産生される、440個のアミノ酸残基からなる重鎖(γ4鎖)2本及び214個のアミノ酸残基からなる軽鎖(κ鎖)2本で構成される糖タンパク質(分子量：約145,000)である。
５　組換え体医薬品である
レノグラスチムは、遺伝子組換え技術により生産されるヒト顆粒球コロニー刺激因子製剤で、チャイニーズハムスター卵巣細胞で産生される174個のアミノ酸残基からなる糖タンパク質である。
