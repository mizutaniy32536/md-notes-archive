# 107-74

**問題文**
薬害事案として、血友病患者への非加熱血液製剤の使用が原因となり生じたのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-74 を参照

**選択肢**
1. HIV感染症
2. クロイツフェルト・ヤコブ病
3. アザラシ肢症
4. スモン病
5. クロロキン網膜症

**正解:** 1

**解説**
解答　１

１　正
HIV感染症は、血友病患者への非加熱血液製剤の使用が原因となり生じた薬害である。

２　誤
クロイツフェルト・ヤコブ病は、脳外科手術を行う際に使用したヒト乾燥硬膜が原因で生じた薬害である。

３　誤
アザラシ肢症は、サリドマイドの使用が原因となり生じた薬害である。

４　誤
スモン病は、キノホルムの使用が原因となり生じた薬害である。

５　誤
クロロキン網膜症は、クロロキンの使用が原因となり生じた薬害である。
