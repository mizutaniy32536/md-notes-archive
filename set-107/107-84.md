# 107-84

**問題文**
循環血液量の減少時における細胞外液の補給に用いる輸液として、適切でないのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-84 を参照

**選択肢**
1. 生理食塩水
2. 5%ブドウ糖液
3. 乳酸リンゲル液
4. 酢酸リンゲル液
5. 5%ブドウ糖加乳酸リンゲル液

**正解:** 2

**解説**
解答　２

循環血液量の減少時における細胞外液の補給に用いる輸液として適切なのは、等張性電解質輸液である。本選択肢のうち、等張性電解質輸液は生理食塩水や乳酸リンゲル液、酢酸リンゲル液、5%ブドウ糖加乳酸リンゲル液である。5%ブドウ糖液は等張性電解質輸液ではないため適切でない。
