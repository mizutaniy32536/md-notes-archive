# 107-58

**問題文**
重症筋無力症の初発症状として、最も頻度が高いのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-58 を参照

**選択肢**
1. 手指の振戦
2. 嚥下障害
3. 四肢の麻痺
4. 眼瞼下垂
5. 発熱

**正解:** 4

**解説**
解答　４

重症筋無力症は、神経筋接合部のアセチルコリン受容体に対する自己免疫疾患で、筋力低下を主症状とし、初発症状として眼症状（眼瞼下垂、複視）などが見られることが多い。その他、筋力低下が全身に及ぶと構音障害、易疲労感、嚥下障害なども見られ、重症例では呼吸麻痺による換気障害が生じることもある。
