# 107-79

**問題文**
ニュルンベルク綱領を踏まえ「人間を対象とする医学研究の倫理的原則」を内容とする、世界医師会が採択した宣言はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-79 を参照

**選択肢**
1. リスボン宣言
2. ヘルシンキ宣言
3. 世界人権宣言
4. ジュネーブ宣言
5. ポツダム宣言

**正解:** 2

**解説**
解答　２

ニュルンベルク綱領を踏まえ「人間を対象とする医学研究の倫理的原則」を内容とする、世界医師会が採択した宣言は、ヘルシンキ宣言である。
