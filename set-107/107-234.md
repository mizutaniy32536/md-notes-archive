## ◆ 抗てんかん薬の長期服用による血清カルシウム低下の機序を問う問題

260226

**問題**
55歳男性。てんかんの治療のため、フェニトインを長期服用。検査にて血清カルシウム値の低下が認められた。この患者の血清カルシウム値が低下した機序として、正しいのはどれか。2つ選べ。

**選択肢**

1. フェニトインが肝臓のCYPを誘導し、ビタミンDの代謝（不活性化）が促進された
​
2. フェニトインがビタミンDの腸管吸収を阻害した
​
3. フェニトインが腎臓でのカルシウム再吸収を抑制した
​
4. 血清カルシウム低下に伴い、副甲状腺ホルモン（PTH）の分泌が促進された
​
5. 血清カルシウム低下に伴い、カルシトニンの分泌が促進された

**正解と理由**
- `1. フェニトインが肝臓のCYPを誘導し、ビタミンDの代謝（不活性化）が促進された`
- `4. 血清カルシウム低下に伴い、副甲状腺ホルモン（PTH）の分泌が促進された`
- **理由の構成**
	- **問題の整理**：フェニトイン長期服用患者の血清Ca低下の機序を2つの因果ステップで問う問題
	- **薬学的知識・仕様・制約**：`フェニトイン`は`CYP誘導薬`であり、ビタミンDの代謝（不活性化）を促進→活性型VitD減少→Ca吸収低下→血清Ca低下、さらにCa低下→`PTH`分泌促進（代償反応）
	- **選択理由**：ステップ1（原因）とステップ2（代償反応）の2段階の機序がそれぞれ選択肢1と4に該当

**解法の手順（思考の流れ）**
1. `フェニトイン`が代表的な`CYP誘導薬`であることを想起
2. CYP誘導 → `ビタミンD`の代謝（25-水酸化体→不活性体への変換）が促進 → 活性型VitD減少 → Ca吸収低下 → 血清Ca低下（選択肢1）
3. 血清Ca低下に対する体の代償反応を考える → Caを上げるホルモン＝`PTH`（副甲状腺ホルモン）の分泌が促進される（選択肢4）
4. カルシトニンはCaを「下げる」ホルモンなので、Ca低下時に分泌が促進されるのは不合理 → 選択肢5は誤り

**他の選択肢が誤りな理由**
- 2. フェニトインがビタミンDの腸管吸収を阻害した
	`フェニトイン`にビタミンDの腸管吸収を阻害する作用は知られていない、VitD低下の原因はCYP誘導による`代謝促進`
- 3. フェニトインが腎臓でのカルシウム再吸収を抑制した
	`フェニトイン`に腎臓でのCa再吸収を抑制する直接的な作用はない、腎Ca排泄を増加させるのは`ループ利尿薬`（フロセミド等）
- 5. 血清カルシウム低下に伴い、カルシトニンの分泌が促進された
	`カルシトニン`はCaを「下げる」ホルモンであり、Ca低下時に分泌が促進されるのではなく、Ca「上昇」時に分泌が促進される、Ca低下時に分泌されるのは`PTH`

**キーポイント**
- `フェニトイン`は`CYP3A4`等を誘導し、ビタミンDの不活性化を促進 → 低Ca血症の原因となる
- `PTH`＝Ca上昇ホルモン（低Ca時に分泌↑）、`カルシトニン`＝Ca低下ホルモン（高Ca時に分泌↑）
- この問題は「原因（CYP誘導→VitD不活性化）」と「代償反応（PTH分泌促進）」の2ステップで構成されている

**関連知識の比較**
- `CYP誘導薬`の代表例：`フェニトイン`、`カルバマゼピン`、`フェノバルビタール`、`リファンピシン`、`セイヨウオトギリソウ`
- `CYP誘導`による相互作用は多数の薬物で問題となり、`ワルファリン`の効果減弱なども同じ機序
- `ビタミンD`の活性化経路：皮膚で産生（VitD3）→ 肝臓で`25-水酸化`（25(OH)D3）→ 腎臓で`1α-水酸化`（1,25(OH)2D3＝活性型）
- `PTH`の作用：骨からCa動員、腎臓でのCa再吸収促進、腎臓での`1α-水酸化酵素`活性化（→活性型VitD産生↑）
- `カルシトニン`は甲状腺`傍濾胞細胞`（C細胞）から分泌され、骨吸収を抑制してCaを低下させる

（一言メモ）
- `PTH`＝Push up（Caを押し上げる）、`カルシトニン`＝カルシウムをとーんと落とす、で作用の方向を記憶
- `フェニトイン`長期服用→`CYP誘導`→VitD不活性化→低Ca→`骨軟化症`のリスクがあり、活性型VitD製剤の補充が必要になることがある
- CYP誘導薬は「`フェノバルビタール`・`フェニトイン`・`カルバマゼピン`・`リファンピシン`」のフ・フ・カ・リで覚える
