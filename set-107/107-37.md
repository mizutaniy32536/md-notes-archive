# 107-37

Cl－チャネル2（ClC-2）を活性化する慢性便秘症治療薬はどれか。１つ選べ。

１ センノシド
２ カルメロース
３ ラクツロース
４ ビサコジル
５ ルビプロストン

**正解:** ５

**解説**
１　誤
センノシドは、大腸の腸内細菌により加水分解されて生成したレインアンスロンが、アウエルバッハ神経叢を刺激して蠕動運動を促進する便秘症治療薬である。
２　誤
カルメロースは、腸管内で水分を吸引して膨張し、腸壁を刺激して蠕動運動を促進する便秘症治療薬である。
３　誤
ラクツロースは、下部消化管で浸透圧を高めて腸管内への腸液分泌を促進する便秘症治療薬である。
４　誤
ビサコジルは、大腸を直接刺激して蠕動運動を促進する便秘症治療薬である。
５　正
ルビプロストンは、小腸粘膜上皮細胞のCl－チャネル2（ClC−2）を活性化することで腸管内への腸液分泌を促進する便秘症治療薬である。
