# 107-188

**問題文**
42歳女性。最近、疲れやすいと感じることが多くなり、また徐々に食欲が低下し、何をするにも億劫でやる気が起こらなくなった。月経周期が乱れたため受診し、検査の結果、橋本病と診断された。この患者の状態として、考えられるのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-188 を参照

**選択肢**
1. 体重が著しく減少している。
2. 頻脈が認められる。
3. 高コレステロール血症が認められる。
4. 血清TSH（甲状腺刺激ホルモン）値が高い。
5. 副甲状腺ホルモンの分泌が亢進している。

**正解:** 3,4

**解説**
解答　３、４

橋本病（慢性甲状腺炎）は、原発性甲状腺機能低下症の一種である。本疾患では、甲状腺ホルモンの作用低下により発汗低下、体重増加、筋力低下、徐脈、高コレステロール血症、粘液水腫、食欲低下、動作緩慢などの症状がみられる。

１　誤
前記参照

２　誤
前記参照

３　正
前記参照

４　正
本疾患では、甲状腺ホルモンの分泌低下に伴う脳下垂体前葉へのネガティブフィードバックの減弱により、血清TSH値は上昇する。

５　誤
本疾患は甲状腺機能低下症の一種であり、副甲状腺ホルモンへの影響はみられない。
