# 107-16

体格指数の1つであるBMIを算出する式はどれか。1つ選べ。

1. 体重（kg）÷ 身長（cm）
2. 体重（kg）÷ 身長（m）
3. 体重（kg）÷ ［身長（cm）］²
4. ［体重（kg）］² ÷ 身長（m）
5. 体重（kg）÷ ［身長（m）］²

## 正解
**5. 体重（kg）÷ ［身長（m）］²**

## 解説

BMI（Body Mass Index）は肥満度を示す体格指数。

- 式：BMI = 体重(kg) ÷ [身長(m)]²
- 単位に注意：身長は「m」（cmではない）、2乗するのは身長のみ
- 判定基準：18.5未満＝低体重、18.5〜25未満＝普通、25以上＝肥満
- 例：身長170cm（1.7m）、体重65kgの場合 → 65÷(1.7)² = 65÷2.89 ≒ 22.5
