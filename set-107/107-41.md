# 107-41

**第107回薬剤師国家試験 問41**

**問題文（全文）**  
下図のセファレキシンの消化管吸収に主として関与するトランスポーターはどれか。1 つ選べ。[1][3][4]

**全選択肢（番号付き）**  
1. 有機アニオントランスポーター **OAT1** [1][3][4]  
2. 有機カチオントランスポーター **OCT2** [1][4]  
3. **P**–糖タンパク質 [1][3][4]  
4. ペプチドトランスポーター **PEPT1** [1][3][4]  
5. グルコーストランスポーター **SGLT2** [1][3][4]

**正解番号**  
**4** [1][3]

**解説**  
セファレキシンは構造中に**ペプチド結合**を有するため、小腸上皮細胞に発現する**ペプチドトランスポーター PEPT1**により認識・輸送され、消化管吸収される。[1][3]  
- 選択肢1: **OAT1**は主にメトトレキサートなどの有機アニオンの輸送を担う（腎臓などに発現）。[1]  
- 選択肢2: **OCT2**は主にシスプラチンなどの有機カチオンの輸送を担う。[1]  
- 選択肢3: **P-gp**（P糖タンパク質）は多剤排出性トランスポーターで吸収阻害に関与。[1]  
- 選択肢5: **SGLT2**は腎臓近位尿細管でのNa+/グルコース共輸送を担う。[1]
