## ◆ HBs抗原陽性妊婦への初期対応を判断する問題

260226

**問題**
妊婦健診における血液検査でHBs抗原が陽性であった。母子感染を防止するために、最初に行うべきことはどれか。1つ選べ。

**選択肢**

1. HBe抗原を検査する
​
2. HBc抗体を検査する
​
3. 速やかに、抗HBs人免疫グロブリンを投与する
​
4. 速やかに、B型肝炎ワクチンを接種する
​
5. 速やかに、インターフェロン製剤を投与する

**正解と理由**
- `1. HBe抗原を検査する`
- **理由の構成**
	- **問題の整理**：HBs抗原陽性の妊婦に対し、母子感染防止のために「最初に」行うべきことを選ぶ
	- **薬学的知識・仕様・制約**：`HBs抗原`陽性はHBV感染を示すスクリーニング結果であり、次のステップは`HBe抗原`検査で感染力（ウイルス複製の活発さ）を評価する
	- **選択理由**：`HBe抗原`が陽性であれば感染力が高く、出生後の新生児への対応方針が決定される、まず「評価」が先で「治療」は後

**解法の手順（思考の流れ）**
1. 「最初に行うべきこと」というキーワードから、治療ではなく`検査`（評価）が先と判断
2. `HBs抗原`陽性＝感染のスクリーニング結果 → 次は感染の活動性を評価する段階
3. 感染力の指標である`HBe抗原`を検査することが最初のステップと結論
4. 選択肢3・4は新生児への対応であり、妊婦本人への「最初の」対応ではないと判断

**他の選択肢が誤りな理由**
- 2. HBc抗体を検査する
	`HBc抗体`は「過去の感染」や「既感染」を示すマーカーであり、感染の`活動性`を評価する指標ではない
- 3. 速やかに、抗HBs人免疫グロブリンを投与する
	`抗HBs人免疫グロブリン`（HBIG）は`出生後の新生児`に投与するものであり、妊婦本人に「速やかに」投与するものではない
- 4. 速やかに、B型肝炎ワクチンを接種する
	`B型肝炎ワクチン`も`出生後の新生児`への対応策であり、既に感染している妊婦には意味がない
- 5. 速やかに、インターフェロン製剤を投与する
	`インターフェロン`は妊婦に`禁忌`（催奇形性リスク）であり、妊娠中に投与してはならない

**キーポイント**
- HBV母子感染防止の最初のステップは`HBe抗原`検査による感染力評価
- `HBs抗原`＝スクリーニング（感染の有無）、`HBe抗原`＝評価（感染の活動性）
- 選択肢3・4は「新生児への対応」を「妊婦への対応」とすり替えたひっかけ

**関連知識の比較**
- HBVマーカーの意味：`HBs抗原`＝現在の感染、`HBe抗原`＝ウイルス複製活発（高感染力）、`HBc抗体`＝過去の感染歴、`HBs抗体`＝免疫獲得（ワクチンor既感染回復）
- 母子感染の主経路は`産道感染`（分娩時の血液・体液曝露）であり、経胎盤感染はまれ
- HBe抗原陽性の場合、新生児への`垂直感染`リスクが約90%と非常に高い
- HBe抗原陰性でも`HBVプレコア変異株`による感染の可能性があるため油断できない
- 「HBs→HBe」の流れ：`Screening`（スクリーニング）→ `Evaluation`（評価）と覚える

（一言メモ）
- HBV検査の流れ：`HBs抗原`（S＝Screening）→ `HBe抗原`（E＝Evaluation）の順で感染力を段階的に評価する
- `インターフェロン`は妊婦`禁忌`、妊娠中のHBV治療が必要な場合は`テノホビル`等の核酸アナログが選択肢
- 選択肢3・4の「速やかに投与」は出生後の`新生児`への対応であり、妊婦への対応と混同させるひっかけが頻出
