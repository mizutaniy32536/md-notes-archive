# 107-127

食物繊維に関する記述のうち、正しいのはどれか。２つ選べ。

１ 水溶性のものと不溶性のものがある。
２ ヒトの消化酵素で消化されず、腸内細菌によっても分解されない。
３ セルロースとリグニンは、どちらも多糖である。
４ 天然の食物繊維は植物性食品由来であり、動物性食品由来のものはない。
５ 「日本人の食事摂取基準（2020年版）」において、食物繊維には目標量が設定されている。

**正解:** １、５

**解説**
１　正
食物繊維は、「人の消化酵素で消化されない食物中の難消化性成分の総体」と定義されている。食物繊維の大部分は多糖であるが、ヒトの消化酵素では分解することができないため、ほとんどエネルギーとして利用できない。しかし、一部は腸内細菌によって分解されエネルギーとして利用される。また、食物繊維には水溶性と不溶性のものがあり代表的なものとして以下に示す。
２　誤
ヒトの消化酵素で消化されないが、腸内細菌には一部分解される（解説1参照）。
３　誤
セルロースはグルコースがβ(1→4)結合で連なった多糖であるが、リグニンは多糖ではなく、高分子フェノール性化合物である。
４　誤
食物繊維のほとんどが植物性食品由来であるが、キチンなどのエビ・カニなどの動物性食品由来も一部存在する。
５　正
「日本人の食事摂取基準（2020年版）」において、食物繊維は生活習慣病の発症予防を目的として、目標量が設定されている。
