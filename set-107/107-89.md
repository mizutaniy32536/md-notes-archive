# 107-89

疾病の一次予防に該当するのはどれか。１つ選べ。

１ がん検診
２ がん患者への緩和ケア
３ うつ病患者に対する社会復帰支援
４ 歩行機能低下患者に対する機能訓練
５ 地域住民を対象とした健康教室

**正解:** ５

**解説**
疾病予防は、予防の段階に応じて一次予防、二次予防、三次予防の3つに分類される。一次予防は「健康な段階」で行う予防であり、健康増進や特異的予防を目的として行われる。二次予防は「疾病が不顕性な段階」で行う予防であり、早期発見・早期治療を目的として行われる。三次予防は「疾病が顕性化した段階」で行う予防であり、能力低下防止や機能回復を目的として行われる。以下に、一次予防、二次予防、三次予防の「予防の段階」及びその例を示す。
１　誤
がん検診は、疾病の二次予防に該当する。
２　誤
がん患者への緩和ケアは、疾病の三次予防に該当する。
３　誤
うつ病患者に対する社会復帰支援は、疾病の三次予防に該当する。
４　誤
歩行機能低下患者に対する機能訓練は、疾病の三次予防に該当する。
５　正
地域住民を対象とした健康教室は、疾病の一次予防に該当する。
