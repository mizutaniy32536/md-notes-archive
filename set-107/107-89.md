# 107-89

**第107回薬剤師国家試験 問89**

**問題文:**  
疾病の一次予防に該当するのはどれか。1つ選べ。[2][3][4][5]

**選択肢:**  
1. がん検診  
2. がん患者への緩和ケア  
3. うつ病患者に対する社会復帰支援  
4. 歩行機能低下患者に対する機能訓練  
5. 地域住民を対象とした健康教室[2][3][4][5]

**正解番号:** 5[2][3]

**解説:**  
一次予防とは、健康な段階で行う疾病の発症予防を指し、栄養改善、環境保全、予防接種、健康教室などが該当する。[2][3]  
選択肢1（がん検診）は二次予防（早期発見・治療）に該当する。[2]  
選択肢2～4はすでに疾患を抱える患者に対するもので、三次予防（リハビリ・社会復帰支援など）に該当し、一次予防ではない。[2][3]  
選択肢5は地域住民を対象とした健康教室で、健康な段階の一次予防の具体例である。[2][3]
