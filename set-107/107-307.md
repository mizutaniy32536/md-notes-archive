



## ◆ 化学療法による副作用発生時の薬剤師の適切な対応を選択する問題
260228

**問題**
- （問306の続き）その後、この患者は歩行困難となり、化学療法による副作用と疑われた。この場合の薬剤師の対応として、正しいのはどれか。2つ選べ。

**選択肢**


1. 副作用が疑われる医薬品の製造販売業者に副作用の可能性があることを報告した。
​
2. 副作用が疑われる医薬品の製造販売業者に副作用の治療費を請求した。
​
3. 適正に使用された医薬品は、全て副作用被害救済制度の対象になるので、申請を患者に勧めた。
​
4. 医薬品と副作用の因果関係が明確ではなかったが、独立行政法人医薬品医療機器総合機構（PMDA）へ報告した。
​
5. 副作用が疑われる医薬品について、病院のホームページにて患者情報を公開して、注意喚起を行った。

**正解と理由**
- `1` と `4`
- **理由の構成**
	- **問題の整理**：化学療法（ビンクリスチン含む）施行中の患者が歩行困難となり、副作用が疑われている状況で、薬剤師としてどのような対応をとるべきかを問われている
	- **薬学的知識・仕様・制約**：薬剤師には`副作用報告義務`がある。報告先は`製造販売業者`および`PMDA`（独立行政法人医薬品医療機器総合機構）の2つ。PMDAへの報告は`因果関係が不明`であっても、疑いの段階で行うことができる。また、`副作用被害救済制度`には除外医薬品があり、`抗悪性腫瘍剤`はその対象外
	- **選択理由**：選択肢1は製造販売業者への副作用報告であり適切。選択肢4はPMDAへの報告で、因果関係が明確でなくても報告可能であり適切

**解法の手順（思考の流れ）**
- まず薬剤師の`副作用報告義務`を想起し、報告先として`製造販売業者`と`PMDA`があることを確認する
- 選択肢1は製造販売業者への報告であり、副作用報告の基本的な義務に合致 → 正しい
- 選択肢4はPMDAへの報告で、「因果関係が明確ではなかった」という条件がついているが、副作用報告は`疑いの段階`で行えることを思い出す → 正しい
- 選択肢2、3、5を消去法で検討し、それぞれ不適切な理由を確認する

**他の選択肢が誤りな理由**
- 2. 副作用が疑われる医薬品の製造販売業者に副作用の治療費を請求した。
	副作用報告と`治療費請求`はまったく別の問題。副作用報告制度は安全性情報の収集を目的としたものであり、製造販売業者への治療費請求の根拠とはならない

- 3. 適正に使用された医薬品は、全て副作用被害救済制度の対象になるので、申請を患者に勧めた。
	`副作用被害救済制度`には除外医薬品が存在する。`抗悪性腫瘍剤`（抗がん薬）は救済制度の`対象外`であるため、「全て対象になる」という記述が誤り。本問ではビンクリスチン等の抗がん薬を使用しており、救済対象にはならない

- 5. 副作用が疑われる医薬品について、病院のホームページにて患者情報を公開して、注意喚起を行った。
	`個人情報保護`の観点から、患者情報をホームページで公開することは不適切。注意喚起の必要性があっても、患者のプライバシーを侵害する方法で行ってはならない

**キーポイント**
- 薬剤師には`製造販売業者`および`PMDA`への副作用報告義務がある
- PMDAへの副作用報告は`因果関係が不明`でも、疑いの段階で報告可能
- `副作用被害救済制度`の対象外となる医薬品：`抗悪性腫瘍剤`、免疫抑制剤（一部）など

**関連知識の比較**
- `副作用報告`の報告期限：予測できない重篤な副作用は`15日以内`にPMDAへ報告が必要
- `副作用被害救済制度`の給付内容：医療費、医療手当、障害年金、障害児養育年金、遺族年金、遺族一時金、葬祭料の7種類
- 救済制度の除外対象には抗悪性腫瘍剤のほか、`法定予防接種`によるものも含まれる（法定予防接種は別制度で救済）
- 医薬品の副作用報告は`薬機法第68条の10第2項`に基づく義務であり、医師・薬剤師・その他の医薬関係者が対象

（一言メモ）
- 「抗がん薬＝副作用被害救済制度の対象外」はひっかけ問題の定番。「全て」「必ず」などの断定表現に注意する
- PMDAへの報告は「疑い」で十分。因果関係の確定は報告の要件ではない
- 副作用報告と治療費請求は制度の趣旨が異なる。報告は安全性情報の収集、治療費は損害賠償の話
