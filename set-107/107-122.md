# 107-122

### 第107回薬剤師国家試験 問122

**問題文**  
検疫に関する記述のうち、**誤っている**のはどれか。1つ選べ。[2][3][4]

**選択肢**  
1. 検疫法は、国内に常在しない感染症の病原体が船舶又は航空機を介して国内に侵入することの防止を目的としている。[3][4]  
2. 検疫感染症の患者は、入国停止、隔離、停留あるいは消毒等の措置がとられる。[3][4]  
3. **新興感染症は、すべて検疫感染症に含まれる。**[2][3][4]  
4. 検疫感染症には、感染症法＊に定める一類感染症が含まれる。  
＊感染症法：感染症の予防及び感染症の患者に対する医療に関する法律[2][3][4]  
5. 検疫感染症には、感染症法＊に定める新型インフルエンザ等感染症が含まれる。  
＊感染症法：感染症の予防及び感染症の患者に対する医療に関する法律[2][3][4]

**正解番号**  
**3**[2][3]

**解説**  
検疫感染症とは、検疫業務の対象となる感染症で、一類感染症、新型インフルエンザ等、MERS、デング熱などの15感染症が該当し、質問、診察、消毒等を実施する。[2]  
選択肢1は正しい。検疫法の目的は、国内に常在しない感染症の病原体が船舶・航空機を介して国内に侵入することの防止と、船舶・航空機に関し感染症の予防に必要な措置を講ずることである。[3]  
選択肢2は正しい。検疫感染症の患者に対し、入国停止、隔離、停留、消毒等の措置がとられる。[3]  
選択肢3は**誤り**。新興感染症とは、過去20年以内に新しく認識された感染症で、公衆衛生上の問題となるものを指すが（例: エボラ出血熱、AIDS、SARS、新型インフルエンザ）、すべてが検疫感染症に含まれるわけではなく、腸管出血性大腸菌感染症やカンピロバクター感染症などは含まれない。[2][3]  
選択肢4・5は正しい。検疫感染症には感染症法の一類感染症および新型インフルエンザ等感染症が含まれる。[2][3]
