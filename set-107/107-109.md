# 107-109

**第107回薬剤師国家試験 問109**

**問題文**  
天然物由来成分の化学構造を基に開発された、血液凝固抑制作用を有するクマリン誘導体はどれか。1つ選べ。[1][2][4][9]

**全選択肢**  
（注：検索結果に化学構造の画像が含まれるため、以下に各選択肢の化合物名と特徴を記載。実際の試験では構造式が提示されていた。）  
1. メトキサレン（フラノクマリン誘導体、尋常性白斑治療薬）[2]  
2. トラニラスト（ナンジノシドを基に開発、アレルギー治療薬）[2]  
3. イプリフラボン（イソフラボノイド誘導体、骨粗鬆症治療薬）[2]  
4. **ワルファリン（クマリン誘導体、血液凝固抑制薬）**[1][2]  
5. クロモグリク酸ナトリウム（ケリンを基に開発、抗アレルギー薬）[2]

**正解番号**  
4[1][2][4]

**解説**  
ワルファリンは、カビが生えたスイートクローバーによるウシの出血性疾患から発見されたジクマロールを基に開発されたクマリン誘導体で、血液凝固第Ⅱ・Ⅶ・Ⅸ・Ⅹ因子のビタミンK依存性γ-カルボキシラーゼを阻害し、抗凝固作用を示す。選択肢4の構造にK（カリウム）が含まれる点（ワルファリンカリウム錠）が特徴で、他の選択肢はクマリン誘導体でないか血液凝固抑制作用を持たない。[1][2]
