# 107-109

天然物由来成分の化学構造を基に開発された、血液凝固抑制作用を有するクマリン誘導体はどれか。１つ選べ。

**正解:** ４

**解説**
１　誤
メトキサレンの構造である。フラノクマリン誘導体であり、皮膚の光線感受性を増強させる作用を有し、尋常性白斑に用いられる。
２　誤
トラニラストの構造である。アントラニル酸誘導体であり、ケミカルメディエーター遊離抑制作用を有し、気管支喘息などに用いられる。
３　誤
イプリフラボンの構造である。イソフラボン誘導体であり、骨吸収抑制作用を有し、骨粗しょう症に用いられる。
４　正
ワルファリンカリウムの構造である。クマリン誘導体であり、血液凝固抑制作用を有し、血栓塞栓症に用いられる。
５　誤
クロモグリク酸ナトリウムの構造である。クロモン誘導体であり、ケミカルメディエーター遊離抑制作用を有し、アレルギー性鼻炎などに用いられる。
