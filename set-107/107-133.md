# 107-133

（NMRスペクトル問題）錠剤の成分として推定される医薬品Aはどれか。1つ選べ。

1. トリアゾラム
2. ブロチゾラム
3. ニトラゼパム
4. フルニトラゼパム
5. エスタゾラム

## 正解
**1. トリアゾラム**

## 解説
NMRスペクトルから芳香族7H、メチル基3H、総プロトン12Hの特徴がトリアゾラムに一致。
