# 107-133

**問題文**
自宅の寝室で倒れている男性が救急搬送された。簡易検査の結果をもとに、直ちに対応する解毒薬が投与された。一方、寝室にはコップの中に大量の錠剤が沈んだ飲料水が残されていた。問133（物理・化学・生物）錠剤の成分が何かを調べるため、コップの中に残された薬剤に適切な処理をしたのち、1H NMRスペクトル（CDCl3溶媒中）を測定した。得られたスペクトルをデータベースと照合したところ、図に示したチャートとシグナル及び積分値が一致した。錠剤の成分として推定される医薬品Aの構造はどれか。1つ選べ。ただし、図はTMSを基準（0 ppm）とし、シグナルを積分曲線と共に示したもので、×は重溶媒中に微量に含まれるCHCl3のシグナルである。＜image1:center＞＜image2:center＞問134（法規・制度・倫理）法令上、この医薬品Aが該当するのはどれか。1つ選べ。

問134　解答　２

医薬品Aはトリアゾラムであり、法令上、向精神薬に該当する。なお、他の選択肢の医薬品も全て向精神薬に該当する。

問135　解答　３

この患者はベンゾジアゼピン系医薬品であるトリアゾラムの中毒と考えられるため、ベンゾジアゼピン受容体拮抗薬であるフルマゼニルが投与されたと考えられる。

１　誤
ネオスチグミンは、コリンエステラーゼを阻害し、アセチルコリンの分解を抑制することで、間接的にニコチン性アセチルコリンNM受容体を刺激するため、ベクロニウムやロクロニウムなどの競合拮抗型筋弛緩薬中毒の解毒に用いられる。

２　誤
ナロキソンは、オピオイドµ受容体拮抗作用を有するため、モルヒネなどの麻薬中毒の解毒に用いられる。

３　正

４　誤
ヨウ化プラリドキシムは、アセチルコリンエステラーゼに結合している有機リン系化合物の脱離作用を有するため、パラチオンやマラチオンなどの有機リン系化合物中毒の解毒に用いられる。

５　誤
メルカプロールは、重金属とキレートを形成して排泄を促進するため、ヒ素（As）、水銀（Hg）、鉛（Pd）、銅（Cu）などの重金属中毒の解毒に用いられる。
