# 107-193

幹細胞に関する記述のうち、正しいのはどれか。２つ選べ。

１ 胚性幹細胞（ES細胞）は、自己複製能を持つ。
２ 胚性幹細胞は、多能性獲得に必要な遺伝子を導入して作製する。
３ 人工多能性幹細胞（iPS細胞）の作製には、受精卵が用いられる。
４ 造血幹細胞は、あらゆる細胞に分化する能力を持つ。
５ 造血幹細胞は、臍帯血にも存在する。

**正解:** １、５

**解説**
幹細胞とは、多分化能と自己複製能を併せ持った細胞であり、体性幹細胞（造血幹細胞や間葉系幹細胞など）や胚性幹細胞（ES細胞）、人工多能性幹細胞（iPS細胞）がこれに該当する。
１　正
前記参照
２　誤
胚性幹細胞は、受精卵から形成された胚盤胞内の内部細胞塊から培養して作製される幹細胞であり、作製過程に遺伝子の導入は必要ない。なお、体細胞（皮膚や血液など）に多能性を獲得させるために必要な遺伝子を人工的に導入して作製する幹細胞は、人工多能性幹細胞（iPS細胞）である。
３　誤
解説2参照
４　誤
造血幹細胞は血球系細胞（白血球や赤血球など）にのみ分化する幹細胞であるため、あらゆる細胞に分化する能力は持たない。
５　正
造血幹細胞は、主に骨髄の中に存在しているが、一部臍帯血や末梢血にも存在する。
