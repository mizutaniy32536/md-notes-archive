# 107-145

次の分類のうち、コンタクトレンズが該当するのはどれか。１つ選べ。

１ 高度管理医療機器
２ 管理医療機器
３ 一般医療機器
４ 再生医療等製品
５ 医薬部外品

**正解:** １

**解説**
コンタクトレンズは高度管理医療機器に分類される。医療機器は人体へのリスクの程度により３種類に分類され、厚生労働大臣が薬事食品衛生審議会の意見を聴いて指定するものである。
なお、上記のリスク分類とは別に、保守点検、修理その他管理に専門的な知識を必要とするものは「特定保守管理医療機器」として指定されることがある。
