# 107-43

血漿タンパク質のうち、プロプラノロールとの親和性が最も高いのはどれか。１つ選べ。

１ アルブミン
２ α−グロブリン
３ γ−グロブリン
４ フィブリノーゲン
５ α1

**正解:** ５

**解説**
薬物と親和性が高い血漿タンパク質として、α1−酸性糖タンパク質や、アルブミンがある。その中で、プロプラノロールのような塩基性薬物との親和性が高い血漿タンパク質は、α1−酸性糖タンパク質である。その他に、α1−酸性糖タンパク質と親和性が高い代表的な薬物として、リドカイン、イミプラミン等の塩基性薬物が挙げられる。一方、アルブミンと親和性が高い代表的な薬物として、ワルファリン、フェニトイン等の酸性薬物が挙げられる。
