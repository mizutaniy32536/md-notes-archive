# 107-26

**問題文**
Cl－チャネル内蔵型受容体はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-26 を参照

**選択肢**
1. アセチルコリンNM受容体
2. γ−アミノ酪酸GABAA受容体
3. セロトニン5−HT3受容体
4. グルタミン酸NMDA受容体
5. ATP P2X受容体

**正解:** 2

**解説**
解答　２

γ−アミノ酪酸GABAA受容体は、Cl－イオンチャネル内蔵型（陰イオンチャネル内蔵型）受容体である。受容体が刺激されると、陰イオンであるCl－が細胞内に流入し過分極を引き起こす。なお、アセチルコリンNM受容体、セロトニン5−HT3受容体、グルタミン酸NMDA受容体やATP P2X受容体は陽イオンチャネル内蔵型受容体である。
