# 107-317

（107-316の続き）この患者家族が、在宅で調剤済みのオキシコドンを管理する場合、麻薬及び向精神薬取締法に照らし合わせ、正しい説明はどれか。1つ選べ。

1. 「家庭麻薬」として管理する。
2. かぎのかかる堅固な保管庫での保管が必要である。
3. 管理者を決めて病院又は薬局に届け出る必要がある。
4. 医師の許可があれば海外旅行に携帯できる。
5. 不要となった残薬は調剤した薬局に返却できる。

## 正解
**5. 不要となった残薬は調剤した薬局に返却できる。**

## 解説

- 選択肢1（誤）: 家庭麻薬とは「1%以下のコデイン、ジヒドロコデイン等を含有するもの」で、オキシコドンは該当しない
- 選択肢2（誤）: 調剤済み麻薬の家庭での管理に鍵のかかる保管庫は不要（施設の保管義務とは異なる）
- 選択肢3（誤）: 管理者の届出義務はない
- 選択肢4（誤）: 麻薬の海外携帯には地方厚生局長の許可が必要。医師の許可だけでは不十分
- 選択肢5（正）: 不要となった調剤済み麻薬は、調剤した薬局に返却できる。薬局が受け取り後に廃棄手続きを行う
