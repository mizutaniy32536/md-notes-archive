# 107-317

# 第107回薬剤師国家試験 問317

## 問題文

この患者家族が、在宅で調剤済みのオキシコドンを管理する場合、麻薬及び向精神薬取締法に照らし合わせ、正しい説明はどれか。1つ選べ。[2]

## 選択肢

1. 「家庭麻薬」として管理する。[2]

2. かぎのかかる堅固な保管庫での保管が必要である。[2]

3. 管理者を決めて病院又は薬局に届け出る必要がある。[2]

4. 医師の許可があれば海外旅行に携帯できる。[2]

5. 不要となった残薬は調剤した薬局に返却できる。[2]

## 正解

**5番**[1][3]

## 解説

**選択肢1 - 誤り**

「家庭麻薬」という概念が存在しますが、これはコデインやジヒドロコデインのように**1％以下の濃度で含有される医療用医薬品**のことを指します。オキシコドンは麻薬そのものであり、「家庭麻薬」ではありません。[1]

**選択肢2 - 誤り**

在宅での調剤済み麻薬（すでに調剤された状態で患者に渡された麻薬）については、施錠が必要な堅固な保管庫での保管は法的に要求されていません。

**選択肢3 - 誤り**

調剤済み麻薬を在宅で管理する場合、管理者を決めて病院又は薬局に届け出る必要はありません。

**選択肢4 - 誤り**

麻薬を医師の許可があっても海外旅行に携帯することはできません。麻薬及び向精神薬取締法により、麻薬の国外持ち出しは禁止されています。

**選択肢5 - 正解**

不要となった調剤済み麻薬の残薬は、調剤した薬局に返却する必要があります。薬局が受け取った調剤済み麻薬は、薬局が廃棄します。[1][3]
