# 107-227

**第107回薬剤師国家試験 問227**

図は、大腸がんの多段階発がんの過程を示したものである。A及びBに該当するがん抑制遺伝子の組合せとして、正しいのはどれか。1つ選べ。[1][5][6]

**選択肢**  
1. BRCA1　WT1[1]  
2. BRCA1　DCC[1]  
3. MYC　MLH1[1]  
4. APC　PTEN[1]  
5. APC　DCC[1]

**正解番号：5** [1][7][9]

**解説**  
大腸がんの多段階発がん過程では、**APC変異** → KRAS活性化 → p53変異 → **DCC変異**の流れが知られており、Aは**APC**、Bは**DCC**に該当するがん抑制遺伝子である。[1]  
選択肢1・2：BRCA1は遺伝性乳がんに関わる遺伝子で、大腸がんとは無関係のため誤り。[1]  
選択肢3：MYCはがん促進遺伝子（オンコジーン）であり、がん抑制遺伝子ではない。[1]  
選択肢4：PTENは他の癌に関与するが、大腸がんのこの過程のBには該当しない。[1]  
以上より、正解は5。[1]
