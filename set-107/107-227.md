# 107-227

大腸がんの多段階発がん。A及びBに該当するがん抑制遺伝子の組合せは？

1. BRCA1/WT1
2. BRCA1/DCC
3. MYC/MLH1
4. APC/PTEN
5. APC/DCC

## 正解
**5. APC/DCC**

## 解説
Vogelsteinモデル：APC変異（ゲートキーパー）→KRAS活性化→DCC変異→p53変異→がん
