# 107-12

尿素回路の代謝中間体であるオルニチンはどれか。１つ選べ。

**正解:** １

**解説**
１　正
L−オルニチンの構造である。L−オルニチンは、尿素回路における代謝中間体であり、L−アルギニンの尿素の放出により生成する。
２　誤
γ−アミノ酪酸（GABA）の構造である。γ−アミノ酪酸（GABA）はグルタミン酸の脱炭酸反応によって生成する。
３　誤
アルギニンの構造である。アルギニンは、尿素回路における代謝中間体であり、アルギニノコハク酸からフマル酸が放出されることにより生成する。
４　誤
尿素の構造である。尿素はピリミジン塩基などの核酸塩基の分解によって生成する。
５　誤
尿酸の構造である。尿酸はプリン塩基などの核酸塩基の分解によって生成する。
