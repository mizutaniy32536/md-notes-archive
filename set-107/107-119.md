# 107-119

### 第107回薬剤師国家試験 問119

**問題文**  
細菌の細胞表面構造に関する記述のうち、正しいのはどれか。2つ選べ。[1][2][3][8]

**選択肢**  
1. グラム陽性細菌には、タイコ酸やリポタイコ酸が結合した厚いペプチドグリカン層が、細胞膜の内側に存在する。[1][2][3]  
2. グラム陰性細菌には、細胞膜（内膜）の外側に薄いペプチドグリカン層があり、さらにその外側には内毒素であるリポ多糖を含む外膜が存在する。[1][2][3]  
3. 淋菌などナイセリア属菌には、ミコール酸と呼ばれる長鎖脂肪酸を多量に含んだ厚い脂質層が存在する。[1][2][3]  
4. マイコプラズマには、ペプチドグリカンや細胞膜が存在しない。[1][2][3]  
5. 肺炎球菌には、多糖を主成分とする莢膜が存在する。[1][2][3]

**正解番号**  
2, 5[1][2]

**解説**  
- **1. 誤**: グラム陽性細菌の厚いペプチドグリカン層は細胞膜の**外側**に存在する。内側ではない。[1][2]  
- **2. 正**: グラム陰性細菌は細胞膜（内膜）の外側に薄いペプチドグリカン層があり、その外側にリポ多糖（内毒素）を含む外膜が存在する。グラム陽性細菌のペプチドグリカン層は厚い。[1][2]  
- **3. 誤**: ミコール酸は結核菌などの最外周部に存在する長鎖脂肪酸。ナイセリア属（淋菌）には存在しない。[1][2]  
- **4. 誤**: マイコプラズマには細胞壁（ペプチドグリカン）が存在しないが、細胞膜は存在する。[1][2]  
- **5. 正**: 肺炎球菌（ストレプトコッカス・ニューモニエ）は多糖を主成分とする莢膜を持つ（グリフィスの実験参照）。[1][2]
