# 107-47

**問題文**
炭酸水素ナトリウムの併用によって、キニジンの血中濃度が上昇する原因として最も適切なのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-47 を参照

**選択肢**
1. 消化管吸収の阻害
2. 肝代謝酵素の阻害
3. 胆汁排泄の促進
4. 腎尿細管分泌の促進
5. 腎尿細管再吸収の促進

**正解:** 5

**解説**
解答　５

炭酸水素ナトリウムは、尿pHを上昇させる薬物である。したがって、塩基性薬物であるキニジンと、炭酸水素ナトリウムとの併用により尿pHが上昇すると、腎尿細管中のキニジンの分子型分率が増加し、腎尿細管再吸収が促進されキニジンの血中濃度が上昇する。
