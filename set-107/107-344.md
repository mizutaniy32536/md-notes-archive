# 107-344

53歳男性。入院中に発症した頻脈性不整脈に対して以下の処方が開始となった。 4日後、朝服用して2時間後に血中濃度を測定すると920 ng/mLとなっていた（目標域：800 ng/mL以下）。なお、患者のeGFRは30 mL/min/1.73m2であり、肝機能値は正常範囲内であった。この症例に対する薬剤師が医師に行う提案として、適切でないのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC を参照

**正解:** ５

**解説**
シベンゾリンは腎排泄型の薬剤であり、腎機能低下時には心機能抑制作用、催不整脈作用、低血糖などの副作用が現れやすいため、腎機能に応じて投与量を減量する必要がある。
なお、シベンゾリンの過量投与への対応として、低血糖症状がみられている場合はブドウ糖の投与を行う。ただし、フロセミドは低カリウム血症を引き起こし、シベンゾリンの催不整脈作用を誘発するおそれがあるため、フロセミドの投与は避けるべきである。
