# 107-81

**問題文**
「インフォームド・コンセント」に関する記述のうち、正しいのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-81 を参照

**選択肢**
1. 治験の被験者に対してのみ実施する。
2. 同意後には撤回できない。
3. 同意後には患者情報は個人名で一般公開する。
4. 本人に意思決定能力があれば、患者の主体性を重んじる。
5. 患者の心情に配慮するため、不安を与える内容は伝えない。

**正解:** 4

**解説**
解答　４

インフォームド・コンセントは、医療行為を受ける患者や治験の被験者などが医師から十分な説明を受け、治療等の有効性や不利益等に関する事項についても理解・納得した上で、患者の自由意志により医療を選択し決定することである。

１　誤
インフォームド・コンセントは、一般に治験の被験者に対して実施するが、家族等に実施する場合もある。

２　誤
インフォームド・コンセントは、いつでも撤回可能である。

３　誤
インフォームド・コンセントで入手した情報は、個人情報保護法に準じた扱いとなるため、患者情報を個人名で一般公開してはならない。

４　正
インフォームド・コンセントは、本人に意思決定能力があれば、原則患者の主体性を重んじる。例外として、患者が未成年や精神病患者などの知的精神的判断能力を持たない場合は、家族や代理人が意思決定をする場合がある。

５　誤
インフォームド・コンセントは、治療等にかかる不利益等に関する事項についても理解・納得して上で、医療を選択し決定することであるため、不安を与える内容も伝えなければならない。
