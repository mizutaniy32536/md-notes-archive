# 107-103

主生成物としてメソ体を与えるのはどれか。１つ選べ。

**正解:** ２

**解説**
１　誤
アルケンにBr2とH2Oを反応させるとBrとOHがトランス付加した化合物が生成する。本化合物にBr2とH2Oを反応させるとラセミ体が生成する。
２　正
アルケンにBr2を反応させるとBrとBrがトランス付加した化合物が生成する。本化合物にBr2を反応させるとメソ体が生成する。
３　誤
アルケンにOsO4を反応させるとOHとOHがシス付加した化合物が生成する。よって、本化合物にOsO4を反応させるとラセミ体が生成する。
４　誤
アルケンにHBrを反応させるとHとBrがマルコフニコフ則に従った化合物が生成する。よって、本化合物にHBrを反応させるとラセミ体が生成する。
５　誤
アルケンにBH3とH2O2を反応させるとHとOHが逆マルコフニコフ則に従いシス付加した化合物が生成する。よって、本化合物にBH3反応後アルカリ性条件下でH2O2を反応させるとラセミ体が生成する。
