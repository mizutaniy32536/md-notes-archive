# 107-7

**問題文**
結合している原子に着目したとき、三角錐型の構造をもつのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-7 を参照

**選択肢**
（選択肢は画像を参照）

**正解:** 2

**解説**
解答　２

中央の原子がsp3混成の場合、分子の形は三角錐型や正四面体型のような立体構造となる。また、中央の原子がsp2混成の場合、分子の形は平面構造となり、中央の原子がsp混成の場合、分子の形は直線構造となる。

１　誤
メチルカチオンの構造である。中央の炭素原子がsp2混成であり、平面構造をもつ。

２　正
メチルアニオンの構造である。中央の炭素原子がsp3混成であり、三角錐型の構造をもつ。

３　誤
三フッ化ホウ素の構造である。中央のホウ素原子がsp2混成であり、平面構造をもつ。

４　誤
メチルラジカルの構造である。中央の炭素原子がsp2混成であり、平面構造をもつ。

５　誤
ホルムアルデヒドの構造である。中央の炭素原子がsp2混成であり、平面構造をもつ。
