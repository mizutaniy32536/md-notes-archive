# 107-63

**問題文**
子宮内膜症の治療に用いる薬剤はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-63 を参照

**選択肢**
1. エチニルエストラジオール錠
2. エンザルタミド錠
3. オキシトシン注
4. リュープロレリン酢酸塩注
5. レトロゾール錠

**正解:** 4

**解説**
解答　４

子宮内膜症は、子宮内膜やその組織が子宮内腔以外で異所性に発生する非悪性腫瘍性疾患である。異所発生部位となるのはダグラス窩（最多）や卵巣、子宮筋層などである。発症には子宮内膜増殖作用を有するエストロゲンが関与するため、薬物治療として、リュープロレリン、ブセレリンなどのGn−RHアゴニストや、ジエノゲストなどの黄体ホルモン製剤、卵胞ホルモン・黄体ホルモン配合剤（低用量ピル）などが用いらえる。
