# 107-48

図は、結晶固体及び非晶質固体の比容積と温度との関係を示したものである。温度アが示すのはどれか。１つ選べ。

１ 凝固点
２ 沸点
３ ガラス転移点
４ 融点
５ 結晶化温度

**正解:** ３

**解説**
液体をゆっくり冷却していくと融点以下では結晶が観察されるが、急速に冷却すると融点以下の温度でも結晶とならず、液体の時と同じ容積変化で推移する過冷却液体となる。さらに冷却すると、ある温度を境に容積変化率が変化し、液体に近い構造を保ったままガラスへと変化する。この過冷却液体からガラスへの転移温度をガラス転移点という。
