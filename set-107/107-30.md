# 107-30

抗てんかん薬ガバペンチンの作用点はどれか。１つ選べ。

１ 電位依存性Ca2＋
２ 電位依存性Na＋
３ シナプス小胞タンパク質SV2A
４ γ−アミノ酪酸GABAA
５ グルタミン酸AMPA受容体

**正解:** １

**解説**
ガバペンチンは、電位依存性Ca2＋チャネルの抑制によるグルタミン酸などの興奮性伝達物質の遊離抑制や、脳内GABA量を増加させ、GABAトランスポーターを活性化することによってGABA神経系機能を維持・増強することにより、抗てんかん作用を示す。
