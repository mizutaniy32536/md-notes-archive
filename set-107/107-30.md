# 107-30

**問題文**
抗てんかん薬ガバペンチンの作用点はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-30 を参照

**選択肢**
1. 電位依存性Ca2＋チャネル
2. 電位依存性Na＋チャネル
3. シナプス小胞タンパク質SV2A
4. γ−アミノ酪酸GABAA受容体
5. グルタミン酸AMPA受容体

**正解:** 1

**解説**
解答　１

ガバペンチンは、電位依存性Ca2＋チャネルの抑制によるグルタミン酸などの興奮性伝達物質の遊離抑制や、脳内GABA量を増加させ、GABAトランスポーターを活性化することによってGABA神経系機能を維持・増強することにより、抗てんかん作用を示す。
