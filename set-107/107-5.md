# 107-5

**問題文**
液体クロマトグラフィー／質量分析法において、用いられるイオン化法はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-5 を参照

**選択肢**
1. エレクトロスプレーイオン化（ESI）法
2. 化学イオン化（CI）法
3. 高速原子衝撃（FAB）法
4. 電子イオン化（EI）法
5. マトリックス支援レーザー脱離イオン化（MALDI）法

**正解:** 1

**解説**
解答　１

液体クロマトグラフィー／質量分析法（LC/MS）では、液体クロマトグラフィーで分離された試料を直接質量分析計に導入するため、液体試料のイオン化に適したエレクトロスプレーイオン化（ESI）法や大気圧化学イオン化（APCI）法が用いられる。
