# 107-5

液体クロマトグラフィー／質量分析法において、用いられるイオン化法はどれか。１つ選べ。

１ エレクトロスプレーイオン化（ESI）法
２ 化学イオン化（CI）法
３ 高速原子衝撃（FAB）法
４ 電子イオン化（EI）法
５ マトリックス支援レーザー脱離イオン化（MALDI）法

**正解:** １

**解説**
液体クロマトグラフィー／質量分析法（LC/MS）では、液体クロマトグラフィーで分離された試料を直接質量分析計に導入するため、液体試料のイオン化に適したエレクトロスプレーイオン化（ESI）法や大気圧化学イオン化（APCI）法が用いられる。
