# 107-83

**問題文**
薬学的管理の経過をSOAP形式で記録する場合、「O」の項目に記載する内容として、適切なのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-83 を参照

**選択肢**
1. 医師への処方提案内容
2. 患者の自覚症状
3. 投与方法の妥当性の評価
4. 薬に対する患者の訴え
5. 血中薬物濃度の測定値

**正解:** 5

**解説**
解答　５

SOAP形式とは、POS（問題志向型システム）を実践するための薬剤服用歴管理記録の記述法であり、患者の情報をS（Subjective data：主観的情報）、O（Objective data：客観的情報）、A（Assessment：評価）、P（Plan：計画）に分けて記載する。

１　誤
医師への処方提案内容は、Pの記載内容である。

２　誤
患者の自覚症状は、Sの記載内容である。

３　誤
投与方法の妥当性の評価は、Aの記載内容である。

４　誤
薬に対する患者の訴えは、Sの記載内容である。

５　正
血中薬物濃度の測定値は、Oの記載内容である。
