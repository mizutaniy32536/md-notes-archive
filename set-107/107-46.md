# 107-46

**問題文**
健常人におけるイヌリンの血漿中濃度と尿中排泄速度との関係を正しく表したグラフはどれか。1つ選べ。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-46 を参照

**選択肢**
（選択肢は画像を参照）

**正解:** 4

**解説**
解答　４

イヌリンは血漿タンパクと結合せず、糸球体ろ過のみにより血漿中から除去される。そのため、尿中排泄速度（dXu/dt）は、血漿中濃度の増加に伴って比例的に増加する（①式）。一方、クリアランスは、血漿中濃度が増加しても比例せず、一定の値になる。
