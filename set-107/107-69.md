# 107-69

厚生労働省が発行する資料はどれか。１つ選べ。

１ 医療用医薬品添付文書
２ 医薬品インタビューフォーム
３ 医薬品リスク管理計画
４ 医薬品・医療機器等安全性情報
５ くすりのしおり

**正解:** ４

**解説**
１　誤
医療用医薬品添付文書は、医薬品医療機器等法第52〜54条の規定に基づいて製薬企業が作成し、個々の包装に添付(封入)されたものである。医薬品の有効性・安全性・品質などの情報を収載したもので、記載項目・内容が規定された文書であり、各種情報の中で唯一法的根拠を持つ資料である。
２　誤
医薬品インタビューフォームは、医療用添付文書等の情報を補完し、医療従事者の日常業務に必要な情報が集約された総合的な医薬品解説書として、日本病院薬剤師会が記載要領を策定し、製薬企業が発行する資料である。
３　誤
医薬品リスク管理計画とは、医薬品の開発から市販後まで一貫したリスク管理を文書にまとめ、調査・試験やリスクを低減するための取り組みの進捗に合わせて、または、定期的に確実に評価が行われるようにする資料である。製薬企業が発行し、PMDAのホームページで閲覧できる。
４　正
医薬品・医療機器等安全性情報は、医薬品・医療機器等安全性情報報告制度により収集された重要な副作用情報を症例とともに記載した資料である。厚生労働省医薬食品局安全対策課が編集し、毎月発行される。
５　誤
くすりのしおりは、医師や薬剤師が患者に処方した薬剤の説明をする際に活用される補助的な資料であり、製薬企業が発行する。
