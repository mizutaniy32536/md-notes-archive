# 107-163

# 第107回薬剤師国家試験 問163

## 問題文

2型糖尿病の治療に使用される薬物に関する記述のうち、正しいのはどれか。**2つ選べ。**[2]

## 全選択肢

1. ミチグリニドは、スルホニル尿素受容体に結合してATP感受性K⁺チャネルを遮断することで、膵β細胞の細胞膜を脱分極させる。[2]

2. ピオグリタゾンは、ペルオキシソーム増殖剤応答性受容体γ（PPARγ）を活性化することで、脂肪細胞の分化を促進する。[2]

3. イプラグリフロジンは、ナトリウム－グルコース共輸送体2（SGLT2）を阻害することで、小腸でのグルコースの吸収を選択的に抑制する。[2]

4. リナグリプチンは、グルカゴン様ペプチド−1（GLP−1）受容体を活性化することで、グルコース濃度依存的にインスリン分泌を促進する。[2]

5. メトホルミンは、AMP活性化プロテインキナーゼ（AMPK）を阻害することで、骨格筋でのグルコーストランスポーター4（GLUT4）の細胞膜への移行を促進する。[2]

## 正解番号

**1、2**[1][5]

## 解説

**選択肢1（正）**

ミチグリニドは速効性インスリン分泌促進薬で、膵臓のβ細胞膜に存在するスルホニル尿素受容体に結合し、ATP感受性K⁺チャネルを遮断することで、膵β細胞の細胞膜を脱分極させる。これによりインスリン分泌を促進する。[1]

**選択肢2（正）**

ピオグリタゾンはインスリン抵抗性改善薬で、ペルオキシソーム増殖剤応答性受容体γ（PPARγ）を活性化することで、脂肪細胞の分化を促進する。[1]

**選択肢3（誤）**

イプラグリフロジンはSGLT2阻害薬で、グルコーストランスポーターを阻害することにより、**腎臓におけるグルコースの再吸収を阻害**し、尿中へのグルコース排出を促進させることにより血糖降下作用を示す。小腸でのグルコースの吸収を選択的に抑制するわけではない。[1]

**選択肢4（誤）**

リナグリプチンはDPP-4（Dipeptidyl Peptidase-4）選択的阻害薬である。GLP-1受容体を活性化するのではなく、DPP-4酵素を阻害することでGLP-1の不活化を防ぐ。[1]

**選択肢5（誤）**

メトホルミンはAMP活性化プロテインキナーゼ（AMPK）を**活性化**するのであって、阻害するのではない。[1]
