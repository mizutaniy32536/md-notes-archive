# 107-108

日本薬局方収載センナに関する記述のうち、誤っているのはどれか。１つ選べ。

１ マメ科植物Cassia angustifolia
２ 確認試験としてマグネシウム－塩酸反応が用いられる。
３ 純度試験として残留農薬（総BHC及び総DDT）の量が規定されている。
４ 瀉下作用を示す活性本体は、腸内細菌により生成したアントロン類である。
５ 妊婦や妊娠している可能性のある女性に使用する場合には流早産の危険性があるため注意を要する。

**正解:** ２

**解説**
１　正
センナは、マメ科植物Cassia angustifolia VahlまたはCassia acutifolia Delileの小葉を基原とする生薬である。
２　誤
センナの主要成分はビアントロン配糖体のセンノシドAであり、ジエチルエーテルを用いたアルカリ性呈色反応が用いられる。日本薬局方センナの確認試験の項目にも、本品の粉末0.5 gにジエチルエーテル10 mLを加えて2分間冷浸した後、ろ過し、ろ液にアンモニア試液5 mLを加えるとき、水層は黄赤色を呈する。と記載されている。
なお、確認試験としてマグネシウム-塩酸反応が用いられる生薬は、フラボノイドを含有するキジツやチンピなどがある。
３　正
センナの純度試験として、残留農薬（総BHC及び総DDT）の量が規定されている。
４　正
センナの主要成分であるビアントロン配糖体のセンノシドAは、腸内細菌による加水分解を受けて活性本体であるアントロン類のレインアンスロンに代謝され、大腸のぜん動運動を亢進させて、瀉下作用を示す。
５　正
センナやダイオウなどのビアントロン配糖体含有生薬は、妊婦や妊娠している可能性のある女性に使用する場合には流早産の危険性があるため注意を要する。
