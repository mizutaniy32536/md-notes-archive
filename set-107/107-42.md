# 107-42

アルミニウム、マグネシウム等の金属を含有する経口剤と同時に経口投与すると、吸収が低下する薬剤はどれか。１つ選べ。

１ オメプラゾール錠
２ レボフロキサシン錠
３ フェノバルビタール散
４ リボフラビン酪酸エステル錠
５ ワルファリンカリウム錠

**正解:** ２

**解説**
アルミニウム、マグネシウム等の2価以上の金属カチオンを含有する経口剤と、ニューキノロン系抗菌薬を同時に経口投与すると、消化管内で難溶性のキレートが形成され、消化管からの吸収が低下する。したがって、ニューキノロン系抗菌薬であるレボフロキサシンが正解となる。なお、2価以上の金属カチオンと難溶性のキレートを形成する代表的な薬物として、テトラサイクリン系抗菌薬、ビスホスホネート製剤などが挙げられる。
