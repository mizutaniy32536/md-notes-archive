# 107-2

### 第107回薬剤師国家試験 必須問題 問2

**問題文（全文）**  
平衡状態にある次の化学反応系に関する記述のうち、正しいのはどれか。1つ選べ。  
ΔfH°は標準生成エンタルピー、(g)は気体状態を表す。  
\[ \ce{N2(g) + 3H2(g) <=> 2NH3(g)} \]  
\[ \Delta_f H^\circ : \ce{0\ (N2),\ 0\ (H2),\ -46\ kJ/mol\ (NH3)} \][1][5][7]

**全選択肢（番号付き）**  
1. 系の温度を下げると、平衡は右側へ移動する。  
2. 系の圧力を下げると、平衡は右側へ移動する。  
3. 系に水素ガスを加えると、平衡は左側へ移動する。  
4. この反応は吸熱反応である。  
5. この反応の平衡定数は系の温度に依存しない。[1][5][7]

**正解番号**  
**1**[1][7]

**解説**  
この反応はハーバー・ボッシュ法のアンモニア合成反応で、生成物(NH3)の標準生成エンタルピーが負(-46 kJ/mol)であるため、**全体として発熱反応**である。  
- **選択肢1 正**: 発熱反応では温度を下げると平衡が生成物側(右側)へ移動する(ルシャトリエの原理)。[1][7]  
- **選択肢2 誤**: 気体分子数が左側(4分子)>右側(2分子)なので、圧力を下げると平衡は気体分子数増加側(左側)へ移動する。[1]  
- **選択肢3 誤**: 水素(H2)を加えると濃度増加により平衡はH2消費側(右側)へ移動する。[1]  
- **選択肢4 誤**: ΔfH°が負のため発熱反応(吸熱反応ではない)。[1][7]  
- **選択肢5 誤**: 平衡定数Kは\(\Delta G = -RT \ln K\)により温度Tに依存する。[1]
