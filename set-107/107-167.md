# 107-167

急性白血病治療薬に関する記述のうち、正しいのはどれか。２つ選べ。

１ シクロホスファミドは、活性酸素を発生させて、DNAを切断する。
２ ビンクリスチンは、チュブリンの重合を促進して、微小管を安定化させる。
３ シタラビンは、細胞内で三リン酸化されて、DNAポリメラーゼを阻害する。
４ ダウノルビシンは、RNAポリメラーゼを特異的に阻害する。
５ イマチニブは、Bcr−Ablチロシンキナーゼを阻害する。

**正解:** ３、５

**解説**
１　誤
シクロホスファミドは、生体内で活性化された後、腫瘍細胞のDNAのグアニン塩基をアルキル化することで、抗がん作用を示す。なお、活性酸素を発生させて、DNAを切断する薬物は、ブレオマイシンである。
２　誤
ビンクリスチンは、チュブリンの重合を阻害して、微小管重合を阻害することで、抗がん作用を示す。なお、チュブリンの重合を促進して、微小管を安定化させる薬物は、パクリタキセルなどのタキサン類である。
３　正
シタラビンは、細胞内で三リン酸化されて、シタラビン三リン酸ヌクレオチドに変換される。これが、デオキシシチジン三リン酸（dCTP）及びデオキシチミジン三リン酸（dTTP）と競合し、DNAポリメラーゼを阻害することで、抗がん作用を示す。
４　誤
ダウノルビシンは、腫瘍細胞のDNA鎖に入り込む（インターカレーション）ことで、DNAポリメラーゼやRNAポリメラーゼなどを阻害することで、抗がん作用を示す。
５　正
イマチニブは、Bcr−Ablチロシンキナーゼを阻害し、腫瘍細胞の増殖を抑制することで、抗がん作用を示す。
