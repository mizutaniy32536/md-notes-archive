# 107-8

次の2つの薬物に関する記述のうち、正しいのはどれか。１つ選べ。

１ エナンチオマーの関係にある。
２ ジアステレオマーの関係にある。
３ 構造異性体の関係にある。
４ 融点が同じである。
５ 比旋光度の絶対値が同じである。

**正解:** ２

**解説**
１　誤
キニジンとキニーネは原子の結合順序が同じで立体配置のみが異なるため、立体異性体の関係にある。さらに、キラル炭素の立体配置が一部異なる関係性にあるため、キニジンとキニーネはジアステレオマーの関係にある。
２　正

３　誤

４　誤

５　誤
