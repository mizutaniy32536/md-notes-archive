# 107-8

**問題文**
次の2つの薬物に関する記述のうち、正しいのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-8 を参照

**選択肢**
1. エナンチオマーの関係にある。
2. ジアステレオマーの関係にある。
3. 構造異性体の関係にある。
4. 融点が同じである。
5. 比旋光度の絶対値が同じである。

**正解:** 2

**解説**
解答　２

１　誤
キニジンとキニーネは原子の結合順序が同じで立体配置のみが異なるため、立体異性体の関係にある。さらに、キラル炭素の立体配置が一部異なる関係性にあるため、キニジンとキニーネはジアステレオマーの関係にある。

２　正
解説1参照

３　誤
キニジンとキニーネは立体異性体の関係にあるため、構造異性体ではない。

４　誤
キニジンとキニーネはジアステレオマーの関係にあるため、融点は異なる。

５　誤
キニジンとキニーネはジアステレオマーの関係にあるため、比旋光度の絶対値は異なる。
