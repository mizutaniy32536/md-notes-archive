# 107-90

# 第107回薬剤師国家試験 問90

## 問題文

直近10年間の世界アンチドーピング規程において、禁止物質として指定されていない薬物はどれか。1つ選べ。[1][3]

## 選択肢

1. アセタゾラミド
2. エリスロポエチン
3. メチルテストステロン
4. カフェイン
5. メチルフェニデート

## 正解番号

**4**[3][4][6]

## 解説

**選択肢1 アセタゾラミド**
アセタゾラミドは**利尿薬**であり、禁止物質として指定されています。利尿薬は尿量を増加させることで禁止物質や代謝物の尿中濃度を下げたり、急激に体重の減量を行えることがあるため、禁止されています。[3][5]

**選択肢2 エリスロポエチン**
エリスロポエチンは**造血薬**です。赤血球数を増加させることから、禁止物質として指定されています。[3][4]

**選択肢3 メチルテストステロン**
メチルテストステロンは**ホルモン系**の薬であり、禁止物質として指定されています。[3]

**選択肢4 カフェイン**
カフェインは禁止物質として指定されていません。これが正解です。[3]

**選択肢5 メチルフェニデート**
メチルフェニデート（リタリン）は第1種向精神薬です。興奮作用があり、禁止物質として指定されています。[3]
