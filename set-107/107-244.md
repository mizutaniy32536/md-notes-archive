# 第107回 問244

**問題文**
校舎が老朽化したため、一部の教室の改築が行われた。改築した教室を利用した生徒から、目、鼻、のどの刺激、めまいの訴えが続いたため、養護教諭から学校薬剤師に相談があった。学校薬剤師がこの教室内の空気中の化学物質を検査したところ、「学校環境衛生基準」で定められている2つの物質が高濃度で検出された。問244（実務）生徒の症状の原因と考えられる物質の組合せとして、正しいのはどれか。1つ選べ。問245（衛生）前問で選択した原因物質1及び原因物質2を測定するための試験法の組合せとして、正しいのはどれか。1つ選べ。

**解説**
問244　解答　３

リード文の「一部の教室の改築後、目、鼻、のどの刺激、めまいの訴えが続いた」より、揮発性有機化合物によるシックハウス症候群（シックスクール症候群）を発症したことが推測される。「学校環境衛生基準」で定められている揮発性有機化合物は、ホルムアルデヒド、キシレン、トルエン、エチルベンゼン、スチレン、パラジクロロベンゼンである。

問245　解答　５

「学校環境衛生基準」で定められている揮発性有機化合物のうち、ホルムアルデヒドはジニトロフェニルヒドラジン誘導体化法を用いた高速液体クロマトグラフ法、キシレン、トルエン、エチルベンゼン、スチレン、パラジクロロベンゼンはガスクロマトグラフ－質量分析法を用いて測定される。なお、酵素免疫測定法は家ダニ及びその糞、検知管法は一酸化炭素及び二酸化炭素、ザルツマン法は二酸化窒素などの測定に用いられる。
