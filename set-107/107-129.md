# 107-129

**問題文**
食品成分の変化に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-129 を参照

**選択肢**
1. メイラード反応とは、還元糖とアミノ酸が酵素的に反応し、シッフ塩基及びα−カルボニル化合物を形成する反応である。
2. チラミンは、アミノ酸脱炭酸酵素によりチロシンから生成され血圧上昇作用を示す。
3. 魚の腐敗臭の原因となるトリメチルアミンは、トリメチルアミンN−オキシドが酸化されることにより生じる。
4. トリプトファンは、脱アミノ反応及び脱炭酸反応によって腐敗臭を有する硫化水素を生じる。
5. 糖質が微生物により分解されて、アルコールや有機酸などの有用な化合物が生成することを発酵という。

**正解:** 2,5

**解説**
解答　２、５

１　誤
メイラード反応とは、還元糖とアミノ酸が非酵素的に反応し、シッフ塩基及びα−カルボニル化合物を形成し、メラノイジンという褐色色素を生成する一連の反応のことである。

２　正
食品に含まれるチロシンは、微生物がもつアミノ酸脱炭酸酵素によりチラミンに変換される。チラミンの生理作用として、血圧上昇作用などがある。

３　誤
魚の旨味成分であるトリメチルアミンN−オキシドは、微生物の腐敗により還元され、魚の腐敗臭の原因となるトリメチルアミンを生成する。

４　誤
腐敗によって硫化水素を生じるアミノ酸はシステインやメチオニンである。なお、トリプトファンの腐敗によって生じる物質としては、スカトールなどがある。

５　正
発酵とは、食品に含まれる糖質などが微生物の分解を受け、アルコールや有機酸（乳酸や酢酸）などの人体にとって有用な化合物を生成することである。
