# 107-181

皮膚に使用する製剤に関する記述のうち、正しいのはどれか。２つ選べ。

１ 油脂性基剤である単軟膏や白色軟膏は、乾燥型及び湿潤型いずれの皮膚疾患にも使用できる。
２ 水溶性基剤であるマクロゴールは、分子量が大きくなると水に不溶になる。
３ 水溶性ゲル基剤は、損傷皮膚を含めた様々な状態の皮膚に安全に使用できる。
４ 吸水クリームと親水クリームは、いずれもサラシミツロウを含んでいる。
５ マトリックス型のテープ剤は、日本薬局方の粘着力試験法により評価される。

**正解:** １、５

**解説**
１　正
油脂性基剤である単軟膏や白色軟膏は、乾燥型及び湿潤型いずれの皮膚疾患にも使用できる。
２　誤
水溶性基剤であるマクロゴールは、分子量にかかわらず水に可溶である。
３　誤
水溶性ゲル基剤は、損傷皮膚を含めた様々な状態の皮膚に必ずしも安全に使用できるとは限らない。
４　誤
吸水クリームには、サラシミツロウが含まれるが、親水クリームには含まれない。
５　正
日本薬局方の粘着力試験法は、貼付剤の粘着力を測定する試験であり、マトリックス型のテープ剤を含む貼付剤全般に規定されている。
