# 107-181

**問題文**
皮膚に使用する製剤に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-181 を参照

**選択肢**
1. 油脂性基剤である単軟膏や白色軟膏は、乾燥型及び湿潤型いずれの皮膚疾患にも使用できる。
2. 水溶性基剤であるマクロゴールは、分子量が大きくなると水に不溶になる。
3. 水溶性ゲル基剤は、損傷皮膚を含めた様々な状態の皮膚に安全に使用できる。
4. 吸水クリームと親水クリームは、いずれもサラシミツロウを含んでいる。
5. マトリックス型のテープ剤は、日本薬局方の粘着力試験法により評価される。

**正解:** 1,5

**解説**
解答　１、５

１　正
油脂性基剤である単軟膏や白色軟膏は、乾燥型及び湿潤型いずれの皮膚疾患にも使用できる。

２　誤
水溶性基剤であるマクロゴールは、分子量にかかわらず水に可溶である。

３　誤
水溶性ゲル基剤は、損傷皮膚を含めた様々な状態の皮膚に必ずしも安全に使用できるとは限らない。

４　誤
吸水クリームには、サラシミツロウが含まれるが、親水クリームには含まれない。

５　正
日本薬局方の粘着力試験法は、貼付剤の粘着力を測定する試験であり、マトリックス型のテープ剤を含む貼付剤全般に規定されている。
