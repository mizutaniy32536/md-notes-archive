# 107-49

分散系における分散相と分散媒の組合せのうち、正しいのはどれか。１つ選べ。

**正解:** ２

**解説**
分散系における分散相とは内相、分散媒とは外相のことである。
１　誤
サスペンションは、分散相が固体、分散媒が液体である懸濁剤である。
２　正
エマルションは、分散相が液体、分散媒が液体である乳剤である。
３　誤
エアゾ一ルは、分散相が液体又は固体、分散媒が気体である煙霧質である。
４　誤
フォームは、分散相が気体、分散媒が液体である泡沫である。
５　誤
キセロゲルは、分散相が気体、分散媒が固体である乾燥ゲルである。
