# 107-85

**問題文**
抗がん剤の無菌調製に関する記述のうち、適切なのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-85 を参照

**選択肢**
1. 調製者は、手袋を二重に装着する。
2. 調製作業は、クリーンベンチ内で行う。
3. 調製に使用するシリンジの先端部は、ルアースリップ式を用いる。
4. バイアルから薬液を吸引する場合は、バイアル内を陽圧にする。
5. 作業台が汚染した場合は、ペーパータオルで中心から外側に向かって拭き取る。

**正解:** 1

**解説**
解答　１

１　正
抗がん剤の調製者は、手袋を二重に装着することで抗がん剤への曝露を防止することができる。

２　誤
抗がん剤の調製には、内部が陰圧となっており作業者の抗がん剤への曝露を防ぐことができ、無菌調剤も可能なクラスⅡ以上の安全キャビネットを使用することが望ましい。なお、クリーンベンチは内部を陽圧に保ち、主に輸液の調製時に使用する。

３　誤
抗がん剤の調製に使用するシリンジの先端部は、注射針の脱落を防止するため、ルアーロック式を用いる。なお、ルアースリップ（ルアーチップ）式は内用シロップ剤の採取やネブライザーへの吸入液注入など、注射針の脱落による危険性が少ない場合に用いる。

４　誤
抗がん剤の調製時、バイアル内を陽圧にした状態で操作すると、薬液の飛散によって作業者が曝露する可能性があるため、抗がん剤の調製時にはバイアル内を陰圧に保つ必要がある。

５　誤
抗がん剤の調製時に作業台が汚染した場合は、手袋を着用し、汚染箇所が広がらないようにペーパータオルで外側から中心に向かって拭き取らなければならない。
