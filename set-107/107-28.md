# 107-28

**問題文**
メピバカインの局所麻酔作用の機序はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-28 を参照

**選択肢**
1. ATP感受性K＋チャネルの刺激
2. アセチルコリンNN受容体の遮断
3. 電位依存性Na＋チャネルの遮断
4. セロトニン5−HT1A受容体の遮断
5. 電位依存性T型Ca2＋チャネルの遮断

**正解:** 3

**解説**
解答　３

メピバカインなどの局所麻酔薬は、知覚神経細胞膜の電位依存性Na＋チャネルを遮断することで局所麻酔作用を示す。局所麻酔薬は投与後、一部が非イオン型（分子型）となり知覚神経細胞膜を通過し、陽イオン型に変換された後に、知覚神経軸索の内側から電位依存性Na＋チャネルを遮断する。
