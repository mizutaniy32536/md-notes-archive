# 107-3

**問題文**
図は、電位の基準となる標準水素電極の模式図である。図中の空欄 　ア　 にあてはまる数値はどれか。1つ選べ。なお、1 atmは1.013×105 Paを表す。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-3 を参照

**解説**
解答　３
