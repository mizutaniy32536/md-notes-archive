# 107-115

**問題文**
テロメアに関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-115 を参照

**選択肢**
1. テロメアは、染色体の末端に存在し、特定のDNA塩基配列の繰り返し構造を含む。
2. テロメアを開始点としてDNAが複製される。
3. 生殖細胞系列の分裂時には、テロメアは短縮する。
4. 多くのがん細胞で、テロメアを伸長させるテロメラーゼが発現している。
5. テロメア伸長の鋳型として、tRNAが利用される。

**正解:** 1,4

**解説**
解答　１、４

１　正
テロメアは、ヒトをはじめとする哺乳類の染色体末端に存在し、特定のDNA塩基配列の繰り返し構造をもつことにより染色体の保護を担っている。

２　誤
ヒトにおけるDNAの複製開始点は複数存在し、両方向に複製を進行させる。そのため、DNA末端にしか存在しないテロメアが複製開始点とはならない。

３　誤
通常の細胞におけるテロメアは、細胞分裂のたびに短縮していくが、生殖細胞やがん細胞では、テロメアの伸長を行うテロメラーゼを発現しているため、結果としてテロメアが短縮することない。そのため、無限に細胞分裂することができる。

４　正
解説3参照

５　誤
テロメアの伸長に用いる鋳型のテロメラーゼはRNAであるが、tRNAではない。なお、tRNAは、翻訳の過程でアミノ酸をリボソームに運ぶ働きをもつRNAである。
