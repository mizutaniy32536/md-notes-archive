# 107-14

**問題文**
図は、細胞分裂後期にある細胞の様子を示している。図中のAで示された構造体はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-14 を参照

**選択肢**
1. 紡錘体
2. 中心体
3. 核小体
4. 動原体
5. 収縮環

**正解:** 4

**解説**
解答　４

本問の図中のAで示された構造体は動原体である。

１　誤
紡錘体は微小管とそれに結合する様々なタンパク質などから構成される細胞骨格装置である。

２　誤
中心体は細胞の分裂の際に２つに分かれて細胞の両極に移動し、これを中心として染色体群が動くため、細胞分裂の中心的な役割を果たすと考えられる顆粒状構造体である。

３　誤
核小体は細胞の核内にみられる小体であり、rRNAの合成やリボソームの組立に関与する。

４　正
動原体はキネトコアとも呼ばれ、細胞分裂時に染色分体を引き離すために紡錘糸が結合する円盤型のタンパク質構造物である。

５　誤
収縮環は細胞分裂後期から終期にかけて形成される繊維構造を指す。収縮環の径が小さくなることで分裂溝ができ、細胞がくびれ始め、最終的にくびれきることで細胞が分裂する。
