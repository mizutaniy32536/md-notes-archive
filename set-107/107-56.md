# 107-56

全身性エリテマトーデス（SLE）に特異性の高い抗体はどれか。１つ選べ。

１ 抗チログロブリン抗体（抗サイログロブリン抗体）
２ 抗二本鎖DNA抗体
３ 抗CCP抗体（抗環状シトルリン化ペプチド抗体）
４ 抗Jo-1抗体
５ 抗GAD抗体（抗グルタミン酸デカルボキシラーゼ抗体）

**正解:** ２

**解説**
１　誤
抗チログロブリン抗体（抗サイログロブリン抗体）は、慢性甲状腺炎（橋本病）などで認められる自己抗体である。
２　正
抗二本鎖DNA抗体は、全身性エリテマトーデスで高率に認められる自己抗体である。
３　誤
抗CCP抗体（抗環状シトルリン化ペプチド抗体）は、慢性関節リウマチで高率に認められる自己抗体である。
４　誤
抗Jo-1抗体は、多発性筋炎/皮膚筋炎で高率に認められる自己抗体である。
５　誤
抗GAD抗体（抗グルタミン酸デカルボキシラーゼ抗体）は、1型糖尿病で高率に認められる自己抗体である。
