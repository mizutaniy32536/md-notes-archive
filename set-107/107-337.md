# 107-337

**問題文**
月齢5ヶ月、体重7.4 kgの患児に以下の処方が出された。患児の保護者への坐薬の使用方法の説明として、適切なのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-337 を参照

**選択肢**
1. 固くなるので、冷蔵庫は避けて部屋の棚の中などで保管してください。
2. 容器から坐剤を取り出した後、図1のように先端から肛門内に深く挿入してください。
3. 半分にする場合は、図2のように切ってください。
4. 挿入してから4〜5秒、肛門部をティッシュ等で押さえてください。
5. 2時間経過しても効果がない場合は、すぐに1回分を追加してください。

**正解:** 2,4

**解説**
解答　２、４

１　誤
本剤は油脂性基剤であるハードファットが用いられており室温では溶ける可能性があるため、直射日光を避けて冷所に保管するように説明する。

２　正
坐剤には向きがあり、先のとがった方（太い方）から肛門に挿入する必要があるため、容器から坐剤を取り出した後、先端から肛門内に深く挿入するように説明する。

３　誤
半分にする場合は、下図のように斜めに切るように説明する。

４　正
挿入した坐剤が肛門から出てくるのを防ぐために、挿入してから4〜5秒、肛門部をティッシュ等で押さえるように説明する。

５　誤
本剤の追加投与は4〜6時間以上空ける必要があるため、効果が無くても短期間のうちに1回分を追加しないように説明する。
