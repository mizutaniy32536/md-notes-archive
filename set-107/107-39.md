# 107-39

**問題文**
腎尿細管の尿酸トランスポーター（URAT1）を阻害して、尿酸再吸収を抑制するのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-39 を参照

**選択肢**
1. コルヒチン
2. フェブキソスタット
3. プロベネシド
4. アロプリノール
5. ラスブリカーゼ

**正解:** 3

**解説**
解答　３

１　誤
コルヒチンは、好中球のチュブリンと結合し、微小管形成を阻害することで好中球の関節への遊走を抑制する。

２　誤
フェブキソスタットは、キサンチンオキシダーゼを選択的かつ非競合的に阻害することで、尿酸生合成を阻害する。

３　正
プロベネシドは、腎尿細管の尿酸トランスポーター（URAT1）を阻害して、尿酸再吸収を抑制する。

４　誤
アロプリノールは、キサンチンオキシダーゼを競合的に阻害することで、尿酸生合成を阻害する。

５　誤
ラスブリカーゼは、尿酸を酸化し、アラントインと過酸化水素に分解することで、血中尿酸値を低下させる。
