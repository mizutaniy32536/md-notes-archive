# 107-54

**問題文**
薬物送達システム（DDS）の概念に基づいて製剤を開発する際の利点として、誤っているのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-54 を参照

**選択肢**
1. 病巣部位への薬物の集積
2. 薬効の持続化
3. 血液脳関門の透過性改善
4. 腎排泄の増大
5. 副作用の軽減

**正解:** 4

**解説**
解答　４

薬物送達システム（DDS）とは、必要な部位に必要な薬物量を必要時間だけ作用させるような最適化を目的に設計させた投与システムである。目的別のDDS技術として、放出制御、標的指向化、吸収改善が挙げられる。
DDSの概念に基づいて製剤を開発する際の主な利点を下記に列挙する。

・放出制御に伴う、薬効の持続化、コンプライアンスの向上
・標的指向化に伴う、病巣部位への薬物の集積性の向上
・吸収改善に伴う、膜透過性（血液脳関門の透過性など）、バイオアベイラビリティーの向上
・副作用の軽減
