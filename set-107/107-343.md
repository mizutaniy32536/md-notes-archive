# 107-343

**問題文**
65歳男性。意識障害により経口摂取困難となったため、非経口投与による栄養管理を開始することになった。主治医より高カロリー輸液の処方設計の依頼があり、以下の処方提案をした。この高カロリー輸液の非タンパク質カロリー／窒素量（NPC/N）を150にするための脂肪乳剤の液量Xに最も近い値はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-343 を参照

**選択肢**
1. 60
2. 150
3. 275
4. 550
5. 690

**正解:** 3

**解説**
解答　３

非タンパク質カロリー／窒素量（NPC/N）は、投与アミノ酸中の窒素1 gあたりの非タンパク質カロリー（kcal）であり、一般的にNPC/Nを150〜200に設定すると、タンパク質を効率よく合成できるとされている。

【NPC（kcal）】
50％ブドウ糖含有基本液400 mLのカロリーは800 kcal（50 g/100 mL×400 mL×4 kcal/g）、20％脂肪乳剤（2 kcal/mL）X mLのカロリーは2X kcal（2 kcal/mL×X mL）であるため、非タンパク質カロリー（NPC）は、800＋2X kcal…① となる。

【窒素量（g）】
10％総合アミノ酸製剤600 mLに含まれる総窒素量は9 g…② である。

【NPC/N（kcal/g）】
NPC/Nを150にするため、①、②より以下の式が成り立つ。

NPC/N＝(800＋2X)／9＝150

これを解いて、X＝275 mLとなる。
