# 107-177

粉体の性質に関する記述のうち、正しいのはどれか。２つ選べ。

１ 顕微鏡法により得られた粒子の投影像を一定方向の2本の平行線で挟んだとき、平行線間の長さに相当する粒子径をマーチン径という。
２ 同一粉体において、質量基準による粒度分布の平均粒子径より、個数基準による粒度分布の平均粒子径の方が小さい。
３ 水溶性の結晶性粉体の臨界相対湿度は、水不溶性の結晶性粉体と混合することで低下する。
４ 真密度1.4 g/cm3
５ 試料粉体の比表面積と平均粒子径が比例することから、比表面積を測定することで試料粉体の平均粒子径を求めることができる。

**正解:** ２、４

**解説**
１　誤
顕微鏡法により得られた粒子の投影像を一定方向の2本の平行線で挟んだとき、平行線間の長さに相当する粒子径をフェレー径という。なお、マーチン径とは、粒子の投影面積を二等分する一定方向の線分の長さを粒子径として表したものである。
２　正
粉体の粒子径分布において、各粒子の個数から得られる分布を個数基準分布、各粒子の質量から得られる分布を質量基準分布という。質量基準分布では、小粒子の質量に対する影響が大粒子の質量に比べて小さいため、同一粉体において、質量基準による粒度分布の平均粒子径より、個数基準による粒度分布の平均粒子径の方が小さい。
３　誤
臨界相対湿度（CRH）とは、水溶性物質において急激に吸湿が増大するときの相対湿度のことであり、CRHが小さい粉体ほど吸湿しやすい。水溶性薬物の結晶性粉体A、Bを混合した場合の臨界相対湿度（CRHAB）は粉体の混合比に関わらず、各粉体の臨界相対湿度（CRHA、CRHB）の積に等しくなり下記の式で表すことができる（エルダーの仮説）。
CRHAB＝CRHA×CRHB
従って、水溶性の結晶性粉体の臨界相対湿度は、水不溶性の結晶性粉体と混合しても変化しない。
４　正
みかけの密度は①式で表すことができる。

設問文より、「真密度1.4 g/cm3」とあるため①式に代入すると②式となる。
みかけの密度＝1.4 g/cm3×（1－空隙率）…②
また、空隙率は③式で表すことができる。
③式を②式に代入すると、④式で表すことができる。
空隙体積は、（みかけの体積－真の体積）で表すことができ④式を⑤式に変換することができる。
真の体積は、⑥式で表すことができ、真密度＝1.4 g/cm3、粉末質量70 gを代入することで求めることができる。
また、みかけの体積は、⑦式で表すことができる。空隙率0.5の粉末の場合、空隙率＝0.5、真の体積＝50 cm3を代入すると、圧縮前のみかけの体積を求めることができる。
