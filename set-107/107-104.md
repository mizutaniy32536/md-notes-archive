# 107-104

**問題文**
次の反応のうち、主生成物の構造を正しく示しているのはどれか。1つ選べ。ただし、各反応はそれぞれ適切な溶媒を用いて行い、反応終了後、適切な後処理を施したものとする。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 107-104 を参照

**選択肢**
（選択肢は画像を参照）

**正解:** 5

**解説**
解答　５

１　誤
アセチルアミノ基はオルト－パラ配向性の置換基であるため、主生成物はオルト置換体またはパラ置換体となる。なお、アセチルアミノ基の立体障害により、オルト置換体が生成しにくいため、本反応の主生成物はパラ置換体となる。

２　誤
メチル基はオルト－パラ配向性の置換基であるため、主生成物はオルト置換体またはパラ置換体となる。

３　誤
アセチル基はメタ配向性の置換基であるため、主生成物はメタ置換体となる。

４　誤
ヒドロキシ基とメチル基はともにオルト－パラ配向性の置換基であるが、ヒドロキシ基の方が強いオルト－パラ配向性の置換基であるため、主生成物はヒドロキシ基から見たオルト置換体となる。

５　正
ブロモ基はオルト－パラ配向性の置換基であるため、主生成物はオルト置換体またはパラ置換体となる。
