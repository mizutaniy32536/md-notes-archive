# 107-339

次の薬物について、治療薬物モニタリング（TDM）を実施する上での適切な採血のタイミングと、中毒域に達した場合に起こり得る副作用（中毒症状）の組合せとして、適切なのはどれか。２つ選べ。

**正解:** １、４

**解説**
治療薬物モニタリング（TDM）の採血は、一般に次回投与直前（トラフ値）に行うとされている。
１　正
腎移植時に適用されるタクロリムス水和物の採血のタイミングは、定常状態到達後の次回投与前（トラフ値）であり、副作用（中毒症状）として腎障害などがあげられる。
２　誤
敗血症に適用されるテイコプラニンの採血のタイミングは、定常状態到達後の次回投与前（トラフ値）であり、副作用（中毒症状）として肝障害などがあげられる。
３　誤
肺アスペルギルス症に適用されるボリコナゾールの採血のタイミングは、定常状態到達後の次回投与前（トラフ値）であり、副作用（中毒症状）として肝障害などがあげられる。
４　正
てんかんに適用されるバルプロ酸ナトリウムの採血のタイミングは、定常状態到達後の次回投与前（トラフ値）であり、副作用（中毒症状）として意識障害などがあげられる。
５　誤
骨肉腫に適用されるメトトレキサートの採血のタイミングは、投与後24 hr、48 hr、72 hrであり、副作用（中毒症状）として骨髄抑制などがあげられる。
