



## ◆ 要指導医薬品の販売時における薬剤師の適切な対応を選択する問題
260228

**問題**
- （問308の続き）この女性は当該医薬品の購入を希望した。販売時の薬剤師の対応として、正しいのはどれか。2つ選べ。

**選択肢**


1. 女性に情報提供及び指導を行い、その内容を理解したこと、質問がないことを確認した。
​
2. 母親も同様の症状があるが、仕事で忙しく来局できないとのことだったため、母親の分も販売した。
​
3. この医薬品は、インターネットでも購入可能であると説明した。
​
4. 別に処方箋を持参していたため、その処方薬と共に配送した。
​
5. 適正な使用のために必要と認められる数量に限り販売した。

**正解と理由**
- `1` と `5`
- **理由の構成**
	- **問題の整理**：要指導医薬品である`フルチカゾン点鼻薬`を購入希望した女性に対する販売時の薬剤師の対応として、正しいものを2つ選ぶ問題
	- **薬学的知識・仕様・制約**：`要指導医薬品`の販売には薬機法上のルールがある。`薬剤師`が`対面`で`情報提供・指導`を行い、購入者が内容を`理解`したことを確認する義務がある。また、`使用者本人`にのみ販売可能で、`インターネット販売`は不可、`配送`も不可（対面で直接手渡し）、販売数量は`適正使用に必要な数量`に限られる
	- **選択理由**：選択肢1は情報提供・指導と理解の確認であり、要指導医薬品の販売義務に合致。選択肢5は適正使用に必要な数量に限った販売であり、薬機法の規定に合致

**解法の手順（思考の流れ）**
- まず「`要指導医薬品`」の販売ルールを想起する
- 要指導医薬品のキーワード：`薬剤師`、`対面`、`情報提供・指導`、`本人のみ`、`ネット不可`、`手渡し`
- 選択肢1は対面での情報提供・指導＋理解の確認であり、販売義務そのもの → 正しい
- 選択肢5は「適正な使用のために必要と認められる数量」に限った販売であり、これも法令に基づく → 正しい
- 選択肢2（本人以外への販売）、3（ネット販売可能と説明）、4（配送）はいずれも要指導医薬品の販売ルールに反する

**他の選択肢が誤りな理由**
- 2. 母親も同様の症状があるが、仕事で忙しく来局できないとのことだったため、母親の分も販売した。
	要指導医薬品は`使用者本人`に対してのみ販売が認められている。来局していない母親の分を代理で販売することはできない。薬剤師が本人に直接、情報提供・指導を行うことが求められるため

- 3. この医薬品は、インターネットでも購入可能であると説明した。
	要指導医薬品は`インターネット販売（特定販売）が禁止`されている。対面での販売が義務づけられているため、「ネットでも購入可能」という説明は誤り。なお、`一般用医薬品`（第一類〜第三類）はネット販売が可能

- 4. 別に処方箋を持参していたため、その処方薬と共に配送した。
	要指導医薬品は`対面で直接手渡し`が原則であり、`配送`は認められていない。処方薬と一緒に配送するという対応は不適切

**キーポイント**
- `要指導医薬品`の販売4原則：`薬剤師`が、`対面`で、`使用者本人`に、`適正使用に必要な数量`に限り販売する
- 要指導医薬品は`特定販売`（インターネット・郵送等）が`禁止`されている
- 販売時には情報提供・指導を行い、購入者の`理解を確認`する義務がある

**関連知識の比較**
- `要指導医薬品`と`第一類医薬品`の違い：どちらも薬剤師が販売するが、第一類医薬品は`ネット販売可`、要指導医薬品は`ネット販売不可`
- `要指導医薬品`はスイッチ直後品目や`劇薬`指定の一般用医薬品が該当し、一定期間の安全性評価を経て一般用医薬品に移行する
- 一般用医薬品の販売区分と対応する販売者：`第一類`＝薬剤師、`第二類`・`第三類`＝薬剤師または`登録販売者`
- 処方箋医薬品は`調剤`として交付するものであり、要指導医薬品の販売とは制度上の位置づけが異なる

（一言メモ）
- 要指導医薬品の販売ルールは「対面・本人・薬剤師・数量制限・ネット不可」の5点セットで覚える
- 「インターネット販売ができるか」は要指導医薬品と一般用医薬品を区別する最も重要なポイント
- 要指導医薬品は「まだ十分な市販後データがない」から厳しいルールが課されている、と制度趣旨を理解すると記憶しやすい
