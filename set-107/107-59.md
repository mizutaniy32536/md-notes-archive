# 107-59

健常者におけるレム睡眠に関する記述のうち、正しいのはどれか。１つ選べ。

１ 急速な眼球運動が特徴である。
２ 筋肉の緊張が亢進する。
３ 入眠直後に多い。
４ 脳波で高振幅徐波を認める。
５ 加齢とともに増加する。

**正解:** １

**解説**
健常者の睡眠には、レム（REM：Rapid Eye Movement）睡眠とノンレム（Non−REM）睡眠という質的に異なる二つの睡眠状態がある。そのうち、レム睡眠とは、浅い眠りで身体は深く眠っているが脳は活発に働いている状態であり、骨格筋活動の低下と急速眼球運動を特徴とする。

４　誤
