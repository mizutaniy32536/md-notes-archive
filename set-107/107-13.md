# 107-13

図は、真核生物においてDNA上の遺伝子からタンパク質が作られるまでの過程を示している。矢印 ア で示す反応はどれか。１つ選べ。

１ 転写
２ 逆転写
３ スプライシング
４ RNA干渉
５ 翻訳

**正解:** ３

**解説**
真核細胞の遺伝子は、遺伝情報をもつエキソンと遺伝情報をもたないイントロンが混在している。mRNAを合成する際に不要なイントロンは図中のアにより切断され取り除かれる。これをスプライシングという。
１　誤
転写とは、DNAの遺伝情報が読み取られ、RNAを合成する段階のことである。
２　誤
逆転写とは、RNAの遺伝情報が読み取られ、cDNAを合成する段階のことである。
３　正
前記参照
４　誤
RNA干渉とは、RNAがいくつかのタンパク質と複合体を形成し、相補的な塩基配列をもつmRNAと特異的に結合し、切断することにより遺伝子発現が抑制される現象のことである。
５　誤
翻訳とは、mRNAの遺伝情報を基にタンパク質を合成することである。
