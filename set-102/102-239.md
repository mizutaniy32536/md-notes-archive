# 第102回薬剤師国家試験 問239

## 問題文

**たばこの煙と喫煙に関する記述のうち、正しいのはどれか。2つ選べ。**

## 選択肢

1. 喫煙によりシトクロムP450のうち、主にCYP3A4が誘導される。

2. （記載なし）

3. たばこの煙には、一酸化炭素や窒素酸化物が含まれている。

4. 妊娠中の喫煙は、低体重児の出産のリスクを高める。

5. アルコールを摂取すると、喫煙によって誘導されるのと同じCYP分子種が誘導される。

## 正解番号

**3、4**[1][2]

## 解説

**選択肢1（誤）**[1]
喫煙により主に誘導されるのはCYP1A2であり、CYP3A4ではありません。

**選択肢3（正）**[2]
たばこの煙には、一酸化炭素や窒素酸化物が含まれているという記述は正しいです。

**選択肢4（正）**[2]
妊娠中の喫煙は、低体重児の出産のリスクを高めるという記述は正しいです。

**選択肢5（誤）**[1][2]
アルコールにより誘導されるのはCYP2E1であり、喫煙によって誘導されるCYP1A2とは異なる酵素です。したがって、同じCYP分子種が誘導されるという記述は誤りです。
