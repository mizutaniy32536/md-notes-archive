# 第102回 問154

**問題文**
下記の化学構造を有する局所麻酔薬に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-154 を参照

**選択肢**
1. 薬物Aは、血中偽性コリンエステラーゼにより容易に分解される。
2. 薬物Bは、血中偽性コリンエステラーゼにより容易に分解される。
3. 薬物Cは、血中偽性コリンエステラーゼにより分解されない。
4. 薬物Bは、生理的pHにおいては、イオン型と非イオン型の平衡状態で存在し、非イオン型が速やかに細胞内に入る。
5. 薬物Cは、生理的pHにおいては、イオン型と非イオン型の平衡状態で存在し、細胞内では非イオン型がより強くNa＋チャネルを遮断する。

**正解:** 1,4

**解説**
解答　１、４

局所麻酔薬は、エステル型とアミド型に大別される。エステル型は構造中にエステル結合を有するため、投与後血中偽性コリンエステラーゼによる分解を受けやすい。一方アミド型は構造中にアミド結合を有するため、血中偽性コリンエステラーゼによる分解を受けにくい。なお、薬物Aはアミノ安息香酸エチル（エステル型）、薬物Bはリドカイン（アミド型）、薬物Cはプロカイン（エステル型）の構造式である。

１　正
薬物A（アミノ安息香酸エチル）はエステル型であり、血中偽性コリンエステラーゼにより容易に分解される。

２　誤
薬物B（リドカイン）はアミド型であり、血中偽性コリンエステラーゼにより分解されない。

３　誤
薬物C（プロカイン）はエステル型であり、血中偽性コリンエステラーゼにより容易に分解される。

４　正
薬物B（リドカイン）などの局所麻酔薬は一般に弱塩基性薬物であり、生理的pHにおいては、イオン型と非イオン型の平衡状態で存在し、非イオン型（分子型）が速やかに知覚神経細胞内に入る。その後知覚神経細胞内では陽イオン型となり、イオン型がより強くNa＋チャネルを遮断する。

５　誤
解説４参照。
