# 第102回 問78

**問題文**
自営業などの人を対象とする医療保険はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-78 を参照

**選択肢**
1. 船員保険
2. 国民健康保険
3. 共済組合
4. 組合管掌健康保険
5. 全国健康保険協会管掌健康保険

**正解:** 2

**解説**
解答　２

自営業などの人を対象とする医療保険は、国民健康保険である。
次に主な公的保険制度の被保険者（対象者）及び保険者を示す。
