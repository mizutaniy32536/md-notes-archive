# 第102回 問85

**問題文**
医療用医薬品としてすでに使われている有効成分が転用された要指導医薬品及び一般用医薬品を何というか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-85 を参照

**選択肢**
1. 指定薬物
2. ジェネリック医薬品
3. オーファンドラッグ
4. スイッチOTC
5. ダイレクトOTC

**正解:** 4

**解説**
解答　４

医薬品は、通常、医療用医薬品として開発され、市販後、有効性や安全性に問題がない場合には、その有効成分が要指導医薬品及び一般用医薬品に転用（スイッチ）されることがある。このような医薬品をスイッチOTCという。これに対し、直接、要指導医薬品及び一般用医薬品として開発されたものをダイレクトOTCという。

１　誤
指定薬物とは、医薬品医療機器等法において「中枢神経系の興奮もしくは抑制又は幻覚の作用を有する蓋然性が高く、かつ、人の身体に使用された場合に保健衛生上の危害が発生するおそれがあるもの」と定義されている。 

２　誤
ジェネリック医薬品（後発医薬品）とは、先発医薬品と同一の有効成分を同一量含有し、同一経路から投与する製剤で、先発医薬品と同等の臨床効果・作用が得られる医薬品をいう。

３　誤
オーファンドラッグ（希少疾病用医薬品）とは、対象者数（患者数）が日本国内において５万人に達しない疾病に対する医薬品で、かつ、その使用価値が特に高いものとして厚生労働大臣が指定したものをいう。

４　正
