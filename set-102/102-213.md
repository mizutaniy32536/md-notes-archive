# 第102回薬剤師国家試験 問213

## 問題文

釣藤散の構成生薬に関する記述のうち、正しいのはどれか。**2つ選べ。**[1][3]

## 全選択肢

1. チョウコウトウは、アルカロイドであるヒルスチンを含有する。[3]

2. セッコウは、天然の含水硫酸カルシウムである。[3]

3. ショウキョウは、辛味成分であるリモニンを含有する。[3]

4. ボウフウは、フラボノイドであるアコニチンを含有する。[3]

5. カンゾウは、ジテルペンサポニンであるグリチルリチン酸を含有する。[3]

## 正解番号

**1、2**[1][3]

## 解説

**選択肢1（正）**
チョウコウトウ（釣藤鈎）はアルカロイドであるヒルスチンを含有する正しい記述です。[1]

**選択肢2（正）**
セッコウ（石膏）は天然の含水硫酸カルシウムである正しい記述です。[1]

**選択肢3（誤）**
ショウキョウ（生姜）は辛味成分としてジンゲノールやショウガオールを含有します。[2]リモニンは柑橘類に含まれる苦味成分であり、辛味成分ではありません。[2]

**選択肢4（誤）**
ボウフウ（防風）はフラボノイドではなく、クマリン誘導体（フラキシジンなど）を含有します。[1]また、アコニチンはアルカロイドであってフラボノイドではありません。

**選択肢5（誤）**
カンゾウ（甘草）が含有するグリチルリチン酸はジテルペンサポニンではなく、**トリテルペンサポニン**です。[1]
