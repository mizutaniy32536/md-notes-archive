# 第102回 問264

**問題文**
58歳男性。手術不能の直腸がんと診断され、以下に示すレジメンに従った化学療法を施行することとなった。問264（薬理）処方薬の作用機序として正しいのはどれか。2つ選べ。

**解説**
問264　解答　２、４

本設問の化学療法は、FOLFIRI（フルオロウラシル、レボホリナートカルシウム、イリノテカン併用療法）＋ベバシズマブ療法である。

１　誤
本処方薬に設問の作用機序に該当する薬物はない。処方薬のベバシズマブは、血管内皮細胞増殖因子（VEGF）受容体ではなく、VEGFに結合するモノクローナル抗体製剤である。なお、VEGF受容体に結合するモノクローナル抗体製剤には、ラムシルマブがある。

２　正
イリノテカンの作用機序である。イリノテカンは、トポイソメラーゼⅠを阻害することでDNA合成を阻害し、がん細胞の増殖を抑制する。

３　誤
本処方薬に設問の作用機序に該当する薬物はない。なお、アルキル化薬には、シクロホスファミドなどの抗悪性腫瘍薬がある。

４　正
フルオロウラシルの作用機序である。フルオロウラシルは、生体内で活性体に変換され、チミジル酸合成酵素を不可逆的に阻害することでDNA合成を阻害し、がん細胞の増殖を抑制する。

問265　解答　１、３

本選択肢のうち、薬剤師による初回面談の際に説明すべき処方薬の副作用として適切なのは、高血圧と下痢である。
本処方薬のうち、重大な副作用として高血圧性脳症、高血圧性クリーゼが報告されている薬物にベバシズマブがあり、重大な副作用として高度な下痢や激しい下痢が報告されている薬物にイリノテカン、フルオロウラシルがある。なお、ざ瘡様皮疹や認知機能障害が報告されている薬物は処方薬にはなく、重大な副作用ではなくその他の副作用として高血糖や耐糖能異常が報告されている薬物はベバシズマブやフルオロウラシルなどが該当するが、初回面談の際に説明すべき副作用ではない。

問266　解答　５
赤血球中のヘモグロビンが分解されて血液中に出てきたものを間接ビリルビン（非抱合型ビリルビン）といい、間接ビリルビンが肝臓でグルクロン酸抱合を受けたものを直接ビリルビン（抱合型ビリルビン）という。直接ビリルビンは、その後分解を受けて排泄される。
体質性黄疸は、グルクロン酸転移酵素（UGT）活性障害による間接ビリルビン上昇型と、直接ビリルビン排泄障害による直接ビリルビン上昇型に分類されるが、本患者では、間接ビリルビン値が高かったことから、UGT活性障害による体質性黄疸と考えられる。
処方薬のうち、イリノテカンは、体内でエステラーゼにより活性代謝物のSN−38に変換され抗悪性腫瘍作用を示す。その後、UGTによりグルクロン酸抱合を受け、主に胆汁中に排泄されるが、UGT活性が低い場合は代謝に遅延が生じるためイリノテカンの副作用発現率が高くなる。よって、処方薬の副作用を予測するために推奨すべき遺伝子診断の対象となるのは、UGTをコードする遺伝子であるUGT1A1遺伝子である。
