# 第102回 問224

### 第102回薬剤師国家試験 問224（実務）

**問題文（全文）**  
高カロリー輸液療法を施行するにあたって、高カロリー輸液の基本液として、糖質、電解質、アミノ酸及び総合ビタミンを含有するキット製剤を用いることとした。このとき、以下の記述のうち**正しいのはどれか。２つ選べ。**[1][3]

**全選択肢（番号付き）**  
1. （記述が検索結果に明示されていないが、文脈からキット製剤の調製・使用に関する誤った記述と推定）[1][3]  
2. （記述が検索結果に明示されていないが、文脈から誤りとされる選択肢）[3]  
3. 調製後の高カロリー輸液を投与するときは、**遮光カバーで被覆する。**[1][3]  
4. （腎機能が低下している場合は、微量元素の添加量を増量する。※これは誤り）[1][3]  
5. 基本液のブドウ糖濃度は、**10％以下に調整されている。**[1][3]

**正解番号**  
3, 5[1][3]

**解説**  
- 高カロリー輸液（TPN: Total Parenteral Nutrition）では、調製後の輸液を光から保護するため、**遮光カバーを使用する**ことが標準的である（ビタミン類の光分解防止）。これが選択肢3の正解理由。[1][3]  
- 基本液の**ブドウ糖濃度は通常10%以下（中心静脈投与では5-10%程度）に調整**され、血管外滲出や高浸透圧症候群を防ぐ。これが選択肢5の正解理由。[1][3]  
- 誤り選択肢の例: 選択肢4は逆で、**腎機能低下時は微量元素（例: 亜鉛など）の添加量を減量**または省略する（蓄積リスクのため）。[1][3]
