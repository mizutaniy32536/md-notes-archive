# 第102回 問227

### 第102回薬剤師国家試験 問227

**問題文**  
65歳女性。慢性腎不全にて通院治療中。最近、時々腰が痛くなり、寝つきも良くないので、整形外科を受診した。骨粗しょう症と診断され、処方箋を薬局に持参した。  
アルファカルシドールカプセルはビタミンD製剤である。ビタミンDの代謝反応のうち、慢性腎不全の患者において低下しているのはどれか。１つ選べ。[1][2]

**全選択肢**  
1. **7-デヒドロコレステロールの開環反応**  
2. **7-デヒドロコレステロールの1位の水酸化反応**  
3. **25-(OH)ビタミンDの1位の水酸化反応**  
4. **1α-(OH)ビタミンDの25位の水酸化反応**  
5. **25-(OH)ビタミンDの開環反応**[1][2]

**正解番号**  
3[1][2]

**解説**  
ビタミンDは肝臓で25位が水酸化されて25-(OH)ビタミンDとなり、その後腎臓で1位が水酸化されて活性型1,25-(OH)₂ビタミンDとなる。慢性腎不全では腎臓での1位水酸化反応が低下するため、アルファカルシドール（1α-(OH)ビタミンD₃）はこのステップをバイパスして使用される。正解は**25-(OH)ビタミンDの1位の水酸化反応**。[1][2]
