# 第102回 問243

### 第102回薬剤師国家試験 問243

**問題文**  
学校薬剤師が小学校の水道水の水質検査を行った。結果は以下の通りであった。  
（注: 実際の問題では水質検査結果の表が掲載されており、一般細菌が基準超過、大腸菌未検出、塩化物イオン濃度上昇、TOC値、トリハロメタン濃度高値、遊離残留塩素基準内などのデータが示されている。[1][2][3]）

この水道水の水質検査に関する次の記述のうち、正しいのはどれか。**2つ選べ。**[1][2][3]

**選択肢**  
1. 大腸菌は検出されていないが、一般細菌が検出されているので、水質基準を満たしていない。  
2. 塩化物イオンの濃度は、し尿等の混入があると値が増加する。  
3. 全有機炭素（TOC）の測定値は、水道水中の還元性無機イオンの影響を受けにくい。  
4. トリハロメタンの濃度が高いと色度、濁度のいずれも高くなる。  
5. 遊離残留塩素が水質基準を超えているため、このままでは飲料に適さない。[1][2][3]

**正解番号**  
**2, 3**[1][2]

**解説**  
- **選択肢1**: 誤り。水質基準では一般細菌は100 CFU/mL以下、大腸菌は検出されなければ基準を満たす。一般細菌が検出されていても基準値内であれば問題ない。[1]  
- **選択肢2**: 正しい。塩化物イオン濃度はし尿、下水、海水、工場排水等の混入で増加する。[1][2]  
- **選択肢3**: 正しい。TOC（総有機炭素）は有機物の指標で、還元性無機イオン（例: 亜硫酸イオンなど）の影響を受けにくい。[2]  
- **選択肢4**: 誤り。トリハロメタンは無色無臭のため、濃度が高くても色度や濁度は変化しない。[1]  
- **選択肢5**: 誤り。遊離残留塩素の基準は0.1 mg/L以上であり、基準を満たしていれば飲料に適する。[1][2]
