# 第102回 問315

### 第102回薬剤師国家試験 問315

**問題文**  
麻薬診療施設における麻薬の管理者に関する記述として、正しいのはどれか。**2つ選べ。**[2][3]

**全選択肢**  
1. 2人以上の麻薬施用者が診療に従事する麻薬診療施設の開設者は、自らが麻薬管理者となる場合を除き、当該麻薬診療施設に麻薬管理者を置かなければならない。[2][3]  
2. 麻薬の廃棄は、調剤済麻薬でないのであれば事前に届け出が必要です。「廃棄してから」では、ありません。（※注: 選択肢2は誤りとして解説あり。詳細は完全問題集参照）[1]  
3. （詳細未記載、誤り）[2]  
4. 麻薬管理者は事故の届け出をした麻薬の品名及び数量を麻薬診療施設に備えた帳簿に記載しなければならない。[3]  
5. 麻薬管理者の免許は、医師でなければ受けることができない。（誤り）[3]

**正解番号**  
**1、4**[2][7]

**解説**  
- **選択肢1 正**: 麻薬及び向精神薬取締法に基づき、2人以上の麻薬施用者が診療に従事する麻薬診療施設の開設者は、自らが麻薬管理者となる場合を除き、麻薬管理者を置くことが義務付けられている。[2][3]  
- **選択肢2 誤**: 調剤済みの麻薬でない廃棄の場合、事前届け出が必要であり、「廃棄してから」報告では不適切。[1]  
- **選択肢3 誤**、**選択肢5 誤**: 麻薬管理者は医師に限定されず、事故届け出後の帳簿記載も正しく選択肢4のみ該当。[2][3]
