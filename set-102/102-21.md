# 第102回 問21

**問題文**
特殊健康診断で、作業者の尿中から下図の構造の物質が検出された。作業者が曝露されたと考えられる物質はどれか。1つ選べ。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-21 を参照

**選択肢**
1. アニリン
2. キシレン
3. スチレン
4. トルエン
5. ベンゼン

**正解:** 4

**解説**
解答　４

本問の構造は馬尿酸である。馬尿酸は、トルエン、ベンジルアルコール、安息香酸などに曝露した時に尿中から検出され、化学物質による曝露レベルや健康影響を推定する生体指標（バイオマーカー）として用いられる。

１　誤
アニリンの曝露によって、尿中にp-ニトロフェノールが排泄される。

２　誤
キシレンの曝露によって、尿中にメチル馬尿酸が排泄される。

３　誤
スチレンの曝露によって、尿中にマンデル酸が排泄される。

４　正
前記参照

５　誤
ベンゼンの曝露によって、尿中にフェノールが排泄される。
