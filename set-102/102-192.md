# 第102回 問192

**問題文**
虚血性心疾患とその治療に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-192 を参照

**選択肢**
1. 労作性狭心症の発作の原因は、冠動脈のれん縮である。
2. 不安定狭心症は、心筋梗塞に移行しやすい。
3. 心筋梗塞発作後、数時間でST波の低下が認められる。
4. 硝酸薬は耐性を生じることがあるため、テープ剤や軟膏剤の場合には休薬期間を設けることが推奨される。
5. β遮断薬は冠動脈がれん縮している狭心症の第1選択薬として用いる。

**正解:** 2,4

**解説**
解答　２、４

１　誤
労作性狭心症は労作時に発作があらわれる狭心症で、主な原因は冠動脈の器質的狭窄（主に動脈硬化によるもの）であり、血液供給が低下した状態で労作によって心筋酸素消費量を増大させることにより発症する。なお、冠動脈のれん縮が原因となるのは、安静時狭心症である。

２　正
不安定狭心症は、不安定プラークの破裂による血栓形成などにより冠動脈の完全閉塞を起こしやすいため、心筋梗塞に移行しやすい。

３　誤
心筋梗塞では、心臓壁の壊死貫通により、発作後数時間でST波の上昇が認められる。

４　正
硝酸薬は長期使用により耐性を生じ、作用が減弱することがあるため、テープ剤や軟膏剤の場合には休薬期間を設けることが推奨される。

５　誤
アドレナリンβ遮断薬は、冠動脈のアドレナリンβ2受容体を遮断することでれん縮を悪化させる恐れがあるため、冠動脈がれん縮している狭心症には適さない。なお、冠動脈がれん縮している狭心症の予防の第一選択には、Ca２＋チャネル遮断薬が用いられる。
