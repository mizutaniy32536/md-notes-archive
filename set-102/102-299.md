# 第102回薬剤師国家試験 問299

## 問題文

56歳男性。一般用医薬品を購入するため薬局を訪れた。現在使用している処方薬について薬剤師が確認したところ、持参したお薬手帳から、点眼薬を使用していることが判明した。

また、お薬手帳には、点眼薬による治療開始前に測定された眼圧が記載されていた。

（眼圧）右28 mmHg、左27 mmHg

**この患者に一般用医薬品を推奨する場合、避けるべき成分はどれか。2つ選べ。**[1][2]

## 全選択肢

1. チキジウム臭化物
2. ジヒドロコデインリン酸塩
3. クロルフェニラミンマレイン酸塩
4. デキストロメトルファン臭化水素酸塩
5. 無水カフェイン

## 正解番号

**1、3**[1][2]

## 解説

患者の眼圧は右28 mmHg、左27 mmHgであり、正常眼圧（10～20 mmHg）より高い状態にあります。したがって、眼圧をさらに上昇させるおそれがある成分は避けるべきです。

**避けるべき理由：**

眼圧の上昇を来たすおそれがあるため、**抗コリン作用を持つ成分を避けるべき**と考えられます。選択肢の中で抗コリン作用を持つのは1と3です。[1][2]

- **選択肢1（チキジウム臭化物）**：胃腸薬に含まれることがあります。[1]
- **選択肢3（クロルフェニラミンマレイン酸塩）**：風邪薬、鼻炎薬、酔い止めなどに含まれることがあります。[1]

選択肢2、4、5は眼圧上昇に関連しない成分であるため、避ける必要がありません。
