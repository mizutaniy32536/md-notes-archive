# 第102回 問303

### 第102回薬剤師国家試験 問303

**問題文（全文）**  
配合変化について検討するために、ブロムヘキシン塩酸塩注射剤の特徴について調査したい。医薬品インタビューフォームから入手できない情報はどれか。１つ選べ。[2][6]

**全選択肢（番号付き）**  
1. 製剤の安定性に関するデータ  
2. 有効成分の安定性に関するデータ  
3. アンプル開封後の使用期限  
4. 浸透圧比  
5. 注射液のpH[1][6]

**正解番号**  
3[1][6]

**解説**  
医薬品インタビューフォーム（IF）には、製剤の安定性（加速試験・過酷試験結果）、有効成分の安定性、注射液のpH、浸透圧比などの製剤情報・臨床試験情報が記載されるが、アンプル開封後の使用期限は添付文書に記載されるもので、IFから入手できない。[1][2][6]  
問302の文脈（ブロムヘキシン塩酸塩注射剤とフロセミド注射剤の側管投与可否）で、IFを活用した配合変化検討が背景にある。[1][2]
