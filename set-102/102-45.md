# 第102回 問45

**問題文**
一般に、高齢者において増大するのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-45 を参照

**選択肢**
1. 胃内容排出速度
2. 血漿中アルブミン濃度
3. 脂溶性薬物の体重あたりの分布容積
4. 肝臓での薬物代謝活性
5. 糸球体ろ過速度

**正解:** 3

**解説**
解答　３

老化によってほとんどの臓器は構成細胞の退化と臓器萎縮をきたし、高齢者では一般的に種々の臓器機能が低下する。そのため有害作用や副作用が発現しやすくなるので、高齢者に薬物を投与する際には注意しなければならない。

１　誤
老化に伴って消化管機能が低下し、胃内容排出速度は低下する。

２　誤
老化に伴って肝機能が低下し、タンパク合成能が低下する。そのため、血漿中アルブミン濃度は低下する。

３　正
老化に伴って総水分量が低下することにより体脂肪の割合が増加するため、脂溶性薬物の体重あたりの分布容積は増大する。

４　誤
一般的に、高齢者では肝臓での薬物代謝活性は低下している。ただし、第Ⅱ相反応による抱合能は、第Ⅰ相反応ほど加齢による影響は受けにくい。

５　誤
老化に伴って腎機能が低下し、糸球体ろ過速度（GFR）は低下する。
