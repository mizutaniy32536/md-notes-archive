# 第102回 問257

### 第102回薬剤師国家試験 問257

**問題文（薬理）**  
前問の「注意すべき副作用」の発現機序として誤っているのはどれか。**２つ選べ。**[1][3]

**全選択肢**  
1. 心臓におけるアドレナリンβ<sub>1</sub>受容体の刺激  
2. 傍糸球体細胞におけるアドレナリンβ<sub>2</sub>受容体の刺激  
3. 心臓におけるアセチルコリンM<sub>2</sub>受容体の遮断  
4. 唾液腺におけるアセチルコリンM<sub>3</sub>受容体の遮断  
5. T細胞でのサイトカイン産生抑制による免疫抑制[1][3]

**正解番号**  
**2、5**[1]

**解説**  
問256の処方薬（インダカテロールマレイン酸塩：長時間作用型β<sub>2</sub>作動薬、及びチオトロピウム臭化物：抗コリン薬）の注意副作用（口渇、動悸）に対する発現機序を問う。[1][3]  

- **1 正しい**：動悸はインダカテロールのβ<sub>1</sub>刺激による。[1]  
- **2 誤り**：血清カリウム濃度上昇は選択肢にない（問256の注意副作用ではない）。傍糸球体細胞のβ<sub>2</sub>刺激はむしろカリウム低下を招く。[1]  
- **3 正しい**：一部の抗コリン薬がM<sub>2</sub>遮断で頻脈（動悸）を起こす可能性。[1]  
- **4 正しい**：口渇はチオトロピウムのM<sub>3</sub>遮断による。[1]  
- **5 誤り**：免疫抑制（口腔カンジダ症）は問256の注意副作用ではなく、吸入ステロイドによるもの（選択肢にない）。[1]
