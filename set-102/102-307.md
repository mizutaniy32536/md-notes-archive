# 第102回 問307

### 第102回薬剤師国家試験 問307

**問題文（全文）**  
55歳男性。喫煙歴30年（1日20本程度）。前回の薬剤服用後に、意識がもうろうとし、ふらついたという。男性が原因を知りたいと薬局を訪れた。男性が服用していた薬剤は以下のとおりである。  
問307（法規・制度・倫理）  
男性から、今回質問した事項について、知り合いの医師にも確認してみたいので、薬歴を開示して欲しいと要望があった。対応として**適切でない**のはどれか。1つ選べ。なお、この薬局は個人情報の保護に関する法律における個人情報取扱事業者である。[1][2][3]

**全選択肢（番号付き）**  
1. 開示にかかった実費相当額の費用を患者に請求した。  
2. 薬局が定める開示手続きの方法にしたがって、開示の請求をするよう指示した。  
3. 薬歴の開示は薬局の義務ではないことを説明した。  
4. 患者の権利や利益を害するおそれのある記述があったため、該当する部分を見えないようにして開示した。  
5. 薬歴の原本は渡すことはできないので、コピーして患者に渡した。[1][2][3]

**正解番号**  
3[1][3]

**解説**  
個人情報保護法では、個人情報取扱事業者に対し、本人からの開示請求があった場合、**原則として開示する義務**がある（個人情報保護法第28条）。したがって、選択肢3の「薬歴の開示は薬局の義務ではないことを説明した」は不適切である。[1][3]  
- 選択肢1：実費相当額の請求は開示手数料として適法（個人情報保護法施行規則第14条）。適切。  
- 選択肢2：事業者が定める手続きに従うよう指示するのは適切。  
- 選択肢4：患者の権利利益を害するおそれがある記述を削除・消去して開示するのは適法（個人情報保護法第28条第5項）。適切。  
- 選択肢5：原本渡与を避けコピー開示するのは適切な運用。[1][2][3]
