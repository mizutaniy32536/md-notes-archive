# 第102回 問337

**問題文**
57歳女性。乾癬に罹患し、以下の処方により治療することになった。健康には気をつけており、日頃から以下の飲み物を常用している。以下の飲み物の中で、同時に摂取したときにこの処方薬の副作用の発現を高める可能性があるのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-337 を参照

**選択肢**
1. ミネラルウォーター
2. グレープフルーツジュース
3. 牛乳
4. トマトジュース
5. 青汁

**正解:** 3

**解説**
解答　３

本処方薬と患者が常用している飲み物のうち、同時に摂取するとこの処方薬の副作用の発現を高める可能性があるのは牛乳である。本処方薬のうち、エトレチナートはビタミンAの誘導体であり、脂溶性が高い医薬品である。そのため、エトレチナートは食事の影響を受けやすく、添付文書に「牛乳又は高脂肪食により本剤の吸収が増大するとの報告がある。」と記載されている。
