# 第102回 問321

### 第102回薬剤師国家試験 問321

**問題文（全文）**  
下記の医薬品を卸売販売業者の担当者が納品に来たので、薬剤師が薬局のカウンターにおいて検収を行った。なお、薬局は、向精神営業者に関して別段の申し出はしていないものとする。  
検収した医薬品のその後の取扱いに関する記述のうち、適切なのはどれか。1つ選べ。[1][2][3]

**全選択肢（番号付き）**  
1. 検収した医薬品を、麻薬を保管している金庫に入れた。  
2. 要指導医薬品を来局者の手に取りやすい場所に陳列した。  
3. 毒薬に相当する医薬品を無施錠で陳列した。  
4. 過剰在庫となった向精神薬を向精神薬卸売業者に返品した。[2][3]

**正解番号**  
4[2][6]

**解説**  
- 選択肢1：麻薬と覚せい剤原料は、同じ場所に保管してはいけません。不適切。[1]  
- 選択肢2：要指導医薬品は、鍵をかけた場所か来局者の手の届かない所に陳列する必要があります。不適切。[1]  
- 選択肢3：毒薬は施錠した場所に保管・陳列する必要があります。不適切。[1]  
- 選択肢4：過剰在庫となった向精神薬は、向精神薬卸売業者に返品可能です。適切。[2][3][6]
