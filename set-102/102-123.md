# 第102回 問123

**問題文**
50歳代の主婦が、家庭菜園で栽培したナスを入れたスパゲティーを食べたところ、ふらつき、ろれつがまわらない、意識混濁などの症状を呈し、救急外来を受診した。摂取したナスは、チョウセンアサガオに接木して栽培したとのことであった。このナスに含まれるチョウセンアサガオ由来の有害成分として考えられるのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-123 を参照

**選択肢**
1. アコニチン
2. アミグダリン
3. スコポラミン
4. チャコニン
5. ℓ-ヒヨスチアミン

**正解:** 3,5

**解説**
解答　３、５

チョウセンアサガオ由来の有害成分として、スコポラミン及びℓ-ヒヨスチアミンが考えられる。スコポラミン及びℓ-ヒヨスチアミンはナス科のチョウセンアサガオに含まれる有毒成分であり、有害成分の摂取によって副交感神経遮断による口渇、ふらつき、意識混濁、頻脈などが起こる。

１　誤
アコニチンは、トリカブトに含まれる有毒成分であり、ナトリウムイオンチャネルを持続的に活性化することによって、口唇や舌のしびれ、四肢のしびれ、不整脈などを引き起こす。

２　誤
アミグダリンは、青梅やアンズなどに含まれる有毒成分であり、体内でβ-グルコシダーゼによる加水分解を受けてシアン化水素を生じる。シアン化水素は、シトクロムオキシダーゼを阻害することによって呼吸抑制などを引き起こす。

３　正
前記参照

４　誤
チャコニンは、ジャガイモの発芽部位や緑皮部に含まれる有毒成分であり、コリンエステラーゼを阻害することによって胃腸障害、縮瞳、けいれん、呼吸困難などを引き起こす。

５　正
前記参照
