# 第102回 問145

**第102回薬剤師国家試験 問145**

**問題文**  
安全な血液製剤の安定供給の確保等に関する法律に関する記述のうち、正しいのはどれか。2つ選べ。[1][2][3]

**全選択肢**  
1. 血液製剤は、医薬品医療機器等法上の医薬品から除外されている。[1][2][3]  
2. 血液製剤を製造販売する場合は、この法律の規定による許可を受けなければならない。[1][2][3]  
3. 病院又は診療所以外の場所において、血液製剤の原料とする目的で、業として人体から採血するには、厚生労働大臣の許可が必要である。[1][2][3]  
4. 業として採血することは、医業にあたる。[1][2]  
5. 血液製剤の原料とする目的で採血するときは、その対価を支払うことができる。[1][2][3]

**正解番号**  
3, 4[1][2]

**解説**  
- **選択肢1（誤）**: 血液製剤は医薬品医療機器等法（薬機法）上の医薬品に該当し、除外されていません。製造販売には薬機法の許可が必要です。[1][2]  
- **選択肢2（誤）**: 血液製剤の製造販売許可は薬機法で規定されており、この法律（安全な血液製剤の安定供給の確保等に関する法律）ではありません。[1][2]  
- **選択肢3（正）**: 病院・診療所以外で血液製剤原料目的の業として採血するには、厚生労働大臣の許可が必要です。[1][2]  
- **選択肢4（正）**: 業として採血することは医業に該当します。[1][2]  
- **選択肢5（誤）**: 売血禁止（法16条）の観点から、血液製剤原料目的の採血で対価を支払うことは禁止されています。[1]
