# 第102回 問233

### 第102回薬剤師国家試験 問233

**問題文（リード文含む）**  
85歳男性。在宅にて要介護度5の寝たきり状態であったが、高熱のため入院した。入院時に患者に触れた看護師等の職員の数名が数日後かゆみを伴う皮膚症状を訴えた。その後、患者が重度の角化型疥癬と診断されたため、院内感染対策委員会にて対応策が検討された。[1][2][3]

**問233（衛生）**  
院内感染を予防するために、陰圧個室に患者を収容するなど空気感染対策をとる必要がある感染症はどれか。**2つ選べ**。[1][2][3]

**全選択肢**  
1. 水痘  
2. 結核  
3. クロストリジウム・ディフィシル感染症  
4. マイコプラズマ感染症  
5. ノロウイルス感染症[1][2][3]

**正解番号**  
1, 2[1][2]

**解説**  
- **選択肢1（水痘）**: 正しい。水痘（水疱瘡）は五類感染症で、空気感染（飛沫核感染）する。陰圧個室などの空気感染対策が必要。[1][2]  
- **選択肢2（結核）**: 正しい。結核は二類感染症で、空気感染する。陰圧個室収容などの対策が標準。[1][2]  
- **選択肢3（クロストリジウム・ディフィシル感染症）**: 誤り。嫌気性菌で空気に弱く、主に接触感染。空気感染しない。[1]  
- **選択肢4（マイコプラズマ感染症）**: 誤り。主に飛沫感染で、空気感染対策（陰圧個室）は不要。[2]  
- **選択肢5（ノロウイルス感染症）**: 誤り。主に接触感染および飛沫感染で、空気感染対策は不要。[2]  

陰圧個室は空気感染（飛沫核感染）対策として、水痘や結核などの代表的な感染症に用いられる。[1][2][3]
