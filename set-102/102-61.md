# 第102回 問61

**問題文**
状況にかかわりなく漫然とした不安が持続する神経症はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-61 を参照

**選択肢**
1. 広場恐怖
2. 強迫性障害
3. 外傷後ストレス障害（PTSD）
4. 全般性不安障害
5. 社会不安障害

**正解:** 4

**解説**
解答　４

１　誤
広場恐怖とは、家を離れたり、雑踏・公衆の場所にいくときに、助けが得られないような状況に対して不安を感じる状態である。

２　誤
強迫性障害とは、本人が無意味、不合理と認識している強迫思考や強迫行為を反復（何度も何度も繰り返し手を洗ってしまうなど）してしまう状態のことである。

３　誤
外傷後ストレス障害（PTSD）とは、強烈な心的外傷（トラウマ）を受けたことにより、強い不安や不眠、抑うつなどの症状がでる状態である。

４　正
全般性不安障害とは、特定の状況にかかわりなく漫然としたコントロール困難な不安が持続する状態である。

５　誤
社会不安障害とは、他人から注目されることへの恐怖で、強い不安や苦痛を感じる状態である。
