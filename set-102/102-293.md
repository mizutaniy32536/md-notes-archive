# 第102回 問293

### 第102回薬剤師国家試験 問293

**問題文（症例）**  
49歳男性。C型慢性肝炎の既往あり。昨年より肝硬変に起因する腹水が出現し、ループ利尿薬とアルブミン製剤が投与されていた。昨日、肝性脳症と診断され入院となり、分岐鎖アミノ酸製剤の点滴静注、ラクツロース及びカナマイシン一硫酸塩の経口投与を開始した。[2][3]

**問293（実務）**  
本症例に用いる薬物に関する記述のうち、正しいのはどれか。**２つ選べ**。[2][3]

**選択肢**  
１．**アルブミン製剤**は**血漿膠質浸透圧**を**低下**させる。  
２．**ループ利尿薬**は**血中Na⁺**を**上昇**させる。  
３．**分岐鎖アミノ酸製剤**は**フィッシャー比**を**低下**させる。  
４．**ラクツロース**は**消化管内**の**pH**を**低下**させる。  
５．**カナマイシン一硫酸塩**は**消化管内**の**アンモニア**の**発生**を**抑制**する。[1][2][3]

**正解番号**  
**４，５**[1][3]

**解説**  
- **選択肢１（誤）**：アルブミン製剤は血中アルブミン濃度を上昇させ、血漿膠質浸透圧を**上昇**させる。血中タンパク質として水分や薬物を保持し、浸透圧維持の中心的役割を担うため、投与により浸透圧は低下しない。[1][3]  
- **選択肢２（誤）**：ループ利尿薬は近位尿細管でのNa⁺再吸収を阻害し、尿中へのNa⁺排泄を促進するため、血中Na⁺は**低下**する。[1]  
- **選択肢３（誤）**：分岐鎖アミノ酸製剤（BCAA）は肝性脳症治療に用いられ、芳香族アミノ酸との比率であるフィッシャー比を**上昇**させる。[1]  
- **選択肢４（正）**：ラクツロースは腸内細菌により代謝され、短鎖脂肪酸を産生し、消化管内のpHを**酸性側（低下）**に傾ける。これによりアンモニアの非吸収性イオン化を促進する。[1][3]  
- **選択肢５（正）**：カナマイシン一硫酸塩は腸内細菌の増殖を抑制し、消化管内でアンモニア産生に関与する細菌の活動を抑え、アンモニア発生を**抑制**する。[1][3]
