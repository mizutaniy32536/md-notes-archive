# 第102回 問319

### 第102回薬剤師国家試験 問319

**問題文**  
87歳女性。寝たきり。この患者に対して発行された処方箋と残薬を、家族が薬局に持参した。  
（前問の状況説明後）その後、この患者の服薬アドヒアランスを向上させるため、処方医の指示により薬剤師が患者宅を訪問した。患者の居宅で行うことができない業務はどれか。**2つ選べ。**[1][2][3]

**全選択肢（番号付き）**  
1. 処方箋を受け取ること  
2. 薬剤を粉砕すること  
3. 疑義照会をすること  
4. 薬剤を一包化すること  
5. 薬剤を交付すること[1][2][3]

**正解番号**  
**2, 4**[1][2][3]

**解説**  
患者宅（居宅）では**調剤**を行うことができません。調剤には薬剤の調製（**粉砕**や**一包化**など）が含まれ、これらは薬局の設備・器具が必要であり、居宅では実施不可です。一方、処方箋の受取、疑義照会（医師等への確認）、薬剤の交付は在宅医療として可能です。[1][3]  
（薬剤師法・医薬品医療機器等法に基づき、調剤業務の場所制限が根拠。[3]）
