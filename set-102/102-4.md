# 第102回 問4

**問題文**
日本薬局方において、容量分析用標準液のファクターfは、通例どの範囲にあるように調製されるか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-4 を参照

**選択肢**
1. 0.850〜1.150
2. 0.900〜1.100
3. 0.950〜1.050
4. 0.970〜1.030
5. 0.990〜1.010

**正解:** 4

**解説**
解答　４

ファクター（f）とは、日本薬局方において、調製した容量分析用標準液の濃度と規定の容量分析用標準液の濃度とのずれの度合いを表す数値である。日本薬局方では通例、ファクターが0.970〜1.030の範囲になるように規定されている。
