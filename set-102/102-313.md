# 第102回 問313

### 第102回薬剤師国家試験 問313

#### 問題文（全文）
65歳男性。肝硬変にて入院となり、ラクツロースゼリーが処方された。ラクツロースゼリーを調剤すべきところを誤ってポリスチレンスルホン酸カルシウムゼリーを調剤した。病棟にて、薬剤師が気づき、服用させずに済んだ。

**問313（法規・制度・倫理）**  
医療事故への対応などは、医療法の医療の安全の確保に関する規定において定められている。この規定において、医療の安全を確保するための措置を講ずる義務又は責務を課されていないのはどれか。1つ選べ。[1][2][3]

#### 全選択肢（番号付き）
1. 薬品棚における外観類似医薬品の配置を工夫する。  
2. 取り違え防止に指さし呼称を導入する。  
3. **調剤した薬剤師個人の責任を追及する。**  
4. 病院内でインシデント事例の情報を共有する。  
5. 個人的要因を明らかにし、発生した誤りを分析する。[1][2][3]

#### 正解番号
**3**[1][2][3][5]

#### 解説
医療法の医療の安全確保に関する規定（第6条の8等）では、医療安全対策として**組織的・システム的な再発防止策**（例: 外観類似薬の配置工夫、指さし呼称導入、インシデント共有、要因分析）が医療機関、医薬品安全管理責任者等に義務付けられているが、**個人の責任追及は義務・責務として定められていない**[1][3]。  
個人の責任追求は、ミスを恐れて調剤を避ける行動を招く可能性があり、システム改善につながらず、再発防止に不適切である。他の選択肢は医療安全確保のための積極的な措置として位置づけられる。[1][2]
