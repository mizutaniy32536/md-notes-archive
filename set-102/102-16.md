# 第102回 問16

**問題文**
リノール酸において、変質の開始反応となる水素の引き抜きが最も起こりやすい部位はどこか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-16 を参照

**選択肢**
（選択肢は画像を参照）

**正解:** 3

**解説**
解答　３

リノール酸のような高度不飽和脂肪酸は、熱や光などによって活性メチレン（二重結合の間に挟まれた炭素）から水素が引き抜かれることで変質が起こる。
