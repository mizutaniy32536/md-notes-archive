# 第102回 問77

**問題文**
ソリブジン事件に関する記述のうち、正しいのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-77 を参照

**選択肢**
1. 市販直後に副作用である間質性肺炎を多数発症した。
2. 長期大量投与により亜急性脊髄視神経症を発症した。
3. アナフィラキシーショックにより死亡例が発生した。
4. 他の医薬品との相互作用により死亡例が発生した。
5. 慢性疾患への投与により網膜症を発症した。

**正解:** 4

**解説**
解答　４

ソリブジン事件では、抗ウイルス薬であるソリブジンを5-フルオロウラシル（5-FU）系抗悪性腫瘍薬と同時に使用することにより、5-FU系抗悪性腫瘍薬の代謝が阻害され、血液障害等の副作用が発生したことが問題となった。なお、この事件を契機として医薬品の併用の危険性が示唆され、医薬品の代謝も含めた情報、複数科受診による情報の共有化、具体的には、添付文書の記載要領が見直され「警告」・「禁忌」等の特に注意する必要があるものを本文冒頭に記載するよう改善されたとともに、薬剤師による薬歴管理やお薬手帳の活用が注目されるようになった。
