# 第102回 問158

**問題文**
高血圧症治療薬に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-158 を参照

**選択肢**
1. プロプラノロールは、傍糸球体細胞からのレニン分泌を抑制する。
2. アリスキレンは、アンギオテンシンIの産生を抑制する。
3. カプトプリルは、一酸化窒素(NO)の産生を抑制する。
4. カンデサルタンは、アンギオテンシンⅡの産生を抑制する。
5. エプレレノンは、アルドステロンの分泌を抑制する。

**正解:** 1,2

**解説**
解答　１、２

１　正
プロプラノロールは、アドレナリンβ受容体遮断薬であり、傍糸球体細胞のβ1受容体を遮断することで、傍糸球体細胞からのレニン分泌を抑制し、血圧を低下させる。

２　正
アリスキレンは、直接レニン阻害薬であり、アンギオテンシノーゲンからアンギオテンシンⅠ（AngⅠ）への変換を阻害することで、AngⅠの産生を抑制する。その結果、AngⅠからアンギオテンシンⅡ（AngⅡ）への合成が抑制されることで、血圧が低下する。

３　誤
カプトプリルは、アンギオテンシン変換酵素（ACE）阻害薬であり、AngⅠからAngⅡへの変換を抑制することで、血圧を低下させる。

４　誤
カンデサルタンは、アンギオテンシンⅡAT1受容体遮断薬（ARB）であり、AT1受容体を介したAngⅡの昇圧作用を抑制し、血圧を低下させる。

５　誤
エプレレノンは、アルドステロン受容体遮断薬であり、遠位尿細管及び集合管においてアルドステロンの作用に拮抗し利尿作用を示すことで、体液量を減少し血圧を低下させる。
