# 第102回 問188

**問題文**
薬物の鎮痛効果について並行群間比較試験を実施することになった。600名の患者を薬物A投与群300名と薬物B投与群300名に分け、各々の薬物について投与後の鎮痛効果をレベル1〜10までの10段階で評価した。この評価データが正規分布していない場合、薬物Aと薬物Bの鎮痛効果の差を比較するための適切な統計手法はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-188 を参照

**選択肢**
1. Chi-square test
2. Student’s t-test
3. Kruskal-Wallis test
4. Mann-Whitney U-test
5. McNemar’s test

**正解:** 4

**解説**
解答　４

本試験のデータは鎮痛効果を10段階の順序カテゴリーで表しているため、ノンパラメトリック法が用いられる。また、本試験では600名の患者を薬物A投与群300名と薬物B投与群300名に分けていることから、対応のないデータの比較を行っている。よって、本問の鎮痛効果の差を比較するための適切な統計手法は、対応のない2群間のノンパラメトリック法であるMann-Whitney U-testである。
