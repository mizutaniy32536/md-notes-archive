# 第102回 問272

**問題文**
30歳女性。排尿痛、頻尿の症状があり、近医を受診した。急性単純性膀胱炎と診断され、以下の処方せんを薬局に持参した。薬歴を確認すると、同一の医師より消化性潰瘍治療のためスクラルファート細粒90%の処方があり、毎食後に服用中であった。問272（実務）医師への疑義照会時に提案する内容として適切なのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-272 を参照

**選択肢**
1. シプロフロキサシンをノルフロキサシンに変更する。
2. シプロフロキサシンをセフジニルに変更する。
3. シプロフロキサシンを服用後、2時間以上あけてスクラルファートを服用するように用法を変更する。
4. シプロフロキサシンを増量する。
5. スクラルファートをアルギン酸ナトリウムに変更する。問273（薬剤）前問での提案の理由となるシプロフロキサシンとスクラルファートの相互作用の機序として、最も適切なのはどれか。1つ選べ。
6. 胃内pHの変化に伴う溶解性の低下
7. 消化管内における難溶解性キレートの形成
8. 消化管吸収に関わるトランスポーターの阻害
9. 血漿タンパク結合の阻害
10. 腎尿細管における再吸収の阻害

**正解:** 3,5

**解説**
問272　解答　３、５

スクラルファートは、アルミニウムイオン（3価の金属イオン）を含む薬剤である。また、ニューキノロン系抗菌薬であるシプロフロキサシンは、2価以上の金属カチオンとキレートを形成する特徴を持つ薬物である。これらを同時に服用すると、消化管内で難溶性のキレートが形成され、消化管からの吸収が遅延または阻害されることが知られている。そのため、両薬物を併用する場合は、2時間以上あけて服用することとされている。

１　誤
ノルフロキサシンは、シプロフロキサシンと同様にニューキノロン系抗菌薬であり、2価以上の金属カチオンとキレートを形成する特徴をもつ薬物である。そのため、シプロフロキサシンをノルフロキサシンに変更するという提案は不適切である。

２　誤
セフジニルは2価以上の金属カチオンとキレートを形成するため、スクラルファートと同時に服用することにより、セフジニルの吸収が低下し効果が減弱することが報告されている。そのため、シプロフロキサシンをセフジニルに変更するという提案は不適切である。

３　正
前記参照

４　誤
シプロフロキサシンを増量しても、スクラルファートと消化管内で難溶性のキレートが形成されることに変わりはないため、シプロフロキサシンを増量するという提案は不適切である。

５　正
アルギン酸ナトリウムは、胃粘膜保護作用があり消化性潰瘍治療薬として用いられる。ナトリウムイオンは1価の金属カチオンであり、シプロフロキサシンとキレートを形成しないため、スクラルファートをアルギン酸ナトリウムに変更するという提案は適切である。
