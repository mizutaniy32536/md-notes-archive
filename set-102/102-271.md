# 第102回 問271

### 第102回薬剤師国家試験 問271

#### 問題文（全文）
76歳男性。1年ほど前から安静時に手足の震えや硬直が認められ、パーキンソン病と診断された。現在まで**レボドパ・ベンセラジド配合錠**が処方され、症状は改善されている。

3日前より尿が出にくいという症状が現れたため、泌尿器科を受診し、**神経因性膀胱による排尿困難**と診断された。本患者に**適切でない薬剤**はどれか。1つ選べ。[1][4]

#### 選択肢
1. **シルドシン**  
2. **ナフトピジル**  
3. **ウラジロガミ**  
4. **タムスロシン**  
5. **プロピベリン**  
（注: 選択肢1～4はすべて**α遮断薬**。選択肢5は抗コリン薬。[1][6]）

#### 正解番号
**5**（プロピベリン）。[1][6]

#### 解説
- 本患者は**パーキンソン病**でレボドパ・ベンセラジド配合錠を服用中であり、**神経因性膀胱による排尿困難**を呈している。[1][4]
- **神経因性膀胱**（過活動膀胱や排尿困難）では、**排尿促進**が目的。**α遮断薬**（選択肢1～4）は**膀胱頸部・括約筋を弛緩**させ、**排尿を促進**するため適切（蓄尿阻害→排尿促進）。[1]
- 一方、**プロピベリン**（選択肢5、抗コリン薬）は**膀胱平滑筋を弛緩**させ**蓄尿を促進**するため、**排尿困難を悪化**させる。**適切でない**。[1]
- 添付文書上、神経因性膀胱の適応は**選択肢1（シルドシン）のみ**明確で、2～4もα遮断薬として選択可能だが、5は不適切。[6]
