# 第102回 問46

**問題文**
患者のクレアチニンクリアランスに基づいて投与設計が行われる治療薬物モニタリング（TDM）対象薬はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-46 を参照

**選択肢**
1. テオフィリン
2. テイコプラニン
3. セファレキシン
4. シンバスタチン
5. リドカイン

**正解:** 2

**解説**
解答　２

クレアチニンクリアランスは、腎機能を評価する指標として用いられ、腎消失型薬物の投与設計に考慮される。

１　誤
テオフィリンは、治療薬物モニタリング(TDM)の対象薬になるが肝消失型薬物であるため、クレアチニンクリアランスに基づいて投与設計は行われない。

２　正
テイコプラニンは、腎消失型薬物であり、患者のクレアチニンクリアランスに基づいて投与設計が行われる治療薬物モニタリング（TDM）対象薬である。

３　誤
セファレキシンは、腎機能が低下した患者に対してはクレアチニンクリアランスを用い投与設計するが、TDM対象薬ではない。

４　誤
シンバスタチンは、肝消失型薬物でありTDMの対象薬でない。

５　誤
リドカインは、TDMの対象薬になるが肝消失型薬物であるため、クレアチニンクリアランスに基づいて投与設計は行われない。
