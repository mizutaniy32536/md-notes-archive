# 第102回 問57

**第102回薬剤師国家試験 問57**

**問題文**  
閉塞性動脈硬化症に関する記述のうち、正しいのはどれか。1つ選べ。[1][2][3]

**全選択肢**  
1. 若い女性に好発する。  
2. **間欠性跛行が特徴的症状である。**  
3. 上肢の動脈に多発する。  
4. 血管炎を伴う。  
5. 患肢に熱感がある。[1][2][3]

**正解番号**  
**2**[1][2]

**解説**  
閉塞性動脈硬化症は主に50歳以上の**中年男性**に好発し、下肢動脈の動脈硬化による閉塞が原因で発症する疾患である。[1][2]  
特徴的な症状として**間欠性跛行**（歩行時に下肢、特にふくらはぎに疼痛・しびれ・冷えが生じ、休息で軽快し再び歩行可能となる状態）が認められる。[1][2]  

- 1は誤り：好発は中年男性であり、若い女性ではない。[1][2]  
- 3は誤り：主に下肢動脈に発生し、上肢ではない。[1]  
- 4は誤り：血管炎を伴わず、これはバージャー病（閉塞性血栓血管炎）の特徴。[1][2]  
- 5は誤り：血流不全により患肢に冷感・しびれが生じ、熱感はない。[1][2]
