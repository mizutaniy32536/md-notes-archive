# 第102回 問5

**問題文**
酵素反応速度を表す式はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-5 を参照

**選択肢**
1. ミカエリス・メンテンの式
2. ファントホッフの式
3. ヤングの式
4. ブラッグの式
5. ストークスの式

**正解:** 1

**解説**
解答　１

酵素反応速度式はミカエリス・メンテンの式で表すことができる。

１　正
上記参照

２　誤
ファントホッフの式は、平衡定数 Kと絶対温度Tの関係を表す式である。

３　誤
ヤングの式は、固体の表面張力γs、液体の表面張力γL、固−液界面張力γSLの3つの界面張力と接触角θの関係を表す式である。

４　誤
X線回析が生じる条件についての式をブラッグの式という。

５　誤
ストークスの式は、粒子の沈降速度と粒子径に関する式である。
