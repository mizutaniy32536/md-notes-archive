# 第102回 問231

### 第102回薬剤師国家試験 問231

#### 問題文
ある医薬品について**免疫能を低下させる可能性**が懸念された。そこで、**治験**に加えて、**PMS**においても感染症などの副作用の発現頻度を調査し、下表の結果を得た。この調査に関する記述のうち、**誤っているのはどれか。2つ選べ。**[1][2][3]

（注: 下表は検索結果から治験N=300例（ニューモシスチス肺炎0例）、PMS N=3,000例（ニューモシスチス肺炎3例、COPD発現率が高いなど）を示す内容と推定される。[1][2]）

#### 選択肢
1. このPMSでは3,000例の調査をしたことにより、治験では見つけられなかった**ニューモシスチス肺炎**の発症を見出すことができた。[1][2]  
2. 治験とは異なり、PMSにおいて**COPD（慢性閉塞性肺疾患）**の発現率が高くなったのは、PMSの対象患者に**喫煙者**が多かったことが一因と推定される。[1][2]  
（注: 検索結果で明示的に2肢までしか記載なし。他の肢3〜5はPMSの有効性調査禁止や治験審査委員会要件などに関する記述とみられ、正誤判定の対象。[1][3] 完全な全選択肢は公式PDF[4]参照推奨。）

#### 正解番号
**4, 5**[3][5]

#### 解説
- **1は正しい**：PMSの症例数N=3,000例により、治験（N=300例、0例）では検出できなかった**ニューモシスチス肺炎**（3例）を発見可能。PMSの利点として稀な副作用検出が挙げられる。[1][2]  
- **2は正しい**（または議論の余地）：PMS対象に喫煙者が多かった可能性を**推定**するのは合理的だが、表から直接のデータなしのため「推定」として許容。後ろ向き調査で検証可能。[1]  
- **4, 5が誤り**：PMSは主に**副作用**調査目的で、有効性調査は行わない[3]。また、治験審査委員会の要件（審査委員会設置者と利害のない者追加、実施施設外の医師・薬剤師参加可など）に関する誤記述が該当。[1][3]  
PMSは承認後大規模調査で稀少副作用を発見するが、患者背景差（例:喫煙率）は推定可能。[1][2] 公式正答は厚労省公表[4][5]。
