# 第102回 問247

### 第102回薬剤師国家試験 問247

**問題文（全文）**  
53歳男性。尿酸値が高く、3週間前より**処方1**で治療を開始。その2週間後、尿酸値が改善されなかったので**処方2**及び**処方3**が追加となった。  

処方2及び処方3を追加して7日後、患者より「新年会が続き、ビールを飲む量が増えており、足の親指が腫れて激しい痛みが生じてきた。」との訴えがあった。  

**問247**（薬理）  
処方された薬物及び前問で挙げた薬物の作用機序として、正しいのはどれか。**2つ選べ**。[3][4]

**全選択肢（番号付き）**  
1. **プロスタグランジンE₂の産生を抑制する。**  
2. 尿酸を酸化してアラントインと過酸化水素に分解する。  
3. 代謝物のオキシプリノール(アロキサンチン)がキサンチンオキシダーゼを阻害する。（※選択肢3以降は検索結果から部分的に確認、完全リストは公式問題集参照）[1][3][4]

**正解番号**  
**1, 4**[7]

**解説**  
- 本問は高尿酸血症治療薬（**処方1**: フェブキソスタット、**処方2**: ベンズブロマロン、**処方3**: クエン酸カリウム・クエン酸ナトリウム配合錠と推定）と問246で挙げられた薬物（コルヒチン、インドメタシン）の作用機序を問う。[3][9]  
- **選択肢1**: 正しい。**インドメタシン**（問246の選択肢5）は非ステロイド性抗炎症薬（NSAIDs）で、シクロオキシゲナーゼ（COX）を阻害し、**プロスタグランジンE₂（PGE₂）の産生を抑制**する。[1][3]  
- **選択肢2**: 誤り。遺伝子組換え型尿酸オキシダーゼ（**ラスブリカーゼ**）の作用機序。[4]  
- **選択肢3**: 誤り。**アロプリノール**の代謝物オキシプリノール（アロキサンチン）がキサンチンオキシダーゼを阻害する作用（本問の処方薬ではない）。[1][4]  
- **選択肢4**: 正しい。**フェブキソスタット**（処方1）はキサンチンオキシダーゼ阻害薬で、尿酸産生を抑制（詳細機序は選択肢4に該当）。[1][7]  
患者は急性痛風発作（アルコール摂取による尿酸上昇が誘因）を起こしており、問246の適切処置はインドメタシン追加（NSAIDsで炎症抑制）。[3][4]
