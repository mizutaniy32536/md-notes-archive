# 第102回 問53

**問題文**
粉末X線回折測定法により評価される医薬品の物性はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-53 を参照

**選択肢**
1. 比表面積
2. 空隙率
3. 結晶性
4. 吸湿性
5. 旋光度

**正解:** 3

**解説**
解答　３

粉末X線回折法測定法により得られる粉末X線回折データから結晶、結晶多形及び溶媒和結晶の同定、判定又は定量、結晶性の定性評価、結晶角などの評価に用いることができる。
本測定法は、原則として無配向化した粉末試料にX線を照射し、その物質中の電子を強制振動させることにより生じる干渉性散乱X線による回折強度を各回折角について測定する方法である。
