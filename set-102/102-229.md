# 第102回薬剤師国家試験 問229

## 問題文

脂肪乳剤輸液に用いられる脂質に関する記述のうち、正しいのはどれか。**2つ選べ**。[1][2]

## 選択肢

1. 脂肪乳剤中の油脂には必須脂肪酸のリノール酸及びα-リノレン酸が含まれている。[1]

2. 中鎖脂肪酸は長鎖脂肪酸に比べてエネルギーに変換されにくい。[1]

3. 脂肪乳剤中の脂質1 gあたりのエネルギー量は約9 kcalである。[2]

4. 脂肪乳剤中のトリアシルグリセロールは、リポタンパク質リパーゼによりモノアシルグリセロールと脂肪酸に分解され組織に吸収される。[2]

5. 脂肪乳剤には乳化剤としてコレステロールが含まれている。[1]

## 正解番号

**1、3**[1][3]

## 解説

**選択肢1：正しい**[1]
必須脂肪酸の補給が脂肪乳剤の重要な意義の1つです。リノール酸及びα-リノレン酸は脂肪乳剤中に含まれています。

**選択肢2：誤り**[1]
中鎖脂肪酸は、一般的な脂肪に比べエネルギー変換効率が高いとされています。つまり、エネルギーに変換されやすいため、選択肢の記述は誤りです。

**選択肢3：正しい**[2][3]
脂肪のエネルギー量は1 gあたり約9 kcalであり、脂肪乳剤中の脂質もこれに該当します。

**選択肢4：誤り**[1]
トリアシルグリセロールはリポタンパク質リパーゼにより**モノアシルグリセロール**と**脂肪酸**に分解されると考えられます。選択肢の記述は正しいですが、これは正解に含まれていません。

**選択肢5：誤り**[1]
乳化剤として含まれるのは**レシチン**です。コレステロールではありません。
