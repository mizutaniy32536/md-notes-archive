# 第102回 問323

### 第102回薬剤師国家試験 問323

**問題文（全文）**  
68歳男性。前立腺がん患者を対象とした治験に参加している。当該患者から担当のCRC（治験コーディネータ）である薬剤師に電話があり「今朝から、お腹のあたりに真っ赤な湿疹ができている。」と連絡があった。湿疹は、この治験薬の代表的な副作用の1つで、参加同意を得る際にも説明していた。[2][5]

**問323（実務）**  
CRCの対応として**誤っている**のはどれか。**２つ**選べ。[2][5]

1. 男性から聴取した情報を記録する。[2][5]  
2. 男性が受診すべきかどうか治験責任医師に相談する。[2][5]  
3. 重篤な有害事象と判断した場合、直ちに実施医療機関の長に報告する。[2][5]  
4. 有害事象及びその治療が、逸脱や中止基準に該当するかどうか治験責任医師と確認する。[2][5]  
5. 治験薬による有害事象と判断された場合、副作用被害救済制度の利用について、男性に情報提供する。[2][5]

**正解番号**  
3、5[4]

**解説**  
CRC（治験コーディネータ）の主な役割は、治験責任医師の指示のもとで被験者からの情報を収集・記録し、医師に報告することである。選択肢1は情報記録として正しい。選択肢2は受診判断を医師に相談する正しい対応。選択肢4は逸脱・中止基準の確認も医師との連携として正しい。一方、選択肢3は**誤り**で、重篤な有害事象（SAE）の報告は治験責任医師を経由してスポンサーや規制当局へ行うもので、直接「実施医療機関の長」に直ちに報告する義務はない。選択肢5も**誤り**で、副作用被害救済制度（医薬品副作用被害救済制度）は市販後承認医薬品に対するもので、治験中の有害事象には適用されないため、情報提供は不適切。[2][5]
