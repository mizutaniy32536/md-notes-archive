# 第102回 問81

**問題文**
「患者が積極的に治療方針の決定に参加し、その決定に従って治療を受けること」を意味するのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-81 を参照

**選択肢**
1. アドヒアランス
2. インフォームド・コンセント
3. カウンセリング
4. クリニカルパス
5. コンプライアンス

**正解:** 1

**解説**
解答　１

１　正
アドヒアランスとは、「患者が積極的に治療方針に参加し治療を受けること」を意味する。

２　誤
インフォームド・コンセントとは、「医療行為（投薬・手術・検査など）や治癒などの対象者（正常人、患者）が、治療や臨床試験・治験の内容についてよく説明を受け十分理解した上で、対象者が自らの自由意思に基づいて医療従事者と方針において合意すること」を意味する。

３　誤
カウンセリングとは、「生活や人間関係において心理的ストレスを溜め込んだ人に対して専門的な知識を得た人が心理学的な資料や経験に基づいて援助すること」を意味する。

４　誤
クリニカルパスとは、「一定の疾患や疾病を持つ患者に対して、入院指導、患者へのオリエンテーション、ケア処置、検査項目、退院指導などを時間軸に沿ってまとめたスケジュール表のこと」を意味する。

５　誤
コンプライアンス（服薬遵守）とは、「患者が服薬指導の内容を理解し服薬指示を守ること」を意味する。
