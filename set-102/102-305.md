# 第102回薬剤師国家試験 問305

## 問題文

45歳男性。1週間前に心筋梗塞の診断により経皮的冠動脈インターベンション（PCI）を施行し、ステントを留置した。梗塞部位は良好に拡張されたが、施行5日後と6日後にステント血栓症が発症した。

その後、カンファレンスにおいて、ステント血栓症の原因が検討され、薬剤師に意見が求められた。薬剤師は服用中のクロピドグレル治療抵抗性の可能性を提起し、代替薬を提案した。

**クロピドグレルの代替薬として、以下の薬剤のうち最も適切なのはどれか。1つ選べ。**

## 選択肢

1. トラネキサム酸
2. プロタミン
3. プラスグレル
4. ワルファリン
5. メナテトレノン

## 正解番号

**3**

## 解説

クロピドグレル（プラビックス）は**抗血小板薬**です。CYP2C19による代謝を受けて活性代謝物になり、血小板表面のP2Y12受容体に作用します。CYP2C19の多型により治療抵抗性が見られることがあります[1]。

本症例では、クロピドグレルの治療抵抗性によるステント血栓症が疑われたため、代替の抗血小板薬が必要です。

選択肢の検討：
- **トラネキサム酸**：止血薬で、抗血小板薬ではない
- **プロタミン**：ヘパリンの拮抗薬で、抗血小板薬ではない
- **プラスグレル**：クロピドグレルと同じP2Y12受容体拮抗薬であり、CYP3A4とCYP2B6により代謝される抗血小板薬。クロピドグレルの代替薬として適切
- **ワルファリン**：抗凝固薬であり、抗血小板薬ではない
- **メナテトレノン**：ビタミンK2製剤で、抗血小板薬ではない

プラスグレルはクロピドグレルと異なる代謝経路を持つため、CYP2C19多型による治療抵抗性の影響を受けにくく、クロピドグレル治療抵抗性の患者における代替薬として最も適切です[1]。
