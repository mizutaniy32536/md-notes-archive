# 第102回 問301

### 第102回薬剤師国家試験 問301

**問題文（全文）**  
HBVが原因のB型肝炎に関する記述として、正しいのはどれか。１つ選べ。[3][8]

**全選択肢**  
1. HBワクチンは感染予防には有効ではない。[1]  
2. 性行為による感染はない。[1]  
3. IgM型HBc抗体は肝炎の後期に現れる。[1]  
4. 初感染で自然治癒するのは半数である。[1]  
5. R-CHOP療法施行中にHBV再活性化が起こりうる。[1][8]

**正解番号**  
5[1][4]

**解説**  
本問は問300の症例（62歳女性、悪性リンパ腫でR-CHOP療法施行後、ALT/AST上昇）を踏まえ、B型肝炎の特徴を問う。10年前の献血時のHBc抗体陽性からB型肝炎キャリアとみられ、R-CHOP療法による免疫抑制でHBV再活性化が起こり、急性肝機能障害（ALT 742 U/L、AST 1,354 U/L）を引き起こした。[1][3]  

- **1誤り**: HBワクチンはB型肝炎ウイルス感染予防に有効。[1]  
- **2誤り**: B型肝炎は血液・体液を介した水平感染（性行為を含む）で感染。[1]  
- **3誤り**: HBc抗体（IgM型）は初感染の**早期**に陽性化（順序: HBc → HBe → HBs抗原）。後期ではない。[1]  
- **4誤り**: 初感染後の自然治癒率は**7～80%**（半数ではない）。[1]  
- **5正しい**: 免疫抑制療法（R-CHOP）でHBVキャリアの再活性化が知られ、肝機能障害の原因となる。[1][3][8]  

正解は5。[1][4]
