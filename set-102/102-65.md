# 第102回 問65

**問題文**
ウイルス性肝炎に関する記述のうち、最も適切なのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-65 を参照

**選択肢**
1. A型は多くが慢性化しやすい。
2. B型は経口感染である。
3. C型は血液感染である。
4. B型はRNAウイルスが原因である。
5. C型はDNAウイルスが原因である。

**正解:** 3

**解説**
解答　３

１　誤
A型肝炎ウイルスは、そのほとんどが急性肝炎の状態で治癒されるため、慢性化しにくい。

２　誤
B型肝炎ウイルスの主な感染経路は、非経口感染（性行為や母子感染、血液感染など）である。

３　正
C型肝炎ウイルスの主な感染経路は、主に血液感染である。

４　誤
B型肝炎ウイルスはDNAウイルスである。なお、A型肝炎ウイルスやC型肝炎ウイルスはRNAウイルスである。

５　誤
