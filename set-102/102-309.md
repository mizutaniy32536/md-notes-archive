# 第102回薬剤師国家試験 問309

## 問題文

73歳女性。感冒様症状を訴え、薬局を訪れた。薬剤師は、薬局製造販売医薬品のかぜ薬1-②を販売することにした。

この薬局で販売する薬局製造販売医薬品に関する記述のうち、正しいのはどれか。**2つ選べ。**[1][2]

## 選択肢

1. 薬局医薬品である。
2. 製造販売業の許可は当該薬局の所在地を管轄する地方厚生局長が与える。
3. 当該薬局の設備及び器具で製造する。
4. 製造業の許可は必要としない。
5. 店舗販売業で販売することができる。[1][2]

## 正解番号

**1、3**[1][2]

## 解説

**選択肢1：正**

薬局医薬品とは、要指導医薬品及び一般用医薬品以外の医薬品であり、薬局製造販売医薬品は薬局医薬品に該当します。[2]

**選択肢2：誤**

製造販売業の許可については、適切な記述ではありません。[2]

**選択肢3：正**

薬局製造販売医薬品は、当該薬局の設備及び器具で製造されるものです。[1][2]

**選択肢4：誤**

製造業の許可の取扱いについて、適切な記述ではありません。[2]

**選択肢5：誤**

薬局製造販売医薬品は薬局でのみ販売でき、店舗販売業者での販売はできません。[2]
