# 第102回 問302

**問題文**
病棟の看護師より、「点滴中の患者に対し側管からブロムへキシン塩酸塩注射剤を投与後、同一の側管より続けてフロセミド注射液を投与してもよいか。」との問い合わせがあった。薬剤師は看護師に回答するため、両薬剤のpH変動スケールに関する情報を収集し、以下の情報を得た。問302（実務）両薬剤のpH変動スケール及び配合変化に関する記述のうち、適切なのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-302 を参照

**選択肢**
1. Aはブロムへキシン塩酸塩、Bはフロセミドである。
2. AとBを比較すると、緩衝性の強いのはBである。
3. 両薬剤が輸液ライン内で混合された場合、混合液のpHは4.7以上6.3以下となる。
4. フロセミド注射剤を先に投与し、続けてブロムへキシン塩酸塩注射剤を投与すれば白濁は生じない。
5. 両薬剤が輸液ライン内で混合されて白濁を生じる可能性が高いので、それぞれ投与前後に生理食塩液等を流す。問303（病態・薬物治療）配合変化について検討するために、ブロムへキシン塩酸塩注射剤の特徴について調査したい。医薬品インタビューフォームから入手できない情報はどれか。1つ選べ。
6. 製剤の安定性に関するデータ
7. 有効成分の安定性に関するデータ
8. アンプル開封後の使用期限
9. 浸透圧比
10. 注射液のpH

**正解:** 1,5

**解説**
問302　解答　１、５

pH変動スケールとは、pH変動試験の結果によって作成されるものであり、pH変動により溶解度が変化することで生じる結晶析出や混濁などの配合変化の予測に用いられる。pH変動試験では、注射剤に0.1 mol/L HClまたは、0.1 mol/L NaOHを加えてpHを変化させ、沈殿や混濁などの外観変化を観察するものである。0.1 mol/L HClまたは、0.1 mol/L NaOHを添加して変化が生じた時点のpHを変化点pHとする。また、注射剤に0.1 mol/L HClまたは、0.1 mol/L NaOHを添加しても何も変化のない場合は、10 mL添加を最大量とし、10 mL添加しても外観変化が現れなかった場合は、その時点のpHを最終pHとする。
本設問で問われているA注射剤の試料pHは2.8であるため、酸性注射剤のブロムヘキシン塩酸塩であることがわかり、0.1 mol/L NaOHを0.2 mL加えるとpH4.7で白濁が起きているため、これが変化点pH、0.1 mol/L HClを10 mL加えて変化のないpHは1.3のため、これが最終pHと読みとれる。同様に、B注射剤で考えた場合、試料pH9.4より塩基性注射剤のフロセミド注射剤であることがわかり、0.1 mol/L HClを0.1 mL加えるとpH6.3で白濁が起きているため、これが変化点pH、0.1 mol/L NaOHを10 mL加えて変化のないpHは12.7のため、これが最終pHと読みとれる。

１　正
前記参照

２　誤
緩衝性を判断する際の指標として、pH移動指数がある。pH移動指数とは、変化点pHと試料pHとの差の絶対値又は最終pHと試料pHとの差の絶対値であらわし、pH移動指数の和が大きいほどその注射剤の緩衝性は弱く、逆に和が小さいほど緩衝性は強くなる。AのpH移動指数の和（3.4）は、BのpH移動指数の和（6.4）より小さいため、AのほうがBに比べ、緩衝性が強い注射剤である。

３　誤
試料pH2.8のA注射剤と、試料pH9.4のB注射剤を混合した場合、pHは2.8以上9.4以下となる。また、A注射剤は、pH4.7以上で白濁し、B注射剤は、pH6.3以下で白濁することから、AとBが輸液ライン内で混合された場合、白濁が生じる。

４　誤
解説３参照
フロセミド注射剤を先に投与し、続けてブロムヘキシン塩酸塩注射剤を投与した場合、輸液ライン内に残っているフロセミド注射剤とブロムヘキシン塩酸塩注射剤が混合され、白濁が生じる。

５　正
解説４参照
両薬剤が輸液ライン内で混合すると白濁が生じるため、それぞれの投与前後に生理食塩液等を流すことで輸液ライン内に残っている注射液を洗い流すことができ、両薬剤の混合を防止できる。

問303　解答　３

医薬品インタビューフォームは、医療用医薬品添付文書等の情報を補完するもので、医療用医薬品に関する様々な情報（製剤の安定性、有効成分の安定性、浸透圧比、注射液のpH、有効期間又は使用期限など）が記載されている。医薬品インタビューフォームは、「有効期間又は使用期限」の項目より、アンプル開封前の使用期限に関する情報は入手できるが、アンプル開封後の使用期限については入手できない。
