# 第102回 問328

**問題文**
注射剤の調製及び使用に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-328 を参照

**選択肢**
1. 注射針の太さは、針管のゲージ番号が小さいほど細い。
2. 輸液セットは、20滴/mLと60滴/mLの2種類の規格がある。
3. カットポイント付きのアンプルをカットする際には、カットポイントを手前に向ける。
4. 孔径0.2 µmの輸液フィルターは、ウイルスの除去を目的としている。
5. 輸液容器のゴム栓を穿刺する際には、消毒用アルコール綿でのゴム栓の清拭は不要である。

**正解:** 2,3

**解説**
解答　２、３

１　誤
注射針の太さは針管の外径のゲージ番号で規格が決められており、ゲージ番号が小さいほど注射針は太い。

２　正
輸液セットは、20滴/mLの成人用輸液セット、60滴/mLの小児用輸液セットがあり、2種類の規格に統一されている。

３　正
カットポイント付きのアンプルをカットする際には、カットポイントを手前に向け、カットポイントとは反対方向に折り取る。

４　誤
輸液の調製時に微生物（細菌、真菌、ウイルスなど）や微小細物（ガラス片、ゴム片など）が混入し、体内に入ると静脈炎や敗血症などの原因となる。そのため、微生物や微小細物の混入を防ぐ目的で輸液フィルターが用いられる。細菌や真菌、微小細物などは孔径0.2 µmの輸液フィルターにより除去できるが、ウイルスは孔径0.2 µmの輸液フィルターを通過するため、除去することができない。よって、孔径0.2 µmの輸液フィルターは、ウイルスの除去を目的とはしていない。

５　誤
輸液容器のゴム栓を穿刺する際には、穿刺部位からの細菌による汚染を防ぐために、消毒用アルコール綿でのゴム栓の清拭が必要である。
