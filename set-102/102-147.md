# 第102回 問147

### 第102回薬剤師国家試験 問147

**問題文**  
我が国の薬価基準制度に関する記述のうち、正しいのはどれか。**2つ選べ**。[1][2][3]

**選択肢**  
1. 薬価とは、国により決定される医薬品の公定価格である。[1][2][3]  
2. 薬価基準は、医療法に基づく厚生労働大臣告示として公表される。[1][2][3]  
3. 新医薬品の薬価算定は、原価計算方式を原則とする。[1][2][3]  
4. 医療用医薬品であっても、薬価基準に収載されていないものがある。[1][2][3]  
5. 薬価改定は、5年ごとに行うよう定められている。[1][2][3]

**正解番号**  
1, 4[1][2]

**解説**  
- **1 正**: 薬価は診療報酬および調剤報酬における薬剤料の算定の基礎となる価格であり、国により決定される医薬品の公定価格である。[2]  
- **2 誤**: 薬価基準は、健康保険法及び療養費担当者指定規則・薬剤給付基準規則に基づき厚生労働省告示として公表される。医療法に基づくものではない。[1]  
- **3 誤**: 新医薬品の薬価算定方式は、原則として類似薬効比較方式（比較対照薬を選定できる場合）。原価計算方式は比較対照薬を選定できない場合の例外的な方法である。[1][2]  
- **4 正**: 薬価基準には保険診療・保険調剤に使用する医薬品が収載されるため、保険対象外の医療用医薬品（例: バイアグラ、プロペシアなど）は収載されていない。[1][2]  
- **5 誤**: 薬価改定は原則として2年ごとに行われる（緊急改定の場合を除く）。[1]
