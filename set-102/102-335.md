# 第102回 問335

**問題文**
2歳男児。体重13 kg。以下の薬剤が処方され、カプセル剤を開封して調剤するよう処方医より指示があった。1包あたり全量0.3 gになるように賦形剤を加えて調剤する場合、この処方を全量調剤するために加える賦形剤の総量（g）として正しいのはどれか。1つ選べ。なお、ダントロレンナトリウム水和物カプセル1カプセルあたりの内容量は0.25 gである。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-335 を参照

**選択肢**
1. 1.4
2. 2.8
3. 4.2
4. 5.6
5. 7.0

**正解:** 5

**解説**
解答　５

まず、ダントロレンナトリウム水和物と賦形剤の総量を求める。1包あたり全量0.3 gになるように賦形剤を加えて調剤するため、14日間での総量は以下のように計算される。
0.3 g/回×2回/日×14日＝8.4 g
ダントロレンナトリウム水和物カプセル（0.25 g/カプセル）の総量は、1日あたり0.4カプセルより、以下のように計算される。
0.25 g/カプセル×0.4カプセル/日×14日＝1.4 g
よって、賦形剤の量は以下のように計算される。
賦形剤の量＝8.4 g（全量）－1.4 g（ダントロレンナトリウム水和物の総量）＝7 g
