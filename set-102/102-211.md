# 第102回 問211

### 第102回薬剤師国家試験 問211

**問題文**  
1歳男児。耳鼻科を受診し中耳炎と診断され、以下の薬剤が処方された。母親が処方箋を持参し、薬局を訪れた。  
（構造式：セフジトレンピボキシルが示されている）[1][2][4][5]  

**セフジトレンピボキシルに関する記述のうち、正しいのはどれか。2つ選べ。**  

1. ペネム骨格を有している。[1][2][3][5]  
2. β－ラクタム環は共鳴が起こりやすく、カルボニル基の電子求引性が高い。[2][3]  
3. 細菌中のペプチドグリカン合成酵素との間で、β－ラクタム環の開環を伴って共有結合を形成することにより、細胞壁の生合成を阻害する。[2][3][4]  
4. セフジトレンのカルボキシ基を構造修飾することにより、経口吸収性が改善されたプロドラッグである。[3][4]  
5. 加水分解により酢酸を生じる。[3][5]  

**正解番号：3, 4** [1][2][3][4]  

**解説**  
- **選択肢1 誤**：本剤はペネム骨格（右側が5員環）ではなく、セフェム骨格（右側が6員環）を有する。[1][2][3][5]  
- **選択肢2 誤**：β－ラクタム環は角度ひずみが大きく、共鳴が起こりづらく、カルボニル基の電子求引性が高い。[2][3]  
- **選択肢3 正**：β－ラクタム系抗生物質の作用機序は、β－ラクタム環の開環反応により細菌のペプチドグリカン合成酵素（トランスペプチダーゼ/PBP）のセリン残基に共有結合し、細胞壁合成を阻害する。[2][3][4]  
- **選択肢4 正**：ピボキシル基をセフジトレンのカルボキシ基に付加したプロドラッグで、小腸吸収率を向上させている。生体内で加水分解され活性体セフジトレンとなる。[1][3][4]  
- **選択肢5 誤**：加水分解によりピバル酸とホルムアルデヒドが生じるが、酢酸は生成しない。[3][5]  

※補足：ピボキシル基由来のピバル酸により、まれに低カルニチン血症→低血糖が報告されているが、国試では不要。[1]
