# 第102回 問79

**問題文**
リスボン宣言に関する記述のうち、正しいのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-79 を参照

**選択肢**
1. 人を対象とする医学系研究の倫理的原則である。
2. 日本では法的拘束力がある。
3. 患者が他の医師の意見を求める権利が示されている。
4. 研究倫理委員会の役割が示されている。
5. 医療従事者の権利が示されている。

**正解:** 3

**解説**
解答　３

１　誤
ヘルシンキ宣言のことである。ヘルシンキ宣言をもとに、世界各国で行われる臨床試験に関する基準が定められている。 

２　誤
リスボン宣言は倫理規範であり、法的拘束力はない。

３　正

４　誤
研究倫理委員会（倫理審査委員会）の役割は、研究倫理が遵守されているかを監視することであり、ヘルシンキ宣言で規定されている。

５　誤
リスボン宣言は、患者の権利を規定したものであり、医療従事者の権利は示されていない。
