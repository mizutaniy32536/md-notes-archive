# 第102回薬剤師国家試験 問253

## 問題文

20歳女性。統合失調症と診断され、3ヶ月間、薬物治療が継続されていた。副作用は特にみられなかったが症状が改善されないため、主治医はクロルプロマジン塩酸塩錠を1日300mgから450mgへ増量した。

この処方が引き起こす可能性がある副作用について、その発現機序はどれか。2つ選べ。[1][2]

## 全選択肢

1. アンギオテンシンⅡAT1受容体遮断により歯肉肥厚が起こる。

2. **アセチルコリンM3受容体遮断により眼圧亢進が起こる。**

3. **ヒスタミンH1受容体遮断により眠気が起こる。**

4. アドレナリンα1受容体遮断により錐体外路障害が起こる。

5. ドパミンD2受容体遮断により起立性低血圧が起こる。

## 正解番号

**2, 3**[1]

## 解説

クロルプロマジン塩酸塩は第一世代の定型抗精神病薬であり、複数の受容体に対する遮断作用を持つため、様々な副作用が生じる可能性があります。[1]

**選択肢2（正答）：** アセチルコリンM3受容体遮断により眼圧亢進が起こるという記述は正しいです。クロルプロマジンのムスカリン受容体遮断作用により、瞳孔散大と眼圧上昇が起こります。[1]

**選択肢3（正答）：** ヒスタミンH1受容体遮断により眠気が起こるという記述は正しいです。クロルプロマジンはH1受容体遮断作用を有し、鎮静作用による眠気が副作用として現れます。[1]

**選択肢1（不正答）：** アンギオテンシンⅡAT1受容体遮断によるカルシウム拮抗薬様の作用は、クロルプロマジンの薬理作用ではありません。[1]

**選択肢4（不正答）：** 受容体の記述が逆です。ドパミンD2受容体遮断により錐体外路症状が起こり、アドレナリンα1受容体遮断により起立性低血圧が起こります。[1]

**選択肢5（不正答）：** ドパミンD2受容体遮断による起立性低血圧という記述は誤りです。受容体の作用が反転しています。[1]
