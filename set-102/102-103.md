# 第102回 問103

**問題文**
次の反応で得られる化合物中のH（a〜c）に関する記述のうち、正しいのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-103 を参照

**選択肢**
1. a、b、cのすべてがLiAlH4由来である。
2. aとbがH3O＋由来、cがLiAlH4由来である。
3. aとbがLiAlH4由来、cがH3O＋由来である。
4. aとbのどちらか一方がH3O＋由来、もう一方とcがLiAlH4由来である。
5. aとbのどちらか一方がLiAlH4由来、もう一方とcがH3O＋由来である。

**正解:** 3

**解説**
解答　３

カルボン酸をLiAlH4を用いて還元すると、アルデヒドを経由して第一級アルコールまで還元される。
カルボン酸にLiAlH4が反応する際、まずはLiAlH4のH原子がヒドリド（H−）となり、ヒドリドが求核剤としてカルボン酸のカルボニル炭素（正電荷を帯びている炭素）を攻撃することで、アルデヒドを生成する。

その後、アルデヒドで反応は止まらず、ヒドリドが求核剤としてアルデヒドのカルボニル炭素を攻撃し、その後H3O＋由来のH＋が結合することで、第一級アルコールが生成する。

以上より、aとbがLiAlH4由来、cがH3O+由来である。よって、選択肢３が正しい。
