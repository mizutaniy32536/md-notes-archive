# 第102回 問171

**問題文**
ある薬物を含む散剤（薬物100 mg/g）を繰り返し経口投与し、定常状態における平均血中濃度を2.0 µg/mLとしたい。投与間隔を8時間とするとき、1回あたりの散剤の投与量（g）として最も適切なのはどれか。1つ選べ。ただし、この薬物の体内動態は線形1−コンパートメントモデルに従い、全身クリアランスは120 mL/min、この散剤における薬物のバイオアベイラビリティは80%とする。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-171 を参照

**選択肢**
1. 0.18
2. 0.36
3. 0.92
4. 1.2
5. 1.4

**正解:** 5

**解説**
解答　５

薬物を一定の投与間隔で繰り返し投与した場合、定常状態の平均血中薬物濃度C ss.av  は①式で表すことができる。投与量を算出するために、①式を②式に変形する。

本文中に「薬物100 mg/gを投与する」とあることから、1回あたり薬物144 mg投与するための散剤の投与量は1.44 gとなる。
