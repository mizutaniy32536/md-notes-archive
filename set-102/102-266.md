# 第102回 問266

### 第102回薬剤師国家試験 問266

#### 問題文
58歳男性。手術不能の直腸がんと診断され、以下に示すレジメン（FOLFIRI＋ベバシズマブ療法）に従った化学療法を施行することとなった。  
患者の検査値を確認したところ、**血中間接ビリルビン値が2.8mg/dL**と高値を示すが、直接ビリルビン値は正常範囲内であった。患者と面談したところ、以前他院にて**体質性黄疸**と診断されたが、特に治療は行っていないことが判明した。この患者の治療上、注意すべき内容として最も適切なのはどれか。1つ選びなさい。[1][2]

#### 選択肢
1. ベバシズマブの**薬効減弱**  
2. ベバシズマブの**副作用発現**  
3. イリノテカンの**薬効減弱**  
4. イリノテカンの**副作用発現**  
5. フルオロウラシルの**薬効減弱**  
6. フルオロウラシルの**副作用発現**[1][2]

#### 正解番号
**5**（イリノテカンの副作用発現）[1][2][3]

#### 解説
本問のレジメンは**FOLFIRI（フルオロウラシル＋レボホリナート＋イリノテカン）＋ベバシズマブ**療法。患者は**体質性黄疸（Gilbert症候群）**による**間接ビリルビン高値**を示す。この状態では**イリノテカンの活性代謝物（SN-38）の排泄が低下**し、**SN-38の血中濃度上昇**により**重篤な副作用（下痢など）発現リスク**が増大するため、**減量または中止**を考慮する（選択肢4）。[1][2][5]  
- 選択肢1,3,5：薬効減弱のエビデンスなし。  
- 選択肢2,6：ベバシズマブやフルオロウラシルはビリルビン値に影響されず、副作用発現リスク増大なし。  
**UGT1A1遺伝子多型**（Gilbert症候群関連）によるイリノテカン毒性増強がポイント。[1][5]
