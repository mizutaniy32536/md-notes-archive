# 第102回 問259

### 第102回薬剤師国家試験 問259

**問題文（問259 薬理）**  
36歳女性。乳がん手術後、以下の薬物療法が開始された。  
この処方の服薬指導として、[A]は、子宮体がんのリスクを上げるため、定期的な検査を行うよう患者に指導した。また、[B]は、ほてりやのぼせ、抑うつなどの更年期症状がみられることがあると患者に説明した。  
**[A]の薬物の作用機序として正しいのはどれか。１つ選べ。**[1][3]

**全選択肢（番号付き）**  
1. 乳腺のエストロゲン受容体（ER）を遮断する。[3]  
2. ヒト上皮増殖因子受容体2型（HER2）を遮断する。[3]  
3. アロマターゼを阻害する。[3]  
4. 活性酸素を発生させ、DNA鎖を切断する。[3]  
5. ゴナドトロピン放出ホルモン（GnRH）受容体を刺激する。[3]

**正解番号**  
**1**[1][3][8]

**解説**  
問258で[A]は**タモキシフェン**と特定されており、本剤は選択的エストロゲン受容体モジュレーター（SERM）として乳腺のエストロゲン受容体（ER）を遮断し、乳がん治療に用いられる。[1][3]  
他の選択肢は誤り：2はトラスツズマブの作用、3はレトロゾールの作用、4はテガフールの作用、5はリュープロレリンの作用機序（ただしリュープロレリンはGnRHアナログとして連続投与で受容体を脱感作させるため刺激ではない）。[1][3][8]
