# 第102回 問258

**問題文**
36歳女性。乳がん手術後、以下の薬物療法が開始された。この処方の服薬指導として、　Ａ　は、子宮体がんのリスクを上げるため、定期的な検査を行うよう患者に指導した。また、　Ｂ　は、ほてりやのぼせ、抑うつなどの更年期症状がみられることがあると患者に説明した。問258（実務）　Ａ　と　Ｂ　に当てはまる薬剤はどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-258 を参照

**選択肢**
1. リュープロレリン酢酸塩
2. レトロゾール
3. トラスツズマブ
4. タモキシフェン
5. テガフール問259（薬理）　Ａ　の薬物の作用機序として正しいのはどれか。1つ選べ。
6. 乳腺のエストロゲン受容体（ER）を遮断する。
7. ヒト上皮増殖因子受容体2型（HER2）を遮断する。
8. アロマターゼを阻害する。
9. 活性酸素を発生させ、DNA鎖を切断する。
10. ゴナドトロピン放出ホルモン（GnRH）受容体を刺激する。

**正解:** 1,4

**解説**
問258　解答　１、４

 　Ａ　 は、子宮体がんのリスクを上げるため、定期的な検査を行うよう患者に指導したことから選択肢の薬剤の中ではタモキシフェンが当てはまる。また、 　Ｂ　 は、ほてりやのぼせ、抑うつなどの更年期症状がみられることがあるという記述から選択肢の薬剤の中ではリュープロレリン酢酸塩が当てはまる。

問259　解答　１
タモキシフェンは、抗エストロゲン薬であり、乳腺のエストロゲン受容体（ER）を遮断することで乳がん細胞の増殖を抑制する。
