# 第102回 問265

### 第102回薬剤師国家試験 問265

**問題文（全文）**  
58歳男性。手術不能の直腸がんと診断され、以下に示すレジメン（FOLFIRI＋ベバシズマブ療法）に従った化学療法を施行することとなった。  
薬剤師による初回面談の際に説明すべき処方薬の副作用として適切なのはどれか。**2つ選べ。**[1][2][4]

**全選択肢（番号付き）**  
1. **高血圧**  
2. **ざ瘡様皮疹**  
3. **下痢**  
4. **認知機能障害**  
5. **高血糖**[1][2][4]

**正解番号**  
1, 3[1][2][4]

**解説**  
本問のレジメンはFOLFIRI（フルオロウラシル＋レボホリナート＋イリノテカン）＋ベバシズマブ療法である。[1][2][4]  
- 選択肢1 **高血圧**：ベバシズマブの主な副作用として血管内皮増殖因子（VEGF）阻害による高血圧が挙げられ、初回面談で説明すべき重篤な副作用。[1][2][4]  
- 選択肢2 **ざ瘡様皮疹**：EGFR阻害薬（例：セツキシマブ）の特徴的な副作用だが、本レジメンには含まれないため不適切。[1][4]  
- 選択肢3 **下痢**：イリノテカンの主な副作用として重篤な下痢が発生しやすく、初回面談で説明必須。[1][2][4]  
- 選択肢4 **認知機能障害**：本レジメンの薬剤で関連が薄く、主要副作用ではない。[1][4]  
- 選択肢5 **高血糖**：ステロイド系薬剤の副作用だが、本レジメンでは該当せず。[1][4]  
以上より、初回面談で特に注意すべきはベバシズマブの高血圧とイリノテカンの下痢。[1][2][4]
