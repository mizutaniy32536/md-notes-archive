# 第102回 問267

### 第102回薬剤師国家試験 問267

**問題文**  
前問の遺伝子診断の結果、酵素活性の低下を伴う遺伝子型であることが判明した。この患者の治療上、注意すべき内容として最も適切なのはどれか。１つ選べ。[1][2]

**全選択肢**  
1. ベバシズマブの薬効減弱[1][2]  
2. ベバシズマブの副作用発現[1][2]  
3. イリノテカンの薬効減弱[1][2]  
4. **イリノテカンの副作用発現**[1][2]  
5. フルオロウラシルの薬効減弱[1][2]

**正解番号**  
4[1][2][5]

**解説**  
本問は問266の続きで、患者は**体質性黄疸（Gilbert症候群）**に加え、**イリノテカン代謝酵素UGT1A1の活性低下遺伝子型（*6/*6など）**が判明したケース。イリノテカンは肝臓のUGT1A1で活性型SN-38がグルクロン酸抱合されて排泄されるが、酵素活性低下時はSN-38が蓄積し、重篤な**副作用（下痢、好中球減少症など）**の発現リスクが高まるため、**減量や投与中止**が必要。したがって、**イリノテカンの副作用発現**に注意すべき。[1][2]  
他の選択肢は誤り：ベバシズマブやフルオロウラシルはUGT1A1依存せず、薬効減弱の関連なし。[1][2]
