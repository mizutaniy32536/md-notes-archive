# 第102回 問83

**問題文**
薬袋に記載する事項として必須なのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-83 を参照

**選択肢**
1. 処方医名
2. 処方箋発行日
3. 薬品名
4. 使用上の注意
5. 調剤年月日

**正解:** 5

**解説**
解答　５

薬袋に記載する事項は薬剤師法第25条同施行規則第14条により①処方せんに記載された患者の氏名、②用法及び用量、③調剤年月日、④調剤した薬剤師の氏名⑤調剤した薬局などの名称及び所在地となっている。
