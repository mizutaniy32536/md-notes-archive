# 第102回 問73

**問題文**
厚生労働大臣が、薬剤師の免許の取消し等の処分をするにあたって、あらかじめ意見を聴かなければならないのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-73 を参照

**選択肢**
1. 医道審議会
2. 都道府県知事
3. 内閣府
4. 薬事・食品衛生審議会
5. 裁判所

**正解:** 1

**解説**
解答　１

厚生労働大臣が、薬剤師の免許の取り消し等の処分をするにあたって、あらかじめ意見を聴かなければならないのは、医道審議会である。
医道審議会は、厚生労働省に設置された厚生労働大臣の諮問機関であり、医師・歯科医師等の行政処分等に関する事項、薬剤師の免許取消し等の行政処分及び薬剤師国家試験試験に関する事項等を審議する。
