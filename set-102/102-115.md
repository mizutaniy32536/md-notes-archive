# 第102回 問115

**問題文**
ピリミジンヌクレオチドの代謝に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-115 を参照

**選択肢**
1. ピリミジン骨格の生合成には、グルタミンとアスパラギン酸が利用される。
2. ピリミジンヌクレオチドの生合成は、最終産物のシチジン5’−三リン酸（CTP）によりフィードバック阻害される。
3. デオキシウリジン5’−一リン酸（dUMP）からデオキシチミジン5’−一リン酸（dTMP）の生合成において、S−アデノシルメチオニンがメチル基供与体として働く。
4. ピリミジンヌクレオチドの分解により、尿酸が生成される。

**正解:** 1,2

**解説**
解答　１、２

１　正
ピリミジンヌクレオチド生合成では、グルタミンと二酸化炭素から生成したカルバモイルリン酸とアスパラギン酸を原料としてカルバモイルアスパラギン酸が生成され、その後ピリミジン骨格を有するオロト酸が生成する。オロト酸はオロチジン5’−一リン酸（OMP）に変換され、ウリジン5’−一リン酸（UMP）になり、ウリジン5’−三リン酸（UTP）を経由してシチジン5’−三リン酸（CTP）となる。最終産物のCTPは、カルバモイルアスパラギン酸を生成する反応を触媒する酵素（アスパラギン酸カルバモイルトランスフェラーゼ）を抑制することでピリミジンヌクレオチドの生合成をフィードバック阻害する。

２　正
解説１参照

３　誤
デオキシウリジン5’−一リン酸（dUMP）からデオキシチミジン5’−一リン酸（dTMP）の生合成は、チミジル酸シンターゼにより触媒され、補酵素であるN５, N10−メチレンテトラヒドロ葉酸（N５, N10−メチレンTHF）がメチル基供与体として働く。THFは、葉酸の還元型であり、一炭素単位の官能基の転移や代謝において中心的役割を果たしている。

４　誤
ピリミジンヌクレオチドの分解によりβ−アラニンやβ−アミノイソ酪酸、アンモニアが生じ、アンモニアは肝臓に運ばれた後、尿素回路により尿素となって体外に排泄される。なお、分解により尿酸が生成されるのは、プリンヌクレオチドである。
