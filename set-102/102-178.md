# 第102回 問178

**問題文**
固形製剤の評価に用いられる一般試験法に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-178 を参照

**選択肢**
1. 腸溶性の崩壊試験では、崩壊試験第1液中で耐酸性を評価した後、試験に用いた錠剤を引き続き使用して、崩壊試験第2液中での崩壊性を判定する。
2. 収着−脱着等温線測定法における水分の収着とは、医薬品粉体に水分の吸着及び吸収の両方が起こることである。
3. 溶出試験の1つであるフロースルーセル法は、徐放性製剤の試験方法であり、即放性製剤には適用されない。
4. 製剤均一性試験法において、医薬品の有効成分含量が25 mg以上かつ有効成分濃度が25%以上の素錠の場合には、質量偏差試験が適用できる。
5. 製剤の粒度の試験法で用いるふるいは、18号のふるいの方が、30号のふるいに比べてより細かい粒子をふるい分けることができる。

**正解:** 2,4

**解説**
解答　２、４

１　誤
崩壊試験とは、錠剤、カプセル剤、顆粒剤、シロップ用剤、丸剤が試験液中、定められた条件で規定時間内に崩壊するかどうかを確認する試験である。腸溶錠の崩壊試験では、胃内pHを想定したpH1.2の崩壊試験第1液、腸内pHを想定したpH6.8の崩壊試験第2液を用いてそれぞれ別々に試験を行う。その際、第1液で使用した錠剤を引き続き第2液で使用してはならない。

２　正
収着−脱着等温線測定法は、固体−水間の相互作用を評価する一般試験法である。原薬又は製剤としての医薬品粉体は、製造工程や保存中に水と接触し相互作用が起こることがある。そのうち、表面においてのみ相互作用が起こることを吸着といい、固体中へ浸透することを吸収という。また、吸着と吸収の両方が起こることきは、収着という用語が用いられる。

３　誤
溶出試験は、経口製剤について溶出規格、溶出基準に適合しているかどうかを判定するために行われるものである。溶出試験には、フロースルーセル法、回転バスケット法、パドル法の３つがあり、徐放性製剤のほか即放性製剤にも適用することができる。

４　正
製剤均一性試験法とは、個々の製剤の間での有効成分含量の均一性の程度を示すための試験法であり、含量均一性試験または質量偏差試験のいずれかの方法で試験される。素錠において、医薬品の有効成分含量が25 mg以上かつ有効成分濃度が25%以上の場合には、質量偏差試験が適用できる。

５　誤
ふるい番号は、数字の大きいものほど、細かい粒子をふるい分けることができる。そのため、製剤の粒度の試験法で用いるふるいは、30号のふるいの方が、18号のふるいに比べてより細かい粒子をふるい分けることができる。
