# 第102回 問41

**問題文**
薬物の腸肝循環の経路はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-41 を参照

**選択肢**
1. 肝臓→門脈→胆管→腸管→肝臓
2. 肝臓→門脈→腸管→胆管→肝臓
3. 肝臓→胆管→門脈→腸管→肝臓
4. 肝臓→胆管→腸管→門脈→肝臓
5. 肝臓→腸管→門脈→胆管→肝臓
6. 肝臓→腸管→胆管→門脈→肝臓

**正解:** 4

**解説**
解答　４

腸管循環とは、肝臓に取り込まれた薬物が代謝を受け胆汁中排泄されて消化管へ移行した後、腸内細菌のβ−グルクロニダーゼにより脱抱合（加水分解）を受けて再び消化管より吸収され、門脈から肝臓に取り込まれる現象である。
