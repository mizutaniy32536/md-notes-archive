# 第102回 問84

**問題文**
妊婦への投与が禁忌である医薬品はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-84 を参照

**選択肢**
1. アセトアミノフェン
2. インスリン
3. 炭酸リチウム
4. プレドニゾロン
5. ヘパリンナトリウム

**正解:** 3

**解説**
解答　３

１　誤
アセトアミノフェンは、妊婦又は妊娠している可能性のある婦人には、治療上の有益性が上回る場合にのみ投与するとされており、禁忌ではない。

２　誤
インスリンは、特に催奇形性等の報告はされておらず、妊娠時の糖尿病治療に用いられる。

３　正
炭酸リチウムは、催奇形性等の報告があるため、妊婦又は妊娠している可能性のある婦人への投与は禁忌とされている。

４　誤
プレドニゾロンは、妊婦又は妊娠している可能性のある婦人には、治療上の有益性が上回る場合にのみ投与するとされており、禁忌ではない。

５　誤
ヘパリンナトリウムは、胎盤通過性が非常に低く、催奇形性等の報告もされていないため、妊娠時の血栓症などの治療に用いられる。
