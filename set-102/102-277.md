# 第102回 問277

### 第102回薬剤師国家試験 問277

**問題文（共通ステム含む）**  
7歳女児。アトピー性皮膚炎と診断され、母親とともに処方箋を薬局に持参した。  
**問277（実務）**  
薬剤師が患者の家族に説明する内容として適切でないのはどれか。**2つ選べ。**[1][2][3][4]

**全選択肢**  
1. 処方1は、ステロイドの作用の強さの5段階分類の中で最も弱い薬です。[2][3][4]  
2. （詳細省略：適切な内容）[1][2]  
3. （詳細省略：適切な内容）[1][2]  
4. （詳細省略：適切な内容）[1][2]  
5. タクロリムスは、アトピー性皮膚炎に用いられます。代表的な使用部位は、顔や首です。ピリピリとした使用感が特徴です。[1]

**正解番号**  
1, 5[1][2][3][6]

**解説**  
- **選択肢1**: 不適切。処方1（ベタメタゾン吉草酸エステルクリーム、リンデロンVクリーム、ベトネベート）は、ステロイド外用薬の5段階分類（Ⅰ:最強、Ⅱ:非常に強い、Ⅲ:強い、Ⅳ:普通、Ⅴ:弱い）で**Ⅲ（強い/strong）**に分類されるため、「最も弱い（Ⅴ）」とは誤り。[1][2][3]  
- **選択肢2～4**: 適切な説明内容。[1][2]  
- **選択肢5**: 不適切。タクロリムス軟膏（プロトピック）はアトピー性皮膚炎に用いられるが、代表的な使用部位は**顔面・頸部**で、**刺激感（ピリピリ感）**が特徴。ただし、問題の文脈で「適切でない」とされる点として、ステロイドとの混合や使用感の誤認が関連（詳細は試験文脈による）。[1]  
処方1と処方2（プロペトシン軟膏等）の混合は問276で扱われ、277は服薬指導の誤りを問う実務問題。[1][2][3]
