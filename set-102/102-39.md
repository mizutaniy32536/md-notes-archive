# 第102回 問39

**問題文**
ゲンタマイシンの作用機序はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-39 を参照

**選択肢**
1. 葉酸合成阻害
2. 細胞壁合成阻害
3. タンパク質合成阻害
4. DNA複製阻害
5. DNAからRNAへの転写阻害

**正解:** 3

**解説**
解答　３

ゲンタマイシンは、アミノグリコシド系抗菌薬であり、主として細菌リボソームの30Sサブユニットに結合することで、タンパク質合成を阻害する。
