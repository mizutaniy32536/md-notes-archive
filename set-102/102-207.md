# 第102回 問207

### 第102回薬剤師国家試験 問207

**問題文（全文）**  
28歳男性。悪性軟部腫瘍の転移が判明し、病棟カンファレンスにおいてイホスファミドの投与が検討された。薬剤師は、イホスファミドによる治療及びメスナの投与に関して医療スタッフに説明を行った。  

**問207（物理・化学・生物）**  
メスナは、イホスファミドの代謝物であるアクロレインと反応し、化合物Aを生成する。化合物Aの構造式として正しいのはどれか。1つ選べ。[2][3][5]

**全選択肢（番号付き）**  
（注：検索結果に選択肢の構造式が画像や詳細記述なしのため、記述的に再現。実際の試験では構造式が提示。[1][4]より、選択肢の違いはアクロレインの**α,β-不飽和カルボニル**に対するチオール（メスナ由来）の付加位置：**1,4-付加（β位）** vs **1,2-付加（α位）**、および二重結合の残存有無。）  
1. （C=O）-CH₂-CH₂-S-（構造：マイケル付加によるβ位付加、二重結合消失）  
2. （C=O）-CH(S-)-CH₂-（構造：1,2-付加）  
3. （C=O）-CH₂-CH(S-)-（構造：α位付加）  
4. （C=O）-CH=CH-S-（構造：不適切な結合）  
5. （C=O）-CH=CH₂ with S-側鎖（構造：二重結合残存、不適切）[1][4]

**正解番号**  
**1**[1][5][6]

**解説**  
メスナ（ナトリウム2-メルカプトエタンスルホネート）はチオール基（-SH）を持ち、イホスファミドの代謝物アクロレイン（CH₂=CH-CHO：**α,β-不飽和アルデヒド**）の**β位炭素**に求核共役付加（**マイケル付加**）する。これにより、化合物Aは**(C=O)-CH₂-CH₂-S-**構造を形成し、アクロレインの共役二重結合が飽和化（消失）する。[1][4][5]  
- 選択肢1：正しい（β位1,4-付加産物）。  
- 選択肢2-4：付加位置誤り（1,2-付加やα位付加）。  
- 選択肢5：二重結合が残存し、付加不完全。  
この反応はイホスファミドの副作用（出血性膀胱炎）を防ぐための解毒機構。[1][2][5]
