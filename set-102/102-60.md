# 第102回 問60

**問題文**
パーキンソン病患者に特徴的な症状はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-60 を参照

**選択肢**
1. 躁状態
2. 下痢
3. 高血圧
4. 上肢の筋弛緩
5. すくみ足

**正解:** 5

**解説**
解答　５

パーキンソン病は、中脳の黒質におけるドパミン作動性神経が変性する神経変性疾患の一種であり、安静時振戦、無動（すくみ足）、筋固縮、姿勢反射障害などの不随意運動障害を主徴とする。そのほか、自律神経症状として便秘や起立性低血圧、精神症状として抑うつや認知症なども見られることがある。
