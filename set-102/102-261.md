# 第102回 問261

### 第102回薬剤師国家試験 問261

**問題文（共通ステム）**  
38歳男性。急性リンパ性白血病と診断され、以下の化学療法が開始された。[2][3]

**問261（薬理）**  
前問（問260）の支持療法に用いる薬物の作用機序で、正しいのはどれか。2つ選べ。[2][3]

**選択肢**  
1. 尿酸トランスポーターを阻害する。[2][3]  
2. DNAをアルキル化し、DNA合成を阻害する。[2][3]  
3. セロトニン5-HT₃受容体を遮断する。[2][3]  
4. G-CSF（顆粒球コロニー刺激因子）受容体を刺激する。[2][3]  
5. PPARγ（ペルオキシソーム増殖剤応答性受容体γ）を阻害する。[1][2][3]

**正解番号**  
3, 4[1][2][3]

**解説**  
- **選択肢1（誤）**: 尿酸トランスポーターを阻害するのは尿酸排泄促進薬（プロベネシド、ベンズブロマロン）。問260の支持療法薬アロプリノールはキサンチンオキシダーゼ阻害薬で尿酸産生を抑制する。[1][3]  
- **選択肢2（誤）**: DNAをアルキル化しDNA合成を阻害するのは抗悪性腫瘍薬（シクロホスファミド、イホスファミド）。支持療法薬には該当しない。[3]  
- **選択肢3（正）**: グラニセトロンの作用機序。セロトニン5-HT₃受容体遮断により化学療法誘発性悪心・嘔吐を抑制。[1][3]  
- **選択肢4（正）**: フィルグラスチムの作用機序。G-CSF受容体を刺激し好中球増加を促す（骨髄抑制対策）。[1][3]  
- **選択肢5（誤）**: PPARγ阻害は支持療法薬ピタバスタチン（問260の非優先薬、スタチン系HMG-CoA還元酵素阻害）の主作用ではない。PPARγ活性化はピオグリタゾンなどの薬に関連。[1]  

問260の支持療法薬（フィルグラスチム、グラニセトロン、アロプリノール、メロペネム）はALL化学療法の骨髄抑制、悪心、腫瘍崩壊症候群、感染対策に用いられ、その作用機序として3と4が正しい。[1][2][3]
