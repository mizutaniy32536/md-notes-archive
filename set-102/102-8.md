# 第102回 問8

**問題文**
ラジカル中間体を経る反応はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-8 を参照

**選択肢**
（選択肢は画像を参照）

**正解:** 3

**解説**
解答　３

１　誤
2−メチルプロパン−2−オール（第三級アルコール）に臭化水素を反応させると、2−ブロモ−2−メチルプロパンが生成する。本反応は、SN1反応（一分子求核置換反応）であり、カルボカチオン中間体を経由し、二段階反応で進行するが、ラジカル中間体は経由しない。

２　誤
2−クロロ−2−メチルプロパン（第三級ハロゲン）を水酸化ナトリウム（塩基）で処理すると、2−メチルプロペンが生成する。本反応は、E2反応（二分子脱離反応）であり、一段階反応で進行するため、中間体は経由しない。

３　正
トルエンに光照射の下、N−ブロモスクシンイミド（NBS）を反応させると、ベンジル位でラジカル置換反応が進行し、HがBrに置換され、ベンジルブロミドが生成する。本反応は、安定なベンジルラジカル中間体を経由する。

４　誤
ヨードメタン（第一級ハロゲン）に水酸化ナトリウム（塩基）を反応させると、メタノールが生成する。本反応は、SN2反応（二分子求核置換反応）であり、一段階反応で進行するため、中間体は経由しない。

５　誤
シクロペンタ−1,3−ジエンにアクリル酸メチルを反応させると、ビシクロ［2,2,1］ヘプタ−5−エン−2−カルボン酸メチルが生成する。本反応はペリ環状反応の一種であるDiels−Alder反応であり、遷移状態を経て一段階反応で進行するため、中間体は経由しない。
