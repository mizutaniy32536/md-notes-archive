# 第102回 問62

**問題文**
アトピー性皮膚炎に関する記述のうち、最も適切なのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-62 を参照

**選択肢**
1. 皮膚のバリア機能が亢進している。
2. 対症療法が治療の中心となる。
3. 湿疹は左右非対称にみられる。
4. おもにⅡ型アレルギーが関与している。
5. 抗ヒスタミン薬が第1選択である。

**正解:** 2

**解説**
解答　２

１　誤
アトピー性皮膚炎では、皮膚のバリア機能は低下している。

２　正
アトピー性皮膚炎は、遺伝的素因も含んだ多病因性の疾患であり、疾患を完治させることのできる治療はない。そのため本症の治療は、皮膚の状態を保ちながら患者のQOLを確保するための対症療法が中心となる。

３　誤
アトピー性皮膚炎の湿疹は、増悪と寛解を繰り返す痒みのある湿疹であり、左右対称にみられる。

４　誤
アトピー性皮膚炎の発症には、おもにⅠ型アレルギーが関与している。

５　誤
アトピー性皮膚炎の治療の第1選択は、副腎皮質ステロイド性薬の外用による炎症の鎮静である。なお、抗ヒスタミン薬は、アレルギー症状や痒みをおさえる補助的な位置づけで用いられる。
