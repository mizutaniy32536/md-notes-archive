# 第102回 問76

**問題文**
医薬品副作用被害救済制度における副作用救済給付の対象として、誤っているのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-76 を参照

**選択肢**
1. 医療費
2. 医療手当
3. 障害年金
4. 休業保障
5. 葬祭料

**正解:** 4

**解説**
解答　４

医薬品副作用被害救済制度における副作用救済給付には、医療費、医療手当、障害年金、障害児養育年金、遺族年金、遺族一時金、葬祭料はあるが、休業保障はない。
