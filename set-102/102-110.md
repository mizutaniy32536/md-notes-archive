# 第102回 問110

**問題文**
生薬に関する記述のうち正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-110 を参照

**選択肢**
1. ロートコンは、ナス科基原植物由来生薬で、トロパンアルカロイドを含有する。
2. リュウタンは、フラボノイド配糖体を主成分とし、苦味健胃薬として用いられる。
3. リュウコツは、硫酸ナトリウムを主成分とし、精神安定を目的として用いられる。
4. インチンコウは、果実を用部にする生薬であり、胆汁分泌促進作用がある。
5. トチュウは、トチュウの樹皮を用部にする生薬であり、リグナンを含有する。

**正解:** 1,5

**解説**
解答　１、５

１　正
ロートコンは、ナス科基原植物由来ハシリドコロなどの根及び根茎を主薬として用いる生薬であり、l−ヒヨスチアミン、スコポラミンなどのトロパンアルカロイドを含有する。

２　誤
リュウタンは、リンドウ科基原植物由来リンドウなどの根及び根茎を主薬として用いる生薬であり、ゲンチオピクロシドなどの苦味配糖体を主成分として、苦味健胃薬として用いられる。

３　誤
リュウコツは、大型ほ乳動物の化石化した骨を主薬として用いる生薬であり、炭酸カルシウムやリン酸カルシウムなどのカルシウム塩を主成分として、精神安定を目的に用いられる。

４　誤
インチンコウは、キク科基原植物由来カワラヨモギの頭花を乾燥させて用いる生薬であり、クマリン誘導体やイソクマリン誘導体を主成分として、利胆・利尿などを目的として用いられる。

５　正
トチュウは、トチュウ科基原植物由来トチュウの乾燥させた樹皮を用いる生薬であり、リグナンを含有する。
