# 第102回 問43

**問題文**
ヒドロキシ基を有する薬物（R−OH）のグルクロン酸抱合体を示す化学構造はどれか。1つ選べ。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-43 を参照

**選択肢**
（選択肢は画像を参照）

**正解:** 2

**解説**
解答　２

１　誤
硫酸抱合体の化学構造である。硫酸抱合は、ヒドロキシ基やアミノ基などを有する薬物が受けやすい。

２　正
グルクロン酸抱合体の化学構造である。グルクロン酸抱合は、ヒドロキシ基、カルボキシ基、アミノ基などを有する薬物が受けやすい。

３　誤
アセチル抱合体の化学構造である。アセチル抱合は、芳香族第一級アミン、ヒドラジン基などを有する薬物が受けやすい。

４　誤
メチル抱合体の化学構造である。メチル抱合は、ヒドロキシ基、チオール基、アミノ基などを有する薬物が受けやすい。

５　誤
アミノ酸抱合体の化学構造である。アミノ酸抱合は、カルボキシ基を有する薬物が受けやすい。
