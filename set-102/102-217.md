# 第102回 問217

### 第102回薬剤師国家試験 問217

**問題文**  
肺血栓塞栓症において、下肢静脈で生じた血栓が肺へ到達する経路として、正しいのはどれか。1つ選べ。[1][2][5]

**全選択肢**  
1. 大腿静脈→総腸骨静脈→下大静脈→**右心房**→**右心室**→**肺静脈**→肺[2][5]  
2. 大腿静脈→総腸骨静脈→下大静脈→**右心房**→**右心室**→**肺動脈**→肺[2][5]  
3. 総腸骨静脈→大腿静脈→**門脈**→下大静脈→右心房→右心室→肺動脈→肺[2][5]  
4. 総腸骨静脈→大腿静脈→下大静脈→**左心房**→**左心室**→肺静脈→肺[1][2]  
5. 大腿静脈→総腸骨静脈→下大静脈→**左心房**→**左心室**→**肺動脈**→肺[1][2][5]

**正解番号**  
**2**[1][2][4][5]

**解説**  
下肢静脈（大腿静脈→総腸骨静脈→下大静脈）で生じた血栓は、下大静脈を経て**右心房**→**右心室**→**肺動脈**→肺へ到達する。これは体循環の正常経路（全身→右心房→右心室→肺動脈→肺）による。[1][2][5]  
- 選択肢1：肺静脈は肺から左心房へ戻る肺循環の静脈のため誤り。  
- 選択肢3：門脈は消化管から肝臓への門脈系で、下肢血栓とは無関係。  
- 選択肢4・5：左心房・左心室は肺循環後の酸素化血液が通るため、下肢血栓は到達しない。[1][2]
