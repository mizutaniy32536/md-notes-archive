## ◆ 豚のレバーを生食することで感染リスクが最も高いウイルスを問うシナリオ

260312

**問題**
ブタのレバーを生で食べることにより感染するリスクが最も高いウイルスはどれか。1つ選べ。

**選択肢**

1. A型肝炎ウイルス
​
2. B型肝炎ウイルス
​
3. C型肝炎ウイルス
​
4. E型肝炎ウイルス
​
5. ノロウイルス

**正解と理由**
- `4. E型肝炎ウイルス`
- **理由の構成**
	- **問題の整理**：「`豚レバーの生食`」から感染する肝炎ウイルスを特定する
	- **薬学的知識・仕様・制約**：`E型肝炎`は動物（豚・猪・鹿）の内臓・肉の生食が主要感染源。人獣共通感染症
	- **選択理由**：`E型肝炎ウイルス`が豚レバーに関連する肝炎の代表的原因

**解法の手順（思考の流れ）**
- 各肝炎の感染経路を確認：
	- A型：`汚染水・魚介類`（カキ等）→ 糞口感染
	- B型：`血液・性行為・母子`
	- C型：`血液`
	- **E型：`豚・猪・鹿の生肉/内臓`** → 人獣共通感染
	- ノロ：`汚染カキ等貝類` → 糞口感染
- 「豚のレバーを生食」→ `E型肝炎ウイルス` → 正しい

**キーポイント**
- `E型肝炎`：豚・猪・鹿のジビエ食品からの感染が日本で増加
- 2003年に豚肝臓のE型肝炎ウイルス汚染が確認→豚レバー生食禁止（2015年に食品衛生法で禁止）
- A型・E型は経口感染（糞口経路）、B型・C型は血液感染

**関連知識の比較**

| 肝炎 | 感染経路 | 食材 |
|---|---|---|
| `A型` | 糞口 | カキ・汚染水 |
| `B型` | 血液・体液 | なし |
| `C型` | 血液 | なし |
| `E型` | 糞口 | **豚レバー・猪・鹿** |
| ノロウイルス | 糞口 | カキ（貝類） |

（一言メモ）
- 豚レバー＝`E型肝炎`。鹿肉＝`E型肝炎`。カキ＝`ノロ`または`A型`。これで食中毒出題は完璧
- 「どの動物から」で感染源ウイルスが決まる国試頻出問題
