# 第102回 問287

### 第102回薬剤師国家試験 問287

**問題文（全文）**  
65歳男性。人間ドックで血中prostate specific antigen（**PSA**）値の高値（30.3 ng/mL）を認めた。  
その後、生検の結果、腫瘤が悪性と診断されたため内分泌療法が施行され、PSA値は正常上限値以下を推移していた。しかし、6年後の受診で再度PSA値が高値となった。このため抗アンドロゲン薬を試みたが、改善が見られず、内分泌療法抵抗性が確認された。そこで、新たな薬物療法を開始することになった。  
本患者に適用することになった薬物はどれか。**２つ選べ。**[1][2][7]

**全選択肢（番号付き）**  
1. シロドシン  
2. **ドセタキセル**  
3. コハク酸ソリフェナシン  
4. **プレドニゾロンコハク酸エステルナトリウム**  
5. クロルマジノン酢酸エステル[1][2]

**正解番号**  
2、4[1][2]

**解説**  
本症例は、PSA高値から生検で**前立腺がん**と診断され、内分泌療法（ホルモン療法）で一旦改善したものの、6年後に再発し、抗アンドロゲン薬（例: クロルマジノン酢酸エステル）で改善せず、**内分泌療法抵抗性（去勢抵抗性前立腺がん：CRPC）**が確認された状態である。[1][2]  
CRPCに対する標準的な新たな薬物療法として、**ドセタキセル**（抗がん剤、タキサン系）と**プレドニゾロン**（ステロイド薬）の併用療法が適用される。これは化学療法の第一選択肢であり、生存期間延長効果が示されている。[1][2]  
他の選択肢の理由：  
- 1. シロドシン：**前立腺肥大症**のα1遮断薬（尿路症状改善）。  
- 3. コハク酸ソリフェナシン：**過活動膀胱**の抗コリン薬。  
- 5. クロルマジノン酢酸エステル：抗アンドロゲン薬（既に抵抗性確認済み）。[1][2]
