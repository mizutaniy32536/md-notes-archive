## ◆ リーフレットの[A]に該当する性感染症を症状から特定するシナリオ

260312

**問題**
20歳女性。性感染症の薬物治療のため処方箋を持参。服薬指導の際にリーフレットを手渡した。
[ A ] は、陰部に潰瘍ができたり、リンパ節の腫れ、全身の発疹などの症状を呈する。
[ A ] にあてはまる感染症はどれか。1つ選べ。

**選択肢**

1. 淋菌感染症
​
2. 性器クラミジア感染症
​
3. 性器ヘルペスウイルス感染症
​
4. 尖圭コンジローマ
​
5. 梅毒

**正解と理由**
- `5. 梅毒`
- **理由の構成**
	- **問題の整理**：「陰部の潰瘍」「リンパ節腫脹」「全身の発疹」=`梅毒`の典型的な症状
	- **薬学的知識・仕様・制約**：梅毒の病期別症状を理解する
	- **選択理由**：記述された症状が梅毒の一期〜二期の症状と一致

**解法の手順（思考の流れ）**
- 陰部の潰瘍=`梅毒一期`（硬性下疳）or`性器ヘルペス`
- リンパ節腫脹=`梅毒一期〜二期`の所見
- 全身の発疹=`梅毒二期`（バラ疹）
- 3つの症状が揃うのは`梅毒`のみ → 5が正解

**各性感染症の主な症状**

| 性感染症 | 主な症状 |
|---|---|
| `淋菌感染症` | 排尿痛、膿性分泌物 |
| `クラミジア` | 不正出血、粘液性分泌物（不顕性感染多い） |
| `性器ヘルペス` | 水疱・潰瘍（局所性、再発する） |
| `尖圭コンジローマ` | 疣贅（いぼ状病変） |
| **`梅毒`** | **硬性下疳（潰瘍）→バラ疹（全身発疹）→ゴム腫** |

**キーポイント**
- `梅毒`の病期：一期（硬性下疳・無痛性潰瘍＋鼠径リンパ節腫脹）→二期（全身性バラ疹）→三期（ゴム腫）→四期（心血管・神経梅毒）
- 近年日本で`梅毒`が急増中（2023年に戦後最多を更新）
- 治療：`アモキシシリン`または`ベンジルペニシリン`

（一言メモ）
- 「陰部潰瘍＋全身発疹」=`梅毒`。局所の潰瘍だけなら性器ヘルペスと鑑別が必要
- 梅毒の一期の特徴は「無痛性」の潰瘍。ヘルペスは有痛性（強い痛み）
