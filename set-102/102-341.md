# 第102回薬剤師国家試験 問341

## 問題文

48歳女性。非小細胞肺がん。以下の処方箋をかかりつけ薬局に持参した。

薬歴からこれまでは処方2の薬剤が3週間毎に処方されており、処方1は初めての処方であることを確認した。

この患者の処方箋と薬歴情報について正しいのはどれか。**2つ選べ。**[1]

## 全選択肢

1. 今回の処方薬は、他の化学療法施行後に開始された薬剤である。

2. 血小板減少による発熱が出現していた可能性がある。

3. 息切れ、咳などのかぜのような症状が出ても自然におさまることを伝える。

4. 発疹、皮膚の乾燥やかゆみが出現する可能性があることを伝える。

## 正解番号

**1、4**[1][5]

## 解説

**選択肢1（正）**

処方1は今回初めて処方された薬剤であり、処方2の薬剤（ジフェンヒドラミン塩酸塩錠、レボフロキサシン錠など）が3週間毎に処方されていたことから、処方2は化学療法に伴う過敏症抑制および白血球減少に伴う易感染状態に対して使用されていたと推察される。したがって、処方1は他の化学療法施行後に開始された薬剤である。[5]

**選択肢2（誤）**

血小板減少は出血傾向を招くが、発熱とは関連しない。ロキソプロフェンNa錠が処方されていることから、**白血球減少に伴う発熱**が出現していた可能性がある。[1][5]

**選択肢3（誤）**

白血球減少により易感染性が表れている可能性があり、息切れや咳などの症状が自然におさまることを伝えるべきではない。むしろ、これらの症状は化学療法の重篤な副作用の初期症状である可能性があるため、患者にはすぐに医師や薬剤師に報告するよう注意してもらう必要がある。[1]

**選択肢4（正）**

分子標的薬（チロシンキナーゼ阻害薬など）の副作用として、**発疹、皮膚の乾燥やかゆみなどの皮膚障害が出現する可能性**がある。これは既知の副作用であり、患者に事前に伝える必要がある。[5]
