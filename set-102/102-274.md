# 第102回薬剤師国家試験 問274

## 問題文

73歳男性。体重60kg。メチシリン耐性黄色ブドウ球菌(MRSA)肺炎の治療目的でアルベカシン硫酸塩の投与が開始された。

投与開始から3日目に血中アルベカシン濃度の測定依頼があり、測定の結果、トラフ値は3.5μg/mL、ピーク値(点滴終了30分後採血)は15μg/mLであった。

検査値(3日目)：白血球数9,500/μL、CRP 4.8 mg/dL、血清クレアチニン2.84 mg/dL

## 選択肢

1. 現在の投与計画を継続する。
2. 点滴速度を速くする。
3. 投与間隔を変えず、1回投与量を増やす。
4. 1回投与量を変えず、投与間隔を延ばす。
5. 1回投与量を変えず、投与間隔を短縮する。

## 正解

**4**

## 解説

アミノグリコシド系抗菌薬であるアルベカシンのトラフ値が3.5μg/mLと高く、また血清クレアチニンが2.84 mg/dLと上昇している状況です。アミノグリコシド系薬物は腎毒性を有するため、トラフ値が高い場合は投与間隔を延ばすことが原則です[1]。

各選択肢の検討：

- **選択肢1**：トラフ値が高いままであり、腎毒性のリスクが増加するため適切ではありません[1]。

- **選択肢2**：点滴速度の変更は血中濃度推移に影響せず、トラフ値の低下につながらないため適切ではありません。

- **選択肢3**：投与量を増やすとトラフ値はさらに上昇するため、不適切です[1]。

- **選択肢4**：1回投与量を変えず投与間隔を延ばすことで、トラフ値を低下させることができます。これにより、腎毒性のリスクを軽減しながら治療効果を維持できるため、最も適切です[1]。

- **選択肢5**：投与間隔を短縮するとトラフ値が上昇するため、不適切です[1]。
