# 第102回 問3

**問題文**
α壊変により放出されるα粒子はどの元素の原子核に相当するか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-3 を参照

**選択肢**
1. 1H
2. 4He
3. 7Li
4. 11B
5. 12C

**正解:** 2

**解説**
解答　２

α壊変は、質量数の大きな核種が安定な核種になるために、α粒子の本体であるヘリウム原子核（4He：陽子2個、中性子2個）を放出する壊変のことである。α壊変した核種は原子番号が2減少、質量数が4減少した娘核種となる。
