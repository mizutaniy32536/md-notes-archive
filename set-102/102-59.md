# 薬剤師国家試験 第102回 問59

## 問題文

インスリン分泌を促進し、食後高血糖を改善する薬物はどれか。1つ選べ。[1][2]

## 選択肢

1. メトホルミン塩酸塩
2. ピオグリタゾン塩酸塩
3. ナテグリニド
4. ボグリボース
5. イプラグリフロジンL-プロリン

## 正解番号

**3番**[1][4]

## 解説

**1番 メトホルミン塩酸塩（誤）**
ビグアニド系薬であり、インスリン分泌作用がないことが特徴です。AMP依存性プロテインキナーゼを活性化し、肝臓での糖新生の抑制や末梢での糖利用促進を介して血糖を低下させます。[1][4]

**2番 ピオグリタゾン塩酸塩（誤）**
ペルオキシソーム増殖剤応答性受容体γ（PPARγ）を活性化し、インスリン抵抗性を軽減することにより高血糖を改善する薬物です。インスリン分泌を促進する作用ではありません。[1][4]

**3番 ナテグリニド（正）**
スルホニルウレア（SU）受容体に結合する薬物であり、膵臓ランゲルハンスβ細胞に存在するSU受容体に作用してインスリンの分泌を促進することにより、食後高血糖を改善します。[1][4]

**4番 ボグリボース（誤）**
α-グルコシダーゼ阻害薬です。糖の吸収を遅延させることにより血糖上昇を抑制する薬物で、インスリン分泌促進作用ではありません。[1]

**5番 イプラグリフロジンL-プロリン（誤）**
SGLT2阻害薬（スーグラ）です。グルコーストランスポーターを阻害することにより、尿中へのグルコース排出を促進させることで血糖降下作用を示します。[1]
