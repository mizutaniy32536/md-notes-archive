# 第102回 問176

### 第102回薬剤師国家試験 問176

**問題文（全文）**  
固形製剤の製造工程と製剤機械に関する記述のうち、正しいのはどれか。**２つ選べ。**[2][5][9]

**全選択肢（番号付き）**  
1. 流動層造粒装置は、混合、造粒、乾燥を1つの装置内で行うことができる。  
2. V型混合機は、容器固定型混合機に分類される。  
3. 糖衣は、フィルムコーティングに比べ、短時間でのコーティング処理が可能である。  
4. 直接打錠法では、原料粉末をそのまま打錠機で圧縮成形するため、滑沢剤の添加を必要としない。  
5. ハンマーミルは、粉砕時に熱が発生するため、熱に弱い医薬品の粉砕には適さない。[2][5]

**正解番号**  
**1 と 5**[2][9]

**解説**  
- **1 正しい**：流動層造粒装置（流動層造粒法）は、粉体を下方から熱気流で流動させ、結合剤溶液を噴霧して造粒する。この装置内で混合、造粒、乾燥を一括で行える。[2][3][9]  
- **2 誤り**：V型混合機は回転型混合機に分類され、容器固定型（例: 二重円錠錘混合機）ではない。[2]  
- **3 誤り**：糖衣コーティングは白糖主成分のシロップを繰り返し塗布するため、フィルムコーティングより工程時間が長い。[2]  
- **4 誤り**：直接打錠法でも、原料粉末に結合剤、賦形剤、崩壊剤、**滑沢剤**を加えて均一化後、圧縮成形する。滑沢剤は打錠器具への付着防止に必要。[2]  
- **5 正しい**：ハンマーミルは高速回転による衝撃粉砕で摩擦熱が発生し、熱に弱い医薬品（例: ビタミン類）には不適。[2][9]
