# 第102回 問186

**問題文**
インフルエンザの病態、診断及び治療に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-186 を参照

**選択肢**
1. インフルエンザウイルスは、A、B、Cの3つの型に分類され、いずれもヒトに感染して典型的なインフルエンザ症状を発症させる。
2. インフルエンザによる死亡率が最も高い年代は、15歳以下の子供である。
3. 迅速診断には、鼻腔・咽頭拭い液を用いた酵素免疫測定法が用いられる。
4. インフルエンザを発症した小児の解熱には、アセトアミノフェンは推奨されない。
5. 慢性呼吸器疾患などのハイリスク患者にはオセルタミビルの予防内服が認められている。

**正解:** 3,5

**解説**
解答　３、５

１　誤
インフルエンザウイルスは、A、B、Cの3つの型に分類され、いずれもヒトに感染するが、関節痛、筋肉痛、高熱などの典型的なインフルエンザ症状がみられるのは一般的にA型とB型のみで、C型はかぜ症候群のような軽い上気道炎を起こす程度の症状しかみられない。

２　誤
インフルエンザによる死亡率が最も高い年代は、体が弱く抵抗力のない65歳以上の高齢者である。

３　正
インフルエンザの迅速診断キットは、鼻腔・咽頭拭い液や鼻腔吸引液中のウイルスを酵素免疫測定法により検出するものである。

４　誤
インフルエンザを発症した小児の解熱には、アセトアミノフェンが推奨される。なお、通常解熱に用いられるNSAIDは、インフルエンザ感染時の小児に使用するとライ症候群を引き起こすことがあるため用いない。

５　正
インフルエンザ感染症を発症している患者の同居家族または共同生活者である高齢者、慢性呼吸器疾患患者、慢性心疾患患者、糖尿病などの代謝性疾患患者、腎機能障害患者に対してはオセルタミビルやザナミビル、ラニナミビルなどのノイラミニダーゼ阻害薬の予防投与が認められている。
