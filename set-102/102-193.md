# 第102回 問193

**問題文**
病態と生体リズムの関係について、正しいのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 102-193 を参照

**選択肢**
1. 喘息発作による呼吸困難が起きやすいのは、午後である。
2. 高血圧症患者において血圧が最も高くなりやすいのは、夕方である。
3. 歯の痛みが発現しやすいのは、正午頃である。
4. うつ病のうつ状態が最も強いのは夕方である。
5. コレステロールの生合成が高まるのは、午前中である。

**正解:** 2

**解説**
解答　２

１　誤
喘息発作による呼吸困難が起こりやすいのは、副交感神経優位で気管支が収縮しやすい夜間〜早朝である。

２　正
一般的に、高血圧症患者において血圧が高くなりやすいのは、昼間〜夕方である。

３　誤
一般的に、歯の痛みが発現しやすいのは、夜間である。

４　誤
うつ病のうつ状態が強いのは朝方であり、夕方になるにつれて症状は改善していく。

５　誤
コレステロールの生合成が高まるのは、夜間である。
