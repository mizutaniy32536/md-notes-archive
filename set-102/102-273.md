# 第102回薬剤師国家試験 問273

## 問題文

前問での提案の理由となるシプロフロキサシンとスクラルファートの相互作用の機序として、最も適切なのはどれか。1つ選べ。[3][5]

## 全選択肢

1. 胃内pHの変化に伴う溶解性の低下[3]

2. 消化管内における難溶性キレートの形成[3]

3. 消化管吸収に関わるトランスポーターの阻害[3]

4. 血漿タンパク結合の阻害[3]

5. 腎尿細管における再吸収の阻害[3]

## 正解番号

**2**[1][5]

## 解説

シプロフロキサシンとスクラルファートの相互作用は、**消化管内におけるキレート形成**により生じます。[1]スクラルファートに含まれるアルミニウムイオンがシプロフロキサシンと難溶性のキレート錯体を形成することで、シプロフロキサシンの吸収が低下します。[1]

前問（問272）での提案である「シプロフロキサシン服用後、2時間以上あけてスクラルファートを服用するように用法を変更する」という疑義照会内容は、この相互作用を回避するためのものであり、服用間隔を設けることでキレート形成を防止することができます。[1]
