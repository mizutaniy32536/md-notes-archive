# 第100回 問50

**問題文**
20℃の条件下で、表面張力が最も大きいのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-50 を参照

**選択肢**
1. エタノール
2. クロロホルム
3. グリセリン
4. 水
5. ベンゼン

**正解:** 4

**解説**
解答 ４
表面張力とは、単位面積の表面をつくるのに要する仕事のことであり、水素結合、クーロン力、ファンデルワールス力などの分子間力の影響を受ける。水は水素結合するため、エタノール、クロロホルム、グリセリン、ベンゼンに比べ、表面張力が大きい。
