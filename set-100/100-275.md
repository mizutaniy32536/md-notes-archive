# 100-275

**問題文**
この薬物0.75 mgを急速静脈内投与するとき、7日後の血中濃度に最も近い値はどれか。1つ選べ。ただし、この薬物の体内動態は線形1−コンパートメントモデルに従うものとし、分布容積は750 Lとする。

**選択肢**
1. 0.25 µg/mL
2. 0.125 µg/mL
3. 0.0625 µg/mL
4. 0.25 ng/mL
5. 0.125 ng/mL
6. 0.0625 ng/mL

**正解:** 6

**解説**
解答　6

初期血中濃度 C0 は `0.75 mg / 750 L = 0.001 mg/L = 1 ng/mL` である。
7日後は168時間後で、半減期42時間の4半減期分に相当する。
したがって、血中濃度は `1 × (1/2)^4 = 0.0625 ng/mL` となる。
