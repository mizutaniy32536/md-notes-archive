# 第100回 問88

**問題文**
インターフェロン製剤を投与中のC型肝炎の患者に禁忌である薬剤はどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-88 を参照

**選択肢**
1. 小柴胡湯
2. 葛根湯
3. ラクツロース
4. ウルソデオキシコール酸
5. リバビリン

**正解:** 1

**解説**
解答 １
インターフェロン投与中の患者に小柴胡湯を投与すると、副作用である間質性肺炎を誘発しやすくなるため、両薬剤は併用禁忌とされている。
よって、インターフェロン製剤を投与中のC型肝炎の患者に禁忌である薬剤は、小柴胡湯である
