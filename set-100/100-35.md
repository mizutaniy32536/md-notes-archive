# 第100回 問35

**問題文**
デスモプレシンの抗利尿作用の機序はどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-35 を参照

**選択肢**
1. バソプレシンV1受容体刺激
2. バソプレシンV1受容体遮断
3. バソプレシンV2受容体刺激
4. バソプレシンV2受容体遮断
5. バソプレシン分泌抑制

**正解:** 3

**解説**
解答 ３
バソプレシン受容体には、血管平滑筋に存在するバソプレシンV1受容体と腎・集合管に存在するバソプレシンV2受容体の2種類が存在する。デスモプレシンは、選択的にバソプレシンV2受容体を刺激することで、腎臓における水の再吸収を促進させることで抗利尿作用を示す。そのため、デスモプレシンは、バソプレシン分泌低下により発症する中枢性尿崩症の治療に用いられる。
なお、バソプレシンV1受容体を刺激すると、血管収縮作用が認められる。
