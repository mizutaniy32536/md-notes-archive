# 第100回 問55

**問題文**
生体に投与後、長時間0次放出を示す製剤はどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-55 を参照

**選択肢**
1. 腸溶性高分子コーティング顆粒
2. 胃溶性高分子コーティング顆粒
3. 腸溶性高分子固体分散体顆粒
4. ワックスマトリックス型錠剤
5. 浸透圧ポンプ型錠剤

**正解:** 5

**解説**
解答 ５
生体に投与後、長時間0次放出を示す製剤は、浸透圧ポンプ型錠剤である。浸透圧ポンプ型錠剤は、薬物及び浸透圧誘発物質を含む錠剤と高分子半透膜で構成されている。水が高分子半透膜を透過すると、薬物あるいは浸透圧誘発物質が溶解し、錠剤内部に浸透圧が発生する。それにより生じた浸透圧によりさらに水が侵入し、薬物が放出される。錠剤内部が飽和濃度以上の濃度を保っている間、薬物は一定速度（0次速度）で放出される。
