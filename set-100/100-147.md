# 第100回 問147

**問題文**
治験に関する記述のうち、正しいのはどれか。２つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-147 を参照

**選択肢**
1. 治験の対象となる薬物について初めて治験の計画の届出をした者は、届出の日から直ちに治験を依頼することができる。
2. 治験の計画を届け出た治験依頼者は、治験を行う医療機関を追加しても、治験計画の変更届を提出する必要はない。
3. 厚生労働大臣は、治験の依頼に関する基準に適合しているかどうかを調査するため、当該職員に治験薬を業務上取り扱う場所に立ち入り、検査させることができる。
4. 治験依頼者は、治験薬の容器に治験用である旨、治験依頼者の氏名及び住所、化学名、用法又は用量、及び効能又は効果を記載しなければならない。
5. 治験依頼者は、治験薬の副作用によるものと疑われる死亡につながるおそれのある症例を知ったときは、定められた期間内に厚生労働大臣（情報の整理を独立行政法人医薬品医療機器総合機構（PMDA）に行わせることとした場合は、PMDA）に報告しなければならない。

**正解:** 3,5

**解説**
解答 ３、５
１ 誤
治験の対象となる薬物について初めて治験の計画の届出をした者は、届出後30日を経過した後でなければ、治験を依頼してはならない。
２ 誤
治験の計画を届け出た治験依頼者は、治験計画に変更が生じた場合、その旨を届け出なければならない。よって、治験の計画を届け出た治験依頼者は、治験を行う医療機関を追加した場合、治験計画の変更届を提出する必要がある。
３ 正
４ 誤
治験依頼者は、治験薬に添付する文書、その治験薬又はその容器若しくは被包に治験用である旨、治験依頼者の氏名及び住所、化学名、用法又は用量、及び効能又は効果を記載してはならない。
５ 正
治験依頼者は、治験薬の副作用によるものと疑われる死亡につながるおそれのある症例を知ったときは、7日以内に厚生労働大臣（情報の整理を独立行政法人医薬品医療機器総合機構（PMDA）に行わせることとした場合は、PMDA）に報告しなければならない。
