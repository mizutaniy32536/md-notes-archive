# 第100回 問68

**問題文**
研究デザインを、エビデンスのレベルの高い順に記載しているのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-68 を参照

**選択肢**
1. ランダム化比較試験＞症例対照研究＞前向きコホート研究
2. 前向きコホート研究＞非ランダム化比較試験＞症例集積
3. ランダム化比較試験＞前向きコホート研究＞症例対照研究
4. 症例集積＞非ランダム化比較試験＞症例報告
5. 症例対照研究＞前向きコホート研究＞ランダム化比較試験

**正解:** 3

**解説**
解答 ３
臨床研究や薬剤疫学研究には種々の研究手法があり、それぞれの研究手法の評価に基づいて適切なエビデンスレベル（科学的根拠の水準）として位置づけを考慮することができる。エビデンスレベルは情報の信頼性と言い換えることができ、次に示す順に上からエビデンスレベルが高い。
