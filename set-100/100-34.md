# 第100回 問34

**問題文**
ガベキサートの急性膵炎治療効果に関わる機序はどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-34 を参照

**選択肢**
1. タンパク質分解酵素阻害
2. H＋,K＋−ATPase阻害
3. ムスカリン性アセチルコリン受容体遮断
4. ヒスタミンH2受容体遮断
5. シクロオキシゲナーゼ阻害

**正解:** 1

**解説**
解答 １
急性膵炎は、膵臓内で活性化された膵液中のトリプシンなどのタンパク質分解酵素により膵臓細胞が自己消化されることで生じる。ガベキサートは、トリプシンなどのタンパク質分解酵素を阻害することで、急性膵炎に治療効果を発現する。なお、ガベキサートと同様の作用を示す薬物には、ナファモスタット、ウリナスタチンなどがある。
