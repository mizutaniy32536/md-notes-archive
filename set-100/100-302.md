## ◆ 手洗い指導で優先度が高い部位を選ぶシナリオ
260312

**問題**
地区の小学校の校長から担当の学校薬剤師に連絡があり、「適切な手洗い方法を児童に指導して欲しい」との依頼があった。
手洗いが不充分になりやすく、指導の優先度が高い部位はどれか。2つ選べ。

**選択肢**

1. 指先
​
2. 手のひら
​
3. 手の甲
​
4. 指の付け根の間
​
5. 手首

**正解と理由**
- `1. 指先`
- `4. 指の付け根の間`
- **理由の構成**
	- 指先と指の間は構造的に洗い残しが起きやすい部位

**解法の手順（思考の流れ）**
- 選択肢1：指先 = 爪の間含め洗い残しやすい → **○**
- 選択肢2：手のひら = 意識して洗う部位 → ✗
- 選択肢3：手の甲 = 比較的意識して洗う → ✗
- 選択肢4：指の付け根の間 = 構造上洗い残しやすい → **○**
- 選択肢5：手首 = 重要だが指先ほどではない → ✗

**キーポイント**
- 手洗いで洗い残しやすい部位：`指先（爪周囲）`、`指の間`、`親指の付け根`
- 手のひらは最も意識されやすい部位
