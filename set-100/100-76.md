# 第100回 問76

**問題文**
次の薬物のうち、他の薬物との相互作用が原因となった薬害事象を引き起こしたのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-76 を参照

**選択肢**
1. サリドマイド
2. クロロキン
3. キノホルム
4. ソリブジン
5. アミノピリン

**正解:** 4

**解説**
解答 ４
選択肢のうち、他の薬物との相互作用が原因となった薬害事象を引き起こしたのは、ソリブジンである。抗ウイルス薬であるソリブジンと抗がん剤であるフルオロウラシルを併用することにより、フルオロウラシルの代謝が阻害され、血液障害等の重篤な副作用が発生した。
