## ◆ 手洗いで予防可能な感染症を選ぶシナリオ
260312

**問題**
地区の小学校の校長から担当の学校薬剤師に連絡があり、「適切な手洗い方法を児童に指導して欲しい」との依頼があった。
手洗いが、感染防止に有効である可能性の高い感染症はどれか。2つ選べ。

**選択肢**

1. C型肝炎
​
2. デング熱
​
3. インフルエンザ
​
4. 日本脳炎
​
5. O-157感染症

**正解と理由**
- `3. インフルエンザ`
- `5. O-157感染症`

**解法の手順（思考の流れ）**
- 選択肢1：C型肝炎 = `血液`媒介 → 手洗いでは防げない → ✗
- 選択肢2：デング熱 = `蚊`媒介 → 手洗い無効 → ✗
- 選択肢3：インフルエンザ = `飛沫・接触`感染 → 手の付着ウイルス除去に有効 → **○**
- 選択肢4：日本脳炎 = `蚊`媒介 → 手洗い無効 → ✗
- 選択肢5：O-157 = `経口`感染 → 手の付着菌除去に有効 → **○**

**キーポイント**
- 手洗いが有効 = `接触感染`または`経口感染`の病原体
- 手洗いが無効 = `蚊媒介`（デング熱、日本脳炎）、`血液媒介`（C型肝炎）
- インフルエンザ：飛沫感染が主だが、接触感染（手→顔）も重要経路

**関連知識の比較**

| 感染症 | 感染経路 | 手洗い有効性 |
|---|---|---|
| インフルエンザ | 飛沫・接触 | ○ |
| O-157 | 経口 | ○ |
| C型肝炎 | 血液 | × |
| デング熱 | 蚊媒介 | × |
| 日本脳炎 | 蚊媒介 | × |

（一言メモ）
- 蚊媒介（デング・日本脳炎）は手洗いでは防げない。虫除け・蚊帳が対策
