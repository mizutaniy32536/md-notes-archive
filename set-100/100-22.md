# 第100回 問22

**問題文**
放射線に対する感受性が最も高い器官又は組織はどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-22 を参照

**選択肢**
1. 脂肪組織
2. 皮膚
3. リンパ組織
4. 肺
5. 神経組織

**正解:** 3

**解説**
解答 ３
生体細胞の放射線感受性は、その細胞の分裂が盛んであるほど、また未分化であるほど高くなる（ベルゴニー・トリボンドーの法則）。
放射線に対する感受性を以下に示す。
リンパ組織、骨髄＞生殖腺＞小腸、皮膚＞肺＞筋肉、脂肪、神経組織
