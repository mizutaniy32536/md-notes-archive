# 第100回 問75

**問題文**
次の物質の原体のうち、覚醒剤原料として規制されているのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-75 を参照

**選択肢**
1. リゼルギン酸
2. フェニルプロパノールアミン
3. サフロール
4. アンフェタミン
5. 無水酢酸

**正解:** 2

**解説**
解答 ２
選択肢のうち、覚醒剤原料として規制されているのは、フェニルプロパノールアミンの原体である。
フェニルプロパノールアミン（50％以下を含有するものを除く）は、覚醒剤取締法により、覚醒剤原料として規制されている。
なお、リゼルギン酸、サフロール、無水酢酸は、麻薬及び向精神薬取締法において麻薬向精神薬原料として規制されており、アンフェタミンは、覚醒剤取締法において覚醒剤として規制されている。
