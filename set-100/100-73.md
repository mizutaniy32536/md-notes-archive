# 第100回 問73

**問題文**
医療法において、医療提供体制の確保を図るための計画（医療計画）を定めると規定されているのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-73 を参照

**選択肢**
1. 国
2. 都道府県
3. 市町村
4. 医療法人
5. 保健所を設置する市又は特別区

**正解:** 2

**解説**
解答 ２
医療法において、医療提供体制の確保を図るための計画（医療計画）を定めると規定されているのは、都道府県である。
都道府県は、基本方針に即して、かつ、地域の実情に応じて、当該都道府県における医療提供体制の確保を図るための計画を定めるものとする。医療提供体制の基本方針は、厚生労働大臣が定める。
