# 100-259

**問題文**
55歳男性。下記の処方せんを持って来局した。患者は半年前にヘリコバクター・ピロリ菌が陽性と診断され、かかりつけ医にて一次除菌を行ったが不成功であった。今回、二次除菌療法を行うことになった。問259（薬理）薬剤1の作用機序として正しいのはどれか。1つ選べ。

**選択肢**
1. ヒスタミンH2受容体を遮断する。
2. DNAの二本鎖を切断する。
3. リボソームの50Sサブユニットと結合し、タンパク質合成を阻害する。
4. H+,K+-ATPaseを非可逆的に阻害する。
5. ウレアーゼを阻害する。

**正解:** 2

**正解と理由**
問258の薬剤1はメトロニダゾールである。メトロニダゾールは嫌気条件下で還元され、活性中間体を介してDNA鎖を損傷し、二本鎖切断を引き起こす。

**他の選択肢が誤りな理由**
1. H2受容体遮断薬の作用であり、メトロニダゾールの作用ではない。
3. マクロライド系などが示すタンパク質合成阻害作用である。
4. プロトンポンプ阻害薬の作用である。
5. ウレアーゼ阻害はメトロニダゾールの主要作用機序ではない。
