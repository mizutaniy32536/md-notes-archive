# 第100回 問145

**問題文**
「医薬品○○」は、製薬企業Bが製造販売承認を得て、製造所Aが製造する一般用医薬品である。下図は「医薬品○○」の販売の流れを示している。「医薬品○○」を販売するにあたり、医薬品の販売業の許可を得る必要があるものとして、正しいのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-145 を参照

**選択肢**
1. A〜Eのすべて
2. A〜Dのみ
3. CとDとEのみ
4. CとDのみ
5. DとEのみ
6. Dのみ

**正解:** 4

**解説**
解答 ４
業として医薬品を販売することができる者は、薬局開設者または医薬品販売業（店舗販売業、配置販売業、卸売販売業）の許可を受けた者である。ただし、①、②の場合には、医薬品販売業の許可を必要としない。
① 医薬品の製造業者がその製造した医薬品を医薬品の製造販売業者または製造業者に販売するとき
② 医薬品の製造販売業者がその製造した医薬品を薬局開設者または医薬品の製造販売者、製造業者もしくは販売業者に販売するとき
よって、「医薬品○○」を販売するにあたり、医薬品卸Cは医薬品の卸売販売業の許可、薬店Dは医薬品店舗販売業の許可が必要である。
なお、「医薬品○○」が製造所Aから消費者に販売されるまでにA、B、Eで必要な許可を以下に示す。
製造所A：医薬品の製造業の許可
製薬企業B：医薬品の製造販売業の許可
薬局E：薬局開設の許可
