## ◆ 感染症法の分類に関する正誤判断シナリオ
260312

**問題**
「感染症の予防及び感染症の患者に対する医療に関する法律（感染症法）」に関する記述のうち、正しいのはどれか。２つ選べ。

**選択肢**

1. 1類感染症の対象疾患は、すべてウイルスが病因の疾患である。
​
2. 3類感染症の対象疾患は、すべて細菌が病因の疾患である。
​
3. 1～4類感染症は、すべて全数把握対象疾患である。
​
4. 1～4類感染症の患者は、すべて特定業種への就業が制限される。
​
5. 4類及び5類感染症の対象疾患は、いずれも人から人へ直接感染することはない。

**正解と理由**
- `2. 3類感染症の対象疾患は、すべて細菌が病因の疾患である`
- `3. 1～4類感染症は、すべて全数把握対象疾患である`

**解法の手順（思考の流れ）**
- 選択肢1：1類にはペスト（`ペスト菌`＝細菌）が含まれる → すべてウイルスではない → **誤り**
- 選択肢2：3類 = コレラ菌、チフス菌、パラチフスA菌、赤痢菌、腸管出血性大腸菌 → すべて`細菌` → **正しい**
- 選択肢3：1～4類はすべて`全数把握`（5類の一部のみ定点把握） → **正しい**
- 選択肢4：就業制限は`1～3類`まで。4類は就業制限なし → **誤り**
- 選択肢5：5類にインフルエンザ（ヒト→ヒト飛沫感染）が含まれる → **誤り**

**キーポイント**
- `1類`のペスト = 細菌（「すべてウイルス」のひっかけ注意）
- `3類` = 5疾患すべて細菌性（コレラ、腸チフス、パラチフス、細菌性赤痢、腸管出血性大腸菌）
- `全数把握` = 1～4類すべて + 5類の一部
- `就業制限` = 1～3類のみ

**関連知識の比較**

| 類型 | 全数把握 | 就業制限 | 入院勧告 |
|---|---|---|---|
| 1類 | ○ | ○ | ○（強制） |
| 2類 | ○ | ○ | ○ |
| 3類 | ○ | ○ | × |
| 4類 | ○ | × | × |
| 5類 | 一部○ | × | × |

（一言メモ）
- 3類の5疾患は全部細菌。覚え方：「コチパセチョ」（コレラ・チフス・パラチフス・赤痢・腸管出血性大腸菌）
- 就業制限は3類まで！4類からは制限なし
