# 第100回 問83

**問題文**
血漿分画製剤に分類されるのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-83 を参照

**選択肢**
1. 人全血液
2. 新鮮凍結人血漿
3. 加熱人血漿たん白
4. 人赤血球濃厚液
5. 人血小板濃厚液

**正解:** 3

**解説**
解答 ３
血液製剤は、輸血用血液製剤（全血製剤、成分製剤）と血漿分画製剤に分類される。
１ 誤
人全血液は、輸液用血液製剤の全血製剤に分類される。
２ 誤
新鮮凍結人血漿は、輸液用血液製剤の成分製剤に分類される。
３ 正
加熱人血漿たん白は、血漿分画製剤に分類される。
４ 誤
人赤血球濃厚液は、輸血用血液製剤の成分製剤に分類される。
５ 誤
人血小板濃厚液は、輸血用血液製剤の成分製剤に分類される。
