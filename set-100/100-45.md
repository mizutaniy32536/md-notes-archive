# 第100回 問45

**問題文**
プロベネシドの併用によってメトトレキサートの血中からの消失が遅延する主要な原因はどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-45 を参照

**選択肢**
1. 肝代謝の飽和
2. 肝取り込みの阻害
3. 血漿タンパク結合の阻害
4. 脳移行の阻害
5. 腎排泄の阻害

**正解:** 5

**解説**
解答 ５
プロベネシド及びメトトレキサートは、腎臓の近位尿細管に存在する有機アニオン輸送系（OAT）により分泌される。そのため、メトトレキサートにプロベネシドを併用すると、プロベネシドがメトトレキサートの尿細管分泌を競合的に阻害し、メトトレキサートの血中からの消失が遅延することがある。
