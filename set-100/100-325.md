# 第100回 問325

**問題文**
80歳女性の家族より「一昨日、処方せんにより調剤してもらった薬を服薬してから母親の体のふらつきがひどくなり、今朝転倒した。」と薬局に連絡があった。調剤した薬剤師が確認したところ、薬剤量を誤って多く調剤した可能性が疑われた。
問325（法規・制度・倫理）
