# 第100回 問80

**問題文**
ヘルシンキ宣言における倫理的原則にあてはまらないのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-80 を参照

**選択肢**
1. 研究の実施内容を研究計画書に明示すること
2. 研究によって生じるリスクを最小化させるための措置を講じること
3. 被験者のプライバシー及び尊厳を守ること
4. 被験者に研究に関する十分な説明を行うこと
5. 被験者の利益にかかわらず、研究目的の重要性を優先すること

**正解:** 5

**解説**
解答 ５
ヘルシンキ宣言とは、ヒトを対象とする医学研究の倫理的原則のことである。
１ あてはまる
ヒトを対象とする研究の実施内容は、研究計画書（プロトコル）に明示すること
２ あてはまる
研究によって生じるリスクを最小化させるための措置を講じること
３ あてはまる
被験者（患者）の生命、健康、尊厳、自己決定権、およびプライバシーを尊重すること
４ あてはまる
研究前に、被験者に対し研究に関する十分な説明を行い、同意を得ること（研究前に、被験者に対し、インフォームド・コンセントを行うこと）
５ あてはまらない
研究目的の重要性よりも被験者の権利および利益が優先されること
