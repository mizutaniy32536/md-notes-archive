# 第100回 問29

**問題文**
麻薬拮抗性鎮痛薬はどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-29 を参照

**選択肢**
1. フェンタニル
2. モルヒネ
3. ペンタゾシン
4. ペチジン
5. オキシコドン

**正解:** 3

**解説**
解答 ３
麻薬拮抗性鎮痛薬とは、オピオイド受容体に結合することで鎮痛作用を示すが、モルヒネなどの麻薬性鎮痛薬の作用に拮抗するものである。ペンタゾシン（選択肢３）は、麻薬拮抗性鎮痛薬であり、オピオイドκ受容体には刺激薬として、μ受容体には弱い拮抗薬あるいは部分的作動薬として作用するため、単独で投与すると鎮痛作用を示すが、μ受容体の完全刺激薬であるモルヒネなどの麻薬性鎮痛薬と併用した場合、モルヒネの鎮痛作用が拮抗される。
なお、フェンタニル、モルヒネ、ペチジン、オキシコドンは、麻薬性鎮痛薬である。
