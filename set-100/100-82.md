# 第100回 問82

**問題文**
薬剤師が保険調剤の調剤録を作成するタイミングとして最も適切なのはどれか。１つ選べ。ただし、分割調剤の場合を除く。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-82 を参照

**選択肢**
1. 処方せん受け付けた時
2. 処方監査を終了した時
3. 計数・計量調剤を終了した時
4. 調剤薬鑑査を終了した時
5. 処方せんが調剤済となった時

**正解:** 5

**解説**
解答 ５
薬剤師が保険調剤の調剤録を作成するタイミングは、患者に対して薬剤を交付した後（処方せんが調剤済みになった時）が適切である。（分割調剤の場合を除く）
