# 100-255

**問題文**
64歳男性。不安定狭心症のため、PCI（経皮的冠動脈インターベンション）を受けた。その後の治療薬として新たに以下の薬剤が処方された。この患者への説明および指導の内容として、適切なのはどれか。2つ選べ。

**選択肢**
1. アスピリン腸溶錠は、術後の経過が良ければ、一週間程度で中止します。
2. チクロピジン塩酸塩錠は、狭くなった血管を広げる作用があります。
3. 定期的に血液検査を必要とする薬剤が処方されていますので、2週間後に受診してください。
4. 筋肉痛や脱力感が起こることがありますが、一時的なものなので心配ありません。
5. 出血しやすくなるので、歯肉や鼻などからの出血があれば受診してください。

**正解:** 3,5

**解説**
1. 誤 アスピリンは原則として継続投与される。
2. 誤 チクロピジンは血小板凝集抑制薬であり、血管拡張薬ではない。
3. 正 チクロピジンは定期的な血液検査が必要である。
4. 誤 スタチンでは横紋筋融解症に注意が必要であり、筋肉痛や脱力感を軽視してはいけない。
5. 正 抗血小板薬により出血傾向が起こりうるため、異常出血時は受診が必要である。
