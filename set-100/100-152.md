# 100-152

**問題文**
レニン-アンギオテンシン系に作用する薬物に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**
1. アドレナリンβ1受容体遮断薬は、レニン分泌量を減少させる。
2. アルドステロン受容体遮断薬は、尿量を減少させる。
3. アンギオテンシンⅡAT1受容体遮断薬は、アルドステロン分泌量を増加させる。
4. レニン阻害薬は、血中のアンギオテンシンⅠとアンギオテンシンⅡの量を減少させる。
5. アンギオテンシン変換酵素阻害薬は、血中のブラジキニン量を減少させる。

**正解:** 1,4

**解説**
1. 正 β1受容体遮断により傍糸球体細胞からのレニン分泌は減少する。
2. 誤 アルドステロン受容体遮断薬は利尿をもたらし、尿量を増加させる。
3. 誤 AT1受容体遮断薬はアルドステロン分泌を抑制する。
4. 正 レニン阻害でアンギオテンシンI産生が減り、結果としてアンギオテンシンIIも減少する。
5. 誤 ACE阻害薬はブラジキニン分解を抑えるため、ブラジキニン量は増加する。

**キーポイント**
- β1遮断薬はRAAS抑制にもつながる。
- ACE阻害薬はブラジキニン増加、副作用として空咳を連想する。
