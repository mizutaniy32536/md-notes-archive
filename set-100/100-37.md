# 第100回 問37

**問題文**
小腸コレステロールトランスポーター阻害薬はどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-37 を参照

**選択肢**
1. イコサペント酸エチル
2. エゼチミブ
3. コレスチミド
4. シンバスタチン
5. フェノフィブラート

**正解:** 2

**解説**
解答 ２
小腸コレステロールトランスポーター（NPC1L1）阻害薬は、エゼチミブである。エゼチミブは、NPC1L1を阻害することにより、食事性および胆汁性コレステロールの吸収を阻害する。
１ 誤
イコサペント酸エチルは、リポタンパクリパーゼ（LPL）を活性化し、トリグリセリド（TG）の加水分解を促進する。
２ 正
３ 誤
コレスチミドは、陰イオン交換樹脂であり、腸管内で胆汁酸を吸着し、胆汁酸の糞便中への排泄を促進することで、胆汁酸によるコレステロールの吸収を抑制する。
４ 誤
シンバスタチンは、ヒドロキシメチルグルタリルCoA（HMG−CoA）還元酵素を競合的に阻害し、肝臓におけるコレステロール合成を阻害する。
５ 誤
フェノフィブラートは、ペルオキシソーム増殖剤応答性受容体（PPAR）αに作用することで、LPLを活性化し、TGを低下させる。
