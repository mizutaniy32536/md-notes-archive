# 第100回 問123

**問題文**
平成20年に、数種の有害化学物質で汚染された事故米を食用の米と偽って転売する事件が起こった。この事例に見られるように、米は化学物質による汚染が比較的多い食品である。米を汚染する可能性が高い有害化学物質はどれか。２つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-123 を参照

**選択肢**
1. ベンゾ［a］ピレン
2. アフラトキシンB1
3. パツリン
4. パラジクロロベンゼン
5. メタミドホス

**正解:** 2,5

**解説**
解答 ２、５
米を汚染する可能性が高い有害物質には、農薬（メタミドホス、アセタミプリドなど）やかびが産生するアフラトキシンB1などがある。なお、事故米とは、汚染米ともいわれ、政府米（政府が生産者から買い入れ、管理する米）のうち、水につかったり、かびが生えたり、基準以上の農薬が残っていたりして食用にならない米のことである。
１ 誤
ベンゾ［a］ピレンは、ヒトに対して発ガン性を有する多環芳香族炭化水素であり、たばこの煙や鰹ぶし、焼き魚、カモの燻製などから検出されることがある。
２ 正
３ 誤
パツリンは、Penicillium expansum等のカビによって産生されるカビ毒であり、主にリンゴ及びその加工品から検出されることがある。
４ 誤
パラジクロロベンゼンは、衣類の防虫剤などに用いられる。平成20年、日清食品のカップヌードルにパラジクロロベンゼンの混入による食品汚染が問題となった。
５ 正
