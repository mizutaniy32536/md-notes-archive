# 第100回 問42

**問題文**
血液脳脊髄液関門の実体を形成している細胞はどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-42 を参照

**選択肢**
1. 神経細胞
2. 毛細血管内皮細胞
3. 脈絡叢上皮細胞
4. アストロサイト
5. 周皮細胞

**正解:** 3

**解説**
解答 ３
血液脳脊髄液関門は、血液と脳脊髄液の間に存在し、血液から脳脊髄液への薬物の移行を制御している。
この関門では、脈絡叢の上皮細胞どうしが密に接着していることから、この関門の実体を形成している細胞は、脈絡叢上皮細胞であるといえる。
