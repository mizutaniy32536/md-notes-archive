# 第100回 問54

**問題文**
以下の添加剤のうち、崩壊剤として用いられるのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-54 を参照

**選択肢**
1. カルメロースカルシウム
2. ヒプロメロースフタル酸エステル
3. 乳酸・グリコール酸共重合体
4. エチルセルロース
5. ステアリン酸マグネシウム

**正解:** 1

**解説**
解答 １
崩壊剤とは、消化管内での固形製剤の崩壊・分散を促進することを目的とした添加剤のことである。選択肢のうち、崩壊剤として用いられるのは、カルメロースカルシウムである。
１ 正
前記参照
２ 誤
ヒプロメロースフタル酸エステルは、腸溶性コーティング剤皮として用いられる。
３ 誤
乳酸・グリコール酸共重合体は、生分解性高分子であり、放出制御型マイクロカプセル製剤の基剤として用いられる。
４ 誤
エチルセルロースは、徐放性コーティング剤として用いられる。
５ 誤
ステアリン酸マグネシウムは、滑沢剤として用いられる。
