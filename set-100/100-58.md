# 第100回 問58

**問題文**
クロルフェニラミンマレイン酸塩を含有する一般用医薬品の添付文書に記載されている、「突然の高熱、さむけ、のどの痛み等があらわれる。」に該当する重篤な副作用はどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-58 を参照

**選択肢**
1. 無顆粒球症
2. 肝機能障害
3. 間質性肺炎
4. うっ血性心不全
5. ショック（アナフィラキシー）

**正解:** 1

**解説**
解答 １
本問の選択肢のうち、初期症状として、「突然の高温、さむけ、のどの痛み等があらわれる。」のは、無顆粒球症である。無顆粒球症では、顆粒球数が500/µL以下に低下しているため、易感染状態となり上気道感染を発症することにより感冒様症状（発熱、悪寒、咽頭痛など）が現れる。
なお、クロルフェニラミンマレイン酸塩の他に、副作用として無顆粒球症が現れる薬剤として、チアマゾール、サラゾスルファピリジン、H2受容体遮断薬、プロトンポンプ阻害薬、チクロピジンなどが挙げられる。
