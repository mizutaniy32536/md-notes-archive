# 第100回 問33

**問題文**
胃腸管に発現する受容体で、刺激されることで消化管運動を亢進させるのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-33 を参照

**選択肢**
1. オピオイドμ受容体
2. アセチルコリンNM受容体
3. アドレナリンβ2受容体
4. セロトニン5−HT4受容体
5. ドパミンD2受容体

**正解:** 4

**解説**
解答 ４
胃腸管に発現する受容体で、刺激されることで消化管運動を亢進させるのは、セロトニン5−HT4受容体（5−HT4）である。5−HT4受容体が刺激されると、アセチルコリンの遊離が促進し、消化管運動が亢進される。
