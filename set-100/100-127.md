## ◆ 肝炎ウイルス感染症に関する正誤判断シナリオ
260312

**問題**
肝炎ウイルス感染症に関する記述のうち、正しいのはどれか。２つ選べ。

**選択肢**

1. 肝細胞がんによる死亡者の多くは、C型肝炎ウイルスの持続感染者（キャリア）である。
​
2. B型肝炎及びC型肝炎は、輸血によりまん延したことがある。
​
3. A型肝炎ウイルスの持続感染者（キャリア）は、B型肝炎やC型肝炎に比べて多い。
​
4. B型肝炎ウイルスは、失活しやすく感染力が弱いため、医療施設内で感染することはない。
​
5. E型肝炎ウイルスは、主に輸血により感染する。

**正解と理由**
- `1. 肝細胞がんによる死亡者の多くは、C型肝炎ウイルスの持続感染者である`
- `2. B型肝炎及びC型肝炎は、輸血によりまん延したことがある`

**解法の手順（思考の流れ）**
- 選択肢1：肝細胞がんの原因 → C型肝炎が約60%、B型肝炎が約15% → 正しい
- 選択肢2：薬害肝炎（血液製剤）として社会問題化した歴史あり → 正しい
- 選択肢3：A型肝炎はキャリア化`しない`（急性感染のみ） → **誤り**
- 選択肢4：B型肝炎ウイルスは消毒薬に`抵抗性が高い`、針刺し事故で院内感染あり → **誤り**
- 選択肢5：E型肝炎 = `経口感染`（汚染水・ブタ肉の生食）、輸血ではない → **誤り**

**キーポイント**
- `A型肝炎` = キャリア化しない（一過性感染）
- `B型肝炎` = 感染力が強い、消毒薬に抵抗性あり
- `C型肝炎` = 肝細胞がんの最大原因、輸血感染の歴史
- `E型肝炎` = 経口感染（ブタ・シカ肉の生食）

**関連知識の比較**

| 肝炎 | 感染経路 | キャリア化 | 特徴 |
|---|---|---|---|
| A型 | 経口（糞口） | しない | 急性のみ |
| B型 | 血液・体液 | する | 感染力強い |
| C型 | 血液 | する（高率） | 肝がん原因No.1 |
| E型 | 経口（ブタ肉等） | しない | 妊婦で劇症化 |

（一言メモ）
- A型は「キャリア化しない」が超重要ポイント
- E型 = ブタ肉の生食。「輸血」のひっかけに注意
