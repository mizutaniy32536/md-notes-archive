# 第100回 問17

**問題文**
水分活性を低下させることにより食品の腐敗を防止する方法はどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-17 を参照

**選択肢**
1. 冷凍
2. 糖漬
3. 酢漬
4. 加熱
5. 真空包装

**正解:** 2

**解説**
解答 ２
選択肢のうち、水分活性を低下させることにより食品の腐敗を防止する方法は、糖漬である。水分活性とは、食品中の自由水の割合のことであり、微生物は増殖するために自由水を使用する。食品を糖漬すると、糖を溶かすのに自由水が使用され、自由水の割合（水分活性）が低下するため、微生物が増殖しにくくなる。よって、糖漬により食品の腐敗を防止することができる。なお、塩漬によっても同様の効果が得られる。
１ 誤
冷凍は、温度を低下させることで微生物の増殖を抑制する方法である。
２ 正
前記参照
３ 誤
酢漬は、pHを低下させることで微生物の増殖を抑制する方法である。
４ 誤
加熱は、熱により微生物の増殖を抑制する方法である。
５ 誤
真空包装は、酸素を低下させることで好気性微生物の増殖を抑制する方法である。
