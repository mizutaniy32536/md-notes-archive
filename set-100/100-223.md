# 第100回 問223

**問題文**
46歳男性。体重90 kg。足の親指が痛みだしたので、近医を受診したところ、痛風と診断されフェブキソスタット製剤が処方された。
問223（実務）
今回の処方薬との相互作用が問題となるものはどれか。1つ選べ。

**選択肢**
1. アザチオプリン
2. テプレノン
3. イミプラミン塩酸塩
4. タムスロシン塩酸塩
5. ジゴキシン

**正解:** 1

**解説**
フェブキソスタット製剤と相互作用が問題となるのは、アザチオプリンである。
アザチオプリンは、生体内で代謝を受けると6−メルカプトプリンとなり作用を示す。6−メルカプトプリン（6−MP）はキサンチンオキシダーゼにより代謝を受けるため、フェブキソスタットのようなキサンチンオキシダーゼ阻害薬と併用すると、6−MPの血中濃度が上昇し、副作用である骨髄抑制が現れやすくなる。そのため、フェブキソスタットとアザチオプリンや6−MPは併用禁忌である。
