# 第100回 問85

**問題文**
感染制御チームにおける薬剤師の役割として適切でないのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-85 を参照

**選択肢**
1. 病棟ラウンドへの参画
2. 注射薬の無菌調製の推進
3. 使用済針のリキャップの推進
4. 耐性菌などの感染関連情報の収集・提供
5. TDMによる抗菌薬の投与設計

**正解:** 3

**解説**
解答 ３
使用済針のリキャップは、針刺し事故の原因になりやすいため、行わないことは望ましい。
なお、感染制御チーム（ICT：Infection Control Team）における薬剤師の役割として、以下のような項目が挙げられる。
・病棟ラウンドの参画
・注射剤の無菌調製の推進
・針指し事故を防止するため、使用済針のリキャップの禁止の推進
・耐性菌などの感染関連情報の収集・提供
・抗菌薬の適正使用の指導（TDMによる抗菌薬の投与設計の提案など）
・消毒薬の適正使用の指導
・手指衛生（手洗い及び手指消毒など）についての指導
など
