# 第100回 問61

**問題文**
頭蓋内圧亢進の状態において見られる病態・症状として、誤っているのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-61 を参照

**選択肢**
1. 頭痛
2. うっ血乳頭
3. 嘔吐
4. 脳ヘルニア
5. 回転性めまい

**正解:** 5

**解説**
解答 ５
頭蓋内圧亢進により、頭痛、うっ血乳頭（視神経乳頭が発赤、腫脹した状態）、視力障害、嘔吐、脳ヘルニア（脳が頭蓋内圧亢進や浮腫によって押し出された状態）、非回転性めまいなどの病態・症状が現れる。
なお、回転性めまいは、一般に内耳障害などの末梢性のめまいで認められる。
