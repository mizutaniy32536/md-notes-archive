# 第100回 問153

**問題文**
交感神経系に作用する薬物に関する記述のうち、正しいのはどれか。２つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-153 を参照

**選択肢**
1. フェニレフリンは、アドレナリンα1受容体を選択的に刺激して、血管を収縮させる。
2. フェントラミンは、非競合的にアドレナリンα1受容体を遮断して、血圧を下降させる。
3. ラベタロールは、アドレナリンβ1受容体遮断作用があるため、反射性頻脈を引き起こす。
4. ミドドリンは、アドレナリンβ2受容体を刺激することで、子宮平滑筋を弛緩させる。
5. チラミンは、短時間内に反復的に静脈内投与されると、その昇圧作用が次第に弱くなる。

**正解:** 1,5

**解説**
解答 １、５
１ 正
フェニレフリンは、選択的アドレナリンα1受容体刺激薬であり、血管平滑筋収縮作用を有する。
２ 誤
フェントラミンは、競合的にアドレナリンα1受容体を遮断することで、ノルアドレナリンによる血圧上昇作用を抑制する。
３ 誤
ラベタロールは、アドレナリンα・β受容体遮断薬である。α受容体遮断による降圧作用は反射性頻脈を引き起こす可能性があるが、ラベタロールはβ1受容体遮断作用を併せもつため、反射性頻脈を起こしにくい。
４ 誤
ミドドリンは、選択的アドレナリンα1受容体刺激薬であり、血管平滑筋収縮作用を有する。なお、β2受容体を刺激して子宮平滑筋を弛緩させ、切迫早・流産の防止に用いられる薬物には、リトドリンなどがある。
５ 正
チラミンは、間接型交感神経興奮薬であり、シナプス小胞からのノルアドレナリン遊離促進作用を有するため昇圧作用を示す。チラミンなどの間接型交感神経興奮薬を短時間内に反復投与すると、シナプス小胞のノルアドレナリンが枯渇し、その昇圧作用が次第に減弱する。この現象をタキフィラキシーという。
