# 第100回 問87

**問題文**
患者の抱える医療上の問題点を問題ごとに明確化し、患者の立場で解決していくシステム（略号）はどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-87 を参照

**選択肢**
1. PEM
2. PMS
3. SOAP
4. POS
5. POMR

**正解:** 4

**解説**
解答 ４
患者の抱える医療上の問題点を問題ごとに明確化し、患者の立場で解決していくシステムのことを問題志向型システム（Problem Oriented System：POS）という。
１ 誤
PEMは、処方イベントモニタリング（Prescription Event Monitoring）の略号である。PEMとは、医薬品の安全性向上に資するため、医薬品を使用して起こった変化、症状などの出来事を収集し記録することである。
２ 誤
PMSは、市販後調査（Post Marketing Surveillance）の略号である。PMSとは、医薬品や医療機器が販売された後に行われる、品質、有効性および安全性の確保を図るための調査のことである。
３ 誤
SOAPとは、経過記録をS（Subjective data：主観的事項）、O（Objective data：客観的事項）、A（Assessment又はAnalysis：評価）、P（Plan：計画）に分けて記載することである。
４ 正
５ 誤
POMRは、問題志向型診療録（Problem Oriented Medical Record）の略号である。POMRとは、POSに沿って、作成される診療録のことである。
