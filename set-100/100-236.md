## ◆ 膣カンジダ症OTC薬販売時に確認すべき事項を判断するシナリオ
260312

**問題**
22歳女性。数日前より下腹部に違和感があり、外陰部のかゆみに加え、おりものに異常がみられた。テレビで膣カンジダ症に使用する薬が発売されたことを知って来局し、薬剤師に相談した。
薬局薬剤師が、膣カンジダ症に使用する薬を販売するにあたり必ず確認しなければならないのはどれか。２つ選べ。

**選択肢**

1. 以前に医師から膣カンジダ症の診断・治療を受けたことがあるか。
​
2. 喫煙したことがあるか。
​
3. 高血圧症と診断されたことがあるか。
​
4. 現在の体重はどのくらいか。
​
5. 妊娠している可能性があるか。

**正解と理由**
- `1. 以前に医師から膣カンジダ症の診断・治療を受けたことがあるか`
- `5. 妊娠している可能性があるか`

**解法の手順（思考の流れ）**
- 選択肢1：OTC薬は`再発例`のみ使用可。初発なら受診が必要 → **必須確認** → 正しい
- 選択肢2：喫煙歴は膣カンジダ症・OTC薬使用と直接関連なし → 不要
- 選択肢3：高血圧は膣カンジダ症OTC薬に影響しない → 不要
- 選択肢4：坐薬は固定量なので体重不要 → 不要
- 選択肢5：妊娠中はOTC薬使用が`禁忌` → **必須確認** → 正しい

**キーポイント**
- 膣カンジダ症OTC薬（クロトリマゾール等） = `再発例のみ`使用可
- 初発 → 他疾患との鑑別が必要 → 受診勧奨
- 妊婦 → 禁忌 → 必ず確認

（一言メモ）
- OTC = 再発のみ。初めての人には「まず病院へ」
- 妊娠確認は薬局業務の基本中の基本
