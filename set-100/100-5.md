# 第100回 問5

**問題文**
0.10 mol/L塩酸水溶液のpHとして最も近い値はどれか。１つ選べ。ただし、塩酸は完全に解離するものとする。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-5 を参照

**選択肢**
1. －0.10
2. 0.00
3. 0.10
4. 1.0
5. 2.0

**正解:** 4

**解説**
解答 ４
pHは以下の式より求めることができる。
pH＝－log[H＋]
塩酸は水中で完全に解離するため、0.10 mol/L塩酸水溶液中では、[H＋]は、0.10 mol/L（10－1 mol/L）存在すると考えられる。そのため、0.10 mol/L塩酸水溶液のpHは1.0（pH＝－log[10－1]＝1）となる。
