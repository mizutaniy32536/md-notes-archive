## ◆ 予防接種法に基づく定期予防接種の正誤判断シナリオ
260312

**問題**
予防接種法に基づく定期予防接種に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**

1. 学校内での集団感染を防ぐため、インフルエンザワクチンは6歳で接種する。
​
2. ワクチン接種により起こる痛み、腫れ、発赤等の軽度な副反応は、完全には防ぐことができない。
​
3. 麻しん及び風しんは、中学校就学以降に感染しやすいため、そのワクチンは11〜12歳で接種する。
​
4. 乳児や小児の間で流行する感染症の定期予防接種は、母子免疫が消失する前の生後早い時期に設定されている。
​
5. BCGワクチンは、予防効果を高めるため1歳と5歳で接種する。

**正解と理由**
- `2. ワクチン接種の軽度な副反応は完全には防げない`
- `4. 定期予防接種は母子免疫が消失する前の生後早い時期に設定されている`

**解法の手順（思考の流れ）**
- 選択肢1：インフルエンザの定期接種対象は`65歳以上`（二類疾病）。6歳は対象外 → **誤り**
- 選択肢2：ワクチン接種の局所反応（痛み・腫れ・発赤）は完全回避不可能 → **正しい**
- 選択肢3：MRワクチンは`1歳〜2歳`と`小学校入学前`の2回。11〜12歳ではない → **誤り**
- 選択肢4：母子免疫消失前に免疫を獲得させるため、生後早い時期に接種 → **正しい**
- 選択肢5：BCGは`生後1歳未満に1回`のみ → **誤り**

**キーポイント**
- インフルエンザの`定期接種`は高齢者のみ（小児は`任意接種`）
- MRワクチン = 1期（1〜2歳）、2期（小学校入学前）
- BCG = 生後1歳未満に1回のみ

（一言メモ）
- 「インフルエンザ＝定期接種（小児）」は**ひっかけ**。小児は任意接種！
- BCGは1回だけ。2回接種するのはMRワクチン
