## ◆ 母乳を介した垂直感染対策が必要な病原体を選ぶシナリオ
260312

**問題**
完全人工栄養を使用するなど、母乳を介した垂直感染を防ぐ対策がなされる病原体はどれか。１つ選べ。

**選択肢**

1. C型肝炎ウイルス
​
2. 梅毒トレポネーマ
​
3. 単純ヘルペスウイルス
​
4. ヒトT細胞白血病ウイルス
​
5. 風しんウイルス

**正解と理由**
- `4. ヒトT細胞白血病ウイルス`
- **理由の構成**
	- HTLV-1は`母乳`を介した垂直感染が主要経路
	- 完全人工栄養に切り替えることで感染をほぼ防げる

**解法の手順（思考の流れ）**
- 選択肢1：C型肝炎ウイルス → 主に`血液`（産道感染あり）、母乳対策は行わない → ✗
- 選択肢2：梅毒トレポネーマ → 主に`胎盤・産道`、母乳対策は行わない → ✗
- 選択肢3：単純ヘルペスウイルス → 主に`産道`、母乳対策は行わない → ✗
- 選択肢4：HTLV-1 → 主に`母乳`、完全人工栄養で予防 → **○**
- 選択肢5：風しんウイルス → 主に`胎盤`（先天性風しん症候群）、母乳対策は行わない → ✗

**キーポイント**
- `HTLV-1`（ヒトT細胞白血病ウイルス）の3大感染経路：母乳、性交、輸血
- 母乳感染が最重要 → `完全人工栄養`で予防
- 他の垂直感染病原体は胎盤・産道経由が主

**関連知識の比較**

| 病原体 | 主な垂直感染経路 | 母乳対策 |
|---|---|---|
| HTLV-1 | **母乳** | ○（人工栄養） |
| C型肝炎ウイルス | 産道（血液） | × |
| 梅毒トレポネーマ | 胎盤 | × |
| 風しんウイルス | 胎盤 | × |
| 単純ヘルペス | 産道 | × |

（一言メモ）
- 「母乳」と聞いたら即HTLV-1。これ一択
- HTLV-1は九州・沖縄に多い（地域偏在）
