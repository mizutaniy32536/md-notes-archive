# 100-254

**問題文**
64歳男性。不安定狭心症のため、PCI（経皮的冠動脈インターベンション）を受けた。その後の治療薬として新たに以下の薬剤が処方された。
処方された薬剤に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**
1. 処方された用量のアスピリンは、血管内皮細胞のプロスタグランジンI2生成よりも、血小板のトロンボキサンA2生成をより強く阻害する。
2. 低用量アスピリンは痛み止めの作用を期待して用いられる。
3. アスピリンを低用量（100〜300 mg/日）で用いた場合、血管内皮細胞のプロスタグランジンI2の生成も阻害されるため、血小板凝集抑制作用が低下する。
4. チクロピジンの副作用として、血栓性血小板減少性紫斑病、無顆粒球症及び重篤な肝障害がある。
5. ロスバスタチンは、HMG-CoAの生合成を阻害し、血清中のLDL量を減少させる。

**正解:** 1,4

**解説**
1. 正 低用量アスピリンは血小板 COX-1 をより強く抑制し、トロンボキサンA2産生を抑える。
2. 誤 低用量アスピリンは主に抗血栓目的で用いられる。
3. 誤 低用量では血管内皮細胞側の作用は相対的に小さく、血小板凝集抑制作用が優位となる。
4. 正 チクロピジンには TTP、無顆粒球症、重篤な肝障害などの副作用がある。
5. 誤 ロスバスタチンは HMG-CoA 還元酵素を阻害する。
