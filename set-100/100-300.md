# 第100回 問300

**問題文**
54歳女性。数年前から関節リウマチを患い、整形外科でメトトレキサートカプセルによる治療を受けていた。患者が処方せんを薬局に持参した際、「今までの薬だけでは改善しないので、次回から飲み薬に加えて点滴する薬を新しく追加すると医師から言われた。」と薬剤師に話した。
問300（病態・薬物治療）
この患者に新しく追加される点滴薬として、考えられるのはどれか。2つ選べ。

**選択肢**
1. オマリズマブ
2. アダリムマブ
3. リツキシマブ
4. インフリキシマブ
5. セツキシマブ

**解説**
問300　解答　解なし
選択肢の中で関節リウマチ患者に適応をもつ薬物としては、アダリムマブとインフリキシマブが該当する。このうち、点滴薬として用いられる薬物はインフリキシマブのみで、アダリムマブは皮下注射でのみ使用されることから、本設問では正答となる選択肢がインフリキシマブ1つのみであるため、国家試験では「解なし」として扱われた。なお、オマリズマブは気管支喘息の治療に、リツキシマブはCD20陽性のB細胞性非ホジキンリンパ腫の治療等に、セツキシマブはEGFR（上皮増殖因子受容体）陽性の治癒切除不能な進行・再発の結腸・直腸がんの治療等に用いられる。
