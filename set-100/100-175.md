# 第100回 問175

**問題文**
高分子及びその溶液に関する記述のうち、正しいのはどれか。２つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-175 を参照

**選択肢**
1. 線状高分子は、良溶媒中で収縮してコイル形状となる。
2. マクロゴール20000（分子量20,000のポリエチレングリコール）は、室温で水に不溶である。
3. 毛細管粘度計は、非ニュートン流体の粘度測定に適する。
4. 高分子溶液の極限粘度から、高分子の平均分子量を求めることができる。
5. Voigt粘弾性の力学的モデルでは、応力一定のとき、ひずみは時間と共に増大し、一定の値に収束する。

**正解:** 4,5

**解説**
解答 ４、５
１ 誤
線状高分子は、良溶媒（高分子と親和性が高い溶媒）中で、広がり線状となって存在し、貧溶媒（高分子と親和性が低い溶媒）中では、収縮してコイル形状で存在する。
２ 誤
マクロゴールは、分子量により状態が異なるが、水溶性であるため、分子量に関係なく水に溶解する。なお、分子量が1000より小さいマクロゴールは液状、1000より大きいマクロゴールは固形状で存在するため、マクロゴール20000（分子量20,000のポリエチレングリコール）は、固形状で存在する。
３ 誤
毛細管粘度計は、ニュートン流体の粘度測定に適しているが、非ニュートン流体の粘度測定には適していない。
４ 正
極限粘度とは、高分子希薄溶液の粘度のことであり、極限粘度を用いて高分子の平均分子量を求めることが可能である。
５ 正
Voigt（フォークト）粘弾性の力学的モデルとは、バネとダッシュポットを並列に結合したモデルである。Voigt粘弾性の力学的モデルでは、応力一定のとき、ひずみは時間と共に増大し、一定の値に収束する。
