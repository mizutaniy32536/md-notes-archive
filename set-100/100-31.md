# 第100回 問31

**問題文**
心室筋の活動電位を下図の実線から破線へ変化させるのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-31 を参照

**選択肢**
1. プロパフェノン
2. メキシレチン
3. プロカインアミド
4. シベンゾリン
5. アミオダロン

**正解:** 2

**解説**
解答 ２
実線から破線への変化では、活動電位の脱分極相（0相）の抑制と活動電位持続時間（APD）の短縮が認められる。このことから、心室筋の活動電位を実線から破線へ変化させる薬物は、Vaughan Williams 分類でⅠb群に属する薬物（メキシレチン、リドカインなど）であると考えられる。
１ 誤
プロパフェノンは、Vaughan Williams 分類でⅠc群に分類され、APDを変化させない。
２ 正
３ 誤
プロカインアミドやシベンゾリンは、Vaughan Williams 分類でⅠa群に分類され、APDを延長する。
４ 誤
解説３参照
５ 誤
アミオダロンは、Vaughan Williams 分類でⅢ群に分類され、APDを延長する。
