# 第100回 問263

**問題文**
66歳男性。最近、腰痛が原因で、寝付きも良くないため、整形外科を受診した。骨粗しょう症と診断され、処方せんを持って薬局を訪れた。

問263（実務）
薬剤師はこの処方に関して医師へ疑義照会を行った。疑義が生じた薬剤として最も可能性が高いのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-263 を参照

**選択肢**
1. ラロキシフェン塩酸塩錠
2. アルファカルシドールカプセル
3. L−アスパラギン酸カルシウム錠
4. メナテトレノンカプセル
5. セレコキシブ錠

**正解:** 1

**解説**
ラロキシフェン塩酸塩錠は、閉経後骨粗鬆症の治療に用いられるため、女性に用いられる。男性（本患者：66歳男性）に処方されている場合には、疑義照会が必要となる。
