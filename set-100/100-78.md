# 第100回 問78

**問題文**
国民医療費に関する記述のうち、正しいのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-78 を参照

**選択肢**
1. 一般用医薬品の購入費用は含まれない。
2. 財源の80％以上は、保険料である。
3. 薬局調剤医療費は、近年横ばい傾向にある。
4. 薬剤料が占める割合は、50％を超えている。
5. 国民所得に占める割合は、1％以下である。

**正解:** 1

**解説**
解答 １
国民医療費とは、医療機関などにおける疾病の治療に要する費用を中心に推計したものである。
診療費・調剤費・入院時食事療養費・訪問看護療養費の他、健康保健等で支給される移送費などは、国民医療費に含まれ、正常な妊娠や分娩などに要する費用、健康の維持・増進を目的とした健康診断、予防接種などに要する費用、一般用医薬品の購入費用などは、国民医療費に含まれない。
平成24年度における国民医療費、国民医療費の財源、薬局調剤医療費、薬剤料の割合、国民所得に占める医療費の割合を以下に示す。
・国民医療費 約39兆2117億円
・国民医療費の財源 保険料：約48.8％、公費：約38.6％、その他：約12.6％
・薬局調剤医療費：約6兆7105億円（近年増加傾向）
・薬剤料の割合：約35.０％
・国民所得に占める割合：約11.2％
