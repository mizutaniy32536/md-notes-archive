# 第100回 問232

**問題文**
41歳男性。山歩きの時に採取したきのこや山菜を、おひたしにしてその日の夕食で食べたところ、1時間後に口唇のしびれが出現した。症状が次第に悪化し、激しい嘔吐、四肢のしびれ、呼吸困難を起こしたため、救急搬送された。病院到着時に重篤な心室性不整脈が出現したため、アミオダロン塩酸塩注射液が投与された。
問232（実務）
アミオダロン塩酸塩注射液を使用する際の注意事項として、正しいのはどれか。2つ選べ。

**選択肢**
1. 希釈する場合は、生理食塩液を用いる。
2. 点滴静注する場合は、滴下型の持続注入ポンプを使用する。
3. 投与開始後は、心電図の連続監視下で患者の状態を把握する。
4. 投与開始後は、肝機能モニタリングを実施する。

**正解:** 3,4

**解説**
1　誤
アミオダロン塩酸塩注射液を生理食塩水で希釈すると、塩析が起こり、沈殿が生じることがあるため、アミオダロン塩酸塩注射液は、ブドウ糖で希釈して用いる。
2　誤
アミオダロン塩酸塩注射液を投与する際、滴下型の注入ポンプを用いると、溶液特性の変化により、液滴サイズが縮小することがあり、過少投与になるおそれがある。そのため、アミオダロン塩酸塩注射液は、容量型の持続ポンプを用いて投与する。
3　正
添付文書の警告に、「本剤では新たな不整脈や不整脈の増悪等を含む重篤な心障害が報告されており、ときに致死的な場合もあるので、CCU、ICU等で心電図及び血圧の連続監視下で使用すること。」と記載されている。
4　正
添付文書の警告に、「本剤投与後24時間以内に重篤な肝機能障害が生じ、肝不全や死亡に至る場合もあるので、患者の状態を慎重に観察するなど、十分に注意すること。」と記載されている。
