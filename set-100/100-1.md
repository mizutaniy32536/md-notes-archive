# 第100回 問1

**問題文**
次に示す単位のうち、SI基本単位でないのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-1 を参照

**選択肢**
1. m（メートル）
2. kg（キログラム）
3. J（ジュール）
4. K（ケルビン）
5. s（秒）

**正解:** 3

**解説**
解答 ３
国際単位系SIには、SI基本単位とSI誘導（組立）単位がある。
SI基本単位でないのは、SI誘導（組立）単位であるJ（ジュール：仕事量、エネルギー）である。なお、SI基本単位には、m（メートル：長さ）、kg（キログラム：質量）、K（ケルビン：温度）、s（秒：時間）、cd（カンデラ：光度）、A（アンペア：電流）、mol（モル：物質量）がある。
