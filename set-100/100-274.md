# 100-274

**問題文**
院内のレジメンを管理する委員会において、制吐薬として用いる薬物について薬剤師が説明した。「この薬物は、5−HT3受容体拮抗薬です。他の5−HT3受容体拮抗薬に比べて消失半減期が約42時間と長く、そのため遅発期の悪心・嘔吐にも効果があります。」薬剤師が説明した薬物はどれか。1つ選べ。

**選択肢**
1. アザセトロン塩酸塩
2. オンダンセトロン塩酸塩水和物
3. グラニセトロン塩酸塩
4. トロピセトロン塩酸塩
5. パロノセトロン塩酸塩

**正解:** 5

**解説**
解答　5

消失半減期が長く、遅発期の悪心・嘔吐にも有効な第二世代5-HT3受容体拮抗薬はパロノセトロンである。
