# 第100回 問121

**問題文**
エネルギー代謝に関する記述のうち、正しいのはどれか。２つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-121 を参照

**選択肢**
1. 非タンパク質呼吸商の値から、脂質と糖質の燃焼割合が推定できる。
2. 基礎代謝量は、安静時エネルギー消費量とも呼ばれる。
3. 糖質、脂質、タンパク質の物理的燃焼価を四捨五入して得られた整数値は、Atwater係数と呼ばれる。
4. 基礎代謝基準値は、年齢や性別にかかわらず一定である。
5. 成人の推定エネルギー必要量は、基礎代謝量に身体活動レベルを乗じて算出される。

**正解:** 1,5

**解説**
解答 １、５
１ 正
非タンパク質呼吸商とは、以下の式で表される。
非タンパク質呼吸商より、脂質と糖質の燃焼割合を推定することができる。
なお、非タンパク質呼吸商が1.0に近いと炭水化物の燃焼割合が多く、0.7に近いと脂質の燃焼割合が多い。
２ 誤
基礎代謝量とは、生きていくために必要な1日の最小限のエネルギー量であり、睡眠時エネルギー消費量（睡眠時代謝量）と同じとされる。
なお、安静時エネルギー消費量（安静時代謝量）は、仰臥位あるいは座位で、安静にしている状態で消費されるエネルギー量のことであり、基礎代謝量よりも高値を示す。
３ 誤
Atwater係数（生物的燃焼値）とは、糖質、脂質、タンパク質1 gを酸素下で燃焼させた熱量（物理的燃焼値）に、吸収率を考慮にいれたエネルギー量のことである。
４ 誤
基礎代謝基準値は、単位体重当たりの基礎代謝量のことであり、年齢や性別によって異なる。
５ 正
推定エネルギー必要量とは、エネルギーの過剰と不足のリスクが最も少なくなる食事摂取量のことであり、推定エネルギー必要量＝基礎代謝量×身体活動レベルで表される。
