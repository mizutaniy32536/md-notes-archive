# 第100回 問67

**問題文**
仮説検定における第一種の過誤はどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-67 を参照

**選択肢**
1. 誤った統計手法で対立仮説を棄却する過誤
2. 棄却すべきでない対立仮説を誤って棄却する過誤
3. 棄却すべきでない帰無仮説を誤って棄却する過誤
4. 棄却すべき対立仮説を棄却し損なう過誤
5. 棄却すべき帰無仮説を棄却し損なう過誤

**正解:** 3

**解説**
解答 ３
仮説検定を行う際には、判定の過誤が生じることがある。判定の過誤には、第一種の過誤と第二種の過誤がある。第一種の過誤とは、本来、2群間に差がなく棄却すべきでない帰無仮説を誤って棄却する過誤のこと（選択肢３）であり、第二種の過誤とは、本来、2群間に差がある棄却すべき帰無仮説を棄却し損なう過誤のこと（選択肢５）である。
