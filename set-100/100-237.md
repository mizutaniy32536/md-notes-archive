## ◆ 膣カンジダ症に関する正しい記述を選ぶシナリオ
260312

**問題**
22歳女性。数日前より下腹部に違和感があり、外陰部のかゆみに加え、おりものに異常がみられた。
膣カンジダ症に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**

1. カンジダ属菌は常在真菌であり、日和見感染を引き起こす。
​
2. 性交経験がない女性にも発症することがある。
​
3. 感染症法では5類感染症に分類される。
​
4. 女性から男性へのカンジダ属菌の感染はない。

**正解と理由**
- `1. カンジダ属菌は常在真菌であり、日和見感染を引き起こす`
- `2. 性交経験がない女性にも発症することがある`

**解法の手順（思考の流れ）**
- 選択肢1：カンジダ = 体表の`常在菌`、免疫低下時に日和見感染 → **正しい**
- 選択肢2：常在菌なので性交経験`なくても`発症する → **正しい**
- 選択肢3：膣カンジダ症は感染症法の`対象外` → **誤り**
- 選択肢4：女性→男性への感染は`起こりうる` → **誤り**

**キーポイント**
- カンジダ属菌 = `常在真菌`（正常な体にも存在）
- `日和見感染`：免疫低下（抗菌薬使用、ステロイド、糖尿病等）で発症
- 性感染症ではなく、常在菌の増殖による → 性交未経験でも発症
- 感染症法の分類対象には含まれない

（一言メモ）
- カンジダ = 常在菌 = 性交経験不問。この2つがセット
- 感染症法の対象外であることも頻出ポイント
