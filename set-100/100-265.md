# 100-265

**問題文**
ダルベポエチンアルファの薬理作用および副作用に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**
1. 単球および単球系前駆細胞に作用して、増殖を促進する。
2. エリスロポエチン（EPO）の EPO 受容体に対する親和性を高める。
3. エポエチンアルファと比較して、持続的な赤血球造血作用を示す。
4. 血圧低下を起こしやすい。
5. 血液粘稠度が上昇し、血栓塞栓症を誘発するおそれがある。

**正解:** 3,5

**解説**
1. 誤 ダルベポエチンアルファは赤芽球系前駆細胞に作用する。
2. 誤 EPO 受容体への親和性を高める薬ではなく、受容体へ直接結合して作用する。
3. 正 糖鎖修飾により半減期が延長され、エポエチンアルファより持続的な赤血球造血作用を示す。
4. 誤 副作用として問題となるのは高血圧であり、低血圧ではない。
5. 正 血液粘稠度上昇に伴い、血栓塞栓症を起こすおそれがある。
