# 100-32

**問題文**
カリウム保持性利尿薬はどれか。1つ選べ。

**選択肢**
1. スピロノラクトン
2. ブメタニド
3. アセタゾラミド
4. D-マンニトール
5. メフルシド

**正解:** 1

**解説**
スピロノラクトンはアルドステロン受容体拮抗薬で、集合管でNa再吸収を抑えつつK排泄を抑制するカリウム保持性利尿薬である。

- 2: ブメタニドはループ利尿薬
- 3: アセタゾラミドは炭酸脱水酵素阻害薬
- 4: D-マンニトールは浸透圧利尿薬
- 5: メフルシドはチアジド系類似薬
