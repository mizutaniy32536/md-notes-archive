# 第100回 問89

**問題文**
注射剤を混合する時に、配合変化の有無を調べる情報源として、最も適切なのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 100-89 を参照

**選択肢**
1. 日本薬局方
2. 医療用医薬品添付文書
3. 医療用医薬品品質情報集
4. 医薬品インタビューフォーム
5. 保険薬事典

**正解:** 4

**解説**
解答 ４
注射剤を混合する時に、配合変化の有無を調べる情報源として、医薬品インタビューフォーム（IF）が用いられる。IFとは、添付文書の情報を補完するために当該医薬品の製造企業が作成、提供している資料である。注射剤の配合変化の有無については、IFの「他剤との配合変化（物理化学的変化）」に記載されている。
