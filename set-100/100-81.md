# 第100回 問81

**問題文**
EBMの実践手順で、 に入る適当な語句はどれか。１つ選べ。
患者の問題の定式化 → 情報の収集 → → 患者への適用

**選択肢**
1. 患者満足度の推察

3. 治療アウトカムの設定
4. 医療者の臨床経験

**正解:** 2

**解説**
解答 ２
EBMとは、Evidence−Based−Medicine（訳：根拠に基づく医療）のことであり、患者にとって最適な医療を行うために、曖昧な経験や直感に頼らず、根拠に基づいて医療・治療を選択し実践していくことである。
EBMは、通常、以下の手順で実践される。
手順1 患者情報の定式化（患者の問題を明らかにする）
手順2 情報の収集（問題解決のための情報を収集する）
手順3 情報の批判的吟味（手順2で得られた情報を吟味する）
手順4 患者への適用（手順3で情報を吟味した結果、患者の問題点を解決できそうなものを適用する）
手順5 手順1〜4の評価（手順1〜4で行った行為が適切であったか評価する）
