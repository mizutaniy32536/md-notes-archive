# 108-310

【シナリオ】
75歳男性。糖尿病及び糖尿病性腎症で在宅療養にて腹膜透析を行い、薬物治療中。処方1はダルベポエチンアルファ（遺伝子組換え）で、ラベルに「生物」と記載。

**患者より、処方1の薬のラベルに記載されている「生物」について質問があった。薬剤師の説明として、正しいのはどれか。1つ選べ。**

1. 感染症のリスクがあるため、処方1の薬の使用に際し、患者からの文書による同意が必要となる。
2. 処方1の薬は、植物由来である。
3. 処方1の薬は、疾病の治療を目的として、人の細胞に導入され、体内で発現する遺伝子を含有するものである。
4. 処方1の薬は、製薬企業に感染症定期報告を義務付けるなど感染症に関する対策が強化されている。
5. この表示は、薬の品質に問題がないことを示す認証マークである。

## 正解
**4. 製薬企業に感染症定期報告を義務付けるなど感染症に関する対策が強化されている。**

## 解説

「生物」ラベル＝**生物由来製品**の表示。

| 選択肢 | 正誤 | 理由 |
|---|---|---|
| 1. 文書による同意が必要 | ❌ | 文書同意が必要なのは「**特定**生物由来製品」。生物由来製品は不要 |
| 2. 植物由来 | ❌ | ダルベポエチンアルファは動物細胞培養で製造される遺伝子組換えタンパク質 |
| 3. 遺伝子治療用医薬品 | ❌ | 遺伝子治療薬の説明。ダルベポエチンは遺伝子組換え技術で「作られた」タンパク質であり、遺伝子治療薬ではない |
| 4. 感染症定期報告の義務 | ⭕ | 生物由来製品は感染症対策が強化され、企業に感染症定期報告が義務付けられている |
| 5. 品質認証マーク | ❌ | そのような制度はない |

### 生物由来製品 vs 特定生物由来製品
| | 生物由来製品 | 特定生物由来製品 |
|---|---|---|
| 表示 | 「生物」 | 「特生物」 |
| 文書同意 | 不要 | **必要** |
| 感染症定期報告 | 義務あり | 義務あり |
| 使用記録保存 | 不要 | **20年間** |
