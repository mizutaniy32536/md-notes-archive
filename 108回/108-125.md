# 108-125

下表は、母子感染する疾患について、その病原体及び主な感染経路を示したものである。正しい組合せはどれか。**2つ選べ。**

| 疾患 | 病原体 | 主な感染経路 |
|---|---|---|
| 1. 風しん | 風疹ウイルス | 胎盤を介す |
| 2. 淋菌感染症 | 淋菌 | 胎盤を介す |
| 3. 成人T細胞白血病 | HTLV-1 | 母乳を介す |
| 4. B型肝炎 | B型肝炎ウイルス | 母乳を介す |
| 5. 風疹 | サイトメガロウイルス | 胎盤を介す |

## 正解
**1と3**

## 解説

### 選択肢1 ⭕ 風しん（風疹）

**病原体：風疹ウイルス（rubella virus）**
**感染経路：胎盤を介す**

- 妊娠初期（特に妊娠20週まで）の母体感染が危険
- **先天性風疹症候群（CRS: Congenital Rubella Syndrome）**を引き起こす

**先天性風疹症候群の3大症状：**
1. 先天性心疾患（動脈管開存症など）
2. 難聴・聴覚障害
3. 白内障・緑内障

**予防：**
- 妊娠前に風疹ワクチン接種（妊娠中は接種不可）
- 妊婦は風疹患者との接触を避ける

### 選択肢2 ❌ 淋菌感染症

**病原体：淋菌（*Neisseria gonorrhoeae*）- 正しい**
**感染経路：胎盤ではなく「産道を介す」- 誤り**

- 分娩時に産道で新生児に感染
- **新生児結膜炎（淋菌性眼炎）**を引き起こす
- 放置すると失明の危険
- **予防：** 出生直後に抗菌点眼薬（エリスロマイシン眼軟膏など）

### 選択肢3 ⭕ 成人T細胞白血病（ATL）

**病原体：HTLV-1（ヒトTリンパ球向性ウイルス1型）- 正しい**
**感染経路：主に母乳を介す - 正しい**

- 母乳中のリンパ球にウイルスが含まれる
- 授乳によって乳児に感染
- キャリアの約5%が数十年後にATL（成人T細胞白血病）を発症
- **予防：** 感染母体は人工栄養（粉ミルク）を推奨

**HTLV-1の感染経路：**
- 母子感染（母乳）：最も重要
- 性行為感染
- 輸血感染（現在はスクリーニングで防止）

### 選択肢4 ❌ B型肝炎

**病原体：B型肝炎ウイルス（HBV）- 正しい**
**感染経路：主に「産道を介す」（母乳ではない）- 誤り**

- 分娩時の血液接触で感染（産道感染）
- 垂直感染率：母体がHBe抗原陽性の場合90%以上
- **予防：** 出生直後にHBワクチン＋抗HBs免疫グロブリン投与

※母乳感染も理論上ありうるが、主な経路は産道

### 選択肢5 ❌ 風疹の病原体

**病原体が間違い：**
- 風疹 → 風疹ウイルス（rubella virus）
- サイトメガロウイルス（CMV） → 別のウイルス

サイトメガロウイルスも胎盤感染して先天性感染症を起こすが、風疹の原因ではない。

## 母子感染の経路まとめ

### 胎盤感染（経胎盤感染）
妊娠中に胎盤を通じて胎児に感染
- **風疹** - 先天性風疹症候群
- **サイトメガロウイルス（CMV）** - 小頭症、難聴
- **梅毒** - 先天梅毒
- **トキソプラズマ** - 水頭症、脈絡網膜炎
- **HIV**
- **水痘・帯状疱疹ウイルス**

### 産道感染（分娩時感染）
分娩時に産道で感染
- **B型肝炎ウイルス（HBV）**
- **C型肝炎ウイルス（HCV）**
- **淋菌** - 新生児結膜炎
- **性器ヘルペスウイルス（HSV-2）**
- **クラミジア** - 新生児結膜炎、肺炎
- **HIV**
- **GBS（B群溶血性連鎖球菌）** - 新生児髄膜炎・敗血症

### 母乳感染
授乳によって感染
- **HTLV-1** - 成人T細胞白血病
- **HIV**
- **サイトメガロウイルス（CMV）**
