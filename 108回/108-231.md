# 108-231

（108-230と連問）この患者に用いる解毒剤として適切なのはどれか。2つ選べ。

1. プラリドキシムヨウ化メチル（PAM）
2. 硫酸アトロピン
3. ヒドロキソコバラミン
4. 亜硝酸アミル
5. ジメルカプロール

## 正解
**3, 4**

## 解説

中毒-解毒剤の対応：
- シアン → ヒドロキソコバラミン、亜硝酸アミル、チオ硫酸Na
- 有機リン → PAM、アトロピン
- 重金属 → ジメルカプロール（BAL）

シアン中毒の解毒メカニズム：
- ヒドロキソコバラミン：シアンと結合→シアノコバラミン（無毒）として排泄
- 亜硝酸アミル：MetHb生成→シアンをMetHbに結合させて解毒
