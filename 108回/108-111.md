## ◆ 生体内の物質移動における輸送機構の正誤を判断するシナリオ

260308

**問題**
- 生体内の物質移動に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**

1. 一酸化窒素は、促進拡散により血管内皮細胞から移動する。
​
2. 水素イオンは、受動輸送により胃壁細胞から分泌される。
​
3. 原尿中のグルコースは、Na⁺との共輸送により腎臓の近位尿細管上皮細胞に取り込まれる。
​
4. 血中のグルコースの大部分は、エンドサイトーシスにより脂肪細胞に取り込まれる。
​
5. ノルアドレナリンは、膜動輸送であるエキソサイトーシスにより、交感神経終末から分泌される。

**正解と理由**
- `3. 原尿中のグルコースは、Na⁺との共輸送により腎臓の近位尿細管上皮細胞に取り込まれる。` と `5. ノルアドレナリンは、膜動輸送であるエキソサイトーシスにより、交感神経終末から分泌される。`
- **理由の構成**
	- **問題の整理**：生体内の5つの物質移動について、輸送機構の記述の正誤を判断する問題
	- **薬学的知識・仕様・制約**：選択肢3は`SGLT`（Na⁺/グルコース共輸送体）による`二次性能動輸送`を正しく記述している。選択肢5は`エキソサイトーシス`（膜動輸送の一種）による神経伝達物質の放出を正しく記述している
	- **選択理由**：1, 2, 4は輸送機構の種類に誤りがあるため不正解

**解法の手順（思考の流れ）**
- 選択肢1：`NO`（一酸化窒素）は脂溶性が高い気体分子 → キャリア不要で膜を直接通過 → `単純拡散`が正しいと判断
- 選択肢2：胃酸のH⁺分泌は細胞内（pH 7）→ 胃内腔（pH 1-2）の方向で濃度勾配に逆らう → `能動輸送`（H⁺/K⁺-ATPase）が必要と判断
- 選択肢3：腎近位尿細管でのグルコース再吸収は`SGLT`によるNa⁺共輸送 → 正しい
- 選択肢4：脂肪細胞へのグルコース取り込みは`GLUT4`による`促進拡散` → `エンドサイトーシス`は誤り
- 選択肢5：シナプス小胞からの`ノルアドレナリン`放出は`エキソサイトーシス` → 正しい

**他の選択肢が誤りな理由**
- 1. 一酸化窒素は、促進拡散により血管内皮細胞から移動する。
	`NO`は分子量30の小さな脂溶性気体であり、キャリア（トランスポーター）を必要とせず`単純拡散`により移動する。促進拡散ではない

- 2. 水素イオンは、受動輸送により胃壁細胞から分泌される。
	胃壁細胞の`H⁺/K⁺-ATPase`（プロトンポンプ）は`一次性能動輸送`であり、ATPのエネルギーを用いてH⁺を濃度勾配に逆らって管腔へ分泌する。プロトンポンプ阻害薬（`PPI`）はこの酵素を阻害する

- 4. 血中のグルコースの大部分は、エンドサイトーシスにより脂肪細胞に取り込まれる。
	脂肪細胞や骨格筋へのグルコース取り込みは`GLUT4`による`促進拡散`で行われる。`インスリン`刺激により細胞内小胞のGLUT4が細胞膜に移行（トランスロケーション）するが、グルコース自体の輸送はエンドサイトーシスではない

**キーポイント**
- `NO`＝脂溶性気体＝`単純拡散`（「促進拡散」とのひっかけ頻出）
- `H⁺`胃酸分泌＝`H⁺/K⁺-ATPase`＝`一次性能動輸送`（「受動輸送」とのひっかけ）
- `GLUT4`はインスリン依存的にトランスロケーションするが、輸送様式自体は`促進拡散`

**関連知識の比較**
- `SGLT1`：小腸上皮の管腔側膜でNa⁺/グルコース共輸送（二次性能動輸送）
- `SGLT2`：腎近位尿細管S1セグメントでNa⁺/グルコース共輸送。`SGLT2阻害薬`（エンパグリフロジンなど）は糖尿病治療薬
- `GLUT1-5`：促進拡散型グルコーストランスポーター。GLUT2（肝臓・膵β細胞）、GLUT4（筋肉・脂肪、インスリン依存）
- エキソサイトーシスの代表：`ノルアドレナリン`、`アセチルコリン`、`インスリン`の分泌

（一言メモ）
- `NO`の輸送は必ず`単純拡散`。脂溶性気体はキャリア不要と即答する
- `GLUT4のトランスロケーション`と「グルコースの輸送様式」を混同させる問題は頻出。GLUT4が「移動」するのではなく、GLUT4を使ってグルコースが「促進拡散」する
- 「受動輸送」と「能動輸送」の判断基準は「濃度勾配に従うか逆らうか」。胃酸は超酸性の胃内腔へ送るので明らかに能動
