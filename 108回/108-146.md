# 108-146

医薬品の製造販売業及び製造業に関する記述のうち、正しいのはどれか。2つ選べ。

1. 製造業の許可は、製造所ごとに受けなければならない。
2. 製造業者は、製造所ごとに医薬品総括製造販売責任者を設置しなければならない。
3. 製造販売業者が、自ら輸入した一般用医薬品を店舗販売業者に販売する場合は、医薬品販売業の許可を必要としない。
4. 製造販売業者が、医薬品を自社工場で製造する場合は、製造業の許可を受けたものとみなされる。
5. 第1種医薬品製造販売業の許可を受けた場合は、第2種医薬品製造販売業対象の医薬品も製造販売することができる。

## 正解
**1, 3**

## 解説

| 選択肢 | 正誤 | 理由 |
|---|---|---|
| 1. 製造業の許可は製造所ごと | ⭕ | 製造業の許可は製造所ごと・区分ごとに必要 |
| 2. 製造業者が総括製造販売責任者を設置 | ❌ | 総括製造販売責任者は「製造販売業者」が設置。製造業者が設置するのは「製造管理者」 |
| 3. 製造販売業者が輸入品を販売業者に販売→販売業許可不要 | ⭕ | 製造販売業者は自ら製造・輸入した医薬品を薬局・販売業者等に販売できる（別途販売業許可は不要） |
| 4. 製造販売業者は自社工場で製造業許可みなし | ❌ | 製造販売業≠製造業。自社工場で製造するなら別途製造業の許可が必要 |
| 5. 第1種があれば第2種もOK | ❌ | 第1種と第2種は別の許可。上位互換ではない |

### 重要ポイント
- **製造販売業**：市場への供給責任（承認取得・市販後管理）→ 総括製造販売責任者
- **製造業**：実際の製造行為 → 製造管理者
- この2つは完全に別の許可制度
