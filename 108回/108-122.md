# 108-122

下図は、我が国における胃、肝臓、膵臓、大腸及び肺の悪性新生物による粗死亡率（人口10万対）の年次推移を示したものである。

**図の傾向：**
- a: 最も高く、増加傾向
- b: かつて最も高かったが減少傾向
- c: 1990年代後半まで増加、その後減少
- d: 増加傾向
- e: やや増加傾向

部位a〜eの悪性新生物に関する記述のうち、正しいのはどれか。2つ選べ。

1. aは大腸である。
2. bの悪性新生物のリスク要因として、食塩の過剰摂取があげられる。
3. cの悪性新生物が1990年代後半まで増加した主な要因として、ヒトパピローマウイルス感染が考えられる。
4. dの悪性新生物のリスク要因として、食生活の欧米化や運動不足などによる肥満があげられる。
5. eは肺である。

## 正解
**2 と 4**

## 解説

### 部位の対応

| 部位 | がんの種類 | 傾向 |
|---|---|---|
| **a** | 肺がん | 最も高く、増加傾向 |
| **b** | 胃がん | かつて最多、減少傾向 |
| **c** | 肝臓がん | 1990年代後半まで増加、その後減少 |
| **d** | 大腸がん | 増加傾向 |
| **e** | 膵臓がん | やや増加 |

### 各選択肢の検証

#### 1. ❌ aは大腸
**aは肺がん**
- 男性の死亡率トップ
- 高齢化と過去の喫煙率の高さが影響

#### 2. ⭕ b（胃がん）：食塩過剰摂取がリスク
**胃がんのリスク要因：**
- 食塩の過剰摂取
- ヘリコバクター・ピロリ菌感染
- 喫煙

**減少した理由：**
- 食塩摂取量の減少
- 冷蔵庫の普及（塩蔵品減少）
- ピロリ菌対策

#### 3. ❌ c：HPV感染が増加要因
**cは肝臓がん**
- 増加要因：**C型肝炎ウイルス（HCV）感染**
- HPV（ヒトパピローマウイルス）は**子宮頸がん**の原因

**減少した理由：**
- C型肝炎治療の進歩
- 輸血時のウイルス検査の徹底

#### 4. ⭕ d（大腸がん）：欧米化・肥満がリスク
**大腸がんのリスク要因：**
- 食生活の欧米化（肉食・脂肪↑）
- 運動不足
- 肥満
- 飲酒、喫煙

**増加した理由：**
- 高脂肪・高タンパク食の増加
- 食物繊維摂取の減少
- 運動不足

#### 5. ❌ eは肺
**eは膵臓がん**
- やや増加傾向
- 高齢化が主因

### がんの死亡率の増減傾向

**減少傾向：**
- **胃がん** — 食塩摂取減少、ピロリ菌対策
- **肝臓がん** — C型肝炎治療の進歩

**増加傾向：**
- **肺がん** — 高齢化、過去の喫煙
- **大腸がん** — 食生活の欧米化、肥満
- **膵臓がん** — 高齢化

### ポイント
- **胃がん** = 食塩、ピロリ菌
- **大腸がん** = 欧米化、肥満
- **肝臓がん** = C型肝炎（HPVではない）
