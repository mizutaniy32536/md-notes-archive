# 108-20

耐熱性芽胞を形成する偏性嫌気性細菌で、大型の深鍋で調理したカレーやシチューによる食中毒の原因となるのはどれか。1つ選べ。

1. 黄色ブドウ球菌
2. カンピロバクター・ジェジュニ/コリ
3. サルモネラ属菌
4. 腸炎ビブリオ
5. ウェルシュ菌

## 正解
**5. ウェルシュ菌**

## 解説

### ウェルシュ菌（Clostridium perfringens）の特徴
- **耐熱性芽胞**を形成 → 100℃の加熱でも死なない
- **偏性嫌気性細菌** → 酸素があると生きられない
- 食肉、魚介類に広く存在
- **カレー・シチュー食中毒**の代表格

### カレー食中毒のメカニズム
1. 大型鍋で大量調理 → 鍋底が嫌気的環境になる
2. 加熱しても芽胞は死滅しない
3. 常温で放置 → 芽胞が発芽して増殖
4. 再加熱しても毒素（エンテロトキシン）が産生される

### 各選択肢の検証
1. ❌ 黄色ブドウ球菌 → 毒素型、芽胞は作らない
2. ❌ カンピロバクター → 鶏肉、らせん状、微好気性
3. ❌ サルモネラ → 卵・肉、芽胞は作らない
4. ❌ 腸炎ビブリオ → 海産物、好塩性、芽胞は作らない
5. ⭕ **ウェルシュ菌** → 全ての条件に一致

### ウェルシュ菌食中毒の特徴
- **潜伏期間**: 6〜18時間
- **症状**: 腹痛、下痢（嘔吐は稀）
- **予防**: 大量調理後は速やかに冷却、小分けして保存

### キーワード
- **耐熱性芽胞** + **偏性嫌気性** + **カレー・シチュー** = **ウェルシュ菌**
