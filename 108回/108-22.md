# 108-22

ある海域の海水中の化学物質Aの濃度は5×10⁻⁸ mg/Lであり、この海域に生息するシャチの体内から検出された化学物質Aの濃度は1×10⁻³ mg/kgであった。このときの生物濃縮係数に最も近い値はどれか。1つ選べ。

ただし、海水の比重は1.0と近似できるものとする。

1. 5×10⁻⁵
2. 5×10⁻³
3. 1×10¹
4. 2×10²
5. 2×10⁴

## 正解
**5. 2×10⁴**

## 解説

### 生物濃縮係数の公式
```
生物濃縮係数 = 生体内濃度 ÷ 環境中濃度
```

### 計算手順

#### ステップ1：単位を揃える
- 海水中濃度：5×10⁻⁸ mg/L
- シャチ体内濃度：1×10⁻³ mg/kg
- 海水の比重 = 1.0 → **1L = 1kg**

単位を揃えると：
- 海水中濃度：5×10⁻⁸ mg/kg
- シャチ体内濃度：1×10⁻³ mg/kg

#### ステップ2：割り算
```
生物濃縮係数 = (1×10⁻³) ÷ (5×10⁻⁸)
            = (1÷5) × 10⁽⁻³⁻⁽⁻⁸⁾⁾
            = 0.2 × 10⁵
            = 2 × 10⁴
```

### 指数の変換
**0.2 × 10⁵ → 2 × 10⁴**

```
0.2 × 10⁵
= (0.2 × 10) × 10⁴    ← 10⁵ = 10 × 10⁴
= 2 × 10⁴
```

**ルール：**
- 小数点を右に1つ動かす → 指数を1減らす
- 小数点を左に1つ動かす → 指数を1増やす

### 生物濃縮とは
環境中の微量な有害物質が、食物連鎖を通じて上位捕食者の体内に高濃度で蓄積される現象。

**代表例：**
- 有機塩素系農薬（DDTなど）
- PCB
- 水銀

この問題では、シャチの体内には海水の**2万倍も濃縮**されている。

### ポイント
- 単位を揃える（比重1.0 → 1L = 1kg）
- 指数計算（引き算）
- 小数点の移動と指数の調整
