# 108-123

下図は、我が国におけるある性感染症の報告数（全数把握）の年次推移を示したものである。この図に該当する感染症はどれか。1つ選べ。

※グラフの特徴：近年上昇傾向で、直近で急上昇している

1. 性器クラミジア感染症
2. 性器ヘルペス感染症
3. 尖圭コンジローマ
4. 梅毒
5. 後天性免疫不全症候群（AIDS）

## 正解
**4. 梅毒**

## 解説

### なぜ梅毒？

**「全数把握」がポイント：**
- 梅毒は感染症法で**全数把握対象**（医師が診断したら全例届出義務）
- AIDS（後天性免疫不全症候群）も全数把握対象だが、近年は横ばい傾向
- 他の選択肢（性器クラミジア、性器ヘルペス、尖圭コンジローマ）は**定点把握対象**

**近年の梅毒の急増：**
- 2010年代から急激に増加
- 特に2013年以降、報告数が右肩上がり
- 若年女性と中高年男性で増加

### 性感染症の届出区分

| 感染症 | 届出区分 | 近年の傾向 |
|---|---|---|
| **梅毒** | 全数把握（5類） | 急増↑↑ |
| **AIDS** | 全数把握（5類） | 横ばい→ |
| 性器クラミジア | 定点把握 | やや減少 |
| 性器ヘルペス | 定点把握 | 横ばい |
| 尖圭コンジローマ | 定点把握 | 横ばい |

### 梅毒の基礎知識

**病原体：** *Treponema pallidum*（梅毒トレポネーマ）

**症状：**
- 第1期：感染部位に硬結（しこり）
- 第2期：全身の皮疹、発熱
- 第3期：ゴム腫
- 第4期：心血管梅毒、神経梅毒

**治療：** ペニシリン系抗生物質

**妊婦感染の危険性：** 先天梅毒（胎児への垂直感染）

### 全数把握と定点把握の違い

**全数把握：**
- 診断した全ての患者を届出
- 重篤な感染症が対象
- 例：結核、梅毒、AIDS、麻疹、COVID-19など

**定点把握：**
- 指定された医療機関のみが届出
- 比較的頻度の高い感染症
- 例：インフルエンザ、性器クラミジア、手足口病など
