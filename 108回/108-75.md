# 108-75

フルオロウラシルとの相互作用により市販直後に多数の死亡例が発生した薬害・事件に関わるのはどれか。1つ選べ。

1. ソリブジン
2. サリドマイド
3. クロロキン
4. キノホルム
5. ゲフィチニブ

## 正解
**1. ソリブジン**

## 解説

ソリブジンは抗ヘルペスウイルス薬で、代謝物がフルオロウラシル（5-FU）の代謝酵素（DPD：ジヒドロピリミジンデヒドロゲナーゼ）を不可逆的に阻害し、5-FUの血中濃度が上昇して重篤な骨髄抑制を引き起こした。1993年、市販直後の薬害事件。

- サリドマイド → 四肢奇形（催奇形性）
- クロロキン → 網膜症（クロロキン網膜症）
- キノホルム → スモン（SMON、亜急性脊髄視神経症）
- ゲフィチニブ → 間質性肺炎
