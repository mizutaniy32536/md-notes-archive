# 第101回 問31

**問題文**
ジゴキシンが直接阻害するのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-31 を参照

**選択肢**
1. Na＋-Ca2＋交換体
2. Na＋,K＋-ATPase
3. アデニル酸シクラーゼ
4. ホスホジエステラーゼ
5. プロテインキナーゼA

**正解:** 2

**解説**
解答　２

ジゴキシンは心筋細胞膜のNa＋,K＋-ATPaseを直接阻害し、細胞内のNa＋の濃度を上昇させる。その結果、間接的にNa＋-Ca2＋交換体が抑制され、Ca2＋の細胞外流出が抑制されることで、細胞内Ca2＋濃度が上昇し、心収縮力増大作用を示す。
