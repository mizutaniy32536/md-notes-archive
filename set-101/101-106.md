# 第101回 問106

**問題文**
プロドラッグAに関する記述のうち、正しいのはどれか。2つ選べ。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-106 を参照

**選択肢**
1. プリン環をもつ。
2. 確認試験では、希塩酸中亜鉛粉末で還元し、芳香族第一アミン（芳香族第一級アミン）の定性反応を利用できる。
3. 生体内でエステラーゼによって加水分解されて薬効を示す。
4. 生体内において、aの位置で切断されて薬効を示す。
5. 代謝された後、主にシトシンリボヌクレオチドの生合成を阻害する。

**正解:** 1,2

**解説**
解答　１、２

プロドラッグAはアザチオプリンで、6-メルカプトプリンのプロドラッグである。アザチオプリンは、生体内でグルタチオン-S-トランスフェラーゼにより、S原子とイミダゾール環との間の結合（構造中のSの左上の結合）が切断され、6-メルカプトプリンに代謝されることで、プリンヌクレオチドの生合成を阻害する。

１　正
アザチオプリンはプリンアナログの前駆物質であり、構造中にプリン環をもつ。

２　正
アザチオプリンを希塩酸中亜鉛粉末で還元すると、イミダゾール環に結合しているニトロ基がアミノ基に還元されるため、芳香族第一級アミンの定性反応を利用できる。

３　誤
アザチオプリンは、生体内でグルタチオン-S-トランスフェラーゼにより分解されて薬効を示す（上記参照）。なお、エステラーゼは、エステル結合を加水分解する酵素である。アザチオプリンは構造中にエステル結合を持たないため、エステラーゼによる加水分解を受けることはない。

４　誤
