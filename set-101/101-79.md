# 第101回 問79

**問題文**
医薬品の安全性に関する非臨床試験の実施の基準はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-79 を参照

**選択肢**
1. GLP
2. GCP
3. GMP
4. GVP
5. GPSP

**正解:** 1

**解説**
解答　１

１　正
GLPは、医薬品の安全性に関する非臨床試験の実施の基準である。

２　誤
GCPは、医薬品の臨床試験の実施の基準である。

３　誤
GMPは、医薬品及び医薬部外品の製造管理及び品質管理の基準である。

４　誤
GVPは、医薬品、医薬部外品、化粧品、医療機器及び再生医療等製品の製造販売後安全管理の基準である。

５　誤
GPSPは、医薬品の製造販売後の調査及び試験の実施の基準である。
