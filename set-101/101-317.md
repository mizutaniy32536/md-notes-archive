# 第101回 問317

### 第101回薬剤師国家試験 問317

**問題文**  
希少疾病用医薬品の指定等に関する手続きのうち、誤っているのはどれか。1つ選べ。[1][3][4]

**選択肢**  
1. 指定申請にあたっては、その用途に係る本邦における対象者の数に関する資料が必要である。  
2. 指定にあたっては、薬事・食品衛生審議会の意見を聴く必要がある。  
3. 指定したとき及び指定を取り消したときは、その旨を公示する。  
4. 再審査期間は最長10年間である。  
5. 承認条件として、再審査のための製造販売後調査において全症例についての調査が義務付けられる。[1][3][4]

**正解番号**  
2[1][4]

**解説**  
希少疾病用医薬品は、対象患者数が本邦で5万人未満で医療上の必要性が高い医薬品として、厚生労働大臣が指定するものであり、指定申請時には対象者数に関する資料が必要（1正しい）。指定は薬事・食品衛生審議会の意見を聴いて行われ（2誤り：実際は厚生労働大臣が直接指定し、審議会の意見聴取は指定前の審査段階で必須ではない）、指定・取消時は公示する（3正しい）。再審査期間は最長10年（4正しい）、承認条件として全症例の製造販売後調査が義務付けられる（5正しい）。[1][4]
