# 第101回 問61

**問題文**
メニエール病の典型的な病態及び症状に該当しないのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-61 を参照

**選択肢**
1. 難聴
2. 耳鳴
3. 浮動性めまい
4. 眼振
5. 耳閉感

**正解:** 3

**解説**
解答　３

メニエール病は、内耳のリンパ液が過剰に増大し、内リンパ腫を生じる末梢性の前庭性めまいの一種である。典型的な病態及び症状としては、繰り返す回転性のめまいや、それに伴う耳鳴、難聴、耳閉感などがある。また、めまい発作時には定方向性の眼振（眼球のぶれ）がみられることが多い。
なお、選択肢中の浮動性めまいとは、体が宙に浮いたような感覚を伴うめまいであり、脳血管疾患や高血圧、うつ病時などにみられる症状であり、メニエール病の典型的な症状とはいえない。
