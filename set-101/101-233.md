# 第101回薬剤師国家試験 問233

## 問題文

特定健康診査におけるメタボリックシンドロームの診断基準となっている検査項目はどれか。**2つ選べ。**[1][2]

## 選択肢

1. 血圧
2. 空腹時血糖値
3. 尿酸値
4. 血清クレアチニン値
5. γ-GTP値

## 正解番号

**1、2**[1][2]

## 解説

特定健康診査におけるメタボリックシンドロームの診断基準となっている検査項目は腹囲、血圧、血糖、脂質です。[2]

**測定項目の基準値：**[2]

- **腹囲**：男性85 cm以上、女性90 cm以上
- **高血圧**：収縮期血圧130 mmHg以上 かつ／または 拡張期血圧85 mmHg以上
- **高血糖**：空腹時血糖100 mg/dL以上
- **脂質異常**：トリグリセリド150 mg/dL以上 または HDLコレステロール40 mg/dL以下

メタボリックシンドロームの診断には、必須条件として内臓脂肪の蓄積（腹囲基準）があり、これに加えて選択項目（血圧、血糖、脂質）が2つ以上該当する必要があります。[1]

したがって、選択肢の中では**血圧（1）**と**空腹時血糖値（2）**が診断基準の検査項目に該当します。尿酸値、血清クレアチニン値、γ-GTP値はメタボリックシンドロームの診断基準には含まれません。
