# 第101回 問177

**問題文**
湿式顆粒圧縮法による錠剤の製造プロセスにおいて、ア〜エに当てはまる単位操作の組合せとして適切なのはどれか。1つ選べ。＜image1:center＞＜image2:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-177 を参照

**選択肢**
（選択肢は画像を参照）

**正解:** 1

**解説**
解答　１

本設問は、湿式顆粒圧縮法により錠剤を製造するプロセスが記載されている。ア〜エに当てはめるべき工程を以下に示す。
ア　粉砕：錠剤を製造するために、まず主薬と添加剤を粉砕する必要がある。粉砕後、粒子径を均一に整えるため分級を行い、その後主薬と添加剤を均一にする目的で混合が行われる。
イ　練合：主薬と添加物を混合したものに結合剤溶液を加えている。これらを均一に混ぜ合わせる工程が必要である。結合剤に液状のものを用いているため練合が適当である。
ウ　乾燥：造粒によって得られた顆粒は結合剤に液状のものを用いているため、多量の水分を含んでいる。多量の水分は錠剤を製する際に、バインディングやスティッキングなど打錠障害の原因となるため、顆粒を乾燥し余分な水分を除去する必要がある。
エ　混合：整粒によって顆粒の大きさを整えた後、打錠を行うために滑沢剤を添加している。顆粒と滑沢剤を均一にする必要があるため混合が行われる。

以上ア〜エに当てはまる単位操作の組合せとして適切なのは１である。
