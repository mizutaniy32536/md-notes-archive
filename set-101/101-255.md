# 第101回 問255

### 第101回薬剤師国家試験 問255

**問題文（薬理）**  
30歳男性。気管支ぜん息の治療のため以下の薬剤が処方された。  
（処方薬：フルチカゾンおよびサルメテロールを含む吸入薬と推定）  
上記の処方薬に関する記述のうち、正しいのはどれか。**２つ**選べ。[1][4][10]

**全選択肢（番号付き）**  
1. フルチカゾンは、細胞質のグルココルチコイド受容体に結合し、核内に移行する。  
2. フルチカゾンは、NF-κBを活性化して、抗炎症作用を示す。  
3. サルメテロールは、気管支平滑筋のアドレナリンβ<sub>2</sub>受容体を刺激し、気管支を拡張する。（部分的に記述あり、完全形は文脈から補完）[1]  
4. フルチカゾンは、血糖上昇作用を有するが、サルメテロールはその作用を減弱する。  
5. サルメテロールは、心機能抑制作用を有するが、フルチカゾンはその作用を減弱する。[4]

**正解番号**  
1、3[1][4]

**解説**  
- **1 正しい**：フルチカゾンは合成副腎皮質ステロイド薬であり、細胞質のグルココルチコイド受容体に結合し、核内に移行することにより、抗炎症作用や免疫抑制作用を示す。[1]  
- **2 誤り**：フルチカゾンはNF-κBを抑制して抗炎症作用を示すため、活性化は逆の作用となる。[1]  
- **3 正しい**：サルメテロールは長時間作用型β<sub>2</sub>刺激薬であり、気管支平滑筋のアドレナリンβ<sub>2</sub>受容体を刺激して気管支拡張作用を示す。[1]  
- **4 誤り**：フルチカゾンは吸入ステロイドとして全身作用が少なく血糖上昇は軽微だが、サルメテロールがその作用を減弱するという証拠はない。[4]  
- **5 誤り**：サルメテロールは心機能抑制作用を持たず、むしろβ<sub>2</sub>刺激による心拍出増加の可能性がある。フルチカゾンとの相互減弱もない。[4][1]
