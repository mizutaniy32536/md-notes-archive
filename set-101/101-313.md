# 第101回 問313

### 第101回薬剤師国家試験 問313

**問題文（全文）**  
50代男性。いままで頭痛を感じたことはなかったが、しばらく前から違和感をおぼえ、妻が持っていたアセトアミノフェン（300 mg/1錠）を主成分とした一般用医薬品（第2類）を1週間前より服用している。  
薬剤師の対応として、最も適切なのはどれか。1つ選べ。[1][3][5][10]

**全選択肢（番号付き）**  
1. このまま服用して、しばらく様子をみてください。  
2. 1日3回まで服用できるので、昼食後も服用してください。  
3. イブプロフェンを主成分とした一般用医薬品に変更してください。  
4. 半月後といわず、すぐに受診してください。  
5. 軽度の痛みなので、1日6回まで服用してください。[3][5][10]

**正解番号**  
4[3]

**解説**  
本症例は、従来頭痛歴のない50代男性がアセトアミノフェン含有第2類医薬品を1週間継続服用しており、二次性頭痛（生命の危機を伴う可能性）の除外が必要。薬剤の漫然使用は薬物乱用頭痛（月10日以内使用が目安）を招く恐れがあり、緊急ではないものの医師受診を速やかに勧奨するのが適切。選択肢1は継続使用を容認し誤り、2・5は過剰服用を助長し誤り、3は他成分変更で二次性頭痛のリスク無視で誤り。正解は4。[1][3][5][10]
