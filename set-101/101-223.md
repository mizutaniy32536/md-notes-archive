# 第101回 問223

### 第101回薬剤師国家試験 問223

**問題文（全文）**  
フィラデルフィア染色体陽性急性リンパ性白血病と診断を受けた患者が、イマチニブメシル酸塩錠の服用を始めたが、効果が不十分であった。[1][2][5]  
**問223（物理・化学・生物）**  
フィラデルフィア染色体が形成される変異の分類として正しいのはどれか。１つ選べ。[1][2][5]

**全選択肢（番号付き）**  
１．置換[1][2][5]  
２．欠失[1][2][5]  
３．逆位[1][2][5]  
４．挿入[1][2][5]  
５．転座[1][2][5]

**正解番号**  
５[2]

**解説**  
フィラデルフィア染色体は、9番染色体と22番染色体の**相互転座**（t(9;22)）により生成されるbcr-abl融合遺伝子を有する染色体であり、**転座**が形成の原因となる変異の分類である。[2][5]  
- 置換：塩基の一部の置換による変異。[5]  
- 欠失：染色体やDNAの一部が失われる変異。[5]  
- 逆位：同一染色体が2箇所で切断され逆転して再結合する変異。[5]  
- 挿入：あるDNA配列に他のDNA配列が組み込まれる変異。[5]  
これらはいずれもフィラデルフィア染色体の形成メカニズムとは異なる。[1][2][5]
