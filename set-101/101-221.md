## ◆ ワクチンの種類と特徴を判断するシナリオ
260312

**問題**
予防接種に用いる抗原（ワクチン）に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**

1. ウイルスに対する生ワクチンは、接種後の免疫応答能を高めるため、天然に存在するウイルスの毒性をより高めたものである。
​
2. 我が国におけるインフルエンザワクチンの主成分は、ウイルスから分離・精製したノイラミニダーゼを不活化したものである。
​
3. ワクチンの中には、病原体構成成分の組換え体タンパク質を主成分とするものがある。
​
4. トキソイドは、病原体が産生する毒素を、免疫原性を残したまま無毒化したものである。

**正解と理由**
- `3. 組換え体タンパク質を主成分とするワクチンがある`
- `4. トキソイドは毒素を免疫原性を残したまま無毒化したもの`

**解法の手順（思考の流れ）**
- 選択肢1：生ワクチン = 毒性を`弱めた`もの（弱毒化）。「高めた」は逆 → **誤り**
- 選択肢2：インフルエンザワクチンの主成分 = `ヘマグルチニン(HA)`。ノイラミニダーゼではない → **誤り**
- 選択肢3：HBVワクチン等は組換え体タンパク質が主成分 → **正しい**
- 選択肢4：トキソイドの定義そのもの。ジフテリア・破傷風トキソイド等 → **正しい**

**キーポイント**
- `生ワクチン` = 弱毒化（毒性を弱めた）ウイルス
- `不活化ワクチン` = ウイルスを殺して抗原性だけ残したもの
- `トキソイド` = 毒素をホルマリン処理で無毒化、免疫原性は維持
- インフルエンザワクチン主成分 = `HA(ヘマグルチニン)`

**関連知識の比較**

| ワクチン種類 | 特徴 | 例 |
|---|---|---|
| 生ワクチン | 弱毒化した病原体 | BCG、麻疹、風疹、水痘 |
| 不活化ワクチン | 殺した病原体/成分 | インフルエンザ、日本脳炎 |
| トキソイド | 無毒化した毒素 | ジフテリア、破傷風 |
| 組換えワクチン | 遺伝子組換えタンパク質 | B型肝炎(HBs抗原) |

（一言メモ）
- 生ワクチンの「弱毒化」は超基本。「強毒化」のひっかけに注意
- インフルエンザ = HA（ヘマグルチニン）。NAじゃない！
