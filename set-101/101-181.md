# 第101回 問181

**問題文**
72歳男性。1年前より、一定の距離を歩行すると右のふくらはぎ（腓腹筋）に痛みを感じていたが、歩行をしばらく中止すると改善するので、放置していた。1ヶ月前から次第に症状が悪化してきたため、近医を受診した。このとき測定した血圧は、以下の通りであった。またMRIで右総腸骨動脈に閉塞を認めた。この患者に対する治療薬について、医師から薬剤師に相談があった。提案すべき適切な薬剤はどれか。2つ選べ。（血圧）左　上腕　138/72 mmHg　　足関節　152/78 mmHg右　上腕　134/70 mmHg　　足関節　 94/52 mmHg

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-181 を参照

**選択肢**
1. ワルファリンカリウム
2. シロスタゾール
3. ダビガトランエテキシラートメタンスルホン酸塩
4. リバーロキサバン
5. サルポグレラート塩酸塩

**正解:** 2,5

**解説**
解答　２、５

本患者は、「歩行をしばらく中止すると右のふくらはぎの痛みが改善する」という間欠性跛行がみられ、その症状が悪化していること、および足関節上腕血圧比（足関節の収縮期血圧/上腕の収縮期血圧：基準値 0.9以上）が低下していることから、閉塞性動脈閉塞症（ASO）だと推測される。
ASOは、四肢（特に腹部以下）の末梢動脈が動脈硬化によって慢性虚血をきたす疾患であり、虚血が進行すると壊死に至ることもある。四肢の慢性的な虚血による間欠性跛行が特徴的な所見である。
治療にはシロスタゾールやサルポグレラート塩酸塩、チクロピジン塩酸塩、クロピドグレル塩酸塩などの抗血小板薬が用いられる。
