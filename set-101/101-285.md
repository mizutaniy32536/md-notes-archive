# 第101回 問285

### 第101回薬剤師国家試験 問285

**問題文**  
糖・電解質水溶液からなる室とアミノ酸水溶液からなる室が、隔壁によって、2室に分けられた構造の高カロリー輸液剤(ダブルバッグ製剤)に関する記述のうち、正しいのはどれか。**2つ選べ。**[1][2][3]

**選択肢**  
1. 通例、保存剤が添加されている。[2]  
2. バッグを両手で強く押すことにより、隔壁部を開通させる。[1][2]  
3. 2室に分かれているため、混合するまでメイラード反応を回避できる。[1][2]  
4. 脂肪乳剤を同時に投与する場合は、糖・電解質水溶液からなる室に混合する。[3]  
5. 混合した製剤は、2時間以内に全量を投与する。[3]

**正解番号**  
**2, 3**[1][2]

**解説**  
- **選択肢1**: 誤り。輸液のように大量に用いる製剤には保存剤は添加しない。[1]  
- **選択肢2**: 正しい。バッグを両手で強く押すことにより隔壁を開通させるのが標準的な操作。[1][2]  
- **選択肢3**: 正しい。糖・電解質水溶液とアミノ酸水溶液を2室に分離することで、混合前のメイラード反応（還元糖とアミノ酸の褐色化反応）を回避できる。[1][2]  
- **選択肢4**: 誤り。脂肪乳剤は高カロリー輸液と混合せず、安定性損傷を避けるため別途末梢静脈から投与する。[3]  
- **選択肢5**: 誤り。混合した製剤は24時間以内に全量投与する。[3]
