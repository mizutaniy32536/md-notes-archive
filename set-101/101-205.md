# 第101回 問205

### 第101回薬剤師国家試験 問205

**問題文**  
74歳男性。意識障害のため救急搬送されてきた。水分貯留をともなう高血圧性緊急症と診断され、治療方針を話し合う中でニカルジピン塩酸塩とフロセミドの投与が検討された。  

処方薬の物性を測定する目的で、種々のpHで水溶液(50μg/mL)を調製し、その5mLずつに、それぞれ1-オクタノール5mLを加えてよく振り混ぜ、分配平衡に達した後、水層中の薬物濃度を測定した。以下の表は、処方されたどちらかの薬物の結果である。この結果に関する記述として正しいのはどれか。**2つ選べ**。  

ただし、この薬物は1-オクタノールとの相互作用を起こさず、また、イオン形薬物の1-オクタノールへの分配は起こらないものとする。[1][2][3][4]  

*(表の詳細は検索結果に記載されていないが、pH増加に伴い水層中濃度が減少する傾向を示すデータであることが複数の解説から推測される。)[1][3]*

**選択肢**  
1. 塩基性薬物ニカルジピンの測定結果である。[2][3]  
2. **酸性薬物フロセミドの測定結果である。**[2][3][4]  
3. この薬物の分配係数は、約10である。[2][3][4]  
4. この薬物のpKaは、約6.0である。[1][2][3][4]  
5. **この薬物のpKaは、約4.0である。**[1][2][3][4]  

**正解番号**  
2, 5[1][2][3][4]

**解説**  
pHが高くなるほど水層中の薬物濃度が減少（オクタノール層へ移行）するため、**酸性薬物**であることがわかる。酸性薬物は塩基性条件下で解離し非イオン化形（分子形）となり、脂溶性が高まってオクタノール層へ分配されやすいため、選択肢2が正しい。[1][3]  

pKaの推定は、分配率が最大化するpH（非イオン化形が支配的な点）から行う。表の結果からpKaは**約4.0**であり、選択肢5が正しい（pH-pKa差が2以上でほぼ完全に非イオン化）。選択肢4（pKa約6.0）は不一致。[1][3]  

選択肢1は誤り（ニカルジピンは塩基性薬物でpH上昇時に水層残存が増えるはず）。選択肢3は分配係数（P = [オクタノール]/[水]）の値が表から約10でない。[1][2][3][4]
