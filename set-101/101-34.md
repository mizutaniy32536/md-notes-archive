# 第101回 問34

**第101回薬剤師国家試験 問34**

**問題文:**  
前立腺がん治療に用いるアンドロゲン受容体遮断薬はどれか。1つ選べ。[1][3][5]

**全選択肢:**  
1. メテノロン  
2. レトロゾール  
3. テストステロン  
4. クロミフェン  
5. フルタミド[1][3][5]

**正解番号:** 5[1][5]

**解説:**  
フルタミド（オダイン）は、非ステロイド性の抗アンドロゲン薬で、前立腺がん治療に用いられ、アンドロゲン受容体を遮断します。[1][5]  
- 1. 誤: メテノロンはタンパク質同化ステロイドで、再生不良性貧血や骨粗鬆症に用いられます。[1][5][6]  
- 2. 誤: レトロゾールはアロマターゼ阻害薬で、閉経後乳がん治療に用いられます。[1][5]  
- 3. 誤: テストステロンはアンドロゲン（男性ホルモン）そのもので、前立腺がん治療には用いられません。[1][5]  
- 4. 誤: クロミフェンは抗エストロゲン薬で、不妊症の排卵誘発に用いられます。[1][5]
