# 第101回 問343

**問題文**
62歳男性。頭痛、のどの痛み、痰の絡む咳、鼻水に加え鼻閉症状を訴えて来局した。男性に質問したところ、仕事で車を運転していること、毎日晩酌し、1日20本の喫煙をしていることがわかった。現在、服用している薬はない。そこで、下記の成分が含まれる一般用医薬品を提案した。＜image1:center＞販売時に薬剤師が情報提供・指導する内容として適切でないのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-343 を参照

**選択肢**
1. 低血糖を起こすことがありますので注意してください。
2. 薬を飲む前後はお酒を飲まないでください。
3. 眠くなる成分は入っていませんので、服用後の車の運転は大丈夫です。
4. 便秘をすることがありますので注意してください。
5. のどが渇くことがあるので、適宜水分を補給してください。

**正解:** 1,3

**解説**
解答　１、３

１　不適切
本剤には血糖値を低下させる成分は含まれていないため、低血糖に関する指導は必要ない。

２　適切
本剤に含まれるd-クロルフェニラミンマレイン酸塩およびジヒドロコデインリン酸塩には中枢抑制作用があり、アルコール摂取によりその作用が増強するため、服薬前後の飲酒は控えるよう指導する必要がある。

３　不適切
本剤に含まれるd-クロルフェニラミンマレイン酸塩には、ヒスタミンH1受容体遮断による中枢抑制作用があり、眠気を引き起こしやすい。そのため、本剤服用後は車の運転を控えるよう指導する必要がある。

４　適切
本剤に含まれるジヒドロコデインリン酸塩には、μ受容体刺激によるアセチルコリン遊離抑制作用があり、腸管運動抑制による便秘を引き起こすことがあるため注意が必要である。

５　適切
本剤に含まれるd-クロルフェニラミンマレイン酸塩には抗コリン作用があり、腺分泌抑制による口渇を引き起こすことがあるので、適宜水分を補給するよう指導する必要がある。
