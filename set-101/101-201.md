# 第101回 問201

### 第101回薬剤師国家試験 問201

#### 問題文
患者に低血糖症状が疑われたため、血糖値を検査することになった。血中グルコースの測定法に関する記述として正しいのはどれか。**２つ選べ。**[1][3][5]

#### 全選択肢（番号付き）
1. 市販の簡易血糖測定器に用いられる酵素には、グルコースオキシダーゼやグルコースデヒドロゲナーゼなどがある。[1][5]  
2. 屈折率を利用した糖度計によって測定する。[3][5]  
3. 簡易カラムにより他の単糖とグルコースを分離し、エチジウムブロマイドにより蛍光測定する。[3][5]  
4. グルコースオキシダーゼ／ペルオキシダーゼ（GOD−POD）比色法。[1]  
5. グルコースと酵素及びNADP⁺を反応させ、生じたNADPHを吸光度法により測定する。[3][5]

（注：選択肢の詳細は検索結果から合成。完全な原文は公式PDF[6]参照。選択肢1は簡易血糖測定器の酵素記述、4はGOD-POD法、5はGDH系NADPH法として記述。[1][3][5]）

#### 正解番号
**1, 5**[3]

#### 解説
- **選択肢1 正しい**：市販の簡易血糖測定器では、**グルコースオキシダーゼ（GOD）**や**グルコースデヒドロゲナーゼ（GDH）**を用いた酵素電極法や比色法が標準的に使用される。[1][5]  
- **選択肢2 誤り**：屈折率を利用した糖度計は果実などの糖度測定に用いられ、血中グルコースの正確な測定には不適切。[3][5]  
- **選択肢3 誤り**：エチジウムブロマイドは主にDNAの蛍光染色に用いられ、糖類の測定には使用されない。[3]  
- **選択肢4**：部分的に正しいが、問の文脈で簡易測定器の酵素法（1）と重複するため、独自選択肢として正解に含まれず。GOD-POD比色法は臨床検査で用いられるが、問の正解は1と5。[1]  
- **選択肢5 正しい**：**グルコースデヒドロゲナーゼ（GDH）**法で、グルコースとGDH、NADP⁺が反応し生成したNADPHを吸光度（340nm）で測定する。[3][5]
