# 第101回 問193

**問題文**
コホート研究の指標の中で、「絶対リスク減少率」に該当するのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-193 を参照

**選択肢**
1. 曝露群と非曝露群の危険度の比
2. 治療することにより、ある転帰がどの程度抑えられたかを減少率で表した値
3. 対照群における結果因子の発生率と治療群における結果因子の発生率の差
4. 症例群が曝露した割合と対照群が曝露した割合の比
5. 治療必要数の逆数

**正解:** 3,5

**解説**
解答　３、５

１　誤
曝露群と非曝露群の危険度（リスク）の比は、相対危険度（relative risk：RR）のことである。RRは、治療群のイベント発生率/対照群のイベント発生率で求めることができる。

２　誤
治療することにより、ある転帰がどの程度抑えられたかを減少率で表した値は、相対リスク減少率（relative risk reduction：RRR）のことである。RRRは、治療群が対照群に対して何％イベント発生率を抑えられたかを示しており、1−RRで求めることができる。

３　正
対照群における結果因子の発生率と治療群における結果因子の発生率の差は、絶対リスク減少率（absolute risk reduction：ARR）のことである。ARRは、対照群のイベント発生率−治療群のイベント発生率で求めることができる。

４　誤
症例群が曝露した割合と対照群が曝露した割合の比は、症例-対照研究（ケースコントロール研究）において使用される、オッズ比のことである。

５　正
治療必要数（number need to treat：NNT）の逆数は、絶対リスク減少率のことである。
