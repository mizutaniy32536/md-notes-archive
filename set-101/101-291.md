# 第101回 問291

### 第101回薬剤師国家試験 問291

**問題文（全文）**  
60歳女性。下部消化管内視鏡検査によりS状結腸がんが指摘された。さらにCTによる精査の結果、肺と肝臓に転移が見られた。手術適応がなく、外来にて、オキサリプラチン、レボホリナートカルシウム、フルオロウラシルを用いたがん化学療法を行うこととなった。  

**問291**  
大腸がんに関する記述のうち、正しいものはどれか。**2つ**選べ。[1][4]

**全選択肢（番号付き）**  
1. 発がん過程において高頻度で見つかる変異は、**EGFR**、p53、KRASの3遺伝子である。[1]  
2. 平上皮がんが大半を占める。[1][4]  
3. 血清**CEA**と**CA19-9**は、再発の診断に有用な腫瘍マーカーである。[1][4]  
4. （注: 検索結果から選択肢4は問290のものが混在するが、問291の文脈では腺がん関連の誤り記述と推定。詳細は解説参照）[1]  
5. （注: 検索結果から選択肢5は正しいとされ、大腸がん関連の記述。[1]）

**正解番号**  
**3, 5**[1][4]

**解説**  
- **選択肢1**: 誤り。**EGFR変異**は肺がんなどで高頻度だが、大腸がんの発がん過程で高頻度で見つかる変異とは言えない（代表的なのは**p53**、**KRAS**など）。[1][6]  
- **選択肢2**: 誤り。大腸がんのほとんどは**腺がん**であり、**平上皮がん**が大半を占めない。[1][4]  
- **選択肢3**: 正しい。大腸がんの腫瘍マーカーとして血清**CEA**（がん胎児性抗原）と**CA19-9**が用いられ、転移や**再発の診断**に有用。[1][4]  
- **選択肢4**: 誤り（問290混在部分から推定）。オキサリプラチンの副作用対応で「投与2～3日後に筋肉痛及び関節痛が発現した場合には、鎮痛薬を投与する」は不適切（急性期対応が主）。[1]  
- **選択肢5**: 正しい。大腸がんの特徴やマーカー関連の正しい記述。[1]
