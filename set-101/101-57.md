# 第101回 問57

**問題文**
心電図上QT間隔の延長を誘発する可能性が最も高い抗不整脈薬はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-57 を参照

**選択肢**
1. メキシレチン
2. ベラパミル
3. アミオダロン
4. プロプラノロール
5. ジゴキシン

**正解:** 3

**解説**
解答　３

抗不整脈薬のうち、K＋チャネル遮断作用により再分極を遅らせる作用を持つ薬剤がQT間隔の延長を誘発する可能性が最も高く、Vaughan Williams分類のⅠa群（キニジンなど）やⅢ群（アミオダロンなど）がこれに該当する。
