# 第101回 問82

**問題文**
医薬品の使用によって健康被害が生じた場合に、医療従事者が厚生労働大臣（情報の整理を独立行政法人医薬品医療機器総合機構（PMDA）に行わせることとした場合には、PMDAに提出するのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-82 を参照

**選択肢**
1. インシデント報告書
2. プレアボイド報告書
3. DIニュース
4. 安全性速報
5. 医薬品安全性情報報告書

**正解:** 5

**解説**
解答　５

１　誤
インシデントとは、誤った医療行為により患者に健康被害を及ぼす可能性がある（があった）出来事のことであり、インシデント報告書は、インシデント（ヒヤリ・ハットした出来事）について報告する際に用いられる報告書である。

２　誤
プレアボイドとは、ファーマシューティカルケアの実践を通じて、薬物療法に伴う患者の不利益（副作用、相互作用、治療効果不十分など）を回避した事例、患者のQOLが向上した事例のことであり、プレアボイド報告書は、プレアボイドを日本薬剤師会に報告する際に用いられる報告書である。

３　誤
DIニュースとは、医療機関の医薬品情報（DI）室から発行される医薬品情報のことである。DIニュースには、医薬品の使用上の注意や新規採用医薬品の情報、緊急安全性情報や安全性速報、医療用医薬品添付文書等の改定のお知らせなどが記載される。

４　誤
安全性速報（ブルーレター）とは、医薬品・医療機器の添付文書が改訂された際、その内容が一般的な使用上の注意の改訂情報よりも迅速な安全対策措置をとる必要がある場合に発出される情報である。

５　正
医薬品安全性情報報告書は、医師、薬剤師などの医療従事者が医薬品による副作用及び感染症の発生について保健衛生上の危害又は拡大を防止する観点から報告が必要と判断した情報を厚生労働大臣（情報の整理を独立行政法人医薬品医療機器総合機構（PMDA）に行わせることとした場合には、PMDA）に報告する際に用いられる報告書である。
