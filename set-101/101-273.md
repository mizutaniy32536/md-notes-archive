# 第101回 問273

### 第101回薬剤師国家試験 問273

**問題文（共通ステム含む）**  
58歳男性。本態性高血圧症及び狭心症に対して外来で薬物治療を受けていたが、急に症状が悪化したため入院となった。薬剤師が面談し、薬物の使用状況等について尋ねたところ、めまいや、一過性の意識障害などの症状が現れることが時々あったため、最近になって自己判断で服薬を止めていたことが判明した。[1][2][3]  

**問273（薬剤）**  
服薬中止のきっかけとなった症状は、この患者が最近摂取し始めた食品あるいは一般用医薬品との相互作用に起因すると考えられた。摂取していた可能性が最も高いのはどれか。１つ選べ。[2][3]  

**全選択肢**  
1. 牛乳  
2. 鉄製剤  
3. アルミニウムを含む制酸剤  
4. **グレープフルーツジュース**  
5. セントジョーンズワートを含む健康食品[3]  

**正解番号**  
4[1][2][8]  

**解説**  
問272で患者が服用していた薬物は**ニフェジピン**（Ca拮抗薬、高血圧症・狭心症に適応）と判明。[1][2][3] ニフェジピンはCYP3A4で代謝され、**グレープフルーツジュース（GFJ）**摂取によりCYP3A4が阻害され血中濃度が上昇し、めまい・意識障害などの副作用（低血圧症状）を引き起こす可能性が高い。[2][8] 他の選択肢はニフェジピンとの関連が低い（牛乳・鉄製剤・制酸剤は吸収影響、セントジョーンズワートはCYP誘導で血中濃度低下）。[1][3][8]
