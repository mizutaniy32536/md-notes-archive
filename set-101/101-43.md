# 第101回 問43

**問題文**
一般に薬物の水溶性が低下する代謝反応はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-43 を参照

**選択肢**
1. アルキル側鎖の水酸化
2. N-脱アルキル化
3. エステルの加水分解
4. グルクロン酸抱合
5. アセチル抱合

**正解:** 5

**解説**
解答　５

一般に、薬物は代謝を受けると水溶性が増大するが、アセチル抱合など一部の代謝反応では、代謝を受けた薬物は水溶性が低下する。なお、アセチル抱合は、アセチル基転移酵素によって芳香族アミンやスルホンアミドなどのアミノ基にアセチル基を導入する代謝反応である。
