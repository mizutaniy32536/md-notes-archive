# 第101回 問335

**問題文**
55歳女性。3日前から腹部膨満感と嘔気があり、昨日からの腹痛と間欠的な嘔吐のため受診した。診察の結果、血圧98/62 mmHg、脈拍104/分、尿量低下、Na＋130 mEq/L、K＋3.5 mEq/L、Cl－90 mEq/Lであり、細胞外液減少を認めた。この患者に投与する輸液の組成として適切なのはどれか。1つ選べ。＜image1:center＞

**正解:** 5

**解説**
解答　５

診察の結果、本患者は細胞外液が減少していることから、本患者には等張性電解質輸液（細胞外液補充液）を投与する必要がある。

１　誤
5％ブドウ糖の組成である。
5％ブドウ糖輸液は、脱水症（特に水欠乏時）の水分補給に用いられる。

２　誤
低張性電解質輸液の4号液（術後回復液）の組成である。
4号液は術後早期の水分・電解質の補給に用いられる。

３　誤
低張性電解質輸液の3号液（維持液）の組成である。
3号液は経口摂取が不能または不十分な場合の水分・電解質の補給に用いられる。

４　誤
低張性電解質輸液の2号液（脱水補給液）の組成である。
2号液は脱水症及び手術前後の水分・電解質の補給・補正に用いられる。

５　正
乳酸リンゲル液の組成である。
乳酸リンゲル液の浸透圧は、血漿（細胞外液）とほぼ等しいため、乳酸リンゲル液は、細胞外液の補給、補正に用いられる。なお、乳酸リンゲル液に含まれる乳酸は肝臓で代謝され重炭酸イオンを生成するため、乳酸リンゲル液は、代謝性アシドーシスの改善にも用いられる。
