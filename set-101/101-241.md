# 第101回 問241

### 第101回薬剤師国家試験 問241

**問題文（全文）**  
36歳男性。CD20陽性の再発低悪性度B細胞性非ホジキンリンパ腫のため、イットリウム（**90Y**）イブリツモマブ チウキセタン（遺伝子組換え）を投与することになり、注射液の調製が依頼された。  
**問241（衛生）**  
**90Y**に関する記述のうち、正しいのはどれか。**2つ選べ**。[1][3]

**全選択肢（番号付き）**  
1. 主に**β線**を放出する。  
2. 半減期は約**1週間**である。  
3. **90Sr**との間に放射平衡が成り立つ。  
4. 神経組織に特異的効果を示す。  
5. 光電効果やコンプトン散乱を引き起こす。[1][3]

**正解番号**  
1 と 3[7][8]

**解説**  
- **90Y**は純粋な**β線**放出核種であり、高エネルギーの**β線**のみを放出する。[1][3][8]  
- 物理学的半減期は**64時間**（約2.7日）であり、選択肢2の「約1週間」（約7日）は誤り。[8]  
- **90Sr**（半減期約28年）は**90Y**の母核種であり、**90Sr** → **90Y**の崩壊系列で放射平衡が成り立つ。[1][3]  
- 選択肢4は誤り（**90Y**は神経組織に特異的でない）。[1]  
- 選択肢5は誤り（**β線**は主に電離作用で、光電効果やコンプトン散乱は**γ線**の相互作用）。[1]  
調製時の文脈として、**90Y**は安全キャビネット内で扱い、**β線**の特性から飛散防止が重要。[3][8]
