# 第101回 問269

### 第101回薬剤師国家試験 問269

**問題文（全文）**  
38歳男性。腰痛により整形外科を受診し、以下の処方箋をもって保険薬局に初めて来局した。（前問問268の処方箋を指す。問268はワルファリン含有処方）。  
**問269 （薬剤）**  
前問で選択した食品あるいは嗜好品の摂取により生じる可能性が最も高いのはどれか。１つ選べ。[1]

**全選択肢（番号付き）**  
1. 処方薬の吸収が促進され、副作用が発現する。  
2. **CYP1A2の誘導により、処方薬の効果が減弱する。**  
3. CYP3A4の阻害により、処方薬の副作用が発現する。  
4. 処方薬の腎排泄が阻害され、副作用が発現する。  
5. 処方薬の血中濃度には影響がないが、効果が減弱する。[1][10]

**正解番号**  
**2**

**解説**  
問268で選択された食品・嗜好品は**オレンジジュース**（問268正解は2）。オレンジジュース（特にグレープフルーツジュースに類似）は**CYP1A2**を誘導し、ワルファリンの代謝を促進するため、ワルファリンの効果が減弱する。ワルファリンはCYP1A2で代謝されるため、この相互作用が生じる。[1]  
他の選択肢は誤り：  
- 1：吸収促進の相互作用ではない。  
- 3：CYP3A4阻害はグレープフルーツジュースの主作用だが、オレンジジュースの主作用ではない。  
- 4：腎排泄阻害の報告なし。  
- 5：血中濃度に影響を与える。[1]
