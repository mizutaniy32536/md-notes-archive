# 第101回 問330

**問題文**
薬局において薬剤師の判断で行ってよい調剤行為はどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-330 を参照

**選択肢**
1. 水剤の調剤において服用しやすくするために賦形剤を添加した。
2. 処方箋に使用期限の指定はなかったが、交付年月日から3日目だったので調剤した。
3. 処方箋中の錠剤を同一銘柄の散剤に変更した。
4. 処方箋中の医薬品名が略語で記載されていたため、薬歴を参考にして調剤した。
5. 処方箋中に疑義内容があったが、他の患者で照会済の内容であったのでそのまま調剤した。

**正解:** 1,2

**解説**
解答　１、２

１　正
賦形剤の添加、保存剤及び安定化剤の添加、溶解補助剤、乳化剤、懸濁剤の添加、等張化剤及び緩衝剤の添加、組合せ剤の調製は、調剤学上当然の措置として薬剤師の判断で行うことができる。

２　正
保険医療機関及び保険医療養担当規則により、保険処方箋の使用期間は、特に記載がある場合を除き、交付日を含めて4日以内と定められている。よって、処方箋に使用期限の指定がなく、交付日から3日目だった場合、疑義照会することなく調剤することが可能である。

３　誤
処方箋中の錠剤を薬剤師の判断で同一銘柄の散剤に変更することはできない。よって、錠剤を同一の銘柄の散剤に変更する場合は、処方医に疑義照会する必要がある。

４　誤
処方箋中の医薬品名が略語で記載されている場合は、医薬品を特定することができないため、処方医に疑義照会し、医薬品を特定してから、調剤する必要がある。

５　誤
処方箋中に疑義内容がある場合には、処方医に疑義照会し、調剤する必要がある。
