# 第101回 問303

### 第101回薬剤師国家試験 問303

**問題文（症例文含む）**  
23歳女性。体重60kg。てんかん発作に対してフェニトイン1日150mgで治療を開始した。2週間後の受診で、治療開始後もてんかん発作が起こったとの訴えがあった。アドヒアランスは良好であった。[1][2][3]  

**問303**  
前問で計算した投与量で治療を続けていたが、中毒症状が発現したため血中濃度を測定したところ30μg/mLであった。原因として考えられる患者の遺伝的特徴はどれか。1つ選べ。[1][2][3]  

**全選択肢**  
1. **CYP2D6の変異型遺伝子をもつ。**  
2. **CYP2C9の変異型遺伝子をもつ。**  
3. **CYP2C19の野生型遺伝子をもつ。**  
4. **CYP3A5の野生型遺伝子をもつ。**  
5. **UGT1A1の変異型遺伝子をもつ。**[1][2][3]  

**正解番号**  
2[1][2][3]  

**解説**  
前問（問302）で計算した投与量を継続したにもかかわらず血中濃度が30μg/mLと高値（中毒域）となり中毒症状が発現した。フェニトインの主な代謝酵素は**CYP2C9**であり、変異型遺伝子（poor metabolizer）を持つ場合、代謝能が低下し血中濃度が上昇しやすいため、これが原因として適切。他の選択肢はフェニトインの主要代謝酵素ではないか、または野生型では代謝能が正常であるため不適切。[1][2][3]
