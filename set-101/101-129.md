## ◆ エボラ出血熱とMERSの共通点を正確に判断するシナリオ

260312

**問題**
エボラ出血熱とMERS（中東呼吸器症候群）の両方にあてはまる記述はどれか。2つ選べ。

**選択肢**

1. 病原体はウイルスである。
​
2. 自然宿主である野生動物を介して感染が拡大する。
​
3. 発症した際の致死率は5％未満である。
​
4. 感染症法で一類感染症に分類される。
​
5. 検疫感染症に指定されている。

**正解と理由**
- `1. 病原体はウイルスである。`
- `5. 検疫感染症に指定されている。`
- **理由の構成**
	- 選択肢1：エボラ出血熱=`エボラウイルス`、MERS=`MERSコロナウイルス`。どちらもウイルス → 正しい
	- 選択肢5：両方とも`検疫感染症`に指定されている → 正しい

**解法の手順（思考の流れ）**
- 選択肢1：エボラ=ウイルス✓、MERS=ウイルス✓ → 正しい
- 選択肢2：感染拡大に野生動物が関与するのはエボラのみ（コウモリ説）。MERSはヒト型コウモリ→ラクダ→ヒトだが「野生動物を介した感染拡大」とは言いにくい → 誤り
- 選択肢3：エボラ致死率50%以上、MERS約40%。5%未満は大きな誤り → 誤り
- 選択肢4：エボラ=`一類`✓、MERS=`二類`✗ → 両方に当てはまらない
- 選択肢5：両方とも`検疫感染症` → 正しい

**キーポイント**
- `エボラ出血熱`=`一類感染症`、`MERS`=`二類感染症`（この差が重要）
- `検疫感染症`：一類感染症、新型インフルエンザ等感染症＋政令で定める感染症（MERS、デング熱、ジカ等）
- 致死率：エボラ50%以上（流行により20〜90%）、MERS約40%

**関連知識の比較**

| | `エボラ出血熱` | `MERS` |
|---|---|---|
| 病原体 | エボラウイルス | MERSコロナウイルス |
| 感染症法 | **一類** | **二類** |
| 致死率 | 50%以上 | 約40% |
| 検疫感染症 | ○ | ○ |
| 自然宿主 | コウモリ（推定） | コウモリ→ラクダ→ヒト |

（一言メモ）
- エボラ=一類、MERS=二類。この差は超重要。「両方一類」は誤り
- 検疫感染症は一類感染症を全て含み、さらにMERS等が追加されている
