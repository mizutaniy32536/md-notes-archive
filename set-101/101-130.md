## ◆ 様々な病原体の母子感染と予防法の正誤を判断するシナリオ

260312

**問題**
母子感染に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**

1. 妊娠初期にサイトメガロウイルスに初感染すると、母子感染を起こすことがある。
​
2. ヒト免疫不全ウイルス（HIV）の母子感染を防ぐため、ワクチンの接種が推奨されている。
​
3. B型肝炎ウイルスは、母体にワクチンを接種することにより、胎児への感染を防ぐことができる。
​
4. 風疹は高率で胎内感染を起こすため、その予防を目的に、妊娠前の抗体検査が推奨されている。
​
5. トキソプラズマ感染による先天異常は、ワクチンにより防ぐことができる。

**正解と理由**
- `1. 妊娠初期にサイトメガロウイルスに初感染すると、母子感染を起こすことがある。`
- `4. 風疹は高率で胎内感染を起こすため、その予防を目的に、妊娠前の抗体検査が推奨されている。`
- **理由の構成**
	- 選択肢1：`CMV`の妊娠初期感染→母子感染リスクあり（`TORCH`症候群のC）→ 正しい
	- 選択肢4：`風疹`の胎内感染=`先天性風しん症候群`（CRS）。妊娠前の抗体検査・ワクチン接種が推奨 → 正しい

**解法の手順（思考の流れ）**
- 選択肢1：CMV初感染（既感染の再活性化より初感染の方が重症）→妊娠初期に母子感染リスク → 正しい
- 選択肢2：`HIVワクチン`は現存しない。予防は`ART`（抗レトロウイルス療法）＋帝王切開＋人工乳 → 誤り
- 選択肢3：B型肝炎の母子感染防止は`母体へのワクチン`ではなく、`出生後の児`への`HBIG`＋`HBワクチン` → 誤り
- 選択肢4：妊娠前の風疹抗体検査・ワクチン接種→CRS予防 → 正しい
- 選択肢5：`トキソプラズマ`に対するワクチンは存在しない。予防は感染経路回避（生肉・猫） → 誤り

**他の選択肢が誤りな理由**
- 2. HIV：有効なワクチンは開発されていない（2024年時点でも未承認）
- 3. B型肝炎の母子感染防止：母体へのワクチンではなく、生まれた`新生児`に`HBIG`＋`HBワクチン`を投与
- 5. トキソプラズマ：ワクチンなし。妊婦は生肉・猫の糞便接触を避けることで予防

**キーポイント**
- `CMV`（サイトメガロウイルス）=`TORCH`のC。妊娠初期の初感染が最もリスク高い
- `HIV母子感染`の予防：ワクチンなし、`ARTが鍵`（感染率<1%まで低減可能）
- `B型肝炎`母子感染：母体へのワクチンではなく、児への`HBIG+HBワクチン`（受動＋能動免疫）

（一言メモ）
- 「HIVワクチンで母子感染予防」は誤り。有効なワクチンは存在しない
- 「母体へのHBワクチン=母子感染防止」も誤り。産後の児への対応が正解
