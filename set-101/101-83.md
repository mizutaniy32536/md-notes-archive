# 第101回 問83

**問題文**
院内感染の原因菌の1つであるメチシリン耐性黄色ブドウ球菌（MRSA）の主要な感染経路はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-83 を参照

**選択肢**
1. 空気感染
2. 飛沫感染
3. 接触感染
4. 昆虫媒介感染
5. 垂直感染

**正解:** 3

**解説**
解答　３

１　誤
空気感染とは、飛沫核感染ともいわれ、主に直径5 µm以下の粒子が空気の流れにより広範囲に拡散し伝播する感染のことである。空気感染する主な病原体には、結核菌、麻疹ウイルス、水痘ウイルスがある。

２　誤
飛沫感染とは、主に直径5 µm以上の粒子が会話、くしゃみ、咳などにより伝播する感染のことである。飛沫感染する主な病原体には、インフルエンザウイルス、マイコプラズマ、風疹ウイルスがある。

３　正
接触感染とは、直接接触あるいは、医療器具などを介した間接的な接触により伝播する感染のことである。接触感染する主な病原体には、メチシリン耐性黄色ブドウ球菌（MRSA）、バンコマイシン耐性陽球菌（VRE）、腸管出血性大腸菌（O157）、緑膿菌がある。

４　誤
昆虫媒介感染とは、自然界にいる蚊やダニなどを通じて伝播する感染のことである。昆虫媒介感染する主な病原体には、デングウイルス、日本脳炎ウイルスがある。

５　誤
垂直感染とは、親から子供へ直接病原体が伝播する感染のことである。垂直感染する主な病原体には、梅毒トレポネーマ、淋菌がある。
