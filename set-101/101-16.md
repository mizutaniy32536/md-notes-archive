# 第101回 問16

**問題文**
ビタミンCの還元作用により、小腸からの吸収が促進されるミネラルはどれか。1つ選べ。1カルシウム2リン3鉄4マンガン5カリウム

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-16 を参照

**選択肢**
（選択肢は画像を参照）

**正解:** 3

**解説**
解答　３

ビタミンCの還元作用により、小腸からの吸収が促進されるミネラルは鉄である。
非へム鉄（無機鉄）は、Fe2＋の形で小腸から吸収されるが、食物中の鉄の大部分はFe3＋であるため、そのままでは吸収されにくい。そのため、ビタミンCの還元作用によりFe3＋がFe2＋へ還元されると吸収が促進される。
