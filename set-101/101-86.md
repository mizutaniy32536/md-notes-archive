# 第101回 問86

**問題文**
熱に不安定な薬物の水溶液を滅菌するのに最も適した方法はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-86 を参照

**選択肢**
1. 高圧蒸気滅菌
2. 乾熱滅菌
3. ろ過滅菌
4. 高周波滅菌
5. ガス滅菌

**正解:** 3

**解説**
解答　３

１　誤
高圧蒸気滅菌は、熱を利用して滅菌する方法であり、熱に不安定な薬物の水溶液を滅菌する方法としては適していない。高圧蒸気滅菌は、ある一定の温度及び圧力で飽和水蒸気を作り加熱することで微生物を殺滅する方法である。

２　誤
乾熱滅菌は、熱を利用して滅菌する方法であり、熱に不安定な薬物の水溶液を滅菌する方法としては適していない。乾熱滅菌は、乾燥空気中で加熱することにより微生物を殺滅する方法である。

３　正
ろ過滅菌は、滅菌時に熱が発生しないため、熱に不安定な薬物の水溶液を滅菌するのに適している。ろ過滅菌は、適当なろ過装置を用いて、ろ過により微生物を除去する方法である。滅菌用フィルターとしては、通例、メンブランフィルターなどが用いられる。

４　誤
高周波滅菌は、熱を利用して滅菌する方法であり、熱に不安定な薬物の水溶液を滅菌する方法としては適していない。高周波滅菌は、2450±50 MHzの高周波により発生する熱により微生物を殺滅する方法である。

５　誤
ガス滅菌は、低温滅菌が可能であるため、熱に不安定な薬物の滅菌には適しているが、滅菌ガスは水溶性が高く水溶液中に残存してしまう恐れがあるため、水溶液を滅菌する方法としては適していない。ガス滅菌は、酸化エチレンガスなどの滅菌ガスを用いて微生物を殺滅する方法である。
