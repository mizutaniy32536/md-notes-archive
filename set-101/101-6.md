# 第101回 問6

**問題文**
カテコールアミンに分類されるのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-6 を参照

**選択肢**
（選択肢は画像を参照）

**正解:** 3

**解説**
解答　３

カテコールアミンの骨格は下記の構造であり、本問では選択肢3のアドレナリンがカテコールアミンに分類される。

１　誤
サリチル酸系解熱鎮痛薬のエテンザミドの構造である。

２　誤
フェノチアジン系抗精神病薬のクロルプロマジンの構造である。

３　正
上記参照 

４　誤
アニリン誘導体の解熱鎮痛薬のアセトアミノフェンの構造である。

５　誤
エステル型局所麻酔薬のアミノ安息香酸エチルの構造である。
