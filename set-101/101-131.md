# 第101回 問131

**問題文**
体内からの化学物質の排泄に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-131 を参照

**選択肢**
1. 尿や胆汁以外にも、唾液腺、汗腺、及び涙腺を介して、化学物質は排泄される。
2. 血液への溶解度が高く、かつ脂溶性が高い化学物質は、呼気中への排泄が速い。
3. 化学物質の胆汁への排泄効率は、その分子量に依存しない。
4. 爪や毛髪も化学物質の排泄経路とみなすことができる。

**正解:** 1,4

**解説**
解答　１、４

１　正
体内に吸収された化学物質は、主に腎臓（尿）と肝臓（胆汁）より排泄されるが、一部は唾液腺、汗腺、涙腺、爪、毛髪などからも排泄される。

２　誤
血液への溶解度が高く、かつ脂溶性が高い化学物質は、呼気中への排泄が遅い。一般に、呼気中へ排泄されやすい化学物質は揮発性の性質を有する。揮発性の化学物質のうち、血液への溶解度が低いものほど呼気中への排泄が速く、血液への溶解度が高いものほど呼気中への排泄が遅い。

３　誤
化学物質の胆汁への排泄効率は、その分子量に依存する。分子量が500以上のものは胆汁中へ排泄されやすく、分子量が500以下のものは尿中へ排泄されやすい。

４　正
