# 第101回 問157

**問題文**
Ca2＋チャネル遮断薬に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-157 を参照

**選択肢**
1. アムロジピンは、心筋のL型Ca2＋チャネルを遮断するが、血管平滑筋のL型Ca2＋チャネルは遮断しない。
2. ベラパミルは、血管平滑筋のL型Ca2＋チャネルを遮断するが、心筋のL型Ca2＋チャネルは遮断しない。
3. シルニジピンは、血管平滑筋のL型Ca2＋チャネル及び交感神経終末のN型Ca2＋チャネルを遮断する。
4. ジルチアゼムは、房室結節のL型Ca2＋チャネルを遮断する。
5. ベプリジルは、Ca2＋チャネル遮断作用に加えて、K＋チャネル活性化作用を有する。

**正解:** 3,4

**解説**
解答　３、４

１　誤
アムロジピンは、ジヒドロピリジン系Ca2＋チャネル遮断薬であり、主に血管平滑筋に存在するL型Ca2＋チャネルを遮断する。心筋のL型Ca2＋チャネル遮断作用は弱い。

２　誤
ベラパミルは、主に心筋のL型Ca2＋チャネルを遮断する。血管平滑筋のL型Ca2＋チャネル遮断作用は弱い。

３　正
シルニジピンは、ジヒドロピリジン系Ca2＋チャネル遮断薬であり、血管平滑筋に存在するL型Ca2＋チャネル及び交感神経終末に存在するN型Ca2＋チャネルを遮断する。その他のジヒドロピリジン系薬と比較すると、N型Ca2＋チャネルの遮断作用を有するため、血管平滑筋に存在するL型Ca2＋チャネル遮断による降圧作用に伴う反射性心拍数増加反応が起こりにくい。

４　正
ジルチアゼムは、血管平滑筋及び房室結節など心筋のL型Ca2＋チャネルを遮断する。

５　誤
ベプリジルは、Ca2＋チャネル遮断作用に加えて、K＋チャネル遮断作用やNa＋チャネル遮断作用を示す。
