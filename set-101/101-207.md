# 第101回 問207

### 第101回薬剤師国家試験 問207

#### 問題文
10歳女児。体重34kg。昨夜より咽頭痛がひどかったため、母親が小児科を受診させ、以下の処方箋をかかりつけの保険薬局へ持参した。処方箋を受け取った薬剤師が薬剤服用歴とお薬手帳で現在服用中の薬剤を確認したところ、以下の①～⑤であった。[1][2][5]

**セフジニルに関する記述のうち、正しいのはどれか。** **2つ選べ。**[1][2][5]

#### 全選択肢（番号付き）
1. **セフェム系抗生物質である。**  
2. **細菌の細胞壁生合成に関わる酵素をアルキル化することで作用を示す。**  
3. **β-ラクタマーゼによって、aの位置で加水分解される。**  
4. **オキシムはZ配置である。**  
5. **アミノイミダゾリル基をもつ。**[1][2][4][5]

#### 正解番号
**1, 4**[1][2]

#### 解説
- **選択肢1**: 正しい。セフジニルは名称に「セフ」が含まれる**セフェム系抗生物質**である。[1][2]  
- **選択肢2**: 誤り。セフェム系抗菌薬の作用機序は、細菌の細胞壁合成酵素（経肽転移酵素）を**アシル化**（ペニシリン結合部位に共有結合）することで阻害するもので、**アルキル化**ではない。[1][7]  
- **選択肢3**: 誤り。β-ラクタマーゼによる加水分解は**β-ラクタム環**（aの位置は図示されていないが、通常β-ラクタム環を指す）で行われ、セフジニルは第3世代セフェムとして耐性を持つが記述が不正確。[1][2][7]  
- **選択肢4**: 正しい。セフジニルの**オキシム基**（C=N-OH）は、二重結合の**Z配置**（優先順位の高い置換基が同側）である。E/Z表示法に基づき、非共有電子対の優先順位が低いためZ。[1][2][4]  
- **選択肢5**: 誤り。セフジニルは**アミノチアゾリル基**を持つが、**アミノイミダゾリル基**ではない。[1][7]

（注: 問題は問206と連続しており、セフジニル細粒の処方箋が背景。公式問題は厚労省PDFで確認可能。[6]）
