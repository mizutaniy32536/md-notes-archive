# 第101回 問50

**問題文**
有効成分を臼歯と頬の間で徐々に溶解させ、口腔粘膜から吸収させる剤形はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-50 を参照

**選択肢**
1. 口腔内崩壊錠
2. 発泡錠
3. チュアブル錠
4. 舌下錠
5. バッカル錠

**正解:** 5

**解説**
解答　５

口腔内に適用する製剤として、口腔用錠剤がある。口腔用錠剤にはトローチ剤、舌下錠、バッカル錠、付着錠、ガム剤があり、本設問中の有効成分を臼歯と頬の間で徐々に溶解させ、口腔粘膜から吸収させる剤形は、バッカル錠が該当する。
