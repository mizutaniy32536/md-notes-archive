# 第101回 問44

**問題文**
腎クリアランス（CLr）と糸球体ろ過速度（GFR）、分泌クリアランス（CLs）、再吸収率（FR）の関係を正しく表しているのはどれか。1つ選べ。ただし、fuは血漿中タンパク非結合形分率を示すものとする。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-44 を参照

**選択肢**
1. CLr＝fu・GFR＋CLs－FR
2. CLr＝fu・GFR＋CLs＋FR
3. CLr＝fu・GFR－CLs＋FR
4. CLr＝（fu・GFR＋CLs）・（1－FR）
5. CLr＝（fu・GFR＋CLs）・FR

**正解:** 4

**解説**
解答　４

腎臓では、糸球体ろ過、尿細管分泌、尿細管再吸収により薬物の排泄が行われており、腎クリアランス（CLr）はこれらの過程により決定される。このことから、CLrは、ろ過クリアランス（CLろ過）、分泌クリアランス（CLs）、再吸収率（FR）により①式で表される。

　　CLr ＝ （CLろ過＋CLs）・（1－FR）・・・①

CLろ過は、血漿中タンパク非結合形分率（fu）・糸球体ろ過速度（GFR）であるため、①式は②式として表される。

　　CLr ＝ （fu・GFR＋CLs）・（1－FR）・・・②
