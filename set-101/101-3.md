# 第101回 問3

**問題文**
示強性状態関数の熱力学的パラメータはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-3 を参照

**選択肢**
1. ギブズ自由エネルギー（G）
2. エンタルピー（H）
3. 圧力（P）
4. エントロピー（S）
5. 内部エネルギー（U）

**正解:** 3

**解説**
解答　３

状態関数とは、系の状態によって一義的に定まる量のことであり、示量性状態関数と示強性状態関数に分類される。示量性状態関数は、系に含まれる物質量に依存し、加成性＊を示すが、示強性状態関数は、系に含まれる物質量には依存せず、加成性を示さない。
加成性＊：単純に足し合わせることができる性質のこと

１　誤
ギブズ自由エネルギー（G）は、示量性状態関数である。

２　誤
エンタルピー（H）は、示量性状態関数である。

３　正
圧力（P）は、示強性状態関数である。

４　誤
エントロピー（S）は、示量性状態関数である。

５　誤
内部エネルギー（U）は、示量性状態関数である。
