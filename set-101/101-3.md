# 第101回 問3

科目: 物理・化学  
問題文: 示強性状態関数の熱力学的パラメータはどれか。1つ選べ。[3][4]  
選択肢:  
1. ギブズ自由エネルギー（G）  
2. エンタルピー（H）  
3. 圧力（P）  
4. エントロピー（S）  
5. 内部エネルギー（U）[3][4]  
正解: 3[3][4]  
解説: 状態関数には示量性状態関数（物質量に依存するもの：ギブズ自由エネルギー、エンタルピー、エントロピー、内部エネルギーなど）と示強性状態関数（物質量に依存しないもの：圧力、温度、密度、濃度など）がある。したがって、示強性状態関数の熱力学的パラメータは圧力（P）である。[3][4]
