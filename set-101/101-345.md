# 第101回 問345

**問題文**
52歳男性。身長170 cm。血液検査の結果、血清ナトリウム濃度が147 mEq/Lであり、高張性脱水と診断され、輸液により水分を補給することになった。この男性の水分欠乏量を血清ナトリウム濃度から算出した場合、最も近い値はどれか。1つ選べ。ただし、この男性の健常時の体重は70 kg、血清ナトリウム濃度の目標値を140 mEq/L、体水分量を体重の60％とする。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-345 を参照

**選択肢**
1. 1.0 L
2. 1.5 L
3. 2.0 L
4. 2.5 L
5. 3.0 L

**正解:** 3

**解説**
解答　３

「この男性の健常時の体重は70 kg」「体水分量を体重の60％とする」と記載されていることから、この男性の健常時の体水分量は70 kg×60％＝42 Lである。また、この男性は高張性脱水であり、水分欠乏していることから、水分欠乏量をx Lとすると、この男性の水分量は（42－x）Lとなる。
「この男性の血清ナトリウム濃度が147 mEq/L」、「血清ナトリウム濃度の目標値を140 mEq/L」とあることから、x Lを以下のように求めることができる。
147 mEq/L×（42－x）L＝140 mEq/L×42 L    x＝2.0（L）
