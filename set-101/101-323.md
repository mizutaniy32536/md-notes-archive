# 第101回 問323

### 第101回薬剤師国家試験 問323

**問題文（全文）:**  
保険薬局の管理薬剤師が、新人の薬剤師に保険調剤について指導を行った。  
保険調剤は、「保険薬局及び保険薬剤師療養担当規則」に基づくものである。この規則の根拠となっている法律はどれか。1つ選べ。[1][2][3]

**全選択肢（番号付き）:**  
1. 介護保険法  
2. 医療法  
3. 健康保険法  
4. 薬剤師法  
5. 医薬品医療機器等法[1][2][3]

**正解番号:**  
3[1][2][3][4]

**解説:**  
「保険薬局及び保険薬剤師療養担当規則」（薬担規則）は、規則第1条において「健康保険法の規定に基づき…保険薬局及び保険薬剤師療養担当規則を次のように定める」と規定されており、この規則の根拠法は**健康保険法**である。他の選択肢（介護保険法、医療法、薬剤師法、医薬品医療機器等法）は該当しない。[1][2]
