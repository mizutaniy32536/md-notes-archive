# 第101回 問36

**問題文**
血小板のプロスタノイドIP受容体を刺激して、血小板凝集を抑制するのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-36 を参照

**選択肢**
1. ナファモスタット
2. チクロピジン
3. ダルテパリン
4. ベラプロスト
5. シロスタゾール

**正解:** 4

**解説**
解答　４

１　誤
ナファモスタットは、タンパク分解酵素であるトロンビンをアンチトロンビンⅢ非依存的に阻害することで、フィブリン形成を抑制する。

２　誤
チクロピジンは代謝活性化を受けて、ADPのP2Y12受容体を遮断し、血小板内のサイクリックAMP（cAMP）を増加させて血小板凝集を抑制する。

３　誤
ダルテパリンは、低分子ヘパリン製剤であり、アンチトロンビンⅢと結合して血液凝固Ⅹa因子を阻害し、フィブリン形成を抑制する。

４　正
ベラプロストは、血小板のプロスタノイドIP受容体を刺激して、血小板内cAMPを増加させて血小板凝集を抑制する。

５　誤
シロスタゾールは、血小板のホスホジエステラーゼⅢを阻害し、血小板内cAMPを増加させて血小板凝集を抑制する。
