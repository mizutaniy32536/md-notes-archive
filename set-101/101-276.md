# 第101回 問276

**問題文**
50歳女性。2型糖尿病と診断され内服薬で治療を行っていたが、血糖コントロール不良のため、インスリン導入目的で入院となり、以下の薬剤をペン型注入器を用いて投与することとなった。＜image1:center＞問276（実務）薬剤師が患者に指導する内容として、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-276 を参照

**選択肢**
1. 十分に混和し、均一にしてから使用してください。
2. 注射を忘れた場合は、次回2倍量を注射してください。
3. 注入ボタンを押したら速やかに針を抜いてください。
4. 腹部、大腿部、上腕部などの投与部位を決め、その中で注射場所を毎回変えてください。問277（薬剤）下図は、今回処方された注射剤（ヒトイソフェンインスリン水性懸濁注射液）とインスリンヒト注射液を、それぞれヒトの皮下に投与した後のインスリン血中濃度推移を示している。処方された注射剤に該当する血中濃度推移と、この製剤の特徴に関する記述の組合せのうち、正しいのはどれか。1つ選べ。なお、処方された注射剤には、添加剤としてプロタミン硫酸塩が含まれている。＜image2:center＞

**正解:** 1,4

**解説**
問276　解答　１、４

１　正
ヒトイソフェンインスリン水性懸濁注射液は懸濁性の注射剤であり、放置すると、白色の沈殿物と無色の上澄液に分離する。懸濁性の製剤は投与する際に、有効成分を均一に分散させる必要があるため、投与前に十分混和し均一にしてから使用するように指導する。

２　誤
注射を忘れた場合にインスリンを2倍量投与してしまうと、過度な血糖降下が起こり、重篤な低血糖の発症につながるおそれがある。そのため、注射を忘れた場合は、次回2倍量を注射するという指導は適切ではない。なお、ヒトイソフェンインスリン水性懸濁注射液は持続性の注射剤であり、注射を忘れた場合には、気付いた時点で1回量を投与する。その後、次の投与は8時間以上間隔を空けて行い、以降は通常通りに投与するように指導する。

３　誤
インスリン注入後、速やかに針を抜いてしまうと、インスリンが充分量注入されない可能性がある。インスリンを確実に注入するためには、注入後5〜10秒以上注入ボタンを押し、その後、注入ボタンを押した状態で針を抜くように指導する。

４　正
インスリン製剤の皮下注射は、投与部位によってインスリンの吸収速度が異なり、作用発現時間に違いが生じるため、おおまかな投与部位（腹部、大腿部、上腕部、臀部）を決定する必要がある。また、同じ場所に注射を行うと、組織の硬結が起こる場合があるので、決定した投与部位の中で注射する場所を変えるように指導する。

問277　解答　６

ヒトイソフェンインスリン水性懸濁注射液中のインスリンは、本剤に添加剤として含まれているプロタミン硫酸塩によって、溶解性の低い複合体を形成し、結晶化している。皮下投与後、結晶化したインスリンが徐々に溶解し血中へ移行していく。そのため、ヒトイソフェンインスリン水性懸濁注射液の血中濃度推移のグラフは、インスリンヒト注射液と比較すると持続的な血中濃度推移を示すので、イのグラフとなる。
