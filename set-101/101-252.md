# 第101回 問252

**問題文**
45歳女性。卵巣がん。がん性疼痛に対して以下の薬剤を使用してきたが、疼痛が増強してきたので、追加処方を検討することにした。問252（実務）ロキソプロフェンNa錠に追加する薬剤として適切なのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-252 を参照

**選択肢**
1. コデインリン酸塩散10％
2. セレコキシブ錠
3. モルヒネ硫酸塩水和物徐放錠
4. メサドン塩酸塩錠問253（薬理）前問で適切と考えた追加処方薬の薬理作用（副作用を含む）として、正しいのはどれか。2つ選べ。
5. 中脳や延髄に作用し、脊髄への下行性抑制系神経を抑制することで鎮痛作用を示す。
6. プロスタグランジンの産生を抑制し、解熱作用を示す。
7. 肝臓の薬物代謝酵素によってモルヒネに変換されて鎮痛作用を示す。
8. 延髄の化学受容器引き金帯（CTZ）を抑制し、制吐作用を示す。
9. 消化管運動を抑制し、便秘を起こす。

**正解:** 1,3

**解説**
問252　解答　１、３

がん性疼痛に用いる鎮痛薬を選択する際には、WHO三段階除痛ラダーに従って適切な効力の鎮痛薬を選択する。WHO三段階除痛ラダーでは、まず軽度な痛みに対しては、第一段階としては非オピオイド系鎮痛薬（非ステロイド性酸性抗炎症薬、アセトアミノフェン）を選択する。鎮痛効果が不十分な軽度から中等度の強さの痛みに対しては、第二段階として弱オピオイド系鎮痛薬（コデイン、ペンタゾシンなど）を選択し、必要に応じて非オピオイド系鎮痛薬を併用する。さらに中等度から高度の強さの痛みに対しては、第三段階として強オピオイド系鎮痛薬（モルヒネ、フェンタニル、オキシコドンなど）を選択し、必要に応じて非オピオイド系鎮痛薬を併用する。
ロキソプロフェンNa錠は、非ステロイド性鎮痛薬であり、第一段階に用いる鎮痛薬に分類されるため、追加処方としては弱オピオイド系か強オピオイド系を痛みの程度に応じて追加処方として検討する。

１　正
コデインリン酸塩散10％は、弱オピオイド系鎮痛薬に分類されるため、ロキソプロフェンNa錠に追加する薬剤として適切である。

２　誤
セレコキシブ錠は、ロキソプロフェンNa錠と同種の非ステロイド性抗炎症薬に分類されるため、追加処方する薬剤としては不適切である。

３　正
モルヒネ硫酸塩水和物徐放錠は、強オピオイド系鎮痛薬に分類されるため、ロキソプロフェンNa錠に追加する薬剤として適切である。

４　誤
メサドン塩酸塩錠は、強オピオイド系鎮痛薬に分類されるが、その他の強オピオイド系鎮痛薬で治療困難な中等度から高度の疼痛を伴う場合に用いられるため、追加処方する薬剤としては不適切である。

問253　解答　３、５

１　誤
コデイン及びモルヒネは、中脳や延髄に作用し、脊髄への下行性抑制系神経を賦活化することで鎮痛作用を示す。

２　誤
ロキソプロフェンやセレコキシブなどの非ステロイド性酸性抗炎症薬は、シクロオキシゲナーゼ（COX）を阻害し、プロスタグランジンの産生を抑制することで、解熱作用を示す。

３　正
コデインは、肝臓の薬物代謝酵素であるCYP2D6によりモルヒネに変換されて鎮痛作用を示す。

４　誤
コデイン及びモルヒネは、延髄の化学受容器引き金帯（CTZ）に存在するドパミンD2受容体を刺激することで、悪心・嘔吐を引き起こす。

５　正
コデイン及びモルヒネは、オピオイドμ受容体を刺激することで、消化管運動を抑制し便秘を引き起こす。
