# 第101回 問94

**問題文**
水溶液中のイオン間相互作用に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-94 を参照

**選択肢**
1. 強電解質のモル伝導率は、濃度に比例して増加する。
2. 難溶解性塩であるAgClの溶解度は、NaNO3の添加によるイオン強度の増大とともに増大する。
3. 高濃度の強電解質溶液におけるイオンの平均活量係数は、1より大きくなることがある。
4. 水中における電解質のイオン間相互作用は、アルコールなどを添加して溶媒の誘電率が低下すると減少する。
5. アルカリ金属における極限モル伝導率は、K＋＜Na＋＜Li＋の順に大きくなっている。

**正解:** 2,3

**解説**
解答　２、３

１　誤
強電解質のモル伝導率Λと濃度Cの関係は、以下のコールラウシュの平方根則で表される。
 Λ=Λ0 −b√C 　Λ0：極限モル伝導率　b：定数
上記の式より、強電解質のモル伝導率は、濃度の平方根に対して減少する。

２　正
難溶解性塩の溶解度は添加する電解質の種類により変化する。
難溶解性塩と共通のイオンを含む電解質を添加した場合、難溶解性塩の溶解度は低下し、難溶解性塩と異なるイオンを含む電解質を添加した場合、難溶解性塩の溶解度は増加する。前者の現象を共通イオン効果といい、後者の現象を異種イオン効果という。
AgClとNaNO3は共通イオンを有しないため、難溶解性塩であるAgClの溶解度は、NaNO3の添加によるイオン強度の増大とともに増大する（異種イオン効果）。

３　正
低濃度（希薄溶液）の強電解質において、イオン強度が増大すると、イオン間の相互作用が増大するため、イオンの平均活量係数は1より小さくなる。しかし、高濃度の強電解質溶液では、イオンと溶媒間に生じる相互作用も考慮する必要があるため、平均活量係数が1より大きくなることがある。

４　誤
イオン間相互作用は、陽イオンと陰イオンに働く相互作用のことであり、誘電率と反比例の関係を示す。そのため、水中における電解質のイオン間相互作用は、アルコールなどを添加して溶媒の誘電率が低下すると増大する。

５　誤
アルカリ金属における極限モル伝導率は、水和イオン半径の大きさに依存する。アルカリ金属において、イオン半径が小さいほど、電子密度が高いため、水分子と強く水和する。水分子と強く水和したイオンは水和半径が大きくなるため、水中の移動度が小さくなり、極限モル伝導率は低くなる。
このことより、アルカリ金属における極限モル伝導率は、K＋＞Na＋＞Li＋の順に小さくなっている。
