# 第101回 問189

**問題文**
口腔咽頭カンジダ症に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-189 を参照

**選択肢**
1. HIV感染者で罹患率が低い。
2. ステロイド吸入患者で罹患率が高い。
3. がん患者で罹患率が低い。
4. 舌痛、口腔内出血、口腔粘膜の白苔、潰瘍などの臨床症状を示す。
5. ニューキノロン系抗菌薬による初期治療が第一選択である。

**正解:** 2,4

**解説**
解答　２、４

口腔咽頭カンジダ症は、口腔内に常在するカンジダ属によって起こり、病原性は低いが、HIV感染、副腎皮質ステロイド性薬や免疫抑制薬などの長期投与およびがん患者などの、免疫力が低下している患者で発症する。

１　誤
HIV感染者などの免疫力が低下している状態だと、罹患率は高い。（前記参照）

２　正
前記参照。

３　誤
がん患者などの免疫力が低下している状態だと、罹患率は高い。（前記参照）

４　正
口腔咽頭カンジダ症では、口腔粘膜に偽膜状の白苔の形成、粘膜感染による口腔潰瘍や口腔内出血、舌乳頭の萎縮による舌痛などの臨床症状を示す。

５　誤
カンジダは真菌の一種であるため、治療には抗真菌薬であるアゾール系抗真菌薬やアムホテリシンBなどが用いられる。なお、ニューキノロン系抗菌薬は、細菌感染症の治療薬であるため、本症には用いられない。
