# 第101回 問49

**問題文**
不揮発性の電解質を溶解させた希薄水溶液において、溶質の濃度上昇とともに値が減少するのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-49 を参照

**選択肢**
1. イオン強度
2. 凝固点
3. 沸点
4. 浸透圧
5. 溶質のモル分率

**正解:** 2

**解説**
解答　２

１　誤
イオン強度とは、イオン間に働く電気的相互作用の強さを表すものであり、希薄水溶液の電解質濃度の上昇とともに、その値は増大する。

２　正
凝固点は束一的性質を示し、希薄水溶液において不揮発性の溶質の濃度の上昇とともに、その値は減少する。

３　誤
沸点は束一的性質を示し、希薄水溶液において不揮発性の溶質の濃度の上昇とともに、その値は増大する。

４　誤
浸透圧は束一的性質を示し、希薄水溶液において不揮発性の溶質の濃度の上昇とともに、その値は増大する。

５　誤
溶質の濃度上昇とともに溶質のモル分率は増大する。
