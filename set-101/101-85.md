# 第101回 問85

**問題文**
血液凝固因子製剤を患者に使用した場合、血液製剤管理簿を作成する意義として最も適切なのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-85 を参照

**選択肢**
1. 在庫切れの防止
2. 使用頻度の把握
3. 購入価格の把握
4. 有効期限切れの防止
5. 保健衛生上の危害の発生又は拡大の防止

**正解:** 5

**解説**
解答　５

血液凝固因子製剤等の特定生物由来製品は、生物由来製品のうち保健衛生上のリスクが特に高く、危害の発生又は拡大を防止するための措置を講じることが必要なものとして厚生労働大臣により指定されたものである。また、血液凝固因子製剤等の血液製剤は、血液製剤管理簿を作成の上、医療機関は少なくとも20年間保管・管理することが規定されている。血液製剤管理簿の記載内容を次に示す。
（血液製剤管理簿への記載内容）
①使用の対象者の氏名及び住所
②使用した製品の名称及び製造番号又は製造記号
③使用した年月日
④保健衛生上の危害の発生又は拡大の防止
