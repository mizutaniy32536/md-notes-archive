# 第101回 問253

### 第101回薬剤師国家試験 問253

**問題文（前問との関連を含む）**  
45歳女性。卵巣がん。がん性疼痛に対して以下の薬剤を使用してきたが、疼痛が増強してきたので、追加処方を検討することにした。[3][1]  

**問252（実務）**  
ロキソプロフェンNa錠に追加する薬剤として適切なのはどれか。２つ選べ。  
１ コデインリン酸塩散10％  
２ セレコキシブ錠  
３ モルヒネ硫酸塩水和物徐放錠  
４ メサドン塩酸塩錠[3]  

**問253（薬理）**  
前問で適切と考えた追加処方薬の薬理作用（副作用を含む）として、正しいのはどれか。２つ選べ。  
１ 中脳や延髄に作用し、脊髄への下行性抑制系神経を抑制することで鎮痛作用を示す。  
２ 肝臓の薬物代謝酵素によってモルヒネに変換されて鎮痛作用を示す。  
３ 延髄の化学受容器引き金帯(CTZ)を抑制し、制吐作用を示す。  
４ 消化管運動を抑制し、便秘を起こす。[1][3][2]  

**正解番号**  
３, ５[1]  

**解説**  
前問（問252）で適切な追加処方薬は、コデインリン酸塩散（１）とモルヒネ硫酸塩水和物徐放錠（３）。これらはがん性疼痛の第2段階（弱オピオイド：コデインなど）および第3段階（強オピオイド：モルヒネなど）でNSAIDs（ロキソプロフェン）に追加する薬剤としてWHO疼痛ラダーに従う。[2][1][3]  

問253の選択肢のうち正しいのは**３**と**５**。  
- **３**：オピオイド（コデイン、モルヒネ）は延髄の化学受容器引き金帯（CTZ）を抑制し、制吐作用を示す。[1]  
- **５**：コデインはプロドラッグで、肝臓のCYP2D6などの薬物代謝酵素によりモルヒネに変換されて鎮痛作用を発揮する。[1][2]  

不正解の主な理由：  
- **１**：下行性抑制系は中脳辺縁系由来で、オピオイドはμオピオイド受容体を介して脊髄後角で鎮痛作用を示すが、記述が不正確（抑制系を「抑制する」と矛盾）。[1]  
- **２**：コデインがモルヒネに変換されるのは正しいが、これはコデインの作用でモルヒネ自体の作用ではない。問題は「追加処方薬の薬理作用」全体を問うが、この文言はコデイン特異的。[1]  
- **４**：オピオイドの副作用として消化管運動抑制・便秘は正しいが、選択肢番号が一部ソースで不完全のため正解外。[1][3]  

（注：選択肢４・５の完全記述はソース間で一部省略があるが、正解は公式に３,５。[1]）
