# 第101回 問152

**問題文**
交感神経系に作用する薬物について、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-152 を参照

**選択肢**
1. エチレフリンは、間接型のアドレナリン作動薬で、タキフィラキシーを生じる。
2. ブナゾシンは、アドレナリンα1受容体を遮断して、末梢血管抵抗を減少させる。
3. ブリモニジンは、アドレナリンα2受容体を遮断して、眼房水の排出を促進する。
4. チモロールは、アドレナリンβ2受容体を刺激して、気管支を拡張させる。
5. ニプラジロールは、アドレナリンβ受容体遮断作用に加えて、ニトログリセリンに類似した血管拡張作用を有する。

**正解:** 2,5

**解説**
解答　２、５

１　誤
エチレフリンは、直接型のアドレナリン作動薬であり、アドレナリンα及びβ受容体を刺激する。タキフィラキシーは、チラミンなどの間接型のアドレナリン作動薬に見られる現象であり、間接型薬物の短時間反復投与により、シナプス伝達が低下し作用の減弱を起こす現象のことをいう。

２　正
ブナゾシンは、選択的にアドレナリンα1受容体を遮断し、ノルアドレナリンによる血管平滑筋収縮作用を抑制し、末梢血管抵抗を減少させる。

３　誤
ブリモニジンは、選択的にアドレナリンα2受容体を刺激し、眼房水の産生を抑制するとともにブドウ膜強膜流出路からの眼房水の排出を促進することで眼内圧を低下させる。

４　誤
チモロールは、非選択的アドレナリンβ受容体遮断薬であり、気管支平滑筋のアドレナリンβ2受容体を遮断することにより気管支平滑筋を収縮させるため、気管支喘息の患者への投与は禁忌である。

５　正
ニプラジロールは、非選択的アドレナリンβ受容体遮断薬である。また、構造中のNOを遊離して、グアニル酸シクラーゼを活性化し細胞内cGMP量を増加させる作用を示すことにより、ニトログリセリンに類似した血管拡張作用を介して血圧を低下させる。
