# 第101回 問101

**問題文**
次の反応のうち、主生成物の構造式を正しく示しているのはどれか。2つ選べ。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-101 を参照

**選択肢**
（選択肢は画像を参照）

**正解:** 1,2

**解説**
解答　１、２

１　正
アルケンへの臭素（Br2）の付加反応は、アンチ付加で進行する。なお、生成した2,3-ジブロモブタンは、メソ体である。

２　正
アルケンに酸触媒下、水（H2O）を付加すると、マルコフニコフ則に従って進行する。

３　誤
ブタ-1,3-ジエンなどの共役ジエンに低温でHBrを反応させると、1,2-付加体（3-ブロモ-ブタ-1-エン）が生成し、高温で反応させると1,4-付加体（1-ブロモ-ブタ-2-エン）が生成する。そのため、本問のような4-ブロモ-ブタ-1-エンを生成することはない。

４　誤
アルケンに過酸（m-クロロ過安息香酸）を反応させると、シン付加によりエポキシドが生成する。その際、アルケンの立体は保持されたまま進行するため、生成するのはトランス配置のエポキシドとなる。

５　誤
アルキンに液体アンモニア中、金属ナトリウムを反応させるとバーチ還元により、trans-アルケンが生成する。
