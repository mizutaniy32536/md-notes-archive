# 第101回 問27

**問題文**
ある受容体への競合的遮断薬A、B、CのpA2値がそれぞれ、9.3、7.9、8.2である場合、拮抗作用の強い順番として正しいのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-27 を参照

**選択肢**
1. A＞B＞C
2. A＞C＞B
3. B＞A＞C
4. B＞C＞A
5. C＞A＞B

**正解:** 2

**解説**
解答　２

pA2値は、競合的遮断薬の効力を示す指標であり、「活性物質の用量―反応曲線を2倍だけ高用量側に平行移動させるのに必要な競合的遮断薬の用量（モル濃度）の負の対数」と定義される。そのため、拮抗作用が強い薬物ほど、低用量で用量―反応曲線を2倍だけ高用量側に平行移動させるため、用量の負の対数であるpA2値は大きくなる。よって、拮抗作用の強い順番は、A＞C＞Bとなる。
