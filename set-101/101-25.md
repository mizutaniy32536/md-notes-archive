# 第101回 問25

**問題文**
閉鎖性水域における富栄養化の制限因子はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-25 を参照

**選択肢**
1. カリウム
2. 亜鉛
3. 鉄
4. 硫黄
5. リン

**正解:** 5

**解説**
解答　５

閉鎖性水域における富栄養化の制限因子となるものは、本問の選択肢のうち、リンである。富栄養化とは、生活排水や工業排水等に由来する窒素やリンによって、これらを栄養素とする植物プランクトンが異常繁殖する現象であり、主に閉鎖性水域（湖沼や海域）で発生しやすい。なお、この現象が海域で発生した場合を赤潮、湖沼で発生した場合を水の華という。
