# 第101回 問71

**問題文**
インフォームド・コンセントに関する記述のうち、正しいのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-71 を参照

**選択肢**
1. 治療におけるデメリットについても説明する必要がある。
2. プライバシーの保護のために必須とされている。
3. 医療従事者が治療を受ける場合には必要とされない。
4. 説明を受ける側の理解度は考慮しなくてよい。
5. パターナリズムに基づくものである。

**正解:** 1

**解説**
解答　１

インフォームド・コンセントとは「患者が医師から十分な説明を受け、治療等の妥当性、有効性及び安全性に関する事項についても理解した上で、患者の自由意志により、患者が受けたい医療を選択・決定し、医師の説明に同意すること」である。そのため、本人の意思に反して行動に介入・干渉するような考え方（パターナリズム）ではなく、患者の意思を尊重するものである。

１　正

２　誤
インフォームド・コンセントは、患者が医師から十分な説明を受け、治療等の妥当性、有効性及び安全性に関する事項についても理解した上で、患者の自由意志により、患者が受けたい医療を選択・決定し、医師の説明に同意することであり、患者のプライバシーの保護を目的とするものではない。

３　誤
医療従事者が治療を受ける場合、治療を受ける医療従事者は患者に該当するため、インフォームド・コンセントが必要である。

４　誤
説明を受ける側の理解度を考慮し、医師は理解しやすい言葉で、分かりやすい説明をすることが重要である。

５　誤
パターナリズムとは、患者の意思に反して医師等が行動に介入・干渉するような考え方である。インフォームド・コンセントは、「患者が医師から十分な説明を受け、治療等の妥当性、有効性及び安全性に関する事項についても理解した上で、患者の自由意志により、患者が受けたい医療を選択・決定する」という患者の自己決定権を尊重する観点が重要視されるので、パターナリズムに基づくものではない。
