# 第101回薬剤師国家試験 問266

## 問題文

リドカインの代謝・消失に関する記述として正しいのはどれか。**2つ選べ。**[1][2]

## 選択肢

1. 肝血流量が低下している患者では、リドカインの血中濃度は減少する。[1][2]

2. リドカインは、肝初回通過効果を受けやすい。[2]

3. リドカインの肝クリアランスは、血中タンパク質非結合率に依存する。[2]

4. リドカインは、肝ミクロソーム酵素により脱エチル化を受ける。[2]

5. リドカインの活性代謝物は、親薬物より毒性が低い。[2]

## 正解

**2、3**[1][2]

## 解説

**選択肢1（誤り）**

リドカインは**肝血流依存型薬物**である。肝血流量の低下に伴って、リドカインの肝クリアランスが低下し、血中濃度が**上昇**することがある。したがって、血中濃度が減少するという記述は誤りである。[1][2]

**選択肢2（正しい）**

リドカインは肝初回通過効果を受けやすく、経口投与では著しく吸収が低下するため、臨床では静脈内投与または硬膜外投与が用いられる。[2]

**選択肢3（正しい）**

リドカインの肝クリアランスは、血中タンパク質非結合率に依存する。肝クリアランスは肝血流量と肝抽出率により決定され、肝血流依存型薬物ではタンパク質結合率の変化がクリアランスに影響を与える。[2]
