# 第101回 問153

### 第101回薬剤師国家試験 問153

**問題文**  
自律神経節遮断薬の効果として正しいのはどれか。２つ選べ。[1][2][4][6]

**全選択肢（番号付き）**  
1. 血圧降下  
2. 頻脈  
3. 唾液分泌亢進  
4. 縮瞳  
5. 消化管運動亢進[1][2][4][6]

**正解番号**  
1, 2[1][2][4]

**解説**  
自律神経節遮断薬は交感神経と副交感神経の両方を遮断するが、各臓器で優位な神経の影響が強く現れる。交感神経優位の臓器（血管、汗腺、心臓）では交感神経の遮断効果が顕著となり、血圧降下（血管拡張）と頻脈（心拍数増加）が起こるため、選択肢1と2は正しい。[1][4]  

一方、副交感神経優位の臓器では副交感神経の遮断によりその作用が抑制される：  
- 選択肢3：唾液腺は副交感神経優位のため、唾液分泌が**減少**（亢進ではない）。[1][4]  
- 選択肢4：瞳孔括約筋は副交感神経優位のため、瞳孔が**散瞳**（縮瞳ではない）。[1][4]  
- 選択肢5：消化管は副交感神経優位のため、消化管運動が**抑制**（亢進ではない）。[1][4]  

以上より、正解は1と2。[1][2][4]
