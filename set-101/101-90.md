# 第101回 問90

**問題文**
大雨の翌日、床上浸水の被害にあった男性が、汚水に浸かった室内を消毒する目的で薬局を訪れた。この男性に提案する消毒剤として最も適切なのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-90 を参照

**選択肢**
1. グルタラール液
2. ベンザルコニウム塩化物液
3. ポピドンヨード液
4. アクリノール液
5. オキシドール

**正解:** 2

**解説**
解答　２

汚水に浸かった室内や、片付け時に汚染された手指を消毒する目的で使用される消毒剤としては、ベンザルコニウム塩化物液を提案する。また、食器類を消毒する場合は、次亜塩素酸ナトリウム液を勧める。
