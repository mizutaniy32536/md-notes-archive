# 第101回 問67

**問題文**
ヘリコバクター・ピロリの除菌治療に用いられない薬物はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-67 を参照

**選択肢**
1. ランソプラゾール
2. アモキシシリン
3. メトロニダゾール
4. ファモチジン
5. クラリスロマイシン

**正解:** 4

**解説**
解答　４

ヘリコバクター・ピロリの除菌治療では、まず一次除菌でプロトンポンプ阻害薬（ランソプラゾールなど）、アモキシシリン、クラリスロマイシンの3剤併用投与を行う。一次除菌に失敗した場合は二次除菌として、クラリスロマイシンをメトロニダゾールに変更しての3剤併用投与を行う。したがって、ヒスタミンH2受容体遮断薬であるファモチジンは、ヘリコバクター・ピロリの除菌治療には用いられない。
