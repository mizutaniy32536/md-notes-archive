# 第101回薬剤師国家試験 問259

## 問題文

62歳女性。3年前に糖尿病と診断され、処方1及び処方2で治療中。最近、手足に痛みやしびれ感があるため処方3が追加となった。

**処方1～3の薬物の作用機序として正しいのはどれか。2つ選べ。**[1][2]

## 選択肢

選択肢の詳細は検索結果に完全に記載されていませんが、以下の内容が確認できます：[1]

1. アルドース還元酵素を阻害し、末梢神経障害を改善する
2. AMP依存性プロテインキナーゼを活性化し、肝臓での糖新生を抑制する
3. ペルオキシソーム（関連内容）
4. 血液を固まりにくくし、血のめぐりを良くすることで痛みをやわらげる
5. DPP-4阻害作用（アログリプチン）

## 正解

**2、3**[1]

（別の解説では **2、5** という記載もあります[1]）

## 解説

**選択肢1（誤り）**[1]
アルドース還元酵素阻害剤はエパルレスタット（キネダック）であり、処方1～3には含まれていない。

**選択肢2（正）**[1]
AMP依存性プロテインキナーゼを活性化し、肝臓での糖新生を抑制する作用機序は処方薬に該当する。

**選択肢3（正）**[1]
ペルオキシソーム関連の作用機序として正しい。

**選択肢4（誤り）**[1][2]
リリカ（プレガバリン、処方3）はCaチャネルα2σリガンドに分類されるGABA誘導体であり、血液を固まりにくくする作用はない。本剤は興奮性神経伝達物質の放出を抑制することにより神経障害性疼痛に対して効果を示す。

**選択肢5**[1]
アログリプチンのDPP-4阻害作用は正しい作用機序。
