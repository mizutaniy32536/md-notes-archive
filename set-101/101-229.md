# 第101回薬剤師国家試験 問229

## 問題文

56歳女性。パーキンソン病及び慢性胃炎で治療中のため、保険薬局に処方箋を持参した。処方1を継続服用していたが、振戦が改善されないため、本日より処方2が追加された。薬剤師は服薬指導の際に、患者にチーズを食べすぎないように指導した。

**チーズの中の成分が問題となる理由として、最も適切なのはどれか。1つ選べ。**[1][2]

## 全選択肢

1. この食品成分が、胃に障害を与え胃炎を悪化させる。[1]

2. この食品成分が、処方された薬物の代謝酵素を阻害する。[1]

3. この食品成分が、処方された薬物の代謝酵素の発現を誘導する。[1]

4. この食品成分が、処方された薬物の消化管からの吸収を阻害する。[1]

5. 処方された薬物が、この食品成分を代謝する酵素を阻害する。[1]

6. 処方された薬物が、この食品成分を代謝する酵素の発現を誘導する。[1]

## 正解番号

**5番**[3]

## 解説

問228の正解はチラミンであり、問229はその理由を問う問題です。パーキンソン病の治療薬として追加処方された**セレギリン**がポイントとなります。セレギリンはモノアミン酸化酵素（MAO）阻害薬であり、チラミンの代謝酵素を阻害します。チーズに含まれる高濃度のチラミンが、セレギリンによって代謝されずに蓄積すると、アドレナリン作動性の過剰反応が生じ、高血圧クリーゼなどの危険な副作用につながる可能性があります。したがって、正解は「処方された薬物（セレギリン）が、この食品成分（チラミン）を代謝する酵素を阻害する」という5番です。[3]
