# 第101回 問225

### 第101回薬剤師国家試験 問225

**問題文（全文）**  
65歳女性。以下の処方箋をもって保険薬局を訪れた。  
（※処方箋内容は問224と共通で、アレンドロン酸経口ゼリーなどが記載。問225は骨代謝に関する独立記述問題。）[1][2][3]  

**骨代謝に関する記述のうち、正しいのはどれか。** **2つ選べ。**（物理・化学・生物分野）[1][2][3]

**全選択肢（番号付き）**  
1. **骨芽細胞は、コラーゲンを細胞外へ分泌する。**[3]  
2. **骨細胞は未分化細胞であり、増殖能を有する。**[1][2][3]  
3. **破骨細胞は、アルカリ性ホスファターゼを細胞外へ分泌する。**[3]  
4. **副甲状腺（上皮小体）ホルモンは、破骨細胞の働きを減弱させる。**[1][2][3]  
5. **1α,25-ジヒドロキシビタミンD₃（活性型ビタミンD₃）は、小腸からのCa²⁺の吸収を促進する。**[1][2]

**正解番号**  
**1, 5**[1][2]

**解説**  
- **選択肢1 正しい**：骨芽細胞は骨基質の形成を担い、コラーゲンなどの基質を細胞外へ分泌する。これにより骨形成が促進される。[1][3]  
- **選択肢2 誤り**：骨細胞は骨芽細胞から分化した成熟した細胞であり、増殖能を有しない。[1][2]  
- **選択肢3 誤り**：破骨細胞は酸性ホスファターゼや加水分解酵素などを細胞外に分泌し、骨吸収を担う。アルカリ性ホスファターゼは骨芽細胞のマーカーである。[2]  
- **選択肢4 誤り**：副甲状腺ホルモン（PTH）は破骨細胞の働きを**活性化**（増強）させ、骨吸収を促進する。減弱させるのは誤り。[1][2]  
- **選択肢5 正しい**：活性型ビタミンD₃（1α,25-ジヒドロキシビタミンD₃）は小腸上皮細胞のカルシウム吸収を促進し、血中Ca²⁺濃度を上昇させる。[1][2][1]
