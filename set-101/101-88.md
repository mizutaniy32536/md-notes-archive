# 第101回 問88

**問題文**
「1件の重大事故の背後には29件の小さな事故があり、その背景には300件の事故に至らない事例がある」という経験則はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-88 を参照

**選択肢**
1. ドミノの法則
2. ハインリッヒの法則
3. マーフィーの法則
4. ムーアの法則
5. メラビアンの法則

**正解:** 2

**解説**
解答　２

１　誤
ドミノの法則とは、ドミノ倒しのように連鎖的に事象が起こっていく様子を表した法則である。

２　正
ハインリッヒの法則とは、労働災害を統計的に調査し導かれた法則であり、「1件の重大事故の背後には29件の小さな事故があり、その背景には300件の事故に至らない事例がある」という経験則である。

３　誤
マーフィーの法則とは、「失敗する余地があるなら、失敗する」など先人の経験から生じた経験則をまとめたものである。

４　誤
ムーアの法則とは、「半導体の集積密度は18〜24ヶ月で倍増する」という経験則である。

５　誤
メラビアンの法則とは、初対面で会ったときに受ける印象を決定付ける要素を分析したものであり、初対面で会ったときに影響を及ぼす割合は、「視覚情報が55％、聴覚情報が38％、言語情報が7％」であるとされている。
