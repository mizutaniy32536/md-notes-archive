# 第101回 問52

**問題文**
噴出する圧縮空気の気流により粒子を加速させて、粒子どうしあるいは粒子と容器壁との衝突により粒子を微細化する粉砕機はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-52 を参照

**選択肢**
1. ローラーミル
2. ボールミル
3. コロイドミル
4. ジェットミル
5. ハンマーミル

**正解:** 4

**解説**
解答　４

粉砕とは粒子に機械的な外力をくわえることで、粒子径を減少させる操作のことをいう。粉砕を行う装置である粉砕機は、原理の異なる装置がいくつか存在し、目的によって適切な装置を選択する必要がある。

１　誤
ローラーミルは、逆方向に回転するローラーの間に粒子を挟み込ませ、粉砕する粉砕機である。

２　誤
ボールミルは、円柱状の容器に粒子とボールをいれて、容器を回転させる。その時にボールと粒子がぶつかる衝撃などで粒子を粉砕する粉砕機である。

３　誤
コロイドミルはせん断応力を与えることで粉砕を行う粉砕機である。懸濁剤中の固体粒子や乳剤の分散相の粉砕に用いられる。

４　正
ジェットミルは、噴出する圧縮空気の気流により粒子を加速させて、粒子どうしあるいは粒子と容器壁との衝突により粒子を微細化する粉砕機である。

５　誤
ハンマーミルは、高速回転するハンマーで粒子に衝撃を与えて粉砕する粉砕機である。
