# 第101回 問55

**問題文**
マトリックス型放出制御製剤からの薬物放出がHiguchi式に従うとき、時間tまでの単位面積当たりの累積薬物放出量について、正しい記述はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-55 を参照

**選択肢**
1. tに比例する。
2. tの平方根に比例する。
3. tの立方根に比例する。
4. tに反比例する。
5. tの平方根に反比例する。
6. tの立方根に反比例する。

**正解:** 2

**解説**
解答　２

Higuchi式は水不溶性マトリックス中に溶解した薬物が、マトリックス表面から放出すると仮定して導かれた式であり、マトリックスからの薬物放出を示すものである。以下の①式で表される。

上記の式より時間tまでの単位面積当たりの累積薬物放出量（Q）は時間の平方根（t1/2）に比例する。
