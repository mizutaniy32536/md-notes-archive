# 第101回 問332

**問題文**
非小細胞肺がんに対して外来化学療法室でカルボプラチン／パクリタキセル（TC）療法を実施することになり、レジメンに従って以下の処方が出された。これら以外には内服薬、注射薬ともに処方されていない。これらの処方に関する記述として誤っているのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-332 を参照

**選択肢**
1. パクリタキセルとカルボプラチンの投与順はどちらが先でもよい。
2. アルコールに過敏であるかを事前に確認する必要がある。
3. カルボプラチンは高度の催吐性リスクに分類される。
4. 重大な副作用として末梢神経障害がある。
5. 腎機能を考慮して投与量を決定する。

**正解:** 1,3

**解説**
解答　１、３

１　誤っている
カルボプラチン／パクリタキセル（TC）療法では、パクリタキセル投与後にカルボプラチンを投与する。なお、カルボプラチンをパクリタキセルの前に投与すると、パクリタキセルの排泄が抑制され、パクリタキセルによる副作用である骨髄抑制が現れやすくなる。

２　正しい
パクリタキセル注射液は添加物として、無水エタノールを含有しているため、アルコールに過敏な患者に投与するとアルコールの中枢神経系への影響が強く現れることがある。そのため、（TC）療法を実施する際には、アルコール過敏であるかを事前に確認する必要がある。

３　誤っている
カルボプラチンは中等度の催吐性リスク（催吐頻度：30〜90％）に分類され、パクリタキセルは軽度の催吐性リスク（催吐頻度：10〜30％）に分類される。

４　正しい
パクリタキセルは重大な副作用として、末梢神経障害を起こすことがある。

５　正しい
カルボプラチンは腎消失型の薬物であるため、投与する際には腎機能を考慮して投与量を決定する必要がある。なお、カルボプラチンの投与量はカルバートの式により算出することができる。
カルバートの式：投与量＝目標AUC×（GFR＋25）
