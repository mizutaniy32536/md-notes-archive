## ◆ ワクチンの知識に関する正誤判断シナリオ
260312

**問題**
薬剤師は疾病予防にも関わるべきであり、ワクチンに関する知識を深める必要がある。
薬剤師として知っておくべきワクチンに関する記述のうち、誤っているのはどれか。1つ選べ。

**選択肢**

1. 麻疹、風疹、水痘及びおたふくかぜのワクチン接種不適当者には妊婦が含まれる。
​
2. B型肝炎ワクチンは抗体価を検査しながら複数回接種することがある。
​
3. 副腎皮質ホルモン薬の使用者に対しては、生ワクチンの接種は避けるべきである。
​
4. インフルエンザワクチンは、卵アレルギーの人に対しては注意して接種する。
​
5. 結核に対する抗体がない場合は、ツベルクリンを接種する。

**正解と理由**
- `5. 結核に対する抗体がない場合は、ツベルクリンを接種する。`
- **理由の構成**
	- ツベルクリンは`検査薬`であり、ワクチンではない
	- 抗体がない場合に接種するのは`BCGワクチン`

**解法の手順（思考の流れ）**
- 選択肢1：麻疹・風疹・水痘・おたふくかぜワクチンはすべて`生ワクチン` → 妊婦は接種不適当 → 正しい
- 選択肢2：B型肝炎ワクチンは3回接種、抗体価を確認しながら追加接種あり → 正しい
- 選択肢3：副腎皮質ホルモン薬 = 免疫抑制 → 生ワクチンで感染リスク → 正しい
- 選択肢4：インフルエンザワクチンは鶏卵で培養 → 卵アレルギーに注意 → 正しい
- 選択肢5：ツベルクリンは`結核感染の有無を調べる検査薬` → ワクチンではない → **誤り**

**キーポイント**
- `ツベルクリン` = 検査薬（結核菌タンパク質の精製物）
- `BCG` = 結核予防ワクチン（弱毒生ワクチン）
- 以前はツベルクリン検査陰性 → BCG接種の流れだった
- 現在は直接BCG接種（生後1歳まで）

**関連知識の比較**

| 項目 | `ツベルクリン` | `BCG` |
|---|---|---|
| 分類 | 検査薬 | 生ワクチン |
| 目的 | 結核感染の有無を調べる | 結核の予防 |
| 接種対象 | （現在は原則不要） | 生後1歳未満 |

（一言メモ）
- ツベルクリン ≠ ワクチン。これは超頻出のひっかけ
- 生ワクチン4つ（麻疹・風疹・水痘・おたふく）は妊婦禁忌
