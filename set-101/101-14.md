# 第101回 問14

**問題文**
DNAウイルスはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-14 を参照

**選択肢**
1. インフルエンザウイルス
2. ポリオウイルス
3. C型肝炎ウイルス
4. ヒトパピローマウイルス（HPV）
5. ヒト免疫不全ウイルス（HIV）

**正解:** 4

**解説**
解答　４

ゲノムとしてDNAを有するものをDNAウイルス、RNAを有するものをRNAウイルスという。
選択肢のうち、DNAウイルスは、ヒトパピローマウイルス（HPV）である。
なお、インフルエンザウイルス、ポリオウイルス、C型肝炎ウイルス、ヒト免疫不全ウイルス（HIV）は、いずれもRNAウイルスである。
