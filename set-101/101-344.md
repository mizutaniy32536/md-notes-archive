# 第101回 問344

**問題文**
7歳女児。近医の皮膚科にてアトピー性皮膚炎と診断され、母親が以下の処方箋を保険薬局に持参した。＜image1:center＞薬剤師が母親に服薬指導する内容として、適切でないのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-344 を参照

**選択肢**
1. タクロリムス軟膏は、傷やただれているところに使用して下さい。
2. ベタメタゾン吉草酸エステルクリームは、症状が改善されても自己判断で中止しないで下さい。
3. ヘパリン類似物質クリームは、皮膚の保湿効果があります。
4. ヘパリン類似物質クリームは、傷やただれがあるところに塗らないで下さい。
5. フェキソフェナジン塩酸塩は、眠気を起こしにくい薬です。

**正解:** 1

**解説**
解答　１

１　不適切
タクロリムス軟膏を傷やただれているところに使用すると、タクロリムスの血中濃度が高くなり腎障害などの副作用が現れやすくなる。そのため、タクロリムス軟膏は、傷やただれているところに使用しないように説明する必要がある。

２　適切
ベタメタゾン吉草酸エステルは副腎皮質ステロイド性薬であり、急な減量や投与中止により、ステロイド離脱症候群が誘発されることがある。そのため、「ベタメタゾン吉草酸エステルクリームは、症状が改善されても自己判断で中止しないで下さい」と説明することは適切である。

３　適切
ヘパリン類似物質は、角質水分保持増強作用を有するため、皮膚の保湿効果を有する。そのため、「ヘパリン類似物質クリームは、皮膚の保湿効果があります」と説明することは適切である。

４　適切
 ヘパリン類似物質の添付文書に「潰瘍、びらん面への直接の塗擦又は塗布を避けること」と記載されている。そのため、「ヘパリン類似物質クリームは、傷やただれがあるところに塗らないで下さい」と説明することは適切である。

５　適切
フェキソフェナジンは他の抗ヒスタミン薬に比べ、中枢移行率が低く眠気を誘発しにくい。そのため、「フェキソフェナジン塩酸塩は、眠気を起こしにくい薬である」と説明することは適切である。
