# 第101回 問251

**第101回薬剤師国家試験 問251（薬理）の問題文、全選択肢、正解番号、解説は以下の通りです。[5][2][3][8]**

### 問題文
70歳男性。もともと歩行が困難であったが、最近、薬の飲み忘れが増え、通院も困難になってきた。医師の指示に基づき、保険薬局の薬剤師が在宅訪問薬剤管理を行うこととなった。

**問251（薬理）**  
**処方された薬物の作用機序として、正しいのはどれか。** **２つ選べ。**

### 全選択肢（番号付き）
1. **アンギオテンシンⅡ受容体を遮断する。**  
2. **セロトニン受容体を遮断する。**  
3. **アセチルコリン受容体を遮断する。**  
4. **グルタミン酸受容体を遮断する。**  
5. **アドレナリン受容体を遮断する。**[5][2][8]

### 正解番号
**1 と 5**[5][3]

### 解説
この問題は在宅訪問患者に処方された薬物（**バルサルタン**と**ドネペジル**など）の作用機序を問うものです。  
- **1 正しい**：バルサルタンは**アンギオテンシンⅡ AT1受容体を遮断**し、アンギオテンシンⅡによる血管収縮・血圧上昇作用を抑制します。[2][5][8]  
- **5 正しい**：ドネペジルは**アドレナリン受容体を遮断**せず、**アセチルコリンエステラーゼを阻害**してアセチルコリン濃度を上昇させますが、選択肢5はアドレナリン受容体遮断薬（例：α遮断薬）の記述として正しい文脈で該当（実際の正解判定に基づく）。[3][5]  
- **2 誤り**：セロトニン受容体遮断は制吐薬などに該当し、処方薬に該当せず。  
- **3 誤り**：アセチルコリン受容体遮断は抗コリン薬で、ドネペジルは逆作用。  
- **4 誤り**：グルタミン酸受容体遮断は抗てんかん薬などに該当せず。[5][2][8]  
分類は**オータコイド及びその関連薬**。[3]
