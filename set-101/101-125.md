# 第101回 問125

**問題文**
ウェルシュ菌及びウェルシュ菌による食中毒に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-125 を参照

**選択肢**
1. ウェルシュ菌は、Clostridium botulinumという学名の偏性嫌気性菌である。
2. 最近（2011年以降）の1年あたりの発生件数は、腸炎ビブリオによる食中毒より少ない。
3. 最近（2011年以降）の食中毒1件あたりの平均患者数は、カンピロバクターによる食中毒に比べて多い。
4. 給食や学生食堂において、食肉を調理したカレーやシチューなどの食品が原因となることが多い。
5. 潜伏期間は3日〜1週間と長く、主要症状は腹痛と水溶性下痢である。

**正解:** 3,4

**解説**
解答　３、４

１　誤
ウェルシュ菌は、Clostridium perfringensという学名であり、芽胞を形成する偏性嫌気性菌である。なお、Clostridium botulinumは、ボツリヌス菌の学名である。

２　誤
最近（2011年以降）の1年あたりの発生件数は、ウェルシュ菌が20件前後、腸炎ビブリオが10件未満である。よって、最近（2011年以降）のウェルシュ菌による食中毒の発生件数は、腸炎ビブリオによる食中毒より多い。

３　正
ウェルシュ菌による食中毒は集団発生を起こしやすいため、食中毒の1件あたりの平均患者数は、ウェルシュ菌による食中毒の方が、カンピロバクターによる食中毒より多い。なお、最近（2011以降）の食中毒1件あたりの平均患者数は、ウェルシュ菌が約50〜100人、カンピロバクターが約10人である。

４　正
ウェルシュ菌は、ヒトや動物の腸管、土壌などに存在するため、食肉を調理したカレーやシチューが主な原因食品となる。給食や学生食堂などで使用する大型鍋は、鍋底が嫌気的になりやすいため、偏性嫌気性菌であるウェルシュ菌は増殖しやすい。また、ウェルシュ菌は芽胞を形成するため加熱しても死滅しない。カレーやシチューなどの食品を室温で放置したときに、菌が発育しやすい至適発育温度（43〜45℃）まで食品の温度が下がると、発芽して急速に菌が増殖する。ゆえに、ウェルシュ菌による食中毒は、給食や学生食堂などで集団発生を起こしやすく、1件あたりの平均患者数が多くなる傾向にある。

５　誤
ウェルシュ菌による食中毒の潜伏期間は6〜18時間であり、菌が産生するエンテロトキシンによって主に腹痛と下痢を引き起こす。
