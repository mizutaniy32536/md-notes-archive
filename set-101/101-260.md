# 第101回 問260

**問題文**
70歳女性。1ヶ月前から四肢の筋力低下と、眠れないほどの四肢及び腰部の痛みがある。整形外科を受診し、以下の薬剤が処方された。＜image1:center＞問260（実務）上記の処方に関する服薬指導として適切なのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-260 を参照

**選択肢**
1. 服用後、なるべく早く食事をとるように指導する。
2. 服用後、横になって安静にするように指導する。
3. 歯科を受診する場合に、服用中の薬剤について歯科医師に告知するように指導する。
4. 自動車の運転など、危険を伴う機械の操作に従事しないように指導する。
5. 腰痛が改善したら、直ちに服薬を中止するように指導する。問261（薬理）リセドロン酸の作用機序として正しいのはどれか。1つ選べ。
6. 骨のエストロゲン受容体に結合し、骨芽細胞を増加させ骨形成を促進する。
7. ヒドロキシアパタイトに結合したのち骨芽細胞に取り込まれ、アポトーシスを起こす。
8. オステオカルシンのカルボキシ化を介し、前駆細胞から破骨細胞への分化を阻害する。
9. メバロン酸代謝経路のファルネシルピロリン酸合成酵素を阻害し、破骨細胞の機能を抑制する。
10. カルシウムの腸管からの吸収を促進し、血中Ca2＋値を上昇させる。

**正解:** 3,4

**解説**
問260　解答　３、４

１　誤
リセドロン酸Na錠は、水以外の飲料（Ca、Mg等の含量の特に高いミネラルウォーターを含む）や食物あるいは他の薬剤と同時に服用すると、本剤の吸収を妨げることがあるため、起床後、最初の飲食前に服用し、かつ服用後少なくとも30分は水以外の飲食を避ける必要がある。そのため、服用後、なるべく早く食事をとるように指導することは適切ではない。

２　誤
リセドロン酸Na錠は、服用後横になることなどによる食道への薬の曝露により、食道炎や食道潰瘍が報告されているため、立位あるいは坐位で、十分量（約180 mL）の水とともに服用し、服用後30分は横たわらないようにする。そのため、服用後、横になって安静にするように指導することは適切ではない。

３　正
リセドロン酸Na錠の重大な副作用として、顎骨壊死・顎骨骨髄炎があらわれることがある。リセドロン酸Na錠服用により現れた症例では、抜歯等の顎骨に対する侵襲的な歯科処置や局所感染に関連して発現しているため、歯科受診時に本剤の使用を歯科医師に告知して侵襲的な歯科処置はできる限り避けることなどを患者に十分説明し、異常が認められた場合には、直ちに歯科・口腔外科を受診するように指導する必要がある。そのため、歯科を受診する場合に、服用中の薬剤について歯科医師に告知するように指導することは適切である。

４　正
アルプラゾラム錠は、眠気、注意力・集中力・反射運動能力等の低下を起こすことがある。そのため、自動車の運転など、危険を伴う機械の操作に従事しないように指導することは適切である。

５　誤
本患者の腰痛は、処方内容から骨粗しょう症が原因であると推察されることから、腰痛が改善しても骨粗しょう症の治療のため、服薬は続けるべきである。そのため、腰痛が改善したら、直ちに服薬を中止するように指導することは適切ではない。

問261　解答　４

リセドロン酸はビスホスホネート製剤であり、骨のヒドロキシアパタイトに結合し破骨細胞に取り込まれることで、メバロン酸代謝経路のファルネシルピロリン酸合成酵素を阻害する。その結果、破骨細胞の機能が抑制され、骨吸収が抑制される。
