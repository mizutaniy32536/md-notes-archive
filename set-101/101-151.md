# 第101回 問151

**問題文**
次の筋細胞に存在する異なった標的分子に働き、収縮機能に対して相反する作用を示す薬物の組合せはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-151 を参照

**選択肢**
1. 血管平滑筋細胞におけるアセチルコリンとフェニレフリン
2. 瞳孔括約筋細胞におけるアセチルコリンとネオスチグミン
3. 気管支平滑筋細胞におけるアセチルコリンとイソプレナリン
4. 骨格筋細胞におけるアセチルコリンとツボクラリン
5. 骨格筋細胞におけるアセチルコリンとダントロレン

**正解:** 3,5

**解説**
解答　３、５

アセチルコリン（ACh）は、血管平滑筋細胞、瞳孔括約筋細胞、気管支平滑筋細胞に存在するムスカリン受容体を刺激することにより、これらの筋肉を収縮させる。また、骨格筋細胞に存在するニコチン受容体を刺激することにより、骨格筋を収縮させる。

１　誤
フェニレフリンは、血管平滑筋細胞に存在するアドレナリンα1受容体を刺激して、ACh同様、血管平滑筋を収縮させる。

２　誤
ネオスチグミンは、可逆的コリンエステラーゼ阻害薬であり、AChの分解を抑制しAChの作用を増強することで、ACh同様、瞳孔括約筋を収縮させる。

３　正
イソプレナリンは、非選択的アドレナリンβ受容体刺激薬であり、気管支平滑筋細胞に存在するアドレナリンβ2受容体を刺激して気管支平滑筋を弛緩させる。

４　誤
ツボクラリンは、骨格筋細胞に存在するAChと同様の標的分子であるニコチン受容体を競合的に遮断することで、AChによる骨格筋収縮作用を抑制する。

５　正
ダントロレンは、骨格筋細胞内の筋小胞体に存在するリアノジン受容体に作用し、筋小胞体からのCa2＋遊離を抑制することで骨格筋を弛緩させる。
