# 第101回 問275

### 第101回薬剤師国家試験 問275

#### 問題文
54歳女性。2年前に高血圧及びうっ血性心不全と診断され、以下の処方による薬物治療を受け、状態は安定していた。昨日、食欲不振と吐き気を訴え受診し、緊急入院となった。ジゴキシンの血中濃度を測定したところ、2.2 ng/mLであった。持参薬確認のため薬剤師が面談したところ、鼻水が出て喉が痛いなど風邪気味の症状のため1週間前に近医を受診し、そこで処方された薬を服用しているとのことであった。[1][2][4]

#### 全選択肢（薬剤区分）
この患者で起きている相互作用として可能性が高いのはどれか。**2つ選べ**。

1. **CYP3A4によるリシノプリルの代謝が抑制された。**  
2. **CYP3A4によるジゴキシンの代謝が促進された。**  
3. **P-糖タンパク質によるリシノプリルの排泄が促進された。**  
4. **P-糖タンパク質によるジゴキシンの排泄が抑制された。**  
5. **腎排泄型有機カチオン輸送体（OCT2）によるジゴキシンの排泄が抑制された。** [1][2][4][7]

#### 正解番号
**4, 5** [1][2][7]

#### 解説
患者はジゴキシン服用中で血中濃度が2.2 ng/mLと高値（中毒域）であり、風邪治療薬との相互作用が疑われる。問274の正解はクラリスロマイシンで、これがP-糖タンパク質（P-gp）およびOCT2を阻害し、ジゴキシンの排泄を抑制して血中濃度を上昇させた。[1][2][7]  
- 選択肢4: クラリスロマイシンがP-gpを阻害し、ジゴキシンの腸管・腎排泄を抑制。[1][7]  
- 選択肢5: クラリスロマイシンがOCT2を阻害し、ジゴキシンの腎排泄を抑制。[1][2]  
他の選択肢は不適切（例: 1はリシノプリルに関連せず、2は促進で逆効果、3はリシノプリルに関連せず）。[1][2]
