### 第101回薬剤師国家試験 問265

**問題文（全文）**  
57歳男性。胸部の激痛、呼吸困難、意識障害にて救急搬送された。心電図所見にて急性心筋梗塞と診断され、直ちにアルテプラーゼが投与された。容態は安定に向かっている。  
発症後24時間が経過した時点で心室性期外収縮が継続していたので、リドカインの投与が決定した。この症例に用いるリドカイン製剤として正しいのはどれか。1つ選べ。なお( )内は投与部位を示す。[1][7]

**全選択肢（番号付き）**

2. 筋注用0.5％溶解液製剤（筋肉内）[7]  
3. **静注用2％製剤（静脈内）**[7]  
4. 注射液2％製剤（硬膜外）[7]

**解説**
