# 第101回薬剤師国家試験 問287

## 問題文

40歳男性。活動期のクローン病と診断された。主治医より患者の栄養状態把握及び改善のため、院内栄養サポートチームに介入の依頼があった。

**クローン病に関する記述のうち、正しいのはどれか。2つ選べ。**[2]

## 全選択肢

1. 緩解と増悪を繰り返す。

2. 小腸及び大腸に病変が限局する。

3. 薬物治療により根治できる。

4. 好発年齢は10歳代後半から20歳代である。

5. ほとんどの症例に粘血便が見られる。

## 正解番号

**1、4**[2]

## 解説

**選択肢1（正）** クローン病は、緩解（寛解）と増悪を繰り返す疾患です。慢性的な経過を辿る特徴があります。[2]

**選択肢2（誤）** クローン病の病変は小腸及び大腸に限定されず、**口腔から肛門までの消化管全体に認められる可能性があります**。[2]

**選択肢3（誤）** クローン病は根治させる治療法がいまだ確立されていないため、薬物治療により根治することはできません。[2]

**選択肢4（正）** クローン病の好発年齢は10歳代後半から20歳代であり、若年層に多く見られる疾患です。[2]

**選択肢5（誤）** ほとんどの症例に粘血便が見られるわけではなく、症状の表れ方は患者ごとに異なります。
