# 第101回 問173

**問題文**
治療薬物モニタリング（TDM）に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-173 を参照

**選択肢**
1. 血清中ジゴキシン濃度を免疫学的測定法で測定する場合、腎障害患者や妊婦では内因性交差物質が測定値に影響を及ぼすことがある。
2. アジスロマイシン投与時には、第8脳神経障害の副作用を回避するためにTDMが行われる。
3. テオフィリンの投与量は、患者のクレアチニンクリアランスを指標に決定される。
4. TDMは、血中薬物濃度と薬効・副作用との間に相関がない薬物において有用である。
5. 薬物によっては血清分離剤に吸着するため、血清分離剤を含む採血管を使用した場合には、血清中濃度を低く見積もる場合がある。

**正解:** 1,5

**解説**
解答　１、５

１　正
血清中ジゴキシン濃度を免疫学的測定法で測定する場合、腎障害患者、肝障害患者、妊婦、新生児などではジゴキシン様免疫反応陽性物質が存在することがあり、測定値を実際よりも高く見積もる場合がある。

２　誤
第8脳神経障害の副作用を回避するためにTDMが行われる薬物として、カナマイシンやゲンタマイシンなどのアミノグリコシド系抗菌薬がある。選択肢中のアジスロマイシンは、マクロライド系抗菌薬であり、TDM対象薬物ではない。

３　誤
テオフィリンは肝消失型薬物であるため、投与量はクレアチニンクリアランスで決定されない。クレアチニンは腎臓の糸球体でろ過を受けて排泄されるため、クレアチニンクリアランスは腎機能の指標となるので、ジゴキシンなどの腎排泄型薬物の投与量を決定する際に用いられる。

４　誤
TDMは、血中薬物濃度と薬効・副作用との間に相関がある薬物において有用である。

５　正
フェニトインなどの薬物は、血清分離剤に吸着するため、血清分離剤を含む採血管を使用した場合には、血清中濃度を低く見積もる場合があるため注意が必要である。
