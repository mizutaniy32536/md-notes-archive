# 第101回 問315

### 第101回薬剤師国家試験 問315

**問題文（法規・制度・倫理）**  
肺炎球菌ワクチンは、生物由来製品として指定されている。生物由来製品に関する記述のうち、正しいのはどれか。**２つ選べ。**[3][1]

**選択肢**  
１ 製造販売業者が、その製品等の感染症に関する知見に基づいた評価を定期的に報告する制度がある。  
２ 遺伝子組換え技術を応用して製造される生物由来製品の添付文書には、その旨を記載しなければならない。  
３ 生物由来製品の添付文書に記載すべき事項として、厚生労働省令で定める事項がある。  
４ 特定生物由来製品に関しては、原材料に由来する感染症を完全に排除できない旨を添付文書に記載する必要がある。  
５ 生物由来製品は、すべて特定生物由来製品である。[3][1]

**正解番号**  
**１、２**[6][3]

**解説**  
- 選択肢１：生物由来製品の製造販売業者は、感染症に関する知見に基づく評価を定期的に報告する**生物由来製品等安全性情報提供制度**がある。これは法令に基づく正しい記述。[3]  
- 選択肢２：遺伝子組換え技術を応用して製造される場合、添付文書にその旨を記載する必要がある（医薬品医療機器等法施行規則）。[1][3]  
- 選択肢３：生物由来製品の添付文書記載事項は厚生労働省令で定められているが、選択肢の表現が不正確または重複するため誤り。実際の記載事項には生物由来製品の特性に関する注意、原料由来成分の名称などが含まれる。[1]  
- 選択肢４：特定生物由来製品に限定される記載義務であり、すべての生物由来製品に適用されないため誤り。[1]  
- 選択肢５：生物由来製品と特定生物由来製品は区別され、全てが特定とは限らないため誤り。[1][3]  
この問題は、生物由来製品の法規制（添付文書記載事項や報告制度）を問うもので、問314（実務）と連動。[3][6]
