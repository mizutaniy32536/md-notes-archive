# 第101回 問165

**問題文**
抗悪性腫瘍薬に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-165 を参照

**選択肢**
1. エトポシドは、DNA鎖に架橋を形成し、DNA合成を阻害する。
2. ビンクリスチンは、チューブリンの重合を阻害して微小管分解を引き起こし、細胞分裂を抑制する。
3. シタラビンは、細胞内で三リン酸ヌクレオチドに変換され、トポイソメラーゼⅡを阻害してDNA合成を阻害する。
4. ゲフィチニブは、上皮増殖因子受容体（EGFR）チロシンキナーゼを選択的に阻害する。
5. イマチニブは、血管内皮細胞増殖因子受容体（VEGFR）チロシンキナーゼを選択的に阻害し、血管新生を阻害する。

**正解:** 2,4

**解説**
解答　２、４

１　誤
エトポシドは、トポイソメラーゼⅡを阻害することで、DNA合成を阻害する。なお、DNA鎖に架橋を形成する抗悪性腫瘍薬としては、シスプラチンなどの白金製剤が挙げられる。

２　正
ビンクリスチンは、微小管の構成タンパク質であるチューブリンに結合し、チューブリンの重合を阻害することで細胞分裂（M期）を抑制する。

３　誤
シタラビンは、ピリミジン代謝拮抗薬であり、細胞内でリン酸化を受け、シタラビン三リン酸ヌクレオチドに変換される。デオキシシチジン三リン酸（dCTP）と競合することで、DNAポリメラーゼを阻害し、DNA合成を阻害する。

４　正
ゲフィチニブは、上皮増殖因子受容体（EGFR）チロシンキナーゼを選択的に阻害することで腫瘍細胞の増殖を抑制し、抗腫瘍作用を示す。

５　誤
イマチニブは、Bcr−Ablチロシンキナーゼを阻害することで、腫瘍細胞の増殖を抑制し、抗腫瘍作用を示す。なお、血管内皮細胞増殖因子受容体（VEGFR）チロシンキナーゼを阻害する抗悪性腫瘍薬としては、アキシチニブなどがある。
