# 第101回 問277

### 第101回薬剤師国家試験 問277

**問題文（全文）**  
50歳女性。2型糖尿病と診断され内服薬で治療を行っていたが、血糖コントロール不良のため、インスリン導入目的で入院となり、以下の薬剤をペン型注入器を用いて投与することとなった。  
下図は、今回処方された注射剤（**ヒトイソフェンインスリン水性懸濁注射液**）とインスリンヒト注射液を、それぞれヒトの皮下に投与した後のインスリン血中濃度推移を示している。  
処方された注射剤に該当する血中濃度推移と、この製剤の特徴に関する記述の組合せのうち、正しいのはどれか。**1つ選べ**。なお、処方された注射剤には、添加剤として**プロタミン硫酸塩**が含まれている。[2][3][5]

**全選択肢（番号付き）**  
（注: 選択肢は血中濃度推移の図示（a〜i?）と特徴記述の組合せ形式。図は原文にあり、ヒトイソフェンインスリン水性懸濁注射液は**イ**（ゆるやかに上昇し維持される曲線）に該当。[1][2]）  
1〜3: 不明（誤り組合せ）  
4: 血中濃度推移 **イ** + 特徴（一部正）  
5: 血中濃度推移 **イ** + 特徴（一部正）  
6: 血中濃度推移 **イ** + **インスリンと添加剤であるプロタミンが「溶解性の低い複合体」を形成することで吸収をゆっくりにした製剤** [1][2][3][5]

**正解番号**  
**6** [1][3]

**解説**  
- **ヒトイソフェンインスリン水性懸濁注射液**は中間型インスリン製剤で、**プロタミン硫酸塩**が添加され、インスリンとプロタミンが**溶解性の低い複合体**を形成することで皮下からの吸収が遅延し、血中濃度が**ゆるやかに上昇し長時間維持**される（曲線 **イ**）。これに対し、インスリンヒト注射液（速効型）は急峻に上昇する。[1][2][3][5]  
- 正解は血中濃度推移 **イ** と上記特徴の組合せである**6**。他の選択肢は曲線または特徴が誤り。[1]
