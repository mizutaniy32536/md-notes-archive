# 第101回 問28

**問題文**
リトドリンの子宮収縮抑制作用機序はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-28 を参照

**選択肢**
1. アドレナリンα1受容体遮断
2. アドレナリンα2受容体刺激
3. アドレナリンβ1受容体遮断
4. アドレナリンβ2受容体刺激
5. アドレナリンβ3受容体遮断

**正解:** 4

**解説**
解答　４

リトドリンは、アドレナリンβ2受容体刺激薬であり、子宮平滑筋のβ2受容体を刺激することにより、子宮収縮抑制作用を示す。切迫流・早産の防止に用いられる。
