# 第101回 問187

**問題文**
58歳男性。21歳のとき、統合失調症を発症。その後、精神科への入退院を繰り返し、昨日、本院に入院。検査の結果、耐糖能異常が診断された。薬歴として以下の処方が記載されていた。＜image1:center＞この患者の処方に対して薬剤師が、医師に処方変更を提案するとき、削除を提案すべき薬剤はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-187 を参照

**選択肢**
1. ハロペリドール錠
2. オランザピン錠
3. フルニトラゼパム錠
4. 酸化マグネシウム錠
5. フルバスタチン錠

**正解:** 2

**解説**
解答　２

本問の処方薬のうち、オランザピン錠は、「血糖上昇による糖尿病性ケトアシドーシス、糖尿病性昏睡を起こすことがあるため、糖尿病の患者あるいは糖尿病の既往がある患者には投与しないこと」との緊急安全性情報（イエローレター）が発出されている。
本患者は、検査により耐糖能異常と診断されており、血糖上昇を起こす状態にあるため、削除を依頼すべき薬剤はオランザピン錠である。
