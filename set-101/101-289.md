# 第101回 問289

### 第101回薬剤師国家試験 問289

**問題文（前問の症例を含む）**  
60歳男性。2年前にうつ病と診断され、薬物治療を行ってきた。ここ数ヶ月、仕事が多忙になり、気分の落ち込みが激しくなった。本日受診した結果、主治医はこれまでの**抗うつ薬**を1錠から2錠に増量した。[2][3][4]

**問289（実務）**  
薬局薬剤師が、前問（問288）の重大な副作用を早期発見するために患者にあらかじめ説明する事項として、**適切でない**のはどれか。１つ選べ。[2][3][4]

**選択肢**  
１ **高熱が出るようでしたら、お知らせください。**  
２ **筋肉がピクピク動くようでしたら、お知らせください。**  
３ **下痢が続くようでしたら、お知らせください。**  
４ **発汗が続くようでしたら、お知らせください。**  
５ **手のひらや足の裏が赤くなり、ひび割れが起こるようでしたら、お知らせください。**[2][3][4]

**正解番号**  
**５**[2][3]

**解説**  
問288の正解は**２ セロトニン症候群**であり、主に**SSRI**（選択的セロトニン再取り込み阻害薬）などの抗うつ薬増量時に注意すべき重大な副作用である。セロトニン症候群の主な症状は**高熱・発汗・下痢・ミオクローヌス（筋肉のピクピクした無意識運動）**などであり、選択肢１～４はこれらを早期発見するための適切な説明である。一方、選択肢５は**手足症候群**（主にカペシタビンやスニチニブなどの抗がん剤による手のひら・足の裏の紅斑・腫脹・ひび割れなどの症状）の特徴であり、セロトニン症候群とは無関係のため適切でない。[2][3]
