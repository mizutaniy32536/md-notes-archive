# 第101回薬剤師国家試験 問295

## 問題文

術後2年経過時に、高カルシウム血症や脊髄圧迫症候など骨転移にともなう合併症状が現れた。骨転移や、その合併症状に対して用いられる薬剤はどれか。**2つ選べ。**[2][4]

## 全選択肢

1. オマリズマブ
2. メナテトレノン
3. ゾレドロン酸水和物
4. デノスマブ
5. ラロキシフェン塩酸塩

## 正解番号

**3（ゾレドロン酸水和物）、4（デノスマブ）**[4]

## 解説

乳がんは骨転移が多いがんであり、骨転移により破骨細胞の活性化が起こります[4]。

**正解の薬剤：**

- **ゾレドロン酸水和物**：ビスホスホネート系薬剤で、破骨細胞の機能を抑制し、骨転移による合併症を予防・治療します。
- **デノスマブ**（プラリア）：RANKLに対するモノクローナル抗体で、破骨細胞の形成と機能を抑制します。[4]

**誤りの選択肢：**

- 選択肢1（オマリズマブ）：喘息やアレルギー疾患に用いられるIgE抗体で、骨転移治療に用いられません。
- 選択肢2（メナテトレノン）：ビタミンK2で、骨代謝改善薬ですが骨転移治療の第一選択ではありません。
- 選択肢5（ラロキシフェン塩酸塩）：選択的エストロゲン受容体調節薬（SERM）で、骨粗鬆症治療に用いられますが、骨転移治療には適切ではありません。[2]
