# 第101回 問331

**問題文**
薬剤師が以下の処方箋を受け取った。薬局には図中1〜5の調剤器具がある。本処方箋の調剤に使用する調剤器具として適切なのはどれか。2つ選べ。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-331 を参照

**選択肢**
（選択肢は画像を参照）

**正解:** 3,4

**解説**
解答　３、４

処方１では、細粒剤が単剤で処方されており、調剤する際、スパーテル（選択肢4）を用いる。
処方2では、軟膏剤が2剤処方されており、両剤を混合するよう指示されていることから、調剤する際、軟膏板と軟膏へら（選択肢3）を用いる。
なお、選択肢１（メートグラス）は、内用液剤の秤量、選択肢2（乳鉢、乳棒）は、散剤の混合、錠剤の粉砕、選択肢5（ふるい）は、錠剤を粉砕した際の篩過に用いられる。
