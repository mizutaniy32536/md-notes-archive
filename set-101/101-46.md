# 第101回 問46

**問題文**
全身クリアランスが50 L/hである薬物を10 mg/hの速度で点滴静注した場合の定常状態における血中濃度（µg/mL）に最も近い値はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-46 を参照

**選択肢**
1. 0.2
2. 0.5
3. 2
4. 5
5. 50

**正解:** 1

**解説**
解答　１

定常状態とは投与速度＝消失速度となる時点のことであり、定常状態における血中濃度Cssは点滴静注の場合、①式で表すことができる。

全身クリアランス50 L/h、点滴速度10 mg/hを代入し、定常状態における血中濃度（µg/mL）を求めると、

Css＝0.2 mg/L＝0.2 µg/mL　となる。
