# 第101回 問203

**第101回薬剤師国家試験 問203（物理・化学・生物）**

ヘリコバクター・ピロリ菌の感染診断には尿素呼気試験法が用いられる。この方法は、**13Cで標識した尿素を服用し、ピロリ菌が持つウレアーゼ活性で生じる呼気中の二酸化炭素を赤外吸収スペクトル法で検出するものである。これに関する記述のうち、正しいのはどれか。2つ選べ。**[3][4]

1. 赤外線は紫外線よりも短波長である。[6]  
2. 赤外線は分子振動によって双極子モーメントが変化するときに吸収される。[2][4][6][8]  
3. 赤外吸収スペクトル法では、紫外可視吸光度法とは異なり、ランベルト－ベール（Lambert-Beer）の法則が成立しない。[2]  
4. **13CO2の逆対称伸縮振動は12CO2と比べ低い波数側に検出される。**[6][9]  
5. **12CO2の測定には2350cm⁻¹付近の吸収が用いられ、これは指紋領域に含まれる。**[4][6]

**正解番号：2, 4**[4][6]

**解説**  
- **選択肢1**：誤。赤外線（波長約0.78μm〜1000μm）は紫外線（200nm〜400nm）より**長波長**である。[6]  
- **選択肢2**：正。赤外吸収は分子の振動（伸縮・変形）により双極子モーメントが変化する際に生じる。[2][4][6][8]  
- **選択肢3**：誤。赤外吸収スペクトル法でもランベルト－ベールの法則が成立する。[2]  
- **選択肢4**：正。13Cは12Cより質量が大きいため、13CO2の振動周波数が低く（波数小）、低い波数側にシフトして検出される。[6][9]  
- **選択肢5**：誤。2350cm⁻¹付近は**基群領域**（>1500cm⁻¹）で、指紋領域（<1500cm⁻¹）ではない。[4][6]  

尿素呼気試験では13C尿素がウレアーゼで分解され、13CO2を生成。この13CO2の赤外吸収を12CO2と比較して同位体比を測定する。[3][6]
