# 第101回 問33

**問題文**
消化管のセロトニン5-HT4受容体を刺激することにより、胃腸運動を促進するのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-33 を参照

**選択肢**
1. ロペラミド
2. ドンペリドン
3. モサプリド
4. ベタネコール
5. トリメブチン

**正解:** 3

**解説**
解答　３

１　誤
ロペラミドは、消化管に存在するオピオイドµ受容体を刺激することで、アセチルコリンの遊離を抑制し、胃腸運動を抑制する。

２　誤
ドンペリドンは、消化管に存在するドパミンD2受容体を遮断することで、アセチルコリン遊離を促進し、胃腸運動を促進する。

３　正
モサプリドは、消化管に存在するセロトニン5-HT4受容体を刺激することで、アセチルコリン遊離を促進し、胃腸運動を促進する。

４　誤
ベタネコールは、消化管に存在するムスカリン性アセチルコリン受容体を刺激することで、消化管平滑筋を収縮させ、胃腸運動を促進する。

５　誤
トリメブチンは、オピオイド受容体に作用し、胃腸運動低下時にはノルアドレナリン遊離を抑制し、胃腸運動を促進する。一方、胃腸運動亢進時はアセチルコリン遊離を抑制し、胃腸運動を抑制するため、消化管運動調律剤として利用されている。
