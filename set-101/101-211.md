# 第101回薬剤師国家試験 問211

## 問題文

散剤の主薬の処方量が少ない場合には、賦形剤の添加により、かさを増し、分包誤差を極力少なくすることができる。賦形をする際は、一般的に賦形剤として乳糖やデンプンなどを、主薬によって選択して用いる。

以下のうち、一般的に乳糖を賦形剤として用いるのはどれか。**2つ選べ。**[3]

## 全選択肢

1. アミノフィリン水和物末
2. イソニアジド末
3. ヨウ化カリウム末
4. β-ガラクトシダーゼ散
5. ロートエキス散

## 正解番号

**3、5**[3][5]

## 解説

乳糖を賦形剤として用いるかどうかは、主薬との**配合変化の有無**で判断します。[5]

**選択肢1（アミノフィリン水和物末）：誤**
乳糖と配合変化を起こして黄色に変色するため、デンプンを賦形剤として用いる必要があります。[5]

**選択肢2（イソニアジド末）：誤**
乳糖と配合変化を起こして黄～褐色に変化または含量が低下するため、デンプンを賦形剤として用いる必要があります。[5]

**選択肢3（ヨウ化カリウム末）：正**
乳糖とは配合変化を起こさないため、乳糖を賦形剤として用いることができます。[5]

**選択肢4（β-ガラクトシダーゼ散）：誤**
β-ガラクトシダーゼと乳糖を混合すると、配合変化を起こして乳糖が分解されてしまうため、デンプンを賦形剤として用いる必要があります。[4]

**選択肢5（ロートエキス散）：正**
乳糖とは配合変化を起こさないため、乳糖を賦形剤として用いることができます。[5]
