# 第101回 問116

**問題文**
下図は真核細胞における二本鎖DNAの複製過程を模式的に表したものである。二方向に複製が進行する際に、不連続なDNA鎖（岡崎フラグメント）の形成を介して複製されている部分の組合せとして正しいのはどれか。1つ選べ。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-116 を参照

**選択肢**
1. アとイ
2. アとウ
3. イとウ
4. イとエ
5. ウとエ

**正解:** 2

**解説**
解答　２

二本鎖DNAの複製は、複製開始点から両方向に進行する。その際、DNAポリメラーゼは、鋳型DNA鎖の3´→5´方向にそって読み取り、新しいDNA鎖を5´→3´方向に合成する。そのため、複製開始点を起点として、連続的に合成されるDNA鎖（リーディング鎖、イとエ）と、不連続的に合成されるDNA鎖（ラギング鎖、アとウ）が形成される。また、ラギング鎖の複製時に生じるDNA断片は岡崎フラグメントとよばれる。
