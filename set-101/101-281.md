# 第101回薬剤師国家試験 問281

## 問題文

フェンタニル貼付剤に関する記述のうち、正しいのはどれか。**2つ選べ。**[1][5]

## 選択肢

1. 貼付部位の温度が上昇するとフェンタニルの吸収量が増大する[5]

2. 急性の疼痛発作時にも有効である[1]

3. 経皮吸収により肝初回通過効果を避けることができる[1]

4. 鎮痛効果は貼付部位周辺に限定される[1]

## 正解番号

**1、3**[1]

## 解説

**選択肢1（正解）**

貼付部位の温度が上昇するとフェンタニルの吸収量が増大します。これはフェンタニル貼付剤の重要な特性であり、患者が入浴時や発熱時に過剰吸収のリスクがあることを示しています。

**選択肢2（誤り）**

急性の疼痛発作時には有効ではありません。フェンタニル貼付剤は緩徐に吸収される製剤であり、急性疼痛発作への対応には適していません。

**選択肢3（正解）**

経皮吸収により肝初回通過効果を避けることができます。これにより、経口投与よりも低用量で効果を得ることができ、肝機能が低下している患者にも使用しやすい利点があります。[1]

**選択肢4（誤り）**

鎮痛効果は貼付部位周辺に限定されず、経皮吸収により全身に作用します。[1]
