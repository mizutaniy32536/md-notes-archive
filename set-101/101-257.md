# 第101回 問257

### 第101回薬剤師国家試験 問257

**問題文（共通ステムを含む）**  
73歳男性。入院中の患者。切除不能の胃がんに対して、S-1（テガフール・ギメラシル・オテラシルカリウム配合剤）／シスプラチン療法を開始し、その後、退院して外来化学療法で治療を継続することとなった。[1][2][4]  

この化学療法の副作用を軽減する薬物に関する記述として**誤っている**のはどれか。１つ選べ。[2][4]

**全選択肢**  
1. アザセトロンは、化学受容器引き金帯（CTZ）と求心性迷走神経終末のセロトニン5-HT3受容体を遮断し、急性の悪心・嘔吐を抑制する。[2]  
2. タンニン酸アルブミンは、大腸のぜん動運動を促進する。[1][4]  
3. パロノセトロンは、セロトニン5-HT3受容体を遮断し、遅発性の悪心・嘔吐を抑制する。[4]  
4. D-マンニトールは、管腔内浸透圧上昇を介して近位尿細管のナトリウム再吸収を抑制し、尿量を増加させる。[4]  
5. レノグラスチムは、顆粒球系幹細胞に作用し、好中球を増加させる。[4]

**正解番号**  
2[1][2][4]

**解説**  
S-1/シスプラチン療法の副作用（下痢、嘔吐、腹痛、骨髄抑制など）を軽減する支持療法薬の作用機序に関する記述で、誤りは選択肢2。[1][2]  
- タンニン酸アルブミンは、腸管内で分解されタンニン酸を遊離し、粘膜タンパク質と結合して不溶性被膜を形成することで**止瀉作用**を示す。大腸ぜん動運動を**促進**するわけではなく、抑制する内容が正しいため誤り。[1][4]  
- 1はアザセトロン（5-HT3拮抗薬）の急性嘔吐抑制作用、正しい。[2]  
- 3はパロノセトロン（5-HT3拮抗薬）の遅発性嘔吐抑制作用、正しい。[4]  
- 4はD-マンニトールの利尿作用（浸透圧利尿）、正しい。[4]  
- 5はレノグラスチム（G-CSF製剤）の好中球増加作用、正しい。[4]
