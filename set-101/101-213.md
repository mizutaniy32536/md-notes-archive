# 第101回 問213

### 第101回薬剤師国家試験 問213

**問題文**  
プレドニゾロンの確認試験に赤外吸収スペクトルが利用できる。以下の記述のうち正しいのはどれか。2つ選べ。[1][4][6][7]

**全選択肢（番号付き）**  
1. 吸収帯Aはヒドロキシ基に由来する。  
2. 吸収帯BはC－O単結合に由来する。  
3. 吸収帯Cはカルボニル基に由来する。  
4. 吸収帯DはC－H結合に由来する。[1][4][6]

**正解番号**  
1, 3[1][6]

**解説**  
- 選択肢1は正しい。吸収帯A（3600〜3200 cm⁻¹付近）はプレドニゾロンのヒドロキシ基（O-H伸縮振動）に由来する。[1][6]  
- 選択肢2は誤り。吸収帯B（3000〜2800 cm⁻¹付近）はアルカンのC-H伸縮振動に由来する（C-O単結合は通常1000〜1300 cm⁻¹付近）。[6]  
- 選択肢3は正しい。吸収帯C（1710 cm⁻¹付近）はプレドニゾロンの20位カルボニル基（C=O伸縮振動）に由来する。[1][6]  
- 選択肢4は誤り。吸収帯D（1660 cm⁻¹付近）はプレドニゾロンの3位α,β-不飽和ケトンのカルボニル基に由来する（C-H結合は通常2800〜3100 cm⁻¹または600〜1500 cm⁻¹付近）。[6]
