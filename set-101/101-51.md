# 第101回 問51

**問題文**
医薬品の酸化を防ぐために加えられる添加剤はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-51 を参照

**選択肢**
1. 結晶セルロース
2. アスコルビン酸
3. ステアリン酸マグネシウム
4. ショ糖
5. パラオキシ安息香酸ブチル

**正解:** 2

**解説**
解答　２

１　誤
結晶セルロースは賦形剤、崩壊剤、結合剤として用いられる。

２　正
アスコルビン酸は抗酸化剤であり、医薬品の酸化を防ぐことができる。

３　誤
ステアリン酸マグネシウムは、固形製剤の打錠の際に用いられる滑沢剤であり、原料粉体の流動性を向上させる目的で使用されている。

４　誤
ショ糖は、固形製剤の賦形剤としてやシロップ剤の矯味を目的に用いられる。

５　誤
パラオキシ安息香酸ブチルは細菌の増殖を抑制する保存剤として、軟膏剤、クリーム剤といった半固形製剤や注射剤、点眼剤といった無菌製剤に用いられる。
