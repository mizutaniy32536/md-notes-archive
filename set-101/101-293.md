# 第101回 問293

### 第101回薬剤師国家試験 問293

**問題文（全文）**  
65歳女性。高血圧症治療のために通院している病院で、慢性心不全（NYHA分類Ⅰ度）と診断された。本日の受診時にむくみなどの自覚症状はないが、心臓超音波検査では左室機能が低下していると指摘された。  
（前問292で、この患者に追加する心不全治療薬として**カルベジロール**が最も推奨される。）  
前問で推奨される追加薬物に関して適切なのはどれか。1つ選べ。[2][3]

**全選択肢（番号付き）**  
1. （詳細未記載だが誤り：解説4参照）[2]  
2. 治療薬物モニタリング（TDM）の対象薬物である。[1][2][3]  
3. 導入時に高用量の負荷投与を行い、続けて維持量を投与する。[2][3]  
4. 導入時に心機能の収縮力が抑制され、心不全が悪化することがある。[1][2][3]  
5. 本剤のβ受容体遮断作用により、レニン分泌は抑制される。[2]

**正解番号**  
**4** [1][2][3]

**解説**  
カルベジロールはβ遮断薬であり、慢性心不全（特にNYHAⅠ度、左室機能低下）の治療に用いられるが、導入時に心機能の収縮力が抑制され、一時的に心不全が悪化する可能性があるため、低用量から開始し徐々に増量する。[1][2][3]  
- 選択肢1：誤り（詳細は解説4参照とされ、具体的内容不明だが不適切）。[2]  
- 選択肢2：誤り。カルベジロールはTDM対象薬ではない（TDM対象はジゴキシンなど）。[1][2]  
- 選択肢3：誤り。高用量負荷投与ではなく、低用量から開始。[2][3]  
- 選択肢5：誤り。β遮断作用によりレニン分泌は抑制されるため、逆。[1][2]  
本患者は過去にACE阻害薬（リシノプリル）で空咳を起こしたため、β遮断薬のカルベジロールが追加推奨。[1][2]
