## ◆ コーヒー摂取と肺がんの関連を喫煙で層別化して解析するシナリオ

260311

**問題**
コーヒー摂取と発がんとの関係を調べるため、10,000人を対象として10年間のコホート研究を行った結果、コーヒー摂取（1日4杯以上）の有無によって肺がん発症率には差が見られないというデータが得られた（表1）。そこで、コーヒーを多飲する人には、喫煙者が多いのではないかと考え、さらに解析を行った（表2）。
この解析に関する記述のうち、正しいのはどれか。2つ選べ。

※表1（コーヒー摂取有無別の肺がん発症データ）・表2（喫煙の有無で層別化した肺がん発症データ）は原本を参照
（参考値：喫煙あり・コーヒーあり 45/3000、喫煙あり・コーヒーなし 50/5000、喫煙なし・コーヒーあり 5/2000、喫煙なし・コーヒーなし 15/3000）

**選択肢**

1. 喫煙（1日5本以上）による肺がん発症の相対危険度は6.0と計算される。
​
2. 喫煙1日5本未満の者では、コーヒー摂取（1日4杯以上）が肺がん発症の相対危険度を低下させることがわかる。
​
3. 喫煙者（1日5本以上）の肺がん発症の相対危険度に、コーヒー摂取は影響を与えないことがわかる。
​
4. コーヒー摂取1日4杯未満かつ喫煙1日5本未満である群が、最も肺がんの罹患率が低いことがわかる。

**正解と理由**
- `2. 喫煙1日5本未満の者では、コーヒー摂取（1日4杯以上）が肺がん発症の相対危険度を低下させることがわかる。`
- `3. 喫煙者（1日5本以上）の肺がん発症の相対危険度に、コーヒー摂取は影響を与えないことがわかる。`
- **理由の構成**
	- **問題の整理**：表1では差がなかったが、`交絡因子`（喫煙）で`層別化`した表2で再解析する
	- **薬学的知識・仕様・制約**：`層別化`は`交絡因子`の影響を排除する解析手法。各層内での`相対危険度`を比較する
	- **選択理由**：
		- 選択肢2：非喫煙層でコーヒーありRR=0.5（0.25%÷0.5%）→ 低下 → 正しい
		- 選択肢3：喫煙層でコーヒーありRR≈1.5、コーヒーなしと大差なし → 影響を与えない → 正しい

**解法の手順（思考の流れ）**
- 表2の各セルから発症率を計算
- 喫煙なし層：コーヒーあり 5/2000=0.25%、コーヒーなし 15/3000=0.50%、RR=0.5 → コーヒーで低下
- 喫煙あり層：コーヒーあり 45/3000=1.50%、コーヒーなし 50/5000=1.00% → ほぼ同等
- 選択肢1：全体のRR=(50/5000)÷(20/5000)=2.5、6.0ではない → 誤り
- 選択肢4：最低罹患率はコーヒーあり・喫煙なし（0.25%）であり、コーヒーなし・喫煙なし（0.50%）ではない → 誤り

**他の選択肢が誤りな理由**
- 1. 喫煙（1日5本以上）による肺がん発症の相対危険度は6.0と計算される。
	喫煙あり群の罹患率50/5000=1.0%、非喫煙群20/5000=0.4%で計算するとRR=2.5。6.0は計算間違い

- 4. コーヒー摂取1日4杯未満かつ喫煙1日5本未満である群が、最も肺がんの罹患率が低いことがわかる。
	最低罹患率はコーヒー4杯以上＋喫煙5本未満群の0.25%。コーヒーなし＋喫煙なしは0.50%でこれより高い

**キーポイント**
- `交絡因子`の存在で見かけの関連が消える・逆転する現象がある（`シンプソンのパラドックス`）
- `層別化`は交絡因子を排除する最も基本的な方法
- この問題では喫煙が`交絡因子`：喫煙はコーヒー摂取と肺がんの両方に関連
- 全体では差がなくても、層別化すると各層で異なる結果が見える

**関連知識の比較**
- `交絡因子`の排除方法：`層別化`、`マッチング`、`多変量解析`、`ランダム化`
- `シンプソンのパラドックス`：部分群の傾向と全体の傾向が逆転する現象
- `効果の修飾`（交互作用）：因子Aの効果が因子Bの有無で変わる場合
- `Mantel-Haenszel法`：層別化データを統合して調整`オッズ比`を算出

（一言メモ）
- 「全体では差がないのに層別化したら差が出た」→ `交絡因子`の存在を示す典型パターン
- `層別化`の計算は各層ごとに`相対危険度`を出すだけ。割り算2回で済む
- 喫煙は多くの疫学研究で`交絡因子`として扱われる代表的因子
