# 第101回 問340

**問題文**
医薬品情報に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-340 を参照

**選択肢**
1. 二重盲検比較試験は、メタアナリシスよりもエビデンスレベルが高い。
2. 医療用医薬品の添付文書は、医薬品医療機器等法によりその記載内容が規定された公的文書である。
3. 能動的情報提供とは、医療従事者や患者などからの質問に対して情報提供を行うことである。
4. 二次資料は、新しい知見の報告を主体とし原著記事を収載した資料である。
5. 緊急安全性情報は、緊急に安全対策上の措置を講じる必要がある場合に、医薬品の製造販売業者等が作成する。

**正解:** 2,5

**解説**
解答　２、５

１　誤
メタアナリシスは、独立して行われた複数の臨床研究のデータを統合し総合評価する統計手法であり、エビデンスレベルは最も高いとされている。そのため、メタアナリシスは二重盲検比較試験よりもエビデンスレベルが高い。

２　正

３　誤
能動的情報提供とは、不特定多数の医療従事者や患者に情報を積極的に提供することである。なお、医療従事者や患者などからの質問に対して情報提供を行うのは、受動的情報提供である。

４　誤
二次資料は、一次資料を要約したり、検索用に加工したものである。なお、新しい知見の報告を主体とし原著記事を収載しているのは一次資料である。

５　正
緊急安全性情報は、緊急に安全対策上の措置を講じる必要がある場合に、厚生労働省の指示に基づき、1ヶ月以内に製造販売業者が作成する情報である。
