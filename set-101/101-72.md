# 第101回 問72

**問題文**
専ら薬局開設者等に医薬品を販売するのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-72 を参照

**選択肢**
1. 店舗販売業
2. 配置販売業
3. 卸売販売業
4. 医薬品製造業
5. 医薬品製造販売業

**正解:** 3

**解説**
解答　３

専ら薬局開設者等に医薬品を販売するのは、卸売販売業である。

１　誤
店舗販売業が医薬品を販売できる相手は一般消費者である。なお、販売できる医薬品は要指導医薬品及び一般用医薬品である。 

２　誤
配置販売業が医薬品を販売できる相手は一般消費者である。なお、販売できる医薬品は一般用医薬品である。

３　正
卸売販売業が医薬品を販売できる相手は薬局開設者等である。なお、販売できる医薬品は医療用医薬品及び要指導医薬品、一般用医薬品である。

４　誤
医薬品製造業が医薬品を販売できる相手は医薬品製造業者、医薬品製造販売業者である。なお、販売できる医薬品は自ら製造した医薬品である。

５　誤
医薬品製造販売業者が医薬品を販売できる相手は薬局開設者、医薬品販売業者、医薬品製造業者、医薬品製造販売業者である。なお、販売できる医薬品は自ら製造販売する医薬品である。
