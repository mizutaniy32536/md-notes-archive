# 第101回 問124

**問題文**
食品の安全に関わる法制度に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-124 を参照

**選択肢**
1. ポジティブリスト制度により、国内で流通しているすべての農薬について、食品中の残留基準が個別に設定されている。
2. HACCPとは、食品製造における最終製品の抜き取り検査による衛生管理の方法である。
3. 特定保健用食品の関与成分の健康影響は、食品安全委員会が評価を行う。
4. 食品表示法は、JAS法、食品衛生法、健康増進法の食品の表示に関する規定を統合して、包括的かつ一元的にしたものである。
5. 食品添加物の規格や使用基準は、食品安全基本法で定められている。

**正解:** 3,4

**解説**
解答　３、４

１　誤
ポジティブリスト制度とは、農薬等が、基準値を超えて残留する食品の販売等を原則禁止する制度である。農薬は、人の健康への影響を考慮して食品衛生法で個別に残留基準を設定するが、残留基準が設定されていないものもあり、残留基準が設定されていないものは一律基準（0.01 ppm）が設定されている。よって、国内で流通しているすべての農薬について、食品中の残留基準が個別に設定されているわけではない。

２　誤
HACCP（Hazard Analysis Critical Control Point）とは、食品の製造・加工工程のすべての段階で発生するおそれのある微生物的、化学的および物理的危害を分析することによって特に危害が発生しやすい場所（重要管理点）を設定し、連続的・重点的に管理することで食中毒を予防する衛生管理の方法である。

３　正

４　正
食品表示法は、食品を摂取する際の安全性及び一般消費者の自主的かつ合理的な食品選択の機会を確保するため、食品衛生法、JAS法及び健康増進法の食品の表示に関する規定を統合して食品の表示に関する包括的かつ一元的にしたものである。

５　誤
食品添加物の規格や使用基準は、食品衛生法で定められている。なお、食品安全基本法では、食品の安全性を確保するため、食品摂取による人の健康への影響の評価（食品健康影響評価）を行うことを定めている。
