# 第101回 問74

**問題文**
医療法の目的として、正しいのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-74 を参照

**選択肢**
1. 保険給付の確保
2. 副作用被害の救済
3. 良質かつ適切な医療を効率的に提供する体制の確保
4. 医薬品等の品質の確保
5. 薬物乱用の防止

**正解:** 3

**解説**
解答　３

医療法は、医療を受ける者による医療に関する適切な選択を支援するために必要な事項、医療の安全を確保するために必要な事項、病院、診療所及び助産所の開設及び管理に関し必要な事項並びにこれらの施設の整備並びに医療提供施設相互間の機能の分担及び業務の連携を推進するために必要な事項を定めること等により、医療を受ける者の利益の保護及び良質かつ適切な医療を効率的に提供する体制の確保を図り、もって国民の健康の保持に寄与することを目的とする。
