# 第101回 問212

**問題文**
63歳女性。日曜日の午後、急に左耳が聞こえにくくなった。翌朝まで様子をみたが改善しなかったため、近隣の耳鼻科を受診したところ、突発性難聴と診断された。以下の薬剤が処方され、保険薬局に来局した。＜image1:center＞問212（実務）薬剤師が服薬指導する内容の記述として適切でないのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-212 を参照

**選択肢**
1. 寝付きが悪くなることがあります。
2. 難聴の症状が改善すれば、プレドニゾロン錠は途中で服用を中止しても構いません。
3. プレドニゾロン錠は、朝食後6錠、昼食後4錠、夕食後2錠を3日間服用してください。
4. メコバラミン錠は光に弱い薬です。光を避けて保管してください。
5. テプレノンカプセルは胃粘膜を保護する薬です。問213（物理・化学・生物）プレドニゾロンの確認試験に赤外吸収スペクトルが利用できる。以下の記述のうち正しいのはどれか。2つ選べ。＜image2:center＞
6. 吸収帯Aはヒドロキシ基に由来する。
7. 吸収帯BはC-O単結合に由来する。
8. 吸収帯Cはカルボニル基に由来する。
9. 吸収帯DはC-H結合に由来する。

**正解:** 2,3

**解説**
問212　解答　２、３

１　適切
本患者に処方されているプレドニゾロン錠は、副作用として不眠が現れることがある。よって、「寝付きが悪くなることがあります。」と説明することは適切である。

２　不適切
プレドニゾロン錠は副腎皮質ステロイド製剤であり、急に中止すると発熱、頭痛、食欲不振、脱力感、筋肉痛、関節痛、ショック等の離脱症状が現れることがある。よって、「症状が改善してもプレドニゾロン錠は途中で服用を中止せず、医師の指示通り服用してください。」と説明する必要がある。

３　不適切
突発性難聴の治療にプレドニゾロン錠を用いる場合、漸減療法が基本となる。よって、「処方1→処方2→処方3の順に服用してください」と説明する必要がある。

４　適切
メコバラミン錠は、光により含量が低下する。よって、「メコバラミン錠は光に弱い薬です。光を避けて保管してください。」と説明することは適切である。

５　適切
テプレノンは胃粘膜保護薬であり、プレドニゾロンの副作用である消化性潰瘍を軽減する目的で処方されている。よって、「テプレノンカプセルは胃粘膜を保護する薬です。」と説明することは適切である。

問213　解答　１、３

１　正
吸収帯A（3400 cm－1付近）は、ヒドロキシ基のO–H結合の伸縮振動に由来するものである。

２　誤
吸収帯B（2950 cm－1付近）は、アルカンのC–H結合の伸縮振動に由来するものである。

３　正
吸収帯C（1700 cm－1付近）は、ケトン（カルボニル基）の伸縮振動に由来するものである。

４　誤
吸収帯D（1660 cm－1付近）は、α、β不飽和ケトン（カルボニル基）の伸縮振動に由来するものである。
