# 第101回 問1

**問題文**
同圧下で最も沸点の高いのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-1 を参照

**選択肢**
1. HF
2. HCl
3. HI
4. H2O
5. H2S

**正解:** 4

**解説**
解答　４

沸点は、一般に分子量の増加や水素結合の影響などにより分子間の結合が強くなるほど高くなる。
そのため、HFやH2Oなどの水素結合を形成する分子は沸点が高く、特にH2Oは1分子あたりに形成できる水素結合の数がHFに比べて多いため、沸点はHFよりH2Oのほうが高くなる。
