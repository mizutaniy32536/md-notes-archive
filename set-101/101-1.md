# 第101回 問1

科目: 物理  
問題文: 同圧下で最も沸点の高いのはどれか。1つ選べ。[2][4][6]  
選択肢:  
1. HF[2][4][6]  
2. HCl[2][4][6]  
3. HI[2][4][6]  
4. H₂O[2][4][6]  
5. H₂S[2][4][6]  
正解: 4[2][6]  
解説: 沸点は原則として分子量が高いほど高くなるが、水素結合を形成する分子は例外的に沸点が高い。H₂OはHFに比べて水素を2つ持ち、Oの電気陰性度が高いため水素結合が強く、最も沸点が高い。[2][6]
