# 第101回 問261

### 第101回薬剤師国家試験 問261

**問題文（薬理）**  
70歳女性。１ヶ月前から四肢の筋力低下と、眠れないほどの四肢及び腰部の痛みがある。整形外科を受診し、以下の薬剤が処方された。  
リセドロン酸の作用機序として正しいのはどれか。１つ選べ。[1][2][8]

**全選択肢**  
１ 骨のエストロゲン受容体に結合し、骨芽細胞を増加させ骨形成を促進する。[2][8]  
２ ヒドロキシアパタイトに結合したのち骨芽細胞に取り込まれ、アポトーシスを起こす。[2]  
３ オステオカルシンのカルボキシ化を介し、前駆細胞から破骨細胞への分化を阻害する。[1][2]  
４ メバロン酸代謝経路のファルネシルピロリン酸合成酵素を阻害し、破骨細胞の機能を抑制する。[1][2]  
５ （検索結果に明記なし。ただし、４が正解のため、他の選択肢と同様に誤り）[1][2]

**正解番号**  
４[1][2]

**解説**  
リセドロン酸Na（ビスフォスフォネート系薬）は、ヒドロキシアパタイトに結合したのち破骨細胞に取り込まれ、メバロン酸代謝経路のファルネシルピロリン酸合成酵素（FPPS）を阻害し、破骨細胞の機能を抑制することでアポトーシスを引き起こす。これにより骨吸収を抑制する。[1][2]  
１はエストロゲン様作用（ラロキシフェンなど）の機序、２は骨芽細胞ではなく破骨細胞に作用、３はストロンチウムラネリートなどの機序で誤り。[1][2]
