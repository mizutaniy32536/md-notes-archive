# 第101回 問2

科目: 物理化学（反応速度論）

問題文: 下図のような反応において、E3が示すものはどれか。1つ選べ。

選択肢:
1. 活性化エネルギー
2. 活性化自由エネルギー
3. 活性化エンタルピー
4. 活性化エントロピー
5. 反応熱

正解: 5

解説: E3は、反応物のエネルギーと生成物のエネルギーの差であり、**反応熱**を示しています[4]。なお、E1は反応物のエネルギーと遷移状態のエネルギーの差であり、正反応の活性化エネルギーを示しており、E2は生成物のエネルギーと遷移状態のエネルギーの差であり、逆反応の活性化エネルギーを示しています[4]。
