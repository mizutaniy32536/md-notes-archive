# 第101回 問2

**問題文**
下図のような反応において、E3が示すものはどれか。1つ選べ。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-2 を参照

**選択肢**
1. 活性化エネルギー
2. 活性化自由エネルギー
3. 活性化エンタルピー
4. 活性化エントロピー
5. 反応熱

**正解:** 5

**解説**
解答　５

E3は反応物のエネルギーと生成物のエネルギーの差を表しており、反応熱（エンタルピー変化量）を示している。なお、E1は反応物のエネルギーと遷移状態のエネルギーの差を表しており、正反応の活性化エネルギーを示している。また、E2は生成物のエネルギーと遷移状態のエネルギーの差を表しており、逆反応の活性化エネルギーを示している。
