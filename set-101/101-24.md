# 第101回 問24

**問題文**
自然放射線被曝のもととなる放射性核種はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-24 を参照

**選択肢**
1. 40K
2. 90Sr
3. 131I
4. 137Cs
5. 239Pu

**正解:** 1

**解説**
解答　１

自然放射線被曝の原因には、宇宙線からの被曝、大地からの被曝、食品からの被曝などがある。40Kは食品に含まれている天然放射性核種であり、食品による自然放射線被曝の原因となる。
