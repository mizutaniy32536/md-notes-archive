# 第101回 問307

### 第101回薬剤師国家試験 問307

#### 問題文（全文）
64歳男性。BMI 28.5。糖尿病で近医に通院中であった。全身倦怠感を訴え、浮腫も認められたため精査目的で入院となった。入院時の持参薬と検査データは以下の通りである。（※持参薬・検査データは問304-306で共有：尿タンパク（3+）、HbA1c高値、血圧高値、LDL-C高値など。）

この患者は、ある臓器の障害により浮腫を生じていたが、その後、同じ臓器の障害による貧血を生じた。この貧血の機序として、正しいのはどれか。**1つ選べ。**[3][5]

#### 全選択肢（番号付き）
1. トランスフェリンの産生が抑制されるため。[2]  
2. 血尿が生じるため。[2]  
3. **エリスロポエチンの分泌が減少するため。**[2]  
4. 内因子の分泌不全が起こるため。[2]  
5. ヘモグロビンの消費が増加するため。[2]

#### 正解番号
**3**[2][3][5]

#### 解説
この患者は尿タンパク（3+）や浮腫から**腎臓**の障害（腎症）を示唆しており、浮腫は腎機能低下によるものである。同様の腎障害で貧血（**腎性貧血**）が生じる機序は、腎臓での**エリスロポエチン（EPO）**分泌減少による赤血球産生低下である。他の選択肢は誤り：1は肝障害関連、2は出血性貧血、4は悪性貧血、5は溶血性貧血の機序。[2][3][5]
