# 第101回 問7

**問題文**
下線で示した元素の酸化数が＋2のものはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-7 を参照

**選択肢**
1. CrO3
2. MnO2
3. K3[Fe(CN)6]
4. CuSO4
5. Ag2O

**正解:** 4

**解説**
解答　４

１　誤
CrO3（三酸化クロム）：（＋6）＋（−2）×3＝0
であるため、Crの酸化数は＋6である。

２　誤
MnO2（二酸化マンガン）：（＋4）＋（−2）×2＝0
であるため、Mnの酸化数は＋4である。

３　誤
K3[Fe(CN)6]（フェリシアン化カリウム）：（＋1）×3＋（＋3）＋（−1）×6＝0
であるため、Feの酸化数は＋3である。

４　正
CuSO4（硫酸銅(Ⅱ)）：（＋2）＋（＋6）＋（−2）×4＝0
であるため、Cuの酸化数は＋2である。

５　誤
Ag2O（酸化銀(Ⅰ)）：（＋1）×2＋（−2）＝0
であるため、Agの酸化数は＋1である。
