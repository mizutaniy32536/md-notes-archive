# 第101回 問237

### 第101回薬剤師国家試験 問237

**問題文（全文）**  
77歳男性。3月中旬のある日、昼食の時間になり、家族が男性を探したところ、男性は農機具等が置いてある小屋でうずくまっていた。顔面は蒼白であり、声をかけてもうなずくだけで言葉が出なかった。また、足下に嘔吐物があった。支えても歩くことができなかったため、救急車で病院に搬送した。家族の話から、この男性は朝10時頃から小屋で畑仕事の準備をしていたという。診察した医師は、化学物質による中毒を疑い、検査を依頼した。[1][2][3]

**問237（衛生）**  
この男性の血液を検査したところ、コリンエステラーゼ活性が著しく低下していた。中毒の原因と考えられる化学物質はどれか。1つ選べ。[1][2][3]

**全選択肢（番号付き）**  
1. クロルピクリン[1][2][3]  
2. グルホシネート[1][2][3]  
3. パラコート[1][2][3]  
4. **メソミル**[1][2][3]  
5. 硫酸タリウム[1][2][3]

**正解番号**  
4[1][2][3]

**解説**  
コリンエステラーゼ活性の低下は、有機リン系またはカルバメート系の農薬中毒を示唆する。**メソミル**はカルバメート系農薬で、可逆的にコリンエステラーゼを阻害し毒性を発揮する。[1][2]  
他の選択肢：  
- 1. クロルピクリン：土壌燻蒸剤でコリンエステラーゼ阻害作用なし。[1]  
- 2. グルホシネート：除草剤で神経毒性なし。[1]  
- 3. パラコート：ビピリジニウム系除草剤で肺障害を引き起こすが、コリンエステラーゼ阻害なし。[2]  
- 5. 硫酸タリウム：神経毒だがコリンエステラーゼ活性低下とは無関係。[1][2]
