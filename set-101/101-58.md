# 第101回 問58

**問題文**
クッシング症候群の臨床症状及び所見に該当するのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-58 を参照

**選択肢**
1. 低血圧
2. 低血糖
3. 中心性肥満
4. 筋力増強
5. 皮膚の肥厚

**正解:** 3

**解説**
解答　３

クッシング症候群は、副腎の腺腫や過形成などにより、副腎皮質から主に糖質コルチコイド（コルチゾール、ヒドロコルチゾン）の過剰分泌を呈する病態の総称であり、その他アルドステロンやアンドロゲンの過剰分泌も起こることで様々な症状を示す。
このうち、糖質コルチコイド作用により高血糖や中心性肥満、皮膚の菲薄化、筋力低下などの臨床症状及び所見が見られる。またアルドステロン作用により高血圧や低カリウム血症などの臨床症状及び所見がみられる。
