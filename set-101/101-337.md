# 第101回 問337

**問題文**
下の図は一般的な外来初回診療の流れを示している。選択肢の語句を　A　〜　E　に当てはめたとき、　C　に入るものはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-337 を参照

**選択肢**
1. 検査
2. 診断
3. 診察
4. 経過観察
5. 治療

**正解:** 2

**解説**
解答　２

以下に一般的な外来初回診療の流れを示す。

受付 → 診察 → 検査 → 診断 → 経過観察又は治療

上記より、 　C　  に入るものは診断である。
