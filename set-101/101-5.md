# 第101回 問5

**問題文**
pHメーターを用いたpH測定に最も関係する物理定数はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-5 を参照

**選択肢**
1. アボガドロ定数
2. ファラデー定数
3. プランク定数
4. ボルツマン定数
5. リュードベリ定数

**正解:** 2

**解説**
解答　２

水溶液のpH測定にはpHメーターが用いられる。pHメーターは、ガラス電極と参照電極により構成され、試料溶液に電極を浸すと、ガラス薄膜を介して電極内部の溶液と試料溶液とのpHの違いにより、膜電位が生じる。
試料溶液のpHは、標準溶液のpH（pHs）と電位を基準値として、次式で表される。

１　誤
アボガドロ定数（NA：6.02×1023 mol-1）は1 mol当たりの粒子（原子、分子、イオンなど）の数を示す定数である。

２　正
ファラデー定数（F：9.65×104 C/mol）は電子1 mol当たりの電気量の絶対値（電気素量）を示す値である。

３　誤
プランク定数（h：6.62606896×10－34 Js）は光量子の持つエネルギーEと振動数に関する比例定数である。

４　誤
ボルツマン定数（kB：1.38066×1023 J/K）は気体定数をアボガドロ数で除した値であり、気体1分子当たりの熱エネルギーを示す。

５　誤
リュードベリ定数（R∞：1.0973731569×107 m-1）は、原子の発光および吸収スペクトルを説明する際に用いられる物理定数である。
