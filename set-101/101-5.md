# 第101回 問5

科目: 物理  
問題文: pHメーターを用いたpH測定に最も関係する物理定数はどれか。１つ選べ。[3][4]  
選択肢:  
1. アボガドロ定数[3][4]  
2. ファラデー定数[3][4]  
3. プランク定数[3][4]  
4. ボルツマン定数[3][4]  
5. リュードベリ定数[3][4]  
正解: 2[3][4]  
解説: pHメーターのpH測定はネルンストの式に基づき、起電力EとpHの関係式にファラデー定数Fが含まれるため。式：E = Es + (RT/F) ln(10) (pHs - pH) （R:気体定数、T:温度）。他の定数は無関係（アボガドロ定数:1molの粒子数、プランク定数:光量子エネルギー、ボルツマン定数:R/NA、リュードベリ定数:原子スペクトル）。[4]
