# 第101回 問231

### 第101回薬剤師国家試験 問231

**問題文（全文）**  
カンピロバクターによる食中毒に関する記述のうち、**誤っている**のはどれか。1つ選べ。[4][10]

**全選択肢**  
1. カンピロバクターは鳥類等の腸管に常在し、鶏肉を汚染しやすい。  
2. 下痢によって菌を排出しているので、下痢止めは使用しない。  
3. 鶏肉の内部は汚染されていないので表面を軽くあぶってから、子供に食べさせる。  
4. 野生生物により汚染された水が飲用に使用された場合に、集団食中毒が発生しやすい。[2][4][10]

**正解番号**  
**3**（誤った記述）[2][4]

**解説**  
カンピロバクターは鶏肉の**内部まで汚染**されているため、表面を軽くあぶるだけでは不十分であり、**中心温度75℃以上で1分間加熱**して完全に加熱殺菌する必要がある。選択肢3はこれを誤っており、誤り。  
- 選択肢1：正しい。鳥類（鶏など）の腸管に常在し、鶏肉を汚染しやすい。[2][4][10]  
- 選択肢2：正しい。下痢は菌の自然排出を促すため、下痢止めは使用せず、整腸剤を推奨。[2][3]  
- 選択肢4：正しい。乾燥に弱いが、水系汚染（野生生物由来）で集団食中毒が発生しやすい。[2]
