# 第101回 問26

**問題文**
Ca2＋に対して高い透過性を示すイオンチャネル内蔵型受容体はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-26 を参照

**選択肢**
1. セロトニン5-HT2受容体
2. グルタミン酸NMDA受容体
3. アセチルコリンNM受容体
4. GABAA受容体
5. グリシン受容体（ストリキニーネ感受性）

**正解:** 2

**解説**
解答　２

生体内の受容体は、主にイオンチャネル内蔵型、Gタンパク質共役型、酵素共役型に大別される。

１　誤
セロトニン5-HT2受容体は、Gqタンパク質共役型受容体である。

２　正
グルタミン酸NMDA受容体は、Ca２＋に対して高い透過性を示す陽イオンチャネル内蔵型受容体である。

３　誤
アセチルコリンNM受容体は、Na＋に対して高い透過性を示す陽イオンチャネル内蔵型受容体である。

４　誤
GABAA受容体は、Cl－に対して高い透過性を示す陰イオンチャネル内蔵型受容体である。

５　誤
グリシン受容体（ストリキニーネ感受性）は、Cl－に対して高い透過性を示す陰イオンチャネル内蔵型受容体である。
