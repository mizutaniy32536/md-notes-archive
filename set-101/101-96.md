# 第101回 問96

**問題文**
L-リシンは、2つのアミノ基と1つのカルボキシ基をもち、水溶液のpHにより4つの化学種が存在する。図に示したL-リシンの化学種が最も多く存在するpHに最も近い値はどれか。1つ選べ。ただし、L-リシンの3つのpKaはそれぞれpKa1＝2.2、pKa2＝9.0、pKa3＝10.5とする。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 101-96 を参照

**選択肢**
1. 2.2
2. 5.6
3. 9.0
4. 9.7
5. 10.5

**正解:** 4

**解説**
解答　４

L-リシンは、2つのアミノ基と1つのカルボキシ基をもち、水溶液のpHにより以下の4つの化学種①〜④が存在する。

本設問では、③が最も存在するpHが問われており、そのpHはpKa2とpKa3の中間の値となる。
③が最も存在する：pH＝（pKa2＋pKa3）÷2＝（9.0＋10.5）÷2＝9.75
なお、アミノ酸の正電荷と負電荷の量が等量存在するpHを等電点といい、③の状態のpHはL-リシンの等電点（L-リシンの等電点：9.75）に該当する。
