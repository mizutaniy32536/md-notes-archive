# 第101回 問299

### 第101回薬剤師国家試験 問299

#### 症例文（問298・299共通）
67歳男性。16年前に**HIV感染**が判明し、ジドブジン（ZDV）とラミブジン（3TC）による治療を開始したが、7年前から服薬を自己中断していた。6年前の**結核罹患**を契機にロピナビル・リトナビル（LPV・RTV）配合剤を追加して治療を再開したが、その2年後から再び服薬を自己中断していた。  
**全身倦怠感**が徐々に進行し、**血液検査**（CD4陽性リンパ球、HIV-RNA定量）の結果、3TC・アバカビル硫酸塩配合剤とLPV・RTVによる治療を開始することになった。[4]

#### 問299（病態・薬物治療）
**HIV感染症**について正しいのはどれか。**2つ選べ**。[4]

1. **母乳**を介した感染はしない。  
2. **無症候期**は、感染後、**数週間**である。  
3. 一過性の**インフルエンザ様症状**が感染初期（感染後数週間）に起こる。  
4. （検索結果で一部言及）無症候期でも血液検査が必要。[1]  
（注: 完全な選択肢4・5は一部ソースで省略されているが、問299の正解は3,4とされ、4は無症候期の血液検査の必要性を指す可能性が高い。[1][4]）

#### 正解番号
**3, 4**[1][4]

#### 解説
- **選択肢1**: 誤り。**母乳感染**が知られており、キャリア母は**断乳**する。[1]  
- **選択肢2**: 誤り。無症候期は**数年間**に及ぶ場合もある。[1]  
- **選択肢3**: 正しい。感染初期（感染後数週間）に**一過性のインフルエンザ様症状**が起こる。[1][4]  
- **選択肢4**: 正しい（文脈から）。無症候期でも**血液検査**（CD4陽性リンパ球、HIV-RNA定量）の値に注目し、治療方針を決定する必要がある。薬剤耐性が生じウイルス増殖が進行する可能性があるため。[1]  

この問題は**HIV感染症**の病態（急性期症状、無症候期の持続性、母乳感染リスク）を問う。症例の服薬中断歴が免疫低下を示唆。[1][4]
