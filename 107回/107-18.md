# 107-18

食品表示法に基づき、用途名と物質名を併記する必要がある食品添加物はどれか。1つ選べ。

1. 香料
2. 甘味料
3. 調味料
4. 乳化剤
5. pH調整剤

## 正解
**2. 甘味料**

## 解説

食品表示法で用途名＋物質名の**併記が必要な8用途**：

| 用途 | 併記必要 | 例 |
|---|---|---|
| 甘味料 | ⭕ | アスパルテーム、サッカリン |
| 着色料 | ⭕ | 赤色2号、β-カロテン |
| 保存料 | ⭕ | ソルビン酸、安息香酸 |
| 酸化防止剤 | ⭕ | ビタミンC、BHA |
| 発色剤 | ⭕ | 亜硝酸Na、硝酸Na |
| 漂白剤 | ⭕ | 亜硫酸Na |
| 防かび剤 | ⭕ | OPP、イマザリル |
| 糊料（増粘剤等） | ⭕ | カラギナン |
| 香料 | ❌ | 「香料」のみでOK |
| 調味料 | ❌ | 「調味料（アミノ酸等）」のみでOK |
| 乳化剤 | ❌ | 「乳化剤」のみでOK |
| pH調整剤 | ❌ | 「pH調整剤」のみでOK |

> 語呂：**「甘・着・保・酸・発・漂・防・糊」**（かん・ちゃく・ほ・さん・はつ・ひょう・ぼう・のり）
