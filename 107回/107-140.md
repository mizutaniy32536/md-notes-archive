



## ◆ 大気中の窒素酸化物の発生源・反応・環境規制に関する正誤を判断するシナリオ
260228

**問題**
- 大気中の窒素酸化物に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**


1. 大気中の窒素酸化物は水分と反応して、酸性雨の原因となる。
​
2. サーマルNOxは、化石燃料中の窒素化合物の燃焼に由来する。
​
3. 大気中の窒素酸化物は、非メタン炭化水素と反応して、光化学オキシダントの原因となる。
​
4. 大気中へ排出される窒素酸化物は、大気汚染防止法により施設単位の排出基準に基づくK値規制が行われている。
​
5. 2010年度以降における二酸化窒素の大気環境基準の達成率は、一般環境大気測定局（一般局）、自動車排出ガス測定局（自排局）のいずれにおいても約80%で推移している。

**正解と理由**
- `1`、`3`
- **理由の構成**
	- **問題の整理**：大気中のNOxについて、酸性雨との関連・発生メカニズム・光化学オキシダント生成・法規制・環境基準達成率の5項目の正誤を判断する
	- **薬学的知識・仕様・制約**：`NOx`は大気中の水分と反応して`硝酸`（HNO₃）を生成し、`酸性雨`の原因となる。また、NOxは`非メタン炭化水素`（NMHC）と紫外線の存在下で反応し、`光化学オキシダント`（O₃等）を生成する
	- **選択理由**：選択肢1のNOx→酸性雨の機序と、選択肢3のNOx＋NMHC→光化学オキシダントの機序はいずれも正しい

**解法の手順（思考の流れ）**
- 選択肢1：NOx→HNO₃→酸性雨 → 正しい（SOx→H₂SO₄も酸性雨の原因）
- 選択肢2：「サーマルNOx」の定義を確認 → 空気中のN₂が高温酸化されて生じるものであり、燃料中の窒素由来ではない → 誤り
- 選択肢3：NOx＋NMHC＋紫外線→光化学オキシダント → 正しい
- 選択肢4：K値規制の適用対象を確認 → SOxに適用される規制であり、NOxではない → 誤り
- 選択肢5：NO₂の環境基準達成率を確認 → ほぼ100%であり、約80%ではない → 誤り

**他の選択肢が誤りな理由**
- 2. サーマルNOxは、化石燃料中の窒素化合物の燃焼に由来する。
	`サーマルNOx`は空気中の`N₂`が高温（1000℃以上）で`O₂`と反応して生成されるNOxであり、「thermal（熱）」の名の通り高温燃焼に由来する。化石燃料中の窒素化合物の燃焼に由来するNOxは`フューエルNOx`と呼ばれる

- 4. 大気中へ排出される窒素酸化物は、大気汚染防止法により施設単位の排出基準に基づくK値規制が行われている。
	`K値規制`は`SOx`（硫黄酸化物）に適用される排出規制であり、煙突の高さと地域ごとのK値から許容排出量を算出する方式。NOxに対しては`総量規制`（特定地域における工場・事業場のNOx排出総量を制限）が適用されている

- 5. 2010年度以降における二酸化窒素の大気環境基準の達成率は、一般環境大気測定局（一般局）、自動車排出ガス測定局（自排局）のいずれにおいても約80%で推移している。
	`NO₂`の大気環境基準達成率は一般局・自排局ともに`ほぼ100%`で推移しており、約80%という記述は誤り。達成率が低いのは`光化学オキシダント`（O₃）であり、ほぼ0%という極めて低い水準にある

**キーポイント**
- NOxの2つの環境影響：`酸性雨`（NOx→HNO₃）と`光化学オキシダント`（NOx＋NMHC＋紫外線→O₃等）
- NOxの発生源区分：`サーマルNOx`＝空気中N₂の高温酸化、`フューエルNOx`＝燃料中窒素化合物の燃焼
- 規制の使い分け：SOx＝`K値規制`、NOx＝`総量規制`
- 環境基準達成率：NO₂＝`ほぼ100%`、光化学オキシダント＝`ほぼ0%`

**関連知識の比較**
- `酸性雨`の原因物質はNOx（→HNO₃）とSOx（→H₂SO₄）の2系統であり、pH 5.6以下の降水を酸性雨と定義する（CO₂の溶解による純水のpHが5.6であるため）
- `光化学オキシダント`の環境基準（0.06 ppm以下）の達成率がほぼ0%である理由は、越境汚染や自然由来のオゾンなど制御困難な要因が含まれるため
- NOxの総量規制は東京・大阪等の`特定地域`に限定して適用される。SOxのK値規制は全国の`ばい煙発生施設`に適用される
- `非メタン炭化水素`（NMHC）はメタン以外の炭化水素の総称であり、光化学オキシダントの前駆物質として環境基準の指針値（午前6〜9時の3時間平均0.20〜0.31 ppmC以下）が設定されている

（一言メモ）
- 「サーマル＝thermal＝熱」→空気中N₂の熱酸化、「フューエル＝fuel＝燃料」→燃料中Nの燃焼、と語源で区別する
- K値規制＝SOx、総量規制＝NOxのペアは規制方式のひっかけ問題の定番
- 環境基準達成率で「ほぼ0%」の光化学オキシダントは要注意ポイント
