# 第99回 問334

**問題文**
下記の処方薬と併用すると、薬物相互作用により、吸収が低下するのはどれか。2つ選べ。
スクリーンショット 2017-04-04 13.21.13.png

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 99-334 を参照

**選択肢**
1. ゲフィチニブ錠
2. シンバスタチン錠
3. イトラコナゾール錠
4. レボチロキシンナトリウム錠
5. ノルフロキサシン錠

**正解:** 1,3

**解説**
ニザチジンはH2受容体遮断薬であり、服用により胃酸分泌が低下するため、胃内pHが上昇する。ゲフィチニブ及びイトラコナゾールは胃内pHが上昇することにより、溶解性が低下するため、吸収が低下する薬剤である。したがって、ニザチジンとゲフィチニブ及びイトラコナゾールを併用すると、ゲフィチニブ及びイトラコナゾールの吸収が低下することがある。
