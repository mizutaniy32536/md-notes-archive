# 第99回 問328

**問題文**
血液製剤の取扱いについて、正しいのはどれか。2つ選べ。

**選択肢**
1. 血液製剤の凝固防止の目的でエチレンジアミン四酢酸ニナトリウム水和物を使うことがある。
2. 放射線を照射した血液製剤は、血清ナトリウム値が上昇するので、照射後速やかに用いる。
3. 人血清アルブミン製剤は凍結保存する。
4. 人血小板濃厚液は採血後、20〜24℃で振とうしながら貯蔵し、その有効期間は96時間である。
5. 人全血液は採血後、2〜6℃に貯蔵し、その有効期間は21日間である。

**解説**
解答　解なし
1　誤
血液製剤の凝固防止の目的で使用されるのは、クエン酸ナトリウムである。なお、エチレンジアミン四酢酸ニナトリウム水和物は、TDMを行う際、採血管内で血液が凝固するのを防止するために使用される。
2　誤
血液に放射線を照射すると、血液中に含まれる細胞が破壊されるため、血清カリウム値が上昇することがある。
3　誤
人血清アルブミン製剤は凍結を避けて室温で保存する。
4　誤
人血小板濃厚液は、20〜24℃で保存し、有効期間は4日間（採血日を1日目として、4日目ま
で）と定められている。本問では、有効期間は96時間となっているが、有効期間4日間＝有効期間96時間ではないため、有効期間96時間は誤りとなる。
5　正
