# 第99回 問91

**問題文**
90Srは以下に示す放射壊変により、放射性核種90Yを経て、90Zrの安定核種になる。90Yの放射能の時間推移を示す曲線はどれか。1つ選べ。ただし、時間ゼロにおける90Srの放射能は5×104 Bqとする。
スクリーンショット 2017-04-03 19.51.06.png

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 99-91 を参照

**正解:** 3

**解説**
放射壊変により90Srが90Yを経て、90Zrになるように、親核種の半減期が娘核種の半減期に比べてはるかに長い（親核種の半減期が娘核種の半減期の1000倍以上を示す）場合、永続平衡が成立する。永続平衡が成立した場合、親核種と娘核種が放出する放射能は同じになる。
90Srが壊変しはじめたとき、90Yは存在しないため、90Yの放射能は0であるが、時間が経過するにともなって、90Yの放射能は90Srが放出する放射能（5×104 Bq）に近づく。したがって、3のグラフが正解である。
