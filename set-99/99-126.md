## ◆ 人口増減率の推移グラフから関連指標の定義と変動要因を判断するシナリオ
260307

**問題**
- 下図は、我が国における人口増減率の推移を示したものである。図に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**

1. 自然増減率は、出生率と死亡率の差から求められる
​
2. 人口増減率から将来の出生率を予測できる
​
3. 昭和25年に人口増減率が一時的に上昇しているが、これには第一次ベビーブームが関係している
​
4. 社会増減率は、人口流入率と人口流出率の和から求められる
​
5. 今後20年間、人口は横ばいのまま推移すると予測されている

**正解と理由**
- `1. 自然増減率は、出生率と死亡率の差から求められる`
- `3. 昭和25年に人口増減率が一時的に上昇しているが、これには第一次ベビーブームが関係している`
- **理由の構成**
	- **問題の整理**：人口増減率の推移グラフから、`自然増減率`・`社会増減率`・`人口増減率`の定義や変動要因について正しい記述を2つ選択する問題
	- **薬学的知識・仕様・制約**：`自然増減率`は出生率から死亡率を差し引いた値であり、人口の自然な増減を示す。`人口増減率` = `自然増減率` + `社会増減率`で算出される。昭和22〜24年（1947〜1949年）の`第一次ベビーブーム`による出生数の急増が、昭和25年前後の人口増減率上昇に反映されている
	- **選択理由**：選択肢1は自然増減率の定義として正しい。選択肢3は第一次ベビーブームと人口増減率上昇の関係が歴史的事実と一致するため正しい

**解法の手順（思考の流れ）**
- まず`自然増減率`の定義を想起し、出生率 − 死亡率であることを確認して選択肢1が正しいと判断する
- 次にグラフの昭和25年付近の上昇について、`第一次ベビーブーム`（1947〜1949年）の影響を想起し、選択肢3が正しいと判断する
- 選択肢2は人口増減率から将来の出生率を直接予測することはできないため誤りと判断する
- 選択肢4は社会増減率の計算が「和」ではなく「差」であるため誤りと判断する
- 選択肢5は日本の人口が減少傾向にあるため誤りと判断する

**他の選択肢が誤りな理由**
- 2. 人口増減率から将来の出生率を予測できる
	`人口増減率`は過去の実績を示す指標であり、将来の`出生率`は経済状況、社会保障制度、未婚率など多様な社会的要因に影響されるため、人口増減率のみから予測することはできない

- 4. 社会増減率は、人口流入率と人口流出率の和から求められる
	`社会増減率`は人口流入率（入国率）から人口流出率（出国率）を差し引いた「差」で求められる。「和」ではなく「差」が正しい計算方法である

- 5. 今後20年間、人口は横ばいのまま推移すると予測されている
	日本の人口は2008年頃をピークに`減少局面`に入っており、国立社会保障・人口問題研究所の推計でも今後数十年にわたり人口減少が続くと予測されている。横ばいという記述は誤り

**キーポイント**
- `自然増減率` = `出生率` − `死亡率` であり、自然増減のみを表す指標である
- `社会増減率` = 入国率 − 出国率（人口流入率 − `人口流出率`）であり、国際人口移動を反映する
- `人口増減率` = `自然増減率` + `社会増減率` であり、人口の総合的な増減を示す
- `第一次ベビーブーム`は1947〜1949年に発生し、年間出生数が約270万人に達した

**関連知識の比較**
- `第一次ベビーブーム`（1947〜1949年）では年間出生数約270万人、`第二次ベビーブーム`（1971〜1974年）では約200万人を記録した
- 日本の`自然増減率`は2005年に初めてマイナスに転じ、以降`人口の自然減`が続いている
- `合計特殊出生率`は1.57ショック（1990年発表、1989年の値）で大きな社会的関心を集めた
- 人口増減率の構成要素として`自然増減`と`社会増減`があるが、日本では社会増減の影響は自然増減に比べて小さい

（一言メモ）
- `自然増減率`は出生率と死亡率の「差」、`社会増減率`は流入率と流出率の「差」であり、いずれも「和」ではなく「差」で求める
- `第一次ベビーブーム`（1947〜1949年）の世代は`団塊の世代`と呼ばれ、その子世代が`第二次ベビーブーム`（1971〜1974年）を形成した
- 日本の人口は2008年の約1億2808万人をピークに`減少局面`に入っており、今後も減少が続くと推計されている
