# 第99回 問255

**問題文**
一般用医薬品であるリアップX5（5％ミノキシジル製剤）を求めて女性が薬局を訪れた。
問255（実務）
販売時の来局者への対応として重要性の最も低いのはどれか。1つ選べ。

**選択肢**
1. 使用者の年齢を確認した。
2. 使用者の性別を確認した。
3. 使用者に気管支ぜん息の既往歴がないことを確認した。
4. 使用者に高血圧の既往歴がないことを確認した。
5. 使用者に心臓・腎臓の障害がないことを確認した。

**正解:** 3

**解説**
1　誤
本剤は、未成年者（20歳未満）に使用することができない。したがって、販売時に使用者の年齢を確認する必要がある。
2　誤
本剤は、女性に使用することができない。したがって、販売時に使用者の性別を確認する必要がある。
3　正
本剤の添付文書には、気管支ぜん息の既往歴がある使用者についての注意事項は表記されていない。したがって、気管支ぜん息の既往歴の有無を確認することは、選択肢中では最も重要性が低いと考えられる。
4　誤
本剤は、高血圧の人、低血圧の人は使用の前に薬剤師等に相談することとされている。したがって、販売時に使用者の高血圧の既往歴がないことを確認する必要がある。
5　誤
本剤は、心臓又は腎臓に障害がある人は使用の前に薬剤師等に相談することとされている。したがって、販売時に使用者に心臓・腎臓の障害がないことを確認する必要がある。
