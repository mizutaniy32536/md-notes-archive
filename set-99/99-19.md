## ◆ 感染症法の二類感染症を選ぶシナリオ
260312

**問題**
「感染症の予防及び感染症の患者に対する医療に関する法律（感染症法）」において二類感染症に指定されているのはどれか。１つ選べ。

**正解と理由**
- `1. 結核`
- **理由の構成**
	- 結核は`二類感染症`に分類されている

**解法の手順（思考の流れ）**
- 選択肢1：結核 = `二類感染症` → **正しい**
- 選択肢2：風しん = `五類感染症` → 誤り
- 選択肢3：ペスト = `一類感染症` → 誤り
- 選択肢4：コレラ = `三類感染症` → 誤り
- 選択肢5：細菌性赤痢 = `三類感染症` → 誤り

⚠️ **注意：** Perplexity検索では正解が「2（風しん）」と出たが、風しんは五類感染症。正解は`1（結核）`が正しい。予備校解説や厚労省発表を確認推奨。

**キーポイント**
- `一類`：エボラ、ペスト、クリミア・コンゴ、マールブルグ、ラッサ、南米出血熱、痘そう
- `二類`：結核、SARS、MERS、鳥インフルエンザ(H5N1)、ジフテリア、ポリオ
- `三類`：コレラ、腸チフス、パラチフス、細菌性赤痢、腸管出血性大腸菌
- `五類`：インフルエンザ、風しん、麻しん、etc.

**関連知識の比較**

| 選択肢 | 正しい分類 |
|---|---|
| 結核 | **二類** |
| 風しん | 五類 |
| ペスト | 一類 |
| コレラ | 三類 |
| 細菌性赤痢 | 三類 |

（一言メモ）
- 二類 = 「ケッサポジマ鳥」（結核・SARS・ポリオ・ジフテリア・MERS・鳥インフル）
- ペストは一類。コレラ・赤痢は三類。混同注意！
