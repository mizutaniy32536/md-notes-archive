# 第99回 問327

**問題文**
45歳男性。アトルバスタチンカルシウム錠10 mgを1日1錠、就寝前に3週間服用していたところ、尿の色が赤褐色になったので、薬の副作用を疑った。留意すべき血液検査項目はどれか。2つ選べ。

**選択肢**
1. 直接ビリルビン
2. クレアチンキナーゼ
3. ミオグロビン
4. 低比重リポタンパクコレステロール
5. プロラクチン

**正解:** 2,3

**解説**
HMG—CoA還元酵素阻害薬であるアトルバスタチンカルシウム錠10 mgを服用中に尿の色が赤褐色に変化していることから、本剤の副作用である横紋筋融解症が現れたと考えられる。横紋筋融解症が現れた場合、筋組織の破壊により血液中のクレアチンキナーゼ及びミオグロビンが上昇する。
