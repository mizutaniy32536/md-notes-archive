# 第99回 問236

**問題文**
10月のある夕方、薬局を男性が訪れた。その日、友人から初物のカキ（牡蠣）をもらったため、昼に友人や家族と自宅の居間で、生ガキと、炭火で焼きながら焼ガキを食したとのこと。全員が頭痛、嘔吐及びめまいを起こしたが、別室に移って休んだところ、少し落ち着いたとのことであった。
問236（衛生）
これらの症状を引き起こした原因として最も可能性が高いのはどれか。1つ選べ。

**選択肢**
1. 腸炎ビブリオ
2. ノロウイルス
3. 腸管出血性大腸菌
4. 二酸化窒素
5. 一酸化炭素

**正解:** 5

**解説**
問題文の記述より、症状を呈するまでの潜伏期がないことが読み取れる。このことから、頭痛、嘔吐及びめまいの原因が「細菌性食中毒」や「毒素型食中毒」の可能性は低い。また、問題文中の「炭火で焼きながら」、「別室に移って休んだところ、少し落ち着いた」という記述より、これらの症状を引き起こした原因として最も可能性が高いのは、「一酸化炭素」であると考えられる。
なお、炭素化合物の不完全燃焼により発生する一酸化炭素の主な中毒症状として、酸素欠乏症による頭痛、嘔吐及びめまいなどがある。
