# 第99回 問224

**問題文**
65歳女性。身長162 cm、体重56 kg。B細胞性非ホジキンリンパ腫と診断され、次の治療を受けることになった。

問224（実務）
これらの処方に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 99-224 を参照

**選択肢**
1. 処方1の薬剤は、infusion reactionを軽減させる目的で投与される。
2. 処方2の薬剤は、特定生物由来製品に指定されている。
3. 処方2の薬剤を使用する前に、B型肝炎ウイルスの有無を確認する。
4. 処方2の希釈時及び希釈後に、成分の凝集を避けるため泡立つまで激しく振とうする。

**正解:** 1,3

**解説**
1　正
リツキシマブを投与すると、infusion reactionが起こることがある。infusion reactionとは、抗体医薬品の投与後24時間以内に生じるアナフィラキシー様症状やインフルエンザ様症状のことである。infusion reactionを軽減するためには、一般に抗ヒスタミン薬（d−クロルフェニラミンマレイン酸塩など）や解熱鎮痛薬（アセトアミノフェンなど）などが用いられる。
2　誤
モノクローナル抗体製剤（リツキシマブ製剤など）は、特定生物由来製品ではなく、生物由来製品に指定されている。
3　正
B型肝炎ウイルスキャリア患者に、リツキシマブ製剤を投与すると劇症肝炎又は肝炎を増悪することがある。そのため、リツキシマブ製剤を投与する前に、B型肝炎ウイルス感染の有無を確認する必要がある。
4　誤
希釈時及び希釈後にリツキシマブ製剤に泡立つような激しい振動を加えるとリツキシマブ同士が凝集するおそれがある。そのため、リツキシマブ製剤には、希釈時及び希釈後に泡立つような激しい振動を加えないこととされている。
