# 第99回 問123

**問題文**
次の図は、ある栄養素の（A）1日摂取量の年次推移（1990年の値を100として表示してある）及び（B）2010年における年齢層別の1日摂取量（g）を示したものである。この栄養素はどれか。1つ選べ。

**選択肢**
1. 炭水化物
2. 鉄
3. カルシウム
4. タンパク質
5. ナトリウム
6. 脂質

**正解:** 5

**解説**
選択肢の栄養素のうち、グラフAより、摂取量（相対値）が減少傾向であり、グラフBより、摂取量が1日数 gとなるものとしてはナトリウムが考えられる。
2010年（平成22年）における食塩の摂取量は、男性11.4 g/日、女性9.8 g/日である。これをナトリウム摂取量に換算すると男性4.49 g/日、女性3.86 g/日となりグラフBと近い値であることが分かる。
