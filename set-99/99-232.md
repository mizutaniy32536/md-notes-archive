## ◆ ダニ咬傷後の軟膏塗布量（1FTU）を判断するシナリオ
260312

**問題**
49歳女性。秋の連休に山へハイキングに行った。帰宅後、上腕部に違和感を感じ約3mmのダニが付着しているのに気付いた。直ちに皮膚科を受診してダニを取り除き、近所の保険薬局を訪れて、処方された副腎皮質ステロイドを含有するチューブ入りの軟膏を受け取った。
女性は、医師より「ワン・フィンガー・チップ・ユニット(1FTU)」の量を塗布するように指示があったと薬剤師に話した。この単位に相当する軟膏の目安量はどれか。1つ選べ。

**選択肢**

1. 人差し指の付け根までの量
​
2. 人差し指の第2関節までの量
​
3. 人差し指の第1関節までの量
​
4. 人差し指の第1関節までの1/2量
​
5. 人差し指の指先に米粒大量

**正解と理由**
- `3. 人差し指の第1関節までの量`
- **理由の構成**
	- 1FTU = 成人の人差し指`先端から第1関節まで`の長さに絞り出した量（約0.5g）
	- 両手のひら2枚分（約500cm²）をカバーする目安

**解法の手順（思考の流れ）**
- 選択肢1：付け根まで = 1FTUを超える過剰量 → ✗
- 選択肢2：第2関節まで = 1FTUを超える → ✗
- 選択肢3：`第1関節まで` = **1FTUの定義そのもの** → **○**
- 選択肢4：第1関節の1/2 = 0.25g（半量） → ✗
- 選択肢5：米粒大 = 定義に合わない → ✗

**キーポイント**
- `1FTU`（Finger Tip Unit）= `人差し指の指先〜第1関節`まで絞り出した量 ≈ 0.5g
- 塗布面積の目安：`両手のひら2枚分`（成人）
- 軟膏・クリームの塗布量指導時に使用

（一言メモ）
- 1FTU = 第1関節。「1」と「第1関節」を対応させて覚える
- 約0.5g、両手のひら2枚分というのも一緒に覚える
