# 第99回 問237

**問題文**
10月のある夕方、薬局を男性が訪れた。その日、友人から初物のカキ（牡蠣）をもらったため、昼に友人や家族と自宅の居間で、生ガキと、炭火で焼きながら焼ガキを食したとのこと。全員が頭痛、嘔吐及びめまいを起こしたが、別室に移って休んだところ、少し落ち着いたとのことであった。
問237（実務）
この男性に対する薬剤師の対応として適切なのはどれか。2つ選べ。

**選択肢**
1. 次亜塩素酸ナトリウムを含む消毒液で調理器具を消毒することを勧めた。
2. 消毒用エタノールで調理器具を消毒することを勧めた。
3. 調理中は部屋の換気を十分に行うことを勧めた。
4. 一般用医薬品の胃腸薬を勧めた。
5. 医療機関への受診を勧めた。

**正解:** 3,5

**解説**
一酸化炭素は、炭素化合物の不完全燃焼により発生するため、一酸化炭素中毒を防止するためには、部屋の換気を十分に行う必要がある。また、一酸化炭素中毒は、急性期の症状（今回みられた症状）の後も遅延性神経障害が現れる可能性があるため、一酸化炭素中毒になった場合は、医療機関を受診する必要がある。
1　誤
次亜塩素酸ナトリウムを含む消毒液で調理器具を消毒することは、ノロウイルスによる食中毒を防止するための方法である。
2　誤
消毒用エタノールで調理器具を消毒することは、腸炎ビブリオや腸管出血性大腸菌による食中毒を防止するための方法である。
3　正
4　誤
一般用医薬品の胃腸薬では、一酸化炭素中毒の症状を改善することはできない。
5　正
