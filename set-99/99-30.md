# 第99回 問30

**問題文**
ブロチゾラムの催眠作用の発現に関わる受容体はどれか。1つ選べ。

**選択肢**
1. アデノシン受容体
2. ヒスタミンH1受容体
3. ムスカリン性アセチルコリン受容体
4. ベンゾジアゼピン受容体
5. カンナビノイド受容体

**正解:** 4

**解説**
ブロチゾラムはチエノジアゼピン系催眠薬であり、GABAA受容体−Cl－チャネル複合体のベンゾジアゼピン受容体（ベンゾジアゼピン結合部位）と結合することでGABAのGABAA受容体を介した中枢抑制作用を増加させ、催眠作用を示す。
