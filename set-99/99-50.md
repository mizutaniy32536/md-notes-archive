# 第99回 問50

**問題文**
製剤総則において、粒状に造粒した経口投与する製剤と規定されているのはどれか。1つ選べ。

**選択肢**
1. 発泡錠
2. 散剤
3. 顆粒剤
4. 分散錠
5. 懸濁剤

**正解:** 3

**解説**
1　誤
発泡錠は、製剤総則において、「水中で急速に発砲しながら溶解又は分散する錠剤である」と規定されている。
2　誤
散剤は、製剤総則において、「経口投与する粉末状の製剤である」と規定されている。
3　正
顆粒剤は、製剤総則において、「経口投与する粒状に造粒した製剤である」と規定されている。
4　誤
分散錠は、製剤総則において、「水に分散して服用する錠剤である」と規定されている。
5　誤
懸濁剤は、製剤総則において、「有効成分を微細均質に懸濁した経口液剤である」と規定されている。
