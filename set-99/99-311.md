# 第99回 問311

**問題文**
学校薬剤師のAさんは、日頃から学校環境の維持改善や医薬品の適正使用、薬物乱用防止などについて学校長に指導助言している。
問311（法規・制度・倫理）
また、Aさんは学校長から、社会問題になっている違法ドラッグについて助言してほしいと頼まれたので、医薬品医療機器等法における指定薬物の規制を確認した。指定薬物に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**
1. 中枢神経系の興奮若しくは抑制又は幻覚の作用を有している蓋然性が高く、身体に使用された場合に保健衛生上の危害の発生のおそれがある物が、指定薬物に指定される。
2. 指定薬物は、いかなる場合でも製造し、輸入し、販売し、授与し、又は販売若しくは授与の目的で貯蔵してはならない。
3. 指定薬物は、有害性が高いと認められた場合、自動的に麻薬としての規制も受ける。
4. 厚生労働大臣は、緊急を要する場合、薬事・食品衛生審議会の意見を聴かずに指定薬物を指定することができる。

**正解:** 1,4

**解説**
1　正
指定薬物とは、中枢神経系の興奮若しくは抑制又は幻覚の作用を有している蓋然性が高く、身体に使用された場合に保健衛生上の危害の発生のおそれがある物（大麻、覚せい剤、麻薬、向精神薬、あへん及びけしがらを除く）として、厚生労働大臣が薬事・食品衛生審議会の意見を聴いて指定するものをいう。
2　誤
指定薬物は、医療等の用途（疾病の診断、治療又は予防の用途及び人の身体に対する危害の発生を伴うおそれがない用途として厚生労働省令で定めるもの）以外の用途に供するために製造し、輸入し、販売し、授与し、所持し、購入し、若しくは譲り受け、又は医療等の用途以外の用途に使用してはならない。
3　誤
指定薬物は、有害性が高いと認められた場合でも、自動的に麻薬としての規制も受けることはない。
4　正
指定薬物を指定する際、原則、厚生労働大臣は、薬事・食品衛生審議会の意見を聴く必要があるが、緊急を要する場合、薬事・食品衛生審議会の意見を聴かずに指定薬物を指定することができる。なお、この場合において、厚生労働大臣は、速やかに、その指定に係る事項を薬事・食品衛生審議会に報告しなければならない。
