# 第99回 問156

**問題文**
抗てんかん薬に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**
1. ガバペンチンは、γ−アミノ酪酸（GABA）トランスポーターを阻害して、シナプス間隙のGABA量を増加させる。
2. クロバザムは、ベンゾジアゼピン受容体に結合し、GABA作動性神経伝達を増強する。
3. スルチアムは、炭酸脱水酵素を阻害し、神経細胞の過剰興奮を抑制する。
4. フェノバルビタールは、神経細胞内へのCl－流入を抑制し、神経細胞膜を過分極させる。

**正解:** 2,3

**解説**
1　誤
ガバペンチンは、電位依存性Ca2＋チャネルのα2δサブユニットに結合し、シナプス前膜におけるCa2＋流入を抑制して、グルタミン酸などの神経伝達物質の遊離を抑制する。
2　正
クロバザムは、ベンゾジアゼピン誘導体であり、GABAA受容体上のベンゾジアゼピン受容体（ベンゾジアゼピン結合部位）に結合して、GABAのGABAA受容体に対する親和性を強め、GABA作動性神経伝達を増強する。
3　正
スルチアムは、炭酸脱水酵素を阻害し、脳内CO2濃度を局所的に増大させることで、神経細胞の過剰興奮を抑制する。
4　誤
フェノバルビタールは、GABAA受容体上のバルビツレート結合部位（ピクロトキシン結合部位）に結合し、Cl－チャネル開口を介して細胞内へのCl－流入を促進することで細胞膜の過分極をおこす。
