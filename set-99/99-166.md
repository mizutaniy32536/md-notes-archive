# 第99回薬剤師国家試験 問166

## 問題文

薬物の生体膜輸送についての記述のうち、正しいのはどれか。**2つ選べ。**[1][2]

## 選択肢

1. 単純拡散による輸送速度は薬物濃度差に比例するが、促進拡散及び能動輸送では飽和性が見られる。[1][2]

2. 単純拡散による輸送は生体エネルギーを必要としないが、促進拡散及び能動輸送では生体エネルギーを必要とする。[1][2]

3. 単純拡散及び促進拡散の場合、薬物の濃度勾配に従って輸送されるが、能動輸送では濃度勾配に逆らって輸送される場合がある。[3][4]

4. 能動輸送はトランスポーターを介して起こるが、単純拡散及び促進拡散にはトランスポーターは関与しない。[4]

5. 単純拡散及び促進拡散の場合、構造類似体の共存による影響は受けないが、能動輸送では影響を受ける場合がある。[2][3]

## 正解番号

**1、3**[2][3]

## 各選択肢の解説

**選択肢1（正）**

単純拡散による輸送速度はFickの法則に従い、細胞内外の薬物濃度差に比例する。一方、促進拡散及び能動輸送はMichaelis-Menten式に従うため、飽和性が見られる。[3]

**選択肢2（誤）**

生体エネルギー（ATP）を必要とするのは能動輸送のみである。単純拡散と促進拡散の両者とも生体エネルギーを必要としない。促進拡散では濃度勾配に従った輸送が起こるが、これはトランスポーターを介して行われ、エネルギーは消費されない。[1][3]

**選択肢3（正）**

単純拡散及び促進拡散では、濃度勾配に従った輸送が起こる。一方、能動輸送では濃度勾配に逆らった輸送が起こることがある。[3]

**選択肢4（誤）**

促進拡散にはトランスポーター（担体）が関与する。したがって、この選択肢は誤りである。[2]

**選択肢5（誤）**

促進拡散はトランスポーターが関与しており、構造類似体の共存による競争的阻害の影響を受ける。したがって、この選択肢は誤りである。[2]
