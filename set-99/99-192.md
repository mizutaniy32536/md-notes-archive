# 第99回 問192

**問題文**
58歳男性。体重55 kg。直腸がんの再発のため、オキサリプラチン、フルオロウラシル、レボホリナートカルシウム、薬物Ｘによる治療を受けた。治療開始10日頃より、顔面にざ瘡様皮膚炎が起こり、その後皮膚亀裂及び爪周囲炎が見られた。この症状は薬物Ｘの副作用と考えられた。薬物Ｘにあてはまるものはどれか。2つ選べ。

**選択肢**
1. セツキシマブ
2. クリゾチニブ
3. テムシロリムス
4. イリノテカン硫酸塩水和物
5. パニツムマブ

**正解:** 1,5

**解説**
記述より、患者は直腸がんに対してオキサリプラチン、フルオロウラシル、レボホリナートカルシウムによるFOLFOX療法を用いている。FOLFOX療法は大腸がんに対する代表的な治療法の一つであり、分子標的薬であるベバシズマブ、セツキシマブ、パニツムマブが併用して投与されることもる。
1　正
セツキシマブはヒト上皮成長増殖因子受容体（EGFR）に対するモノクローナル抗体製剤であり、直腸がん等の治療に用いる。副作用としてざ瘡様皮膚炎、皮膚亀裂、爪周囲炎等がある。
2　誤
クリゾチニブはチロシンキナーゼ阻害薬であり、非小細胞肺がんの治療に用いられる。直腸がんの治療には用いられない。
3　誤
テムシロリムスは哺乳類ラパマイシン標的タンパク質（mTOR）阻害薬であり、根治切除不能又は転移性の腎細胞がんの治療に用いられる。直腸がんの治療には用いられない。
4　誤
イリノテカン硫酸塩水和物はトポイソメラーゼⅠ阻害薬であり、直腸がん等の治療に用いられる。代表的な副作用は下痢、骨髄抑制であり、ざ瘡様皮膚炎、皮膚亀裂、爪周囲炎等は見られない。
5　正
パニツムマブはヒト上皮成長増殖因子受容体（EGFR）に対するモノクローナル抗体製剤であり、直腸がん等の治療に用いる。副作用としてざ瘡様皮膚炎、皮膚亀裂、爪周囲炎等がある。
