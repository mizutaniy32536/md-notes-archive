## ◆ 塩素消毒が効かない病原体を選ぶシナリオ
260312

**問題**
塩素消毒に強い抵抗性を示す病原体はどれか。1つ選べ。

**選択肢**

1. インフルエンザウイルス
​
2. レジオネラ属菌
​
3. 大腸菌
​
4. クリプトスポリジウム
​
5. 赤痢菌

**正解と理由**
- `4. クリプトスポリジウム`
- **理由の構成**
	- クリプトスポリジウムは`オーシスト（胞嚢体）`を形成し、塩素消毒に強い抵抗性を持つ

**解法の手順（思考の流れ）**
- 選択肢1：インフルエンザウイルス → 塩素消毒で容易に不活化 → ✗
- 選択肢2：レジオネラ属菌 → 塩素消毒で死滅（ただしレジオネラは熱に強い） → ✗
- 選択肢3：大腸菌 → 塩素消毒に弱い → ✗
- 選択肢4：クリプトスポリジウム → `オーシスト`形成 → **塩素消毒に強い抵抗性** → **正しい**
- 選択肢5：赤痢菌 → 塩素消毒に弱い → ✗

**キーポイント**
- `クリプトスポリジウム` = 原虫、オーシストが塩素消毒に抵抗性
- 上水道への混入による集団感染事例あり（1996年 埼玉県越生町）
- 対策：`紫外線照射`または`煮沸`が有効

**関連知識の比較**

| 病原体 | 塩素消毒への抵抗性 | 対策 |
|---|---|---|
| クリプトスポリジウム | **強い** | 紫外線・煮沸 |
| 大腸菌 | 弱い | 塩素消毒で対応 |
| レジオネラ | 弱い | 塩素・熱処理 |
| インフルエンザウイルス | 弱い | 塩素消毒で対応 |

（一言メモ）
- クリプトスポリジウム = 「塩素が効かない」の代名詞
- 越生町の集団感染は超有名事例。覚えておく
