## ◆ 高齢者施設入居者の結核に関する記述を判断するシナリオ
260312

**問題**
80歳男性。特別養護老人ホームに入居中。発熱、食欲不振、全身の倦怠感、体重減少などの症状があり、病院で検査を受けた。胸部X線撮影で肺尖部陰影が認められ、喀痰検査で結核菌が検出されて結核と診断された。
結核に関する記述のうち、正しいのはどれか。２つ選べ。

**選択肢**

1. 主な感染経路は接触感染である。
​
2. 最近(2009～2011年)、我が国では年間約千人の患者が新たに発生している。
​
3. 最近(2009～2011年)、新たに発生した患者の約半数は70歳以上の高齢者である。
​
4. DOTS(直接服薬確認療法)が行われている。
​
5. 結核菌の消毒には消毒用エタノールは無効である。

**正解と理由**
- `3. 新規結核患者の約半数は70歳以上の高齢者`
- `4. DOTSが行われている`

**解法の手順（思考の流れ）**
- 選択肢1：結核の主な感染経路 = `空気感染（飛沫核感染）`。接触感染ではない → **誤り**
- 選択肢2：2009〜2011年の新規患者数 = 年間`約2万人以上`。「約千人」は大きく過小 → **誤り**
- 選択肢3：高齢化に伴い、新規結核患者の約半数が70歳以上 → **正しい**
- 選択肢4：DOTS（直接服薬確認療法）= 服薬確認と治療成功率向上のため導入 → **正しい**
- 選択肢5：消毒用エタノールは結核菌に`有効` → **誤り**

**キーポイント**
- 結核 = `空気感染（飛沫核感染）`。接触感染ではない
- `DOTS`：Direct Observation Treatment Short-course（短期直接監視下治療）
- 結核菌 = 消毒用エタノール`有効`（ただし芽胞には無効）
- 高齢者が感染者の半数を占める（老人施設でのリスク）

**関連知識の比較**

| 項目 | 内容 |
|---|---|
| 感染経路 | **空気感染（飛沫核）** |
| 感染症法分類 | 二類感染症 |
| 治療 | 抗結核薬（HRZE）＋DOTS |
| エタノール消毒 | 有効 |

（一言メモ）
- 「接触感染」は絶対ひっかかってはいけない。結核 = 空気感染
- DOTSは「確実に飲ませる仕組み」
