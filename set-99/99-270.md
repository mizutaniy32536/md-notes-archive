# 第99回 問270

**問題文**
40歳男性。体重65 kg。病院で腎移植後、シクロスポリンを含む処方による治療を継続中である。1年後の定期検診で脂質異常症と高血圧症を指摘された。
問270（実務）
これらの症状を改善する次の薬物のうち、シクロスポリンと併用禁忌なのはどれか。1つ選べ。

**選択肢**
1. アムロジピンベシル酸塩
2. イコサペント酸エチル
3. カルテオロール塩酸塩
4. コレスチラミン
5. ロスバスタチンカルシウム

**正解:** 5

**解説**
シクロスポリン投与中にロスバスタチンを併用すると、ロスバスタチンの血漿中濃度が上昇することがある。そのため、シクロスポリン服用中にロスバスタチンを投与するとロスバスタチンの副作用である横紋筋融解症が発現しやすくなるため、シクロスポリンとロスバスタチンは併用禁忌となっている。
