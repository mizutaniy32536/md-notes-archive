# 第99回 問61

**問題文**
不眠症の適応を有する薬物のうち、メラトニン受容体を介して効果を発現するのはどれか。1つ選べ。

**選択肢**
1. ゾルピデム酒石酸塩
2. ゾピクロン
3. ラメルテオン
4. リルマザホン塩酸塩水和物
5. トリアゾラム

**正解:** 3

**解説**
メラトニン受容体にはMT1、MT2受容体が存在し、これらが刺激されると睡眠−覚醒リズムの調整により睡眠が誘発される。ラメルテオンはMT1、MT2受容体の両方に高い親和性を有するメラトニン受容体刺激薬であり、不眠症の治療に用いられる。なお、選択肢にあるラメルテオン以外の薬物はベンゾジアゼピン系及びベンゾジアゼピン類似薬であり、いずれもGABAA受容体を介して睡眠作用を発現する。
