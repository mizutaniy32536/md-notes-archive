# 第99回 問52

**問題文**
無菌製剤と規定されているのはどれか。1つ選べ。

**選択肢**
1. 含嗽剤
2. 吸入液剤
3. 注腸剤
4. 眼軟膏剤
5. 軟膏剤

**正解:** 4

**解説**
製剤総則において無菌製剤と規定されている剤形には、注射剤、腹膜透析用剤、点眼剤、眼軟膏剤などがある。
1　誤
含嗽剤は、口腔、咽頭などの局所に適応する液状の製剤である。
2　誤
吸入液剤は、ネブライザなどにより適応する液状の吸入剤である。
3　誤
注腸剤は、肛門を通して適応する液状又は粘稠なゲル上の製剤である。
4　正
眼軟膏剤は、結膜嚢などの眼組織に適応する半固形の無菌製剤である。
5　誤
軟膏剤は、皮膚に塗布する有効成分を基剤に溶解又は分散させた半固形状の製剤である。
