# 第99回 問101

**問題文**
次の構造式で示される日本薬局方収載医薬品クロラムフェニコールの正しい化学名はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 99-101 を参照

**選択肢**
1. 2,2−Dichloro−N−[(1S,2S)-1,3−dihydroxy-3-(4-nitrophenyl)propan−1−yl]acetamide
2. 2,2−Dichloro−N−[(1S,2S)-1,3-dihydroxy-1-(4-nitrophenyl)propan−2−yl]acetamide
3. 2,2−Dichloro−N−[(1S,2S)-1,3-dihydroxy-3-(4-nitrophenyl)propan−2−yl]acetamide
4. 2,2−Dichloro−N−[(1R,2R)-1,3-dihydroxy-3-(4-nitrophenyl)propan−1−yl]acetamide
5. 2,2−Dichloro−N−[(1R,2R)-1,3-dihydroxy-1-(4-nitrophenyl)propan−2−yl]acetamide

**正解:** 5

**解説**
本構造の主たる官能基はアミドであるため、主鎖はアセトアミド（acetamide）である。
主鎖のN 位に結合している側鎖は、主骨格が炭素3つで真ん中の炭素と主鎖が結合しているところから、2−プロピル基（propan−2−yl）であり、これに、2つのヒドロキシ基（hydroxy）と1つの4−ニトロフェニル基（4−nitrophenyl）が結合しているため、1,3−dihydroxy−1−(4−nitrophenyl)propan−2−ylと命名される。（置換基が多い炭素の位置番号が小さくなるように命名するため、ヒドロキシ基と4−ニトロフェニル基の両方が結合した炭素原子を1位としている。）
また、本側鎖の不斉炭素原子はどちらもR配置であるため、クロラムフェニコールの正しい化学名は、2,2−Dichloro−N−[(1R,2R)−1,3−dihydroxy−1−(4−nitrophenyl)propan−2−yl]acetamideとなる。
