# 第99回 問44

**問題文**
腸肝循環を受けやすい薬物はどれか。1つ選べ。

**選択肢**
1. リチウム
2. ゲンタマイシン
3. セファレキシン
4. プラバスタチン
5. アシクロビル

**正解:** 4

**解説**
腸肝循環とは、胆汁中排泄された薬物が消化管へ移行した後、消化管より再び吸収され、門脈から肝臓に取り込まれることである。腸管循環されやすい代表的な薬物には、インドメタシン、ジゴキシン、モルヒネ、プラバスタチンなどがある。なお、プラバスタチンは有機アニオン輸送系により肝臓に取り込まれ、MRP2（multidrug resistance associated protein2）により胆汁中に排泄された後、消化管内で再度吸収されることがある。
