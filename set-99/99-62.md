# 第99回 問62

**問題文**
バセドウ病に関する記述のうち、正しいのはどれか。1つ選べ。

**選択肢**
1. 体重増加を認める。
2. 脈拍数が減少する。
3. 発汗が抑制される。
4. 血清TSH値が増加する。
5. 手指の振戦を認める。

**正解:** 5

**解説**
バセドウ病は、免疫の異常により体内で抗TSH受容体抗体が産生され、TSH受容体が持続的に刺激されることによっておこる甲状腺機能亢進症である。そのため、体重減少、頻脈、発汗、手足震戦など、甲状腺ホルモン作用の増強に基づく諸症状を認める。また、体内のチロキシン（T4）、トリヨードチロニン（T3）が増加するため、負のフィードバックによって血清TSH値は低下する。
