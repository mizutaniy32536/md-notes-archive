# 第99回 問47

**問題文**
治療薬物モニタリング（TDM）が必要とされる代表的な抗生物質はどれか。1つ選べ。

**選択肢**
1. アンピシリン
2. イミペネム
3. エリスロマイシン
4. テイコプラニン
5. セフジニル

**正解:** 4

**解説**
治療薬物モニタリング（TDM）が必要とされる代表的な抗生物質は、グリコペプチド系抗生物質（バンコマイシン、テイコプラニン等）やアミノグリコシド系抗生物質（アルベカシン、アミカシン、ゲンタマイシン、ストレプトマイシン等）である。グリコペプチド系抗生物質であるテイコプラニンは、副作用である腎障害を防止するためにTDMが必要とされる。
