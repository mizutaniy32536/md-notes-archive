# 第99回 問339

**問題文**
女性患者から、現在の肥満の程度を教えて欲しいと尋ねられた。この患者の身長は150 cm、体重は75 kgである。現在の体重は標準体重の何倍か。最も近い値を1つ選べ。ただし、標準体重はBMI法で求めることとする。

**選択肢**
1. 1.2
2. 1.3
3. 1.4
4. 1.5
5. 1.6

**正解:** 4

**解説**
肥満の判定に用いられるBMIは、体重（kg）÷身長（m）2で算出することができる。
BMIの標準値は22であるため、BMI法により標準体重X（kg）を以下のように求めることができる。
22＝X÷1.52
X＝49.5（kg）
患者の体重が75 kgであり、75÷49.5＝1.5となるため、患者の体重は標準体重の約1.5倍となる。
