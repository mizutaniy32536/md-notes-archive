## ◆ 我が国の性感染症の疫学に関する正誤判断シナリオ
260312

**問題**
我が国における性感染症に関する記述のうち、正しいのはどれか。２つ選べ。

**選択肢**

1. 性器クラミジア感染症の患者数は、性感染症の中で最も多く、その対策が急務となっている。
​
2. 新規HIV感染者(平成20-24年)の大半は男性であり、感染の原因としては、同性間の性的接触によるものが最も多い。
​
3. B型肝炎については、輸血を介した新規の発症者も増え続けている。
​
4. 淋菌感染症の患者数は公衆衛生の向上により30年前に比べ激減しており、平成20年以降感染の報告はない。
​
5. 梅毒の予防に有効なワクチンが実用化されている。

**正解と理由**
- `1. 性器クラミジア感染症の患者数は性感染症の中で最も多い`
- `2. 新規HIV感染者の大半は男性で、同性間の性的接触が最も多い`

**解法の手順（思考の流れ）**
- 選択肢1：性感染症の患者数 = クラミジアが断トツ1位（男性の40%、女性の60%） → **正しい**
- 選択肢2：HIV新規感染者の大半は男性、感染経路は同性間性的接触が最多 → **正しい**
- 選択肢3：B型肝炎の輸血感染は`減少`傾向（ワクチン普及・検査強化で）→ **誤り**
- 選択肢4：淋菌感染症は減少したが平成20年以降も報告`あり`（ゼロではない）→ **誤り**
- 選択肢5：梅毒の予防ワクチンは現在`実用化されていない` → **誤り**

**キーポイント**
- `クラミジア`：日本の性感染症No.1
- `HIV`：男性が大半、同性間接触が最多感染経路
- 梅毒ワクチンは`存在しない`（抗菌薬で治療）

**関連知識の比較**

| 性感染症 | 病原体 | 患者数順位 |
|---|---|---|
| 性器クラミジア | クラミジア・トラコマチス | **1位** |
| 淋病 | 淋菌 | 2位 |
| 梅毒 | 梅毒トレポネーマ | 近年急増中 |

- 梅毒ワクチンは存在しない！ペニシリンで治療
