# 第99回 問288

**問題文**
65歳女性。身長160 cm、体重50 kg。てんかんの既往があり、現在フェニトイン100 mg錠を1回1錠、1日2回朝夕食後服用している。34歳時に子宮筋腫の手術を受け輸血された。55歳からC型慢性肝炎による代償期肝硬変の診断で近医に通院していた。今回、以下の薬剤が追加となった。

問288（実務）
この処方に関する記述のうち、正しいのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 99-288 を参照

**選択肢**
1. 溶解する水は、80℃以上の湯を用いる。
2. 溶解後、室温で24時間まで保存できる。
3. 溶解後の浸透圧が高いので下痢に注意する。
4. 主成分は芳香族アミノ酸である。
5. この処方だけで1日に必要な熱量を摂取できる。

**正解:** 3

**解説**
1　誤
本剤を熱湯で溶解するとタンパク質が変性してしまうおそれがある。そのため、本剤を溶解する際は水又は温湯（約50℃）を用いる。
2　誤
本剤の添付文書には、「用時調製するが、調製後10時間以内に使用すること。また、調製後やむなく保存する場合は冷所保存が望ましい。」と記載されているため、溶解後、室温で24時間保存することはできない。
3　正
本剤溶解後の浸透圧比（生理食塩水の浸透圧を1としたときの比率）は約2である。本剤は浸透圧比が大きいため、消化管内で水分を吸収することから、副作用として下痢、腹部膨満感といった消化器症状が現れることがある。
4　誤
本剤の主成分は分岐鎖アミノ酸（バリン、ロイシン、イソロイシン）である。なお、本剤は、芳香族アミノ酸（フェニルアラニン、チロシン）を制限し、分岐鎖アミノ酸（バリン、ロイシン、イソロイシン）を多く含有することで、Fischer比（分岐鎖アミノ酸／芳香族アミノ酸）の低下による肝性脳症の治療に用いられている。
5　誤
本剤を1日3包用いた場合の摂取熱量は630 kcal／日であり、本症例における患者の1日に必要な熱量（1,700 kcal／日）を摂取することはできない。そのため、残りの必要熱量については、食事により摂取する必要がある。
