# 第99回 問27

**問題文**
部分受容体刺激薬に関する記述として正しいのはどれか。1つ選べ。

**選択肢**
1. 受容体の立体構造を変化させる力は、完全受容体刺激薬と同等である。
2. 完全受容体刺激薬の存在下で相乗作用を示す。
3. 固有活性によって最大作用が決まる。
4. 内因性受容体刺激物質の作用を変化させない。
5. 濃度を上げれば受容体を最大限に活性化する。

**正解:** 3

**解説**
受容体刺激薬は、受容体と結合した際の受容体刺激能によって、完全受容体刺激薬（完全アゴニスト）と部分受容体刺激薬（部分アゴニスト）に分けられる。薬物が受容体に100％結合した時、その受容体を最大限（100％）活性化させることができるもの（内活性＝1）を完全受容体刺激薬といい、100％未満の強さで活性化するもの（0＜内活性＜1）を部分受容体刺激薬という。
1　誤
部分受容体刺激薬の立体構造を変化させる力（受容体を活性化する力）は、完全受容体刺激薬よりも弱い。
2　誤
部分受容体刺激薬は完全受容体刺激薬と拮抗的に作用するため、完全受容体刺激薬の作用を減弱させるおそれがある。
3　正
固有活性（内活性）とは、各受容体刺激薬により生じる最大作用の強さを表す値のことであり、薬物固有の値である。したがって、固有活性によって最大作用が決まる。
4　誤
部分受容体刺激薬は内因性受容体刺激物質と拮抗的に作用するため、内因性受容体刺激物質の作用を減弱させるおそれがある。
5　誤
濃度を増加させ、受容体に100％結合させても、部分受容体刺激薬は、100％の活性作用を示す（受容体を最大限に活性化する）ことはできない。
