# 第99回 問333

**問題文**
56歳男性。以前より内科で処方されているワルファリンカリウム錠を服用している。以前の服薬指導時には、何も問題はなかったが、今回の来局時に「2日前から歯磨き時に歯茎の出血が止まりにくくなった｡」との訴えがあった。出血の原因として疑われるのはどれか。2つ選べ。

**選択肢**
1. シメチジン含有の一般用医薬品の服用
2. クロレラの摂取
3. イコサペント酸エチル含有のサプリメントの摂取
4. 納豆の摂取

**正解:** 1,3

**解説**
本患者は、ワルファリンカリウム錠を服用しており、来局時に「2日前から歯磨き時に歯茎の出血が止まりにくくなった。」と訴えがあったことから、ワルファリンカリウムの作用が増強していると考えられる。
1　正
ワルファリンカリウムはCYPにより代謝される薬物であるため、CYP阻害作用を有するシメチジン含有の一般用医薬品と併用すると、ワルファリンカリウムの代謝が抑制され、作用が増強することがある。
2　誤
ワルファリンカリウムは、肝臓においてビタミンKと拮抗し、血液凝固因子の産生を抑制することにより抗凝血作用を示す。そのため、クロレラや納豆などのビタミンK含有食品の摂取により、ワルファリンカリウムの作用が減弱する。
3　正
イコサペント酸エチルは、血小板凝集抑制作用を有する。よって、ワルファリンカリウム服用中の患者がイコサペント酸エチル含有のサプリメントを摂取すると、出血が生じやすくなる。
4　誤
