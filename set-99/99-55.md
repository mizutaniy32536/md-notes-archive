# 第99回 問55

**問題文**
消化管障害の軽減を目的としたプロドラッグはどれか。1つ選べ。

**選択肢**
1. テモカプリル塩酸塩
2. ヒドロコルチゾンコハク酸エステルナトリウム
3. アセメタシン
4. カンデサルタンシレキセチル
5. エリスロマイシンステアリン酸塩

**正解:** 3

**解説**
1　誤
テモカプリル塩酸塩は、テモカプリラートの脂溶性を高め、吸収性を改善することを目的としたプロドラッグである。
2　誤
ヒドロコルチゾンコハク酸エステルナトリウムは、ヒドロコルチゾンにコハク酸基を導入しナトリウム塩とすることで、水溶性を増大させ作用を持続化することを目的としたプロドラッグである。
3　正
アセメタシンは、インドメタシンをエステル誘導体化することで、消化管障害を軽減させることを目的としたプロドラッグである。
4　誤
カンデサルタンシレキセチルは、カンデサルタンに脂溶性を付与することで生体膜透過性を高め、吸収性を改善することを目的としたプロドラッグである。
5　誤
エリスロマイシンステアリン酸塩は、エリスロマイシンをステアリン酸塩とすることで、胃内安定性を高めることを目的としたプロドラッグである。
