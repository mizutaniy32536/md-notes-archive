# 第99回 問45

**問題文**
フェロジピン服用患者が避けるべき飲食物はどれか。1つ選べ。

**選択肢**
1. グレープフルーツジュース
2. 牛乳
3. コーヒー
4. ブロッコリー
5. 納豆

**正解:** 1

**解説**
Ca拮抗薬であるフェロジピンは、CYP3A4で代謝される。グレープフルーツジュースに含まれるフラノクマリン類は、小腸上皮細胞に存在するCYP3A4を阻害する。よって、フェロジピンとグレープフルーツジュースを併用すると、フェロジピンの血中濃度が上昇することがある。
なお、Ca拮抗薬以外にグレープフルーツジュースと相互作用を示す薬物には、抗血小板薬のシロスタゾール、免疫抑制薬のシクロスポリン、HIVプロテアーゼ阻害薬のサキナビルなどが挙げられる。
