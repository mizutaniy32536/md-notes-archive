# 第99回 問22

**問題文**
生体への影響を考慮した電離放射線の実効線量の単位はどれか。1つ選べ。

**選択肢**
1. クーロン
2. グレイ
3. シーベルト
4. ベクレル
5. カンデラ

**正解:** 3

**解説**
実効線量とは、各臓器、各組織の放射線感受性を考慮した放射線量のことであり、その単位は、シーベルト（Sv）で表される。
1　誤
クーロン（C）は、電気量及び電荷量を表す単位である。また、クーロンは放射線の照射線量の単位にも用いられ、その際にはクーロン毎キログラム（C/kg）で表す。
2　誤
グレイ（Gy）は、吸収線量を表す単位である。
3　正
4　誤
ベクレル（Bq）は、放射能量を表す単位である。
5　誤
カンデラ（Cd）は、光度を表す単位である。
