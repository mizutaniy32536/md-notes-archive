## ◆ 小児の髄膜炎予防のため平成25年から定期接種になったワクチンを選ぶシナリオ
260312

**問題**
小児の髄膜炎による死亡や後遺症を予防することを目的として、平成25年から定期接種が行われることになったワクチンはどれか。２つ選べ。

**選択肢**

1. 不活化ポリオワクチン
​
2. 麻しん・風しん混合ワクチン
​
3. インフルエンザ菌b型(Hib)ワクチン
​
4. ヒトパピローマウイルスワクチン
​
5. 小児用肺炎球菌ワクチン

**正解と理由**
- `3. インフルエンザ菌b型(Hib)ワクチン`
- `5. 小児用肺炎球菌ワクチン`
- **理由の構成**
	- 両者とも`細菌性髄膜炎`の主要原因菌に対するワクチン
	- 平成25年（2013年）から定期接種化

**解法の手順（思考の流れ）**
- 選択肢1：不活化ポリオワクチン → 定期接種開始は`平成24年9月`→ 髄膜炎予防目的でもない → ✗
- 選択肢2：MRワクチン → 定期接種開始は`平成18年4月` → ✗
- 選択肢3：Hibワクチン → `平成25年から定期接種`、細菌性髄膜炎の主要原因 → **○**
- 選択肢4：HPVワクチン → 子宮頸がん予防が目的、髄膜炎とは無関係 → ✗
- 選択肢5：小児用肺炎球菌ワクチン → `平成25年から定期接種`、細菌性髄膜炎の主要原因 → **○**

**キーポイント**
- 小児細菌性髄膜炎の2大原因：`Hib`（インフルエンザ菌b型）、`肺炎球菌`
- 平成25年に両者が同時に定期接種化
- HPVワクチンは子宮頸がん予防（髄膜炎とは別）

**関連知識の比較**

| ワクチン | 定期接種開始時期 | 目的 |
|---|---|---|
| Hibワクチン | **平成25年** | 細菌性髄膜炎予防 |
| 小児用肺炎球菌 | **平成25年** | 細菌性髄膜炎・肺炎予防 |
| 不活化ポリオ | 平成24年9月 | ポリオ予防 |
| MRワクチン | 平成18年 | 麻しん・風しん予防 |
| HPVワクチン | 平成25年（同年だが目的が違う） | 子宮頸がん予防 |

（一言メモ）
- Hib + 肺炎球菌 = セットで「細菌性髄膜炎予防」として平成25年定期接種化
- HPVは同じ平成25年でも目的が「子宮頸がん予防」。ひっかけ注意
