## ◆ ダニ媒介感染症を選ぶシナリオ
260312

**問題**
49歳女性。秋の連休に山へハイキングに行った。（前問と同一症例）
ダニが媒介する感染症はどれか。２つ選べ。

**選択肢**

1. 破傷風
​
2. 重症急性呼吸器症候群(SARS)
​
3. 日本脳炎
​
4. 重症熱性血小板減少症候群(SFTS)
​
5. つつが虫病

**正解と理由**
- `4. 重症熱性血小板減少症候群(SFTS)`
- `5. つつが虫病`

**解法の手順（思考の流れ）**
- 選択肢1：破傷風 → `傷口から破傷風菌`が感染。ダニ媒介ではない → ✗
- 選択肢2：SARS → `飛沫感染`。ダニ媒介ではない → ✗
- 選択肢3：日本脳炎 → `蚊`が媒介。ダニではない → ✗
- 選択肢4：SFTS（重症熱性血小板減少症候群） → `マダニ`が媒介 → **○**
- 選択肢5：つつが虫病 → `ツツガムシ（ダニの一種）`が媒介 → **○**

**キーポイント**
- `SFTS`：マダニ媒介のウイルス感染症、致死率10〜30%
- `つつが虫病`：ツツガムシリケッチアを保有するツツガムシ（幼虫が吸着）
- 日本脳炎 → 蚊。混同注意

**関連知識の比較**

| 感染症 | 媒介生物 |
|---|---|
| SFTS | **マダニ** |
| つつが虫病 | **ツツガムシ（ダニ類）** |
| 日本脳炎 | コガタアカイエカ（蚊） |
| デング熱 | ヒトスジシマカ（蚊） |
| 破傷風 | 媒介なし（土壌の芽胞） |

（一言メモ）
- ダニ媒介 = SFTS・つつが虫病・ライム病（日本国試ではSFTSとつつが虫病が頻出）
- 日本脳炎=蚊。ダニと蚊の混同はよくあるひっかけ
