# 第99回 問31

**問題文**
ペロスピロンが統合失調症の陽性症状を改善する機序はどれか。1つ選べ。

**選択肢**
1. アドレナリンα2受容体刺激
2. セロトニン5−HT2受容体遮断
3. セロトニン5−HT4受容体刺激
4. ドパミンD1受容体遮断
5. ドパミンD2受容体遮断

**正解:** 5

**解説**
ペロスピロンはドパミンD2受容体遮断作用とセロトニン5−HT2受容体遮断作用を併せもつセロトニン・ドパミンアンタゴニスト（SDA）であり、統合失調症の治療に用いられる。ドパミンD2受容体遮断作用により陽性症状（幻覚、妄想など）を、セロトニン5−HT2受容体遮断作用により陰性症状（感情の平板化、自閉など）を改善する。
