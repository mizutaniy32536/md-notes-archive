# 99-38

**問題文**
ヒスタミンH1受容体遮断作用を有する抗アレルギー薬はどれか。1つ選べ。

**選択肢**
1. ザフィルルカスト
2. トラニラスト
3. フェキソフェナジン
4. セラトロダスト
5. ラマトロバン

**正解:** 3

**解説**
1. 誤 ザフィルルカストはロイコトリエン受容体遮断薬である。
2. 誤 トラニラストはケミカルメディエーター遊離抑制薬である。
3. 正 フェキソフェナジンはヒスタミンH1受容体遮断薬である。
4. 誤 セラトロダストはトロンボキサンA2受容体遮断薬である。
5. 誤 ラマトロバンはトロンボキサンA2受容体遮断薬である。

**キーポイント**
- フェキソフェナジンは第二世代抗ヒスタミン薬。
- 抗アレルギー薬は薬名から系統を即答できるようにしておく。
