# 第99回 問88

**問題文**
注射用バンコマイシン塩酸塩の添付文書に、使用上の注意として「60分以上かけて点滴静注すること」と記載がある。これによって予防できる副作用はどれか。1つ選べ。

**選択肢**
1. 回転性のめまい
2. 水様便を伴う下痢
3. 下肢の浮腫
4. 顔や頸部の発赤
5. 眼球結膜の黄変

**正解:** 4

**解説**
バンコマイシンを急速なワンショット静注又は短時間での点滴静注と行うとヒスタミンが遊離され、顔、頸などに紅斑性充血、掻痒等のレッドネック（レッドマン）症候群や血圧低下の副作用が発現することがある。このため、注射用バンコマイシン塩酸塩は60分以上かけて点滴静注する必要がある。
