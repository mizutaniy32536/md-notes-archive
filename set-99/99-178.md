# 第99回 問178

**問題文**
ターゲティングに関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**
1. 受動的ターゲティングとは、標的部位を特異的に認識できる抗体や糖タンパク質などを薬物に結合させて体内分布を制御する手法である。
2. 逆ターゲティングとは、副作用を発現する部位への薬物分布を回避する手法である。
3. リポソームは、内部の疎水性コアに薬物を含有させた高分子ミセル製剤である。
4. 昇圧化学療法とは、抗がん薬をマイクロカプセルなどのキャリアーに封入して、腫瘍の栄養動脈に注入する治療法である。
5. 標的細胞内で特異的に発現する酵素により親薬物に変換されるプロドラッグを用いることで、薬物の標的細胞への選択的作用が得られる。

**正解:** 2,5

**解説**
1　誤
能動的ターゲティングに関する記述である。受動的ターゲティングは、生体の血管透過性や異物処理機構などの生理学的特徴や、正常組織と病態組織の違いを利用して、目的部位に薬物を受動的に送達させる手法である。
2　正
逆ターゲィングとは、正常細胞や副作用を発現する部位への薬物分布を回避する手法である。
3　誤
リポソームは、脂質二分子膜より構成される高分子ミセル製剤であり、内部に親水性コア（水相）をもち、親水性薬物は内部の親水性コアに、疎水性薬物は疎水性二重膜に保持される。
4　誤
化学塞栓療法に関する記述である。昇圧化学療法とは、アンギオテンシンⅡに対する血管の感受性の違いを利用した薬物送達システムである。腫瘍血管はアンギオテンシンⅡに対する血管収縮性に乏しいため、アンギオテンシンⅡを投与すると正常血管のみが収縮することで、腫瘍組織への血流量が特異的に増加し、抗がん薬を腫瘍組織へ送達することが可能となる。
5　正
記述通りである。例えば、ドキシフルリジンやカペシタビンは、腫瘍組織において活性が高い酵素により5−フルオロウラシルに変換されることで腫瘍細胞への選択的作用が得られる抗がん薬である。
