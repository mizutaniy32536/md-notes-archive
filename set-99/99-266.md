# 第99回 問266

**問題文**
40歳男性。薬局で下記の成分を含有する一般用医薬品を購入した。
スクリーンショット 2017-07-10 16.34.56.png
問266（実務）
この一般用医薬品の主な効能・効果として適切なのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 99-266 を参照

**選択肢**
1. 胃酸過多、胸やけ、胃部不快感、胃痛
2. 下痢、軟便
3. かぜの諸症状（のどの痛み、鼻水、鼻づまり、くしゃみなど）の緩和
4. 急性鼻炎、アレルギー性鼻炎又は副鼻腔炎による諸症状の緩和
5. 慢性便秘、常習性便秘

**正解:** 1

**解説**
本剤に含まれる成分の作用を以下にまとめる。
メチルメチオニンスルホニウムクロライド：粘膜修復作用
水酸化アルミニウム・炭酸水素ナトリウム共沈物、酸化マグネシウム：制酸作用
ロートエキス3倍散：胃酸分泌及び胃腸運動抑制作用
ウイキョウ末、ショウキョウ末、チョウジ末：健胃作用
ビオヂアスターゼ2000（消化酵素）：消化促進作用
これらのことから、この一般用医薬品の主な効能・効果は、胃酸過多、胸やけ、胃部不快感、胃痛であると考えられる。
