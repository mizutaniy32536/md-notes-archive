# 第99回 問122

**問題文**
「日本人の食事摂取基準（2010年版）」において耐容上限量が設定されているビタミンはどれか。2つ選べ。

**選択肢**
1. ビタミンB1
2. ビタミンE
3. ビオチン
4. ビタミンC
5. ビタミンA

**正解:** 2,5

**解説**
耐容上限量は、過剰に摂取した場合に健康障害をもたらす物質について定められている。ビタミンでは、ビタミンA（過剰摂取で頭痛などの症状がみられる）、ビタミンD、ビタミンE（過剰摂取で出血傾向がみられる）、ビタミンB6、ナイアシン、葉酸の6種に耐容上限量が設定されている。
