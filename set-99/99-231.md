## ◆ 抗結核薬エタンブトールの最重大副作用を選ぶシナリオ
260312

**問題**
80歳男性。特別養護老人ホームに入居中。（前問と同一症例）結核と診断された。この患者には、イソニアジド、リファンピシン、エタンブトール塩酸塩が処方された。エタンブトール塩酸塩の最も重大な副作用として、薬剤師が留意しなければならないのはどれか。1つ選べ。

**選択肢**

1. 聴覚障害
​
2. 腎機能障害
​
3. 視力障害
​
4. 嗅覚障害
​
5. 味覚障害

**正解と理由**
- `3. 視力障害`
- **理由の構成**
	- エタンブトール = `視神経障害`による視力低下・視野狭窄が最重大副作用
	- 不可逆的になる可能性があり、定期的な眼科検査が必要

**解法の手順（思考の流れ）**
- 選択肢1：聴覚障害 → `ストレプトマイシン`の副作用 → ✗
- 選択肢2：腎機能障害 → あるが最重大ではない → ✗
- 選択肢3：視力障害（視神経障害） → **エタンブトール固有の最重大副作用** → **○**
- 選択肢4：嗅覚障害 → エタンブトールとは無関係 → ✗
- 選択肢5：味覚障害 → まれで最重大ではない → ✗

**キーポイント**
- `エタンブトール（EB）` → `視力障害`（視神経炎）
- 定期的な`眼科検査`が必須

**関連知識の比較**

| 抗結核薬 | 略号 | 主な副作用 |
|---|---|---|
| イソニアジド | H | 末梢神経炎、肝障害 |
| リファンピシン | R | 肝障害、体液が橙赤色に |
| エタンブトール | **E** | **視力障害（視神経炎）** |
