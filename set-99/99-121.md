# 第99回 問121

**問題文**
タンパク質とその栄養価に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**
1. タンパク質100 gに含まれる窒素の量は平均6.25 gである。
2. 生物価が高いタンパク質ほど正味タンパク質利用率も高い。
3. 生物価に対する正味タンパク質利用率の割合が消化吸収率である。
4. タンパク質中の必須アミノ酸の中で、含量が最も少ないアミノ酸を第一制限アミノ酸という。
5. 一定時間に尿中に排泄される窒素量から、タンパク質の体内での燃焼量を推定することができる。

**正解:** 3,5

**解説**
1　誤
タンパク質の平均窒素含有量は約16％であり、タンパク質100 gに含まれる窒素の量は約16 gである。なお、タンパク質の平均窒素含有量が16％であることから、窒素係数（タンパク質の平均窒素含有率の逆数）は、6.25（100/16）であり、窒素含有量が分かれば窒素係数より、タンパク質量を逆算できる。
2　誤
タンパク質の栄養価の指標として生物価と正味タンパク質利用率があり、以下のように求めることができる。
生物価＝体内保留窒素量／吸収窒素量
正味タンパク質利用率＝体内保留窒素量／摂取窒素量　又は　生物価×消化吸収率
上記の式より、生物価が高くても消化吸収率が低い場合、正味タンパク質利用率が低くなることがある。
3　正
正味タンパク質利用率＝生物価×消化吸収率であることから、消化吸収率＝正味タンパク質利用率÷生物価である。
4　誤
タンパク質の理想的なアミノ酸組成に対して、不足している必須アミノ酸を制限アミノ酸といい、最も不足率が大きい必須アミノ酸を第一制限アミノ酸という。
5　正
タンパク質の窒素成分はすべて尿中に排泄される。そのため、尿中に排泄される窒素量に窒素係数6.25を乗じることで、体内で燃焼したタンパク質量を推定することができる。
