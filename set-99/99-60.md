# 第99回 問60

**問題文**
統合失調症の陰性症状として正しいのはどれか。1つ選べ。

**選択肢**
1. 妄想
2. 幻覚
3. 失見当識
4. 食欲亢進
5. 意欲欠如

**正解:** 5

**解説**
統合失調症は、ドパミンやセロトニン、グルタミン酸などの脳内伝達物質の作用異常によっておこる内因性精神疾患である。症状は幻覚、幻聴、妄想、思考の混乱、異常行動などの陽性症状と、意欲欠如、無気力、無関心などの陰性症状に大別される。よって、陰性症状として正しいのは選択肢5の意欲欠如である。また、選択肢3の失見当識は認知障害の一種であり、陽性・陰性症状とは別のものである。
