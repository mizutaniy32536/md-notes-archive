# 第99回 問193

**問題文**
医薬品の承認申請時に作成されるCTD（Common Technical Document）に含まれないのはどれか。2つ選べ。

**選択肢**
1. 品質に関する文書
2. 非臨床試験報告書
3. 臨床試験報告書
4. 薬価設定に関する文書
5. 販売計画書

**正解:** 4,5

**解説**
CTD（Common Technical Document）とは、医薬品承認申請のための国際共通化資料のことである。CTDの構成（5部構成）を以下に示す。
第1部：申請書など行政情報及び添付文書に関する情報
第2部：CTDの概要
第3部：品質に関する文書（選択肢1）
第4部：非臨床試験報告書（選択肢2）
第5部：臨床試験報告書（選択肢3）
よって、薬価設定に関する文書（選択肢4）、販売計画書（選択肢5）はCDTには含まれない。
