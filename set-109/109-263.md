# 109-263

### 第109回薬剤師国家試験 問263

**問題文**  
しばらくして白内障の手術（眼内レンズ挿入術）のため入院となり、**処方2**が中止となった。**処方2**に替えて、新たに異なる作用機序の薬物を追加することになった。追加する薬物の作用機序として、適切なのはどれか。**2つ選べ**。[1][4]

**全選択肢**  
1. アドレナリンα<sub>2</sub>受容体作動薬：**〇** ブリモニジン（アイファガン®）：アドレナリンα<sub>2</sub>受容体を刺激して、毛様体における房水産生を抑制するとともに、ぶどう膜強膜流出経路からの房水流出を促進する。[1][3]  
2. β遮断薬：**×** チモロール（チモプトール®）：β遮断薬についての記述（非選択的β受容体遮断薬であり、房水産生抑制作用を示す）。[1]  
3. 炭酸脱水酵素阻害薬：**×** ブリンゾラミドに関する記述（毛様体上皮細胞の炭酸脱水酵素を阻害することにより房水産生を抑制する）。選択肢3の炭酸脱水酵素を阻害するような機序の薬物はなく、誤答肢としては適切ではない。[5][7]  
4. Rhoキナーゼ阻害薬：**〇** リパスジル（グラナテック®）：Rhoキナーゼ阻害薬についての記述。[1]  
5. プロスタノイドEP<sub>2</sub>受容体作動薬：**×** 処方2のオミデネパグ（エイベリス®）：プロスタノイドEP<sub>2</sub>受容体作動薬についての記述。[1]

**正解番号**  
**1、4**[1][3][8]

**解説**  
- **処方2**（オミデネパグ：プロスタノイドEP<sub>2</sub>受容体作動薬）を中止し、異なる作用機序の薬物を追加する場合、正しい作用機序は1（ブリモニジン：アドレナリンα<sub>2</sub>受容体作動薬）と4（リパスジル：Rhoキナーゼ阻害薬）。これらは房水産生抑制や流出促進の異なるメカニズムを示す。[1][3]  
- 2は処方3のチモロール（β遮断薬）と重複、3は存在しない機序（炭酸脱水酵素阻害はブリンゾラミドだが不適切）、5は処方2のオミデネパグと同一機序のため誤り。[1][5][7]
