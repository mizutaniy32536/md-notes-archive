# 109-315

### 第109回薬剤師国家試験 問315

**問題文（全文）**  
大学病院で循環器内科を担当している薬剤師が、循環器内科病棟の医師から「糖尿病の治療薬である医薬品A（カプセル剤）が心不全にも効果があるという報告を聞いたので、当院でも効果を検証することを目的として、無作為化二重盲検比較試験を実施したいと考えている。  
＜比較試験の計画（一部抜粋）＞  
対象患者：成人の心不全患者（心機能はNYHA分類Ⅱ度かつ左室駆出率が低下（40%未満）した心不全（HFrEF））  
この研究は、臨床研究法の臨床研究のうち**特定臨床研究**に該当する。該当する理由として正しいのはどれか。**1つ選べ**。[1][3][4]

**全選択肢（番号付き）**  
1. 大学病院で行う研究であるため。[3]  
2. 無作為化二重盲検比較試験であるため。[3]  
3. **医薬品の適応外使用の有効性を検証する研究であるため。**[1][3]  
4. 医師主導で行う研究であるため。[3]

**正解番号**  
**3**[1][3][6]

**解説**  
この研究は、糖尿病治療薬である医薬品Aの**適応外使用（心不全に対する効果）**の有効性を検証するものであるため、臨床研究法における**特定臨床研究**に該当する。特定臨床研究は、既承認医薬品の適応外使用に関する有効性・安全性を評価する研究を指す。[1][3]  
他の選択肢の誤り：  
- 1：実施施設（大学病院）が基準ではなく、研究内容による。[1][3]  
- 2：試験デザイン（無作為化二重盲検）が基準ではない。[1][3]  
- 4：主導者（医師）が基準ではなく、研究の目的・内容による。[1][3]
