# 109-205

# 第109回薬剤師国家試験 問205

## 問題文

イオフルパン（123I）注射液によるドパミントランスポーターシンチグラフィーはSPECTの1つである。この診断法に関連する記述のうち、正しいのはどれか。**2つ選べ**。[1][2]

## 全選択肢

1. 123Iから放出されるγ線を検出する。[1][2][3]

2. SPECTは、放射型のCTの一種である。[1][2][3]

3. 放出される2本の消滅放射線を同時計測する。[1][2][3]

4. イオフルパン（123I）注射液は、病院内のサイクロトロンで製造する必要がある。[1][2][3]

5. ドパミントランスポーターに結合しなかった遊離の123Iを測定する。[3]

## 正解番号

**1、2**[1][2][3]

## 解説

**選択肢1（正）**：123Iは放射性ヨウ素であり、γ線を放出します。イオフルパン（123I）注射液によるドパミントランスポーターシンチグラフィーは、この123Iから放出されるγ線を検出することで画像化する診断法です。[3]

**選択肢2（正）**：SPECTは単一光子放射型コンピュータ断層撮影（Single Photon Emission Computed Tomography）の略称で、放射型のCTの一種です。[1][2]

**選択肢3（誤）**：2本の消滅放射線を同時計測するのはPET（陽電子放射断層撮影）の特徴です。SPECTではγ線を検出します。

**選択肢4（誤）**：123Iは比較的半減期が長い（約13時間）ため、サイクロトロンではなく、ジェネレータから製造される放射性医薬品です。サイクロトロンで製造する必要はありません。

**選択肢5（誤）**：この診断法は、ドパミントランスポーターに**結合した**イオフルパン（123I）を測定することで、ドパミン神経系の機能を評価します。遊離の123Iを測定するのではなく、結合したものを測定対象としています。
