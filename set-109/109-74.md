# 109-74

**問題文**
新有効成分含有医薬品等の市販直後調査の実施は、どの時点から6ヶ月と規定されているか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-74 を参照

**選択肢**
1. 承認申請した時
2. 承認を受けた時
3. 販売を開始した時
4. 薬価収載された時
5. 医薬品リスク管理計画を策定した時

**正解:** 3

**解説**
解答　３

新有効成分含有医薬品等の市販直後調査の実施は、販売を開始した時点から6ヶ月と規定されている。
なお、市販直後調査とは、GVP省令に基づき、医薬品の適正使用のための情報提供と副作用情報の把握のために、製造販売業者が行う調査である。
