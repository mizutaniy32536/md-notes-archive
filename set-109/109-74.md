# 109-74

新有効成分含有医薬品等の市販直後調査の実施は、どの時点から6ヶ月と規定されているか。１つ選べ。

１ 承認申請した時
２ 承認を受けた時
３ 販売を開始した時
４ 薬価収載された時
５ 医薬品リスク管理計画を策定した時

**正解:** ３

**解説**
新有効成分含有医薬品等の市販直後調査の実施は、販売を開始した時点から6ヶ月と規定されている。
なお、市販直後調査とは、GVP省令に基づき、医薬品の適正使用のための情報提供と副作用情報の把握のために、製造販売業者が行う調査である。
