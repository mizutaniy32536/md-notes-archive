# 109-345

**問題文**
47歳男性。体重62 kg。20年以上前から習慣として毎日ビール（1缶500 mL）を3缶飲んでいた。腹部膨満感、嘔吐、四肢の浮腫、左足親指の関節痛を訴えて受診したところ、アルコールの過剰摂取による肝硬変及び痛風と診断され、以下の処方1、2、3により治療中である。治療開始から2週間目に、薬局の薬剤師が服薬状況の確認のために電話をしたところ、患者から「発熱があり、眼が充血して両下肢にじん麻疹ができている」という訴えがあった。薬局の薬剤師は、すぐに主治医への受診を勧め、その後入院加療となった。（入院時所見等）Na 140 mEq/L、K 3.8 mEq/L、総ビリルビン2.1 mg/dL、血清アルブミン2.8 g/dL、PT−INR 2.0、AST 78 IU/L、ALT 66 IU/L、血小板8×104 /µL、γ−GTP 120 IU/L、アンモニア111 µg/dL、BUN 18 mg/dL、血清クレアチニン1.1 mg/dL、eGFR 42.8 mL/min/1.73 m2、尿酸8.1 mg/dL入院精査の結果、アロプリノールによるアレルギー反応と疑われ、アロプリノールの使用は中止となった。入院中、この患者への対応として適切でないのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-345 を参照

**選択肢**
1. スピロノラクトン錠をトルバプタン錠に変更
2. ベンズブロマロン錠の追加
3. フェブキソスタット錠の追加
4. ベタメタゾン点眼液の追加
5. オロパタジン塩酸塩錠の追加

**正解:** 2

**解説**
解答　２

１　適切
トルバプタンは、ループ利尿薬等の他の利尿薬で効果不十分な肝硬変における体液貯留に対して適応があり、他の利尿薬で効果不十分な場合は、腎機能が温存された早期の段階でトルバプタンの投与を開始することが推奨されている。本患者の訴えに浮腫に関する情報はないが、血清アルブミン値は2.8 g/dL（基準値：4.0 g/dL以上）と低値を示しており、浮腫の改善が見られないことが推測される。したがって、フロセミド錠とスピロノラクトン錠では効果不十分と考えられるため、スピロノラクトン錠をトルバプタン錠に変更することは適切である。

２　不適切
アロプリノールの使用が中止となったため、他の痛風治療薬の追加を検討することは適切であるが、ベンズブロマロンは「劇症肝炎」で緊急安全性情報（イエローレター）が発出されており、本患者のような肝硬変患者への投与は禁忌である。

３　適切
アロプリノールの使用が中止となったため、他の痛風治療薬の追加を検討することは適切である。アロプリノールとフェブキソスタットはどちらも尿酸生成阻害薬であるが、アロプリノールはプリン型のキサンチンオキシダーゼ阻害薬であるのに対し、フェブキソスタットは非プリン型のキサンチンオキシダーゼ阻害薬である。よって、アロプリノールと構造式が大きく異なるフェブキソスタットは、アレルギー反応を起こす可能性が低いと考えられるため、本提案は適切である。

４　適切
アロプリノールのアレルギー反応による眼の充血に対して副腎皮質ステロイドであるベタメタゾン点眼液を追加することは適切である。

５　適切
アロプリノールのアレルギー反応による両下肢のじん麻疹に対してヒスタミンH1受容体拮抗薬オロパタジン塩酸塩錠を追加することは適切である。
