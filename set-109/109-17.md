# 109-17

**問題文**
予防接種が勧奨されている子宮頸がんのリスク要因はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-17 を参照

**選択肢**
1. ヒト単純ヘルペスウイルス
2. 梅毒トレポネーマ
3. クラミジア・トラコマチス
4. ヒトパピローマウイルス
5. 淋菌

**正解:** 4

**解説**
解答　４

子宮頸がんは、ヒトパピローマウイルス（HPV）感染により発症することが知られており、HPVは子宮頸がんをはじめ、膣がん、肛門がん、尖圭コンジローマなど様々な疾患の発生に関与する。
