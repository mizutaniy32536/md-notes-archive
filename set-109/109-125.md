# 109-125

**問題文**
予防接種に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-125 を参照

**選択肢**
1. 予防接種で得られる免疫は、人工能動免疫に含まれる。
2. 予防接種によって免疫能を獲得することは、宿主に対する疾病予防である。
3. 予防接種法におけるA類疾病については、接種の努力義務はない。
4. 予防接種法におけるB類疾病の予防接種は、個人予防よりも集団予防を主な目的としている。
5. 予防接種法に定められた予防接種による健康被害は、医薬品副作用被害救済制度により救済される。

**正解:** 1,2

**解説**
解答　１、２

１　正
宿主の獲得免疫には、宿主自身が曝露を受けて獲得した能動免疫と、他の宿主が獲得した能動免疫を免疫グロブリンの形で受けて得る受動免疫に分けられる。さらに能動免疫には、病原体の感染によって成立する自然能動免疫と、ワクチンを用いた予防接種によって成立する人工能動免疫に分けられ、受動免疫には、母体から抗体を受け取る自然受動免疫と、免疫グロブリンの投与による人工受動免疫に分けられる。

２　正
感染には、病気を引き起こす病原体（感染源）、宿主、及び両者を仲介する感染経路の3つの因子が関わり、感染が成立するためにはこの3因子全てが必要である。そのうち、予防接種によって免疫能を獲得することは、宿主に対する疾病予防に該当する。

３　誤
予防接種法におけるA類疾病については、接種の努力義務がある。一方、予防接種法におけるB類疾病については、接種の努力義務はない。

４　誤
予防接種法におけるA類疾病の予防接種は、個人予防よりも集団予防を主な目的としており、B類疾病の予防接種は、集団予防よりも個人予防を主な目的としている。

５　誤
予防接種法に定められた予防接種による健康被害は、予防接種被害救済制度により救済される。なお、予防接種法に基づかない任意接種による健康被害は、医薬品副作用被害救済制度により救済される。
