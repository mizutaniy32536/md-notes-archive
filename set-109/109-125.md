## ◆ 予防接種に関するシナリオ
260308

**問題**
- 予防接種に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**

1. 予防接種で得られる免疫は、人工能動免疫に含まれる。
​
2. 予防接種によって免疫能を獲得することは、宿主に対する疾病予防である。
​
3. 予防接種法におけるA類疾病については、接種の努力義務はない。
​
4. 予防接種法におけるB類疾病の予防接種は、個人予防よりも集団予防を主な目的としている。
​
5. 予防接種法に定められた予防接種による健康被害は、医薬品副作用被害救済制度により救済される。

**正解と理由**
- `1, 2`
- **理由の構成**
	- **問題の整理**：予防接種の免疫学的分類・疾病予防の概念・予防接種法におけるA類/B類の目的と義務・健康被害救済制度の適用先を判断する
	- **薬学的知識・仕様・制約**：ワクチン接種は自己の免疫系を刺激して抗体産生させる`人工能動免疫` — A類疾病は`集団予防（流行阻止）`が主目的で`努力義務あり` — B類疾病は`個人予防（重症化防止）`が主目的で`努力義務なし` — 予防接種法による健康被害は`予防接種健康被害救済制度`が適用（医薬品副作用被害救済制度ではない）
	- **選択理由**：選択肢1は「ワクチン接種＝自己免疫系を刺激して抗体産生＝人工能動免疫」と一致。選択肢2は「免疫獲得→宿主個人の疾病予防」という定義と一致

**解法の手順（思考の流れ）**
- 免疫の分類（人工/自然、能動/受動）とA類/B類の特徴を軸に各選択肢を照合
- 選択肢1：ワクチン接種 = 抗原を投与して自己の免疫系に抗体を産生させる → `人工能動免疫`に分類される → 正しい
- 選択肢2：免疫獲得によって宿主（個人）が疾病に罹りにくくなる → `宿主に対する疾病予防`として正しい
- 選択肢3：A類疾病には`努力義務あり`。努力義務がないのはB類 → 誤り
- 選択肢4：B類は`個人予防（重症化防止）`が主目的。`集団予防（流行阻止）`が主目的なのはA類 → 誤り
- 選択肢5：予防接種法による健康被害は`予防接種健康被害救済制度`が適用。医薬品副作用被害救済制度は対象外 → 誤り

**他の選択肢が誤りな理由**
- 3. A類疾病には努力義務がある
	予防接種法においてA類疾病は`集団予防（流行阻止）`が主目的であり、接種の`努力義務が課されている` — 努力義務がないのはB類疾病（個人予防が主目的）

- 4. B類疾病の主目的は個人予防（集団予防ではない）
	B類疾病（インフルエンザ・高齢者肺炎球菌など）の予防接種は`個人予防（重症化防止）`が主目的 — `集団予防（流行阻止）`が主目的なのはA類疾病（麻疹・風疹・日本脳炎・百日咳・ポリオ・Hib・HPVなど）

- 5. 予防接種法による健康被害は専用制度が適用される
	予防接種法に基づく健康被害には`予防接種健康被害救済制度`（市区町村経由でPMDA審査）が適用される — `医薬品副作用被害救済制度`は通常の医薬品副作用が対象であり、予防接種による被害には適用されない

**キーポイント**
- ワクチン接種 = `人工能動免疫`（自己の免疫系を刺激して抗体産生）
- 予防接種により免疫獲得 = `宿主に対する疾病予防`
- A類疾病：主目的＝`集団予防（流行阻止）`、`努力義務あり`
- B類疾病：主目的＝`個人予防（重症化防止）`、`努力義務なし`
- 予防接種法による健康被害：`予防接種健康被害救済制度`が適用（医薬品副作用被害救済制度ではない）

**関連知識の比較（A類・B類）**
- A類：主目的＝集団予防（流行阻止） — 努力義務あり — 対象：麻疹・風疹・日本脳炎・百日咳・ポリオ・Hib・HPVなど
- B類：主目的＝個人予防（重症化防止） — 努力義務なし — 対象：インフルエンザ（65歳以上）・肺炎球菌（65歳以上）

**関連知識の比較（救済制度）**
- 予防接種健康被害救済制度：予防接種法に基づく健康被害 — 市区町村経由でPMDA審査
- 医薬品副作用被害救済制度：通常の医薬品副作用 — PMDAへ直接申請 — 予防接種は対象外
- 感染等被害救済制度：血液製剤による感染被害 — PMDAへ申請

（一言メモ）
- A類＝`集団予防・努力義務あり`、B類＝`個人予防・努力義務なし`をセットで暗記
- 予防接種の健康被害は`予防接種健康被害救済制度`専用。医薬品副作用被害救済制度と混同しない
