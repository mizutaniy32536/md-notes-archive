# 109-279

### 第109回薬剤師国家試験 問279

#### 問題文
翌日、小児科を再受診し、父親が以下の処方箋（**処方1**及び**2**）を持って薬局を訪れた。

薬剤師が父親と面談したところ、患児は、8ヶ月前にも発熱後に痙攣を起こし、今回と同じ小児科を受診したが、坐剤を2種類処方されたのは初めてとのことであった。

**処方1及び2の坐剤に関する記述のうち、正しいのはどれか。2つ選べ。**[1][3][4][8][10]

#### 全選択肢（番号付き）
1. 処方1の基剤は、マクロゴール400とマクロゴール4000を1：1で混合したものである。[1][3][4][8]  
2. 処方1の基剤は、直腸内で体温により溶融して薬物を放出する。[3][4]  
3. 処方2の基剤は、モノ、ジ、トリグリセリドの混合物である。[1][4]  
4. 処方2の基剤は、直腸内の水分で溶解して薬物を放出する。[1]  
5. いずれの坐剤も薬物の肝初回通過効果を回避できる。[1][3][4][10]

#### 正解番号
**3, 5**[1][3][4]

#### 解説
- **選択肢1（誤）**: 処方1の基剤はマクロゴール（マクロゴール4000とマクロゴール1540の混合）であり、マクロゴール400とマクロゴール4000を1:1で混合したものはマクロゴール軟膏で坐剤基剤として柔らかすぎるため不適切。処方箋に比率は記載されていない。[1][3][4]  
- **選択肢2（誤）**: 処方1（マクロゴール基剤）は直腸内の水分で溶解して薬物を放出する（体温溶融ではない）。[1][3][4]  
- **選択肢3（正）**: 処方2の基剤はハードファットで、モノ・ジ・トリグリセリドの混合物であり、体温で溶融して薬物を放出する。[1][4]  
- **選択肢4（誤）**: 処方2（ハードファット基剤）は体温で溶融し、直腸内の水分で溶解しない。[1][4]  
- **選択肢5（正）**: 坐剤は直腸投与のため、いずれも薬物の肝初回通過効果（first-pass effect）を回避できる。[1][3][4][10]

**処方1**: ジアゼパム坐剤（マクロゴール基剤）。**処方2**: フェノバルビタール坐薬（ハードファット基剤）。[1][3][4]
