# 109-178

### 第109回薬剤師国家試験 問178

**問題文（全文）**  
イリノテカンは酵素（ア）によって活性代謝物（イ）に変換され、さらに酵素（ウ）によって代謝物（エ）に変換され不活化される。（ア）の酵素及び（イ）の構造式、（ウ）の酵素及び（エ）の構造式のそれぞれの組合せとして、正しいのはどれか。**1つ選べ。**[1][3]

**全選択肢（番号付き）**  
|選択肢|（ア）の酵素|(イ)の構造式|(ウ）の酵素|(エ）の構造式|  
|------|-------------|-------------|-------------|-------------|  
|1|CYP3A4|A|カルボキシエステラーゼ|C|  
|2|硫酸転移酵素|A|UDP-グルクロン酸転移酵素|B|  
|3|**カルボキシエステラーゼ**|**A**|**UDP-グルクロン酸転移酵素**|**B**|  
|4|CYP3A4|B|硫酸転移酵素|C|  
|5|UDP-グルクロン酸転移酵素|B|カルボキシエステラーゼ|A|[1][2][3][6]

**正解番号**  
**3**[2][4][6]

**解説**  
イリノテカンはプロドラッグであり、**カルボキシエステラーゼ**（ア）によって活性代謝物**SN-38（構造式A）**に変換され、抗腫瘍作用を示す。その後、**UDP-グルクロン酸転移酵素（UGT1A1）**（ウ）によって不活性代謝物**SN-38G（構造式B）**に変換され、排泄される。他の選択肢は酵素や構造式の対応が誤り（例: CYP3A4はイリノテカンの不活性化に関与するが活性化酵素ではない）。[2][3][4][6]
