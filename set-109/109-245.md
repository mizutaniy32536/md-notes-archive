# 109-245

### 第109回薬剤師国家試験 問245

**問題文（全文）**  
夫婦と小学生2人の4人家族。冬休みに、スキー場近くのキャンプ場で大型テントを張り、その中で炭火を使って鉄板でバーベキューをしていた。また、テントの入り口付近では、暖房器具の電源として携帯型のガソリンエンジン発電機を使用していた。しばらくして、通りがかった隣のテントの男性が、昏睡状態で倒れているこの家族を発見した。  

前問の検査で診断が確定した。この中毒の原因物質に関する記述として、正しいのはどれか。**2つ選べ。**[1][2][3][6][9]

**全選択肢（番号付き）**  
1. 独特の腐敗臭を有する気体で、窒息性の呼吸器障害を引き起こす。[1][2][3][4][6]  
2. 無色で、呼吸器に対する刺激性を有する気体である。[1][2][3][4]  
3. 無色無臭で、空気よりもわずかに軽い可燃性の気体である。[1][2][3][4]  
4. ヘモグロビンに対する結合親和力が酸素の約200倍である。[1][2][4]  
5. ヘモグロビンの二価の鉄を酸化し、酸素への結合能力を消失させる。[1][2][3][4][5]

**正解番号**  
**3, 4**[1][2][3]

**解説**  
この問題は一酸化炭素（CO）中毒の原因物質（CO）の性質を問う。問244でカルボキシヘモグロビン（CO-Hb）検査によりCO中毒が確定している。[1][2][6][9]  

- **1. 誤り**：COは無味無臭の気体であり、「独特の腐敗臭」はない。窒息性だが呼吸器障害の刺激性はない。[1][2]  
- **2. 誤り**：COは無色無臭・無刺激性であり、呼吸器に対する刺激性はない。[1][2]  
- **3. 正しい**：COは無色無臭、空気の平均分子量（約28.8）より分子量28でわずかに軽く、可燃性である。[1][2][3]  
- **4. 正しい**：COはヘモグロビン（Hb）に対する結合親和力が酸素（O₂）の約200倍以上であり、CO-Hb形成により酸素運搬能が低下する。[1][2][4]  
- **5. 誤り**：COはHbの二価鉄（Fe²⁺）を酸化せず、競合結合により酸素結合を阻害する。Fe²⁺を三価（Fe³⁺）に酸化するのは一酸化窒素（NO）などでメトヘモグロビン形成である。[1][2][5]  

正解はCOの物理化学的性質と毒性機序を正確に記述した3と4。[1][2][3][4]
