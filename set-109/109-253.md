# 109-253

### 第109回薬剤師国家試験 問253

**問題文（共通の症例文を含む）**  
73歳女性。左前頭部が柔らかく腫張しているのを自覚し、かかりつけ医を受診した。腫瘤性病変を指摘され、精査加療目的で紹介入院となった。CT検査で頭蓋及び四肢に骨病変が認められた。（多発性骨髄腫の症例）。治療開始後、血清カルシウム値が12 mg/dLを超えたため、薬物を追加することとなった。追加する薬物の作用機序として、適切なのはどれか。**2つ選べ。**[1][2][4][5]

**全選択肢**  
1. 骨芽細胞の副甲状腺ホルモン受容体を遮断する。  
2. 副甲状腺細胞のカルシウム受容体を遮断する。  
3. **破骨細胞のファルネシルピロリン酸合成酵素を阻害する。**  
4. **RANKLを阻害して骨吸収を抑制する。**  
5. ビタミンDの活性型を阻害する。[1][2][4][5]

**正解番号**  
**3、4**[1][2][5]

**解説**  
この問題は、多発性骨髄腫に伴う高カルシウム血症に対する治療薬（ゾレドロン酸とデノスマブ）の作用機序を問う。  
- 選択肢3：ゾレドロン酸（ビスフォスフォネート系）は、破骨細胞のファルネシルピロリン酸合成酵素（FPPS）を阻害し、骨吸収を抑制する。[1][2][5][8]  
- 選択肢4：デノスマブはRANKL（RANKリガンド）を阻害し、破骨細胞の分化・活性化を抑えて骨吸収を抑制する。[1][5][8]  
- 誤り選択肢：  
  1. 骨芽細胞のPTH受容体遮断は該当せず。  
  2. 副甲状腺細胞のCa受容体遮断はシナカルセト（二次性副甲状腺機能亢進症用）。  
  5. ビタミンD活性型阻害は関連なし。[2][5]  
高Ca血症治療として、これら2剤の機序が正しい。[1][2][4][5][8]
