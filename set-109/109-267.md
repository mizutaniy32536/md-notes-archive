# 109-267

### 第109回薬剤師国家試験 問267

**問題文（全文）**  
39歳、閉経前女性。身長155cm、体重43kg、体表面積1.38m2。右乳房に腫瘤を触知し、乳腺外科を受診、針生検の結果、T1N1M0のStageⅡA、**エストロゲン受容体（ER）陽性**、**プロゲステロン受容体（PgR）陽性**、**HER2陰性**の浸潤性乳管がん（乳がん）と診断された。術後補助療法として以下の薬物療法が施行された。（処方1および処方2の詳細あり。処方2はリュープロレリン酢酸塩注射用キット3.75mg）。[1][2][3][5]

**処方2の製剤に関する記述として、正しいのはどれか。2つ選べ。**  
1. 乳酸・グリコール酸共重合体が使用されている。  
2. 高分子で薬物結晶をコーティングしている。  
3. マトリックスを形成している高分子が分解することにより薬物を放出する。  
4. 自己乳化型マイクロエマルション製剤である。  
5. 粉末部の薬剤を液体部の溶液で溶解して使用する製剤である。[1][2][3][5][6]

**正解番号：1と3**[1][2][3][5]

**解説**  
- **1. ○** 処方2の製剤（リュープロレリン酢酸塩注射用キット3.75mg）は、乳酸・グリコール酸共重合体（生体分解性高分子）のマイクロカプセルの中にリュープロレリンを封入した持続性注射剤である。[1][2][3][5]  
- **2. ×** 高分子で薬物結晶をコーティングしているわけではなく、マイクロカプセル内に薬物を分散・封入したマトリックス型である。[1][5]  
- **3. ○** 皮下投与後、マトリックスを形成する乳酸・グリコール酸共重合体が高分子が徐々に分解され、リュープロレリンが持続的に放出される。[1][2][3][5]  
- **4. ×** 自己乳化型マイクロエマルション製剤ではない。[1][2]  
- **5. ×** 粉末部と液体部を溶解して使用する製剤ではない（マイクロカプセル形態）。[5]
