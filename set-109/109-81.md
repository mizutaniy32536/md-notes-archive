# 109-81

**問題文**
一般用医薬品の薬効分類と成分の組合せで正しいのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-81 を参照

**選択肢**
（選択肢は画像を参照）

**正解:** 1

**解説**
解答　１

１　正
クロタミトンは、鎮痒薬として一般用医薬品に含まれる成分である。

２　誤
クロルフェニラミンマレイン酸塩は、抗ヒスタミン薬主薬製剤として一般用医薬品に含まれる成分である。なお、催眠鎮静薬として一般用医薬品に含まれる成分には、ジフェンヒドラミンなどがある。

３　誤
ペミロラストカリウムは、アレルギー用点眼薬として一般用医薬品に含まれる成分である。なお、鼻炎用点鼻薬として一般用医薬品に含まれる成分には、ナファゾリンなどがある。

４　誤
テルビナフィン塩酸塩は、みずむし・たむし用薬として一般用医薬品に含まれる成分である。

５　誤
ビサコジルは、瀉下薬（下剤）として一般用医薬品に含まれる成分である。なお、含嗽薬として一般用医薬品に含まれる成分には、ポビドンヨードなどがある。
