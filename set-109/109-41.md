## ◆ Fickの法則における透過速度に影響するパラメータを問うシナリオ

260308

**問題**

**正解と理由**
- `3. 膜の厚さ（L）`
- **理由の構成**
	- **問題の整理**：Fickの法則の式において、透過速度と「反比例」する因子を選ぶ問題
	- **薬学的知識・仕様・制約**：Fickの第一法則による膜透過速度は J = D × S × Km ×（C₁ − C₂）/ L で表される。ここで D = 拡散係数、S = 有効表面積、Km = 分配係数、L = `膜の厚さ`。Lのみが分母にあるため、透過速度と`反比例`する
	- **選択理由**：式の分母に位置するパラメータはLのみであり、他はすべて分子（比例関係）にあるため

**解法の手順（思考の流れ）**
- まず`Fickの法則`の基本式 J = D × S × Km × ΔC / L を想起
- 「反比例」= 分母に位置するパラメータを探す
- 分子の因子: D（拡散係数）、S（表面積）、Km（分配係数）、ΔC（濃度差）→ すべて比例
- 分母の因子: L（膜の厚さ）→ 唯一の反比例パラメータ
- よって`膜の厚さ`が正解と即座に結論づけた

**他の選択肢が誤りな理由**
- 1. 膜の有効表面積
	Fickの式の分子に位置するため、透過速度と`正比例`する。表面積が広いほど透過速度は大きくなる

- 2. 薬物の膜中の拡散係数
	Fickの式の分子に位置するため、`正比例`。拡散係数Dは温度や薬物の分子量に依存し、分子量が大きいほど小さくなる

- 4. 薬物の膜への分配係数（C₁/Cout）
	Fickの式の分子に位置するため、`正比例`。`脂溶性`が高い薬物ほど分配係数が大きく、膜透過しやすい

- 5. 細胞外及び細胞内溶液と接する面の膜中薬物濃度の差（C₁ - C₂）
	Fickの式の分子に位置する濃度勾配であり、`正比例`。濃度差が大きいほど透過の駆動力が大きくなる

**キーポイント**
- `Fickの第一法則`: J = D × S × Km × ΔC / L を確実に暗記する
- 式中で唯一分母にあるのは`L`（膜の厚さ）のみ → 反比例
- 他の全因子（D, S, Km, ΔC）は分子 → 比例

**関連知識の比較**
- `単純拡散`はFickの法則に従い、濃度勾配が駆動力となる。`受動輸送`に分類される
- `促進拡散`はトランスポーター介在だがエネルギー不要で、`飽和性`を示す点がFick型と異なる
- `Michaelis-Menten式`は飽和性のある担体輸送に適用される。v = Vmax × C / (Km + C)
- `膜の厚さ`は通常7〜10 nmであり、生体膜としては比較的均一だが、UWL（非撹拌水層）を含めると実効的な厚さは増大する

（一言メモ）
- Fickの式は「分母にL、分子にそれ以外」と覚えれば、比例・反比例の問題は即答できる
- `拡散係数D`は Stokes-Einstein式で D = kT / 6πηr と表される。`分子量`が大きい（半径rが大きい）ほどDは小さくなる
- 生体膜透過では`脂溶性`（分配係数Km）と`分子量`（拡散係数D）が2大決定因子
