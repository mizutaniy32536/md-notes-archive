# 109-41

**問題文**
下図に示すFickの法則に従う細胞膜透過において、薬物の透過速度と反比例するのはどれか。1つ選べ。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-41 を参照

**選択肢**
1. 膜の有効表面積
2. 薬物の膜中の拡散係数
3. 膜の厚さ（L）
4. 薬物の膜への分配係数（C1/Cout）
5. 細胞外及び細胞内溶液と接する面の膜中薬物濃度の差（C1－C2）

**正解:** 3

**解説**
解答　３

Fickの第一法則に従う薬物の膜透過速度Vは①式で表される。

①式より、Fickの第一法則に従う薬物の透過速度と反比例するのは、膜の厚さ（L）である。なお、Fickの第一法則に従う薬物の透過速度と比例するのは、薬物の膜中の拡散係数（D）、薬物の膜への分配係数（K）、膜の有効表面積（S）、細胞外及び細胞内溶液と接する面の膜中薬物濃度の差（C1－C2）である。
