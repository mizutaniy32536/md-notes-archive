# 109-174

薬物代謝酵素の遺伝子多型に関する記述のうち、正しいのはどれか。２つ選べ。

１ CYP2C19の遺伝子多型によってオメプラゾールの血中濃度時間曲線下面積（AUC）は変化するが、代謝物のAUCは変化しない。
２ CYP2D6の遺伝子多型が関与するイミプラミンのpoor metabolizer（PM）では、活性代謝物の血中濃度が高い。
３ アザチオプリンを使用する前には、UDP−グルクロン酸転移酵素（UGT1A1）の遺伝子多型の診断が行われている。
４ N
５ アルデヒド脱水素酵素（ALDH2）はPMの頻度に人種差があり、白人と比べて日本人ではPMの出現率が高い。

**正解:** ２、５

**解説**
１　誤
オメプラゾールは、主にCYP2C19及び一部CYP3A4により代謝される。CYP2C19には遺伝子多型が存在し、CYP2C19の遺伝子多型によって、オメプラゾールの血中濃度時間曲線下面積（AUC）は変化し、代謝物のAUCも変化する。
２　正
イミプラミンはCYP2D6により不活性化され、その他のCYP1A2、CYP2C19やCYP3A4などの分子種により活性代謝物であるデシプラミンに代謝される。そのため、CYP2D6の遺伝子多型が関与するイミプラミンのpoor metabolizer（PM）では、活性代謝物の血中濃度が高くなる。
３　誤
アザチオプリンを使用する前には、NUDT15（Nudix Hydrolase 15）の遺伝子多型の診断が行われている。NUDT15は、アザチオプリンの代謝酵素の一つであり、その遺伝子多型とアザチオプリンの重篤な副作用である白血球減少等の発現率には、関連性が認められている。なお、使用前にUDP−グルクロン酸転移酵素（UGT1A1）の遺伝子多型の診断を行う薬物には、イリノテカンなどがある。
４　誤
N−アセチル転移酵素（NAT2）には遺伝子多型が存在し、日本人では約10%がイソニアジドのアセチル化反応速度が遅い群（PM）に属する。
５　正
アルデヒド脱水素酵素（ALDH2）はPMの頻度に人種差があり、白人と比べて日本人ではPMの出現率が高く、PMの割合が約40％であるのに対して、白人ではほとんどPMが見られない。
