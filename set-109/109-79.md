# 109-79

**問題文**
介護保険の保険者に該当するのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-79 を参照

**選択肢**
1. 国民健康保険組合
2. 健康保険組合
3. 地方厚生局
4. 都道府県
5. 市町村及び特別区

**正解:** 5

**解説**
解答　５

介護保険の保険者に該当するのは、市町村及び特別区である。
