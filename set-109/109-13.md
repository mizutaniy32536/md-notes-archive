# 109-13

核内受容体のリガンドの前駆体となるビタミンはどれか。１つ選べ。

１ ナイアシン
２ ピリドキシン
３ パントテン酸
４ レチノール
５ ビオチン

**正解:** ４

**解説**
リガンドとは、生体分子と複合体を形成して生物学的な目的を果たす物質をいう。核内受容体にリガンドとして結合することで核内受容体を活性化させるものには、脂溶性ビタミン、脂肪酸などの代謝物やステロイドホルモンなどがある。
選択肢のうち、脂溶性ビタミンに該当するのは、レチノール（ビタミンA）である。
