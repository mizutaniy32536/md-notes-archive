# 109-91

分子間相互作用に関する記述のうち、正しいのはどれか。２つ選べ。

１ クーロン力は電荷間距離の2乗に反比例する。
２ 分散力は分子間にはたらく反発力である。
３ 水中における界面活性剤のミセル形成はイオン結合による。
４ 疎水性相互作用は水溶液中のタンパク質の高次構造の形成及び安定化に寄与している。
５ 核酸塩基対は配位結合により形成される。

**正解:** １、４

**解説**
１　正
クーロン力Fは下式で示される。このためクーロン力は電荷間距離（r）の2乗に反比例する。
２　誤
分散力は、分子間に働く引力である。
３　誤
水中における界面活性剤のミセル形成は疎水性相互作用によるものである。疎水性相互作用は疎水性分子表面から水を追い出して疎水性分子が集まろうとする力が駆動力となっている。
４　正
疎水性相互作用は水溶液中のタンパク質の高次構造の形成及び安定化に寄与している。例えばタンパク質中の疎水性アミノ酸残基は、疎水的相互作用により疎水的な側鎖をタンパク質の内部に向かって押し込むことで安定化する。
５　誤
核酸塩基対は水素結合により形成される。DNAの二重らせん構造において2本の鎖をつなぐ力であり、アデニン−チミン間では2箇所、グアニン−シトシン間では3箇所で水素結合が形成される。
