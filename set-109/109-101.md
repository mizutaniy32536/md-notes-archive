# 109-101

以下のアンモニウムイオンのうち、pKa値が最小なのはどれか。１つ選べ。

**正解:** ５

**解説**
アンモニウムイオンのpKa値は置換基の性質によって影響を受ける。選択肢１と比較して−OCH3、−CH3などの電子供与性基は酸性を弱めるためpKa値は大きくなる。すなわち選択肢３、４は解答としては不適切である。
更に、−Cl、−NO2などの電子求引性基は酸性を強めるためpKa値は小さくなる（選択肢２、５）。電子求引性基の強さを比較すると−Cl＜−NO2であり、強い電子求引性基がついているほどpKaは小さくなるため、選択肢５のpKa値が最小である。
