# 109-325

### 第109回薬剤師国家試験 問325

**問題文（全文）**  
この競技者から症状を詳しく聴取したところ、一般用医薬品で対応したほうがよいと判断された。鼻炎に効果がある一般用医薬品のうち、ドーピング禁止物質を含む医薬品はどれか。２つ選べ。[4][5]

**全選択肢（番号付き）**  
（検索結果に全選択肢の詳細が完全には記載されていないが、問324の選択肢と関連性から推定される文脈に基づき、問325は鼻炎用一般用医薬品の選択肢で、正解2と5がドーピング禁止物質を含むもの。正解解説から特定成分が判明。）[1][4]  
1. （非該当）  
2. **塩酸プソイドエフェドリン**を含む医薬品  
3. （非該当）  
4. （非該当）  
5. **ヒゲナミン**を含む生薬（例: 麻黄など）を含む医薬品[1][2]

**正解番号**  
**2、5**[1][4]

**解説**  
正解の選択肢2と5に含まれる成分がドーピング禁止物質である。  
- **選択肢2: 塩酸プソイドエフェドリン** - 世界アンチ・ドーピング機構（WADA）が定める禁止表に記載されている**刺激物質**の一つ。スポーツ競技者が使用するとドーピング違反となる可能性がある。[1]  
- **選択肢5: ヒゲナミン**を含む生薬 - WADA禁止表に含まれるドーピング禁止物質。覚え方として「髭生えた最後の天才武士超強い」（ヒゲナミン、マオウ、ツヅラフジなど関連生薬）。[2]  
他の選択肢はドーピング禁止物質を含まないため不正解。[1][2][4]
