# 109-176

TDMの実施が望ましい薬物の性質として、正しいのはどれか。２つ選べ。

１ 有効血中濃度の範囲が広い。
２ 体内動態の個人差が小さい。
３ 薬物相互作用を受ける可能性が低い。
４ 体内動態に非線形性が認められる。
５ 副作用と対象疾患の症状の区別が難しい。

**正解:** ４、５

**解説**
TDMの実施が望ましい薬物の性質としては、有効血中濃度の範囲が狭い（安全域が狭い）、体内動態の個人差が大きい、薬物相互作用を受ける可能性が高い、体内動態に非線形性が認められる、副作用と対象疾患の症状の区別が難しいなどのほか、重篤な副作用を示す、繰り返し投与により体内動態に変化が予想される、などがある。
