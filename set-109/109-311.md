# 109-311

（法規・制度・倫理） この女性（75歳）の夫は、自身が利用しているかかりつけ薬局を訪れた。夫は薬剤師に、 妻の入院や妻宛てに郵送された医療保険証のことなどを相談した。その医療保険証 の医療保険制度（後期高齢者医療制度）に関する薬剤師の説明内容として適切なのはどれか。2つ選べ。
1 所得により医療費の窓口負担割合が異なります。
2 運営主体は全国健康保険協会です。
3 保険料の支払いは免除されます。
4 ₆₅ 歳以上 ₇₅ 歳未満の方は一定の障害と認定された場合、加入することができ
ます。
5 当該制度の加入者は高額療養費制度を利用できません。

## 正解
**1 と 4**

## 解説

### 後期高齢者医療制度の基本

**対象者：**
- 原則：**75歳以上**
- 例外：**65歳以上75歳未満で一定の障害がある者**

**運営主体：**
- **後期高齢者医療広域連合**（都道府県単位の組織）

**財源：**
- 患者負担（窓口負担）
- 公費（国・都道府県・市町村）
- 現役世代からの支援金
- **高齢者の保険料**

### 各選択肢の検証

#### 1. ⭕ 所得により窓口負担割合が異なる

**負担割合（2022年10月以降）：**
| 区分 | 窓口負担割合 |
|---|---|
| 一般所得者 | **1割** |
| 一定以上の所得 | **2割** |
| 現役並み所得者 | **3割** |

#### 2. ❌ 運営主体は後期高齢者医療広域連合

**医療保険の運営主体：**
| 制度 | 運営主体 |
|---|---|
| 後期高齢者医療制度 | **後期高齢者医療広域連合**（都道府県単位） |
| 健康保険（協会けんぽ） | 全国健康保険協会 |
| 健康保険（組合健保） | 健康保険組合 |
| 国民健康保険 | 市町村・都道府県 |

#### 3. ❌ 保険料の支払いは必要

**保険料：**
- 所得に応じて保険料を負担
- 原則、年金から天引き（特別徴収）
- 年金額が少ない場合は納付書払い（普通徴収）
- **免除されない**

#### 4. ⭕ 65歳以上75歳未満で一定の障害があれば加入可能

**一定の障害（政令で定める程度）：**
- 障害年金1級・2級の受給者
- 身体障害者手帳1〜3級および4級の一部
- 精神障害者保健福祉手帳1級・2級
- 療育手帳A（重度）

→ **広域連合の認定**を受ければ加入できる（任意）

#### 5. ❌ 高額療養費制度を利用できる

**後期高齢者医療制度でも高額療養費制度は利用可能**
- 自己負担限度額は所得区分に応じて設定
- 通常の医療保険と同様に利用できる

### 後期高齢者医療制度まとめ

| 項目 | 内容 |
|---|---|
| **対象** | 75歳以上（65〜74歳で一定の障害も可） |
| **運営主体** | 後期高齢者医療広域連合 |
| **窓口負担** | 1割・2割・3割（所得による） |
| **保険料** | 所得に応じて負担（免除なし） |
| **高額療養費** | 利用可能 |
