# 109-179

**問題文**
2種類の粉体A及びBの粉体層について、図1に示すように、種々の垂直応力（σ）を加えた状態で可動セルを水平方向に引っ張ることでせん断試験を行った。得られたせん断応力（τ）の値をσに対してプロットした結果を図2に示す。以下の記述のうち、正しいのはどれか。2つ選べ。ただし、τとσの間にはクーロンの式が成立するものとする。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-179 を参照

**選択肢**
1. 粉体Aは粉体Bよりも安息角が大きい。
2. 粉体Aを造粒するとグラフの傾きが大きくなる。
3. 粉体Bは粉体Aよりも付着力が強い。
4. 粉体Bに滑沢剤を適量添加すると内部摩擦係数が大きくなる。
5. 粉体Bは粉体Aよりもオリフィスからの流出速度が大きい。

**正解:** 1,5

**解説**
解答　１、５

クーロンの式と図2より、傾きが内部摩擦係数（μ）、縦軸切片が付着力（C）の一次関数のグラフであることが分かる。
一般に粉体は、内部摩擦係数が小さく、付着力が小さいほど、流動性が良いため、粉体Aと粉体Bのグラフを比較すると、流動性が良い粉体は粉体Bといえる。

１　正
安息角とは、粉体を重力により自由落下させることで山形に層を形成させた際、粉体層表面が水平面となす角度を示す。一般に、安息角が小さいほど流動性が良いため、粉体Aは粉体Bよりも安息角が大きい。

２　誤
粉体は、造粒すると粒子径が大きくなるため、流動性は良くなる。流動性が良くなると、内部摩擦係数は小さくなるため、図2の粉体Aの傾きは小さくなる。

３　誤
図2の縦軸切片より、粉体Bは粉体Aよりも付着力が弱い。

４　誤
粉体Bに滑沢剤を適量添加すると、流動性が良くなるため、内部摩擦係数は小さくなる。

５　正
オリフィスからの流出速度とは、円筒形容器の底の中心部に設けたオリフィス（小孔）から粉体が排出する速度のことを示す。一般に、オリフィスからの流出速度が大きいほど流動性が良いため、粉体Bは粉体Aよりもオリフィスからの流出速度が大きい。
