# 109-207

# 第109回薬剤師国家試験 問207

## 問題文

ドパミンは、FAD（フラビンアデニンジヌクレオチド）を補酵素としてモノアミン酸化酵素B（MAO-B）により代謝される。この代謝反応は、以下の図に示す機構で進行すると考えられている。ただし、図中のFADの化学構造はRとして一部省略している。

以下の記述のうち、正しいのはどれか。**2つ選べ。**[1][2]

## 選択肢

| 番号 | 選択肢 |
|------|--------|
| 1 | 反応①において、FADは還元剤として働く。 |
| 2 | 反応②は、置換反応である。 |
| 3 | 反応③において、カテコール骨格をもつイミンが生成する。 |
| 4 | カテコール骨格をもつ第一級アミンが最終産物となる。 |
| 5 | セレギリンはMAO-Bに酸化された後、補酵素FADと共有結合を形成する。 |

## 正解番号

**3、5**[1][3]

## 解説

**選択肢1（誤り）**
FADは水素を受け取って還元されており、ドパミンから水素を奪って酸化しています。つまり、FADは**酸化剤**として働いています。還元剤ではありません。[1][3]

**選択肢2（誤り）**
反応②は、FADが還元される酸化還元反応であり、置換反応ではありません。[3]

**選択肢3（正解）**
反応③において、カテコール骨格をもつイミンが生成します。[1][3]

**選択肢4（誤り）**
最終産物は、カテコール骨格をもつ第一級アミンではなく、**カテコール骨格を持つアルデヒドとアンモニア**です。[1]

**選択肢5（正解）**
セレギリンはMAO-Bに酸化された後、補酵素FADと共有結合を形成します。これはセレギリンが可逆的阻害薬ではなく、不可逆的阻害薬として作用することを示しています。[1][3]
