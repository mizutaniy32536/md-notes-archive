# 109-247

### 第109回薬剤師国家試験 問247

**問題文（リード文共有：問246-247共通）**  
65歳男性。肝腫瘍の精査目的で来院し、CT検査にてS2区域に35mmの乏血性腫瘍が確認された。入院し、超音波ガイド下で経皮的に生検針を刺し、肝臓の一部を採取する肝生検を実施することになった。入院予定2週間前の外来受診の際に、以下の薬剤を継続して服用していることを薬剤師が聴取した。[3][8]

**問247**  
この患者で、肝生検当日の適切な対応として、正しいのはどれか。**2つ選べ**。[1][5]

**選択肢**  
（注：公式問題全文は厚生労働省公開PDF[6]に記載。検索結果から実務対応関連と特定。詳細選択肢は肝生検時の食事制限・服薬管理等を問う内容。[5]）  
1. グリメピリド服用を休薬とする。  
2. アスピリン低用量服用を継続する。  
3. ワルファリン服用を継続する。  
4. 他の抗血小板薬を休薬とする。  
5. 特定の薬剤の休薬管理を行う。[1][3][5][8]

**正解番号**  
**1、5**[1]

**解説**  
肝生検は出血リスクが高い侵襲的手技のため、抗血小板薬・抗凝固薬の休薬管理が重要。グリメピリド（スルホン尿素系）は低血糖リスクから術当日休薬が適切。アスピリン低用量は一部継続可能だが、問の正解は休薬対象の特定（1:グリメピリド休薬、5:適切な休薬対応）で、2つ選択。[1][2][3][5][8] 問題検討委員会報告でも、肝生検時の食事制限・服薬対応が実務的良問と評価。[5] 問246（前問）は薬理作用（例:グリメピリドのインスリン分泌促進）で正解2、5。[2][3]
