# 109-67

我が国において、肝硬変の原因となるウイルス性肝炎として最も頻度が高いのはどれか。１つ選べ。

１ A型肝炎
２ B型肝炎
３ C型肝炎
４ D型肝炎
５ E型肝炎

**正解:** ３

**解説**
C型肝炎のほとんどは慢性肝炎へと移行し、慢性化率は他のウイルス性肝炎に比べて最も高く、肝硬変への進展率も高い。
