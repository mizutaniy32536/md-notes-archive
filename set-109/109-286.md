# 109-286

**問題文**
86歳男性。76歳時に妻と死別し独居中であるが、近所に住む娘が介護にあたってきた。死別5年後の81歳の頃から、徐々に物忘れが出現し、時々つじつまが合わない発言があったが放置していた。84歳頃より、物忘れがひどくなり、一人になると不安感が強くなった。娘の姿が見えないと、すぐに名前を呼び、片時も離れられない状況になったため、物忘れ外来を受診した。来院時、新しいことが覚えられず、取り繕うような話し方であった。尿失禁や歩行障害はなし。長谷川式簡易知能評価（HDS−R）は30点満点中18点であった。頭部CTで海馬の萎縮を指摘されたが、梗塞巣所見はなく、血液検査も異常はなかった。また、この男性は不整脈に対して服薬しており、骨粗しょう症の治療のため3年前から昨年までの24ヶ月間テリパラチド皮下注キットによる治療が実施された。現在の処方は以下のとおりである。＜image1:center＞問286（病態・薬物治療）この患者の経過及び処方から考えられる疾患はどれか。2つ選べ。

**正解:** 1,3

**解説**
問286　解答　１、３

本患者は、

・86歳の高齢である
・物忘れがひどくなり新しいことが覚えられない
・長谷川式簡易知能評価（HDS−R）が30点満点中18点（20点以下は認知症の疑いあり）
・頭部CTで海馬の萎縮を指摘されたが、梗塞巣所見はなく、血液検査も異常はない
・ドネペジル塩酸塩が処方されている

以上のことからアルツハイマー型認知症であると推測される。
また、

・86歳の高齢である
・不整脈に対して服薬している事に併せて、ワルファリンカリウム錠が処方されている

以上のことから心房細動であることも推測される。

問287　解答　３、５

本患者はベッド上での生活状態となっており、自分一人の状況ではほぼ薬を服用できないことから、内服薬よりも、投与回数の少ない注射薬での治療を行うのが望ましい。

１　誤
メナテトレノンカプセルは内服薬であるため、追加する治療薬として適切ではない。
また、メナテトレノンはビタミンK2製剤であり、本患者はビタミンKと拮抗作用を有するワルファリンカリウム錠を服用中であるという点からも、追加する治療薬として適切ではない。

２　誤
ラロキシフェン塩酸塩錠は内服薬であるため、追加する治療薬として適切ではない。
また、ラロキシフェンは選択的エストロゲン受容体モジュレーター（SERM）であり、閉経後骨粗しょう症に用いるが、本患者は男性である点からも、追加する治療薬として適切ではない。さらに、ラロキシフェンは、静脈血栓塞栓症のリスクを上昇させるため、長期不動状態にある患者には投与禁忌である点からも、追加する治療薬として適切ではない。

３　正
デノスマブ（遺伝子組換え）皮下注は抗RANKLモノクローナル抗体であり、6ヶ月に1回の皮下注射で骨粗しょう症の治療に用いられる。そのため、本患者に追加する治療薬として適切である。

４　誤
テリパラチド（遺伝子組換え）皮下注はヒト副甲状腺ホルモンのN末端フラグメントであり、皮下注射で骨粗しょう症の治療に用いられるが、その投与は24ヶ月を超えないこととされている。本患者は3年前から昨年までの24ヶ月間テリパラチド皮下注キットによる治療を行なっているため、追加する治療薬として適切ではない。

５　正
ゾレドロン酸水和物注射液はビスホスホネート製剤であり、年に1回の点滴静脈注射で骨粗しょう症の治療に用いられる。そのため、本患者に追加する治療薬として適切である。
