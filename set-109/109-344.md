# 109-344

48歳男性。身長174 cm、体重52 kg。慢性骨髄性白血病、高血圧、消化性潰瘍並びに脂質異常症と診断され、これまで処方1及び処方2にて治療していた。今回、イマチニブ錠にて効果不十分であったため、ニロチニブ塩酸塩水和物カプセルに変更になり、処方3〜5の処方箋を持参し来局した。 （検査値） 血圧125/72 mmHg、赤血球410×104 /µL、白血球4,600 /µL、 総コレステロール202 mg/dL、HDL−C 55 mg/dL、LDL−C 90 mg/dL、 TG 105 mg/dL 薬局薬剤師の対応として適切なのはどれか。２つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC を参照

**正解:** ３、５

**解説**
１　誤
ニロチニブとテルミサルタンの相互作用は報告されていないため、変更の提案は不適切である。
２　誤
ニロチニブとプラバスタチンの相互作用は報告されていないため、変更の提案は不適切である。
３　正
ニロチニブは食事の影響を受けやすいため、食事の1時間以上前か食後2時間以降に1日2回経口投与する必要がある。したがって、ニロチニブ塩酸塩水和物カプセルの用法を食後2時間以降に変更を提案することは適切である。
４　誤
ニロチニブはファモチジンやエソメプラゾールなどの胃内pHを上昇させる薬剤と併用すると、溶解度が低下して吸収が低下するため、ファモチジン錠から、エソメプラゾールカプセルへの変更を提案することは不適切である。なお、ニロチニブとファモチジンの併用については、服用時間をずらすことで本剤のCmax及びAUCに影響がなかったという報告があるため、ファモチジンの用法は就寝前のまま変更なしで問題ないと考えられる。
５　正
ニロチニブはQT間隔延長があらわれることがあるため、本剤投与開始前と本剤投与中は適宜心電図検査を行う必要がある。したがって、心電図検査の実施の有無を確認することは適切である。
