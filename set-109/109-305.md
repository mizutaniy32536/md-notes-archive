# 109-305

# 第109回薬剤師国家試験 問305

## 問題文

退院後の治療経過観察において、薬剤師が留意すべき事項として正しいのはどれか。**2つ選べ。**[2][4]

## 全選択肢

1. 検査結果と処方内容の確認後の調剤は、クロザピン専任登録管理薬剤師が担当する。[2][4]

2. 心理教育や作業療法プログラムを立案する。[2][4]

3. 口渇、多飲、頻尿症状が発現した場合には直ちに受診するように指導する。[2][4]

4. クロザピンに残薬が生じた場合は、速やかに自己廃棄するよう指導する。[2][4]

5. 体重減少をきたすことがあるので、セルフモニタリングできるよう指導する。[1][2]

## 正解番号

**1、3、5**[2][3][5]

※問題不備により、正解の選択肢は3つあります。うち2つを選択できれば正解です。[3][7]

## 解説

**選択肢1（正解）**
クロザピンは登録された医療機関、医療従事者でなければ扱えません。CPMS規定に従った血液検査が実施されたことをeCPMS上で確認してから、クロザピン専任登録管理薬剤師が調剤します。[2]

**選択肢2（誤り）**
心理教育や作業療法プログラムの立案は薬剤師の範囲ではなく、多職種連携における他職種の役割です。[2][6]

**選択肢3（正解）**
クロザピンは新たに糖尿病になったり、糖尿病が悪化したりする可能性があるため、糖尿病の症状（体重減少、口渇、多飲、頻尿）が発現した場合は、直ちに受診するように指導すべき事項です。[1][2]

**選択肢4（誤り）**
クロザピンに残薬が生じた場合、専任薬剤師が速やかに回収・廃棄します。患者に自己廃棄するよう指導することは不適切です。[2]

**選択肢5（正解）**
クロザピンの副作用で高血糖が生じることがあるため、体重減少などの糖尿病症状をセルフモニタリングするよう指導することは重要な留意事項です。[1][2]
