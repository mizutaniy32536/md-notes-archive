# 109-24

**問題文**
地上部での光化学オキシダントの生成に関与する主な非電離放射線はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-24 を参照

**選択肢**
1. 遠赤外線
2. 近赤外線
3. 可視光線
4. UV−A
5. UV−C

**正解:** 4

**解説**
解答　４

光化学オキシダントは、工場や自動車から排出される窒素酸化物や炭化水素に、紫外線が反応することで生成する。選択肢中の紫外線UV−AとUV−Cのうち、UV−Cは、オゾン層に吸収され、ほとんど地上に到達しないため、地上部での光化学オキシダントの生成に関与する主な非電離放射線はUV−Aである。
