# 109-163

**問題文**
肝疾患、膵疾患及び胆道疾患の治療に用いられる薬物に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-163 を参照

**選択肢**
1. フロプロピオンは、カテコール−O−メチルトランスフェラーゼ（COMT）を阻害してノルアドレナリンの代謝を抑制することで、Oddi括約筋を弛緩させる。
2. ウルソデオキシコール酸は、胆汁酸の増加を伴わずに胆石表面のコレステロールをミセル化することで、コレステロール胆石を溶解する。
3. カモスタットは、膵外分泌を促進して膵機能を活性化することで、慢性膵炎を改善する。
4. テノホビルは、NS5A複製複合体を阻害することで、B型肝炎ウイルス（HBV）の複製を抑制する。
5. ソホスブビルは、NS5B RNA依存性RNAポリメラーゼを阻害することで、C型肝炎ウイルス（HCV）の複製を抑制する。

**正解:** 1,5

**解説**
解答　１、５

１　正
フロプロピオンは、カテコール−O−メチルトランスフェラーゼ（COMT）を阻害してノルアドレナリンの代謝を抑制することで、Oddi括約筋を弛緩させて、膵液排出を促進する。

２　誤
ウルソデオキシコール酸は、胆汁中の胆汁酸を増加させて、胆石表面のコレステロールをミセル化することで、コレステロール胆石を溶解する。

３　誤
カモスタットは、膵液に含まれるタンパク分解酵素を阻害することで、慢性膵炎における急性症状を改善する。

４　誤
テノホビルは、体内で活性化され、逆転写酵素（RNA依存性DNAポリメラーゼ）を阻害することで、B型肝炎ウイルス（HBV）の増殖を抑制する。なお、NS5A複製複合体を阻害することで、C型肝炎ウイルス（HBC）の複製を抑制するのは、レジパスビルなどである。

５　正
ソホスブビルは、体内で活性化され、NS5B RNA依存性RNAポリメラーゼを阻害することで、C型肝炎ウイルス（HCV）の複製を抑制する。
