# 109-119

**問題文**
2種類の染色法を用いて細菌Aについて調べた。染色法1（グラム染色）では明瞭な染色像が観察されなかったため、染色法2（抗酸染色）での染色を行ったところ、染色法2では陽性であった。染色法1、2の染色操作を図に示した。＜image1:center＞細菌Aとこれらの染色法に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-119 を参照

**選択肢**
1. 細菌Aは、ペプチドグリカン層に加えて染色法1での染色に抵抗性の細胞壁成分を持っている。
2. ペプチドグリカン層が厚い細菌では、クリスタルバイオレットとルゴール液でつくられた色素がエタノールで除去される。
3. 高級脂肪酸やワックスに富む細胞壁成分を持つ細菌は、染色法2で染色されない。
4. 細菌Aの細胞壁は、融点の低い脂質に富んでおり、加温すると石炭酸フクシンの透過性が増す。
5. 染色法1の陽性細菌は青紫色、染色法2の陽性細菌は赤色に染まる。

**正解:** 1,5

**解説**
解答　１、５

設問で、「染色法1（グラム染色）では明瞭な染色像が観察されなかったため、染色法2（抗酸染色）での染色を行ったところ、染色法2では陽性であった。」と記載されていることから細菌Aは、抗酸菌であると予想される。抗酸菌は細胞壁にペプチドグリカン層に加えて、高級脂肪酸であるミコール酸などを含む脂質が多量に存在するため、グラム染色での染色に抵抗性を示すが、抗酸染色では陽性を示す。

１　正
前記参照

２　誤
ペプチドグリカン層が厚い細菌（グラム陽性菌）では、クリスタルバイオレットとルゴール液でつくられた色素がエタノールで除去されにくい。そのため、グラム陽性菌はグラム染色により紫色に染色されたままになる。また、グラム陰性菌は、一般にペプチドグリカン層が薄く、クリスタルバイオレットとルゴール液でつくられた色素がエタノールで除去されやすいため、脱色される。脱色されたままでは観察しにくいため、その後サフラニン液により対比染色し、赤色に染色して観察する。

３　誤
高級脂肪酸やワックスに富む細胞壁成分を持つ細菌（抗酸菌）は、染色法2（抗酸染色）で染色される。抗酸菌は抗酸染色において、石炭酸フクシンを加温しながら加えることで赤色に染色されて、アルコールを加えても脱色されない特徴がある。

４　誤
細菌A（抗酸菌）の細胞壁に含まれるミコール酸は高級脂肪酸に分類されており、他の脂肪酸よりも融点が高い。したがって、抗酸菌の細胞壁は、融点の高い脂質に富んでおり、加温すると石炭酸フクシンの透過性が増す。

５　正
染色法1（グラム染色）の陽性菌（グラム陽性菌）は紫色に、陰性菌（グラム陰性菌）は赤色になる。また、染色法2（抗酸染色）の陽性菌（抗酸菌）は赤色、陰性菌（抗酸菌以外）は青色に染色される。
