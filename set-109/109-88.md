# 109-88

**問題文**
ノロウイルスで汚染された病室の床を消毒するのに最も適するのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-88 を参照

**選択肢**
1. クロルヘキシジングルコン酸塩
2. グルタラール
3. 次亜塩素酸ナトリウム
4. ポビドンヨード
5. ベンザルコニウム塩化物

**正解:** 3

**解説**
解答　３

消毒薬は、芽胞形成菌を含むほぼ全ての微生物に有効な高水準消毒薬、ウイルスや結核菌に対して有効な中水準消毒薬、一般細菌に有効な低水準消毒薬に分類される。ノロウイルスに対して有効な消毒薬は、高水準消毒薬と中水準消毒薬である。

１　誤
クロルヘキシジングルコン酸塩は低水準消毒薬であり、ノロウイルスに対する有効性が低い。

２　誤
グルタラールは高水準消毒薬であり、ノロウイルスに有効であるが、グルタラールの蒸気は眼や呼吸器の粘膜を刺激するため、病室の床の消毒には適さない。

３　正
次亜塩素酸ナトリウムは中水準消毒薬であり、ノロウイルスで汚染された病室の床を消毒するのに最も適している。

４　誤
ポビドンヨードは中水準消毒薬であり、ノロウイルスに有効であるが、ポビドンヨードは暗赤褐色の液体であり、病室の床の消毒には適さない。

５　誤
ベンザルコニウム塩化物は低水準消毒薬であり、ノロウイルスに対する有効性が低い。
