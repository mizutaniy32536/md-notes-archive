# 109-255

### 第109回薬剤師国家試験 問255

**問題文（症例共有部分を含む）**  
70歳男性。身長165cm、体重50kg。1年前に心筋梗塞を起こし、心室細動による心停止で救命救急センターに搬送され、心蘇生術を施し心機能が回復した。その後、意識消失発作を起こしたため、半年前に埋め込み型除細動器(ICD)を植え込んだ。しかし、その後もICDが作動することがあり、医師は心室性不整脈予防のための追加薬を検討している。  
**問255（薬理）**  
前問（問254）で追加する薬物が有する作用機序はどれか。**2つ選べ。**[1][3][4][6]

**全選択肢（番号付き）**  
1. 可溶性グアニル酸シクラーゼを活性化する。  
2. アデノシンA<sub>1</sub>受容体を刺激する。  
3. アセチルコリンM<sub>2</sub>受容体を遮断する。  
4. L型Ca<sup>2+</sup>チャネルを遮断する。  
5. K<sup>+</sup>チャネルを遮断する。[1][3][4]

**正解番号**  
**4、5**[1][3][4]

**解説**  
問254の正解は**4. アミオダロン**であり、ショック抵抗性心室細動（VF）に対する追加治療薬として適切。アミオダロンはクラスIII抗不整脈薬で、主作用は**K<sup>+</sup>チャネル遮断（選択肢5）**による作用延長効果を有する。また、Na<sup>+</sup>チャネル遮断作用、**L型Ca<sup>2+</sup>チャネル遮断作用（選択肢4）**、および抗アドレナリン作用（β遮断作用）を併せ持つため、心室性不整脈予防に有効。[2][3][6]  
- 選択肢1：誤り。可溶性グアニル酸シクラーゼ活性化は硝酸薬（例: 一硝酸イソソルビド）の作用でアミオダロンではない。[1][3]  
- 選択肢2：誤り。アデノシンA<sub>1</sub>受容体刺激はアデノシンなどの作用でアミオダロンに該当しない。[1][3]  
- 選択肢3：誤り。アセチルコリンM<sub>2</sub>受容体遮断は一部抗不整脈薬に見られるが、アミオダロンの主作用ではない。[1]  
ICD作動の続く再発性心室細動・心室頻拍予防にアミオダロンが標準的に用いられる。[3][6]
