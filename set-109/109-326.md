# 109-326

**問題文**
84歳男性。体重減少と身体活動の低下のため介護保険施設に入所中である。歩行中に転倒し、大腿骨頸部骨折により整形外科病院に入院となった。患者は、オルメサルタン、アゼルニジピン、エソメプラゾール、アトルバスタチン、センノシドを服用中であった。（現病歴）高血圧症、脂質異常症、便秘症（入院2週間前の血圧記録）血圧手帳より、平均112/64 mmHg（入院時検査値）血圧 110/62 mmHg、心拍66拍/分、LDL−C 82 mg/dL、HDL−C 56 mg/dL、TG（トリグリセリド）110 mg/dL病棟担当薬剤師は、患者が持参した薬剤の継続について医師から相談を受け、本患者の生命予後に関して、文献の情報を参考に検討した。＜image1:center＞生命予後の改善を念頭に、薬剤師が医師に伝える内容として、最も優先順位が高いのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-326 を参照

**選択肢**
1. オルメサルタンを中止する。
2. アトルバスタチンを中止する。
3. オルメサルタンとアゼルニジピンを同時に中止する。
4. オルメサルタンとアゼルニジピンの合剤に変更する。
5. 現在の治療を継続する。

**正解:** 1

**解説**
解答　１

本患者は、高血圧症の治療のためにアンギオテンシンⅡAT1受容体遮断薬のオルメサルタンとCa2＋チャネル遮断薬のアゼルニジピンの2剤を併用している。また、本患者は生存率の低い収縮期血圧が130 mmHg未満かつ降圧薬2剤以上の群に該当するため、生命予後の改善に必要なことは、降圧剤を2剤未満にすることで生存率の高い収縮期血圧が130 mmHg未満かつ降圧薬2剤未満の群にすることであると読み取れる。つまり、薬剤師が医師に伝える内容として最も優先順位が高いのは、降圧剤のオルメサルタンまたはアゼルニジピンを中止することである。なお、オルメサルタンとアゼルニジピンを同時に中止することは、高血圧の治療を中止することであるため不適切である。
