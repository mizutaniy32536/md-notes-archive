# 109-77

大麻取締法で規定される「大麻」に該当しない大麻草（カンナビス・サティ バ・エル）の部位はどれか。1つ選べ。
1 花穂
2 未熟な茎
3 葉
4 種子
5 根

## 正解
**4. 種子**

## 解説
大麻取締法で「大麻」に該当しないもの：
- **成熟した茎**
- **種子**

それ以外は全て「大麻」に該当する。

❌ 花穂、葉、根、未熟な茎 → 全て「大麻」  
⭕ 種子、成熟した茎 → 「大麻」に該当しない

**理由：** 種子や成熟した茎にはTHC（テトラヒドロカンナビノール）がほとんど含まれないため。  
例：七味唐辛子の麻の実（種子）、麻縄（成熟した茎）はOK。

**引っかけポイント：** 「未熟な茎」は×。「成熟した茎」だけが除外される。
