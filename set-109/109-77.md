# 109-77

大麻取締法で規定される「大麻」に該当しない大麻草（カンナビス・サティバ・エル）の部位はどれか。１つ選べ。

１ 花穂
２ 未熟な茎
３ 葉
４ 種子
５ 根

**正解:** ４

**解説**
大麻取締法で規定される「大麻」とは、大麻草(カンナビス・サティバ・エル)及びその製品をいう。ただし、大麻草の成熟した茎及びその製品(樹脂を除く。)並びに大麻草の種子及びその製品を除くとされている。
したがって、大麻草の種子や成熟した茎は、「大麻」に該当しない。
