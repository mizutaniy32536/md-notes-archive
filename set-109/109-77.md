# 109-77

**問題文**
大麻取締法で規定される「大麻」に該当しない大麻草（カンナビス・サティバ・エル）の部位はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-77 を参照

**選択肢**
1. 花穂
2. 未熟な茎
3. 葉
4. 種子
5. 根

**正解:** 4

**解説**
解答　４

大麻取締法で規定される「大麻」とは、大麻草(カンナビス・サティバ・エル)及びその製品をいう。ただし、大麻草の成熟した茎及びその製品(樹脂を除く。)並びに大麻草の種子及びその製品を除くとされている。
したがって、大麻草の種子や成熟した茎は、「大麻」に該当しない。
