# 109-47

TDMにおける薬物血中濃度の測定において、全血試料を用いて分析する必要がある薬物はどれか。１つ選べ。

１ テイコプラニン
２ バルプロ酸ナトリウム
３ フェニトイン
４ シクロスポリン
５ ジゴキシン

**正解:** ４

**解説**
TDMにおける薬物血中濃度の測定において、血清や血漿ではなく全血試料を用いて分析する必要がある選択肢中の薬物は、選択肢４のシクロスポリンである。シクロスポリンは、血液中の赤血球画分に多く分布するため、血清や血漿を用いて分析を行うと正しい薬物血中濃度が測定できない。
なお、同様の理由でタクロリムスとエベロリムスも全血試料を用いて分析する必要がある。
