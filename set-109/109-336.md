# 109-336

**問題文**
42歳男性。身長165 cm。血液検査の結果、血清ナトリウム濃度が147 mEq/Lであり、高張性脱水と診断され、輸液により水分を補給することになった。この患者の水分欠乏量に最も近い値はどれか。1つ選べ。ただし、この患者の体重は60 kg、血清ナトリウム濃度の目標値を140 mEq/L、体水分量を体重の50％とする。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-336 を参照

**選択肢**
1. 1.0 L
2. 1.5 L
3. 2.0 L
4. 2.5 L
5. 3.0 L

**正解:** 2

**解説**
解答　２

現在（高張性脱水時）の体重は60 kgで、体水分量を体重の50％とすると現在の体水分量は30 kg＝30 Lである。現在のナトリウム量は、血清ナトリウム濃度147 mEq/Lより、147 mEq/L×30 L＝4410 mEqである。血清ナトリウム濃度の目標値が140 mEq/Lであることから、目標とする体水分量は4410 mEq÷140 mEq/L＝31.5 Lである。よって、水分欠乏量は、目標とする体水分量31.5 Lから現在の体水分量30 Lを引いて1.5 Lとなる。
