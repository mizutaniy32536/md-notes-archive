# 109-92

**問題文**
状態関数に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-92 を参照

**選択肢**
1. 状態関数の変化量は系の変化の経路に依存する。
2. 示量性状態関数においては加成性が成立する。
3. 示強性状態関数は物質量に依存する。
4. 体積は示量性状態関数である。
5. エントロピーは示強性状態関数である。

**正解:** 2,4

**解説**
解答　２、４

物理量は大きく経路関数と状態関数の2つに分けられる。経路関数は系の変化の経路に依存し、状態関数は系の変化の経路に依存しない。
更に、状態関数は示量性と示強性の2つに大別できる。示量性状態関数は加成性が成立し、物質量[mol]に比例する。示強性状態関数は加成性が成立せず、物質量[mol]に依存しない。（加成性：単純に足し合わせることができる性質）

１　誤
状態関数は、系の状態が定まると一義的に定まる量である。よって変化量は系の変化の経路に依存しない（始状態と終状態の系の状態のみに依存する）。 

２　正
示量性状態関数においては加成性が成立する。

３　誤
示強性状態関数は物質量[mol]に依存しない。

４　正
示量性状態関数の例として体積、質量、内部エネルギー、エントロピー、エンタルピー、ギブズエネルギーなどがある。

５　誤
示強性状態関数の例として圧力、温度、濃度、モルエントロピー、モルエンタルピー、モルギブズエネルギーなどがあるが、エントロピーは示量性状態関数である。選択肢４の解説参照。
