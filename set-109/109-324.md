# 109-324

**問題文**
21歳男性。スポーツ競技者。花粉症による鼻炎症状がつらく、練習に影響するので自分で何か良いものはないかと調べたところ、ある健康食品が良いとインターネット上で評判になっていることを知り、当該健康食品について相談するため薬局を訪れた。この競技者から、ドーピング禁止物質（注）は摂取できないという申し出があり、薬剤師が対応した。（注）ドーピング禁止物質：世界アンチ・ドーピング機構が定める禁止表に記載されている物質問324（法規・制度・倫理）この薬局では、この健康食品以外にも複数の健康食品を取り扱っており、店舗内での健康食品の広告を検討している。次のうち、医薬品的な効果に該当せず、医薬品でなくても広告が可能なのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-324 を参照

**選択肢**
1. がんの治癒
2. 花粉症の予防
3. 健康維持
4. 細胞の活性化
5. 肝機能の向上問325（実務）この競技者から症状を詳しく聴取したところ、一般用医薬品で対応したほうがよいと判断された。鼻炎に効果がある一般用医薬品のうち、ドーピング禁止物質を含む医薬品はどれか。2つ選べ。＜image1:center＞

**正解:** 3

**解説**
問324　解答　３

健康食品の販売において、医薬品として承認を受けていないにもかかわらず、医薬品的な効能・効果の広告を行うことは、承認前の医薬品の広告に該当するため禁止されている。
医薬品的な効能・効果について以下に示す。
【医薬品的な効能・効果】
①	疾病の治療又は予防を目的とする効果
例：「がんの治療」、「花粉症の予防」など
②	身体の構造又は機能に影響を与えることを目的とする効果
例：「細胞の活性化」、「肝機能の向上」など
したがって、本問の選択肢のうち、健康食品の広告として可能なのは、「健康維持」である。

問325　解答　２、５

ドーピング禁止物質については、世界アンチ・ドーピング機構によって公表されており、各物質の特徴にあわせて「常に禁止される物質」「競技会（時）に禁止される物質」「特定競技において禁止される物質」に分けられる。

１　誤
成分中にドーピング禁止物質は含まれていない。 

２　正
成分中には、競技会（時）に禁止されるドーピング禁止物質である塩酸プソイドエフェドリンが含まれている。

３　誤
成分中にドーピング禁止物質は含まれていない。

４　誤
成分中にドーピング禁止物質は含まれていない。

５　正
成分中のマオウにはエフェドリンが含まれており、エフェドリンは競技会（時）に禁止されるドーピング禁止物質に該当する。
