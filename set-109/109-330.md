# 109-330

**問題文**
28歳女性。体重50 kg。5年前に双極性障害と診断され、近隣の精神科通院加療中であった。炭酸リチウム800 mg（21.6 mEq）/日の内服でコントロール良好であった（血清リチウム濃度：0.8〜1.0 mEq/L）が、夫と口論になりその後希死念慮を認めた。翌朝午前5時ごろに炭酸リチウム錠200 mgを100錠過量服用した。その後、夫からの通報で救急搬送され、直ちに経鼻胃管による胃洗浄等の処置を施したが心電図でQT延長を認めたことから、リチウム中毒を疑い、入院の上で同日16時より持続的血液透析（CHD）を開始した。来院時に1.6 mEq/Lであった血清リチウム濃度はCHD開始時には3.6 mEq/Lに上昇していた。その後、20時、22時、翌日12時に採血と心電図検査を実施した。患者の経過を図に示す。（来院時所見等）血圧75/45 mmHg、脈拍92拍/分、QTc（Bazett補正式）0.455秒（基準値：0.36秒以上0.44秒未満）、血清リチウム濃度1.6 mEq/L（有効治療域：0.6〜1.2 mEq/L）、血清クレアチニン0.75 mg/dL、Na 140 mEq/L、K 3.8 mEq/L＜image1:center＞CHDは第2病日12時に離脱、経過観察とした。第3病日12時の心電図ではQTcは0.431秒にまで改善し、患者の容態も安定した。担当薬剤師のこの患者に対するアセスメント等として適切なのはどれか。2つ選べ。ただし、CHD離脱によって炭酸リチウムの全身クリアランスは半分になること、炭酸リチウムの分布容積はCHDによらず0.84 L/kgとする。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-330 を参照

**選択肢**
1. 服用した炭酸リチウムの半分に相当する量が吸収されていた。
2. 腸に達した炭酸リチウム除去のための薬用炭投与は無効である。
3. CHDの離脱時には心電図は正常化している。
4. リチウムの体内動態は非線形性を特徴とする。
5. 第3病日12時の血清リチウム濃度は0.5 mEq/L 未満と推定される。

**正解:** 不明

**解説**
解答　解なし

１　誤
本患者が服用した炭酸リチウムの量は、200 mg/錠×100錠＝20,000 mgであり、炭酸リチウム800 mgはリチウムとして21.6 mEqであるため、炭酸リチウム20,000 mgはリチウムとして540 mEqに相当する（800 mg：21.6 mEq＝20,000 mg：X mEqより、X＝540 mEq）。本患者が服用した炭酸リチウムが全て吸収されたと仮定すると、炭酸リチウムの分布容積が0.84 L/kg×50 kg＝42 Lであることから、血中濃度が540 mEq／42 L≒12.86 mEq/L上昇すると考えられ、服用した炭酸リチウムの半分が吸収されたと仮定すると、血中濃度が6.43 mEq/L上昇すると考えられる。本患者のCHD開始時の血中濃度が3.6 mEq/Lであり元々0.8〜1.0 mEq/Lでコントロールされていたことを考慮すると上昇した血中濃度は2.6〜2.8 mEq/Lであり、服用した炭酸リチウムの半分に相当する量は吸収されていないと考えられる。

２　正
薬用炭（活性炭）は薬毒物中毒における吸着や解毒に用いられるが、リチウムを吸着することはできないため、腸に達した炭酸リチウム除去のための薬用炭投与は無効である。

３　誤
グラフより、CHDの離脱時のQTc時間は0.444秒であり、QTc時間の基準値（0.36秒以上0.44秒未満）から逸脱しているため、心電図が正常化しているとは言えない。

４　誤
リチウムの体内動態は線形性を特徴とする。

５　正
グラフより、CHDを開始した第1病日16時のリチウム濃度が3.6 mEq/Lで、CHDを離脱した第2病日12時のリチウム濃度が0.96 mEq/Lであり、20時間で約4分の1になっていることから、CHDを実施中の半減期は約10時間と考えられる。
ここで、半減期t1/2と消失速度定数keの関係式（1）と全身クリアランスCLtotと分布容積Vdの関係式（2）、及び、CHD離脱によって炭酸リチウムの全身クリアランスが半分になることと炭酸リチウムの分布容積はCHDによらず0.84 L/kgで一定であることから、CHD離脱によって消失速度定数keが半分になるため、半減期t1/2が2倍の約20時間になると考えられる。

よって、CHDを離脱した第2病日12時のリチウム濃度が0.96 mEq/Lでその20時間後の第3病日8時の時点で半分の0.48 mEq/Lになると予測されるため、第3病日12時には0.5 mEq/L未満になっていると推定される。

※本問は国家試験当日に配布されたグラフの左縦軸のラベルの単位が誤っていたため解なしとなった問題である。
