# 109-45

**問題文**
胆汁中への排泄が主な消失経路である薬物はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-45 を参照

**選択肢**
1. カナマイシン
2. プラバスタチン
3. リチウム
4. メトトレキサート
5. エナラプリル

**正解:** 2

**解説**
解答　２

選択肢の薬物のうち、胆汁中への排泄が主な消失経路である薬物は、プラバスタチンである。なお、その他の選択肢の薬物の主な消失経路は、すべて尿中への排泄（腎消失）である。
