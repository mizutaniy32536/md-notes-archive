# 109-308

（法規・制度・倫理） 病院薬剤師が、患児及び患児の両親に説明するにあたり、これまでに収集した情 報の確認を行った。情報として、誤っているのはどれか。1つ選べ。
1 バイオ後続品は、先行バイオ医薬品と同等／同質の品質、安全性、有効性を有
する医薬品である。
2 バイオ後続品は、先行バイオ医薬品に比べて開発費が低く抑えられ、薬の価格
が安く設定されている。
3 生物学的同等性試験により先行バイオ医薬品との同等性が証明できれば、バイ
オ後続品の臨床試験は不要である。
4 原則として、バイオ後続品の製造販売後調査が実施される。
5 バイオ後続品について医薬品リスク管理計画の作成が求められている。

## 正解
**3. 生物学的同等性試験により先行バイオ医薬品との同等性が証明できれば、バイオ後続品の臨床試験は不要である → 誤り**

## 解説

### バイオ後続品（バイオシミラー）とは
先行バイオ医薬品の特許切れ後に開発される、**同等／同質**の品質・安全性・有効性を有する医薬品

**代表例：**
- インフリキシマブBS（レミケードの後続品）
- エタネルセプトBS（エンブレルの後続品）

### バイオ後続品とジェネリック医薬品の違い

| 項目 | ジェネリック医薬品 | バイオ後続品 |
|---|---|---|
| **種類** | 化学合成医薬品 | 生物由来医薬品（抗体医薬品等） |
| **先発品との関係** | **完全に同じ** | **同等／同質**（完全同一ではない） |
| **承認に必要な試験** | 生物学的同等性試験のみ | 品質試験 + 非臨床試験 + **臨床試験** |
| **開発費** | 低い | ジェネリックより高いが先行品よりは低い |
| **価格** | 先発品の約40〜50% | 先発品の約70% |

### 各選択肢の検証

1. ⭕ **同等／同質の品質・安全性・有効性**
   - バイオ医薬品は生物由来のため完全同一は不可能
   - 「同等／同質」という表現が正しい

2. ⭕ **価格が安く設定**
   - 開発費が先行品より低い
   - 薬価は先行品の約70%

3. ❌ **臨床試験は不要 → 誤り**
   - バイオ後続品の承認には**臨床試験が必須**
   - ジェネリック医薬品とは異なる
   - 品質試験 + 非臨床試験 + **臨床試験**（薬物動態、有効性、安全性）

4. ⭕ **製造販売後調査が実施される**
   - 市販後の安全性確保のため

5. ⭕ **医薬品リスク管理計画（RMP）の作成**
   - 全ての新医薬品に必要

### なぜ臨床試験が必要？

バイオ医薬品の特徴：
- タンパク質などの高分子化合物
- 製造工程が複雑
- 微妙な違いが免疫原性などに影響する可能性

→ **完全に同じものを作ることは不可能**
→ だから臨床試験で有効性・安全性を確認する必要がある
