# 109-251

### 第109回薬剤師国家試験 問251

**問題文（共通ステム）**  
50歳男性。会社員。人事異動で1年前に本社の営業課長を命じられた。しかし仕事に順応できず、ストレス、不安感及び過食が3ヶ月続いた。上司のすすめもあり心療内科を受診し、うつ病と診断され以下の**処方1**で治療中である。内服開始後、特に副作用は現れていないが、十分な効果が認められないため、医師は処方に新たに薬剤を追加して併用療法を行いたいと考えている。なお、男性は現在排尿障害を伴う前立腺肥大症で**処方2**を内服中である。[1][2][3]

**問251（薬理）**  
前問において禁忌のため用いることができないと判断された薬物は、この患者の**排尿障害**を悪化させるおそれがある。その理由はどれか。**2つ選べ**。[3][4]

**全選択肢**  
1. ドパミンD<sub>2</sub>受容体が遮断されるため。  
2. **ムスカリンM<sub>3</sub>受容体が遮断されるため。**  
3. セロトニン5-HT<sub>3</sub>受容体が遮断されるため。  
4. **アドレナリンα<sub>1</sub>受容体が刺激されるため。**  
5. アンドロゲン受容体が刺激されるため。[1][2][3][4]

**正解番号**  
**2、4**[1][2][3][4]

**解説**  
問250では、排尿障害を伴う前立腺肥大症の患者に抗コリン作用（ムスカリンM<sub>3</sub>受容体遮断）やα<sub>1</sub>受容体刺激作用を持つ三環系・四環系抗うつ薬（アミトリプチリン、マプロチリン、クロミプラミン、ミルナシプラン）が禁忌となる。一方、アリピプラゾールはこれらの作用がなく併用可能（問250正解：2）。[1][2][3]  
問251では、禁忌薬の理由として、**2. ムスカリンM<sub>3</sub>受容体遮断**（抗コリン作用による膀胱平滑筋弛緩・尿閉悪化）と**4. アドレナリンα<sub>1</sub>受容体刺激**（尿道平滑筋収縮・排尿困難悪化）が正しい。三環系・四環系薬の抗コリン作用とミルナシプランのα<sub>1</sub>刺激作用が該当。[1][2][3][4]
