# 109-53

**問題文**
等張化剤として用いられるのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-53 を参照

**選択肢**
1. エタノール
2. ポリソルベート80
3. 塩酸
4. クロロブタノール
5. 濃グリセリン

**正解:** 5

**解説**
解答　５

１　誤
エタノールは、難溶性薬物の溶解性を改善するために、補助溶媒（コソルベント）として用いられる。なお、補助溶媒により難溶性薬物の溶解性が改善する現象をコソルベンシーという。

２　誤
ポリソルベート80は、非イオン性界面活性剤であり、懸濁化剤や乳化剤として用いられる。

３　誤
塩酸は、注射剤や点眼剤などでpH調節剤として用いられる。

４　誤
クロロブタノールは、微生物の発育を阻止するために、保存剤として用いられる。

５　正
濃グリセリンは、注射剤や点眼剤などで等張化剤として用いられる。
