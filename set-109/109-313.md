# 109-313

### 第109回薬剤師国家試験 問313

**問題文**  
80歳男性。脊柱管狭窄症があり、自宅療養中。骨粗しょう症があり、定期的にイバンドロン酸ナトリウム水和物の注射を受けていたが、腰痛がひどく、新たに処方1の薬剤が追加され使用している。この男性は、娘が介護を行うにあたり近々娘の家で同居することとなり、娘から近隣の薬局に処方1の薬剤の取扱いがあるか問合せがあった。現在、当該薬局では処方1の取扱いはなかったが、男性が引っ越してくるまでに時間があるため、処方1の製造販売業者が提供するシステムへの調剤施設の登録などの手続きを行い、調剤できる準備を整えることになった。  
引越し後、処方1を当該薬局において調剤した際、患者の娘から使い方の説明を求められた。薬剤師が行う説明として適切なのはどれか。**2つ選べ**。[1][3][4]

**全選択肢**  
1. 腰の痛い場所に貼ってください。  
2. **皮膚に傷のあるところには貼付しないでください。**  
3. **半分に切って使用しないでください。**  
4. 使用済のお薬は、接着面を表にして処分してください。  
5. このお薬がよく効くように、貼った部位を電気毛布等でよく温めてください。[3][4]

**正解番号**  
**2、3**[1][8][10]

**解説**  
- **2 正しい**：ブプレノルフィン経皮吸収型製剤（処方1はこれに該当）は皮膚から吸収されるため、皮膚に傷がある箇所に貼付すると薬剤吸収が過度に増加する可能性がある。傷のある箇所を避ける説明が適切。[1]  
- **3 正しい**：経皮吸収型製剤は指定の用量・面積で設計されており、半分に切ると吸収量が不均一になるため、使用禁止。[1]  
- **1 誤り**：痛みの部位に直接貼付せず、無毛で清潔な健常皮膚（例: 腰部、上腕外側等）に貼付。[3]  
- **4 誤り**：使用済み製剤は接着面（粘着面）を内側にして貼り合わせ処分するのが適切（表向きは不適切）。[7]  
- **5 誤り**：貼付部位を加温すると吸収が過度に促進されるため避ける。[1][3]  
（領域: 実務。処方1はブプレノルフィン経皮吸収型鎮痛剤で、麻薬及び向精神薬取締法関連の問312と連動。）[1][3][4][7]
