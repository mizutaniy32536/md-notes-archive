# 109-153

**問題文**
痛みを伴う疾患に用いられる薬物に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-153 を参照

**選択肢**
1. タペンタドールは、下行性疼痛抑制系の神経終末でのセロトニン再取り込みを選択的に阻害して、痛覚情報伝達を抑制する。
2. プレガバリンは、求心性一次知覚神経の電位依存性Ca2＋チャネルを構成するα1サブユニットに結合して、神経伝達物質の遊離を抑制する。
3. レバロルファンは、オピオイドµ受容体を刺激して、モルヒネの鎮痛効果を増強する。
4. エレヌマブは、カルシトニン遺伝子関連ペプチド（CGRP）受容体に結合してCGRPの作用を阻害することで、片頭痛発作の発症を抑制する。
5. チザニジンは、アドレナリンα2受容体を刺激して、筋緊張を伴う疼痛を緩和する。

**正解:** 4,5

**解説**
解答　４、５

１　誤
タペンタドールは、オピオイド鎮痛薬であり、オピオイドµ受容体を刺激する作用と下行性疼痛抑制系の神経終末でのノルアドレナリン再取り込みを阻害する作用によって、痛覚情報伝達を抑制する。

２　誤
プレガバリンは、神経障害性疼痛治療薬であり、求心性一次知覚神経の電位依存性Ca2＋チャネルを構成するα2δサブユニットに結合することで、神経伝達物質の遊離を抑制する。

３　誤
レバロルファンは、オピオイド受容体拮抗薬であり、オピオイドµ受容体を遮断することで、モルヒネの急性中毒である呼吸抑制を改善する。

４　正
エレヌマブは、ヒト抗カルシトニン遺伝子関連ペプチド（CGRP）受容体モノクローナル抗体製剤であり、CGRP受容体に結合して、片頭痛発作の原因であるCGRPのCGRP受容体への結合を阻害することで、片頭痛発作の発症を抑制する。

５　正
チザニジンは、中枢性筋弛緩薬であり、アドレナリンα2受容体を刺激して、脊髄多シナプス反射を抑制することで、筋緊張を伴う疼痛を緩和する。
