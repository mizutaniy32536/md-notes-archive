# 109-148

**問題文**
健康保険制度に基づく調剤報酬に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-148 を参照

**選択肢**
1. 調剤報酬点数表は、保険薬局における保険調剤に関する算定の要件と点数を定めている。
2. 健康保険と国民健康保険では、それぞれ異なる調剤報酬点数表を用いている。
3. 調剤報酬改定は、3年に1度行われる。
4. 地域医療への貢献を評価した加算が設けられている。
5. 近年、調剤医療費の内訳において、薬剤料の構成割合は約50%である。

**正解:** 1,4

**解説**
解答　１、４

１　正
調剤報酬点数表は、保険薬局における保険調剤に関する算定の要件と点数を定めている。

２　誤
健康保険と国民健康保険では、それぞれ同じ調剤報酬点数表を用いる。患者が加入する保険の種類にかかわらず、使用される薬剤の品目や価格は同じである。

３　誤
調剤報酬改定は、通常2年に1度行われる。

４　正
地域医療への貢献を評価する加算として、地域支援体制加算が設けられている。

５　誤
令和4年度の調剤医療費の内訳において、薬剤料の構成割合は72.6%である。
