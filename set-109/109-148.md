# 109-148

健康保険制度に基づく調剤報酬に関する記述のうち、正しいのはどれか。２つ選べ。

１ 調剤報酬点数表は、保険薬局における保険調剤に関する算定の要件と点数を定めている。
２ 健康保険と国民健康保険では、それぞれ異なる調剤報酬点数表を用いている。
３ 調剤報酬改定は、3年に1度行われる。
４ 地域医療への貢献を評価した加算が設けられている。
５ 近年、調剤医療費の内訳において、薬剤料の構成割合は約50%である。

**正解:** １、４

**解説**
１　正
調剤報酬点数表は、保険薬局における保険調剤に関する算定の要件と点数を定めている。
２　誤
健康保険と国民健康保険では、それぞれ同じ調剤報酬点数表を用いる。患者が加入する保険の種類にかかわらず、使用される薬剤の品目や価格は同じである。
３　誤
調剤報酬改定は、通常2年に1度行われる。
４　正
地域医療への貢献を評価する加算として、地域支援体制加算が設けられている。
５　誤
令和4年度の調剤医療費の内訳において、薬剤料の構成割合は72.6%である。
