# 109-301

２ β−D−グルカンが陽性となる。
３ 抗ガングリオシド抗体が陽性となる。
４ 迅速ウレアーゼ試験で診断できる。
５ PCR検査で菌の同定ができる。

３ 全身倦怠感、黄疸、皮膚のかゆみが出現したら、すぐに連絡するように指導する。
