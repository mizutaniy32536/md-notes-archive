# 109-301

# 第109回薬剤師国家試験 問301

## 問題文

38歳男性。身長175cm、体重65kg。腎機能及び肝機能は正常。仕事で海外出張が多く疲労気味だった。帰国後、37℃台の微熱と痰がからむ咳が2週間続き、近医を受診した。胸部X線検査の結果、肺に空洞を伴う結節性陰影を認めたため総合病院を紹介受診し、諸検査を受けたところ肺結核と診断された。

この患者に下記の薬剤が処方された。薬剤師による服薬指導内容として**誤っているのはどれか。1つ選べ。**[1][3][4]

## 全選択肢

1. 視覚障害を起こすことがあるので、視力の低下に注意するよう指導する。[3][4]

2. マグロを食べると頭痛が出現することがあるので、食べ過ぎないよう指導する。[3][4]

3. 全身倦怠感、黄疸、皮膚のかゆみが出現したら、すぐに連絡するように指導する。[3][4]

4. 尿や汗が赤く着色することを説明する。[2][3][4]

5. 処方2の薬剤は貧血予防のために処方されていることを説明する。[2][3][4]

## 正解番号

**5**[2][3][4]

## 解説

選択肢1は妥当です。エタンブトールは視覚障害（色覚異常など）を起こす可能性があるため、視力の低下に注意するよう指導することは適切な服薬指導です。[3]

選択肢2は妥当です。イソニアジドはモノアミン酸化酵素阻害薬のような作用があり、マグロなどに含まれるヒスタミンとの相互作用により頭痛が生じる可能性があるため、食べ過ぎないよう指導することは適切です。[3]

選択肢3は妥当です。リファンピシンなどの結核治療薬は肝障害を起こす可能性があるため、全身倦怠感、黄疸、皮膚のかゆみなどの初期症状が出現した場合は医師に連絡するよう指導することは重要です。[3]

選択肢4は妥当です。リファンピシンは尿や汗を赤く着色することが知られており、これは副作用ではなく薬物の特性であるため、患者が安心できるよう説明する必要があります。[2][3]

選択肢5は誤りです。処方2の薬剤はイソニアジドの副作用である**末梢神経障害を予防するために処方されている**のであり、貧血予防のためではありません。[3][4]したがって、選択肢5の服薬指導内容は誤っています。
