# 109-44

**問題文**
ヒト肝臓において、最も発現量が多いシトクロムP450分子種はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-44 を参照

**選択肢**
1. CYP1A2
2. CYP2C19
3. CYP2D6
4. CYP2E1
5. CYP3A4

**正解:** 5

**解説**
解答　５

ヒト肝臓において、最も発現量が多いシトクロムP450の分子種はCYP3A4である。なお、それぞれの分子種の発現量については個体差があるが、CYP1A2が約13％、CYP2C19が約1％、CYP2D6が約2％、CYP2E1が約14％、CYP3A4が約30％程度である。
