# 109-116

### 第109回薬剤師国家試験 問116

**問題文（全文）**  
プリオン病では、プリオンタンパク質（PrP）の感染性を有した異常型が蓄積する。プリオンタンパク質の特徴として、正常型であるPrP<sup>c</sup>と異常型であるPrP<sup>Sc</sup>が相互作用（会合）すると、正常型が異常型へと変換されることが知られている。図1はNMRにより解析されたPrP<sup>c</sup>の立体構造、図2はPrP<sup>Sc</sup>の仮想の立体構造をそれぞれ安定な二次構造をわかりやすくリボン図で描いている。なお、図中には示されていないが、PrP<sup>c</sup>のC末端にはGPIアンカーが付加されている。  
図3は、PrPの遺伝子配列から予想される翻訳産物をアミノ酸の一文字表記で示している。アミノ酸配列のうち、プロリン残基は黒点（●）で、グリシン残基は2重下線で示している。  
以下の記述のうち、正しいのはどれか。２つ選べ。[3][5]

**全選択肢（番号付き）**  
１ 図2に示すPrP<sup>Sc</sup>は、図1に示すPrP<sup>c</sup>に比べてβシート構造が多い。  
２ βシート構造の安定性は、並行するペプチド鎖間で形成されるジスルフィド（S−S）結合によっている。  
３ 図3に示すPrPに点在するプロリンは、その環状構造により二次構造の形成とその規則性に影響を与えている。  
４ 図3に示すPrPに多く存在するグリシンは、特定の二次構造からループ構造への変化を妨げる要因となっている。  
５ PrP<sup>c</sup>においてGPIアンカーが結合するアミノ酸残基は、図3で1番目のメチオニンである。[1][3][5]

**正解番号**  
１と３[1][3][5][7]

**解説**  
- **１ 正**：図1（PrP<sup>c</sup>）は主にαヘリックス（らせん構造）で示され、図2（PrP<sup>Sc</sup>）はβシート（矢印構造）が多く描かれているため、PrP<sup>Sc</sup>の方がβシート構造が多い。[1][3][7]  
- **２ 誤**：βシート構造の安定性は、並行するペプチド鎖間（または反並行）の**水素結合**によるもので、ジスルフィド結合ではない。ジスルフィド結合は3次構造の安定化に関与する。[1][3][5][7]  
- **３ 正**：プロリンは環状構造のため、φ角・ψ角の回転が制限され、αヘリックスやβシートの形成を妨げ、二次構造の規則性に影響を与える。アミノ酸配列に点在するプロリンがこれを示す。[1][3][5]  
- **４ 誤**：グリシンは側鎖がHのみで柔軟性が高く、ループ構造形成を**促進**する要因となる。妨げるものではない。[1][5]  
- **５ 誤**：GPIアンカーはC末端側に結合し、図3の1番目のメチオニン（N末端付近）ではない。[1][5]  

（注：図1・2はリボン図で二次構造を視覚化、図3はアミノ酸配列（N末端左→C末端右）でプロリン（●）・グリシン（==）を強調。問題は生物分野のタンパク質構造知識を問う。）[1][3][5]
