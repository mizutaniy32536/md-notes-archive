# 109-215

# 第109回薬剤師国家試験 問215

## 問題文

処方されたエルデカルシトール（C₃₀H₅₀O₅）に関する記述として**誤っているのはどれか。1つ選べ。**[1][3]

## 全選択肢

1. 共役トリエン構造をもち、遮光保存が必要である。

2. ステロイド骨格を構成する環状構造の一部が開裂した構造をもつ。

3. C環とD環の結合は、trans配置である。

4. 本品は炭素数30に対しヒドロキシ基を4つ含み、水に溶けやすい。

5. 標的受容体と結合する際、ヒドロキシ基は水素結合の形成に関与する。[1]

## 正解番号

**4番**[2][3]

## 解説

選択肢4が誤りです。エルデカルシトールは**活性型ビタミンD₃誘導体**であり、ビタミンDは**脂溶性ビタミン**に分類されます[3]。そのため、ヒドロキシ基を4つ含んでいるにもかかわらず、疎水性を示す部分が多いため、**水に溶けにくい性質**を持っています[1]。「水に溶けやすい」という記述が誤りです。

その他の選択肢は正しいです[1]：
- 選択肢1：共役トリエン構造により光に不安定なため遮光保存が必要
- 選択肢2：ステロイド骨格の一部が開裂した構造を持つ
- 選択肢3：C環とD環間のメチル基と水素原子の配置がtrans配置
- 選択肢5：ヒドロキシ基は標的受容体との結合時に水素結合を形成する
