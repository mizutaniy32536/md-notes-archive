# 109-214

**問題文**
65歳女性。腰背部痛のため近隣の整形外科クリニックを受診したところ、骨粗しょう症と診断され、以下の処方箋を持って薬局を訪れた。問214（実務）処方薬を使用する際に、定期的に検査する必要がある項目はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-214 を参照

**選択肢**
1. エルデカルシトールの血中濃度
2. 血清カルシウム
3. 血中TSH（甲状腺刺激ホルモン）
4. 血清カリウム
5. 血糖問215（物理・化学・生物）処方されたエルデカルシトール（C30H50O5）に関する記述として誤っているのはどれか。1つ選べ。＜image2:center＞
6. 共役トリエン構造をもち、遮光保存が必要である。
7. ステロイド骨格を構成する環状構造の一部が開裂した構造をもつ。
8. C環とD環の結合は、trans配置である。
9. 本品は炭素数30に対しヒドロキシ基を4つ含み、水に溶けやすい。
10. 標的受容体と結合する際、ヒドロキシ基は水素結合の形成に関与する。

**正解:** 2

**解説**
問214　解答　２

エルデカルシトールは活性型ビタミンD3製剤であり、腸管からのカルシウム吸収を促進する。このため、血清カルシウム値を定期的（3～6カ月に1回程度）に測定（選択肢２）し、異常が認められた場合には、休薬する必要がある。

問215　解答　４

１　適切
2つ以上の多重結合が単結合を挟んで交互に連なる部分構造を共役系という。エルデカルシトールは3つの二重結合が単結合を挟んで連なった共役トリエン構造を持つ。なお、共役トリエン構造を持つエルデカルシトールは遮光保存が必要である。

２　適切
エルデカルシトールはステロイド骨格を構成する環状構造の一部が開裂した構造（図の赤部分）をもつ。

３　適切
C環及びD環の結合部位にあるメチル基および水素原子がくさびと破線表記でtransの関係であることから、C環とD環の結合は、trans配置である。

４　不適切
一般に、炭素数3に対しヒドロキシ基を1つ以上含む構造は水に溶けやすい。本品は炭素数30に対しヒドロキシ基を4つしか含まないため、水に溶けにくい。

５　適切
ヒドロキシ基は水素結合のドナー（プロトン供与体）としても、アクセプター（プロトン受容体）としても働くことができるため、標的受容体との水素結合形成に関与する。
