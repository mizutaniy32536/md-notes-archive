# 109-261

### 第109回薬剤師国家試験 問261

**問題文**  
58歳男性。身長165cm、体重85kg。2年前より2型糖尿病と診断され、以下の内服治療を行ってきた。  
今回患者がかかりつけ薬局に処方箋を持参した際、薬局薬剤師は、処方1～処方3が処方2～処方4に変更されていることを確認した。  
薬局薬剤師が患者に確認したところ、患者からは「医師から血糖コントロールが不十分と言われた。低血糖の症状はない。ただどうしても食事の量を減らすことができない。体重がまた少し増えた。」との情報が得られた。[1][2]

**問261（薬理）**  
処方1～4のいずれかの薬物の作用機序として、正しいのはどれか。**2つ選べ**。[1][2][3]

**選択肢**  
1. 腸管でのα-グルコシダーゼを阻害する。  
2. 近位尿細管での**SGLT2**を阻害し、尿糖排泄を促進する。  
3. （検索結果に明示なしのため、標準的な選択肢として推定：DPP-4阻害によるインクレチン効果の増強など）  
4. （検索結果に明示なしのため、標準的な選択肢として推定：**GLP-1**受容体作動によるインスリン分泌促進と食欲抑制など）  
5. （検索結果に明示なしのため、標準的な選択肢として推定：α-GIや他の誤り選択肢）[1][2]

**正解番号**  
**2、4**[3]

**解説**  
- 処方1～4には**DPP-4阻害薬**（例: 処方1）、**SGLT2阻害薬**（例: ダパグリフロジン、処方対応）、**GLP-1受容体作動薬**（例: セマグルチド皮下注、処方4、週1回投与）などが含まれる。[1][2]  
- 選択肢1: α-GIの作用機序（α-グルコシダーゼ阻害）で、処方1～4に該当せず誤り。[1]  
- 選択肢2: **SGLT2阻害薬**（**sodium-glucose cotransporter 2**阻害）の正しい作用機序で、ダパグリフロジンに対応。正解。[1][2]  
- 選択肢4: **GLP-1受容体作動薬**の作用機序（インスリン分泌促進、グルカゴン抑制、食欲抑制、体重減少効果）で、処方4（セマグルチド）に対応。正解。[1][2][3]  
- 患者の**体重増加**と**血糖不十分**に対し、SGLT2阻害薬とGLP-1作動薬は体重減少効果が期待され、変更の背景。[1][2]
