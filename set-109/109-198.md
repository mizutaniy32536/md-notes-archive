# 109-198

**問題文**

**解説**
問198　解答　５

問題文より、与えられた式に［BUNpre］＝80 mg/dL、［BUNpost］＝10 mg/dLを代入すると標準化透析量（Kt/V）が求められる。

よってKt/Vは2.07であり、選択肢５が正しい。

問199　解答　４

透析後の起立性低血圧を予防するために薬物を投与する場合、一般的にはアメジニウムメチル硫酸塩やドロキシドパ（選択肢４）などの昇圧薬が使用される。なお、ドロキシドパは通常、透析終了後の起立時に収縮期血圧が15mmHg以上低下する患者の起立性低血圧の改善に適用される。
