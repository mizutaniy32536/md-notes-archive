# 109-59

**問題文**
高ナトリウム血症の患者に禁忌である薬物はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-59 を参照

**選択肢**
1. トルバプタン
2. エプレレノン
3. エナラプリルマレイン酸塩
4. カルベジロール
5. フロセミド

**正解:** 1

**解説**
解答　１

トルバプタンは、バソプレシンV2受容体遮断薬であり、その水利尿作用により、血液濃縮をきたし、高ナトリウム血症に伴う意識障害を認めることがある。そのため、高ナトリウム血症の患者には投与禁忌とされている。
