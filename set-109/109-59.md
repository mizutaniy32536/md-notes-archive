# 109-59

高ナトリウム血症の患者に禁忌である薬物はどれか。１つ選べ。

１ トルバプタン
２ エプレレノン
３ エナラプリルマレイン酸塩
４ カルベジロール
５ フロセミド

**正解:** １

**解説**
トルバプタンは、バソプレシンV2受容体遮断薬であり、その水利尿作用により、血液濃縮をきたし、高ナトリウム血症に伴う意識障害を認めることがある。そのため、高ナトリウム血症の患者には投与禁忌とされている。
