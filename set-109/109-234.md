# 109-234

# 第109回薬剤師国家試験 問234

## 問題文

55歳男性。身長165 cm、体重46 kg。脳血管障害により3年前から寝たきりの状態となり自宅療養中であった。ベッドから誤って転落し、大腿骨を骨折したため、入院となった。

入院時、仙骨部に褥瘡が認められ、病態としては**黄色期**と判断されたことから、褥瘡チームが介入した。褥瘡チームが回診で確認したところ滲出液、創周囲の感染徴候が認められた。

## 設問

褥瘡チームの薬剤師による医師への提案として適切なのはどれか。**2つ選べ。**

## 選択肢

1. 急性期の病態と考えられることから、積極的な薬物治療は行わず、経過観察する。

2. 抗菌薬を全身投与する。

3. スルファジアジン銀クリームを患部に塗布する。

4. 精製白糖・ポビドンヨード配合軟膏を患部に塗布する。

5. クロベタゾールプロピオン酸エステル軟膏を患部に塗布する。

## 正解

**3、4**[1][2]

## 解説

**選択肢1（誤り）**：患者の褥瘡は黄色期に進行しており、組織は既に壊死している状態です。そのため、「急性期」ではなく、積極的な治療介入が必要です。[1]

**選択肢2（誤り）**：発熱などの全身的な感染の徴候が見られず、創周囲に局所的な感染徴候が認められているため、外用抗菌薬の使用が適切です。全身投与の必要はありません。[1]

**選択肢3（正解）**：スルファジアジン銀クリームは外用抗菌薬であり、壊死組織の除去と感染治療に用いられる適切な選択肢です。[1]

**選択肢4（正解）**：精製白糖・ポビドンヨード配合軟膏も外用薬として、感染管理と創傷治療に適しています。[1]

**選択肢5（誤り）**：クロベタゾールプロピオン酸エステルはステロイド軟膏であり、感染部位に使用してはいけません。[1]

黄色期の褥瘡治療では、壊死組織を取り除き、感染を管理することが重要であり、外用抗菌薬による局所治療が有効な対応です。[1]
