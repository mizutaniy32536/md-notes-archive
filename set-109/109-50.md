# 109-50

**問題文**
複合体を形成することによって、以下の疎水性薬物を安定化するのに用いられる化合物はどれか。1つ選べ。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-50 を参照

**選択肢**
1. アスコルビン酸
2. α−シクロデキストリン
3. カルメロースナトリウム
4. エデト酸ナトリウム水和物
5. パラオキシ安息香酸ブチル

**正解:** 2

**解説**
解答　２

シクロデキストリンは、グルコピラノースがα（1→4）グリコシド結合で環状に結合した構造を有している。グルコースの数が6個のものをα−シクロデキストリンという。シクロデキストリンによる包接化により、アルプロスタジルの水溶性と安定性を改善できる。模式図を下記に示す。

１　誤
アスコルビン酸は、医薬品の酸化を防ぐための抗酸化剤として用いられる。

２　正
上記参照

３　誤
カルメロースナトリウムは、結合剤、増粘剤、分散安定剤などとして用いられる。

４　誤
エデト酸ナトリウム水和物は、キレート剤であり、金属類を不活化するための安定化剤として用いられる。

５　誤
パラオキシ安息香酸ブチルは、細菌の増殖を抑制する保存剤として用いられる。
