# 109-197

# 第109回薬剤師国家試験 問197

## 問題文

二酸化炭素の検出法とその原理に関連する記述のうち、正しいのはどれか。2つ選べ。[1][2]

## 全選択肢

1. 検知管法は、二酸化炭素が酸性溶液に吸収される性質を利用している。[1]

2. 検知管法では、検知管に充てんした検知剤中のpH指示薬の色の変化によって二酸化炭素を検出する。[2]

3. 二酸化炭素は赤外線を吸収する。[1]

4. ガスクロマトグラフィーの水素炎イオン化検出器を用いて、二酸化炭素を検出できる。[1]

5. （選択肢5の詳細は検索結果に記載されていません）

## 正解番号

**2、3**[1][2]

## 解説

**選択肢1は誤り**：検知管法では、検知剤が二酸化炭素と反応して色が変化する性質を利用しています。「酸性溶液に吸収される性質」ではなく、**pH指示薬の色変化**を利用するため、この記述は誤りです。[1]

**選択肢2は妥当**：検知管法についての正しい記述です。検知管に充てんされたpH指示薬が二酸化炭素と反応することで色が変化し、その色の濃淡から濃度を測定します。[1][2]

**選択肢3は妥当**：二酸化炭素が赤外線を吸収するのは基礎知識です。この性質は赤外線吸収分光法など、他の分析手法の原理となります。[1]

**選択肢4は誤り**：水素炎イオン化検出器は「C-H結合を有する有機物を高感度で検出」します。CO₂は検出できません。[1]
