# 109-288

**問題文**
42歳女性。半年前に両側手指関節及び両側膝関節の痛みを自覚し病院を受診した。検査の結果、関節リウマチと診断され、処方1による治療を受けていた。症状のコントロールが不十分だったためメトトレキサートが漸増されたが、多発性関節炎は持続した。最近では仕事にも支障をきたすようになったため、治療方針が変更されることになった。この患者は小児期に水痘に罹患した既往がある。＜image1:center＞問288（病態・薬物治療）この患者に追加する治療薬として、適切なのはどれか。2つ選べ。

**正解:** 2,5

**解説**
問288　解答　２、５

本患者は、関節リウマチと診断されメトトレキサート（従来型合成DMARDs）を服用中だが、症状のコントロールが不十分であるため、トファシチニブ（分子標的型合成DMARDs）やアダリムマブ（生物学的DMARDs）の追加投与を検討する。

問289　解答　１、２

本患者は、

・小児期に水痘に罹患した既往がある
・「左腰部から背部に沿ってかゆみがあって、昨日から痛み出した」と訴えている
・医師が診察後バラシクロビルを処方している
・免疫抑制薬であるメトトレキサートカプセルを服用している
・トファシチニブもしくはアダリムマブを追加後に症状が出現している

以上のことから、帯状疱疹を発症したと推測される。
帯状疱疹は、免疫が低下した際に発症しやすいため、免疫抑制作用のあるメトトレキサートやトファシチニブ、アダリムマブの使用状況より副作用による帯状疱疹を疑ったと考えられる。
