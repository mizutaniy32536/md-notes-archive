# 109-15

**問題文**
主要組織適合遺伝子複合体（MHC）のクラスI分子とクラスⅡ分子の両方で抗原提示できる免疫担当細胞はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-15 を参照

**選択肢**
1. 樹状細胞
2. キラーT細胞
3. 好中球
4. 巨核球
5. 制御性T細胞

**正解:** 1

**解説**
解答　１

主要組織適合遺伝子複合体（MHC）由来のタンパク質であるMHC分子は、自己と非自己の識別に関与する細胞膜結合型糖タンパク質であり、クラスI分子とクラスⅡ分子の２つに大別される。このうち、主要組織適合遺伝子複合体（MHC）のクラスI分子は、ほとんど全ての有核細胞が有しており、クラスⅡ分子は樹状細胞、マクロファージ、B細胞などの抗原提示細胞が有している。
したがって、選択肢のうち、主要組織適合遺伝子複合体（MHC）のクラスI分子とクラスⅡ分子の両方で抗原提示できる免疫担当細胞は、樹状細胞である。
