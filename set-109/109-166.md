# 109-166

### 第109回薬剤師国家試験 問166

**問題文（症例の続き）**  
50歳女性。5年前に近医にて高血圧を指摘され、アムロジピン5 mg/日を服用していた。最近の血圧は155/95 mmHg程度と高値が持続しており、頭痛や脱力を自覚し今回受診した。  
二次性高血圧が疑われたため、腹部CT検査が実施されて左副腎に腫瘍を認めたが、血中コルチゾール値や血中カテコールアミン値の上昇は認めなかった。[1][2][3]

**問166**  
本症例で認められる血液検査所見として、正しいのはどれか。**2つ選べ**。[1][2][3][5]

1. カリウム低値  
2. カルシウム高値  
3. LDLコレステロール高値  
4. 遊離チロキシン(FT4)高値  
5. レニン活性低値[1][2][3][5]

**正解番号**：**1, 5**[1][2]

**解説**  
この症例は副腎腫瘍による二次性高血圧で、コルチゾール値とカテコールアミン値の上昇がないため、**原発性アルドステロン症（コーン症候群）**が疑われる。アルドステロン過剰分泌により、ナトリウム貯留・カリウム排泄促進が起こり、**カリウム低値（低カリウム血症）**と血漿量増加によるレニン抑制で**レニン活性低値**を示す。[1][2][4]  

- 2. カルシウム高値：原発性アルドステロン症との直接的関連なし。[1]  
- 3. LDLコレステロール高値：クッシング症候群で関連するが、本症例はコルチゾール正常。[1]  
- 4. 遊離チロキシン(FT4)高値：甲状腺機能と無関係。[2]  

これにより、問166の正解は**1, 5**。[1][2]
