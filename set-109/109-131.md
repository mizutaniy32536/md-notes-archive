# 109-131

**第109回薬剤師国家試験 問131**

食中毒の原因となる微生物に関する記述のうち、正しいのはどれか。**2つ選べ**。[1][2][3]

**１** 黄色ブドウ球菌が産生する毒素は、100℃で30分間加熱しても毒性を失わない。[1][2][3]  
**２** ボツリヌス菌は、低酸素状態ではボツリヌス毒素を産生しない。[1][2][3]  
**３** 腸管出血性大腸菌は、酸に抵抗性を示し、腸管でベロ毒素を産生する。[1][2][3]  
**４** セレウス菌の嘔吐型食中毒の原因となる毒素は、100℃で30分間の加熱で毒性を失う。[1][3]  
**５** エルシニア・エンテロコリチカは、4℃の低温条件下では発育できない。[1][2][3]

**正解番号：１，３**[1][2]

**解説**  
**１ 正しい**：黄色ブドウ球菌が産生するエンテロトキシンは耐熱性で、菌体は易熱性だが毒素自体は100℃で30分加熱しても毒性を失わない。[1][2]  
**２ 誤り**：ボツリヌス菌は偏性嫌気性菌で、低酸素状態（嫌気性条件下）でボツリヌストキシンを産生する。[1][2]  
**３ 正しい**：腸管出血性大腸菌（EHEC）は胃酸に抵抗性を持ち、腸管内でベロ毒素（志賀毒素）を産生し、出血性大腸炎や溶血性尿毒症症候群を引き起こす。[1][2]  
**４ 誤り**：セレウス菌の嘔吐型食中毒の原因毒素（セレウス菌嘔吐毒素）は耐熱性で、100℃加熱でも不活化しない。[1]  
**５ 誤り**：エルシニア・エンテロコリチカは低温性菌で、4℃の冷蔵条件下でも発育する。[1][2]
