# 109-55

**問題文**
下の模式図で表される経口徐放性製剤の放出制御膜として、適切なのはどれか。1つ選べ。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-55 を参照

**選択肢**
1. 糖衣
2. 半透膜
3. 腸溶性皮膜
4. 生分解性皮膜
5. イオン交換樹脂膜

**正解:** 2

**解説**
解答　２

模式図で表される経口徐放性製剤の放出制御膜として適切なのは、半透膜である。製剤外部から水が半透膜を通して浸透すると、溶解した電解質との浸透圧差によってさらに水が侵入する。それによるプッシュ層の体積増大により、薬物放出口から薬物が押し出される。なお、模式図のような放出制御システムをOROS（Osmotic controlled release oral delivery system）という。
