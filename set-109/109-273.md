# 109-273

### 第109回薬剤師国家試験 問273

**問題文（全文）**  
53歳男性。以前から、A病院内科で高血圧症、脂質異常症、糖尿病、胃潰瘍、腰痛症に対して治療を受けており、症状は安定していた。最近、仕事のストレスのためか気分が落ち込むことが多く、食欲不振、不眠などが続くため、B心療内科を受診した。うつ病と診断され、処方箋を持って、かかりつけの薬局を訪れた。  

**問273（薬剤）**  
前問の薬剤を選択した理由に最も深く関係する機序はどれか。1つ選べ。[1][3][5][6]

**全選択肢（番号付き）**  
1. **胃内pHの上昇**  
2. **小腸CYP3A4発現量の増加**  
3. **小腸P-糖タンパク質発現量の増加**  
4. **肝CYP1A2の阻害**  
5. **肝CYP3A4の阻害**[1][3][5][6]

**正解番号**  
**4**[1][3]

**解説**  
前問（問272）の正解は**チザニジン錠**であり、心療内科の処方薬**フルボキサミン**との併用禁忌である。チザニジンは肝臓で**CYP1A2**によって代謝されるため、フルボキサミンのような**CYP1A2阻害**によりチザニジンの血中濃度が上昇し、副作用（例: 眠気、筋力低下）増加のリスクがある。他の選択肢は本相互作用の主機序とは関係しない（例: 1はファモチジン関連、2・3・5はCYP3A4やP-gp関連）。[1][3][5][6]
