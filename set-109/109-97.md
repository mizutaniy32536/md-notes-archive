# 109-97

確認試験と検出される医薬品の組合せのうち、正しいのはどれか。２つ選べ。

**正解:** ３、５

**解説**
１　誤
記載の医薬品はナプロキセンであり、カルボン酸を含む。塩化鉄（Ⅲ）との反応による呈色はフェノール性水酸基またはヒドロキサム酸を確認するものである。
２　誤
記載の医薬品はレボドパであり、フェノール性水酸基、アミノ酸の構造を含む。フェーリング試液との反応による呈色はアルデヒド基またはα−ケトール基を確認するものである。
３　正
記載の医薬品はd−カンフルであり、ケトンを含む。2,4−ジニトロフェニルヒドラジンとの反応による呈色はアルデヒド、ケトンを確認するものである。
４　誤
記載の医薬品はブドウ糖であり、環状構造では水酸基を含む。ニンヒドリンとの反応による呈色はアミノ酸またはイミノ酸を確認するものである。
５　正
記載の医薬品はアミノ安息香酸エチルであり、芳香族第一級アミン、エステルを含む。亜硝酸ナトリウムでジアゾ化後、N−(1−ナフチル)−N´−ジエチルエチレンジアミンとの反応による呈色は芳香族第一級アミンを確認するものである。
