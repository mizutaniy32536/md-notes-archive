# 109-127

**問題文**
ビタミンに関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-127 を参照

**選択肢**
1. レチノイン酸は、光を感知するロドプシンの構成成分で視覚機能を正常に保つのに必要である。
2. α−トコフェロールは、ヒトでは腸内細菌によって主に供給される。
3. コレカルシフェロールは、紫外線照射により動物の皮膚で7−デヒドロコレステロールから合成される。
4. リボフラビンは生体内で活性化され、糖質や脂質のエネルギー代謝に関与する酵素の補酵素としてはたらく。
5. チアミンは、生体内で補酵素NAD＋やNADP＋として酸化還元反応に関与する。

**正解:** 3,4

**解説**
解答　３、４

１　誤
ビタミンAのうちレチノイン酸は、核内受容体を刺激し、タンパク質の遺伝子発現を転写レベルで調節することで、細胞の正常維持、細胞の分化誘導に関与する。一方、ビタミンAのうちレチノールやレチナールは、暗所での物の識別に関与するロドプシンの原料となり、視覚機能の正常化に関与する。

２　誤
α−トコフェロール（ビタミンE）は、植物油に多く存在するが、動物体内ではほとんど合成されず、腸内細菌による供給もほとんどない。

３　正
生体内で合成された7−デヒドロコレステロール（プロビタミンD）は、紫外線照射により皮膚で開環反応を受けコレカルシフェロール（ビタミンD）になる。その後、肝臓、腎臓の順に水酸化反応を受け、カルシトリオール（活性型ビタミンD）になる。

４　正
リボフラビン（ビタミンB2）は、生体内で活性化されFADやFMNとなり、糖代謝におけるクエン酸回路でのエネルギー産生や、脂肪酸のβ酸化などに関与する。

５　誤
チアミン（ビタミンB1）は生体内で活性化されチアミンピロリン酸（TPP）となり、ピルビン酸の酸化的脱炭酸反応やα-ケトグルタル酸の脱炭酸反応といった糖代謝に関与する。なお、ナイアシンは生体内で活性化されNAD＋やNADP＋となり、酸化還元反応に関与する。
