# 109-169

抗悪性腫瘍薬に関する記述のうち、正しいのはどれか。２つ選べ。

１ シタラビンは、標的細胞で三リン酸化体となり、DNAポリメラーゼを阻害して、DNA合成を低下させる。
２ テガフールは、活性代謝産物が活性酸素を発生させて、DNAを酸化的に傷害する。
３ フルダラビンは、トポイソメラーゼⅠを阻害して、がん細胞のG2期への移行を抑制する。
４ ドキソルビシンは、DNAポリメラーゼ及びRNAポリメラーゼを阻害して、DNA及びRNAの合成を低下させる。
５ アファチニブは、血管内皮細胞増殖因子受容体（VEGFR）チロシンキナーゼを阻害して、血管新生を抑制する。

**正解:** １、４

**解説**
１　正
シタラビンは、ピリミジン代謝拮抗薬であり、標的細胞で三リン酸化体（シタラビン三リン酸ヌクレオチド）となり、デオキシシチジン三リン酸（dCTP）と競合してDNAに取り込まれることで、DNAポリメラーゼを阻害して、DNA合成を低下させる。
２　誤
テガフールは、フルオロウラシルのプロドラッグであり、フルオロウラシルの活性代謝産物であるフルオロデオキシウリジル酸（FdUMP）となり、デオキシウリジン一リン酸（dUMP）に拮抗することで、チミジル酸合成酵素を阻害して、DNA合成を低下させる。なお、活性酸素を発生させて、DNAを酸化的に傷害する薬は、ブレオマイシンなどである。
３　誤
フルダラビンは、プリン代謝拮抗薬であり、活性代謝物がDNA及びRNAに取り込まれることで、DNAポリメラーゼ及びRNAポリメラーゼを阻害して、DNA及びRNAの合成を低下させる。なお、トポイソメラーゼⅠを阻害して、がん細胞のG2期への移行を抑制する薬は、イリノテカンなどである。
４　正
ドキソルビシンは、アントラサイクリン系薬であり、DNA鎖の塩基対間に入り込み（インターカレーション）、DNAポリメラーゼ、RNAポリメラーゼ及びトポイソメラーゼⅡを阻害して、DNA及びRNAの合成を低下させる。
５　誤
アファチニブは、上皮増殖因子受容体（EGFR）チロシンキナーゼを阻害して、抗悪性腫瘍作用を示す。なお、血管内皮細胞増殖因子受容体（VEGFR）チロシンキナーゼを阻害して、血管新生を抑制する薬は、ソラフェニブなどである。
