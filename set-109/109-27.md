# 109-27

自律神経節遮断薬の効果として、正しいのはどれか。１つ選べ。

１ 心拍数低下
２ 散瞳
３ 消化管運動促進
４ 排尿促進
５ 発汗促進

**正解:** ２

**解説**
自律神経節遮断薬の効果は、自律神経の節に存在するニコチン性アセチルコリンNN受容体を遮断することで、各臓器で特に優位な自律神経を遮断する。各臓器は自律神経に支配されており、交感神経が優位に働いている臓器は血管と汗腺であり、その他の臓器（心臓、瞳孔、消化管、膀胱など）は副交感神経が優位に働いている。
１　誤
心臓は副交感神経優位であり、自律神経節遮断薬により副交感神経節が遮断されることで、心拍数増加が現れる。
２　正
瞳孔は副交感神経優位であり、自律神経節遮断薬により副交感神経節が遮断されることで、瞳孔括約筋弛緩による散瞳が現れる。
３　誤
消化管は副交感神経優位であり、自律神経節遮断薬により副交感神経節が遮断されることで、消化管運動低下が現れる。
４　誤
膀胱は副交感神経優位であり、自律神経節遮断薬により副交感神経節が遮断されることで、膀胱排尿筋弛緩による排尿抑制が現れる。
５　誤
汗腺は交感神経優位であり、自律神経節遮断薬により交感神経節が遮断されることで、発汗抑制が現れる。
