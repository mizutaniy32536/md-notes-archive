# 109-7

**問題文**
活性酸素種でないのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-7 を参照

**選択肢**
1. 1O2
2. 3O2
3. ・O2－
4. ・OH
5. H2O2

**正解:** 2

**解説**
解答　２

活性酸素種とは、酸素由来の分子種で反応性が三重項酸素より高いものと定義される。
よって、活性酸素種でないのは3O2（三重項酸素）となる。
