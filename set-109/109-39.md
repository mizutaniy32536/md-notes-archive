# 109-39

DNAジャイレース及びトポイソメラーゼⅣを阻害することで、細菌のDNA複製を抑制する抗菌薬はどれか。１つ選べ。

**正解:** １

**解説**
DNAジャイレース及びトポイソメラーゼⅣを阻害する抗菌薬は、ニューキノロン系である。選択肢のうち、ニューキノロン系に該当するのは１のシプロフロキサシンである。なお、ニューキノロン系の構造において特徴的なところは、フッ素を含有している点である。
１　正
ニューキノロン系抗菌薬のシプロフロキサシンの構造である。シプロフロキサシンは、DNAジャイレース及びトポイソメラーゼⅣを阻害し、DNA合成を抑制することで抗菌作用を示す。
２　誤
テトラサイクリン系抗菌薬のテトラサイクリンの構造である。テトラサイクリンは、細菌リボソーム30Sサブユニットに結合し、タンパク質合成を阻害することで抗菌作用を示す。
３　誤
アミノグリコシド系抗菌薬のカナマイシンの構造である。カナマイシンは、細菌リボソーム30Sサブユニットに結合し、タンパク質合成を阻害することで抗菌作用を示す。
４　誤
ペニシリン系抗菌薬のアモキシシリンの構造である。アモキシシリンは、ペニシリン結合タンパク質のトランスペプチダーゼに結合し、細胞壁合成を阻害することで抗菌作用を示す。
５　誤
マクロライド系抗菌薬のアジスロマイシンの構造である。アジスロマイシンは、細菌リボソーム50Sサブユニットに結合し、タンパク質合成を阻害することで抗菌作用を示す。
