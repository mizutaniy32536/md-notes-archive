# 109-3

**問題文**
質量数は変わらず原子番号が1増加した娘核種が生成するのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-3 を参照

**選択肢**
1. α壊変
2. β－壊変
3. β＋壊変
4. 軌道電子捕獲
5. 核異性体転移

**正解:** 2

**解説**
解答　２

１　誤
α壊変では質量数が4減少し、原子番号が2減少した娘核種が生成する。

２　正

３　誤
β＋壊変では質量数が変わらず、原子番号が1減少した娘核種が生成する。

４　誤
軌道電子捕獲では質量数が変わらず、原子番号が1減少した娘核種が生成する。

５　誤
核異性体転移は核異性体がより安定な状態に変化することをいい、一般的に核異性体がγ粒子放射の代わりに壊変のエネルギーを軌道電子に与えて、この軌道電子が放出されることが多い。核異性体転移の前後では、質量数も原子番号も変化しない。
