# 109-141

**問題文**
廃棄物に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-141 を参照

**選択肢**
1. 産業廃棄物の種類別排出量として最も多いのは、汚泥である。
2. 有害廃棄物の国境を越える移動及びその処理によって生じるヒトの健康被害並びに環境汚染を防止するために、ロンドン条約が制定されている。
3. 循環型社会における廃棄物・リサイクル対策を総合的に推進し、削減対策の優先順位を明確化するために、循環型社会形成推進基本法が制定されている。
4. 海洋汚染の原因となるマイクロプラスチックは、プラスチックゴミの生分解により生じた微小粒子である。
5. 海洋への廃棄物投棄による海洋汚染を防止するための国際的対応として、バーゼル条約が制定されている。

**正解:** 1,3

**解説**
解答　１、３

１　正
2020年における産業廃棄物の種類別排出量として最も多いのは汚泥であり、次いで動物のふん尿、がれきの順となっている。

２　誤
有害廃棄物の国境を越える移動及びその処理によって生じるヒトの健康被害並びに環境汚染を防止するために、バーゼル条約が制定されている。なお、ロンドン条約は海洋への廃棄物投棄による海洋汚染を防止するための国際的対応として制定されている。

３　正
循環型社会形成推進基本法では、発生抑制（リデュース）、再使用（リユース）、再生利用（リサイクル）、熱回収、適正処分の順に削減対策の優先順位を明確化し、循環型社会における廃棄物・リサイクル対策を総合的に推進している。

４　誤
海洋汚染の原因となるマイクロプラスチックは、プラスチックゴミが波や太陽光（紫外線）などで破砕されることにより生じた微小粒子である。

５　誤
