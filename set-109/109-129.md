# 109-129

食品成分に関する記述のうち、正しいのはどれか。２つ選べ。

１ キトサンは、エビやカニの殻に含まれる成分を原料としたもので、血中コレステロール値を低下させる作用がある。
２ アスタキサンチンは、ブドウの果皮に含まれており、コラーゲンの合成を促す作用がある。
３ 難消化性デキストリンは、コンニャクに含まれており、脂肪の燃焼を促す作用がある。
４ ダイゼインは、ダイズに含まれており、抗酸化作用がある。
５ β−カロテンは、ニンジンやカボチャに含まれており、体内で分解してビタミンEが生成する。

**正解:** １、４

**解説**
１　正
キトサンは、エビやカニなどの甲殻類に含まれるキチンを加工することで得られる食物繊維の一種であり、コレステロールの吸収を抑え、血中LDLコレステロールを低下させる作用などを有する。
２　誤
アスタキサンチンは、エビ、カニ、サケなどに多く含まれている赤色の天然色素（カロテノイド）の一種であり、抗酸化作用を有する。なお、ブドウの果皮に含まれる食品成分としては、アントシアニンやレスベラトロールなどがある。
３　誤
難消化性デキストリンは、トウモロコシのデンプンの分解物から製造される水溶性食物繊維の一種であり、腸内環境の改善、コレステロール値の低下、糖の吸収抑制作用などを有する。なお、コンニャクに含まれるのは、食物繊維の一種であるグルコマンナンである。
４　正
ダイゼインは、ダイズなどのマメ科植物に含まれるイソフラボンの一種であり、抗酸化作用や女性ホルモン様作用などを有する。
５　誤
β-カロテンは、ニンジンやカボチャなどの緑黄色野菜に含まれており、体内でビタミンAに変換されるプロビタミンAである。
