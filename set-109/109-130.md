# 109-130

**問題文**
甘味料のうち、フェニルケトン尿症患者に対する注意喚起の表示が義務づけられているのはどれか。1つ選べ。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-130 を参照

**選択肢**
（選択肢は画像を参照）

**正解:** 4

**解説**
解答　４

フェニルケトン尿症患者に対する注意喚起の表示が義務付けられているのは、選択肢４のアスパルテームである。フェニルケトン尿症とは、フェニルアラニンをチロシンに変換するフェニルアラニン水酸化酵素が欠損することで、チロシン由来のドパミン、ノルアドレナリン、アドレナリンなどが低下し、精神発達遅延などの脳の発達障害をきたす。アスパルテームは、アスパラギン酸とフェニルアラニンから合成される甘味料であり、体内でフェニルアラニンに代謝される。よって、アスパルテームを含む食品については、L-フェニルアラニン化合物を含む旨の表示が食品表示法により義務付けられている。なお、選択肢２のアドバンテームおよび選択肢３のネオテームも構造中にフェニルアラニンを含有するが、アスパルテームよりも体内でフェニルアラニンに代謝される可能性が低いことから、表示の義務はない。
