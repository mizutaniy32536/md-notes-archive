# 109-183

下図は湿式顆粒圧縮法の製造工程である。A、Bの単位操作で使用する装置の組合せのうち、正しいのはどれか。１つ選べ。

**正解:** ３

**解説**
湿式顆粒圧縮法の製造工程において、（A）は粉砕、（B）は混合を示す。設問中に示されている装置のうち、粉砕（A）で使用することができる装置はボールミル（イ）、混合（B）で使用することができる装置は二重円錐型混合機（ウ）である。
ア　コーティングに用いるコーティングパンである。コーティングパンは、転動する製剤部分にコーティング液を噴霧することで、錠剤や顆粒剤などの製剤の表面を適当な物質で被覆する機械である。
イ　粉砕に用いるボールミルである。ボールミルは、ボールと粉体を円筒形容器内で回転させ、円筒壁から落下するボールの衝撃と摩擦によって粉砕する機械である。
ウ　混合に用いる二重円錐型混合機である。二重円錐型混合機は、本体容器が回転することにより、2種以上の物質を混ぜ合わせて均一にする機械である。
エ　湿式造粒に用いる押出し造粒機である。押出し造粒機は、医薬品混合物に結合剤を添加して練合した湿潤物を、一定の孔径のスクリーンから押出して顆粒を得る機械である。
オ　乾燥に用いる箱型乾燥機である。箱型乾燥機は、湿った原料を棚上に載せ、温風を棚の間に送ることで原料を乾燥させる装置である。
