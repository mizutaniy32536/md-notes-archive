# 109-16

対象集団について、ある一時点における疾病の有無と要因の保有状況を調査し、その関連を明らかにする疫学研究手法はどれか。１つ選べ。

１ 症例対照研究
２ 縦断的研究
３ 介入研究
４ コホート研究
５ 横断的研究

**正解:** ５

**解説**
１　誤
症例対照研究とは、ある疾病の患者群と対照群を設定し、現在から過去にさかのぼって要因の曝露を両集団で比較する疫学研究手法である。
２　誤
縦断的研究とは、症例対照研究や要因対照研究などに代表される、二時点以上にわたって要因群と対照群の比較をする疫学的研究手法である。
３　誤
介入研究とは、前向き研究の一つであり、分析疫学によって要因と疾病の関連性が明らかにされた場合、その要因を取り除いたり、あるいは人為的に与えたりすることにより、疾病の発生率がどのように変化するのかを観察し、因果関係を決定する疫学研究手法である。
４　誤
コホート研究は、要因対照研究ともよばれ、対象集団を要因曝露の有無によって分け、現在から将来に向かって疾病の発生率（罹患率）を両集団で比較する疫学研究手法である。
５　正
横断的研究とは、ある一時点における要因と疾病との関連性について有病率を用いて調べる疫学研究手法である。調査時点における観察であるため、要因曝露と疾病発生の時間的前後関係が不明であるため、因果関係は言及できない。
