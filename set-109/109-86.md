# 109-86

漢方薬と代表的な副作用の組合せとして正しいのはどれか。１つ選べ。

**正解:** ４

**解説**
小柴胡湯の副作用として間質性肺炎が起こり、死亡等の重篤な転帰に至ることがあるので、発熱、咳嗽、呼吸困難などが現れた場合には直ちに投与を中止する。さらに、インターフェロン製剤との併用で間質性肺炎が現れやすくなることがあるため、小柴胡湯とインターフェロン製剤は併用禁忌である。
