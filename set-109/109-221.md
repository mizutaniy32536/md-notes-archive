# 109-221

### 第109回薬剤師国家試験 問221

**問題文（リード文含む）**  
89歳女性。体重40kg。高血圧症及び慢性心不全に対して処方1で薬物治療を行っている。独居で入院拒否があるため、医師と薬剤師、看護師が訪問している。  
（中略：処方1の詳細、検査結果など。訪問医は継続中だった処方1のうち、エナラプリルのみを中止して、新たに**処方2**を追加した。）[1][2][4][5]  

**問221（実務）**  
処方2の薬剤の服用によって生じる可能性が高いのはどれか。**2つ選べ。**[2][4][5]  

**全選択肢（番号付き）**  
1. 血中TSHの上昇  
2. **高カリウム血症**  
3. 血圧上昇  
4. 血中NT-proBNPの上昇  
5. **脱水症状**[2][4][5]  

**正解番号**  
**2、5**[1][2]  

**解説**  
処方2の薬剤は**サクビトリルバルサルタンNa水和物（エンレスト）**であり、ネプリライシン阻害によるサクビトリルの作用とARB（バルサルタン）の作用が関与する。[1][2][3]  
- **選択肢2（高カリウム血症）**: バルサルタンによるレニン-アンギオテンシン-アルドステロン系（RAAS）阻害でアルドステロン分泌が低下し、低アルドステロン血症を招き、高カリウム血症の可能性が高い。[2]  
- **選択肢5（脱水症状）**: サクビトリル（ネプリライシン阻害）によりBNPなどのナトリウム利尿ペプチドの分解が抑制され、利尿促進作用が強まり、脱水症状の可能性が高い。[2]  
- **誤り（1: 血中TSHの上昇）**: TSH上昇との関連はなく、甲状腺機能への直接影響は報告されていない。[2]  
- **誤り（3: 血圧上昇）**: ARB成分により血圧低下が主作用で、上昇の可能性は低い。[2]  
- **誤り（4: 血中NT-proBNPの上昇）**: NT-proBNPはネプリライシンで分解されないため値は上昇せず、心不全治療でむしろ低下傾向。[2]  
サクビトリルバルサルタンは心不全治療薬として添付文書にこれらの副作用が記載されている。[3][6]
