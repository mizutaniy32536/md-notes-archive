# 109-237

# 第109回薬剤師国家試験 問237

## 問題文

この経口補水液に表示されているマークはどれか。**１つ選べ。**[1]

## 選択肢

1. 特定保健用食品のマーク[1]
2. 条件付き特定保健用食品のマーク[1]
3. JAS規格に合格した食品に表示されるマーク[1]
4. 特別用途食品のマーク[1]
5. 総合衛生管理（HACCP）厚生労働大臣の承認マーク[1]

## 正解番号

**4**[1][2]

## 解説

特別用途食品には、病者用食品、妊産婦・授乳婦用粉乳、幼児用調製乳、嚥下困難者用食品、特定保健用食品がある。特別用途食品のうち、**特定保健用食品を除く特別用途食品には、選択肢4（特別用途食品）のマークが表示される**。[1]

問題文で言及されている経口補水液は病者用食品（特別用途食品の一種）であるため、特別用途食品のマークが表示されることになる。[1][6]

**各選択肢の説明：**

- **選択肢1**：特定保健用食品のマーク（個別の許可マーク）[1]
- **選択肢2**：条件付き特定保健用食品のマーク（条件付きの許可マーク）[1]
- **選択肢3**：JAS規格に合格した食品に表示されるマーク（品質規格）[1]
- **選択肢5**：総合衛生管理（HACCP）厚生労働大臣の承認マーク（衛生管理方式）[1]
