## ◆ 食事により速度的バイオアベイラビリティのみ低下する薬物を選ぶシナリオ

260308

**問題**
- 経口投与する薬物について、食事により量的バイオアベイラビリティは変わらないが、速度的バイオアベイラビリティが低下するのはどれか。2つ選べ。

**選択肢**

1. アセトアミノフェン
​
2. セファクロル
​
3. リボフラビン
​
4. フェニトイン
​

**正解と理由**
- `2. セファクロル` と `3. リボフラビン`
- **理由の構成**
	- **問題の整理**：食事によって「量的BA（AUC）は変わらない」が「速度的BA（Cmax、Tmax）が低下」する薬物を2つ選ぶ問題
	- **薬学的知識・仕様・制約**：`セファクロル`は`PEPT1`（ペプチドトランスポーター）で吸収される。食後は食事中のペプチドとPEPT1上で競合するため吸収速度が低下するが、最終的な吸収量は変わらない。`リボフラビン`（ビタミンB₂）は`RFVT`（リボフラビントランスポーター）で吸収され、食後は胃内滞留が延長してトランスポーターに到達するのが遅れるため速度が低下するが量は変わらない
	- **選択理由**：いずれも特異的トランスポーターに依存して吸収されるため、食事の影響で吸収速度が低下するが、最終的な吸収量（AUC）には影響しない

**解法の手順（思考の流れ）**
- まず「量は変わらないが速度が低下」するメカニズムを考える。トランスポーターの`飽和`や食事成分との`競合`が想起される
- 各選択肢の吸収経路を確認：トランスポーター依存かどうかが判断基準
- `セファクロル`は`β-ラクタム系`抗菌薬で`PEPT1`基質であると判断 → 食事中ペプチドと競合
- `リボフラビン`は`RFVT`基質であると判断 → 食後は速度低下
- 他の選択肢は単純拡散や食事で量自体が変動するタイプと判断

**他の選択肢が誤りな理由**
- 1. アセトアミノフェン
	`単純拡散`で吸収される。食後は`胃内容排出速度`の低下により吸収速度が低下するが、これは一般的な食事の影響であり、特異的トランスポーターの飽和とは異なる。問題のポイントはトランスポーター介在の飽和型

- 4. フェニトイン
	`溶解度`が低い薬物であり、食後に溶解環境が変わるため、量的BAが変動する可能性がある。「量は変わらず速度だけ低下」のパターンに当てはまらない

- 5. インドメタシン ファルネシル
	`脂溶性プロドラッグ`であり、食後は胆汁分泌増加によりミセル形成が促進され、量的BAが`増大`する。速度だけでなく量も変わるため条件に合わない

**キーポイント**
- 「量は変わらないが速度が低下」→ 特異的`トランスポーター`介在の吸収で食事成分と`競合`するパターン
- `PEPT1`基質の代表例：`セファクロル`、セファレキシン、アモキシシリン、`バラシクロビル`
- `RFVT`基質：リボフラビン（吸収部位が上部小腸に限定、飽和性あり）

**関連知識の比較**
- 食事で「量的BAが増大」する薬物：`インドメタシンファルネシル`（脂溶性PD、ミセル促進）、`グリセオフルビン`（脂溶性、脂肪食で吸収増大）
- 食事で「量的BAが低下」する薬物：`アジスロマイシン`（食事により分解が促進）
- 食事で「速度的BAが低下」する薬物：`セファクロル`（PEPT1競合）、`リボフラビン`（RFVT飽和）、一般的に胃排出速度低下の影響を受ける薬物全般
- `アセトアミノフェン`は食後BA変動の「指標薬」として胃排出速度の評価に用いられることがある

（一言メモ）
- 「速度だけ下がって量は同じ」のメカニズムは`トランスポーター競合・飽和`。「PEPT1とRFVT」がキーワード
- `インドメタシンファルネシル`は「脂溶性プロドラッグ → 食後BA増大」のセットで覚える。ファルネシル基＝脂溶性↑の目印
- `セファクロル`と`セファレキシン`はともに`PEPT1`基質。セファ系＋PEPT1は頻出の組み合わせ
