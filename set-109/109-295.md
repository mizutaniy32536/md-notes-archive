# 109-295

# 第109回薬剤師国家試験 問295

## 問題文

1クール目Day8に発熱があり再来院した。処方1を服用中であり、全身倦怠感が強く入院加療となった。担当医から追加処方（処方2）の連絡を受けて、カンファレンスで今後の方針について協議することになった。

この患者の病態に関する記述のうち、**正しいのはどれか。2つ選べ。**[1][2][3]

## 全選択肢

1. 末梢における好中球の破壊が亢進している。
2. CRPが上昇している可能性が高い。
3. 口内炎や咽頭痛を発症しやすい。
4. 高度な貧血が認められる。
5. 薬剤に対するアレルギーにより発症したと考えられる。[1][2][3]

## 正解番号

**2、3**[1][2][3][7]

## 解説

**選択肢1：× 誤り**

本患者はFOLFIRINOX療法（化学療法）を受けているため、**骨髄での好中球産生抑制**が考えられます。末梢における好中球の破壊が亢進しているのではなく、骨髄で産生される好中球が減少しているため、選択肢1は誤りです。[1][2][6]

**選択肢2：◎ 正解**

患者は発熱と好中球の数が著しく減少しています。CRP（C反応性タンパク）は炎症や細胞破壊が起こると増加するマーカーであり、発熱性好中球減少症の状態では感染の兆候があるため、**CRPが上昇している可能性が高い**と考えられます。[2][3]

**選択肢3：◎ 正解**

好中球の著しい減少により、感染防御機能が低下します。その結果として、**口内炎や咽頭痛などの口腔内感染症が発症しやすくなります。**[1][2]

**選択肢4：× 誤り**

本問のケースでは高度な貧血は認められないと判断されます。[1]

**選択肢5：× 誤り**

本患者の発熱性好中球減少症はFOLFIRINOX療法（化学療法）による骨髄抑制に起因するものであり、**薬剤に対するアレルギーにより発症したのではありません。**[1][2]
