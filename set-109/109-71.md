# 109-71

### 第109回薬剤師国家試験 問71

**問題文**  
次のうち、医療における患者のQOLを考えるときに最も重要視すべきなのはどれか。1つ選べ。[1][2][6]

**全選択肢**  
1. 患者の人間としての尊厳  
2. 家族の意思  
3. 費用対効果  
4. 医療者の経済的利益  
5. 医学の進歩[1][2][3][6][9]

**正解番号**  
**1**[1][2][3]

**解説**  
QOL（Quality of Life：生活の質）を向上させるためには、患者の人間としての尊厳を最優先に考えることが必要です。これは患者の自己決定権や価値観を尊重し、患者中心のケアを提供する基盤となります。[1][2][3]  
家族の意思や費用対効果はQOLに影響する要素ですが、患者の尊厳より優先されません。医療者の経済的利益や医学の進歩は、患者のQOLを直接向上させるものではなく、場合によっては損なう可能性があります。[1][3]
