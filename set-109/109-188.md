# 109-188

### 第109回薬剤師国家試験 問188

**問題文（全文）**  
72歳男性。本態性高血圧症のために、処方1及び処方2を服用していた。最近、血圧が上昇したため、薬物による降圧療法を強化することになった。追加する治療薬として、最も適切なのはどれか。1つ選べ。[1][2][3]

**全選択肢（番号付き）**  
1. アムロジピンベシル酸塩  
2. アメジニウムメチル硫酸塩  
3. シベンゾリンコハク酸塩  
4. イルベサルタン  
5. ドキサゾシンメシル酸塩[1][2][3][4][5]

**正解番号**  
**5**（ドキサゾシンメシル酸塩）[1][2][4]

**解説**  
高齢者の本態性高血圧症に対し、既存の処方（検索結果からCa拮抗薬（ニフェジピン）やARB/ACE阻害薬、利尿薬などが含まれると推定）に追加する降圧薬を選択する問題。異なる作用機序の薬を追加するのが原則で、高齢男性では**α1遮断薬**が適切。[1][2]  

- 1. **×** アムロジピンはCa拮抗薬で、既存のニフェジピン（処方1）と同クラス。重複のため不適切。[1]  
- 2. **×** アメジニウムメチル硫酸塩は交感神経刺激薬で、降圧薬ではない。[2]  
- 3. **×** シベンゾリンは不整脈治療薬で、降圧療法に不適切。[1][2]  
- 4. **×** イルベサルタンはARBで、既存処方と同クラス（または類似）のため重複。[1][2]  
- 5. **〇** ドキサゾシンは**α1遮断薬**で、既存薬と作用機序が異なり、72歳男性の高血圧強化療法に適する（前立腺肥大症の可能性も考慮）。[1][2][4]  

高血圧ガイドラインでは、コントロール不十分時、異なるクラスの薬（例: Ca拮抗薬＋ARB＋利尿薬後、α遮断薬追加）を推奨。[5]
