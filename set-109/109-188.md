# 109-188

72歳男性。本態性高血圧症のために、処方1及び処方2を服用していた。最近、血圧が上昇したため、薬物による降圧療法を強化することになった。追加する治療薬として、最も適切なのはどれか。１つ選べ。

１ アムロジピンベシル酸塩
２ アメジニウムメチル硫酸塩
３ シベンゾリンコハク酸塩
４ イルベサルタン
５ ドキサゾシンメシル酸塩

**正解:** ５

**解説**
１　誤
アムロジピンベシル酸塩は、ジヒドロピリジン系Ca2＋チャネル遮断作用を有する降圧薬である。ただし、本患者に処方されているニフェジピン徐放錠と同種同効薬であるため、本剤の追加提案は不適切である。
２　誤
アメジニウムメチル硫酸塩は、交感神経機能亢進作用を有する昇圧薬である。そのため本剤の追加提案は不適切である。
３　誤
シベンゾリンコハク酸塩はVaughan Williams分類Ⅰa群に分類される抗不整脈薬である。そのため本剤の追加提案は不適切である。
４　誤
イルベサルタンは、アンギオテンシンAT1受容体遮断作用を有する降圧薬である。ただし、本患者に処方されているテルミサルタン錠と同種同効薬であるため、本剤の追加提案は不適切である。
５　正
ドキサゾシンメシル酸塩は、アドレナリンα1受容体遮断作用を有する降圧薬であり、本患者の降圧療法を強化するための追加提案は適切である。
