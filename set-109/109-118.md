# 109-118

炎症に関する記述のうち、正しいのはどれか。2つ選べ。
1 発赤は、炎症の兆候の一つで、赤血球が血管外に浸潤する現象である。
2 補体成分の C₃b と C₅b は、肥満細胞からヒスタミンを遊離させるアナフィラ
トキシンである。
3 P︲セレクチンは、血管内皮細胞の表面に発現し、白血球の炎症部位への動員に
関わる。
4 Toll 様受容体は、主に炎症後期に線維芽細胞の活性化に関わり組織修復を促
す。
5 炎症時には、肝臓での C 反応性タンパク質の産生が亢進する。
