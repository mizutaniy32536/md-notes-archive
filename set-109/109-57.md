# 109-57

### 第109回薬剤師国家試験 問57

**問題文**  
身体の一部又は全身に瞬間的な筋収縮が起こり、発作持続時間が短く、意識消失の自覚症状が無いことが多いてんかん発作はどれか。１つ選べ。[2][3][4]

**全選択肢**  
１ **複雑部分発作**  
２ **欠神発作**  
３ **強直間代発作**  
４ **ミオクロニー発作**  
５ **脱力発作**[2][3][4][9]

**正解番号**  
**４**[1][4]

**解説**  
- **４ 正（ミオクロニー発作）**: 全般発作であり、身体の一部又は全身に**瞬間的な筋収縮**が起こり、発作持続時間が短く、**意識消失の自覚症状が無い**ことが多い。[1][4]  
- **１ 誤（複雑部分発作）**: 意識消失を伴う部分発作で、自動症（無意識の動作）が認められる。[4]  
- **２ 誤（欠神発作）**: 全般発作で、痙攣を伴わず短時間の**意識障害**が起きる。[1][4]  
- **３ 誤（強直間代発作）**: 全般発作で、突然の**意識消失**後、強直性・間代性痙攣、昏睡が続く。[4]  
- **５ 誤（脱力発作）**: 全般発作で、突然の**瞬間的な脱力**（筋肉の弛緩）が起きる。[1][4]
