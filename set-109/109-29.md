# 109-29

アバタセプトの関節リウマチ治療効果発現の標的分子はどれか。１つ選べ。

１ CD28
２ 細胞傷害性Tリンパ球抗原−4（CTLA−4）
３ CD80/CD86
４ IL−6受容体
５ TNF−α

**解説**
