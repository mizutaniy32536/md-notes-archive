# 109-29

**問題文**
アバタセプトの関節リウマチ治療効果発現の標的分子はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-29 を参照

**選択肢**
1. CD28
2. 細胞傷害性Tリンパ球抗原−4（CTLA−4）
3. CD80/CD86
4. IL−6受容体
5. TNF−α

**正解:** 3

**解説**
解答　３

アバタセプトの関節リウマチ治療効果発現の標的分子は、CD80/CD86である。アバタセプトは、抗原提示細胞膜のCD80/CD86に結合し、CD28を介した共刺激シグナルを阻害することでT細胞の活性化を抑制する。
