# 109-243

### 第109回薬剤師国家試験 問243

#### 問題文
ある市立中学校で、線状降水帯による大雨のため床下浸水の被害が発生し、休校となった。この学校では、市から供給される水道水のみを水源とし、地下の受水槽に一旦貯めたのちに、高置水槽に揚水して給水栓に飲料水を供給している。学校を再開するにあたり、臨時検査として給水栓の飲料水の水質を検査することになり、学校薬剤師が以下の項目について検査を行った。検査項目：一般細菌、大腸菌、塩化物イオン、全有機炭素（TOC）の量、pH値、味、臭気、色度、濁度及び遊離残留塩素。この検査結果から、薬剤師が学校に報告する内容として適切なのはどれか。2つ選べ。[1][2][3][4][5]

#### 全選択肢（番号付き）
1. 臨時検査において一般細菌が検出されていることから、地下の受水槽に汚染された雨水などが流入したおそれがある。  
2. 塩化物イオン濃度と大腸菌の検査結果から、地下の受水槽がし尿で汚染されたおそれがある。  
3. （詳細未記載だが誤り）  
4. （詳細未記載だが誤り）  
5. （詳細未記載だが正解の一つ、学校環境衛生基準に基づく適切な報告内容）[1][2][4][5]

#### 正解番号
**1 と 5**[4]

#### 解説
- **選択肢1（正解）**: 臨時検査で一般細菌が検出されているため、地下の受水槽に汚染された雨水などが流入した可能性があると報告するのが適切。直近の検査で一般細菌が検出された事実に基づく。[1][2][4]  
- **選択肢2（誤り）**: 大腸菌は臨時検査で不検出のため、し尿汚染の可能性は低い。塩化物イオン濃度も基準内と推測され、し尿汚染を示唆しない。[1][2][4]  
- **選択肢5（正解）**: 学校環境衛生基準に基づき、検査結果全体から適切な対応（例: 受水槽清掃勧告など）を報告。[2][4]  
全体として、水道水源の飲料水基準（一般細菌検出は異常、大腸菌不検出は正常）を考慮し、雨水流入リスクを指摘するのが薬剤師の役割。[1][2][5]
