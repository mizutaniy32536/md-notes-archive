# 109-126

**問題文**
新生児マススクリーニングとその対象疾患に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-126 を参照

**選択肢**
1. 対象疾患は、後天性代謝異常症及び内分泌疾患である。
2. 全ての対象疾患のスクリーニングには、タンデムマス法が用いられている。
3. ガラクトース血症の発見率は、この10年間減少し続けている。
4. メープルシロップ尿症の治療には、分岐鎖アミノ酸制限ミルクの食事療法が行われる。
5. クレチン症（先天性甲状腺機能低下症）の治療には、ホルモン補充が行われる。

**正解:** 4,5

**解説**
解答　４、５

１　誤
新生児マススクリーニングの対象疾患は、フェニルケトン尿症やメープルシロップ尿症、ホモシスチン尿症、ガラクトース血症といった先天性代謝異常症と、先天性甲状腺機能低下症（クレチン症）や先天性副腎過形成症といった内分泌疾患である。

２　誤
新生児マススクリーニングの対象疾患のうち、アミノ酸代謝異常症や有機酸代謝異常症、脂肪酸代謝異常症はタンデムマス法によるスクリーニングが可能であるが、ガラクトース血症や先天性甲状腺機能低下症（クレチン症）、先天性副腎過形成症はタンデムマス法によるスクリーニングが不可能なため、別の方法を用いている。

３　誤
ガラクトース血症の発見率は、年度により変動するため、10年間減少し続けているわけではない。

４　正
メープルシロップ尿症は、分岐鎖α−ケト酸デヒドロゲナーゼの先天性異常により、分岐鎖アミノ酸に由来するα−ケト酸が分解されず、α−ケト酸及び分岐鎖アミノ酸が血中に増加することで精神運動発達の遅延を認める疾患である。そのため治療には、分岐鎖アミノ酸制限ミルクの食事療法が行われる。

５　正
クレチン症（先天性甲状腺機能低下症）は、甲状腺の形態または機能異常により甲状腺ホルモンの分泌不全が生じる疾患である。そのため治療には、甲状腺ホルモンの補充療法が行われる。
