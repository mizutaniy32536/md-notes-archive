# 109-33

**問題文**
メフルシドの利尿作用の機序はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-33 を参照

**選択肢**
1. 心房性ナトリウム利尿ペプチド（ANP）受容体刺激
2. バソプレシンV2受容体遮断
3. 上皮性Na＋チャネル遮断
4. Na＋−Cl－共輸送体阻害
5. Na＋−K＋−2Cl－共輸送体阻害

**正解:** 4

**解説**
解答　４

メフルシドは非チアジド系利尿薬であり、チアジド系利尿薬と化学構造は異なるが作用機序は類似しており、遠位尿細管のNa＋−Cl－共輸送体を阻害し、Na＋及び水の再吸収を抑制することで、利尿作用を示す。なお、ヘンレ係蹄上行脚のNa＋及び水の再吸収抑制作用も有する。
