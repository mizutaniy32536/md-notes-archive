# 109-65

手術が適応とならないクッシング症候群の治療に用いる薬物はどれか。１つ選べ。

１ チアマゾール
２ ミトタン
３ デスモプレシン酢酸塩水和物
４ レボチロキシンナトリウム水和物
５ メチロシン

**正解:** ２

**解説**
クッシング症候群は慢性的なコルチゾールの過剰分泌を起こす疾患で、下垂体腺腫が原因となるクッシング病、小細胞がんなどの異所性ACTH産生腫瘍が原因となる異所性ACTH症候群、副腎皮質腺腫、副腎癌、副腎皮質過形成などに分類される。
治療としては、原則手術による副腎や下垂体の腫瘍の摘出が行われるが、手術不能例では副腎皮質ホルモン合成阻害薬（ミトタン、メチラポン、トリロスタン）によりコルチゾール産生抑制を行う。
