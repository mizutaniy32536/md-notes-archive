# 109-211

### 第109回薬剤師国家試験 問211

#### 問題文
43歳男性。身長170cm、体重75kg。双極性障害で処方1の薬剤を服用していた。抑うつ症状が再燃してきたため、今回、処方2が追加された処方箋と以下の検査値が記載された情報用紙を持って患者が来局した。[1][2][3][4][5]

この処方に関し、医師へ疑義照会する内容として、適切なのはどれか。1つ選べ。[1][2][3][4][5]

#### 全選択肢
1. バルプロ酸Na徐放錠200mgの投与量を、1日200mgに減量する。[1][2][3][4][5]  
2. バルプロ酸Na徐放錠200mgの投与量を、1日1200mgに増量する。[1][2][3][4][5]  
3. ラモトリギン錠の投与量を、1日50mgに増量する。[2][3][4][5]  
4. ラモトリギン錠の用法を、朝食後2時間以降投与に変更する。[3][4][5]  
5. ラモトリギン錠の用法を、隔日投与にする。[3][4]

#### 正解番号
**5** [1][2][3][4][9]

#### 解説
ラモトリギン（ラミクタール）は双極性障害の維持療法に用いられるが、バルプロ酸Na併用時は血中濃度が上昇し、重篤な皮膚障害（スティーブンス・ジョンソン症候群など）のリスクが高まるため、投与開始時は1日25mgを1日1回（毎日投与）から始め、増量間隔を厳守する必要がある。[2][3][5]  
処方2でラモトリギンが追加された場合、隔日投与は用法・用量に反し、疑義照会すべき内容である。[1][2][3][4]  
他の選択肢は不適切：1は血清濃度が治療域のため減量不要、2は濃度が治療域で増量による肝障害リスク高、3はバルプロ酸併用時の増量規定違反、4は投与タイミング変更不要。[1][2][3]
