# 109-23

**問題文**
以下の生体組織のうち、放射線に対する感受性が最も高いのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-23 を参照

**選択肢**
1. 神経組織
2. 造血組織
3. 皮膚組織
4. 筋肉組織
5. 脂肪組織

**正解:** 2

**解説**
解答　２

放射線に対する感受性は、①細胞分裂の頻度が高く、②将来行う細胞分裂の数が多く、③形態、機能が未分化な生体組織ほど高くなる（ベルゴニー・トリボンドーの法則）。選択肢のうち、放射線に対する感受性が最も高いのは、細胞分裂の頻度が高く、将来行う細胞分裂の数が多い造血組織である。
