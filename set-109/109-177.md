# 109-177

**問題文**
抗悪性腫瘍薬であるイリノテカンはプロドラッグである。この薬物の製剤と代謝に関する以下の問に答えよ。問177イリノテカン塩酸塩水和物を有効成分とするオニバイド点滴静注に関する記述のうち、正しいのはどれか。2つ選べ。なお、本製剤は以下の添加物を含む。添加物1,2−ジステアロイル−sn−グリセロ−3−ホスホコリン、コレステロール、N−(カルボニル−メトキシポリエチレングリコール−2000)−1,2−ジステアロイル−sn−グリセロ−3−ホスホエタノールアミンナトリウム塩、4−(2−ヒドロキシエチル)−1−ピペラジンエタンスルホン酸

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-177 を参照

**選択肢**
1. イリノテカンをポリエチレングリコール誘導体で可溶化した製剤である。
2. ポリエチレングリコールで表面が修飾されたリポソーム製剤である。
3. 能動的ターゲティングにより腫瘍に集積する。
4. 細網内皮系を回避することにより血中に滞留する。
5. 製剤中に含まれる微粒子は150 µm程度である。問178イリノテカンは酵素（ア）によって活性代謝物（イ）に変換され、さらに酵素（ウ）によって代謝物（エ）に変換され不活化される。（ア）の酵素及び（イ）の構造式、（ウ）の酵素及び（エ）の構造式のそれぞれの組合せとして、正しいのはどれか。1つ選べ。

**正解:** 2,4

**解説**
問177　解答　２、４

オニバイド点滴静注は、ポリエチレングリコール（PEG）で表面が修飾されたリポソーム製剤である。イリノテカンをPEG化したリポソームベースのナノ粒子に封入することで、血中滞留時間を延長し、EPR効果（血管透過性及び滞留性亢進）に伴う腫瘍細胞への集積を増加させ、腫瘍細胞内でのイリノテカンの活性代謝物であるSN−38の暴露期間の延長による抗腫瘍活性を増強することを目的として開発された製剤である。

１　誤
解説４参照。PEG化の目的は可溶化ではなく、血中滞留時間の延長である。

２　正
オニバイド点滴静注は、PEGで表面が修飾されたリポソーム製剤である。

３　誤
受動的ターゲティングにより腫瘍に集積する。EPR効果は正常細胞と腫瘍細胞の特性の違いを利用したターゲティングであり、受動的ターゲティングに分類される。

４　正
オニバイド点滴静注は、PEGで表面が修飾されていることにより、通常のリポソーム製剤と比較すると細胞内皮系を回避することで血中滞留性が向上している。

５　誤
リポソームはナノ粒子であり、製剤中に含まれる微粒子は100 nm程度である。

問178　解答　３

イリノテカンはプロドラッグであり、体内でカルボキシルエステラーゼにより活性代謝物であるSN−38（A）に変換され抗腫瘍作用を示す。その後、UDP−グルクロン酸転移酵素（UGT1A1）により不活性代謝物であるSN−38G（B）に変換され排泄される。
