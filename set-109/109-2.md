# 109-2

**問題文**
SDSポリアクリルアミドゲル電気泳動を行った後、タンパク質の染色に用いる最も適切な化合物はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-2 を参照

**選択肢**
1. ニンヒドリン
2. 臭化エチジウム
3. o−フタルアルデヒド
4. フルオレセインイソチオシアネート
5. クーマシーブリリアントブルー（クマシーブリリアントブルー）

**正解:** 5

**解説**
解答　５

SDSポリアクリルアミドゲル電気泳動（SDS−PAGE）を行った後のタンパク質の染色に用いる主な化合物は、クーマシーブリリアントブルー（クマシーブリリアントブルー）である。

１　誤
ニンヒドリンは、アミノ酸やタンパク質の染色などに用いられるが、SDS−PAGEを行った後のタンパク質の染色に用いることは一般的ではない。

２　誤
臭化エチジウムは、アガロースゲル電気泳動を行った後の核酸の染色に用いられる。

３　誤
o−フタルアルデヒドは、液体クロマトグラフィーにおけるアミノ酸の染色に用いられる。

４　誤
フルオレセインイソチオシアネートは、核酸、抗体、タンパク質などの蛍光標識に用いられる。

５　正
