# 109-139

**問題文**
一般環境大気測定局及び自動車排出ガス測定局における大気汚染物質の測定法に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-139 を参照

**選択肢**
1. 一酸化炭素は、照射した赤外線の吸収量に基づいて測定される。
2. 二酸化窒素は、紫外線の照射によって励起した二酸化窒素分子が発する蛍光の強度に基づいて測定される。
3. 光化学オキシダントは、ザルツマン試薬との反応により生じる生成物の吸光度に基づいて測定される。
4. 二酸化硫黄は、エチレンとの反応により生じる近紫外線領域の発光の強度に基づいて測定される。
5. 浮遊粒子状物質は、ろ紙上に粒子を捕集して、β線を照射し、その透過量に基づいて測定される。

**正解:** 1,5

**解説**
解答　１、５

１　正
一酸化炭素の測定には赤外線吸収法が用いられており、照射した赤外線の吸収量に基づいて測定される。

２　誤
一酸化窒素、二酸化窒素の測定には、ザルツマン法や化学発光法が用いられる。化学発光法では、一酸化窒素とオゾンを反応させ、生成する励起した二酸化窒素分子が発する蛍光の強度に基づいて測定される。なお、二酸化窒素を測定する場合は、二酸化窒素を一酸化窒素に還元してから同様の方法で測定する。

３　誤
光化学オキシダントであるオゾンの測定には中性ヨウ化カリウム法、化学発光法が用いられ、ペルオキシアシルナイトレート（PAN）の測定には中性ヨウ化カリウム法が用いられる。なお、一酸化窒素、二酸化窒素の測定にはザルツマン法が用いられており、二酸化窒素とザルツマン試薬との反応により生じる生成物の吸光度に基づいて測定される。ただし、一酸化窒素はザルツマン試薬と反応しないため、過マンガン酸カリウムで二酸化窒素に酸化させてから同様の方法で測定する。

４　誤
二酸化硫黄の測定には、溶液導電率法、トリエタノールアミン・パラロザ二リン法などが用いられる。なお、オゾンの測定には化学発光法が用いられており、エチレンとの反応により生じる近紫外線領域の発光の強度に基づいて測定される。

５　正
浮遊粒子状物質の測定には、重量法やβ線吸収法が用いられる。β線吸収法では、ろ紙上に浮遊粒子状物質を捕集して、β線を照射し、その透過量から浮遊粒子状物質の質量を測定する。
