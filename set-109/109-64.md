# 109-64

尿路結石を有する痛風患者に対して、最も適切な尿酸降下薬はどれか。１つ選べ。

１ ベンズブロマロン
２ コルヒチン
３ ドチヌラド
４ トピロキソスタット
５ プロベネシド

**正解:** ４

**解説**
尿酸排泄促進薬（ベンズブロマロン、ドチヌラド、プロベネシド）は、尿中への尿酸排泄を増加させるため尿路結石を誘発しやすくなる。そのため、尿路結石を有する痛風患者に対しては、尿酸排泄促進薬は使用せず、トピロキソスタットなどの尿酸生成抑制薬を用いる。なお、コルヒチンは、痛風発作の緩解および予防に用いられ、尿酸効果作用はもたない。
