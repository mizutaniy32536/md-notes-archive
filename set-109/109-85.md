# 109-85

**問題文**
生後3ヶ月の乳児に対する薬の使い方として適切なのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-85 を参照

**選択肢**
1. 散剤を熱湯で溶かしてから飲ませる。
2. 散剤をハチミツに混ぜて飲ませる。
3. 散剤に数滴の水を加えてペースト状にし、頬の内側か上あごに塗りつけ、すぐに水を飲ませる。
4. 散剤とシロップ剤が一緒に出ている時は、翌日分をあらかじめ混ぜて作り置きしておく。
5. 坐剤は冷凍庫に保管し、冷えた状態で使用する。

**正解:** 3

**解説**
解答　３

１　誤
熱湯により成分変質のリスクがあるため不適切である。

２　誤
1歳未満の乳児はハチミツを摂取することによって乳児ボツリヌス症のリスクがあるため不適切である。

３　正

４　誤
配合変化による含量低下などを招くリスクがあるため不適切である。

５　誤
坐剤を冷えた状態で使用すると腸を刺激するリスクがあるため、挿入前に室温に戻す、または手で少し温めてから使用することが好ましい。
