# 109-275

# 第109回薬剤師国家試験 問275

## 問題文

34歳既婚女性。高血圧症、逆流性食道炎治療中。かかりつけ薬局の薬剤師がいつもの薬をお渡しする際に副作用の発現状況などを確認していると、患者は半年前に結婚し、そろそろ子供が欲しいと思うようになったとのことであったが、患者背景については、薬歴が更新されていなかった。

**今後の薬物療法にあたり、妊娠の可能性があるこの患者に説明しておく情報はどれか。__2つ__選べ。**[1][3]

## 全選択肢

1. 血圧コントロールのため、生活習慣や食習慣に気をつける。
2. 現在服用中の血圧の薬は妊娠中でも服用可能である。
3. 妊娠がわかったら、すぐに連絡してもらう。
4. 逆流性食道炎の治療薬を市販の六君子湯などの漢方薬に変更する。
5. 下剤は、一般用医薬品であれば、妊娠中でも服用可能である。

## 正解番号

**1、3**[1][3]

## 解説

妊娠の可能性がある患者に対する薬剤師の説明内容として適切なのは、以下の2つです：

**選択肢1が正解の理由：** 妊娠前から血圧を適切にコントロールすることは、妊娠中の高血圧管理の基礎となるため、生活習慣や食習慣の改善について説明することは重要です。

**選択肢3が正解の理由：** 妊娠が判明した場合、現在服用中の医薬品の妊娠への影響を評価する必要があるため、患者が妊娠したことを直ちに薬剤師に報告することは必須です。

**その他の選択肢が不正解の理由：**
- 選択肢2：ACE阻害薬など一部の高血圧薬は妊娠中（特に妊娠後期）に使用すべきではないため、「妊娠中でも服用可能」という説明は誤りです。
- 選択肢4：逆流性食道炎の治療薬を勝手に変更することは適切ではなく、医師との相談が必要です。
- 選択肢5：下剤の中には妊娠中に避けるべきものもあるため、「一般用医薬品であれば服用可能」という説明は不正確です。
