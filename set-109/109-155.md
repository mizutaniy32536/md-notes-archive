# 109-155

# 第109回薬剤師国家試験 問155

## 問題文

この患者の症状改善を目的として使用される可能性のある薬物に関する記述のうち、正しいのはどれか。**2つ選べ。**[1][2]

## 選択肢

1. ブロモクリプチンは、線条体においてドパミンD2受容体を刺激することで、間接路のGABA作動性神経を抑制する。[1]

2. ゾニサミドは、グルタミン酸AMPA受容体を刺激することで、抗パーキンソン作用を示す。[1]

3. セレギリンは、モノアミン酸化酵素（MAO-B）を阻害することで、ドパミン分解を抑制する。[1]

4. 「選択肢4の内容」[記載なし]

5. クロナゼパム（リボトリール）は、GABA_A受容体のベンゾジアゼピン結合部位に結合し、GABA神経系の活動性を高める。[2]

## 正解

**1、5**[1][2]

## 解説

**選択肢1（正答）**：ブロモクリプチンはドパミン受容体刺激薬であり、麦角アルカロイドの一種です。パーキンソン病治療薬として使用され、線条体においてドパミンD2受容体を刺激することで、間接路のGABA作動性神経を抑制する作用があります。[1][2]

**選択肢2（誤答）**：ゾニサミドについては、AMPA受容体刺激ではなく、別の機序で作用します。[1]

**選択肢3（誤答）**：セレギリンはMAO-B阻害作用を持ち、ドパミン分解を抑制する作用がありますが、本問では正答ではありません。[5]

**選択肢5（正答）**：クロナゼパム（リボトリール）はベンゾジアゼピン系の薬物であり、GABA_A受容体のベンゾジアゼピン結合部位に結合して、GABA神経系の活動性を高めます。これにより、REM睡眠行動異常などの症状改善に用いられます。[2]
