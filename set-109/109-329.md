# 109-329

**問題文**
72歳男性。体重60 kg。10年前に大学病院の腎臓内科にてネフローゼ症候群と診断されプレドニゾロン錠とフロセミド錠による治療が開始された。昨年ステロイド抵抗性との診断を受けてシクロスポリンカプセルが追加され、治療中である。某年8月9日に近隣の皮膚科を受診したところ、爪白癬との診断を受け、イトラコナゾールの投与を推奨された。同日、大学病院の腎臓内科よりイトラコナゾールカプセルの処方が追加され、最終的に処方1〜4のように調整された。9月6日の診察にて経過良好とのことから、治療は同じ処方で継続された。＜image1:center＞（9月6日の検査値）血清クレアチニン1.3 mg/dL、eGFR 42.7 mL/min/1.73 m2、血清アルブミン3.1 g/dL、尿蛋白（±）、下肢の浮腫（±）本日9月13日、昨夕より四肢のむくみと尿の泡立ちがひどくなったと自覚したため、かかりつけ薬局へ相談に訪れた。対応した薬剤師は患者からの聞き取りに加え、9月6日の検査値を確認した。（聞き取り内容）・処方1のお薬は以前と変わらず飲んでいる。・処方2のお薬は全て残っている。・処方3のお薬は残っていない。・処方4のお薬は全て残っている。今回の患者の主訴である四肢のむくみと尿の泡立ちについて、腎臓内科医に提案する内容として適切なのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-329 を参照

**選択肢**
1. シクロスポリンの血中濃度測定
2. イトラコナゾールの血中濃度測定
3. 処方1の中止
4. 処方2及び処方4の開始
5. 処方4の開始

**正解:** 1,5

**解説**
解答　１、５

本患者は、ネフローゼ症候群に対してプレドニゾロン錠による治療を行っているが、ステロイド抵抗性のためシクロスポリンカプセルを併用している。また、爪白癬治療のためにイトラコナゾールの投与も開始した。シクロスポリンはCYP3A4により代謝され、イトラコナゾールはCYP3A4阻害作用をもつことから、処方2,3の用量は、イトラコナゾール併用によるシクロスポリンの血中薬物濃度上昇を考慮し、シクロスポリン単独投与の処方4に比べ、低用量で処方されている。
9月13日の本患者からの自覚症状と聞き取り内容より、9月6日に処方された薬剤のうち、イトラコナゾールを服用せず、低用量のシクロスポリンのみを服用していることから、今回の主訴の原因は、シクロスポリンの効果が不十分であることが考えられる。よって薬剤師が腎臓内科医に提案する内容として適切なのは、シクロスポリンの血中濃度測定と処方4の開始である。なお、全て残っている処方2と処方4は、同時に開始すると相互作用によりシクロスポリンの血中濃度が上がりすぎて副作用発現のリスクが上昇するため、現時点で処方2の投与は開始しない。
