# 109-259

### 第109回薬剤師国家試験 問259

#### 問題文
25歳女性。身長153cm、体重40kg。19歳のときにクローン病と診断され、メサラジンとアザチオプリンによる併用療法を実施していたが、効果不十分のため1年前よりアダリムマブ(遺伝子組換え)皮下注が追加となった。2週間前より、発熱、腹痛及び下痢があり検査目的で入院となった。内視鏡検査の結果、症状が悪化していることが分かり、アダリムマブが以下の**処方1**に変更されるとともに、**処方2**が追加された。[1][2][4][6][8]

**問259**  
クローン病の症状寛解を目的とした薬物の作用機序のうち、今回までに処方されてきたのとは異なるのはどれか。**2つ選べ**。[1][2][6]

#### 全選択肢（番号付き）
1. リンパ球表面に発現する**α4β7インテグリン**に結合することで、リンパ球の腸管粘膜への浸潤を阻害する。  
2. TNF-αに結合してその作用を阻害する。  
3. IL-12およびIL-23のp40サブユニットに結合してその作用を阻害する。  
4. プリン合成酵素を阻害する。  
5. 炎症性サイトカインの転写を抑制する。[1][2][4][6]

#### 正解番号
**1, 5**[1][2][4]

#### 解説
- この患者はこれまで**メサラジン**、**アザチオプリン**、**アダリムマブ**を処方されていた。[1][2][4]  
- **選択肢1**: **ベドリズマブ**（処方1）の作用機序（α4β7インテグリン阻害）。これまで処方されていなかったため正解。[1][2][4][6]  
- **選択肢2**: **アダリムマブ**の作用機序（TNF-α阻害）。これまで処方されていたため誤り。[1][2]  
- **選択肢3**: **ウステキヌマブ**の作用機序（IL-12/23 p40阻害）。これまで処方されていたため誤り（注: 検索結果で一部言及あり）。[1][2]  
- **選択肢4**: **アザチオプリン**の作用機序（プリン合成酵素阻害）。これまで処方されていたため誤り。[1][2]  
- **選択肢5**: **ステロイド**（処方2）の作用機序（炎症性サイトカイン転写抑制）。これまで処方されていなかったため正解。[1][2][4]
