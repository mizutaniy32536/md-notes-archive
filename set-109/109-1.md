# 109-1

永久双極子モーメントをもつ分子はどれか。１つ選べ。

１ ベンゼン
２ メタン
３ 二酸化炭素
４ 水
５ 四塩化炭素

**正解:** ４

**解説**
永久双極子モーメントとは、分子内部での電荷の偏りの程度を表す。
なお、分子内の一部分に電荷の偏りがあっても、その電荷を打ち消しあう構造があれば分子全体で永久双極子モーメントをもたないことがある。
１　誤
ベンゼンは正六角形構造であり、分子内で電荷を打ち消しあうので永久双極子モーメントをもたない。
２　誤
メタンは正四面体構造であり、分子内で電荷を打ち消しあうので永久双極子モーメントをもたない。
３　誤
二酸化炭素は直線形構造であり、分子内で電荷を打ち消しあうので永久双極子モーメントをもたない。
４　正
水は折れ線形構造であり、永久双極子モーメントをもつ。
５　誤
四塩化炭素は正四面体構造であり、分子内で電荷を打ち消しあうので永久双極子モーメントをもたない。
