# 109-213

### 第109回薬剤師国家試験 問213

**問題文**  
カペシタビンは、段階的にフルオロウラシルに代謝されるプロドラッグである。体内におけるカペシタビンの代謝を示した下図に関する記述のうち、正しいのはどれか。２つ選べ。[3][4][5]

**選択肢**  
1. カペシタビンの点線で囲った構造は、分子の疎水性を高める。  
2. Aは二酸化炭素（CO₂）である。  
3. Bの糖部はD-リボースである。  
4. Cの点線で囲った酸素原子は、水に溶けている酸素分子（O₂）に由来する。  
5. 加リン酸分解で生じるDの構造は（図示省略）。[3][4][5]

**正解番号**  
**1, 2**[1][2][4][5]

**解説**  
- **1 正しい**：カペシタビンの点線で囲った構造（ペンチルオキシカルボニル基）は疎水性が高く、分子全体の疎水性を高め、水溶性を低下させて経口吸収を向上させるプロドラッグ設計である。[1][2][5]  
- **2 正しい**：代謝過程でAは二酸化炭素（CO₂）として放出される。[1][4][5]  
- **3 誤り**：Bの糖部は5'-デオキシ-D-リボースであり、D-リボース（5位にOH基がある）とは異なる。[1][2][5]  
- **4 誤り**：Cの点線で囲った酸素原子は水分子の酸素原子に由来し、溶けている酸素分子（O₂）ではない。[1]  
- **5 誤り**：加リン酸分解で生じるDの構造にはリン酸（P）が含まれるはずだが、選択肢の構造にPが見当たらない。[1]  

複数の解説源で正解が**1, 2**と一致し、図の構造解析に基づく。[1][2][4][5]
