# 109-309

（実務） 病院薬剤師の説明を受けて、患児及び患児の両親はバイオ後続品を使用すること にした。この成長ホルモン製剤の説明として適切なのはどれか。2つ選べ。
1 毎日通院の上、医師または看護師に注射してもらってください。
2 用量を決めるためにはお子さんの体重の情報が必須です。
3 このお薬の使用中は、生ワクチンの接種はできません。
4 このお薬を始めることによって、夜尿症（おねしょ）になります。
5 定期的に血糖や HbAlc の検査が必要となります。

## 正解
**2 と 5**

## 解説

### 成長ホルモン製剤の基本

**適応：**
- 成長ホルモン分泌不全性低身長症
- ターナー症候群
- プラダー・ウィリ症候群
- SGA性低身長症など

**投与方法：**
- **皮下注射（毎日1回、就寝前）**
- **在宅自己注射が可能**
- ペン型注射器を使用

**用量設定：**
- **体重1kgあたり○○mg**で計算
- 定期的に体重測定して用量調整

### 各選択肢の検証

#### 1. ❌ 毎日通院は不要
**在宅自己注射が可能**
- 毎日病院に通うのは非現実的（学校もあるし）
- インスリンと同様、自己注射指導を受けて自宅で注射
- ペン型注射器で簡便に投与可能

#### 2. ⭕ 用量は体重で決める
**体重1kgあたりで用量を計算**
- 例：ソマトロピン 0.175mg/kg/週 を週7回に分割
- 体重が増えれば用量も増やす必要がある
- 定期的な体重測定が必須

#### 3. ❌ 生ワクチン接種は可能
**成長ホルモンは免疫抑制作用なし**
- 生ワクチンが禁忌なのは：
  - 免疫抑制剤
  - 抗がん剤
  - ステロイド大量投与
- 成長ホルモンは該当しない

#### 4. ❌ 夜尿症は副作用ではない
成長ホルモンの主な副作用：
- **耐糖能異常（血糖↑）** ← 重要
- 甲状腺機能低下
- 頭蓋内圧亢進
- 関節痛、筋肉痛
- 浮腫

夜尿症（おねしょ）は副作用ではない

#### 5. ⭕ 血糖・HbA1cの検査が必要
**成長ホルモンはインスリン拮抗作用がある**
- 血糖値を上昇させる
- 耐糖能異常、糖尿病のリスク
- **定期的な血糖・HbA1cのモニタリングが必須**
- 特に肥満、糖尿病家族歴がある場合は注意

### 成長ホルモンの作用

| 作用 | 効果 | 臨床的意義 |
|---|---|---|
| 成長促進 | 骨・筋肉の成長 | 低身長症の治療 |
| タンパク質合成↑ | 筋肉量増加 | 同化作用 |
| 脂肪分解↑ | 体脂肪減少 | 痩身効果 |
| **血糖↑** | **インスリン拮抗** | **耐糖能異常のリスク** |

### 患者指導のポイント
- 在宅自己注射の手技指導
- 定期的な体重測定
- 血糖モニタリングの重要性
- 副作用の早期発見（頭痛、視力障害など）
