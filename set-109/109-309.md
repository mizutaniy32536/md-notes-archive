# 109-309

### 第109回薬剤師国家試験 問309

**問題文（リード文含む）**  
13歳男性。身長135cm、体重40kg。近隣の公立病院脳神経外科にて頭蓋咽頭腫の摘出術を受けたものの、血中下垂体ホルモン濃度の異常は改善せず、汎下垂体機能低下症と診断され、いくつかのホルモン補充療法の一つとして成長ホルモン補充療法を行うこととなった。  
処方医より「ソマトロピン(遺伝子組換え)製剤を使用したいが、患児及び患児の両親が先行バイオ医薬品とバイオ後続品(バイオシミラー)のどちらを選択するかを判断できるように説明してほしい。」との依頼を受け、病院薬剤師から先行バイオ医薬品とバイオ後続品の相違について説明を行うこととなった。  
病院薬剤師の説明を受けて、患児及び患児の両親はバイオ後続品を使用することにした。この成長ホルモン製剤の説明として適切なのはどれか。2つ選べ。[2][3]

**全選択肢**  
1. 毎日通院の上、医師または看護師に注射してもらってください。  
2. 用量を決めるためにはお子さんの体重の情報が必須です。  
3. このお薬の使用中は、生ワクチン接種を控えてください。  
4. このお薬を始めることによって、夜尿症 (おねしょ) になります。  
5. 定期的に血糖や HbA1c の検査が必要となります。[1][2][9]

**正解番号**  
2, 5[1][2]

**解説**  
- **選択肢1**: 誤り。自己注射する製剤であり、通院は不要。[1]  
- **選択肢2**: 正しい。体重あたりで用量が設定されるため、体重情報が必須。[1][2]  
- **選択肢3**: 不適切（詳細不明だが、標準的な成長ホルモン製剤の説明として選ばれていない）。[2]  
- **選択肢4**: 誤り（問308関連で夜尿症は副作用として関連しない情報）。[1]  
- **選択肢5**: 正しい。成長ホルモン製剤使用中は血糖上昇の可能性があり、定期的な血糖・HbA1c検査が必要。[1]  
ソマトロピン(遺伝子組換え)製剤（バイオ後続品）は皮下注射で自宅自己投与可能で、体重換算投与が標準。用量調整と血糖モニタリングが重要。[1][2][3][9]
