# 109-133

ヒ素に関する記述のうち、正しいのはどれか。２つ選べ。

１ 無機ヒ素化合物の毒性は、5価のヒ素の方が、3価のヒ素より強い。
２ アルセノベタインは、無機ヒ素と比べて急性毒性が強い。
３ ヒ素及び無機ヒ素化合物は、国際がん研究機関（IARC）の発がん性分類グループ1（ヒトに対する発がん性が認められる）に分類されている。
４ 無機ヒ素化合物による急性中毒時には、解毒薬として亜硝酸アミルが用いられる。
５ 無機ヒ素化合物は、地殻中に広く分布し、汚染された地下水の飲用による健康被害が世界的に問題となっている。

**正解:** ３、５

**解説**
１　誤
無機ヒ素化合物の毒性は、3価のヒ素（亜ヒ酸）の方が、5価のヒ素（ヒ酸）より強い。
２　誤
アルセノベタインは、エビなどの甲殻類で合成される無機ヒ素のメチル体であり、低毒性の化合物である。なお、アルセノベタインはヒト体内では合成できない。
３　正
国際がん研究機関（IARC）は主にヒトに対する発がん性に関する様々な物質・要因（作用因子）を評価し、4つのグループに分類している。そのうち、グループ1（ヒトに対する発がん性が認められる）に分類される有害金属は、ヒ素及び無機ヒ素化合物やカドミウム及びカドミウム化合物、六価クロム化合物、ニッケル化合物である。
４　誤
無機ヒ素化合物による急性中毒時には、解毒薬としてジメルカプロール（BAL）が用いられる。なお、亜硝酸アミルは、シアン中毒の解毒薬として用いられる。
５　正
無機ヒ素化合物は、地殻中に広く分布しているため、自然由来のヒ素が地下水などを汚染することで健康被害が生じることがある。
