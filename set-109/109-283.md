# 109-283

### 第109回薬剤師国家試験 問283

#### 問題文
患児の母親への服薬指導にあたって、事前に処方3の**ツロブテロールテープ剤**に関する医薬品情報を調べたところ、有効成分及び製剤に関して、以下の情報が得られた。（問題文中の具体的な情報、図、グラフは省略形で記載。実際の問題ではインタビューフォームに基づく半減期3.56時間、小児データ、血清中濃度推移のグラフ選択が含まれる）。  
ツロブテロールテープ剤を患児に単回貼付して24時間後に剥離した場合の血清中薬物濃度の時間推移として、最も妥当なのはどれか。[1][2][5][6]

#### 全選択肢（番号付き）
1. （血中濃度が貼付後徐々に上昇し、4時間後にピーク、以降急速に低下するグラフパターン）[1][6]  
2. （血中濃度が貼付後徐々に上昇し、約12時間でピーク、その後徐々に低下し24時間後に剥離するグラフパターン）[1][6]  
3. （血中濃度が貼付後急速に上昇し、4時間後にピーク、以降低下するグラフパターン）[1][6]  
4. （血中濃度が貼付後上昇し24時間後にピーク、剥離後12時間でさらに増加する不自然なグラフパターン）[1][6]  
5. （血中濃度が貼付後上昇し、4時間後に低下開始、24時間後に剥離するグラフパターン）[1][6]

#### 正解番号
**2**[1][2][8]

#### 解説
ツロブテロールテープ剤は24時間持続放出製剤で、貼付後血中濃度が徐々に上昇し約12時間でピークに達した後、24時間後に剥離しても半減期3.56時間のため急激に低下せず安定推移する。選択肢1,3,5は剥離後4時間で血中濃度が1/3程度に低下する点が誤り（半減期計算上不適切）。選択肢4は24時間後にピークまたは剥離後12時間の不自然な増加が製剤特性に反する。[1][2][6]  
正解の2は、貼付後上昇→12時間ピーク→24時間剥離後の緩やか低下がテープ剤の制御徐放性と一致する。[1][6]
