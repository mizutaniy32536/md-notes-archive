# 109-6

**問題文**
同圧下で沸点が最も高いのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-6 を参照

**選択肢**
1. CH3CH2CH2CH3
2. CH3CH2CH2CH2CH3
3. (CH3)2CHCH3
4. (CH3)2CHCH2CH3
5. (CH3)4C

**正解:** 2

**解説**
解答　２

アルカンの沸点はファンデルワールス力が大きくなるに従って高くなる。
・分子量
ファンデルワールス力は一般に分子量が大きいほど大きい。
今回の場合、沸点の大小関係は炭素数５（分子量C5H12=72）の選択肢２、４、５＞炭素数４（分子量C4H10=58）の選択肢１、３となる。
・分子の形
ファンデルワールス力は２つの分子が接触する表面積が大きいほど大きい。
直鎖アルカンと分岐鎖アルカンを比較すると、直鎖アルカンの方が表面積が大きいのでファンデルワールス力が大きくなり、沸点が高くなる。
すなわち炭素数５の選択肢２、４、５のうち沸点が最も高いのは選択肢２となる。

なお、沸点はそれぞれ選択肢１（ブタン）：－0.5℃、選択肢２（ペンタン）：36.1℃、選択肢３（2−メチルプロパン）：－11.7℃、選択肢４（2−メチルブタン）：27.8℃、選択肢５（2,2−ジメチルプロパン）：9.5℃である。
