# 109-20

**問題文**
ホルムアルデヒドや有機酸の殺菌・静菌作用によって食品の保存性を高める方法はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-20 を参照

**選択肢**
1. 乾燥
2. 塩蔵
3. くん煙
4. 冷蔵
5. 糖漬

**正解:** 3

**解説**
解答　３

１　誤
乾燥は、水分活性を低下させることで食品の保存性を高める方法である。

２　誤
塩蔵（塩漬）は、水分活性を低下させることで食品の保存性を高める方法である。

３　正
くん煙は、煙に含まれるホルムアルデヒドや有機酸による殺菌・静菌作用により食品の保存性を高める方法である。

４　誤
冷蔵は、低温にて微生物の活動性を低下させることで食品の保存性を高める方法である。

５　誤
糖漬は、水分活性を低下させることで食品の保存性を高める方法である。
