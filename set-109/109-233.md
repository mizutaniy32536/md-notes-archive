# 109-233

### 第109回薬剤師国家試験 問233

**問題文（共通ステム含む）**  
乳児（生後1ケ月、女児）を連れた母親が、女児の1ケ月検診を受けた後に、女児が服用している薬について相談したいことがあるということで、かかりつけ薬局に来局した。  
女児の出生時の体重は2,940gであり、母乳栄養で現在は3,890gである。  
女児は、メナテトレノンシロップ1mLを哺乳確立時、生後1週目（産科退院時）、今回と合計3回内服している。母親は、女児へのメナテトレノン投与の理由について薬剤師に質問をした。  

**問233（衛生）**  
薬剤師は母親にメナテトレノン投与によって、あるビタミンを補充できることを説明した。  
このビタミンが新生児や幼若乳児で不足する理由として、誤っているのはどれか。1つ選べ。[1][2][3][4]

1. 胎盤を通過しにくいため。[2][3][4]  
2. 母乳中の含有量が少ないため。[2][3]  
3. 胆汁分泌が低下している場合には、吸収されにくいため。[1][2]  
4. **新生児や乳児は代謝及び排泄が亢進しているため。**[2][3]  
5. 新生児や乳児は、腸内細菌叢が未発達であるため。[1][2][3]

**正解番号：4**[1][2][3]

**解説**  
メナテトレノン（ビタミンK<sub>2</sub>）は新生児出血予防に用いられ、ビタミンKを補充する。ビタミンKが新生児・幼若乳児で不足する主な理由は以下の通り（正しいもの）：  
- 胎盤を通過しにくい（分子の脂溶性が高いため胎盤通過が不十分）。[3]  
- 母乳中の含有量が少ない。[2]  
- 胆汁分泌が低下（例: 胆道閉鎖）すると吸収不良。[1][2]  
- 腸内細菌叢が未熟でビタミンK産生菌が少ない。[1][2]  

**4は誤り**。新生児・乳児のビタミンK代謝・排泄は亢進しておらず、不足の主因ではない。[2][3]
