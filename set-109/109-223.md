# 109-223

### 第109回薬剤師国家試験 問223

#### 問題文（共通ステム）
65歳女性。高血圧症、骨粗しょう症、慢性便秘症及び脂質異常症の治療でかかりつけ薬局を利用し、処方1及び処方2の薬剤を2年間服用している。さらに、脂質異常症については食事療法の効果が不十分で、処方3を2ヶ月前に開始した。  
昨夜から全身の筋肉痛が生じているため、一般用医薬品の痛み止めを求めて来局した。昨日は、日課としている約2kmの散歩しかしておらず、普段はその程度の運動で筋肉痛は起こらないと言う。また、胃が弱いので内服ではなく、テレビコマーシャルで見たロキソプロフェン含有テープ剤の購入を希望している。[1][4][7]

（注: 処方3にフェノフィブラートが含まれ、横紋筋融解症の疑いがある文脈。[1]）

#### 問223（物理・化学・生物）
この患者の**筋肉の状態**を確認するために注目すべき生体成分として、適切なのはどれか。  
**2つ選べ。**  
1. クレアチンキナーゼ  
2. アルブミン  
3. 乳酸デヒドロゲナーゼ  
4. 総コレステロール  
5. アミラーゼ[1][6][10]

#### 正解番号
**1, 3**[1]

#### 解説
フェノフィブラートによる横紋筋融解症が疑われる場合、筋肉の状態を確認するために注目すべき生体成分として、**クレアチンキナーゼ（CK）**と**乳酸デヒドロゲナーゼ（LDH）**、およびミオグロビンが重要である。**クレアチンキナーゼ**は筋肉に多く含まれる転移酵素で、筋損傷時に血中濃度が上昇する。**乳酸デヒドロゲナーゼ**も筋肉由来の酵素で、横紋筋融解症の指標となる。  
選択肢2のアルブミンは肝機能・栄養状態に関連し、4の総コレステロールは脂質異常症の指標、5のアミラーゼは膵臓・唾液腺関連のため不適切。[1][6][10]
