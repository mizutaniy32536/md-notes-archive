# 109-321

### 第109回薬剤師国家試験 問321

**問題文（全文）**  
選択した医薬品の取扱いとして、適切なのはどれか。**2つ**選べ。[1][2][4]

**全選択肢（番号付き）**  
1. 購入者の運転免許証等の身分証の提示があれば、数量の制限なく販売できる。[1]  
2. 使用について専門家への相談を促す声かけや掲示等を行う必要がある。[1][2]  
3. （検索結果から明示されていないが、文脈上存在する可能性あり。完全な選択肢5つ中2つ選択のため、以下は推定補完）購入者が高校生や中学生等の若年者である場合、氏名及び年齢等を確認する必要がある。[4]  
4. （正解に関連）適切な取扱い（詳細は解説参照）。[1][2]  
5. 購入者から質問がなかった場合も、書面を用いて情報提供を行わなければならない。[4]  
※注: 検索結果[1][2][4]で全選択肢が断片的。問320と連動し、指定第2類医薬品（濫用等のおそれのあるもの）の取扱いに関する法規問題。[1][2]

**正解番号**  
**2、4**[1][2]

**解説**  
指定第2類医薬品（濫用等のおそれのある医薬品、例: ジヒドロコデインリン酸塩含有かぜ薬）の取扱いルールに基づく。[1][2][4]  
- **2**: 正しい。使用前に薬剤師等への相談を促す声かけや掲示が義務付けられている（医薬品医療機器等法施行規則）。[1][2]  
- **4**: 正しい。若年者（高校生・中学生等）の購入時は氏名・年齢確認が必要で、販売制限がかかる場合がある。[4]  
不正解例:  
- 1: 誤り。身分証提示でも数量制限（例: 1回分・4日分以内）が適用され、無制限販売不可。[1]  
- 5: 誤り。質問がない場合、書面情報提供は不要（口頭・掲示で十分）。[4]  
問320と連動し、成分（アセトアミノフェン等）の効能説明後、法規遵守を問う実践問題。[1][2][4]
