# 109-228

**問題文**
70歳男性と60歳女性の夫婦。2人で、打たせ湯のある温泉施設に日帰り旅行へ行ったところ、翌日夜に女性は全身倦怠感と頭痛を伴う発熱の症状を呈したが、自宅療養し、2日後に回復した。一方、男性は旅行から帰った5日後に発熱し、呼吸困難感、湿性咳嗽と胸痛の症状が見られたため、医療機関を受診したところ、肺炎と診断され入院となった。男性は身長165 cm、体重53 kg。入院時の各種検査結果は以下のとおりである。さらに、この男性が罹患している疾病に関して、尿中抗原検査及び喀痰の遺伝子増幅法検査を追加で施行した結果、いずれも陽性となった。夫婦は同じ病原体に罹患し、女性はポンティアック熱であったと考えられた。（身体所見）体温38.2℃、血圧101/62 mmHg、脈拍95拍/分（整）、SpO2（ルームエアー）93%（検査所見）白血球12,000 /µL、赤血球440×104 /µL、Hb 14.3 g/dL、血小板20×104 /µL、AST 25 IU/L、ALT 20 IU/L、総ビリルビン0.8 mg/dL、血清クレアチニン2.0 mg/dL、CRP 18.2 mg/dL、SARS−CoV2−PCR（鼻咽頭ぬぐい液）陰性、尿中肺炎球菌抗原検査陰性問228（実務）この男性患者に対する治療薬として最も適切なのはどれか。1つ選べ。

**正解:** 1

**解説**
問228　解答　１

温泉施設に行ったのちに発熱、呼吸困難感、湿性咳嗽、胸痛の症状が見られ、肺炎と診断されたことから、この男性はレジオネラ症に罹患したと考えられる。レジオネラ症の治療には、レボフロキサシンなどのニューキノロン系抗菌薬やアジスロマイシンなどのマクロライド系抗菌薬が用いられる。なお、レジオネラ症には肺炎型（レジオネラ肺炎）と非肺炎型（ポンティアック熱）の病型がある。肺炎型は、潜伏期が2〜10日程度であり、全身倦怠感、頭痛、食欲不振、筋肉痛などの症状に始まり、咳や38℃以上の高熱、寒気、胸痛、呼吸困難などがみられ重症化しやすい。一方、非肺炎型は、潜伏期間が数時間から48時間程度であり、発熱、悪寒、筋肉痛などの症状がみられるが、これらの症状は一過性であり自然に治癒することが多い。

問229　解答　２、３

１　誤
感染症法において、レジオネラ症は四類感染症に分類される。

２　正
レジオネラ症は、日和見感染症であり、高齢者や免疫力が低下した人などは発症のリスクが高い。

３　正
レジオネラ症は、レジオネラ属菌に汚染された空調の冷却塔水、加湿器、循環式浴槽などから発生するエアロゾルを吸入することで発症することから、公衆浴場などで集団発生を起こしやすい。

４　誤
レジオネラ症は、レジオネラ属菌に汚染されたエアロゾルの吸入により発症し、ヒトからヒトへ直接感染しない。

５　誤
現在、レジオネラ症に対するワクチンは実用化されていない。
