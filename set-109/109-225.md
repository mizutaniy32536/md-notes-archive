# 109-225

### 第109回薬剤師国家試験 問225

**問題文（全文）**  
3歳5ヶ月女児。台所で沸かしていたやかんのお湯をかぶり、体表の熱傷(Ⅱ度)で救急搬送された。入院14日目に39℃の発熱を認め、熱傷創部からセフェム系抗菌薬に感受性のある「A」（問224で緑膿菌と同定）が同定されたため、セフタジジムを2週間投与し、症状が改善したため終了した。抗菌薬終了7日後に、再び38℃の発熱及び熱傷創部に膿を認めた。膿を検査したところ、【A】（緑膿菌）が同定された。検出された【A】の薬剤感受性試験及び患者の情報は以下のとおりである（なお、この患者はアモキシシリン水和物に対するアレルギー歴がある）。この患者の検査結果を受けて、医師と感染制御チーム (ICT) の薬剤師が協議した。薬剤師が医師に提案する抗菌薬として、適切なのはどれか。**2 つ選べ。**[1][2][3][4][5]

**全選択肢（番号付き）**  
1. シプロフロキサシン  
2. メロペネム  
3. アミカシン  
4. セフタジジム  
5. タゾバクタム・ピペラシリン[1][2][4][5]

**正解番号**  
2, 3[1][2][5]

**解説**  
患者は熱傷創部から**緑膿菌**（好気性グラム陰性菌）が同定され、薬剤感受性試験でメロペネムとアミカシンが感性、耐性としてシプロフロキサシン、セフタジジムが耐性を示した。また、βラクタム系（アモキシシリン）アレルギー歴があるため、タゾバクタム・ピペラシリン（ペニシリン系）は不適切。[1][2][3]  
- **1. シプロフロキサシン**: 薬剤感受性試験で耐性。[1][2]  
- **2. メロペネム**: 広域スペクトルのカルバペネム系で、緑膿菌に有効。感受性あり。[1][2]  
- **3. アミカシン**: アミノグリコシド系で緑膿菌に有効（静菌的）。感受性あり。[1][2][7]  
- **4. セフタジジム**: 過去投与歴があり耐性獲得。[1][2]  
- **5. タゾバクタム・ピペラシリン**: βラクタム系（ペニシリン系）でアレルギー歴から不適切。[1][2]  
問224でAを緑膿菌と特定した症例の継続問題。[1][3]
