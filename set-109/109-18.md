# 109-18

**問題文**
事業者が、常時従事する労働者に対して特殊健康診断を受診させることが義務付けられているのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-18 を参照

**選択肢**
1. 石綿（アスベスト）の粉じんを発散する場所における業務
2. 重量物を取り扱う業務
3. 著しく暑熱な場所における業務
4. 電子計算機への入力を反復して行う業務
5. 著しい騒音を発する場所における業務

**正解:** 1

**解説**
解答　１

特殊健康診断とは、有害な化学物質や物理的因子に曝露されている作業者を対象に、健康影響を早期発見して職業病を予防するために実施されるのものであり、具体的には、①特定の有機溶剤業務、②鉛業務、③四アルキル鉛業務、④特定化学物質の製造・取扱業務、⑤高圧室内業務又は潜水業務、⑥放射線業務、⑦石綿（アスベスト）業務、⑧粉じん業務、⑨除染業務、⑩歯に有害な物質を扱う業務に従事する作業者に対して原則、6ヶ月に一度の実施が義務付けられている。
