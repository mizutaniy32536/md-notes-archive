# 109-291

### 第109回薬剤師国家試験 問291

**問題文（全文）**  
79歳女性。3年前、階段を昇る時に息切れを感じるようになり受診したところ、胸部レントゲン写真で心拡大と肺うっ血を認め、心エコー検査等の結果、慢性心不全と診断された。処方1と処方2で治療されていたが、慢性心不全の増悪により入院した。その後、処方1を処方3へ変更し目立った副作用なく病態が安定した。医師は検査値等を確認し、さらに**処方4**を退院時に追加し退院することになった。退院時の検査値等は以下のとおりである。[1][2][3][5]

**問291**  
追加された**処方4**の薬剤を服用するにあたり、この患者自身が特に注意する副作用症状として、適切なのはどれか。**2つ選べ**。[1][2][3]

**全選択肢**  
1. **ふらつき**  
2. **陰部搔痒感**  
3. **歯肉肥厚**  
4. **喀痰**  
5. **難聴**[1][2][3]

**正解番号**  
1, 2[1][2]

**解説**  
追加された処方4の薬剤は**ダパグリフロジンプロピレングリコール水和物（SGLT2阻害薬、フォシーガ）**である。この薬剤の主な副作用として、低血糖による**ふらつき（1）**と、尿路生殖器感染症による**陰部搔痒感（2）**に注意が必要である。高齢者であるため、低血糖リスクが高く、患者自身がふらつきを感じたら医師に相談するよう指導する。陰部搔痒感はカンジダ症などの感染症を示唆し、清潔保持を促す。また、脱水によるめまいも関連するが、選択肢ではふらつきが該当。他の選択肢（歯肉肥厚：カルシウム拮抗薬関連、喀痰・難聴：無関係）は不適切。[1][2][10]
