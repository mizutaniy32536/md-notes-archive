# 109-61

心電図上でQT間隔の延長により生じやすくなる不整脈はどれか。１つ選べ。

１ 洞停止
２ 洞房ブロック
３ 心房細動
４ 発作性上室頻拍
５ 多形性心室頻拍

**正解:** ５

**解説**
心筋活動電位時間の延長によりQT間隔が延長するものをQT延長症候群と呼び、Torsades de Pointes（トルサード・ド・ポアンツ）という特殊な多形性心室頻拍を生じるのが特徴的である。トルサード・ド・ポアンツは心臓のポンプ機能を著しく低下させ、脳への血流が低下することでめまいや意識消失を繰り返し引き起こす。
