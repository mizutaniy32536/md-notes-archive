# 109-257

### 第109回薬剤師国家試験 問257

**問題文（全文）**  
52歳女性。40歳時に気管支ぜん息と診断された。現在、処方1～5の薬剤を服用しており、アドヒアランスは良好であるが、ぜん息発作で頻回に入院していた。今回、ぜん息発作のコントロール目的で入院となり、処方6が追加された。  

処方1～6のいずれかの薬物の作用機序として、正しいのはどれか。2つ選べ。[1][2][4]

**処方内容（関連部分）**  
- 処方1: サルメテロール/フルチカゾンプロピオン酸エステル（注1: 1回サルメテロールとして50μg及びフルチカゾンプロピオン酸エステルとして500μg）[1]  
- 処方2: スピリーバ2.5μgレスピマット（注2: 60吸入 1本）[1]  
- 処方3: テオフィリン（詳細省略）[4]  
- 処方4: モンテルカスト（キプレス®・シングレア®）：ロイコトリエン受容体拮抗薬[1]  
- 処方5: プロカテロール（メプチン®）：β₂作動薬[1]  
- 処方6: オマリズマブ（遺伝子組換え）皮下注150mgシリンジ 3本（1回450mg 11:00に皮下投与）[1][2]

**全選択肢（番号付き）**  
1. トロンボキサン合成酵素を阻害することで、気管支平滑筋の収縮を抑制する。[2][4]  
2. アデノシン受容体に対して拮抗薬として作用する。[4]  
3. IL-5に結合して、IL-5が好酸球の細胞膜上のIL-5受容体に結合するのを阻害することで、血中の好酸球数を減少させる。[2]  
4. ロイコトリエン受容体を阻害することで、気管支平滑筋の収縮を抑制する。[1][4]  
5. （詳細未記載だが、誤りとしてβ₂作動薬関連の不適切記述と推定）[1][2]

**正解番号**  
2と4[1][2][4]

**解説**  
- **選択肢1（誤り）**: トロンボキサン合成酵素阻害薬（例: オザグレル）は処方1～6のいずれにも該当せず、気管支平滑筋収縮抑制の記述も不正確。[1][2][4]  
- **選択肢2（正解）**: 処方3のテオフィリンはアデノシン受容体拮抗薬として気管支拡張作用を示す。[4]  
- **選択肢3（誤り）**: IL-5結合による好酸球減少は処方1～6のいずれの薬物（オマリズマブはIgEに結合）にも該当しない。[2]  
- **選択肢4（正解）**: 処方4のモンテルカストはロイコトリエン受容体拮抗薬として気管支平滑筋収縮を抑制する。[1][4]  
- **選択肢5（誤り）**: β₂作動薬（処方5のプロカテロール）関連の記述が不正確で、該当なし。[1][2]  
オマリズマブ（処方6）は血清中総IgE濃度と体重に基づく投与量設定が問256の論点。[2][4]
