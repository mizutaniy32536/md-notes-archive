# 109-199

### 第109回薬剤師国家試験 問199

**問題文（全文）**  
65歳女性。身長155cm。パート勤務。糖尿病性腎症が進行し、1年前から血液透析が導入された。透析後目標体重（ドライウエイト）43kg、透析間体重増加量平均2.0kg。  
透析効率を評価した結果、ドライウエイトの上方修正を行った。しかし、その後も透析後の起立時に収縮期血圧が25mmHg以上低下しており、症状の改善もみられなかった。  
したがって、医師は透析後の血圧低下を予防するために薬物を投与することにした。予防投与される薬物として、最も適切なのはどれか。1つ選べ。[2][3][4]

**全選択肢（番号付き）**  
1. イソプレナリン塩酸塩  
2. アデノシン三リン酸二ナトリウム水和物  
3. ジゴキシン  
4. **ドロキシドパ**  
5. ニトログリセリン[2][3][4]

**正解番号**  
4[2][3][4]

**解説**  
ドロキシドパは、中枢神経系に作用して血圧を上昇させる薬剤であり、透析後の低血圧（起立性低血圧）を予防するために使用される。正解のドロキシドパは、立ちくらみやふらつきなどの症状を改善するのに適する。他の選択肢は不適切：イソプレナリンは心刺激薬、アデノシンは徐脈誘発、ジゴキシンは心不全治療、ニトログリセリンは血管拡張による血圧低下を助長する可能性がある。[4][7]
