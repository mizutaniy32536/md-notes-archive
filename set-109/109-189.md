# 109-189

**問題文**
間質性肺炎に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-189 を参照

**選択肢**
1. 閉塞性換気障害のために肺活量が減少する。
2. 原因として最も頻度が高いのは、副腎皮質ステロイド性薬の副作用である。
3. 乾性咳嗽が出現する。
4. 胸部聴診で呼息時にブツブツという粗い断続性ラ音が聴取される。
5. 特発性肺線維症に対しては、ニンテダニブエタンスルホン酸塩を投与することがある。

**正解:** 3,5

**解説**
解答　３、５

１　誤
間質性肺炎は、肺間質（肺胞隔壁）に炎症や線維化病変を認める拘束性換気障害であり、肺活量が減少する。

２　誤
副腎皮質ステロイド性薬は、間質性肺炎の治療に用いられる。なお、間質性肺炎は、原因不明の特発性間質性肺炎と薬剤や放射線治療が原因となる二次性間質性肺炎に分類される。

３　正
間質性肺炎の主な症状として、乾性咳嗽、呼吸困難、発熱などがある。

４　誤
間質性肺炎の胸部聴診所見として、吸気時に「バリバリ」といった断続性ラ音（捻髪音）が認められる。なお、吸気時に「ブツブツ」といった断続性ラ音（水泡音）が認められるのは肺水腫などである。

５　正
ニンテダニブエタンスルホン酸塩は、特発性肺線維症、全身性強皮症を伴う間質性肺炎、進行性線維化を伴う間質性肺炎に適応を有するチロシンキナーゼ阻害薬である。
