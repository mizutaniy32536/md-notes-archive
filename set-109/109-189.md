# 109-189

間質性肺炎に関する記述のうち、正しいのはどれか。2つ選べ。
1 閉塞性換気障害のために肺活量が減少する。
2 原因として最も頻度が高いのは、副腎皮質ステロイド性薬の副作用である。
3 乾性咳嗽が出現する。
4 胸部聴診で呼息時にブツブツという粗い断続性ラ音が聴取される。
5 特発性肺線維症に対しては、ニンテダニブエタンスルホン酸塩を投与すること
がある。
