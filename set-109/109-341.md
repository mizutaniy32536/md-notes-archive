# 109-341

**問題文**
38歳男性。体重80 kg。数年前より動悸及び息切れを自覚し、最近では歯茎からの出血や不意の鼻血などで不安になったため近医を受診した。引き続き近医より紹介された大学病院にて骨髄穿刺を受けた。病理検査の結果、骨髄異形成症候群と診断され、全身放射線照射に引き続き無菌病室で骨髄幹細胞移植治療を受けた。移植後の移植片対宿主病及び真菌感染症の予防目的で以下の薬物が投与開始された。なお、タクロリムスの血中濃度は15 ng/mLを目標とされた。投与開始後2週間目にタクロリムス、ボリコナゾールは経口投与に変更となった。骨髄幹細胞移植を実施した後の病棟担当薬剤師によるアセスメントとして適切なのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-341 を参照

**選択肢**
1. 投与開始直後の発熱に備えて頓用のイブプロフェンを準備する。
2. タクロリムス水和物注射液は1 mL/hの速度で投与する。
3. ボリコナゾールは血中濃度を参考に用量を調節する。
4. 移植片対宿主病を疑う所見が現れた場合はシクロスポリン注射液の追加を考慮する。
5. 移植片対宿主病の発症を認めなくてもタクロリムスの投与は生涯必要である。

**正解:** 2,3

**解説**
解答　２、３

１　誤
本患者は、全身放射線照射により全ての造血幹細胞を破壊したのちに正常な造血幹細胞を移植するため、免疫低下による感染症のリスクが高い状態にある。従って、発熱があった場合は、頓用のイブプロフェンではなく、感染症を考慮した抗菌薬の投与などが考えられる。

２　正
処方2より、体重80 kgの本患者へのタクロリムスの1日投与量（mg/日）は、0.03 mg/kg/日×80 kg＝2.4 mg/日である。生理食塩液を用いて濃度が0.1 mg/mLになるように調製しているため、タクロリムスの1日投与量（mL/日）は、2.4 mg/日÷0.1 mg/mL＝24 mL/日である。よって、タクロリムスの1時間あたりの投与量（mL/h）は、24 mL/日＝24 mL/24h＝1 mL/hである。

３　正
ボリコナゾールは血中薬物モニタリング（TDM）対象薬剤であるため、血中濃度を参考に用量を調節することは適切である。

４　誤
シクロスポリン注射液は、タクロリムスを投与している患者に禁忌であるため、本患者にシクロスポリン注射液の追加を考慮することは不適切である。なお、移植片対宿主病（GVHD）を疑う所見が現れた場合、軽度の場合はステロイドの局所投与など、重度の場合はステロイドの全身投与などGVHDの程度に応じた処置が行われる。

５　誤
移植片対宿主病（GVHD）の発症を認めない場合、タクロリムスを徐々に減量し、移植後約6ヶ月頃に投与を中止することができる。
