# 109-83

**問題文**
アフタ性口内炎に用いられる一般用医薬品に含まれる有効成分はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-83 を参照

**選択肢**
1. フルオシノロンアセトニド
2. トリアムシノロンアセトニド
3. ベタメタゾン吉草酸エステル
4. デキサメタゾン酢酸エステル
5. ヒドロコルチゾン

**正解:** 2

**解説**
解答　２

アフタ性口内炎に用いられる一般用医薬品に含まれる有効成分は、トリアムシノロンアセトニドである。アフタ性口内炎（潰瘍性口内炎）は、ストレスや疲れによる免疫力の低下、睡眠不足、栄養不足（ビタミンB2欠乏）などにより起こるとされている。
なお、他の選択肢は、湿疹などの皮膚症状に用いられる一般用医薬品に含まれる有効成分である。
