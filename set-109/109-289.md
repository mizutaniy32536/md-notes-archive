# 109-289

# 第109回薬剤師国家試験 問289

## 問題文

治療方針が変更されて2週間後、患者が来院し、診察前の薬剤師との面談で「左腰部から背部に沿ってかゆみがあって、昨日から痛み出した」と訴えた。薬剤師は、患者情報を収集して副作用を疑い、医師へ情報提供を行った。医師は診察後バラシクロビルを処方した。

**薬剤師が副作用を疑うきっかけになった患者情報はどれか。2つ選べ。**[1][2]

## 選択肢

1. メトトレキサートの服用状況
2. 追加した治療薬の処方内容
3. 年齢と性別
4. 水痘ワクチンの接種歴
5. 肝機能、腎機能

## 正解

**1、2**[1][2][3]

## 解説

患者が訴えた「左腰部から背部に沿ってかゆみがあって、昨日から痛み出した」という症状は、**帯状疱疹**の特徴的な症状です。神経に沿って広がり、多くは片側に発症することが帯状疱疹の典型的な臨床像です。[2]

医師がバラシクロビル（抗ウイルス薬）を処方したことからも、帯状疱疹と診断されたことが示唆されます。

副作用を疑うきっかけになった患者情報は以下の通りです：

**選択肢1「メトトレキサートの服用状況」：○**
メトトレキサートは免疫抑制剤の一つであり、免疫機能を低下させるため、帯状疱疹のリスク因子となります。[3]

**選択肢2「追加した治療薬の処方内容」：○**
治療方針の変更時に追加された治療薬（生物学的製剤など免疫抑制作用のある薬物）も免疫機能を低下させ、帯状疱疹の発症リスクを高めます。[3]

**選択肢3「年齢と性別」：×**
年齢と性別のみでは帯状疱疹発症の直接的な根拠にはなりません。

**選択肢4「水痘ワクチンの接種歴」：×**
接種歴の有無よりも、現在の免疫抑制状態が重要です。

**選択肢5「肝機能、腎機能」：×**
肝機能・腎機能検査値は帯状疱疹発症の直接的な原因ではありません。
