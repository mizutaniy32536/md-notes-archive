# 109-124

**問題文**
下図の曲線ア〜エは、15歳未満、15〜64歳、65歳以上、75歳以上のいずれかの年齢区分別人口の割合の推移を、右の数値は2022年における各年齢区分別人口の割合を示している。年齢区分別人口に関する記述のうち、正しいのはどれか。2つ選べ。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-124 を参照

**選択肢**
1. アは生産年齢人口の割合を示している。
2. イは65歳以上の老年人口の割合を示している。
3. アとウの和は、従属人口の割合である。
4. イをエで割った値に100を乗じた値は、老年人口指数である。
5. 2022年における年少人口指数は、およそ19.5である。

**正解:** 1,5

**解説**
解答　１、５

総務省統計局が実施する国勢調査では、15歳未満の人口割合を年少人口割合（イ）、15〜64歳の人口割合を生産年齢人口割合（ア）、65歳以上の人口割合を老年人口割合（ウ）としている。なお、エは75歳以上老年人口割合を示している。

１　正
前記参照

２　誤
前記参照

３　誤
従属人口割合は、イ（年少人口割合）とウ（老年人口割合）の和である。

４　誤
老年人口指数は、ウ（老年人口割合）をア（生産年齢人口）で割った値に100を乗じることで求められる。

５　正
年少人口指数は、年少人口割合を生産人口割合で割った値に100を乗じることで求められる。よって、2022年における年少人口指数は、(11.6÷59.4)×100 ≒19.5である。
