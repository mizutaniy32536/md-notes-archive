# 109-156

**問題文**
花粉症の治療に用いられる薬物に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-156 を参照

**選択肢**
1. レボセチリジンは、知覚神経のヒスタミンH1受容体を遮断し、くしゃみを抑制する。
2. クロモグリク酸は、ロイコトリエンCysLT1受容体を遮断し、眼のかゆみを軽減する。
3. ラマトロバンは、プロスタノイドTP受容体及びプロスタノイドDP2受容体（CRTH2受容体）を遮断し、鼻粘膜の炎症を軽減する。
4. ナファゾリンは、鼻粘膜のアドレナリンα1受容体を遮断することで血管平滑筋を弛緩させ、鼻閉を軽減する。
5. トラニラストは、ヤヌスキナーゼ（JAK）を阻害することで炎症細胞の活性化を抑制し、結膜の充血を軽減する。

**正解:** 1,3

**解説**
解答　１、３

１　正
レボセチリジンは、第二世代抗ヒスタミン薬であり、知覚神経のヒスタミンH1受容体を遮断し、くしゃみを抑制する。

２　誤
クロモグリク酸は、肥満細胞等からのケミカルメディエーターの遊離を抑制し、眼のかゆみを軽減する。なお、ロイコトリエンCysLT1受容体を遮断し気管支喘息などの治療に用いる薬は、モンテルカストなどである。

３　正
ラマトロバンは、鼻粘膜血管のプロスタノイドTP受容体遮断による血管透過性亢進に対する抑制作用、及び好酸球のプロスタ
ノイドDP2受容体（CRTH2受容体）遮断による炎症細胞の遊走に対する抑制作用により、鼻粘膜の炎症を軽減する。

４　誤
ナファゾリンは、鼻粘膜のアドレナリンα1受容体を刺激することで血管平滑筋を収縮させ、鼻閉を軽減する。

５　誤
トラニラストは、肥満細胞等からのケミカルメディエーターの遊離を抑制し、結膜の充血を軽減する。なお、ヤヌスキナーゼ（JAK）を阻害することで炎症やかゆみを抑制しアトピー性皮膚炎などの治療に用いる薬は、デルゴシチニブである。
