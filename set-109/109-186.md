# 109-186

骨粗しょう症に関する記述のうち、正しいのはどれか。２つ選べ。

１ 閉経後は、骨吸収の相対的な低下により、骨量の減少をきたす。
２ 典型的なX線所見として、頭蓋骨の打ち抜き像がある。
３ 骨代謝マーカーは、骨折リスクの予測に有用である。
４ 閉経後骨粗しょう症の治療には、エストロゲンの補充療法を行う。
５ 選択的エストロゲン受容体モジュレーター（SERM）は、乳腺や子宮内膜に対してエストロゲン様作用を示す。

**正解:** ３、４

**解説**
１　誤
閉経後骨粗しょう症では、エストロゲンの分泌低下に伴い、相対的に骨吸収が亢進することにより、骨量の減少をきたす。
２　誤
典型的なX線所見として、頭蓋骨の打ち抜き像を認めるのは、多発性骨髄腫である。なお、骨粗しょう症のX線所見では、骨折に伴う骨変形などを認める。
３　正
骨代謝マーカーは、一般的に骨粗しょう症の治療方針や治療効果の判定、骨折リスクの予測などに用いられる。
４　正
閉経後骨粗しょう症の治療には、エストロゲンの補充療法や、選択的エストロゲン受容体モジュレーター（SERM）であるラロキシフェン、バゼドキシフェンなどが用いられる。
５　誤
選択的エストロゲン受容体モジュレーター（SERM）は、骨に対してはエストロゲン様作用を示すが、乳腺や子宮内膜に対してはエストロゲン様作用を示さない。そのため、エストロゲン製剤に比べ、乳がんや子宮がんの発症リスクが低い。
