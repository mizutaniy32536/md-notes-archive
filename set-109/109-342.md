# 109-342

来局者の訴える症状及び現在の服用薬、それに対して薬剤師が選択した一般用漢方製剤として適切なのはどれか。２つ選べ。

**正解:** ２、４

**解説**
１　誤
「風邪のひき始めで寒気がする」という症状には、葛根湯などの提案が考えられる。なお、防風通聖散は、「食べ過ぎや飲み過ぎで太ってしまった」「お通じが悪い」などの症状に用いられる。
２　正
「アレルギー性の鼻炎」という症状には、小青竜湯などの提案が考えられる。
３　誤
「疲れやすく、食欲不振である」という症状には、補中益気湯などの提案が考えられる。しかし、補中益気湯にはグリチルリチン酸を含む甘草（カンゾウ）が含まれているため、すでに他のグリチルリチン含有製剤を服用している患者に補中益気湯を提案することは、偽アルドステロン症の発症リスクが高くなるため不適切である。
４　正
「排尿が困難であり、いつも残尿感がある」という症状には、八味地黄丸などの提案が考えられる。
５　誤
「夜就寝中に足がつる」という症状には、芍薬甘草湯などの提案が考えられる。なお、当帰芍薬散は、「冷え症でむくみやすい」「貧血の傾向があり顔色が悪い」「生理不順がある」などの症状に用いられる。
