# 109-184

図1に示すニフェジピン徐放錠を8分割又は分割せずに、1錠分をヒトに経口投与したときの血漿中ニフェジピン濃度の推移を図2に示す。この錠剤に関する記述のうち、正しいのはどれか。２つ選べ。

１ スパンタブ型の錠剤である。
２ 8分割した錠剤の血中濃度推移はAである
３ 外層部は徐放性マトリックスである。
４ 内核錠は徐放性コーティングされている。
５ 錠剤全体が腸溶性コーティングされている。

**正解:** ２、３

**解説**
本剤はアダラートCR錠（ニフェジピン徐放錠）であり、消化管液の豊富な小腸上部でゆっくりニフェジピンが溶出する外層部と、消化管液の少ない結腸で比較的すみやかに溶出する内核錠を組み合わせた有核二層の徐放化製剤である。そのため、分割せず経口投与した場合、図2のBのように全体的になだらかな血漿中濃度の推移となる。
１　誤
本剤は有核錠であり、スパンタブ型の錠剤ではない。なお、スパンタブは、速溶層と徐放層を重ね合わせた2層から成る錠剤である。
２　正
本剤を分割した場合の血漿中濃度は、すみやかに溶出する内核錠の部分がはじめから消化管液に触れることになるため、分割せず投与する場合と比べ、溶出初期に急激に上昇してしまう。よって、8分割した際の血中濃度推移は図2のAのようになる。
３　正
本剤の外層部の徐放化システムとして、薬物分子を高分子基剤中に分散させたマトリックスが用いられている。
４　誤
本剤の内核錠はすみやかに溶出するため、徐放性コーティングはされていない。
５　誤
図2より、ニフェジピンの血漿中濃度は、投与後すぐに上昇していることが読み取れる。そのため、錠剤全体に腸溶性コーティングはされていないとわかる。
