# 109-281

### 第109回薬剤師国家試験 問281

**問題文（全文）**  
提案された製剤は、以下の添加剤を含み、図のような構造をしている。この製剤に関する記述のうち、正しいのはどれか。**2つ選べ**。[1][2][3][6]  

添加剤：ステアリン酸カルシウム、ヒドロキシプロピルセルロース、カルボキシビニルポリマー、エチルセルロース[2][4][6]  

（注）セレニカR顆粒40%：1g中バルプロ酸ナトリウムを400mg含有する**膜制御型**の徐放性顆粒[1][3][5]  

**全選択肢（番号付き）**  
（検索結果から選択肢の完全な記述は一部省略されているが、正誤が明記されたものを基に再現）[1][2][5][6]  
1. 水不溶性フィルムとしてヒドロキシプロピルセルロースが用いられている。（×）[6]  
2. （詳細不明だが誤り。飽和濃度関連か0次放出関連で一部正誤議論あり）[2][5]  
3. 製剤内部の薬物が飽和濃度で、シンク条件が保たれる間は、薬物が**一定速度（0次速度）**で放出される。（〇）[2][5]  
4. **Higuchi式**に従う。（×）Higuchi式は不溶性マトリックス型徐放製剤に適用され、本製剤は膜制御型であるため誤り。[1][2]  
5. 薬物を放出した後の**白い粒（残渣、ゴーストピル）**が便中に排出される。（〇）[1][2]  

**正解番号**：**3 と 5**[1][2][5]  

**解説**  
セレニカR顆粒は膜制御型の徐放性製剤で、エチルセルロースなどが膜を形成し、内部の飽和薬物が一定速度（0次放出）で放出されるため、選択肢3が正しい。[1][2][5]  
Higuchi式はマトリックス型（薬物がマトリックス全体から拡散）に適用されるため、膜制御型の本製剤には不適切で選択肢4は誤り。[1][2]  
薬物放出後、空の白い粒（ゴーストピル）が便中に見られるため、選択肢5が正しい。[1][2]  
選択肢1は誤り（ヒドロキシプロピルセルロースは水溶性で膜には不適切）。選択肢2は詳細不明だが全体正解から誤りと判断。[1][2][6]
