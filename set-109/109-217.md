# 109-217

# 第109回薬剤師国家試験 問217

## 問題文

54歳女性。数ケ月前より、咽頭痛及び頸部リンパ節腫脹を認めた。精査の結果、悪性リンパ腫(びまん性大細胞型B細胞リンパ腫)と診断され、初回治療としてR-CHOP療法を開始するために入院となった。化学療法施行前にB型肝炎ウイルスのスクリーニング検査を実施したところ、特定の検査結果であったため、以下の処方が開始となった。

**この処方に関して薬剤師が留意しておく内容として、正しいのはどれか。2つ選べ。**[1][2]

## 選択肢

1. 飲み忘れた場合、次の就寝前に2回分をまとめて服用する。[1][2]

2. 食事により吸収が低下するため、服用前後2時間は食事をしない。[1][2]

3. 処方された薬剤を服用中でも、化学療法を開始する。[1][2]

4. 腎機能に応じた用量調整の必要はない。[1][2]

5. この処方は継続せず、1週間で終了する。[1]

## 正解番号

**2、3**[1][2]

## 解説

**選択肢1（誤り）**
飲み忘れた場合に2回分をまとめて服用することはありません。通常、飲み忘れた場合は、次の定められた時間に通常の1回分を服用するよう指導します。[1][2]

**選択肢2（正しい）**
食後2時間以降かつ次の食事の2時間以上前に服用するよう指導します。食事により吸収が低下するため、服用前後2時間は食事をしないことが重要です。[1]

**選択肢3（正しい）**
処方されている薬剤はエンテカビル錠で、B型肝炎ウイルスの増殖を抑制する逆転写酵素阻害薬です。化学療法施行前のB型肝炎スクリーニング陽性患者に対して、化学療法による免疫抑制に伴うB型肝炎の再活性化を予防するため、化学療法を開始する際にも服用を継続する必要があります。[1][2]

**選択肢4（誤り）**
エンテカビルは主に腎排泄される薬物です。クレアチニンクリアランスに応じた用法用量調節が必要な薬です。[1]

**選択肢5（誤り）**
エンテカビルは長期間の継続投与が必要です。B型肝炎の再活性化予防目的で、化学療法期間中および終了後も継続する必要があります。[1]
