# 109-48

経口固形製剤からの弱電解質薬物の溶出性に影響を及ぼさない因子はどれか。１つ選べ。

１ 薬物の粒子径
２ 薬物の結晶性
３ タンパク結合率
４ 溶液のpH
５ 溶液の粘度

**正解:** ３

**解説**
経口固形製剤からの弱電解質薬物の溶出性は、薬物の粒子径、薬物の結晶性、溶液のpH、溶液の粘度の影響を受けるが、溶出過程に血漿タンパク質への結合は関与しないため、タンパク結合率は溶出性に影響を及ぼさない。
