## ◆ 国内HIV感染者・AIDS患者の発生動向を図から読み取るシナリオ
260308

**問題**
- （衛生） 図1は、国内の HIV 感染者及び AIDS 患者の年間新規報告数の推移を示したものであり、図2は、2021 年における国内の HIV 感染者の新規報告の感染経路別内訳である。HIV 感染者の発生動向に関する記述のうち、正しいのはどれか。2つ選べ。

**図データ（概要）**
- 図1：HIV感染者（A・上折れ線）とAIDS患者（B・下折れ線）の年間新規報告数推移（1985〜2021年）
- 図2（2021年・感染経路別内訳）
	- 経路ア：10.5%（異性間性的接触）
	- 経路イ：71.6%（同性間性的接触）
	- 経路ウ：5.4〜5.5%（その他）
	- 不明：12.3%
	- その他：0.1%

**選択肢**

1. 図1のＡは HIV 感染者で、Ｂは AIDS 患者を表している。
​
2. 2021 年における HIV 感染者と AIDS 患者を合わせた新規報告数に占める AIDS 患者の割合は、約 80% と高い水準である。
​
3. HIV 感染者年間新規報告数は、男性よりも女性の方が多い。
​
4. 図2の経路アは、母子感染によるものである。
​
5. 図2の経路イは、同性間の性的接触によるものである。

**正解と理由**
- `1. 図1のＡはHIV感染者、ＢはAIDS患者` / `5. 経路イは同性間の性的接触`
- **理由の構成**
	- **問題の整理**：図1（推移グラフ）と図2（感染経路別内訳）を読み取り、HIV発生動向に関する正誤を判断する
	- **薬学的知識・仕様・制約**：HIV感染者数はAIDS患者数より多い（感染→発症まで数年〜10年のラグがある）、日本では男性・同性間接触が主要感染経路
	- **選択理由**：選択肢1は図の読み取り（HIV感染者数＞AIDS患者数）から正しい、選択肢5は経路イが71.6%で最大・同性間性的接触と一致して正しい

**解法の手順（思考の流れ）**
- 図1：AとBのうち報告数が多い（上側）折れ線がA → HIVに感染しているが発症していない人も含むため、HIV感染者数はAIDS患者数より多くなる → AがHIV感染者、BがAIDS患者 → 選択肢1は正しい
- 2021年のAIDS患者割合：図1の数値を読むと、AIDS患者（B）はHIV感染者と合計した中で約30%程度 → 約80%という記述は誤り → 選択肢2は誤り
- 感染者の男女比：日本のHIV感染は男性が圧倒的多数（男性同士の性的接触が主要経路のため） → 選択肢3は誤り
- 経路ア（10.5%）：異性間の性的接触 → 母子感染ではない → 選択肢4は誤り
- 経路イ（71.6%）：最大割合の経路 → 同性間の性的接触（主に男性同士）→ 選択肢5は正しい

**他の選択肢が誤りな理由**
- 2. AIDS患者の割合が約 80%
	HIV感染から`AIDSを発症`するまでには平均`数年〜10年`の潜伏期間がある — そのため未発症のHIV感染者数はAIDS患者数を大きく上回る — 2021年データではAIDS患者は全体の`約30%程度`であり、約80%という記述は誤り

- 3. 男性より女性の方が多い
	日本のHIV感染は`男性が圧倒的多数`（2021年のHIV感染者新規報告のほとんどが男性） — 経路イ（同性間性的接触・71.6%）が示すように、男性同士の性的接触が主要感染経路であることがその背景にある

- 4. 経路アは母子感染
	図2の経路ア（10.5%）は`異性間の性的接触`による感染 — `母子感染`（垂直感染）は報告数が非常に少なく、独立したカテゴリとして別に示されるか、その他に含まれる — 経路アを母子感染と混同しないよう注意

**キーポイント**
- 図1の読み取り：`HIV感染者数（A）＞AIDS患者数（B）` — 感染から発症まで潜伏期間があるため
- 日本のHIV感染経路：`経路イ（同性間性的接触）が71.6%で最大` — 男性同士が主
- `経路ア（10.5%）＝異性間性的接触` — 母子感染ではない点を混同しないこと
- AIDS患者の割合：全体の`約30%程度` — 80%という数値は誤り

**関連知識の比較**
- HIV感染の潜伏期間：平均`5〜10年`（無症候期） → その後CD4陽性T細胞が200/μL未満になるとAIDSと診断
- 日本の感染経路別内訳（2021年）：`同性間性的接触71.6%` > 不明12.3% > 異性間性的接触10.5% > その他5.4%
- 世界全体との比較：世界では`異性間性的接触`が主要経路だが、日本では`同性間性的接触`が圧倒的多数という点が特徴的

（一言メモ）
- 経路ア（異性間）と経路イ（同性間）の割合と内容をセットで覚える — `イ＝71.6%＝同性間`が核心
- HIV感染者とAIDS患者の区別：`感染者＞患者`は「潜伏期間があるから」という理由とセットで理解する
