# 109-82

入院患者のバンコマイシンによるMRSA感染症治療プロセスのうち、薬剤師が主に担当する役割はどれか。１つ選べ。

１ バンコマイシンを用いた治療の決定
２ バンコマイシン注射液静脈内投与後の経時的採血
３ 皮膚症状発現時の処置
４ 血中CRP及び血中バンコマイシン濃度の測定
５ 測定結果に基づくバンコマイシン投与設計の提案

**正解:** ５

**解説**
１　誤
バンコマイシンを用いた治療の決定は、医師が主に担当する役割である。
２　誤
バンコマイシン注射液静脈内投与後の経時的採血は、医師又は医師の指示を受けた看護師、臨床検査技師が主に担当する役割である。
３　誤
皮膚症状発現時の処置は、看護師が主に担当する役割である。
４　誤
血中CRP及び血中バンコマイシン濃度の測定は、臨床検査技師が主に担当する役割である。
５　正
測定結果に基づくバンコマイシン投与設計の提案は、薬剤師が主に担当する役割である。
