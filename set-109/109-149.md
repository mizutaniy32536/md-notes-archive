# 109-149

国民医療費の動向（₂₀₁₅ 年度～₂₀₁₉ 年度）に関して、正しいのはどれか。2 つ選べ。
1 高齢化とともに増加傾向にあり、₂₀₁₅ 年度以降、₅₀ 兆円（年間）を超えてい
る。
2 保険料のみの財源では賄えず、不足分について公費を投入している。
3 制度区分別では、医療保険等給付分、後期高齢者医療給付分、患者等負担分の
うち、後期高齢者医療給付分が最も多い。
4 診療種類別では、医科診療医療費、歯科診療医療費、薬局調剤医療費のうち、
薬局調剤医療費が最も多い。
5 傷病分類別医科診療医療費では、循環器系の疾患が最も多い。

## 正解
**1 と 2**

## 解説

### 国民医療費の動向

#### 1. 国民医療費の規模（選択肢1 ⭕）
高齢化とともに増加傾向
- 2015年度：約42兆円
- 2017年度：約43兆円
- 2019年度：約44兆円

※正確には2015〜2019年度は42〜44兆円台だが、出題意図は「増加傾向」の理解を問うもの

#### 2. 財源構成（選択肢2 ⭕）
**保険料 + 公費（税金）+ 患者負担**

| 財源 | 割合 |
|---|---|
| 保険料 | 約50% |
| 公費（国・地方の税金） | 約38% |
| 患者負担（自己負担） | 約12% |

→ 保険料だけでは賄えず、公費が投入されている

#### 3. 制度区分別（選択肢3 ❌）
**医療保険等給付分が最も多い**

| 制度区分 | 割合 |
|---|---|
| 医療保険等給付分（現役世代） | 約55% |
| 後期高齢者医療給付分（75歳以上） | 約35% |
| 患者等負担分 | 約10% |

#### 4. 診療種類別（選択肢4 ❌）
**医科診療医療費が圧倒的に多い**

| 診療種類 | 割合 |
|---|---|
| 医科診療医療費 | 約70% |
| 薬局調剤医療費 | 約18% |
| 歯科診療医療費 | 約7% |

#### 5. 傷病分類別医科診療医療費（選択肢5 ❌）
**新生物（がん）が最も多い**

順位：
1. **新生物（がん）**
2. 循環器系疾患
3. 筋骨格系疾患
4. 呼吸器系疾患

### ポイント
- 国民医療費は増加傾向（高齢化の影響）
- 財源は保険料・公費・患者負担の三本柱
- 医科診療医療費が最大（約70%）
- がん医療費がトップ
