# 109-123

**問題文**
保健統計に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-123 を参照

**選択肢**
1. 保健統計は、人口統計と死亡統計に大別される。
2. 人口統計は、人口動態統計と疾病統計に大別される。
3. 人口統計は、集団の健康水準の向上を図る上で重要な情報である。
4. 人口動態統計は、ある一時点における人口の規模や構造などを調査するものである。
5. 人口静態統計の代表的なものは、国勢調査である。

**正解:** 3,5

**解説**
解答　３、５

保健統計は、人口統計と疾病統計に大別される。さらに、人口統計は人口静態統計と人口動態統計に大別され、疾病統計には、国民生活基礎調査、患者調査、全国在宅障害児・者等実態調査などが含まれる。

１　誤
前記参照

２　誤
前記参照

３　正
人口統計によって得られる人口規模や人口構成、出生や死亡などに関する情報は、健康水準を測定するための指標となる健康指標（平均寿命や乳児死亡率など）の算出に必要であり、健康指標に基づき集団の健康状態を評価し、必要に応じて対策を講じる。ゆえに、人口統計は集団の健康水準の向上を図る上で重要な情報である。

４　誤
人口動態統計は、一定期間内における人口の変動に関する項目（出生、死亡、死産、婚姻、離婚）を調査するものである。一方、人口静態統計は、ある一時点における人口の規模や構造などを調査するものである。

５　正
人口静態統計の代表例である国勢調査では、5年ごとに氏名、性別、出生年月日、配偶の関係、国籍、5年前居住地、教育状況、職業、居住種などを調査する。
