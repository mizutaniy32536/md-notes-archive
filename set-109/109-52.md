# 109-52

**問題文**
日本薬局方の製剤に関する一般試験法のうち、下図の装置を用いるのはどれか。1つ選べ。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-52 を参照

**選択肢**
1. 崩壊試験法
2. 眼軟膏剤の金属性異物試験法
3. 半固形製剤の流動学的測定法
4. 粘着力試験法
5. 吸入剤の空気力学的粒度測定法

**正解:** 3

**解説**
解答　３

図の装置であるペネトロメーター（稠度計）を用いる一般試験法は、半固形製剤の流動学的測定法における稠度試験法である。試料台に乗せた半固形製剤に対して円錐が進入する距離を測定することで、半固形製剤の硬さ・柔らかさを測定する。なお、半固形製剤の流動学的測定法は、半固形製剤に対し、力を加えることで流動性と変形を測定する方法であり、展延性試験法及び稠度試験法がある。
