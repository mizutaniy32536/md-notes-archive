# 109-51

有効成分を速やかに溶解させ、口腔粘膜から吸収させる口腔用錠剤はどれか。１つ選べ。

１ 溶解錠
２ 舌下錠
３ チュアブル錠
４ 口腔内崩壊錠
５ 発泡錠

**正解:** ２

**解説**
日本薬局方において、各剤形は下記のように規定されている。
１　誤
溶解錠は、水に溶解して服用する錠剤である。
２　正
舌下錠は、有効成分を舌下で速やかに溶解させ、口腔粘膜から吸収させる口腔用錠剤である。
３　誤
チュアブル錠は、咀嚼して服用する錠剤である。
４　誤
口腔内崩壊錠は、口腔内で速やかに溶解又は崩壊させて服用できる錠剤である。
５　誤
発泡錠は、水中で急速に発砲しながら溶解又は分散する錠剤である。
