# 109-209

### 第109回薬剤師国家試験 問209

**問題文（全文）**  
後日、同患者が再び来局し、不安やイライラが強いことから漢方薬を紹介してほしいと依頼された。この患者に加味逍遙散（注）を紹介したが、この漢方薬を長期服用する場合、留意すべき副作用として**腸間膜静脈硬化症**が報告されている。  
（注）サイコ、シャクヤク、トウキ、ソウジュツ、ブクリョウ、**サンシシ**、ボタンピ、カンゾウ、ショウキョウ、ハッカから構成される処方  

この副作用の原因と考えられる生薬として正しいのはどれか。1つ選べ。なお、写真左下のスケールバーは1cmである。[1][2][3][4]

**全選択肢（番号付き）**  
写真はそれぞれ以下の生薬を示す（スケールバー1cmで識別）：  
1. **甘草（カンゾウ）**  
2. **山梔子（サンシシ）**  
3. **生姜（ショウキョウ）**  
4. **芍薬（シャクヤク）**  
5. **柴胡（サイコ）**[2][3]

**正解番号**  
**2**[1][2][3]

**解説**  
加味逍遙散の構成生薬のうち、**山梔子（サンシシ）**を長期間（目安5年以上）服用すると**腸間膜静脈硬化症**の発症リスクが高くなることが知られている。[3]  
写真からサンシシ（選択肢2）を識別する問題であり、過去問（例:第106-108回）と類似の出題形式。[2][3]  
他の選択肢（甘草、ショウキョウ、シャクヤク、サイコ）はこの副作用との関連が報告されていない。[1][2]
