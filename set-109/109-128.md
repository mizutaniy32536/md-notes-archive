# 109-128

三大栄養素としての糖質、脂質、タンパク質に関する記述のうち、正しいのはどれか。２つ選べ。

１ 単位重量当たりで得られるエネルギーが最も多いのは、糖質である。
２ 糖質はエネルギー源となり、エネルギー貯蔵体にも変換される。
３ 脂質から単位重量当たりに得られるエネルギーは、タンパク質の約4倍である。
４ タンパク質はエネルギー源にはならない。
５ 脂質やタンパク質の代謝物の中には、ホルモンなどの生理活性物質の前駆体になるものがある。

**正解:** ２、５

**解説**
１　誤
単位重量当たりで得られるエネルギーが最も多いのは、脂質である。単位重量当たりで得られるエネルギーはAtwater係数と呼ばれ、糖質、脂質、タンパク質それぞれのAtwater係数は4 kcal/g,9 kcal/g,4 kcal/gである。
２　正
糖質は、解糖系やTCA回路、電子伝達系を経てエネルギー源となるが、余剰の糖質は肝臓や筋肉でグリコーゲンとしてエネルギー貯蔵体にもなる。
３　誤
脂質とタンパク質の単位重量当たりに得られるエネルギー（Atwater係数）は、それぞれ9 kcal/g,4 kcal/gである。よって、脂質から単位重量当たりに得られるエネルギーは、タンパク質の約2倍である。
４　誤
飢餓時など糖質の摂取不足により血中グルコース濃度が低下すると、タンパク質が代謝されて生じる糖原性アミノ酸（アラニン、セリン、グルタミン酸など）が糖新生を促進し、エネルギー源になる。
５　正
脂質の代謝物の1つであるアラキドン酸は、プロスタグランジンやロイコトリエンなどの生理活性物質の前駆体になり、タンパク質の代謝物のアミノ酸は、甲状腺ホルモンなどの生理活性物質の前駆体になる。
