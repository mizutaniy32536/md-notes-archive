# 109-173

**問題文**
薬物のみかけの分布容積とその変動に関与する血漿タンパク結合に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 109-173 を参照

**選択肢**
1. みかけの分布容積は、体内薬物量と血漿中薬物濃度の平衡定数として定義される。
2. 特定の臓器や細胞内小器官（核やリソソーム、ミトコンドリアなど）に分布する薬物は、体重1 kgあたりの分布容積が10 Lを越えることがある。
3. 脂溶性の高い薬物の分布容積は加齢に伴って減少する。
4. 血漿タンパク結合率が高い薬物のみかけの分布容積は体内水分量とほぼ等しい。
5. タンパク非結合型薬物の濃度は、定常状態において血漿中と組織間隙液中との間でほぼ等しい。

**正解:** 2,5

**解説**
解答　２、５

１　誤
みかけの分布容積Vd（L）は、薬物が血漿中薬物濃度C（mg/L）で全身に分布すると仮定したときの容積であり、体内薬物量X（mg）を血漿中薬物濃度C（mg/L）で除した値で定義される定数である（分布容積Vd＝体内薬物量X／血漿中薬物濃度C）。なお、平衡定数は可逆反応における反応物と生成物の濃度比であり、体内薬物量X（mg）は濃度ではないため、みかけの分布容積Vdは平衡定数とは言えない。

２　正
特定の臓器や細胞内小器官（核やリソソーム、ミトコンドリアなど）に分布する薬物は、分布容積が増大する特徴を示すため、体重1 kgあたりの分布容積が10 Lを越えることがある。

３　誤
加齢に伴い体内水分量が低下するため、脂溶性の高い薬物の分布容積は増大する。

４　誤
血漿タンパク結合率が高い薬物のみかけの分布容積は、血漿容積とほぼ等しくなる。

５　正
定常状態（分布平衡）時における血漿中と組織間隙液中とのタンパク非結合型薬物の濃度は、ほぼ等しくなる。
