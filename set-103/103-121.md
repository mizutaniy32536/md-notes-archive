# 第103回 問121

**問題文**
交通事故により顔面を損傷し食事を経口摂取できない入院患者（20歳男性、60 kg、基礎疾患なし）に、高カロリー輸液を行うことになった。身動きできないので身体活動レベルは1.2と評価された。グルコースのみでエネルギー補給する場合、1日に必要と推定されるグルコース量（g）はどれか。１つ選べ。ただし、20歳の男性の基礎代謝基準値を25.0 kcal/kg体重とする。

**選択肢**
1. 200
2. 375
3. 450
4. 1,800
5. 7,200

**正解:** 3

**解説**
人が1日に必要とされるエネルギー量は以下の式で求められる。
推定エネルギー必要量（kcal/day）＝基礎代謝量（kcal/day）×身体活動レベル・・・①

また、基礎代謝量は以下の式で求められる。
基礎代謝量（kcal/day）＝基礎代謝基準値（kcal/kg体重/day）×体重（kg）・・・②

①及び②式より、この男性の推定エネルギー必要量は以下のように求められる。
推定エネルギー必要量(kcal/day)＝25.0（kcal/kg体重/day）×60（kg体重）×1.2
＝1,800（kcal/day）
よって、この男性は、1,800 kcal/dayをグルコースのみで補給する。

グルコース1 gの利用エネルギー（Atwater係数）は4 kcalであるため、1日に必要と推定されるグルコース量（g）は以下のように求められる。
1,800 (kcal/day)÷4（kcal/g）＝450（g/day）
