# 第103回 問303

**問題文**

68歳男性。肝細胞がんによる肝部分切除後に痛みが出現したため疼痛治療を開始した。1ヶ月前から医療用麻薬が導入され、2週間前に増量された。今回、肝細胞がん再発の治療のため入院となった。緩和ケアチームの薬剤師は、患者へのインタビューにより、「痛みのコントロールは良好だが、2週間ほど前から眠気が強くなり昼間でも傾眠傾向あり」との情報を得た。検査データ：NH3 50 µg/dL、Alb 3.0 g/dL、Na 137 mEq/L、Cl 104 mEq/L、K 5.3 mEq/L、Ca 8.7 mg/dL、BUN 25mg/dL、Cr 1.28 mg/dL、Ccr 38.2 mL/min、腹水（−）、脳への転移（−）

この患者の病態と薬物治療に関する記述のうち、正しいのはどれか。２つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等は e-REC の原ページを確認してください。

**選択肢**

- １. 他の鎮痛薬へ変更しても、便秘は軽減できない。
- ２. 排泄障害により、モルヒネの血中濃度が上昇し傾眠傾向となっている。
- ３. モルヒネの代謝物が、オピオイド受容体に対する作用増強の原因となっている。
- ４. 鎮痛薬の変更と同時にナロキソンを投与して傾眠を改善させる。
- ５. 腎機能の悪化が、眠気を引き起こすことになった要因として考えられる。

**正解:** 3、5

**解説**

１ 誤
フェンタニルクエン酸塩やオキシコドン塩酸塩水和物の方がモルヒネ硫酸塩水和物よりも便秘のリスクが低いため、オピオイドスイッチングで他の鎮痛薬へ変更することで便秘の軽減が可能である。

２ 誤
モルヒネは、肝臓でグルクロン酸抱合を受けてモルヒネ−6−グルクロニド（M6G）に代謝されることで強い作用を示す。また、このM6Gは腎臓から排泄される。
本患者は、BUN とCrが上昇（基準値：BUN 20 mg/dL以下・Cr 1.3 mg/dL以下）していることから腎機能が低下しており、M6Gの排泄障害で血中濃度が上昇し、オピオイド受容体に対し強く作用してしまったことで傾眠傾向となっていると考えられる。

３ 正解説3参照

４ 誤
ナロキソンはオピオイド拮抗薬であり、モルヒネなどによる急性呼吸抑制の改善には用いられるが、傾眠の改善には用いられない。

５ 正
