# 第103回 問60

**問題文**
双極性障害の躁状態の症状として誤っているのはどれか。１つ選べ。

**選択肢**
1. 観念奔逸
2. 多弁
3. 希死（自殺）念慮
4. 誇大妄想
5. 気分爽快

**正解:** 3

**解説**
双極性障害の躁状態では、気分が高揚することで爽快な気分になり、活動性が異常亢進する。症状としては、観念奔逸（次々と考えが浮かんで思考がまとまらない状態）、誇大妄想、多弁、睡眠欲求の減少などがあげられる。
なお、希死（自殺）念慮のような悲壮的な思考は、うつ状態においてみられる症状である。
