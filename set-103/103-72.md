# 第103回 問72

**問題文**
病院が必ず有しなければならない施設はどれか。１つ選べ。

**選択肢**
1. 集中治療室
2. 病理の検査施設
3. 調剤所
4. 研究室
5. 病理解剖室

**正解:** 3

**解説**
病院が有しなければならない施設は以下の通りである。
① 各科専門の診察室
② 手術室
③ 処置室
④ 臨床検査施設
⑤ エックス線室
⑥ 調剤所
⑦ 給食施設
⑧ 診療科名中に産婦人科又は産科を有する病院にあっては、分べん室及び新生児の入浴施設
⑨ 療養病床を有する病院にあっては、機能訓練室
⑩ その他都道府県の条例で定める施設
