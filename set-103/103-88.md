# 第103回 問88

**問題文**
内視鏡の消毒に使用されるのはどれか。１つ選べ。

**選択肢**
1. グルタラール
2. 消毒用エタノール
3. 次亜塩素酸ナトリウム
4. ベンザルコニウム塩化物
5. ポビドンヨード

**正解:** 1

**解説**
内視鏡は、臓器内腔の血液や体液に接するため、ウイルスや微生物による汚染を受ける可能性がある。そのため、ウイルスや各種微生物に幅広く有効な高水準消毒薬のグルタラールが一般に用いられている。

１ 正
前記参照

２ 誤
消毒用エタノールは中水準消毒薬に分類され、各種微生物に幅広く効果を示すが、芽胞形成菌に有効ではないため、内視鏡の消毒には用いない。

３ 誤
次亜塩素酸ナトリウムは中水準消毒薬に分類され、各種微生物に幅広く有効なものの、酸化作用により金属を腐食させるため、内視鏡の消毒には用いない。

４ 誤
ベンザルコニウム塩化物は低水準消毒薬に分類され、手指や粘膜の消毒、リネン類や床上などの消毒に用いられるが、抗菌スペクトルが狭いため、内視鏡の消毒には用いない。

５ 誤
ポビドンヨードは中水準消毒薬に分類され、各種微生物に幅広く効果を示すが、酸化作用により金属を腐食させるため、内視鏡の消毒には用いない。
