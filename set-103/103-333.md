# 第103回 問333

**問題文**
52歳男性。腰痛のためロキソプロフェンNaを服用している。全身倦怠感が続いたため受診した。検査の結果、薬物の長期服用による慢性肝疾患が疑われ入院した。肝機能に関する検査値は以下の通りである。ただし、（　）内は正常上限値とする。AST 1,260 IU/L（35）、ALT 1,330 IU/L（35）、ALP 264 IU/L（330）、T−Bil 0.9 mg/dL（1.0）、γ−GTP 40 IU/L（50）薬物性肝障害の種類は以下のように分類される。N：正常上限値、ALT比＝ALT値/N、ALP比＝ALP値/Nこの患者の治療に推奨する薬物はどれか。２つ選べ。

**選択肢**
1. タウリン
2. ウルソデオキシコール酸
3. グリチルリチン酸
4. ソホスブビル
5. リバビリン

**正解:** 2、3

**解説**
ALT＞2N(1330＞2×35)かつALP≦N（264≦330）という検査値から、本患者は肝細胞障害型の薬剤性肝障害であることがわかる。
肝細胞障害型の薬剤性肝障害では、ウルソデオキシコール酸、グリチルリチン酸などを用いた薬物治療が行われる。なお、胆汁うっ滞型の薬剤性肝障害では、ウルソデオキシコール酸、タウリンなどを用いた薬物治療が行われる。
