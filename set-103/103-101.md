# 第103回 問101

**問題文**
化合物Ａの最も安定な立体配座に関する記述のうち、誤っているのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 103-101 を参照

**選択肢**
1. シクロヘキサン環は、いす形配座である。
2. ヒドロキシ基aは、アキシアル位にある。
3. イソプロピル基bは、エクアトリアル位にある。
4. 炭素−炭素結合cとdは、ほぼ平行である。
5. 炭素−水素結合eとfは、ほぼ平行である。

**正解:** 2

**解説**
シクロヘキサン環にはいす形配座とふね形配座があり、一般的に最も安定な配座はいす形配座である。いす形配座の置換基の結合方向には、垂直方向のアキシアル位と斜め方向のエクアトリアル位があり、立体障害の大きい置換基（イソプロピル基など）がエクアトリアル位にあると安定である。化合物A（l−メントール）の最も安定な立体配座は図1である。また、いす形配座のシクロヘキサン（図２）において、それぞれ太線で表された結合はほぼ平行である。
