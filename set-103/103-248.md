# 第103回 問248

**問題文**

62歳女性。毎日午前1時に就寝し、午前6時に起床する規則的な生活をしている。最近寝付きが悪い日が続いた。薬局で睡眠改善薬の一般用医薬品を購入して服用したが、改善されなかった。そこで専門医を受診し、すぐに眠れて朝すっきり起きられるような薬を希望し、睡眠薬が処方させることになった。

処方される薬剤として最も適切なのはどれか。１つ選べ。

**選択肢**

- １. アモバルビタール錠
- ２. スボレキサント錠
- ３. エスタゾラム錠
- ４. フルニトラゼパム錠
- ５. ゾルピデム酒石酸塩錠

**正解:** 5

**解説**

不眠症は、入眠困難型と睡眠維持障害型（中途覚醒、早朝覚醒）に分類され、入眠困難型には消失半減期の短い睡眠薬、睡眠維持障害型には消失半減期の長い睡眠薬が推奨される。本患者は、「最近寝付きの悪い日が続いた」状態にあることから、入眠困難型であると考えられる。よって、本患者に処方される薬剤として最も適切なのは、消失半減期の短い睡眠薬である。

１ 誤
アモバルビタールは、中時間型のバルビツール酸系睡眠薬であり、睡眠維持障害に用いられるため、本患者には適していない。アモバルビタールは、脳幹網様体に分布するγ-アミノ酪酸GABAA受容体のバルビツレート結合部位に結合し、GABAA受容体の機能を亢進することで催眠作用を示す。

２ 誤
スボレキサントは、中時間型の睡眠薬であり、睡眠維持障害に用いられるため、本患者には適していない。スボレキサントは、オレキシン受容体を遮断することにより脳内におけるモノアミン神経系を抑制し、脳を覚醒状態から睡眠状態へ移行させることで睡眠を誘発する。

３ 誤
エスタゾラムは、中時間型のベンゾジアゼピン系睡眠薬であり、睡眠維持障害に用いられるため、本患者には適していない。エスタゾラムは、大脳辺縁系に分布するγ-アミノ酪酸GABAA受容体のベンゾジアゼピン結合部位に結合し、GABAA受容体のGABAに対する親和性を高めることで催眠作用を示す。

４ 誤
フルニトラゼパムは、中時間型のベンゾジアゼピン系睡眠薬であり、睡眠維持障害に用いられるため、本患者には適していない。フルニトラゼパムは、大脳辺縁系に分布するγ-アミノ酪酸GABAA受容体のベンゾジアゼピン結合部位に結合し、GABAA受容体のGABAに対する親和性を高めることがある。

５ 正
ゾルピデムは、超短時間型の非ベンゾジアゼピン系睡眠薬であり、入眠困難に用いられるため、本患者に適している。ゾルピデムは、大脳辺縁系に分布するγ-アミノ酪酸GABAA受容体のベンゾジアゼピン結合部位に結合し、GABAA受容体のGABAに対する親和性を高める。
