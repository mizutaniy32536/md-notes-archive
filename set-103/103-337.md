# 第103回 問337

**問題文**
サリドマイドを服用する患者への説明として、適切なのはどれか。２つ選べ。

**選択肢**
1. 紛失した場合は、処方医又は調剤した薬剤師に連絡してください。
2. 服用の必要がなくなった場合は、残った薬を速やかに破棄してください。
3. （男性の場合）服用中でも避妊する必要はありません。
4. （女性の場合）服用開始4週間前から服用終了4週間後まで必ず避妊してください。
5. 服用中でも授乳してかまいません。

**正解:** 1、4

**解説**
サリドマイドはかつて睡眠薬として使用されていたが、サリドマイドの持つ催奇形性が社会問題となり、販売が中止となった。しかし、サリドマイドの研究が進み、多発性骨髄腫やらい性結節性紅斑への有効性が認められたため、現在は『サリドマイド製剤安全管理手順』に従い厳格に管理されながら再び販売されている。なお、『サリドマイド製剤安全管理手順』には、サリドマイドの胎児への曝露を防止するために、「薬剤管理及び妊娠回避の徹底等」などについて記載されている。

１ 正
患者がサリドマイドを紛失した場合は、患者（または薬剤管理者）が調剤元の医療機関でサリドマイドの管理を行っている薬剤師（責任薬剤師）等へ紛失したことを報告する必要があるため、本選択肢の説明は適切である。なお、薬剤管理者とは、患者の薬剤管理を行い、患者以外の者の誤飲防止や不要になった薬剤の返却を徹底するため等に選定される患者の身近な者又は医療関係者である。

２ 誤
サリドマイドは数量の管理が必要な薬剤である。服用中止等で残薬が出た場合は調剤元の医療機関の責任薬剤師等へ返却する必要があるため、本選択肢の説明は不適切である。

３ 誤
サリドマイド服用中は、男女ともに避妊する必要がある。男性患者の場合、服用開始時から服用終了4週間後まで避妊する必要があるため、本選択肢の説明は不適切である。

４ 正
女性患者の場合、服用開始4週間前から服用終了4週間後まで避妊する必要があるため、本選択肢の説明は適切である。

５ 誤
サリドマイドは乳汁中への移行が報告されており、服用開始時から服用終了4週間後まで授乳することが禁止されているため、本選択肢の説明は不適切である。
