# 第103回 問221

**問題文**

48歳女性。既往歴 乳がん、StageⅠ。2年前の術後より再発予防の目的で以下の処方にてホルモン療法を受けている。

この患者は、最近、精神的に不安定となり、不安発作が頻回になった。本症状の改善のための処方追加を検討するにあたり、タモキシフェンとの併用の観点から問題となる薬物について医師から問い合わせがあった。本症例に対して併用を注意すべき薬物はどれか。１つ選べ。

**選択肢**

- １. セチプチリンマレイン酸塩
- ２. ミルナシプラン塩酸塩
- ３. パロキセチン塩酸塩水和物
- ４. ロラゼパム
- ５. ロルメタゼパム

**正解:** 3

**解説**

選択肢の薬物のうち、パロキセチン塩酸塩水和物はCYP2D6阻害作用を有するため、タモキシフェンと併用すると、タモキシフェンの抗悪性腫瘍効果が減弱するおそれがあるため注意が必要である。
