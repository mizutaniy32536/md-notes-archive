# 第103回薬剤師国家試験 問42

## 問題文

親水性薬物の経皮吸収における最大の障壁はどれか。1つ選べ。[1][2]

## 選択肢

1. 皮脂腺
2. 真皮層
3. 毛嚢
4. 角質層
5. 毛細血管

## 正解番号

**4. 角質層**[1][2]

## 各選択肢の解説

**1. 皮脂腺** ✕
皮脂腺は皮膚の付属器官ですが、経皮吸収における主要な障壁ではありません。

**2. 真皮層** ✕
真皮層は表皮の下に位置する層ですが、経皮吸収の律速段階（最大の障壁）ではありません。

**3. 毛嚢** ✕
毛嚢は皮膚の付属器官で、経皮吸収における主要な障壁ではありません。

**4. 角質層** ◎
**角質層は皮膚の最表面に位置し、バリア機能を有しています。**親水性薬物の経皮吸収における最大の障壁であり、経皮からの薬物吸収過程における律速段階となります。[1][4]

**5. 毛細血管** ✕
毛細血管は皮膚内に存在しますが、経皮吸収における主要な障壁ではありません。
