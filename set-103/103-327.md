# 第103回 問327

**問題文**
26歳男性。1日数回の下痢を繰り返し、また、血便が出ていたので近医を受診した。検査の結果、潰瘍性大腸炎と診断され、メサラジン錠を用いた治療を開始した。2年後、出血性下痢の増加と腹痛を認めるようになり、薬物はメサラジン錠とプレドニゾロン錠の併用に変更になった。この患者の病態と薬理学的管理について適切でないのはどれか。２つ選べ。

**選択肢**
1. 服用困難な場合には、メサラジン錠を粉砕する。
2. 感染症にかかりやすい。
3. メサラジンの副作用として、消化器症状に気をつける。
4. 定期的に大腸がんの検査を受ける。
5. メサラジン錠服用により、潰瘍性大腸炎の完治が期待できる。

**正解:** 1、5

**解説**
１ 不適切
メサラジン錠は、大腸において抗炎症作用を示す5-アミノサリチル酸製剤であり、小腸などで吸収されず大腸まで到達できるように放出制御型製剤にされている。そのため粉砕はしてはならず、服用困難な場合には、剤形の変更などを提案するべきである。

２ 適切
プレドニゾロン錠は副腎皮質ステロイド性薬であり、免疫抑制作用をもつため、感染症にかかりやすくなる。

３ 適切
メサラジンは副作用として、腹痛、下痢、血便などの消化器症状がみられることがあるため、注意が必要である。

４ 適切
潰瘍性大腸炎は発症後長期経過すると大腸がんのリスクが高くなる。そのため、定期的に内視鏡検査を受ける必要がある。

５ 不適切
潰瘍性大腸炎は、原因不明の非特異的な炎症性腸疾患（IBD）であるため完治療法は見つかっておらず、消化管炎症を抑えながら患者のQOLを維持することが治療の目的となる。
