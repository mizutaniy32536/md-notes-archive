# 第103回 問210

**問題文**

6歳男児。体重20 kg。身長120 cm。扁桃炎と診断され、この男児の処方せんを、母親が薬局に持参した（処方1）。

セフジニルは鉄イオンに配位し、キレートを形成する。矢印で示したセフジニルの原子のうち、鉄イオンに最も配位しにくいのはどれか。１つ選べ。

**正解:** 4

**解説**

セフジニルと鉄イオンは配位結合によってキレートを形成する。配位結合するためにはローンペア（非共有電子対）が必要となるが、選択肢４の窒素原子は共鳴によりローンペアが非局在化しているため鉄イオンに最も配位しにくいと考えられる。
