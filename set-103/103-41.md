## ◆ 胃排出速度低下で吸収量が増大する薬物を選ぶシナリオ

260310

**問題**
- 食事による胃内容排出速度の低下によって、吸収量が増大するのはどれか。1つ選べ。

**選択肢**

1. アセトアミノフェン
​
2. エリスロマイシン
​
3. セファクロル
​
4. リファンピシン
​
5. リボフラビン

**正解と理由**
- `5. リボフラビン`
- **理由の構成**
	- `リボフラビン`（ビタミンB₂）は小腸上部の`RFVT`（リボフラビントランスポーター）で吸収される飽和性トランスポーター基質
	- 胃排出速度↓ → 薬物が小腸上部に`ゆっくり到達` → トランスポーターの`飽和を回避` → 結果的に`吸収量が増大`

**他の選択肢が誤りな理由**
- 1. `アセトアミノフェン`は単純拡散で吸収。胃排出遅延で速度は低下するが吸収量は変わらない
- 2. `エリスロマイシン`は胃酸で分解されやすく、胃滞留延長はむしろ不利
- 3. `セファクロル`はPEPT1基質だが、胃排出遅延で吸収「速度」は低下しても量は変わらない
- 4. `リファンピシン`は単純拡散で吸収。胃排出遅延の影響は限定的

**キーポイント**
- 「胃排出速度低下で吸収量が`増大`」→ 飽和性トランスポーター基質で吸収部位が限定されている薬物
- `リボフラビン`は吸収部位が小腸上部に限定（absorption window）

**関連知識の比較**
- 胃排出速度低下で「速度↓・量同じ」→ セファクロル（Q5で出題済み）
- 胃排出速度低下で「量↑」→ `リボフラビン`（飽和性＋吸収窓）
- 胃排出速度低下で「量↓」→ エリスロマイシン（胃酸分解↑）

（一言メモ）
- `リボフラビン`は2パターンで出題される: ①速度的BA低下（109-171）、②吸収量増大（103-41）。いずれもRFVTの飽和性がキー
- 「吸収量が増大」のキーワードがあればリボフラビンを思い出す
