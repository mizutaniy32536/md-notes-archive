# 第103回 問319

**問題文**

72歳男性。通院困難なため在宅医療を受けている。医師の訪問時に男性の家族より、夜になると咳が止まらなくなり本人が眠れていないことが伝えられた。診察の結果、次の薬剤が処方され、薬剤師が在宅にて対応することとなった。

当該患者に対して、保険薬局の保険薬剤師が医療保険で行う訪問薬剤管理指導に関する記述のうち、適切なのはどれか。２つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等は e-REC の原ページを確認してください。

**選択肢**

- １. 訪問薬剤管理指導を行う場合、保険薬局は都道府県知事の許可を受ける必要がある。
- ２. 当該患者が介護認定を受けている場合でも、原則として医療保険が優先される。
- ３. 訪問薬剤管理指導は、医師の指示に基づいて行う。
- ４. 薬学的管理指導計画は、訪問前に策定する。
- ５. 訪問薬剤管理指導を行うにあたり、患者と薬局との契約書の作成が必要である。

**正解:** 3、4

**解説**

１ 誤
保険薬局の保険薬剤師が医療保険における訪問薬剤管理指導を行う場合、保険薬局は地方厚生局長に届け出る必要がある。
なお、介護保険における居宅療養管理指導を行う場合、保険薬局は居宅療養管理指導を行う指定居宅サービス事業者の指定を受けたものとみなされるため、別段の手続きを行う必要はない。

２ 誤
当該患者が介護認定を受けている場合は訪問薬剤管理指導に係る給付（サービス）について、介護保険が優先される。

３ 正
保険薬局が行う訪問薬剤管理指導は、医師の指示が必要である。また、指示のあった医師に対しての訪問結果の報告は文書により行う。

４ 正
訪問薬剤管理指導を行う場合には、医師の指示に基づき、患者宅へ訪問前に保険薬剤師が薬学的管理指導計画を策定してから、保険薬剤師が患家を訪問して、薬学的管理指導を行う必要がある。

５ 誤
医療保険における訪問薬剤管理指導を行うにあたり、患者と薬局との契約書の作成は必要ない。なお、介護保険における居宅療養管理指導を行うには、患者と薬局との契約書を作成する必要がある。
