# 第103回 問285

**問題文**

52歳男性。食道がんの手術後に完全静脈栄養による治療を受けていた。ビタミンB1不足による乳酸アシドーシスの疑いでチアミン塩化物塩酸塩を急速静注したが、効果が不十分であったため7 ％炭酸水素ナトリウム注射液40 mLを輸液500 mLに混合して点滴投与する予定である。

表は、各輸液の成分濃度を示している。炭酸水素ナトリウム注射液との混合で、配合変化が生じる可能性が最も高い輸液剤はどれか。１つ選べ。ただし、電解質の濃度はmEq/L、ブドウ糖の濃度はw/v％である。

> ⚠️ 画像を含む問題のため、図・構造式等は e-REC の原ページを確認してください。

**正解:** 1

**解説**

炭酸水素ナトリウム注射液と、カルシウムイオンを含む注射剤を混合すると、炭酸カルシウムが生成し沈殿物として析出する場合がある。したがって、炭酸水素ナトリウム注射液との混合で配合変化が生じる可能性が最も高い輸液剤は、カルシウムイオンを含有している選択肢１である。
