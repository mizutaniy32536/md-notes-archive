# 第103回 問13

**問題文**
図はヒト染色体を表す。矢印の部分（黒色）に存在する構造の名称はどれか。１つ選べ。

**選択肢**
1. 動原体
2. テロメア
3. 核小体
4. セントロメア
5. 紡錘糸

**正解:** 2

**解説**
１ 誤
動原体（キネトコア）は、有糸分裂において、染色体のセントロメア部分に複数のタンパク質が結合することで形成される構造体であり、紡錘糸が結合する。

２ 正
テロメアは、染色体の末端に存在する構造であり、反復配列を含むテロメアの構造は染色体末端の安定性に関与する。

３ 誤
核小体（仁）は、真核細胞の核内に存在する球状の構造体である。核小体には、RNAポリメラーゼⅠが存在し、リボソームRNA（rRNA）の合成や、rRNAとタンパク質を用いたリボソームの組み立てが行われる。

４ 誤
セントロメアは、染色体の中心部分の狭窄した領域であり、紡錘糸が結合する動原体が形成される。

５ 誤
紡錘糸は、微小管が重合した繊維であり、有糸分裂において染色分体の分離等に関与する。
