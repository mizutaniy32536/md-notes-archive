# 第103回 問58

**問題文**
子宮内膜症に関する記述のうち、正しいのはどれか。１つ選べ。

**選択肢**
1. 無痛性の疾患である。
2. 受精卵が着床しやすくなる。
3. 血清中のCA125が低値を示す。
4. エストロゲン分泌が減少する。
5. 薬物治療には低用量黄体ホルモン・卵胞ホルモン配合剤を用いる。

**正解:** 5

**解説**
子宮内膜症とは、卵管、卵巣、膣等の子宮粘膜上皮以外の場所に子宮内膜様組織が発生する疾患である。

１ 誤
子宮内膜症の症状として、生理痛、下腹部痛、性交痛などがあげられる。

２ 誤
子宮内膜症により、排卵不全や受精卵の着床不全等がみられる。そのため、子宮内膜症は不妊症の原因となる。

３ 誤
子宮内膜症では、子宮がんや卵巣がんなどの腫瘍マーカーであるCA125が高値を示すことがある。

４ 誤
子宮内膜症は、エストロゲンにより子宮内膜が増殖するエストロゲン依存性疾患であり、本症でのエストロゲン分泌量の減少はみられない。

５ 正
子宮内膜症の治療には、低用量黄体ホルモン・卵胞ホルモン配合剤（低用量ピル）やジエノゲスト（黄体ホルモン製剤）、GnRH（性腺刺激ホルモン放出ホルモン）アゴニストなどが用いられる。
