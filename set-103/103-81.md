## ◆ 皮下注射で投与される薬剤を選ぶシナリオ

260312

**問題**
皮下注射で投与されるのはどれか。1つ選べ。

**選択肢**

1. BCGワクチン
​
2. インフルエンザワクチン
​
3. パクリタキセル注射液
​
4. ハロペリドール注射液
​
5. フェニトインナトリウム注射液

**正解と理由**
- `2. インフルエンザワクチン`
- **理由の構成**
	- **問題の整理**：各薬剤の`投与経路`を正確に答える
	- **薬学的知識・仕様・制約**：`インフルエンザワクチン`=`皮下注射`（日本の標準）
	- **選択理由**：インフルエンザワクチンのみが皮下注射

**解法の手順（思考の流れ）**
- 選択肢1：`BCGワクチン`=`経皮接種`（スタンプ式、ハンコ注射）→ 皮下注射ではない
- 選択肢2：`インフルエンザワクチン`=`皮下注射` → 正しい
- 選択肢3：`パクリタキセル`=`点滴静注`（抗がん剤）→ 皮下注射ではない
- 選択肢4：`ハロペリドール`=`筋注`または`静注`（抗精神病薬）→ 皮下注射ではない
- 選択肢5：`フェニトインナトリウム`=`静脈内注射`（抗てんかん薬）→ 皮下注射ではない

**キーポイント**

| 薬剤 | 投与経路 |
|---|---|
| `BCGワクチン` | 経皮（スタンプ） |
| `インフルエンザワクチン` | **皮下注射** |
| `MRワクチン`、水痘ワクチン | 皮下注射 |
| `ロタウイルスワクチン` | 経口 |
| `点鼻インフルエンザワクチン` | 点鼻（欧米、日本未承認） |

- `BCG`は特殊な経皮接種（1回9本×4列）。皮下に入れない

（一言メモ）
- 「BCG=皮下注射」は典型ひっかけ。正しくは`経皮接種`（ハンコ）
- インフルエンザワクチン=皮下注射。欧米では筋注も使うが日本は皮下注が標準
