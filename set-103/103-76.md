# 第103回 問76

**問題文**
国民医療費に含まれるのはどれか。１つ選べ。

**選択肢**
1. 帝王切開による分娩
2. 一般用医薬品の購入
3. 特定健康診査の受診
4. 肺炎球菌感染症の予防接種
5. 介護保険法におけるリハビリテーション

**正解:** 1

**解説**
国民医療費は、各年度に医療機関等で行われた傷病の治療に要した費用を中心に推計したものを取りまとめたものである。帝王切開による分娩は、正常な妊娠・分娩ではなく治療等の医療行為の介入が必要であるため、国民医療費に含まれる。
なお、国民医療費に含まれない主なものとして、正常な妊娠・分娩の費用、健康診断・予防接種、要指導医薬品の購入費用、介護保険の費用等がある。
