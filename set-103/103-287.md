# 第103回 問287

**問題文**

24歳男性。悪性リンパ腫に対して外来化学療法を実施予定である。外来化学療法室で、薬剤師がレジメンチェックを行った。

R-CHOP療法の実施により、急に尿量の減少と浮腫を認めたため外来受診した。その際に血液検査で認められる異常所見として可能性が高いのはどれか。２つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等は e-REC の原ページを確認してください。

**選択肢**

- １. 高尿酸血症
- ２. 低カリウム血症
- ３. 高ナトリウム血症
- ４. 高カルシウム血症
- ５. 高リン血症

**正解:** 1、5

**解説**

悪性リンパ腫のような造血腫瘍へのR-CHOP療法の実施において、急に尿量の減少と浮腫を認めたことから、本患者は腫瘍崩壊症候群による急性腎不全を発症していると推測される。
腫瘍崩壊症候群は、血液中における腫瘍細胞の急速な崩壊により、細胞内成分とその代謝物が腎の生理的排泄能力を超えて体内に蓄積し、尿酸・リン・カリウムの血中濃度上昇、乳酸アシドーシス、リン酸の増加に伴い血中カルシウムが消費され生じる低カルシウム血症が起こる。さらには、高尿酸血症による痛風腎に伴う急性腎不全など、様々な病態を生じる。
