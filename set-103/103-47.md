# 第103回 問47

**問題文**
肝クリアランスに関する記述のうち、正しいのはどれか。１つ選べ。

**選択肢**
1. 上限は肝固有クリアランスである。
2. 上限は肝血流速度である。
3. 下限は肝固有クリアランスである。
4. 下限は肝血流速度である。
5. 肝血流速度に反比例する。

**正解:** 2

**解説**
肝クリアランス（CL 肝）は①式で表すことができる。

①式より、CL 肝は、Q肝に比例するパラメーターであり、E肝は、0≦E肝≦1である。そのため、CL肝の上限はE肝 ＝ 1の時であり、上限はQ肝となる。一方、CL肝の下限はE肝 ＝ 0の時であり、下限は0となる。
