# 第103回 問230

**問題文**

65歳男性。長期にわたるアルコール依存症と診断されて、入院治療中。食事が摂れず栄養不良の状態であったが、さらに担当看護師より、眼球運動の異常やふらつき、意識障害が確認されるようになったと報告があった。この患者の症状の原因としてビタミン欠乏の可能性が考えられた。

この患者で欠乏し、症状の原因となっている可能性が最も高いビタミンはどれか。１つ選べ。

**選択肢**

- １. ビタミンB1
- ２. ビタミンB2
- ３. ビタミンB6
- ４. ビタミンB12
- ５. ビタミンE

**正解:** 1

**解説**

アルコールの代謝の一部はビタミンB1が関与しているため、慢性アルコール中毒者ではビタミンB1が欠乏し易い状態にある。本患者は、長期にわたるアルコール依存症と診断されているほか、食事が摂れず栄養不良の状態であることから、ビタミンB1が欠乏していると考えられる。また、眼球運動の異常やふらつき、意識障害が確認されていることから、ビタミンB1の欠乏によるウェルニッケ脳症が発現していると考えられる。

１ 正
ビタミンB1の欠乏症として、脚気やウェルニッケ脳症が知られている。ウェルニッケ脳症では、眼球運動麻痺、運動障害、意識障害などが起こる。

２ 誤
ビタミンB2の欠乏症として、皮膚炎、口内炎などが知られている。

３ 誤
ビタミンB6の欠乏症として、舌炎、貧血、ペラグラ様症状などが知られている。

４ 誤
ビタミンB12の欠乏症として、巨赤芽球性貧血などが知られている。

５ 誤
ビタミンEの欠乏症は、ヒトではほとんどみられない。
