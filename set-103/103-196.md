# 第103回 問196

**問題文**

68歳男性。慢性腎不全にて入院中。今回とこれまでの血液検査の結果から、eGFRの低下が認められたため、クレメジン®細粒分包2 g（注）が追加となり、薬剤師に服薬説明の依頼があった。（注：1包中に球形吸着炭2 gを含有する）

球形吸着炭は、その表面に種々の物質を吸着することができる。本薬物の吸着現象に関する記述のうち、正しいのはどれか。２つ選べ。

**選択肢**

- １. 本薬物への吸着に飽和現象が観察された場合、単分子層吸着が主であると判断できる。
- ２. 本薬物への最大吸着量は、ノイエス−ホイットニー（Noyes−Whitney）の式から求めることができる。
- ３. 本薬物への物質の吸着は、吸着速度と脱離速度が等しくなるまで進行する。
- ４. 本薬物による物質の吸着は、主に静電的相互作用による。

**正解:** 1、3

**解説**

薬物への物質の吸着様式には、物質が薬物に一層で吸着する単分子層吸着と、吸着した物質の上にさらに物質が重なる多分子層吸着が知られている。

単分子層（飽和）吸着量Vmを求めるには、単分子層吸着ではLangmuir式（ラングミュアー式）、多分子層吸着ではBET式を用いる。

１ 正
本問では、「本薬物への吸着に飽和現象が観察された場合」とあるため、本薬物への物質の吸着は単分子層吸着であると予想される。

２ 誤
本薬物への最大吸着量は、単分子層飽和吸着量であるため、Langmuir式（ラングミュアー式）が用いられる。また、Noyes−Whitney式（ノイエス−ホイットニー式）は、溶解過程が拡散律速の場合の溶解速度を表した式である。

３ 正
本薬物への物質の吸着は、吸着平衡が成り立つまで進行する。吸着平衡では、吸着速度と脱離速度が等しくなる。

４ 誤
球形吸着炭は、慢性腎不全における尿毒症毒素を消化管内で吸着し、糞便中に排泄させることにより、尿毒症症状を改善し、透析導入を遅らせることができるため、尿毒症症状の改善及び透析導入の遅延に用いられる。本薬物には小さな穴があいており、その穴に毒素がファンデルワールス力等により引きつけられることにより効果を発揮する。
