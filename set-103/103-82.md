# 第103回 問82

**問題文**
添付文書の「警告」や「禁忌」に追加する情報を迅速に伝達するために、厚生労働省の指示のもとに製造販売業者が作成する文書はどれか。１つ選べ。

**選択肢**
1. 医療用医薬品製品情報概要
2. イエローレター
3. 医薬品安全対策情報
4. 医薬品・医療機器等安全性情報
5. 医薬品インタビューフォーム

**正解:** 2

**解説**
１ 誤
医療用医薬品製品情報概要は、医療関係者が医薬品を適正に使用するために必要な情報の概要であり、日本製薬工業協会の作成要領をもとに製造販売業者が作成する文書である。

２ 正
イエローレター（緊急安全性情報）は、重要かつ緊急な連絡を要する副作用情報を１ヶ月以内に医療関係者に伝達するためのものであり、厚生労働省の指示のもとに製造販売業者が作成する文書である。

３ 誤
医薬品安全対策情報（Drug Safety Update：DSU）は、各医薬品における「使用上の注意」等の改訂内容をまとめたものであり、日本製薬団体連合会が作成する文書である。

４ 誤
医薬品・医療機器等安全性情報は、厚生労働省が収集した副作用情報と症例を医療関係者に情報提供するために発行されるものであり、厚生労働省が作成する文書である。

５ 誤
医薬品インタビューフォームは、添付文書等の情報を補完する総合的な医薬品の解説書であり、日本病院薬剤師会の策定する記載要領をもとに、製造販売業者が作成する文書である。
