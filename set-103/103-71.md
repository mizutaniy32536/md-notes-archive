# 第103回 問71

**問題文**
法律の規定を実施するため内閣が定めるのはどれか。１つ選べ。

**選択肢**
1. 憲法
2. 政令
3. 省令
4. 条例
5. 条約

**正解:** 2

**解説**
１ 誤
憲法は、統治の根本規範となる基本的な原理原則に関して定めた法規範等のことである。

２ 正
政令は、法律の規定を実行するため、閣議決定を経て、内閣が定める。

３ 誤
省令は、法律、政令の規定を実行するため、各省大臣が発する。

４ 誤
条例は、地域限定で拘束力をもつものであり、地方自治体の議決により制定される。

５ 誤
条約は、文書による国家間の取決めであり、内閣が締結し、国会が承認する。
