# 第103回 問79

**問題文**
生前に、終末期の延命措置に関することについて、意思表示をしておくことを表す語句として最も適切なのはどれか。１つ選べ。

**選択肢**
1. セカンドオピニオン
2. インフォームドコンセント
3. クリニカルパス
4. リビングウィル
5. コンプライアンス

**正解:** 4

**解説**
回復の見込みがない重篤な疾病に罹患した患者が、自分の意思で、自分が望む形の最期を迎えることに尊厳死の意味があると考えられている。そのため、リビングウィル（尊厳死の権利を主張して、延命治療の打ち切りを希望するといった意思表示）を、生きている間に（living）自分自身で遺書（will）を書き、医師に伝えることを勧めている。

１ 誤
セカンドオピニオンとは、患者が自分の病気についてよりよい決断を行うために、主治医以外の医師に意見を求めることである。

２ 誤
インフォームドコンセントとは、患者が医師から十分な説明を受け、治療等の有効性や不利益等に関する事項についても理解・納得した上で、患者の自由意思により医療を選択し決定することである。

３ 誤
クリニカルパスとは、治療の標準化を図るために作成された各病態別の診療スケジュールのことである。

５ 誤
コンプライアンスとは、要求や命令などに応じること、人の願いなどを受け入れることをいう。そのため、服薬においては、患者が薬剤を指示どおりに服薬することを指す。 また、患者が薬剤を指示どおりに服薬していない場合をノンコンプライアンスという。
