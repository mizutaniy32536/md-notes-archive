# 第103回 問14

**問題文**
T細胞が分化・成熟する一次リンパ器官はどれか。１つ選べ。

**選択肢**
1. リンパ節
2. 胸腺
3. 脾臓
4. 副腎
5. 骨髄

**正解:** 2

**解説**
T細胞が分化・成熟する一次リンパ器官は、胸腺（選択肢2）である。
リンパ器官は、一次リンパ器官と二次リンパ器官の2つに大別され、一次リンパ器官はリンパ球（T細胞やB細胞など）の産生・分化が行われる場であり、骨髄や胸腺が該当する。一方、二次リンパ器官は、免疫担当細胞による免疫応答が行われる場であり、リンパ節や脾臓、パイエル板などが該当する。

１ 誤
リンパ節は、リンパ液中の抗原に対して免疫応答が行われる場であり、二次リンパ器官に該当する。

２ 正
胸腺は、T細胞が分化・成熟する場であり、一次リンパ器官に該当する。

３ 誤
脾臓は、赤脾臓と白脾臓に区画され、二次リンパ器官に該当する。赤脾臓は血液中の異物や古くなった赤血球を除去・排除する区画であり、白脾臓は主として血行性の抗原に対する免疫反応の場であるとともに、古くなったリンパ球の除去などが行われる区画である。

４ 誤
副腎は、皮質と髄質からなり、内分泌器官に該当する。副腎皮質からは鉱質コルチコイドや糖質コルチコイドなどが血中に分泌され、副腎髄質からはアドレナリンやノルアドレナリンが血中に分泌される。

５ 誤
骨髄は、一次リンパ器官に該当する。骨髄では、ほとんどの血球成分に分化可能な多能性造血幹細胞が存在し、種々の血球細胞が産生されるとともに、B細胞の前駆細胞がB細胞へと機能的に分化・成熟されるが、T細胞への分化は行われない。
