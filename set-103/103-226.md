# 第103回 問226

**問題文**

74歳女性。年齢を考えるとそろそろ骨がもろくなり、寝たきりになるのではないかと心配になった。

この女性が薬局を訪れ、サプリメントの摂取について相談した。薬局に取りそろえている以下の成分を含むサプリメントのうち、カルシウム（Ca）のほかに摂取する成分として優先度が高いのはどれか。２つ選べ。

**選択肢**

- １. ビタミンA
- ２. ビタミンC
- ３. ビタミンD
- ４. ビタミンE
- ５. ビタミンK

**正解:** 3、5

**解説**

選択肢のうち、カルシウムのほかに摂取する成分として優先度が高いのは、ビタミンDとビタミンKである。
ビタミンDは、体内で活性型ビタミンDに代謝されることで、血中のビタミンD結合タンパク質と結合し、標的組織に運ばれて細胞に取り込まれた後に、核内の受容体に結合して遺伝子の発現を亢進する。その結果、小腸ではカルシウム結合タンパク質の生合成が促進され、カルシウムの吸収が促進される。ビタミンDが不足すると、骨や歯の発育が不良となり、幼児ではくる病、成人では骨軟化症といった欠乏症が現れる。
ビタミンKは、骨基質タンパク質であるオステオカルシンのグルタミン酸残基のγ-カルボキシル化反応の補酵素として作用し、骨形成に関与している。
