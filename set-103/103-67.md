# 第103回 問67

**問題文**
エビデンスの明らかな情報を臨床現場で活用することを目的とし、治療や予防に関する無作為化比較試験のシステマティックレビューなどを収載したものはどれか。１つ選べ。

**選択肢**
1. TOXLINE
2. Drug Safety Update
3. MEDLINE
4. Cochrane Library
5. Physicians’ Desk Reference

**正解:** 4

**解説**
システマティックレビューとは、臨床での疑問（クリニカルクエスチョン）に対して、文献および研究を網羅的に調査し、無作為化比較試験（ランダム化比較試験）のような質の高い研究をまとめ、バイアスを評価しながら分析・統合することであり、その結果を収載している二次資料データベースがCochrane Library（コクランライブラリー）である。

１ 誤
TOXLINEは、副作用や中毒などの毒物学を収載している二次資料データベースであり、米国国立医学図書館が作成している。

２ 誤
Drug Safety Update（DSU：医薬品安全対策情報）は、添付文書の「使用上の注意」の改訂があったものを収載し毎月発行される速報誌であり、日本製薬団体連合会が作成している。

３ 誤
MEDLINEは、医学情報を収載している二次資料データベースであり、米国国立医学図書館が作成している。

４ 正
前記参照

５ 誤
Physicians’ Desk Reference（PDR）は、米国の代表的な医薬品の添付文書を要約・収載している三次資料である。
