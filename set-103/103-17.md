# 第103回 問17

**問題文**
食品中のトリプトファンから生じる腐敗臭の原因物質はどれか。１つ選べ。

**選択肢**
1. エチルメルカプタン
2. スカトール
3. スペルミジン
4. トリメチルアミン
5. ヒスタミン

**正解:** 2

**解説**
１ 誤
エチルメルカプタンは、食品中のシステインから生じる腐敗臭の原因物質である。

２ 正
スカトールは、食品中のトリプトファンから生じる腐敗臭の原因物質である。

３ 誤
スペルミジンは、納豆やヨーグルトに多く含まれている物質であり、分解されることにより独特のにおいが生じると考えられている。

４ 誤
トリメチルアミンは、魚に含まれるトリメチルアミンオキシドが還元されて生じる魚の生臭さの原因物質である。

５ 誤
ヒスタミンは、食品中のヒスチジンから生じる腐敗アミンである。
