# 第103回 問227

**問題文**

74歳女性。年齢を考えるとそろそろ骨がもろくなり、寝たきりになるのではないかと心配になった。

この女性に「骨粗しょう症は加齢とともに骨がもろくなり、進行しやすい病気なので、無理のない軽い運動を心がけてください」と指導した。この指導の根拠となる骨のリモデリングに関する記述のうち、正しいのはどれか。２つ選べ。

**選択肢**

- １. エストロゲンは、骨からのCa２＋遊離を促進する。
- ２. 骨細胞は、増殖能を有する未分化細胞であり、Ca２＋を細胞外へ分泌する。
- ３. 骨芽細胞は、コラーゲンを細胞外へ分泌して骨基質をつくる。
- ４. カルシトニンは、破骨細胞の機能を抑制して、骨形成に働く。
- ５. 負荷がかかる運動は、破骨細胞を活性化することで骨量を増加させる。

**正解:** 3、4

**解説**

骨は、常に骨芽細胞による骨形成と破骨細胞による骨吸収を繰り返して再構築（リモデリング）を行っている組織である。骨量は、骨芽細胞と破骨細胞の機能的平衡状態により維持されている。

１ 誤
エストロゲンは、カルシトニン分泌を促進することで骨からのCa2＋遊離（骨吸収）を抑制する。

２ 誤骨細胞は、骨芽細胞に骨基質が埋め込まれた細胞であり、骨組織において最も多く存在する細胞である。骨細胞は細胞分裂を行わないため、増殖能を持たない。また、骨芽細胞も細胞分裂を行わないため、増殖能を持たない。なお、骨芽細胞を形成する骨形成細胞は、間葉に由来した未分化細胞であり、増殖能を有する。

３ 正
骨芽細胞は、Ⅰ型コラーゲン、オステオカルシン、ヒドロキシアパタイト、アルカリ性ホスファターゼなどを分泌して骨基質をつくる。

４ 正
カルシトニンは、甲状腺傍ろ胞細胞から分泌されるペプチドホルモンであり、破骨細胞の働きを抑制することで骨吸収を抑制し、骨基質へのCa2＋の取り込みを促進して骨形成を促す。

５ 誤
負荷がかかる運動は、骨芽細胞を活性化することで骨形成を促進させ、骨量を増加させる。
