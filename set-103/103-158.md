# 第103回 問158

**問題文**
消化器に作用する薬物に関する記述のうち、正しいのはどれか。２つ選べ。

**選択肢**
1. モサプリドは、副交感神経のセロトニン5-HT4受容体を刺激することでアセチルコリンの遊離を増大させ、胃排出を促進する。
2. スクラルファートは、壁細胞のヒスタミンH2受容体を遮断することで胃酸分泌を抑制する。
3. ボノプラザンは、壁細胞のH＋,K＋-ATPaseのSH基と酸性環境で共有結合を形成することで胃酸分泌を抑制する。
4. カルメロースは、小腸粘膜上皮細胞のCl－チャネル（ClC-2）を活性化することで腸管内への水分分泌を促進する。
5. メトクロプラミドは、副交感神経終末のドパミンD2受容体を遮断し、ドパミンによるアセチルコリンの遊離抑制を解除することで胃運動を促進する。

**正解:** 1、5

**解説**
１ 正
モサプリドは、副交感神経のセロトニン5-HT4受容体を刺激し、副交感神経終末からのアセチルコリン遊離を増大させることで、消化管運動を亢進するため、胃内容物の胃排出を促進する。

２ 誤
スクラルファートは、防御因子増強薬であり、潰瘍底タンパク質と結合することで、保護層を形成し胃粘膜を保護する。なお、壁細胞のヒスタミンH2受容体を遮断することで胃酸分泌を抑制するのは、ファモチジンなどのヒスタミンH2受容体遮断薬である。

３ 誤
ボノプラザンは、壁細胞のH＋,K＋−ATPase（プロトンポンプ） のK＋と競合することにより、プロトンポンプを阻害して胃酸分泌を抑制する。なお、壁細胞のプロトンポンプのSH基と酸性環境で共有結合を形成することで胃酸分泌を抑制するのは、ランソプラゾールやオメプラゾールなどである。

４ 誤
カルメロースは、膨張性下剤であり、腸管内で水分を吸収して膨張し、腸内容物の容積を増大して腸管壁を刺激することで、腸管運動を促進する。なお、小腸粘膜上皮細胞のCl－チャネル（ClC-2）を活性化することで腸管内への水分分泌を促進するのは、ルビプロストンである。

５ 正
メトクロプラミドは、消化管における副交感神経終末のドパミンD2受容体を遮断することで、ドパミンによるアセチルコリンの遊離抑制を解除し、アセチルコリン遊離を増大させ胃運動を促進する。
