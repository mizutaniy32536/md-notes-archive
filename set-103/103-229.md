# 第103回 問229

**問題文**

74歳女性。年齢を考えるとそろそろ骨がもろくなり、寝たきりになるのではないかと心配になった。

骨の構成成分であるCaに関する記述のうち、正しいのはどれか。２つ選べ。

**選択肢**

- １. Caは、主に遊離イオン状態で骨や歯に存在する。
- ２. 骨に存在するオステオカルシンは、Caと結合する。
- ３. 腸管からのCaの吸収は、カゼインホスホペプチドにより阻害される。
- ４. 腸管からのCaの吸収は、シュウ酸やフィチン酸により亢進する。
- ５. パラトルモン（副甲状腺ホルモン）の過剰分泌により高Ca血症となることがある。

**正解:** 2、5

**解説**

１ 誤
Caは生体内で最も多く存在する無機質で、99％がヒドロキシアパタイト（リン酸カルシウムの一種）として骨や歯といった硬組織中に存在する。

２ 正
オステオカルシンは、骨芽細胞が分泌する骨基質タンパク質であり、ビタミンK依存性カルボキシラーゼによってグルタミン酸残基がγ-カルボキシル化を受けると、Ca結合能を獲得し、ヒドロキシアパタイトを吸着する。

３ 誤
カゼインホスホペプチド（CPP）は、Caが腸内でリン酸イオンと不溶性塩を形成するのを抑制し、腸管からのCa吸収を促進する。

４ 誤
シュウ酸やフィチン酸は、Caと不溶性塩を形成するため腸管からのCa吸収を阻害する。

５ 正
パラトルモンは、副甲状腺（上皮小体）から分泌されるペプチドホルモンであり、骨吸収促進作用や腎臓におけるCaの再吸収促進作用をもち、血中Ca濃度を上昇させる。よって、パラトルモンの過剰分泌により高Ca血症となることがある。
