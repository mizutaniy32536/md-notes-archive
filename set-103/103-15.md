# 第103回 問15

**問題文**
ヘリコバクター・ピロリが、アンモニアを産生する際に関与する主な酵素はどれか。１つ選べ。

**選択肢**
1. アルギナーゼ
2. ウレアーゼ
3. グルタミナーゼ
4. クレアチンキナーゼ
5. デアミナーゼ

**正解:** 2

**解説**
ヘリコバクター・ピロリが、アンモニアを産生する際に関与する主な酵素は、ウレアーゼである。ヘリコバクター・ピロリは、らせん状のグラム陰性微好気性の桿菌であり、ウレアーゼを産生して尿素を分解することでアンモニアを作り、胃酸を中和することにより胃粘膜〜粘膜下で増殖する。

１ 誤
アルギナーゼは、L-アルギニンをL-オルニチンと尿素に加水分解する酵素である。

グルタミナーゼは、L-グルタミンをL-グルタミン酸とアンモニアに加水分解する酵素である。

４ 誤
