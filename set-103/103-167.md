### 第103回薬剤師国家試験 問167

**問題文**  
薬物の消化管吸収とバイオアベイラビリティに関する記述のうち、正しいのはどれか。2つ選べ。[1][2][3]

**選択肢**  
1. カルバペネム系抗生物質であるメロペネムは、小腸粘膜での透過性が高いため、経口製剤として用いられる。[2][3]  
2. 陰イオン交換樹脂であるコレスチラミンは、酸性物質であるプラバスタチンを吸着し、その吸収を阻害する。[1][4]  
3. 経口製剤の絶対的バイオアベイラビリティは、その製剤を経口投与した際の血中濃度時間曲線下面積（AUC）を、等量の同一薬物を静脈内投与した際のAUCで除したものに等しい。[2][4]  
4. 小腸において、親水性薬物のみかけの吸収速度は、非撹拌水層の拡散速度に依存する。[1][2]  
5. 小腸下部から吸収された薬物は、門脈を経ずに下大静脈に入るために、肝初回通過効果を受けない。[1][2][3]

**正解番号**  
2, 3[1][2][4]

**各選択肢の解説**  
- **1**: 誤り。メロペネムはカルバペネム系抗生物質で注射剤であり、経口製剤ではない。小腸粘膜での透過性が高いという記述も不正確。[1]  
- **2**: 正しい。コレスチラミンは陰イオン交換樹脂で、酸性のプラバスタチンを吸着し、消化管からの吸収を阻害する。[1][4]  
- **3**: 正しい。絶対的バイオアベイラビリティ（F）は、経口投与時のAUCpoを静脈内投与時のAUCivで除した値（F = AUCpo / AUCiv）に等しい。[1][2][4]  
- **4**: 誤り。小腸での親水性薬物のみかけ吸収速度は、非撹拌水層ではなく消化管粘膜透過速度に依存する（非撹拌水層依存は疎水性薬物）。[1][4]  
- **5**: 誤り。小腸下部から吸収された薬物も門脈を経由し肝臓を通過するため、肝初回通過効果を受ける（直腸下部が例外）。[1][3][5]
