## ◆ インフルエンザ流行期の6歳女児への薬局薬剤師の適切な対応を問うシナリオ

260312

**問題**
2月のインフルエンザ流行期。6歳の娘が咳・38.0℃発熱・筋肉痛・倦怠感で母親が来局。
薬局の薬剤師の対応として適切なのはどれか。2つ選べ。

**選択肢**

1. アスピリンを解熱鎮痛成分として含有する一般用医薬品を販売した。
​
2. インフルエンザの疑いがあるとして、医療機関への受診勧奨を行った。
​
3. 半年前に近隣の医療機関から本人に処方された風邪薬を服用するように指導した。
​
4. 筋肉痛を緩和するために、一般用医薬品のジクロフェナク貼付剤を販売した。
​
5. 高熱が続くと脱水症状を起こすことがあるので、水分補給に努めるように指導した。

**正解と理由**
- `2. インフルエンザの疑いがあるとして、医療機関への受診勧奨を行った。`
- `5. 高熱が続くと脱水症状を起こすことがあるので、水分補給に努めるように指導した。`
- **理由の構成**
	- 選択肢2：インフルエンザ流行期＋典型症状→`受診勧奨`が適切
	- 選択肢5：脱水予防の`水分補給指導`は基本的なOTC対応として適切

**解法の手順（思考の流れ）**
- 選択肢1：`アスピリン`=小児のインフルエンザに`禁忌`（`ライ症候群`リスク）→ 絶対ダメ
- 選択肢2：流行期＋典型症状→インフルエンザ疑い→`受診勧奨` → 正しい
- 選択肢3：半年前の処方薬=期限・状態不明。自己判断での服用指導は不適切 → 誤り
- 選択肢4：`ジクロフェナク`（NSAIDs）のインフルエンザへの使用→重症化リスク（脳炎・脳症）の可能性 → 不適切
- 選択肢5：水分補給指導は適切 → 正しい

**他の選択肢が誤りな理由**
- 1. 小児へのアスピリン=`ライ症候群`（脳症＋肝障害）のリスク。インフルエンザ・水痘時は特に禁忌
- 3. 他者への処方薬を使用させる指導は不適切。処方薬は本人に処方されたものでも使い回しは原則NG
- 4. NSAIDs（ジクロフェナク等）のインフルエンザへの投与→インフルエンザ脳炎・脳症の悪化リスクがあるため注意が必要

**キーポイント**
- 小児+インフルエンザ疑い=`アスピリン禁忌`（`ライ症候群`）
- OTC薬剤師の基本：疑わしい症状→`受診勧奨`。自己解決させようとしない
- `解熱鎮痛薬`の選択：小児には`アセトアミノフェン`が安全

**関連知識の比較**
- `ライ症候群`：サリチル酸系（アスピリン等）×ウイルス感染（インフルエンザ・水痘）→ 急性脳症＋肝脂肪変性
- NSAIDs全般：インフルエンザ脳炎・脳症の重症化リスクがある（特に小児）
- `アセトアミノフェン`：年齢制限なし（適切用量で）。インフルエンザ時の解熱薬として最も安全

（一言メモ）
- 「インフルエンザ=アスピリン禁忌」は絶対覚える。ライ症候群はインフルエンザ+サリチル酸系
- 「流行期+典型症状→受診勧奨」は薬剤師の基本姿勢
