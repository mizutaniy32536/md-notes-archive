# 第103回 問23

**問題文**
微量元素セレンを含む生体防御因子はどれか。１つ選べ。

**選択肢**
1. スーパーオキシドジスムターゼ
2. NADPHオキシダーゼ
3. グルタチオンペルオキシダーゼ
4. カタラーゼ
5. メタロチオネイン

**正解:** 3

**解説**
１ 誤
スーパーオキシドジスムターゼは、構造中に銅及び亜鉛、マンガン又は鉄を含有し、スーパーオキシドを過酸化水素と酸素に変換する生体防御因子である。

２ 誤
NADPHオキシダーゼは、構造中にヘム鉄とFADを含有し、生体内に侵入した微生物の殺菌に用いられる活性酸素の生成に関与する生体防御因子である。

３ 正
グルタチオンペルオキシダーゼは、構造中にセレンを含有し、グルタチオンの存在下で過酸化水素を水に還元する生体防御因子である。

４ 誤
カタラーゼは、構造中に鉄を含有し、過酸化水素を酸素と水に変換する生体防御因子である。

５ 誤
メタロチオネインは、システインを中心としたアミノ酸から構成されるタンパク質であり、カドミウムや無機水銀などの重金属の解毒や活性酸素の除去を行う生体防御因子である。
