# 第103回 問54

**問題文**
腹膜透析用剤に関する記述のうち、正しいのはどれか。１つ選べ。

**選択肢**
1. pH調節剤を加えることはできない。
2. 等張化剤を加えることはできない。
3. 別に規定するもののほか、無菌試験法に適合する。
4. 静脈内に投与される。
5. 通例、密閉容器に保存する。

**正解:** 3

**解説**
腹膜透析用剤は、腹膜透析に用いる用剤である。腹膜透析は、患者自身の腹膜を使って、体内で血液を浄化する透析法である。

１ 誤
腹膜透析用剤は、塩酸や水酸化ナトリウムなどのpH調節剤を加えることができる。

２ 誤
腹膜透析用剤は、塩化ナトリウムなどの等張化剤を加えることができる。

３ 正
腹膜透析用剤は、腹膜透析に用いる無菌の透析用剤である。腹膜透析用剤は、患者の腹膜腔に注入するため、別に規定するもののほか、無菌試験法に適合する。

４ 誤
腹膜透析用剤は、静脈内に投与せず、カテーテルを通じて、患者自身の腹膜腔に注入する。

５ 誤
腹膜透析用剤に用いる容器は、通例、密封容器、又は必要に応じて、微生物の混入を防ぐことができる気密容器に保存する。
