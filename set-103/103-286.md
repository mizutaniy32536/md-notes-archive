# 第103回 問286

**問題文**

24歳男性。悪性リンパ腫に対して外来化学療法を実施予定である。外来化学療法室で、薬剤師がレジメンチェックを行った。

医師に確認又は提案すべき内容として適切なのはどれか。２つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等は e-REC の原ページを確認してください。

**選択肢**

- １. リツキシマブ注射剤の投与前に、B型肝炎ウィルス感染の有無を確認する。
- ２. シクロホスファミド水和物注射剤の投与後は、しびれなどの末梢神経障害の発現に注意する。
- ３. ドキソルビシン塩酸塩注射剤の投与が長期化する際には、総投与量（累積投与量）に注意する。
- ４. ビンクリスチン硫酸塩注射剤の投与後は、出血性膀胱炎の発現に注意する。

**正解:** 1、3

**解説**

１ 正
リツキシマブは治療中に、B型肝炎ウィルスキャリアの患者又は既往感染者での劇症肝炎や肝炎の増悪による死亡例により安全性報告（ブルーレター）が発出されている。
そのため、リツキシマブ注射剤の投与前には、B型肝炎ウィルス感染の有無を確認する必要がある。

２ 誤
シクロホスファミド水和物注射剤の投与後に注意する必要のある副作用としては、出血性膀胱炎や骨髄抑制があげられる。なお、本レジメンにおいて、しびれなどの末梢神経障害の発現に注意する必要のある薬剤はビンクリスチン硫酸塩である。

３ 正
ドキソルビシン塩酸塩は、総投与量が500 mg/m2を越えると重篤な心筋障害を生じる危険がある。そのため、投与が長期化する際には、総投与量（累積投与量）に注意する必要がある。
