# 第103回 問312

**問題文**
入院手術前に医師と協議の上、この薬剤師が薬学的管理をすることになった。上記の処方の中で、休薬の必要性が高いのはどれか。２つ選べ。

**選択肢**
1. 処方1
2. 処方2
3. 処方3
4. 処方4
5. 処方5

**正解:** 2、3

**解説**
本患者は、大腸がんが見つかり３ヶ月後に切除手術を受けることになっており、抗凝血作用や血小板凝集抑制作用を有する薬物を手術時に服用していると手術に伴う出血が助長されるおそれがあるため、手術前に休薬する必要がある。そのため、本問の選択肢のうち手術前に休薬する必要があるのは、血小板凝集抑制作用を有する薬物であるイコサペント酸エチル粒状カプセルとシロスタゾール錠である。

問313 解答 ２

１ 誤
当該患者は76歳であるため、第1号被保険者である。介護保険において、保険者である市町村又は特別区の区域内に住所を有する65歳以上の者を第1号被保険者、40歳以上65歳未満の医療保険加入者を第2号被保険者という。

２ 正
介護保険制度では、要介護認定を受けた場合に介護サービスを受けられ、要支援認定を受けた場合には介護予防サービスを受けられる。

３ 誤
要介護認定や要支援認定は、介護認定審査会の判定に基づいて、市町村又は特別区が行う。

４ 誤
要介護認定は疾病の重症度が判定基準とされるわけではなく、要介護認定等基準時間を基準として判定される。なお、要介護認定等基準時間とは、被保険者に対して行われる5分野（直接生活介助、間接生活介助、問題行動関連行為、機能訓練関連行為、医療関連行為）に要する一日当たりの時間として厚生労働大臣の定める方法により推計される時間である。

５ 誤
第1号被保険者の保険料は、市町村又は特別区が徴収する。なお、第2号被保険者の保険料は、被保険者の加入する医療保険の保険者が徴収し、市町村又は特別区に納付する。
