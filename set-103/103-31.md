# 第103回 問31

**問題文**
全身麻酔の導入時に、前投与薬（麻酔前投薬）として使用されない薬物はどれか。１つ選べ。

**選択肢**
1. ベクロニウム
2. ジアゼパム
3. レミフェンタニル
4. ピロカルピン
5. ドロペリドール

**正解:** 4

**解説**
１ 使用される
ベクロニウムは、骨格筋弛緩薬であり、麻酔時の筋弛緩の目的で麻酔前投薬として使用される。

２ 使用される
ジアゼパムは、ベンゾジアゼピン系抗不安薬であり、手術前の不安を除去する目的で麻酔前投薬として使用される。

３ 使用される
レミフェンタニルは、麻薬性鎮痛薬であり、全身麻酔薬の導入及び維持における鎮痛の目的で麻酔前投薬として使用される。

４ 使用されない
ピロカルピンは、副交感神経興奮薬であり、緑内障などに用いられるが、麻酔前投薬としては使用されない。なお、全身麻酔薬投与時には、気道分泌や唾液分泌などを抑制する目的で、抗コリン薬であるアトロピンやスコポラミンなどが麻酔前投薬として使用される。

５ 使用される
ドロペリドールは、ブチロフェノン系神経遮断薬であり、手術時の鎮痛の目的で麻酔前投薬として使用される。
