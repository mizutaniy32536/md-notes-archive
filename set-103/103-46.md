# 第103回 問46

**問題文**
糸球体ろ過速度に対する腎クリアランスの比がほぼ1に等しいのはどれか。１つ選べ。

**選択肢**
1. アンピシリン
2. イヌリン
3. インドシアニングリーン
4. グルコース
5. パラアミノ馬尿酸

**正解:** 2

**解説**
腎排泄は一般的に、糸球体ろ過、尿細管分泌、尿細管再吸収により行われるが、イヌリンの腎排泄は、糸球体ろ過のみによって行われる。糸球体ろ過による腎クリアランス（CL腎）は、糸球体ろ過速度（GFR）と血漿タンパク非結合率（fp）の積（CL腎 ＝ GRF × fp）によって算出されるが、イヌリンは血漿タンパクとはほとんど結合しないためfp ≒ 1となる。そのため、イヌリンの腎クリアランスはGFRに相当し、糸球体ろ過速度に対する腎クリアランスの比はほぼ1となる。
