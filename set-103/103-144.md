# 第103回 問144

**問題文**
薬剤師の業務に関する記述のうち、正しいのはどれか。２つ選べ。

**選択肢**
1. 処方された医薬品を備蓄していなかったので、調剤を拒否した。
2. 患者から薬袋不要の申出があったので、調剤した薬剤だけを交付した。
3. 処方せんを交付した医師の同意を得て薬剤を変更して調剤した。
4. 処方せんに発行の年月日の記載がなかったが調剤した。
5. 薬剤師免許証を紛失し、再交付申請中であるが、調剤した。

**正解:** 3、5

**解説**
１ 誤
調剤に従事する薬剤師は、調剤の求めがあった場合には、正当な理由がなければ、これを拒んではならない。処方された医薬品を備蓄していないだけでは、調剤を拒む正当な理由には該当しないため、調剤を拒否するのではなく、卸売業者や近隣の薬局等から医薬品を取り寄せて調剤する必要がある。調剤に従事する薬剤師が調剤を拒むことができる正当な理由を以下に示す。
【調剤を拒む正当な理由の例】
・冠婚葬祭、急病等で薬剤師が不在の場合
・病院・診療所の調剤所に、他の病院等の処方箋が持ち込まれた場合
・災害、事故等により調剤ができない場合 など

２ 誤
薬剤師は調剤した薬剤の容器又は被包（薬袋）に患者の氏名、用法、用量等の必要事項を記載することが必要であり、患者から薬袋不要の申出があった場合においても、調剤した薬剤を薬袋に入れて交付する必要がある。

３ 正
処方せんに記載された医薬品を変更して調剤する際は、その処方せんを交付した医師、歯科医師又は獣医師の同意を得る必要がある。

４ 誤
処方せんに発行の年月日などの法令で定められた事項が記載されていない場合には、薬剤師は処方医に問い合わせて、これを確認した後でなければ調剤してはならない。

５ 正
薬剤師名簿に登録されている者であれば、薬剤師業務に従事することができるため、薬剤師免許証の紛失等により薬剤師免許証の再交付申請中であっても、薬剤師は調剤を行うことができる。
