# 第103回 問53

**問題文**
下に示す包装形態の名称はどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 103-53 を参照

**選択肢**
1. SP（Strip Package）包装
2. バラ包装
3. PTP（Press Through Package）包装
4. ピロー包装
5. スティック包装

**正解:** 3

**解説**
１ 誤
SP包装は、アルミ箔あるいはセロファンに低密度ポリエチレンなどの熱可塑性高分子フィルムを重ね合わせたラミネートフィルムで作られた包装形態のことである。

２ 誤
バラ包装は、錠剤、カプセル剤などをプラスチック製ボトルなどの容器に直接入れた包装形態のことである。

３ 正
PTP包装は、凸面のある表面がポリ塩化ビニル（PVC）、裏面がアルミ箔からできており、表から押すことによって破れ、製剤を容易に取り出すことができるように作られた包装形態のことである。

４ 誤
ピロー包装は、SP包装や、PTP包装を施した医薬品を、さらに防湿性を図るため一定数量ずつ束ねラミネートフィルムなどで包装したものである。

５ 誤
スティック包装は、包材を筒状に成形して、上下を加熱圧着した包装形態のことである。
