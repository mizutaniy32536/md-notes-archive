# 第103回 問85

**問題文**
病棟で保管する以下の薬物を含む注射液のうち、夜間や休日等で必要な注意をする医療従事者がいない場合、保管するロッカーや引き出し等に施錠しなければならないのはどれか。１つ選べ。

**選択肢**
1. ゲンタマイシン硫酸塩
2. コンドロイチン硫酸エステルナトリウム
3. サリチル酸ナトリウム
4. フルルビプロフェン アキセチル
5. ペンタゾシン

**正解:** 5

**解説**
「夜間や休日等で必要な注意をする医療従事者がいない場合、保管するロッカーや引き出し等に施錠しなければならない」医薬品は向精神薬である。

１ 誤
ゲンタマイシン硫酸塩は、向精神薬に該当しない。

２ 誤
コンドロイチン硫酸エステルナトリウムは、向精神薬に該当しない。

３ 誤
サリチル酸ナトリウムは、 向精神薬に該当しない。

４ 誤
フルルビプロフェン アキセチルは、向精神薬に該当しない。

５ 正
ペンタゾシンは、第二種向精神薬に指定されているため、盗難防止について必要な注意をしている場合を除いて、鍵のかかる設備に保管しなければならない。
