# 第103回 問290

**問題文**

74歳男性。喘息にて近医から下記の薬剤（処方1及び処方2）が処方されていた。呼吸困難を自覚しており、禁煙したにもかかわらず、症状が改善しないため、呼吸器内科を受診したところ、新たにCOPD（慢性閉塞性肺疾患）と診断され、追加の処方（処方3）が行われた。

吸入剤の服薬指導に関する記述のうち、正しいのはどれか。２つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等は e-REC の原ページを確認してください。

**選択肢**

- １. 加圧式定量噴霧吸入器は吸気と噴霧の同調が必要でないため、任意のタイミングで吸入するように説明する。
- ２. ドライパウダー吸入器は自己の吸気で吸入を行うため、十分な吸気力があるかを確認する。
- ３. 吸入薬は、内服薬と同等の全身性の副作用があると伝える。
- ４. 吸入指導を行う場合は、口頭説明だけではなく、吸入練習器具を用いて実践させることが望ましい。
- ５. 喘息発作時にはオルベスコを使用するように伝える。

**正解:** 2、4

**解説**

１ 誤
加圧式定量噴霧吸入器は、容器に入った噴射剤により薬剤をエアゾールとして噴射する吸入剤であり、吸気と噴霧の同調が必要なため、薬剤噴射のタイミングに合わせてゆっくり吸入するように説明する。

２ 正
ドライパウダー吸入器は、充填された固体粒子をエアゾールとして自己の吸気で吸入を行う吸入剤であるため、十分な吸気力があるかを確認する必要がある。

３ 誤
吸入薬は、局所作用を目的とした剤型であり、内服薬と比べて全身性の副作用は少ないことを伝える。

４ 正
吸入薬は、吸入方法を理解せず手技を誤ると十分な効果が得られず、副作用の発現などにつながるおそれがある。そのため吸入指導を行う場合は、口頭説明だけではなく、吸入練習器具を用いて実践させ、しっかりと吸入方法を理解させることが望ましい。

５ 誤
オルベスコ（シクレソニド）は、副腎皮質ステロイド性薬であり、気道の慢性炎症を改善することで、気管支ぜん息の長期管理に用いられる薬剤（コントローラー）である。喘息発作時に使用する薬剤（リリーバー）ではない。
