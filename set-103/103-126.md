



## ◆ 悪性新生物の部位別年齢調整死亡率の推移とリスク要因を読み取る問題
260307

**問題**
図は、1955年から2015年までの全悪性新生物及び部位別にみた悪性新生物の年齢調整死亡率の年次推移を示したものである。A〜Fは、乳房、肺（気管、気管支及び肺）、胃、肝臓、大腸及び子宮のいずれかに対応している。これらの年次推移に関する記述のうち、適切なのはどれか。2つ選べ。

**選択肢**


1. Aの年齢調整死亡率が低下し続けている要因として、がんの早期発見や食生活の変化が考えられる。
​
2. Bの年齢調整死亡率が1990年代後半まで上昇した主な要因として、飲酒やウイルス感染の関与が考えられる。
​
3. Cの年齢調整死亡率が1990年代後半まで上昇した要因の1つとして、食事内容の欧米化が考えられる。
​
4. Eの年齢調整死亡率の低下の主な要因として、ワクチンの定期接種によるEの罹患率の低下が考えられる。
​
5. 近年、全悪性新生物の年齢調整死亡率が男女とも低下しているが、粗死亡率も同様に低下している。

**正解と理由**
- `1. Aの年齢調整死亡率が低下し続けている要因として、がんの早期発見や食生活の変化が考えられる。`
- `3. Cの年齢調整死亡率が1990年代後半まで上昇した要因の1つとして、食事内容の欧米化が考えられる。`
- **理由の構成**
	- **問題の整理**：図のA〜Fを部位に対応させると、A＝胃、B＝肺、C＝大腸、D＝肝臓、E＝子宮、F＝乳房となる
	- **薬学的知識・仕様・制約**：`胃がん`の死亡率低下は内視鏡検査の普及による早期発見と、`減塩`や冷蔵庫の普及による食生活の変化（塩蔵品の減少）が要因。`大腸がん`の死亡率上昇は食事の`欧米化`（肉類・脂肪の増加、食物繊維の減少）が主なリスク要因
	- **選択理由**：選択肢1はA（胃がん）の死亡率低下の要因として早期発見と食生活の変化を正しく指摘。選択肢3はC（大腸がん）の死亡率上昇の要因として食事内容の欧米化を正しく指摘

**解法の手順（思考の流れ）**
- まず図のパターンからA〜Fを部位に対応させる。一貫して低下しているAは`胃がん`、1990年代後半まで急上昇したBは`肺がん`と判断
- 1990年代後半まで上昇傾向のCは`大腸がん`、1990年代後半にピークを迎えたDは`肝臓がん`と判断
- 一貫して低下傾向のEは`子宮がん`、緩やかに上昇傾向のFは`乳がん`と判断
- 選択肢1：A（胃がん）の低下要因は早期発見と食生活の変化 → 正しい
- 選択肢2：B（肺がん）の主因は`喫煙`であり、飲酒やウイルス感染ではない → 誤り
- 選択肢3：C（大腸がん）の上昇要因は`食事の欧米化` → 正しい
- 選択肢4：E（子宮がん）の低下はHPVワクチン普及前であり、主因は`検診の普及` → 誤り
- 選択肢5：年齢調整死亡率は低下しているが、`粗死亡率`は高齢化により上昇 → 誤り

**他の選択肢が誤りな理由**
- 2. Bの年齢調整死亡率が1990年代後半まで上昇した主な要因として、飲酒やウイルス感染の関与が考えられる。
	B（`肺がん`）の死亡率上昇の主因は`喫煙`であり、特に男性の喫煙率の高さが大きく影響している。飲酒は`肝がん`、ウイルス感染は`肝がん`（B型・C型肝炎ウイルス）や`子宮頸がん`（HPV）のリスク要因であり、肺がんの主因ではない

- 4. Eの年齢調整死亡率の低下の主な要因として、ワクチンの定期接種によるEの罹患率の低下が考えられる。
	E（`子宮がん`）の死亡率低下の主因は子宮がん検診の普及による`早期発見・早期治療`である。`HPVワクチン`の定期接種は2013年に開始されたが、問題の図（2015年まで）の長期低下傾向はワクチン導入以前から続いており、ワクチンが主因とは言えない

- 5. 近年、全悪性新生物の年齢調整死亡率が男女とも低下しているが、粗死亡率も同様に低下している。
	全悪性新生物の`年齢調整死亡率`は低下しているが、`粗死亡率`は高齢化の進行によりむしろ上昇している。年齢構成の影響を受ける粗死亡率と、それを排除した年齢調整死亡率は異なる動きを示す

**キーポイント**
- 悪性新生物の部位別`年齢調整死亡率`の推移パターンを把握し、各がん種のリスク要因と対応させることが重要
- `胃がん`低下の要因：内視鏡検査による早期発見、`減塩`、冷蔵庫の普及（塩蔵品・漬物の減少）、`ヘリコバクター・ピロリ菌`除菌治療の普及
- `肺がん`上昇の主因は`喫煙`であり、1990年代後半以降の低下は禁煙運動の成果を反映している
- `大腸がん`上昇の要因は`食事の欧米化`（動物性脂肪の増加、食物繊維の減少）
- 全悪性新生物の粗死亡率は上昇しているが年齢調整死亡率は低下しており、高齢化の影響と治療技術の向上を反映している

**関連知識の比較**
- `肝がん`の死亡率は1990年代後半にピークを迎えた後に低下しており、`C型肝炎ウイルス`に対するインターフェロン療法や直接作用型抗ウイルス薬（DAA）の普及が寄与している
- `乳がん`の年齢調整死亡率は緩やかに上昇傾向にあり、`食生活の欧米化`や出産年齢の高齢化、授乳期間の短縮が要因として考えられている
- `子宮頸がん`のリスク要因は`HPV`（ヒトパピローマウイルス）感染であり、`HPVワクチン`は2013年から定期接種に位置づけられたが積極的勧奨が一時差し控えられた経緯がある
- 日本人のがん死亡原因の第1位は男性が`肺がん`、女性が`大腸がん`である（粗死亡率ベース）

（一言メモ）
- `胃がん`のリスク要因は`ヘリコバクター・ピロリ菌`感染と塩分の過剰摂取であり、除菌治療と`減塩`が予防の柱となる
- がんの`年齢調整死亡率`が低下しても`粗死亡率`が上昇するのは、高齢化によるがん罹患者数の増加が原因である
- `肺がん`の最大のリスク要因は`喫煙`であり、受動喫煙も含めた喫煙対策ががん死亡率低下の鍵となる
