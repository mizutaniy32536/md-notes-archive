# 第103回 問145

**問題文**
麻薬に関する記述のうち、正しいのはどれか。２つ選べ。

**選択肢**
1. ジアセチルモルヒネを輸入する場合、その都度、都道府県知事の許可を得なければならない。
2. 麻薬処方せんには、麻薬施用者の免許証の番号が記載されていなければならない。
3. 麻薬を用いた研究を行うには、都道府県知事から麻薬研究者の免許を受けなければならない。
4. ジヒドロコデインリン酸塩を10％含有する散剤は家庭麻薬に該当する。
5. 治療のため、麻薬を施用中の患者が外国に旅行する場合、麻薬の輸出ができないため処方変更が必要になる。

**正解:** 2、3

**解説**
１ 誤
ジアセチルモルヒネは、何人も輸入及び輸出することが禁止されている。また、同様に何人も輸入及び輸出することが禁止されているものには、あへん末や覚せい剤がある。

２ 正
麻薬施用者は、麻薬を記載した処方せんを交付するときには、その処方せんに患者の氏名、麻薬の品名、分量、用法用量、麻薬施用者の氏名、麻薬施用者の免許証の番号、患者の住所等を記載しなければならない。

３ 正
麻薬研究者とは、都道府県知事の免許を受けて、学術研究のため、麻薬原料植物を栽培し、麻薬を製造し、又は麻薬、あへん若しくはけしがらを使用する者をいう。

４ 誤
家庭麻薬とは、千分中十分以下（1％以下）のコデイン、ジヒドロコデイン又はこれらの塩類を含有するものであって、その他の麻薬を含有しないものをいう。

５ 誤
自己疾病の治療目的で麻薬を携帯して出国（輸出）や入国（輸入）する場合は、厚生労働大臣の許可を受けて行うことができる。
