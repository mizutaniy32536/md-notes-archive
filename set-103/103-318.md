# 第103回 問318

**問題文**
3日後、薬剤師が訪問したところ、家族から症状も改善されず痰もつまっていると報告を受けた。薬剤師が医師に対して行う提案として適切なのはどれか。２つ選べ。

**選択肢**
1. アジスロマイシン錠を2日間延長する。
2. アジスロマイシン錠をクラリスロマイシン錠に変更する。
3. L−カルボシステイン錠を追加する。
4. レバミピド錠を追加する。
5. ツロブテロール経皮吸収型テープを追加する。

**正解:** 3、5

**解説**
１ 誤
抗菌薬のアジスロマイシンは成人に1回500mgを1日1回3日間服用することにより、感受性菌に対して有効な組織内濃度が約7日間持続するため、処方延長は不要であると考えられる。

２ 誤
アジスロマイシンやクラリスロマイシンは、マクロライド系抗菌薬である。抗菌薬の変更や追加を行うのであれば、マクロライド系以外の抗菌薬を選択することが望ましい。

３ 正
L−カルボシステインは去痰薬である。本患者の家族から痰もつまっているとの報告があるため、設問の提案は適切である。

４ 誤
レバミピドは胃粘膜保護薬であり、本患者は胃痛や胃の不快感を訴えていないため、設問の提案は適切ではない。

５ 正
ツロブテロールは気管支拡張薬であり、本患者は気管支拡張薬であるテオフィリン徐放錠を3日間服用したにもかかわらず症状が改善していないため、設問の提案は適切である。

問319 解答 ３、４

１ 誤
保険薬局の保険薬剤師が医療保険における訪問薬剤管理指導を行う場合、保険薬局は地方厚生局長に届け出る必要がある。
なお、介護保険における居宅療養管理指導を行う場合、保険薬局は居宅療養管理指導を行う指定居宅サービス事業者の指定を受けたものとみなされるため、別段の手続きを行う必要はない。

２ 誤
当該患者が介護認定を受けている場合は訪問薬剤管理指導に係る給付（サービス）について、介護保険が優先される。

３ 正
保険薬局が行う訪問薬剤管理指導は、医師の指示が必要である。また、指示のあった医師に対しての訪問結果の報告は文書により行う。

４ 正
訪問薬剤管理指導を行う場合には、医師の指示に基づき、患者宅へ訪問前に保険薬剤師が薬学的管理指導計画を策定してから、保険薬剤師が患家を訪問して、薬学的管理指導を行う必要がある。

５ 誤
医療保険における訪問薬剤管理指導を行うにあたり、患者と薬局との契約書の作成は必要ない。なお、介護保険における居宅療養管理指導を行うには、患者と薬局との契約書を作成する必要がある。
