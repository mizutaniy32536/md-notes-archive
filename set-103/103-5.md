# 第103回 問5

**問題文**
有効数字を考慮した、2つの測定値1.231と0.32132の和はどれか。１つ選べ。

**選択肢**
1. 1.6
2. 1.55
3. 1.552
4. 1.5523
5. 1.55232

**正解:** 3

**解説**
有効数字の計算には、足し算・引き算による計算（加減法）と掛け算・割り算による計算（乗除法）がある。

本設問は加減法であり、有効数字は各数値の小数点以下の桁数が最も少ないものに合わせて計算する。
1.231（小数点以下3桁）＋0.32132（小数点以下5桁）＝1.55232≒1.552（小数点以下3桁に合わせる）
よって、選択肢３が正答となる。

なお、乗除法であれば、有効数字は各数値の有効数字の桁数が最も少ないものに合わせて計算する。
例）
1.231（有効数字4桁）×0.32132（有効数字5桁）＝0.39553392≒0.3955（有効数字4桁に合わせる）
