# 第103回 問56

**問題文**
肺腺がんの診断に有用な腫瘍マーカーはどれか。１つ選べ。

**選択肢**
1. NSE
2. CA125
3. CEA
4. PSA
5. AFP

**正解:** 3

**解説**
１ 誤
NSE(neuron specific enolase)は、主に小細胞肺がんや神経芽細胞腫において陽性を示す腫瘍マーカーである。

２ 誤
CA 125（carbohydrate antigen 125）は、主に卵巣がんや子宮がんにおいて陽性を示す腫瘍マーカーである。

３ 正
CEA(carcino embryonic antigen) は、主に肺腺がんや、胃がんや大腸がんなどの消化器系がんにおいて陽性を示す腫瘍マーカーである。

４ 誤
PSA（prostate specific antigen）は、主に前立腺がんにおいて陽性を示す腫瘍マーカーである。

５ 誤
AFP（α−fetoprotein）は、主に肝細胞がんにおいて陽性を示す腫瘍マーカーである。
