# 第103回 問116

**問題文**
ヒト免疫不全ウイルス（HIV）に関する記述のうち、正しいのはどれか。２つ選べ。

**選択肢**
1. HIVは、ゲノムとして二本鎖RNAをもつ。
2. HIVは宿主細胞表面のToll様受容体に結合し、細胞内に侵入する。
3. ウイルスがもつ逆転写酵素により生成したDNAは、宿主細胞の染色体に組み込まれる。
4. ウイルス表面に発現するノイラミニダーゼが、宿主細胞への吸着に必要である。
5. HIVは、CD4陽性T細胞に感染する。

**正解:** 3、5

**解説**
ヒト免疫不全ウイルス（HIV）は、レトロウイルス科に属するRNAウイルスであり、粒子中に逆転写酵素を有することにより、ウイルスRNAを鋳型にして相補的なDNA（cDNA）を合成する。合成されたDNAは、ウイルス由来のインテグラーゼによって宿主細胞の染色体に組み込まれる。

１ 誤
HIVは、ゲノムとして一本鎖RNAをもつ。

２ 誤
HIVは、宿主細胞表面のケモカイン受容体（CCR5など）及びCD4に結合し、細胞内に侵入する。なお、Toll様受容体は、マクロファージや好中球などの食細胞や樹状細胞などの免疫細胞が微生物の特徴的な構造（分子パターン）を認識する受容体である。

３ 正
前記参照

４ 誤
HIVの宿主細胞への吸着に関与するのは、HIVのエンベロープに含まれるgp120である。なお、ノイラミニダーゼは、A型及びB型インフルエンザウイルスのエンベロープに存在し、宿主細胞表面に存在するシアル酸を切断することで宿主細胞からの放出に関与する酵素である。

５ 正
HIVは、宿主細胞表面のケモカイン受容体（CCR5など）及びCD4に結合することから、これらの受容体を発現するCD4陽性T細胞やマクロファージに感染する。
