# 第103回 問169

**問題文**
Poor metabolizer（PM）において薬効が低下する薬物と代謝酵素の組合せとして正しいのはどれか。１つ選べ。

**選択肢**
（選択肢は画像を参照）

**正解:** 3

**解説**
１ 誤
チザニジンは、主にCYP1A2により代謝される。CYP1A2のPM（代謝活性の低い人）では、チザニジンの代謝が抑制されるため、薬効は増大する。

２ 誤
ワルファリンは、主にCYP2C9により代謝される。CYP2C9のPMでは、ワルファリンの代謝が抑制されるため、薬効は増大する。

３ 正
クロピドグレルは、主にCYP2C19により活性代謝物に代謝される。CYP2C19のPMでは、クロピドグレルの代謝が抑制され、活性代謝物の生成が抑制されるため、薬効は低下する。

４ 誤
イミプラミンは、CYP2D6により代謝されることで不活化されるほか、CYP1A2、CYP3A4及びCYP2C19により代謝されてデシプラミン（活性代謝物）となる。CYP2D6のPMでは、イミプラミンの不活化が抑制され、デシプラミンの生成が増大するため、薬効は増大する。

５ 誤
イリノテカンは、主にグルクロン酸転移酵素の一種であるUGT1A1により代謝される。UCT1A1のPMでは、イリノテカンの代謝が抑制されるため、薬効は増大する。
