# 第103回 問329

**問題文**
58歳男性。CD20陽性のびまん性大細胞型B細胞性リンパ種と診断され、R−CHOP療法による治療が行われることになり、薬剤師は以下の処方を確認した。1コース期間：3週間総コース数：6〜8コースd−クロルフェニラミンマレイン酸塩錠2 mg 1錠及びイブプロフェン錠200 mg 1錠を服用する。担当医師に提案すべき内容として、適切なのはどれか。２つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 103-329 を参照

**選択肢**
1. リツキシマブの点滴速度は少しずつ上げていく。
2. グラニセトロンは、リツキシマブの後に投与する。
3. ドキソルビシン塩酸塩の点滴速度は少しずつ上げていく。
4. d−クロルフェニラミンマレイン酸塩とイブプロフェンは、リツキシマブの投与開始30分前に投与する。

**正解:** 1、4

**解説**
１ 正
リツキシマブの初回投与時は、最初の30分は50 mg/hの速度で点滴静注を開始し、患者の状態を十分観察しながら、その後注入速度を30分毎に50 mg/hずつ上げて、最大400 mg/hまで速度を上げることができる。よって、点滴速度を200 mg/hからではなく、少しずつ上げていくよう提案するのは適切である。

２ 誤
グラニセトロンは、R−CHOP療法に使う薬剤のうち、シクロホスファミド水和物やドキソルビシン塩酸塩などの催吐リスクの高い抗悪性腫瘍薬の悪心・嘔吐の予防のために用いられ、シクロホスファミド水和物の投与前（リツキシマブの投与後）に投与する。ただし、本レジメンはそのように正しく記載されているため、薬剤師が担当医師に提案すべき内容としては適切ではない。

３ 誤
ドキソルビシン塩酸塩は、投与速度を変えることにより血管痛が生じることがあるため、投与速度は一定にしておく必要がある。よって、点滴速度を少しずつ上げていくよう提案するのは適切ではない。

４ 正
リツキシマブは初回投与時、高頻度（90％程度）でinfusion reactionを起こすため、その予防としてリツキシマブの投与開始30分前にd−クロルフェニラミンマレイン酸塩とイブプロフェンを投与する必要がある。よって、本提案は適切である。
