# 第103回 問51

**問題文**
日本薬局方において、口腔内で徐々に溶解又は崩壊させ、口腔、咽頭などの局所に適用する口腔用錠剤と規定されているのはどれか。１つ選べ。

**選択肢**
1. 溶解錠
2. トローチ剤
3. チュアブル錠
4. 口腔内崩壊錠
5. 分散錠

**正解:** 2

**解説**
日本薬局方において、各剤形は下記のように規定されている。

１ 誤
溶解錠は、水に溶解して服用する錠剤である。

２ 正
トローチ剤は、口腔内で徐々に溶解又は崩壊させ、口腔、咽頭などの局所に適用する口腔用錠剤である。

３ 誤
チュアブル錠は、咀嚼して服用する錠剤である。

４ 誤
口腔内崩壊錠は、口腔内で速やかに溶解又は崩壊させて服用できる錠剤である。

５ 誤
分散錠は、水に分散して服用する錠剤である。
