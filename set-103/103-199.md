# 第103回 問199

**問題文**

83歳男性。高齢者介護施設に入所しているが、肺炎のため入院となった。入院時、仙骨部に褥瘡が認められたことから、褥瘡ケアチームが対応した。感染の可能性がある黄色の滲出液が多かったため、精製白糖・ポピドンヨード配合軟膏を滅菌ガーゼに塗布し、創部への貼付処置をした。1週間後、褥瘡の診断を行ったところ、黄色の滲出液はなくなり、一部が黒色化した壊死組織と褥瘡部分の両方に乾燥傾向が認められた。

軟膏剤やクリーム剤は流体としての性質をもつ。図は流体におけるせん断応力（S）とせん断速度（D）の関係を表したグラフである。次の記述のうち、正しいのはどれか。２つ選べ。＜image1:center＞

**選択肢**

- １. ①の直線の傾きの逆数は、流体の粘度を表す。
- ２. ②の特性を示すものに、精製白糖・ポピドンヨード配合軟膏がある。
- ３. ③の特性を示すものに、デンプンの高濃度（50％以上）水性懸濁液がある。
- ４. ④の特性を示すものに、スルファジアジン銀クリームがある。
- ５. ⑤のグラフは、チキソトロピーを表す。

**正解:** 1、3

**解説**

縦軸にせん断速度D、横軸にせん断応力Sをとり、D−S関係を示すグラフのことをレオグラムという。レオグラムはニュートン流動と非ニュートン流動とで大きく二つに分けられる。ニュートン流動は設問のグラフ①で示され、非ニュートン流動は設問のグラフ②、③、⑤で示される。
なお、グラフ②は準粘性流動、グラフ③はダイラタント流動、グラフ⑤は塑性（ビンガム）流動である。

１ 正
グラフ①で示されるニュートン流動（粘性流動）は、せん断速度Dとせん断応力Sが比例する関係にあり、レオグラムは原点を通る直線であるため、以下の関係式が成り立つ。

したがって、縦軸にせん断速度D、横軸にせん断応力Sをとりグラフを作成する場合、直線の傾きの逆数は粘度（η）を示す。

２ 誤
グラフ②で示される準粘性流動を示す物質の例には、約1％のメチルセルロース、約1％のカルメロースナトリウムなどがある。

３ 正
グラフ③で示されるダイラタント流動を示す物質の例には、デンプンの高濃度（50％以上）水性懸濁液がある。

４ 誤
一般に軟膏やクリーム剤は塑性流動を示す。したがって、スルファジアジン銀クリームは塑性（ビンガム）流動を示すため、グラフ⑤のレオグラムになる。

５ 誤
グラフ⑤は塑性（ビンガム）流動を示す。本問のチキソトロピーとは、力を加えると粘度が減少し、流動しやすくなり、これを放置するとゆるやかに元の状態に戻ることをいう。そのため、上昇曲線（応力増加時の流動曲線）と下降曲線（応力減少時の流動曲線）は一致しない。チキソトロピーを示すのは準粘性流動や準塑性流動などであり、塑性流動はチキソトロピーを示さない。
