# 第103回 問100

**問題文**
図は、ある酸素化学種の分子軌道のエネルギー準位と電子配置を模式的に表したものである。この酸素化学種はどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 103-100 を参照

**選択肢**
1. 一重項酸素
2. 三重項酸素
3. スーパーオキシド（O2－）
4. 過酸化物イオン（O22－）
5. オゾン

**正解:** 3

**解説**
図の横線は分子軌道を表し、上下の短い矢印はそれぞれ電子を表している。短い矢印が17本描かれていることから、電子を17個有する酸素化学種である。

１ 誤
一重項酸素（1O2）は酸素分子であり、電子を16個有している。

２ 誤
三重項酸素（3O2）は酸素分子であり、電子を16個有している。

３ 正
スーパーオキシド（O2－）は電子を17個有している。

４ 誤
過酸化物イオン（O22－）は電子を18個有している。

５ 誤
オゾン（O3）は電子を24個有している。
※分子軌道において、一重項酸素（1O2）と三重項酸素（3O2）はスピンの向きは異なるが酸素（O2）分子であり、電子数は同じである。
