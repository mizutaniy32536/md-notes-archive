# 第103回 問35

**問題文**
バソプレシンV2受容体を遮断する利尿薬はどれか。１つ選べ。

**選択肢**
1. スピロノラクトン
2. ヒドロクロロチアジド
3. アゾセミド
4. トリアムテレン
5. トルバプタン

**正解:** 5

**解説**
１ 誤
スピロノラクトンは、アルドステロン受容体を遮断することで、遠位尿細管から集合管においてNa＋-K＋交換系を抑制し利尿作用を示す。

２ 誤
ヒドロクロロチアジドは、チアジド系利尿薬であり、遠位尿細管においてNa＋−Cl－共輸送系を抑制し利尿作用を示す。

３ 誤
アゾセミドは、ループ利尿薬であり、ヘンレ係蹄上行脚におけるNa＋−K＋−2Cl－共輸送系を抑制し利尿作用を示す。

４ 誤
トリアムテレンは、遠位尿細管及び集合管において、アルドステロンとは無関係に管腔側膜のNa＋チャネルを抑制することで、Na＋-K＋交換系を抑制し利尿作用を示す。

５ 正
トルバプタンは、バソプレシンV2受容体を遮断することで、腎集合管において水の再吸収を抑制し利尿作用を示す。
