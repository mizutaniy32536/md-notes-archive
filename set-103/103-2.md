# 第103回 問2

**問題文**
化合物の親水性や疎水性を表す指標となる分配係数の測定において、水と組み合わせて用いられる有機溶媒はどれか。１つ選べ。

**選択肢**
1. アセトニトリル
2. アセトン
3. 1−オクタノール
4. グリセロール
5. メタノール

**正解:** 3

**解説**
分配係数PO/Wは、化合物の親水性や疎水性を表す指標として用いられ、互いに混ざり合わない2液相（有機溶媒と水）間で、平衡時における分配される薬物の割合を示し、以下の式で表される。ただし、平衡時における有機溶媒相中の溶質の分子形濃度を［分子形］油、水相中の溶質の分子形濃度を［分子形］水とする。

分配係数の測定に使用される有機溶媒は、水と混和しない溶媒を用いる必要がある。

１ 誤
アセトニトリルは水と混和するため、分配係数の測定に用いない。

２ 誤
アセトンは水と混和するため、分配係数の測定に用いない。

３ 正
1−オクタノールは水と混和しないため、分配係数の測定に用いる。

４ 誤
グリセロールは水と混和するため、分配係数の測定に用いない。

５ 誤
メタノールは水と混和するため、分配係数の測定に用いない。
