# 第103回 問212

**問題文**

85歳女性。再発・再燃を繰り返す逆流性食道炎の治療を受けている。この患者の処方箋を、家族が薬局に持参した。家族より、最近飲み込む力が弱まり服用の際に苦労しているので、薬が飲みやすくなる方法はないかとの相談を受けた。

薬剤師の対応として、適切なのはどれか。２つ選べ。

**選択肢**

- １. オメプラゾール錠を正確に半錠に分割して調剤する。
- ２. オメプラゾール錠を乳鉢で粉砕して調剤する。
- ３. ラベプラゾールNa錠へ変更し、乳鉢で粉砕して調剤することを医師に提案する。
- ４. ランソプラゾールカプセルへ変更し、脱カプセルして調剤することを医師に提案する。
- ５. ランソプラゾール口腔内崩壊錠への変更を医師に提案する。

**正解:** 4、5

**解説**

１ 誤
オメプラゾール錠等のプロトンポンプ阻害剤は、胃酸による分解を抑える目的で腸溶性コーティングが施されている製剤である。そのため、半錠に分割した場合、コーティングが損なわれ、分割または粉砕して調剤すべきではない。

４ 正
ランソプラゾールカプセルは、腸溶性顆粒を充てんした製剤であるため、脱カプセルによる作用の影響は小さいと考えられる。そのため、脱カプセルしての調剤を医師に提案することは適切である。

５ 正
ランソプラゾール口腔内崩壊錠は、腸溶性の粒を含む製剤であり、唾液で崩壊させてから服用する。本剤は飲み込むための力があまり必要なく、嚥下力が弱っている患者でも服用しやすいため、ランソプラゾール口腔内崩壊錠への変更を医師に提案することは適切である。
