# 第103回 問345

**問題文**
35歳男性。乗物酔い防止薬の購入のため薬局を訪れた。男性は、自分と5歳の子供の両方が服用できる一般用医薬品を希望している。子供は錠剤やカプセル剤を服用できる。薬剤師がこの男性に勧める医薬品の成分と用量として適切なのはどれか。２つ選べ。

**正解:** 1、2

**解説**
１ 正
選択肢の成分を含む一般用医薬品は、乗り物酔い防止薬のマイトラベル錠である。本剤は成人で1回3錠、5歳以上11歳未満の小児で1回1錠を服用する。よって、本問の男性と5歳の子供の両方が服用できる乗り物酔い防止薬として、薬剤師がこの男性に勧めるのは適切である。

２ 正
選択肢の成分を含む一般用医薬品は、乗り物酔い防止薬のトラベルミン®ファミリーである。本剤は成人で1回2錠、5歳以上11歳未満の小児で1回1錠を服用するとなっている。よって、本問の男性と5歳の子供の両方が服用できる乗り物酔い防止薬として、薬剤師がこの男性に勧めるのは適切である。

３ 誤
選択肢の成分を含む一般用医薬品は、乗り物酔い防止薬のアネロン「ニスキャップ」である。選択肢の成分のうち、アミノ安息香酸エチルは、乳幼児（6歳未満）への投与によりメトヘモグロビン血症を引き起こすとの報告があり、医療用医薬品において6歳未満の乳幼児に禁忌となっている。また、本剤は、15歳未満の小児は服用しないこととされている。よって、薬剤師がこの男性に勧めるのは適切ではない。

４ 誤
選択肢の成分を含む一般用医薬品は、乗り物酔い防止薬のウット®である。選択肢の成分のうち、ブロモバレリル尿素は、小児における安全性が確立しておらず、医療用医薬品において小児に慎重投与となっている。また、本剤は、15歳未満の小児は服用しないこととされている。よって、薬剤師がこの男性に勧めるのは適切ではない。

５ 誤
選択肢の成分を含む一般用医薬品は、総合風邪薬のコルゲンコーワIB2である。本剤は乗り物酔い防止薬ではないので、薬剤師がこの男性に勧めるのは適切ではない。
