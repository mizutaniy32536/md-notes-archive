# 第103回 問340

**問題文**
58歳男性。尿路結石の既往歴あり。健康診断で尿酸値が高いことを指摘され、受診を勧められた。現在は痛風関節炎等の症状は認められないが、近医を受診した。検査の結果、尿酸排泄低下型の高尿酸血症と診断され、薬物治療を行うことになった。検査データ：尿酸値 9.3 mg/dL、eGFR 23 mL/分/1.73 m2、AST 35 U/L、ALT 33 U/L、LDH 230 U/L、ALP 340 U/L、γ−GTP 65 U/L本患者の治療に用いられる薬物として最も適切なのはどれか。１つ選べ。

**選択肢**
1. フェブキソスタット
2. ブコローム
3. プロベネシド
4. ベンズブロマロン
5. ラスブリカーゼ（遺伝子組換え）

**正解:** 1

**解説**
本患者は検査データのeGFRが23 mL/分/1.73 m2であることから高度に腎機能が低下しており、また尿路結石の既往歴もあるため、尿酸排泄促進薬は禁忌となる。そのため、本患者の治療には尿酸合成阻害薬を用いる。

１ 正
フェブキソスタットは尿酸合成阻害薬であるため、本患者の治療に用いられる。

２ 誤
ブコロームは尿酸排泄促進薬であるため、本患者の治療には用いられない。

３ 誤
プロベネシドは尿酸排泄促進薬であるため、本患者の治療には用いられない。

４ 誤
ベンズブロマロンは尿酸排泄促進薬であるため、本患者の治療には用いられない。

５ 誤
ラスブリカーゼは尿酸分解酵素製剤であり、がん化学療法に伴う高尿酸血症の改善には用いられるが、本症例のような通常の高尿酸血症の治療には用いられない。
