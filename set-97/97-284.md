# 第97回 問284

**問題文**
70歳男性。マイコプラズマ肺炎の治療のため、以下の薬剤が処方された。
スクリーンショット 2017-09-26 16.46.44.png
問284（実務）
両薬剤をそれぞれ秤量した後、分包する方法として、最も適切なのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 97-284 を参照

**選択肢**
1. 乳鉢と乳棒を用いて混和し、分包する。
2. 乳鉢とスパーテルを用いて混和し、分包する。
3. 混和せずに2段分割（2度撒き）で分包する。
4. 混和せずに別々に分包する。

**正解:** 4

**解説**
ドライシロップは、原則、散剤と混和せず、単独で分包する。よって、エリスロマイシンドライシロップ10％とビオフェルミンR散が同時に処方されている場合、両剤を混合せず別々に分包する。
