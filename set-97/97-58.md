# 第97回 問58

**問題文**
インフルエンザの薬物治療に関する記述のうち、正しいのはどれか。1つ選べ。

**選択肢**
1. ザナミビル水和物は、B型の患者に有効である。
2. アスピリンは、小児の解熱薬として推奨される。
3. アマンタジン塩酸塩は、B型の患者に有効である。
4. ニューキノロン系抗菌薬が第一選択薬である。
5. オセルタミビルリン酸塩は、症状発現直後の使用では有効性がない。

**正解:** 1

**解説**
1　正
ザナミビル水和物は、ノイラミニダーゼ阻害薬であり、A型及びB型両方の患者に有効である。
2　誤
アスピリンは、ライ症候群を引き起こすとの報告例があるため、15歳未満のインフルエンザ患者には原則投与しない。なお、インフルエンザにおける小児の解熱薬としては、アセトアミノフェンが推奨される。
3　誤
アマンタジン塩酸塩が阻害するM2イオンチャネルはA型のみにしか存在しないため、B型には無効である。
4　誤
ニューキノロン系抗菌薬は抗細菌薬であるため、インフルエンザウイルスには無効である。
5　誤
本剤は、症状発現から48時間経過後に投与を開始した患者における有効性を裏付けるデータが得られていないため、症状発現後速やかに投与を開始する。
