# 97-248

**問題文**
67歳男性。災害時、救護所に本人のお薬手帳を持参し、医師に処方を求めた。お薬手帳を確認したところ、エナラプリルマレイン酸塩錠を服用していたことが判明した。救護所にはエナラプリルマレイン酸塩錠が置いていなかった。エナラプリルマレイン酸塩錠の代替薬として、以下の在庫品目のうち、薬剤師が医師に提案する最も適切な薬剤はどれか。1つ選べ。

**選択肢**
1. トラネキサム酸カプセル
2. バルサルタン錠
3. デキストロメトルファン臭化水素酸塩水和物錠
4. ニトログリセリン舌下錠
5. セフジニルカプセル

**正解:** 2

**解説**
1. 誤 トラネキサム酸は止血薬であり代替にならない。
2. 正 バルサルタンはARBで、ACE阻害薬エナラプリルの代替候補として適切である。
3. 誤 デキストロメトルファンは鎮咳薬である。
4. 誤 ニトログリセリンは狭心症治療薬である。
5. 誤 セフジニルは抗菌薬である。

**キーポイント**
- エナラプリルはACE阻害薬、高血圧治療薬。
- 災害時の代替提案では同系統または近い降圧機序の薬を選ぶ。
