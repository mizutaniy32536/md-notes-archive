## ◆ 高木兼寛の脚気研究の疫学的手法と歴史的事実を問うシナリオ

260311

**問題**
以下の研究に関する記述のうち、正しいのはどれか。2つ選べ。
1883年、遠洋航海に出ていた練習船の乗員376名のうち169名が脚気に罹るという事態が発生した。海軍医務局長の高木兼寛は、脚気の原因が食事の質であると考え、特に食事中の炭素：窒素比の値が15を超えると発生することに注目した。
1884年、季節、乗組員、航路などの条件を一致させ、白米主体の食事から麦飯や洋食に変えただけで航海を実施したところ、脚気はほとんど発生せず、全員無事に帰国した。鈴木梅太郎が脚気の予防因子を米糠から抽出するのに成功したのは、その25年以上も後のことである。

**選択肢**

1. この研究は、疫学の症例対照研究に相当する。
​
2. この研究では、脚気の予防に有効な因子は炭水化物である可能性が示唆される。
​
3. この研究では、白米に脚気の原因物質が含まれている可能性が排除できない。
​
4. 鈴木梅太郎が得た抽出物の有効成分は、のちにビタミンB1であることがわかった。

**正解と理由**
- `3. この研究では、白米に脚気の原因物質が含まれている可能性が排除できない。`
- `4. 鈴木梅太郎が得た抽出物の有効成分は、のちにビタミンB1であることがわかった。`
- **理由の構成**
	- **問題の整理**：高木兼寛の脚気研究の研究デザインと、脚気の原因に関する歴史的事実の正誤を判断する
	- **薬学的知識・仕様・制約**：高木の研究は「条件を一致させて食事のみを変える」という`介入研究`（準実験的研究）に相当する。`ビタミンB1`（チアミン）欠乏が脚気の原因であり、鈴木梅太郎が米糠から`オリザニン`を抽出し、後にビタミンB1と同定された
	- **選択理由**：選択肢3は「白米に原因物質が含まれる可能性の排除不可能性」→ 白米精製でビタミンB1が除かれる可能性を排除できない → 正しい。選択肢4は`ビタミンB1`の歴史的事実 → 正しい

**解法の手順（思考の流れ）**
- 選択肢1「症例対照研究」→ 条件を揃えて食事のみ変えた → `介入研究`（準実験）に相当 → 誤り
- 選択肢2「炭水化物が予防因子」→ 白米（炭水化物中心）で脚気多発、麦飯・洋食で予防 → 炭水化物は予防因子ではない → 誤り
- 選択肢3「白米に原因物質がある可能性」→ 食事変更で予防されたが、白米に有害物質があるか、白米にはない栄養素が欠けているかは区別できない → 両方の可能性があり排除不可 → 正しい
- 選択肢4「鈴木梅太郎の抽出物＝ビタミンB1」→ `オリザニン`=`チアミン`（ビタミンB1）は歴史的事実 → 正しい

**他の選択肢が誤りな理由**
- 1. この研究は、疫学の症例対照研究に相当する。
	症例対照研究は疾患の有無で群を分けて過去の曝露を遡る研究。高木の研究は「条件を一致させて食事だけを変えた」前向きの`介入研究`（半実験的研究）に相当する

- 2. この研究では、脚気の予防に有効な因子は炭水化物である可能性が示唆される。
	白米主体（高C/N比＝炭水化物過多）で脚気が多発し、麦飯・洋食（タンパク質多含）で予防された。炭水化物は「予防因子」ではなく「リスク因子または欠乏を促す因子」として解釈される

**キーポイント**
- 高木兼寛の研究は`介入研究`（準実験的研究）の先駆け。「条件を揃えて1因子だけ変える」が特徴
- `脚気`の原因：`ビタミンB1`（チアミン）欠乏。白米精製によりB1を含む米糠が除去される
- `鈴木梅太郎`が米糠から`オリザニン`（後のビタミンB1）を1910年に抽出
- 研究デザインの歴史：高木（1884年実験）→ 鈴木梅太郎（1910年オリザニン抽出）→ ビタミンB1同定

**関連知識の比較**
- `ビタミンB1`（チアミン）欠乏：`脚気`（末梢神経炎、心不全）、`ウェルニッケ脳症`（アルコール依存症に合併）
- `ビタミンB2`（リボフラビン）欠乏：`口角炎`、`舌炎`、`皮膚炎`
- `ビタミンB3`（ナイアシン）欠乏：`ペラグラ`（皮膚炎・下痢・認知症の3D）
- `ビタミンB12`欠乏：`悪性貧血`（巨赤芽球性貧血）、神経障害

（一言メモ）
- 高木兼寛の研究は「`症例対照研究`ではなく`介入研究`」が頻出ひっかけ。「条件を揃えた」「食事を変えた」= 介入と覚える
- 「`白米に原因物質`が含まれる可能性が排除できない」は、ビタミン欠乏説（精製により除去）と毒物混入説の両方を区別できないという意味
- `オリザニン`=`チアミン`=`ビタミンB1`：薬剤師試験では「オリザニン」の名前も問われることがある
