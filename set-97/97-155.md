# 第97回 問155

**問題文**
全身麻酔薬に関する記述のうち、正しいのはどれか。1つ選べ。

**選択肢**
1. 亜酸化窒素は、最小肺胞内濃度（MAC）が大きく、酸素欠乏症を起こしやすい。
2. エンフルランは、ハロタンに比べ、カテコールアミンによる心室性不整脈を誘発しやすい。
3. プロポフォールは、GABAB受容体を刺激し、速やかな麻酔作用を示す。
4. チオペンタールは、代謝及び排泄が速やかなため、作用が短時間で消失する。
5. ケタミンは、グルタミン酸NMDA受容体を刺激し、意識の解離をもたらす。

**正解:** 1

**解説**
1　正
亜酸化窒素は、最小肺胞濃度（MAC）が大きい吸入麻酔薬であるため、麻酔作用を示すには高濃度の吸入が必要となる。そのため、酸素欠乏症をきたしやすく、通常20%程度の酸素と併用する。
2　誤
エンフルランは、ハロタンと比較してカテコールアミン感受性増大作用が弱く、心室性不整脈を誘発しにくい。
3　誤
プロポフォールは、麻酔の導入及び覚醒が速やかな静脈麻酔薬であり、GABAA受容体-Cl－チャネル複合体において、GABAの作用を増強することで麻酔作用を示す。
4　誤
チオペンタールは、脂溶性が高く、脳以外の脂肪組織への再分配が速やかなため作用が短時間で消失する。
5　誤
ケタミンは、グルタミン酸NMDA型受容体を遮断する静脈麻酔薬であり、大脳皮質は徐波化する一方で、大脳辺縁系は覚醒化し、意識の解離状態をもたらす。
