# 第97回 問72

**問題文**
薬剤師のみが資格要件を満たすのはどれか。1つ選べ。

**選択肢**
1. 薬局の管理者
2. 店舗販売業の店舗管理者
3. 医薬部外品製造販売業の総括製造販売責任者
4. 生物由来製品の製造管理者
5. 麻薬管理者

**正解:** 1

**解説**
薬剤師のみが資格要件を満たすのは、薬局の管理者である。
1　正
2　誤
店舗販売業の店舗管理者は、薬剤師又は登録販売者である。
3　誤
医薬部外品製造販売業の総括製造販売責任者は、厚生労働省令で定める基準に該当する者である。
4　誤
生物由来製品の製造管理者は、医師、細菌学的知識を有する者その他の技術者として厚生労働大臣の承認を受けた者である。
5　誤
麻薬管理者は、医師、歯科医師、獣医師又は薬剤師である。
