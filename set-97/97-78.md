# 第97回 問78

**問題文**
地域薬局の役割に該当しないのはどれか。1つ選べ。

**選択肢**
1. セルフメディケーションの支援
2. 医薬品の販売
3. 調剤
4. 地域住民の健康診断
5. 在宅医療への参画

**正解:** 4

**解説**
地域薬局の役割に該当しないのは、地域住民の健康診断である。地域住民の健康診断は、主に病院や診療所で行われる。
地域薬局の役割として、医薬品の販売、医薬品の調剤、医薬品の適正使用の推進、在宅医療・居宅医療への参画、一般用医薬品の販売、健康相談によるセルフメディケーションの支援などがある。
