# 第97回 問16

**問題文**
過剰に摂取すると、悪心、嘔吐、頭痛などを主症状とする急性中毒を起こすのはどれか。1つ選べ。

**選択肢**
1. ビタミンA
2. ビタミンB12
3. ビタミンD
4. ビタミンE
5. ビタミンK

**正解:** 1

**解説**
過剰に摂取すると、悪心、嘔吐、頭痛などを主症状とする急性中毒を起こすのは、ビタミンAである。ビタミンAの過剰摂取により、急性中毒の症状として、悪心、嘔吐、頭痛などが起こることがあり、慢性中毒の症状として、頭蓋内圧亢進、脱力などが起こることがある。
なお、ビタミンDの過剰摂取により、中毒症状として、高カルシウム血症による悪心・嘔吐、腎不全などが起こることがあるが、ビタミンB12、ビタミンE、ビタミンKの過剰摂取による中毒症状は、特に知られていない。
