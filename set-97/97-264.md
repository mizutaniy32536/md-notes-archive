# 97-264

**問題文**
36歳女性。術後の病理検査により卵巣癌Ⅰc期と診断され、パクリタキセルとカルボプラチンの併用療法が予定されている。処方2のデキサメタゾン錠の投与目的として、正しいのはどれか。1つ選べ。

**選択肢**
1. 化学療法に伴う骨髄抑制の軽減
2. 化学療法に伴う消化器症状（悪心・嘔吐）の軽減
3. 化学療法に伴う感染症の予防
4. 抗炎症作用による化学療法の効果の増強
5. 化学療法に伴う血栓形成の予防

**正解:** 2

**解説**
解答　2

パクリタキセル及びカルボプラチンでは悪心・嘔吐が起こりうるため、デキサメタゾンは5-HT3受容体遮断薬などとともに前投与され、制吐目的で用いられる。
