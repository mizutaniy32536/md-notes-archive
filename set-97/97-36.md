# 97-36

**問題文**
ヒスタミンH2受容体遮断作用を示すのはどれか。1つ選べ。

**選択肢**
1. メトクロプラミド
2. ファモチジン
3. モサプリド
4. スルピリド
5. プログルミド

**正解:** 2

**正解と理由**
ファモチジンは代表的なヒスタミンH2受容体遮断薬であり、胃酸分泌を抑制する。

**他の選択肢が誤りな理由**
1. メトクロプラミドは主にドパミンD2受容体遮断作用を示す。
3. モサプリドは5-HT4受容体刺激作用を示す。
4. スルピリドはドパミンD2受容体遮断作用を示す。
5. プログルミドはガストリン受容体遮断薬であり、H2遮断薬ではない。
