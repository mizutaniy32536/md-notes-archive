# 第97回 問136

**問題文**
フロン類について、数値が0であるのはどれか。2つ選べ。

**選択肢**
1. 特定フロン分子内の水素原子数
2. 代替フロン分子内のフッ素原子数
3. 我が国で工業的製造が許可されている特定フロン種の数
4. パーフルオロカーボンの地球温暖化係数

**正解:** 1,3

**解説**
フロンとは、メタンやエタンなどのハイドロカーボンの水素が塩素やフッ素などのハロゲンで置換された
化合物の総称である。
1　正
特定フロンであるクロロフルオロカーボン（CFC）は、Cl、F、Cから構成されるフロンであり、分子内に水素原子を含まない。
2　誤
代替フロンには、ハイドロクロロフルオロカーボン（HCFC）とハイドロフルオロカーボン（HFC）があり、いずれも分子内にフッ素原子を含む。
3　正
モントリオール議定書では、オゾン層を破壊しやすい特定フロンの生産量と消費量を段階的に削減するスケジュールが示されており、現在、我が国では特定フロンの工業的製造が禁止されている。
4　誤
地球温暖化係数とは、二酸化炭素を基準（二酸化炭素の地球温暖化係数：1）として、他の温室効果ガスがどれだけ地球温暖化作用を有するかを表した数字である。パーフルオロカーボンの地球温暖化係数は6,500〜9,200である。
