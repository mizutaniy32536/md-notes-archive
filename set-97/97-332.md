# 第97回 問332

**問題文**
42歳男性。重症筋無力症に対して、以下の薬剤が処方された。重大な副作用の典型的な初期症状はどれか。2つ選べ。
スクリーンショット 2017-04-28 16.27.27.png

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 97-332 を参照

**選択肢**
1. 排尿困難
2. 便秘
3. 発汗
4. 頻脈
5. 縮瞳

**正解:** 3,5

**解説**
ジスチグミン臭化物は、可逆的コリンエステラーゼ阻害薬であり、重大な副作用として、コリン作動性クリーゼを起こすことがある。コリン作動性クリーゼの初期症状として、下痢、発汗、徐脈、縮瞳、唾液分泌過多などが認められる。
