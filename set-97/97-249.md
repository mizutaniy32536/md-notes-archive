# 97-249

**問題文**
67歳男性。災害時、救護所に本人のお薬手帳を持参し、医師に処方を求めた。お薬手帳を確認したところ、エナラプリルマレイン酸塩錠を服用していたことが判明した。救護所にはエナラプリルマレイン酸塩錠が置いていなかった。エナラプリルの薬理作用の機序として、正しいのはどれか。2つ選べ。

**選択肢**
1. アンギオテンシン変換酵素を阻害して、アンギオテンシンⅡの生成を抑制する。
2. 一酸化窒素合成酵素を阻害して、一酸化窒素の生成を抑制する。
3. キニナーゼⅡを阻害して、ブラジキニン量を増加させる。
4. ホスホリパーゼA2を阻害して、プロスタグランジンの生成を抑制する。

**正解:** 1,3

**解説**
1. 正 エナラプリルはACEを阻害し、アンギオテンシンII生成を抑制する。
2. 誤 一酸化窒素合成酵素阻害はしない。
3. 正 ACEはキニナーゼIIでもあるため、その阻害でブラジキニン量が増加する。
4. 誤 ホスホリパーゼA2阻害はしない。

**キーポイント**
- ACE阻害薬はアンギオテンシンII低下とブラジキニン増加の2本柱で覚える。
- 空咳の副作用はブラジキニン増加と結び付ける。
