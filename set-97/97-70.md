# 第97回 問70

**問題文**
加齢に伴い増大するのはどれか。1つ選べ。

**選択肢**
1. 肺活量
2. 体脂肪率
3. 腎血漿流量
4. 胃酸分泌量
5. 血漿中アルブミン濃度

**正解:** 2

**解説**
1　誤
加齢に伴い肺機能が低下するため、肺活量は減少する。
2　正
加齢に伴い筋肉が脂肪に代謝されるため、体脂肪率は増大する。
3　誤
加齢に伴い腎機能が低下するため、腎血漿流量は減少する。
4　誤
加齢に伴い胃酸分泌機能が低下するため、胃酸分泌量は低下し、胃内pHが上昇する。
5　誤
加齢に伴い肝機能が低下するため、アルブミン合成能が低下し、血漿中アルブミン濃度は減少する。
