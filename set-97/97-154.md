# 第97回 問154

**問題文**
知覚神経系に作用する薬物に関する記述のうち、正しいのはどれか。1つ選べ。

**選択肢**
1. コカインは、血管拡張作用を持つため、局所麻酔作用の持続時間が短い。
2. プロカインは、皮膚・粘膜浸透力が強いエステル型局所麻酔薬で、表面麻酔に用いられる。
3. テトラカインは、非イオン型が神経細胞膜の内側から作用し、電位依存性Na＋チャネルを遮断する。
4. オキセサゼインは、強酸性下でも局所麻酔作用を示し、胃潰瘍に伴う疼痛を緩和する。
5. リドカインは、血中エステラーゼによる代謝物がアレルギー反応を起こしやすい。

**正解:** 4

**解説**
1　誤
コカインは、交感神経終末のアミントランスポーター阻害作用を有するため、交感神経興奮作用を介して血管収縮作用を示す。血管拡張作用を有する局所麻酔薬と比較すると作用持続時間が延長されるが、血漿コリンエステラーゼ（血中エステラーゼ）による分解を受けるため、その作用持続時間は30分程度と短い。
2　誤
プロカインは、皮膚・粘膜浸透力が弱く表面麻酔としては用いられない。伝達麻酔、硬膜外麻酔、浸潤麻酔などにより用いられる。
3　誤
テトラカインなどの局所麻酔薬は、非イオン型で知覚神経細胞内に入り、神経細胞膜の内側で陽イオン型となり電位依存性Na＋チャネルを遮断する。
4　正
オキセサゼインは、その他の局所麻酔薬と異なり強酸性下でも局所麻酔作用を発揮するため、経口投与が可能である。ガストリン分泌を抑制し胃酸分泌を低下させるため、胃潰瘍に伴う疼痛の緩和に用いられる。
5　誤
リドカインは、アミド型局所麻酔薬であり血漿コリンエステラーゼによる分解を受けない。局所麻酔薬によるアレルギー反応は、プロカインなどのエステル型局所麻酔薬で見られやすい。
