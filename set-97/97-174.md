# 第97回 問174

**問題文**
薬物の溶解及び製剤からの放出に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**
1. ヒグチ（Higuchi）の式において、放出される薬物の累積量は時間の平方根に比例する。
2. ヒクソン-クロウェル（Hixson-Crowell）の式は、粒度分布を持つ粉体の溶解現象を表す式である。
3. 固体分散体中の薬物は、その薬物結晶に比べて溶解速度が小さい。
4. 安定形の結晶は、準安定形の結晶に比べて溶解速度が大きい。
5. 無水物は、水和物に比べて水中での溶解速度が大きい。

**正解:** 1,5

**解説**
1　正
ヒグチ（Higuchi）の式は、マトリックス製剤の累積薬物放出量Qを表す式であり、次式で表される。
Q＝[D・（2A－Cs）]・Cs・t1/2
D：マトリックス中の薬物の拡散係数、A：マトリックス中の全薬物量
Cs：マトリックス中の薬物の溶解度、t：時間
2　誤
ヒクソン-クロウェル（Hixson-Crowell）の式は、シンク条件下、薬物が同一粒子径の球形粒子からなると仮定した場合の粉体の溶解現象を表す式である。
3　誤
固体分散体とは、薬物を水溶性高分子に分散させたものであり、その薬物結晶に比べて溶解速度が大きい。
4　誤
安定形の結晶は、準安定形の結晶に比べて溶解度が小さいため、溶解速度が小さい。
5　正
無水物は、水和物に比べ溶解度が大きいため、無水物は、水和物に比べ水中での溶解速度が大きい。
