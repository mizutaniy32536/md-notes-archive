# 第97回 問47

**問題文**
薬物を除去する能力を表すパラメーターで、血流速度と同じ単位をもつのはどれか。1つ選べ。

**選択肢**
1. 分布容積
2. 消失半減期
3. 消失速度定数
4. 血中濃度－時間曲線下面積
5. クリアランス

**正解:** 5

**解説**
薬物を除去する能力を表すパラメーターには、消失半減期、消失速度定数、クリアランスなどがあり、それぞれの単位には、時間、時間—1、容積・時間—1を用いる。血流速度の単位は、容積・時間—1であることから、血流速度と同じ単位をもつ薬物を除去する能力を表すパラメーターは、クリアランスである。
