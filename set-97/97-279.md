# 第97回 問279

**問題文**
医薬品として未承認のヒドロキノンの軟膏剤を院内製剤として調製してほしいと、皮膚科の医師から薬剤部に依頼があった。
問279（薬剤）
薬剤部で油脂性基剤を用いて調製することになった。基剤として、正しいのはどれか。2つ選べ。

**選択肢**
1. バニシングクリーム
2. コールドクリーム
3. マクロゴール軟膏
4. プラスチベース
5. 白色ワセリン

**正解:** 4,5

**解説**
油脂性基剤として用いられるものには、白色ワセリン、プラスチベース、白色軟膏、流動パラフィンなどがある。
1　誤
バニシングクリームは、o/w型のクリーム剤である。
2　誤
コールドクリームは、w/o型のクリーム剤である。
3　誤
マクロゴール軟膏は、水溶性基剤である。
4　正
