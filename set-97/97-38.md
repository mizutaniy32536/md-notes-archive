# 97-38

**問題文**
ヒスタミンH1受容体遮断作用を持たないケミカルメディエーター遊離抑制薬はどれか。1つ選べ。

**選択肢**
1. クロルフェニラミン
2. プロメタジン
3. クロモグリク酸
4. ジフェンヒドラミン
5. ケトチフェン

**正解:** 3

**解説**
1. 誤 クロルフェニラミンは第一世代抗ヒスタミン薬である。
2. 誤 プロメタジンはH1受容体遮断作用を持つ。
3. 正 クロモグリク酸はケミカルメディエーター遊離抑制薬であり、H1受容体遮断作用は持たない。
4. 誤 ジフェンヒドラミンは第一世代抗ヒスタミン薬である。
5. 誤 ケトチフェンはH1受容体遮断作用を持つ。

**キーポイント**
- クロモグリク酸は「遊離抑制」で覚える。
- 第一世代抗ヒスタミン薬との見分けが頻出。
