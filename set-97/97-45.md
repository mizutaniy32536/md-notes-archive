# 第97回 問45

**問題文**
主として未変化体のまま体内から尿中に排泄されるのはどれか。1つ選べ。

**選択肢**
1. ゲンタマイシン
2. テオフィリン
3. ニフェジピン
4. フェニトイン
5. リドカイン

**正解:** 1

**解説**
主として未変化体のまま体内から尿中に排泄される薬物（腎消失型薬物）には、アミノグリコシド系抗生物質（ゲンタマイシン、カナマイシンなど）、アセタゾラミド、ジゴキシン、炭酸リチウムなどがある。また、主として肝の代謝により消失する薬物（肝消失型薬物）には、テオフィリン、ニフェジピン、フェニトイン、リドカイン、プロプラノロール、アミオダロン、ワルファリンなどがある。
