# 第97回 問189

**問題文**

**正解:** 4

**解説**
患者より「1週間前からトイレによく行き水をたくさん飲むようになりました。」との訴えがあり、また、空腹時血糖が358 mg/dL（基準値：70〜110 mg/dL）であることから、統合失調症治療薬であるオランザピンによる副作用である高血糖が出現していると考えられる。
SOAP形式では、S（Subjuctive：主訴や患者の訴えなどの主観的情報）、O（Objective：理学的所見などの客観的情報）、A（Assessment：評価、分析など）、P（Plan：検査や治療の指針、今後の治療計画など）にわけて患者情報を記載する。
SOAP形式でこの患者の指導記録を作成した場合、Sには「1週間前からトイレによく行き水をたくさん飲むようになりました。」（患者の訴え）、Oには「空腹時血糖358 mg/dL」（検査値）、Aには「血糖値が高く、自覚症状から、薬剤性高血糖が疑われる。」（SやOから得られる薬剤師の判断）、Pには「オランザピン錠を直ちに中止し、代替薬を考慮する。」（Aに基づいた問題解決のための計画）を記入する。
