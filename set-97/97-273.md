問273（薬剤）
定常状態におけるフェニトインの体内からの消失速度はMichaelis−Menten式で表される。この患者における最大消失速度(mg/day)に最も近い値はどれか。1つ選べ。ただし、Michaelis定数を8 mg/L、バイオアベイラビリティを100％とする。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 97-273 を参照

2. 240
3. 420
4. 1,500

**正解:** 3

**解説**
Michaelis-Menten式は、①式で表される。
