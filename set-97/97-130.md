## ◆ 性行為感染症の中でウイルスが病原体のものを選ぶシナリオ
260312

**問題**
次の性行為感染症のうち、病原体がウイルスであるのはどれか。2つ選べ。

**選択肢**

1. 梅毒
​
2. 軟性下疳
​
3. 淋病
​
4. 尖圭コンジローマ
​
5. B型肝炎

**正解と理由**
- `4. 尖圭コンジローマ`
- `5. B型肝炎`

**解法の手順（思考の流れ）**
- 選択肢1：梅毒 → 梅毒トレポネーマ（`スピロヘータ`＝細菌の一種）→ ✗
- 選択肢2：軟性下疳 → 軟性下疳菌（Haemophilus ducreyi）`細菌` → ✗
- 選択肢3：淋病 → 淋菌（Neisseria gonorrhoeae）`細菌` → ✗
- 選択肢4：尖圭コンジローマ → `ヒトパピローマウイルス(HPV)`＝ウイルス → **○**
- 選択肢5：B型肝炎 → `B型肝炎ウイルス(HBV)`＝ウイルス → **○**

**キーポイント**
- 性感染症のウイルス性：`HPV`（尖圭コンジローマ・子宮頸がん）、`HBV`（B型肝炎）、`HIV`、`HSV`（性器ヘルペス）
- 梅毒の病原体は「ウイルス」と誤答しやすい → `スピロヘータ（細菌の一種）`

**関連知識の比較**

| 性感染症 | 病原体 | 種類 |
|---|---|---|
| 梅毒 | 梅毒トレポネーマ | スピロヘータ（細菌） |
| 軟性下疳 | H. ducreyi | 細菌 |
| 淋病 | 淋菌 | 細菌 |
| 尖圭コンジローマ | **HPV** | **ウイルス** |
| B型肝炎 | **HBV** | **ウイルス** |
| 性器クラミジア | クラミジア・トラコマチス | 細菌（偏性細胞内寄生） |

（一言メモ）
- 梅毒 = スピロヘータ（細菌）。「ウイルスじゃないの？」は典型的な誤解
- HPV = 子宮頸がんの原因でもある。ワクチンあり（定期接種）
