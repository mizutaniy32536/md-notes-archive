# 第97回 問118

**問題文**
免疫に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**
1. ナチュラルキラー（NK）細胞は、あらかじめ抗原感作を受けなくとも、腫瘍細胞やウイルス感染細胞を傷害する。
2. マクロファージの表面にあるToll様レセプターは、細菌の菌体成分の識別のための受容体としてはたらく。
3. 好中球が同一の異物により反復刺激を受けると、記憶細胞となり食作用が増強される。
4. T細胞は、細胞表面に免疫グロブリンをもたないため、抗原を認識することができない。
5. 肥満細胞（マスト細胞）は、細胞表面の主要組織適合遺伝子複合体（MHC）クラスⅡ分子により抗原を認識する。

**正解:** 1,2

**解説**
1　正
ナチュラルキラー（NK）細胞は、自然免疫を担う細胞であり、あらかじめ抗原感作を受けなくとも、腫瘍細胞やウイルス感染細胞に対して細胞傷害活性を示す。
2　正
マクロファージの表面にあるToll様レセプター（TLR）は、抗原（細菌の菌体成分やウイルス産物）を識別するパターン認識受容体としてはたらいている。
3　誤
好中球は、自然免疫を担っており、同一の異物により反復刺激を受けても記憶細胞とはならない。なお、同一の異物により反復刺激を受けることにより記憶細胞となるのは、B細胞やT細胞である。
4　誤
T細胞は、細胞表面に免疫グロブリンをもたないが、T細胞受容体（TCR）を有するため、抗原を認識することが可能である。なお、B細胞は、細胞表面に免疫グロブリンを有し、抗原を認識することが可能である。
5　誤
肥満細胞（マスト細胞）は、その細胞表面にIgE抗体のFc領域が結合できる受容体が発現しており、この受容体にIgE抗体が結合することで、抗原を認識する。なお、主要組織適合遺伝子複合体（MHC）クラスⅡ分子は、樹状細胞、B細胞、マクロファージなどの抗原提示細胞で発現する。
