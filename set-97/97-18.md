# 第97回 問18

**問題文**
疾病の二次予防に該当するのはどれか。1つ選べ。

**選択肢**
1. 健康教室
2. 予防接種
3. 集団検診
4. 在宅機能訓練
5. 職場環境の改善

**正解:** 3

**解説**
疾病の予防は、健康な者を対象に健康増進及び特定疾患の発症予防、生活環境の改善によって病気の発症を未然に防ぐことを目的とした一次予防、疾病が不顕性の段階で早期発見・早期治療を行うことを目的とした二次予防、患者を対象に合併症等を併発して悪化するのを防止し、障害された機能を正常に戻し、早く社会復帰できるようにすることを目的とした三次予防の3つに大別される。
1　誤
健康教室は、健康な者を対象に健康増進を目的とする一次予防に該当する。
2　誤
予防接種は、健康な者を対象に特定疾患の発症予防を目的とする一次予防に該当する。
3　正
集団検診は、疾病が不顕性の段階で早期発見・早期治療を行うことを目的とする二次予防に該当する。
4　誤
在宅機能訓練は、患者を対象に合併症等を併発して悪化するのを防止し、障害された機能を正常に戻し、早く社会復帰できるようにすることを目的とする三次予防に該当する。
5　誤
職場環境の改善は、職場環境の改善によって病気を未然に防ぐことを目的とした一次予防に該当する。
