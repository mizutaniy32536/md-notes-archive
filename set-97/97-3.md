# 第97回 問3

**問題文**
Ag2CrO4の溶解度がS（mol/L）であるとき、溶解度積（Ksp）と溶解度の関係式として、正しいのはどれか。1つ選べ。

**選択肢**
1. Ksp ＝ 2S
2. Ksp ＝ S 2
3. Ksp ＝ 2S 2
4. Ksp ＝ 2S 3
5. Ksp ＝ 4S 3

**正解:** 5

**解説**
難溶性電解質MaXbの溶解度をSとすると、溶解したMaXbは完全に解離していると考えられるため、溶液中の[Mb＋]はaS、[Xa−]の濃度は、bSとなる。また、MaXbの溶解度積Kspは飽和溶液中における陽イオンと陰イオンの濃度の積（[Mb＋]a[Xa−]b＝[aS]a[bS]b）で表される。したがって、Ag2CrO4（溶解度：S）の溶解度積Kspは以下のように表すことができる。
Ksp＝[Ag＋]2[CrO42−]＝[2S]2[S]＝4S 3
