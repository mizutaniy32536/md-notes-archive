# 第97回 問294

**問題文**
62歳男性。高血圧症。冠動脈疾患治療のため、2年前に経皮的冠動脈形成術を受け、薬物を服用していた。その後、症状悪化のため、開胸心臓手術の適応となり手術目的で入院した。
問294（病態・薬物治療）
以下の薬物のうち、手術前に出血予防のため休薬期間を要し、さらにCYP2C19遺伝子多型により体内動態が影響を受ける薬物はどれか。1つ選べ。

**選択肢**
1. マニジピン塩酸塩
2. アスピリン
3. アテノロール
4. テルミサルタン
5. クロピドグレル硫酸塩

**正解:** 5

**解説**
手術前に出血予防のため休薬期間を要する薬物として、抗凝固薬（ワルファリンカリウムやヘパリンなど）や抗血小板薬（アスピリンやクロピドグレル硫酸塩など）のような抗血栓薬が挙げられる。
選択肢の薬物の中で手術前に休薬する必要があるものは、アスピリン、クロピドグレル硫酸塩であり、このうちクロピドグレル硫酸塩が主としてCYP2C19で代謝される。よって、手術前に出血予防のため休薬期間を要し、さらにCYP2C19遺伝子多型により体内動態が影響を受ける薬物はクロピドグレル硫酸塩である。
