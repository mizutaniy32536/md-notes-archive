# 第97回 問69

**問題文**
遺伝子多型がワルファリンの薬効に最も影響する薬物代謝酵素はどれか。1つ選べ。

**選択肢**
1. CYP1A2
2. CYP2C9
3. CYP2C19
4. CYP2D6
5. CYP3A4

**正解:** 2

**解説**
CYPの分子種の中で、遺伝子多型が確認されているものとして、CYP2C9、CYP2C19、CYP2D6があり、そのうちワルファリンの代謝・薬効に最も影響するのはCYP2C9である。なお、CYP2C19の代謝にはオメプラゾールやクロピドグレルなどが、CYP2D6の代謝にはイミプラミンやコデインなどが影響する。
