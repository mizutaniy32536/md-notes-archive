# 第97回 問276

**問題文**
病院薬剤部において､以下のA〜Dの注射剤を購入して保管することになった。
スクリーンショット 2017-05-01 9.06.26.png
問276（実務）
保管管理方法について、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 97-276 を参照

**選択肢**
1. Aは、常に施錠して保管しなければならない。
2. Bは、常に施錠して保管しなければならない。
3. Cは、室温で保存可能であり、その有効期間は検定合格の日から3年である。
4. Dは、医療従事者が常時在室するなど注意している場合を除き、施錠して保管しなければならない。

**正解:** 1,4

**解説**
1　正
毒薬は、他の物と区別して貯蔵し、貯蔵する場所には施錠しなければならない。
2　誤
劇薬は、他の物と区別して貯蔵する必要があるが、貯蔵する場所に施錠する必要はない。
3　誤
人血清アルブミン製剤は、室温での保存が可能であり、その有効期限は検定合格の日から2年である。
4　正
向精神薬は、医療従事者が常時在室するなど注意している場合を除き、施錠して保管しなければならない。
