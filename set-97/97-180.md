# 第97回 問180

**問題文**
日本薬局方の製剤試験法に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**
1. 注射剤の採取容量試験法は、内容物が容器に表示量どおりに正確に充てんされていることを確認する試験法である。
2. 点眼剤の不溶性異物検査法は、不溶性異物の大きさ及び数を測定する方法である。
3. 眼軟膏剤には、無菌試験法が適用される。
4. 軟膏剤には、鉱油試験法は適用されない。

**正解:** 3,4

**解説**
1　誤
注射剤の採取容量試験法は、表示量よりやや過剰に採取できる量が容器に充てんされていることを確認する試験法である。
2　誤
点眼剤の不溶性異物検査法は、点眼剤中の不溶性異物の有無を調べる検査法である。なお、点眼剤中の不溶性異物の大きさ及び数を測定する方法は、点眼剤の不溶性微粒子試験法である。
3　正
無菌試験法は、眼軟膏剤以外に点眼剤、注射剤に適用される。
4　正
鉱油試験法は、注射剤及び点眼剤に用いる非水性溶剤中の鉱油を試験する方法であり、軟膏剤には適用されない。
