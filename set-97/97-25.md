# 第97回 問25

**問題文**
医療機関により廃棄される“血液の付着したガーゼ（未滅菌）”が該当する区分として、最も適切なのはどれか。1つ選べ。

**選択肢**
1. 産業廃棄物
2. 特別管理産業廃棄物
3. 事業系一般廃棄物
4. 家庭系一般廃棄物
5. 特別管理一般廃棄物

**正解:** 5

**解説**
廃棄物処理法において、廃棄物は、産業廃棄物（事業活動に伴って生じた廃棄物であり、廃棄物処理法で規定された20種類）と一般廃棄物（産業廃棄物以外のもの）に大別される。また、廃棄物のうち、爆発性、毒性、感染性その他の人の健康又は生活環境に係る被害を生ずるおそれがある性状を有する廃棄物を特別管理一般廃棄物及び特別管理産業廃棄物として扱う。
医療機関により廃棄される“血液の付着したガーゼ（未滅菌）”は、感染性を有する一般廃棄物であるため、特別管理一般廃棄物に該当する。
