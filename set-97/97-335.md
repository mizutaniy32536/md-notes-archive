# 第97回 問335

**問題文**
循環器系疾患の薬物治療に関して、薬剤師が医師へ提案する内容として、最も適切なのはどれか。1つ選べ。

**選択肢**
1. 徐脈を伴う重症心不全の患者に対して、心機能改善を期待してプロプラノロール塩酸塩を使用する。
2. 心房細動を伴うWPW（Wolff–Parkinson–White）症候群のある患者に対して、抗不整脈作用を期待してベラパミル塩酸塩を使用する。
3. QT時間が延長している患者に対して、心室頻拍への移行防止を期待してソタロール塩酸塩を使用する。
4. 肥大型閉塞性心筋症の患者に対して、心筋収縮力の増強を期待してジゴキシンを使用する。
5. アンギオテンシン変換酵素阻害薬による治療を受けている慢性心不全の患者に対して、予後改善効果を期待してカルベジロールを使用する。

**正解:** 5

**解説**
1　誤
徐脈を伴う重症心不全の患者に対して、プロプラノロール塩酸塩などのアドレナリンβ受容体遮断薬を投与すると、徐脈及び重症心不全が悪化することがある。そのため、プロプラノロール塩酸塩は、高度な徐脈及び重症心不全の患者に対して投与禁忌である。
2　誤
心房細動を伴うWPW（Wolff–Parkinson–White）症候群のある患者に対して、ベラパミル塩酸塩を投与すると、副伝導路への伝導を促進してしまい、心室細動を生じることがある。そのため、ベラパミル塩酸塩は、WPW症候群の患者に対して慎重投与である。
3　誤
QT時間が延長している患者に対して、ソタロール塩酸塩を投与すると、過度のQT延長により不整脈を起こすことがある。そのため、ソタロール塩酸塩は、QT時間が延長している患者に対して投与禁忌である。
4　誤
肥大型閉塞性心筋症の患者に対して、ジゴキシンを投与すると、左室流出路閉塞を悪化させることがある。そのため、ジゴキシンは、肥大型閉塞性心筋症の患者に対して投与禁忌である。
5　正
アンギオテンシン変換酵素阻害薬による治療を受けている慢性心不全の患者に対して、カルベジロールを投与すると、慢性心不全の予後を改善することがあるため、予後改善効果を期待して用いられる。
