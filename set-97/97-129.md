## ◆ 感染性疾患と感染源の正しい組合せを選ぶシナリオ
260312

**問題**
感染性疾患と、主な感染源の組合せのうち、正しいのはどれか。2つ選べ。

**選択肢**

1. ジフテリア　－　気道由来の飛沫
​
2. 成人T細胞白血病　－　母乳
​
3. エキノコックス症　－　ノミ
​
4. E型肝炎　－　血液

**正解と理由**
- `1. ジフテリア　－　気道由来の飛沫`
- `2. 成人T細胞白血病　－　母乳`

**解法の手順（思考の流れ）**
- 選択肢1：ジフテリア = ジフテリア菌による上気道感染症 → `飛沫感染` → **正しい**
- 選択肢2：成人T細胞白血病（ATL）= HTLV-1感染 → `母乳・性交・輸血`が主経路 → **正しい**
- 選択肢3：エキノコックス症 = 寄生虫エキノコックス → キツネ・犬の糞便から経口感染。`ノミ`ではない → **誤り**
- 選択肢4：E型肝炎 = E型肝炎ウイルス → `経口感染`（汚染食物・ブタ肉）。血液ではない → **誤り**

**キーポイント**
- `HTLV-1`（成人T細胞白血病の原因）→ 母乳感染が重要（九州・沖縄に多い）
- `エキノコックス`→ キツネの糞便に汚染された山菜・水 → 経口感染
- `E型肝炎`→ ブタ・シカ肉の生食 → 経口感染

**関連知識の比較**

| 感染症 | 正しい感染源 |
|---|---|
| ジフテリア | 飛沫 |
| 成人T細胞白血病 | **母乳**（性交・輸血も） |
| エキノコックス症 | キツネ糞便からの経口感染 |
| E型肝炎 | 経口（ブタ肉等） |

（一言メモ）
- エキノコックス = 北海道に多い、キツネが媒体。ノミは関係ない
- HTLV-1の母乳感染は100-19と共通の知識
