# 第97回 問313

**問題文**
薬局における医薬品の取扱いについて、以下の問に答えよ。
問313（法規・制度・倫理）
メチルフェニデート塩酸塩錠に当てはまり、かつ、トリアゾラム錠に当てはまらないのはどれか。1つ選べ。

**選択肢**
1. 保管は、業務に従事する者が実地に盗難防止に必要な注意をしている場合以外は、鍵をかけた施設内で行わなければならない。
2. 廃棄は、焼却、酸・アルカリ等による分解、希釈、他の薬剤との混合等、回収が困難な方法で行わなければならない。
3. 卸売業者から譲り受けたときは、品名（販売名）・数量、譲り受けの年月日、譲り受けの相手方の営業所等の名称・住所を記録し、2年間保存しなければならない。
4. 一定数量以上の盗難、紛失等が生じたときは、速やかに都道府県知事に届け出なければならない。

**正解:** 3

**解説**
向精神薬は、第一種向精神薬、第二種向精神薬、第三種向精神薬のいずれかに分類される。メチルフェニデート塩酸塩錠は、第一種向精神薬であり、トリアゾラムは第三種向精神薬である。
1　メチルフェニデート塩酸塩及びトリアゾラムいずれにも当てはまる
向精神薬を薬局で保管する場合、業務に従事する者が実地に盗難防止に必要な注意をしている場合以外は、鍵をかけた施設内で行わなければならない。
2　メチルフェニデート塩酸塩及びトリアゾラムいずれにも当てはまる
向精神薬を廃棄する場合、焼却、酸・アルカリ等による分解、希釈、他の薬剤との混合等、回収が困難な方法で行わなければならない。
3　メチルフェニデート塩酸塩には当てはまるがトリアゾラムには当てはまらない
第一種向精神薬及び第二種向精神薬を卸売業者から譲り受けたときは、品名（販売名）・数量、譲り受けの年月日、 譲り受けの相手方の営業所等の名称・住所を記録し、2年間保存しなければならない。
4　メチルフェニデート塩酸塩及びトリアゾラムいずれにも当てはまる
