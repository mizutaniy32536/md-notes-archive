# 第97回 問23

**問題文**
ジエチル−p−フェニレンジアミン（DPD）法による水道水中の残留塩素の測定において、DPDと速やかに反応して赤色を呈するのはどれか。1つ選べ。

**選択肢**
1. HClO
2. NH2Cl
3. NHCl2
4. NCl3
5. CHCl3

**正解:** 1

**解説**
ジエチル−p−フェニレンジアミン（DPD）法は、DPDを試薬として用い、DPDと残留塩素が反応することによって生じる赤桃色を比色定量することによって水道水中の残留塩素を測定する方法である。残留塩素は遊離残留塩素（HClO、ClO－）と結合残留塩素（NH2Cl、NHCl2、NCl3）に大別され、DPDと速やかに反応するのは遊離残留塩素である。
