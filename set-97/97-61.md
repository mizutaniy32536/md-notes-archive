# 第97回 問61

**問題文**
関節リウマチの治療薬はどれか。1つ選べ。

**選択肢**
1. セツキシマブ
2. インフリキシマブ
3. トラスツズマブ
4. ベバシズマブ
5. ゲフィチニブ

**正解:** 2

**解説**
1　誤
セツキシマブは、上皮成長因子受容体（EGFR）に対するモノクローナル抗体で、EGFR陽性の治癒切除不能な進行・再発の結腸・直腸がんに用いられる。
2　正
インフリキシマブは、腫瘍壊死因子α（TNFα）に対するモノクローナル抗体で、関節リウマチ、クローン病、潰瘍性大腸炎などに用いられる。
3　誤
トラスツズマブは、ヒト上皮成長因子2型受容体（HER2）に対するモノクローナル抗体で、HER2過剰発現が確認された乳がんや転移性乳がん、治癒切除不能な進行・再発の胃がんなどに用いられる。
4　誤
ベバシズマブは、ヒト血管内皮増殖因子に対するモノクローナル抗体で、治癒切除不能な進行・再発の結腸・直腸がんや手術不能又は再発乳がんに用いられる。
5　誤
ゲフィチニブは、EGFRチロシンキナーゼ阻害薬で、非小細胞肺がんに用いられる。
