# 第97回 問182

**問題文**
ウイルス性肝炎（A型、B型、C型）に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**
1. C型肝炎の感染経路は、非経ロ感染である。
2. B型肝炎ウイルスは、RNAウイルスである。
3. A型、B型、C型のいずれも、ウイルスが肝細胞を直接破壊して発症する。
4. B型肝炎ウイルスの抗体は、HBc→HBe→HBsの順で陽性化する。
5. C型急性肝炎の慢性化率は他の肝炎に比べて低い。

**正解:** 1,4

**解説**
1　正
C型肝炎の感染経路は、非経口感染（主に血液感染）である。なお、感染経路が経口感染なのは、A型、E型肝炎である。
2　誤
B型肝炎ウイルスは、DNAウイルスである。なお、A型、C型肝炎ウイルスは、RNAウイルスである。
3　誤
A型、B型、C型肝炎では、肝臓に存在する肝炎ウイルスが細胞性免疫により認識され、免疫反応によりウイルスを排除する際、肝細胞が破壊され炎症を起こす。
4　正
5　誤
C型急性肝炎の患者のほとんどが慢性肝炎へと移行し、慢性化率は他の肝炎に比べて最も高い。
