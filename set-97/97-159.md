# 97-159

**問題文**
消化管に作用する薬物に関する記述のうち、誤っているのはどれか。1つ選べ。

**選択肢**
1. カルメロース（カルボキシメチルセルロース）は、腸管内で水分を吸収して膨張し、腸管運動を促進する。
2. ロペラミドは、腸管のオピオイドμ受容体を刺激し、腸管運動を抑制する。
3. メペンゾラートは、アセチルコリンM3受容体を遮断し、腸管運動を抑制する。
4. 次硝酸ビスマスは、腸粘膜表面のタンパク質に結合することで被膜を形成し、腸粘膜を保護する。
5. トリメブチンは、腸管のドパミンD2受容体を遮断し、低下した腸管運動を促進する。

**正解:** 5

**解説**
解答　５

1　正
カルメロースは膨張性下剤であり、腸管内で水分を吸収して膨張し、腸内容物の容積を増大して腸管壁を刺激することで、腸管運動を促進する。

2　正
ロペラミドはコリン作動性神経のオピオイドμ受容体を刺激することで、アセチルコリン遊離を抑制し、腸管運動を抑制する止瀉薬である。

3　正
メペンゾラートは抗コリン薬であり、アセチルコリンM3受容体を遮断することで、腸管運動を抑制する。

4　正
次硝酸ビスマスは、腸粘膜のタンパク質と結合して被膜を形成し、腸粘膜を保護する。また、腸内異常発酵により生じる硫化水素と結合し、止瀉作用を示す。

5　誤
トリメブチンは、オピオイドμ受容体刺激作用を介し、亢進した腸管運動を抑制し、低下した腸管運動を促進する。ドパミンD2受容体遮断ではない。
