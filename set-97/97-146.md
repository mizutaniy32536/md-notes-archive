# 第97回 問146

**問題文**
我が国の社会保障制度に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**
1. 社会保障は、憲法に根拠を置いている。
2. 年金制度は、社会保険方式の社会保障である。
3. 医療保険制度は、社会扶助方式の社会保障である。
4. 生活保護制度は、社会保険方式の社会保障である。

**正解:** 1,2

**解説**
1　正
社会保障は、日本国憲法第25条に根拠を置いている。
＜日本国憲法第25条＞
すべて国民は、健康で文化的な最低限度の生活を営む権利を有する。
国は、すべての生活部面について、社会福祉、社会保障及び公衆衛生の向上及び増進に努めなければならない。
2　正
年金制度、医療保険制度は、社会保険方式＊1の社会保障である。
3　誤
解説参照
4　誤
生活保護制度は、社会扶助方式＊2の社会保障である。
社会保険方式＊1：保険料を財源として、保険技術を用いて給付を行う方式
社会扶助方式＊2：租税を財源として、保険技術を用いず給付を行う方式
