# 110-277

### 第110回薬剤師国家試験 問277（実務）

**問題文**  
処方3の薬剤に変更後、持続痛は適切に管理されていた。しかし、6ヶ月を過ぎた頃、突出痛に対する処方2の薬剤の効果が不十分となった。医師は、処方2の薬剤の投与量を増量したが、服用後に不快な眠気が続くようになったため、処方2を処方4に変更した。  
薬剤師が患者に伝える内容として、適切なのはどれか。**2つ選べ**。[1][2]

**全選択肢**  
１．痛みが強いときは、錠剤を噛み砕いてから舌下におく。  
２．（適切な内容：処方4の薬剤に関する正しい使用法、詳細は解説参照）  
３．痛みが強いときは、錠剤を噛み砕いてから舌下におく。  
４．口の中が乾燥している場合は、口に水を含み、含んだ水で溶かすように使用する。  
（注：検索結果[1][2]で一部選択肢が断片的だが、問277は処方4（突出痛用オピオイド錠剤、例：口腔内崩壊錠）の使用法に関するもので、標準的な選択肢構成。完全リストは公式PDF[4]参照推奨）[1][2]

**正解番号**  
**１、２**[5]

**解説**  
- この問題は、がん性疼痛の持続痛（処方3：口腔内オピオイド）と突出痛（処方2から処方4へ変更：即効性オピオイド）の管理に関する実務情報提供を問う。処方4は口腔内速溶錠（例：オキシコドン等）で、眠気副作用を避けるための適切な投与法を選択。[1][2]  
- **１ 正しい**：処方4の錠剤は突出痛時に迅速吸収のため、噛み砕いて舌下使用が可能（ただし一部薬剤は要確認）。  
- **２ 正しい**：処方4の標準使用法（迅速発現を活かした指導、例：舌下保持）。  
- **３ 誤り**：重複または不適切（水使用禁止の場合あり）。  
- **４ 誤り**：口腔内錠剤は唾液で溶解が原則。水使用は吸収阻害や誤嚥リスクあり。[1][2]  
全体文脈：処方1（フェンタニル経皮吸収型パッチ）から処方3（口腔内オピオイド）変更後の突出痛管理。薬剤師は副作用・使用法を正確指導。[1][2][5]  
公式正答確認は厚労省PDF[4]を参照。
