# 110-8

以下の化合物のうち、最も酸性度が高いのはどれか。１つ選べ。

**正解:** ５

**解説**
選択肢１のアルコール（中性）と選択肢２〜５のフェノール及びフェノール誘導体（弱酸性）を比較した場合、選択肢２〜５の酸性度が高い。
フェノール及びフェノール誘導体の酸性度の大きさは置換基の性質によって影響を受ける。
−Cl、−NO2などの電子求引基は酸性度を高める。電子求引基の強さを比較すると−Cl＜−NO2であり、強い電子求引基がついているほど酸性度は高まるため、選択肢５の酸性度が最も高い。
なお、−NH2などの電子供与基は酸性度を低下させるため、酸性度は選択肢１＜３＜２＜４＜５の順に高くなる。
