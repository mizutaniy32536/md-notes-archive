# 110-147

**問題文**
我が国の医療保険制度に関する記述として、正しいのはどれか。2つ選べ。なお、公費負担医療を受けている場合を除く。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-147 を参照

**選択肢**
1. 国民は公的保険に加入しなければならない。
2. 保険者が支払う療養の給付に関する費用は、被保険者が支払う一部負担金を除いた残額である。
3. 75歳以上の被保険者が支払う一部負担金の割合は、所得に関わらず同じである。
4. 健康保険の被保険者が扶養する配偶者や子供は、国民健康保険に加入しなければならない。
5. 健康保険では、勤務中や業務により発生した病気や怪我等を対象に療養の給付を行う。

**正解:** 1,2

**解説**
解答　１、２

１　正
我が国の医療保険制度においては、基本的にすべての国民が被用者保険や国民健康保険等の公的医療保険に加入する国民皆保険制度が採用されている。

２　正
保険者が支払う療養の給付に関する費用は、保険医療機関又は保険薬局における療養の給付に要する費用の額から、被保険者が支払う一部負担金に相当する額を控除した額（被保険者が支払う一部負担金を除いた残額）である。

３　誤
75歳以上の被保険者が支払う一部負担金の割合は原則１割であるが、一定以上の所得者は2割、現役並所得者は3割であり、所得により異なる。

４　誤
健康保険の被保険者が扶養する配偶者や子供は、被扶養者として健康保険の給付の対象となるため、国民健康保険に加入する必要はない。

５　誤
健康保険など被用者保険では、業務災害以外（業務外の事由）による傷病等を保険給付の対象とする。なお、勤務中や業務により発生した病気や怪我等は業務災害（業務上の事由）に該当し、労働者災害補償（労災）保険の対象としている。
