# 110-243

### 第110回薬剤師国家試験 問243

**問題文**  
野球大会当日、複数の生徒が体調不良を訴え、保健室で手当てを受けた。この高校では、教育委員会が作成した「学校における熱中症対策ガイドライン」をもとに今後の対策を話し合うこととなり、**熱中症を防止するために必要な注意事項について学校薬剤師に助言を求めた。学校薬剤師が学校に対して行う助言の内容として、正しいのはどれか。2つ選べ。**[1][2][4][6]

**全選択肢（番号付き）**  
1. 乾球温度と気温が同じ場合には、湿球温度が高いほど熱中症のリスクは減少する。[1]  
2. WBGTの算出に用いる黒球温度は、紫外線の影響を最も強く受ける。[1]  
3. 風速が1m/s以上の場合、風速が大きいほど暑さ指数は高くなる。[1]  
4. 室内では熱中症のリスクがないため、WBGTの測定は不要である。[1]  
5. 熱中症の疑いがある生徒に対しては、まず冷たい飲料水を飲用させる。[1]

**正解番号**  
**3、5**[1][2][8]

**解説**  
- **選択肢1（誤り）**: 乾球温度と気温が同じ場合、湿球温度が高いほど汗の蒸発が妨げられ、熱中症リスクは**増加**する。「減少」ではない。[1]  
- **選択肢2（誤り）**: 黒球温度は**輻射熱**（赤外線）の影響を主に受け、紫外線の影響ではない。[1]  
- **選択肢3（正しい）**: 風速が1m/s以上の場合、風速が大きいほど体感温度が上がり、暑さ指数（WBGTなど）が**高くなる**（風が熱を運ぶため）。[1]  
- **選択肢4（誤り）**: 室内でも熱中症リスクがあり、WBGT測定や注意が必要。[1]  
- **選択肢5（正しい）**: 熱中症疑いの場合、まず**体を冷やし**、水分・電解質補給を行うが、ガイドラインでは飲料水の適切な補給を助言（ただし意識がない場合は飲用せず）。[1]  

問242との関連で、WBGT計算式は**WBGT = 0.7×湿球温度 + 0.2×黒球温度 + 0.1×乾球温度**（屋外用）。[4][6]
