# 110-157

### 第110回薬剤師国家試験 問157

**問題文（全文）**  
63歳男性。慢性心不全と診断され、治療中である。その他の既往歴及び常用薬はない。  
服薬は正しく継続され、副作用もみられなかったが、1ヶ月前から労作時の息切れが徐々に増悪するようになった。2～3日前からは安静時にも息苦しさを自覚するようになり、昨夜突然、強い咳を伴った呼吸困難が出現したため救急搬送された。下肢に浮腫が認められ、血圧は88/60 mmHgであった。心臓超音波検査を行ったところ、左室駆出率（LVEF）は30%に低下していた。さらに、胸部X線検査により肺うっ血と軽度な心拡大の所見が認められ、慢性心不全の急性増悪と診断され入院となった。[3][4]  
この患者の来院時の病態及び症状に関する記述として、正しいのはどれか。**2つ選べ。**[3]

**全選択肢（番号付き）**  
1. 右心機能は正常である。  
2. 心臓からのナトリウム利尿ペプチドの分泌が亢進している。  
3. 心電図でST上昇が認められる。  
4. 呼吸症状は、起坐位よりも仰臥位で増悪する。  
5. 尿量は増加している可能性が高い。[1][3][4]

**正解番号**  
2, 4[1][9]

**解説**  
- **選択肢1**: 誤り。下肢の浮腫が静脈血の循環不全を示唆しており、右心機能が正常とは判断できない。[1]  
- **選択肢2**: 正しい。心不全の急性増悪により、心臓（特に心室伸展）からのナトリウム利尿ペプチド（ANP/BNP）の分泌が亢進する。[1][3]  
- **選択肢3**: 誤り（記載なし）。急性心筋梗塞のようなST上昇は本症例の記述に合致せず、心電図所見は明記されていない。[3]  
- **選択肢4**: 正しい。左心不全による肺うっ血で起坐呼吸（きざこきゅう：仰臥位で呼吸困難が増悪し、起坐位で軽減）がみられる。[1]  
- **選択肢5**: 誤り。心不全による血流減少で尿量は減少する。[1]  
以上より、正解は**2, 4**。[1][3][4][9]
