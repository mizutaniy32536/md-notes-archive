# 110-60

重症筋無力症で認められる症状はどれか。１つ選べ。

１ 突発性発熱
２ 前向性健忘
３ 聴覚障害
４ 振戦
５ 構音障害

**正解:** ５

**解説**
重症筋無力症は、運動神経筋接合部のシナプス後膜に存在するアセチルコリンNM受容体に対する自己抗体により、骨格筋の刺激伝導が障害される自己免疫性疾患である。主な症状としては、眼瞼下垂、筋易疲労感、構音障害などがある。
