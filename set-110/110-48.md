# 110-48

**問題文**
ある薬物の肝固有クリアランス（CLint）が肝血流量（Qh）と比較して十分に小さい場合、その薬物の肝クリアランスはどのように近似できるか。1つ選べ。ただし、fuは血漿タンパク非結合形分率とする。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-48 を参照

**選択肢**
1. Qh
2. CLint
3. fu・Qh
4. fu・CLint
5. CLint / fu

**正解:** 4

**解説**
解答　４

well−stirredモデルにおいて、薬物の肝クリアランスCLhは、肝固有クリアランスCLintを用いて以下の式で表される。

問題文より、「薬物の肝固有クリアランス（CLint）が肝血流量（Qh）と比較して十分に小さい」とあることから、CLint ＜＜ Qhであることが分かる。よって、Qh＋fu・CLint≒Qhと近似できる。
このことから肝クリアランスCLhは以下のように近似することができる。
