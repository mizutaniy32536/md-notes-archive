# 110-275

### 第110回薬剤師国家試験 問275

#### 問題文
65歳男性。高血圧症及び**気管支ぜん息**に対して処方1及び処方2の薬剤を使用していた。  
血圧が160/95 mmHg前後で推移していたため、**テルミサルタン錠**を80 mgに増量した。数日後、患者はめまいや頭痛を自覚したため、再来院した。血圧を測定したところ92/66 mmHgと低値を示した。  
テルミサルタン錠は40 mgに減量され自覚症状は消失したが、血圧が160/95 mmHgまで上昇した。そのため、他の降圧薬を追加することになった。**追加する薬物として適切なのはどれか。2つ選べ。**[1][2][6]

#### 正解番号
**2、5**[2][8][9]

#### 解説
本症例の患者は**高血圧症**と**気管支ぜん息**を有し、**テルミサルタン（ARB）**80 mg増量で低血圧（92/66 mmHg、めまい・頭痛）を起こしたため40 mgに減量。自覚症状は消失したが血圧が再上昇（160/95 mmHg）したため、追加降圧薬を選択。[2]  

- **選択肢1（プロプラノロール）**: **β遮断薬**のため、**喘息患者に禁忌**。誤り。[1]  
- **選択肢2（トリクロルメチアジド）**: **チアジド系利尿薬**。喘息に影響なく追加可能で妥当。[1]  
- **選択肢5**: 正解に含まれる（詳細未明記だが、患者背景に適合する降圧薬、例: カルシウム拮抗薬など喘息非禁忌のもの）。[2][8][9]  

**テルミサルタン**の増量で過度な血圧降下が生じたため、相加・相乗効果を考慮し、喘息禁忌薬（β遮断薬）を避け、**チアジド系**など安全なものを追加。[1][2]
