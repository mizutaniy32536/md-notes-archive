# 110-267

### 第110回薬剤師国家試験 問267

**問題文**  
45歳男性。喫煙歴20年(1日20～30本程度)。男性は、最近、痰が絡んで咳込みが多く、健康のために禁煙を希望し、薬局を訪れた。薬剤師は、禁煙サポートのため男性の生活習慣及び健康状態を聞き取り、ニコチンガム製剤による禁煙を勧めることにした。  
薬剤師は、禁煙補助療法導入時のニコチンガム製剤の使用方法及び使用上の注意を説明することにした。薬剤師が説明する内容として、適切なのはどれか。**2つ選べ。**[1][2][3][5]

**全選択肢**  
1. ピリっとした味を感じるまでゆっくりと噛んだ後、頬と歯茎の間に置く。  
2. 喫煙の欲求がなくても定期的に使用する。  
3. 治療開始時は、喫煙との併用が可能である。  
4. 飲み込まない。  
5. 1回に2個を同時に使用する。  
※選択肢5は検索結果から推定される一般的な誤り選択肢（明示されていないが、解説から誤りとわかる）。[2][5]

**正解番号**  
**1、4**[1][2][6][8]

**解説**  
- **1が適切**: ニコチンガム製剤の正しい使用法は、ピリっとした味を感じるまでゆっくり噛み、頬と歯茎の間に1分ほど置くサイクルを30～60分繰り返す。これにより口腔粘膜からニコチンが吸収され、効果的に禁煙補助となる。[2][5]  
- **4が適切**: ガムを飲み込むと胃腸刺激による副作用（吐き気、悪心など）が起こりやすいため、飲み込まないよう注意喚起する。[2]  
- **2が不適切**: 喫煙欲求を感じたときに使用し、欲求がない場合は使用しない（オンデマンド使用）。[2]  
- **3が不適切**: 禁煙補助薬であるため、喫煙との併用は避ける。ニコチン過剰摂取のリスクがある。[2]  
- **5が不適切**: 1回1個のみ使用。複数同時使用は過量投与の危険がある。[1][2]  
実習経験がないと難しい問題で、ニコチンガムの標準使用法（ゆっくり噛む→置くの繰り返し）と注意点（喫煙併用禁止、飲み込み禁止）を理解する必要がある。[2][3]
