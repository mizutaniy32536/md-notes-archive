# 110-219

### 第110回薬剤師国家試験 問219

**問題文**  
36歳男性。一昨日食事会があり中華料理とアルコールを摂取した。昨日から胸やけと胃の痛みがあり、まだ治まらなかったので、一般用医薬品の購入を希望し来局した。薬剤師は以下の成分を含む医薬品を提案した。男性は、この薬の成分、副作用や飲み方などについての詳しい説明を求めた。  
＜提案した一般用医薬品に含まれる成分＞  
ピレンゼピン塩酸塩水和物  
メタケイ酸アルミン酸マグネシウム  
炭酸水素ナトリウム  
ビオヂアスターゼ2000  

この患者の胸やけや胃の痛みにつながった可能性が高いのはどれか。**2つ選べ。**[1][2][3][8]

**全選択肢**  
1. 胃酸が下部食道壁を刺激している。  
2. 胃腺の壁細胞からの胃酸分泌が減少している。  
3. 胃粘膜表面を覆う粘液の分泌が増加している。  
4. 胃内でアセチルコリンやヒスタミンの分泌量が増加している。  
5. 幽門括約筋が拡張し、胃内容物の排出が促進されている。[1][2][3][8]

**正解番号**  
**1, 4**[1][2][3]

**解説**  
- **選択肢1**: 正しい。胃酸が下部食道壁を刺激することで胸やけや胃の痛みが生じる可能性が高い（胃食道逆流の機序）。[1][2]  
- **選択肢2**: 誤り。胸やけや胃の痛みは胃酸分泌の**増加**が原因で、分泌減少は胃もたれや消化不良を引き起こす。[1][3]  
- **選択肢3**: 誤り。胃粘膜保護のための粘液分泌が**増加**すると痛みは軽減され、減少時が痛みの原因。[1]  
- **選択肢4**: 正しい。胃内でアセチルコリンやヒスタミンの分泌量が増加すると壁細胞からの胃酸分泌が促進され、症状につながる（抗コリン薬やH2ブロッカーがこれを抑制）。[1][3]  
- **選択肢5**: 誤り。幽門括約筋拡張による排出促進は逆流を防ぐが、実際は排出抑制が逆流を助長する。[1]  

以上より、正解は**1, 4**。[1][2][3]
