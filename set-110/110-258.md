# 110-258

**問題文**
65歳男性。最近、尿意を我慢できなくなってきたため、近くの泌尿器科を受診したところ、過活動膀胱と診断され、行動療法とともに以下の処方で治療を開始することとなった。当薬局への来局は今回が初めてであったことから、お薬手帳の確認と聞き取りを行った。問258（実務）眼科において治療中と考えられる疾患として、最も可能性が高いのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-258 を参照

**選択肢**
1. 加齢黄斑変性
2. 白内障
3. 緑内障
4. ぶどう膜炎
5. 流行性角結膜炎問259（薬理）前問で最も可能性が高い疾患を患者が治療中であることを眼科医に確認できたので、泌尿器科の医師に処方変更を提案することにした。提案する薬物の作用機序として適切なのはどれか。1つ選べ。
6. アドレナリンα1受容体刺激
7. アドレナリンβ3受容体刺激
8. アセチルコリンM3受容体遮断
9. アンドロゲン受容体遮断
10. ミネラルコルチコイド受容体（アルドステロン受容体）遮断

**正解:** 3

**解説**
問258　解答　３

お薬手帳の内容から緑内障治療薬であるラタノプロスト点眼液、ドルゾラミド塩酸塩点眼液を使用しているため、眼科において治療中と考えられる疾患として、最も可能性が高いのは緑内障である。ラタノプロスト点眼液は、プロスタノイドFP受容体を刺激してぶどう膜強膜からの眼房水流出を促進し、ドルゾラミド塩酸塩点眼液は、炭酸脱水酵素を阻害して眼房水の産生を抑制する。

問259　解答　２

本患者が緑内障の治療中であると眼科医に確認できたことから、今回処方されたアセチルコリンM3受容体遮断薬であるソリフェナシンコハク酸塩は緑内障を悪化させる恐れがあるため、処方変更が必要である。選択肢のうち、緑内障患者に使用できる過活動膀胱治療薬は、アドレナリンβ3受容体刺激作用を示すミラベグロンなどが該当する。
