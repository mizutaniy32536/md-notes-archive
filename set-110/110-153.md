# 110-153

**問題文**
外科的手術時に用いられる薬物に関する記述として、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-153 を参照

**選択肢**
1. 血液／ガス分配係数の大きい吸入麻酔薬ほど、麻酔の導入は速い。
2. 最小肺胞濃度（MAC）の大きい吸入麻酔薬ほど、麻酔作用は強い。
3. デクスメデトミジンは、アドレナリンα2受容体を刺激することで、鎮痛及び鎮静作用を生じる。
4. チアミラールは、γ−アミノ酪酸GABAA受容体のバルビツール酸結合部位に結合することで、意識消失を生じる。
5. ケタミンは、ヒスタミンH1受容体を遮断することで、不動化（筋弛緩）を起こす。

**正解:** 3,4

**解説**
解答　３、４

１　誤
血液／ガス分配係数は吸入麻酔薬の導入と覚醒の速さの指標であり、値が小さいほど麻酔の導入・覚醒は速い。

２　誤
最小肺胞濃度（MAC）は吸入麻酔薬の麻酔作用の強さの指標であり、値が小さいほど麻酔作用は強い。

３　正
デクスメデトミジンは、アドレナリンα2受容体を刺激することで、中枢神経からノルアドレナリンの遊離を抑制するため、鎮痛及び鎮静作用を生じる。

４　正
チアミラールは、γ−アミノ酪酸GABAA受容体のバルビツール酸結合部位に結合することで、GABAの作用を増強し、意識消失を生じる。

５　誤
ケタミンは、NMDA型グルタミン酸受容体を遮断することで、麻酔・鎮痛作用を生じる。
