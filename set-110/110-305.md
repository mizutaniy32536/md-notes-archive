# 110-305

### 第110回薬剤師国家試験 問305

**問題文（共通ステム）**  
58歳女性。乳がんによる骨転移性疼痛に対して、デノスマブ皮下注とモルヒネ硫酸塩水和物徐放錠にて治療中であった。腰痛の悪化により、モルヒネ硫酸塩水和物徐放錠が増量された。FDG-PETで腰椎に集積像が確認された。[1][2][3]

**問305（実務）**  
緩和ケアチームの薬剤師の対応として、適切なのはどれか。**2つ**選べ。[2][3]

**全選択肢（番号付き）**  
1. デノスマブ皮下注の投与中は、低カルシウム血症の発現に注意するよう、情報共有する。[2][3]  
2. デノスマブ皮下注からゾレドロン酸水和物注射液への変更を検討する。[2][3]  
3. モルヒネ硫酸塩水和物徐放錠の投与中は、QT延長の副作用の発現に注意するよう、情報共有する。[1][2]  
4. モルヒネ硫酸塩水和物徐放錠からフェンタニルクエン酸塩テープへの変更を検討する。[1][2]  
5. モルヒネ硫酸塩水和物徐放錠からロキソプロフェンナトリウム錠への変更を検討する。[2]

**正解番号**  
**1、4**[2][9]

**解説**  
- **1が適切**: デノスマブ（ランマーク）はRANKL阻害薬で、骨吸収を抑制するが、低カルシウム血症の副作用が知られているため、投与中は低カルシウム血症に注意し、情報共有するのが適切。[1][3]  
- **4が適切**: モルヒネ徐放錠増量後も腰痛が悪化（忍容不良の可能性）しているため、オピオイドの種別変更として経皮吸収型鎮痛剤であるフェンタニルテープへの切り替えを検討するのが緩和ケアの標準対応。[1][2]  
- **2が不適切**: デノスマブとゾレドロン酸（ビスフォスフォネート）は骨転移治療薬として併用可能で、乳がん骨転移ではデノスマブが推奨され、変更の必要はない。[1]  
- **3が不適切**: モルヒネの主な副作用は便秘・悪心・眠気などで、QT延長はメサドンなどの一部オピオイドにみられるが、モルヒネでは主な懸念ではない。[1][2]  
- **5が不適切**: ロキソプロフェンはNSAIDsで、オピオイド抵抗性疼痛や骨転移性疼痛の第1選択ではなく、増量後のオピオイドからの変更先として不適切。[2]  
複数の解説サイトで正解が**1、4**と一致。[2][9]
