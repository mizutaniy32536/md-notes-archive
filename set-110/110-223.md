# 110-223

### 第110回薬剤師国家試験 問223

**問題文**  
68歳男性。パーキンソン病及びうつ病の治療のため継続して薬剤を服用し、パーキンソン病の症状は軽快していたが、1ヶ月前より時間帯によって歩くことができたりできなかったりする症状が認められ、生活に支障をきたすようになった。薬の調節とリハビリテーションを行う目的で4週間の入院となった。  

（入院時持参薬）  
- レボドパ100mg・カルビドパ配合錠  
- ペルゴリドメシル酸塩錠 250μg  
- パロキセチン錠20mg  

入院後、レボドパ・カルビドパ配合錠を1回1錠、1日3回から1回1錠、1日5回に増量したが、症状が改善しなかったため、さらに薬剤を追加することとなった。この患者に追加する薬剤の候補として適切なのはどれか。**２つ**選べ。[1][3][4]

**全選択肢（番号付き）**  
1. イストラデフィリン錠  
2. エンタカポン錠  
3. セレギリン塩酸塩錠  
4. サフィナミドメシル酸塩錠  
5. ラサギリンメシル酸塩錠[1][4]

**正解番号**  
**1、2**[3][6][9]

**解説**  
この患者はパーキンソン病に対しレボドパ・カルビドパ配合剤とペルゴリド（ドパミンアゴニスト）を投与中であり、時間帯による症状変動（おそらく**on-off現象**）が問題。レボドパ増量後も改善せず追加薬が必要。[1][3]  

- **1. イストラデフィリン錠**：アデノシンA2A受容体拮抗薬。ドパミン系を間接的に活性化し、on-off現象の改善に有効。[3]  
- **2. エンタカポン錠**：COMT阻害薬。レボドパの末梢代謝を抑制し、脳内ドパミン濃度を延長・安定化。レボドパ頻回投与時のon-off改善に適する。[3]  

他の選択肢の不適切さ：  
- **3. セレギリン塩酸塩錠**、**4. サフィナミドメシル酸塩錠**、**5. ラサギリンメシル酸塩錠**：いずれもMAO-B阻害薬だが、患者は**パロキセチン（SSRI）**併用中。SSRIとの併用でセロトニン症候群リスクが高いため不適切。[3]  
ペルゴリド継続中でも、イストラデフィリンとエンタカポンは併用可能で、症状改善に寄与。[1][3][4]
