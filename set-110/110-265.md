# 110-265

# 第110回薬剤師国家試験 問265

## 問題文

処方1及び処方2のいずれかの薬物の結核菌に対する作用として、正しいのはどれか。**2つ選べ。**[1][3]

## 選択肢

1. DNA依存性RNAポリメラーゼを阻害して、RNA合成を抑制する。[1][3]

2. ミコール酸の合成を阻害して、細胞壁合成を抑制する。[1][3]

3. トポイソメラーゼⅣとDNAジャイレースを阻害して、核酸合成を抑制する。[3]

4. ATP合成酵素を阻害して、結核菌細胞内のATP量を低下させる。[3]

5. リボソーム30Sサブユニットに作用して、タンパク質合成を阻害する。[2]

## 正解番号

**1、2**[1][2][7]

## 解説

**選択肢1 ◉正解**

**リファンピシン**はDNA依存性RNAポリメラーゼを阻害し、RNA合成を抑制することが作用機序である。[1][2]

**選択肢2 ◉正解**

**イソニアジド**は結核菌細胞壁の構成成分であるミコール酸の生合成を阻害する。結核菌などの抗酸菌の細胞壁は、分子量の大きな脂肪酸であるミコール酸を含むことが特徴である。[1][2]

**選択肢3 ✗誤り**

トポイソメラーゼⅣとDNAジャイレース阻害は、キノロン系抗菌薬の作用機序であり、処方1、2の薬物の結核菌に対する作用ではない。

**選択肢4 ✗誤り**

ATP合成酵素阻害は、処方1、2の薬物の作用機序ではない。

**選択肢5 ✗誤り**

リボソーム30Sサブユニットに作用するのは**アミノグリコシド系**の作用機序である。処方1、2の薬物の結核菌に対する作用ではない。[1]
