# 110-98

### 第110回薬剤師国家試験 問98

**問題文（全文）**  
次の記述は、X線造影剤として用いられる日本薬局方アミドトリゾ酸（C₁₁H₉I₃N₂O₄：613.91）の純度試験と定量法に関するものである。  

（純度試験の記述は省略。定量法の記述は以下の通り：アルカリ性条件下でアミドトリゾ酸を還元しヨウ化物イオンを遊離させた後、0.1 mol/L硝酸銀液（下線部ア）で滴定。指示薬はイテトラブロモフェノールフタレインエチルエステル試液（下線部イ）1 mLを使用し、終点は沈殿の黄色が**ウ**に変わるときとする。0.1 mol/L硝酸銀液1 mL＝**エ**mg C₁₁H₉I₃N₂O₄）[1][2][3][4][5]  

**問98**  
定量法に関する記述として正しいのはどれか。**2つ**選べ。[1][2][3][5]  

1. 本定量法の原理は、Volhard法に基づいている。  
2. 下線部アの標準液は、0.1 mol/Lチオシアン酸アンモニウム液の標定に用いられる。  
3. 下線部イの指示薬の代わりにフルオレセインナトリウムを用いることができる。  
4. **ウ**に入る色は、橙色である。  
5. **エ**に入る数字は、20.46である。[1][2][3][4][5]  

**正解番号**  
**2、5**[1][2][3][5]  

**解説**  
- **1誤**：本定量法は吸着指示薬（イテトラブロモフェノールフタレインエチルエステル）を使用した沈殿滴定法で、**Fajans法（ファヤンス法）**に基づく。Volhard法は残渣滴定法であり異なる。[2][3][4]  
- **2正**：下線部ア（0.1 mol/L硝酸銀液）の標定には、0.1 mol/Lチオシアン酸アンモニウム液を使用する。[1][2][3]  
- **3誤**：指示薬の代替としてフルオレセインナトリウムは使用できない。Fajans法では特定の吸着指示薬が必要。[2][3]  
- **4誤**：**ウ**（終点の色）は緑色。橙色ではない。[2][3]  
- **5正**：アミドトリゾ酸1分子あたりヨウ素3原子（I: 126.90×3=380.70）を含むため、換算係数は(613.91×0.1)/0.38070=**20.46**（mg）。[2][3][5]
