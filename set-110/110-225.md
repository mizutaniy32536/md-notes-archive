# 110-225

### 第110回薬剤師国家試験 問225

**問題文（物理・化学・生物）**  
前問で参照された検査項目で確かめられたのはどれか。1つ選べ。[1][2][3][5]

1. 特定のがん遺伝子が存在すること。  
2. 特定のがん抑制遺伝子に変異が存在すること。  
3. 特定のがん原遺伝子（注）に変異が存在しないこと。  
4. 特定の薬物代謝酵素遺伝子の一塩基多型のタイプが存在すること。  
5. 転座した異常染色体が存在すること。  

（注）がん原遺伝子：原がん遺伝子ともいう[1][2][3][5]

**正解番号：3** [1][2][3][5][8][9]

**解説**  
問224では、コンパニオン診断として**RAS遺伝子**（がん原遺伝子）が参照され、mFOLFOX6（レボホリナート、フルオロウラシル、オキサリプラチン）に**セツキシマブ**を組合せた治療を選択。[2][3][5]  
RAS遺伝子に変異がある場合、セツキシマブの有効性が低下するため、**RAS遺伝子に変異が存在しないこと**を確認した。[2][3]  
- 1はALK融合遺伝子陽性（アレクチニブ使用時）[1]  
- 2はがん抑制遺伝子変異（例: TP53）[1]  
- 4はUGT1A1多型（イリノテカン使用時）[1][3]  
- 5はフィラデルフィア染色体など[1]  
よって、3が正しい。[1][2][3]
