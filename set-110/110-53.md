# 110-53

下の模式図で示される経口徐放カプセル剤の型はどれか。１つ選べ。

１ スパンスル型
２ ロンタブ型
３ グラデュメット型
４ スパスタブ型
５ レジネート型

**正解:** １

**解説**
１　正
スパンスル型は、薬物を含有する顆粒を高分子被膜でコーティングし、これをカプセルに充填した製剤である。コーティングの厚さが異なる顆粒を数種類充填することで、薬物の放出速度を最適化している。
２　誤
ロンタブ型は、徐放性マトリックスに薬物を分散した核錠を、速放性の薬物層で被包した錠剤である。
３　誤
グラデュメット型は、水に不溶性の多孔質プラスチック格子に薬物を分散させた錠剤である。
４　誤
スパスタブ型は、速放性顆粒と徐放性皮膜でコーティングされた数種の徐放性顆粒を打錠した錠剤である。
５　誤
レジネート型は、イオン交換樹脂に薬物を吸着させて、消化管内のイオンとの交換反応により薬物を放出することで徐放化した製剤である。
