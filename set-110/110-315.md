# 110-315

### 第110回薬剤師国家試験 問315

**問題文（背景を含む）**  
ICUに常駐する薬剤師が、この病棟で使用している輸血用血液製剤のうちの血液成分の製剤に関する病棟スタッフ向けの講義を実施し、血液製剤に関する内容やその留意点について質問を受けた。この病院で採用されている血液成分の製剤は、赤血球液-LR「日赤」、照射赤血球液-LR「日赤」、濃厚血小板-LR「日赤」、照射濃厚血小板-LR「日赤」、新鮮凍結血漿-LR「日赤」である。[3][6]

**問315（法規・制度・倫理）**  
薬剤師が講義時に受けた質問に対する回答として、**誤っている**のはどれか。1つ選べ。[2][3][6]

**全選択肢**  
1. 赤血球製剤の原料となる血液の確保は献血により行われています。  
2. 輸血用血液製剤の国内自給率は100%です。  
3. 日本赤十字社が輸血用血液製剤の原料となる献血の採血業務を一元的に担っています。  
4. 照射されている血液製剤とは、輸血による移植片対宿主病（GVHD）予防目的で、放射線が照射された製品です。  
5. 血液成分の製剤の使用に関する記録は、医療機関において使用日から**30年**の保存義務があります。[3][6]

**正解番号**  
**5**[3][7]

**解説**  
この問題は、薬剤師の回答として**誤っている**ものを選ぶ。  
- **1 正しい**：赤血球製剤の原料となる血液は、献血によって確保されている。[3]  
- **2 正しい**：日本において、輸血用血液製剤の原料となる献血の採血業務は、原則として日本赤十字社（日赤）が一元的に担っており、国内自給率は100%である。[1][3]  
- **3 正しい**：血液製剤の原料とする目的で事業として採血するためには厚生労働大臣の許可が必要で、日赤が一元的に担う。[3]  
- **4 正しい**：照射赤血球液や照射濃厚血小板は、GVHD予防のため放射線照射された製品。[6]  
- **5 誤り（正解）**：血液成分製剤の使用記録の保存義務は、使用日から**5年**（安全な血液製剤の提供等に関する省令に基づく）。30年は誤り。[3][6]  

※問314は複合問題の前半（実務）で、血液成分製剤の内容として適切なものを2つ選ぶが、本問は独立して誤りを問う。[3][5]
