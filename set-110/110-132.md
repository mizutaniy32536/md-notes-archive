# 110-132

**問題文**
活性酸素による傷害を防ぐための生体防御因子に関する記述として、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-132 を参照

**選択肢**
1. カタラーゼは、過酸化水素を水と酸素に分解する。
2. スーパーオキシドジスムターゼは、活性中心にヘム鉄をもつ。
3. グルタチオンペルオキシダーゼは、活性中心にマンガンをもつ。
4. β-カロテンは、体内で代謝されてビタミンAとなる。
5. ビタミンEは、主に細胞質で抗酸化作用を示す。

**正解:** 1,4

**解説**
解答　１、４

１　正
カタラーゼは、過酸化水素を水と水素に分解する酵素であり、活性中心にヘム鉄をもつ。

２　誤
スーパーオキシドジスムターゼ（SOD）は、スーパーオキシドアニオンを過酸化水素と酸素に変換する酵素であり、活性中心に銅および亜鉛をもつSOD1、鉄あるいはマンガンをもつSOD2、ニッケルをもつSOD3に分類される。

３　誤
グルタチオンペルオキシダーゼは、グルタチオン存在下で過酸化水素を水に分解する酵素であり、活性中心にセレンをもつ。

４　正
β-カロテンは、小腸上皮細胞でビタミンAに変換される。なお、β-カロテンは活性酸素である一重項酸素（1O2）を消去する作用を有する。

５　誤
ビタミンEは、主に細胞膜に含まれる不飽和脂肪酸の酸化を抑制する。
