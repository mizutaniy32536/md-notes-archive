# 110-59

心房と心室を直接連絡する副伝導路により、心室の早期興奮が生じるのはどれか。１つ選べ。

１ Adams−Stokes（アダムス・ストークス）症候群
２ Brugada（ブルガダ）症候群
３ QT延長症候群
４ WPW（Wolff−Parkinson−White）症候群
５ 洞不全症候群

**正解:** ４
