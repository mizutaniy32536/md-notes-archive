# 110-283

### 第110回薬剤師国家試験 問283

**問題文（全文）**  
65歳男性。帯状疱疹の予防のためにワクチン接種を希望した。男性は、かかりつけの病院で接種可能かを確認し、接種の予約を行った。接種の当日は平熱で、問診の結果からワクチン接種が可能と判定され、以下のワクチン製剤の1回目の接種を行うことになった。[1][2][4][5]  

ワクチン接種後、被接種者が待合室に向かう途中で気分が悪いとその場に倒れこんだ。その場に居合わせた薬剤師と看護師は、待機している医師に対応を要請した。患者は呼びかけには反応するが、頻脈と蒼白が観察された。駆けつけた医師は被接種者の症状からアナフィラキシーを疑ったため、その場に居合わせた薬剤師と看護師とともにチームとして患者に対応した。[1][2]  

この患者に対する迅速な対応として、適切なのはどれか。**２つ選べ**。[2]

**全選択肢（番号付き）**  
１．患者を仰臥位に保持  
２．アドレナリンの静脈内投与  
３．急速輸液に使用する等張電解質液の準備  
４．二相性反応予防に使用する抗ヒスタミン薬の準備[2]

**正解番号**  
**１、３**[1][8]

**解説**  
アナフィラキシーの疑いがある場合の迅速な対応として、患者を**仰臥位（横臥位）に保持**し（１）、循環不全対策として**急速輸液用の等張電解質液（例：生理食塩水や乳酸リンゲル液）を準備**（３）するのが適切である。[2]  
アドレナリンの**静脈内投与（２）は、皮下注射や筋注が第一選択であり、静脈内は重症時や効果不十分時の代替で迅速対応の優先ではない**。抗ヒスタミン薬（４）は二相性反応予防に有用だが、**一次対応ではなく補助的**であり、迅速対応の優先順位が低い。[1][2]  
ガイドライン（日本アレルギー学会など）に基づき、チーム対応では気道確保・循環維持が優先され、薬剤師は医師支援として輸液準備などに携わる。[1][2]
