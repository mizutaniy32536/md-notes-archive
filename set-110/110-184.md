# 110-184

アトピー性皮膚炎の病態と治療に関する記述として、正しいのはどれか。2つ 選べ。
1. 乳幼児期には、体幹や四肢に乾燥や皮疹が発症したのちに頭部や顔面に拡大す る。
2. 掻痒、左右対称性の湿疹及び慢性・反復性経過を特徴とする。
3. 血液検査では、血中 IgA 値が高値を示す。
4. 外用療法の第一選択は、非ステロイド性抗炎症薬（NSAIDs）である。
5. タクロリムス軟膏を使用する場合は、塗布部位の灼熱感やほてり、感染症に注 意する。
