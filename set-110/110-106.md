# 110-106

下図に示すブレオマイシンA2はDNA鎖を切断することができる。このために必要な化学的性質として、最も適切なのはどれか。１つ選べ。

１ 部分構造アにおけるFe（Ⅱ）との配位能
２ 部分構造イにおけるMg（Ⅱ）との配位能
３ 部分構造ウにおけるMg（Ⅱ）との配位能
４ 部分構造エにおける光増感作用
５ 部分構造オにおけるFe（Ⅱ）との配位能

**正解:** １

**解説**
ブレオマイシンA2は、部分構造アにおいてFe（Ⅱ）【2価の鉄イオン：Fe2＋】とキレート錯体を形成し、この錯体が分子状酸素O2を取り込んで活性酸素に変換し、この活性酸素がDNA鎖を切断することで抗がん作用を示す。なお、部分構造エにおいてはがん細胞のDNAとインターカレーションすることが報告されている。
