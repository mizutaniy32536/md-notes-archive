# 110-337

**問題文**
67歳女性。身長163 cm、体重62 kg。乳がんの術後化学療法としてペルツズマブ、トラスツズマブ、ドセタキセルによる化学療法を施行している。外来日の検査値の結果を踏まえ、医師はペグフィルグラスチム（遺伝子組換え）の投与を行った。また患者は即時入院となった。この患者に使用する抗菌薬として適切なのはどれか。2つ選べ。（検査値）体温38.1℃、好中球数450 /µL、eGFR 72.5 mL/min/1.73 m2

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-337 を参照

**選択肢**
1. セフェピム
2. エリスロマイシン
3. メロペネム
4. ミノサイクリン
5. ダプトマイシン

**正解:** 1,3

**解説**
解答　１、３

乳がんの術後化学療法を施行中の患者で、好中球が450 /µLで38.1℃の発熱があり、医師がペグフィルグラスチムの投与を行っている状況から、本患者は発熱性好中球減少症（FN：Febrile Neutropenia）を発症していると考えられる。
FN患者には、広域スペクトラムを有する抗菌薬（レボフロキサシン、セフタジジム、セフェピム、タゾバクタム・ピペラシリン、メロペネムなど）を用いる。
