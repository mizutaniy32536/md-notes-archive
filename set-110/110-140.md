# 110-140

**問題文**
下図の汚染物質A〜Cは、全国の一般環境大気測定局における窒素酸化物、二酸化硫黄又は光化学オキシダントのいずれかの季節変動を示したものである。汚染物質A〜Cに関する記述のうち、正しいのはどれか。2つ選べ。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-140 を参照

**選択肢**
1. 汚染物質Aは、揮発性有機化合物や非メタン炭化水素に太陽光、特に紫外線が照射されることによって生成する。
2. 汚染物質Bは、移動発生源である自動車のほかに、一般家庭で使用される燃焼器具も発生源となる。
3. 汚染物質Cは、主に固定発生源における化石燃料の燃焼によって生成する。
4. 汚染物質Aの測定には、ザルツマン法が用いられる。
5. 汚染物質Cの測定には、中性ヨウ化カリウム法が用いられる。

**正解:** 2,5

**解説**
解答　２、５

汚染物質Aは二酸化硫黄、Bは窒素酸化物、Cは光化学オキシダントである。二酸化硫黄は、環境基準の設定、排出規制の実施による燃料の低硫黄化及び排煙脱硫装置の設置等により大気中濃度は非常に低く季節変動はほとんどない。窒素酸化物は、燃焼器具の使用量が増加する冬季に大気中濃度が高くなる傾向がある。光化学オキシダントは、生成に紫外線が関与するため、日照時間の長い春から夏にかけて大気中濃度が高くなる傾向がある。

１　誤
汚染物質A（二酸化硫黄）は、主に工場などの固定発生源における化石燃料の燃焼によって生成する。なお。揮発性有機化合物や非メタン炭化水素に太陽光、特に紫外線が照射されることによって生成するのは、汚染物質C（光化学オキシダント）である。

２　正

３　誤
解説１参照

４　誤
汚染物質A（二酸化硫黄）の測定には、溶液導電率法や紫外線蛍光法が用いられる。なお、ザルツマン法は、汚染物質B（窒素酸化物）の測定に用いられる。

５　正
中性ヨウ化カリウム法では、試料空気を中性のヨウ化カリウム溶液に通じ、生成するヨウ素（I2）に由来する呈色物質（I3−）の吸光度を測定することで光化学オキシダント濃度を求める。
