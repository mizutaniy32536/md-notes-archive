# 110-320

（法規・制度・倫理） 薬局薬剤師が薬剤管理指導を請求する保険の算定要件等に関する記述として、適 切なのはどれか。2つ選べ。
1. 在宅患者訪問薬剤管理指導料で請求する。
2. 管理指導に携わる薬剤師は、 ₅ 年以上の実務経験が求められる。
3. 保険薬局の指定を受けていれば、この管理指導に関する実施について、あらた めて届出をする必要はない。

4. ₁ ケ月に請求できる訪問回数に制限が設けられている。
5. 訪問時に居宅サービスの変更につながる情報を得た場合でも、医師以外の関係 者に報告する必要はない。

## 正解
**1 と 4**

## 解説（在宅患者訪問薬剤管理指導）

### 各選択肢の検証

1. ⭕ **在宅患者訪問薬剤管理指導料で請求**
   - 薬局薬剤師が患者宅を訪問して管理指導を行った際の報酬

2. ❌ **実務経験年数の要件はない**
   - 5年以上という要件は設定されていない
   - ただし、在宅医療に関する研修受講は推奨される

3. ❌ **届出が必要**
   - 保険薬局の指定だけでは不十分
   - 地方厚生局に「在宅患者訪問薬剤管理指導の実施に関する届出」が必要

4. ⭕ **訪問回数に制限がある**
   - **原則：月4回まで**
   - **末期がん患者等：週2回・月8回まで**
   - 厚生労働大臣が定める疾病等の患者は例外あり

5. ❌ **医師以外の関係者にも報告が必要**
   - ケアマネジャー、訪問看護師など多職種連携が重要
   - 居宅サービス計画の変更につながる情報は共有すべき

### 在宅患者訪問薬剤管理指導のポイント
- **届出**: 地方厚生局への届出が必須
- **訪問回数**: 月4回まで（末期がん等は月8回まで）
- **多職種連携**: 医師、ケアマネジャー等への情報提供が重要
- **記録**: 訪問薬剤管理指導記録の作成・保管
