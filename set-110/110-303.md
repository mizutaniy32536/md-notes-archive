# 第110回薬剤師国家試験 問303

## 問題文

75歳男性。慢性胃炎の既往がある。2年前に脳梗塞を発症し、それ以来、処方1及び処方2の薬剤を継続的に服用している。遺伝子検査も実施され、EGFR遺伝子変異陽性と判明した。パフォーマンスステータス（PS）1。患者に喫煙歴はなく、機会飲酒のみ。外来通院治療を強く希望したため、ゲフィチニブ（処方3）での治療を開始することになり、処方1、処方2とも総合病院で一括して処方することになった。状態が安定したら退院し、処方1～3の薬剤での治療を継続する予定である。

**処方3の開始にあたり、病棟薬剤師の対応として、適切なのはどれか。2つ選べ。**[1][4]

2. レバミピドをオメプラゾールに変更するよう医師に処方提案する。[1][4]

3. プロトロンビン時間が延長する可能性があるので、ワルファリンカリウムの用量調節を医師に提案する。[4]

## 正解番号

## 解説

**選択肢1について**

手足症候群は、抗がん剤や分子標的薬が正常な細胞に作用し、手足の皮膚や爪の細胞が障害されることにより生じます。代表的症状は、手のひら、足の裏が赤くなる、痛みが生じる、かかとがカサカサするなどです。レボフロキサシンは手足症候群の予防薬ではないため、この提案は適切ではありません。[1]

**選択肢2について**

レバミピドをオメプラゾールに変更するという提案は、検索結果に具体的な根拠が示されていません。[1]

**選択肢3について**

ゲフィチニブはCYP3A4で主に代謝される薬剤であり、CYP2C9阻害作用も有しています。患者が服用しているワルファリンカリウムはCYP2C9で代謝されるため、ゲフィチニブとの相互作用によりプロトロンビン時間が延長する可能性があります。したがって、用量調節を医師に提案することは適切です。[1]

**選択肢5について**

ゲフィチニブは主にCYP3A4で代謝されるため、グレープフルーツジュース（GFJ）摂取によりCYP3A4が阻害される→ゲフィチニブの血中濃度上昇→効果「増強」の可能性があります。「減弱」ではありません。[1]
