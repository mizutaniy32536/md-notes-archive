# 110-252

**問題文**
28歳女性。高校教員として勤務している。ここ数ヶ月、仕事のストレスから風邪をひきやすく、体調が優れない日が続いていた。最近になり、左胸背部に赤い発疹が多く現れ、ピリピリとする痛みが出てきたことから、近医を受診した。診断の結果、帯状疱疹と診断され、以下の処方箋を持って薬局を訪れた。問252（実務）この患者への説明として、適切なのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-252 を参照

**選択肢**
1. 他の教職員や生徒へ感染させる危険性があるため、出勤停止となります。
2. むくみが現れたり、尿量が減った場合は、すぐに知らせてください。
3. 帯状疱疹は再発する可能性があります。
4. 服用期間中は水分摂取を控えるようにしてください。
5. 痛みが強い時は患部を冷やしてください。問253（薬理）処方薬で治療を継続した結果、皮疹は治まったものの、その部位で断続的な刺すような痛みや持続的で焼けるような痛みが生じるようになった。これらの症状に対して使用する薬物の作用機序として、正しいのはどれか。2つ選べ。
6. 電位依存性Ca2＋チャネルの機能の抑制
7. γ-アミノ酪酸GABAB受容体の刺激
8. オピオイドµ受容体の刺激
9. セロトニン5-HT1B/1D受容体の刺激
10. カルシトニン遺伝子関連ペプチド（CGRP）受容体の遮断

**正解:** 2,3

**解説**
問252　解答　２、３

１　誤
帯状疱疹ウイルスは飛沫・接触感染するが、他の教職員や生徒へ感染させる危険性は比較的低いため、必ずしも感染者が出勤停止になるわけではない。

２　正
バラシクロビルは副作用に腎障害があるため、むくみや尿量が減るなどの腎機能低下の兆候があればすぐに知らせるように指導する。

３　正
帯状疱疹は発症後もウイルスが体内に潜伏しているため、免疫力低下時に再発する可能性がある。

４　誤
バラシクロビルは、脱水状態では腎障害を起こしやすくなるため、バラシクロビルの腎排泄を促進させるために適切に水分摂取を行う。

５　誤
帯状疱疹の疼痛は、患部を冷やすと神経の刺激で痛みが悪化するため、温めることで疼痛が軽減する。

問253　解答　１、３

本患者の症状から帯状疱疹後神経痛を発症していると考えられる。帯状疱疹後神経痛の治療薬としてはプレガバリンや、トラマドールなどが用いられる。

１　正
プレガバリンなどに関する記述である。プレガバリンは、電位依存性Ca2＋チャネルのα2δサブユニットと結合しCa2＋流入を抑制することで、グルタミン酸等の神経伝達物質遊離を抑制するため、鎮痛作用を示す。

２　誤
バクロフェンに関する記述である。バクロフェンは、γ-アミノ酪酸GABAB受容体を刺激し、脊髄の単シナプス及び多シナプス反射を抑制することで、筋弛緩作用を示す。

３　正
トラマドールなどに関する記述である。トラマドールは、オピオイドµ受容体刺激作用とセロトニン、ノルアドレナリン再取り込み阻害作用による下行性疼痛抑制系の活性化により鎮痛作用を示す。

４　誤
スマトリプタンなどに関する記述である。スマトリプタンは、セロトニン5-HT1B/1D受容体を刺激して、頭痛発作時に拡張した血管を収縮させることにより片頭痛発作を改善する。

５　誤
エレヌマブの記述である。エレヌマブは、抗カルシトニン遺伝子関連ペプチド（CGRP）受容体抗体であり、CGRPのCGRP受容体への結合を抑制し、片頭痛発作の発症を抑制する。
