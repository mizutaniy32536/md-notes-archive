# 110-199

### 第110回薬剤師国家試験 問199

**問題文（全文）**  
38歳女性。アレルギー性鼻炎で近医を受診し、処方1による治療を受けていた。しかし、薬剤師は処方1の薬剤の供給が不安定になったとの情報を得たため、医師と他剤への変更を検討し、処方1は処方2へ変更された。なお、処方1の薬剤と処方2の薬剤は、添加剤が同じものを調剤した。[1][2]  

この原末中のレボセチリジン塩酸塩の含量に最も近いのはどれか。１つ選べ。ただし、原末中にはR−エナンチオマー以外に旋光性を示す物質は含まないものとする。[2]  

（注：問199は物理・化学・生物分野の実践問題で、比旋光度と実測旋光度を用いた含量計算問題。詳細な数値データやグラフは検索結果に記載なしのため、標準的な比旋光度公式 \([ \alpha ] = \frac{\alpha \times 100}{c \times l}\) を基に解く基礎知識が必要。[1][2]）

**全選択肢（番号付き）**  
1. 92.2 %  
2. 94.1 %  
3. 96.3 %  
4. 98.4 %  
5. 100.7 %[1]

**正解番号**  
3[1]

**解説**  
問199は比旋光度 \([ \alpha ]\) と実測旋光度 \(\alpha\) の関係（\([ \alpha ] = \frac{\alpha \times 100}{c \times l}\), ここで \(l\) は試料セルの長さ[mm]）を基礎知識として用いた含量計算問題。正解は96.3%で、原末中のレボセチリジン塩酸塩（S-エナンチオマー）の含量を求め、R-エナンチオマーは旋光性を示さない前提で計算。[1][2] これは物理化学の標準問題で、与えられた旋光度データから最も近い値を選択。[1]
