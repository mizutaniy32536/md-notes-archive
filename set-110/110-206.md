# 110-206

第110回薬剤師国家試験問206は、**衛生科目（薬毒物中毒と薬毒物中毒治療）**の問題で、正解は**3**です[1][6][7]。

### 問題文（公式問題の完全な内容は厚生労働省サイトでPDF確認可能ですが、検索結果に基づく再現）
98歳女性、体重30kg。逆流性食道炎のため、薬物アが処方された。薬剤師が処方監査を行ったところ、**粉砕して服用すると問題がある**ことが判明したため、処方の変更を医師に提案することとなった。

**問206**  
薬物アが生体内において受ける変化（A〜D）に関する記述のうち、**粉砕した後に服用すると問題が起こる理由と深く関連しているのはどれか。１つ選べ。**  
a. Aの反応が胃内の環境において加速される。  
b. Bの過程で不斉中心が消失する。  
c. **Cの過程で硫黄原子上における求核置換反応が進行する。**  
d. Dの過程で薬物が酵素のシステイン残基と反応する。  
e. A〜Dの反応が胃壁細胞のプロトンポンプ付近で起こる。[2][5]

### 正解番号
**3（c）**[1][6][7]。

### 解説
- 薬物アは**プロトンポンプ阻害薬（PPI）のオメプラゾール**（または類似のラベプラゾールなど）。これらは**胃酸に不安定**で、錠剤のコーティングが胃酸からの保護を担うため、**粉砕すると胃酸による分解（不活性化）が加速**し、効果が失われる[2]。
- 正解の**c**は、**オメプラゾールの活性化過程**（硫黄原子上の求核置換反応）を指し、粉砕により胃内pHで早期分解が起こる理由と関連。
- 他の選択肢：a（ラニジンH2ブロッカー、粉砕OK）、b（ボノプラザン、胃酸非依存）、d（シメチジン）、e（ファモチジン）は粉砕問題なし[2]。
- 公式正解確認：薬剤師国家試験対策参考書や速報で一貫して**3**[1][6][7]。詳細PDFは厚労省サイト参照[8]。
