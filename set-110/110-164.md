# 110-164

脂質異常症治療薬に関する記述として、正しいのはどれか。２つ選べ。

１ アトルバスタチンは、3−ヒドロキシ−3−メチルグルタリルCoA（HMG−CoA）還元酵素を阻害して、肝細胞の低密度リポタンパク質（LDL）受容体を増加させる。
２ コレスチミドは、転写因子SREBP−1cの活性を抑制して、トリグリセリドの合成を抑制する。
３ プロブコールは、小腸コレステロールトランスポーターを阻害して、腸管からのコレステロール吸収を抑制する。
４ ベザフィブラートは、ミクロソームトリグリセリド輸送タンパク質（MTP）を阻害して、超低密度リポタンパク質（VLDL）の産生を抑制する。
５ ニコモールは、ニコチン酸受容体を刺激し、脂肪組織における脂肪分解を抑制して、肝臓への遊離脂肪酸の動員を減少させる。

**正解:** １、５

**解説**
１　正
アトルバスタチンは、3−ヒドロキシ−3−メチルグルタリルCoA（HMG−CoA）還元酵素を阻害して、肝細胞の低密度リポタンパク質（LDL）受容体を増加させることで、血中LDLコレステロールを低下させる。
２　誤
コレスチミドは、陰イオン交換樹脂であり、腸管内で胆汁酸と結合し胆汁酸の排泄を促進する。これにより、肝細胞でコレステロールから胆汁酸への異化が亢進し、肝臓でのコレステロールの需要が高まることで、血中LDLコレステロール値を低下させる。なお、脂質合成系の諸酵素の活性を調節している転写因子SREBP−1cの活性を抑制して、トリグリセリドの合成を抑制する薬剤は、イコサペント酸エチルである。
３　誤
プロブコールは、コレステロールの胆汁酸への異化排泄を促進して、血中LDLコレステロールを低下させる。なお、小腸コレステロールトランスポーター（NPC1L1）を阻害して、腸管からのコレステロール吸収を抑制する薬剤は、エゼチミブである。
４　誤
ベザフィブラートは、脂肪細胞のペルオキシソーム増殖剤応答性受容体α（PPARα）を刺激することにより、リポタンパクリパーゼ（LPL）を活性化し、血中トリグリセリド（TG）を低下させる。なお、ミクロソームトリグリセリド輸送タンパク質（MTP）を阻害して、超低密度リポタンパク質（VLDL）の産生を抑制する薬剤は、ロミタピドである。
５　正
ニコモールは、ニコチン酸受容体を刺激し、脂肪組織における脂肪分解を抑制して、肝臓への遊離脂肪酸の動員を減少させることで、肝臓での超低比重リポタンパク質（VLDL）の合成を抑制する。
