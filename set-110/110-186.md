# 110-186

### 第110回薬剤師国家試験 問186

**問題文**  
17歳男性。14歳時、起床時に右肩がぴくついた後、意識が消失した。病院において脳波検査で異常が指摘され、てんかんと診断された。バルプロ酸ナトリウムによる治療が開始され、問題なく経過していた。今回、全般性強直間代発作が出現し、重積状態になったため救急搬送された。20分ほどけいれん発作が持続している。本患者のけいれん抑制のために最初に静脈内投与される薬物として、適切なのはどれか。２つ選べ。[2][4][9]

**全選択肢（番号付き）**  
１ メタンフェタミン  
２ ジアゼパム  
３ プロポフォール  
４ ロラゼパム  
５ フェノバルビタール[2][4]

**正解番号**  
２、４[2][5][6]

**解説**  
本問は、てんかん重積状態（発作が20分持続）に対する初期静脈内投与薬を選択する問題である。てんかん重積状態の初期治療ガイドラインでは、ベンゾジアゼピン系薬剤（ジアゼパム、ロラゼパム）が第一選択として静注され、迅速なけいれん抑制を図る。メタンフェタミンは覚醒剤であり不適切、プロポフォールやフェノバルビタールはベンゾジアゼピン抵抗性重積状態の第2選択以降に用いられる。[2][4][5][6]
