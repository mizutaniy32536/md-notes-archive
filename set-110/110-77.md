# 110-77

介護保険法において、第2号被保険者が要介護認定を受けられる特定疾病に該当するのはどれか。１つ選べ。

1. 関節リウマチ
2. 結核
3. C型肝炎
4. 高血圧症
5. 脂質異常症

## 解答

**１（関節リウマチ）**

## e-REC解説

介護保険法において、被保険者は第1号被保険者と第2号被保険者に分けられる。

- **第1号被保険者：** 市町村の区域内に住所を有する65歳以上の者
- **第2号被保険者：** 市町村の区域内に住所を有する40歳以上65歳未満の医療保険加入者

介護保険法による保険給付を受給するためには、要介護認定もしくは要支援認定を受けなければならないが、第2号被保険者は要介護状態もしくは要支援状態の原因である身体上又は精神上の障害が**特定疾病**によって生じたものでなければならない。
