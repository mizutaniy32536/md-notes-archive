# 110-197

### 第110回薬剤師国家試験 問197

**問題文（共通症例）**  
70歳男性。陳旧性心筋梗塞、胆管結石の既往歴があり、交通外傷で救急搬送され、入院となった。気管挿管後人工呼吸器管理とし、緊急輸血を行い、生理食塩液とアドレナリン注1 mgを投与した。[1][2]

**問197（実務）**  
生理食塩液の持続投与に加え、8.4 w/v％NaHCO3注射液が処方された理由として最も適切なのはどれか。１つ選べ。[1][2]

１．ケトアシドーシスであったため。  
２．呼吸性アシドーシスであったため。  
３．代謝性アシドーシスであったため。  
４．PaO2が高値であったため。  
５．ヘモグロビンが低値であったため。[1][2]

**正解番号：３**

**解説**  
患者は交通外傷によるショック状態で、血ガス分析から点滴開始前の血中HCO₃⁻濃度が10.1 mEq/Lと低値を示し、pHも低下しており、**代謝性アシドーシス**が認められる。NaHCO₃注射液は代謝性アシドーシスの補正に用いられるため、これが最も適切。[1][2]  
１．誤：ケトアシドーシスを示唆する血糖値やケトン体の上昇は見られない。  
２．誤：人工呼吸器管理下でPaCO₂が正常範囲であり、呼吸性アシドーシスではない。  
４．誤：PaO₂は130 mmHg以上と高値だが、これは酸素投与や人工呼吸の影響で、NaHCO₃投与の適応外。  
５．誤：ヘモグロビン低値は輸血で対応しており、NaHCO₃の理由ではない。[2]
