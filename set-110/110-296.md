# 110-296

（病態・薬物治療） この患者の病態に関する記述として正しいのはどれか。2つ選べ。
1. 関節リウマチの典型的な初期症状が発現している。
2. 発症には、主に細胞性免疫（Ⅳ型アレルギー）が関与する。
3. 検査結果より、ループス腎炎は否定できる。
4. 手指に Raynaud（レイノー）現象が認められる。
5. 粘膜症状や精神神経系症状など多様な全身症状が現れることがある。
