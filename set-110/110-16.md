# 110-16

**問題文**
下図の年次推移を示す我が国の人口統計指標はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-16 を参照

**選択肢**
1. 出生率
2. 合計特殊出生率
3. 総再生産率
4. 粗死亡率
5. 年齢調整死亡率

**正解:** 4

**解説**
解答　４

１　誤
出生率は、人口1000人あたりの年間の出生数である。非婚化、晩婚化の影響により近年は減少傾向である。

２　誤
合計特殊出生率は、15〜49歳の女性の年齢別出生率の合計であり、1人の母親が残す平均男女児数を表す。非婚化の影響により近年は減少傾向である。

３　誤
総再生産率は、15〜49歳の女性の年齢別女児出生率の合計であり、１人の母親が残す平均女児数を表す。非婚化の影響により近年は減少傾向である。

４　正
粗死亡率は、人口1000人あたりの年間の死亡数である。高齢化の影響により1985年頃から増加傾向である。

５　誤
年齢調整死亡率は、年齢構成に差がある集団間での死亡状況を比較できるように、年齢構成を調整し統一させた死亡率のことをいう。年齢調整死亡率は減少傾向である。
