# 110-91

理想気体からなる閉じた系における熱力学第一法則に関する記述として、正しいのはどれか。２つ選べ。

１ 定温過程において、系に加えられた熱量はすべて系がする仕事になる。
２ 内部エネルギー変化は、系に加えられた熱量と系が外界からされた仕事の和で表される。
３ 断熱過程において、系の体積が増加すると内部エネルギーも増加する。
４ 内部エネルギー変化は経路関数である熱と仕事からなり、それ自体も経路関数である。
５ 系が外界に対してする仕事は、不可逆過程の方が可逆過程より大きい。

**正解:** １、２

**解説**
熱力学第一法則とは「内部エネルギー変化（ΔU）は、系に加えられた熱量（Q）と系が外界からされた仕事（W）の和で表される」という法則であり、一般にΔU＝Q＋Wと表される。
１　正
内部エネルギーは絶対温度に比例するため、定温過程では内部エネルギー変化がゼロ（ΔU＝0）となる。したがって、定温過程における熱力学第一法則はQ＝－Wと表現できるため、系に加えられた熱量（Q）はすべて系がする仕事（－W）になる。
２　正
前記参照
３　誤
断熱過程では熱の出入りがないためQ＝0となる。したがって、断熱過程における熱力学第一法則はΔU＝Wと表現できるため、内部エネルギー変化（ΔU）はすべて系がされた仕事（W）になる。系の体積が増加すると、系が外界に仕事をするため、内部エネルギーは減少する。
４　誤
内部エネルギー変化（ΔU）は経路関数である熱（Q）と仕事（W）からなるが、内部エネルギー変化（ΔU）自体は状態関数（示量性状態関数）である。
５　誤
系が外界に対してする仕事は、常に平衡状態を保ちながら進む理想的な変化である可逆過程の方が不可逆過程より大きい。
