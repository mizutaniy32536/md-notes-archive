# 110-279

# 第110回薬剤師国家試験 問279

## 問題文

処方1の薬剤が持効性を示す機構として、正しいのはどれか。**1つ選べ**。[4]

## 全選択肢

1. インスリン分子を結晶化することで、溶解性を低下させた。[4]

2. 投与後、皮下組織において、インスリン分子が安定した可溶性のマルチヘキサマーを形成するようにした。[4]

3. インスリン分子の等電点を改変することで、生理的なpHで微細な沈殿物を形成するようにした。[4]

4. インスリン分子に脂肪酸を結合させることで、血中でアルブミンと複合体を形成するようにした。[4]

## 正解番号

**4**[7]

## 解説

本問は妊娠糖尿病患者に処方されたインスリンデテミル（処方1）の持効性機構に関する問題です。[2]

インスリンデテミルは、インスリン分子の30番目のリシン残基に**ミリスチン酸（脂肪酸）を共有結合**させた長時間作用型インスリンです。[4] この脂肪酸の結合により、血中でアルブミンと複合体を形成することで、インスリンの放出が遅延し、**約24時間の持効性**を示します。[4]

他の選択肢について：
- 選択肢1：結晶化はNPH（中間型）インスリンの機構です。
- 選択肢2：マルチヘキサマー形成はインスリングラルギンなど他の製剤の機構です。
- 選択肢3：等電点の改変による沈殿物形成もNPHインスリンの機構です。
