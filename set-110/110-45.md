# 110-45

**問題文**
下図は腎臓のネフロンを模式的に示したものである。図中の1〜5のうち、有機アニオントランスポーター（OAT1、OAT3）を介した薬物の輸送を表すのはどれか。1つ選べ。ただし、図中の矢印の向きは薬物が移行する方向を示す。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-45 を参照

**選択肢**
（選択肢は画像を参照）

**正解:** 3

**解説**
解答　３

有機アニオントランスポーター（OAT1、OAT3）は、主に腎臓のネフロンの近位尿細管上皮細胞の側底膜上に存在し、主に酸性薬物のような生体内で有機アニオンとなる薬物を血管側から近位尿細管上皮細胞内へ輸送することで、尿細管管腔側への薬物輸送に関与する。
