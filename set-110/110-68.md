# 110-68

**問題文**
胆石症の疝痛発作時に、疼痛緩和のために使用される薬物として最も適切なのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-68 を参照

**選択肢**
1. ウルソデオキシコール酸
2. エストラジオール
3. ブチルスコポラミン臭化物
4. ベザフィブラート
5. モルヒネ塩酸塩

**正解:** 3

**解説**
解答　３

胆石症の疝痛発作は、胆嚢頸部や胆嚢管に胆石が嵌頓することにより生じ、心窩部痛、右季肋部痛が多く、しばしば右背中・右肩に放散する。疝痛発作の疼痛緩和には、抗コリン薬（ブチルスコポラミンなど）やNSAIDs（ジクロフェナクなど）が使用される。
