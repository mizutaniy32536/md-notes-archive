# 110-68

胆石症の疝痛発作時に、疼痛緩和のために使用される薬物として最も適切なのはどれか。１つ選べ。

１ ウルソデオキシコール酸
２ エストラジオール
３ ブチルスコポラミン臭化物
４ ベザフィブラート
５ モルヒネ塩酸塩

**正解:** ３

**解説**
胆石症の疝痛発作は、胆嚢頸部や胆嚢管に胆石が嵌頓することにより生じ、心窩部痛、右季肋部痛が多く、しばしば右背中・右肩に放散する。疝痛発作の疼痛緩和には、抗コリン薬（ブチルスコポラミンなど）やNSAIDs（ジクロフェナクなど）が使用される。
