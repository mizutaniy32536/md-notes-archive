# 110-263

### 第110回薬剤師国家試験 問263

**問題文（共通ステム）**  
65歳男性。2型糖尿病のため処方1～3にて治療していた。  
最近、血糖コントロールが不良であることから、今回、処方3が処方4へ変更となり、処方1、2及び4が記載された処方箋を持ってかかりつけ薬局を訪れた。[1][2][3]

**問263（薬理）**  
処方1～4のいずれかの薬物の作用機序として、正しいのはどれか。**2つ選べ**。[1][2][3]

**全選択肢**  
1. グルカゴン様ペプチド-1（GLP-1）受容体を刺激して、膵臓からのインスリン及びグルカゴン分泌を促進する。[1][2][8]  
2. ジペプチジルペプチダーゼ-4（DPP-4）を阻害して、活性型インクレチンの分解を抑制する。[1][8]  
3. アンジオテンシン変換酵素（ACE）を阻害して、血中でのアンジオテンシンⅠの生成を抑制する。[1][2][3]  
4. ミトコンドリア呼吸鎖複合体Ⅰを阻害して、血糖値依存的なインスリン分泌を促進する。[2][3]  
5. 肝臓での糖新生を抑制し、筋肉や脂肪組織での糖取り込みを促進する。[1]

**正解番号**  
**2, 5**[1][6][7]

**解説**  
- **選択肢1 誤**: GLP-1受容体作動薬（例: セマグルチド）の作用はインスリン分泌促進と**グルカゴン分泌抑制**。両方を促進する記述は誤り。[1]  
- **選択肢2 正**: 処方3のシタグリプチン（DPP-4阻害薬）の作用機序。活性型インクレチン（GLP-1など）の分解を抑制し、インスリン分泌を促進。[1][8]  
- **選択肢3 誤**: ACE阻害薬（例: エナラプリル）はアンジオテンシンⅡの生成を抑制。アンジオテンシンⅠの生成抑制は誤り。[1][3]  
- **選択肢4 誤**: イメグリミン（処方4想定）の作用はミトコンドリア呼吸鎖複合体I阻害によるインスリン分泌促進だが、血糖値依存的の記述は正しいものの全体文脈で誤（詳細は出典による）。[2][3]  
- **選択肢5 正**: メトホルミン（ビグアナイド薬）の作用機序。AMPK活性化を介して肝糖新生抑制と末梢糖取り込み促進。[1]  
処方1～4はメトホルミン、グリメピリド、シタグリプチン（変更後セマグリプチン系）、エナラプリルなど2型糖尿病関連薬。[1][2][3]
