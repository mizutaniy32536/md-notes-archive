# 110-246

（薬理） 処方 ₁ ～ ₃ のいずれかの薬物に関する記述として、正しいのはどれか。2つ選 べ。
1. ドパミン D2 受容体に部分アゴニストとして作用することで、ドパミン神経系 を安定化させる。
2. ヒスタミン H1 受容体及びアドレナリン a1 受容体を刺激することで、統合失調 症の陰性症状を改善する。
3. ドパミン D2 受容体を遮断することで、統合失調症の陽性症状を改善する。
4. c︲アミノ酪酸 GABAA 受容体のベンゾジアゼピン結合部位に結合することで、 睡眠を誘発する。
5. メラトニン MT1 及び MT2 受容体を刺激することで、睡眠︲覚醒リズムを正常 化する。
