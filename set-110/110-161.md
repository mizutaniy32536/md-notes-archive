# 110-161

過敏性腸症候群治療薬に関する記述として、正しいのはどれか。２つ選べ。

１ ポリカルボフィルカルシウムは、胃内の酸性条件下でカルシウムを脱離し、腸管腔内において膨潤・ゲル化することで、水分バランスを調節する。
２ ラモセトロンは、求心性神経終末に存在するセロトニン5−HT3
３ メペンゾラートは、副交感神経のセロトニン5−HT4
４ トリメブチンは、消化管運動亢進時には、アドレナリン作動性神経のオピオイドµ
５ リナクロチドは、胆汁酸トランスポーターを阻害し、胆汁酸の再吸収を抑制することで、腸管内に水分及び電解質を分泌させる。

**正解:** １、２

**解説**
１　正
ポリカルボフィルカルシウムは、胃内の酸性条件下でカルシウムを脱離し、腸管腔内において膨潤・ゲル化することで、水分バランスを調節するため、過敏性腸症候群における下痢、便秘に対して効果を示す。
２　正
ラモセトロンは、求心性神経終末のセロトニン5−HT3受容体遮断による大腸痛覚の伝達抑制及び、腸管のセロトニン5-HT3受容体遮断による下痢抑制により、過敏性腸症候群における下痢に対して効果を示す。
３　誤
メペンゾラートは、アセチルコリンM3受容体を遮断することで、腸運動を抑制する。なお、副交感神経のセロトニン5−HT4受容体を刺激し、アセチルコリン遊離を促進することで、腸運動を亢進する薬剤は、モサプリドである。
４　誤
トリメブチンは、消化管運動亢進時には、コリン作動性神経のオピオイドµ受容体を刺激することで、アセチルコリンの遊離を抑制し、腸運動を抑制する。一方、消化管運動低下時には、アドレナリン作動性神経のオピオイドµ受容体を刺激することで、ノルアドレナリンの遊離を抑制する。その結果、コリン作動性神経からのアセチルコリンの遊離が増加し、腸運動は亢進する。
５　誤
リナクロチドは、腸管表面に存在するグアニル酸シクラーゼC受容体を活性化し、サイクリックGMP（cGMP）濃度を増加させることで、腸管分泌及び腸管輸送を促進する。なお、胆汁酸トランスポーターを阻害し、胆汁酸の再吸収を抑制することで、腸管内に水分及び電解質を分泌させる薬剤は、エロビキシバットである。
