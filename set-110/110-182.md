# 110-182

**問題文**
粘膜に適用する製剤に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-182 を参照

**選択肢**
1. バッカル錠は、咀嚼により、有効成分を放出する口腔用錠剤である。
2. 点鼻粉末剤には、通例、密閉容器が用いられる。
3. 点眼剤の非水性溶剤には、通例、植物油が用いられる。
4. 眼軟膏剤に含まれる粒子の最大粒子径は、通例、150 µm以下である。
5. 坐剤の油脂性基剤として、マクロゴールが用いられる。

**正解:** 2,3

**解説**
解答　２、３

１　誤
日本薬局方において、バッカル錠は、有効成分を臼歯と頬の間で徐々に溶解させ、口腔粘膜から吸収させる口腔用錠剤と定義されている。なお、咀嚼により、有効成分を放出する口腔用錠剤と定義されているのは、ガム剤である。

２　正
点鼻粉末剤には、通例、密閉容器が用いられる。

３　正
点眼剤の非水性溶剤には、通例、植物油が用いられる。また、そのほかの適切な有機溶剤が用いられることもある。

４　誤
眼軟膏剤に含まれる粒子の最大粒子径は、通例、75 µm以下である。

５　誤
坐剤の油脂性基剤として、ハードファットなどが用いられる。なお、マクロゴールは、坐剤の水溶性基剤として用いられる。
