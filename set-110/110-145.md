# 110-145

### 第110回薬剤師国家試験 問145

**問題文**  
劇薬の取扱いについて、正しいのはどれか。２つ選べ。[1][3][5]

**選択肢**  
1. 貯蔵又は陳列の際には、他の医薬品と区別して貯蔵し、又は陳列しなければならない。  
2. 貯蔵する場所には、かぎを施さなければならない。  
3. 18歳未満の者に交付してはならない。  
4. 劇薬を譲渡する際に譲受人から交付される文書には、品名、数量、使用目的等の記載が必要である。  
5. 譲受人から交付された文書の保存期間は3年間である。[1][3][5]

**正解番号**  
1, 4[1][3][5][6]

**解説**  
- **1 正**: 劇薬（毒薬も同様）は、他の医薬品と区別して貯蔵・陳列しなければならない（医薬品医療機器等法施行規則）。[1][3][5]  
- **2 誤**: 貯蔵場所に鍵を施す義務はない（施錠は望ましいが規定なし）。[1][2][3][5]  
- **3 誤**: 劇薬の交付禁止は14歳未満であり、18歳未満ではない。[1]  
- **4 正**: 劇薬譲渡時、譲受人から品名・数量・使用目的等を記載した文書を受け取る必要がある。[1][2][3][5]  
- **5 誤**: 文書の保存期間は2年間であり、3年間ではない。[1][2]  

正解は劇薬取扱いの法令規定に基づく1と4。[1][3][5][6]
