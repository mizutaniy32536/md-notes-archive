# 110-142

**問題文**
薬局の開設者が、患者の同意なしでも患者の個人情報を第三者に提供できるのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-142 を参照

**選択肢**
1. 患者が通学する学校の養護教諭から、今後の健康管理のため患者の処方薬について照会があった場合
2. 民間保険会社から、保険加入の勧誘のため患者の連絡先について照会があった場合
3. 患者が意識不明で緊急搬送された医療機関から、患者の薬剤服用歴について照会があった場合
4. 服薬指導時に得た患者情報から処方内容に疑義が生じ、処方医に照会する場合
5. 患者が勤める勤務先から、健康状態の確認のため既往歴について照会があった場合

**正解:** 3,4

**解説**
解答　３、４

薬局は個人情報取扱事業者に該当し、個人情報取扱事業者は次に掲げる場合を除いて、あらかじめ本人の同意を得ないで、薬剤服用歴等の個人データを第三者に提供することはできない。

①　法令に基づく場合
②　人の生命、身体又は財産の保護のために必要がある場合であって、本人の同意を得ることが困難である場合　　など

よって、本問の選択肢のうち、薬局開設者が、患者の同意なしでも患者の個人情報を第三者に提供できるのは、患者が意識不明で緊急搬送された医療機関から、患者の薬剤服用歴について照会があった場合（選択肢３）、及び、服薬指導時に得た患者情報から処方内容に疑義が生じ、処方医に照会する場合（選択肢４）である。
なお、選択肢４に関して、処方内容に疑義が生じたとき、その疑わしい点を確かめた後でなければ調剤してはならないことについては、薬剤師法において規定されている。
