## ◆ CKD患者における食塩摂取とESRD発症のオッズ比を算出するシナリオ

260311

**問題**
この患者が内科で出された処方箋を持って家族と来局した。この患者に服薬指導した際に、家族から食生活に関する質問があり、食塩の摂取を控えるよう助言した。薬剤師が腎臓病に関連するリスク因子について論文検索したところ、食塩摂取量と末期腎不全（End Stage Renal Disease：ESRD）発症との関連性について検討した論文があった。その論文においては、慢性腎臓病の患者500人のうち、100人がESRDを発症し、食塩6.0g/日以上を摂取していた患者は85人であった。また、ESRD を発症した100人のうち、食塩6.0g/日以上を摂取していた患者が20人であった。
食塩摂取（6.0g/日以上）によるESRD発症のオッズ比として最も近い値はどれか。1つ選べ。

**正解と理由**
- `1. 1.3`
- **理由の構成**
	- **問題の整理**：CKD患者500人のデータから`オッズ比`を計算する。まず2×2表を正確に作成する
	- **薬学的知識・仕様・制約**：`オッズ比`（OR）= (a×d)/(b×c)。問題文から各セルの値を正確に読み取ることが最重要
	- **選択理由**：2×2表を作成して計算するとOR≈1.3

**解法の手順（思考の流れ）**
- まず問題文から数値を整理する
	- CKD患者総数：500人
	- ESRD発症者：100人 → ESRD非発症者：400人
	- 食塩6.0g/日以上の全患者：85人 → 食塩6.0g/日未満：415人
	- ESRD発症かつ食塩≧6.0g/日：20人
- 2×2表を作成：

|  | ESRD発症 | ESRD非発症 | 計 |
|---|---|---|---|
| 食塩≧6.0g/日 | a=20 | b=65 | 85 |
| 食塩<6.0g/日 | c=80 | d=335 | 415 |
| 計 | 100 | 400 | 500 |

- b = 85 - 20 = 65（食塩≧6.0g/日の全85人 − ESRD発症20人）
- c = 100 - 20 = 80（ESRD発症100人 − 食塩≧6.0g/日の20人）
- d = 400 - 65 = 335（ESRD非発症400人 − 食塩≧6.0g/日のESRD非発症65人）
- OR = (a×d)/(b×c) = (20×335)/(65×80) = 6700/5200 ≈ **1.288 ≈ 1.3**

**他の選択肢が誤りな理由**
- 2. 2.3
	2×2表の数値を誤って配置した場合や、計算過程での間違いによる値

- 3. 3.7
	問題文の数値の読み違えによる計算結果

- 4. 4.1
	食塩≧6.0g/日の85人をすべてESRD発症群に誤って配置した場合の計算に近い

- 5. 4.9
	2×2表の作成で大きな誤りがある場合の値

**キーポイント**
- `オッズ比`計算の最大のポイントは2×2表の正確な作成。問題文から各セルを丁寧に読み取る
- この問題は「食塩6.0g/日以上の全患者数」と「ESRD発症者中の食塩6.0g/日以上の人数」が別々に与えられている
- OR = (a×d)/(b×c) の公式を正確に適用するだけ
- OR≈1.3はリスクがやや増加するが大きくはない値

**関連知識の比較**
- `オッズ比`=1：関連なし、>1：リスク増加、<1：防御的
- `CKD`患者の食塩制限：6g/日未満が推奨（CKD診療ガイドライン）
- `ESRD`（末期腎不全）：eGFR<15（ステージG5）、透析導入の対象
- 腎臓病と食事療法：食塩制限（6g/日未満）、タンパク質制限、カリウム制限

（一言メモ）
- 2×2表の作成は「引き算で埋める」のがコツ。全体から部分を引いて残りのセルを計算する
- 問題文の「500人のうち食塩≧6.0gが85人」と「ESRD発症100人中食塩≧6.0gが20人」を混同しないこと
- `CKD`患者の`食塩`制限は6g/日未満。日本人の平均食塩摂取量は約10g/日で大幅に超過している
