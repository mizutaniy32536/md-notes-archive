# 110-261

### 第110回薬剤師国家試験 問261

**問題文（全文）**  
48歳男性。2ヶ月前から食欲の低下を感じ、消化器内科で精査したところ、ヘリコバクター・ピロリ陽性と診断され、一次除菌を行うこととなった。  
**問261（薬理）**  
ヘリコバクター・ピロリの除菌療法において、ボノプラザン以外にラベプラゾールも用いられてきた。ボノプラザン又はラベプラゾールに関する記述として、正しいのはどれか。2つ選べ。[1][4][9][10]

2. （正：妥当な記述）[1]  
3. （誤：詳細不明だが、解説で不適切）[1]  
4. （誤：K⁺と競合するのはボノプラザン。PPIの活性化に関連）[1]  
5. （正：PPIであるラベプラゾールについての記述）[1][4]

**正解番号**  
**2, 5** [1][7]

**解説**  
PPI（プロトンポンプ阻害薬）とP-CAB（カリウム競合的酸阻害薬）の比較に関する問題。以下の3つのポイントに注目：  
1. **酸による活性化**：PPI（ラベプラゾールを含む）は胃内酸で活性化されるが、ボノプラザン（P-CAB）は酸による活性化を必要としない。[1][4]  
2. **プロトンポンプとの結合**：PPIはシスチン残基と共有結合するが、ボノプラザンは非共有結合。[1]  
3. **分泌細管への貯留**：PPIは胃壁細胞の分泌細管に貯留するが、ボノプラザンは貯留しない。[1]  

- 選択肢2：妥当（PPI/P-CABの特性に合致）。[1]  
- 選択肢5：妥当（ラベプラゾールのPPIとしての記述）。[1]  
- 選択肢4：誤り（K⁺と競合するのはボノプラザンの特徴で、ラベプラゾールには該当せず）。[1]  
問260（服薬指導）と連動し、ヘリコバクター・ピロリ除菌療法の薬理を問う。[4][9][10]
