# 110-245

### 第110回薬剤師国家試験 問245

**問題文（全文）**  
病院で、以下の事故事例を用いて、抗がん剤シクロホスファミド水和物の調製に関する新人薬剤師の研修会を行うことになった。  
＜事故事例＞  
シクロホスファミド水和物の溶解に閉鎖式薬物移送システム（CSTD）で調製するところ、その器具がなかった。薬剤師は安全キャビネットを使用して、シクロホスファミド水和物500 mg（凍結乾燥粉末）を生理食塩液25 mLで溶解させているときに18Gの注射針がシリンジから外れ、誤って薬液を飛散させてしまった。  

**問245（衛生）**  
この過程で排出されたものの廃棄に関して、新人薬剤師が備えておくべき知識として正しいのはどれか。**2つ選べ。**[1][2][3][4]

**全選択肢（番号付き）**  
1. 使用した注射針は、図のように針カバーに収納して廃棄する。  
2. 飛散した薬液を拭き取ったガーゼは、一般廃棄物として廃棄する。  
3. 調製に使用した手袋、ガウン、フェイスシールド等は、焼却廃棄物として廃棄する。  
4. 安全キャビネットの表面を清拭したガーゼは、一般廃棄物として廃棄する。  
5. 使用済みシリンジは、針を曲げて破壊してから廃棄する。[1][2][3][4]

**正解番号**  
**1、3**[3][9]

**解説**  
この問題は、抗がん剤調製過程で発生した廃棄物の適切な処理に関する知識を問う。抗がん剤曝露リスクを考慮し、感染性廃棄物や有害廃棄物として分類される物品の廃棄方法がポイント。  
- **1番正解**: 使用済み注射針は、飛散・刺創防止のため針カバーに収納して焼却廃棄物として処理する。直接手で触れず安全に管理。[3][4]  
- **3番正解**: 手袋、ガウン、フェイスシールド等は抗がん剤汚染の可能性があるため、焼却廃棄物（感染性廃棄物相当）として処理。[3][9]  
- **誤り選択肢**:  
  - 2番: 飛散薬液拭き取りガーゼは抗がん剤汚染のため焼却廃棄物、一般廃棄物不可。  
  - 4番: 清拭ガーゼも同様に汚染リスクあり、一般廃棄物不可。  
  - 5番: 使用済みシリンジは針を曲げず、収納または専用カッターで処理。[2][3]  
正解は公式解答および複数解説サイトで**1、3**と一致。[3][6][9]
