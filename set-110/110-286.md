# 110-286

### 第110回薬剤師国家試験 問286

#### 問題文
67歳女性。既往歴及び服薬歴はない。大腿骨近位部骨折のため、1ヶ月間入院加療することになった。入院時の大腿骨骨密度は、若年成人平均値（YAM）の65％であり、血清カルシウム値9.6 mg/dL、血清リン値3.5 mg/dLであった。退院時に以下の治療薬が処方された。  
（処方内容：リセドロン酸ナトリウム錠、エルデカルシトール錠など。詳細は省略）  
退院から約半年経過してようやく持参した薬剤を確認したところ、リセドロン酸ナトリウム錠は10錠、エルデカルシトール錠は50錠の残薬が認められた。[1][2][4]

この患者の病態及び治療に関する記述として、正しいのはどれか。**2つ選べ**。[1][2][4]

#### 選択肢
1. **原発性の骨粗しょう症と考えられる**。  
2. 発症に、カルシウム不足による骨基質の石灰化障害が関与している。  
3. **入院前は、骨吸収が骨形成を上回った状態と考えられる**。  
4. 処方薬はいずれも横臥状態での服用が可能である。  
5. エルデカルシトールは、リセドロン酸による低カルシウム血症を増強する。[1][2][4]

#### 正解番号
**1, 3**[1]

#### 解説
- **選択肢1**: 正しい。骨密度がYAMの65%と低下しており、大腿骨近位部骨折を起こした高齢女性で他の疾患歴がないため、原発性骨粗鬆症（一次性骨粗鬆症）と診断される。他の病気や原因に起因しない。[1][4]  
- **選択肢2**: 誤り。骨粗鬆症の発症は主に骨吸収が骨形成を上回ることで骨量が減少した結果であり、カルシウム不足による石灰化障害が主因ではない。血清カルシウム値・リン値も正常範囲。[1][2][4]  
- **選択肢3**: 正しい。骨粗鬆症では骨吸収（破骨細胞による）が骨形成（骨形成細胞による）を上回る状態が持続し、骨密度低下・骨折に至る。[1][2][4]  
- **選択肢4**: 誤り。リセドロン酸ナトリウム錠（ビスフォスフォネート系）は上臥位（座った状態）で服用し、横臥時は食道停留・潰瘍のリスクがあるため不可。[1]  
- **選択肢5**: 誤り。エルデカルシトールは活性型ビタミンD₃製剤で、腸管からのカルシウム吸収を促進し、リセドロン酸による低カルシウム血症を低減・予防する。高カルシウム血症に注意が必要。[1][2]  

以上より、正解は**1, 3**。[1][2]
