# 110-286

（病態・薬物治療） この患者の病態及び治療に関する記述として、正しいのはどれか。2つ選べ。
1. 原発性の骨粗しょう症と考えられる。
2. 発症に、カルシウム不足による骨基質の石灰化障害が関与している。
3. 入院前は、骨吸収が骨形成を上回った状態と考えられる。
4. 処方薬はいずれも横臥状態での服用が可能である。
5. エルデカルシトールは、リセドロン酸による低カルシウム血症を増強する。
