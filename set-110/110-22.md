# 110-22

**問題文**
非電離放射線のうち、眼の水晶体タンパク質の変性によってガラス工白内障を引き起こすのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-22 を参照

**選択肢**
1. UVA
2. UVB
3. UVC
4. 赤外線
5. 可視光線

**正解:** 4

**解説**
解答　４

眼の水晶体タンパク質の変性によってガラス工白内障を引き起こすのは、赤外線である。ガラス工白内障とは、ガラス職人など、強い赤外線を長時間浴び続ける職業の人に発生する白内障のことである。

１　誤：UVAは、皮膚に色素沈着（サンタン）を引き起こす。
２　誤：UVBは、皮膚に炎症（サンバーン）を引き起こす。
３　誤：UVCは、皮膚がんや白内障を引き起こす。
４　正：前記参照
５　誤：可視光線は、目の疲れやドライアイなどを引き起こす。
