# 110-159

**問題文**
抗血小板薬に関する記述として、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-159 を参照

**選択肢**
1. シロスタゾールは、ADP P2Y12受容体を遮断することで、血小板内サイクリックAMP（cAMP）濃度を増加させる。
2. オザグレルは、アデノシンA2受容体を遮断することで、血小板内Ca2＋濃度の上昇を抑制する。
3. ベラプロストは、プロスタノイドIP受容体を刺激することで、血小板内cAMP濃度を増加させる。
4. サルポグレラートは、セロトニン5−HT2受容体を遮断することで、血小板内Ca2＋濃度の上昇を抑制する。
5. プラスグレルは、ホスホジエステラーゼⅢ（PDEⅢ）を阻害することで、血小板内cAMP濃度を増加させる。

**正解:** 3,4

**解説**
解答　３、４

１　誤
シロスタゾールは、ホスホジエステラーゼⅢを阻害することで、血小板内サイクリックAMP（cAMP）濃度を増加させ、血小板凝集を抑制する。なお、ADP P2Y12受容体を遮断することで、血小板内cAMP濃度を増加させ血小板凝集を抑制する薬剤は、プラスグレルなどである。

２　誤
オザグレルは、トロンボキサンA2（TXA2）合成酵素を阻害することで、TXA2産生を抑制し、血小板凝集を抑制する。なお、アデノシンA2受容体を間接的に刺激することで、血小板内Ca2＋濃度の増加を抑制し血小板凝集を抑制する薬剤は、ジピリダモールである。

３　正
ベラプロストは、プロスタノイドIP受容体を刺激することで、血小板内cAMP濃度を増加させ、血小板凝集を抑制する。

４　正
サルポグレラートは、セロトニン5−HT2受容体を遮断することで、血小板内Ca2＋濃度の上昇を抑制し、血小板凝集を抑制する。

５　誤
