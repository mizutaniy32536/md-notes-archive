# 110-88

**問題文**
以下のうち、Common Terminology Criteria for Adverse Events（CTCAE）が用いられる指標として適切なのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-88 を参照

**選択肢**
1. 医薬品自主回収時の健康危険度
2. インシデント発生時の患者への影響度分類レベル
3. 医薬品・医療機器等安全性情報の緊急度
4. 医薬品による有害事象の重症度
5. 患者の日常生活における制限の程度

**正解:** 4

**解説**
解答　４

Common Terminology Criteria for Adverse Events（CTCAE）は、アメリカ国立がん研究所（NCI）が主導し世界共通で使用されることを意図して作成された有害事象に関しての共通用語基準であり、医薬品による有害事象の重症度をGrade1（軽症）〜Grade5（死亡）の5段階で評価する際に用いられる。
