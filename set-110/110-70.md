# 110-70

**問題文**
レジオネラ肺炎に対して最も有効性が期待できる抗菌薬はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-70 を参照

**選択肢**
1. シプロフロキサシン
2. タゾバクタム・ピペラシリン
3. メロペネム
4. アミカシン
5. バンコマイシン

**正解:** 1

**解説**
解答　１

レジオネラ肺炎の原因菌である、レジオネラ属菌は細胞内寄生細菌であるため、宿主細胞に浸透するニューキノロン系抗菌薬（シプロフロキサシンなど）やマクロライド系抗菌薬（アジスロマイシンなど）が有効である。
