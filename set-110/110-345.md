# 110-345

**問題文**
62歳女性。卵巣がんにて、パクリタキセル、カルボプラチン、ベバシズマブを用いた外来化学療法を施行している。ベバシズマブの重大な副作用である血栓塞栓症の早期発見のために、患者に伝えておくべき自覚症状として適切なのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-345 を参照

**選択肢**
1. 口渇
2. 息苦しさ
3. 黒色便
4. 胸の痛み
5. 味覚障害

**正解:** 2,4

**解説**
解答　２、４

ベバシズマブはヒト血管内皮増殖因子（VEGF）に対するモノクローナル抗体製剤で、腫瘍組織での血管新生を抑制することにより悪性腫瘍の増殖を阻害する薬剤であるが、正常な血管内皮も傷つきやすくなり血栓ができやすくなる。
血栓塞栓症の自覚症状は血栓が詰まった場所によって様々であるが、肺に血栓が詰まると息苦しさや胸の痛みなどの症状が現れるため、選択肢の中で血栓塞栓症の早期発見のために、患者に伝えておくべき自覚症状として適切なのは息苦しさと胸の痛みである。
