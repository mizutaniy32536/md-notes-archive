# 110-75

**問題文**
医薬品副作用被害救済制度の概要を示した下図において、健康被害者が給付を請求する先の A として正しいのはどれか。1つ選べ。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-75 を参照

**選択肢**
1. 主治医
2. 都道府県知事
3. 製造販売業者
4. 社会保険診療報酬支払基金
5. 独立行政法人医薬品医療機器総合機構（PMDA）

**正解:** 5

**解説**
解答　５

医薬品副作用被害救済制度において、健康被害者が給付を請求する先は独立行政法人医薬品医療機器総合機構（PMDA）（選択肢５）である。
