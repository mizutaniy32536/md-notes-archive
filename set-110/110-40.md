# 110-40

HER2（ヒト上皮増殖因子受容体2型）と結合することにより、乳がん細胞の増殖を抑制するのはどれか。１つ選べ。

１ エキセメスタン
２ トラスツズマブ
３ ベバシズマブ
４ ダウノルビシン
５ ブスルファン

**正解:** ２

**解説**
HER2（ヒト上皮増殖因子受容体2型）と結合することにより、乳がん細胞の増殖を抑制するのは、トラスツズマブである。トラスツズマブは、HER2に対するモノクローナル抗体製剤であり、抗体依存性細胞障害作用により、乳がんなどの治療に用いられる。
１　誤
エキセメスタンは、非可逆的にアロマターゼを阻害し、アンドロゲンからエストロゲンへの合成を阻害することにより、閉経後乳がんの治療に用いられる。
２　正
上記参照
３　誤
ベバシズマブは、ヒト血管内皮増殖因子（VEGF）に対するモノクローナル抗体であり、VEGFと結合し腫瘍組織での血管新生を抑制することにより、乳がんなどの治療に用いられる。
４　誤
ダウノルビシンは、腫瘍細胞のDNA鎖に入り込み（インターカレーション）、DNAポリメラーゼやRNAポリメラーゼなどを阻害することにより、急性白血病の治療に用いられる。
５　誤
ブスルファンは、腫瘍細胞のDNAをアルキル化することにより、慢性骨髄性白血病などの治療に用いられる。
