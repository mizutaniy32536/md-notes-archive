# 110-40

**第110回薬剤師国家試験 問40の問題文、全選択肢、正解番号、解説は以下の通りです。[1][3]**

**問題文:**  
HER2（ヒト上皮増殖因子受容体2型）と結合することにより、乳がん細胞の増殖を抑制するのはどれか。1つ選べ。[1][3]

**全選択肢:**  
1. エキセメスタン  
2. トラスツズマブ  
3. ベバシズマブ  
4. ダウノルビシン  
5. ブスルファン[1][3]

**正解番号:**  
2[1][10]

**解説:**  
- **1. エキセメスタン**: アロマターゼ阻害薬であり、HER2と結合しない。[1][5]  
- **2. トラスツズマブ**: HER2が標的の分子標的薬で、HER2と結合し乳がん細胞の増殖を抑制する。[1][5]  
- **3. ベバシズマブ**: VEGFが標的の分子標的薬で、血管新生を抑制するがHER2と結合しない。[1][5]  
- **4. ダウノルビシン**: アントラサイクリン系の抗がん剤でHER2と結合しない。[1]  
- **5. ブスルファン**: アルキル化剤でHER2と結合しない。[1]
