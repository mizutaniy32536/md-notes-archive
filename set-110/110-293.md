# 110-293

### 第110回薬剤師国家試験 問293

#### 問題文（共通ステム）
50歳男性。4ヶ月前に、僧帽弁閉鎖不全症に対して、自己の僧帽弁を温存する僧帽弁形成術が施行された。その後、外来で経過観察を行っていたが、継続する38℃台の発熱、手掌や足底に紅斑が認められ、精査目的で入院となった。

（入院時検査所見）  
白血球12,500/μL、CRP 7.5 mg/dL（基準値0.14 mg/dL以下）、Hb 15.6 g/dL、AST 25 IU/L、ALT 15 IU/L、BUN 16.3 mg/dL、血清クレアチニン 0.85 mg/dL、尿タンパク（－）、尿潜血（－）[2][4][5]

脳膿瘍や髄膜炎の合併症は認められなかった。主治医からの依頼があり、抗菌薬適正使用支援チームとして介入することとなった。[2][4][5]

#### 問293  
この患者の抗菌薬を用いた治療において、抗菌薬適正使用支援チームが主治医に助言する内容として最も適切なのはどれか。**2つ選べ**。[1][2][5]

1. 指定感染症であるため、速やかに保健所に届出をする。  
2. **血液培養により陰性化が確認された場合、速やかに抗菌薬治療を終了する。**  
3. **血液培養結果が陽性であっても、CRPが基準値内まで低下すれば抗菌薬治療を終了する。**  
4. （一部ソースで記載あり）血液培養結果が陽性であっても、CRPが基準値内まで低下すれば抗菌薬治療を終了する。（重複・変形）[1]  
（注: 完全な全選択肢リストはソース間で一部不完全だが、上記が主なもの。正解は2,3）[1][2][5][9]

#### 正解番号
**2、3**[1][2][4][5][9]

#### 解説
この症例は、僧帽弁形成術後の**感染性心内膜炎（IE）**疑い（発熱、紅斑、僧帽弁手術歴）。血液培養でメチシリン感受性黄色ブドウ球菌（MSSA）が検出され、抗菌薬治療中。[1][4][5]

- **選択肢1 誤**: MSSAによるIEは指定感染症ではないため、保健所届出不要。感染症法上、IE自体は2類感染症相当だが本ケースは指定外。[1][4]
- **選択肢2 正**: IE治療ガイドライン（日本循環器学会等）では、**血液培養陰性化確認後、治療終了**が標準。複数セット採取し、陰性化を治療効果判定の基準とする。[1][2]
- **選択肢3 正**: 培養陽性でも**CRP正常化**で終了可能。臨床指標（CRP低下）で治療継続を判断し、過剰使用回避。抗菌薬適正使用支援の観点で適切。[1][2]
- 全体: 問292正解は3,5（疣贅による心不全リスク、セファゾリン推奨）。治療はMSSAカバーでセファゾリン等、チーム助言はガイドラインベースのde-escalation/escalationを促す。[1][2][4][5][9]
