# 110-299

### 第110回薬剤師国家試験 問299

**問題文（全文）**  
検査所見と診察の結果から判断してロスバスタチンを変更することになった。変更後の治療薬として適切なのはどれか。**2つ選べ。**[1][2][3]

**全選択肢（番号付き）**  
1. 別のスタチン（例: アトルバスタチン）への変更  
2. **エゼチミブ**（スタチンの副作用が疑われる症例での併用が適切）[1]  
3. （詳細未記載だが、問298関連で誤りの例として類似）  
4. **エボロクマブ**（PCSK9阻害薬としてLDL受容体分解抑制に有効）[1]  
5. （詳細未記載だが、問298の5「筋肉痛と脱力感は、ロスバスタチンを継続投与しても自然に消失する。」が関連し不適切）[3][8]

**正解番号**  
**2、4**[1][2][8]

**解説**  
- 患者は家族性高コレステロール血症（FH）を疑う所見（LDL-C 262 mg/dL、皮膚結節性黄色腫、アキレス腱肥厚9.0 mm、父親の早発性心筋梗塞既往）があり、CK 4,215 IU/L上昇、AST 102 IU/L上昇からロスバスタチンによる横紋筋融解症（筋肉痛・脱力感）が疑われるため、スタチン継続や変更は不適切（選択肢1誤り）。[1][3][4]  
- **選択肢2（エゼチミブ）**：コレステロール吸収阻害薬としてスタチン不耐例に適し、単独または併用でLDL-C低下効果あり。妥当。[1]  
- **選択肢4（エボロクマブ）**：ヒト抗PCSK9モノクローナル抗体製剤。PCSK9（LDL受容体分解促進タンパク質）のLDL受容体結合を阻害し、肝細胞表面LDL受容体数を増加させてLDL-Cを低下。FHの高LDL-C血症に有効でスタチン不耐に適する。妥当。[1]  
- 問298正解は1、2（食事療法の炭水化物比50-60%、運動前のスクリーニング）が関連し、LDL目標はFHで100 mg/dL未満（心血管疾患既往なし）。[2][4][8]
