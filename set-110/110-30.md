# 110-30

**問題文**
オレキシン受容体を選択的に遮断することで、睡眠を誘導するのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-30 を参照

**選択肢**
1. ヒドロキシジン
2. リルマザホン
3. レンボレキサント
4. ゾピクロン
5. ペントバルビタール

**正解:** 3

**解説**
解答　３

１　誤
ヒドロキシジンは、ヒスタミンH1受容体を遮断することで、抗不安作用を示す。

２　誤
リルマザホンは、生体内で代謝されベンゾジアゼピン誘導体となり、GABAA受容体のベンゾジアゼピン結合部位に結合することで、睡眠を誘導する。

３　正
レンボレキサントは、オレキシン受容体を遮断することで、睡眠を誘導する。

４　誤
ゾピクロンは、GABAA受容体のベンゾジアゼピン結合部位に結合することで、睡眠を誘導する。

５　誤
ペントバルビタールは、GABAA受容体のバルビツール酸結合部位に結合することで、睡眠を誘導する。
