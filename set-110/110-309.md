# 110-309

### 第110回薬剤師国家試験 問309

#### 問題文
50歳女性。身長155cm、体重48kg。B型肝炎の既往あり。起床時の手指関節のこわばり、肘関節、膝関節の痛みを主訴として受診し、関節リウマチと診断され、処方1と処方2の薬剤で治療していた。その後、炎症がコントロールできず、受診した際に注射薬の追加が提案されたが、患者から「自分で注射するのは怖い。病院内で点滴してほしい。」との要望があったため、**処方3**が追加されることとなった。なお、処方3の薬剤は、製造販売業者から卸売販売業者を経て、納入されたものである。[1][2][3][5]

**問309（法規・制度・倫理）**  
処方3の関節リウマチ治療薬の規制に関する記述として、正しいのはどれか。**2つ選べ。**[3][5]

#### 選択肢
1. この薬剤を使用した医療機関の開設者の氏名及び住所は、卸売販売業者から製造販売業者に報告される。[1][3]  
2. 直接の容器・被包には、白地に赤枠、赤字で「生物」の文字が記載されている。[1][3][5]  
3. 取り扱う医療関係者は、使用対象者の氏名、住所等の記録を保存しなければならない。[5]  
4. （検索結果に明記なしのため省略。解説参照）[1]  
5. 製造に用いた生物の名称が記載されています。[1]

#### 正解番号
**1、5**[1][2][5][6][8]

#### 解説
- **選択肢1 正しい**：処方3の薬剤（アバタセプト（遺伝子組換え）点滴静注用）は生物由来製品（特定生物由来製品）であり、生物由来製品の販売業者は、販売した薬局開設者や病院・診療所の開設者の氏名・住所を製造販売業者に報告する。[1][5][9]  
- **選択肢2 誤り**：直接の容器・被包には**白地に黒枠、黒字**で「生物」の文字が記載される。「白地に赤枠、赤字」は誤り（赤枠・赤字は毒薬や劇薬の場合）。[1][4][5]  
- **選択肢3 誤り**：詳細は検索結果に明記なしだが、解説で正解外。[5]  
- **選択肢4 誤り**：15日以内に厚生労働大臣に報告しなければならないとする記述は不適切。新設予定の30日以内の報告義務を含めても誤り。[1]  
- **選択肢5 正しい**：生物由来製品の容器・被包に製造に用いた生物の名称が記載される。[1]  

処方3の薬剤はアバタセプト（遺伝子組換え）で、生物由来製品の法規（医薬品医療機器等法）が問われている。[1][5][9]
