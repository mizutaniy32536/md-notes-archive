# 110-139

**第110回薬剤師国家試験 問139**

環境汚染（大気汚染、水質汚濁、土壌汚染等）を防止するための法規制に関する記述として、正しいのはどれか。２つ選べ。[1][2][7][8]

1. 大気汚染防止法では、自動車などの移動発生源から排出されるばい煙や揮発性有機化合物及び粉じんについて排出基準が定められている。[2][7][8]  
2. 水質汚濁防止法では、健康に係る有害物質及び生活環境に係る汚染状態について排水基準が定められている。[2][7]  
3. 都道府県は、国が定めた一般排出基準あるいは一律排水基準よりも厳しい「上乗せ排出基準」あるいは「上乗せ排水基準」を定めることができる。[2]  
4. 土壌汚染対策法では、すべての特定有害物質について「土壌含有基準」が定められている。[2]  
5. ダイオキシン類対策特別措置法では、大気、水質、土壌及び農作物を対象に環境基準が定められている。[2]

**正解番号：2, 3** [1][4]

**解説**  
- 選択肢1：大気汚染防止法では、ばい煙等について**発生施設ごと**（工場など）に国が定める一般排出基準が設定されており、自動車などの移動発生源からの排出については基準が設定されていないため誤り。[1]  
- 選択肢2：水質汚濁防止法では、健康に係る有害物質及び生活環境に係る汚染状態について排水基準が定められているため正しい。[1][2]  
- 選択肢3：都道府県は、国が定めた一般排出基準あるいは一律排水基準よりも厳しい「上乗せ排出基準」あるいは「上乗せ排水基準」を条例で定めることができるため正しい。[1][2]  
- 選択肢4：詳細な誤りの記述は検索結果にないが、正解が2,3であることから誤り。[1][4]  
- 選択肢5：ダイオキシン類対策特別措置法では、農作物には環境基準が定められていない（農作物による人体への取り込み量が少なく、健康影響が問題ないため）ため誤り。[1][2]
