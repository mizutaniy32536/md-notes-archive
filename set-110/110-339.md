# 110-339

**問題文**
院内において、病棟で使用するネブライザーの消毒用の次亜塩素酸ナトリウムの調製を薬剤部で行い、払い出すこととなった。次亜塩素酸ナトリウム濃度6％の消毒薬を購入し、0.01％（100 ppm）に調製して1 Lのボトルで払い出すこととした。このとき、原液 　ア　 に水を加えた全量を 　イ　 とし、この液 　ウ　 を採取し、これに水を加えて全量を1 Lとした。容量の組み合わせとして正しいのはどれか。1つ選べ。

**解説**
解答　４

希釈すると希釈倍率に応じて濃度が低下する。例えば、ある溶液の体積をn倍に希釈した場合、その濃度は1/n倍になる。
本問では、濃度6％の次亜塩素酸ナトリウムを0.01％に調整するためには、600倍の希釈を行わなければならない。

１　誤
原液50 mLに水を加え全量を1 Lとすることで、20倍に希釈されたことになる。この液100 mLを採取し、これに水を加えて全量を1 Lとすることで、さらに10倍希釈されたことになる。よって、最終的に次亜塩素酸ナトリウムは200倍に希釈されている。

２　誤
原液30 mLに水を加え全量を300 mLとすることで、10倍に希釈されたことになる。この液50 mLを採取し、これに水を加えて全量を1 Lとすることで、さらに20倍希釈されたことになる。よって、最終的に次亜塩素酸ナトリウムは200倍に希釈されている。

３　誤
原液10 mLに水を加え全量を200 mLとすることで、20倍に希釈されたことになる。この液10 mLを採取し、これに水を加えて全量を1 Lとすることで、さらに100倍希釈されたことになる。よって、最終的に次亜塩素酸ナトリウムは2000倍に希釈されている。

４　正
原液5 mLに水を加え全量を300 mLとすることで、60倍に希釈されたことになる。この液100 mLを採取し、これに水を加えて全量を1 Lとすることで、さらに10倍希釈されたことになる。よって、最終的に次亜塩素酸ナトリウムは600倍に希釈されている。

５　誤
原液5 mLに水を加え全量を1.5 Lとすることで、300倍に希釈されたことになる。この液50 mLを採取し、これに水を加えて全量を1 Lとすることで、さらに20倍希釈されたことになる。よって、最終的に次亜塩素酸ナトリウムは6000倍に希釈されている。
