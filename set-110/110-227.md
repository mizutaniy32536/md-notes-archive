# 110-227

# 第110回薬剤師国家試験 問227

## 問題文

34歳女性。この女性の12歳の娘に市からヒトパピローマウイルス（HPV）ワクチン接種についてのお知らせが届いた。娘はHPVワクチンの接種歴がない。この女性はHPVワクチンの説明を聞くため、市から送られた成分表を含む資料を持って娘と一緒にかかりつけ薬局に来局した。

**問227（物理・化学・生物）**

このワクチンに含まれる添加物に関する記述として、誤っているのはどれか。１つ選べ。[3]

## 全選択肢

1. 添加物Aは、細菌由来成分の誘導体である。

2. 添加物Aと添加物Bは、ともにアジュバントである。

3. 添加物Bは、抗原を吸着し、投与部位での抗原の滞留時間を延ばす。

4. 添加物Aは、抗原提示細胞の抗原提示機能を促進する。

5. 添加物Aと添加物Bは、ともに自然免疫応答を抑制し、獲得免疫応答を促進する。

## 正解番号

**5**[2][9]

## 解説

選択肢1は正しい。添加物Aは、グラム陰性菌由来成分の誘導体である。[2]

選択肢2は正しい。アジュバントは、抗原と一緒に投与することで、免疫系の反応を高める物質であり、添加物Aと添加物Bはともにアジュバントである。[2]

選択肢3は正しい。添加物Bは、抗原を吸着し、投与部位での抗原の滞留時間を延ばす。[3]

選択肢4は正しい。添加物Aは、抗原提示細胞の抗原提示機能を促進する。[2]

選択肢5は**誤り**である。各アジュバントの作用機序については不確定なことも多く、添加物Aと添加物Bがともに自然免疫応答を抑制するという記述は不適切である。[5][10]特に、選択肢3に関しては、アルミニウム塩が接種された抗原を局所部位に長時間留めておく効果を否定する研究報告があることも指摘されている。[10]
