# 110-115

**問題文**
コラーゲンは、下図に示すように、ポリペプチド鎖（コラーゲン単量体）の3本が、らせん状に絡み合って三重らせん構造（コラーゲン三量体）を形成する。図上段に示すように、単量体ポリペプチド鎖のアミノ酸配列では、3個おきにグリシン残基（Gly）が配置する。Xはプロリン（Pro）の翻訳後修飾で生じたアミノ酸である。このコラーゲンに関する記述として、正しいのはどれか。2つ選べ。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-115 を参照

**選択肢**
1. コラーゲンの三重らせん構造は、α−ヘリックスである。
2. Xは、ヒドロキシ化されたプロリンである。
3. グリシン残基が繰り返し存在することが、三重らせん構造の形成に重要である。
4. ビタミンCは、コラーゲン産生細胞の核内に入り、コラーゲン遺伝子の転写を促進する。
5. コラーゲン三量体は、細胞内の細胞骨格を構成することにより組織の強度を保つ。

**正解:** 2,3

**解説**
解答　２、３

１　誤
α−ヘリックスとは、1本のポリペプチド鎖が分子内水素結合することで形成される細長いらせん構造を指す。コラーゲンの三重らせん構造は、3本のポリペプチド鎖が形成している特殊ならせん構造であり、α−ヘリックスとは異なる。

２　正
Xはプロリン（Pro）の翻訳後修飾で生じたアミノ酸であることから、ヒドロキシプロリンであると考えられる。

３　正
グリシン（Gly）は側鎖が水素原子であるため、タンパク質を構成する他のアミノ酸と比べて、空間を占める割合が小さい。コラーゲンの三重らせん構造はグリシン、プロリン、ヒドロキシプロリンが多く含まれ、グリシン以外は比較的大きな側鎖を有している。コラーゲンの三重らせん構造は比較的大きな側鎖をもつアミノ酸が多い中、グリシンは側鎖が小さく、混み合った中でも安定して収まることができるため、三重らせん構造の形成に重要な役割を担っている。

４　誤
ビタミンCは、コラーゲン遺伝子の転写を促進させることでコラーゲンの発現を誘導する。しかし、コラーゲン遺伝子の転写を促進させる際にビタミンCが核内に移行するかどうかは、現在知られておらず不明である。

５　誤
コラーゲンは細胞外マトリックスの代表的な構成タンパク質であり、組織の強度を保つ役割を担うが、細胞内の細胞骨格を構成するわけではない。
