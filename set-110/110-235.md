# 110-235

### 第110回薬剤師国家試験 問235

**問題文（実務）**  
この女性は生活習慣に改善がみられず、再び健康フェアに参加した際に受診勧奨され、近医を受診した。血圧は160/95 mmHg、空腹時血糖値は145 mg/dL、HbA1c値は6.8%であった。患者は運動療法・食事療法を実施していたが、改善しなかったため、**処方1**と**処方2**が出され来局した。  
（注：処方1はエナラプリルを含む高血圧治療薬、処方2はメトホルミン含有の糖尿病治療薬と推定される。詳細処方は検索結果に明記されていないが、文脈から血管拡張薬とビグアナイド系薬の副作用指導に関する問題。）[1][2][3]

**全選択肢**  
1. まぶたや唇の腫れなどを認めた場合には教えてください。  
2. 悪心、下痢などを認めた場合には教えてください。  
3. 空咳などを認めた場合には教えてください。  
4. 副作用予防のため適度に水分を摂取してください。  
5. バリウム造影剤を用いる検査は受けないでください。[1]

**正解番号**  
5[1][5][6]

**解説**  
選択肢1,3はエナラプリル（ACE阻害薬）の主な副作用である**血管浮腫**（まぶたや唇の腫れ）や**空咳**についての指導であり、本問の正解ではない。[1]  
選択肢2,4はメトホルミン（ビグアナイド系薬）の副作用（**悪心・下痢**）への対応や予防（**水分摂取**）として適切だが、本問の正解ではない。[1]  
正解の**選択肢5**はメトホルミンの使用で乳酸アシドーシスリスクを考慮し、**バリウム造影剤検査（ヨウ化造影剤使用時）**を避ける指導が適切である。ヨウ化造影剤は腎機能低下を招き、メトホルミンと併用で重篤な副作用を誘発する可能性があるため、検査前後にメトホルミンの中止を指導する標準的な内容。[1][2][3]
