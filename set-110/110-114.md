# 110-114

1. ミトコンドリアで行われる。
2. アシル鎖の伸長過程で、補酵素として NADH が利用される。
3. クエン酸により促進される。
4. アセチル CoA を前駆物質としてリノール酸が合成される。
5. アセチル CoA をマロニル CoA に転換する反応が律速段階である。
