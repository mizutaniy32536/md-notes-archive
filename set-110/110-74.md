# 110-74

毒物劇物営業者が行う劇物の容器及び被包への表示について、「医薬用外」の文字に加えて表示する内容として正しいのはどれか。１つ選べ。

１ 白地に赤色をもって「劇」の文字
２ 白地に黒色をもって「劇」の文字
３ 白地に赤色をもって「劇物」の文字
４ 赤地に白色をもって「劇物」の文字
５ 白地に黒色をもって「劇物」の文字

**正解:** ３

**解説**
