# 110-28

**問題文**
ピロカルピンが毛様体筋を収縮させる機序はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-28 を参照

**選択肢**
1. アドレナリンα1受容体遮断
2. アドレナリンα2受容体刺激
3. アドレナリンβ2受容体遮断
4. アセチルコリンM2受容体遮断
5. アセチルコリンM3受容体刺激

**正解:** 5

**解説**
解答　５

ピロカルピンは、アセチルコリンM3受容体を刺激することで、毛様体筋を収縮させる。
