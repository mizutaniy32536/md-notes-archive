# 110-6

以下のアルドヘキソースAのアノマーはどれか。１つ選べ。

**正解:** ４

**解説**
鎖状の単糖が閉環する際、カルボニルであった炭素に新しいキラル中心が生じる。この新たに生じた不斉炭素をアノマー炭素と呼ぶ。また、アノマー炭素の置換基の立体化学のみが異なるジアステレオマーをアノマーと呼ぶ。
アルドヘキソースA（β−D−グルコース）の場合、1位の炭素がアノマー炭素となるため、Aと比較して1位のみの立体配置が異なる選択肢４（α−D−グルコース）がアノマーの関係にある化合物である。
