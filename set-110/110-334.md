# 110-334

**問題文**
68歳女性。1週間前より右下肢に腫脹が見られるようになり、かかりつけ医から大学病院を紹介され、受診した。下肢静脈超音波検査による血栓の確認及びD-ダイマーの上昇により、深部静脈血栓症と診断され、即日入院となった。薬剤師は、患者が携帯していたお薬手帳より、現在の使用中の薬剤情報を入手した。また、入院後も服用継続の予定である。（臨床検査値）D-ダイマー8.9 µg/mL（基準値：1.0 µg/mL未満）、eGFR 72 mL/min/1.73 m2この患者に適切な深部静脈血栓症の治療薬はどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-334 を参照

**選択肢**
1. シロスタゾール
2. クロピドグレル
3. ワルファリンカリウム
4. アピキサバン
5. ダルテパリン

**正解:** 4,5

**解説**
問334　解答　４、５

深部静脈血栓症の治療には、抗凝固薬であるワルファリンカリウムやアピキサバン、ダルテパリンなどを用いる。なお、この患者はワルファリンカリウムと併用禁忌のミコナゾールを継続して服用予定のため、この患者に適切な深部静脈血栓症の治療薬は、アピキサバンとダルテパリンである。
なお、ミコナゾールはアゾール系抗真菌薬であり、ワルファリンカリウムの代謝酵素であるCYP2C9を阻害することで、ワルファリンカリウムの作用を増強し、重篤な出血などが現れるおそれがあるため併用禁忌となっている。
