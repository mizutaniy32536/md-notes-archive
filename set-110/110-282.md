# 110-282

（薬剤） このワクチン製剤に関する記述として、正しいのはどれか。2つ選べ。
1. 専用溶解用液中の脂質粒子は、内水層を有する。
2. ポリソルベート ₈₀ は、微生物の発育を阻止する目的で添加されている。
3. 精製白糖は、無痛化を目的に添加されている。
4. 凍結を避けて、 ₂ ～ ₈ ℃で保存する。
5. 血液中に移行後、製剤が徐々に分解されて有効成分を放出する。
