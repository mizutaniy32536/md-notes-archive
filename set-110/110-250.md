# 110-250

（薬理） 新たに開始する候補薬剤の作用機序として、適切なのはどれか。2つ選べ。
1. IL︲₁₂ 及び IL︲₂₃ の p₄₀ サブユニットに特異的に結合して、ヘルパー T 細胞の 活性化を抑制する。
2. TNF︲a に特異的に結合して、TNF︲a の作用を抑制する。
3. カルシニューリンを阻害して、IL︲₂ の産生を抑制する。
4. プリン塩基の合成を阻害して、リンパ球の増殖を抑制する。
5. ヤヌスキナーゼ（JAK）を阻害して、リンパ球の活性化を抑制する。
