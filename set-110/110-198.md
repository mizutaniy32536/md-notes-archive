# 110-198

（実務） 変更後の処方 ₂ に関して薬剤師が患者に行う説明として、適切なのはどれか。 2つ選べ。
1. 処方 ₂ の薬剤は、処方 ₁ の薬剤と同じように服用してください。
2. 処方 ₂ の薬剤は、処方 ₁ の薬剤と効果が同等です。
3. 処方 ₂ の薬剤は、処方 ₁ の薬剤より副作用が軽くなります。
4. 処方 ₁ の薬剤の供給が安定すれば、医師の同意がなくても処方 ₁ に戻せます。
5. 飲合せに注意する薬が処方 ₁ の薬剤と異なりますので、今後別の薬を服用する 場合は、相談してください。
