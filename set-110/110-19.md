# 110-19

**問題文**
以下の構造式で示す腐敗アミンの前駆体となるアミノ酸はどれか。1つ選べ。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-19 を参照

**選択肢**
1. リシン
2. チロシン
3. ヒスチジン
4. アルギニン
5. トリプトファン

**正解:** 3

**解説**
解答　３

本問の構造はヒスタミンである。ヒスタミンは、ヒスチジンの脱炭酸反応によって生じる腐敗アミンである。

１　誤
リシンから生じる腐敗アミンは、カダベリンである。

２　誤
チロシンから生じる腐敗アミンは、チラミンである。

３　正
前記参照

４　誤
アルギニンから生じる腐敗アミンは、アグマチンである。

５　誤
トリプトファンから生じる腐敗アミンは、トリプタミンである。
