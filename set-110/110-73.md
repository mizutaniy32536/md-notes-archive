# 110-73

**問題文**
医薬品の製造販売承認申請のための一般毒性試験を実施する際に、遵守しなければならないのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-73 を参照

**選択肢**
1. GCP
2. GLP
3. GMP
4. GQP
5. GVP

**正解:** 2

**解説**
解答　２

医薬品の製造販売承認申請のための一般毒性試験を実施する際に遵守しなければならないのは、GLP（選択肢２）である。
なお、GLPは、医薬品の安全性に関する非臨床試験の実施の基準であり、単回投与毒性試験や反復投与毒性試験などの一般毒性試験に加えて、生殖発生毒性試験やがん原性試験などの特殊毒性試験や安全性薬理試験も遵守対象となる。
