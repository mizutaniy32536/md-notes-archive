# 110-2

**問題文**
熱力学における「閉じた系」において、熱、物質及び仕事のうち、外界とやりとりができるものだけをすべて挙げたのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-2 を参照

**選択肢**
1. 熱
2. 物質
3. 仕事
4. 熱、仕事
5. 物質、仕事
6. 熱、物質

**正解:** 4

**解説**
解答　４

熱力学では、系と外界の境界において、「物質」及びエネルギーである「熱」と「仕事」のやりとりの可否によって系を分類する。
閉じた系（閉鎖系）とは、物質のやりとりはできないが、熱と仕事をやりとりできる系である。
