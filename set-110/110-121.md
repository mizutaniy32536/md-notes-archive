# 110-121

# 第110回薬剤師国家試験 問121

## 問題文

ナイアシンから合成される下図の補酵素について、正しいのはどれか。**2つ選べ。**[3][4][5]

## 全選択肢

1. 酸化型の構造を示している。[3][4][5]

2. 340 nmの吸収極大を有する。[3][4][5]

3. 3−ヒドロキシ−3−メチルグルタリルCoA（HMG−CoA）からメバロン酸を産生する反応に関与する。[3][4][5]

4. ピルビン酸からアセチルCoAを産生する反応に関与する。[3][4][5]

5. ミトコンドリア内膜上での電子の受け渡しに関与する。[3][4][5]

## 正解番号

**2、3**[3][4][8][10]

## 解説

**選択肢1（誤り）**
設問の構造はNADPHであり、電子を受け取った後の状態（**還元型**）である。なお、NADPHから生成するNADP⁺は電子を受け取る前の状態（酸化型）である。[3]

**選択肢2（正解）**
NADPH（還元型）は、**340 nmに吸収極大を有する**。[3]

**選択肢3（正解）**
3−ヒドロキシ−3−メチルグルタリルCoA（HMG−CoA）からメバロン酸を産生する反応には、HMG−CoA還元酵素が関与し、その補酵素は**NADPH**である。[3]

**選択肢4（誤り）**
ピルビン酸からアセチルCoAを産生する反応に関与するのは、NADPHではなく**NAD⁺**である。この反応にはピルビン酸脱水素酵素複合体が関与する。[1]

**選択肢5（誤り）**
ミトコンドリア内膜上での電子の受け渡しに関与するのはNADPHではなく、**NAD⁺/NADH**である。[1]
