# 110-37

**問題文**
高尿酸血症治療薬トピロキソスタットが阻害するのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-37 を参照

**選択肢**
1. シクロオキシゲナーゼ
2. キサンチンオキシダーゼ
3. 尿酸オキシダーゼ
4. 尿酸トランスポーター
5. 有機アニオントランスポーター

**正解:** 2

**解説**
解答　２

トピロキソスタットは、尿酸合成に関わるキサンチンオキシダーゼを阻害することで、高尿酸血症を改善する。
