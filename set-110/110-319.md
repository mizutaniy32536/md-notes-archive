# 110-319

### 第110回薬剤師国家試験 問319

**問題文（共通ステムからの続き）**  
79歳男性。難治性の気管支ぜん息と診断され、吸入ステロイド薬及び長時間作用性気管支拡張薬による吸入療法を行っていた。しかし、効果不十分のため3ヶ月前よりメポリズマブ（遺伝子組換え）皮下注が追加され、院内処方にて病院で使用法の指導・確認を行っていた。なお、この男性は、同一世帯に75歳の妻がおり、両者とも要支援・要介護認定は受けていない。この夫婦は年金受給者でその他の収入はなく、窓口負担割合が1割である。[1]

**問319（法規・制度・倫理）**  
この患者の**医療費の自己負担に関する制度**について、薬剤師として患者に説明する際に適切なものはどれか。**2つ**選べ。[1][4]

- 1. 多数回重複受療時の自己負担限度額適用認定証を提示することで月間の自己負担限度額が軽減される。
- 2. 高額療養費制度の自己負担限度額を超えた場合、翌月の自己負担限度額が軽減される。
- 3. 同一世帯内に70歳以上の高齢者が複数いる場合、世帯単位で高額療養費制度の限度額適用認定証が発行される。
- **4. 年収約370万円～約770万円の世帯主がいる場合、区分アとみなされる。**[6][7]
- **5. 低所得者Ⅱの場合、月額外来80,100円＋αを超えた場合に限度額適用認定証が使用できる。**[6][7]

**正解番号：4、5** [6][7]

**解説**  
この問題は医療費の自己負担に関する制度、特に**高額療養費制度**の詳細を問う。患者は79歳（70歳以上1割負担）、妻75歳（同）、年金受給者で低所得の可能性が高い。[1][4]  
- 4は正しい：**区分ア**は被保険者または世帯主の年収が約370万円～約770万円の場合。[6][7]  
- 5は正しい：**低所得者Ⅱ**（住民税非課税世帯等）は、月額外来**80,100円＋α**（αは1割負担の1,000円×受療回数）を超過で限度額適用認定証使用可能。[6][7]  
不正解肢：1は多数回重複受療時限度額適用認定証の誤り（高額療養費とは別）。2は翌月軽減の誤り（現月計算）。3は世帯主が70歳未満なら単独計算。[1][4][6]  
なお、問題文が「医療費の自己負担に関する制度」と曖昧で高額療養費を断定しにくいとの指摘もあるが、正解は4・5で確定。[4]
