# 110-179

薬物の溶解性を高める製剤的手法に関する記述として、正しいのはどれか。２つ選べ。

１ 微粉化により、比表面積を小さくする。
２ シクロデキストリンにより、ミセルを形成させる。
３ 水と混和する有機溶媒を添加して、溶媒の極性を変化させる。
４ 結晶性薬物の場合、非晶質化して熱力学的に安定化させる。
５ テオフィリンの場合、エチレンジアミンと複合体を形成させる。

**正解:** ３、５

**解説**
１　誤
微粉化により粒子径は小さくなり、比表面積を大きくすることで、薬物の溶解速度を高めることができる。
２　誤
シクロデキストリンによる包接化により、難溶性薬物の溶解性を高めることができる。なお、界面活性剤であるポリオキシエチレン硬化ヒマシ油やポリソルベート80により、ミセルを形成することでも、難溶性薬物の溶解性を高めることができる。
３　正
薬物が単独の水に溶けにくい場合、エタノールなどの有機溶媒（コソルベント）を添加することで、難溶性薬物の溶解性を高めることができる。この現象をコソルベンシーという。
４　誤
結晶性薬物の場合、非晶質化して熱力学的に不安定化することにより、薬物の溶解性を高めることができる。
５　正
テオフィリンの場合、エチレンジアミンと可溶性の複合体を形成することにより、溶解性を高めることができる。
