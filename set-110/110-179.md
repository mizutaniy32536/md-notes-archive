# 110-179

薬物の溶解性を高める製剤的手法に関する記述として、正しいのはどれか。 2つ選べ。
1. 微粉化により、比表面積を小さくする。
2. シクロデキストリンにより、ミセルを形成させる。
3. 水と混和する有機溶媒を添加して、溶媒の極性を変化させる。
4. 結晶性薬物の場合、非晶質化して熱力学的に安定化させる。
5. テオフィリンの場合、エチレンジアミンと複合体を形成させる。
