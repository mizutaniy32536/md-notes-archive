# 110-239

### 第110回薬剤師国家試験 問239（実務）

**問題文**  
NDMAが検出されたメトホルミン塩酸塩を服用したことによるリスクは極めて低く、これまでに国内外において重篤な健康被害が発生したとの報告はない。この患者に対する薬剤師の説明内容として、適切なのはどれか。**2つ**選べ。[1][2]

**選択肢**  
1. NDMAによるがんの発症には、閾値があること。  
2. 摂取したNDMAの量は、実質安全量を下回っていること。  
3. 他の同種同効薬に変更する必要があること。  
4. 血糖値が落ち着いていれば服用を止め、次回受診時に医師に報告すること。  
5. 必要に応じて、発がんリスクに関する情報を医師に提供できること。[1][2]

**正解番号**  
2、5[2][4][8]

**解説**  
この問題は、問238の続きで、メトホルミン塩酸塩に検出されたNDMA（N-ニトロソジメチルアミン）の低リスクを患者に適切に説明する内容を問う。リスクは極めて低く、重篤な健康被害の報告はないため、不安を煽らず事実に基づく説明が適切。[1][2]  
- **2**：問238の計算で、患者のNDMA摂取量（1日0.096μg、10年で約35μg）は実質安全量（例: 96ng/日相当）を下回るため正しい。[1]  
- **5**：薬剤師は医師への情報提供が可能であり、患者にその役割を伝えるのは適切。[2][4]  
- **不正解理由**：1はNDMAの発がん性に閾値がない非閾値発がん物質のため誤り。3は低リスクで変更不要。4は糖尿病治療継続が必要で中止は不適切。[1][2]
