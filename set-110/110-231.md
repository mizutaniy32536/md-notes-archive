# 110-231

### 第110回薬剤師国家試験 問231

**問題文**  
薬剤師はこの親子に、皮膚の露出を少なくするために長袖・長ズボンなどの衣服を着用することに加えて、虫よけ剤の使用を勧めた。虫よけ剤の活性成分として適切なのはどれか。**2つ選べ。**[1][3][4][5]

**選択肢**  
1. イミダクロプリド  
2. **ディート**  
3. **イカリジン**  
4. ジクロルボス  
5. ジフェンヒドラミン[3][4][5]

**正解番号**  
2、3[1][3][4][5][7][10]

**解説**  
マダニに対する忌避剤（虫よけ剤）として、**ディート**と**イカリジン**の2種類の有効成分を含む製品が市販されており、SFTS予防に適する。[1][3]  
- 1. **誤**：イミダクロプリドはネオニコチノイド系の殺虫剤で、虫よけ剤ではない。[3]  
- 4. **誤**：ジクロルボスは有機リン系の殺虫剤で、忌避効果はない。[1][3]  
- 5. **誤**：ジフェンヒドラミンは抗ヒスタミン薬で、虫よけ剤ではない。[1][3]  
SFTSはマダニ媒介のウイルス感染症であり、虫よけ剤の使用が感染予防に有効。[3][4]
