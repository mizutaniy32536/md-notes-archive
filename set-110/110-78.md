# 110-78

**問題文**
世界医師会総会にて採択された、ヒポクラテスの誓いを現代化したものであるジュネーブ宣言の根幹となるのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-78 を参照

**選択肢**
1. 患者の権利
2. 生命倫理の4原則
3. 医師の遵守すべき倫理
4. 非人道的な人体実験の実施条件
5. 人を対象とした医学研究の倫理的原則

**正解:** 3

**解説**
解答　３

１　誤
患者の権利に関する宣言として、リスボン宣言があげられる。

２　誤
生命倫理の4原則とは、トム・L・ビーチャムとジェイムズ・F・チルドレスが提唱した原則であり、自律尊重原則、無危害原則、善行原則、正義原則から成り立つ。

３　正
医師の遵守すべき倫理に関する宣言として、ジュネーブ宣言があげられる。ジュネーブ宣言は、古代より続く伝統的な医師の倫理などについてのギリシャ神への宣誓文であるヒポクラテスの誓いを現代の状況に合わせたものである。

４　誤
非人道（非倫理）的な人体実験の実施条件について記載された倫理原則には、ニュルンベルク綱領などがあげられる。

５　誤
人を対象とした医学研究の倫理的原則に関する宣言として、ニュルンベルク綱領を踏襲したヘルシンキ宣言があげられる。
