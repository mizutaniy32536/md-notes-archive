# 110-160

**問題文**
呼吸器系に作用する薬物に関する記述として、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-160 を参照

**選択肢**
1. オキシメテバノールは、オピオイド受容体を刺激して、鎮咳作用を示す。
2. L−カルボシステインは、構造中にSH基を有し、ムコタンパク質のペプチド鎖の連結を切断して、去痰作用を示す。
3. フルマゼニルは、末梢性化学受容器を刺激して、間接的に呼吸中枢を興奮させる。
4. ブロムヘキシンは、アンブロキソールの活性代謝物であり、肺サーファクタント分泌を促進する。
5. ニンテダニブは、血管内皮増殖因子受容体（VEGFR）、線維芽細胞増殖因子受容体（FGFR）及び血小板由来増殖因子受容体（PDGFR）のチロシンキナーゼを阻害して、肺の線維化を抑制する。

**正解:** 1,5

**解説**
解答　１、５

１　正
オキシメテバノールは、麻薬性鎮咳薬であり、オピオイド受容体を刺激して、鎮咳作用を示す。

２　誤
L−カルボシステインは、構造中にSH基を有さず、ムコタンパク質の構成成分であるシアル酸、フコースのバランスを改善して、去痰作用を示す。なお、構造中にSH基を有し、ムコタンパク質のペプチド鎖の連結を切断して、去痰作用を示す薬剤は、アセチルシステインなどである。

３　誤
フルマゼニルは、ベンゾジアゼピン受容体に結合して、ベンゾジアゼピン系薬の作用に拮抗することで、呼吸抑制を改善する。なお、末梢性化学受容器を刺激して、間接的に延髄呼吸中枢を興奮させる薬剤は、ドキサプラムである。

４　誤
アンブロキソールは、ブロムヘキシンの活性代謝物であり、肺サーファクタント分泌の促進、線毛運動の亢進により、去痰作用を示す。

５　正
ニンテダニブは、低分子チロシンキナーゼ阻害薬であり、血管内皮増殖因子受容体（VEGFR）、線維芽細胞増殖因子受容体（FGFR）及び血小板由来増殖因子受容体（PDGFR）のチロシンキナーゼを阻害して、肺の線維化を抑制する。
