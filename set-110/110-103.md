# 110-103

**第110回薬剤師国家試験 問103**

**問題文**  
以下の反応で、生成物を与える電子移動を示す一連の矢印が正しいのはどれか。2 つ選べ。[1][3][5][6]

**選択肢**  
1. NaBH₄によるヒドリド還元反応の矢印（ヒドリドイオン (H⁻) が C=O の炭素に攻撃しない誤った示し方）[1][5]  
2. α,β 不飽和カルボニルへの共役付加反応の矢印（窒素原子上の孤立電子対が β 炭素に攻撃しない誤った示し方）[1][5]  
3. SN₂反応の正しい電子移動矢印[1]  
4. 求電子置換反応の正しい電子移動矢印[1]  
5. 電子移動の矢印の向きや始点が誤ったもの（矢印の先が正電荷でない、始点が炭素原子からではなく結合線からでない誤り）[1]

**正解番号**  
3, 4[1][3][5][6][8]

**解説**  
- 選択肢1: 水素化ホウ素ナトリウム (NaBH₄) によるヒドリド還元なので、ヒドリドイオン (H⁻) が C=O の炭素に攻撃するが、矢印が誤り。[1][5]  
- 選択肢2: α,β 不飽和カルボニルへの共役付加反応なので、窒素原子上の孤立電子対が β 炭素に攻撃するが、矢印が誤り。[1][5]  
- 選択肢3: SN₂反応として電子移動の矢印が妥当。[1]  
- 選択肢4: 求電子置換反応として電子移動の矢印が妥当。[1]  
- 選択肢5: 電子移動の矢印では、先が正電荷でなければ向きがおかしく、始点は炭素原子からではなく結合線部分からが妥当。[1]  
以上より、正解は3,4。[1][3][5][6][8]
