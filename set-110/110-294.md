# 110-294

### 第110回薬剤師国家試験 問294

#### 問題文
29歳女性。既婚。3年前に夫をドナーとした生体腎移植（ABO血液型適合）を受け、拒絶反応を抑制するため免疫抑制薬を以下の処方で服用しながら、生活をしている。腎移植後、体調がよくなり生活も安定してきたので、子供を欲しいと思うようになった。

（処方）  
シクロスポリンカプセル25mg  
・1回3カプセル（1日6カプセル）  
・1日2回 朝夕食後 28日分[2][5][6]

**問294（病態・薬物治療）**  
患者の希望を考慮すると、今後の免疫抑制薬として選択可能なのはどれか。**2つ選べ。**[1][2][3][5][6]

#### 全選択肢（番号付き）
1. **シクロスポリン**  
2. **タクロリムス**  
3. **バシリキシマブ**  
4. **ミゾリビン**  
5. **エベロリムス**[2][3][5][6]

#### 正解番号
**1、2**[1][2][6][9]

#### 解説
患者は妊娠を希望しており、腎移植後の免疫抑制継続が必要。「妊婦・授乳婦を対象とした薬の適正使用推進事業」（2016年度開始）に基づき、**シクロスポリン**と**タクロリムス**は「治療上の有益性が危険性を上回ると判断される場合にのみ投与可能」であり、妊婦使用が可能（アザチオプリンも同様）。[1]

一方、**バシリキシマブ**（抗CD25モノクローナル抗体、急性拒絶反応用）、**ミゾリビン**（核酸合成阻害薬）、**エベロリムス**（mTOR阻害薬）は妊婦投与禁忌のため選択不可。[1][3]
