# 110-338

**問題文**
手袋を外す時の手技について、感染対策上、誤っているのはどれか。1つ選べ。ただし、手袋の外側を濃い色で、内側を薄い色で示している。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-338 を参照

**選択肢**
1. 手袋の外側の手首のあたりを反対の手でつまむ。
2. 手袋を中表にして外し、まだ手袋を着用している手で外した手袋を持っておく。
3. 素手になった手で反対側の手袋の外側をつまむ。
4. 手袋を引き上げるように裏返しにして外していく。
5. 外した後、2枚の手袋をひとかたまりになった状態で廃棄し、速乾性手指消毒剤で手指消毒を行う。

**正解:** 3

**解説**
解答　３

手袋を外す際は、感染対策上、手袋の外側が汚染されていることを想定し、素手で手袋の外側に触れないようにするべきである。選択肢３では、素手で手袋の外側に触れてしまっているため感染対策上不適切である。
