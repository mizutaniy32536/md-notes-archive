# 110-55

**問題文**
レクチンによる認識機構を介した能動的ターゲティングのための修飾基として用いられるのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-55 を参照

**選択肢**
1. 糖鎖
2. ポリペプチド
3. 長鎖不飽和脂肪酸
4. オリゴヌクレオチド
5. ポリエチレングリコール

**正解:** 1

**解説**
解答　１

レクチンによる認識機構を介した能動的ターゲティングのための修飾基として用いられるのは、糖鎖である。レクチンは、標的細胞の糖鎖を認識し結合するタンパク質の総称であり、薬物を能動的に標的細胞に送達する技術に応用されることが期待されている。
