# 110-128

**問題文**
食品添加物A〜Eに関する記述として正しいのはどれか。2つ選べ。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-128 を参照

**選択肢**
1. Aは、食品に甘味を与える目的で使用される指定添加物である。
2. Bは、鮮魚介類や食肉、野菜類の色調を調節する目的で使用される。
3. Cは、食品を漂白する目的で使用され、殺菌料としても用いられる。
4. Dは、金属イオンと錯体を形成することによって、油脂などの酸化を防ぐ目的で使用される。
5. Eは、カビや細菌などの発育を抑制し、保存性を高める目的で乳製品などの食品に添加して使用される。

**正解:** 1,3

**解説**
解答　１、３

１　正
Aは、甘味料のグリチルリチン酸二ナトリウムである。甘草から得た甘味成分で、消費者庁によって指定された指定添加物である。

２　誤
Bは、タール系着色料の食用赤色2号（アマランス）である。着色料は、食品に着色し嗜好的価値を高める目的で添加されるが、食肉・鮮魚介類・生鮮野菜などの色によって品質や鮮度を判断する食品には使用が禁じられている。なお、食肉などの動物性食品中の色素を安定化させる目的では発色剤の亜硝酸ナトリウムなどが、野菜などの植物性食品中の色素を安定化させる目的では色調調整剤の硫酸第一鉄が用いられる。

３　正
Cは、漂白剤の次亜塩素酸ナトリウムである。食品の色素を酸化分解することで漂白作用を示す。また、殺菌料としての用途もあり、カット野菜などの殺菌にも用いられている。

４　誤
Dは、酸化防止剤のエリソルビン酸である。エリソルビン酸は還元型の酸化防止剤であり、自身が油脂の代わりに酸化されることで油脂の酸化を防止する。金属イオンと錯体を形成することで油脂の酸化を防止するのは、金属封鎖型の酸化防止剤で、エチレンジアミン四酢酸二ナトリウム（EDTA・2Na）などがある。

５　誤
Eは、防かび剤のチアベンダゾールである。カビの発育を抑制し、食品の品質を保持する目的で、かんきつ類やバナナに使用される。
