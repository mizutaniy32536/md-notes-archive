# 110-276

### 第110回薬剤師国家試験 問276

**問題文（全文）**  
処方1及び処方3の薬剤に関する記述として、正しいのはどれか。**2つ選べ**。[1][2]

**全選択肢（番号付き）**  
１ 処方1の薬剤は、定常状態で薬物を0次放出する。[1][2]  
２ 処方1の薬剤の1枚当たりの有効成分含量は、フェンタニルに換算して2.55 mgである。[1][2]  
３ 処方1の薬剤からの有効成分の吸収量は、貼付部位の温度が上昇すると増大する。[1][2]  
４ 処方3の薬剤は、シングルユニット型のリザーバー型製剤である。[1][2]  
５ 処方3の薬剤は、初回分を処方1の薬剤の剥離直後に服用する。[1][2]

**正解番号**  
**２、３**[2][8]

**解説**  
- **選択肢２**: 処方1の薬剤（フェンタニル透皮吸収型パッチ、例: デュロテップMTパッチなど）は、1枚（25μg/h、72時間使用）当たりのフェンタニル含量が2.55mgである。これは製品の標準含量に基づく。[1][2]  
- **選択肢３**: フェンタニル透皮パッチは経皮吸収製剤であり、貼付部位の温度上昇（例: 発熱や加温）により皮膚透過率が増大し、吸収量が増えることが知られる。[1][2]  
- **誤り選択肢１**: 定常状態での放出は**1次放出**（膜透過制御による一定速度）であり、0次放出（一定量/時間）ではない。[1][2]  
- **誤り選択肢４**: 処方3の薬剤（オキシコドン経口即放錠、例: オキノームなど）はマトリックス型ではなく、経口錠剤であるためリザーバー型ではない。[1][2]  
- **誤り選択肢５**: 処方3への変更時、初回分は貼付剥離**12時間後**に服用するのが標準（オピオイドローテーション時の忍容性確保のため）。[1][2]  

**背景文脈（問題の前段）**: 患者に持続痛・突出痛があり、処方1（フェンタニル透皮パッチ）使用中に皮膚副作用で処方3（オキシコドン経口製剤）に変更。[1][2]
