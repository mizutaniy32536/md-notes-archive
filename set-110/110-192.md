# 110-192

**問題文**
生薬とその主な副作用の組合せのうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-192 を参照

**選択肢**
（選択肢は画像を参照）

**正解:** 2,5

**解説**
解答　２、５

１　誤
カンゾウの主な副作用は低カリウム血症である。カンゾウはグリチルリチン酸を含有し、低カリウム血症、血圧上昇、浮腫などの症状を呈する偽アルドステロン症やミオパチーを起こすことがある。

２　正

３　誤
オウゴンの主な副作用は胆汁うっ滞である。

４　誤
サンシシの主な副作用は腸管膜静脈硬化症ある。

５　正
マオウに含まれるエフェドリンには交感神経や中枢神経の興奮作用があり、動悸、発汗過多、不眠などを引き起こすことがある。
