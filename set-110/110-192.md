# 110-192

生薬とその主な副作用の組合せのうち、正しいのはどれか。２つ選べ。

**正解:** ２、５

**解説**
１　誤
カンゾウの主な副作用は低カリウム血症である。カンゾウはグリチルリチン酸を含有し、低カリウム血症、血圧上昇、浮腫などの症状を呈する偽アルドステロン症やミオパチーを起こすことがある。
２　正
３　誤
オウゴンの主な副作用は胆汁うっ滞である。
４　誤

マオウに含まれるエフェドリンには交感神経や中枢神経の興奮作用があり、動悸、発汗過多、不眠などを引き起こすことがある。
