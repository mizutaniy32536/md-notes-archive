# 110-42

経口投与された薬物が吸収される経路として、正しいのはどれか。１つ選べ。

１ 小腸 → リンパ管 → 大腸 → 全身循環系
２ 小腸 → リンパ管 → 肝臓 → 全身循環系
３ 小腸 → 門脈 → 大腸 → 全身循環系
４ 小腸 → 門脈 → 肝臓 → 全身循環系
５ 小腸 → 胆管 → 大腸 → 全身循環系
６ 小腸 → 胆管 → 肝臓 → 全身循環系

**正解:** ４

**解説**
吸収とは、薬物が投与された部位から全身循環系へと移行する過程のことである。経口投与された薬物の多くは、小腸で吸収された後、門脈を経て肝臓へ運ばれた後、全身循環系へ移行する。
