# 110-42

**問題文**
経口投与された薬物が吸収される経路として、正しいのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-42 を参照

**選択肢**
1. 小腸　→　リンパ管　→　大腸　→　全身循環系
2. 小腸　→　リンパ管　→　肝臓　→　全身循環系
3. 小腸　→　門脈　→　大腸　→　全身循環系
4. 小腸　→　門脈　→　肝臓　→　全身循環系
5. 小腸　→　胆管　→　大腸　→　全身循環系
6. 小腸　→　胆管　→　肝臓　→　全身循環系

**正解:** 4

**解説**
解答　４

吸収とは、薬物が投与された部位から全身循環系へと移行する過程のことである。経口投与された薬物の多くは、小腸で吸収された後、門脈を経て肝臓へ運ばれた後、全身循環系へ移行する。
