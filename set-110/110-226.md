# 110-226

**問題文**
34歳女性。この女性の12歳の娘に市からヒトパピローマウイルス（HPV）ワクチン接種についてのお知らせが届いた。娘はHPVワクチンの接種歴がない。この女性はHPVワクチンの説明を聞くため、市から送られた下の成分表を含む資料を持って娘と一緒にかかりつけ薬局に来局した。そのHPVワクチンの成分、投与方法、そして免疫応答を高める成分（アジュバント）が入っていることなどを、下の成分表を確認しながら薬剤師が説明した。問226（実務）このワクチンに関する説明として正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-226 を参照

**選択肢**
1. これは弱毒生ワクチンです。
2. 子宮頸がんを予防するものです。
3. 筋肉注射で接種します。
4. 接種は1回で完了します。
5. HPV（16型／18型）感染後に接種しても、がんを予防する効果があります。問227（物理・化学・生物）このワクチンに含まれる添加物に関する記述として、誤っているのはどれか。1つ選べ。
6. 添加物Aは、細菌由来成分の誘導体である。
7. 添加物Aと添加物Bは、ともにアジュバントである。
8. 添加物Bは、抗原を吸着し、投与部位での抗原の滞留時間を延ばす。
9. 添加物Aは、抗原提示細胞の抗原提示機能を促進する。
10. 添加物Aと添加物Bは、ともに自然免疫応答を抑制し、獲得免疫応答を促進する。

**正解:** 2,3

**解説**
問226　解答　２、３

薬効成分がHPV16型および18型であることから、本ワクチンは、子宮頸がんの予防に用いられる組換え沈降2価ヒトパピローマウイルス様粒子ワクチン（サーバリックス®）であると考えられる。

１　誤
現在、日本国内で承認されているHPVワクチンは全て不活化ワクチンである。

２　正
前記参照

３　正
本ワクチンは、10歳以上の女性に、通常、1回0.5 mLを0、1、6ヶ月後に3回、上腕の三角筋部に筋肉内接種する。

４　誤
解説３参照

５　誤
本ワクチンは、接種時に感染が成立しているHPVの排除及び既に生じているHPV関連の病変の進行予防効果は期待できない。

問227　解答　５

本ワクチンに添加されている添加物A（3-脱アシル化-4´-モノホスホリルリピッドA）および添加物B（水酸化アルミニウム懸濁液）は、どちらもアジュバントとして添加されている。アジュバントとは、ワクチンと併用することにより、ワクチンの免疫原性を高めることができる物質である。なお、アジュバントには、アルミニウム塩や病原体のリポ多糖、鞭毛、核酸などから生成された誘導体がある。

１　正
3-脱アシル化-4´-モノホスホリルリピッドは、グラム陰性菌Salmonella minnesota R595株の細胞壁に含まれるリポ多糖の誘導体である。

２　正
前記参照

３　正
水酸化アルミニウムは、抗原を吸着し、体内の滞留時間を長くすることで免疫反応を促進する目的で添加されているアジュバントである。

４　正
3-脱アシル化-4´-モノホスホリルリピッドAは、Toll様受容体4（TLR4）を介した抗原提示細胞の活性化を目的で添加されているアジュバントである。

５　誤
アジュバントは、自然免疫応答の活性化を介して特異的な獲得免疫応答を促進する。
