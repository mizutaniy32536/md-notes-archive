# 110-195

薬物動態の変化に関連する記述として、正しいのはどれか。2つ選べ。
1. 心不全患者では、水溶性薬物の腎排泄速度が低下する。
2. 肥満患者では、脂溶性薬物の分布容積が小さくなる。
3. 小児では、水溶性薬物の体重当たりの分布容積が成人よりも大きい。
4. 妊娠に伴って、糸球体ろ過速度が低下する。
5. 血清アルブミン値低下患者では、薬物の分布容積が低下する。
