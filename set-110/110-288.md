# 110-288

**問題文**
65歳男性。毎年2月から6月頃にかけて、鼻水、くしゃみ、鼻づまりの症状に悩むため、2月になり、一般用医薬品購入のため、来局した。初めての来局だったので、聞き取りを行い、以下の情報が得られた。・出勤や訪問などの外出時に、くしゃみや鼻水がひどくなるが、その症状に対する薬は飲んだことがない。・数年前に市販の花粉症に対する内服薬を購入したことがあるが、品名は忘れた。・就寝時布団に入り暖まると、鼻づまりがひどくなり、寝つきが悪くなることが多い。・仕事で週に2〜3回、社用車を運転して取引先を訪問することがある。・泌尿器科クリニックで前立腺肥大症と診断され、治療を受けている。・頻尿と残尿感を改善する内服薬を1日1回夕食後に服用している。ただし、薬剤名は不明である。問288（病態・薬物治療）この来局者の病態に関する記述として、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-288 を参照

**選択肢**
1. 鼻閉には、鼻粘膜における血管透過性の亢進が関与している。
2. 免疫複合体が鼻粘膜に沈着して発症した。
3. 鼻汁中の好酸球が減少している。
4. 症状の原因を特定するには、スクラッチテストが有用である。
5. 通年性に、鼻粘膜の腫脹が認められる可能性が高い。問289（実務）聞き取りの内容を踏まえて、この来局者に提案する一般用医薬品として最も適切なのはどれか。1つ選べ。

**正解:** 1,4

**解説**
問288　解答　１、４

来局者は、
・毎年2月から6月頃にかけて、鼻水、くしゃみ、鼻づまりなどのアレルギー症状が出ている
・これらの症状は出勤や訪問などの外出時にひどくなる
以上のことから、季節性アレルギー性鼻炎（花粉症）であると推測される。

１　正
鼻閉（鼻詰まり）は、鼻粘膜における血管拡張や血管透過性亢進によって浮腫が起こり、鼻気道が狭くなることで生じる。

２　誤
季節性アレルギー性鼻炎は、I型アレルギーの機序で発症する。I型アレルギーは、マスト細胞（肥満細胞）上のIgE抗体を介したケミカルメディエーターの放出により症状が出現する。なお、免疫複合体が関与するのはⅢ型アレルギーである。

３　誤
I型アレルギーでは、好酸球の増加がみられる。

４　正
スクラッチテストはI型アレルギーにおける皮膚テストの一種で、針で引っ掻いた皮膚の上にアレルゲン溶液を垂らし、膨疹または紅斑の出現具合によって、アレルギーの原因物質を特定する検査である。

５　誤
来局者は季節性アレルギー性鼻炎（花粉症）であると推測されるため、通年性に症状が認められる可能性は低い。

問289　解答　１

来局者は季節性アレルギー性鼻炎（花粉症）であり、治療には抗ヒスタミン薬が用いられるが、下記の背景を考慮して一般用医薬品の提案をする必要がある。
・仕事で週に2〜3回、社用車を運転して取引先を訪問することがある。
→眠気が少ない薬が望ましい。
・泌尿器科クリニックで前立腺肥大症と診断され、治療を受けている。
→尿閉を悪化させない薬が望ましい。

１　正
エピナスチン塩酸塩は非鎮静型第二世代抗ヒスタミン薬であり、眠気が非常に少なく、抗コリン作用も弱いため前立腺肥大症にも比較的安全に使用できる。来局者は本剤の服用は初めてであるため、初回は6錠包装から販売して様子を見るのが妥当である。

２　誤
解説1参照。初回から48錠包装で販売するのではなく、6錠包装から販売して、問題なければ後日48錠包装へ切り替えるのが妥当である。

３　誤
本選択肢の一般用医薬品に含まれるクロルフェニラミンマレイン酸塩は第一世代抗ヒスタミン薬であり、眠気が強く、抗コリン作用による排尿困難の副作用も有するため、来局者に提案する一般用医薬品としては適切ではない。

４　誤
本選択肢の一般用医薬品に含まれるマレイン酸カルビノキサミンは第一世代抗ヒスタミン薬であり、眠気が強く、抗コリン作用による排尿困難の副作用も有する。さらに、ベラドンナ総アルカロイドは抗コリン成分であり、排尿困難の副作用を有するため、来局者に提案する一般用医薬品としては適切ではない。

５　誤
