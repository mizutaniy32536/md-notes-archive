# 110-288

（病態・薬物治療） この来局者の病態に関する記述として、正しいのはどれか。2つ選べ。
1. 鼻閉には、鼻粘膜における血管透過性の亢進が関与している。
2. 免疫複合体が鼻粘膜に沈着して発症した。
3. 鼻汁中の好酸球が減少している。
4. 症状の原因を特定するには、スクラッチテストが有用である。
5. 通年性に、鼻粘膜の腫脹が認められる可能性が高い。
