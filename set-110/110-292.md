# 110-292

（病態・薬物治療） この患者の合併症と治療に関する記述として、正しいのはどれか。2つ選べ。
1. 貧血の所見があり、出血性病変の合併が疑われる。
2. 腎機能が低下し、腎梗塞の合併が疑われる。
3. 疣贅による僧帽弁の障害により、心不全を合併するリスクがある。
4. 脳塞栓症よりも、肺塞栓症を合併するリスクが高い。
5. 血液培養の結果から、抗菌薬としてセファゾリンが推奨される。
