# 110-259

### 第110回薬剤師国家試験 問259

**問題文（全文）**  
前問で最も可能性が高い疾患を患者が治療中であることを眼科医に確認できたので、泌尿器科の医師に処方変更を提案することにした。提案する薬物の作用機序として適切なのはどれか。1つ選べ。[1][2][4]

**全選択肢（番号付き）**  
1. アドレナリンα<sub>1</sub>受容体刺激  
2. アドレナリンβ<sub>3</sub>受容体刺激  
3. アセチルコリンM<sub>3</sub>受容体遮断  
4. アンドロゲン受容体遮断  
5. ミネラルコルチコイド受容体（アルドステロン受容体）遮断[1][2][4][9]

**正解番号**  
**2**[2][6]

**解説**  
問258の正解は**緑内障**（3）であり、患者はドルゾラミド塩酸塩点眼液（緑内障治療薬）を使用中。[1][2] 過活動膀胱の初回処方はアセチルコリンM<sub>3</sub>受容体遮断薬（例: ソリフェナシンなど）が多いが、緑内障患者では閉角緑内障を悪化させる可能性があるため、**アドレナリンβ<sub>3</sub>受容体刺激薬（例: ミラベグロン、ビベグロン）への変更を提案**。[1][2][4] これにより膀胱収縮を抑制し、尿意切迫を改善する。[2][6][9] 他の選択肢は過活動膀胱治療の主要機序に該当しない。[1][4]
