# 110-190

**問題文**
慢性閉塞性肺疾患（COPD）に関する記述として、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-190 を参照

**選択肢**
1. 末梢気道病変と気腫性病変が複合的に関与して発症する。
2. 中高年期より若年期での発症が多い。
3. 病期分類には、対標準1秒量（%FEV1）を用いる。
4. 気管支ぜん息を合併している場合には、副腎皮質ステロイド製剤の吸入よりも内服が推奨される。
5. インフルエンザワクチン接種は禁忌である。

**正解:** 1,3

**解説**
解答　１、３

１　正
慢性閉塞性肺疾患（COPD）は、末梢気道壁が炎症で肥厚・狭窄し気道分泌が貯留する末梢気道病変と、肺胞が炎症の持続により破壊され気腔が拡大する気腫性病変が、様々な割合で複合的に関与して発症する。

２　誤
COPDは、たばこ煙を主とする有害物質を長期に吸入曝露することなどにより生じるため、若年期より中高年期での発症が多い。

３　正
COPDの病期分類は、下記の対標準1秒量（％FEV1）により分類される。
Ⅰ期（軽度）：％FEV180％以上
Ⅱ期（中等度）：％FEV1 50〜80％
Ⅲ期（高度）：％FEV1 30〜50％
Ⅳ期（極めて高度）：％FEV1 30％以下
なお％FEV1は、健常人の1秒量の予測値に対する患者の1秒量の実測値の比率である。

４　誤
COPDの薬物療法では、主に気管支拡張薬（長時間作用性抗コリン薬や長時間作用性アドレナリンβ2受容体刺激薬）の吸入が用いられる。なお、気管支ぜん息を合併している場合には、長時間作用性アドレナリンβ2受容体刺激薬・副腎皮質ステロイド薬の合剤の吸入が推奨されるが、内服ではない。

５　誤
COPD患者では、感染の重症化を防ぐために、インフルエンザワクチンや肺炎球菌ワクチンなどを接種するように指導する。
