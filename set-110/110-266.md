# 110-266

1. a 口腔内において製剤からのニコチンの溶出が低下するため。
2. a 口腔内が酸性に傾くことによりニコチンの吸収が低下するため。
3. b 胃内 pH の上昇によりニコチンの吸収が低下するため。
4. b 消化管においてキレート形成によりニコチンの吸収が低下するため。 小腸における CYP₃A₄ の誘導によりニコチンの代謝が促進される
5. c ため。 小腸における P︲糖タンパク質の誘導によりニコチンの消化管分泌
6. c が促進されるため。
