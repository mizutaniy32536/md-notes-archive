# 110-266

**問題文**
45歳男性。喫煙歴20年（1日20〜30本程度）。男性は、最近、痰が絡んで咳込みが多く、健康のために禁煙を希望し、薬局を訪れた。薬剤師は、禁煙サポートのため男性の生活習慣及び健康状態を聞き取り、ニコチンガム製剤による禁煙を勧めることにした。男性と話し合いのうえ、禁煙期間の目標を90日に設定し、最初の1週間は1日の使用個数の目安を以下とした。（販売する一般用医薬品）ニコチンガム製剤（ニコチン2 mg含有／個）（使用個数の目安）1日9個また、男性に対して習慣的に摂取しているか飲食物を質問したところ、以下の情報が得られた。・炭酸水（a）を頻繁に引用している。・亜鉛含有サプリメント（b）を食後に摂取している。・気分を落ち着かせるために、セントジョーンズワート含有食品（c）を毎日摂取している。問266（薬剤）ニコチンガム製剤による禁煙補助療法の十分な効果を得るために、摂取を控える必要がある飲食物とその理由の組合せとして正しいのはどれか。1つ選べ。問267（実務）薬剤師は、禁煙補助療法導入時のニコチンガム製剤の使用方法及び使用上の注意を説明することにした。薬剤師が説明する内容として、適切なのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-266 を参照

**選択肢**
1. ピリっとした味をかんじるまでゆっくりと噛んだ後、頬と歯茎の間に置く。
2. 喫煙の欲求がなくても定期的に使用する。
3. 治療開始時は、喫煙との併用が可能である。
4. 使用中に唾液が多くなった場合は、唾液を飲み込まず、ティッシュペーパーなどに唾液を出す。
5. 使用後のガムは、医療廃棄物として薬局で回収する。

**解説**
問266　解答　２

ニコチンガム製剤は、有効成分であるニコチンを咀嚼により口腔内で溶出させ、口腔粘膜で吸収させることを目的とした製剤である。有効成分であるニコチンは、弱塩基性薬物であるためpHの低い環境下では分子形の割合が低下する。また、口腔粘膜からの薬物の吸収はpH分配仮説に基づいて行われている。そのため、口腔粘膜からのニコチンの吸収は低下する。 
患者が習慣的に摂取している飲食物の中には、薬物の吸収や代謝に影響を及ぼすものがある。
たとえば、炭酸水（a）は口腔内のpHを低下させるため、ニコチンなどの塩基性薬物の吸収を低下させる可能性がある。
また、亜鉛含有サプリメント（b）は、ペニシラミンなどの薬物 と併用することでキレートを形成し、これらの薬物の吸収を低下させることがある。
さらに、セントジョーンズワート含有食品（c）は、種々のCYPを誘導する作用があり、特にCYP3A4を誘導する作用が強いため、CYP3A4で代謝される薬物と併用するとその代謝が促進され、薬効が減弱することがある。また、P−糖タンパク質 も誘導することが知られており、これにより薬物の消化管からの吸収が低下することがある。

１　誤
炭酸水を飲用することによって口腔内pHが低下するが、製剤からのニコチンの溶出が低下するとの報告はない。

２　正
前記参照
 
３　誤
亜鉛含有サプリメントによって、胃内pHが上昇するという報告はない。

４　誤
ニコチンが亜鉛含有サプリメントとキレートを形成するという報告はない。

５　誤
ニコチンは、主にCYP2A6で代謝される。そのため、ニコチンはセントジョーンズワートと相互作用を起こしにくい。

６　誤
ニコチンはP−糖タンパク質の基質ではない。そのため、ニコチンはP−糖タンパク質を介した相互作用を起こしにくい。

問267　解答　１、４

ニコチンガム製剤は、ニコチンの離脱症状（イライラや集中困難）を軽減するためにニコチンの摂取を喫煙以外の方法で行い、徐々に摂取量を減らしながら最終的にニコチンの摂取をゼロにすることを目的としている。そのため、離脱症状（イライラや集中困難）がない時には摂取するべきではない。また、喫煙時に摂取するとニコチンの過量摂取となる恐れがあるため喫煙と併用するべきではない。

１　正
ニコチンガム製剤は、有効成分であるニコチンを口腔粘膜から吸収させるため、ピリッとした味を感じるまでゆっくり噛んだ後、頬と歯茎に置くよう説明する。

２　誤

４　正
ニコチンガム製剤使用中は、咀嚼行為によって唾液が多く分泌されることがある。この唾液を飲み込んでしまうことにより、胃粘膜が刺激され吐き気や腹痛などの副作用を生じる可能性がある。そのため、ニコチンガム製剤使用中に唾液が多くなった場合は、唾液を飲み込まず、ティッシュペーパーなどに唾液を出すよう説明する。

５　誤
使用後のガムは、医療廃棄物ではないため紙などに包んで小児の手の届かない所に捨てるよう説明する。
