# 110-65

**問題文**
空気感染が主な伝播経路となるウイルス感染症はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-65 を参照

**選択肢**
1. 麻しん
2. 風しん
3. HIV感染症
4. 流行性耳下腺炎（ムンプス）
5. インフルエンザ

**正解:** 1

**解説**
解答　１

１　正
麻しんは主に、空気感染（飛沫核感染）により伝播する。

２　誤
風しんは主に、飛沫感染により伝播する。

３　誤
HIV感染症は、主に同性間同士の性的接触により伝播する。

４　誤
流行性耳下腺炎（ムンプス）は主に、飛沫感染により伝播する。

５　誤
インフルエンザは主に、飛沫感染により伝播する。
