# 110-65

空気感染が主な伝播経路となるウイルス感染症はどれか。１つ選べ。

１ 麻しん
２ 風しん
３ HIV感染症
４ 流行性耳下腺炎（ムンプス）
５ インフルエンザ

**正解:** １

**解説**
１　正
麻しんは主に、空気感染（飛沫核感染）により伝播する。
２　誤
風しんは主に、飛沫感染により伝播する。
３　誤
HIV感染症は、主に同性間同士の性的接触により伝播する。
４　誤
流行性耳下腺炎（ムンプス）は主に、飛沫感染により伝播する。
５　誤
インフルエンザは主に、飛沫感染により伝播する。
