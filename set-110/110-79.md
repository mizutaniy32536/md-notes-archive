# 110-79

**問題文**
汚染されたヒト乾燥硬膜の使用により感染した疾患であって、生物由来製品感染等被害救済制度の創設の契機となったのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-79 を参照

**選択肢**
1. 成人T細胞白血病
2. クロイツフェルト・ヤコブ病
3. トキソプラズマ症
4. 大腿四頭筋短縮症
5. 後天性免疫不全症候群

**正解:** 2

**解説**
解答　２

ヒト乾燥硬膜の使用により感染した疾患であって、生物由来製品感染等被害救済制度の創設の契機となった疾患は、クロイツフェルト・ヤコブ病（選択肢２）である。
