# 110-251

（実務） 患者は内服薬による治療を希望し、新たに内服薬を開始することになった。医師 より、開始する薬剤に関する留意点について、患者への説明を依頼された。説明す る内容として適切なのはどれか。2つ選べ。
1. 寛解が得られた後、寛解維持期においては隔日投与となること。
2. 腎機能障害が現れやすいこと。
3. 脂質検査値の異常が現れる可能性があること。
4. 眠気が起こりやすいこと。
5. 感染症を起こしやすいこと。
