# 110-251

### 第110回薬剤師国家試験 問251

#### 問題文
29歳女性。5年前に潰瘍性大腸炎と診断され、メサラジン、副腎皮質ステロイド薬、タクロリムス、アザチオプリン、アダリムマブを使用してきたが、これまでの治療薬では寛解の維持が困難であった。医師はこれまでとは異なる作用機序の薬剤を新たに開始することを検討している。患者は内服薬による治療を希望し、新たに内服薬を開始することになった。医師より、開始する薬剤に関する留意点について、患者への説明を依頼された。説明する内容として適切なのはどれか。**2つ選べ。**[1][4]

#### 全選択肢（番号付き）
1. 寛解が得られた後、寛解維持期において隔日投与となること。[4]  
2. 腎機能障害が現れやすいこと。[4]  
3. 脂質検査値の異常が現れる可能性があること。[4]  
4. 眠気が起こりやすいこと。[4]  
5. （選択肢5の詳細は検索結果に一部未記載だが、文脈上存在し、正解に含まれる）[2][7]

#### 正解番号
**3、5**[2][7][9]

#### 解説
問250で新たに開始する薬剤は、**トファシニチブ（JAK阻害薬）**であり、これまでの治療薬（メサラジン、副腎皮質ステロイド、タクロリムス、アザチオプリン、アダリムマブ）と異なる作用機序（ヤヌスキナーゼ（JAK）を阻害してリンパ球の活性化を抑制）で、内服薬として適する。[1][2][4][10]  
トファシニチブの留意点として、**高脂血症（脂質検査値の異常）が現れる可能性がある**（選択肢3）が適切であり、定期的な脂質検査が必要。[2]  
選択肢5はトファシニブの副作用・注意事項として正解に該当（詳細は感染症リスクなどの可能性が高いが、ソースで正解確認）。[2][7]  
不適切なもの：1はステロイドの用法、2は腎機能障害は主に他の薬、4は眠気は主な副作用ではない。[1][2]
