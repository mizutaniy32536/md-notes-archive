# 110-167

抗悪性腫瘍薬に関する記述として、正しいのはどれか。2つ選べ。
1. オキサリプラチンは、がん細胞の DNA 鎖内及び鎖間に架橋を形成して、DNA の複製及び転写を阻害する。
2. パクリタキセルは、微小管の重合を阻害して、紡錘糸の機能を抑制する。
3. イリノテカンは、体内で活性代謝物に変化し、トポイソメラーゼⅠ活性を阻害 して、切断された一本鎖 DNA の再結合を抑制する。
4. ニボルマブは、T 細胞上の PD︲₁ に結合し、PD︲₁ と PD︲L₁ 及び PD︲L₂ の結 合を阻害して、T 細胞の活性化を抑制する。
5. ラムシルマブは、上皮増殖因子受容体（EGFR）に結合し、EGF の作用を抑制 して、腫瘍血管新生を抑制する。
