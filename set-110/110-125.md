# 110-125

この ₂₀ 年間の生活習慣病に関する記述として正しいのはどれか。2つ選べ。
1. 糖尿病の患者数は ₄₀ 歳代より急激に増加するが、そのほとんどは ₁ 型糖尿病 である。
2. 慢性閉塞性肺疾患（COPD）による総死亡者数は減少傾向にある。
3. 心疾患による粗死亡率は増加傾向にある。
4. 脳血管疾患のうち、粗死亡率が最も高いのは、脳内出血である。
5. 全悪性新生物死亡のうち、膵臓がんの死亡者数は男女とも増加傾向にある。
