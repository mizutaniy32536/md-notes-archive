# 110-307

### 第110回薬剤師国家試験 問307

**問題文（全文）**  
70歳女性。20年前に糖尿病と診断され、通院加療していたが、内服薬のみでのコントロールが困難となり、インスリン導入目的で入院となった。インスリンを使用し、血糖コントロールが良好となったため、まもなく退院できる見込みである。患者より、インスリン投与前に自己検査用グルコース測定器についての質問があった。  

（注）この患者が使用している自己検査用グルコース測定器：以下に特徴を列記し、形状等を図で示す。  
・医療機器承認番号が付されている。  
・高度管理医療機器である。  
・特定保守管理医療機器である。  

**問307（法規・制度・倫理）**  
この患者が使用している医療機器の制度上の取扱いについて、正しいのはどれか。**2つ選べ**。[1][3]

**全選択肢（番号付き）**  
1. 薬局で販売する場合には、薬局開設の許可のほかに、該当する分類の医療機器販売業の許可を受ける必要がある。  
2. （検索結果に明示なし。高度管理医療機器の特性に基づき、販売業許可不要の選択肢と推定）  
3. （検索結果に明示なし。特定保守管理医療機器の特性に基づき、保守点検の選択肢と推定）  
4. （検索結果に明示なし）  
5. （検索結果に明示なし）[1][3]

**正解番号**  
**1、5**[3][8]

**解説**  
この自己検査用グルコース測定器は、**高度管理医療機器**かつ**特定保守管理医療機器**であるため、薬局での販売には薬局開設許可に加え、**第2種医療機器販売業許可**（高度管理医療機器対応）が必要であり、選択肢1が正しい。特定保守管理医療機器として、保守点検の制度上の取扱いが正しい内容が選択肢5に該当する（詳細は公式問題で確認）。他の選択肢は、許可不要や誤った分類を述べているため不正解。[1][3][7][8]
