# 110-35

**問題文**
アンチトロンビン非依存的に血液凝固第Xa因子の活性を直接阻害する抗凝固薬はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-35 を参照

**選択肢**
1. エノキサパリン
2. フォンダパリヌクス
3. ダビガトランエテキシラート
4. ワルファリン
5. エドキサバン

**正解:** 5

**解説**
解答　５

アンチトロンビン非依存的に血液凝固第Xa因子の活性を直接阻害する抗凝固薬は、エドキサバンである。エドキサバンは、直接血液凝固第Xa因子に結合してその活性を選択的に阻害することで、プロトロンビンからトロンビンへの変換を抑制する。
なお、エノキサパリンとフォンダパリヌクスはアンチトロンビン依存的に血液凝固第Xa因子の活性を阻害する抗凝固薬であり、ダビガトランエテキシラートは直接トロンビン活性を抑制する抗凝固薬、ワルファリンはビタミンKを抑制することでプロトロンビンの生合成を抑制する抗凝固薬である。
