# 110-213

### 第110回薬剤師国家試験 問213

#### 問題文（リード文含む）
31歳女性。1ヶ月前に粘血便、下痢が出現したため近所の消化器内科を受診し、大腸内視鏡検査により**全大腸炎型潰瘍性大腸炎**と診断された。  
医師との面談で、患者が安価な治療を望んだため、**処方1**による治療が開始され症状は改善した。その後、病状が安定していたが、たびたびめまいや頭痛が生じたとの訴えがあったため、医師は**処方1**を**処方2**へ変更し、患者が**処方2**の処方箋を持って薬局を訪れた。[1][2][4]

**薬剤師が、処方2の薬剤の服用及び注意事項について患者に説明する内容として、適切なのはどれか。2つ選べ。**（実務問題）[1][2][3][4]

#### 全選択肢（番号付き）
1. 錠剤が大きくて飲みにくい場合は、粉砕し水や微温湯に懸濁して服用してもよいこと。[1][3][4]  
2. 糞便中に白いものがみられる可能性があること。[4]  
3. グレープフルーツジュースの飲用は避けること。[4]  
4. **症状が改善しても、再燃しないように服薬を継続する必要があること。**[1][2][4]  
5. 処方2の薬剤の服用中は、大腸がん検査を避けること。[1][2]

#### 正解番号
**2, 4**[1][2][3][7][9]

#### 解説
- **選択肢1**: 誤り。処方2の薬剤は放出調整のためコーティングされた錠剤であり、粉砕すると効果が損なわれるため不可。[1]  
- **選択肢2**: 正解。処方2の薬剤（例: メサラジンなどの腸溶錠剤）のコーティング剤が糞便中に白いものとして現れる可能性がある。[4]（複数ソースで正解確認）[1][2][3]  
- **選択肢3**: 不適切（詳細未記載だが、正解外）。[4]  
- **選択肢4**: 正解。潰瘍性大腸炎は寛解期でも服薬継続が再燃予防に必要。[1]  
- **選択肢5**: 誤り。処方2服用中でも定期的な大腸がん検査は推奨され、避ける必要はない。[1]  

処方2は潰瘍性大腸炎治療薬（例: 5-ASA製剤の腸溶錠）で、問212と関連し、処方1からの変更（副作用回避）背景がある。[2][4] 公式正答表でも2,4確認。[7]
