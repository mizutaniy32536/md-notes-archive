# 110-203

### 第110回薬剤師国家試験 問203

**問題文（全文）**  
4歳男児。体重14 kg。事故で低酸素脳症となり大学病院に入院していたが退院し、在宅で人工呼吸器と胃瘻及び導尿による管理を行っている。脳下垂体機能不全に対しては3ヶ月に1回、大学病院の内分泌・代謝内科を外来受診することになっており、初回受診時に、以下の薬剤が処方された。[処方内容は図示、詳細は問202に関連][3][4]。  
**初回受診後の患者家族に対して薬局薬剤師が行う服薬指導として正しいのはどれか。** **2 つ選べ。**[1][3][4]

**全選択肢（番号付き）**  
1. 発熱時には処方1の薬剤を増量する必要があるので、医師の指示を受けてください。[4]  
2. 1 日 合計 2 L 以上の水を薬剤と共に与える。[1]  
3. 大豆製品は処方 3 の薬剤の効果を強めるので、摂らせないようにしてください。[1][3]  
4. 痙れんや嘔吐は薬剤による副作用の可能性があるので、医師に連絡してください。[1][3][4]  
5. [詳細未明示だが、大豆製品関連の誤り選択肢][1]

**正解番号**  
1, 4[1][3][9]

**解説**  
- **選択肢1**: 妥当。発熱時は処方1の薬剤（おそらく成長ホルモン関連）を増量する必要があり、医師の指示を仰ぐ指導が適切[1][4]。  
- **選択肢2**: 誤り。デスモプレシン（処方関連薬剤）服用時は水中毒に注意し、水分摂取を1日合計2L以上とする指導は不適切[1]。  
- **選択肢3**: 誤り。大豆製品は処方3の薬剤（おそらくレボチロキシン）の効果を強めるという根拠がなく、通常量では問題ない[1][3]。  
- **選択肢4**: 妥当。痙れんや嘔吐は薬剤副作用（例: デスモプレシンによる水中毒）の可能性があるため、医師連絡を指導[1][3]。  
- **選択肢5**: 誤り。大豆製品の大量摂取がない限り問題ない[1]。  
以上より、正解は1と4[1][3]。
