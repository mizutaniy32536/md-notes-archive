# 110-203

（実務） 初回受診後の患者家族に対して薬局薬剤師が行う服薬指導として正しいのはどれ か。2つ選べ。
1. 発熱時には処方 ₁ の薬剤を増量する必要があるので、医師の指示を受けてくだ さい。
2. 幼児は成人よりも体重あたりの必要水分量が多いので、 ₁ 日合計 ₂ L 以上の水 を薬剤と共に与えてください。
3. 決められた服用時刻を遵守すれば、食事の時間はいつでもかまいません。
4. 痙れんや嘔吐は薬剤による副作用の可能性があるので、医師に連絡してください。
5. 大豆製品は処方 ₃ の薬剤の効果を強めるので、摂らせないようにしてください。
