# 110-241

### 第110回薬剤師国家試験 問241

**問題文（共通リード文含む）**  
2歳8ヶ月女児。1週間前から腹痛を訴えることがあり、今朝になって、嘔吐したため、近医を受診した。母親からの聞き取りでは、日頃から鉛を含む金属製アクセサリーを舐める癖があり、誤飲したおそれがあるとのことであった。  
レントゲン検査では異物は見当たらなかった。尿中のδ-アミノレブリン酸濃度は10mg/Lと高値であった。また、血液検査では有核赤血球数の増加がみられ、赤血球遊離プロトポルフィリン濃度は180μg/dL（基準値30〜86μg/dL）、原子吸光光度計で測定した血中鉛濃度は75μg/dLであった。[1][2]

**問241（衛生）**  
ヘムの生合成経路を表した下図中の1〜5のうち、この女児の中毒原因物質によって抑制される反応はどれか。**2つ選べ。**[1][2]

**選択肢**  
（注: 図中の反応番号1〜5に対応。図は提供されていないが、鉛中毒によるヘム生合成阻害の標準経路に基づく）  
1〜5（具体的な反応名は図示されていないが、正解は2と5）[1][2][9]

**正解番号**  
2、5[1][2][8]

**解説**  
この女児は血中鉛濃度75μg/dLと高値、尿中δ-アミノレブリン酸高値、有核赤血球増加、赤血球遊離プロトポルフィリン高値から**鉛中毒**が疑われる。[1][2][7]  
鉛はヘム生合成経路で**δ-アミノレブリン酸デアミナーゼ（ALA-D）**と**フェロケラターゼ**を阻害する。これが図中の**2（ALA-D）**と**5（フェロケラターゼ）**に対応。[1][7][9]  
阻害によりALA蓄積（尿中高値）、プロトポルフィリンIX蓄積（赤血球高値）が起こり、ヘム合成障害による貧血（有核赤血球増加）を呈する。[1][2][7]
