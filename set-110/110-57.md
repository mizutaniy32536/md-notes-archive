# 110-57

デキストラン硫酸固定化セルロースを用いた吸着器によるアフェレシス（注）施行中の患者への投与禁忌薬はどれか。１つ選べ。 （注）アフェレシス：生体内のさまざまな血液関連因子を分離・除去して治療する広範囲な医療技術の総称

１ ニフェジピン
２ エナラプリルマレイン酸塩
３ イルベサルタン
４ エサキセレノン
５ ドキサゾシンメシル酸塩

**正解:** ２

**解説**
エナラプリルは、ACE（アンギオテンシン変換酵素）阻害薬である。エナラプリルは、ACEと同一酵素であるキニナーゼⅡも同時に阻害することでブラジキニンの分解を抑制する作用を併せもつ。
一方、デキストラン硫酸固定化セルロースは、血中キニン系の代謝亢進によりブラジキニンの生成を促進する作用を持ち、エナラプリルとの併用でブラジキニンの過剰蓄積が起こることがあるため、併用禁忌とされている。
