# 110-57

**問題文**
デキストラン硫酸固定化セルロースを用いた吸着器によるアフェレシス（注）施行中の患者への投与禁忌薬はどれか。1つ選べ。（注）アフェレシス：生体内のさまざまな血液関連因子を分離・除去して治療する広範囲な医療技術の総称

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-57 を参照

**選択肢**
1. ニフェジピン
2. エナラプリルマレイン酸塩
3. イルベサルタン
4. エサキセレノン
5. ドキサゾシンメシル酸塩

**正解:** 2

**解説**
解答　２

エナラプリルは、ACE（アンギオテンシン変換酵素）阻害薬である。エナラプリルは、ACEと同一酵素であるキニナーゼⅡも同時に阻害することでブラジキニンの分解を抑制する作用を併せもつ。
一方、デキストラン硫酸固定化セルロースは、血中キニン系の代謝亢進によりブラジキニンの生成を促進する作用を持ち、エナラプリルとの併用でブラジキニンの過剰蓄積が起こることがあるため、併用禁忌とされている。
