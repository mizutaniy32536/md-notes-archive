# 110-13

**問題文**
真核生物において、転写開始の際にRNAポリメラーゼⅡの結合に必要なDNA上の領域はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-13 を参照

**選択肢**
1. イントロン
2. エクソン
3. プロモーター
4. テロメア
5. ターミネーター

**正解:** 3

**解説**
解答　３

１　誤
イントロンは、成熟mRNAを生成する前にスプライシング反応によって除去される塩基配列の部分である。

２　誤
エクソンは、スプライシング反応により除去されず、最終的にアミノ酸配列に翻訳される部分である。

３　正
プロモーターは、DNAにおける転写制御を担う領域である。プロモーター領域に基本転写因子とRNAポリメラーゼが結合し、複合体を形成することで転写が開始される。

４　誤
テロメアは、真核細胞の染色体の末端にある構造で、染色体末端を保護する役割をもつ。

５　誤
ターミネーターとは、転写が行われている遺伝子またはオペロンの終端を示す塩基配列部分である。
