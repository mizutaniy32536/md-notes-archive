# 110-254

**問題文**
56歳男性。身長165 cm、体重63 kg。1年前に健診で心電図異常が見つかり、循環器内科で精密検査となった。その結果、頻脈性不整脈と診断され、ジソピラミド（50 mgカプセルを1回1カプセル、1日3回朝昼夕食後服用）による治療が開始され、継続して服用していた。2週間前に動悸や立ちくらみを自覚するようになり、精密検査と治療の見直しのため入院となった。入院から4日目にジソピラミドを1日150 mgから300 mgに増量したが、症状が改善されなかったことから、ジソピラミドをソタロール（80 mg錠を1回1錠、1日2回朝夕食後服用）に変更となった。ソタロール服用開始から5日目に、担当薬剤師が病室を訪れた際に、下図に示す心電図の異常を発見した。問254（薬理）この患者に使用された2つの抗不整脈薬に共通する作用機序はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-254 を参照

**選択肢**
1. 電位依存性Na＋チャネル遮断
2. 電位依存性Ca2＋チャネル遮断
3. 電位依存性K＋チャネル遮断
4. アドレナリンβ受容体遮断
5. アセチルコリンM2受容体遮断問255（実務）認められた心電図異常はどれか。2つ選べ。
6. ST上昇
7. 房室ブロック
8. 徐脈
9. QT延長
10. テント状T波の出現

**正解:** 3

**解説**
問254　解答　３

本患者に用いられたジソピラミドとソタロールに共通する作用機序は、電位依存性K＋チャネル遮断作用である。ジソピラミドはVaughan Williams分類Ⅰa群の抗不整脈薬で電位依存性Na＋チャネル遮断作用、電位依存性K＋チャネル遮断作用を有しており、ソタロールはVaughan Williams分類Ⅲ群の抗不整脈薬で電位依存性K＋チャネル遮断作用を有する。
なお、電位依存性Ca2＋チャネル遮断作用を有するのはベラパミルなどのVaughan Williams分類Ⅳ群の抗不整脈薬、アドレナリンβ受容体遮断作用を有するのはメトプロロールなどのVaughan Williams分類Ⅱ群の抗不整脈薬、アセチルコリンM2受容体遮断作用を有するのは徐脈性不整脈治療薬のアトロピンである。

問255　解答　３、４

心電図より本患者は徐脈及びQT延長が認められる。心拍数を算出する方法としては、心室の収縮から次の心室の収縮を示すRR間隔を計り、1拍にかかる時間を導く。心拍数は1分あたりであらわすため、1分（60秒）をRR間隔（秒）で徐することで導く。心電図よりRR間隔は約32マスであるため、32マス×0.04秒／マス＝1.28秒となる。よって、1分あたりの心拍数は60秒÷1.28秒＝約47回となり、徐脈（心拍数50回／分未満）が認められる。
また、心電図よりQT間隔は約18マスであるため、18マス×0.04秒／マス＝0.72秒となる。QT間隔の正常上限値は0.45秒のため、QT延長が認められる。
