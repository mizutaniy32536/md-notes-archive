# 110-27

**問題文**
フェニレフリンの昇圧作用の機序はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-27 を参照

**選択肢**
1. アドレナリンα1受容体刺激
2. アドレナリンα2受容体遮断
3. アドレナリンβ1受容体刺激
4. カテコール−O−メチルトランスフェラーゼ阻害
5. ノルアドレナリン再取り込み阻害

**正解:** 1

**解説**
解答　１
