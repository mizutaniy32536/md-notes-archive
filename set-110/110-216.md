# 110-216

**問題文**
保険薬局に勤務している薬剤師が、看護学校の講師を担当することになった。講義では、植物に含まれている麻薬成分が医療用麻薬として使用されていることを紹介することにした。最近、高度のがん疼痛のある患者が、その初回の服薬を躊躇した事例を経験したことから、その場合の対応や、現場での医療用麻薬の有効な使い方についても講義しようと考えている。患者が服薬を躊躇した事例の処方薬は、モルヒネ塩酸塩水和物徐放性カプセルとモルヒネ塩酸塩水和物内用液であった。問216（実務）講義で伝える内容として、適切なのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-216 を参照

**選択肢**
1. オピオイドを定期的に鎮痛に必要な量で投与すれば、がん患者の生命予後に影響を与えない。
2. がん疼痛管理にオピオイドを使用する場合、精神依存が生じる可能性は低く、オピオイドの使用を控える理由とはならない。
3. 服薬を躊躇したこの高度のがん疼痛のある患者に対しては、弱オピオイドを提案する。
4. 高度の腎機能障害がある患者に対しては、この事例の処方薬の投与を推奨する。
5. がん疼痛の突出痛がある場合、オピオイドの徐放製剤をレスキュー薬として投与する。問217（物理・化学・生物）また、講義ではモルヒネと同じ植物に含まれる麻薬成分である化合物Xについても紹介し、化合物Xは一般用医薬品の鎮咳薬としても利用され、一部ではその濫用が問題になっていることを述べることにした。化合物Xの化学構造式と、その基原となる植物写真の組合せとして、正しいのはどれか。1つ選べ。

**正解:** 1,2

**解説**
問216　解答　１、２

がん疼痛の治療において、オピオイド（麻薬性鎮痛薬）は中心的な役割を担う。WHO方式がん疼痛治療法では、患者の痛みの強さに応じて段階的に鎮痛薬を選択、使用することが推奨されている。

１　正
モルヒネをはじめとするオピオイドは、鎮痛目的で適切に使用した場合、生命予後（生存期間）には影響しないことが確認されている。また、疼痛コントロールを良好に保つことで、痛みによるQOL低下を防ぎ、活動性や食欲の改善に寄与する可能性がある。

２　正
がん疼痛管理でのオピオイド使用においては、精神依存のリスクは非常に低く、依存の懸念からオピオイドを避けるべきではない。身体依存（離脱症状）や耐性は起こり得るが、疼痛管理の妨げとはならない。

３　誤
すでに高度のがん疼痛がある患者であれば、モルヒネなどの強オピオイドが第一選択である。また、弱オピオイドは中等度疼痛に用いるものであり、高度のがん疼痛に対しては効果が不十分な可能性が高い。したがって、弱オピオイドへの提案は不適切である。

４　誤
モルヒネは腎排泄性代謝物を生成するため、腎機能が高度に低下している患者ではこれらの代謝物が体内に蓄積し、副作用リスクが高まる。よって、フェンタニルやオキシコドンなど、腎機能の影響を受けにくい薬剤への切り替えが推奨される。すなわち、高度の腎機能障害がある患者に対してはモルヒネの投与を推奨せず、薬剤変更を検討すべきである。

５　誤
がん疼痛の突出痛には、速効性のある即放性製剤をレスキュー薬として使用する。なお、徐放製剤は定時投与で血中濃度を安定させる目的であり、突出痛への対応には不適切である。

問217　解答　４

化合物Xは一般用医薬品の鎮咳薬としても利用され、一部ではその濫用が問題になっていることから、コデインと推定される。コデインは、ケシ（Papaver somniferum）に含まれる天然由来の成分であり、医療用および一般用医薬品（鎮咳薬）として使用されている。
化学構造式Aはコデイン、化学構造式Bはデキストロメトルファン（合成鎮咳薬）、化学構造式Cはナロキソン（オピオイド拮抗薬）である。
また、写真アは３〜９枚の小葉が集まった手のひらのようのな形であることからアサ、写真イは花が直立した茎の先端につき、写真の右側に楕円形で、上部が平たい皿のような形をしている果実があることからケシと推定される。
すなわち、化合物Xの化学構造式はA、基原植物の写真はイである。
