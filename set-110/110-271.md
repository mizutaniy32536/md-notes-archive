# 110-271

### 第110回薬剤師国家試験 問271

**問題文（全文）**  
前問で懸念された薬物相互作用を回避するために、薬剤師が皮膚科の医師に提案する処方変更の内容として、適切なのはどれか。1つ選べ。[1][2]

**全選択肢（番号付き）**  
1. 投与量の減量  
2. 投与タイミングの就寝前への変更  
3. **テルビナフィン塩酸塩錠への変更**  
4. イトラコナゾール内用液への変更  
5. ボリコナゾール錠への変更[1][2]

**正解番号**  
**3**[7][8]

**解説**  
問270では、イトラコナゾールカプセルと患者の現在服用薬（お薬手帳記載、詳細は問270でグラペフルーツジュースなどCYP3A4阻害物質が示唆）との相互作用機序として「**5. ヘム鉄への配位結合によるCYP3A4の阻害**」が正解であり、イトラコナゾールのCYP3A4阻害が問題となる。[2][7]  
この相互作用を回避するため、抗真菌薬を**CYP3A4阻害作用のないテルビナフィン塩酸塩錠**に変更するのが適切。頭部白癬（皮膚真菌症）に対しテルビナフィンは有効で、CYP3A4を介さない経路で代謝されるため相互作用リスクが低い。他の選択肢は不適切：1は阻害を避けず、2はタイミング変更で不十分、4はイトラコナゾール系のため同問題、5はボリコナゾールもCYP3A4阻害薬で相互作用増強の可能性あり。[1][2][7]
