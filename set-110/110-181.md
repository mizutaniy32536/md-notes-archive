# 110-181

**問題文**
下図はロータリー打錠機による打錠工程を模式的に示したものである。工程Aはどれか。1つ選べ。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-181 を参照

**選択肢**
1. 臼への充填
2. 予備圧縮
3. 重量調節
4. 圧縮成形
5. 錠剤放出

**正解:** 3

**解説**
解答　３

ロータリー打錠機による打錠工程は、臼への充填→重量調節→予備圧縮→圧縮成形→錠剤放出のサイクルで連続的に行われる。工程Aは重量調節であり、秤重レールを上下させることにより原料粉体の重量を調節する。
