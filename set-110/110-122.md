## ◆ 飲酒量と大腸がんリスクのコホート研究結果を読み解くシナリオ

260311

**問題**
下表は、日本人を対象とした5つのコホート研究のデータを統合して解析し、男性の飲酒量と大腸がん、結腸がん及び直腸がんの関係について調べた結果である。この表に関する記述として正しいのはどれか。2つ選べ。
ただし、下表を基に算出した「飲酒しない群」の大腸がん発症率は10万観察人年当たり142人、「23.0g/日以上のアルコール摂取群」の大腸がん発症率は10万観察人年当たり196人である。

※表（飲酒量別のハザード比、95%信頼区間、症例数、観察人年のデータ）は原本を参照

**選択肢**

1. 週に1回未満の飲酒であっても、飲酒は大腸がんのリスクを有意に増加させる。
​
2. 22.9g/日以下のアルコール摂取であっても、週に1回以上の飲酒は直腸がんのリスクを有意に増加させる。
​
3. 23.0g/日以上のアルコール摂取は、結腸がんのリスクを有意に増加させる。
​
4. 飲酒しない群と比較して、23.0g/日以上アルコールを摂取する日本人男性では、飲酒によって10万観察人年当たり338人が過剰に大腸がんを発症すると推定される。
​
5. 飲酒しない群と比較して、23.0g/日以上アルコールを摂取する日本人男性の大腸がんのうち、28%は飲酒によるものであると推定される。

**正解と理由**
- `3. 23.0g/日以上のアルコール摂取は、結腸がんのリスクを有意に増加させる。`
- `5. 飲酒しない群と比較して、23.0g/日以上アルコールを摂取する日本人男性の大腸がんのうち、28%は飲酒によるものであると推定される。`
- **理由の構成**
	- **問題の整理**：コホート研究の統合解析データから、飲酒量と各種大腸がんの関連を`ハザード比`と`95%信頼区間`で評価し、さらに`寄与危険度`・`寄与危険割合`の計算を行う
	- **薬学的知識・仕様・制約**：`ハザード比`の95%信頼区間が1をまたがなければ統計的に有意。`寄与危険度`=曝露群発症率−非曝露群発症率。`寄与危険割合`（帰属危険割合）=寄与危険度÷曝露群発症率
	- **選択理由**：選択肢3は23.0g/日以上群の結腸がんハザード比1.60（信頼区間が1をまたがない）→ 有意。選択肢5は(196-142)/196≈0.275（約28%）→ 正しい

**解法の手順（思考の流れ）**
- 選択肢1「週1回未満で有意」→ ハザード比1.00で信頼区間が1をまたいでいる → 有意ではない → 誤り
- 選択肢2「22.9g/日以下で直腸がんリスク有意増加」→ 信頼区間が1をまたぐ → 有意ではない → 誤り
- 選択肢3「23.0g/日以上で結腸がん有意増加」→ ハザード比1.60、信頼区間が1をまたがない → 有意 → 正しい
- 選択肢4「過剰発症338人」→ `寄与危険度`=196-142=54人/10万観察人年。338は計算間違い → 誤り
- 選択肢5「28%は飲酒による」→ `寄与危険割合`=(196-142)/196=54/196≈0.275=約28% → 正しい

**他の選択肢が誤りな理由**
- 1. 週に1回未満の飲酒であっても、飲酒は大腸がんのリスクを有意に増加させる。
	週1回未満の飲酒群の大腸がんハザード比は1.00であり、`95%信頼区間`が1をまたいでいるため、統計的に有意な増加とは言えない

- 2. 22.9g/日以下のアルコール摂取であっても、週に1回以上の飲酒は直腸がんのリスクを有意に増加させる。
	0.1-22.9g/日群の直腸がんハザード比の`95%信頼区間`が1をまたいでおり、統計的に有意ではない

- 4. 飲酒しない群と比較して、23.0g/日以上アルコールを摂取する日本人男性では、飲酒によって10万観察人年当たり338人が過剰に大腸がんを発症すると推定される。
	`寄与危険度`=196-142=54人/10万観察人年。338は196+142の誤計算であり、正しくは54人

**キーポイント**
- `95%信頼区間`が1をまたぐ → 統計的に有意ではない、またがない → 有意
- `寄与危険度`=曝露群の発症率−非曝露群の発症率（絶対的な超過リスク）
- `寄与危険割合`（帰属危険割合）=寄与危険度÷曝露群の発症率。曝露群における曝露に起因する割合
- `ハザード比`≒`相対危険度`。生存時間解析で用いる指標

**関連知識の比較**
- `相対危険度`（RR）= 曝露群罹患率/非曝露群罹患率。関連の強さの指標
- `寄与危険度`（AR）= 曝露群罹患率 − 非曝露群罹患率。絶対リスクの差
- `寄与危険割合`= AR/曝露群罹患率。曝露群での曝露に起因する割合
- `集団寄与危険割合`（PAF）= 集団全体で曝露を除去した場合に予防できる割合
- アルコール23.0g/日≒瓶ビール1本（中瓶500mL）、日本酒1合相当

（一言メモ）
- 「95%信頼区間が1をまたぐか」で有意性を判断するパターンは国試頻出。表の読み方を練習しておく
- `寄与危険度`と`寄与危険割合`の計算は、問題文に発症率が直接与えられる場合が多い。引き算と割り算だけ
- 「338人」のようなひっかけ選択肢は、足し算と引き算を混同させる典型パターン
