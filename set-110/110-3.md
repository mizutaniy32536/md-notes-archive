# 110-3

難溶性塩である水酸化鉄（Ⅲ）（Fe(OH)3）の純水中での溶解度をS mol/Lとすると、その溶解度積（Ksp）を正しく表しているのはどれか。１つ選べ。

１ 3S
２ 4S
３ 9S
４ 9S
５ 27S

**正解:** ５

**解説**
一般に、難溶性塩MmAnの溶解度積Kspは次のように表せる。
Ksp＝[Mn＋]m[Am－]n
よって、Fe(OH)3の溶解度積は以下のとおりである。
Ksp＝[Fe3＋][OH－]3
また、Fe(OH)3が溶解平衡にあるときの各イオンの濃度は次のように表せる。
Fe(OH)3 ⇄ Fe3＋ ＋ 3OH－ より、[Fe3＋]＝S mol/L 、[OH－]＝ 3S mol/L
よって、Ksp＝[Fe3＋][OH－]3＝S・(3S)3＝27S4
