# 110-3

**問題文**
難溶性塩である水酸化鉄（Ⅲ）（Fe(OH)3）の純水中での溶解度をS mol/Lとすると、その溶解度積（Ksp）を正しく表しているのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-3 を参照

**選択肢**
1. 3S3 (mol/L)3
2. 4S3 (mol/L)3
3. 9S3 (mol/L)3
4. 9S4 (mol/L)4
5. 27S4 (mol/L)4

**正解:** 5

**解説**
解答　５

一般に、難溶性塩MmAnの溶解度積Kspは次のように表せる。
Ksp＝[Mn＋]m[Am－]n
よって、Fe(OH)3の溶解度積は以下のとおりである。
Ksp＝[Fe3＋][OH－]3

また、Fe(OH)3が溶解平衡にあるときの各イオンの濃度は次のように表せる。
Fe(OH)3 ⇄ Fe3＋ ＋ 3OH－ より、[Fe3＋]＝S mol/L 、[OH－]＝ 3S mol/L

よって、Ksp＝[Fe3＋][OH－]3＝S・(3S)3＝27S4
