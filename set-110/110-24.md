# 110-24

**問題文**
湖沼で藍藻類や放線菌が産生するカビ臭の原因物質はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-24 を参照

**選択肢**
1. クロラミン
2. ジェオスミン
3. トリハロメタン
4. クロロフェノール
5. ミクロシスチン

**正解:** 2

**解説**
解答　２

湖沼や海域のような閉鎖的水域では、水の循環が起こりにくいため富栄養化が起こりやすい。富栄養化状態では、藍藻類や放線菌が増殖しやすく、藍藻類などが産生するジェオスミンや2-メチルイソボルネオールはカビ臭の原因、ミクロシスチンは肝毒性の原因となる。
