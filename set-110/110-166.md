# 110-166

### 第110回薬剤師国家試験 問166

**問題文（全文）**  
レンサ球菌属に対して抗菌作用を示す薬物に関する記述として、正しいのはどれか。**2つ選べ。**[1][3][7]

**全選択肢（番号付き）**  
1. セファレキシンは、細菌の**ピルビン酸転移酵素**を阻害して、UDP-N-アセチルムラミン酸の合成を抑制する。[1][3]  
2. エリスロマイシンは、マクロライド系抗菌薬であり、細菌のリボソーム**50Sサブユニット**に結合して、タンパク質の合成を阻害する。[3]  
3. アモキシシリンは、ペニシリン系抗菌薬であり、細菌のペニシリン結合タンパク質に共有結合して、不可逆的に**トランスペプチダーゼ**活性を阻害する。[3]  
4. （選択肢4の記述は検索結果で部分的にしか確認できず、完全な文言は公式問題集参照を推奨。ただし正解に含まれる）[3][4]  

※選択肢の完全なリストは厚生労働省公式問題（[5]）で確認可能。検索結果では1〜3が主に記載。[1][3]

**正解番号**  
**3、4**[4][6]

**解説**  
- **1 誤**: セファレキシンは第1世代セファロスポリンで、ペニシリン結合タンパク質を阻害して細胞壁合成を抑制。ピルビン酸転移酵素阻害はバンコマイシンの作用機序。[1][3]  
- **2 誤**: エリスロマイシンはレンサ球菌に抗菌作用を示すが、A群溶連菌では耐性例が多く、第一選択ではない。記述自体は正しいが、問の文脈で不適切（または完全正解でない）。[3]  
- **3 正**: アモキシシリンはペニシリン系βラクタム薬で、トランスペプチダーゼを不可逆阻害し、レンサ球菌に優れた抗菌作用を示す。溶連菌咽頭炎の第一選択薬。[1][3]  
- **4 正**: （詳細記述未詳だが、正解確認済み。薬理的正しい記述で、レンサ球菌有効薬の作用機序を正確に述べたものと推定。[4][6]）  

A群溶連菌咽頭炎治療ではペニシリン系（アモキシシリン等）が標準で、セファレキシンも代替可能。[1][3]
