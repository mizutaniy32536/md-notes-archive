# 110-201

### 第110回薬剤師国家試験 問201

**問題文**  
スマトリプタンはセロトニン (5-HT) 受容体のアゴニストとして薬効を発揮するが、受容体のサブタイプによって結合の強さが異なる。3H標識した5-HT1D受容体を用いた結合試験の結果から、スマトリプタンの5-HT1D受容体に対する解離定数Kdを求めよ。  
（前文の症例：45歳女性。慢性的に片頭痛を繰り返しており、現在、処方1及び2の薬剤を服用している。病院再診時、患者の片頭痛日誌を確認したところ、月4回以上通勤できないほどの片頭痛発作があった。医師は処方2の薬剤を変更するため、病院薬剤師に薬剤の選択について意見を求めた。[1][2][3][9]）

スマトリプタンが結合していない5-HT1D受容体をR、スマトリプタンの遊離形をS、スマトリプタンと5-HT1D受容体の結合形をRSとし、RとSは次のように1：1で反応するものとする。  
\[ \ce{R + S <=> RS} \]  
解離定数Kdは次の式で表される。  
\[ K_d = \frac{[R][S]}{[RS]} \]  

与えられたデータ：  
- スマトリプタンの総濃度（[S] + [RS]）：**100 nmol/L**  
- 受容体の総濃度（[R] + [RS]）：**100 nmol/L**  
- 結合した受容体濃度（[RS]）：**80 nmol/L**[1][2][3]

スマトリプタンと5-HT1D受容体の**解離定数Kd**として高いのはどれか。[1][2][3]

**全選択肢**  
1. 0.20 nmol/L  
2. 0.80 nmol/L  
3. **5.0 nmol/L**  
4. 80 nmol/L  
5. 100 nmol/L[1][2][3]

**正解番号**  
3[1][2]

**解説**  
スマトリプタンの総濃度が100 nmol/L、[RS] = 80 nmol/L であるため、遊離スマトリプタン濃度[S] = 総濃度 - [RS] = 100 - 80 = **20 nmol/L**。  
受容体の総濃度が100 nmol/L であるため、結合していない受容体濃度[R] = 総濃度 - [RS] = 100 - 80 = **20 nmol/L**。  
解離定数 \( K_d = \frac{[R][S]}{[RS]} = \frac{20 \times 20}{80} = 5 \) nmol/L。  
したがって、正解は**3（5.0 nmol/L）**。[1][2]
