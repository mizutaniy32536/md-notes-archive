# 110-72

**問題文**
医薬品医療機器等法において、「薬局医薬品」とは、「要指導医薬品及び A 以外の医薬品をいう。」となっている。Aにあてはまるのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-72 を参照

**選択肢**
1. 医療用医薬品
2. 処方箋医薬品
3. 一般用医薬品
4. 体外診断用医薬品
5. 薬局製造販売医薬品

**正解:** 3

**解説**
解答　３

医薬品医療機器等法において、「医薬品」は3種類に大別され、それぞれ「薬局医薬品」、「要指導医薬品」、「一般用医薬品」とよばれる。
したがって、「薬局医薬品」とは、「要指導医薬品及び一般用医薬品以外の医薬品」であり、本問では選択肢３が正答となる。
なお、「薬局医薬品」には、薬局製造販売医薬品（薬局製剤）及び医療用医薬品（処方箋医薬品を含む）が含まれる。
