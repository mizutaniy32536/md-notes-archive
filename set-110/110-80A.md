## ◆ 傾聴における適切なコミュニケーション技法を判断するシナリオ
260307

**問題**
- コミュニケーションを円滑にするため取り入れる傾聴の技能や態度として、適切なのはどれか 1つ選べ

**選択肢**

1. ブロッキング

2. ミラーリング

3. エンコーディング

4. パターナリズム

5. デコーディング

**正解と理由**
- `2. ミラーリング`
- **理由の構成**
	- **問題の整理**: 問いは「傾聴の技能や態度」として適切なものの特定
	- **薬学的知識・仕様・制約**: 傾聴は共感的理解 無条件の肯定的関心 自己一致を軸に相手理解を深める技能であり、相手の非言語情報へ注意を向ける姿勢が重要
	- **選択理由**: ミラーリングは相手のしぐさ 表情 姿勢などを自然に合わせて心理的距離を縮める技法であり、傾聴実践に合致

**解法の手順（思考の流れ）**
1. まず「傾聴の技能」という語から、相手理解と関係形成を促進する行動を探すと判断
2. 次に選択肢を「技能」「情報処理用語」「意思決定態度」に分類して絞り込む
3. パターナリズムは意思決定モデルであり傾聴技法ではないため除外
4. ブロッキングは会話遮断の意味合いで傾聴と逆方向と判断
5. 残る候補の中で、対人コミュニケーション技法として成立するミラーリングを選択

**他の選択肢が誤りな理由**
- 1. ブロッキング
	傾聴では相手の発話を妨げず受容的に聴くことが基本であり、`ブロッキング`は`傾聴`の方向性と一致しない

- 3. エンコーディング
	`エンコーディング`は情報を符号化する一般概念であり、対人場面での`傾聴技能`を直接示す用語ではない

- 4. パターナリズム
	`パターナリズム`は医療者主導の意思決定態度を指し、相手の語りを引き出す`傾聴技法`とは別概念

- 5. デコーディング
	`デコーディング`は情報の復号に関する一般概念であり、設問で求める`技能や態度`としては不適切

**キーポイント**
- `傾聴`は相手理解と信頼形成を目的にした対人コミュニケーション実践
- `ミラーリング`は非言語レベルの同調でラポール形成を補助
- `パターナリズム`は意思決定モデルであり傾聴技法とは区別して覚える

**関連知識の比較**
- `ミラーリング`は動作 姿勢 表情の同調、`マッチング`は声量 速度 トーンなど音声面の同調
- `オープンクエスチョン`は相手の語りを広げ、`クローズドクエスチョン`は情報確認に有効
- `共感的理解` `無条件の肯定的関心` `自己一致`はロジャーズの中核概念として併記して整理

（一言メモ）
- `傾聴`では助言より先に相手の語りを最後まで受け止める姿勢を優先
- `ミラーリング`はやりすぎると不自然になるため自然な範囲で使う
- `パターナリズム`と`共有意思決定`の違いは国試頻出論点として横断整理
