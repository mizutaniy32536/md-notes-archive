# 110-92

ファントホッフプロットは直線を示す。このことに関する記述として正しいの はどれか。2つ選べ。
1. 温度の逆数に対して平衡定数の対数をプロットしたものである。
2. 切片から標準反応エンタルピーが求まる。
3. 傾きから標準反応エントロピーが求まる。
4. 吸熱反応のとき、傾きは負である。
5. 傾きが正のとき、温度が上がるにつれて平衡定数は大きくなる。
