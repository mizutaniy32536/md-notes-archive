# 110-253

1. 電位依存性 Ca2+ チャネルの機能の抑制
2. c︲アミノ酪酸 GABAB 受容体の刺激
3. オピオイド n 受容体の刺激
4. セロトニン ₅︲HT1B/1D 受容体の刺激
5. カルシトニン遺伝子関連ペプチド（CGRP）受容体の遮断
