# 110-211

（実務） 処方 ₂ の薬剤に関連したアドバイスとして、最も適切なのはどれか。1つ選べ。
1. 健康のため、青汁やクロレラなどを多めに摂取しましょう。
2. チーズの摂取は避けてください。
3. 便の色が黒くなったらすぐに医師や薬剤師に連絡してください。
4. 骨を丈夫にすると言われるビタミン K のサプリメントを、積極的に摂取しま しょう。
5. ナットウキナーゼは血栓を溶かすと言われるので、納豆を食べましょう。
