# 110-211

### 第110回薬剤師国家試験 問211

**問題文（全文）**  
86歳女性。高血圧症と2型糖尿病。フレイルによる通院困難のため、多職種の訪問による自宅でのケアと処方1による薬物治療を受けていた。訪問医から病院の循環器科での検査入院を勧められ、心房細動の診断を受け、ワルファリンカリウムによる治療が開始された。  
退院後、娘が以下の処方箋を持って薬局を訪れた。  

（処方箋の詳細：処方2にワルファリンカリウムを含む）  

患者は独居だが、近所に住む娘が食事や服薬の管理をしており、今後の在宅療養に合わせて再度アドバイスして欲しいと依頼があった。  
**問211（実務）**  
処方2の薬剤に関連したアドバイスとして、最も適切なのはどれか。１つ選べ。[2][3]

**全選択肢（番号付き）**  
1. 健康のため、青汁やクロレラなどを多めに摂取しましょう。  
2. チーズの摂取は避けてください。  
3. **便の色が黒くなったらすぐに医師や薬剤師に連絡してください。**  
4. 骨を丈夫にすると言われるビタミンKのサプリメントを、積極的に摂取しましょう。  
5. 納豆を食べましょう。[1][2]

**正解番号**  
**3**[1][7]

**解説**  
処方2の薬剤はワルファリンカリウムであり、ビタミンKエポキシド還元酵素を阻害して還元型ビタミンKの生成を抑制し、抗凝固作用を示す。[3]  
ワルファリン投与中はビタミンKを多く含む食品（納豆、クロレラ、青汁、ビタミンKサプリメント）の過剰摂取で薬効が減弱するため、選択肢1、4、5は不適切。[1][2]  
選択肢2のチーズはビタミンK含有量が少なく、ワルファリンとの関連が薄いため不適切。[1]  
選択肢3はワルファリンによる出血傾向（消化管出血など）を監視するための適切なアドバイスであり、便が黒くなる（タール便）は出血の兆候。[1][2]
