# 110-291

# 第110回薬剤師国家試験 問291

## 問題文

65歳男性。2型糖尿病の既往があり処方1の薬剤を服用している。約15年前にC型肝炎ウイルス（HCV）感染症と診断され、インターフェロン治療を受けたがウイルスは陰性化しなかった。

その後、C型慢性肝炎に対して処方2の薬剤を内服し経過観察中であったが、AST及びALTの軽度上昇が認められたため、直接作用型抗ウイルス薬（処方3）が追加された。

処方3追加時とその2ヶ月後受診時の検査値は下表のとおりであった。

検査結果を見た医師から、院内における採用薬の確認と治療薬の変更について、薬剤部に問合せがあった。

**検査値から判断し、処方3の薬剤を変更することにした。変更後の治療薬として、最も適切なのはどれか。1つ選べ。**[2][3]

## 全選択肢

1. ラミブジン[3]
2. グレカプレビル・ピブレンタスビル配合錠[3]
3. エンテカビル[3]
4. テノホビルアラフェナミド[3]
5. ソホスブビル・ベルパタスビル配合錠+リバビリン[3]

## 正解番号

**2**[1][2][7][8]

## 解説

処方3追加時とその2ヶ月後受診時の検査値から、**eGFRが29 mL/min/1.73 m²に低下**しており、**重度の腎機能障害**が確認されました。[2]

選択肢の検討：

- **選択肢1、3、4**：ラミブジン、エンテカビル、テノホビルアラフェナミドは**B型肝炎治療薬**であり、C型肝炎治療のための薬剤として不適切です。[1]

- **選択肢5**：ソホスブビル・ベルパタスビル配合錠は、**重度の腎機能障害（eGFR<30 mL/分/1.73 m²）に禁忌**です。[1]患者のeGFRが30を切っているため、不適切です。[1]

- **選択肢2**：グレカプレビル・ピブレンタスビル配合錠は、重度の腎機能障害がある患者でも使用可能であり、C型肝炎の直接作用型抗ウイルス薬として最も適切です。[1]
