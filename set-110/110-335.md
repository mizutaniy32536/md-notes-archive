# 110-335

**問題文**
63歳男性。数日前より全身倦怠感があり、血圧低下、体重減少が出現したため近医を受診し、大学病院を紹介された。外観上皮膚の色素沈着が認められた。検査の結果、コルチゾールの低値及びACTHの高値が認められ、アジソン病の確定診断を受け、以下の処方が開始となった。（入院時検査値）身長165 cm、体重53 kg、血圧99/59 mmHg、Na 134 mEq/L、K 5.3 mEq/L、空腹時血糖75 mg/dL、コルチゾール1.0 µg/dL、ACTH 1,796 pg/mLしかし、投薬開始後、低血圧及び低Na血症の改善が不十分であり、処方に追加する治療薬の検討が必要となった。この患者に追加する最も適切な治療薬はどれか。1つ選べ。

**正解:** 3

**解説**
問335　解答　３

アジソン病は、後天性の慢性副腎皮質機能低下症であり、副腎からの糖質コルチコイド、鉱質コルチコイドの分泌が低下し、それに伴うフィードバック機構により副腎皮質刺激ホルモン（ACTH）の分泌が増加する。一般に治療には、糖質コルチコイドと鉱質コルチコイド両方の作用を持つヒドロコルチゾンを投与する。ただし、ヒドロコルチゾンの投与でも鉱質コルチコイド欠乏症状（低血圧、低Na血症、高K血症）が改善しない場合は、強い鉱質コルチコイド作用を有するフルドロコルチゾンの投与を検討する。
本患者においても、ヒドロコルチゾンの投与後も低血圧及び低Na血症の改善が不十分であることから、追加するのに適切な治療薬はフルドロコルチゾンである。
