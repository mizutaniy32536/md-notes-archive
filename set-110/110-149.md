# 110-149

我が国の医薬分業について正しいのはどれか。2つ選べ。

1. いわゆる医薬分業率とは、全患者のうち投薬が必要とされた患者への処方件数 に対する院外処方箋枚数の割合である。
2. 都道府県による医薬分業率の地域差は、令和元年（₂₀₁₉ 年）以降認められなく なった。

3. かかりつけ薬局において薬歴管理を行うことにより、重複投薬や相互作用の有 無の確認ができ、薬物療法の有効性・安全性の向上が期待される。
4. 西洋の医療制度が導入された明治 ₂ 年（₁₈₆₉ 年）を医薬分業元年として、急速 に分業が進んだ。
5. 医師は、患者に必要な医薬品を病院・診療所にある医薬品に限定されることな く処方することができる。
