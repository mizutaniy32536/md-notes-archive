# 110-25

**問題文**
逆転層は大気汚染物質の滞留を引き起こす。盆地や谷間などの低地に冷たい空気が流入して発生するのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-25 を参照

**選択肢**
1. 沈降性逆転層
2. 地形性逆転層
3. 放射性逆転層
4. 移流性逆転層
5. 前線性逆転層

**正解:** 2

**解説**
解答　２

１　誤
沈降性逆転層は、高気圧圏内の上空、下降気流があるところにおいて、下降した空気が断熱圧縮により温度上昇し、下層空気の気温よりも上層空気の方が高くなることで発生する。

２　正
地形性逆転層は、夜間に冷却された空気が斜面に沿って盆地や谷間などの低地に流れ込むことで発生する。

３　誤
放射性逆転層は、日没後、地表面から急速な熱放射により地表面付近の気温が上層より低くなることで発生する。

４　誤
移流性逆転層は、風がある場合に冷却した地表面の上に暖かい空気が流れ込むことで発生する。

５　誤
前線性逆転層は、前線の移動に伴い、暖気が寒気の上層になることで発生する。
