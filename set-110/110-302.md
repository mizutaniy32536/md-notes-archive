# 110-302

# 第110回薬剤師国家試験 問302

## 問題文

75歳男性。慢性胃炎の既往がある。2年前に脳梗塞を発症し、それ以来、処方1及び処方2の薬剤を継続的に服用している。

咳と嗄声が続き、血痰を認めたため近医を受診し、胸部X線で右肺腫瘤を指摘された。総合病院呼吸器内科を紹介受診し、入院して精査した結果、StageⅣBの非小細胞肺がん（腺がん）と診断された。状態が安定したら退院し、処方1～3の薬剤での治療を継続する予定である。

**この患者の病態と治療に関する記述として、正しいのはどれか。2つ選べ。**

## 選択肢

1. 肺野部に発生することが多いがんである。
2. 発症前と同じ日常生活が制限なく行える。
3. 他臓器に遠隔転移している。
4. 手術での根治切除が可能である。
5. 腫瘍マーカーとして、SCC抗原（squamous cell carcinoma related antigen）が用いられる。

## 正解番号

**1、3**[1][7][10]

## 解説

**選択肢1：正解**
腺がんは肺野部（肺の周辺部）に発生することが多いがんです。[1]

**選択肢2：誤り**
StageⅣBは進行した肺がんであり、ECOG PSが1程度と考えられます。PS1では「肉体的に激しい活動は制限されるが、歩行可能、軽作業や座っての作業は可能」という状態です。したがって、発症前と同じ日常生活が制限なく行えるわけではありません。[1]

**選択肢3：正解**
StageⅣBは他臓器に遠隔転移している進行がんの分類です。[1]

**選択肢4：誤り**
StageⅣBのような遠隔転移がある進行がんでは、手術での根治切除は不可能です。

**選択肢5：誤り**
SCC抗原は扁平上皮がんのマーカーです。本症例は腺がんであるため、SCC抗原は用いられません。腺がんではCEAやCA19-9などのマーカーが用いられます。
