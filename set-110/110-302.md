# 110-302

（病態・薬物治療） この患者の病態と治療に関する記述として、正しいのはどれか。2つ選べ。
1. 肺野部に発生することが多いがんである。
2. 発症前と同じ日常生活が制限なく行える。
3. 他臓器に遠隔転移している。
4. 手術での根治切除が可能である。
5. 腫瘍マーカーとして、SCC 抗原（squamous cell carcinoma related antigen） が用いられる。
