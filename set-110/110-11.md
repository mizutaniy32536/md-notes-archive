# 110-11

**問題文**
血管平滑筋細胞を持たず血管内皮細胞と基底膜から構成され、栄養成分や老廃物の物質交換が行われるのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-11 を参照

**選択肢**
1. 大動脈
2. 細動脈
3. 毛細血管
4. 大静脈
5. 胸管

**正解:** 3

**解説**
解答　３

血管平滑筋細胞を持たず血管内皮細胞と基底膜から構成され、栄養成分や老廃物の物質交換を担うのは、毛細血管である。毛細血管は動脈と静脈の間に位置し、身体中の組織細胞に網の目状に分布し、細胞に必要な酸素や栄養を供給し、また不要となった二酸化炭素や老廃物を回収する。
