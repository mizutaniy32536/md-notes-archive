# 110-84

災害派遣医療チーム（DMAT）が最も優先すべき活動はどれか。１つ選べ。

１ 広域医療搬送対象患者の選出
２ 心的外傷後ストレス障害（PTSD）の診療
３ 長期的な医療支援
４ 一般用医薬品の販売
５ 被災地の復興支援

**正解:** １

**解説**
災害派遣医療チーム（DMAT）は、災害の急性期（概ね48時間以内）に活動できる機動性を持った、専門的な研修・訓練を受けた医療チームであり、選択肢の中でDMATが最も優先すべき活動は、広域医療搬送対象患者の選出である。
災害の急性期では、被災地の医療需要の増加に対して医療供給が不足するため、被災地での対応が困難な重症患者を被災地の外に搬送して治療を行うこと（広域医療搬送）が重要である。
