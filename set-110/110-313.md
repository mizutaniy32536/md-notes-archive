# 110-313

### 第110回薬剤師国家試験 問313

**問題文**  
保険薬局における薬剤服用歴と調剤録（注）に関する内容として、正しいのはどれか。**2つ選べ**。  
（注）薬剤師法で規定されている調剤録[1][2][3]

**選択肢**  
1. 調剤後、患者の薬剤の使用状況を継続的に把握し、その際指導した内容は、処方箋が調剤済みの場合、調剤録への記入事項に該当しない。[2]  
2. 調剤録は、調剤した薬剤師ではなく、薬局の管理者が記載しなければならない。[2][3]  
3. 薬局の開設者は、調剤録を備え、最終記載日から3年間保存する義務がある。[1][2]  
4. （詳細未記載だが、誤りとされる選択肢。例: 薬剤服用歴の記録・管理の実施は、薬局の義務であり、調剤報酬の対象とはならない。[3]）  
5. （詳細未記載だが、正解選択肢。薬担規則第8条2項関連。[1]）

**正解番号**  
**3, 5**[1][5][6]

**解説**  
- **選択肢3**: 正しい。薬剤師法第28条により、薬局開設者は調剤録を薬局に備え付け、最終記載日から3年間保存する義務がある[1][2]。  
- **選択肢5**: 正しい。薬担規則第8条2項の内容で、関連する正しい記述[1]。  
- **その他の選択肢**:  
  - 選択肢1: 誤り。調剤後継続的な把握・指導内容は調剤録に記入すべき事項に該当[2]。  
  - 選択肢2: 誤り。調剤録は調剤行為を行った薬剤師が記載する[2]。  
  - 選択肢4: 誤り（詳細により異なるが、全体として不正解）[3]。  
本問は薬剤師法・薬担規則に基づく保険薬局の記録義務を問う。問312と共通の症例（68歳男性、高血圧・心不全、処方変更後フォローアップ記載）を使用[1][2][3]。
