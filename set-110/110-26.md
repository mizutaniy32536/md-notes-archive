# 110-26

**問題文**
モルモット摘出回腸標本の収縮反応について、アセチルコリン単独による濃度－反応曲線（破線）と、薬物X存在下でのアセチルコリンによる濃度－反応曲線（実線）を作成したところ、下図のようになった。薬物Xはどれか。1つ選べ。ただし、アセチルコリン単独により生じる最大収縮を100％とした。また、薬物のXの単独では収縮反応が生じなかった。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 110-26 を参照

**選択肢**
1. ヒスタミン
2. アトロピン
3. ピリドスチグミン
4. パパベリン
5. ベタネコール

**正解:** 4

**解説**
解答　４

アセチルコリンは、回腸のムスカリンM3受容体を刺激することで回腸を収縮させる。グラフからアセチルコリン単独では回腸を収縮させる反応率が最大約100％であるが、薬物Xを併用したところ最大反応率は約50％と頭打ちとなるため、薬物Xはアセチルコリンに対する非競合的阻害薬であるパパベリンが該当する。

１　誤
ヒスタミンは、ヒスタミンH1受容体を刺激することで、回腸を収縮させるため、アセチルコリンに対して協力作用を示す。

２　誤
アトロピンは、ムスカリン受容体遮断作用により回腸を弛緩させるため、アセチルコリンの競合的阻害薬である。

３　誤
ピリドスチグミンは、コリンエステラーゼを阻害し、アセチルコリンの分解を抑制することで、回腸を収縮させるため、アセチルコリンに対して協力作用を示す。

４　正
パパベリンは、ホスホジエステラーゼ阻害によりcAMPの分解を抑制することで、回腸を弛緩させるため、アセチルコリンの非競合的阻害薬である。

５　誤
ベタネコールは、ムスカリン受容体刺激作用により回腸を収縮させるため、アセチルコリンに対して協力作用を示す。
