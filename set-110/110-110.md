# 110-110

**第110回薬剤師国家試験 問110**

以下の漢方処方が示す作用と、それに主に関与する構成生薬との組合せのうち、誤っているのはどれか。1つ選べ。[1][4][5]

**選択肢**  
1. 六君子湯の**補気作用** ニンジン  
2. 桂枝茯苓丸の**駆瘀血作用** トウニン  
3. 加味逍遙散の**補血作用** トウキ  
4. 葛根湯の**発汗作用** シャクヤク  
5. 五苓散の**利水作用** ブクリョウ[1]

**正解番号：4**[1][4]

**解説**  
- 1は正しい：六君子湯の補気作用にニンジンが関与する。[1][5]  
- 2は正しい：桂枝茯苓丸の駆瘀血作用にトウニンが関与する。[1]  
- 3は正しい：加味逍遙散の補血作用にトウキ（当帰）が関与し、血虚を改善する。[1][4]  
- 4は誤り：葛根湯の発汗作用に主に関与するのは**ケイヒ（桂皮）、マオウ（麻黄）、ショウキョウ**で、シャクヤク（芍薬）は筋肉の緊張緩和などに寄与するが発汗には関与しない。[1][4]  
- 5は正しい：五苓散の利水作用にブクリョウが関与する。[1]
