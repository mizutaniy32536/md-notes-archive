# 第105回 問205

**第105回薬剤師国家試験 問205（実務）**

**問題文（前問の問204と共通の導入部含む）**  
細胞膜などの半透膜を介した水の移動は、膜で隔てられた溶液の浸透圧の差によって起こる。浸透圧の原理を利用した薬物も用いられている。  
**前問の浸透圧の原理に基づいた水の動きを利用した薬物はどれか。1つ選べ。**[1][2][3]

**全選択肢**  
1. 酸化マグネシウム  
2. ナルデメジン  
3. センノシダ  
4. ヒマシ油  
5. ビサコジル[1][2][3]

**正解番号**  
1[2]

**解説**  
酸化マグネシウムは腸管内で不溶性の酸化マグネシウムが水に溶けにくいため高浸透圧を生じ、腸管内に水を引き込んで便を軟化させる**浸透性下剤**である。これが浸透圧の原理に基づいた水の動きを利用した機序である。[1][2]  
- 2. 誤：ナルデメジンは末梢μオピオイド受容体アンタゴニストとしてオピオイド誘発性便秘を改善する。[1]  
- 3. 誤：センノシドは腸内細菌で加水分解されレインアントロンとなり、腸粘膜やアウエルバッハ神経叢を刺激して蠕動運動を促進する**大腸刺激性下剤**。[1][2]  
- 4. 誤：ヒマシ油はリシン油酸により小腸を刺激する**小腸刺激性下剤**。[1][2]  
- 5. 誤：ビサコジルは腸粘膜を刺激する**大腸刺激性下剤**。[2][1]
