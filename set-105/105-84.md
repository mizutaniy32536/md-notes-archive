# 第105回 問84（薬剤）

**問題文**
日本薬局方で規定されている温度の範囲として正しいのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-84 を参照

**選択肢**
1. 常温とは10〜30℃である。
2. 室温とは1〜30℃である。
3. 微温とは20〜30℃である。
4. 冷水とは20℃以下の水である。
5. 温湯とは40〜50℃の水である。

**正解:** 2

**解説**
解答　２

日本薬局方で規定されている温度の範囲は下記の通りである。

・常温とは15〜25℃である。
・室温とは1〜30℃である。
・微温とは30〜40℃である。
・冷水とは10℃以下の水である。
・温湯とは60〜70℃の水である。
