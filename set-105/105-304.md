# 第105回 問304（実務）

**問題文**
70歳女性。糖尿病の既往歴はない。非小細胞肺がん（扁平上皮がん、PD−L1発現率15％）と診断され、ニボルマブ点滴静注240 mg、2週間毎の投与が開始された。経過良好だったが、6回目の投与後、自宅で強い倦怠感、食欲低下、口渇と多尿が出現し、水分摂取も困難であったため、緊急受診した。受診時、朝食をとらずに行った検査値は以下の通りである。検査値Na 135 mEq/L、Cl 96 mEq/L、K 5.4 mEq/L、 BUN 23 mg/dL、HbA1c 6.0％（NGSP値）、血糖値571 mg/dL、血液pH 7.1、尿糖4＋、尿ケトン3＋問304（病態・薬物治療）この患者に関する記述のうち、正しいのはどれか。２つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-304 を参照

**選択肢**
1. Tリンパ球の機能が低下している。
2. 血中C−ペプチドが低値である。
3. 次回以降のニボルマブの投与を中止すべきである。
4. インフュージョンリアクションが起きている。
5. 2型糖尿病を発症している。問305（実務）この患者に実施されるべき治療の目的と薬物治療の組合せとして適切なのはどれか。２つ選べ。

**正解:** 2、3

**解説**
問304　解答　２、３

ニボルマブの重大な副作用に1型糖尿病があり、血糖値571 mg/dLと高値であること、強い倦怠感、食欲低下、口渇と多尿といった所見を認めていることから、本患者はニボルマブの副作用である1型糖尿病を発症したと考えられる。また、尿糖4＋、尿ケトン3＋、血液pH 7.1の所見から１型糖尿病に伴う糖尿病性ケトアシドーシスも生じている。そのため、ニボルマブの投与を中止し、糖尿病性ケトアシドーシスの処置としてインスリン製剤の投与などを行う必要がある。

１　誤
ニボルマブは、ヒト型抗ヒトPD−1モノクローナル抗体製剤であり、PD−1とそのリガンドであるPD−L1及びPD−L2との結合を阻害し、がん抗原特異的なT細胞の増殖、活性化及び細胞障害活性の増強などを示す。よって、本患者のTリンパ球の機能は亢進していると考えられる。

２　正
冒頭解説文参照。1型糖尿病を発症している可能性が高いため、インスリン分泌能を反映する血中C−ペプチドは低値であると考えられる。

３　正
冒頭解説文参照。

４　誤
インフュージョンリアクションとは、モノクローナル抗体製剤投与後に認められるアナフィラキシー様症状などのことであり、代表的な症状として悪寒、発熱、頭痛などを認める。本患者の所見よりインフュージョンリアクションが生じているとは考えにくい。

５　誤
冒頭解説文参照。

問305　解答　１、２

本患者は、ニボルマブ投与に伴う重大な副作用である1型糖尿病、それに伴う糖尿病性ケトアシドーシスを発症している。そのため、治療として、高血糖の改善に速効型インスリン製剤、高血糖の伴う脱水及び電解質欠乏の改善に生理食塩水を第一選択とし、その他、諸症状の改善を目的に薬物治療を行う。

１　正
冒頭解説文参照。

２　正
冒頭解説文参照。

３　誤
糖尿病性アシドーシスの補正のため、インスリン注射が用いられ、効果不十分の際は、炭酸水素ナトリウムなどのアルカリ化薬が用いられることがある。

４　誤
本患者は、高ナトリウム血症ではないと考えらえる。本患者のような高血糖時には、血漿浸透圧が上昇することで細胞内から水分が移動するため、血清ナトリウム値は正常域〜低値を示す。

５　誤
解説3参照。なお、電解質バランスの改善には生理食塩液が用いられる。
