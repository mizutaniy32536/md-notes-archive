# 第105回 問199

### 第105回薬剤師国家試験 問199

**科目名**: 物理・化学・生物（実践問題）

**問題文全文**:  
（問198と共通の症例文）  
60歳男性。骨折治療のため入院中。逆流性食道炎のため、1ヶ月前からオメプラゾール腸溶錠20 mgを1日1回1錠、朝食後に服用している。患者の服薬アドヒアランスは良好であったが、症状の改善がみられなかった。  
そのため、医師から他に有効なプロトンポンプ阻害薬（PPI）がないか薬剤師に相談があった。薬剤師がPPIと薬物代謝酵素CYP2C19に関する文献などを調べたところ、図のデータを見つけた。  
（図：CYP2C19遺伝子型によるPPIの血中濃度データ。オメプラゾールはPMで高濃度、EMで低濃度。エソメプラゾールはS体のみでPMでも高い効果。詳細は公式問題PDF参照）[1][2][5]  
エソメプラゾールは、オメプラゾールの光学異性体のS体のみの薬物である。  
問199（物理・化学・生物）  
図のデータから、オメプラゾール腸溶錠20mg 1日1回投与で症状改善がみられなかった理由として、最も適切なのはどれか。１つ選べ。[1][2]

**全選択肢**:  
１．オメプラゾールは粉砕して服用したため、腸溶衣が機能せず効果が低下した。  
２．患者はHomo-EM型であり、オメプラゾールが速やかに代謝されて血中濃度が低かった。  
３．患者はHetero-EM型であり、オメプラゾールが速やかに代謝されて血中濃度が低かった。  
４．患者はPM型であり、オメプラゾールが過剰に代謝されて血中濃度が低かった。  
５．朝食後投与のため、胃酸によりオメプラゾールが分解されて血中濃度が低かった。[1][2][7][8]

**正解番号**: ２[2]

**解説**:  
図のデータから、オメプラゾールはCYP2C19の**Homo-EM**（ホモ接合型広代謝者）で血中濃度が著しく低く、治療効果が不十分となる。患者は服薬アドヒアランス良好で1ヶ月継続したが改善せず、Homo-EM型の可能性が高い。PM（poor metabolizer）は逆に高濃度、Hetero-EMは中間。エソメプラゾールはS体のみのため、遺伝子型に関わらず安定した効果が期待できる（問198と連動）。腸溶錠は粉砕不可（構造式から酸不安定）が理由ではない（問199の誤答誘導）。[1][2][7][8]
