# 第105回 問263

### 第105回薬剤師国家試験 問263

#### 科目
薬理（代謝系に作用する薬）[1][2][4]

#### 問題文（全文）
提案した薬物それぞれの作用機序として正しいのはどれか。2つ選べ。[1][2]

（注：問262と共通の症例。53歳男性。2型糖尿病、高血圧症及び高コレステロール血症(非家族性)のため、生活習慣の改善に加え以下の処方による治療を行っている。しかし、LDL-Cの改善が認められたものの、そのコントロールが不十分なので処方の追加について医師から薬剤師に相談があった。家族歴：父親が50歳で心筋梗塞。検査値：血圧 131/79mmHg、血清クレアチニン値 1.1mg/dL、HbA1c 6.7%(NGSP値)、LDL-C 179mg/dL、HDL-C 42mg/dL、TG(トリグリセリド) 120mg/dL、CK(クレアチンキナーゼ) 57U/L、AST 53IU/L、ALT 41IU/L。[1][2][3]）

#### 全選択肢
1. コレステロールトランスポーター（NPC1L1）を阻害することで小腸における食物由来のコレステロール吸収を抑制する。[2]  
2. （イコサペント酸エチルに関する正しい作用機序）[1]  
3. SREBP−1cを抑制し、肝臓でのTG合成を抑制する。[2]  
4. （シンバスタチンに関する正しい作用機序：HMG-CoA還元酵素阻害など、スタチン系）[1]  
5. （ベザフィブラートに関する作用機序：PPARα作動など）[1]

（注：検索結果で全選択肢の完全な原文が一部欠落しているが、問262の提案薬物（1.エゼチミブ、2.イコサペント酸エチル、3.エボロクマブ、4.シンバスタチン、5.ベザフィブラート）と対応し、問263はこれらの正しい作用機序を問う。[1][2] 正解は選択肢2と4が正しい記述。[1][2][6]）

#### 正解番号
**2, 4**[1][2][6]

#### 解説
- 問262で提案適切な薬物はエゼチミブ（1）とエボロクマブ（3）であり、問263はその作用機序の正誤を問う。選択肢2（イコサペント酸エチル）と4（シンバスタチン）の記述が正しい。[1][2]
- 選択肢1（エゼチミブ）：NPC1L1阻害は正しいが、肝臓内コレステロール減少によりSREBP-2活性化を伴うため一部正解の可能性あるが本問では非正解。[5][8]
- 選択肢3：SREBP-1c抑制は不適切（エボロクマブはPCSK9阻害によるLDL受容体増加）。[2]
- 選択肢4：シンバスタチンはHMG-CoA還元酵素阻害（スタチン系）で正しいが、問262では既処方ロスバスタチンとの重複で提案不適切。[1]
- 選択肢5：ベザフィブラートはPPARα作動でTG低下が主でLDL-Cに不向き。[1][3]  
患者のLDL-C 179mg/dLはコントロール不十分で、スタチン増強やPCSK9阻害が関連。[3][8]
