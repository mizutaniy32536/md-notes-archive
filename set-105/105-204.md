# 第105回 問204（実務）

**問題文**
細胞膜などの半透膜を介した水の移動は、膜で隔てられた溶液の浸透圧の差によって起こる。浸透圧の原理を利用した薬物も用いられている。問204（物理・化学・生物）膜により隔てられた二つの水相A、Bの浸透圧ΠA、ΠBと水の移動する方向を表す模式図として正しいのはどれか。１つ選べ。なお、図中の●は水に溶けている溶質で、その数の違いは濃度の大小を表し、矢印は水の移動する方向を表す。問205（実務）前問の浸透圧の原理に基づいた水の動きを利用した薬物はどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-204 を参照

**選択肢**
1. 酸化マグネシウム
2. ナルデメジン
3. センノシド
4. ヒマシ油
5. ビサコジル

**正解:** 2

**解説**
問204　解答　２

浸透現象とは、濃度の異なる溶液を半透膜で隔てたとき、溶液の濃度を均一にしようと濃度の低い溶液から濃度の高い溶液に向かって溶媒分子が移動する現象である。
したがって、本問においては浸透現象により、水相B（低濃度）の溶媒が水相A（高濃度）の方へ移動する。
また、浸透圧は、溶質が多いほど濃度の高い溶液となるため高くなる。そのため、水相A（高濃度）の方が水相B（低濃度）に比べて高くなる（ΠA＞ΠB）。

問205　解答　１

本問の選択肢の薬物はすべて緩下薬（下剤）であるが、その中で浸透圧の原理に基づいた水の動きを利用したものは酸化マグネシウムである。

１　正
酸化マグネシウムは、腸内で難吸収性の重炭酸塩となり、腸管内浸透圧を高める。このため、組織側から腸管内へ水分が引き寄せられ、腸内容物を膨張・軟化させることで緩下作用を示す。

２　誤
ナルデメジンは、消化管のオピオイド受容体に結合し、オピオイド鎮痛薬に拮抗することにより消化管運動および消化管神経活動を活性化し、緩下作用を示すため、オピオイド誘発性便秘症に用いられる。

３　誤
センノシドは、腸内細菌の作用によりレインアンスロンを遊離し、大腸の蠕動運動を亢進することで緩下作用を示す。

４　誤
ヒマシ油は、小腸内でリパーゼにより、リシノール酸とグリセリンに分解され、リシノール酸が小腸を刺激し、蠕動運動を亢進させることで緩下作用を示す。

５　誤
ビサコジルは、結腸・直腸の粘膜に作用して蠕動運動を促進し、腸粘膜への直接作用により排便反射を高め緩下作用を示す。
