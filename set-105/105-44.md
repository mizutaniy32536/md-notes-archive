# 第105回 問44（生物）

**問題文**
CYP3A4の活性を不可逆的に阻害するのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-44 を参照

**選択肢**
1. イソニアジド
2. エリスロマイシン
3. シメチジン
4. ファモチジン
5. リファンピシン

**正解:** 2

**解説**
解答　２

選択肢のうち、CYP3A4の活性を不可逆的に阻害するのは、エリスロマイシンである。
エリスロマイシンなどのマクロライド系抗生物質は、CYP3A4により代謝を受ける際に生成する中間体、または代謝物自身が、CYP3A4と共有結合し、解離しにくい複合体を形成してCYP3A4を不可逆的に不活化する。なお、シメチジンは、構造中にイミダゾール骨格を有し、CYPのヘム鉄に配位結合することでCYPの活性を可逆的に阻害する。
