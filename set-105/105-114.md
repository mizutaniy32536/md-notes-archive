# 第105回 問114（化学）

**問題文**
下図のヌクレオチドに関する記述のうち、正しいのはどれか。２つ選べ。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-114 を参照

**選択肢**
1. RNAの構成成分である。
2. デオキシウリジン5´−一リン酸のメチル化により生成する。
3. 塩基のメチル基の導入には、ピリドキサールリン酸が補酵素として必要である。
4. 塩基部分の生合成には、アスパラギン酸及びグルタミンが利用される。
5. 分解されて生じた塩基は酸化されて尿酸となる。

**正解:** 2、4

**解説**
解答　２、４

１　誤
本問の構造はデオキシチミジン5´−一リン酸（dTMP：チミジル酸）であり、RNAの構成成分ではない。デオキシチミジン5´−一リン酸などの2–デオキシリボースを含むデオキシリボヌクレオチドは、主にDNAの構成成分となる。また、RNAの構成成分は、リボースを含むリボヌクレオチドである。

２　正
デオキシチミジン5´−一リン酸は、デオキシウリジン5´−一リン酸の構造中のウラシルの5位がメチル化されることにより生成する。

３　誤
デオキシウリジン5´−一リン酸の構造中の塩基をメチル化する際には、チミジル酸シンターゼが用いられ、補酵素としてN 5,N 10–メチレンテトラヒドロ葉酸がメチル基供与体として働く。テトラヒドロ葉酸は葉酸の還元型であり、一炭素単位の転移反応などに関与する。なお、ピリドキサールリン酸はビタミンB6の活性型であり、アミノ基転移反応などのアミノ酸代謝に関与する。

４　正
本問のデオキシチミジン5´−一リン酸の塩基はチミンであり、ピリミジン塩基に該当する。ピリミジン塩基部分の生合成には、アスパラギン酸及びグルタミンと二酸化炭素より合成されたカルバモイルリン酸が利用される。

５　誤
デオキシチミジン5´−一リン酸が分解されて生じた塩基（チミン）は代謝されることで、β−アミノイソ酪酸、尿素及びCO2になる。なお、分解の最終産物が尿酸となるのは、アデニル酸やグアニル酸などが分解されてできるプリン塩基である。
