# 第105回 問91（物理）

**問題文**
日本薬局方において、L−エチルシステイン塩酸塩の純度試験は以下のように規定されている。この純度試験に関する記述のうち、正しいのはどれか。２つ選べ。純度試験（1）硫酸塩　本品0.6 gをとり、試験を行う。比較液には0.005 mol/L 　ア　 0.35 mLを加える（0.028％以下）。（2）重金属　本品1.0 gをとり、第1法により操作し、試験を行う。比較液には 　イ　 標準液1.0 mLを加える（ 　ウ　 ppm以下）。ただし、重金属試験法第1法では、医薬品各条に規定する量の試料をネスラー管にとり、水適量に溶かし、40 mLとする。これに希酢酸2 mL及び水を加えて50 mLとし、検液とする。比較液は、医薬品各条に規定する量の 　イ　 標準液をネスラー管にとり、希酢酸2 mL及び水を加えて50 mLとする。また、　イ　 標準液1.0 mL中には 　イ　 0.01 mgが含まれる。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-91 を参照

**選択肢**
1. 硫酸塩試験法においては、検液及び比較液に、2,2’−ビピリジル試液2 mLずつを加えて混和し、黒色の背景を用いて混濁を比較する。
2. 重金属試験法においては、検液及び比較液に、硫化ナトリウム試液1滴ずつを加えて混和し、白色の背景を用いて液の色を比較する。
3. ア　 は塩化バリウムである。
4. イ　 は鉛である。
5. ウ　 に入る数値は100である。

**正解:** 2、4

**解説**
解答　２、４

日本薬局方においてL−エチルシステイン塩酸塩の純度試験は、硫酸塩試験法や重金属試験法などを行うと規定されている。
重金属試験法は、医薬品中に混在する重金属の限度試験である。この重金属とは、酸性で硫化ナトリウム試液によって呈色する金属性混在物をいい、その量は鉛（Pb）の量として表す。医薬品各条には、重金属（Pbとして）の限度をppmで（　）内に付記する。
硫酸塩試験法は、医薬品中に混在する硫酸塩の限度試験である。医薬品各条には、硫酸塩（SO4として）の限度をパーセント（%）で（　）内に付記する。

１　誤
硫酸塩試験法においては、検液及び比較液に、塩化バリウム試液2 mLずつを加えて混和し、黒色の背景を用いて混濁を比較する。

２　正
重金属試験法においては、検液及び比較液に、硫化ナトリウム試液1滴ずつを加えて混和すると、重金属の濃度に比例して暗系色を呈するため、その濃さを白色の背景を用いて比較する。

３　誤
アは硫酸である。日本薬局方において、L−エチルシステイン塩酸塩の硫酸塩試験法は、次のように規定されている。「本品0.6 gをとり、試験を行う。比較液には、0.005 mol/L硫酸0.35 mLを加える（0.028%以下）。

４　正
イは鉛である。日本薬局方において、L−エチルシステイン塩酸塩の重金属試験法は、次のように規定されている。本品1.0 gをとり、第1法により操作し、試験を行う。比較液には鉛標準液1.0 mLを加える（10 ppm以下）。

５　誤
ウは10である。比較液に含まれる重金属が鉛0.01 mgであるため、本品1.0 g中に含まれる重金属は鉛に換算すると0.01 mg以下となる。したがって、0.01 mg/1.0 g＝10 µg/1.0 g＝10 ppmとなる。
