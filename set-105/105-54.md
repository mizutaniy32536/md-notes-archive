# 第105回 問54（衛生）

**問題文**
図に示す構造を有し、医療用医薬品として用いられる製造の適用部位はどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-54 を参照

**選択肢**
1. 口腔
2. 子宮
3. 直腸
4. 皮膚
5. 眼

**正解:** 1

**解説**
解答　１

設問中の図は、トリアムシノロンアセトニドを有効成分として口内炎治療に用いる医療用医薬品として製造された、口腔粘膜に適用する付着錠である。したがって、適用部位は口腔である。なお、本剤の付着層に用いられているヒドロキシプロピルセルロースやカルビキシビニルポリマーは、水に溶解し粘稠性を示す親水性高分子であり、患部に製剤を付着させる目的で使用されている。
