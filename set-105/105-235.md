# 第105回 問235（実務）

**問題文**
（105-234の続き）この感染症（HIV/AIDS）の治療薬に含まれる成分として正しいのはどれか。1つ選べ。

**選択肢**
1. バラシクロビル塩酸塩水和物
2. アジスロマイシン水和物
3. ミノサイクリン塩酸塩
4. エルビテグラビル
5. ソホスブビル

**正解:** 4

**解説**

1. ❌ **誤** — バラシクロビルは抗ヘルペスウイルス薬。体内でアシクロビルに変換され、ウイルス由来チミジンキナーゼによりリン酸化→アシクロビル三リン酸としてDNAポリメラーゼを阻害。

2. ❌ **誤** — アジスロマイシンはマクロライド系抗生物質。細菌リボソーム50Sサブユニットに結合し、タンパク質合成を阻害。

3. ❌ **誤** — ミノサイクリンはテトラサイクリン系抗生物質。70Sリボソームの30Sサブユニットに結合し、タンパク質合成を阻害。

4. ✅ **正** — エルビテグラビルはインテグラーゼ阻害薬。HIV−DNAの宿主DNAへの組み込みを抑え、プロウイルス形成・ウイルス増殖を阻止する。

5. ❌ **誤** — ソホスブビルはC型肝炎ウイルス（HCV）治療薬。NS5B RNA依存性RNAポリメラーゼを阻害しHCVの増殖を抑制。
