# 第105回 問312

### 第105回薬剤師国家試験 問312（法規・制度・倫理）

**問題文（全文）**  
46歳男性。2年前に甲状腺全摘出手術を受けた後、レボチロキシンナトリウム錠内服による薬物治療を行なっている。通院間隔が6ヶ月に一度に変更になり、180日分の処方箋を持って来局した。  
この患者の薬剤服用歴を確認すると、過去に服用忘れや、自己判断で服用を中断していた可能性が疑われた。長期処方への変更に伴い、薬剤師が服薬アドヒアランスに関連した注意事項を説明することになった。  
（注）PECOやPICOは疑問を構造化するための手法の1つ。PはPatient、EはExposure、IはIntervention、CはComparison、OはOutcomeの頭文字のこと。  
作成したPECO又はPICOに従って、実際に介入研究を行うことになった。この研究を実施するにあたり薬剤師が注意すべき点として、適切なのはどれか。**２つ選べ。**[2][4][7][9]

**全選択肢**  
1. 薬剤服用歴から、調剤した薬剤種類数を調べる。  
2. 患者に持参してもらった残薬数から服薬率を算出する。  
3. 患者から飲み忘れの有無を聞き取る。  
4. 処方した医師に処方意図を確認する。  
5. 併用薬との相互作用の有無を調べる。[2][4][9]

**正解番号**  
**2, 3**[2][4][6][9]

**解説**  
PECO/PICOに基づく介入研究（服薬アドヒアランス改善のための電話フォローアップ）を実施する際、服薬率の客観的・主観的評価として、患者持参の残薬数から服薬率を算出（選択肢2）し、飲み忘れの有無を直接聞き取る（選択肢3）が適切。選択肢1は調剤実績の確認に留まり、服薬実態を反映しない。選択肢4は研究外の医師確認、選択肢5は相互作用調査でアドヒアランス評価に直接関連しない。[2][4][7][9]
