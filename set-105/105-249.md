# 第105回 問249

# 第105回薬剤師国家試験 問249

**科目：薬理**

## 問題文

55歳男性。コンピューター関連企業に勤務しており、勤務時間中は長時間コンピューターの画面を見ることが多い。1年前、目のかすみや視野がぼやけることがあり眼科を受診したところ、緑内障と診断され処方1にて治療を行っていた。今回の受診の際、眼圧が高くなっていることを指摘され、処方2が追加となった。

薬剤師による適切な指導内容の根拠として正しいのはどれか。1つ選べ。[1]

## 全選択肢

1. ぶどう膜強膜流出経路からの眼房水流出効果が持続する。[1]

2. 虹彩や眼瞼への色素沈着が起こる。[1]

3. 血漿浸透圧の上昇により眼房水産生が増加する。[1]

4. アドレナリンβ受容体遮断効果が全身に及ぶ。[2]

5. 眼房水産生に対する抑制効果が持続する。[2]

## 正解番号

**4番**[2]

## 解説

問248で選択された指導内容「点眼直後に息苦しい感じがあったら、すぐに医師又は薬剤師に連絡すること」は、処方2がアドレナリンβ受容体遮断薬（ベータブロッカー）であることに基づいています。この薬剤のアドレナリンβ受容体遮断効果が全身に及ぶことで、呼吸困難などの全身副作用が起こる可能性があるため、この指導が必要となります。[2]
