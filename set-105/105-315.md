# 第105回 問315

# 第105回薬剤師国家試験 問315

## 科目名
実務

## 問題文
薬剤師が「禁煙をサポートしましょうか」と声をかけたところ、「禁煙は初めてだけどやってみます」と回答があった。この男性への薬剤師の対応として、適切でないのはどれか。1つ選べ。[1][2][3]

## 全選択肢
1. 禁煙開始日を決めるよう伝える。
2. タバコを吸いたくなったときの対策について一緒に考える。
3. 定期的にフォローアップを行う。
4. タバコには離脱症状がないと伝える。
5. 公的医療保険を使って禁煙外来での治療が受けられる場合があると伝える。[1][2][3]

## 正解番号
**4**[1][2][3]

## 解説
薬剤師が患者の禁煙をサポートする際、選択肢1～3および5は適切な対応です。禁煙開始日の設定、吸いたくなったときの対策検討、定期的なフォローアップ、そして禁煙外来の保険診療の情報提供は、いずれも効果的な禁煙サポートの要素です。

一方、選択肢4の「タバコには離脱症状がないと伝える」は適切ではありません。[1]禁煙すると、イライラするといった離脱症状が見られるためです。患者に対して誤った情報を提供することは、禁煙継続の意思を損なわせる可能性があり、薬剤師としての対応として不適切です。
