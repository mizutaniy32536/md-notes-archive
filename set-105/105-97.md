# 第105回 問97（物理）

**問題文**
以下の記述は、L−バリン（C5H11NO2）の薄層クロマトグラフィー（TLC）に関するものである。本品0.10 gを水25 mLに溶かし、試料溶液とする。試料溶液5 µLをTLC用シリカゲルを用いて調製した薄層板にスポットする。次に1−ブタノール/水/酢酸（100）混液（3：1：1）を展開溶媒として約10 cm展開した後、薄層板を80℃で30分間乾燥する。これに試薬 　A　 のアセトン溶液（1→50）を均等に噴霧した後、80℃で5分間加熱する。このクロマトグラフィーに関する記述のうち、正しいのはどれか。２つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-97 を参照

**選択肢**
1. このクロマトグラフィーの分離モードはイオン交換である。
2. 試薬 　A　 はニンヒドリンである。
3. L−バリンのスポットは黄色を呈する。
4. 試料にL−ロイシン（C6H13NO2）が混在するとき、そのRf値はL−バリンのRf値より小さい。
5. この試験でL−バリンのRf値より大きなRf値を与える不純物は、逆相分配クロマトグラフィーにおいては保持時間が一般にL−バリンより大きくなる。

**正解:** 2、5

**解説**
解答　２、５

１　誤
固定相にTLC用シリカゲルを用いて薄層板を調製しているため、このクロマトグラフィーの分離モードは吸着クロマトグラフィー（順相クロマトグラフィー）である。

２　正
ニンヒドリンはアミノ酸の呈色試薬であり、本問のL−バリン（アミノ酸）を検出するために加える。

３　誤
アミノ酸やペプチドなどの第一級アミノ基を含む医薬品に、ニンヒドリン試薬を加えると青〜紫色を呈する。なお、L−バリン（アミノ酸）のスポットは、紫色を呈する。

４　誤
本問では固定相にシリカゲルを用いているため、極性の低い物質は固定相に保持されにくく、移動度が大きくなるため、下記のRf値は大きくなる。

したがって、試料にL−ロイシン（C6H13NO2）が混在するとき、L−ロイシン（C6H13NO2）の方がL−バリン（C5H11NO2）に比べて炭素数が多く極性が低いため、固定相に保持され
