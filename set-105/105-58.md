# 第105回 問58（衛生）

**問題文**
横紋筋融解症で高値を示す血液検査所見はどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-58 を参照

**選択肢**
1. 白血球数
2. 血小板数
3. ビリルビン値
4. アルブミン値
5. ミオグロビン値

**正解:** 5

**解説**
解答　５

横紋筋融解症とは、薬物の副作用や外傷によって、骨格筋細胞の壊死、融解が生じた状態をいう。骨格筋細胞の壊死、融解に伴い筋細胞内成分のミオグロビンやクレアチンキナーゼ（CK）が血液中に流出するため、血液検査所見にて、血中ミオグロビン値や血中CK値が高値を示す。
