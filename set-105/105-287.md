# 第105回 問287

# 第105回薬剤師国家試験 問287

## 科目名
実務（病院薬学）

## 問題文
処方変更後3日目には咳は軽快し解熱傾向を認めたものの、37.5℃前後の微熱が継続している。薬剤師が今後の治療方針を医師と確認した。その内容として最も適切なのはどれか。1つ選べ。[1][4]

## 全選択肢
1. ロキソプロフェンナトリウム水和物の追加投与
2. フルコナゾールへの変更
3. 現在の処方薬による14日間の治療完遂
4. テオフィリンの追加投与
5. インフルエンザウイルス感染の追加検査[1][4]

## 正解番号
**3**[1][2][4]

## 解説

**選択肢1について**
解熱傾向にあり、さらに解熱の必要性はないと考えられます。[1]

**選択肢2について**
抗真菌薬に変更する理由は見当たりません。[1]

**選択肢3について**
妥当な記述です。本症例はマイコプラズマ肺炎と考えられ、細胞壁を有していないため**βラクタム系抗生物質が無効**です。[2]セフカペンピボキシル塩酸塩から変更された処方薬（マクロライド系抗生物質）による14日間の治療完遂が標準的な治療方針となります。[1]
