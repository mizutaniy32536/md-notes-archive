# 第105回 問83（薬剤）

**問題文**
器官形成期に該当し、薬剤による催奇形性のリスクが最も高いと考えられる妊娠週はどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-83 を参照

**選択肢**
1. 2週未満
2. 2〜3週
3. 4〜7週
4. 8〜12週
5. 13週以降

**正解:** 3

**解説**
解答　３

妊娠4〜7週目（妊娠2ヶ月目）は絶対過敏期といい、胎児の中枢神経、心臓、消化器などの重要な臓器や四肢などが形成されていく器官形成期に該当するため、薬剤による催奇形性のリスクが最も高いと考えられる。
なお、妊娠週数は、最終月経の初日を妊娠0週0日として計算するため、実際に受精・着床した時期（妊娠2〜3週目頃）とずれが生じることに注意する必要がある。
