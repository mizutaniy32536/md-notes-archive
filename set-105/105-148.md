# 第105回 問148

### 第105回薬剤師国家試験 問148

**問題文**  
医療法に基づき、医療機関の管理者に求められる医療安全の確保に関する記述のうち、適切なのはどれか。**2つ選べ**。[1][2][3]

**全選択肢**  
1. 医療に係る安全管理のための指針を整備しなければならない。  
2. 医薬品安全管理責任者を配置しなければならない。  
3. 医療事故が発生した場合、第三者委員会による調査を実施しなければならない。  
4. 医薬品の安全使用のために、患者を対象とした研修を実施しなければならない。  
5. 医療事故が発生した場合、当該医療事故の日時、場所、状況等を公表しなければならない。[1][2][3]

**正解番号**  
1, 2[1][2][3][9]

**解説**  
医療法第6条の9～第6条の14において、医療機関の管理者（病院等）は医療の安全確保を義務付けられている。具体的には、①医療に係る安全管理のための指針を整備すること、②医薬品安全管理責任者を配置すること、③従事者に対する医薬品の安全使用のための研修の実施、④医薬品の安全使用のための業務に関する手順書の作成及び実施、⑤医薬品の安全使用のための情報収集・改善策の実施が求められる。[1][3][9]  
- 選択肢3は誤り。「第三者委員会による調査」は義務付けられていない。  
- 選択肢4は誤り。研修の対象は「従業者」（医療従事者）であり、「患者」ではない。  
- 選択肢5は誤り。医療事故の「公表」義務はない。[1]
