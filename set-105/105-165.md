# 第105回 問165（薬理）

**問題文**
35歳女性。身長150 cm、体重65 kg、血圧138/90 mmHg、喫煙歴あり（30本／日）。仕事が忙しくなりストレスが溜まり、暴飲暴食で早食いとなった。ある日、突然、今までに経験したことがない心窩部痛におそわれ、心配になり近医を受診した。検査の結果、逆流性食道炎と診断され、下記の処方で治療が開始され、2ヶ月ほどで軽快した。問165（薬理）この患者に処方されたそれぞれの薬物の作用機序として正しいのはどれか。２つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-165 を参照

**選択肢**
1. ヒスタミンH2受容体遮断
2. アセチルコリンM3受容体遮断
3. H＋,K＋−ATPase阻害
4. セロトニン5−HT4受容体刺激
5. ドパミンD2受容体遮断問166（病態）この患者に関する記述のうち、適切なのはどれか。２つ選べ。
6. 食道下部括約筋の収縮により、胃酸が逆流することにより発症した。
7. 内視鏡検査で食道に炎症やびらんが認められた。
8. Ca2＋チャネル遮断薬を併用するとより症状が改善する。
9. 再発防止のため、この処方2を継続する必要がある。
10. 薬物療法に加えて禁煙や体重の減量などの生活指導が奏功する。

**正解:** 3、4

**解説**
問165　解答　３、４

それぞれの処方薬の作用機序は以下の通りである。
ランソプラゾールは、胃壁細胞で活性体となったのち、H＋,K＋−ATPase（プロトンポンプ）のSH基と共有結合を形成することで、プロトンポンプを非可逆的に阻害して胃酸分泌を抑制する。
モサプリドは、副交感神経のセロトニン5-HT4受容体を刺激し、副交感神経終末からのアセチルコリン遊離を増大させることで、消化管運動を亢進するため、胃内容物の胃排出を促進する。

問166　解答　２、５

逆流性食道炎とは、胃食道逆流症のうち、内視鏡検査にて食道に炎症やびらんを認めるものをいう。なお、胃食道逆流症は、主に胃酸をはじめとする酸性の胃内容物が食道や口腔内に逆流することで、胸やけや呑酸、胸痛などの症状を呈する病態で、食道病変が明確な逆流性食道炎（びらん性胃食道逆流症）と食道病変が明確でない非びらん性胃食道逆流症に分類される。

１　誤
逆流性食道炎において胃内容物が逆流する要因として、下部食道括約筋の機能低下に伴う弛緩があげられる。下部食道括約筋が弛緩する要因として、加齢、肥満、食道裂孔ヘルニアなどがある。

２　正
冒頭解説文参照

３　誤
Ca2＋チャネル遮断薬には下部食道括約筋の弛緩作用があるため、併用すると症状が悪化するおそれがある。

４　誤
逆流性食道炎の治療は、禁煙、減量、節酒などの生活指導とともに薬物療法を行う。薬物療法は、プロトンポンプ阻害薬（ランソプラゾールなど）が初期治療の第一選択薬として推奨されており、再発防止のための維持療法においても、プロトンポンプ阻害薬が推奨されている。
また、補助的に消化管運動機能改善薬（モサプリドなど）や、制酸薬、粘膜保護薬などが用いられることもあるが、再発予防のためには使用しない。
したがって、本患者の再発防止として継続投与する必要があるのは処方1のランソプラゾールカプセルである。

５　正
