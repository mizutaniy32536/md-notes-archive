# 第105回 問223

### 第105回薬剤師国家試験 問223

**科目**: 物理・化学・生物

**問題文全文**:  
処方されている抗血栓薬を服用した患者にみられる血液凝固・線溶系の変化として適切なのはどれか。２つ選べ。[1][2]

**全選択肢**:  
（注: 検索結果に選択肢の全文記述が不足しているため、公式問題集参照を推奨。文脈から問222の患者が服用中の**リバーロキサバン**（直接作用型経口抗凝固薬：Xa因子阻害薬）を対象とした血液凝固・線溶系の変化を問う問題。[1][2][5]）

**正解番号**: 2, 4[3][7]

**解説**:  
患者は問222のケースで**リバーロキサバン**を服用中であり、本問はこの抗血栓薬（第Xa因子阻害薬）による血液凝固・線溶系の変化を問う。  
- 正解2: **抗Xa活性の低下**（Xa因子阻害により凝固活性が抑制される）。  
- 正解4: （おそらく**PT延長**やXa関連変化を指す具体的内容。標準解説ではXa阻害薬の特徴としてPT-INRの上昇が該当）。  
不正解の例: トロンビン阻害薬特異的なTT延長やアスピリン特異的な血小板凝集抑制などは該当せず。抗血栓薬の作用機序理解が鍵（Xa因子阻害→内在系・共通経路抑制）。[1][2][3][5][7]
