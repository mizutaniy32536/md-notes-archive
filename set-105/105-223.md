# 第105回 問223

### 第105回薬剤師国家試験 問223

**科目**: 物理・化学・生物

**問題文全文**:  
処方されている抗血栓薬を服用した患者にみられる血液凝固・線溶系の変化として適切なのはどれか。２つ選べ。[1][2]

**全選択肢**:  
1. 組織トロンボプラスチンの生成が抑制されている。  
2. トロンビンの生成が抑制されている。  
3. プロトロンビンの生成が抑制されている。  
4. フィブリンの生成が抑制されている。  
5. プラスミンの生成が促進されている。

**正解番号**: 2, 4[3][7]

**解説**:  
患者は問222のケースで**リバーロキサバン**を服用中であり、本問はこの抗血栓薬（第Xa因子阻害薬）による血液凝固・線溶系の変化を問う。  
- 選択肢2: 第Xa因子阻害によりトロンビン生成が抑制されるため正しい。  
- 選択肢4: トロンビン生成低下に続いてフィブリン生成も抑制されるため正しい。  
- 選択肢1・3: 組織トロンボプラスチンやプロトロンビンの生成そのものを抑える薬ではない。  
- 選択肢5: 線溶促進薬ではないためプラスミン生成促進は起こらない。  
抗血栓薬の作用機序理解が鍵であり、Xa因子阻害によって共通経路が抑制される。[1][2][3][5][7]
