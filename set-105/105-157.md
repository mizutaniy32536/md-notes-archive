# 第105回 問157（薬理）

**問題文**
インドメタシンの副作用である消化性潰瘍の発症機序として正しいのはどれか。２つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-157 を参照

**選択肢**
1. ガストリン受容体の刺激
2. ムスカリン性アセチルコリン受容体の刺激
3. 胃蠕動運動の抑制
4. 胃粘膜血流の低下
5. 胃粘液分泌の抑制

**正解:** 4、5

**解説**
解答　４、５

インドメタシンは、非ステロイド性抗炎症薬（NSAIDs）であり、シクロオキシゲナーゼ（COX）を阻害することで、プロスタグランジンの生合成を抑制し、抗炎症作用を示す。
また、NSAIDsは副作用として消化性潰瘍などの胃腸障害が有名であり、プロスタグランジンによる胃粘膜血流量の増加作用や胃粘液分泌促進作用などを抑制することにより発症する。
