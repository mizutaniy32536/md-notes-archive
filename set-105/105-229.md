# 第105回 問229

### 第105回薬剤師国家試験 問229（衛生）

#### 問題文（全文）
小学生の男児がサッカークラブに加入した。母親は、これを機に自宅の救急箱を充実したいと考え、かかりつけの薬局を訪れた。男児は過去に栗きんとんや大量の甘栗を摂取した際に呼吸困難、全身にかゆみを伴うむくみとじん麻疹を経験したことがある。バナナ、アボカドを摂取しても同様の症状が現れたことがある。[2][3]

この男児への使用を避けることが適切な製品及び関連する食物アレルギーに関する記述のうち、正しいのはどれか。**２つ選べ。**[2][3]

#### 全選択肢
1. この製品の主成分と、栗やバナナに含まれる成分には、共通の構造をもつエピトープが存在する。  
2. この製品の主成分から脱炭酸反応によって生じるヒスタミンが、アレルギーの原因である。  
3. この製品をこの男児が使うと、Ⅰ型アレルギーが誘発される可能性がある。  
4. 栗やバナナは、重篤なアレルギー症状を引き起こす可能性があるため、これらを含む食品には特定原材料（7品目）として表示する義務がある。  
5. ラテックスアレルギーは幼児期のみに発症する。[1][2]

#### 正解番号
**1, 3**[1]

#### 解説
- **選択肢1 正**: この男児は栗・バナナ・アボカドによる**ラテックスフルーツ症候群**を示唆しており、ラテックス（ゴム製品、主成分はポリイソプレン）のエピトープとこれらの果実成分に共通構造が存在する（交叉反応性）。問228で避けるべき製品はラテックス手袋等。[1][2][3]  
- **選択肢2 誤**: ラテックスアレルギーはⅠ型アレルギーで、IgE抗体が関与。主成分からヒスタミンが直接生じるわけではなく、脱炭酸反応は無関係。[1]  
- **選択肢3 正**: ラテックスアレルギーは即時型（Ⅰ型）アレルギーのため、使用でアナフィラキシー等が誘発される可能性がある。[1][2]  
- **選択肢4 誤**: 特定原材料7品目はえび・かに・卵・乳・小麦・そば・落花生。栗・バナナは含まない（21品目にも非該当）。[1]  
- **選択肢5 誤**: ラテックスアレルギーは幼児期に限らず成人期も発症可能。[1]  

**正解は1,3**。ラテックスフルーツ症候群の知識が鍵。[1][2]
