# 第105回 問219

# 第105回薬剤師国家試験 問219

## 科目
物理・化学・生物

## 問題文
前問で検討している薬物に関する記述のうち、正しいのはどれか。**2つ選べ。**[2][3]

## 全選択肢
1. EGFRのチロシンキナーゼを特異的に阻害し、細胞内シグナル伝達を抑制する。[2][3]

2. 高分子型分子標的薬である。[2][3]

3. がん細胞が分泌する増殖因子に結合して、その分解を促進する。[2][3]

4. 標的タンパク質のアミノ酸配列の違いにより有効性が異なる場合がある。[1]

5. がん細胞中の変異した遺伝子に結合して、その遺伝子を切断する制限酵素としての働きをもつ。[1]

## 正解番号
**1、4**[1][2]

## 解説
前問(問218)で検討されている薬物は**ゲフィチニブ（商品名イレッサ）**です。[1]

ゲフィチニブは、上皮成長因子受容体（EGFR）のチロシンキナーゼを選択的に阻害する**低分子抗がん剤**であり、非小細胞肺がんに対する治療薬として使用されます。[1]EGFR遺伝子変異陽性であることを確認した上で用いる薬です。[1]

**正解の根拠：**

- **選択肢1（正解）**：ゲフィチニブはEGFRのチロシンキナーゼを特異的に阻害し、細胞内シグナル伝達を抑制するというのが本薬の作用機序です。[1]

- **選択肢2（誤り）**：ゲフィチニブは低分子型の分子標的薬であり、高分子型ではありません。[1]

- **選択肢3（誤り）**：ゲフィチニブはEGFR受容体に直接作用するもので、がん細胞が分泌する増殖因子に結合するのではありません。

- **選択肢4（正解）**：EGFR遺伝子変異の状態によって、ゲフィチニブの有効性が異なります。標的タンパク質（EGFR）のアミノ酸配列の違いにより有効性が異なる場合があります。[1]

- **選択肢5（誤り）**：ゲフィチニブは制限酵素ではなく、チロシンキナーゼ阻害薬です。
