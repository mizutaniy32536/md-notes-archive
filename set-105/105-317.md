# 第105回 問317

# 第105回薬剤師国家試験 問317

## 問題文

この薬剤を取り扱う薬剤師が行わなければならないこととして、適切なのはどれか。**2つ選べ。**[2][4]

（背景：68歳男性、肝硬変。低タンパク血症による難治性腹水に対して高張アルブミン製剤（献血アルブミン20%静注）による治療が開始された。なお、この薬剤の容器には「特生物」の表示がある。）

## 全選択肢

1. 鍵のかかる場所で保管する。[4]

2. 保管場所に白地に赤枠、赤字をもって「特生物」の表示を行う。[4]

3. 使用した患者の氏名及び住所、使用した薬剤の名称及び製造番号又は製造記号、使用年月日、その他必要な事項を記録する。[4]

4. 使用に関する記録を、その使用した日から少なくとも10年間保存する。[1]

5. 使用による感染症の発生について、危害の発生を防止するために必要があると認めるときは、その旨を厚生労働大臣に報告する。[1]

## 正解番号

**3、5**[1][2][7][8]

## 解説

**特定生物由来製品（特生物）**とは、生物由来製品のうち、販売・授与後において保健衛生上の危害の発生や拡大を防止するための措置を講ずることが必要なものであり、厚生労働大臣が薬事・食品衛生審議会の意見を聴いて指定するものです。[1]

特定生物由来製品の取扱いに関して、薬剤師が行わなければならないことは以下の通りです：

**選択肢3（正解）**：使用した患者の氏名及び住所、使用した薬剤の名称及び製造番号又は製造記号、使用年月日、その他必要な事項を記録することは、特定生物由来製品の法定義務です。[1][4]

**選択肢5（正解）**：使用による感染症の発生について、危害の発生を防止するために必要があると認めるときは、その旨を厚生労働大臣に報告する義務があります。ヒト血液原料のため、感染リスクを完全に排除できないからです。[1]

**選択肢1（不適切）**：鍵のかかる場所での保管は、特定生物由来製品に対する一般的な要件ではありません。

**選択肢2（不適切）**：保管場所の表示方法として「白地に赤枠、赤字」の表示は、特定生物由来製品の標準的な表示方法ではありません。

**選択肢4（不適切）**：記録の保存期間は10年間ですが、これは「行わなければならないこと」というより管理義務に該当し、この問いの対象外です。
