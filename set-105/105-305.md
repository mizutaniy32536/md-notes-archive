# 第105回 問305

### 第105回薬剤師国家試験 問305

#### 問題文（症例）
70歳女性。糖尿病の既往歴はない。非小細胞肺がん（扁平上皮がん、PD−L1発現率15％）と診断され、ニボルマブ点滴静注240 mg、2週間毎の投与が開始された。経過良好だったが、6回目の投与後、自宅で強い倦怠感、食欲低下、口渇と多尿が出現し、水分摂取も困難であったため、緊急受診した。受診時、朝食をとらずに行った検査値は以下の通りである。[2][3]

**検査値**  
Na 135 mEq/L、Cl 96 mEq/L、K 5.4 mEq/L、BUN 23 mg/dL、HbA1c 6.0％（NGSP値）、血糖値571 mg/dL、血液pH 7.1、尿糖4＋、尿ケトン3＋[2][3]

**問305（実務）**  
この患者に実施されるべき治療の目的と薬物治療の組合せとして適切なのはどれか。**2つ選べ。**[1][2]

#### 全選択肢
（検索結果から問305の全選択肢が部分的にしか記載されていないが、適切な組合せとして以下のものが正解に該当し、解説で言及されているものを含む。[1][2]）  
1. **高血糖改善 速効型インスリンを投与**（正解）[1]  
2. **脱水補正 生理食塩液を点滴投与**（正解、脱水補正は高血糖改善にもつながる）[1]  
3. **高ナトリウム血症改善 5%ブドウ糖を点滴投与**（誤り）[1]  
4. **電解質バランス改善 炭酸水素ナトリウムを点滴投与**（誤り、この患者は低ナトリウム血症で酸逆変性）[1]  
5. **膵炎治療 ガベキサートを投与**（誤り、ガベキサートはタンパク質分解酵素阻害薬で膵炎治療薬だが、この患者に不適切）[1]

#### 正解番号
**1, 2**[1][2]

#### 解説
本患者はニボルマブ投与に伴う重大な副作用として**1型糖尿病**とそれに伴う**糖尿病性ケトアシドーシス（DKA）**を発症している。HbA1cが低く急性発症、尿ケトン陽性、pH低下がその特徴。[2][3]  
- **選択肢1**: 高血糖（血糖値571 mg/dL）改善のため**速効型インスリン**投与が適切。インスリン欠乏によるDKA治療の基本。[1][2]  
- **選択肢2**: 脱水症状（口渇、多尿、BUN上昇）補正のため**生理食塩液**点滴投与が適切。脱水補正は高血糖改善にも寄与する。[1]  
- **選択肢3**: Na 135 mEq/Lは低ナトリウム血症のため不適切。[1]  
- **選択肢4**: 電解質バランス改善に炭酸水素ナトリウムは酸逆変性で使用されるが、この組合せは不適切。[1]  
- **選択肢5**: ガベキサートは膵炎治療薬だが、本症例はニボルマブ誘発性1型糖尿病で膵炎ではないため不適切。[1]  
（注: 問304の正解は2,3（血中C-ペプチド低値、次回ニボルマブ中止）で関連。[1][3]）
