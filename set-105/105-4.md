# 第105回 問4（物理）

**問題文**
元素の原子量をH＝1.0079、C＝12.0107、O＝15.9994、Pb＝207.2とするとき、酢酸鉛（Ⅱ）の式量の有効数字の桁数として正しいのはどれか。１つ選べ。

**選択肢**
1. 3
2. 4
3. 5
4. 6
5. 7

**正解:** 2

**解説**
解答　２

有効数字の桁数を見る際は、0でない最高位の数字を一桁目と数え始める（設問にあるH＝1.0079の場合、有効数字五桁となる）。
酢酸鉛（Pb（C2H3O2）2）の式量は文中の原子量（H＝1.0079、C＝12.0107、O＝15.9994、Pb＝207.2）を用いて、各元素の原子量の総和より以下のように計算される。
Pb（C2H3O2）2の式量＝207.2＋（12.0107×2＋1.0079×3＋15.9994×2）×2＝325.2878
本計算は各元素の原子量の総和より求めているため、有効数字の加減法より桁数を考慮する。加減法は、各数値の小数点以下の桁数が最も少ないものに合わせて計算する。
小数点以下の桁数が一番少ないのは、Pb＝207.2（小数点以下1桁）のため、Pb（C2H3O2）2の式量は小数点以下1桁まで表し、2桁目以降は四捨五入して325.3（有効数字4桁）となる。
