# 第105回 問92

### 第105回薬剤師国家試験 問92（一般理論問題・物理：局方酸化亜鉛の定量法）

#### 問題文（全文）
酢酸亜鉛は、ウィルソン病や低亜鉛血症の治療薬として用いられているが、副作用として銅欠乏症を生じる場合がある。亜鉛化合物の定量にはキレート滴定が用いられることがある。本品を850℃で1時間強熱し、その約0.8 gを精密に量り、水2 mL及び塩酸3 mLに溶かし、水を加えて正確に100 mLとする。この液10 mLを正確に量り、水80 mLを加え、水酸化ナトリウム溶液（1→50）をわずかに①**沈殿**を生じるまで加え、次に②**pH10.7のアンモニア・塩化アンモニウム緩衝液5 mLを加え**た後、0.05 mol/Lエチレンジアミン四酢酸二水素二ナトリウム液で滴定する（指示薬：**（ア）** 0.04 g）。  
0.05 mol/Lエチレンジアミン四酢酸二水素二ナトリウム液1 mL ＝ **（イ）** mg ZnO[1][2][3]

#### 全選択肢
1. 下線部①の沈殿は水酸化亜鉛(Zn(OH)<sub>2</sub>)である。[3]  
2. 下線部②の操作は、エチレンジアミン四酢酸と金属の錯体を作りやすくするために行う。[3]  
3. （ア）は、クリスタルバイオレットである。[1][2][3]  
4. 滴定終点において、指示薬（ア）がエチレンジアミン四酢酸と結合して変色する。[1][3]  
5. （イ）に入る数値は2.035である。[1][3]

#### 正解番号
**1, 2**（複数選択問題）[1][2][6]

#### 解説
- **選択肢1 正**：酸化亜鉛を溶かした酸性の水溶液に水酸化ナトリウム溶液を加えると、水酸化亜鉛（Zn(OH)<sub>2</sub>）の沈殿を生じる。これはZn<sup>2+</sup>イオンの沈殿操作で、滴定前の不純物除去や調整を目的とする。[1][2][3]  
- **選択肢2 正**：pH10.7のアンモニア・塩化アンモニウム緩衝液を加える操作は、EDTA（エチレンジアミン四酢酸）とZn<sup>2+</sup>の安定な錯体を形成しやすくするため（条件pH調整）。Zn-EDTA錯体の安定定数が最大になるpH領域である。[1][2][3]  
- **選択肢3 誤**：指示薬（ア）は**エリオクロムブラックT**（または類似の金属指示薬）であり、クリスタルバイオレットではない。クリスタルバイオレットは主に他の滴定で用いられる。[1]  
- **選択肢4 誤**：終点では指示薬がZn<sup>2+</sup>から解離し、遊離型EDTA存在下で色が変わる。指示薬はEDTAと直接結合して変色するわけではなく、金属イオンとの結合状態の変化による。[1][2]  
- **選択肢5 誤**：（イ）の計算は以下の通り。  
  EDTA 0.05 mol/L × 1 mL = 0.05 mmol（対応数1:1）。  
  ZnO分子量 81.38 g/mol → 0.05 mmol × 81.38 = **4.069 mg**。[1][2][7]  
EDTAによるキレート滴定の標準的手法で、日本薬局方酸化亜鉛定量法に基づく。[1][2][3][7]
