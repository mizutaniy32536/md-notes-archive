# 第105回 問71（薬理）

**問題文**
薬局開設者が、都道府県知事に別段の申出をしない限り、免許を受けたとみなされるのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-71 を参照

**選択肢**
1. 向精神薬輸入業者
2. 向精神薬輸出業者
3. 向精神薬製造製剤業者
4. 向精神薬使用業者
5. 向精神薬小売業者

**正解:** 5

**解説**
解答　５

薬局開設の許可を受けた者は、向精神薬卸売業者及び向精神薬小売業者の免許を受けた者とみなされる。ただし、当該薬局開設者が、都道府県知事に別段の申出をしたときは、この限りではない。
