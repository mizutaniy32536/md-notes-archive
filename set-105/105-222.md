# 第105回 問222（実務）

**問題文**
72歳男性。A病院の泌尿器科及びB病院の循環器科を受診している。A病院において、侵襲危険度の高い経尿道的膀胱腫瘍切除術（TURBT）実施のため、泌尿器科医師から、現在服用中の薬を確認し、術前中止薬の有無を調査するよう、A病院の入退院支援センター担当の薬剤師に依頼があった。患者が持参したお薬手帳の内容、患者へのインタビューなどから、患者の服用薬が判明した。問222（実務）患者は、手術の前日に入院することが決まった。入退院支援センター担当薬剤師の対応として、適切なのはどれか。２つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-222 を参照

**選択肢**
1. 薬剤師の判断により、入院前日の朝から全ての薬剤を服用中止するように患者に指示した。
2. 泌尿器科医師に、ファモチジンをラベプラゾールナトリウムに変更するように提案した。
3. リバーロキサバンの服用を中止する必要があることを泌尿器科医師に事前に説明した。
4. B病院の循環器医師にオルメサルタンメドキソミル口腔内崩壊錠の休薬の可否を確認し、泌尿器科医師に内容を伝達した。
5. 泌尿器科医師にシロドシンの服用中止を提案した。問223（物理・化学・生物）処方されている抗血栓薬を服用した患者にみられる血液凝固・線溶系の変化として適切なのはどれか。２つ選べ。
6. 組織トロンボプラスチンの生成が抑制されている。
7. トロンビンの生成が抑制されている。
8. プロトロンビンの生成が抑制されている。
9. フィブリンの生成が抑制されている。
10. プラスミンの生成が促進されている。

**正解:** 3、4

**解説**
問222　解答　３、４

患者は、侵襲危険度の高い経尿道的膀胱腫瘍切除術（TURBT）を実施する予定であり、手術中の出血を助長する抗血栓薬は、一般的に術前中止が必要である。また、手術時、麻酔、出血の影響で血圧が低下するが、代償的にレニン–アンギオテンシン系の機能が亢進し、血圧が維持される。しかし、レニン−アンギオテンシン系の機能を抑制するアンギオテンシン変換酵素（ACE）阻害薬やアンギオテンシンⅡAT1受容体遮断薬（ARB）を使用していると、代償反応が機能せず、過度の血圧低下が生じるおそれがあるため、ACE阻害薬、ARBなども術前中止が必要である。

１　誤
薬剤師の自己判断で、服用中止するように患者に指示することは適切ではない。処方医に患者の持参薬についてフィードバックを行い、処方医に確認した後に指示を行うことが適切である。

２　誤
ファモチジンは、本手術への影響は少ないと考えられるため、ラベプラゾールナトリウムへ処方変更する提案は不要である。

３　正
前記参照。リバーロキサバン錠は、選択的かつ直接的にXa阻害することで、抗血栓作用を示す。手術を開始する24時間前から休薬を行うように泌尿器科医師に事前に説明することは適切である。

４　正
前記参照。オルメサルタンメドキソミル口腔内崩壊錠は、アンギオテンシンⅡAT1受容体遮断薬であり、アンギオテンシン系の機能を抑制するため、手術を開始する24時間前から休薬することが望ましいとされている。

５　誤
シロドシンは、選択的α1A受容体遮断作用により前立腺肥大症に伴う排尿困難を改善する。本手術への影響は少ないと考えられるため、服用中止を提案することは適切ではない。

問223　解答　２、４

リバーロキサバン錠は、選択的かつ直接的にXa阻害することで、トロンビン生成、フィブリン生成を抑制し、抗凝固作用を示す。
