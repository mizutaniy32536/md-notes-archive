# 第105回 問181（病態・薬物治療）

**問題文**
光に対して不安定な薬物Aを含有するフィルムコーティング錠に含まれる添加剤を以下に示す。添加剤の種類や量を変更したときに起こり得る錠剤特性の変化に関する記述のうち、正しいのはどれか。２つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-181 を参照

**選択肢**
1. カルメロースカルシウムをカルメロースナトリウムに変更すると、錠剤は崩壊しにくくなる。
2. ヒドロキシプロピルセルロースの配合量を相対的に減らすと、素錠の硬度は下がりやすくなる。
3. ステアリン酸マグネシウムの配合量を相対的に減らすと、キャッピングが起こりやすくなる。
4. ヒプロメロースをヒプロメロースフタル酸エステルに変更すると、日本薬局方崩壊試験第2液中における薬物Aの溶出は起こらない。
5. 酸化チタンを除いても、本錠剤に光を当てた際の薬物Aの安定性は変わらない。

**正解:** 1、2

**解説**
解答　１、２

１　正
カルメロースカルシウムは、水を吸収することで膨張する性質を持つ崩壊剤であり、主に錠剤の崩壊性を高めるために用いられる。一方、カルメロースナトリウムは水溶性の結合剤であり、主に錠剤内の結合性を高めるために用いられる。したがって、カルメロースカルシウムをカルメロースナトリウムに変更すると、錠剤は崩壊しにくくなる。

２　正
ヒドロキシプロピルセルロースは水溶性の結合剤であり、主に錠剤内の結合性を高めるために用いられる。したがって、ヒドロキシプロピルセルロースの配合量を相対的に減らすと、錠剤内の結合力が低下し、素錠の硬度は下がりやすくなる。

３　誤
ステアリン酸マグネシウムは滑沢剤であり、錠剤の製造時に打錠工程前に適量添加することで粉体の流動性の向上、臼や杵への付着防止などのために用いられる。また、滑沢剤などの添加剤の過不足等が原因で、下表のような打錠障害が発生する。

したがって、ステアリン酸マグネシウムの配合量を相対的に減らすと、バインディング、スティッキング等が起こりやすくなる。

４　誤
ヒプロメロースは水溶性高分子であり、胃溶性のコーティング剤として利用される。一方、ヒプロメロースフタル酸エステルは、酸性条件下ではあまり溶解せず、中性から塩基性条件下では溶解するため、主に腸溶性のコーティング剤として用いられる。したがって、ヒプロメロースフタル酸エステルに変更すると、日本薬局方崩壊試験第2液（pH6.8の溶液）中において薬物Aは溶出する。

５　誤
酸化チタンは遮光剤であり、主に光分解しやすい医薬品の光安定性を向上させるために用いられる。したがって、酸化チタンを除いた場合、本錠剤に光を当てた際の薬物Aの安定性は低下する。
