# 第105回 問323

# 第105回薬剤師国家試験 問323

## 問題文

62歳男性。妻と死別し独居である。認知症があり、介護保険制度による要支援2のサービスを受けている。前回の処方からドネペジル塩酸塩が5mgから10mgに増量になった。薬剤師が医師の指示により患者宅を訪問したところ、患者から最近尿が出にくく、吐き気があると訴えがあった。また、3日前から風邪気味のため、市販薬であるA顆粒を服用していることがわかった。

**介護保険制度に照らしたこの患者に関する記述のうち、適切なのはどれか。2つ選べ。**[1][3]

## 選択肢

1. この患者は第2号被保険者である。[1][3]

2. 薬局において居宅療養管理指導料を算定する。[2]

3. 薬局において在宅患者訪問薬剤管理指導料を算定する。[2]

4. この患者は介護給付を受けることができる。[2]

5. この患者の介護保険料は、医療保険料に上乗せして保険者が一括徴収する。[1][3]

## 正解番号

**1、5**[1][3][6]

## 解説

**選択肢1（正）**：介護保険制度の被保険者には第1号被保険者（65歳以上）と第2号被保険者（40～64歳の医療保険加入者）がいます。この患者は62歳であり、医療保険加入者であるため、第2号被保険者に該当します。[1][3]

**選択肢2（誤）**：居宅療養管理指導料は医療保険の適用であり、介護保険のサービスではありません。[2]

**選択肢3（誤）**：在宅患者訪問薬剤管理指導料は医療保険の適用です。介護保険としてのサービスではありません。[2]

**選択肢4（誤）**：この患者は「要支援2」の認定を受けており、介護給付は「要介護認定」を受けた人のみが対象です。また、特定疾病に「認知症」は含まれていません。[1]

**選択肢5（正）**：介護保険料は、医療保険料に上乗せして保険者が一括徴収されます。[1][3]
