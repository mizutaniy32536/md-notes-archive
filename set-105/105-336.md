# 第105回 問336（実務）

**問題文**
ある薬局で採用予定の医薬品は口腔内崩壊（OD）錠で、5 mg、10 mg、20 mg、40 mgの4規格が存在する。取扱い方法を検討するため当該医薬品のインタビューフォームを確認したところ、以下に示すデータが記載されていた。このデータの解釈及び対応として適切なのはどれか。２つ選べ。選択肢中、特に規格を示さないときは4規格に共通するものとする。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-336 を参照

**選択肢**
1. 室温で60％RHを越えない場合PTPシート包装で3年間安定に保管できる。
2. 25℃/75％RHで、アルミ袋（乾燥剤入り）で6ヶ月間安定に保管できる。
3. 無包装状態で25℃/60％RHで、3ヶ月では硬度が低下する。
4. 25℃/50%RHの条件では、無包装状態でも光には比較的安定である。
5. OD錠20 mgは、プラスチックボトル（乾燥剤入り）で40℃/75％RHで6ヶ月間外観変化はないが、含量は低下する。

**正解:** 2、4

**解説**
解答　２、４

１　誤
長期保存試験のデータによると、25℃/60%RH、PTP、アルミ袋（乾燥剤入り）の条件で保存した場合、OD錠10 mg、OD錠20 mg、OD錠40 mgは36ヶ月間（3年間）変化がなかったことから、室温（1〜30℃）で60％RHを越えない場合はPTPシート包装で3年間安定に保管できると解釈できる。しかし、OD錠5 mgは同条件で保存した場合のデータが、24ヶ月（2年間）分しかないため、3年間安定に保管できると解釈することはできない。

２　正
加速試験のデータによると、40℃/75%RH、PTP、アルミ袋（乾燥剤入り）の条件で保存した場合、全OD錠は6ヶ月間変化がなかったことから、より穏やかな条件である25℃/75％RHであれば、アルミ袋（乾燥剤入り）で6ヶ月間安定に保管できると解釈することができる。

３　誤
過酷試験のデータによると、25℃/75％RH、ガラス製シャーレ（解放）の条件で保存した場合、全OD錠が3ヶ月で硬度が低下している。より穏やかな条件である25℃/60％RHであれば、3ヶ月で硬度が低下しない可能性もあるが、3ヶ月で硬度が低下すると解釈することはできない。

４　正
過酷試験のデータによると、2000 lx、25℃/60%RH、ガラス製シャーレ（解放）の条件で保存した場合、全OD錠が25日間（総照度120万lx・hr÷2000 lx＝600 hr＝25日）変化がなかったことから、より穏やかな条件である25℃/50%RHの条件であれば、無包装状態でも光には比較的安定であると解釈することができる。

５　誤
加速試験のデータによると、40℃/75%RH、プラスチックボトル（乾燥剤入り）の条件で保存した場合、OD錠20 mgは6ヶ月間変化がなかったことから、6ヶ月間外観変化はないと解釈できる。しかし、含量の低下については本データから確認することはできない。
