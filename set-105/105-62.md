# 第105回 問62（薬理）

**問題文**
骨格筋の壊死と再生を繰り返し、徐々に筋萎縮や筋力低下が進行する遺伝性の疾患はどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-62 を参照

**選択肢**
1. 重症筋無力症
2. 筋ジストロフィー
3. 皮膚筋炎
4. 多発性硬化症
5. ギラン・バレー症候群

**正解:** 2

**解説**
解答　２

筋ジストロフィーとは、骨格筋の形成に必要な細胞質タンパクのジストロフィンが変性し、骨格筋の壊死と再生を繰り返し、徐々に筋萎縮や筋力低下が進行する遺伝性疾患である。筋肉が萎縮するタイプと筋肉が弛緩できないタイプ（筋強直性）がある。
