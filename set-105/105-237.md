# 第105回 問237

# 第105回薬剤師国家試験 問237

## 科目
衛生

## 問題文
特定健康診査の結果に基づき、この男性に対して行われる特定保健指導に関する記述のうち、正しいのはどれか。１つ選べ。[1][2]

（※前問の患者：60歳男性、喫煙歴なし、極端な運動不足。健診結果は身長165cm、体重81.7kg、BMI 30、腹囲100cm、収縮期血圧155mmHg、拡張期血圧95mmHg、中性脂肪220mg/dL、HDL-C 35mg/dL、空腹時血糖値90mg/dL、HbA1c 5.2%）

## 選択肢
1. 生活習慣の改善の意識付けを行うための「情報提供」の段階である。
2. 生活習慣の改善に関する「動機付け支援」の段階である。
3. 生活習慣の改善に関する「動機付け支援」に加え、糖尿病に対する栄養指導を受ける段階である。
4. 早期に介入し、行動変容を促す「積極的支援」の段階である。
5. この男性への特定保健指導の階層化において、年齢を考慮する必要はない。

## 正解番号
**4番**[1][2]

## 解説
本患者は腹囲100cmであることからSTEP1の肥満リスクに該当し、さらに収縮期血圧155mmHg、中性脂肪220mg/dLであることからSTEP2の追加リスク数が2つあります。[1]これらの基準により、本患者は**積極的支援（選択肢４）の対象**となります。[1][2]
