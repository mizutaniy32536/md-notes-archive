# 第105回 問28（化学）

**問題文**
腎臓の傍糸球体細胞で、レニン分泌を抑制する機序はどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-28 を参照

**選択肢**
1. アドレナリンα1受容体刺激
2. アドレナリンα1受容体遮断
3. アドレナリンα2受容体遮断
4. アドレナリンβ1受容体刺激
5. アドレナリンβ1受容体遮断

**正解:** 5

**解説**
解答　５

腎臓の傍糸球体細胞にはアドレナリンβ1受容体が存在し、交感神経興奮により放出されたノルアドレナリンによって受容体が刺激されることで、レニン分泌が亢進する。そのため、アドレナリンβ1受容体を遮断すると、レニン分泌が低下し、アンギオテンシノーゲンからアンギオテンシンⅠへの変換が抑制される。
