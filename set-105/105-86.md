# 第105回 問86（薬剤）

**問題文**
薬局間で譲渡を行う際、譲受・譲渡の記録が法律上必要なのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-86 を参照

**選択肢**
1. 化粧品
2. 医薬部外品
3. 第二類医薬品
4. 指定第二類医薬品
5. 第二種向精神薬

**正解:** 5

**解説**
解答　５

向精神薬のうち、第一種向精神薬と第二種向精神薬を薬局間で譲渡を行う際、麻薬及び向精神薬取締法により、譲受・譲渡の記録（品目、数量、年月日、相手の氏名又は名称及び住所）を記載して2年間保存する必要である。
