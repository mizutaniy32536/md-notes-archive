# 第105回 問274（実務）

**問題文**
14歳女児。身長160 cm、体重52 kg。造血幹細胞移植後の真菌感染症予防のため、フルコナゾールカプセルで管理を行っていた。しかし、画像診断や検査値などからアスペルギルス症が疑われ、注射用ボリコナゾールが投与されることになった。検査値：AST 25 IU/L、ALT 37 IU/L、γ−GTP 40 IU/L、 血清クレアチニン値 0.7 mg/dL問274（実務）病棟担当薬剤師は、注射用ボリコナゾールの投与にあたり、処方監査を行い、投与後のモニタリングについて検討した。薬剤師の対応として適切なのはどれか。２つ選べ。

**解説**
問274　解答　３、５

１　誤
ボリコナゾールは小児に対して投与する場合、原則注射剤から開始することとされている。したがって、初回投与から注射剤から経口投与剤への変更の提案は行わない。

２　誤
ボリコナゾールの用法及び用量（mg/kg）は、年齢と体重に応じて下記の①〜③に分類される。本問の患者は14歳、体重52 kgであり③に該当する。したがって、本問の患者の体重あたりの用量（③）は、成人の標準用量（①）と同量で開始されていることを確認する。

３　正
ボリコナゾールは、CYP2C19、2C9及び3A4の阻害作用を有する。したがって、ボリコナゾールによって代謝が強く阻害される薬剤が併用されていないことを確認する必要がある。

４　誤
ボリコナゾールは、通常投与では5〜7日目に定常状態に達すると考えられており、採血はそれ以降に行う。

５　正
ボリコナゾールの重篤な肝障害（肝炎、黄疸、肝不全、肝性昏睡等）が現れることがあるので、必要に応じて肝機能を定期的（月に1〜2回）にモニタリングする必要がある。
