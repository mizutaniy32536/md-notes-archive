# 第105回 問344（実務）

**問題文**
56歳男性。下記の処方薬を使用中であるが、市販の胃薬を購入するために来局した。患者は普段から胃が弱いことを訴えており、過日より親の介護でストレスを感じるようになったせいか、少しキリキリと胃の痛みを感じることがあるとのことであった。表の成分を含む医薬品のうち、推奨するものとして最も適切なのはどれか。１つ選べ。

**解説**
解答　４

本患者はピロカルピン塩酸塩点眼液を使用していることから、緑内障を治療中であると考えられるため、抗コリン作用を有する成分を含まない胃薬を推奨する必要がある。

１　誤
本医薬品は、タンニン酸ベルベリンなどの成分を含有する下痢止めであり、ロートエキスが抗コリン作用を有するため、本患者に推奨すべきではない。

２　誤
本医薬品は、ピコスルファートナトリウム水和物などの成分を含有する便秘薬であり、本患者に推奨すべきではない。

３　誤
本医薬品は、ブチルスコポラミン臭化物などの成分を含有する胃薬であるが、ブチルスコポラミン臭化物が抗コリン作用を有するため、本患者に推奨すべきではない。

４　正
本医薬品は、健胃成分や制酸成分を含有する胃薬であり、本患者に推奨できる。

５　誤
本医薬品は、イブプロフェンなどの成分を含有する解熱鎮痛薬であり、本患者に推奨すべきではない。
