# 第105回 問89（薬剤）

**問題文**
壊死組織を除去して創部を清浄化する行為はどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-89 を参照

**選択肢**
1. ドレナージ
2. スクラビング
3. トリアージ
4. デ・エスカレーション
5. デブリードマン

**正解:** 5

**解説**
解答　５

１　誤
ドレナージとは、ドレーンという管を用いて、体内に貯留した血液、膿、滲出液、消化液などを体外に排出することである。

２　誤
スクラビングとは、スクラブ法（洗浄法）とも呼ばれ、洗浄剤を配合した消毒薬と流水を用いて手指を消毒することである。

３　誤
トリアージとは、患者の重症度に基づき、治療の優先度を決めることである。一般に、重症群（赤色）、中等症群（黄色）、軽傷群（緑色）、死亡群（黒色）の4つに区分される。

４　誤
デ・エスカレーションとは、最初に広域スペクトルを有する抗菌薬を投与し、培養結果と臨床的効果を考慮した後に、狭域スペクトルを有する抗菌薬に変更する治療戦略である。

５　正
デブリードマンとは、壊死組織を除去して創部を清浄化する行為である。
