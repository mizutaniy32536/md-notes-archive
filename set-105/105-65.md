# 第105回 問65（薬理）

**問題文**
全身性エリテマトーデス（SLE）の症状として発現頻度が最も低いのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-65 を参照

**選択肢**
1. 低体温
2. 光線（日光）過敏症
3. 関節痛
4. 蝶形紅斑
5. 口腔内潰瘍

**正解:** 1

**解説**
解答　１

全身性エリテマトーデス（SLE：systemic lupus erythematosus）は、自己免疫疾患の一つであり、自己抗体により形成される免疫複合体が組織に沈着することによって全身の臓器に慢性の炎症を生じる疾患である。SLEの特徴的な症状として、発熱、両側頬部にわたる蝶形紅斑、光線過敏症状、口腔潰瘍、破壊や変形を伴わない多発性関節炎・関節痛があげられる。
よって、発熱の症状から、低体温が発現頻度の最も低い選択肢である。
