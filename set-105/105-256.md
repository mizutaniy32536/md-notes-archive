# 第105回 問256（実務）

**問題文**
52歳女性。全身倦怠感と微熱、手足の関節痛と朝のこわばりを訴えて受診したところ、関節リウマチと診断された。メトトレキサート過敏症の既往歴があるため、代替薬について、医師より薬剤師に相談があった。問256（実務）医師に推奨すべきメトトレキサートの代替薬として適切なのはどれか。２つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-256 を参照

**選択肢**
1. レフルノミド
2. モルヒネ塩酸塩
3. サラゾスルファピリジン
4. インフリキシマブ
5. スリンダク問257（薬理）医師に推奨すべきそれぞれの薬物の作用機序はどれか。２つ選べ。
6. シクロオキシゲナーゼ（COX）の阻害
7. ピリミジンヌクレオチドの合成抑制
8. T細胞及びマクロファージでのサイトカイン産生抑制
9. 腫瘍壊死因子（TNF）−αの捕捉
10. オピオイドµ受容体の刺激

**正解:** 1、3

**解説**
問256　解答　１、３

関節リウマチの治療としては、メトトレキサートが中心に用いられるが、メトトレキサートが用いられない場合には、その他の疾患修飾性抗リウマチ薬（DMARDs）が選択される。
本選択肢のうち、関節リウマチの治療に適応があるのは、レフルノミド、サラゾスルファピリジン、インフリキシマブであるが、インフリキシマブはキメラ型モノクローナル製剤であり、インフリキシマブに対する中和抗体の産生を抑制する目的でメトトレキサートとの併用が必須であるため、本患者には用いることができない。よって、医師に推奨すべきメトトレキサートの代替薬として適切なのは、レフルノミド、サラゾスルファピリジンである。

問257　解答　２、３

レフルノミドはプロドラッグであり、生体内で代謝されて生成した活性代謝物が、ジヒドロオロテートデヒドロゲナーゼを阻害することで、de novoピリミジンヌクレオチドの生合成を阻害し、リンパ球の増殖を抑制する。
また、サラゾスルファピリジンは、T細胞及びマクロファージからのサイトカイン産生を抑制することで抗リウマチ作用を示す。
