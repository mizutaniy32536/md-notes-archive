# 第105回 問132（生物）

**問題文**
モルヒネ及びその類似化合物が、生体内で受ける代謝に関する記述のうち、正しいのはどれか。２つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-132 を参照

**選択肢**
1. コデインは、O−脱メチル化されてモルヒネになる。
2. モルヒネは、アセチル化されてヘロインになる。
3. モルヒネは、メチル化されてコデインになる。
4. ヘロインは、加水分解されてモルヒネになる。
5. コデインは、還元されてジヒドロコデインになる。＜image1:center＞

**正解:** 1、4

**解説**
解答　１、４

１　正
コデインは生体内において、主に6位の水酸基がグルクロン酸抱合を受けることでコデイン-6-グルクロニドへと代謝される。一部はシトクロムP450によりO−脱メチル化されてモルヒネへと代謝される。

２　誤
ヘロインは、モルヒネをアセチル化することで得られる合成品である。生体内でモルヒネがアセチル化されてヘロインになる経路はない。

３　誤
モルヒネはメチル化されてコデインになるが、生体内でモルヒネがメチル化されてコデインになる経路はない。

４　正
ヘロインは生体内において、カルボキシエステラーゼによって加水分解されてモルヒネになる。

５　誤
コデインは還元されてジヒドロコデインになるが、生体内でコデインが還元されジヒドロコデインになる経路はない。
