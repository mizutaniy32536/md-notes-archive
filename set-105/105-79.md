# 第105回 問79（薬剤）

**問題文**
薬剤師が業務上知り得た人の秘密を漏らすと、秘密漏示罪に問われる場合があるが、その根拠となる法律はどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-79 を参照

**選択肢**
1. 民法
2. 薬剤師法
3. 刑法
4. 医薬品医療機器等法
5. 個人情報の保護に関する法律（個人情報保護法）

**正解:** 3

**解説**
解答　３

薬剤師が業務上知り得た人の秘密を正当な理由なく、漏らした場合、秘密漏示罪に問われる場合がある。秘密漏示罪は刑法（第134条）によって規定されている。したがって、本問の薬剤師は刑法違反の罪に問われる可能性がある。
