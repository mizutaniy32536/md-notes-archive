


## ◆ 男女別の死亡統計の推移から粗死亡率と年齢調整死亡率の関係を読み取る問題
260307

**問題**
表は、我が国における男女別の死亡統計（人口千対）の推移を示したものである。この表及び我が国の死亡統計に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**


1. 1950年から1980年までの間、粗死亡率が低下しているのは、この期間に出生率が上昇したためである。
​
2. 1990年から2016年まで粗死亡率が上昇しているのは、健康水準の低下によるものである。
​
3. 2016年における年齢調整死亡率の性差が、粗死亡率の性差より顕著であるのは、その年の年齢別人口構成の性差を反映している。
​
4. 年齢調整死亡率は、「昭和60年（1985年）の人口の年齢構成」を基準として補正した死亡率である。
​
5. 高齢における死亡割合が高くなると、年齢調整死亡率は低くなる傾向にあるため、男性に比べ女性の方が高齢で死亡していると推察できる。

**正解と理由**
- `3. 2016年における年齢調整死亡率の性差が、粗死亡率の性差より顕著であるのは、その年の年齢別人口構成の性差を反映している。`
- `5. 高齢における死亡割合が高くなると、年齢調整死亡率は低くなる傾向にあるため、男性に比べ女性の方が高齢で死亡していると推察できる。`
- **理由の構成**
	- **問題の整理**：表から男女別の`粗死亡率`と`年齢調整死亡率`の推移を読み取り、1950〜1980年は低下傾向、1990年以降は粗死亡率が上昇傾向、年齢調整死亡率は低下傾向という動きを把握する
	- **薬学的知識・仕様・制約**：`年齢調整死亡率`は1985年（昭和60年）モデル人口を基準集団として、年齢構成の影響を除去した死亡率である。粗死亡率は人口構造（高齢化）の影響を直接受けるため、高齢化が進むと粗死亡率は上昇するが、年齢調整死亡率は健康水準そのものの変化を反映する
	- **選択理由**：選択肢3は年齢調整死亡率が年齢構成の影響を除去する指標であることから、性差が粗死亡率より顕著に現れるのは実際の年齢別人口構成の性差（女性の方が高齢者の割合が高い）を反映していると判断できる。選択肢5は女性の方が長寿で高齢死亡割合が高いことが、年齢調整死亡率の性差として表れることを正しく述べている

**解法の手順（思考の流れ）**
- まず表から`粗死亡率`と`年齢調整死亡率`の2種類の指標の推移パターンを確認する
- 次に粗死亡率の変動要因として「人口構造の変化（高齢化）」と「健康水準の変化」の2つを区別する
- 粗死亡率の1950〜1980年の低下は健康水準の向上（感染症対策、医療進歩）が主因であり、出生率上昇が主因ではないと判断 → 選択肢1は誤り
- 1990年以降の粗死亡率上昇は`少子高齢化`による高齢者割合の増加が主因であり、健康水準の低下ではないと判断 → 選択肢2は誤り
- 選択肢4について、年齢調整死亡率の基準は「昭和60年モデル人口」であり、「人口の年齢構成」という表現自体は概ね正しいが、問題文の文脈では選択肢3・5がより正確な記述と判断する
- 選択肢3は年齢調整死亡率の本質（年齢構成の影響除去）を正しく述べている → 正解
- 選択肢5は女性が男性より高齢で死亡する傾向と年齢調整死亡率の関係を正しく述べている → 正解

**他の選択肢が誤りな理由**
- 1. 1950年から1980年までの間、粗死亡率が低下しているのは、この期間に出生率が上昇したためである。
	粗死亡率の低下の主因は`結核`などの感染症死亡の減少、生活環境・衛生水準の改善、医療の進歩による`健康水準の向上`である。出生率の上昇で分母（人口）が増えれば理論上は低下要因となるが、主要因ではない

- 2. 1990年から2016年まで粗死亡率が上昇しているのは、健康水準の低下によるものである。
	粗死亡率の上昇は`少子高齢化`によって死亡リスクの高い高齢者の人口割合が増加したことが主因である。年齢調整死亡率は同期間で低下しており、実際の健康水準はむしろ向上している

- 4. 年齢調整死亡率は、「昭和60年（1985年）の人口の年齢構成」を基準として補正した死亡率である。
	年齢調整死亡率の基準は「昭和60年モデル人口」であり、実際の1985年の人口構成そのものではない。`モデル人口`は統計上の標準的な人口構成として設定されたもので、実測人口構成とは異なる

**キーポイント**
- `粗死亡率`は人口構造（特に高齢化）の影響を直接受けるため、集団間や時系列の比較には不向きである
- `年齢調整死亡率`は`昭和60年モデル人口`を基準集団として年齢構成の影響を除去した指標であり、実質的な死亡リスクの比較に用いる
- 高齢化が進行すると粗死亡率は上昇するが、年齢調整死亡率は低下することがあり、これは医療水準や生活環境の改善を反映している
- 女性は男性より平均寿命が長く、高齢での死亡割合が高いため、年齢調整死亡率の性差が粗死亡率の性差より大きくなる

**関連知識の比較**
- `直接法`による年齢調整死亡率は、各年齢階級の死亡率に基準集団の人口構成を適用して算出する。一方、`間接法`では標準化死亡比（SMR）を用いる
- `合計特殊出生率`は1人の女性が生涯に産む子どもの平均数を示す指標で、2005年に過去最低の1.26を記録した後、やや回復傾向にあるが依然として`人口置換水準`（約2.07）を大きく下回っている
- `平均寿命`（0歳の平均余命）と`健康寿命`の差は「不健康な期間」を表し、この差を縮小することが公衆衛生上の課題である

（一言メモ）
- `年齢調整死亡率`は「昭和60年モデル人口」を基準とする直接法で算出され、人口の`高齢化`の影響を排除して死亡リスクを比較できる
- 粗死亡率が上昇しても`年齢調整死亡率`が低下していれば、健康水準は改善していると解釈できる
- 日本の女性は男性より`平均寿命`が約6年長く、この差が`年齢調整死亡率`の性差にも影響する
