# 第105回 問149（衛生）

**問題文**
我が国の医療保険制度に関する記述のうち、正しいのはどれか。２つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-149 を参照

**選択肢**
1. 国民はいずれかの公的医療保険に加入する。
2. 自らが将来使用する医療費を予め積み立てておく自助の原則による。
3. 被用者保険と国民健康保険とでは、現物給付される医療の内容は異なる。
4. 医療保険制度による医療の財源に、公費は含まれない。
5. 後期高齢者医療制度の被保険者には、75歳以上の者及び65歳以上75歳未満の寝たきり状態にある者が含まれる。

**正解:** 1、5

**解説**
解答　１、５

１　正
わが国の医療保険制度においては、基本的にすべての国民が被用者保険又は国民健康保険等の医療保険制度に加入する国民皆保険制度がとられている。
 
２　誤
わが国の医療保険制度は、収められた保険料を必要な人が使えるようにし、保険加入者全体でリスクを分散し支えあう、共助の原則による方式が採用されている。

３　誤
被用者保険と国民健康保険とでは、現物給付される医療の内容は同じである。

４　誤
医療保険制度による医療の財源は、保険料、公費、その他（患者負担など）から成り立っている。

５　正
後期高齢者医療制度の被保険者には、75歳以上の者及び65歳以上75歳未満の者で後期高齢者医療広域連合から、寝たきり等一定の障害認定を受けた者である。
