# 第105回 問64（薬理）

**問題文**
アトピー性皮膚炎の初期治療として適切なのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-64 を参照

**選択肢**
1. タクロリムス水和物の内服
2. テルビナフィン塩酸塩の外用
3. ヒドロコルチゾン酪酸エステルの外用
4. モンテルカストナトリウムの内服
5. ナジフロキサシンの外用

**正解:** 3

**解説**
解答　３

アトピー性皮膚炎は、皮膚に炎症や痒み等の症状が増悪・寛解を繰り返しながら発症する疾患である。本疾患の多くの患者は、アレルギー性疾患の家族歴・既往歴やIgE抗体を産生しやすい体質などのアトピー素因を有する。本疾患の治療としては、保湿などのスキンケア、原因となるアレルゲンの除去、薬物療法があげられる。
薬物療法としては、初期治療から副腎皮質ステロイド性薬（ヒドロコルチゾン酪酸エステルなど）の外用剤を抗炎症目的で用いる。また、その他必要に応じて抗ヒスタミン薬の内服なども補助的に行う場合もある。
