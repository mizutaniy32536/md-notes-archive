# 第105回 問56（衛生）

**問題文**
「ショック」という用語で表される状態として適切なのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-56 を参照

**選択肢**
1. 一過性の意識消失
2. 組織間液の増加による臓器不全
3. 酸素欠乏による血中の還元型ヘモグロビン上昇
4. 急激に発生する組織循環不全
5. 心拍の乱れによる不快感

**正解:** 4

**解説**
解答　４

「ショック」とは、生体に対する侵襲あるいは侵襲の対する生体反応の結果、重要臓器の血流が維持できなくなり、細胞の代謝障害や臓器障害が起こり、生命の危機に至る急性の症候群をいう。

１　誤
「失神」に関する記述である。

２　誤
「浮腫」に関する記述である。

３　誤
「チアノーゼ」に関する記述である。

４　正
冒頭解説文参照

５　誤
「動悸」に関する記述である。
