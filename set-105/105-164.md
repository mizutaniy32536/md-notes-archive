# 第105回 問164

### 第105回薬剤師国家試験 問164（薬理）

#### 問題文（リード文含む）
58歳男性。5年前より健康診断にて高血圧症を指摘されていたが放置していた。1年前には心肥大も指摘され、その頃から労作時に呼吸が苦しくなるようになった。ある日、発作性夜間呼吸困難のため、緊急入院した。入院時に浮腫が認められ、胸部レントゲンで、心肥大の増悪と肺うっ血像が認められた。

この患者に対して、症状の改善や心臓への負荷を軽減するため、作用機序の異なる2つの薬物が治療薬の候補となった。それぞれの主な作用点と作用、主な細胞内の反応、前負荷及び後負荷に及ぼす影響の組合せのうち、正しいのはどれか。**2つ選べ。**[2][3]

#### 正解番号

#### 解説
- **選択肢1**: Na⁺,K⁺-ATPase阻害（ジゴキシン）。心筋収縮力を増強するため、後負荷が増大する。誤り。[1]  
- **選択肢5**: 可溶性グアニル酸シクラーゼ活性化（リオシグアト、アデムパス）。血管拡張により前負荷・後負荷を軽減。心不全に適した記述で正しい。[1]  
- 問全体: 心不全患者に対する作用機序の異なる薬（例: ジゴキシン類似とリオシグアト）の作用点・細胞内反応・負荷影響の組合せ。正解は心臓負荷軽減に合致する2と5。[1][2][5]
