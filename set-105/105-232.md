# 第105回 問232（実務）

**問題文**
7歳男児。インフルエンザと診断され吸入薬が処方された。両親もインフルエンザで寝込んでおり、70歳の祖母が処方せんを持って来局。
薬剤師が、本剤の使用に際して祖母に行う説明として適切なのはどれか。2つ選べ。

**選択肢**
1. 吸入直後にうがいをしないと、舌にカンジダという菌が増えることがあります。
2. 吸入が上手くいかない場合には、ブリスター内の薬を飲ませてください。
3. 本剤は吸湿性が高いので、吸入器を操作してブリスターに穴をあけるのは、吸入する直前にしてください。
4. この薬剤を吸入し、平熱に戻った場合には、直ちに薬の使用を中止してください。
5. 平熱に戻っても、しばらくはウイルスの感染力が残っているため、他人に感染させる可能性があります。

**正解:** 3, 5

**解説**

1. ❌ **誤** — 本剤は免疫抑制作用をもたないため、舌にカンジダ菌が増えることはない。吸入直後にうがいをする必要はない。口腔カンジダの恐れがあるのは副腎皮質ステロイド吸入薬。

2. ❌ **誤** — 本剤は吸入により気道に直接薬剤を到達させることで効果を発揮する。ザナミビル水和物の消化管吸収率は非常に悪く、経口投与による効果はほとんど得られない。

3. ✅ **正** — 吸湿性が高いため、吸入する直前にブリスターに穴をあける。

4. ❌ **誤** — 平熱に戻っても体内にウイルスが残っている可能性がある。医師の指示通り5日間服用するよう指導する。

5. ✅ **正** — 平熱に戻っても、しばらくはウイルスの感染力が残っている。
