# 第105回 問209

### 第105回薬剤師国家試験 問209（物理・化学・生物）

**問題文**  
65歳男性。がんで入院中。当初、医療チームの方針として、アプレピタントカプセルを制吐剤として投与することが計画されていたが、口内炎が悪化したため、ホスアプレピタントの点滴静注への変更について再度検討することとなった。  

アプレピタントからホスアプレピタントが創製されたのと同様な目的で開発されたプロドラッグはどれか。1つ選べ。[1][2][4][5]

**全選択肢**  
1. オセルタミビルリン酸塩  
2. バラシクロビル塩酸塩  
3. ドカルパミン  
4. **クロラムフェニコールコハク酸エステルナトリウム**  
5. ロキソプロフェンナトリウム[2][4]

**正解番号**  
4[2][4][5]

**解説**  
アプレピタントは経口製剤（カプセル）であるが、口内炎により経口投与が困難となったため、水溶性を高めて点滴静注可能な**ホスアプレピタント**（リン酸プロドラッグ）が開発された。同様の目的（水溶性向上による注射剤化）で開発されたプロドラッグを選択する。[1][4]  

- 1: 経口吸収改善目的のプロドラッグ（エステル加水分解で活性化）。[2][4]  
- 2: 経口吸収改善目的（ペプチドトランスポーター利用）。[4][5]  
- 3: 経口投与可能化目的。[4]  
- **4: 正解。水溶性増大で注射剤化（コハク酸エステルプロドラッグ）。**[2][4]  
- 5: 消化管障害軽減目的。[2][4]
