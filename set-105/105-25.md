# 第105回 問25（化学）

**問題文**
生物化学的酸素要求量（BOD）を測定する際に用いる試験法はどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-25 を参照

**選択肢**
1. ウィンクラー法
2. 酸性高温過マンガン酸法
3. オルトフェナントロリン法
4. 重量法
5. インドフェノール法

**正解:** 1

**解説**
解答　１

生物学的酸素要求量（BOD）は20℃、5日間の溶存酸素消費量である。BODの測定は、溶存酸素測定法のウィンクラー法を用いて、前処理を行なった試料水15分後の溶存酸素と5日後の溶存酸素の差より求められる。

１　正
前記参照

２　誤
酸性高温過マンガン酸法は、化学的酸素要求量（COD）を測定する際に用いる試験法である。

３　誤
オルトフェナントロリン法は、Fe2＋を測定する際に用いる試験法である。

４　誤
重量法は、水質汚濁に係る環境基準の浮遊物質（SS）やn−ヘキサンを測定する際に用いる試験法である。なお、大気汚染に係る環境基準の浮遊粒子状物質や微小粒子状物質を測定する際に用いる試験法にも重量法がある。

５　誤
インドフェノール法は、アンモニア態窒素を測定する際に用いる試験法である。
