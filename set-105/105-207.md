# 第105回 問207

### 第105回薬剤師国家試験 問207

**科目名：** 実務（薬学実践問題）[1][2][3]

**問題文全文：**  
粉砕して服用する場合の不都合を回避するために、当該病院の採用薬の中から薬剤師が提案する薬物として、**適切でない**のはどれか。１つ選べ。ただし、これらの薬剤は全て錠剤であり粉砕して用いるものとする。[1][2][3]

**全選択肢：**  
1. ラニチジン[2]  
2. ボノプラザン[2]  
3. **オメプラゾール**[2]  
4. シメチジン[2]  
5. ファモチジン[2]

**正解番号：** 3[2][9]

**解説：**  
薬物アはラベプラゾール（PPI）で、粉砕すると胃内環境で活性化が加速され効果が低下する（問206正解1）。代替として粉砕しても問題ないH2ブロッカー（1:ラニチジン、4:シメチジン、5:ファモチジン）やカリウムイオン競合型酸ブロッカー（2:ボノプラザン）を提案すべきだが、同じPPIのオメプラゾール（3）は粉砕不可で不適切。[2][4]
