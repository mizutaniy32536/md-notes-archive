## ◆ 薬物の吸収に関する正しい記述を選ぶシナリオ

260310

**問題**
- 薬物の吸収に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**

1. 抗コリン薬は、胃内容排出速度を増大させ、経口投与された併用薬の最高血中濃度到達時間の短縮や最高血中濃度の上昇を引き起こす。
​
2. 鼻粘膜は、重層扁平上皮細胞からなり、細胞間は密着結合により強固に結合しているため、バリアー機能は高く、高分子薬物の吸収部位としては不適である。
​
3. 皮膚の角質層は皮膚を構成する層の中で最も厚いため、薬物の経皮吸収における最大の障壁となる。
​
4. 口腔粘膜からの薬物吸収は、一般に受動拡散によって起こるが、その吸収速度は部位により異なり、舌下粘膜で大きい。
​
5. 直腸下部からの薬物吸収は、門脈や肝臓を通過せずに全身循環に移行するため、肝初回通過効果を回避できる。

**正解と理由**
- `4` と `5`
- 4: 口腔粘膜吸収は受動拡散が主体。`舌下粘膜`は血流豊富で吸収速度が大きい
- 5: `直腸下部`→下直腸静脈→内腸骨静脈→門脈を迂回→肝初回通過効果回避

**他の選択肢が誤りな理由**
- 1. 抗コリン薬は胃排出速度を`低下`させる（増大ではない）→ Tmax延長、Cmax低下
- 2. 鼻粘膜は`多列線毛上皮`で薄くバリア低い→高分子（ペプチド点鼻薬等）の吸収に適する
- 3. 角質層は最も`薄い`層だが経皮吸収の最大の障壁（厚さではなく脂質構造がバリア）

**キーポイント**
- `抗コリン薬`＝胃排出速度「低下」（プロパンテリンと同じ機序）
- `舌下投与`＝受動拡散+血流豊富+初回通過効果回避

（一言メモ）
- 選択肢1の「増大」→「低下」のひっかけは108-42（プロパンテリン）と同じパターン
