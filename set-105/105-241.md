# 第105回 問241

### 第105回薬剤師国家試験 問241（衛生）

#### 問題文
有名芸能人がコカインを所持し、使用していた事件が報道された。地域の自治会より、健康サポート薬局の薬剤師にコカインの特徴や問題点について講演依頼があった。  
コカインの構造、代謝及び作用機序に関する記述のうち、正しいのはどれか。**2つ選べ**。[3][4]

#### 全選択肢
1. 神経伝達物質ドパミンと同様の骨格を有している。  
2. バルビツール酸誘導体と同様に、中枢抑制作用を示す。  
3. 耐性が生じるのは、代謝物としてΔ9−テトラヒドロカンナビノールが生成するためである。[3]

#### 正解番号
**4, 5**[5][6]

#### 解説
- **選択肢1 誤**: コカインはトロパン骨格を有しており、ドパミン（カテコールアミン骨格を有する）と異なる骨格を有する。[4]  
- **選択肢2 誤**: コカインは中枢興奮作用を示す。中枢抑制作用ではない。[1][3]  
- **選択肢3 誤**: Δ9−テトラヒドロカンナビノールは大麻の主成分であり、コカインの耐性とは関係が大きくない。[1]  
- **選択肢4 正**: （詳細は検索結果に一部欠如だが、正解として確認）[5][6]  
- **選択肢5 正**: （詳細は検索結果に一部欠如だが、正解として確認）[5][6]  
コカインはドパミン再取り込み阻害により中枢刺激作用を示し、耐性形成に関与するが、身体依存は生じにくい。[1][4]
