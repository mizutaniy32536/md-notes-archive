# 第105回 問173（薬剤）

**問題文**
薬物の胆汁中排泄に関する記述のうち、正しいのはどれか。２つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-173 を参照

**選択肢**
1. 肝実質細胞の胆管側膜上に発現し、薬物や薬物の代謝物を胆汁中へ排出するトランスポーターの多くは、ATPの加水分解エネルギーを直接利用した輸送を行う。
2. 一般に、分子量が小さい薬物ほど、胆汁中に排泄されやすい。
3. 肝実質細胞から毛細胆管中に排出された薬物は、総胆管を経て十二指腸内に分泌される。
4. グルクロン酸抱合体となることで腸管循環を受ける薬物は、腸内細菌がもつβ−グルクロニダーゼを阻害すると、血中濃度時間曲線下面積が増加する。
5. 薬物の肝クリアランスは肝臓での代謝クリアランスで表され、胆汁中への排泄クリアランスは考慮されない。

**正解:** 1、3

**解説**
解答　１、３

１　正
肝実質細胞の胆管側膜上に発現し、薬物や薬物の代謝物を胆汁中へ排出するトランスポーターの多くは1次性能動輸送担体であり、ATPの加水分解エネルギーを直接利用した輸送を行う。

２　誤
一般に、分子量が小さい薬物は、尿中に排泄されやすく、胆汁中に排泄されにくい。胆汁中排泄されやすのは、一定以上の分子量（分子量約500〜1500程度）をもつ薬物である。

３　正
肝実質細胞から毛細胆管中に排出（胆汁中排泄）された薬物は、総胆管を経て十二指腸内に分泌される。

４　誤
グルクロン酸抱合体として胆汁中に排泄された薬物は、腸内細菌がもつβ−グルクロニダーゼにより脱抱合を受け、腸管から再び吸収されることがある（腸管循環）。したがって、腸内細菌がもつβ−グルクロニダーゼを阻害すると、腸管循環による再吸収を受けにくくなるため、血中濃度時間曲線下面積は減少する。

５　誤
薬物の肝クリアランスは、肝臓での代謝クリアランスと未変化体の胆汁中への排泄クリアランスの和で表される。
