# 第105回 問296（実務）

**問題文**
59歳男性。162 cm、51 kg。慢性腎炎が進行し、13年前より血液透析（HD）治療を受けている。HDに際しては、HD開始前にそう痒予防のためにジフェンヒドラミン塩酸塩錠10 mgを4錠内服している。また、体外循環中の凝固防止の目的でナファモスタットメシル酸塩注（後発品）を用いている。最近15日間における患者の体温とCRP値は図のようになった。問296（病態・薬物治療）HD実施日に発熱が見られたため検査した結果、抗ナファモスタットIgEが陽性であることが第9日に明らかとなった。抗ナファモスタットIgEが陽性になった機序に関する記述のうち、適切なのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-296 を参照

**選択肢**
1. Th1細胞（1型ヘルパーT細胞）の指令を受けB細胞から分化した形質細胞により、ナファモスタットに特異的なIgEが産生された。
2. Th2細胞（2型ヘルパーT細胞）の指令を受けB細胞から分化した形質細胞により、ナファモスタットに特異的なIgEが産生された。
3. Th1細胞の指令を受けたT細胞から分化した形質細胞により、ナファモスタットに特異的なIgEが産生された。
4. Th2細胞の指令を受けたT細胞から分化した形質細胞により、ナファモスタットに特異的なIgEが産生された。
5. 特異的受容体と結合した感作T細胞により、ナファモスタットに特異的なIgEが産生された。問297（実務）検査結果を受けて対処した結果、第11日目以後はHD後の症状が見られなくなった。対処した内容として考えられるのはどれか。１つ選べ。
6. HD前の投与薬としてデキサメタゾン錠を追加した。
7. HD前の投与薬としてナルフラフィン塩酸塩カプセルを追加した。
8. HD中のナファモスタットメシル酸塩注射用をダルテパリンナトリウム注射液（低分子ヘパリン）に変更した。
9. 経過観察した。
10. ナファモスタットメシル酸塩注射用を後発品から先発品に変更した。

**正解:** 2

**解説**
問296　解答　２

樹状細胞などの抗原提示細胞は、Th0細胞（ナイーブヘルパーT細胞）に対して抗原提示を行い、抗原提示されたTh0細胞は、Th1細胞（1型ヘルパーT細胞）やTh2細胞（2型ヘルパーT細胞）へ分化する。その後、Th2細胞は、サイトカインを分泌してB細胞を形質細胞へ分化させ、抗原特異的な抗体を産生する。本患者のようなIgEが陽性となるⅠ型アレルギー反応では、この機構が関与している。なお、Th1細胞は、サイトカインを分泌してキラーT細胞の活性化を促し、細胞性免疫を誘導する。

問297　解答　３

問題文の図より、4日目以降のHD実施日に、発熱やCRP上昇が認められること、抗ナファモスタットIgEが陽性を示していることから、本症例ではⅠ型アレルギー反応が起きていると考えられる。ナファモスタットメシル酸塩注のアレルギー反応の発現に対しては、症状発現を予測することが困難であり、有効な予防方法はないとされている。そのため、アレルギー反応を疑った際は、ナファモスタットメシル酸塩注を中止し、代替薬を用いるなどの対応が行われる。

１　誤
前記参照。HD前に、症状発現の予防的投与としてデキサメタゾン錠を追加すること適切ではない。また、デキサメタゾン錠は血栓塞栓症の副作用を有するため、本患者での血栓形成を促進するおそれがあるため追加は不適切である。

２　誤
ナルフラフィン塩酸塩は、既存治療で効果不十分な血液透析患者におけるそう痒の改善に用いられる。本患者のHD後の症状はそう痒の増悪ではないため、ナルフラフィン塩酸塩の追加は不適切である。

３　正
前記参照。ナファモスタットメシル酸塩注によるアレルギー反応が生じた際、ナファモスタットメシル酸塩注を中止後、ダルテパリンナトリウム注射液（低分子ヘパリン）に変更することでアレルギー症状が軽減したとの報告がある。よって、本患者に対する対応として適切である。

４　誤
前記参照。経過観察のみでアレルギー症状の軽減が図れたとは考えにくい。

５　誤
ナファモスタットメシル酸塩注を、同有効成分の先発品に変更することは、対処内容として不適切である。
