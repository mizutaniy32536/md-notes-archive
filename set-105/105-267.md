# 第105回 問267

### 第105回薬剤師国家試験 問267

**科目：** 薬剤[1][2]

**問題文（全文）：**  
68歳男性。狭心症。かかりつけ医を受診し、定期的に処方1の薬剤を服用している。来局時の聞き取りにより、この患者は最近、他の医療機関で非小細胞肺がんと診断され、エルロチニブ塩酸塩錠による化学療法の実施が予定されているとのことであった。  
**問267（薬剤）**  
処方変更をしない場合に問題となる、エルロチニブの体内動態の変化として適切なのはどれか。１つ選べ。[1][2]

**全選択肢：**  
1. 吸収の低下  
2. 肝代謝の阻害  
3. 肝代謝の亢進  
4. 尿中排泄の阻害  
5. 胆汁排泄の阻害[1][2]

**正解番号：** 1[1][2][4][7]

**解説：**  
患者は狭心症治療で**エソメプラゾールマグネシウム水和物カプセル**（プロトンポンプ阻害薬、PPI）を服用中。PPIは胃内pHを上昇させ、エルロチニブの溶解・吸収を低下させるため、**吸収の低下**が生じ血中濃度が低下し効果が減弱する問題となる（併用注意）。他の選択肢は主な相互作用ではない。[1][2][9][10]  
※問266（実務）では、この相互作用を避けるためエソメプラゾールを中止する提案が正解（2）[1][2]。一部検討で問267の適切性が議論されたが、公式正解は1。[8]
