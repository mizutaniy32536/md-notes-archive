# 第105回 問31（生物）

**問題文**
ベンゾジアゼピン骨格を有し、筋弛緩作用に基づく転倒などの副作用が少ない催眠薬はどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-31 を参照

**選択肢**
1. クアゼパム
2. ゾルピデム
3. ゾピクロン
4. リルマザホン
5. トリアゾラム

**正解:** 1

**解説**
解答　１

１　正
クアゼパムは、ベンゾジアゼピンω1受容体選択性の高いベンゾジアゼピン系催眠薬であるため、ω2受容体を介した筋弛緩作用に基づく転倒などの副作用が少ない。

２　誤
ゾルピデムやゾピクロンは、作用機序はベンゾジアゼピン系催眠薬と同じではあるが、ベンゾジアゼピン骨格を有さない非ベンゾジアゼピン系催眠薬である。なお、これらの薬剤は、クアゼパム同様ω1受容体選択性が高いため、ω2受容体を介した筋弛緩作用に基づく転倒などの副作用は少ない。

３　誤
解説2参照

４　誤
リルマザホンは、生体内で代謝を受け閉環することで作用を示すベンゾジアゼピン系催眠薬であるため、元々の構造としてはベンゾジアゼピン骨格を有さない。

５　誤
トリアゾラムは、ω1、ω2受容体ともに作用しやすいベンゾジアゼピン系催眠薬であるため、筋弛緩作用に基づく転倒などの副作用に注意が必要である。
