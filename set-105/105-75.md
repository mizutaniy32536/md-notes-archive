# 第105回 問75（薬理）

**問題文**
法律において、条件及び期限付き承認の仕組みが規定されているのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-75 を参照

**選択肢**
1. 医薬品
2. 医療機器
3. 医薬部外品
4. 再生医療等製品
5. 化粧品

**正解:** 4

**解説**
解答　４

法律において、条件及び期限付き承認の仕組みが規定されているのは、再生医療等製品である。
再生医療等製品は、主に人又は動物の細胞に培養その他の加工を施した製品であり、原材料の細胞によっては品質が不均一となることがあるため、有効性の予測が困難であり、評価に時間を要する。そのため、承認申請に必要な資料を揃えるのに時間がかかり、通常の承認制度では開発されてから患者の手元まで届くのにかなりのタイムラグが出来てしまう。このような、タイムラグを避ける目的で、条件及び期限を決めて早期に承認し、市販しながらデータを集め、期限内に再度正式に承認申請を行うという条件及び期限付き承認制度が導入されることになった。
