# 第105回 問121

### 第105回薬剤師国家試験 問121（物理・化学・生物）

**問題文（科目：物理・化学・生物）**  
α–グルコシダーゼ阻害薬であるアカルボースに関する記述のうち、**誤っている**のはどれか。**1つ**選べ。[2][1]

**全選択肢**  
1. マルトース型の部分構造が含まれる。  
2. 破線で囲んだ部分の結合様式はβ1 → 4結合である。  
3. 水に溶けやすい。  
4. ヘミアセタール構造をもつため、還元性を示す。  
5. ラクターゼの活性を阻害する。[1][2]

**正解番号**  
**2**（誤っている記述は選択肢2）。[1][2][3]

**解説**  
- **選択肢1**: 正しい。問題文の構造における右半分が**マルトース型の部分構造**。[1]  
- **選択肢2**: **誤り**。破線部は**α-1,4結合**であり、β-1,4結合ではない。[1]  
- **選択肢3**: 正しい。**OH基が多い**ことから水に溶けやすい。[1]  
- **選択肢4**: 正しい。右端部分の**エーテル結合（C-O）とC-OHを有するCの部分**がヘミアセタール構造で、還元性を示す。[1]  
- **選択肢5**: 正しい。アカルボースは**二糖類分解酵素阻害薬**および**アミラーゼ阻害薬**だが、**ラクターゼは阻害しない**（酵素AとBの活性阻害）。[1][2]  

問全体の背景：問119-121は**糖質の消化・吸収過程**と**アカルボースの構造式**（図3）を用いた3連問。[2][4]
