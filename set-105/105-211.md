# 第105回 問211

# 第105回薬剤師国家試験 問211

**科目：** 実務[1]

**問題文：**

73歳男性。高血圧と糖尿病のため以下の薬剤が処方されていた。薬剤師が患者宅を訪問した際、この患者に末梢神経障害などがみられ、薬剤をPTPシートから取り出すことに不自由していた。そのため、薬剤師は、一包化することを医師に提案することにした。患者が服用しているオルメサルタンメドキソミル錠の添付文書を確認したところ、下記のような記載があった。

この処方を調剤する場合に、薬剤師の対応として**適切でないのはどれか。1つ選べ。なお、それぞれのケースにおいて患者の了承はあるものとする。**[1]

**全選択肢：**

| 番号 | 選択肢 |
|------|--------|
| 1 | オルメサルタンメドキソミル錠とそれ以外の薬剤を別々に分包する。 |
| 2 | 医薬品インタビューフォームなどを参考にし、変色が起きないと考えられる日数で分割調剤する。 |
| 3 | （選択肢3は検索結果に記載されていません） |
| 4 | （ビグアナイド系薬剤とオルメサルタンメドキソミルの一包化） |
| 5 | （選択肢5は検索結果に記載されていません） |

**正解番号：** 4[5]

**解説：**

ビグアナイド系薬（メトホルミン以外のビグアナイド系薬剤、特にブホルミン）は、グアニジノ基を有しており、オルメサルタンメドキソミルの分解産物（Ⅲ）と同様の反応を起こすため、これらの薬剤と一包化すると着色が生じます[5]。そのため、ビグアナイド系薬とオルメサルタンメドキソミルを一緒に分包することは適切ではありません[1]。

一方、オルメサルタンメドキソミル錠を他の薬剤と別々に分包したり、医薬品インタビューフォームを参考にして変色が起きないと考えられる日数での分割調剤は、適切な対応です。
