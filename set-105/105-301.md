# 第105回 問301

### 第105回薬剤師国家試験 問301

#### 問題文
75歳女性。153cm、48kg。心疾患の治療中に骨密度の低下を認め、以下の処方薬を継続して服用している。家の中で転倒後、腰痛が持続するため、かかりつけ医を受診し、整形外科病院に紹介入院となった。MRI検査の結果、腰椎圧迫骨折と診断され、1ヶ月間の入院加療と安静が指示された。患者は60歳頃から趣味で編み物をしている。喫煙歴はなく、時折、グラスに2〜3杯のワインを飲むことを楽しみにしている。[1][2][3]

**患者の退院時に薬剤師が行う指導として適切なのはどれか。2つ選べ。**[1][2][3]

#### 全選択肢
1. アレンドロン酸ナトリウム錠を服用後、30分間は飲食を控えることを伝える。[1][3]  
2. さらなる骨折はQOLを低下させるため、退院後は軽度な運動も控えることを伝える。[1][3]  
3. カルシウム製剤が処方されているので、乳製品は控えることを伝える。[1][2][3]  
4. アルコール飲料の過度の摂取は骨粗鬆症のリスク因子であることを伝える。[1][2][3]  
5. アレンドロン酸ナトリウム錠は水なしで服用することを伝える。[1][2][3]

#### 正解番号
**1, 4**[1][2]

#### 解説
アレンドロン酸ナトリウム錠は、コップ1杯程度の多めの水で服用し、服用後最低30分は横にならず飲食を控えるよう指導する。これにより食道への薬残留や胃内容物による吸収不良を防ぐため、選択肢1が適切。[1]  
患者は時折ワインを飲む習慣があるが、アルコール飲料の過度の摂取は骨粗鬆症のリスク因子であるため、選択肢4が適切。[1][2][3]  
選択肢2は誤り（適度な運動が骨折予防に有効）。選択肢3は誤り（乳製品はカルシウム源として推奨）。選択肢5は誤り（多めの水で服用が必要）。[1][2][3]
