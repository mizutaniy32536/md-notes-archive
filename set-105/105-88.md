# 第105回 問88（薬剤）

**問題文**
在宅訪問し、薬剤管理指導を実施するにあたり、医師から入手した診療情報提供書の内容を踏まえて、薬剤師が策定し、必要に応じて見直しをしていくのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-88 を参照

**選択肢**
1. サービス提供に係わる重要事項説明書
2. 居宅療養管理指導契約書
3. 薬学的管理指導計画書
4. 訪問薬剤管理指導報告書
5. 訪問薬剤管理指導記録簿

**正解:** 3

**解説**
解答　３

薬剤師が在宅訪問し、薬剤管理指導を実施する際は、医師から入手した診療情報提供書の内容を踏まえ、訪問回数や実施予定の指導項目（服薬状況の確認、副作用のチェックなど）を記載した「薬学的管理指導計画書」をあらかじめ策定しておく必要がある。この「薬学的管理指導計画書」は必要に応じ（少なくとも月に1回）見直しを行う必要がある。
