# 第105回 問166

### 第105回薬剤師国家試験 問166（病態）

#### 問題文
35歳女性。身長150 cm、体重65 kg、血圧138/90 mmHg、喫煙歴あり（30本／日）。仕事が忙しくなりストレスが溜まり、暴飲暴食で早食いとなった。ある日、突然、今までに経験したことがない心窩部痛におそわれ、心配になり近医を受診した。検査の結果、**逆流性食道炎**と診断され、下記の処方で治療が開始され、2ヶ月ほどで軽快した。

この患者に関する記述のうち、適切なのはどれか。**２つ選べ**。[1][2][4]

#### 全選択肢
1. 食道下部括約筋の**収縮**により、胃酸が逆流することにより発症した。[1][2][4]  
2. 内視鏡検査で食道に炎症やびらんが認められた。[1][2][4]  
3. **Ca拮抗薬**は、本問の患者に、現処方に併用してもより症状が改善するとは考えられない。[1]  
4. 症状がおさまれば、薬物療法の継続は必要ありません。[1]  
5. 薬物療法に加えて**禁煙や体重の減量**などの生活指導が奏功する。[1][4]

#### 正解番号
**2, 5**[1][2]

#### 解説
- **選択肢1**: 誤り。食道下部括約筋（噴門）の**緩み**により胃酸逆流が起こりやすくなる。「収縮により」ではない。[1]  
- **選択肢2**: 正しい。胃酸が逆流し、食道の粘膜に炎症やびらんが見られると逆流性食道炎と診断される。内視鏡検査で確認される。[1][2]  
- **選択肢3**: 誤り。Ca拮抗薬は降圧・狭心症などに用いられ、本患者の症状改善に寄与しない。[1]  
- **選択肢4**: 誤り。症状軽快後も再発防止のため、必要に応じて薬物療法の継続を考慮する。[1]  
- **選択肢5**: 正しい。喫煙・肥満（BMI≈28.9）が逆流性食道炎の危険因子であり、禁煙・減量などの生活指導が有効。[1][4]
