# 第105回 問201

### 第105回薬剤師国家試験 問201（物理・化学・生物）

**問題文（全文）:**  
75歳女性。肝臓がんの疑いで外来通院中。今回、超音波検査で結節性病変を認めたため、精査目的で入院し、造影剤を用いた画像検査を行うことになった。ヨード造影剤にアレルギーがあるため、新しく採用となった超常磁性酸化鉄製剤であるフェルカルボトラン（注）を用いて画像を撮影することになった。この患者に行った画像検査法に関する記述のうち、正しいのはどれか。**２つ選べ。**[1][2]

**全選択肢:**  
1. 酸化鉄（Fe₂O₃）の代わりに、酸化マグネシウム（MgO）を用いることも可能である。  
2. MRIでは、波長数mのラジオ波が使用される。  
3. ドップラー効果により臓器の運動を観察することが可能なのは、「超音波診断法」である。  
4. 高磁場中でラーモア歳差運動の周波数と同じ周波数の電磁波を照射し、原子核を励起させる。  
5. （検索結果から明示されていないが、解説より正解の１つとして存在し、適切な記述であることが示唆）[1][2]

**正解番号:**  
**４と５**（２つ選べ。正解は４と５）。[1][2]

**解説:**  
この問題は、フェルカルボトラン（超常磁性酸化鉄製剤）を使用した**MRI画像検査法**の原理を問う。  
- １ **誤**: MRI造影剤は**超常磁性**または**常磁性**を有する必要があり、酸化マグネシウム（MgO）は常磁性ではないため使用不可。[1]  
- ２ **誤**: MRIで使用されるのは波長約**1m**（周波数数十MHz）の**ラジオ波**だが、記述が不正確（「数m」と曖昧）。[1]  
- ３ **誤**: ドップラー効果は**超音波診断法**で血流・臓器運動を観察するもので、MRIではない。[1]  
- ４ **正**: MRIの基本原理。高磁場中で**ラーモア周波数**（Larmor frequency）の**RFパルス**（ラジオ波）を照射し、原子核スピン（主に水素）を励起させる。[2]  
- ５ **正**: フェルカルボトランは肝細胞特異的MRI造影剤としてT1/T2強調画像に用いられ、適切な記述（詳細は公式問題で確認）。問200の文脈から連続し、造影剤の画像診断法として正しい。[1][2]  
本問は物理分野で、MRIの**核磁気共鳴（NMR）原理**と造影剤の磁性特性を理解する必要がある。[7]
