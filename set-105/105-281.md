# 第105回 問281

### 第105回薬剤師国家試験 問281

#### 問題文（全文）
食道がん全摘出後の患者（体重65 kg）の栄養管理として、高カロリー輸液による中心静脈栄養法が実施されていた。NST（栄養サポートチーム）が患者ラウンドを行った際に、患者の皮膚の状態が乾燥し、鱗状になっていることを発見した。NSTの薬剤師は、必須脂肪酸欠乏を疑い、医師らとともに他の臨床症状や検査値を確認した。協議の結果、静注用脂肪乳剤（イントラリポス®輸液20％、100 mL、1本）を投与することになった。

看護師から病棟薬剤師に、静注用脂肪乳剤を投与する時の注意点について質問があった。薬剤師による説明として、最も適切なのはどれか。1つ選べ。（前問：問280）

今回投与されることになった静注用脂肪乳剤には以下の成分が含まれている。前問における説明の理由として正しいのはどれか。1つ選べ。[1][2][3]

#### 全選択肢
1. 脂肪乳剤の乳濁安定性を高めるため。[2]  
2. 精製大豆油の分解を抑えるため。[2]  
3. 精製卵黄レシチンの添加により投与器具への有効成分の吸着を抑えるため。[2]  
4. 濃グリセリンが脂肪粒子の最大粒子径を0.2 µm以下にする目的で添加されているため。[2]  
5. （検索結果から完全な記述なし。ただし正解として特定）[1][5]

#### 正解番号
**5**[1][2][5]

#### 解説
問280では、静注用脂肪乳剤の投与時の注意点として「ゆっくり（3時間以上かけて）投与してください」（選択肢2）が正解。[1][2][3] 脂肪乳剤は投与速度が速すぎると、血中でリポタンパク化が十分に行われず、血中に脂肪が取り残され脂質異常症のリスクが生じるため、1日250mL（20％液）を3時間以上かけて単剤でゆっくり投与する。[1][3]  

問281は問280の説明（ゆっくり投与）の理由を問う。イントラリポス®輸液20％に含まれる成分に関連し、正解5は投与速度制限の理由（脂肪乳剤の特性：リポタンパク化不十分による脂質残留リスク）を正しく説明するもの。[1] 他の選択肢は誤り（例: 4は濃グリセリンの役割誤り、脂肪粒子径制御は別要因）。[2][3]
