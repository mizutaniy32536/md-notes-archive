# 第105回 問110（化学）

**問題文**
骨の成長とリモデリングに関する記述のうち、正しいのはどれか。２つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-110 を参照

**選択肢**
1. 骨基質の生成に必要なコラーゲンは、主に骨細胞から分泌される。
2. 骨芽細胞は、アルカリホスファターゼの作用により、石灰化に必要なリン酸の局所濃度を上昇させる。
3. 破骨細胞は、骨基質を分解する働きをもつ。
4. エストロゲンは、思春期において骨芽細胞の働きを抑制する。
5. カルシトニンは、破骨細胞を活性化する。

**正解:** 2、3

**解説**
解答　２、３

骨のリモデリングとは、破骨細胞による骨吸収と骨芽細胞による骨形成を繰り返すことにより、新しい骨へと置換する現象をいう。破骨細胞による骨吸収と骨芽細胞による骨形成のバランスが、骨の成長に関与している。

１　誤
骨基質の生成に必要なコラーゲンは、主に骨芽細胞から分泌される。

２　正
骨芽細胞は、アルカリホスファターゼ活性を有しており、様々な有機リン酸エステルを分解してリン酸の局所濃度を上昇させる。生じたリン酸はカルシウムと結合して、リン酸カルシウムからなるヒドロキシアパタイトが形成され、石灰化が進行する。

３　正
破骨細胞は、酸性ホスファターゼを分泌し、古くなった骨基質を溶解し骨吸収を促進させる。

４　誤
思春期になると、女性では卵巣でエストロゲンが産生され始め、骨芽細胞の働きを促進及び破骨細胞の働きを抑制することで、骨の形成を促進する。つまり、女性における思春期の急速な成長にはエストロゲンが深く関与している。

５　誤
カルシトニンは甲状腺傍ろ胞細胞から分泌されるホルモンで、破骨細胞の働きを抑制する。
