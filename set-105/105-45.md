# 第105回 問45（生物）

**問題文**
図の1〜5のうち、イヌリンの血漿中濃度と腎クリアランスとの関係を示すのはどれか。１つ選べ。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-45 を参照

**選択肢**
（選択肢は画像を参照）

**正解:** 3

**解説**
解答　３

イヌリンは、血漿タンパク質とほとんど結合せず、腎臓の糸球体ろ過のみにより排泄される(尿細管分泌、尿細管再吸収を受けない)ので、イヌリンの腎クリアランスは糸球体ろ過速度（GFR）にほぼ等しくなる。GFRは、血漿中濃度が増加しても変化せず一定であるため、イヌリンの血漿中濃度が変化してもその腎クリアランスは一定である。よって該当するグラフは3となる。
