# 第105回 問120

### 第105回薬剤師国家試験 問120

**科目**: 物理・化学・生物（生物）[1][2][3][4]

**問題文全文**:  
小腸粘膜上皮細胞における糖の輸送過程に関する記述のうち、正しいのはどれか。2つ選べ。[1][3][4]

**全選択肢**:  
1. ガラクトースは、単純拡散により細胞膜を通過して細胞内に取り込まれる。[1][3][4]  
2. 輸送体Dは、細胞内外のNa⁺の濃度勾配を利用して、グルコースを細胞内に取り込む。[1][3]  
3. （選択肢3の詳細は検索結果に明記なし。ただし一部ソースで妥当と記述あり。[1]）  
4. （選択肢4の詳細は検索結果に明記なし。ただし一部ソースで妥当と記述あり。[1]）  
5. Na⁺,K⁺–ATPaseについての記述（細胞内外のNa⁺を細胞内に、K⁺を細胞外に輸送するなど）。[1][3]

**正解番号**: 2, 5[1][4][5][6]

**解説**:  
- 選択肢1: 誤り。ガラクトースはSGLT1により能動輸送され、単純拡散ではない。[1][4]  
- 選択肢2: 正しい。輸送体D（SGLT1）はNa⁺濃度勾配を利用してグルコースを細胞内に取り込む。[1][3]  
- 選択肢5: 正しい。Na⁺,K⁺–ATPaseはNa⁺を細胞外へ、K⁺を細胞内へ輸送し、二次的能動輸送の駆動力となる。[1]  
（注: 図2（グルコース輸送過程）を基にした問題で、選択肢3・4の詳細は一部ソースで正しいとされるが、正解は2と5で一致。[1][3][4][5][6] 公式問題は厚生労働省PDFで確認可能。[7]）
