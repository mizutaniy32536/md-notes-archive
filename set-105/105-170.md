# 第105回 問170（薬剤）

**問題文**
薬物の生体内分布に関する記述のうち、正しいのはどれか。２つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-170 を参照

**選択肢**
1. 腎臓や小腸粘膜の毛細血管は有窓内皮であるため、低分子薬物は窓構造に存在する小孔を介して毛細血管を透過できる。
2. 皮下に投与された分子量5,000以下の薬物は、毛細血管よりも毛細リンパ管に移行しやすい。
3. アルブミンは血漿中に約4 g/dL存在し、プロプラノロールやイミプラミンのような塩基性薬物と強く結合する。
4. プラバスタチンは肝細胞の血液側細胞膜に発現する有機アニオントランスポーターOAT1B1を介して細胞内に取り込まれる。
5. 血漿タンパク結合率が高い弱酸性薬物は、乳汁中への移行性が高い。

**正解:** 1、4

**解説**
解答　１、４

１　正
腎臓や小腸粘膜の毛細血管は有窓内皮であり、構成する内皮細胞同士が密に接しているが、細胞膜が融合した薄い部分である窓構造をもつ。低分子薬物は窓構造に存在する小孔を介して毛細血管を透過できる。

２　誤
一般に、皮下に投与された分子量5,000以下の薬物は、毛細リンパ管よりも毛細血管に移行しやすく、分子量5,000以上の薬物は、毛細血管よりも毛細リンパ管に移行しやすい。

３　誤
アルブミンは血漿中に約4 g/dL存在する血漿タンパク質であり、ワルファリンやインドメタシンなどの酸性薬物と結合しやすい。一方、プロプラノロールやイミプラミンのような塩基性薬物は、α1−酸性糖タンパク質と結合しやすい。

４　正
プラバスタチンは肝細胞の血液側細胞膜に発現する有機アニオントランスポーターOAT1B1を介して細胞内に取り込まれた後、胆管側細胞膜に発現するMRP2（multidrug resistance associated protein 2）を介して胆汁中に排泄される。

５　誤
薬物の乳汁中への移行は一般に受動拡散（単純拡散）に従い、水溶性薬物やイオン性薬物、血漿タンパク結合率が高い薬物などは乳汁中への移行性が低く、脂溶性薬物や非イオン性薬物、血漿タンパク結合率が低い薬物などは乳汁中への移行性が高い。また、乳汁中のpHは約6.8であり、血漿のpHの約7.4より酸性よりであるため、塩基性薬物は血漿中において分子形分率が高くなり、母乳中においてイオン形分率が高くなるため、乳汁中に移行しやすい。したがって、血漿タンパク結合率が低い塩基性薬物は、乳汁中への移行性が高い。
