# 第105回 問251

### 第105回薬剤師国家試験 問251（実務）

**問題文（全文）**  
75歳男性。大腸がんステージⅣに対して、mFOLFOX6療法により治療中である。原発巣の痛みに対し、処方1の薬剤が投与されていたが痛みのコントロールが不十分であった。しかし、開始18日目に下痢、発汗、発熱、見当識障害が出現し、開始25日目に不眠、焦燥感、体重減少を訴え、処方薬による副作用が疑われ、血液検査を行った。（注：処方1はアセトアミノフェン、アミトリプチリン、トラマドールを含む痛み止め処方。問250で副作用の機序がトラマドールのセロトニン再取り込み阻害作用等と関連。）[1][2][9]

**副作用の改善を目的とした医師への処方提案として、最も適切なのはどれか。１つ選べ。**  
1. アセトアミノフェンを中止し、オキシコドンに切り替える。  
2. アミトリプチリンを中止し、トラマドールを増量する。  
3. アミトリプチリンを中止し、デュロキセチンに切り替える。  
4. **トラマドールを中止し、オキシコドンに切り替える。**  
5. トラマドールを中止し、メサドンに切り替える。[1][2]

**正解番号**  
4[1][2][7]

**解説**  
患者はトラマドールのセロトニン再取り込み阻害作用によるセロトニン症候群様の副作用（下痢、発汗、発熱、見当識障害、不眠、焦燥感、体重減少）を呈している。高齢者で腎機能正常（クレアチニン0.9 mg/dL）だが、トラマドールの活性代謝物O-デスメチルトラマドールが蓄積しやすいため中止が必要。オキシコドンは純粋なオピオイドμ受容体刺激薬でセロトニン作用がなく、痛みコントロールが可能で適切な代替。1はアセトアミノフェン関連なし、2はトラマドール増量で悪化、3はデュロキセチンもSNRIでセロトニン作用あり、5はメサドンもセロトニン再取り込み阻害作用ありで不適。[1][2][9]
