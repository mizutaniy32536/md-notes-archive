# 第105回 問289

### 第105回薬剤師国家試験 問289

**問題文（全文）**  
28歳女性。日頃から月経による出血量が多く、痛みも強い。食事は炭水化物中心で不規則だった。3ヶ月前から階段を上がるときに息切れを感じていた。2週間前より動悸、息切れ、めまいなどの自覚症状が増強した。かかりつけ医を受診したところ、顔面や眼瞼結膜の蒼白などの他覚症状が認められ、血液検査でヘモグロビン値7.2 g/dL、赤血球数260万/μLであった。下記の薬剤が処方され来局した。  
薬局の薬剤師は患者から、膝の外傷部の化膿に対して整形外科からセフジトレンピボキシル錠と耐性乳酸菌製剤が処方され、数日前から服用中であることを聴取した。  
薬物治療によってヘモグロビン値が11.6 g/dL、赤血球数が390万/μLとなった。改善が期待できる臨床所見として適切なのはどれか。**2つ選べ。**[1][2][3]

**全選択肢**  
1. 点状出血  
2. 歩行障害  
3. スプーン状爪  
4. 舌炎  
5. 脾腫[1][2][5]

**正解番号**  
3, 4[1][2][3]

**解説**  
本症例は鉄欠乏性貧血（月経過多・不規則な炭水化物中心の食事による鉄不足が原因）と診断され、鉄剤投与によりヘモグロビン値・赤血球数が改善した。鉄欠乏性貧血の特徴的な臨床所見である**スプーン状爪（こいれんとう甲）**と**舌炎**が改善が期待できる。[1][2][3]  
- 1. 点状出血：再生不良性貧血などで見られるため不適切。  
- 2. 歩行障害：本症例の症状とは関連なし。  
- 5. 脾腫：溶血性貧血などで見られるため不適切。[1]
