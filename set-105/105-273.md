# 第105回 問273

### 第105回薬剤師国家試験 問273

**科目名**: 薬学実践問題（実務）[1][2][3]

**問題文全文**:  
54歳女性。152cm、48kg。高血圧、脂質異常症、深在性皮膚真菌症の治療のため処方1と処方2の薬剤を服用していた。その後、深部静脈血栓塞栓症を発症し、その治療のため処方3が追加となった。  
遺伝子多型検査の結果、ホモの変異を有することが判明し、医師は代替薬について薬剤師に相談した。医師に提案すべき抗血栓薬として最も適切なのはどれか。1つ選べ。[1][2][3]

**全選択肢**:  
- 1. アピキサバン錠  
- 2. シロスタゾール錠  
- 3. ダビガトランエテキシラートメタンスルホン酸塩カプセル  
- 4. チクロピジン塩酸塩錠  
- 5. リバーロキサバン錠[1][2][3]

**正解番号**: 1[1][2][3][10]

**解説**:  
処方3に含まれるワルファリン（抗凝固薬）の代替薬を提案する必要がある。選択肢のうち、アピキサバン、リバーロキサバン、ダビガトランは直接作用型経口抗凝固薬（DOAC）として抗凝固作用を示すため適切。一方、シロスタゾールとチクロピジンは抗血小板薬であるため、深部静脈血栓塞栓症治療の代替としては不適切。したがって、最も適切な提案はアピキサバン（エリキュース）である。[1][3]
