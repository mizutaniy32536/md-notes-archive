# 第105回 問197

### 第105回薬剤師国家試験 問197

**科目**: 物理・化学・生物

**問題文全文**:  
処方2の薬剤は、酸化水酸化鉄(FeO(OH))が主成分である。酸化水酸化鉄は水酸化鉄(Ⅲ)(Fe(OH)₃)から水(H₂O)が脱離したものである。酸化水酸化鉄及び水酸化鉄(Ⅲ)に関する記述のうち、正しいのはどれか。**２つ選べ。**[1][2][3]

**全選択肢**:  
1. 純水中における水酸化鉄(III)の溶解度積は、その濃度に依存して変化する。  
2. 純水中における水酸化鉄(III)の溶解度積は、溶液の温度によって変化する。  
3. 水酸化鉄(III)の溶解度は、溶液のpHによって変化しない。  
4. 酸化水酸化鉄によるリン酸の吸着にはpHが影響する。  
5. 酸化水酸化鉄によるリン酸の吸着に剤形や比表面積は影響しない。

**正解番号**: **2・4**[3][4][7]

**解説**:  
- 選択肢1: 溶解度積(Ksp)は温度によって決まる定数であり、濃度に依存して変化しないため誤り。[2]  
- 選択肢2: 正しい記述（溶解度積に関する正しい特性）。[2]  
- 選択肢4: 正しい記述（酸化水酸化鉄および水酸化鉄(Ⅲ)の特性に関する正解）。[4][7]  
酸化水酸化鉄はスクロオキシ水酸化鉄（例: ピートル）などの高リン血症治療剤で、リン吸収抑制に用いられ、透析患者の文脈で胃部不快感から処方変更。[1][2][3]
