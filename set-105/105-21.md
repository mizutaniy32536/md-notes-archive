## ◆ 我が国の死因別死亡率の順位から該当する死因を判断するシナリオ
260307

**問題**
- 平成28年における我が国の死因別死亡率の第2位に該当する死因はどれか。1つ選べ。

**選択肢**

1. 自殺
​
2. 不慮の事故
​
3. 肺炎
​
4. 心疾患
​
5. 悪性新生物

**正解と理由**
- `4. 心疾患`
- **理由の構成**
	- **問題の整理**：平成28年における日本の`死因別死亡率`の第2位に該当する死因を選択する問題
	- **薬学的知識・仕様・制約**：平成28年の死因別死亡率の順位は、1位 `悪性新生物`（28.5%）→ 2位 `心疾患`（15.1%）→ 3位 `肺炎`（9.1%）→ 4位 `脳血管疾患`（8.4%）→ 5位 `老衰`（7.1%）である。心疾患には`虚血性心疾患`（心筋梗塞・狭心症）や`心不全`などが含まれ、高齢化に伴い死亡率は増加傾向にある
	- **選択理由**：平成28年の統計データにおいて`心疾患`は死亡率15.1%で第2位であり、問題の条件に合致するため選択肢4が正解と判断される

**解法の手順（思考の流れ）**
- まず「死因別死亡率の第2位」という条件を確認し、上位の死因順位を想起する
- 日本の死因別死亡率の不動の1位は`悪性新生物`であり、選択肢5を第1位として除外する
- 次に、第2位は`心疾患`であることを想起し、選択肢4が該当すると判断する
- `肺炎`は第3位、`脳血管疾患`は第4位であることも確認し、選択肢3も除外する
- `自殺`や`不慮の事故`は上位5位には入らないため、選択肢1・2も除外する
- これにより`選択肢4`（心疾患）が正解であると結論づけた

**他の選択肢が誤りな理由**
- 1. 自殺
	`自殺`は平成28年の死亡率1.6%で8位相当であり、第2位には該当しない。日本の自殺者数は2003年をピークに減少傾向にあるが、依然として`社会的問題`として重要視されている

- 2. 不慮の事故
	`不慮の事故`は平成28年の死亡率2.9%で6位相当であり、第2位には該当しない。不慮の事故には`交通事故`、転倒・転落、溺死などが含まれる

- 3. 肺炎
	`肺炎`は平成28年の死亡率9.1%で`第3位`であり、第2位ではない。なお平成29年以降、`誤嚥性肺炎`が肺炎から独立して集計されるようになったため、肺炎の順位は変動している

- 5. 悪性新生物
	`悪性新生物`は平成28年の死亡率28.5%で`第1位`であり、第2位ではない。悪性新生物は1981年以降、日本の死因第1位を維持し続けている

**キーポイント**
- 日本の死因別死亡率の上位3位は「`悪性新生物` → `心疾患` → `肺炎`」の順で、この順位は長年安定していた
- 平成29年以降、`誤嚥性肺炎`の独立集計により肺炎の順位が低下し、`老衰`が上位に浮上する変動が起きている
- `心疾患`による死亡は`高齢化`に伴い増加傾向にあり、虚血性心疾患と心不全が主な内訳である
- 死因統計の問題では「何年の統計か」に注意が必要であり、年次によって3位以降の順位は変動する

**関連知識の比較**
- `粗死亡率`と`年齢調整死亡率`：粗死亡率では悪性新生物が1位だが、年齢調整死亡率では悪性新生物の死亡率も低下傾向にあり、高齢化の影響を除くと実質的な死亡リスクは改善している
- `脳血管疾患`はかつて日本の死因第1位であったが、`降圧薬`の普及や`食塩摂取量`の減少により1981年以降は順位を下げた
- 平成30年以降の死因順位の変動：1位 `悪性新生物` → 2位 `心疾患` → 3位 `老衰` → 4位 `脳血管疾患` → 5位 `肺炎`と、`老衰`の上昇が顕著である
- `老衰`の増加は超高齢社会を反映しており、死亡診断書における`老衰`の記載基準の変化も影響している

（一言メモ）
- 日本の死因別死亡率は「`悪性新生物` → `心疾患`」の1-2位が長年不動であり、3位以降は`誤嚥性肺炎`の独立集計や`老衰`の増加により年次変動がある
- `脳血管疾患`は1951年から1980年まで死因第1位であったが、`降圧療法`の進歩と減塩指導の普及により大幅に減少した
- 死因統計の問題では出題年と統計年の対応に注意し、特に平成29年前後の`肺炎`と`老衰`の順位逆転を意識することが重要
