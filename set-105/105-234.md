# 第105回 問234（衛生）

**問題文**
性的接触で感染し、同性間性的接触が異性間より多く、ニューモシスチス肺炎やカンジダ症を合併する感染症（HIV/AIDS）。我が国におけるこの感染症の男女別の発生動向を示した図はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図はe-REC 105-234 を参照

**選択肢**
1. 図1
2. 図2
3. 図3
4. 図4
5. 図5

**正解:** 1

**解説**

HIV感染症は全数把握対象疾患。我が国のHIV感染症の特徴：
- **男性が女性よりも圧倒的に多い**
- 女性の患者数は横ばい
- 男性の患者数は2000〜2008年は増加傾向、それ以降は横ばい
- 縦軸が患者数（全数把握）

したがって、選択肢1が正答。

各選択肢の図が示す感染症：
- 1: **HIV感染症**（正答）
- 2: 性器クラミジア感染症
- 3: 性器ヘルペス感染症
- 4: 淋菌感染症
- 5: 梅毒
