# 第105回 問277

### 第105回薬剤師国家試験 問277

#### 問題文（全文）
アザチオプリンで治療を継続していたところ、9ヶ月後にウイルスの再活性化が確認され、患者の腎機能を考慮して、核酸アナログである**テノホビルアラフェナミドフマル酸塩**に変更されることになった。

「**ア**」及び「**イ**」に適する語句の組合せとして最も適切なのはどれか。１つ選びなさい。[1][2][4][5]

（注：問題はテノホビルアラフェナミドフマル酸塩の特徴を問うもので、問276の症例（HBc抗体陽性患者のB型肝炎ウイルス再活性化）を継続。eGFR 39mL/min/1.73m²の腎機能低下を考慮。[1][2]）

#### 全選択肢
|  | ア | イ |
|---|----|----|
| **1** | テノホビルジソプロキシルフマル酸塩 | 血漿中で未変化体でいるものが、標的細胞内に入り薬効を発揮する |
| **2** | テノホビルジソプロキシルフマル酸塩 | 血漿中で加水分解されてテノホビルになると、細胞内に入れず、薬効を消失する |
| **3** | テノホビル | 血漿中で未変化体でいるものが、標的細胞内に入り薬効を発揮する |
| **4** | テノホビル | 血漿中で加水分解されてテノホビルになると、細胞内に入れず、薬効を消失する |
| **5** | テノホビルアラフェナミド | 標的細胞内において、テノホビル二リン酸がより高い濃度で産生される |[1][4][5]

（注：検索結果から選択肢の構造を再現。問277はア・イ穴埋め形式で、テノホビル系薬のプロドラッグ特性と腎機能考慮を比較。[5]）

#### 正解番号
**2**[1][2][4]

#### 解説
HBc抗体陽性でB型肝炎ウイルス再活性化が起こり、腎機能低下（eGFR 39）を考慮してテノホビルアラフェナミドフマル酸塩へ変更。テノホビルアラフェナミドはテノホビルの**プロドラッグ**で、経口吸収性を改善し、血漿中で未変化体として標的細胞（肝細胞等）に入り、細胞内でテノホビル二リン酸に変換されて高い濃度で産生される。一方、**テノホビルジソプロキシルフマル酸塩**は血漿中で加水分解されてテノホビルになると細胞内移行が悪く薬効消失し、腎負荷が高いため不適。アラフェナミドは腎排泄が少なく腎機能低下患者に適する。正解**2**は「ア：テノホビルジソプロキシルフマル酸塩」「イ：血漿中で加水分解されてテノホビルになると、細胞内に入れず、薬効を消失する」を指し、これを対比してアラフェナミドの優位性を問う。[1][2][4][5]
