# 第105回 問278（実務）

**問題文**
5歳女児。熱や咳の症状があり母親に連れられて受診し、下記内容の処方箋を持って来局した。母親との会話において、子どもは以前にも散剤や顆粒剤を服用したことがあり、問題なく飲めたとのことであった。また、母親が右手に包帯をしており、物を取り扱いにくいため、一包にまとめて欲しいと医師に申し入れ済みとのことであった。問278（薬剤）薬剤師はこの処方の調剤方法を検討するため、アスベリン®散と幼児用PL配合顆粒の処方量を十分に振とう混合してみた。このときの状態を模式的に表す図として、最も適切なのはどれか。１つ選べ。問279（実務）前問での検討結果を踏まえて、この患者に対する最も適切な調剤方法はどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-278 を参照

**選択肢**
1. 両剤を計量し、乳鉢にて、乳棒で混和し分包した。
2. アスベリン®散を乳糖で賦形したものと幼児用PL配合顆粒を、2段分割（2度撒き）で分包した。
3. 両剤を計量し、乳鉢にて、スパーテルで混和し分包した。
4. 幼児用PL配合顆粒を調剤用ミルで粉砕してから、アスベリン®散と乳棒・乳鉢にて混和し分包した。
5. アスベリン®散を乳糖で賦形したものをふるいで篩過してから、幼児用PL配合顆粒と乳棒・乳鉢にて混和し分包した。

**正解:** 1

**解説**
問278　解答　１

一般に、粒子径の異なる粒子を容器に充填し十分に振とうすると、幼児用PL配合顆粒のような大粒子が飛び上がった隙間にアスベリン®散のような小粒子が入り込むことで、下方に小粒子、上方に大粒子が次第に集まり分離する。
したがって、アスベリン®散と幼児用PL配合顆粒を十分に振とう混合すると、選択肢1のような状態となる。

問279　解答　２

前問の結果より粒子径の異なる粒子（アスベリン®散と幼児用PL配合顆粒）を混和すると、分離する可能性がある。そのため、アスベリン®散を乳糖で賦形したものと幼児用PL配合顆粒を、直接混和せず2段分割（2度撒き）で分包する。
なお、一般に散剤と顆粒剤を一包化する際は2剤の混和は行わず、2段分割（2度撒き）で分包する。
