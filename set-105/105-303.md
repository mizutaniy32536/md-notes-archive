# 第105回 問303

### 第105回薬剤師国家試験 問303

**問題文（実務）**  
薬剤師が主治医に提案すべき薬物として最も適切なのはどれか。１つ選べ。[1][8]

**全選択肢**  
1. アセトアミノフェン  
2. コデインリン酸塩水和物  
3. トラマドール塩酸塩  
4. アスピリン  
5. **プレガバリン**[1][5]

**正解番号**  
5[1][5]

**解説**  
58歳女性の左乳がん再発・転移患者で、AC療法4コース後、再発時に鋭い強い痛みに対し鎮痛薬が処方され、5コース目来院時に指先のしびれ・感覚消失・電撃痛（神経障害性疼痛）を訴え。神経障害性疼痛に**プレガバリン**が第一選択薬として適しており、Ca阻害による鎮痛効果が期待される。他の選択肢（アセトアミノフェン、コデイン、トラマドール、アスピリン）は主に非オピオイド鎮痛薬やオピオイドで、神経障害性疼痛への特異性に劣る。[1][5][9]
