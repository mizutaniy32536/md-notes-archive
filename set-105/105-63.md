# 第105回 問63（薬理）

**問題文**
活動電位持続時間を短縮するのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-63 を参照

**選択肢**
1. アミオダロン
2. ジソピラミド
3. ソタロール
4. フレカイニド
5. メキシレチン

**正解:** 5

**解説**
解答　５

１　誤
アミオダロンやソタロールは、Vaughan Williams 分類のⅢ群に分類され、活動電位持続時間（APD）を延長する。

２　誤
ジソピラミドは、Vaughan Williams 分類のⅠa群に分類され、APDを延長する。

３　誤
解説1参照

４　誤
フレカイニドは、Vaughan Williams 分類のⅠc群に分類され、APDを変化させない。

５　正
メキシレチンは、Vaughan Williams 分類のIb群に分類され、APDを短縮する。
