# 第105回 問291

### 第105回薬剤師国家試験 問291

#### 問題文（全文）
17歳男性。身長170cm。断続的に続く腹痛と下痢を呈し、3ヶ月間で体重が60kgから54kgへと減少した。最近は、38℃前後の発熱を認めることがある。内視鏡検査の結果、回盲部と空腸から横行結腸にかけて非連続的な潰瘍病変が観察された。そこで、プレドニゾロン（50 mg/日）とメトロニダゾール（750 mg/日）による治療が開始された。なお、この患者はB型及びC型肝炎ウイルスには感染していない。

治療開始後も症状改善が見られないため、10月中旬より入院して**アダリムマブ**による治療を開始することになり、患者の治療方針を医療チームで話し合うことになった。薬剤師がチームに提案することとして適切なのはどれか。**2つ選べ。**[1][2][4][8]

#### 全選択肢
1. 抗アダリムマブ抗体の有無を検査すること。[2][4]  
2. 流行に備えて、インフルエンザワクチンを接種すること。[2][4]  
3. 胸部レントゲン検査を行い、結核感染の有無を調べること。[2][4]  
4. レジパスビル/ソホスブビル配合錠を投与すること。[1][2][4]  
5. 抗ミトコンドリア抗体の有無を検査すること。[1][2][4]

#### 正解番号
**2, 3**[1][2][5][6]

#### 解説
本問の症例はクローン病（回盲部と空腸から横行結腸にかけての非連続性潰瘍）と診断され、プレドニゾロンとメトロニダゾールで改善せず、アダリムマブ（ヒト型抗ヒトTNF-αモノクローナル抗体）へ移行。[1][2][4]  

- **選択肢1**: アダリムマブの効果減弱要因として抗アダリムマブ抗体が考えられるが、投与開始前に検査は通常不要。誤り。[2]  
- **選択肢2（正解）**: アダリムマブは免疫抑制薬のため、生ワクチン接種は禁忌だが、不活化ワクチン（インフルエンザワクチン）は投与前に推奨。10月中旬は流行期に備えるのに適切。[1][2][4]  
- **選択肢3（正解）**: TNF阻害薬のアダリムマブ開始前に結核感染スクリーニング（胸部レントゲン、インターフェロンγ遊離試験など）が必須。活動性結核を除外。[1][2][4]  
- **選択肢4**: C型肝炎治療薬だが、問題文でC型肝炎非感染のため不要。誤り。[1][2][4]  
- **選択肢5**: 原発性胆汁性胆硬化症などの自己免疫性肝疾患関連だが、本症例に無関係。誤り。[1][2][4]  

アダリムマブ使用時の安全性確保（ワクチン・結核スクリーニング）が薬剤師の適切な提案。[1][2]
