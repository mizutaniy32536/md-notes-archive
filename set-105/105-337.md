# 第105回 問337（実務）

**問題文**
アドレナリン自己注射用キット製剤において、使用時に針が出ないという不具合が報告され、独立行政法人医薬品医療機器総合機構（PMDA）より、クラスⅠの回収情報が出された。この報告を受けて、ある病院の薬剤師が該当ロットの製剤の納品履歴があるかどうかを確認したところ、6ヶ月前に1本納品されていたが、調剤済みで在庫はなかった。該当ロットの製剤の使用期限はあと9ヶ月程度残っていることが判明した。この病院がとるべき対応として最も適切なのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-337 を参照

**選択肢**
1. クラスⅠの回収であるため、今後の様子を見守る。
2. 該当ロットの製剤を調剤された患者に連絡をし、未使用であれば代替品と交換する。
3. 病院ホームページにおいて、該当ロットを公表し、患者からの連絡を待つ。
4. 患者から該当ロットの製剤を回収し、代替品を提供し実費を請求する。
5. 該当ロット以外の製剤についても、可能な限り回収する。

**正解:** 2

**解説**
解答　２

医薬品の回収は、回収される製品によりもたらされる健康への危険性の程度により、以下のとおり個別回収ごとに、Ⅰ、Ⅱ又はⅢの3つのクラスに分類されている。

クラスⅠの回収情報が出されているので、この製品の使用等が、重篤な健康被害又は死亡の原因となり得る状況である。この病院では、該当ロットの製剤が1本患者の手に渡っているので、速やかに患者に連絡をとり、使用した患者の状況を把握するなど、しかるべき対応をとらなければならない。

１　誤
クラスⅠに分類される医薬品のため、患者の状態を確認し必要に応じて受診勧奨を行うとともに、該当する医薬品の回収および代替品と交換（使用の有無に関わらず）を行うことが適切である。

２　正
解説１参照。

３　誤
回収情報が出された場合、該当する医薬品を薬剤師が直ちに回収する。

４　誤
回収情報が出された場合、該当ロットの製剤を回収し、代替品の提供は無償で行う。

５　誤
回収は、該当ロットの製剤のみ行う。
