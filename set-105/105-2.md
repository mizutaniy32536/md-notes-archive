# 第105回 問2（物理）

**問題文**
日本薬局方アスピリンの逆滴定による定量法で使用する標準液で、ファクターの値が必要なのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-2 を参照

**選択肢**
1. 0.1 mol/Lエチレンジアミン四酢酸二水素二ナトリウム液
2. 0.1 mol/L硝酸銀液
3. 0.5 mol/L水酸化ナトリウム液
4. 0.1 mol/Lチオ硫酸ナトリウム液
5. 0.25 mol/L硫酸

**正解:** 5

**解説**
解答　５

逆滴定とは、医薬品に過量の標準液を反応させ、反応しなかった標準液を別の標準液で滴定する方法である。
アスピリン（別名：アセチルサリチル酸）は酸性化合物であるため、アスピリンの逆滴定では最初に過量の塩基性標準液（0.5 mol/L水酸化ナトリウム液）と反応させ、反応しなかった塩基性標準液を酸性標準液（0.25 mol/L硫酸）で滴定すると考えられる。
最後に滴定する標準液のファクターが必要であるため、日本薬局方アスピリンの逆滴定による定量法で使用する標準液で、ファクターの値が必要なのは0.25 mol/L硫酸である。
