# 第105回 問141（衛生）

**問題文**
薬剤師法に関する記述のうち、正しいのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-141 を参照

**選択肢**
1. 薬剤師の任務は「医療及び保健指導をつかさどることによって、公衆衛生の向上及び増進に寄与し、もって国民の健康な生活を確保する。」と定められている。
2. 薬剤師でなければ、薬剤師又はこれにまぎらわしい名称を用いてはならない。
3. 薬剤師免許は、薬剤師国家試験の合格によって発行される。
4. 薬剤師は罪を犯しても、免許を取り消されることはない。
5. 調剤に従事する薬剤師に限り、資格を確認できるよう氏名が公表される。

**正解:** 2

**解説**
解答　２

１　誤
薬剤師法において、薬剤師の任務は、「薬剤師は、調剤、医薬品の供給その他薬事衛生をつかさどることによって、公衆衛生の向上及び増進に寄与し、もって国民健康を確保するものとする。」となっている。なお、「医療及び保健指導をつかさどることによって公衆衛生の向上及び増進に寄与し、もって国民の健康な生活を確保するものとする」と規定されているのは、医師である。

２　正
薬剤師でなければ、薬剤師又はこれにまぎらわしい名称を用いてはならない。

３　誤
薬剤師免許は、薬剤師国家試験に合格した者の申請により、薬剤師名簿に登録されたときに発行される。

４　誤
薬剤師が相対的欠格事由（罰金以上の受刑者、薬事に関する不正行為のあった者など）のいずれかに該当し、又は薬剤師としての品位を損ねるような行為があったとき、薬剤師は、免許を取り消されることがある。

５　誤
調剤に従事する薬剤師に限らず、すべての薬剤師は資格を確認できるよう氏名が公表される。
