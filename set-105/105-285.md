# 第105回 問285

### 第105回薬剤師国家試験 問285

#### 問題文（実践問題）
68歳女性。肝及び腎機能の検査値は正常範囲内であった。変形性関節症と膀胱炎のため、処方1の薬剤で治療を受けていた。外来診察時に患者より、「痛みが治まらず、もっと効く薬が欲しい。」と訴えがあり、以下の処方2に変更された。[1][2][3]

**問285（実務）**  
処方を受け取った薬剤師が行う内容として、優先度が高いのはどれか。**２つ**選べ。[1][2][3]

#### 全選択肢
1. ロコア®テープの用法・用量について医師に照会する。[1][2][3]  
2. ロキソプロフェンナトリウム錠とロコア®テープの併用について医師に照会する。[1][2][3]  
3. ノルフロキサシンとエスフルルビプロフェンの相互作用について医師に照会する。[1][2][3]  
4. 妊娠の有無について患者に確認する。[1][2][3]  
5. ノルフロキサシン錠からロメフロキサシン塩酸塩錠への変更について医師に照会する。[1][3][4]

#### 正解番号
**2, 3** [2][4]

#### 解説
- **選択肢2**: ロコア®テープ（エスフルルビプロフェン外用）は吸収率が高く、全身作用が強いNSAIDsに相当するため、ロキソプロフェンナトリウム錠（経口NSAIDs）との併用は重複となり、可能な限り避けるべき。添付文書でも「他の全身作用を期待する消炎鎮痛剤との併用は避け、やむを得ず併用時は必要最小限に留め、患者状態に注意」と記載されているため、疑義照会が必要。[1][2]  
- **選択肢3**: ノルフロキサシン（ニューキノロン系）とエスフルルビプロフェン（NSAIDs）には相互作用があり（中枢神経系抑制作用の増強など）、照会が必要。[4]  
- **その他の選択肢**:  
  - 1: ロコア®テープの貼付枚数2枚は用法・用量の範囲内。[1]  
  - 4: 68歳女性で妊娠の可能性は低く、優先度低い。[2]  
  - 5: ノルフロキサシンからロメフロキサシンへの変更は同系統で問題なく、照会不要。[4]
