# 第105回 問311

### 第105回薬剤師国家試験 問311

#### 科目名
法規・制度・倫理[1][2][3]

#### 問題文全文
近隣に専門クリニックが開院したこともあり、甲状腺治療薬の長期処方が増加している。一方で、この患者のように、継続治療が必要なのに、服用を忘れたり、勝手に中断する患者が多い。この状況を改善するため、薬剤師が研究を行うことになった。問題文に「長期処方の患者に対して（Pに該当）、薬剤師が電話によるフォローアップを行う（Iに該当）ことで、患者の服薬アドヒアランスの改善又は症状悪化の早期発見につながるか（Oに該当）を検討することにした。この研究のPECO又はPICOの組合せとして、適切なのはどれか。１つ選べ。」（注）PECOやPICOは疑問を構造化するための手法の1つ。PはPatient、EはExposure、IはIntervention、CはComparison、OはOutcomeの頭文字のこと。[1][2]

#### 全選択肢
1. 服薬アドヒアランスの悪い患者（P）、電話によるフォローアップ（E）、服薬アドヒアランスの改善（O）  
2. 長期処方の患者（P）、電話によるフォローアップ（I）、服薬アドヒアランスの改善又は症状悪化の早期発見（O）  
3. 甲状腺治療薬服用患者（P）、薬剤師によるフォローアップ（I）、症状悪化の早期発見（O）  
4. 長期処方の患者（P）、服薬アドヒアランスの悪い患者（C）、電話によるフォローアップ（I）  
5. 甲状腺治療薬長期処方患者（P）、電話によるフォローアップ（E）、服薬アドヒアランスの改善（O）[1][2][3]

#### 正解番号
**2**[1][2][3][6]

#### 解説
問題文に「**長期処方の患者** （P）、**薬剤師が電話によるフォローアップを行う** （I）ことで、患者の**服薬アドヒアランスの改善又は症状悪化の早期発見** （O）を検討することにした」とあることから、選択肢**2**が適切。「甲状腺治療薬長期処方の患者」ではなく「長期処方の患者」である点、PICOのI（Intervention）が電話フォローアップに該当する点が正解の根拠。他の選択肢はPの定義が不正確（例:1は「服薬アドヒアランスの悪い患者」で漠然としすぎ[3]）、またはEを誤用（5）など不適切。[1][2][3]
