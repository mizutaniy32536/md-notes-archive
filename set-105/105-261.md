# 第105回 問261

### 第105回薬剤師国家試験 問261（薬理）

**問題文（共通ステムを含む）**  
57歳女性。大腿骨頸部骨折の治療を目的とした手術のため本日入院した。手術は4日後に予定している。病棟担当薬剤師が患者の持参薬を確認したところ、下記の5種類の薬剤を所持していた。  
持参薬  
- アムロジピンベシル酸塩錠  
- アトルバスタチンカルシウム水和物錠  
- アルファカルシドールカプセル  
- ラロキシフェン塩酸塩錠  
- ロキソプロフェンナトリウム水和物錠  

手術前から休薬すべき薬剤のその理由となる副作用はどれか。1つ選べ。[1][2][3]

**全選択肢**  
1. 房室ブロック  
2. 高血糖  
3. 高Na⁺血症  
4. **血栓形成**  
5. 消化器障害[1][2][3]

**正解番号**  
4[1][2][3][4][6]

**解説**  
問260では医師に休薬を提案すべき薬剤として**ラロキシフェン塩酸塩錠（選択肢4）**が適切であり、これは女性ホルモン系薬剤（選択的エストロゲン受容体モジュレーター：SERM）である。手術前休薬の理由は、ラロキシフェンが**血栓形成**リスクを高めるためで、手術後の安静状態（横臥位）が血栓形成を促進する可能性がある。エストロゲン様作用により静脈血栓塞栓症のリスクが増大する。[1][2][8]  
他の選択肢は該当せず（例: 房室ブロックはアムロジピン関連、高血糖はアルファカルシドール関連、消化器障害はロキソプロフェン関連）。[1][2]
