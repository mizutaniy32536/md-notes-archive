# 第105回 問14（物理）

**問題文**
胚性幹細胞（ES細胞）の樹立に用いられるのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-14 を参照

**選択肢**
1. 精子
2. 原始卵胞
3. 造血幹細胞
4. 皮膚の線維芽細胞
5. 初期胚の内部細胞塊

**正解:** 5

**解説**
解答　５

胚性幹細胞（ES細胞）は、受精卵から発生が少し進んだ胚盤胞の中の内部細胞塊を取り出して培養することにより作製される多能性幹細胞である。よって、胚性幹細胞（ES細胞）の樹立に用いられるのは、初期胚の内部細胞塊である。
