# 第105回 問191（病態・薬物治療）

**問題文**
HIV感染症に関する記述のうち、正しいのはどれか。２つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-191 を参照

**選択肢**
1. HIVは唾液を介して感染する。
2. 日本ではHIV感染者とAIDS発症者の数はほぼ等しい。
3. 感染初期には発熱などのインフルエンザ様症状が出現し、通常数年持続する。
4. 感染後、一般に数年〜十数年は無症候期が続く。
5. AIDS期には悪性腫瘍や脳症などが発症する。

**正解:** 4、5

**解説**
解答　４、５

１　誤
HIV（ヒト免疫不全ウイルス）の主な感染経路は性行為感染、母子感染（経胎盤、経産道、経母乳）、注射針などを介した血液感染であり、HIVがほぼ含まれていない唾液は感染源とはならない。

２　誤
AIDS（後天性免疫不全症候群）発症とは、HIV感染によりCD4陽性Tリンパ球数が減少し、日和見感染症などを発症する免疫不全状態のことをいう。このAIDS発症は、HIV感染の検査、抗HIV薬の服用などにより抑えることが可能であるため、HIV感染者のうちのAIDS発症者数を比較すると少なくなる。

３　誤
HIV感染初期には発熱や咽頭痛、関節痛などのインフルエンザ様症状が出現し、これらの症状は数日から数週間でおさまる。また、その後一般に数年〜十数年は無症候期が続き、CD4陽性Tリンパ球数の減少が著しくなるとAIDS発症期となる。AIDS発症期には、免疫力の低下から日和見感染症（ニューモシスチス肺炎など）、悪性腫瘍（カポジ肉腫など）、HIV脳症（HIVそのものやHIVへの免疫反応により引き起こされる中枢神経障害）などが発症する。

４　正
