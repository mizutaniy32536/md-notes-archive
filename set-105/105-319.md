# 第105回 問319

### 第105回薬剤師国家試験 問319

#### 科目: 実務（地域医療・学校薬剤師関連）

#### 問題文（全文）
保険薬局に勤務して3年が経過した薬剤師が、今年度から近隣の中学校の**学校薬剤師**を担当することになった。1ヶ月後には薬物乱用防止教室での講師を担当することになっている。そのため、学校保健安全法の内容を確認することにした。[5][2][7]

#### 全選択肢
この薬剤師が行う薬物乱用防止教室に関する記述のうち、適切なのはどれか。**2つ選べ**。[2][5]

1. 薬物乱用防止に関する講演は薬剤師に特化した職務である。[2]
2. 麻薬の廃棄方法について説明する。[2]
3. 薬物乱用が心身に及ぼす影響について説明する。[2]
4. 麻薬・覚せい剤に関する基礎知識について説明する。[2]
5. 薬物乱用とは、何回も繰り返して薬物を使用することであると説明する。[2]

#### 正解番号
**3, 4**（選択肢3と4が適切。薬物乱用の心身影響説明と適正使用啓発が学校薬剤師の職務に該当）[2][5]

#### 解説
- 学校薬剤師の職務として、学校保健安全法に基づき、**薬物乱用防止・医薬品の適正使用に関する啓発**が適切。[5]
- 選択肢1: 薬物乱用防止講演は警察官や行政官も行うため、薬剤師特化ではない（誤り）。[2]
- 選択肢2: 麻薬廃棄方法の説明は乱用防止に直接つながらず不適切（誤り）。[2]
- 選択肢3,4: 薬物乱用の心身影響説明と関連啓発は妥当（正解）。[2]
- 選択肢5: 健康診断実施は学校薬剤師の職務外（誤り）。[2]
