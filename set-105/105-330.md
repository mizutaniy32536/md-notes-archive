# 第105回 問330（実務）

**問題文**
4歳男児。体重17 kg。脳性麻痺で以下の薬剤が処方された。1包あたり0.3 gになるように賦形剤を加えて調剤する場合、この処方を全量調剤するために加える賦形剤の総量（g）として正しいのはどれか。１つ選べ。なお、ダントロレンナトリウム水和物カプセル1カプセルあたりの内容量は0.25 gである。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-330 を参照

**選択肢**
1. 2.1
2. 4.2
3. 6.3
4. 8.4
5. 9.1

**正解:** 3

**解説**
解答　３

まず、ダントロレンナトリウム水和物と賦形剤の総量を求める。1包あたり0.3 gになるように賦形剤を加えて調剤するため、14日間での総量は以下のように計算される。
　0.3 g/回×2回/日×14日＝8.4 g
ダントロレンナトリウム水和物カプセル（0.25 g/カプセル）の総量は、1日あたり0.6カプセルより、以下のように計算される。
　0.25 g/カプセル×0.6カプセル/日×14日＝2.1 g
よって、賦形剤の量は以下のように計算される。
　賦形剤の量＝8.4 g（全量）－2.1 g（ダントロレンナトリウム水和物の総量）＝6.3 g
