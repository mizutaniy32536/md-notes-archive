# 第105回 問49（衛生）

**問題文**
一定温度の水に溶解したとき、表面張力が図中のAの挙動を示す物質はどれか。１つ選べ。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-49 を参照

**選択肢**
1. 1−オクタノール
2. エタノール
3. 塩化ナトリウム
4. パルミチン酸
5. ラウリル硫酸ナトリウム

**正解:** 3

**解説**
解答　３

ある物質を一定の温度の水に溶解したとき、水溶液中の物質（溶質）の濃度変化に伴う表面張力の挙動は下図のようになる。

Ⅰ型の挙動を示す物質は、無機塩類（塩化ナトリウムなど）や、極性化合物（ブドウ糖、ショ糖など）である。
Ⅱ型の挙動を示す物質は、脂肪酸（パルミチン酸など）、アルコール（1−オクタノール、エタノールなど）である。
Ⅲ型の挙動を示す物質は、界面活性剤（ラウリル硫酸ナトリウムなど）である。
したがって、本問の表面張力が図中のA（Ⅰ型）の挙動を示す物質は、塩化ナトリウムである。
