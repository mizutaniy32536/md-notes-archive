# 第105回 問299

### 第105回薬剤師国家試験 問299

**問題文（症例背景含む）**  
60歳男性。身長160cm、体重60kg。40歳頃に糖尿病と診断され、経口糖尿病薬の内服を開始した。50歳頃に腎障害を指摘され、55歳頃に、目のかすみ、眼性疲労、眼痛を自覚したため眼科を受診した。[2]

**問299（薬物治療）**  
この患者に対する急性期の治療として、適切なのはどれか。1つ選べ。  
１．20%マンニトール300mLを60分かけて点滴静注する。  
２．ロキソプロフェンナトリウム錠60mgを内服する。  
３．濃グリセリン・果糖配合製剤500mLを60分かけて点滴静注する。  
４．アセタゾラミドナトリウム注射用500mgを静注する。  
５．ピノレキシン点眼液0.005%を点眼する。[1][2]

**正解番号**  
**1, 4**（複数の正解があるため、いずれか1つ選択で正解とする。採点上、複数の選択肢を正解として扱う。[3][4][5]）

**解説**  
- **選択肢１**：高浸透圧製剤（20%マンニトール）。房水排出を促し、眼圧低下を促すため妥当。[1]  
- **選択肢２**：頭痛抑制の意図だが、房水排出を優先すべきで誤り。[1]  
- **選択肢３**：果糖配合製剤は急激な血糖上昇の恐れがあり、糖尿病患者に不適切。[1]  
- **選択肢４**：炭酸脱水酵素阻害薬。アセタゾラミドは房水産生を抑制し眼圧低下に有効。[2]  
- **選択肢５**：ピノレキシンは緑内障急性発作に使用可能だが、本問では1,4が主正解。[1][2]  
（問298の病態は閉角緑内障で、隅角狭窄による眼圧上昇・網膜視神経線維欠損。[2]）[1][2][5]
