# 第105回 問297

### 科目名
**実務**（服薬指導と副作用の回避）[1][2][3]

### 問題文全文
59歳男性。162cm、51kg。慢性腎炎が進行し、13年前より血液透析（HD）治療を受けている。HDに際しては、HD開始前にそう痒予防のためにジフェンヒドラミン塩酸塩錠10mgを4錠内服している。また、体外循環中の凝固防止の目的でナファモスタットメシル酸塩注（後発品）を用いている。最近15日間における患者の体温とCRP値は図のようになった。  
HD実施日に発熱が見られたため検査した結果、抗ナファモスタットIgEが陽性であることが第9日に明らかとなった。検査結果を受けて対処した結果、第11日以後はHD後の症状が見られなくなった。**対処した内容として考えられるのはどれか。１つ選べ。**[1][3][8]

### 全選択肢
1. HD前の投与薬として**デキサメタゾン錠**を追加した。[1][3]  
2. HD前の投与薬として**ナルフラフィン塩酸塩カプセル**を追加した。[1][3]  
3. HD中の**ナファモスタットメシル酸塩注射用をダルテパリンナトリウム注射液（低分子ヘパリン）に変更した**。[1][3]  
4. **経過観察した**。[1][3]  
5. **ナファモスタットメシル酸塩注射用を後発品から先発品に変更した**。[1][3]

### 正解番号
**3**[1][2][3][5]

### 解説
本症例では、抗ナファモスタットIgEが陽性であり、ナファモスタット投与により発熱および炎症（CRP上昇）が現れていた。対処として、抗原（ナファモスタット）を除去するため、HD中の抗凝固薬を**ダルテパリンナトリウム注射液（低分子ヘパリン）に変更**した。これにより第11日以後、症状が消失した。[1][3]  
- 選択肢1: デキサメタゾンは化学療法の前投与薬との混同。IgE陽性への直接対処ではない。[4]  
- 選択肢2: ナルフラフィンはそう痒治療薬で無関係。[1]  
- 選択肢4: 経過観察では症状改善しない。[1]  
- 選択肢5: 先発品変更でも抗原曝露が続くため不適切。[1][3]
