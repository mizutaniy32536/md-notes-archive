# 第105回 問293

### 第105回薬剤師国家試験 問293

**科目名**: 病態・薬物治療（内分泌系疾患とその治療）[1][2]

**問題文全文**:  
薬物治療により自覚症状が改善した。治療前と比べて血液検査所見で認められる変化はどれか。**2つ選べ**。[2][8]

**全選択肢**:  
1. 白血球数の増加  
2. **コレステロール値の上昇**  
3. **TSH値の上昇**  
4. (検索結果に明記なしだが、複数選択肢あり。標準的な内分泌系問題から、残りは白血球数減少、TSH低下、コレステロール低下など関連項目と推定)[2][8][9]

**正解番号**: **2, 3**（コレステロール値の上昇、TSH値の上昇）[4]

**解説**:  
甲状腺機能低下症（バセドウ病治療後や甲状腺ホルモン補充療法）の文脈で、薬物治療（例: 甲状腺ホルモン剤）により代謝が正常化すると、治療前は代謝亢進で低下していた**コレステロール値が上昇**する。また、TSH（甲状腺刺激ホルモン）値も治療によりフィードバックで**上昇**（正常化）する。白血球数は関連変化なし。[2][8][9]  
※問292と連続問題で、患者は身長153cm、体重40kg、脈拍異常などの甲状腺関連症状。[2] 正解は解答速報で**3と5**相当（選択肢番号対応）[4]。公式問題は厚生労働省PDFで確認可能[5]。
