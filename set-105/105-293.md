# 第105回 問293

### 第105回薬剤師国家試験 問293

**科目名**: 病態・薬物治療（内分泌系疾患とその治療）[1][2]

**問題文全文**:  
薬物治療により自覚症状が改善した。治療前と比べて血液検査所見で認められる変化はどれか。**2つ選べ**。[2][8]

**全選択肢**:  
1. 白血球数の増加  
2. 赤血球数の増加  
3. 血清コレステロール値の上昇  
4. 空腹時血糖値の上昇  
5. 血中TSH値の上昇

**正解番号**: **3, 5**（血清コレステロール値の上昇、血中TSH値の上昇）[4]

**解説**:  
甲状腺機能低下症の文脈で、薬物治療により自覚症状が改善すると、治療前は代謝亢進で低下していた**血清コレステロール値が上昇**し、TSH（甲状腺刺激ホルモン）値も**上昇**する。白血球数や赤血球数、空腹時血糖値は本問の正解ではない。[2][8][9]  
※問292と連続問題で、患者は身長153 cm、体重40 kg、脈拍異常などの甲状腺関連症状。[2] 正解は解答速報で**3と5**。[4]
