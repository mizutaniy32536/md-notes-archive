# 第105回 問117（化学）

**問題文**
補体の活性化と機能に関する記述のうち、正しいのはどれか。２つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-117 を参照

**選択肢**
1. 補体の3つの活性化経路には、いずれもキナーゼ（リン酸化酵素）の連鎖反応が関わっている。
2. 補体活性化の古典経路は、レクチンが微生物表面のマンノースやマンナンを認識することで始まる。
3. C3aやC5aは、過剰な炎症反応を抑制する。
4. 病原体の表面にC3bが結合すると、C3b受容体を介して食細胞による病原体の貪食が促される。
5. C5bの生成は、膜侵襲複合体（MAC）形成の引き金となり、病原体が破壊される。

**正解:** 4、5

**解説**
解答　４、５

１　誤
補体の活性化経路には、古典経路、代替経路（第2経路）、レクチン経路（第3経路）の3つがあり、各経路の活性化因子は異なるが、いずれの経路にもプロテアーゼ（タンパク質分解酵素）による加水分解反応が関与する。 

２　誤
補体活性化の古典経路は、抗原抗体複合体のFc領域に補体のC1が結合し、C1が活性化することで始まる。なお、レクチンが微生物表面のマンノースやマンナンを認識することで補体の活性化が始まるのは、レクチン経路（第3経路）である。

３　誤
補体分解生成物であるC3aやC5aなどはアナフィラトキシンとよばれ、好塩基球や肥満細胞などに作用してヒスタミンやロイコトリエンなどの炎症性化学伝達物質を分泌させることにより、過剰な炎症反応を引き起こす。

４　正
補体分解生成物であるC3bはオプソニンとよばれ、病原体の表面に結合することでC3b受容体を有する食細胞による病原体の貪食が促進される。

５　正
補体分解生成物であるC5bは、C6、C7、C8、C9と共にC5b6789複合体である膜侵襲複合体（MAC）形成の引き金となる。MACは、病原体の細胞膜表面に形成され、病原体の細胞膜に小孔を形成し、病原体を破壊する。
