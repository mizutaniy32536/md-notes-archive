# 第105回 問307

### 第105回薬剤師国家試験 問307

#### 科目名
実務（法規・制度・倫理に関連）[1][2][5]

#### 問題文全文
焼き鳥屋に勤務している従業員が油で汚れた換気扇の掃除を行うため、20 w/v％水酸化ナトリウム溶液を買い求めに来局した。この焼き鳥屋は、毒物劇物営業者ではなく、今回初めてこの薬局を利用した。なお、この20 w/v％水酸化ナトリウム溶液は劇物である。

販売することになった20 w/v％水酸化ナトリウム溶液の使用に関する薬剤師のアドバイスとして、**適切でない**のはどれか。1つ選べ。[1][2]

#### 全選択肢
1. 腐食性があるので、素手で触らないでください。  
2. 失明のおそれがあるので、保護メガネなどを使ってください。  
3. 危害防止のため、希薄な水溶液とし、中和した後、多量の水で希釈して廃棄してください。  
4. 誤飲防止のため、飲食をしながら使用しないでください。  
5. **引火性があるので、火のそばで使わないでください。**[2]

#### 正解番号
**5**（公式解答および複数ソースで確認）[2][5][6]

#### 解説
20 w/v％水酸化ナトリウム溶液（劇物）は強アルカリ性の**腐食性**物質であり、皮膚接触や眼接触で重篤な損傷を引き起こすため、選択肢1・2は適切（保護具着用を勧める）。誤飲防止のための飲食禁止（4）も適切。廃棄方法（3）は中和・希釈が標準的で適切。

一方、選択肢**5は不適切**。水酸化ナトリウムは**水溶液**であり、引火性を持たない（非可燃性）。火気注意は誤ったアドバイスで、実務上の知識不足を示す問題。[1][2]
