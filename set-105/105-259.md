# 第105回薬剤師国家試験 問259

**科目：** 薬理

**問題文：**
候補となるそれぞれの薬物の作用機序として正しいのはどれか。2つ選べ。[1][4]

**選択肢：**
1. ドパミンD2受容体遮断[4]
2. セロトニン5–HT3受容体遮断[4]
3. オピオイドμ受容体刺激[4]
4. ロイコトリエンB4産生阻害[4]
5. アセチルコリンM3受容体遮断[4]

**正解番号：** 2、3[1][4]

**解説：**
本問は問258で選択された追加薬剤候補（ロペラミド塩酸塩とラモセトロン塩酸塩）の作用機序を問うものです。[1]

**選択肢2（セロトニン5–HT3受容体遮断）** は正しいです。ラモセトロン塩酸塩（イリボー）は5–HT3受容体遮断作用を示し、下痢型過敏性腸症候群の症状を改善します。[1]

**選択肢3（オピオイドμ受容体刺激）** は正しいです。ロペラミド塩酸塩（ロペミン）は止瀉薬であり、オピオイドμ受容体刺激作用を示します。[1]
