# 第105回 問213

### 第105回薬剤師国家試験 問213

#### 問題文（物理・化学・生物）
イロプロストやベラプロストナトリウムはプロスタグランジンI₂の構造をもとに開発された薬物である。これらに関する記述のうち、最も適切なのはどれか。**２つ選べ。**[2][3]

#### 全選択肢
1. イロプロストはプロスタグランジンI₂の部分構造aの酸素原子を炭素原子に置換することにより、シトクロムP450による代謝を受けやすくしている。[1][2][3]  
2. ベラプロストナトリウムはプロスタグランジンI₂のプロドラッグである。[2][4]  
3. ベラプロストナトリウムはカルボン酸部位を塩とすることにより、水溶性を向上させている。[1][2][3][4]  
4. プロスタグランジンI₂の部分構造aの二重結合の立体化学はE配置である。[1][2][3][4][7]

#### 正解番号
**3 と 5**（注: 検索結果[4]で選択肢5が確認され、正解として解説。標準4択+αの形式で、3と5が正しい。[4][6]）

#### 解説
- **選択肢1 誤**: イロプロストはプロスタグランジンI₂のフラン環（部分構造a）を飽和炭素五員環に変換したが、これによりCYP代謝を受けやすくはならない。[1]  
- **選択肢2 誤**: ベラプロストナトリウムはプロスタグランジンI₂のプロドラッグではない。[2][4]  
- **選択肢3 正**: カルボン酸をナトリウム塩にすることでイオン化し、水溶性が向上する。[1][2][3]  
- **選択肢4 誤**: プロスタグランジンI₂の部分構造aの二重結合はZ配置（優先順位の高い置換基が同じ側）。E配置ではない。[1][2][3][4][7]  
- **選択肢5 正**: ベラプロストナトリウムは部分構造aに芳香環を導入し、酸性条件下での安定性を向上させている。[4]  

（背景: 35歳女性の肺動脈性肺高血圧症患者がイロプロスト吸入液から経口薬へ変更のケース。[1][2][3]）
