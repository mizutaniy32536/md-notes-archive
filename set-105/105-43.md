# 第105回 問43（生物）

**問題文**
薬物代謝における第Ⅱ相反応はどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-43 を参照

**選択肢**
1. アルキル化
2. 加水分解
3. 還元
4. 酸化
5. 抱合

**正解:** 5

**解説**
解答　５

薬物代謝は、主に脂溶性薬物の極性を増大させることで薬物を体外へ排泄することが目的であり、第Ⅰ相反応（酸化、還元、加水分解）と第Ⅱ相反応（抱合反応）に大別される。
