


## ◆ 年齢区分別人口割合の将来推計と薬剤種類数のデータを読み取る問題
260307

**問題**
表1は、年齢区分別人口割合の将来推計である。表2は、全国の保険薬局での処方調査から明らかになった同一の保険薬局で調剤された薬剤種類数の年齢区分別のデータである。これらの表に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**


1. 表1から、2035年における老年化指数は約200%になると予測される。
​
2. 2015年から2035年までにおける老年人口割合の増加には、75歳以上人口割合の増加が大きく寄与している。
​
3. 表2から、75歳以上の患者のうち、ほぼ4人に1人が7種類以上/月の薬剤を処方されていることがわかる。
​
4. 7種類以上/月の割合が、65〜74歳に比べて75歳以上で約2倍であることは、65〜74歳に比べて75歳以上の患者の医療機関受診率が約2倍であることを示している。
​
5. 人口割合の将来推計は、将来にわたって総人口が変化しないものとして計算されている。

**正解と理由**
- `2. 2015年から2035年までにおける老年人口割合の増加には、75歳以上人口割合の増加が大きく寄与している。`
- `3. 表2から、75歳以上の患者のうち、ほぼ4人に1人が7種類以上/月の薬剤を処方されていることがわかる。`
- **理由の構成**
	- **問題の整理**：表1は年齢区分別の人口割合推計（2015年〜2035年）、表2は保険薬局での年齢区分別の薬剤種類数データを読み取る
	- **薬学的知識・仕様・制約**：`老年化指数`は（老年人口 / 年少人口）× 100で算出する。`ポリファーマシー`（多剤併用）は一般に6種類以上の薬剤を服用している状態を指し、75歳以上で特に問題となる
	- **選択理由**：選択肢2は表1から2015〜2035年の老年人口割合増加において、65〜74歳よりも`75歳以上`の後期高齢者人口の増加が大きく寄与していることが読み取れるため正しい。選択肢3は表2から75歳以上で7種類以上/月の薬剤処方割合が約25%（4人に1人）であることが読み取れるため正しい

**解法の手順（思考の流れ）**
- まず表1から各年の年齢区分別人口割合を確認し、老年化指数の計算に必要な数値を特定する
- 選択肢1について`老年化指数`を計算すると、2035年の値は約200%ではなく約300%程度と判断 → 誤り
- 選択肢2について表1の65〜74歳と75歳以上の割合の変化を比較し、75歳以上の増加が大きいことを確認 → 正しい
- 選択肢3について表2の75歳以上の7種類以上/月の割合が約25%であることを確認 → 正しい
- 選択肢4について薬剤種類数の増加は受診率だけでは説明できないと判断 → 誤り
- 選択肢5について将来推計人口は総人口の変化（減少）を考慮しているため → 誤り

**他の選択肢が誤りな理由**
- 1. 表1から、2035年における老年化指数は約200%になると予測される。
	`老年化指数`は（老年人口 / 年少人口）× 100で計算する。2035年の推計値から計算すると老年化指数は約300%程度となり、約200%という記述は誤りである

- 4. 7種類以上/月の割合が、65〜74歳に比べて75歳以上で約2倍であることは、65〜74歳に比べて75歳以上の患者の医療機関受診率が約2倍であることを示している。
	薬剤種類数の増加は`受診率`だけでは説明できない。75歳以上では複数の`慢性疾患`を併存することが多く、1回の受診でも処方される薬剤数が多くなる。受診率が2倍でも同じ疾患に対する同じ薬剤であれば種類数は変わらない

- 5. 人口割合の将来推計は、将来にわたって総人口が変化しないものとして計算されている。
	国立社会保障・人口問題研究所による`将来推計人口`は、出生率・死亡率・国際人口移動の仮定に基づいて総人口の変化を考慮した上で計算されている。総人口不変の仮定ではない

**キーポイント**
- `老年化指数`＝（老年人口 / 年少人口）× 100 で、高齢化の進行度を年少人口との比で示す
- `老年人口指数`＝（老年人口 / 生産年齢人口）× 100 であり、老年化指数とは分母が異なる点に注意
- 75歳以上の`後期高齢者`ではポリファーマシーの割合が高く、約4人に1人が7種類以上の薬剤を処方されている
- 将来推計人口は総人口の減少を前提として算出されており、「割合」の変化と「実数」の変化は別物である

**関連知識の比較**
- `従属人口指数`＝（年少人口 + 老年人口）/ 生産年齢人口 × 100 で、生産年齢層の負担度を示す指標である
- `年少人口指数`＝（年少人口 / 生産年齢人口）× 100 で、子どもの扶養負担を示す
- `ポリファーマシー`は一般に6種類以上の薬剤併用を指すが、明確な定義はなく、5種類以上とする文献もある。副作用リスクの増大、`アドヒアランス`低下、薬物相互作用の増加が問題となる
- 2025年問題として`団塊の世代`が全員75歳以上となることで、医療費・介護費の急増が懸念されている

（一言メモ）
- `老年化指数`と`老年人口指数`は名称が似ているが、分母が「年少人口」か「生産年齢人口」かという違いがある
- 75歳以上の`後期高齢者`では`ポリファーマシー`が深刻で、薬剤師による処方の一元管理と減薬提案が重要な役割となる
- 将来推計人口は`国立社会保障・人口問題研究所`が出生・死亡・移動の仮定を置いて算出するもので、総人口不変の仮定ではない
