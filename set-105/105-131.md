## ◆ 予防接種法に定める定期予防接種の種類・回数・分類を問うシナリオ

260312

**問題**
予防接種法に定める定期予防接種に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**

1. B型肝炎の予防接種は、生後12ヶ月までの間に1回のみ接種する。
​
2. 4種混合ワクチンであるDPT−IPVは、ジフテリア、百日咳、破傷風及びポリオ（急性灰白髄炎）の予防に用いられる。
​
3. 2019年以降、風しんワクチン接種の公費助成の対象者を拡大したのは、近年の風しんの流行及び先天性風しん症候群の報告数の増加によるものである。
​
4. 水痘の予防接種を受けた場合、日本脳炎の予防接種は翌日であれば受けることができる。
​
5. 肺炎球菌感染症は、小児及び高齢者の個人予防を主な目的とするB類疾病に位置付けられている。

**正解と理由**
- `2. 4種混合ワクチンであるDPT−IPVは、ジフテリア、百日咳、破傷風及びポリオの予防に用いられる。`
- `3. 2019年以降、風しんワクチン接種の公費助成の対象者を拡大したのは、近年の風しんの流行及び先天性風しん症候群の報告数の増加によるものである。`
- **理由の構成**
	- **薬学的知識・仕様・制約**：
		- `DPT-IPV`=D（ジフテリア）＋P（百日咳）＋T（破傷風）＋IPV（不活化ポリオ）の4種混合
		- 2019年に風しんワクチンの公費助成対象を拡大（1962〜1979年生まれの男性）

**解法の手順（思考の流れ）**
- 選択肢1：B型肝炎=生後12ヶ月までに`3回`接種（1回ではない）→ 誤り
- 選択肢2：DPT-IPV=ジフテリア＋百日咳＋破傷風＋ポリオ → 正しい
- 選択肢3：風しん公費助成拡大（2019年〜）=近年の流行・CRS増加が背景 → 正しい
- 選択肢4：水痘（生ワクチン）→日本脳炎（不活化）は同時接種可能。「翌日」という制限は不正確 → 誤り
- 選択肢5：小児肺炎球菌は`A類`。高齢者肺炎球菌は`B類`。「小児及び高齢者のB類」は誤り → 誤り

**他の選択肢が誤りな理由**
- 1. B型肝炎の予防接種は、生後12ヶ月までの間に1回のみ接種する。
	B型肝炎ワクチンは生後12ヶ月までに**3回**接種（0, 1, 6ヶ月後）

- 4. 水痘の予防接種を受けた場合、日本脳炎の予防接種は翌日であれば受けることができる。
	異なるワクチンの接種間隔：注射生ワクチン同士は27日以上空ける必要があるが、注射生ワクチン→不活化ワクチンの間隔制限は撤廃されている（2020年10月改正）

- 5. 肺炎球菌感染症は、小児及び高齢者の個人予防を主な目的とするB類疾病に位置付けられている。
	小児の肺炎球菌感染症は`A類疾病`（集団予防目的）。高齢者の肺炎球菌は`B類疾病`。「小児及び」をつけてB類とするのは誤り

**キーポイント**
- `DPT-IPV`：D（ジフテリア）P（百日咳）T（破傷風）IPV（不活化ポリオ）の4種混合ワクチン
- `B型肝炎`ワクチン：0, 1, 6ヶ月の**3回接種**（A類定期接種、2016年〜）
- 小児肺炎球菌=`A類`、高齢者肺炎球菌=`B類`。同じ疾患でも年齢で分類が異なる

**関連知識の比較**
- ワクチン接種間隔（2020年改正後）：
	- 注射`生ワクチン`→注射`生ワクチン`：27日以上
	- それ以外の組み合わせ：**制限なし**（同時接種も可能）
- `風しん`公費助成拡大の背景：40代前後の男性に抗体保有率が低い世代が存在

（一言メモ）
- DPT-IPV=「ジフテリア・百日咳・破傷風・ポリオ」。「IPV=不活化ポリオ」のIがInactivatedのI
- B型肝炎は1回じゃなくて3回。0-1-6ヶ月スケジュール
- 小児肺炎球菌=A類、高齢者肺炎球菌=B類。この使い分けは引っかかりやすい
