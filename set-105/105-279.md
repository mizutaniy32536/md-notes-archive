# 第105回 問279

### 第105回薬剤師国家試験 問279（実務）

**問題文（全文）**  
前問での検討結果を踏まえて、この患者に対する最も適切な調剤方法はどれか。１つ選べ。[1][2][3]

**全選択肢**  
1. 両剤を計量し、乳鉢にて、乳棒で混和し分包した。[1][2][3]  
2. アスベリン®散を乳糖で賦形したものと幼児用PL配合顆粒を、2段分割（2度撒き）で分包した。[1][2][3]  
3. 両剤を計量し、乳鉢にて、スパーテルで混和し分包した。[1][2][3]  
4. 幼児用PL配合顆粒を調剤用ミルで粉砕してから、アスベリン®散と乳棒・乳鉢にて混和し分包した。[2]  
5. アスベリン®散を乳糖で賦形したものをふるいで篩過してから、幼児用PL配合顆粒と乳棒・乳鉢にて混和し分包した。[2]

**正解番号**  
**2** [2][5]

**解説**  
問278では、アスベリン®散（粒子径が小さい）と幼児用PL配合顆粒（粒子径が大きい）の振とう混合で均等混和せず、小粒子が下部に偏在する状態が確認されたため、直接混和は不適切。[2]  
選択肢1,3,4,5は「混和し分包」とあり、偏在を防げないため誤り。[2][5]  
正解の2は、アスベリン®散を乳糖で賦形（粒子径を調整）し、顆粒と2段分割（2度撒き）で分包するため、内容均一性が保たれ、母親の包帯による取り扱いやすさも考慮した適切な方法。[1][2][5]
