# 第105回 問217

### 第105回薬剤師国家試験 問217

**科目**: 物理・化学・生物（実践問題）[1][2][3]

**問題文（全文）**:  
問216の続き。68歳男性、心房細動とLVEF低下した心不全（心拍数120回/分など）。心拍数調節のため薬剤A（問216でアミオダロン塩酸塩錠）が追加。下図は薬剤Aの投与前と投与後の心電図（Ⅱ誘導）を示している。この変化が起こる理由として適切なのはどれか。**２つ選べ**。[1][2][3]

**全選択肢**:  
1. 心室筋細胞からのNa⁺流出の直接的抑制[2]  
2. 心室筋細胞からのK⁺流出の直接的抑制[2]  
3. 心室筋細胞の活動電位持続時間の延長[2]  
4. 洞房結節の脱分極の直接的促進[2]  
5. 不応期の短縮[2]

**正解番号**: **2, 3**[2][4][8]

**解説**:  
問216の薬剤Aは**アミオダロン塩酸塩錠（選択肢3）**。クラスIII抗不整脈薬で、心拍数調節に適し、心房細動時の速い心拍を抑制。問217はアミオダロンの心電図変化（おそらくQT間隔延長など）を問う。[1][2][3]  
- 正解2: K⁺流出直接抑制（K⁺チャネル遮断）。  
- 正解3: 活動電位持続時間延長（クラスIIIの特徴）。  
不正解1: Na⁺流出抑制はクラスI。4: 洞房結節促進は心拍数上昇要因で不適切。5: 不応期短縮は逆効果。フロセミド・トルバプタンは抗不整脈薬でない。[2]
