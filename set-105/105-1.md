# 第105回 問1（物理）

**問題文**
以下のガラス器具のうち、溶質を溶媒に溶かして正確に一定の液量の溶液を調製するために用いられるのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-1 を参照

**選択肢**
（選択肢は画像を参照）

**正解:** 2

**解説**
解答　２

容量分析に用いられる体積測定用の器具を化学用体積計という。主な体積計はホールピペット（全量ピペット）、メスフラスコ （全量フラスコ）、メスシリンダー、メートルグラスである。これら体積計には、メスフラスコやメスシリンダーのように内部に溶液を入れた時の体積が表示体積である受用と、ホールピペットのように体積計から溶液を出した時の体積が表示体積の出用とがある。本問は「溶質を溶媒に溶かして正確に一定の液量の溶液を調製する」とあり、器具内部で溶質を溶液に溶かす場合は、受用器具を用いる。

１　誤
出用器具のホールピペットである。ホールピペットは一定量の正確な液量を計測する場合に用いる。出用器具のため、溶質を溶媒に溶かして調製するのには適さない。

２　正
受用器具のメスフラスコである。メスフラスコは、溶質を溶解させ、正確に全量を一定の液量とする場合に用いられる。

３　誤
受用器具のメスシリンダー（無栓形）である。メスシリンダーは目盛りのついた円筒形の容器である。メスシリンダーで測れるのは、目盛りの10分の1程度までのおおよその体積なので、正確に一定の液量の溶液を調製するために用いるのは適さない。

４　誤
受用器具のメスシリンダー（有栓形）である。メスシリンダー内で溶質を溶媒に溶解させたい場合、無栓形ではなく有栓形を用いる。

５　誤
受用器具のメートルグラスである。メートルグラスは目盛りのついた円錐形の容器である。容器の下部に行くほど細かい目盛りがついているため、少ない容量の測定に適しているが、精度はメスシリンダーより劣る。正確に一定の液量の溶液を調製するために用いるのは適さない。
