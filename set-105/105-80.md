# 第105回 問80（薬剤）

**問題文**
希少疾病用医薬品に関する記述のうち、適切なのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-80 を参照

**選択肢**
1. 指定難病の患者に対する治療薬のことである。
2. 国が主体となって製品化を行う。
3. 指定された後に、取り消されることはない。
4. 承認されるまで、指定されたことは公開されない。
5. 他の医薬品に優先して承認審査を受けられる。

**正解:** 5

**解説**
解答　５

１　誤
希少疾病用医薬品とは、その用途に係る対象者の数が我が国において5万人に達せず、かつ、その用途に関し特に優れた使用価値を有する医薬品である。
一部の希少疾病用医薬品は指定難病の治療薬に該当するものもあるが、すべての希少疾病用医薬品が指定難病の治療薬であるというわけではない。

２　誤
希少疾病用医薬品の製品化の主体になるのは、製薬企業（製造販売業者）等である。なお、国は希少疾病用医薬品等について、試験研究を促進するのに必要な資金の確保に努め、試験研究を促進するのに必要な税制上の措置を講ずるなどの支援措置を行う。

３　誤
希少疾病用医薬品の指定は、対象患者数の要件を欠くと認められる場合や、他の医薬品等が新たに承認されること等により、医療上の必要性の要件を欠くと認められた場合などに取り消されることがある。

４　誤
厚生労働大臣は、希少疾病用医薬品を指定したときは、その旨を官報に掲載することで公示する。

５　正
優先審査の対象には、希少疾病用医薬品等がある。優先審査とは、できるだけ迅速に医療現場に提供できるように他の医薬品等に優先して承認審査を行う希少疾病用医薬品の開発支援措置のことである。
