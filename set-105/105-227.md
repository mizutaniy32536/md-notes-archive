# 第105回 問227

### 第105回薬剤師国家試験 問227

#### 問題文（衛生）
62歳女性。身長156 cm、体重54 kg。慢性腎不全、2型糖尿病、高血圧症で外来治療中。骨粗しょう症はない。今回、慢性腎不全の病状が進行し、入院加療することになった。入院時の持参薬と検査値は以下の通りであった。[1][2][3]

持参薬：アムロジピンベシル酸塩錠5 mg、フロセミド錠20 mg、リナグリプチン錠5 mg、ボグリボース錠0.2 mg、ポリスチレンスルホン酸カルシウム20%ゼリー25 g、球形吸着炭細粒2 g／包[2]

検査値：Na 140 mEq/L、K 5.2 mEq/L、Cl 108 mEq/L、P 5.9 mg/dL、補正Ca 7.5 mg/dL、血清アルブミン 3.7 g/dL、AST 24 IU/L、ALT 26 IU/L、BUN 50.5 mg/dL、血清クレアチニン 1.8 mg/dL、eGFR 23 mL/min/1.73 m²、intact-PTH 210 pg/mL（標準値：10～65 pg/mL）[2][3]

この患者の病態に関連するビタミンやミネラルに関する記述のうち、正しいのはどれか。**2つ選べ。**[1][2][3][5]

#### 全選択肢
1. カルシトリオールは、肝臓で25位が水酸化されて活性型に変換される。[1][2][5]  
2. ビタミンKは、骨のCa結合タンパク質であるオステオカルシンの遺伝子の転写を活性化する。[1][2][5]  
3. リン酸は、フィチン酸やシュウ酸とキレートを生成することにより、消化管での吸収が抑制される。[1][2][5]  
4. 腎不全の進行によって腎臓での活性型ビタミンDの生成が低下すると、消化管からのカルシウム吸収が低下する。[1][2][5]  
5. この患者のintact-PTHの値から、血中カルシウム濃度を正常に維持しようとして副甲状腺機能亢進状態になっていることがわかる。[1][2][5]

#### 正解番号
**4, 5**[1][8]

#### 解説
- この患者は慢性腎不全進行により、補正Ca低値（7.5 mg/dL）、P高値（5.9 mg/dL）、intact-PTH高値（210 pg/mL）で二次性副甲状腺機能亢進症を示唆。[1][3]
- 選択肢1：誤り。カルシトリオールは既に活性型ビタミンD製剤であり、肝臓での25位水酸化は不要（活性型のため）。[1][2]
- 選択肢4：正しい。腎不全では活性型ビタミンD生成低下により消化管Ca吸収が低下する。[1][2]
- 選択肢5：正しい。低Caを代償してPTHが上昇し、副甲状腺機能亢進状態になる。[1][2]
