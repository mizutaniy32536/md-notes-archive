# 第105回 問325

### 第105回薬剤師国家試験 問325

#### 問題文（全文）
その後、この患者が死亡し、患者の相続人から、薬剤が残っているので、薬局に返却したいとの申し出があった。確認したところ、残薬は**フェンタニル貼付剤**及び**酸化マグネシウム**であった。これらの薬剤の取扱いに関する記述のうち、正しいのはどれか。**2つ選べ**。[1][2][5]

#### 全選択肢
1. 酸化マグネシウムは、まだ使用期限を過ぎていなかったので、必要に応じて相続人が服用してもよいと指導した。[1][2][5]  
2. （正解）フェンタニル貼付剤は、麻薬として都道府県知事の許可を得て焼却した。[1][2]  
3. （正解）酸化マグネシウムは、薬局で管理し、必要に応じて他の患者に使用した。[1][2]  
4. 返却されたフェンタニル貼付剤は、まだ使用期限を過ぎていなかったので、仕入れをした卸売販売業者に返品した。[5]  
5. 返却されたフェンタニル貼付剤を薬局で廃棄したので、廃棄後30日以内に都道府県知事に届出を行った。[5][1]

#### 正解番号
**2, 3** [1][2][7]

#### 解説
- 選択肢1：誤り。薬剤の返却に許可は不要だが、酸化マグネシウムを相続人に譲渡・服用指導は不適切（一般薬でも返品後の再利用は制限）。[1]  
- 選択肢2：正しい。フェンタニル貼付剤は**麻薬**のため、相続人からの返却後、都道府県知事の許可を得て焼却処分（麻薬及び向精神薬取締法97条の313関連）。[1][2]  
- 選択肢3：正しい。酸化マグネシウムは麻薬でないため、返却後薬局で管理し他の患者に使用可能。[1][2]  
- 選択肢4：誤り。麻薬の返品は卸へ不可。[5]  
- 選択肢5：誤り。廃棄時の届出は不要で、許可焼却が必要。[1][5]  
（問の背景：71歳男性膵がん緩和ケア患者の退院時処方から派生。フェンタニル貼付剤は**オピオイド麻薬**、酸化マグネシウムは便秘薬）。[1][2][5]
