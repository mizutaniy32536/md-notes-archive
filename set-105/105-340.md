# 第105回 問340（実務）

**問題文**
58歳男性。糖尿病で近医にてインスリン治療を継続している。前回と同じ内容の処方箋を持って薬局を訪れた。残薬を確認したところ、インスリン注射液の残薬はないが、使用可能な注射針が13本自宅にあるとのことであった。薬剤師が注射針の必要数を計算したところ、処方量が不足していることに気付いた。注射時に空打ちで2単位使用する。医師に提案する追加すべき注射針数（1袋7本入り）として最も適切なのはどれか。１つ選べ。

**選択肢**

5. 50袋

**正解:** 1

**解説**
解答　１

処方１におけるインスリンの総単位数は、300単位×3＝900単位である。皮下注射1回につき、空打ちと合わせて5単位（3単位＋2単位）使用するため、900単位÷5単位/回＝180回分の注射針が必要である。
処方2におけるインスリンの総単位数は、300単位×2＝600単位である。皮下注射1回につき、空打ちと合わせて10単位（8単位＋2単位）使用するため、600単位÷10単位/回＝60回分の注射針が必要である。
注射針は、処方3の28本（7本/袋×4袋）と自宅に余っている13本の計41本であるが、本処方で必要な注射針の本数は240本（180本＋60本）であるため、240本－41本＝199本の注射針が不足している。
したがって、199本÷7本/袋≒28.4袋より、29袋の追加を提案すべきである。
