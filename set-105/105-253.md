# 第105回 問253

# 第105回薬剤師国家試験 問253

## 問題文

医師に提案したそれぞれの薬物のもつ作用の特徴として、適切なのはどれか。2つ選べ。[2]

## 選択肢

1. セロトニン5−HT2A受容体遮断作用
2. ヒスタミンH1受容体遮断作用
3. ドパミンD2受容体遮断作用
4. アセチルコリンM1受容体遮断作用
5. アドレナリンα1受容体部分刺激作用[2]

## 正解番号

**1, 3**[1][4]

## 解説

本症例は統合失調症治療薬のハロペリドール使用中に錐体外路症状が生じたため、代替薬の相談があった場合です。[1]

問252で推奨された薬物はアリピプラゾール（選択肢3）とリスペリドン（選択肢4）です。[1][4]これらの非定型抗精神病薬の特徴は以下の通りです：

**セロトニン5−HT2A受容体遮断作用（選択肢1）**：アリピプラゾールとリスペリドンの両者が有する作用です。この作用により、ドパミン放出を促進し、ドパミンD2受容体遮断による副作用（錐体外路症状）を軽減します。[9]

**ドパミンD2受容体遮断作用（選択肢3）**：統合失調症の治療効果をもたらす主要な作用機序です。アリピプラゾールとリスペリドンは、ハロペリドールよりもD2受容体への親和性が低いか、部分刺激作用を有することで、錐体外路症状を軽減しながら治療効果を維持します。[9]

なお、選択肢2のヒスタミンH1受容体遮断作用についても、リスペリドン及びアリピプラゾールに示唆されているため、正解になる可能性も指摘されています。[6]
