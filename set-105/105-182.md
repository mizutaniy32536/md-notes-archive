# 第105回 問182（病態・薬物治療）

**問題文**
医薬品の容器に関する記述のうち、正しいのはどれか。２つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-182 を参照

**選択肢**
1. 吸入液剤を製するには、通例、有効成分に溶剤を加えて溶液とし、液状の噴射剤とともに耐圧性の容器に充填し、定量バルブを装着する。
2. 坐剤に用いる容器は、通例、密閉容器とする。
3. 血液透析用剤に用いる容器は、通例、密封容器とする。
4. 点眼剤に用いる容器は、通例、点眼剤の不溶性異物検査法の試験に支障をきたさない透明性のある気密容器とする。
5. 外用エアゾール剤及びポンプスプレー剤に用いる容器は、通例、耐圧性の容器とする。

**正解:** 2、4

**解説**
解答　２、４

１　誤
吸入エアゾール剤を製する際の記述である。吸入液剤を製するには、通例、有効成分に溶剤及び適切な等張化剤、pH調節剤などを加え、混和して均質に溶解又は懸濁し、必要に応じてろ過する。

２　正
坐剤に用いる容器は、通例、密閉容器とする。

３　誤
血液透析用剤に用いる容器は、通例、微生物の混入を防ぐことのできる気密容器とする。

４　正
点眼剤の不溶性異物検査法は、点眼剤中の不溶性異物の有無を肉眼観察し、調べる方法である。したがって、点眼剤に用いる容器は、異物の観察に支障をきたさない透明性のある気密容器とする。

５　誤
外用エアゾール剤は、容器に充填した液化ガス又は圧縮ガスと共に有効成分を噴霧するスプレー剤であり、通例、耐圧性の容器を用いる。一方、ポンプスプレー剤は、ポンプにより容器内の有効成分を噴霧するスプレー剤であり、通例、気密容器を用いる。
