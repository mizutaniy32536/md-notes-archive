# 第105回 問3（物理）

**問題文**
次の原子のうち、核スピンをもたない（核スピン量子数＝0）のはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-3 を参照

**選択肢**
1. 1H
2. 12C
3. 13C
4. 14N
5. 15N

**正解:** 2

**解説**
解答　２

陽子の数あるいは中性子の数のいずれかが奇数の原子核はあたかも自転している回転体のような性質を示し、核スピン角運動量とこれに平行な磁気モーメントをもつ。核スピン角運動量を表す量子数が核スピン量子数であり、その原子核に固有の値である。核スピン量子数は、質量数が奇数の場合は半整数（1/2、3/2、5/2…）、偶数の場合は整数の値をとる。また、12C、16Oのように陽子数と中性子数がいずれも偶数の原子核はスピン量子数が0である。
