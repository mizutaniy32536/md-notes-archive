# 第105回 問77（薬剤）

**問題文**
覚せい剤取締法で規制されるのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-77 を参照

**選択肢**
1. 大麻
2. モルヒネ
3. 亜硝酸イソブチル
4. フェニルアミノプロパン
5. ペンタゾシン

**正解:** 4

**解説**
解答　４

１　誤
大麻は大麻取締法で規制される。

２　誤
モルヒネは麻薬であり、麻薬及び向精神薬取締法で規制される。

３　誤
亜硝酸イソブチルは指定薬物であり、医薬品医療機器等法で規制される。

４　正
フェニルアミノプロパンは覚せい剤であり、覚せい剤取締法で規制される。

５　誤
ペンタゾシンは第二種向精神薬であり、麻薬及び向精神薬取締法で規制される。
