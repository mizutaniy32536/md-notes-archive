# 第105回 問215

# 第105回薬剤師国家試験 問215

## 科目
実務（薬局における業務）

## 問題文
22歳男性。身長175cm、体重60kg。花粉症の症状がひどくなったので、家族が使用していた一般用医薬品の小青竜湯エキス顆粒の服用を考えたが、陸上競技の国体選手であったため、かかりつけ薬剤師に相談した。
薬剤師は、小青竜湯エキス顆粒にはアンチ・ドーピング規程における禁止物質が含まれるため、服用しないよう指示した上で、近隣の医療機関への受診を勧奨した。その結果、次の薬剤が処方されたので、薬剤師が処方監査を行った。

**処方された薬剤のうち、アンチ・ドーピングの観点から、処方変更を医師に提案すべき薬剤はどれか。1つ選べ。**[1][2]

## 全選択肢
1. フェキソフェナジン塩酸塩錠
2. ベタメタゾン錠
3. フルチカゾンフランカルボン酸エステル点鼻液
4. フルオロメトロン点眼液
5. エピナスチン塩酸塩点眼液

## 正解番号
**2**[1][2]

## 解説
**ベタメタゾン錠**が正解です。ベタメタゾンはホルモン関連薬に分類され、アンチ・ドーピング規程の**S4（ホルモン関連薬及び代謝調節薬）**に該当する禁止薬物です。[1][2]

一方、フェキソフェナジンのような**抗ヒスタミン薬はアンチ・ドーピングの観点から気にする必要がありません**。また、点鼻薬や点眼薬は通常の用法・用量であれば、アンチ・ドーピングの観点から制限されません。したがって、選択肢1、3、4、5は誤りです。[2]
