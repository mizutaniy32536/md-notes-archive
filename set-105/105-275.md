# 第105回 問275

# 第105回薬剤師国家試験 問275

## 問題文

初日はボリコナゾールとして1回300mgを1日2回、2日目以降は1回200mgを1日2回点滴静注し、治療を行ったが、症状やレントゲン陰影の改善はみられなかった。この患者におけるボリコナゾールの定常状態での平均血漿中濃度は1.0mg/Lであったため、薬剤師は治療域に達していないと判断した。この患者における定常状態における平均血漿中濃度を2.5mg/Lとしたい。m値は0.50mg/Lとする。[1]

## 選択肢

- 1. 500
- 2. 600
- 3. 700
- 4. 800
- 5. 1,000

## 正解番号

**1**[1][6][8]

## 解説

この問題はミカエリス・メンテン式を用いた薬物動態の計算問題です。[1]

**ミカエリス・メンテン式：**

\[ v = \frac{V_{max} \cdot [S]}{K_m + [S]} \]

定常状態においては、投与量＝消失速度であることを利用します。[1]

現在の状況から、定常状態での平均血漿中濃度が1.0mg/Lの場合の消失速度と、目標濃度2.5mg/Lでの消失速度の関係式を立てます。Km値が0.50mg/Lという低い値であるため、濃度が増加するに従って消失速度が増加します。

目標濃度を達成するために必要な新しい投与量は、現在の投与量（1日の合計では初日600mg、2日目以降400mg）を、定常状態での濃度比に基づいて調整する必要があります。計算結果として、新しい投与量は500mgとなります。
