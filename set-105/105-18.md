# 第105回 問18（化学）

**問題文**
食中毒を引き起こす自然毒のうち、植物に由来するのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-18 を参照

**選択肢**
1. サキシトキシン
2. シガトキシン
3. チャコニン
4. ジノフィシストキシン
5. テトロドトキシン

**正解:** 3

**解説**
解答　３

１　誤
サキシトキシンは、イガイやホタテガイなどの二枚貝に蓄積する麻痺性貝毒であり、摂取により神経麻痺などを引き起こす。

２　誤
シガトキシンは、熱帯や亜熱帯のサンゴ礁周辺に生息する魚類に蓄積する麻痺性魚毒であり、摂取によりドライアイスセンセーションなどの神経障害などを引き起こす。

３　正
チャコニンは、ジャガイモの芽や緑皮部に含まれる植物性自然毒であり、摂取によりコリンエステラーゼを阻害することで縮瞳、下痢、嘔吐などを引き起こす。

４　誤
ジノフィシストキシンは、イガイやホタテガイなどの二枚貝に蓄積する麻痺性貝毒であり、摂取により下痢などを引き起こす。

５　誤
テトロドトキシンは、フグなどに蓄積する麻痺性魚毒であり、摂取により神経障害などを引き起こす。
