# 第105回 問136（衛生）

**問題文**
放射性物質を取り扱う際の放射線被ばくの防護に関する記述のうち、誤っているのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-136 を参照

**選択肢**
1. γ線の遮へいには鉛遮へい用具が適している。
2. γ線による被ばく量は、被ばく時間が同じ場合、線源からの距離を20 cmから40 cmにすると、1/4になる。
3. α線は透過性が強いので、外部被ばくを防護する必要がある。
4. 内部被ばくを防ぐには、放射性物質の経口摂取、吸入摂取、経皮吸収を防ぐことが重要である。
5. 体内に取り込まれた放射性核種の放射線による内部被ばくを防ぐことは困難である。

**正解:** 3

**解説**
解答　３

１　正しい
γ線は、透過性が強いため、密度の大きい金属（鉛など）の厚い板で遮へいする。

２　正しい
外部被ばく線量は、距離の2乗に反比例する。線源からの距離を20 cmから40 cmと2倍にした場合、被ばく量は1/4（1/22）倍になる。

３　誤っている
α線は、透過性が弱く、ダンボール、厚紙で遮へいすることができる。また、外部被ばくとは、体外から放射線を浴びた時に起こる被ばくのことであり、透過性の強い放射線（γ線）ほど影響が大きく、防護する必要性が高い。

４　正しい
内部被ばくとは、放射性核種を体内に取り込んだ時に起こる被ばくのことであり、放射性物質の経口摂取、吸入摂取、経皮吸収を防ぐことが重要である。内部被ばくは、電離作用の大きい放射線（α線）ほど影響が大きい。

５　正しい
体内に取り込まれた放射性核種は、特定の臓器に集積しやすいものなどがあり、排泄が容易ではない。また、放射性核種が体内に取り込まれている間、内部被ばくを受け続けることになるため、内部被ばくを防ぐことは困難である。
