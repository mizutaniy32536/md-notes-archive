# 第105回 問224（実務）

**問題文**
62歳男性。肺炎感染症の治療のため、スルバクタムナトリウム・アンピシリンナトリウムの点滴投与が開始された。肺炎は改善されたが、投与5日目から、腹痛、頻回の水様性の下痢、発熱、白血球数及びCRP値の上昇が認められた。直腸内視鏡検査を行ったところ、多発する黄白色の偽膜、浮腫やびらんが認められ、偽膜性大腸炎と診断された。このため、スルバクタムナトリウム・アンピシリンナトリウムの点滴投与を中止し、抗菌薬の変更についてカンファレンスが開かれた。問224（物理・化学・生物）この患者で新たに発症した腸疾患とその原因菌に関する説明のうち、誤っているのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-224 を参照

**選択肢**
1. 原因菌は、腸内において常在細菌叢を形成している。
2. 原因菌は、経口感染する。
3. 原因菌は、空気中で生存できない芽胞非形成菌である。
4. 発症には、肺炎感染症の治療薬の投与による菌交代現象が関与する。
5. 症状は、原因菌が産生する外毒素により起こる。問225（実務）このカンファレンスにおいて、薬剤師が提案する抗菌剤として適切なのはどれか。２つ選べ。
6. セフジニルカプセル
7. クラブラン酸カリウム・アモキシシリン水和物配合錠
8. メトロニダゾール錠
9. バンコマイシン塩酸塩散
10. レボフロキサシン水和物錠

**正解:** 3

**解説**
問224　解答　３

本患者の主訴や検査所見より、肺炎感染症の治療のために点滴投与された抗菌薬のスルバクタムナトリウム・アンピシリンが、腸内細菌叢の変化（菌交代現象）が引き起こし、偽膜性大腸炎を発症したと考えられる。一般的に、菌交代現象による偽膜性大腸炎の原因菌は、クリストリディウム（クリストリディオイデス）・ディフィシル菌である。
ディフィシル菌は、偏性嫌気性のグラム陽性桿菌であり、健常人の腸管内において少数生息し、常在細菌叢を形成している。特徴として、熱、乾燥、アルコールに抵抗性を示す芽胞を形成する。
菌交代現象を起こしやすい抗菌薬として、アンピシリンなどのペニシリン系抗菌薬、クリンダマイシンなどのリンコマイシン系抗菌薬などがある。

１　正
前記参照

２　正
ディフィシル菌は、排泄物（便）で汚染された衣類、器具に触れた手などから経口感染する。

３　誤
ディフィシル菌は、偏性嫌気性の芽胞形成菌であり、通常空気中では生存することはできない。

４　正
前記参照

５　正
ディフィシル菌は外毒素を産生することで、大腸粘膜への潰瘍形成を引き起こし、偽膜性大腸炎を引き起こす。

問225　解答　３、４

偽膜性大腸炎を発症した場合、使用している抗菌薬の投与を中止し、バンコマイシン塩酸塩または、メトロニダゾールの経口投与を行う。よって、今回のカンファレンスで薬剤師が提案する抗菌剤として適切なのは、バンコマイシン塩酸塩散、メトロニダゾール錠である。
