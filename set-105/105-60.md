# 第105回 問60（衛生）

**問題文**
切迫早産の治療に用いられる薬物はどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-60 を参照

**選択肢**
1. オキシトシン
2. クロミフェンクエン酸塩
3. クロルマジノン酢酸エステル
4. メチルエルゴメトリンマレイン酸塩
5. リトドリン塩酸塩

**正解:** 5

**解説**
解答　５

１　誤
オキシトシンは、脳下垂体後葉ホルモン製剤であり、律動的子宮平滑筋収縮作用により、分娩誘発などに用いられる。

２　誤
クロミフェンクエン酸塩は、視床下部におけるエストロゲン受容体遮断によりGn-RH（ゴナドトロピン放出ホルモン）を分泌し、LH（黄体形成ホルモン）を分泌させるため、排卵障害に基づく不妊症の排卵誘発に用いられる。

３　誤
クロルマジノン酢酸エステルは、抗アンドロゲン作用を有する黄体ホルモン製剤であり、前立腺癌の治療などに用いられる。

４　誤
メチルエルゴメトリンマレイン酸塩は、持続的子宮平滑筋収縮作用により、子宮収縮の促進や子宮出血の予防・治療に用いられる。

５　正
リトドリン塩酸塩は、選択的アドレナリンβ2受容体刺激薬であり、子宮平滑筋弛緩作用により、切迫早産の治療用いられる。
