# 第105回 問158（薬理）

**問題文**
下部消化管に作用する薬物に関する記述のうち、正しいのはどれか。２つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-158 を参照

**選択肢**
1. ピコスルファートは、腸内細菌の作用でレインアンスロンを生成し、アウエルバッハ神経叢を刺激することで、大腸運動を促進する。
2. ラクツロースは、界面活性作用により腸内容物の表面張力を低下させ、水分を浸潤させることで、硬便を軟化させる。
3. ルビプロストンは、小腸上皮に存在するCl－チャネル2（ClC−2）を活性化することで、腸管腔内への水分分泌を促進する。
4. リナクロチドは、グアニル酸シクラーゼC受容体を活性化し、サイクリックGMP（cGMP）濃度を増加させることで、腸管分泌および腸管運動を促進する。
5. センノシドは、管腔内で水分を吸収して膨張し、腸壁を刺激することで、蠕動運動を促進する。

**正解:** 3、4

**解説**
解答　３、４

１　誤
ピコスルファートは、大腸刺激性下剤であり、腸内細菌の作用でジフェノール体を生成し、大腸粘膜を刺激し、蠕動運動を亢進すると共に水分吸収を阻害することにより緩下作用を示す。なお、腸内細菌の作用でレインアンスロンを生成し、アウエルバッハ神経叢を刺激することで、大腸運動を促進するのは、センノシドである。

２　誤
ラクツロースは、経口投与後ほとんど消化吸収されることなく下部消化管に到達し、腸管内の浸透圧を高めることで緩下作用を示す。なお、界面活性作用により腸内容物の表面張力を低下させ、水分を浸潤させることで、硬便を軟化させるのは、ジオクチルソジウムスルホサクシネートである。

３　正
ルビプロストンは、小腸粘膜上皮細胞に存在するCl－チャネル2（ClC−2）を活性化することで、腸管腔内への水分分泌を促進し、便を軟化させる。

４　正
リナクロチドは、腸管の管腔表面に存在するグアニル酸シクラーゼC受容体を活性化し、サイクリックGMP（cGMP）濃度を増加させることで、腸管分泌および腸管運動を促進する作用や、大腸痛覚過敏改善作用を示す。

５　誤
センノシドは、大腸刺激性下剤であり、腸内細菌の作用でレインアンスロンを生成し、アウエルバッハ神経叢を刺激することで、大腸運動を促進させ緩下作用を示す。なお、管腔内で水分を吸収して膨張し、腸壁を刺激することで、蠕動運動を促進するのは、カルメロースである。
