# 第105回 問76（薬剤）

**問題文**
医療法の規定に基づく「医療計画」を定めることが義務づけられているのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-76 を参照

**選択肢**
1. 厚生労働省
2. 都道府県
3. 地方厚生局長
4. 保健所
5. 市区町村

**正解:** 2

**解説**
解答　２

医療法において、厚生労働大臣の基本方針に従い、医療計画を定めるのは都道府県と規定されている。
よって設問の医療計画を定めるのは都道府県となる。
