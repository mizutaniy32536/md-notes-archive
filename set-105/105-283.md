# 第105回 問283

### 第105回薬剤師国家試験 問283

#### 問題文（実務）
73歳女性。卵巣がんStageⅢcに対してTC（パクリタキセル＋カルボプラチン）療法を施行していたが6ヶ月後に再発した。そこで2次療法として、ドキソルビシン塩酸塩をMPEG−DSPE修飾リポソームに封入した注射剤（ドキシル®注）を導入することとなった。  
5コース目の投与中に、患者から、刺入部に耐え難い焼かれるような痛みを感じ、赤く腫れているとの訴えがあり、ドキシル®注の血管外漏出が疑われた。本剤の血管外漏出の対処法として、適切なのはどれか。**2つ選べ。**[1][2][3][8]

#### 全選択肢
1. 患部を温める。  
2. すぐに留置針を抜く。  
3. 患部を生理食塩液でフラッシュする。  
4. デクスラゾキサンを静脈内投与する。  
5. 漏出部周囲から薬液や血液を吸引・除去する。[1][2][3]

#### 正解番号
**4, 5**[1][2][3][5]

#### 解説
ドキシル®注（ドキソルビシン塩酸塩リポソーム製剤）は起壊死性抗がん剤に分類され、血管外漏出により周囲組織に炎症や壊死を起こす可能性がある。[2]  
血管外漏出が疑われた場合の対処は、薬剤滴下を中止し、漏出部周囲から薬液や血液を吸引・除去した後、留置針を抜針する（選択肢5）。壊死性抗がん剤の場合、デクスラゾキサンを静脈内投与する（選択肢4）。[1][2][3]  
選択肢1は誤り（冷やすのが一般的）。選択肢2は誤り（吸引後抜針）。選択肢3は誤り（フラッシュは不適切）。[1][3]
