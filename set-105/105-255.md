# 第105回 問255

### 第105回薬剤師国家試験 問255

#### 科目名
**薬理**

#### 問題文全文
前問で推奨された薬物の1つ(薬物Aとする)を追加して治療を行っていたが、狭心症発作を起こした。そこでジルチアゼムが追加処方されたが、徐脈が起きたため、ジルチアゼムとの相互作用を疑い薬物Aを中止した。中止した薬物Aの作用として正しいのはどれか。１つ選べ。[1][2][3]

#### 全選択肢
1. **α1遮断作用**  
2. **ACE阻害作用**  
3. **直接的血管拡張作用**  
4. **Ca拮抗作用**  
5. **β遮断作用** [3][2]

#### 正解番号
**5** [2][4][9]

#### 解説
問254（前問）で心筋梗塞後の降圧薬として推奨された薬物Aは**メトプロロール酒石酸塩**（β遮断薬）である。ジルチアゼム（Ca拮抗薬）は心筋細胞のCa<sup>2+</sup>チャネルを遮断し心機能を抑制する。メトプロロールはβ遮断作用により心拍数を抑制するため、併用により心機能が過度に抑制され徐脈が生じた。よって、薬物Aの作用は**β遮断作用**が正しい。[2][7]
