# 第105回 問36（生物）

**問題文**
メテノロンのタンパク質同化作用の機序はどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-36 を参照

**選択肢**
1. アロマターゼ阻害
2. エストロゲン受容体刺激
3. ペルオキシダーゼ阻害
4. バソプレシンV2受容体刺激
5. アンドロゲン受容体刺激

**正解:** 5

**解説**
解答　５

メテノロンは男性ホルモン製剤あり、アンドロゲン受容体を刺激することにより、男性ホルモン様のタンパク質同化作用を示すため、骨粗しょう症などの治療に用いられる。
