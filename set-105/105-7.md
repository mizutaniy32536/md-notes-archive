# 第105回 問7（物理）

**問題文**
以下に示す薬物のIUPAC名として正しいのはどれか。１つ選べ。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-7 を参照

**選択肢**
1. (1S,2S )−1−Methylamino−2−phenylpropan−2−ol
2. (1S,2R )−1−Methylamino−2−phenylpropan−2−ol
3. (1S,2S )−2−Methylamino−1−phenylpropan−1−ol
4. (1R,2S )−1−Methyl−1−methylamino−2−phenylethan−2−ol
5. (1S,2S )−2−Methyl−2−methylamino−1−phenylethan−1−ol

**正解:** 3

**解説**
解答　３

IUPAC命名法では、主たる基（OH基）を含む最長の炭素鎖を主鎖（propan−1−ol）、その他の置換基（Methylamino、phenyl）を側鎖とする。また、主たる基に結合する炭素の位置番号を最小にするため、本化合物のIUPAC名は、(1S,2S )−2−Methylamino−1−phenylpropan−1−olとなるため、選択肢３が正答となる。
