# 第105回 問81（薬剤）

**問題文**
薬局薬剤師の役割として正しいのはどれか。１つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 105-81 を参照

**選択肢**
1. 入院患者の薬物療法を決定する。
2. 国民の主体的な健康管理を支援する。
3. 医師の指示に基づき、在宅患者に治療行為をする。
4. 親交のある患者に対し、供給不足の医薬品を優先的に配分する。
5. 来局者の健康診断の結果から糖尿病の早期診断をする。

**正解:** 2

**解説**
解答　２

日本薬剤師会が制定する薬剤師行動規範において、「薬剤師は、国民が自分自身の健康に責任を持ち、個人の意思又は判断のもとに健康を維持、管理するセルフケアを積極的に支援する」と規定されているため、国民の主体的な健康管理を支援することは、薬局薬剤師の役割として適切である。
また、薬剤師行動規範において、薬剤師には医薬品の供給を適切につかさどる任務があることも規定されているため、親交のある患者に対し、供給不足の医薬品を優先的に配分する行為は不適切である。
なお、薬物療法の決定、治療行為、病気の診断は医師の役割であり、薬剤師が行うことはできない。
