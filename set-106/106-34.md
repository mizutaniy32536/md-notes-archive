# 第106回 問34

**問題文**
妊娠末期の子宮平滑筋を収縮させる脳下垂体後葉ホルモン薬はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-34 を参照

**選択肢**
1. オキシトシン
2. エルゴメトリン
3. エストラジオール
4. ジノプロストン
5. プロゲステロン

**正解:** 1

**解説**
解答　１

１　正
オキシトシンは、下垂体後葉ホルモン製剤であり、律動的に子宮平滑筋を収縮させるため、分娩誘発などに用いられる。

２　誤
エルゴメトリンは、麦角アルカロイド製剤であり、持続的に子宮平滑筋を収縮させるため、子宮収縮の促進や子宮出血の予防・治療に用いられる。

３　誤
エストラジオールは、卵胞ホルモン製剤であり、月経困難症や更年期障害などに用いられる。

４　誤
ジノプロストンは、プロスタグランジンE2製剤であり、律動的に子宮平滑筋を収縮させるため、陣痛誘発などに用いられる。

５　誤
プロゲステロンは、黄体ホルモン製剤であり、月経困難症などに用いられる。
