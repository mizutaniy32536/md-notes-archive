# 第106回 問21

**問題文**
大腸がんのプロモーターはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-21 を参照

**選択肢**
1. 12−O−テトラデカノイルホルボール−13−アセテート（TPA）
2. 食塩
3. デオキシコール酸
4. フェノバルビタール
5. 2,3,7,8−テトラクロロジベンゾ−p−ジオキシン

**正解:** 3

**解説**
解答　３

１　誤
12−O−テトラデカノイルホルボール−13−アセテート（TPA）は、皮膚がんのプロモーターである。 

２　誤
食塩は、胃がんのプロモーターである。

３　正
デオキシコール酸は、大腸がんのプロモーターである。

４　誤
フェノバルビタールは、肝臓がんのプロモーターである。

５　誤
2,3,7,8−テトラクロロジベンゾ−p−ジオキシン（TCDD）は、肝臓がん、肺がん、皮膚がんなどの発がんプロモーターである。
