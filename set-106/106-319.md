## ◆ 調剤時の薬剤師の対応と処方箋への記載義務を判断するシナリオ
260306

**問題**
- （問318-319 連問）75歳男性。1年前から、高血圧症、脂質異常症、高尿酸血症と不眠症のため、近医を受診し、前回同様の薬剤が処方された。患者のお薬手帳には、3日前に初めて受診した別の泌尿器科から処方された薬が記載されていた。この患者に対し、調剤を行うにあたり薬剤師が行った対応のうち、薬剤師法に基づいて処方箋に記入しなければならないとされているのはどれか。1つ選べ。

​

**正解と理由**
- `1. 疑義照会に対する処方医の回答内容`
- **理由の構成**
	- **問題の整理**：75歳男性が近医と泌尿器科の2つの医療機関を受診しており、調剤を行う際に薬剤師が行った対応のうち、`薬剤師法`に基づいて「処方箋に記入しなければならない」ものを問う
	- **薬学的知識・仕様・制約**：`薬剤師法第24条`は、処方箋中に疑わしい点があるとき、その処方箋を交付した医師等に問い合わせて確認した後でなければ調剤してはならないと規定している。さらに`薬剤師法第26条`では、薬剤師は調剤したときは処方箋に調剤済みの旨・調剤年月日等を記入し、`疑義照会`を行った場合はその回答内容を処方箋に記入しなければならないと定められている
	- **選択理由**：`疑義照会`に対する処方医の回答内容は`薬剤師法第26条`により処方箋への記載が法的に義務づけられている唯一の選択肢であり、他の対応は薬歴等への記録事項であっても処方箋記載義務はない

**解法の手順（思考の流れ）**
- まず「薬剤師法に基づいて処方箋に記入しなければならない」という問いのキーワードから、`薬剤師法`上の処方箋記載義務に関する条文を想起する
- `薬剤師法第26条`が処方箋への記載事項を定めていることを確認し、調剤済みの旨・調剤年月日・`疑義照会`の回答内容が記載義務の対象であると判断する
- 各選択肢について、処方箋記載義務があるか否かを検討する
- 選択肢2〜5はいずれも`薬歴`（薬剤服用歴管理記録）への記録事項であるが、処方箋への記載義務は法的に規定されていないと判断する
- これにより`選択肢1`のみが`薬剤師法`に基づく処方箋記載義務に該当すると結論づけた

**他の選択肢が誤りな理由**
- 2. お薬手帳に関する指導の内容
	`お薬手帳`に関する指導内容は`薬歴`に記録すべき事項であるが、`薬剤師法`上の処方箋記載義務には含まれていない

- 3. 患者の家族から聞き取りをした内容
	患者の家族からの聞き取り内容は服薬指導や`薬歴`管理の一環として記録されるべきものであるが、処方箋に記入する法的義務はない

- 4. かかりつけ薬剤師に関する説明内容
	`かかりつけ薬剤師`に関する説明は患者への情報提供サービスであり、`薬歴`への記録は必要であるが、`薬剤師法`に基づく処方箋記載義務の対象ではない

- 5. 患者に行った受診勧奨の内容
	`受診勧奨`は薬剤師として適切な対応であるが、その内容を処方箋に記入する義務は`薬剤師法`上規定されていない

**キーポイント**
- `薬剤師法第24条`は`疑義照会`の義務を定め、処方箋中に疑わしい点がある場合は処方医への確認なしに調剤してはならないと規定している
- `薬剤師法第26条`は処方箋への記載義務を定め、`調剤済みの旨`・調剤年月日・`疑義照会`の回答内容の記入を義務づけている
- 処方箋記載義務と`薬歴`記録義務は法的根拠が異なり、混同しないことが重要である

**関連知識の比較**
- `薬剤師法第23条`は調剤の場所に関する規定で、薬局以外の場所での調剤を原則禁止している
- `薬剤師法第25条`は調剤した薬剤の容器・被包への`記載義務`（患者氏名・用法用量・調剤年月日・調剤した薬局名称等）を定めている
- `薬歴`（薬剤服用歴管理記録）は`薬剤服用歴管理指導料`の算定要件として整備が求められるものであり、患者情報・指導内容・聞き取り内容等を記録するが、これは処方箋記載義務とは別の法的枠組みに基づく
- `処方箋の保存義務`は`薬剤師法第28条`に規定されており、調剤済みの処方箋を`3年間`保存しなければならないとされている

（一言メモ）
- `疑義照会`（`薬剤師法第24条`）は処方箋中の疑義に対し処方医に確認する義務であり、その回答内容は`第26条`により処方箋に記入しなければならない
- 処方箋記載義務（`薬剤師法第26条`）と`薬歴`記録義務は法的根拠が異なるため、「どこに記録するか」を正確に区別することが試験対策上重要である
- `薬剤師法`の条文番号（第23条〜第28条）は調剤業務の根幹を規定しており、各条の対象と義務内容をセットで整理しておくと法規問題で強い
