# 第106回 問20

**問題文**
日本人の平均摂取量が、「日本人の食事摂取基準（2015年版）」における「目標量」よりも多いのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-20 を参照

**選択肢**
1. 炭水化物
2. 飽和脂肪酸
3. コレステロール
4. カリウム
5. 食物織維

**正解:** 2

**解説**
解答　２

「日本人の食事摂取基準（2015年版）」における目標量が設定されている栄養素は、炭水化物、脂質、飽和脂肪酸、カリウム及び食物繊維であり、コレステロールには目標量が設定されていない。
このうち、日本人の平均摂取量が、「日本人の食事摂取基準（2015年版）」における「目標量」よりも多いのは飽和脂肪酸である。
