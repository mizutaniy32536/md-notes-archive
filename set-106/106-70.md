# 第106回 問70

**問題文**
パラメトリック法に分類される仮説検定法はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-70 を参照

**選択肢**
1. Mann−WhitneyのU検定
2. t検定
3. カイ二乗検定
4. フィッシャーの直接確率検定
5. ログランク検定

**正解:** 2

**解説**
解答　２

パラメトリック法とは、母集団の分布が正規分布に従うと仮定した検定法であり、t検定や分散分析などが該当する。なお、母集団の分布が特定の分布を仮定しない検定法を、ノンパラメトリック法という。

１　誤
Mann−WhitneyのU検定は、ノンパラメトリック法の一つである。 

２　正
冒頭解説文参照。

３　誤
カイ二乗検定は、ノンパラメトリック法の一つである。

４　誤
フィッシャーの直接確率検定は、ノンパラメトリック法の一つである。  

５　誤
ログランク検定は、ノンパラメトリック法の一つである。
