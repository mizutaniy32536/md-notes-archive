# 106-330

**問題文**
32歳男性。双極性障害の患者。調剤薬の交付時には、いつも妻が横に付き添っている状況であった。今回、衝動コントロールが効かない状態になることがあり、次回の予約診察日より前に受診し、以下の処方が追加された。服薬指導の内容として最も適切なのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-330 を参照

**選択肢**
1. 口腔粘膜からの吸収による効果が期待できる。
2. 予めブリスターシートから取り出して保管する。
3. 口腔内の速溶により吐き出し防止が期待できる。
4. 服用時は空腹を避ける。
5. 緊急に服用する場合はブリスターシートの上から押し出してもよい。

**正解:** 3

**解説**
解答　３

１　誤
本剤は、口腔内崩壊錠であり、口腔内で速やかに崩壊することから唾液のみ（水なし）でも服用可能であるが、口腔粘膜からの吸収により効果発現を期待する製剤ではないため、口腔粘膜からの吸収による効果が期待できるという服薬指導の内容は不適切である。

２　誤
本剤は、吸湿性を有するためブリスター包装のまま保存することとされているため、予めブリスターシートから取り出して保管するという服薬指導の内容は不適切である。なお、本剤を投与する際は、乾かした手でブリスターシートから取り出し直ちに口内に入れるよう指導する。

３　正
本患者は、双極性障害であり、衝動コントロールが効かない状態であることから、処方薬を服用する際に吐き出してしまい、きちんと服用できないことにより病態が悪化する可能性が考えられる。そのため、付き添いできている妻に対しても、きちんと薬を服用することの重要性を伝えることも重要となる。本剤は、口腔内崩壊錠であり、口内で速やかに崩壊することから唾液のみ（水なし）でも服用可能であるため、服用時の吐き出し防止が期待できるという服薬指導の内容は適切である。

４　誤
本剤の食事による吸収の影響は、認められないため、服用時は空腹を避けるという服薬指導の内容は不適切である。

５　誤
本剤は、錠剤と比較してやわらかいため、シートを剥がさずに押し出そうとすると割れることがある。そのため、緊急に服用する場合はブリスターシートの上から押し出してもよいという服薬指導の内容は不適切である。もし、ブリスターシートから取り出す際に、欠けや割れが生じた場合は、全量服用することも併せて指導する。
