# 第106回 問121（衛生）

**問題文**
我が国における性感染症に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**
1. 尖圭コンジローマは、ヒトパピローマウイルスによって引き起こされる。
2. 定点把握報告の対象となっている性感染症のうち、膣カンジダ症が最も報告数が多い。
3. 淋菌感染症の報告数は、公衆衛生の向上により減少し、平成25年以降は感染の報告がない。
4. B型肝炎は、母子感染に加え、性的接触によっても起こる。
5. 感染症法では、梅毒への対応として特定職種への就業が制限されている。

**正解:** 1, 4

**解説**

1. ✅ **正** — 尖圭コンジローマはヒトパピローマウイルス（主に6型、11型などのDNAウイルス）による性感染症。

2. ❌ **誤** — 定点把握報告対象（淋菌感染症、性器クラミジア感染症、性器ヘルペス感染症、尖圭コンジローマ）のうち、最も報告数が多いのは**性器クラミジア感染症**。膣カンジダ症は対象外。

3. ❌ **誤** — 淋菌感染症の報告数は減少傾向だが、平成25年以降も報告あり。

4. ✅ **正** — B型肝炎は血液・体液を介した感染で、母子感染、血液感染、性行為による感染経路がある。

5. ❌ **誤** — 感染症法の就業制限対象は一類〜三類感染症等。梅毒は**五類感染症**のため就業制限なし。
