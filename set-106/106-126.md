# 106-126

### 第106回薬剤師国家試験 問126

**問題文（物理・化学・生物）**  
ビタミンKの生理作用に関する記述のうち、正しいのはどれか。**2つ選べ**。[3][2]

**選択肢**  
1. ビタミンKは、プロトロンビンの生成に関与する。  
2. ワルファリンによりビタミンK依存性凝固因子の生成が促進される。  
3. ビタミンKは、タンパク質のグルタミン酸残基の修飾に関与する。[3]

**正解番号**  
1, 3[1][2][3]

**解説**  
- 選択肢1: 正しい。ビタミンKはプロトロンビンの合成において補酵素として作用し、生成に関与する（過去問98-121参照）。[1][3]  
- 選択肢2: 誤り。ワルファリンはビタミンK依存性凝固因子の生成を阻害し、血液をサラサラにする薬であるため、「促進」ではない。[1][2]  
- 選択肢3: 正しい。ビタミンKは血液凝固因子などのタンパク質のグルタミン酸残基をγ-カルボキシグルタミン酸に修飾する（γ-カルボキシレーション）。[3]
