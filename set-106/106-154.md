# 106-154

# 第106回薬剤師国家試験 問154

## 問題文

26歳女性。以下の処方箋を持って来局した。患者からの聞き取りによると、「会社の部署の異動により、寝付けなくなった。眠りにつくことができれば朝まで眠れるが、寝付けないときには、ついスマートフォンで動画を見てしまう。寝坊するのが怖くて眠れない日もある。他に病気はない。」という。

この患者の睡眠障害の型として考えられるのはどれか。1つ選べ。[1][2][3]

## 全選択肢

1. 入眠障害
2. 中途覚醒
3. 熟眠障害
4. 早朝覚醒
5. 過眠障害

## 正解番号

**1（入眠障害）**[1][2][3]

## 解説

患者の主訴は「寝付けなくなった」という点です。これは睡眠の開始に困難を示す典型的な**入眠障害**の特徴です。[2]

他の睡眠障害型が除外される理由は以下の通りです：[1]

- **中途覚醒**：患者は「眠りにつくことができれば朝まで眠れる」と述べており、一度眠りについた後の途中覚醒がないため除外される
- **早朝覚醒**：同様に、朝まで眠れることから除外される
- **熟眠障害**：「ぐっすり眠れた気がしない」といった訴えがないため除外される
- **過眠障害**：「寝過ごしてしまう」といった訴えがないため除外される

したがって、この患者の睡眠障害は**入眠障害**（寝付けない状態）と診断されます。[1]
