# 106-167

# 第106回薬剤師国家試験 問167

## 問題文

35歳男性。身長173cm、体重85kg。父親が糖尿病。既往歴なし。喫煙歴なし。機会飲酒。会社事務職で日頃より運動不足であり、毎日1L以上の甘い清涼飲料水を飲用していた。

この1年間で体重が3kg増加したが、ここ数ヶ月は体重の減少を自覚している。7日前より全身倦怠感、口渇及び多尿を認めたため外来受診した。受診時の意識は清明であり、血糖値480mg/dL、HbA1c11.0％（NGSP値）、尿糖（4＋）、尿蛋白（－）、尿中ケトン体（3＋）であった。

**この患者で、血中濃度が顕著に上昇していると考えられるのはどれか。2つ選べ。**[2]

## 全選択肢

1. 3-ヒドロキシ酪酸
2. アセト酢酸
3. γ-アミノ酪酸
4. 水酸化物イオン
5. ナトリウムイオン

## 正解番号

**1, 2**[3]

## 解説

本症例は血糖値480mg/dL、尿中ケトン体（3＋）という所見から、**糖尿病性ケトアシドーシス（DKA）** の状態にあります。[10]

インスリン不足により、脂肪の分解が亢進し、肝臓での脂肪酸の酸化が増加します。その結果、**ケトン体が大量に産生・蓄積** されます。ケトン体の主な成分は以下の通りです：

- **3-ヒドロキシ酪酸**：ケトン体の約70～80%を占める主要なケトン体
- **アセト酢酸**：ケトン体の約20～30%を占める
- **アセトン**：ケトン体の一部で呼気で排出される

本症例の尿中ケトン体が（3＋）と高値を示しており、血中でも3-ヒドロキシ酪酸とアセト酢酸が顕著に上昇していると考えられます。

γ-アミノ酪酸、水酸化物イオン、ナトリウムイオンは、この病態では特に血中濃度が上昇する理由がありません。
