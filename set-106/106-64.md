# 第106回 問64

**問題文**
動揺病による嘔気の予防に用いられるのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-64 を参照

**選択肢**
1. アデノシン三リン酸二ナトリウム水和物
2. イソソルビド
3. エチゾラム
4. ジフェニドール塩酸塩
5. ジメンヒドリナート

**正解:** 5

**解説**
解答　５

動揺病とは、乗り物酔いのことであり、漠然とした腹部不快感を伴う悪心・嘔吐、めまい、蒼白、発汗および関連症状を含む症候群である。動揺病に伴う諸症状の予防には、ジメンヒドリナートやジフェンヒドラミンなどが用いられる。
