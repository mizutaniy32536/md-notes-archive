# 106-1

**問題文**
日本薬局方一般試験法において、次の装置が用いられる試験法はどれか。1つ選べ。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-1 を参照

**選択肢**
1. 融点測定法
2. 沸点測定法及び蒸留試験法
3. 凝固点測定法
4. 屈折率測定法
5. 比重及び密度測定法

**正解:** 2

**解説**
解答　２

本装置は、蒸留フラスコに液体試料を入れ適当な熱源で加熱することで、液体試料が気体になる時の温度（沸点）を測定し、気体となった試料を冷却器により液体に戻し、蒸留液としてメスシリンダーに集める装置であり、沸点測定法及び蒸留試験法に用いられる。
