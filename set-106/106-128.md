## ◆ 絶食時のグルコース供給源と代謝変動
260305

**問題**
図は、体重70kgの男性がグルコース100gを摂取後に絶食した場合の血中グルコース濃度に基づいた理論的なグルコース消費量の経時変化を示している。縦軸はグルコース消費量、横軸はグルコース摂取後の時間経過をステージI〜IVに分けて示す。「A」及び「B」は血中グルコースの供給源を示す。

この図に関する記述のうち、誤っているのはどれか。1つ選べ。

**選択肢**

1. Aは、グリコーゲンである。
​
2. Bは、糖新生であり、ステージII、IIIでは筋肉においてタンパク質が分解されて生じたアラニンが肝臓に運ばれて起こる。
​
3. ステージIII、IVでは、脂肪組織中のトリアシルグリセロールが血中に放出され、血中トリアシルグリセロール濃度が上昇する。
​
4. ステージIVでは、肝臓だけでなく腎臓においても糖新生が亢進する。
​
5. ステージIVでは、脳でケトン体がエネルギーとして利用される。

**正解と理由**
- `3. ステージIII、IVでは、脂肪組織中のトリアシルグリセロールが血中に放出され、血中トリアシルグリセロール濃度が上昇する。`
- **理由の構成**
	- **問題の整理**：絶食時のグルコース供給源（グリコーゲン分解・糖新生）と各ステージの代謝変動について、誤った記述を1つ選ぶ問題
	- **薬学的知識・仕様・制約**
		- 絶食時、脂肪組織では `ホルモン感受性リパーゼ` が活性化され、貯蔵TGを加水分解する
		- 分解産物は `遊離脂肪酸`（FFA）と `グリセロール` であり、TGがそのまま血中に放出されるわけではない
		- 血中に放出された遊離脂肪酸はアルブミンに結合して各組織に運ばれ、β酸化でエネルギー源となる
	- **選択理由**：選択肢3は「TGが血中に放出される」と記述しており、実際にはTGが加水分解されてFFA＋グリセロールとして放出される点が誤り

**解法の手順（思考の流れ）**
- まず図の構造を把握する → Aは絶食初期に減少する供給源 ＝ 肝`グリコーゲン`、Bは後期にかけて増加する供給源 ＝ `糖新生` と判断
- 選択肢1（A＝グリコーゲン）と選択肢2（B＝糖新生、アラニン-グルコースサイクル）は正しいと確認
- 選択肢3で「TGが血中に放出」に違和感 → 脂肪動員では`ホルモン感受性リパーゼ`がTGを加水分解し、遊離脂肪酸とグリセロールとして放出されることを想起
- 選択肢4（腎臓でも糖新生が亢進）と選択肢5（脳がケトン体を利用）は長期絶食の基本知識として正しいと確認
- 以上から選択肢3が唯一の誤りと結論づけた

**他の選択肢が誤りな理由**
- 1. Aは、グリコーゲンである。
	正しい記述。絶食初期（ステージI〜II）では主に肝臓の`グリコーゲン`が分解されてグルコースが供給される。肝グリコーゲンは約12〜18時間で枯渇する

- 2. Bは、糖新生であり、ステージII、IIIでは筋肉においてタンパク質が分解されて生じたアラニンが肝臓に運ばれて起こる。
	正しい記述。`アラニン-グルコースサイクル` により筋肉でタンパク質が分解されて生じた`アラニン`が肝臓に運ばれ、糖新生の基質となる

- 4. ステージIVでは、肝臓だけでなく腎臓においても糖新生が亢進する。
	正しい記述。長期絶食（ステージIV）では`腎臓`でも糖新生が亢進し、全体の糖新生の約40〜50%を担うようになる

- 5. ステージIVでは、脳でケトン体がエネルギーとして利用される。
	正しい記述。長期絶食時には肝臓で`ケトン体`（アセト酢酸・`β-ヒドロキシ酪酸`）が産生され、脳がこれをエネルギー源として利用することでグルコース消費を節約する

**キーポイント**
- 脂肪動員のプロセス：`ホルモン感受性リパーゼ` がTGを加水分解 → `遊離脂肪酸`＋グリセロール → TGそのものは血中に放出されない
- 遊離脂肪酸は`アルブミン`に結合して血中輸送され、各組織でβ酸化を受ける
- 肝グリコーゲンは絶食開始後約12〜18時間で枯渇し、以降は`糖新生`が主なグルコース供給源となる
- 長期絶食（数日〜数週間）では腎臓の糖新生が全体の40〜50%を占めるまで亢進する
- 脳は通常グルコースのみをエネルギー源とするが、長期絶食時には`ケトン体`もエネルギー源として利用可能

**関連知識の比較**
- `ホルモン感受性リパーゼ`（HSL）と`リポタンパク質リパーゼ`（LPL）の違い：HSLは脂肪組織内でTGを加水分解する細胞内酵素。LPLは血管内皮に存在し、リポタンパク質中のTGを加水分解する
- 糖新生の基質の種類：`アラニン`（筋肉由来）、`グリセロール`（脂肪動員由来）、`乳酸`（嫌気的解糖由来、コリ回路）、オキサロ酢酸など
- `ケトン体` の種類と生成：アセト酢酸、β-ヒドロキシ酪酸、アセトンの3種。肝臓のミトコンドリアでアセチルCoAから`HMG-CoA`を経て生成される
- 絶食ステージの代謝変化：ステージI（食後吸収期）→ II（グリコーゲン分解優位）→ III（糖新生優位、脂肪動員開始）→ IV（`ケトン体`利用、腎臓の糖新生亢進）

（一言メモ）
- 脂肪組織からの脂肪動員は「TGそのまま放出」ではなく「`ホルモン感受性リパーゼ` による加水分解→FFA＋グリセロール」が正しいプロセス
- `アラニン-グルコースサイクル` と `コリ回路`（乳酸-グルコースサイクル）は糖新生の二大経路として区別して覚える
- 長期絶食時の`腎臓`の糖新生寄与率（40〜50%）は頻出ポイント
