# 第106回 問36

**問題文**
アプレピタントの制吐作用に関わる作用点はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-36 を参照

**選択肢**
1. ヒスタミンH1受容体
2. ドパミンD2受容体
3. タキキニンNK1受容体
4. セロトニン5−HT3受容体
5. オピオイドμ受容体

**正解:** 3

**解説**
解答　３

アプレピタントは、嘔吐中枢やCTZのタキキニンNK1受容体を選択的に遮断して、抗悪性腫瘍薬による急性および遅発性悪心、嘔吐を抑制する。
