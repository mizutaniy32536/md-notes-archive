# 第106回 問88

**問題文**
ドライパウダー吸入器の使用法に関する説明として適切なのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-88 を参照

**選択肢**
1. 吸入前に容器をよく振ってください。
2. 吸入前に息を吐かないでください。
3. 吸入時は勢いよく深く息を吸い込んでください。
4. 吸入してから息を止めないでください。
5. 吸入口を下に向けて吸い込んでください。

**正解:** 3

**解説**
解答　３

ドライパウダー吸入器は、吸入量が一定になるように調製された粉末製剤を吸入する器具である。ドライパウダー吸入器の使用法を下記に示す。
【ドライパウダー吸入器の使用法】
 １）確実に１回分が吸入できるようにセットする。
 ２）吸入前に息を吐く(吸入器(デバイス)に向けて息を吐かないように注意する)。
 ３）水平にして、勢いよく深く息を吸い込む。
 ４）吸入口(マウスピース)から口を離して、軽く息を止める。
 ５）ゆっくりと息を吐き出す。
 ６）吸入後にうがいを行う。

１　誤
吸入前に容器を振ると、粉末製剤がこぼれる可能性があるため、容器は振らないよう説明する。なお、吸入前に容器をよく振る必要があるのは吸入エアゾール剤である。

２　誤
前記参照。

３　正
前記参照。

４　誤
前記参照。

５　誤
吸入口を下に向けると、粉末製剤がこぼれる可能性があるため、吸入口は下に向けないよう指導する。
