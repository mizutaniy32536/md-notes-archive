# 106-113

**問題文**
検体中における、ある微生物の存在を調べるために、リアルタイムPCR法を実施した。以下に示す測定手順で行い、測定した結果を図に示す。この実験に関する記述のうち、正しいのはどれか。2つ選べ。測定手順3つの検体（試料1〜3）をサンプルチューブに別々に採取し、それぞれにDNA抽出用の試薬を加える。       ↓抽出したDNAを定量する。       ↓各検体から一定量のDNAを別々のチューブに取り、それぞれのチューブに二本鎖DNAを検出する蛍光色素、デオキシヌクレオチド混合物、プライマー1組、酵素を含む反応液を加えて、PCRを開始する。↓蛍光強度を測定することで、反応産物が増幅されていく経過を追いながら、PCRを約40サイクルまで繰り返す。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-113 を参照

**選択肢**
1. 目的とする微生物の存在量が最も多かったのは、試料3である。
2. この測定手順で、細菌、DNAウイルス及びRNAウイルスの検出が可能である。
3. 試料1と試料3に含まれる微生物の存在量は、約106倍異なると推定される。
4. 各試料とも25サイクルを超えるあたりから曲線が頭打ちになる主な原因は、デオキシヌクレオチドが枯渇するためである。
5. PCR開始時に反応液に加える酵素には耐熱性のものを用いる。

**正解:** 4,5

**解説**
解答　４、５

PCR法は、通常試験管内における温度変化を利用して目的DNAを複製して増幅する方法である。

①	二本鎖DNAを熱変性により水素結合を切り、一本鎖にする。
②	アニーリングによるプライマーの結合
③	耐熱性DNAポリメラーゼによるDNAの合成
④	①〜③の手順を１サイクルとして、20〜30回サイクル繰り返すことにより、目的のDNA領域を220〜230倍に複製することが可能となる。

リアルタイムPCR法は、PCR法により増幅されるDNAを蛍光物質で標識することにより、リアルタイムでDNAを定量できる測定法である。PCR法におけるDNA量はサイクル数の増加により指数関数的に増幅され、最終的には定常状態に達する。そのため、用いるDNA量が多ければ、設定した蛍光強度に達する時間も短くなり、必要なサイクル数も少なくなる。また、細菌DNAやウイルス由来のDNAをPCR法により増幅することで、感染の有無を調べることが可能となる。

１　誤
PCR法に用いるDNA量が多いほど、少ないサイクル数で増幅曲線が起き上がり閾値に達するため、目的とする微生物の存在量が最も多かったのは、試料1である。

２　誤
本測定手順により、細菌DNAやDNAウイルスの検出は可能であるが、RNAウイルスを検出することはできない。リアルタイムPCR法の原理によるRNAの検出を行う場合は、目的のRNAを逆転写によるcDNA変換とリアルタイムPCR法を組み合わせたリアルタイム逆転写PCR法を行う必要がある。

３　誤
リアルタイムPCR法によるDNA量は、サイクル数をn回としたとき、2n倍に増幅される。試料1と試料3のサイクル数は6回異なるため、試料1と試料3に含まれる微生物の存在量は、約26倍異なると考えられる。

４　正
PCR法でDNAが増幅する際、基質のデオキシヌクレオチド、プライマー、DNAポリメラーゼなどを必要とする。DNAを増幅する過程でサイクル数が増えていくと、反応液におけるこれらの量は減少し、最終的に枯渇する。そのため、サイクル数が増え蛍光強度が閾値を超えたあたりからDNAの増幅率は低下していき、最終的には頭打ちがみられる。

５　正
PCR法では、熱変性によるDNAポリメラーゼの失活を防ぐ目的で、耐熱性DNAポリメラーゼを用いる。
