# 106-60

**問題文**
続発性副甲状腺機能低下症の治療に用いられるのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-60 を参照

**選択肢**
1. アルファカルシドール
2. エルカトニン
3. チアマゾール
4. ブロモクリプチン塩酸塩
5. プレドニゾロン

**正解:** 1

**解説**
解答　１

続発性副甲状腺機能低下症は、甲状腺手術や放射線療法による副甲状腺ホルモン（パラトルモン）分泌不全に伴う低Ca血症などをきたす病態であり、治療には腸管からのCa吸収促進作用をもつ活性型VD3製剤（アルファカルシドールなど）や、Ca製剤（グルコン酸カルシウムなど）が用いられる。
