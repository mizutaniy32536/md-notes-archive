# 106-86

**問題文**
ラモトリギンに対して発出された安全性速報（ブルーレター）を契機に添付文書の「警告」に記載された重篤な副作用はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-86 を参照

**選択肢**
1. 低カルシウム血症
2. 高ビリルビン血症
3. 間質性肺疾患
4. 腎機能障害
5. 皮膚障害

**正解:** 5

**解説**
解答　５

１　誤
低カルシウム血症に安全性速報（ブルーレター）が発出されたのはデノスマブである。

２　誤
高ビリルビン血症に安全性速報（ブルーレター）が発出されたのはシメプレビルである。

３　誤
間質性肺疾患に安全性速報（ブルーレター）が発出されたのはアベマシクリブである。

４　誤
腎機能障害（急性腎不全）に緊急安全性情報（イエローレター）が発出されたのはエダラボンである。

５　正
皮膚障害に安全性速報（ブルーレター）が発出されたのはラモトリギンである。
