# 106-342

**問題文**
30歳男性。眼に違和感があり近医を受診したところ、麦粒腫と診断され、下記処方箋を持って来局した。薬剤師が患者に説明する眼軟膏の使用方法として、最も適切なのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-342 を参照

**選択肢**
1. 眼を閉じて、まぶたの外側に均等に塗布する。
2. 眼を閉じて、目元に塗布する。
3. 眼を閉じて、目尻に塗布する。
4. 下まぶたを下にひき、目の中央に塗布する。＜image6＞
5. 下まぶたを下にひき、まぶたの内側に塗布する。

**正解:** 5

**解説**
解答　５

眼軟膏剤は、結膜嚢などの眼組織に適応する半固形状の無菌製剤である。
眼軟膏剤の使用方法は以下の通りである。
①手を石鹸でよく洗う。
②鏡を見ながら、下まぶたを軽く引き、チューブの先がまぶたやまつげ、眼球に触れないように注意しながら、チューブを少し押して下まぶたに薬を塗布する。
③目を閉じ、軟膏が溶けて全体に広がるまで少し待つ。
④目の外にあふれた軟膏は、清潔なガーゼやティッシュで拭き取る。
⑤使用後は、チューブの先が不潔にならないよう注意してふたをする。
