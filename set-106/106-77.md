# 106-77

**問題文**
患者が自らの治療のために、2週間程度の海外旅行に携帯する場合、地方厚生（支）局長の許可が必要となるのはどれか。1つ選べ。なお、地方厚生（支）局長は、厚生労働大臣から権限が委任されているものとする。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-77 を参照

**選択肢**
1. 麻薬
2. 向精神薬
3. あへん
4. 覚醒剤
5. 大麻

**正解:** 1

**解説**
解答　１

麻薬輸出業者でなければ、麻薬を輸出してはならない。ただし、本邦から出国する者が厚生労働大臣（厚生労働大臣から権限が委任されている場合には、地方厚生（支）局長）の許可を受けて、自己の疾病の治療の目的で携帯して出国する場合はこの限りではない（麻薬及び向精神薬取締法第17条）。
したがって、地方厚生（支）局長が厚生労働大臣から権限が委任されており、地方厚生（支）局長の許可を受けている場合、患者が自らの治療のために、海外旅行に携帯できるのは麻薬である。
なお、麻薬と同様に地方厚生（支）局長が厚生労働大臣から権限が委任されており、地方厚生（支）局長の許可を受けている場合、患者が自らの治療のために、海外旅行に携帯できるものには、覚醒剤原料もある。
