# 第106回 問4

**問題文**
塩化水素（気体）のH原子とCl原子の間の結合として正しいのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-4 を参照

**選択肢**
1. 共有結合
2. イオン結合
3. 水素結合
4. 金属結合
5. 疎水結合

**正解:** 1

**解説**
解答　１

１　正
共有結合は、一般に非金属原子と非金属原子の間に形成される原子間結合である。H原子とCl原子はどちらも非金属原子であるため、塩化水素の原子間結合は共有結合である。

２　誤
イオン結合は、一般に金属原子と非金属原子の間に形成される原子間結合である。

３　誤
水素結合は、電気陰性度の大きな原子（F原子、O原子、N原子）に直接結合したH原子と他分子の電気陰性度の大きな原子（F原子、O原子、N原子）の間に形成される分子間相互作用である。

４　誤
金属結合は、金属原子と金属原子の間に形成される原子間結合である。

５　誤
疎水結合は、水中において、疎水性分子が集まるように見える分子間相互作用である。
