## ◆ 橋梁塗装作業者の有害物質中毒における解毒薬を選択するシナリオ
260305

**問題**
- 〔106-236〜237 共通設問〕44歳男性。高速道路などの橋梁塗装作業に従事している。6ヶ月に一度の特殊健康診断時に、自覚症状として、全身倦怠感、食欲不振、腹痛やめまいなどを申告した。その後、血液検査結果から産業医が有害物質による中毒と診断し、医療機関での診察を勧めた。
	- 尿中δ−アミノレブリン酸濃度：5.5 mg/L
	- 赤血球数 420×10⁴/µL、白血球数 4,700/µL、血小板数 27×10⁴/µL
	- Hb 10.2 g/dL、Ht 36%
- この患者に使用する解毒薬として適切なのはどれか。2つ選べ。

**選択肢**


1. ジメルカプロール

2. プラリドキシムヨウ化物

3. エデト酸カルシウムナトリウム水和物

4. 亜硝酸アミル

5. アトロピン硫酸塩水和物

**正解と理由**
- `1. ジメルカプロール`
- `3. エデト酸カルシウムナトリウム水和物`
- **理由の構成**
	- **問題の整理**：橋梁塗装作業者が全身倦怠感、食欲不振、腹痛、めまいを訴え、尿中`δ-アミノレブリン酸`（δ-ALA）が高値、`Hb 10.2 g/dL`と貧血を呈しており、これらの所見から`鉛中毒`と判断される
	- **薬学的知識・仕様・制約**：橋梁塗装には`鉛系塗料`が使用されており、慢性的な鉛曝露による職業性`鉛中毒`が起こりうる。鉛はヘム合成経路の`δ-ALA脱水酵素`および`フェロケラターゼ`を阻害し、δ-ALAの蓄積と`小球性低色素性貧血`を引き起こす。鉛中毒の解毒にはSH基を持つキレート剤`ジメルカプロール`（BAL）および`CaNa₂EDTA`（エデト酸カルシウムナトリウム水和物）が用いられる
	- **選択理由**：`ジメルカプロール`はSH基で鉛をキレートして排泄を促進し、`CaNa₂EDTA`はカルシウムを鉛と交換してキレート錯体を形成し腎排泄させる。いずれも鉛中毒の標準的な治療薬である

**解法の手順（思考の流れ）**
- まず「橋梁塗装作業」「特殊健康診断」というキーワードから、職業性有害物質曝露を想起する
- 次に尿中`δ-ALA`高値と`貧血`（Hb 10.2 g/dL）の組み合わせから、ヘム合成障害を引き起こす`鉛中毒`と判断する
- 鉛中毒の解毒薬として`ジメルカプロール`（BAL）と`CaNa₂EDTA`が想起される
- 選択肢2の`プラリドキシム`と選択肢5の`アトロピン`は`有機リン中毒`の治療薬であり除外する
- 選択肢4の`亜硝酸アミル`は`シアン中毒`の解毒薬（MetHb形成）であり除外する
- これにより選択肢1と3が鉛中毒の解毒薬として適切であると結論づけた

**他の選択肢が誤りな理由**
- 2. プラリドキシムヨウ化物
	`プラリドキシム`（PAM）は`有機リン中毒`に用いるコリンエステラーゼ再賦活薬である。有機リンにより`リン酸化`されたAChEの活性部位を求核攻撃して再活性化する薬物であり、鉛中毒には無効である

- 4. 亜硝酸アミル
	`亜硝酸アミル`は`シアン中毒`の解毒に用いられる。亜硝酸により`メトヘモグロビン`（MetHb）を形成し、MetHbのFe³⁺がCN⁻を捕捉して`シアンメトヘモグロビン`を生成することでシトクロム酸化酵素からCN⁻を解離させる機序であり、鉛中毒には適応がない

- 5. アトロピン硫酸塩水和物
	`アトロピン`は`有機リン中毒`における`ムスカリン様症状`（縮瞳、流涎、徐脈、気管支収縮など）の拮抗薬として用いられる。ムスカリン受容体を遮断して副交感神経の過剰興奮症状を抑えるが、鉛中毒の治療機序とは無関係である

**キーポイント**
- `鉛中毒`の診断指標：尿中`δ-ALA`増加、尿中`コプロポルフィリン`増加、赤血球中`プロトポルフィリン`増加、`好塩基性斑点赤血球`の出現
- 鉛はヘム合成経路の`δ-ALA脱水酵素`と`フェロケラターゼ`を阻害し、δ-ALAやコプロポルフィリンが蓄積する
- 鉛中毒の解毒薬は`ジメルカプロール`（BAL）と`CaNa₂EDTA`の2つをセットで覚える
- `特殊健康診断`は鉛、有機溶剤、特定化学物質などの業務に従事する労働者に対して実施が義務づけられている

**関連知識の比較**
- `ジメルカプロール`（BAL）はSH基を持つキレート剤で、鉛のほかに`ヒ素`、`水銀`、`銅`（ウィルソン病）の中毒にも使用される
- `D-ペニシラミン`もSH基を持つキレート剤で、鉛中毒（経口投与可能な場合）や`ウィルソン病`、`関節リウマチ`に用いられる
- `デフェロキサミン`はSH基ではなくヒドロキサム酸基で`鉄`を特異的にキレートする解毒薬であり、鉛には使用しない
- `プラリドキシム`＋`アトロピン`は`有機リン中毒`のセット治療であり、PAMはChE再賦活（ニコチン様症状にも有効）、アトロピンはムスカリン様症状の拮抗と役割が異なる
- シアン中毒の解毒には`亜硝酸アミル`→`亜硝酸ナトリウム`→`チオ硫酸ナトリウム`の順に投与する方法と、`ヒドロキソコバラミン`を単独投与する方法がある

（一言メモ）
- `鉛中毒`の解毒薬は`BAL`（ジメルカプロール）と`CaNa₂EDTA`の2剤セット、`有機リン中毒`は`PAM`と`アトロピン`の2剤セットで覚えると混同を防げる
- 尿中`δ-ALA`高値＝鉛中毒のサインと覚えておく。δ-ALA脱水酵素が鉛で阻害されるため基質のδ-ALAが蓄積して尿中排泄が増加する
- 橋梁塗装や蓄電池製造など`鉛取扱い業務`では6ヶ月ごとの`特殊健康診断`が義務であり、血中鉛濃度や尿中δ-ALAが検査項目に含まれる
