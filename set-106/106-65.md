## ◆ 空気感染（飛沫核感染）する病原体を選ぶシナリオ

260312

**問題**
空気感染する病原体はどれか。1つ選べ。

**選択肢**

1. インフルエンザウイルス
​
2. ヒト免疫不全ウイルス（HIV）
​
3. ポリオウイルス
​
4. 風しんウイルス
​
5. 麻しんウイルス

**正解と理由**
- `5. 麻しんウイルス`
- **理由の構成**
	- **問題の整理**：`空気感染`（飛沫核感染）する病原体を1つ選ぶ
	- **薬学的知識・仕様・制約**：空気感染する病原体は「`結核菌`・`麻疹ウイルス`・`水痘帯状疱疹ウイルス`」の3つ（＝`ケッ・マ・ミ`と覚える）
	- **選択理由**：選択肢の中で空気感染するのは`麻しんウイルス`のみ

**解法の手順（思考の流れ）**
- 空気感染の代表3つ=`結核`・`麻疹`・`水痘`を想起
- 選択肢1：`インフルエンザ`=`飛沫感染`（主） → ×
- 選択肢2：`HIV`=`血液・体液感染` → ×
- 選択肢3：`ポリオウイルス`=`糞口感染`（経口感染） → ×
- 選択肢4：`風しん`=`飛沫感染` → ×
- 選択肢5：`麻しん`=`空気感染` → ○

**他の選択肢が誤りな理由**
- 1. インフルエンザウイルス
	主に`飛沫感染`。空気感染の可能性は密閉空間で限定的に議論されるが、主要な感染経路は飛沫

- 2. ヒト免疫不全ウイルス（HIV）
	`血液・体液感染`（性的接触、血液曝露、母子感染）

- 3. ポリオウイルス
	`糞口感染`（経口感染）。腸管内で増殖するエンテロウイルス属

- 4. 風しんウイルス
	`飛沫感染`。空気感染はしない

**キーポイント**
- 空気感染（飛沫核感染）の3大病原体：**`結核菌`・`麻疹ウイルス`・`水痘帯状疱疹ウイルス`**
- `飛沫感染`と`空気感染`の違い：飛沫（5μm以上）はすぐ落下、飛沫核（5μm未満）は空気中に長時間浮遊
- `麻疹`は感染力が極めて強い（基本再生産数R₀=12〜18）

**関連知識の比較**

| 感染経路 | 代表的病原体 |
|---|---|
| `空気感染` | 結核菌、麻疹ウイルス、水痘帯状疱疹ウイルス |
| `飛沫感染` | インフルエンザ、風しん、百日咳、ムンプス |
| `接触感染` | MRSA、ノロウイルス、腸管出血性大腸菌 |
| `血液・体液感染` | HIV、HBV、HCV |
| `糞口感染` | ポリオ、ロタ、A型肝炎、E型肝炎 |

（一言メモ）
- 空気感染=`ケッ・マ・ミ`（結核・麻疹・水痘（みずぼうそう））。この3つだけ覚えれば必須問題は即答
- 麻疹の感染力はインフルエンザの6倍以上。同じ部屋にいるだけで感染しうる
- `N95マスク`が必要なのは空気感染対策。飛沫感染ならサージカルマスクで十分
