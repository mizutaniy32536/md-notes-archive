# 第106回 問81

**問題文**
チーム医療における医療者同士の関係性として適切なのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-81 を参照

**選択肢**
1. 独立
2. 依存
3. 協働
4. 主従
5. 競合

**正解:** 3

**解説**
解答　３

チーム医療とは、患者をチームの中心またはチームの一員として、多種多様な医療スタッフが、各々の高い専門性を前提に、目的と情報を共有し、業務を分担しつつも互いに連携・補完し合い、患者の状況に的確に対応した医療を提供することであるため、チーム医療における医療者同士の関係性として適切なのは協働である。
