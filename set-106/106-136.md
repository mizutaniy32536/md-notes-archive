## ◆ 電離放射線の防御に関する正誤を問うシナリオ
260303

**問題**
- 電離放射線の防御に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**

1. 防御の3原則である「時間」、「距離」、「遮へい」は、内部被曝の低減を目的としている。
​
2. α線はX線やγ線に比べて電離作用が強いので、外部被曝の危険性が高い。
​
3. ³²Pから放出されるエネルギーの強いβ線を遮へいする場合は、鉛板や鉛ブロックを用いる。
​
4. 安定ヨウ素剤は、甲状腺への¹³¹Iの蓄積を防ぐために投与される。
​
5. 放射性同位体を摂取した後、体内の放射線量が半分になるまでの期間を有効半減期という。

**正解と理由**
- `4. 安定ヨウ素剤は、甲状腺への¹³¹Iの蓄積を防ぐために投与される。` と `5. 放射性同位体を摂取した後、体内の放射線量が半分になるまでの期間を有効半減期という。`
- **理由の構成**
	- **問題の整理**：電離放射線の防御に関する正しい記述を2つ選ぶ問題
	- **薬学的知識・仕様・制約**：`安定ヨウ素剤` は非放射性ヨウ素で甲状腺を飽和させ、放射性 `¹³¹I` の取り込みを競合的に阻害する。`有効半減期` は物理的半減期と生物学的半減期の両方を考慮した値
	- **選択理由**：選択肢1は外部被曝の誤り、2はα線の透過力の誤り、3はβ線の遮へい材の誤り

**解法の手順（思考の流れ）**
- 選択肢1：防御の3原則は「`外部被曝`」の低減が目的であり、内部被曝ではないと判断
- 選択肢2：α線は電離作用が強いが `透過力が低い`（紙で遮断可能）ため、外部被曝リスクは低いと判断
- 選択肢3：β線に鉛を使うと `制動放射線` が発生するため不適切。`アクリル板` が正しいと判断
- 選択肢4・5は正しい記述であると結論づけた

**他の選択肢が誤りな理由**
- 1. 防御の3原則である「時間」、「距離」、「遮へい」は、内部被曝の低減を目的としている。
	防御の3原則は `外部被曝` の低減が目的。内部被曝は体内に取り込まれた放射性物質によるものであり、距離をとっても低減できない

- 2. α線はX線やγ線に比べて電離作用が強いので、外部被曝の危険性が高い。
	`α線` は電離作用は強いが `透過力` が極めて低い（紙1枚で遮断可能）。外部被曝リスクは低く、`内部被曝` が危険

- 3. ³²Pから放出されるエネルギーの強いβ線を遮へいする場合は、鉛板や鉛ブロックを用いる。
	β線を鉛など高原子番号物質で遮へいすると `制動放射線`（ブレムスシュトラールング）が発生して逆に危険。β線には `アクリル板` など低原子番号物質を使用する

**キーポイント**
- 防御の3原則（時間・距離・遮へい）は `外部被曝` の低減が目的
- `α線`：電離作用強い・透過力低い → 内部被曝が危険
- `β線`の遮へい：アクリル板（鉛は制動放射線発生のため不可）
- `有効半減期`の公式：1/T_eff = 1/T_phys + 1/T_bio

**関連知識の比較**
- 放射線の遮へい材：α線＝紙、`β線`＝アクリル板、`γ線`・X線＝鉛・コンクリート、`中性子線`＝水・パラフィン
- `線量当量`（Sv）= 吸収線量（Gy）× 放射線荷重係数。α線の荷重係数は20、β線・γ線は1
- `確定的影響`（しきい値あり：白内障、不妊等）と `確率的影響`（しきい値なし：がん、遺伝的影響）の区別が重要

（一言メモ）
- `α線` は「電離作用が強い＝体内で最も危険（内部被曝）」「透過力が低い＝体外では安全（外部被曝リスク低い）」のセットで覚える
- `制動放射線` はβ線が重い原子核の近くで急減速した際に発生するX線であり、鉛でβ線を遮へいしてはいけない理由
- `安定ヨウ素剤` は原発事故時の `¹³¹I` 被曝防護に使用され、事故後できるだけ早期に服用することが重要
