## ◆ 副腎皮質束状層のホルモンを選ぶシナリオ
260312

**問題**
副腎皮質の束状層から分泌されるホルモンはどれか。1つ選べ。

**選択肢**

1. コルチゾール
​
2. アルドステロン
​
3. テストステロン
​
4. ノルアドレナリン
​
5. アドレナリン

**正解と理由**
- `1. コルチゾール`
- **理由の構成**
	- 副腎皮質`束状層` → `糖質コルチコイド（コルチゾール）`を分泌

**解法の手順（思考の流れ）**
- 選択肢1：コルチゾール = 束状層から分泌される`糖質コルチコイド` → **○**
- 選択肢2：アルドステロン = `球状層`から分泌（鉱質コルチコイド）→ ✗
- 選択肢3：テストステロン = 主に`精巣`から（副腎では`網状層`から少量）→ ✗
- 選択肢4：ノルアドレナリン = `副腎髄質`から分泌 → ✗
- 選択肢5：アドレナリン = `副腎髄質`から分泌 → ✗

**キーポイント**
- 副腎皮質3層の覚え方：「球(きゅう)束(そく)網(もう)」= 外側から内側へ

**関連知識の比較**

| 層 | ホルモン | 種類 |
|---|---|---|
| **球状層** | アルドステロン | 鉱質コルチコイド |
| **束状層** | **コルチゾール** | 糖質コルチコイド |
| **網状層** | DHEA、アンドロゲン | 性ホルモン前駆体 |
| 副腎**髄質** | アドレナリン、ノルアドレナリン | カテコールアミン |

（一言メモ）
- 覚え方：「球→束→網」「鉱→糖→性」（外→内の順）
- 皮質 = ステロイドホルモン、髄質 = カテコールアミン
