# 106-93

**問題文**
日本薬局方において、ヒドロコルチゾンコハク酸エステル（下図）の確認試験及び純度試験は以下のように規定されている（一部省略）。この確認試験及び純度試験に関する記述のうち、正しいのはどれか。2つ選べ。＜image1:center＞確認試験（1）本品3 mgに硫酸2 mLを加えるとき、液は初め帯黄緑色の蛍光を発し、徐々に橙黄色を経て暗赤色に変わる。この液は紫外線を照射するとき、強い淡緑色の蛍光を発する。この液に注意して水10 mLを加えるとき、液は黄色から橙黄色に変わり、淡緑色の蛍光を発し、黄褐色綿状の浮遊物を生じる。（2）略純度試験類縁物質　本品25 mgをとり、メタノール10 mLを正確に加えて溶かし、試料溶液とする。別にヒドロコルチゾン25 mgをとり、メタノール10 mLを正確に加えて溶かす。この液1 mLを正確に量り、メタノールを加えて正確に50 mLとし、標準溶液とする。これらの液につき、薄層クロマトグラフィーにより試験を行う。試料溶液及び標準溶液3 µLずつを薄層クロマトグラフィー用 　ア　（蛍光剤入り）を用いて調製した薄層板にスポットする。次にクロロホルム／エタノール（99.5）／ギ酸混液（150：10：1）を展開溶媒として約10 cm展開した後、薄層板を風乾する。これに紫外線（主波長 　イ　 nm）を照射するとき、試料溶液から得た主スポット以外のスポットは、標準溶液から得たスポットより濃くない。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-93 を参照

**選択肢**
1. 確認試験は、試料中に含まれる不純物の限度あるいは量を調べる試験法である。
2. 確認試験(1)は、ステロイドの確認反応である。
3. ア　 に入るのは、「シリカゲル」である。
4. 純度試験では、類縁ステロイドの混在が許される限度は5％である。
5. イ　 に入る数値は、450である。

**正解:** 2,3

**解説**
解答　２、３

１　誤
確認試験は、医薬品又は医薬品中に含有されている主成分などを、その特性に基づいて確認するための試験である。なお、試料中に含まれる不純物の限度あるいは量を調べる試験法は純度試験である。

２　正
確認試験(1)は、ステロイドが硫酸により呈色することを利用したステロイドの確認反応である。なお、本反応はコーベル反応と呼ばれ、確認するステロイドの種類によって、色調や色の変化が異なる。

３　正
アに入るのは、「シリカゲル」である。薄層クロマトグラフィーにおける固定相は、一般的にシリカゲルを用いることが多い。

４　誤
試料溶液から得た主スポット以外のスポットとは、ヒドロコルチゾンコハク酸エステルに含まれる不純物（類縁ステロイドなど）由来のスポットであり、標準溶液から得たスポットとは、混在が許される限度の濃度に調製された不純物としてのヒドロコルチゾン由来のスポットである。

スポットされた3 µLに含まれる主成分（ヒドロコルチゾンコハク酸エステル）の量と不純物（ヒドロコルチゾン）の量は以下の通りである。

・主成分（ヒドロコルチゾンコハク酸エステル）の量
（25 mg/10 mL）×3 µL＝7.5 µg
・不純物（ヒドロコルチゾン）の量
　[｛（25 mg/10 mL）×1 mL｝/50 mL]×3 µL＝0.15 µg

したがって、不純物としての類縁ステロイドの混在が許される限度は、主成分7.5 µgに対して不純物0.15 µgとなるため、以下のように計算でき、2％となる。

（0.15 µg/7.5 µg）×100＝2％

５　誤
イに入る数値は、254 nmである。なお、波長が450 nmの電磁波は、紫外線ではなく可視光線である。
