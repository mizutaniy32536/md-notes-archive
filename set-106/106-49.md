# 106-49

**問題文**
図の構造をもつ経口固形製剤が示すと考えられる、水中での水溶性薬物の放出パターンはどれか。1つ選べ。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-49 を参照

**選択肢**
（選択肢は画像を参照）

**正解:** 4

**解説**
解答　４

図中の経口固形製剤は、水溶性薬物結晶粒子を不溶性の徐放性のコーティング剤であるエチルセルロース中に分散させている水不溶性マトリックス型放出制御型製剤であると考えられる。
水不溶性マトリックス型放出制御型製剤は、マトリックス中の拡散過程が薬物放出の律速となるため、表面近くの薬物の放出は速いが、内部の薬物は表面へ拡散してから放出される。したがって、時間経過とともに薬物放出率は低下する。
