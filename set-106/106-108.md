# 106-108

**問題文**
写真A〜Eに示した生薬に関する記述のうち、正しいのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-108 を参照

**選択肢**
1. Aはキョウニンで、鎮咳作用を期待して麻黄湯に配合される。
2. Bはサンショウで、腹部を温めることや健胃作用を期待して大建中湯に配合される。
3. Cはサンシシで、利胆作用を期待して葛根湯に配合される。
4. Dはハンゲで、瀉下作用を期待して半夏瀉心湯に配合される。
5. Eはショウキョウで、鎮嘔作用や健胃作用を期待して六君子湯に配合される。

**正解:** 2

**解説**
解答　２

Aはショウキョウ、Bはサンショウ、Cはサンシシ、Dはハンゲ、Eはキョウニンである。

１　誤
ショウキョウは、ショウガ科植物ショウガの根茎を用いる生薬であり、健胃作用や鎮嘔作用を期待して六君子湯などに配合される。
 
２　正
サンショウは、ミカン科植物サンショウの成熟した果皮を用いる生薬であり、健胃作用や腹部を温めることを期待して大建中湯などに配合される。

３　誤
サンシシは、アカネ科植物クチナシの果実を用いる生薬であり、利胆作用を期待して黄連解毒湯などに配合される。

４　誤
ハンゲは、サトイモ科植物カラスビシャク塊茎を用いる生薬であり、鎮吐作用を期待して半夏瀉心湯などに配合される。

５　誤
キョウニンは、バラ科植物ホンアンズ又はアンズなどの種子を用いる生薬であり、鎮咳作用を期待して麻黄湯などに配合される。
