## ◆ 母子感染（TORCH症候群等）の感染経路と予防を問うシナリオ

260312

**問題**
母子感染に関する記述のうち、誤っているのはどれか。1つ選べ。

**選択肢**

1. 風しんウイルスは、主に経胎盤感染で胎児に感染し、先天性風しん症候群を引き起こすことがある。
​
2. 妊娠の初期に妊婦がトキソプラズマ原虫に感染した場合、経胎盤感染によって胎児に重篤な症状が引き起こされることがある。
​
3. 先天梅毒は、梅毒トレポネーマを原因菌とし、産道感染を介して新生児に伝播する感染症である。
​
4. 成人T細胞白血病の原因ウイルスであるヒトT細胞白血病ウイルス−1型（HTLV−1）に妊婦が感染している場合、出生後は人工乳を授乳する。
​
5. ヒト免疫不全ウイルス（HIV）の母子感染には、経胎盤感染、産道感染及び母乳感染がある。

**正解と理由**
- `3. 先天梅毒は、梅毒トレポネーマを原因菌とし、産道感染を介して新生児に伝播する感染症である。`
- **理由の構成**
	- **問題の整理**：母子感染の各病原体の感染経路を正確に区別する。`先天梅毒`の感染経路がポイント
	- **薬学的知識・仕様・制約**：`先天梅毒`は`経胎盤感染`（垂直感染）であり、`産道感染`ではない。梅毒トレポネーマが胎盤を通過して胎児に感染する
	- **選択理由**：「産道感染」が誤り。正しくは「経胎盤感染」

**解法の手順（思考の流れ）**
- 選択肢1：風しん→`経胎盤感染`→`先天性風しん症候群`（CRS）→ 正しい
- 選択肢2：`トキソプラズマ`→妊娠初期の`経胎盤感染`→胎児に重篤な症状 → 正しい
- 選択肢3：`先天梅毒`=「産道感染」？→ 先天梅毒は`経胎盤感染`。産道感染ではない → **誤り**
- 選択肢4：`HTLV-1`=母乳感染が主要経路→人工乳で予防 → 正しい
- 選択肢5：`HIV`=経胎盤＋産道＋母乳の3経路 → 正しい

**他の選択肢が誤りな理由（本問では「正しい選択肢」）**
- 1. 風しんウイルスは、主に経胎盤感染で胎児に感染し、先天性風しん症候群を引き起こすことがある。
	`TORCH症候群`のR=Rubella（風しん）。妊娠初期の感染で難聴・先天性心疾患・白内障のリスク

- 2. 妊娠の初期に妊婦がトキソプラズマ原虫に感染した場合、経胎盤感染によって胎児に重篤な症状が引き起こされることがある。
	`TORCH症候群`のT=Toxoplasma。生肉・猫の糞便が感染源。妊娠初期ほど胎児への影響が重篤

- 4. HTLV-1に妊婦が感染している場合、出生後は人工乳を授乳する。
	`HTLV-1`は`母乳感染`が主要経路。九州・沖縄に多い。人工乳への切り替えで予防

- 5. HIVの母子感染には、経胎盤感染、産道感染及び母乳感染がある。
	`HIV`の母子感染3経路は正しい。抗レトロウイルス療法＋帝王切開＋人工乳で母子感染率を<1%に低減可能

**キーポイント**
- `TORCH症候群`：T（トキソプラズマ）、O（Other=梅毒、水痘、パルボB19等）、R（風しん）、C（サイトメガロウイルス）、H（ヘルペス）
- **`先天梅毒`＝`経胎盤感染`**。「産道感染」ではない。これがこの問題の核心
- `HTLV-1`の感染経路：①`母乳感染`（最重要）、②性行為、③輸血

**関連知識の比較**

| 病原体 | 主な母子感染経路 |
|---|---|
| `風しんウイルス` | 経胎盤 |
| `トキソプラズマ` | 経胎盤 |
| `梅毒トレポネーマ` | **経胎盤** |
| `HTLV-1` | 母乳 |
| `HIV` | 経胎盤＋産道＋母乳 |
| `B群溶血性レンサ球菌` | 産道 |
| `クラミジア` | 産道 |
| `HBV` | 産道（主）＋経胎盤 |

（一言メモ）
- `先天梅毒`=`経胎盤`感染。「先天」の名の通り、出生前にすでに感染している
- `HTLV-1`=母乳がメイン。日本では九州・沖縄で陽性率が高い
- TORCH症候群のゴロ：「**T**oキソ、**O**その他（梅毒等）、**R**ubella、**C**MV、**H**SV」
