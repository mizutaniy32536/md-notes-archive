## ◆ 薬局製造販売医薬品と製造物責任法の適用を判断するシナリオ
260306

**問題**
- （問320-321 連問）薬局において「かぜ薬4-②」（薬局製造販売医薬品）を販売した。10歳の息子が服用後に呼吸困難・意識低下。調査の結果、一部の成分の投入量を誤って製造していたことが判明した。製造物責任に関する記述のうち、正しいのはどれか。1つ選べ。

**選択肢**

1. この医薬品は、製造物責任法における「製造物」に該当しうる。
​
2. この医薬品の成分投入量が正しければ、添付文書等に不備があっても、製造物責任法における「欠陥」には該当しない。
​
3. 購入者は、製造物責任に基づく損害賠償請求を行う場合、医薬品の「欠陥」の他に薬局の「過失」を立証する必要がある。
​
4. 医薬品については、製造物責任法における「欠陥」は、最高の科学水準で求められる安全性を欠いていることをいう。

**正解と理由**
- `1. この医薬品は、製造物責任法における「製造物」に該当しうる。`
- **理由の構成**
	- **問題の整理**：薬局で販売された`薬局製造販売医薬品`（かぜ薬）の成分投入量誤りにより健康被害が発生した事例において、`製造物責任法`（PL法）に関する正しい記述を問う
	- **薬学的知識・仕様・制約**：`製造物責任法`（PL法）における「製造物」とは、製造又は加工された動産をいう（第2条第1項）。`薬局製造販売医薬品`は薬局が製造・販売する医薬品であり、製造又は加工された動産に該当するため`PL法`の適用対象となる。一方、薬局での`調剤行為`（処方箋に基づく調剤）や`院内製剤`は個別の役務提供に近く、PL法の「製造物」には該当しないとされている
	- **選択理由**：`薬局製造販売医薬品`は一般用医薬品と同様に製造・加工された動産であり、`PL法`の「製造物」に該当しうるため、選択肢1が正しい

**解法の手順（思考の流れ）**
- まず「製造物責任法（PL法）」の基本要件である「製造物」「欠陥」「無過失責任」の3つのキーワードを想起する
- `PL法`の「製造物」の定義が「製造又は加工された動産」であることを確認し、`薬局製造販売医薬品`がこれに該当するかを検討する
- `薬局製造販売医薬品`は調剤とは異なり、薬局が自ら製造して販売する医薬品であるため「製造又は加工された動産」に該当すると判断する
- 選択肢2〜4について、`PL法`の「欠陥」の定義や立証責任の仕組みと照合し、それぞれ誤りであると判断する
- これにより`選択肢1`のみが正しいと結論づけた

**他の選択肢が誤りな理由**
- 2. この医薬品の成分投入量が正しければ、添付文書等に不備があっても、製造物責任法における「欠陥」には該当しない。
	`PL法`における「欠陥」には、`製造上の欠陥`（成分投入量の誤り等）だけでなく、`設計上の欠陥`や`表示・警告上の欠陥`（添付文書の不備等）も含まれる。したがって成分投入量が正しくても、添付文書等に不備があれば`PL法`上の「欠陥」に該当しうる

- 3. 購入者は、製造物責任に基づく損害賠償請求を行う場合、医薬品の「欠陥」の他に薬局の「過失」を立証する必要がある。
	`PL法`は`無過失責任`の原則に基づいており、被害者は製造物の「欠陥」の存在と、欠陥と損害との間の`因果関係`を立証すれば足りる。製造者の「過失」を立証する必要はなく、これが民法上の`不法行為`（過失責任）との大きな違いである

- 4. 医薬品については、製造物責任法における「欠陥」は、最高の科学水準で求められる安全性を欠いていることをいう。
	`PL法`第2条第2項における「欠陥」とは、当該製造物が`通常有すべき安全性`を欠いていることをいう。「最高の科学水準」で求められる安全性ではなく、製造物の特性・通常予見される使用形態・製造業者が引き渡した時期等の事情を考慮して判断される

**キーポイント**
- `PL法`の「製造物」は「製造又は加工された動産」と定義されており、`薬局製造販売医薬品`はこれに該当するが、`調剤行為`や`院内製剤`は該当しない
- `PL法`の「欠陥」は3類型（`製造上の欠陥`・`設計上の欠陥`・`表示・警告上の欠陥`）に分類され、添付文書の不備も欠陥に含まれる
- `PL法`は`無過失責任`であり、被害者は「欠陥」と「因果関係」の立証で足り、製造者の「過失」の立証は不要である
- `PL法`の安全性基準は「`通常有すべき安全性`」であり、「最高の科学水準」ではない

**関連知識の比較**
- `PL法`（`無過失責任`）と`民法709条`の`不法行為`（`過失責任`）は立証責任の構造が異なり、PL法では被害者の立証負担が軽減されている
- `PL法`の免責事由として`開発危険の抗弁`（製造物を引き渡した時の科学技術の知見では欠陥を認識できなかった場合）が認められている（第4条第1号）
- `薬局製造販売医薬品`は`都道府県知事`の許可を受けた薬局で製造販売される医薬品であり、`薬局製造販売業`の許可が必要となる
- 医薬品による健康被害の救済制度としては`医薬品副作用被害救済制度`（PMDA）があるが、これは適正使用下の副作用が対象であり、製造上の欠陥による被害とは制度の趣旨が異なる

（一言メモ）
- `PL法`の適用対象は「製造又は加工された動産」であり、`薬局製造販売医薬品`は該当するが`調剤`・`院内製剤`・不動産・未加工の農産物は対象外である
- `PL法`の「欠陥」は`通常有すべき安全性`の欠如であり、「最高水準」ではない点が頻出の引っかけポイントである
- `PL法`は`無過失責任`を採用しているため、被害者は製造者の「過失」ではなく「欠陥」と「因果関係」を立証すればよい
