# 106-138

**問題文**
地球環境問題とそれに対する国際的な取組の組合せとして、正しいのはどれか。2つ選べ。　地球環境問題　　　　　　国際的な取組

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-138 を参照

**選択肢**
1. オゾン層の破壊　・・・　ロンドン条約
2. 地球温暖化　　　・・・　パリ協定
3. 水銀による汚染　・・・　水俣条約
4. 海洋汚染　　　　・・・　京都議定書
5. 酸性雨　　　　　・・・　ストックホルム条約

**正解:** 2,3

**解説**
解答　２、３

１　誤
オゾン層の破壊に対する国際的な取組として、モントリオール議定書がある。

２　正
地球温暖化に対する国際的な取組として、パリ協定や京都議定書がある。

３　正
水銀による汚染に対する国際的な取組として、水俣条約がある。 

４　誤
海洋汚染に対する国際的な取組として、ロンドン条約がある。 

５　誤
酸性雨に対する国際的な取組として、東アジア酸性雨モニタリングネットワークや長距離越境大気汚染条約（ジュネーブ条約）がある。なおストックホルム条約は、残留性有機汚染物質（POPs：Persistent Organic Pollutants）に対する国際的な取組である。
