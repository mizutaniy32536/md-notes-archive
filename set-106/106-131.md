## ◆ 化学物質と有害作用の対応関係を判断するシナリオ
260305

**問題**
化学物質の主な有害作用に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**

1. 塩化ビニルモノマーは、膀胱がんを引き起こす
​
2. フェニトロチオンは、近位尿細管細胞に蓄積して腎障害を引き起こす
​
3. アスベストは、肺がんや悪性中皮腫の原因となる
​
4. ベンジジンは、造血機能障害を引き起こす
​
5. シアン化水素は、シトクロムオキシダーゼを阻害して細胞呼吸を停止させる

**正解と理由**
- `3` と `5`
- **理由の構成**
	- **問題の整理**：5つの化学物質とその有害作用の対応関係から正しいものを2つ選ぶ
	- **薬学的知識・仕様・制約**：
		- `アスベスト`（石綿）は吸入により肺がんおよび悪性中皮腫を引き起こす発がん物質として確立されている
		- `シアン化水素` はミトコンドリア電子伝達系の `シトクロムオキシダーゼ`（複合体IV）を阻害し、細胞呼吸を停止させる
		- 塩化ビニルモノマーの標的臓器は肝臓（肝血管肉腫）であり膀胱ではない
		- フェニトロチオンは有機リン系殺虫剤でありChE阻害が主作用、腎障害ではない
		- ベンジジンは膀胱がんの原因物質であり、造血機能障害はベンゼンの作用
	- **選択理由**：選択肢3はアスベストと肺がん・悪性中皮腫の正しい対応、選択肢5はシアン化水素とシトクロムオキシダーゼ阻害の正しい対応であるため、この2つが正しい

**解法の手順（思考の流れ）**
- まず各選択肢の「化学物質名」と「有害作用」の対応が正しいかを一つずつ照合する
- 選択肢1：`塩化ビニルモノマー` → 膀胱がんではなく `肝血管肉腫` が正しい対応と想起される
- 選択肢2：`フェニトロチオン` → 有機リン系 → `ChE阻害` が主作用であり腎障害ではないと判断
- 選択肢3：`アスベスト` → 肺がん・悪性中皮腫 → 教科書的な正しい対応と判断
- 選択肢4：`ベンジジン` → 造血機能障害はベンゼンの作用であり、ベンジジンは膀胱がんと想起される → 入れ替えトラップと判断
- 選択肢5：`シアン化水素` → シトクロムオキシダーゼ阻害 → 正しい対応と判断
- 以上から選択肢3と5が正しいと結論づけた

**他の選択肢が誤りな理由**
- 1. 塩化ビニルモノマーは、膀胱がんを引き起こす
	`塩化ビニルモノマー` が引き起こすのは `肝血管肉腫`（肝臓の血管内皮由来の悪性腫瘍）であり、膀胱がんではない
	膀胱がんの原因物質は `ベンジジン` やβ-ナフチルアミンなどの芳香族アミン類

- 2. フェニトロチオンは、近位尿細管細胞に蓄積して腎障害を引き起こす
	`フェニトロチオン`（スミチオン）は有機リン系殺虫剤であり、主作用は `コリンエステラーゼ`（ChE）の阻害
	近位尿細管に蓄積して腎障害を引き起こすのは `カドミウム` の特徴

- 4. ベンジジンは、造血機能障害を引き起こす
	`ベンジジン` は芳香族アミンの一種で、`膀胱がん` の原因物質
	造血機能障害（再生不良性貧血・白血病）を引き起こすのは `ベンゼン` であり、名前が似ているため入れ替えトラップになりやすい

**キーポイント**
- 化学物質と標的臓器・疾患の正確な対応を覚える必要がある（入れ替え問題が頻出）
- `ベンジジン`＝膀胱がん、`ベンゼン`＝造血機能障害 → 名前が似ているため混同注意
- `塩化ビニルモノマー`＝`肝血管肉腫` → 膀胱がんとの入れ替えに注意
- `シアン化水素` は電子伝達系 `複合体IV`（シトクロムオキシダーゼ）を阻害して内窒息を引き起こす
- `カドミウム` は近位尿細管障害 → フェニトロチオンとの入れ替えに注意

**関連知識の比較**
- 発がん物質と標的臓器の主な対応：
	`塩化ビニルモノマー` → 肝血管肉腫
	`ベンジジン`・β-ナフチルアミン → 膀胱がん
	`ベンゼン` → 造血機能障害（白血病）
	`アスベスト` → 肺がん・悪性中皮腫
	`ヒ素` → 皮膚がん・肺がん・ボーエン病
	`クロム（VI）` → 肺がん・鼻中隔穿孔
- 電子伝達系の阻害物質：
	`シアン化水素`・硫化水素 → 複合体IV阻害
	`一酸化炭素` → ヘモグロビンのCO結合（酸素運搬阻害）
	`ロテノン` → 複合体I阻害
- 有機リン系の中毒機序：`フェニトロチオン`・パラチオン → ChE阻害 → アセチルコリン蓄積 → ムスカリン様症状・ニコチン様症状

（一言メモ）
- `ベンジジン`＝膀胱がん、`ベンゼン`＝造血障害 → 「ジン」がつく方が「じんぞう」ではなく「ぼうこう」と覚える
- `シアン化水素` は `複合体IV` 阻害で「内窒息」、一酸化炭素はHb結合で「外窒息」→ 窒息の機序が異なる
- 化学物質の有害作用は「物質名と標的臓器の入れ替え」が定番の出題パターンなので、正確な対応表を頭に入れておく
