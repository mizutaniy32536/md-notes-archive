## ◆ 胃及び周辺の模式図から幽門を選ぶシナリオ
260312

**問題**
ヒトの胃及びその周辺の模式図において、幽門はどれか。1つ選べ。

**選択肢**

1. 噴門
​
2. 胃底部
​
3. 胃体部
​
4. 幽門
​
5. 十二指腸

⚠️ 図が必要な問題です。実際の問題では胃の模式図が示されます。

**正解と理由**
- `4. 幽門`
- **理由の構成**
	- 幽門 = 胃と十二指腸の`境界部`

**解法の手順（思考の流れ）**
- 選択肢1：噴門 = 食道と胃の境界（入口側）→ ✗
- 選択肢2：胃底部 = 胃の上部（噴門より上の膨らみ）→ ✗
- 選択肢3：胃体部 = 胃の中央部分 → ✗
- 選択肢4：幽門 = 胃と十二指腸の境界（出口側）→ **○**
- 選択肢5：十二指腸 = 小腸の最初の部分 → ✗

**キーポイント**
- 胃の構造：`噴門`（入口）→ `胃底部` → `胃体部` → `幽門`（出口）→ 十二指腸
- 「噴門」と「幽門」を混同しない！
  - 噴門 = 食道側（入口）
  - 幽門 = 十二指腸側（出口）

（一言メモ）
- 「幽門」= 出口。「幽」は「奥深い」→ 食べ物が奥（腸）へ出ていく門
