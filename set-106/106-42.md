## ◆ 肝初回通過効果を回避できる剤形を選ぶシナリオ

260310

**問題**
- 以下の剤形のうち、薬物の肝初回通過効果を回避するのに最も適しているのはどれか。1つ選べ。

**選択肢**

1. 経口徐放錠
​
2. 口腔内崩壊錠
​
3. 腸溶錠
​
4. 経口ゼリー剤
​
5. 坐剤

#####################################

**正解と理由**
- `5. 坐剤`
- **理由の構成**
	- `坐剤`は直腸に投与される。`直腸下部`から吸収された薬物は`下直腸静脈`→`内腸骨静脈`→`下大静脈`経由で全身循環に入り、`門脈`を経由しないため`肝初回通過効果`を回避できる

**他の選択肢が誤りな理由**
- 1〜4はすべて経口投与 → 消化管吸収 → 門脈 → 肝臓 → 初回通過効果を受ける。口腔内崩壊錠も大部分は嚥下後に消化管で吸収

**キーポイント**
- 初回通過効果を回避できる投与経路: `坐剤`（直腸下部）、`舌下投与`、`経皮投与`、`吸入`、`注射`
- 直腸上部からの吸収は上直腸静脈→門脈→肝臓のため初回通過効果を`受ける`

（一言メモ）
- 「初回通過効果の回避」＝門脈を通らない経路。坐剤は直腸「下部」がポイント
