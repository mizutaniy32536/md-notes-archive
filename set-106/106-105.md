# 106-105

**問題文**
次の化合物のうち、カルボキシ基のバイオアイソスター（生物学的等価体）を含むのはどれか。2つ選べ。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-105 を参照

**選択肢**
（選択肢は画像を参照）

**正解:** 3,5

**解説**
解答　３、５

医薬品の設計において、構造式の一部を他の原子団などに置き換えることによって薬効の増強・持続化、副作用の軽減などを図ることがある。注目した部分構造と同様の生物活性をもち、置き換えることが可能な原子団を、その部分構造のバイオアイソスター（生物学的等価体）という。以下に代表的なカルボキシ基（−COOH）のバイオアイソスターの構造を記載する。

【カルボキシ基のバイオアイソスター】

１　誤
イマチニブの構造であり、カルボキシ基のバイオアイソスターを含まない。

２　誤
フルジアゼパムの構造であり、カルボキシ基のバイオアイソスターを含まない。

３　正
カンデサルタン シレキセチルの構造であり、カルボキシ基のバイオアイソスターであるテトラゾールを含む。

４　誤
サキナビルの構造であり、カルボキシ基のバイオアイソスターを含まない。

５　正
オキサメタシンの構造であり、カルボキシ基のバイオアイソスターであるヒドロキサム酸を含む。
