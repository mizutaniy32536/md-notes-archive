# 第106回 問29

**問題文**
中枢神経でオピオイドμ受容体を遮断して、モルヒネが引き起こす呼吸抑制を改善するのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-29 を参照

**選択肢**
1. ドキサプラム
2. ナルデメジン
3. ナロキソン
4. ナルフラフィン
5. フルマゼニル

**正解:** 3

**解説**
解答　３

１　誤
ドキサプラムは、頸動脈小体及び大動脈小体の化学受容器を刺激して呼吸興奮を起こす呼吸興奮薬であり、麻酔時や中枢神経抑制剤による中毒時などの呼吸抑制ならびに覚醒遅延に用いられる。

２　誤
ナルデメジンは、末梢性オピオイドμ受容体拮抗薬であり、オピオイド誘発性便秘症に用いられる。

３　正
ナロキソンは、中枢神経でオピオイドμ受容体を遮断して、モルヒネが引き起こす呼吸抑制を改善する。

４　誤
ナルフラフィンは、選択的オピオイドκ受容体作動薬であり、血液透析患者におけるそう痒症の改善に用いられる。

５　誤
フルマゼニルは、ベンゾジアゼピン受容体拮抗薬であり、ベンゾジアゼピン系薬剤による鎮静の解除及び呼吸抑制の改善に用いられる。
