# 106-173

**問題文**
体内動態が線形性を示す薬物Aは、肝代謝と腎排泄によって体内から消失し、正常時における腎クリアランスは全身クリアランスの60％である。また、腎疾患時に薬物Aの肝クリアランスは変化しないが、腎クリアランスは糸球体ろ過速度（GFR）に比例して変化する。薬物Aを投与中の患者において、GFRが正常時の50%に低下したとする。薬物Aの血中濃度時間曲線下面積（AUC）を腎機能正常時と同じにするには、投与量を腎機能正常時の何％に変更すればよいか。1つ選べ。

**選択肢**
1. 20
2. 40

4. 140

**正解:** 3

**解説**
解答　３

設問文に「体内動態が線形性を示す薬物Aは、肝代謝と腎排泄によって体内から消失し、正常時における腎クリアランスは全身クリアランスの60％である」とあることから薬物Aの正常時における腎クリアランスは全身クリアランスの60％、肝クリアランスは全身クリアランスの40％であるといえる。
　CL腎（正常）＝0.6 CLtot（正常）
　CL肝（正常）＝0.4 CLtot（正常）
　CLtot（正常）：腎機能正常時の全身クリアランス
　CL肝（正常）：腎機能正常時の肝クリアランス
　CL腎（正常）：腎機能正常時の腎クリアランス

また、「腎疾患時に薬物Aの肝クリアランスは変化しないが、腎クリアランスは糸球体ろ過速度（GFR）に比例して変化する。」とあることから、GFRが正常時の50％に低下した場合の患者の全身クリアランスは、以下のように求めることができる。
　CL肝（腎疾患）＝CL肝（正常）＝0.4 CLtot（正常）
　CL腎（腎疾患）＝0.6 CLtot（正常）×0.5＝0.3 CLtot（正常）
　CLtot（腎疾患）＝CL肝（腎疾患）＋CL腎（腎疾患）＝0.4 CLtot（正常）＋0.3 CLtot（正常）＝0.7 CLtot（正常）
　CLtot（腎疾患）：腎疾患時の全身クリアランス
　CL肝（腎疾患）：腎疾患時の肝クリアランス
