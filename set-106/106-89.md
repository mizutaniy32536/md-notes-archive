## ◆ 動脈硬化を伴う脂質異常症患者における不適切な生活指導の判断
260305

**問題**
動脈硬化をきたしている脂質異常症患者において、実施すべきでない食事・生活習慣はどれか。1つ選べ。

**選択肢**

1. 禁煙
​
2. n-3系多価不飽和脂肪酸の摂取制限
​
3. コレステロールの摂取制限
​
4. 塩分の摂取制限
​
5. 飲酒制限

**正解と理由**
- `2. n-3系多価不飽和脂肪酸の摂取制限`
- **理由の構成**
	- **問題の整理**：動脈硬化を伴う脂質異常症患者に対して、実施すべきでない食事・生活習慣を1つ選ぶ問題（否定形の問い）
	- **薬学的知識・仕様・制約**：`n-3系多価不飽和脂肪酸`（DHA・EPA）は`抗血栓作用`、`中性脂肪低下作用`、抗炎症作用を有し、動脈硬化の予防・改善に寄与する EPA製剤（`イコサペント酸エチル`）は高脂血症治療薬として臨床使用されている
	- **選択理由**：n-3系多価不飽和脂肪酸は制限ではなく積極的に摂取すべき栄養素であり、その摂取を制限することは不適切な指導に該当する

**解法の手順（思考の流れ）**
- まず「実施すべきでない」という否定形の問いであることを確認する
- 各選択肢について、動脈硬化を伴う脂質異常症患者に対して実施すべきかどうかを判断する
- 禁煙、コレステロール制限、塩分制限、飲酒制限はいずれも動脈硬化予防の基本的な生活指導として適切と判断
- `n-3系多価不飽和脂肪酸`は抗動脈硬化作用を有し、摂取を制限するのは逆効果と想起
- よって「n-3系多価不飽和脂肪酸の摂取制限」が実施すべきでない項目であると結論づけた

**他の選択肢が誤りな理由**
- 1. 禁煙
	喫煙は`血管内皮機能`を障害し、`LDL`の酸化を促進して動脈硬化を進行させる 禁煙は実施すべき指導であるため「実施すべきでない」には該当しない

- 3. コレステロールの摂取制限
	食事性コレステロールの過剰摂取は`LDLコレステロール`の上昇を招き、動脈硬化を促進する 摂取制限は実施すべき指導である

- 4. 塩分の摂取制限
	過剰な塩分摂取は`高血圧`を引き起こし、高血圧は動脈硬化の重要な危険因子である 塩分制限は実施すべき指導である

- 5. 飲酒制限
	過剰飲酒は`中性脂肪`（トリグリセリド）の上昇を招き、脂質異常症を悪化させる 飲酒制限は実施すべき指導である

**キーポイント**
- `n-3系多価不飽和脂肪酸`（DHA・EPA）は動脈硬化予防に有益であり、摂取制限は不適切
- EPA製剤（`イコサペント酸エチル`）は高トリグリセリド血症の治療薬として使用される
- 動脈硬化の`危険因子`：喫煙、高LDLコレステロール、高血圧、糖尿病、肥満、過剰飲酒
- `n-3系`は血小板凝集抑制・中性脂肪低下・抗炎症作用の3つの機序で抗動脈硬化作用を発揮する

**関連知識の比較**
- 脂肪酸の分類と動脈硬化への影響
	- `n-3系多価不飽和脂肪酸`（EPA、DHA、α-リノレン酸）：抗動脈硬化作用 摂取推奨
	- `n-6系多価不飽和脂肪酸`（リノール酸、アラキドン酸）：過剰摂取は炎症を促進する可能性がある 適量摂取が推奨
	- `飽和脂肪酸`（パルミチン酸、ステアリン酸）：LDLコレステロールを上昇させ、動脈硬化を促進 摂取制限が推奨
	- `トランス脂肪酸`：LDL上昇＋HDL低下の二重作用で動脈硬化リスクを高める 摂取回避が推奨
- `EPA/AA比`：EPAとアラキドン酸の血中比率は心血管イベントリスクの指標として注目されている
- DHA・EPAの供給源：青魚（サバ、イワシ、サンマ等）が代表的 `α-リノレン酸`（亜麻仁油、えごま油）は体内でEPA・DHAに変換される

（一言メモ）
- `n-3系多価不飽和脂肪酸`は「制限」ではなく「積極摂取」が原則 動脈硬化・脂質異常症の両方に有益
- `飽和脂肪酸`はLDLを上げ、`n-3系不飽和脂肪酸`は中性脂肪を下げる 脂肪酸の種類による作用の違いを整理しておくこと
- `EPA製剤`（イコサペント酸エチル）は医薬品としても使用されており、食事指導と薬物療法の両面からn-3系の重要性を理解すること
