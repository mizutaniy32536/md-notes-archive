# 106-59

**問題文**
パーキンソン様症状はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-59 を参照

**選択肢**
1. 口渇
2. 立ちくらみ
3. 小刻み歩行
4. 体重増加
5. 勃起障害

**正解:** 3

**解説**
解答　３

パーキンソン様症状は錐体外路症状を主症状とし、安静時振戦（手足が震える）、無動・寡動（動けない・動きが遅い）、筋固縮（筋肉がこわばる）、姿勢反射障害（前屈みになり転びやすい）、歩行障害（小刻み歩行・すくみ足）などの症状が特徴的である。
