## ◆ Michaelis-Menten式のグラフからKmとVmaxを読み取るシナリオ
260312

**問題**
Michaelis-Mentenの速度論に従う酵素について、至適温度における基質濃度[S]と反応初速度vの関係、及び、この酵素のミカエリス定数Kmと最大反応速度Vmaxの値を示す図が与えられた。正しいものを選べ。1つ選べ。

**選択肢**

1. Kmが小さく、Vmaxが大きい
​
2. Kmが小さく、Vmaxが小さい
​
3. Kmが大きく、Vmaxが大きい
​
4. Kmが大きく、Vmaxが小さい
​
5. KmとVmaxの値が判断できない

⚠️ グラフが必要な問題です。実際の問題ではMichaelis-Menten曲線が示されます。

**正解と理由**
- `1. Kmが小さく、Vmaxが大きい`（グラフによる。予備校解説を要確認）

**解法の手順（思考の流れ）**
- Km = v = Vmax/2 の時の基質濃度[S]
  - Km小 → 低い[S]でVmax/2に達する → 酵素の基質親和性が`高い`
  - Km大 → 高い[S]でやっとVmax/2に達する → 親和性が`低い`
- Vmax = 反応初速度の上限（漸近値）
  - グラフの上限が高い = Vmax大
  - グラフの上限が低い = Vmax小
- グラフの曲線が左側で急上昇 → Km小
- グラフの漸近値が高い → Vmax大

**キーポイント**
- `v = Vmax × [S] / (Km + [S])`
- Km = `Vmax/2` の時の[S]
- Km小 = 基質親和性高い（酵素が基質をがっちり掴む）
- Lineweaver-Burk plot: 1/v vs 1/[S]の直線グラフでもKm, Vmaxを求められる

（一言メモ）
- Kmは「半分の速度になる基質濃度」。小さいほど酵素が優秀
- グラフ問題は図が見えないと正確に答えられないので、予備校解説を確認推奨
