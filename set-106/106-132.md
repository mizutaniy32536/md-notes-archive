## ◆ 覚醒剤に関する記述の正誤を判断するシナリオ
260305

**問題**
覚醒剤に関する記述のうち、誤っているのはどれか。1つ選べ。

**選択肢**

1. メタンフェタミンは、アンフェタミンのアミノ基がメチル化されたものである
​
2. 摂取を中断しても、摂取時と同様の精神症状が現れるフラッシュバックがみられることがある
​
3. 摂取を中断すると、身体依存性が強く現れる
​
4. 中枢神経を興奮させ、発汗、悪心、けいれんなどの症状が現れる
​
5. 摂取の確認には、尿や毛髪中の未変化体を調べる

**正解と理由**
- `3`（誤りの選択肢）
- **理由の構成**
	- **問題の整理**：覚醒剤に関する5つの記述から誤っているものを1つ選ぶ（誤り選択問題）
	- **薬学的知識・仕様・制約**：
		- 覚醒剤（`メタンフェタミン`・アンフェタミン）は `精神依存` が非常に強いが、`身体依存` は弱い
		- 身体依存が強い薬物は `モルヒネ`（オピオイド系）、`アルコール`、`バルビツール酸系` などが代表的
		- 覚醒剤の中断時に現れるのは抑うつ・疲労感・過眠などの精神症状であり、激しい離脱症候群（身体依存の指標）は生じにくい
	- **選択理由**：選択肢3は「身体依存性が強く現れる」としているが、覚醒剤は精神依存が主体で身体依存は弱いため、この記述が誤り

**解法の手順（思考の流れ）**
- 誤り選択問題であることを確認し、各選択肢を順に検証する
- 選択肢1：`メタンフェタミン` は `アンフェタミン` のN-メチル化体 → 構造的に正しい
- 選択肢2：`フラッシュバック` は覚醒剤やLSDで認められる現象 → 正しい記述と判断
- 選択肢3：覚醒剤の依存性 → `精神依存` が強く `身体依存` は弱い → 「身体依存性が強く現れる」は誤りと判断
- 選択肢4：覚醒剤は中枢神経興奮薬であり、交感神経刺激作用による発汗・悪心・けいれん → 正しい
- 選択肢5：覚醒剤の検出 → 尿中・毛髪中の `未変化体` を検出する → 正しい
- 選択肢3のみが誤りであると結論づけた

**他の選択肢が誤りな理由**
- （本問は誤り選択問題のため、正しい選択肢の根拠を記載）

- 1. メタンフェタミンは、アンフェタミンのアミノ基がメチル化されたものである
	`メタンフェタミン` はアンフェタミンのアミノ基にメチル基が導入された構造（N-メチルアンフェタミン）であり、この記述は正しい

- 2. 摂取を中断しても、摂取時と同様の精神症状が現れるフラッシュバックがみられることがある
	`フラッシュバック` は薬物使用中断後に、再使用していないにもかかわらず幻覚・妄想などの精神症状が再燃する現象
	覚醒剤やLSDで特に認められ、この記述は正しい

- 4. 中枢神経を興奮させ、発汗、悪心、けいれんなどの症状が現れる
	覚醒剤はモノアミン（`ドパミン`・ノルアドレナリン）の遊離促進・再取り込み阻害により中枢興奮作用を示す
	交感神経刺激に伴い発汗・悪心・けいれんなどが出現し、この記述は正しい

- 5. 摂取の確認には、尿や毛髪中の未変化体を調べる
	覚醒剤は尿中や毛髪中の `未変化体`（メタンフェタミン・アンフェタミン）を検出して確認する
	毛髪は長期間の使用履歴を反映でき、この記述は正しい

**キーポイント**
- 覚醒剤は `精神依存` が極めて強いが `身体依存` は弱い → 「精神依存 vs 身体依存」の区別は頻出
- 身体依存が強い薬物：`モルヒネ`・アルコール・`バルビツール酸系` → これらと覚醒剤を対比して覚える
- `フラッシュバック` は覚醒剤・LSD・大麻で認められる現象
- 覚醒剤の検出は `未変化体` を尿・毛髪から検出する方法が用いられる

**関連知識の比較**
- 依存性の分類（精神依存 vs 身体依存）：
	精神依存が強い：`覚醒剤`・コカイン・LSD・大麻・ニコチン
	身体依存が強い：`モルヒネ`・ヘロイン・`アルコール`・バルビツール酸系・ベンゾジアゼピン系
	両方強い：モルヒネ・ヘロイン・アルコール
- 覚醒剤取締法の規制対象：`メタンフェタミン` とアンフェタミンの2物質
- 耐性と交差耐性：覚醒剤には耐性が形成されるが、逆耐性（感作）も報告されており、フラッシュバックの一因と考えられている
- 薬物検出法の比較：尿は短期間の使用検出に有用、`毛髪` は長期間（数か月）の使用履歴を反映できる

（一言メモ）
- `覚醒剤`＝精神依存が強い・身体依存は弱い、`モルヒネ`＝精神依存も身体依存も強い → 依存性の「質」の違いを押さえる
- `フラッシュバック` は「使用をやめた後に症状が再燃する」現象であり、覚醒剤・LSD・大麻が代表的
- 薬物の検出で「未変化体」を調べるのは `覚醒剤`、代謝物を調べるのは大麻（11-nor-9-carboxy-THC）→ 検出対象の違いも出題される
