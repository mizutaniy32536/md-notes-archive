# 106-332

**問題文**
63歳男性。体重60 kg。症候性の急性低ナトリウム血症（血清ナトリウム濃度108 mEq/L）のため、医師より3％高張食塩水により一定時間内に血清ナトリウム濃度を120 mEq/Lまで上昇させる必要があると判断された。3％高張食塩水の必要量（mL）に最も近い値はどれか。1つ選べ。なお、ナトリウムの必要量の計算において、以下の関係式が成り立つとする。式：Na必要量（mEq）＝（目標血清Na濃度 － 現在の血清Na濃度）× 体内水分量（L）ここで、濃度の単位はmEq/Lである。また、体内水分量は体重の60％、3％高張食塩水のナトリウム濃度は0.513 mEq/mLとする。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-332 を参照

**選択肢**
1. 400
2. 550
3. 700
4. 850
5. 1,000
6. 1,150

**正解:** 4

**解説**
解答　４

この男性の体内水分量は、60 kg×0.6＝36 kg＝36 Lとなる。（水1 kg＝1 L）

与えられた関係式より、この男性のNa必要量（mEq）を求める。
Na必要量（mEq）：（120 mEq/L －108 mEq/L）×36 L ＝ 432 mEq

よって、ナトリウム濃度が0.513 mEq/mLである3％高張食塩水の必要量（mL）は、
432 mEq÷0.513 mEq/mL≒842 mL

より、約850 mLである。
