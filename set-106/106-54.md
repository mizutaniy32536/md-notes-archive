# 第106回 問54

**問題文**
図に示す注射剤の容器の名称はどれか。1つ選べ。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-54 を参照

**選択肢**
1. アンプル
2. カートリッジ
3. ダブルバッグ
4. プレフィルドシリンジ
5. バイアル

**正解:** 4

**解説**
解答　４

イラストには薬液が充填されたシリンジが描かれているため、この容器はプレフィルドシリンジ（薬剤充填済みシリンジ）である。プレフィルドシリンジを用いる利点として、注射剤の調製時における業務の効率化、薬剤の取り違えによる誤投与の防止、異物混入・細菌汚染のリスクの軽減などが挙げられる。
なお、他の容器のイラストを下記に示す。
