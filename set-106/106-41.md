## ◆ 単純拡散による細胞膜透過の特徴を選ぶシナリオ

260310

**問題**
- 単純拡散による薬物の細胞膜透過に関する記述のうち、正しいのはどれか。1つ選べ。

**選択肢**

1. 濃度勾配に従う。
​
2. 透過速度はMichaelis-Menten式で表される。
​
3. トランスポーターを介する。
​
4. ATPの加水分解エネルギーを利用する。
​
5. タンパク質の細胞内取り込みに関与する。

**正解と理由**
- `1. 濃度勾配に従う。`
- **理由の構成**
	- `単純拡散`は`受動輸送`の一種で、濃度勾配に従って膜を直接透過する。エネルギー不要、トランスポーター不要、Fickの法則に従い飽和性なし

**他の選択肢が誤りな理由**
- 2. `Michaelis-Menten式`は担体介在輸送（飽和性あり）に適用。単純拡散にはFickの法則が適用される
- 3. 単純拡散は`トランスポーター不要`。脂質二重層を直接通過
- 4. ATP利用は`一次性能動輸送`。単純拡散はエネルギー不要
- 5. タンパク質（高分子）の取り込みは`エンドサイトーシス`（膜動輸送）

**キーポイント**
- 単純拡散の3条件: ①`濃度勾配`に従う ②エネルギー`不要` ③`飽和性なし`

（一言メモ）
- 「単純拡散」の問題は消去法でも解ける。2〜5はすべて別の輸送機構の特徴
