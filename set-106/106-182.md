# 106-182

**問題文**
単位操作と製剤機械に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-182 を参照

**選択肢**
1. ジェットミルは、ジュール・トムソン効果により粉砕時の温度の上昇を抑えられるので、熱に不安定な化合物や低融点化合物の粉砕に適している。
2. 旋回スクリュー型混合機は、本体容器が回転することにより、粉粒体の集合と分割を交互に繰り返すことで混合が進行する。
3. 流動層造粒装置は、熱風気流中に吹き上げた粉末に結合剤を噴霧するので、装置内で圧密化を受けて重質な造粒物が得られる。
4. ロータリー型打錠機は、複数組の上下杵と臼を組み込むことができるので、大量生産に適している。
5. オーガ式のカプセル充てん装置は、瞬間的に薬物溶液をゼラチンで包み込めるので、シームレスカプセルを製することができる。

**正解:** 1,4

**解説**
解答　１、４

１　正
ジェットミルは、噴出する圧縮空気に試料を巻き込み、粒子同士あるいは容器壁との衝突によって粉砕する粉砕機である。本機は、ジュールトムソン効果（冷却効果）によって熱の発生を抑えられるため、熱に不安定な医薬品の粉砕に適している。

２　誤
混合機には、容器回転型及び容器固定型があり、容器固定型には旋回スクリュー型混合機、容器回転型にはV型混合機がある。本問では本体容器が回転すると記載されているので、旋回スクリュー型混合機ではなく、V型混合機の記述である。

３　誤
流動層造粒装置は、熱風気流中に吹き上げた粉末に結合剤を噴霧するので、装置内で圧密化を受けず、軽質で不定形な造粒物が得られる。

４　正
ロータリー型打錠機は、複数組の上下杵と臼を組み込むことができるので、錠剤の大量生産に用いられる。

５　誤
滴下法（二重ノズル法）に関する記述である。滴下法（二重ノズル法）は、軟カプセルを製する方法で、瞬間的に薬物溶液をゼラチンで包み込めるので、シームレスカプセルを製することができる装置である。一方、オーガ式のカプセル充填装置は、硬カプセルの充填に使用される。
