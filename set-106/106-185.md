# 106-185

**問題文**
浮腫の病態に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-185 を参照

**選択肢**
1. ネフローゼ症候群による浮腫では、血漿膠質浸透圧が上昇している。
2. 腎炎による浮腫は、最初に顔面、眼瞼に顕著に表れる。
3. 肝性浮腫は、上肢に顕著に表れる。
4. 心性浮腫は、朝方に生じる場合が多い。
5. 粘液水腫は、指圧で圧痕を残さない。

**正解:** 2,5

**解説**
解答　２、５

浮腫とは、細胞外液のうち間質液が病的に増加した状態をいう。

１　誤
ネフローゼ症候群では、高タンパク尿に伴う低タンパク血症を呈するため、血漿膠質浸透圧が低下し、浮腫を生じる。

２　正
急性糸球体腎炎などの腎炎では、尿量減少に伴う体液貯留により浮腫を生じる。一般に本症のような腎性浮腫は、顔面や眼瞼に多くみられ、臥位が続いた朝方起床時に認められやすい。

３　誤
肝性浮腫は、肝硬変などの肝疾患により引き起こされる浮腫である。肝硬変などの肝疾患では、アルブミン合成が低下し、血漿膠質浸透圧が低下することにより下肢浮腫や腹水を生じる。

４　誤
心性浮腫は、心不全などの心疾患により引き起こされる浮腫である。心不全などの心疾患では、心機能低下に伴い、大静脈うっ血による下肢浮腫など、肺静脈うっ血による肺水腫などを認める。一般に心性浮腫は、夕方に強く認められる。

５　正
粘液水腫は、グリコサミノグリカンが皮下組織に蓄積することで生じる圧痕を残さない浮腫であり、慢性甲状腺炎などの甲状腺機能低下症において認められる。
