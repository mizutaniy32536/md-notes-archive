# 第106回 問58

**問題文**
遅発性ジスキネジアの典型的な症状はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-58 を参照

**選択肢**
1. 高熱
2. 手指関節のこわばり
3. 無意識に口をもぐもぐさせる
4. 筋肉痛
5. じっと座っていられない

**正解:** 3

**解説**
解答　３

遅発性ジスキネジアは、抗精神病薬に関連して発症する不随意運動であり、無意識に口をもぐもぐさせる、繰り返し唇をすぼめる、舌を左右に動かすなどの症状が特徴的である。
