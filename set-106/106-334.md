# 106-334

**問題文**
72歳男性。体重60 kg、身長165 cm。薬局に処方箋を持参した。患者から力が入りにくくなったとの訴えがあったため、対応した薬剤師は血液検査のデータを確認した。検査値：赤血球数350×104 /µL、Hb 13.5 g/dL、白血球数4,200 /µL、血小板数15×104 / µL、HbA1c 5.6％（NGSP値）、Scr 1.4 mg/dL、eGFR 42.6 mL/min/1.73 m2、ALT 42 IU/L、AST 30 IU/L、ALP 100 IU/L、T−Bil 0.5 mg/dL 疑義照会すべき内容として適切なのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-334 を参照

**選択肢**
1. 血清マグネシウム値の確認
2. ファモチジン塩酸塩錠の減量
3. プレガバリンカプセルの減量
4. シタグリプチンリン酸塩錠の減量
5. バルサルタン錠の減量

**正解:** 1,3

**解説**
解答　１、３

本患者は、72歳の高齢であること、eGFRが42.6 mL/min/1.73 m2と低値を示していること（基準値：90 mL/min/1.73 m2以上）、Scr が1.4 mg/dLと高値を示していること（基準値：1.2 mg/dL以下）から、腎機能が低下していると考えられる。また、併せて力が入りにくくなったとの訴えがあることから、酸化マグネシウム錠による高マグネシウム血症や、プレガバリンカプセルによる脱力・めまいが生じている可能性が高い。
よって、疑義照会すべき内容としては、血清マグネシウム値の確認やプレガバリンカプセルの減量が適切である。
