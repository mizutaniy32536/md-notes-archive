# 第106回薬剤師国家試験 問227

## 問題文

70歳男性。10年前から2型糖尿病と前立腺がんに罹患し治療を受けている。また、慢性閉塞性肺疾患（COPD）の診断を受け、治療中である。今回友人より、「あなたは70歳だけど肺炎球菌ワクチンの接種をしないのか」と聞かれ、ワクチン接種の相談に薬局を訪れた。患者はインフルエンザワクチンを接種したことはあるが、肺炎球菌ワクチンを接種した経験はなかった。

**肺炎球菌ワクチン及びこの患者のワクチン接種に関する注意点について、正しいのはどれか。1つ選べ。**[3]

## 選択肢

1. 糖尿病の治療中のため、肺炎球菌ワクチンの接種不適当者である。[3]

2. 前立腺がんの治療中のため、肺炎球菌ワクチンの接種不適当者である。[3]

3. 23価肺炎球菌ワクチンを他のワクチンと混合接種することができる。

4. 23価肺炎球菌ワクチンは常温で保存することができる。

5. 23価肺炎球菌ワクチンは8度以下で凍結を避けて保存する。[2]

## 正解番号

**5**[2]

## 解説

**選択肢1（誤り）**：糖尿病は肺炎球菌ワクチン接種の禁忌ではなく、接種不適当者には該当しません。[4]

**選択肢2（誤り）**：前立腺がんの治療中でも肺炎球菌ワクチンを接種することは可能であり、接種不適当者には該当しません。[3]

**選択肢3（誤り）**：23価ワクチンを他のワクチンと混合接種はできません。[2]

**選択肢4（誤り）**：常温保存はできません。

**選択肢5（正解）**：23価肺炎球菌ワクチンは**8度以下で凍結を避けて保存**する必要があります。加速試験では23～27度での保存期間は8ヶ月となります。[2]
