## ◆ 複数医療機関を受診する高齢患者への薬剤師の対応を判断するシナリオ
260306

**問題**
- 75歳男性。1年前から、高血圧症、脂質異常症、高尿酸血症と不眠症のため、近医を受診し、前回同様の薬剤が処方された。患者のお薬手帳には、3日前に初めて受診した別の泌尿器科から処方された薬が記載されていた。薬剤師が患者の家族に確認したところ、一昨日から、特に午前中にぼんやりしていることが多く、最近は物忘れが目立つため、認知症を発症したのではないかと心配していた。この患者に薬剤師が行った対応として、適切でないのはどれか。1つ選べ。

**選択肢**

1. 睡眠薬の副作用の可能性もあるため、今回の処方について医師に確認を行った。
​
2. 医療機関ごとに異なるお薬手帳を使用するよう指導した。
​
3. 睡眠薬などを自己調整して服薬していないか患者に確認した。
​
4. かかりつけ薬剤師の制度を説明し、かかりつけ薬剤師のいる薬局で全て調剤してもらうように勧めた。
​
5. 認知症の可能性も否定できないため、専門医へ行くように勧めた。

**正解と理由**
- `2. 医療機関ごとに異なるお薬手帳を使用するよう指導した。`
- **理由の構成**
	- **問題の整理**：75歳男性が近医（高血圧症・脂質異常症・高尿酸血症・不眠症）と泌尿器科の2つの医療機関を受診しており、家族から午前中のぼんやり・物忘れの訴えがある状況で、薬剤師の対応として「適切でないもの」を問う
	- **薬学的知識・仕様・制約**：`お薬手帳`は患者が服用しているすべての薬剤情報を一元管理するためのツールであり、`1冊にまとめて管理`することが原則である。複数の医療機関を受診する場合こそ1冊のお薬手帳で情報を集約し、`相互作用`や`重複投与`の確認を可能にする必要がある
	- **選択理由**：医療機関ごとに異なるお薬手帳を使用するよう指導することは、薬剤情報の一元管理を阻害し、`相互作用`や`重複投与`のチェックができなくなるため、薬剤師の対応として不適切である

**解法の手順（思考の流れ）**
- まず「75歳・複数医療機関受診・午前中のぼんやり・物忘れ」というキーワードから、`ポリファーマシー`と`睡眠薬の持ち越し効果`の問題が想起される
- 次に「適切でないもの」を選ぶ問題であることを確認し、各選択肢が薬剤師の対応として妥当かを検討する
- 選択肢1は`睡眠薬`の副作用（持ち越し効果）を疑っての`疑義照会`であり、適切と判断
- 選択肢2は`お薬手帳`を医療機関ごとに分けるという指導であり、薬剤情報の一元管理に反するため不適切と判断
- 選択肢3〜5はいずれも薬剤師として妥当な確認・指導・受診勧奨であり、適切と判断
- これにより選択肢2のみが「適切でない」対応と結論づけた

**他の選択肢が誤りな理由**
- 1. 睡眠薬の副作用の可能性もあるため、今回の処方について医師に確認を行った。
	高齢者の午前中のぼんやりは`睡眠薬`の`持ち越し効果`（翌朝への効果残存）として典型的な症状であり、処方内容について医師に`疑義照会`を行うことは適切な対応である

- 3. 睡眠薬などを自己調整して服薬していないか患者に確認した。
	高齢者が`睡眠薬`を自己判断で増量・変更して服用している可能性があるため、`服薬コンプライアンス`の確認は薬剤師として適切な対応である

- 4. かかりつけ薬剤師の制度を説明し、かかりつけ薬剤師のいる薬局で全て調剤してもらうように勧めた。
	`かかりつけ薬剤師制度`は患者の服薬情報を一元的に管理し、`ポリファーマシー`の防止や`相互作用`の確認を行う制度であり、複数医療機関を受診する患者への利用勧奨は適切な対応である

- 5. 認知症の可能性も否定できないため、専門医へ行くように勧めた。
	物忘れが目立つという症状から`認知症`の可能性も考慮し、`専門医`への受診勧奨を行うことは薬剤師として適切な対応である。薬の副作用だけでなく疾患の可能性も視野に入れた判断が求められる

**キーポイント**
- `お薬手帳`は医療機関や薬局が異なっても必ず`1冊にまとめる`ことが原則であり、薬剤情報の一元管理により`相互作用`や重複投与の防止が可能になる
- 高齢者における午前中のぼんやり・ふらつきは`睡眠薬`の`持ち越し効果`（hangover effect）を疑うべき典型的な症状である
- `かかりつけ薬剤師制度`は複数医療機関を受診する患者の薬学的管理を一元化し、`ポリファーマシー`対策として有効な仕組みである

**関連知識の比較**
- `お薬手帳`は2016年度の調剤報酬改定により、電子版も正式に認められ、`電子お薬手帳`でも紙と同様に1つのアプリで全医療機関の情報を集約管理できる
- `かかりつけ薬剤師指導料`の算定要件として、患者の同意取得、薬剤師の実務経験・研修履歴、在籍期間などが定められている
- `睡眠薬`の持ち越し効果は`長時間作用型`のベンゾジアゼピン系で特に起こりやすく、高齢者では`転倒リスク`の増大や認知機能低下様の症状をもたらすことがある
- `ポリファーマシー`は一般に6剤以上の多剤併用を指し、`薬物有害事象`のリスク増大、`服薬アドヒアランス`低下、残薬増加などの問題を引き起こす

（一言メモ）
- `お薬手帳`は必ず1冊にまとめ、すべての医療機関・薬局で提示することで`相互作用`チェックや重複投与防止の基盤となる
- 高齢者の「午前中のぼんやり」は`睡眠薬`の`持ち越し効果`を第一に疑い、認知症との鑑別が重要である
- `かかりつけ薬剤師`は服薬情報の一元管理・24時間対応・在宅対応を担い、複数科受診の高齢者にとって`ポリファーマシー`防止の要となる
