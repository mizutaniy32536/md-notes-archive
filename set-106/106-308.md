## ◆ 安全性速報の性質を問うシナリオ
260303

**問題**
- 病院の医薬品情報室の薬剤師が医薬品医療機器情報配信サービス（PMDAメディナビ）より医薬品の安全性速報を受信した。薬剤師は、その情報をもとに院内での必要な対応を行うとともに、直ちに当該医薬品の使用患者及び使用状況を把握するため電子カルテより使用患者の抽出を行った。
- この安全性速報について正しいのはどれか。1つ選べ。

**選択肢**

1. 医薬品の使用上の注意の改訂情報に基づいて、製造販売業者が作成する情報である
​
2. 医薬品等の安全性に関する重要な情報であり、緊急安全性情報に準じ厚生労働省の指示等で、製造販売業者が作成する情報である
​
3. 医薬品の使用によって、健康被害が生じた場合に、医療従事者が厚生労働大臣に提出する情報である
​
4. PBRER（Periodic Benefit-Risk Evaluation Report）とも呼ばれる国際的な安全性評価情報である
​
5. 医薬品の有効性・安全性に関する情報を定期的に整理・評価し、リスク・ベネフィットバランスの観点から提出される情報である

**正解と理由**
- `2. 医薬品等の安全性に関する重要な情報であり、緊急安全性情報に準じ厚生労働省の指示等で、製造販売業者が作成する情報である`
- **理由の構成**
	- **問題の整理**：安全性速報の定義・性質を問う問題
	- **薬学的知識・仕様・制約**：`安全性速報`（ブルーレター）は `緊急安全性情報`（イエローレター）に準じた重要な安全性情報であり、厚生労働省の指示等により `製造販売業者` が作成する
	- **選択理由**：安全性速報の定義そのものが選択肢2に記載されている

**解法の手順（思考の流れ）**
- 「安全性速報」= 「ブルーレター」であることを想起
- 作成者は `製造販売業者`、指示元は `厚生労働省` であることを確認
- 緊急安全性情報（イエローレター）に「準じた」位置づけであることを確認
- 選択肢2が安全性速報の定義に完全に合致すると結論

**他の選択肢が誤りな理由**
- 1. 医薬品の使用上の注意の改訂情報に基づいて、製造販売業者が作成する情報である
	これは `DSU`（Drug Safety Update）の説明であり、安全性速報とは異なる情報提供制度

- 3. 医薬品の使用によって、健康被害が生じた場合に、医療従事者が厚生労働大臣に提出する情報である
	これは `副作用報告`（医療従事者からの安全性情報報告）の説明であり、安全性速報とは方向が逆（医療従事者→厚労省）

- 4. PBRER（Periodic Benefit-Risk Evaluation Report）とも呼ばれる国際的な安全性評価情報である
	`PBRER` は定期的ベネフィット・リスク評価報告であり、安全性速報とは別の制度

- 5. 医薬品の有効性・安全性に関する情報を定期的に整理・評価し、リスク・ベネフィットバランスの観点から提出される情報である
	これも `PBRER`/`PSUR` の説明であり、安全性速報は「定期的」ではなく「必要時」に発出される

**キーポイント**
- `安全性速報`（ブルーレター）：緊急安全性情報に準じ、厚労省の指示で製造販売業者が作成
- `緊急安全性情報`（イエローレター）：最も緊急性の高い安全性情報
- `DSU`：使用上の注意の改訂情報を提供する制度
- `PMDAメディナビ`：医薬品安全性情報の電子メール配信サービス

**関連知識の比較**
- 安全性情報の緊急度：`イエローレター` ＞ `ブルーレター` ＞ `DSU`
- `副作用報告` は医療従事者の義務（医薬品医療機器等法第68条の10第2項）であり、重篤な場合は15日以内に報告
- `RMP`（医薬品リスク管理計画）は製造販売業者が作成し、安全性監視活動・リスク最小化活動を計画的に実施するもの

（一言メモ）
- `ブルーレター` は「黄色（イエロー）ほどではないが重要」という位置づけで、色の序列で緊急度を覚える
- `DSU` は「添付文書の改訂情報のお知らせ」であり、安全性速報ほどの緊急性はない通常の情報提供
- 安全性速報は `PMDAメディナビ` で医療従事者に電子メール配信される
