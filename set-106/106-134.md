## ◆ 化審法における第一種特定化学物質を問うシナリオ
260303

**問題**
- 「化学物質の審査及び製造等の規制に関する法律（化審法）」の第一種特定化学物質はどれか。2つ選べ。

**選択肢**

1. ポリ塩化ジベンゾ-p-ジオキシン（PCDD）
​
2. ポリ塩化ジベンゾフラン（PCDF）
​
3. ポリ塩化ビフェニル（PCB）
​
4. クロロニトロフェン（CNP）
​
5. ポリ塩化ナフタレン（PCN）

**正解と理由**
- `3. ポリ塩化ビフェニル（PCB）` と `5. ポリ塩化ナフタレン（PCN）`
- **理由の構成**
	- **問題の整理**：化審法の第一種特定化学物質に該当するものを2つ選ぶ問題
	- **薬学的知識・仕様・制約**：化審法は「意図的に製造・輸入される化学物質」を対象とする。`PCB` と `PCN` は意図的に製造された物質であり、難分解性・高蓄積性・長期毒性を持つため第一種特定化学物質に指定
	- **選択理由**：PCDD・PCDFは非意図的生成物であり化審法の対象外（`ダイオキシン類対策特別措置法` で規制）。CNPは農薬であり第一種特定化学物質ではない

**解法の手順（思考の流れ）**
- まず化審法の対象が「意図的に製造・輸入される化学物質」であることを想起する
- PCDD・PCDFは燃焼過程で `非意図的` に生成される物質であり、化審法ではなくダイオキシン法の対象と判断
- PCB（カネミ油症事件の原因物質）とPCNが化審法の第一種特定化学物質であると結論づけた

**他の選択肢が誤りな理由**
- 1. ポリ塩化ジベンゾ-p-ジオキシン（PCDD）
	`PCDD` は廃棄物焼却等で非意図的に生成されるダイオキシン類であり、化審法ではなく `ダイオキシン類対策特別措置法` で規制される

- 2. ポリ塩化ジベンゾフラン（PCDF）
	`PCDF` もPCDDと同様に非意図的生成物であり、`ダイオキシン類対策特別措置法` の対象

- 4. クロロニトロフェン（CNP）
	`CNP` は除草剤（農薬）として使用されていたが、化審法の第一種特定化学物質には指定されていない。CNP自体がダイオキシン類を不純物として含む

**キーポイント**
- 化審法の対象：`意図的に製造・輸入` される化学物質
- 非意図的生成物（PCDD・PCDF等）は `ダイオキシン類対策特別措置法` で規制
- 第一種特定化学物質の3要件：`難分解性` + `高蓄積性` + `長期毒性`（人への）
- `カネミ油症事件`（1968年）が化審法制定（1973年）の契機

**関連知識の比較**
- 化審法の物質分類：第一種特定化学物質（製造・輸入禁止）、`第二種特定化学物質`（製造・輸入の制限）、`監視化学物質`（監視対象）
- `ダイオキシン類` にはPCDD（75種）・PCDF（135種）・コプラナーPCB（十数種）が含まれ、毒性は `TEF`（毒性等価係数）で2,3,7,8-TCDDを基準に評価する
- `POPs条約`（ストックホルム条約）は残留性有機汚染物質の国際的規制枠組みであり、PCB・DDT等が対象

（一言メモ）
- 化審法は「作って売る物質」が対象であり、燃焼等で `非意図的に発生` する `PCDD`・`PCDF` は対象外
- 第一種特定化学物質の代表例：`PCB`・DDT・アルドリン・ディルドリン・クロルデン・ヘキサクロロベンゼン・`PCN`
- `カネミ油症事件` では米ぬか油にPCBとPCDFが混入し、皮膚症状・肝障害・免疫異常等が発生した
