# 第106回 問35

**問題文**
ラフチジンの胃酸分泌抑制作用に関わる機序はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-35 を参照

**選択肢**
1. H＋,K＋−ATPase阻害
2. ヒスタミンH2受容体遮断
3. アセチルコリンM1受容体遮断
4. プロスタノイドEP受容体遮断
5. ガストリン遊離抑制

**正解:** 2

**解説**
解答　２

ラフチジンは、胃壁細胞のヒスタミンH2受容体を遮断することで胃酸分泌を抑制する。
