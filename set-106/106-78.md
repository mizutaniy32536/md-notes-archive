# 106-78

**問題文**
日本における社会保障制度に含まれないのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-78 を参照

**選択肢**
1. 生活保護
2. 児童福祉
3. 年金
4. 感染症予防
5. 学校教育

**正解:** 5

**解説**
解答　５

社会保障制度とは、国民の生活の安定が損なわれた場合、社会的な仕組みとして生活の保障や自立を支援する体制・制度を指す。
社会保障制度は社会保険、社会福祉、公的扶助、保健医療・公衆衛生の４つから構成される。

１　誤
生活保護は、社会保障制度の公的扶助に該当する。

２　誤
児童福祉は、社会保障制度の社会福祉に該当する。

３　誤
年金は、社会保障制度の社会保険に該当する。

４　誤
感染症予防は、社会保障制度の保健医療、公衆衛生に該当する。

５　正
