## ◆ トランスポーター介在薬物輸送の正しい記述を選ぶシナリオ

260310

**問題**
- トランスポーターを介した薬物輸送に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**

1. ペプチドトランスポーターPEPT1によるバラシクロビル輸送の駆動力は、プロトン濃度勾配である。
​
2. 有機アニオントランスポーターOAT1によるメトトレキサート輸送は、ATPの加水分解エネルギーを駆動力として直接利用する。
​
3. シクロスポリンは有機アニオントランスポーターOATP1B1を阻害するため、プラバスタチンの肝臓への移行を抑制し、血中濃度を上昇させる。
​
4. カルビドパは血液脳関門に発現する中性アミノ酸トランスポーターLAT1を介して、脳へ移行する。
​
5. シスプラチンは有機カチオントランスポーターOCT2の基質であるため、ジゴキシンの尿細管分泌を競合的に阻害する。

**正解と理由**
- `1` と `3`
- **理由の構成**
	- 1: `PEPT1`はH⁺/ペプチド共輸送体であり、`プロトン濃度勾配`が駆動力（二次性能動輸送）
	- 3: `シクロスポリン`は`OATP1B1`阻害薬。`プラバスタチン`の肝取り込みを抑制→血中濃度上昇

**他の選択肢が誤りな理由**
- 2. `OAT1`は二次性能動輸送。α-ケトグルタル酸の濃度勾配が駆動力。ATP直接利用ではない
- 4. `LAT1`を介して脳に移行するのは`レボドパ`。`カルビドパ`はBBBを通過しない（だから末梢の脱炭酸を阻害できる）
- 5. `ジゴキシン`の尿細管分泌に関わるのは`P-gp`であり`OCT2`ではない。トランスポーターの種類が異なる

**キーポイント**
- カルビドパ vs レボドパ: BBBを通過するのは`レボドパのみ`
- OATP1B1阻害: シクロスポリン→プラバスタチン血中濃度↑（横紋筋融解症のリスク）

（一言メモ）
- 「カルビドパ＝BBB通過しない」は超頻出。レボドパとセットで覚える
