# 106-343

**問題文**
35歳女性。仕事が忙しく、差し込むような胃痛が時々起こるため、近隣の薬局を訪れ薬剤師にお薬手帳を見せて相談した。薬剤師が手帳を確認したところ、半年前からフェキソフェナジン塩酸塩・塩酸プソイドエフェドリン配合錠（朝タ空腹時服用）、エソメプラゾールカプセル20 mg（夕食後服用）が処方され、女性はこれらの薬剤を指示どおりに服用していた。薬剤師は、薬局に常備している一般用医薬品の中からこの女性に提供する医薬品を選択した。この薬剤師が選択した最も適切な一般用医薬品の成分はどれか。1つ選べ。

**解説**
解答　４

本患者は、時々起こる差し込むような胃痛のため、一般用医薬品の胃腸薬を購入しようと来局している。ただし、本患者は、半年前から抗アレルギー薬であるフェキソフェナジン塩酸塩・塩酸プソイドエフェドリン配合錠と、胃酸分泌抑制薬であるエソメプラゾールカプセルを服用しているため、これらの薬剤との適応や相互作用に注意して、女性に提供する医薬品を選択する必要がある。

１　誤
本成分を含む一般用医薬品は、胃痛や胃もたれ、胸焼け、むかつきなどに用いられる。本患者は、すでにファモチジンと同効薬であるエソメプラゾールカプセルを服用しており、この2剤は、それぞれ単独使用で効果が期待できるため、併用はしてはいけないとされている。そのため、この女性に提供する医薬品としては不適切である。

２　誤
本成分を含む一般用医薬品は、総合感冒薬として風邪の諸症状に用いられる。本患者は胃腸薬を求めて来局しているため、この女性に提供する医薬品としては不適切である。

３　誤
本成分を含む一般用医薬品は、頭痛・月経痛や悪寒・発熱時の解熱などに用いられる。また、ロキソプロフェンを含む薬剤は、副作用に消化性潰瘍の可能性があり、本患者に投与すると胃痛が悪化するリスクがあるため、この女性に提供する医薬品としては不適切である。

４　正
本成分を含む一般用医薬品は、鎮痛鎮痙薬として胃痛や腹痛、さしこみ、胸焼けなどに用いられる。そのため、本患者の時々起こる差し込むような胃痛に対して提供する医薬品としては適切である。

５　誤
本成分を含む一般用医薬品は、胃もたれや食欲不振に用いられる。また、本成分に含まれる乾燥水酸化アルミニウムゲルと水酸化マグネシウムは、現在女性が服用中のフェキソフェナジン塩酸塩と併用することで、フェキソフェナジン塩酸塩を一時的に吸着し、吸収量が低下する。そのため、この女性に提供する医薬品としては不適切である。
