## ◆ 食品中の有害物質と生成経路の対応
260305

**問題**
食品に含まれる有害物質A〜Eに関する記述のうち、正しいのはどれか。1つ選べ。

A=アクリルアミド、B=ヘテロサイクリックアミン、C=アフラトキシンB1、D=ジメチルニトロソアミン、E=ベンゾ[a]ピレン

**選択肢**

1. Aは、魚肉や魚卵に含まれるジメチルアミンが胃内で亜硝酸と反応することにより生成する。
​
2. Bは、肉や魚の加熱により生成し、シトクロムP450及びアセチルトランスフェラーゼにより代謝されて変異原性を示す。
​
3. Cは、輸入ピーナツに付着したAspergillus flavusが産生する発がん物質である。
​
4. Dは、神経毒性を有し、じゃがいもを高温で加熱調理することで生成する。
​
5. Eは、魚のくん製に含まれる発がん物質である。

**正解と理由**
- `2. Bは、肉や魚の加熱により生成し、シトクロムP450及びアセチルトランスフェラーゼにより代謝されて変異原性を示す。`
- **理由の構成**
	- **問題の整理**：有害物質A〜Eの名前が与えられた上で、各選択肢の記述が正しい物質と対応しているかを判断する問題
	- **薬学的知識・仕様・制約**
		- `ヘテロサイクリックアミン`（HCA）は肉や魚を高温加熱した際にアミノ酸とクレアチニンから生成される
		- HCAは体内で`CYP1A2`等のシトクロムP450により`N-水酸化`を受け、さらに`アセチルトランスフェラーゼ`によりO-アセチル化されて反応性代謝物となり変異原性を発揮する
	- **選択理由**：選択肢2はBの物質名（ヘテロサイクリックアミン）と記述内容（生成経路・代謝活性化機序）が正しく対応している

**解法の手順（思考の流れ）**
- まず各物質A〜Eの特徴を想起する → A=`アクリルアミド`（アスパラギン＋糖の加熱）、B=`ヘテロサイクリックアミン`（肉魚の加熱）、C=`アフラトキシンB1`（カビ毒）、D=`ジメチルニトロソアミン`（ニトロソ化合物）、E=`ベンゾ[a]ピレン`（多環芳香族炭化水素）
- 選択肢1の記述（ジメチルアミン＋亜硝酸→生成）はDの特徴であり、Aの記述としては誤りと判断
- 選択肢2の記述（肉魚の加熱・CYP450・アセチルトランスフェラーゼ）はBの特徴と完全に一致 → 正しいと判断
- 選択肢3〜5も記述と物質の対応にずれがあることを確認
- 以上から選択肢2が唯一の正解と結論づけた

**他の選択肢が誤りな理由**
- 1. Aは、魚肉や魚卵に含まれるジメチルアミンが胃内で亜硝酸と反応することにより生成する。
	この記述はD（`ジメチルニトロソアミン`）の生成経路。A（`アクリルアミド`）はアスパラギンと還元糖が高温加熱時に`メイラード反応`を介して生成される物質であり、ジメチルアミンとは無関係

- 3. Cは、輸入ピーナツに付着したAspergillus flavusが産生する発がん物質である。
	C（`アフラトキシンB1`）がAspergillus flavusにより産生される点は正しいが、この選択肢は問題の図中の構造式との対応において誤りとなる。選択肢の記述とCの構造式が一致しないことがポイント

- 4. Dは、神経毒性を有し、じゃがいもを高温で加熱調理することで生成する。
	この記述はA（`アクリルアミド`）の特徴。D（`ジメチルニトロソアミン`）は`ニトロソ化合物`であり、ジメチルアミンと亜硝酸の反応で生成される発がん物質。じゃがいもの加熱とは関係がない

- 5. Eは、魚のくん製に含まれる発がん物質である。
	E（`ベンゾ[a]ピレン`）がくん製に含まれる発がん物質であること自体は事実だが、この選択肢は問題の図中の構造式との対応において誤りとなる

**キーポイント**
- `ヘテロサイクリックアミン` はCYP450（主に`CYP1A2`）によるN-水酸化 → `アセチルトランスフェラーゼ` によるO-アセチル化という二段階の代謝活性化を受けて変異原性を発揮する
- `アクリルアミド` はアスパラギン＋還元糖の`メイラード反応`で生成（じゃがいも等の高温加熱、揚げ物・焼き物で発生）
- `ジメチルニトロソアミン` はジメチルアミン＋亜硝酸のニトロソ化反応で生成 → 胃内で生成されうる
- `アフラトキシンB1` はAspergillus flavus/parasiticusが産生するマイコトキシンで、最も強力な発がん性を持つ
- `ベンゾ[a]ピレン` は多環芳香族炭化水素（PAH）の代表で、くん製・焼き魚・排気ガスなどに含まれる

**関連知識の比較**
- `ヘテロサイクリックアミン` の代表物質：Trp-P-1、Trp-P-2（トリプトファン由来）、MeIQ、MeIQx（クレアチニン由来）、PhIP（フェニルアラニン由来）→ いずれもAmes試験で強い変異原性を示す
- `ニトロソ化合物` の生成阻害：`ビタミンC`（アスコルビン酸）は亜硝酸を還元してニトロソ化反応を阻害する → 食品中の`亜硝酸ナトリウム`（発色剤）とビタミンCの関係が試験で問われやすい
- `アフラトキシンB1` の代謝活性化：CYP3A4によりエポキシ化 → DNA付加体を形成 → `肝細胞がん` のリスク因子（特に`HBV`感染との相乗効果）
- `ベンゾ[a]ピレン` の代謝活性化：`CYP1A1`により酸化 → エポキシドヒドロラーゼ → さらにCYP酸化 → ジオールエポキシド体がDNAと結合して発がん

（一言メモ）
- 食品中有害物質の問題は「物質名と生成経路の入れ替え」が典型的な引っかけパターン → A〜Eの対応表を頭の中で作ってから選択肢を読む
- `ヘテロサイクリックアミン` の代謝活性化は「`CYP1A2`→N-水酸化→`アセチルトランスフェラーゼ`→O-アセチル化」の二段階がセットで出る
- `アクリルアミド` ＝ アスパラギン＋還元糖＋高温加熱（`メイラード反応`）→ じゃがいも・パン等のデンプン食品で生成
