# 106-96

**問題文**
血中のグルコースの定量法として、次のような酵素反応を利用した方法がある。この方法では、波長340 nmの吸光度からグルコースの濃度を求める。この方法に関する記述のうち、正しいのはどれか。2つ選べ。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-96 を参照

**選択肢**
1. 試料に入射する光と透過する光の強度をそれぞれI0、Iとすると吸光度はlog(I0/I)で表される。
2. 340 nmの波長の電磁波は、分子の振動状態の変化を直接引き起こす。
3. 240 nmから400 nmまでの吸収スペクトルを測定する場合には、ガラス製のセルが用いられる。
4. 340 nmの吸収は、NADPHに由来する。
5. グルコース濃度は340 nmの吸光度の二乗に比例する。

**正解:** 1,4

**解説**
解答　１、４

本問のグルコースの定量法は、1 molのグルコースから1 molのNADPHが生成するという量的関係と、NADPHとNADP＋の吸光度の違いを利用している。
NADPHは波長340 nmに吸収極大を持つが、NADP＋は波長340 nmに吸収極大を持たないため、340 nmの吸光度を測定することにより、NADPHの濃度を求めることができる。NADPHのモル濃度は反応したグルコースのモル濃度と等しいため、グルコースの濃度を求めることができる。

１　正
吸光度は、透過度(t)の逆数の常用対数と定義される。透過度(t)は、t＝I/I0（I：透過光の強度、I0：入射光の強度）と表せるため、吸光度＝log(1/t)＝log(I0/I)で表される。

２　誤
340 nmの波長の電磁波は紫外線であり、主に分子の持つ電子状態の変化を引き起こす。なお、主に分子の振動状態の変化を引き起こす電磁波は、赤外線である。

３　誤
240 nmから400 nmの波長の電磁波は紫外線であり、吸収スペクトルの測定にガラス製のセルを用いることができないため、石英製のセルを用いる。

４　正
上記参照

５　誤
反応したグルコースのモル濃度と生成したNADPHのモル濃度が等しいことから、グルコース濃度とNADPHの濃度は比例関係にある。ランベルト－ベールの法則より、NADPHの濃度は吸光度に比例するため、グルコース濃度は340 nmの吸光度に比例する。
