# 第106回 問45

**問題文**
腸肝循環するのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-45 を参照

**選択肢**
1. アルベカシン
2. イソニアジド
3. エナラプリル
4. オセルタミビル
5. モルヒネ

**正解:** 5

**解説**
解答　５

腸管循環とは、胆汁中排泄された薬物が消化管へ移行した後、消化管より再び吸収され、門脈から肝臓に取り込まれることである。腸管循環されやすい代表的な薬物として、モルヒネ、インドメタシン、ジゴキシンプラバスタチンなどがある。

１　誤
アルベカシンは、主に腎臓から排泄される薬物である。

２　誤
イソニアジドは、主に肝臓で代謝された後、腎臓から排泄される薬物である。

３　誤
エナラプリルは、主に肝臓で代謝された後、腎臓から排泄される薬物である。

４　誤
オセルタミビルは、主に肝臓で代謝された後、腎臓及び糞中から排泄される薬物である

５　正
モルヒネは、主に肝臓で代謝された後、腎臓から排泄されるが、一部、胆汁排泄され消化管へ移行した後、消化管より再び吸収され腸肝循環する薬物である。
