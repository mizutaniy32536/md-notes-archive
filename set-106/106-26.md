# 第106回 問26

**問題文**
ある受容体の完全刺激薬である化合物Aに化学修飾を加え、受容体への親和性がより高い化合物Bを得た。化合物Aの摘出腸管標本における収縮の濃度−反応曲線がa（破線）であるとき、化合物Bの濃度−反応曲線b（実線）が正しく示されている図はどれか。1つ選べ。ただし、化学修飾により受容体への選択性や内活性には変化がなかったものとする。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-26 を参照

**選択肢**
（選択肢は画像を参照）

**正解:** 2

**解説**
解答　２

化合物Aは完全刺激薬なので、高濃度での反応率は100%となる。化合物Bも、化合物Aと受容体への選択性、内活性が同じことから完全刺激薬であり、高濃度での反応率は100%となる。また化合物Bは、化合物Aより受容体への親和性が高いので、少ない薬物濃度で受容体に反応する。よって化合物Bの濃度−反応曲線bは、化合物Aの濃度−反応曲線aより左側にシフトしたグラフとなり、該当するグラフは2となる。
