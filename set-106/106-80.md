# 第106回 問80

**問題文**
法令で以下のとおり定義されているのはどれか。1つ選べ。「患者が継続して利用するために必要な機能及び個人の主体的な健康の保持増進への取組を積極的に支援する機能を有する薬局」

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-80 を参照

**選択肢**
1. 保険薬局
2. 薬剤師会営薬局
3. 健康サポート薬局
4. 地域包括支援センター
5. ドラッグストア

**正解:** 3

**解説**
解答　３

患者が継続して利用するために必要な機能及び個人の主体的な健康の保持増進への取組を積極的に支援する機能を有する薬局に該当するのは、健康サポート薬局である。
