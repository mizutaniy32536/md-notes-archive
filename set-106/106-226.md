## ◆ 70歳COPD患者への肺炎球菌ワクチンの正しい知識を問うシナリオ

260312

**問題**
70歳男性。10年前から2型糖尿病と前立腺がんに罹患し治療を受けている。また、慢性閉塞性肺疾患（COPD）の診断を受け、治療中である。友人から肺炎球菌ワクチンの接種を勧められ薬局を訪れた。患者はインフルエンザワクチンを接種したことはあるが、肺炎球菌ワクチンは未接種。
この患者に接種が検討される肺炎球菌ワクチンに関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**

1. このワクチンは、肺炎球菌の病原性を弱毒化した生ワクチンである。
​
2. このワクチンには、血清型の異なる肺炎球菌の莢膜多糖が含まれている。
​
3. この患者には、予防接種法による集団予防を目的として肺炎球菌ワクチンが接種される。
​
4. ジフテリアの予防にもなる。
​
5. 定期接種による健康被害が生じた場合は、予防接種健康被害救済制度の対象となる。

**正解と理由**
- `2. このワクチンには、血清型の異なる肺炎球菌の莢膜多糖が含まれている。`
- `5. 定期接種による健康被害が生じた場合は、予防接種健康被害救済制度の対象となる。`
- **理由の構成**
	- **問題の整理**：高齢者向け`肺炎球菌ワクチン`の種類・分類・救済制度を問う
	- **薬学的知識・仕様・制約**：
		- `PPSV23`（23価莢膜ポリサッカライドワクチン）=23種の莢膜多糖を含む`不活化ワクチン`
		- 高齢者の肺炎球菌感染症は`B類疾病`=`個人予防`目的
		- `定期接種`で健康被害が生じた場合は`予防接種健康被害救済制度`が適用される
	- **選択理由**：2（莢膜多糖含有）と5（救済制度適用）が正しい

**解法の手順（思考の流れ）**
- 選択肢1：肺炎球菌ワクチン=`不活化ワクチン`（莢膜多糖ワクチン）。生ワクチンではない → 誤り
- 選択肢2：`PPSV23`=23種の莢膜多糖を含む → 正しい
- 選択肢3：高齢者肺炎球菌は`B類疾病`=`個人予防`目的。`集団予防`はA類 → 誤り
- 選択肢4：`PCV13`にはジフテリア毒素がキャリアタンパクとして含まれるが、ジフテリアの予防効果はない → 誤り
- 選択肢5：`定期接種`=`予防接種健康被害救済制度`の対象 → 正しい

**他の選択肢が誤りな理由**
- 1. このワクチンは、肺炎球菌の病原性を弱毒化した生ワクチンである。
	肺炎球菌ワクチンは`不活化ワクチン`（莢膜多糖または結合型ワクチン）。生菌は含まない

- 3. 集団予防を目的として接種される。
	高齢者の肺炎球菌感染症は`B類疾病`。B類=`個人予防`目的。`A類`が`集団予防`目的

- 4. ジフテリアの予防にもなる。
	`PCV13`（結合型ワクチン）にはジフテリア毒素（CRM197）がキャリアタンパクとして結合しているが、ジフテリアに対する`予防効果はない`（免疫を誘導するのは莢膜多糖部分）

**キーポイント**
- `PPSV23`（ニューモバックス）：23価莢膜ポリサッカライドワクチン。高齢者の`定期接種`（B類）
- `PCV13`（プレベナー13）：13価結合型ワクチン。CRM197（ジフテリア毒素変異体）とのタンパク結合
- A類=`集団予防`＝接種勧奨、B類=`個人予防`＝努力義務なし

**関連知識の比較**
- B類定期接種：`インフルエンザ`（65歳以上）、`高齢者肺炎球菌`
- `予防接種健康被害救済制度`：A類→ 手厚い給付、B類→ やや低い給付基準だが対象ではある
- 肺炎球菌の莢膜多糖=`T細胞非依存性抗原`。結合型ワクチン→`T細胞依存性`に変換→免疫記憶が成立

（一言メモ）
- 「PCV13にジフテリア毒素入り→ジフテリア予防できる？」→ できない。キャリアタンパクとして使ってるだけ。ひっかけ注意
- B類=個人予防、A類=集団予防。「高齢者ワクチン=B類」と覚える
- 莢膜多糖ワクチン→T非依存性→免疫記憶なし→ブースター効果が弱い。結合型→T依存性→記憶あり
