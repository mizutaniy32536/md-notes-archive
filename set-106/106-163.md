# 106-163

**問題文**
呼吸器系に作用する薬物に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-163 を参照

**選択肢**
1. オキシメテバノールは、アドレナリンβ2受容体を刺激することで、気管支平滑筋を弛緩させる。
2. カルボシステインは、ムコタンパク質中のジスルフィド（S−S）結合を開裂することで、痰の粘性を低下させる。
3. ジメモルファンは、延髄の咳中枢を抑制することで、咳反射を抑制する。
4. ブロムヘキシンは、アンブロキソールの活性代謝産物で、肺サーファクタントの分泌を促進する。
5. ジプロフィリンは、ホスホジエステラーゼを阻害してサイクリックAMP（cAMP）濃度を増加させることで、気管支平滑筋を弛緩させる。

**正解:** 3,5

**解説**
解答　３、５

１　誤
オキシメテバノールは、延髄の咳中枢を抑制することで、咳反射を抑制する麻薬性鎮咳薬である。なお、アドレナリンβ2受容体を刺激することで、気管支平滑筋を弛緩させるのは、サルメテロールなどの気管支喘息治療薬である。

２　誤
カルボシステインは、粘液の構成成分であるシアル酸、フコースのバランスを改善し、痰の粘性を低下させる去痰薬である。なお、ムコタンパク質中のジスルフィド（S−S）結合を開裂することで、痰の粘性を低下させるのは、アセチルシステインなどの去痰薬である。

３　正
ジメモルファンは、延髄の咳中枢を抑制することで、咳反射を抑制する非麻薬性鎮咳薬である。

４　誤
ブロムヘキシンは、肺サーファクタントや気道分泌を促進させる去痰薬である。なお、アンブロキソールは、ブロムヘキシンの活性代謝産物であり、アンブロキソール自体も医薬品として用いられている。

５　正
ジプロフィリンは、ホスホジエステラーゼを阻害してサイクリックAMP（cAMP）濃度を増加させることで、気管支平滑筋を弛緩させる気管支喘息治療薬である。
