## ◆ 医薬品の添付文書等に関する規定を判断するシナリオ

260305

**問題**
医薬品の添付文書等（医薬品に添付する文書又はその容器若しくは被包）に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**


1. 製造販売業者は、添付文書等記載事項について、法令で定められた方法によって公表しなければならない。

2. 添付文書等記載事項は、薬価改定にあわせて変更しなければならない。

3. 添付文書等記載事項は、医薬品に関する最新の論文その他により得られた知見に基づき、記載されていなければならない。

4. 製造販売業者は、厚生労働大臣が指定する医薬品の添付文書等記載事項の内容について、あらかじめ厚生労働大臣の許可を受けなければならない。

5. 添付文書等に承認を受けていない効能又は効果を記載するためには、承認を受けていない旨を注記しなければならない。

**正解と理由**
- `1. 製造販売業者は、添付文書等記載事項について、法令で定められた方法によって公表しなければならない。`
- `3. 添付文書等記載事項は、医薬品に関する最新の論文その他により得られた知見に基づき、記載されていなければならない。`
- **理由の構成**
	- **問題の整理**：添付文書等に関する5つの記述から、`薬機法` の規定に照らして正しいものを2つ選ぶ問題
	- **薬学的知識・仕様・制約**：
		- `薬機法第52条の2` により、製造販売業者は添付文書等記載事項を`厚生労働大臣`に届け出た上で、`PMDA`のホームページ等、法令で定められた方法で公表する義務がある
		- `薬機法第52条` により、添付文書等記載事項は医薬品に関する最新の論文その他により得られた知見に基づき記載されなければならない
		- 添付文書等記載事項に薬価の記載はなく、薬価改定に伴う変更義務は存在しない
		- 厚生労働大臣が指定する医薬品については、添付文書等記載事項を事前に`届出`する必要があるが、「許可」ではない
		- `薬機法第54条` により、承認を受けていない効能・効果の記載は注記の有無にかかわらず禁止されている
	- **選択理由**：選択肢1は公表義務の規定として正しく、選択肢3は最新知見に基づく記載義務として正しいため、この2つが正解となる

**解法の手順（思考の流れ）**
- まず添付文書等に関する`薬機法`の条文構造を想起し、各選択肢がどの条文に対応するかを整理する
- 選択肢1は`薬機法第52条の2`の公表義務に該当し、製造販売業者の法定義務として正しいと判断
- 選択肢2は薬価と添付文書の関係を問うが、添付文書に薬価の記載義務はないため誤りと判断
- 選択肢3は`薬機法第52条`の最新知見記載義務に合致するため正しいと判断
- 選択肢4は「許可」と「届出」の違いに着目し、正しくは`届出`であるため誤りと判断
- 選択肢5は`薬機法第54条`の記載禁止事項に該当し、注記すれば記載できるというのは誤りと判断
- これにより選択肢1と3が正解と結論づけた

**他の選択肢が誤りな理由**
- 2. 添付文書等記載事項は、薬価改定にあわせて変更しなければならない。
	添付文書等には具体的な`薬価`の記載はなく、薬価改定に連動した変更義務は存在しない。添付文書等の変更が必要となるのは、新たな`副作用`情報や安全性に関する知見が得られた場合など、医薬品の品質・有効性・安全性に関わる事項が変更された場合である

- 4. 製造販売業者は、厚生労働大臣が指定する医薬品の添付文書等記載事項の内容について、あらかじめ厚生労働大臣の許可を受けなければならない。
	厚生労働大臣が指定する医薬品の添付文書等記載事項については、あらかじめ厚生労働大臣への`届出`が必要であり、「許可」ではない。`届出`と`許可`は行政手続上まったく異なる概念であり、届出は一方的な通知行為、許可は行政庁の判断を伴う行為である

- 5. 添付文書等に承認を受けていない効能又は効果を記載するためには、承認を受けていない旨を注記しなければならない。
	`薬機法第54条`により、承認を受けていない効能・効果の記載は注記の有無にかかわらず`禁止`されている。未承認の効能・効果を注記付きで記載することも認められておらず、そもそも記載すること自体が違法となる

**キーポイント**
- `薬機法第52条`：添付文書等記載事項は`最新の論文`その他により得られた知見に基づき記載する義務がある
- `薬機法第52条の2`：製造販売業者は添付文書等記載事項を届出後、法令で定められた方法（`PMDA`ホームページ等）で公表しなければならない
- `薬機法第54条`：承認を受けていない効能・効果、`虚偽`・誇大な記載は添付文書等への記載が禁止されている
- 厚生労働大臣が指定する医薬品の添付文書等記載事項は事前`届出`制であり、許可制ではない

**関連知識の比較**
- `届出`と`許可`と`認可`の違い：届出は行政庁への一方的通知（受理義務あり）、許可は一般的禁止の解除（行政庁の判断を伴う）、認可は法律行為の効力を補充する行政行為
- 添付文書等の電子化：2021年8月の`薬機法改正`により、紙の添付文書の同梱が廃止され、電子的方法（`PMDA`ホームページ等）による情報提供が原則化された
- `薬機法第54条`の記載禁止事項：承認外の効能・効果のほか、保健衛生上危険がある用法・用量・使用期間の記載も禁止されている
- 添付文書等記載事項の変更時には、変更届を`PMDA`経由で厚生労働大臣に届け出る必要がある

（一言メモ）
- `薬機法第54条`は添付文書の「書いてはいけないこと」を定めた条文で、`未承認効能`の記載や`虚偽・誇大`な表現が対象となる
- 添付文書の`届出`と`公表`はセットで覚える：まず厚生労働大臣に届出→次に`PMDA`ホームページで公表、という流れ
- 「許可」と「届出」の混同は法規の頻出ひっかけパターンで、`届出`は行政庁の判断を待たず効力が生じる点が決定的な違い
