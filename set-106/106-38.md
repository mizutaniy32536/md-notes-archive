# 第106回 問38

**問題文**
眼房水の産生抑制により眼圧を下げる炭酸脱水酵素阻害薬はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-38 を参照

**選択肢**
1. ピロカルピン
2. ブナゾシン
3. ビマトプロスト
4. リパスジル
5. ドルゾラミド

**正解:** 5

**解説**
解答　５

１　誤
ピロカルピンはムスカリンM３受容体刺激薬であり、毛様体筋収縮による眼房水流出促進作用により眼圧を低下させる。

２　誤
ブナゾシンはアドレナリンα1受容体選択的遮断薬であり、ぶどう膜強膜流出路からの眼房水流出促進作用により眼圧を低下させる。

３　誤
ビマトプロストはPGF2α誘導体であり、ぶどう膜強膜流出路からの眼房水流出促進作用により眼圧を低下させる。

４　誤
リパスジルはRhoキナーゼ阻害薬であり、線維柱帯−シュレム菅からの眼房水流出促進作用により眼圧を低下させる。

５　正
ドルゾラミドは炭酸脱水酵素阻害薬であり、眼房水産生抑制作用により眼圧を低下させる。
