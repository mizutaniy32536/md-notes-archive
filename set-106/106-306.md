## ◆ バイオ後続品への切り替え時の患者説明を問うシナリオ
260303

**問題**
- 60歳女性。10年前に2型糖尿病の診断を受けた後、インスリン補充療法による薬物治療を行っている。薬剤師が面談したところ、患者は毎月の支払費用を負担に感じていたとのことであった。そこで、医師にインスリン グラルギンのバイオ後続品に切り替えることで医療費軽減になることを提案し、医師はバイオ後続品への切り替えを認めた。薬剤師は、このバイオ後続品の添付文書等を確認した上で、患者に説明することになった。なお、併用薬はない。
- このバイオ後続品に切り替えるにあたり薬剤師が説明する内容として正しいのはどれか。2つ選べ。

**選択肢**

1. 今まで使っていたお薬と比べて効果に変わりないことが確認されています
​
2. 今まで使っていたお薬と比べて副作用に変わりないことが確認されています
​
3. 今まで使っていたお薬と比べて安定性が悪いため、冷蔵庫での保管が必須です
​
4. 今まで使っていたお薬と主成分と添加剤は全く同じです
​
5. 今まで使っていたお薬と同じ製造販売業者のものです

**正解と理由**
- `1. 効果に変わりないことが確認されています` と `2. 副作用に変わりないことが確認されています`
- **理由の構成**
	- **問題の整理**：バイオ後続品への切り替え時に薬剤師が患者に説明する正しい内容を2つ選ぶ
	- **薬学的知識・仕様・制約**：`バイオ後続品` は先行バイオ医薬品と `有効性`・`安全性` の同等性/同質性が審査で確認されている。ただし「全く同じ」ではなく「同等/同質」
	- **選択理由**：効果（有効性）と副作用（安全性）の同等性は確認済みであり、患者への説明として正しい

**解法の手順（思考の流れ）**
- バイオ後続品の定義「先行品と `同等/同質`」を想起する
- 「全く同じ」と「同等/同質」の違いに注意し、選択肢4（全く同じ）は誤りと判断
- 選択肢5（同じ製造販売業者）はバイオ後続品の定義上ありえないと判断
- 選択肢3の安定性が劣るという記述には根拠がないと判断
- 有効性（選択肢1）と安全性（選択肢2）の同等性が確認されていることから正解と結論

**他の選択肢が誤りな理由**
- 3. 今まで使っていたお薬と比べて安定性が悪いため、冷蔵庫での保管が必須です
	`バイオ後続品` の安定性が先行品より劣るとは限らない。保管条件は添付文書の記載に従うべきであり、安定性が悪いという一般化は不適切

- 4. 今まで使っていたお薬と主成分と添加剤は全く同じです
	`バイオ医薬品` は生物由来で製法による微小な差異があり、「`全く同じ`」ではなく「`同等/同質`」と表現される。添加剤も異なる場合がある

- 5. 今まで使っていたお薬と同じ製造販売業者のものです
	`バイオ後続品` は先行品とは `異なる製造販売業者` が開発・製造するものであり、同じ製造販売業者という説明は誤り

**キーポイント**
- `バイオ後続品`（バイオシミラー）は先行品と「`同等/同質`」であり「全く同じ」ではない
- 品質・`有効性`・`安全性` の同等性/同質性が審査で確認される
- `ジェネリック医薬品` との違い：承認に `臨床試験` が必要
- 患者への説明ポイント：効果・安全性は同等、医療費軽減のメリット

**関連知識の比較**
- `ジェネリック医薬品`：化学合成による低分子医薬品のコピー品。BE試験で同等性を確認。臨床試験は不要
- `バイオ後続品`：生物由来の高分子医薬品の後続品。品質＋非臨床＋ `臨床試験` で同等性/同質性を確認
- `オーソライズドジェネリック`（AG）：先行品と同じ原薬・添加剤・製造方法のジェネリック。バイオ後続品とは異なる概念

（一言メモ）
- `バイオ後続品` は「同等/同質」であり「全く同じ」ではない。この違いが国試頻出ポイント
- バイオ後続品の薬価は先行品の約70%程度であり、`ジェネリック` ほどの値下げ幅はない（開発に臨床試験が必要なため）
- `インスリン グラルギン` のバイオ後続品は日本で承認されたバイオ後続品の代表例の一つ
