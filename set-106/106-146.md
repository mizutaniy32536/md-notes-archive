## ◆ 医薬品の製造販売業と製造業の許可制度に関する正誤判断

260305

**問題**
医薬品の製造販売業及び製造業に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**


1. 第一種医薬品製造販売業の許可を受ければ、処方箋医薬品の製造販売を行うことができる。

2. 製造販売業者が自社製品を製造する自社の製造所は、製造業の許可を受けているものとみなされる。

3. 製造販売業者が、自ら輸入した医薬品を薬局開設者に販売する場合には、医薬品販売業の許可を必要としない。

4. 製造業者が、自ら製造した医薬品を店舗販売業者に販売する場合には、医薬品販売業の許可を必要としない。

5. 製造業者は、製造しようとする医薬品の品目ごとに許可を受けなければならない。

**正解と理由**
- `1. 第一種医薬品製造販売業の許可を受ければ、処方箋医薬品の製造販売を行うことができる。`
- `3. 製造販売業者が、自ら輸入した医薬品を薬局開設者に販売する場合には、医薬品販売業の許可を必要としない。`
- **理由の構成**
	- **問題の整理**：医薬品の`製造販売業`と`製造業`は別個の許可制度であり、それぞれの許可の範囲・単位・行為の区別を正確に理解する必要がある
	- **薬学的知識・仕様・制約**：
		- `製造販売業`の許可は第一種と第二種に区分される
		- `第一種`は`処方箋医薬品`を含むすべての医薬品の製造販売が可能
		- `第二種`は処方箋医薬品以外の医薬品のみ製造販売が可能
		- 「製造販売」とは、製造等（輸入を含む）した医薬品を市場に出荷する行為であり、製造販売業者が自ら輸入して薬局開設者等に販売する行為は`製造販売行為`そのものに該当する
		- そのため別途`販売業`の許可は不要となる
	- **選択理由**：選択肢1は第一種製造販売業の許可範囲の正確な記述であり、選択肢3は製造販売行為の定義に照らして販売業許可が不要であることが正しい

**解法の手順（思考の流れ）**
- まず`製造販売業`の許可区分（第一種・第二種）の違いを想起し、選択肢1が第一種の許可範囲として正しいと判断
- 次に選択肢2の「みなし規定」について、製造販売業と製造業は`別個の許可`であり、自社製造所でも製造業の許可が別途必要と判断
- 選択肢3は`製造販売行為`の定義（製造等した医薬品を市場出荷する行為）に該当するため、販売業許可は不要と判断
- 選択肢4は製造業者が行う「販売」は`販売業`に該当し、別途許可が必要と判断
- 選択肢5は製造業の許可単位が「品目ごと」ではなく`製造所ごと`（区分ごと）であることから誤りと判断
- これにより選択肢1と3が正しいと結論づけた

**他の選択肢が誤りな理由**
- 2. 製造販売業者が自社製品を製造する自社の製造所は、製造業の許可を受けているものとみなされる。
	`製造販売業`の許可と`製造業`の許可はまったく別の許可制度であり、自社の製造所であっても`製造業の許可`を別途取得しなければならない
	みなし規定は存在しない

- 4. 製造業者が、自ら製造した医薬品を店舗販売業者に販売する場合には、医薬品販売業の許可を必要としない。
	`製造業`の許可はあくまで医薬品を「製造」するための許可であり、製造した医薬品を他者に販売する行為は`販売業`に該当する
	製造業者が`店舗販売業者`に販売するには、別途販売業の許可が必要

- 5. 製造業者は、製造しようとする医薬品の品目ごとに許可を受けなければならない。
	製造業の許可は`品目ごと`ではなく、`製造所ごと`（区分ごと）に与えられる
	品目ごとに必要なのは製造販売業における`製造販売承認`であり、製造業の許可とは異なる

**キーポイント**
- `製造販売業`は第一種（処方箋医薬品を含む全医薬品）と第二種（処方箋医薬品以外）に区分される
- 「製造販売」とは製造等した医薬品を`市場出荷`する行為であり、輸入品の販売も製造販売行為に含まれる
- `製造業`の許可は製造所ごと（区分ごと）であり、品目ごとの許可は不要
- 製造販売業の許可と製造業の許可は`別個の許可`であり、みなし規定はない
- 品目ごとに必要なのは`製造販売承認`（医薬品医療機器等法第14条）であり、製造業の許可とは別制度

**関連知識の比較**
- `製造販売業者`は市場出荷の最終責任を負う（GQP・GVP体制の整備が必要）
- `製造業者`は委託を受けて製造する立場であり、市場出荷の判断権限は持たない
- `卸売販売業`の許可を持てば、製造業者でも医薬品の販売が可能になる
- 外国製造業者が日本に輸出する場合は`外国製造業者の認定`が必要（許可ではなく認定）
- `製造販売承認`は品目ごとに審査を受けて取得するものであり、製造業許可（製造所単位）とは許可の単位が異なる

（一言メモ）
- `製造販売業`＝市場出荷の責任者、`製造業`＝モノを作る工場の許可、と役割を分けて覚える
- 品目ごとに必要なのは「`製造販売承認`」であり、「製造業の`許可`」は製造所単位であることを混同しない
- 製造販売業者が自ら輸入・販売する行為は`製造販売行為`に該当するため、別途販売業許可は不要
