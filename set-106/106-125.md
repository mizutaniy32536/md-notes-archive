# 106-125

### 第106回薬剤師国家試験 問125

**問題文（全文）**  
ビタミンKの一種であるビタミンK₂（メナテトレノン）に関する記述のうち、**誤っている**のはどれか。1つ選べ。[1][2]

**全選択肢**  
（注：検索結果から全選択肢の完全なリストが一部しか確認できず、特に選択肢4が誤りと明記されている。公式問題冊子[8]参照推奨だが、利用可能な情報に基づき再現）  
1. ヘキサンには極めて溶けやすいが、水にはほとんど溶けない。  
2. （詳細不明だが正しいとされる記述）  
3. （詳細不明だが正しいとされる記述）  
4. **IRスペクトルにおいて、カルボニル基（C=O）に由来する吸収は、1,650～1,800 cm⁻¹くらいのところである。**  
5. 2−メチル−1,4−ナフトキノンにイソプレン単位で構成されるプレニル基が結合している。[2]

**正解番号**  
**4**[1][2]

**解説**  
メナテトレノン（ビタミンK₂）はナフトキノン誘導体であり、**カルボニル基を持たない**構造のため、IRスペクトルにC=O吸収（通常1,650～1,800 cm⁻¹）は観察されない。選択肢4は構造的に誤り。[1][2]  
他の選択肢は正しく、例えば選択肢1は脂溶性ビタミンの溶解性、選択肢5はメナテトレノンの化学構造を正確に記述している。[1][2]
