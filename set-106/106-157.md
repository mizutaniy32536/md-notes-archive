# 106-157

# 第106回薬剤師国家試験 問157

## 問題文

40歳女性。3年前に多発性関節炎を認め外来受診したところ、関節リウマチと診断された。メトトレキサートとプレドニゾロンによる治療が開始され、徐々に増量することにより症状の改善を認めていたが、最近、関節痛が再燃した。

**再燃時に、値が上昇していると考えられる検査項目はどれか。2つ選べ。**[1]

## 全選択肢

1. CEA
2. CPK
3. KL-6
4. MMP3
5. 白血球数

## 正解番号

**4、5**[1]

## 解説

**選択肢1（CEA）** - 誤り
CEA（がん胎児性抗原）は、大腸がんなどの消化器がんや肺腺がんなど様々ながんで増加する腫瘍マーカーであり、関節リウマチの再燃では上昇しません。[1]

**選択肢2（CPK）** - 誤り
CPK（クレアチンキナーゼ）は関節リウマチの再燃時に上昇するとは考えられません。[1]

**選択肢3（KL-6）** - 誤り
KL-6（シアル化糖鎖抗原）は、間質性肺炎などで上昇する検査項目であり、関節リウマチの再燃では上昇しません。[1]

**選択肢4（MMP3）** - 正解
MMP3（マトリックスメタロプロテイナーゼ3）は、関節リウマチの再燃時に値が上昇します。これは関節リウマチの活動性を反映する検査項目です。[1]

**選択肢5（白血球数）** - 正解
関節リウマチの再燃時には、炎症反応が増強され白血球数が上昇します。[1]
