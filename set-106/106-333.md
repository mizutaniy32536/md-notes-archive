# 106-333

**問題文**
フェンタニルクエン酸塩注射液を投与開始から10分後、担当医が別の患者に誤って投与されていることに気づき、すぐに投与を中止した。その後、麻薬管理者がこの麻薬の誤投与に対して行う届出に関する記述のうち、正しいのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-333 を参照

**選択肢**
1. 麻薬廃棄届を届け出る。
2. 所有麻薬届を届け出る。
3. 麻薬事故届を届け出る。
4. 調剤済麻薬廃棄届を届け出る。
5. 届出は必要ない。

**正解:** 3

**解説**
解答　３

フェンタニルクエン酸塩注射液などの麻薬注射液を、別の患者に誤って投与してしまった場合には、麻薬管理者が都道府県知事に麻薬事故届を届け出る必要がある。
