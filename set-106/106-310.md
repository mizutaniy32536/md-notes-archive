## ◆ 抗体医薬品投与時の副作用軽減処置を問うシナリオ
260303

**問題**
- 81歳男性。以前、ダラツムマブ（遺伝子組換え）の治験に参加していた。この医薬品の製造販売が承認され、薬価収載されるまでの期間に限り、無償提供プログラムがあることを知り、参加することになった。なお、この医薬品の治験では症例数が限られていたことから、製造販売業者が作成した医薬品リスク管理計画（RMP）には、全例を対象にした特定使用成績調査の実施と安全性検討事項としてInfusion reaction、骨髄抑制、感染症が設定されていた。
- 本剤投与によって起こりうる副作用を軽減するための処置として適切なのはどれか。2つ選べ。

**選択肢**

1. Infusion reactionを軽減させるために、本剤投与前に副腎皮質ホルモンを投与する
​
2. 遅延性のInfusion reactionを軽減させるために、本剤投与後に抗ヒスタミン薬を投与する
​
3. 骨髄抑制のためにABO式血液型の検査を定期的に行う
​
4. 慢性閉塞性肺疾患もしくは気管支ぜん息のある患者には、投与後処置として気管支拡張薬及び吸入ステロイド薬の投与を検討する

**正解と理由**
- `1. Infusion reactionを軽減させるために、本剤投与前に副腎皮質ホルモンを投与する` と `4. COPD・気管支ぜん息患者には気管支拡張薬及び吸入ステロイド薬の投与を検討する`
- **理由の構成**
	- **問題の整理**：ダラツムマブ投与時の副作用軽減処置として適切なものを2つ選ぶ
	- **薬学的知識・仕様・制約**：`Infusion reaction` の予防には投与前の `プレメディケーション`（副腎皮質ホルモン＋抗ヒスタミン薬＋解熱鎮痛薬）が推奨される。COPD・喘息患者では `気管支攣縮` リスクが高く、投与後に気管支拡張薬等の追加処置が推奨
	- **選択理由**：選択肢1はプレメディケーションの基本、選択肢4はリスク患者への追加対応として適切

**解法の手順（思考の流れ）**
- 選択肢3の「骨髄抑制にABO式血液型検査」は血液型は変化しないため的外れと判断
- 選択肢2の「遅延性に抗ヒスタミン薬」は、遅延性Infusion reactionには `副腎皮質ホルモン` が適切であり、抗ヒスタミン薬は `即時型` 反応の予防に使用と判断
- 選択肢1と4が適切な処置であると結論

**他の選択肢が誤りな理由**
- 2. 遅延性のInfusion reactionを軽減させるために、本剤投与後に抗ヒスタミン薬を投与する
	`遅延性` Infusion reactionの軽減には `副腎皮質ホルモン` が適切。`抗ヒスタミン薬` は即時型反応の予防に投与前に使用するもの

- 3. 骨髄抑制のためにABO式血液型の検査を定期的に行う
	`ABO式血液型` は変化しないため、骨髄抑制のモニタリングには不適切。骨髄抑制の評価には `白血球数`・`好中球数`・`血小板数` 等の血液検査が適切

**キーポイント**
- `プレメディケーション`：抗体医薬品投与前に副腎皮質ホルモン＋抗ヒスタミン薬＋解熱鎮痛薬を投与
- `遅延性Infusion reaction`：投与後に出現する可能性があり、`副腎皮質ホルモン` で対処
- COPD・喘息患者では `気管支攣縮` リスクが高く、追加の気管支拡張薬等が必要
- 骨髄抑制のモニタリングは `血球数` の定期検査で行う（血液型検査ではない）

**関連知識の比較**
- `ダラツムマブ` は抗CD38モノクローナル抗体であり、多発性骨髄腫の治療に使用される
- `RMP`（医薬品リスク管理計画）は製造販売業者が策定し、安全性監視活動とリスク最小化活動を計画的に実施するもの
- `特定使用成績調査`：特定の患者集団や使用条件下での安全性・有効性を確認する市販後調査。治験で症例数が限られている場合に全例調査が求められることがある

（一言メモ）
- 抗体医薬品の `Infusion reaction` 対策は「投与前に `プレメディケーション`（ステロイド＋抗ヒスタミン薬＋解熱鎮痛薬）」が基本セット
- `遅延性` の反応には副腎皮質ホルモン、`即時型` の反応には抗ヒスタミン薬という使い分けを覚える
- 骨髄抑制のモニタリングは `CBC`（全血球計算）であり、ABO血液型検査は関係ない
