## ◆ 不凍液誤飲による中毒の代謝機序と治療法を判断するシナリオ
260305

**問題**
- 34歳男性。仕事中に不凍液を誤飲した。数時間後、下腹部痛、下痢、嘔気、嘔吐を伴い、救急外来を受診したところ、徐々に意識レベルが低下してきた。血中エチレングリコール濃度を測定したところ50 mg/dLであり、血液検査と血液ガス分析結果は以下のとおりであった。
	- 血清クレアチニン値：1.8 mg/dL
	- 血液ガス分析：pH 7.01、PaO₂ 99 mmHg、PaCO₂ 32 mmHg、HCO₃⁻ 7.8 mEq/L
- この患者で起こった中毒及びその治療に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**


1. エチレングリコールは、主として肝臓でアルコール脱水素酵素により代謝される。

2. エチレングリコールは、主として肝臓でCYPにより代謝される。

3. グリコール酸がカルシウムと結合して不溶性の沈着物を形成する。

4. ホメピゾールが用意されていなかった場合、エタノールを投与する。

5. ホメピゾールは、肝臓のアルデヒド脱水素酵素を阻害することにより、エチレングリコールを解毒する。

**正解と理由**
- `1. エチレングリコールは、主として肝臓でアルコール脱水素酵素により代謝される。`
- `4. ホメピゾールが用意されていなかった場合、エタノールを投与する。`
- **理由の構成**
	- **問題の整理**：`エチレングリコール`（EG）中毒における代謝経路、毒性代謝物の作用、および治療薬の薬理機序について正誤を判断する
	- **薬学的知識・仕様・制約**：エチレングリコールは肝臓で`アルコール脱水素酵素`（ADH）により`グリコールアルデヒド`に酸化され、さらに`アルデヒド脱水素酵素`（ALDH）により`グリコール酸`へ、最終的に`シュウ酸`へと代謝される。CYPはこの代謝経路の主酵素ではない。`エタノール`はADHに対してエチレングリコールより高い親和性を持ち、競合的にADHを阻害してEGの毒性代謝物生成を抑制するため、`ホメピゾール`が用意できない場合の代替治療として用いられる
	- **選択理由**：選択肢1はEGの主代謝酵素がADHであることを正しく述べており、選択肢4はホメピゾール不在時の代替としてエタノール投与が適切であることを正しく述べている

**解法の手順（思考の流れ）**
- まず「エチレングリコールの代謝経路」に着目し、EGは`ADH`→`ALDH`の経路で代謝されることを想起する
- 選択肢1と2を比較し、主代謝酵素は`ADH`であって`CYP`ではないと判断。選択肢1が正しく、選択肢2は誤りと絞り込む
- 選択肢3について、Caと結合して不溶性沈着物を形成するのは`シュウ酸`であり、`グリコール酸`ではないと判断
- 選択肢4について、`エタノール`がADHに対してEGより高い親和性を持つことから、ホメピゾールの代替としてADH阻害に使用できると判断
- 選択肢5について、`ホメピゾール`の阻害対象は`ADH`であり、`ALDH`ではないことを確認
- これにより選択肢1と4が正しい記述であると結論づけた

**他の選択肢が誤りな理由**
- 2. エチレングリコールは、主として肝臓でCYPにより代謝される。
	`エチレングリコール`の主代謝経路は`CYP`ではなく`アルコール脱水素酵素`（ADH）による酸化である。ADHがEGをグリコールアルデヒドに変換し、続いてALDHがグリコール酸へ変換する。CYPはエチレングリコール代謝にほとんど関与しない

- 3. グリコール酸がカルシウムと結合して不溶性の沈着物を形成する。
	カルシウムと結合して不溶性の`シュウ酸カルシウム`結晶を形成するのは`シュウ酸`であり、グリコール酸ではない。シュウ酸カルシウム結晶は腎尿細管に沈着して`急性腎障害`の原因となる

- 5. ホメピゾールは、肝臓のアルデヒド脱水素酵素を阻害することにより、エチレングリコールを解毒する。
	`ホメピゾール`が阻害するのは`アルコール脱水素酵素`（ADH）であり、`アルデヒド脱水素酵素`（ALDH）ではない。ADH阻害によりエチレングリコールから毒性代謝物への変換を最初のステップで遮断することが解毒の機序である

**キーポイント**
- `エチレングリコール`の代謝経路：EG →（`ADH`）→ グリコールアルデヒド →（`ALDH`）→ グリコール酸 → グリオキシル酸 → `シュウ酸`
- `ホメピゾール`は`ADH`の競合的阻害薬であり、エチレングリコールおよび`メタノール`中毒の第一選択解毒薬である
- `エタノール`はADHに対してEGより親和性が高いため、ホメピゾールが入手できない場合の代替として`競合的阻害`に利用される
- `シュウ酸`がCa²⁺と結合して`シュウ酸カルシウム`結晶を形成し、腎尿細管沈着による急性腎障害や`低カルシウム血症`を引き起こす

**関連知識の比較**
- `メタノール`中毒もADHで代謝（→ホルムアルデヒド→`ギ酸`）され、同様にホメピゾールまたはエタノールで治療する。メタノール特有の毒性として`ギ酸`による視神経障害（失明）がある
- `ジスルフィラム`はALDH阻害薬であり、アルコール依存症治療に用いられる。ホメピゾール（ADH阻害）との阻害対象の違いを整理しておく
- ADH阻害薬（`ホメピゾール`）は代謝の最初のステップを遮断するため、毒性代謝物の生成を根本から防ぐ。ALDH阻害では中間代謝物（アルデヒド）が蓄積するため解毒にならない
- エチレングリコール中毒で生じる`低カルシウム血症`は、シュウ酸がCa²⁺をキレートすることによるもので、テタニーや`QT延長`などの症状を呈する場合がある

（一言メモ）
- `エチレングリコール`の毒性代謝物で覚えるべきは`グリコール酸`（代謝性アシドーシスの主因）と`シュウ酸`（腎障害・低Ca血症の原因）の2つ
- `ホメピゾール`＝ADH阻害、`ジスルフィラム`＝ALDH阻害と対比して覚えると、選択肢5のようなひっかけに対応できる
- `エタノール`のADH親和性がEGより高い性質は、ホメピゾール不在時の代替治療の根拠であり、同時にアルコール飲酒者でEG中毒の進行が遅れる理由でもある
