# 106-109

**問題文**
次の構造式で示す化合物Aに関する記述のうち、正しいのはどれか。2つ選べ。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-109 を参照

**選択肢**
1. 放線菌によって生産されるマクロライドである。
2. 真菌によって生産される環状ペプチドである。
3. Z 配置の二重結合をもつ。
4. ピロール環をもつ。
5. ヘミアセタール構造をもつ。

**正解:** 1,5

**解説**
解答　１、５

１　正
化合物Aはタクロリムスの構造であり、放線菌によって生産されるマクロライド（巨大な環を有するラクトン）である。

２　誤
環状ペプチドとは、複数のアミノ酸がペプチド結合（アミド結合）することで環を形成した化合物である。化合物Aはアミド結合が1ヶ所のみであり、環状ペプチドとは言えない。なお、真菌によって生産される環状ペプチドにはシクロスポリンがある。

３　誤
E/Z 表記とは、順位則による優先順位の高い置換基が二重結合に対して反対側であればE配置、同じ側であればZ配置とする幾何異性体の表記法である。化合物Aの有するE/Z 配置を考えることが可能な二重結合は、全てE配置である。（下図参照）

４　誤
ピロール環は、窒素原子を1つ含有する5員環の芳香族であり、化合物Aには含まれない。なお、化合物Aが有する複素環としては、ピペリジン環とテトラヒドロピラン環がある。（下図参照）

５　正
ヘミアセタール構造は、1つの炭素原子がヒドロキシ基とエーテル基（アルコキシ基）を両方もつ構造であり、化合物Aはヘミアセタール構造を有する。（下図参照）
