# 第106回 問66

**問題文**
帯状疱疹の治療薬はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 106-66 を参照

**選択肢**
1. ガンシクロビル
2. ザナミビル水和物
3. バラシクロビル塩酸塩
4. ラルテグラビルカリウム
5. リトナビル

**正解:** 3

**解説**
解答　３

帯状疱疹は、ヘルペスウイルス3型（水痘・帯状疱疹ウイルス）の回帰感染により発症する。治療には、抗ウイルス薬のアシクロビルやバラシクロビル、ファムシクロビルなどが用いられる。

１　誤
ガンシクロビルはサイトメガロウイルス感染症に用いられる。

２　誤
ノイラミニダーゼ阻害薬であるザナミビル水和物は、インフルエンザ感染症に用いられる。

３　正
冒頭解説文参照。

４　誤
HIVインテグラーゼ阻害薬であるラルテグラビルカリウムは、HIV感染症に用いられる。

５　誤
プロテアーゼ阻害薬であるリトナビルはHIV感染症やC型肝炎に用いられる。
