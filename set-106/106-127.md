## ◆ エネルギー代謝に関する正誤判断
260305

**問題**
エネルギー代謝に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**

1. 呼吸商とは、二酸化炭素排出量から酸素消費量を差し引いた値である。
​
2. Atwater係数は、糖質、脂質、タンパク質の物理的燃焼値、消化吸収率及び未利用エネルギーをもとに設定された値である。
​
3. 非タンパク質呼吸商は、尿中に排泄される窒素量に窒素係数6.25を乗じて求められる。
​
4. 基礎代謝基準値は、男女ともに10代で最大となる。
​
5. 成人の推定エネルギー必要量は、基礎代謝量に身体活動レベルを乗じて求められる。

**正解と理由**
- `2. Atwater係数は、糖質、脂質、タンパク質の物理的燃焼値、消化吸収率及び未利用エネルギーをもとに設定された値である。`
- `5. 成人の推定エネルギー必要量は、基礎代謝量に身体活動レベルを乗じて求められる。`
- **理由の構成**
	- **問題の整理**：エネルギー代謝に関する5つの記述から正しいものを2つ選ぶ問題
	- **薬学的知識・仕様・制約**
		- `Atwater係数` は物理的燃焼値、消化吸収率、未利用エネルギーの3要素から算出される（糖質4、脂質9、タンパク質4 kcal/g）
		- `推定エネルギー必要量` は基礎代謝量（BMR）に身体活動レベル（PAL）を乗じて求められる
	- **選択理由**：選択肢2はAtwater係数の定義そのものであり正しい。選択肢5はPAL法による推定エネルギー必要量の算出式として正しい

**解法の手順（思考の流れ）**
- まず各選択肢がエネルギー代謝のどの概念に関するものかを分類する
- 選択肢1は `呼吸商` の定義 → CO₂排出量÷O₂消費量（比であり差ではない）と想起し、誤りと判断
- 選択肢2は `Atwater係数` の設定根拠 → 物理的燃焼値・消化吸収率・未利用エネルギーの3要素から成ることを確認し、正しいと判断
- 選択肢3は `非タンパク質呼吸商` の定義 → 尿中N×6.25はタンパク質の体内燃焼量を求める式であり、非タンパク質呼吸商そのものではないと判断
- 選択肢4は `基礎代謝基準値` の年齢変化 → 体重あたりの基準値は1〜2歳で最大であり、10代で最大になるのは基礎代謝量（絶対値）と判断
- 選択肢5は `推定エネルギー必要量` の算出法 → 基礎代謝量×身体活動レベル（PAL法）であり正しいと確認
- 以上から選択肢2と5が正解と結論づけた

**他の選択肢が誤りな理由**
- 1. 呼吸商とは、二酸化炭素排出量から酸素消費量を差し引いた値である。
	`呼吸商`（RQ）はCO₂排出量÷O₂消費量の「比」であり、差し引き（減算）ではない。糖質のRQ＝1.0、脂質＝0.7、タンパク質＝0.8が代表値

- 3. 非タンパク質呼吸商は、尿中に排泄される窒素量に窒素係数6.25を乗じて求められる。
	尿中窒素量×`窒素係数6.25` で求められるのはタンパク質の体内燃焼量であり、`非タンパク質呼吸商`（npRQ）ではない。npRQはタンパク質由来のCO₂とO₂を全体から差し引いた上で算出する

- 4. 基礎代謝基準値は、男女ともに10代で最大となる。
	`基礎代謝基準値`（体重あたりの基礎代謝量、kcal/kg/日）は男女ともに1〜2歳で最大。10代で最大となるのは`基礎代謝量`（kcal/日の絶対値）であり、基準値と混同しないことが重要

**キーポイント**
- `呼吸商` はCO₂排出量÷O₂消費量の比であり、糖質1.0、脂質0.7、タンパク質0.8が基本値
- `Atwater係数` は物理的燃焼値・消化吸収率・未利用エネルギーから設定され、糖質4、`脂質9`、タンパク質4 kcal/g
- `基礎代謝基準値`（体重あたり）は1〜2歳で最大、基礎代謝量（絶対値）は10代で最大 → 「基準値」と「量」の区別が頻出
- `推定エネルギー必要量` ＝ 基礎代謝量 × 身体活動レベル（PAL）
- `非タンパク質呼吸商` は全体のガス交換量からタンパク質由来を除いて算出するもので、尿中窒素量×6.25とは別の概念

**関連知識の比較**
- `呼吸商` と `非タンパク質呼吸商` の違い：RQは全体のCO₂/O₂比、npRQはタンパク質由来を除いたCO₂/O₂比で、体内でどの栄養素が優位に燃焼しているかを判定するのに使う
- `Atwater係数` と `物理的燃焼値` の違い：物理的燃焼値はボンベ熱量計で測定した値（糖質4.1、脂質9.4、タンパク質5.6 kcal/g）であり、消化吸収率と未利用エネルギーを補正したものがAtwater係数
- `基礎代謝量`（BMR）と `安静時代謝量`（RMR）の違い：BMRは早朝空腹・覚醒・安静の厳密な条件下で測定。RMRはやや緩い条件で測定され、BMRより約10%高い
- `身体活動レベル`（PAL）は日本人の食事摂取基準ではI（低い：1.50）、II（ふつう：1.75）、III（高い：2.00）の3段階に分類される

（一言メモ）
- `呼吸商` は「割り算」、差し引きは引っかけ → 選択肢で「差」「引く」と出たら要注意
- `基礎代謝基準値`（体重あたり）のピークは1〜2歳、`基礎代謝量`（絶対値）のピークは10代 → 「基準値」か「量」かで答えが変わる頻出論点
- `Atwater係数` の3要素（物理的燃焼値・消化吸収率・`未利用エネルギー`）はセットで覚える
