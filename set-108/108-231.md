# 108-231

### 第108回薬剤師国家試験 問231

**問題文**  
28歳男性。勤務先のメッキ工場で、ビンに小分けしてあった液体を誤って飲用した。その直後に苦しがったため、近くにいた同僚が救急車を要請し、救急搬送された。来院時、頭痛、悪心、嘔吐及び頻脈を認めた。胃洗浄時に採取した胃内容物をシェーンバイン法により検査したところ、グアヤク試験紙が青色へ変化した。[1][2][3]

**問231（実務）**  
この患者に用いる解毒剤として適切なのはどれか。**2つ選べ**。[1][2][3]

1. プラリドキシムヨウ化メチル (PAM)  
2. 硫酸アトロピン  
3. ヒドロキソコバラミン  
4. 亜硝酸アミル  
5. ジメルカプロール[1][2][3]

**正解番号**  
3, 4[1][2][5]

**解説**  
本問は問230でシアン化カリウム（シアン中毒）と判明した患者に対する解毒剤を選択する問題である。メッキ工場での誤飲、頭痛・悪心・嘔吐・頻脈、シェーンバイン法（グアヤク試験紙が青色変化：シアン化物検出）からシアン中毒が確定。[1][2][3]  

- **1. 誤り**：プラリドキシムヨウ化メチル（PAM）は有機リン剤中毒（例: ジクロルボス）やサリンに用いる。[1][3]  
- **2. 誤り**：硫酸アトロピンはカルバメート系殺虫剤中毒に用いる。[1]  
- **3. 正しい**：ヒドロキソコバラミンはシアン中毒の解毒剤で、シアンをビタミンB12に結合させて無毒化する。[1][3]  
- **4. 正しい**：亜硝酸アミルはシアン中毒の解毒剤で、メトヘモグロビンを生成しシアンをメトヘモグロビン・シアン複合体として解毒（その後チオ硫酸ナトリウム等で排泄促進）。[1][3]  
- **5. 誤り**：ジメルカプロール（BAL）はヒ素や重金属中毒に用いる。[1][3]  

シアン中毒の標準解毒療法として**3と4**が適切。[1][2][3][10]
