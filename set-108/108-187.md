# 108-187

**問題文**
薬物動態に影響を与える因子のうち、高齢者において起きていると考えにくいのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-187 を参照

**選択肢**
1. 体脂肪率の増加
2. 血漿中のアルブミン濃度の低下
3. 血漿中のα1−酸性糖タンパク濃度の低下
4. 肝血流量の減少
5. 糸球体ろ過量の減少

**正解:** 3

**解説**
解答　３

１　考えられる
高齢者では、筋肉量の減少に伴い脂肪量が増加するため、体脂肪率の増加がみられる。

２　考えられる
高齢者では、肝機能低下によりアルブミン合成能が低下するため、血漿中のアルブミン濃度の低下がみられる。

３　考えにくい
高齢者では、血漿中のα1−酸性糖タンパク濃度の上昇がみられる。

４　考えられる
高齢者では、肝機能低下により肝血流量の減少がみられる。

５　考えられる
高齢者では、腎機能低下により糸球体ろ過量の減少の減少がみられる。
