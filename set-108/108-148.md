# 108-148

**問題文**
介護保険に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-148 を参照

**選択肢**
1. 保険者は、市町村（特別区含む）である。
2. 保険給付を受けていない第2号被保険者は、保険料を支払う必要はない。
3. 第2号被保険者は、特定疾病によって生じた障害による要支援・介護状態でなければ、保険給付を受けることはできない。
4. 要介護認定（要支援認定を含む）は、主治医が行う。
5. 要支援状態と認定された者はサービスの利用に際し費用の自己負担があるが、要介護状態と認定された者は自己負担がない。

**正解:** 1,3

**解説**
解答　１、３

１　正
介護保険の保険者は、市町村（特別区含む）である。

２　誤
介護保険の保険料は、保険給付を受けているかに関わらず、保険料を支払う必要がある。従って、第2号被保険者は、保険料を支払う必要がある。

３　正
介護保険の給付を受けるためには、要介護認定又は要支援認定を受ける必要がある。第2号被保険者は、特定疾病によって生じた障害による要支援・介護状態でなければ、要介護又は要支援認定を受けることができないため、保険給付を受けることはできない。

４　誤
要介護認定（要支援認定を含む）は、保険者である市町村（特別区含む）が行う。

５　誤
要支援状態と認定された者及び要介護状態と認定された者は、サービスの利用に際し費用の自己負担がある。
