# 108-148

介護保険に関する記述のうち、正しいのはどれか。2つ選べ。

1. 保険者は、市町村（特別区含む）である。
2. 保険給付を受けていない第2号被保険者は、保険料を支払う必要はない。
3. 第2号被保険者は、特定疾病によって生じた障害による要支援・介護状態でなければ、保険給付を受けることはできない。
4. 要介護認定（要支援認定を含む）は、主治医が行う。
5. 要支援状態と認定された者はサービスの利用に際し費用の自己負担があるが、要介護状態と認定された者は自己負担がない。

## 正解
**1, 3**

## 解説

| 選択肢 | 正誤 | 理由 |
|---|---|---|
| 1. 保険者は市町村 | ⭕ | 介護保険の保険者は市町村（特別区含む） |
| 2. 給付なしの第2号は保険料不要 | ❌ | 給付の有無に関係なく保険料の支払い義務あり |
| 3. 第2号は特定疾病が条件 | ⭕ | 第2号被保険者（40〜64歳）は特定疾病による要支援・要介護状態のみ給付対象 |
| 4. 要介護認定は主治医が行う | ❌ | 認定は市区町村が行う（介護認定審査会の審査を経て）。主治医は「意見書」を書くだけ |
| 5. 要介護は自己負担なし | ❌ | 要支援・要介護どちらも原則1割（所得により2〜3割）の自己負担あり |

### 介護保険の被保険者まとめ
| | 第1号被保険者 | 第2号被保険者 |
|---|---|---|
| 年齢 | 65歳以上 | 40〜64歳 |
| 給付条件 | 原因を問わず | **特定疾病**が原因の場合のみ |
