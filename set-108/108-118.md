# 108-118

**問題文**
細菌の抗菌薬耐性化には種々の遺伝子もしくはその産物が関わっている。次の記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-118 を参照

**選択肢**
1. グラム陰性菌の外膜にあるポーリンの増加は、カルバペネム系抗菌薬への耐性化を促す。
2. 腸球菌のバンコマイシン耐性遺伝子VanAの産物は、細胞壁のペプチドグリカン合成の代替経路で働き、グリコペプチド系抗菌薬の作用を回避する。
3. リファンピシン耐性菌では、DNAポリメラーゼ遺伝子の変異により、耐性が獲得されている。
4. カナマイシン耐性遺伝子産物の中には、抗菌薬をリン酸化する酵素として働き、RNAポリメラーゼへの結合を消失させるものがある。
5. Rプラスミドなどの伝達性因子を介して薬物排出タンパク質が過剰発現することは、多剤耐性化の一因となる。

**正解:** 2,5

**解説**
解答　２、５

細菌の抗菌薬耐性化には種々の遺伝子もしくはその産物が関わっており、その仕組みとしては主に以下の4つがあげられる。
①	抗菌薬を分解・修飾する
②	細菌構造を変えて、抗菌薬を作用させなくする
③	抗菌薬を細菌細胞質から排出しやすくする
④	抗菌薬を細菌細胞質に入れない

１　誤
カルバペネム系抗菌薬がグラム陰性菌に作用する際には、細菌外膜にあるポーリンと呼ばれる管状タンパクから侵入する必要がある。そのため、このポーリンの減少や欠損がカルバペネム系抗菌薬への耐性化を促す要因となり、仕組みとしては上記の④に該当する。

２　正
バンコマイシンなどのグリコペプチド系抗菌薬は、細胞壁前駆体のD−Alanyl−D−Alanine部分と結合し、細胞壁（ペプチドグリカン）合成を阻害することで抗菌作用を示す。バンコマイシン耐性腸球菌（VRE）は、バンコマイシン耐性遺伝子VanAの産物により、D−Alanyl−D−Alanine部分をD−Alanyl−D−Lactate（乳酸塩）に変化させて、バンコマイシンを結合できなくすることで、グリコペプチド系抗菌薬の作用を回避している。なお、仕組みとしては上記の②に該当する。

３　誤
リファンピシンは、RNAポリメラーゼ阻害作用を有する抗結核薬であり、耐性菌ではその作用点であるRNAポリメラーゼのβサブユニットをコードする遺伝子の変異により耐性を獲得している。なお、仕組みとしては、上記の②に該当する。

４　誤
カナマイシンなどのアミノグリコシド系抗菌薬は、細菌リボソームに結合し、タンパク質合成を阻害することで抗菌作用を示す。アミノグリコシド系抗菌薬耐性菌は、耐性遺伝子により産生したリン酸化酵素やアセチル化酵素などにより、抗菌薬を不活化させて細菌リボソームへの結合を消失させる。なお、仕組みとしては、上記の①に該当する。

５　正
薬物排出タンパク質が過剰発現して抗菌薬を細菌細胞質から排出しやすくすることは、抗菌薬耐性化の一因となり、仕組みとしては、上記の③に該当する。なお、Rプラスミドとは、薬剤耐性遺伝子が組み込まれているプラスミドであり、薬剤耐性菌が性線毛を介して感受性菌に接合し、Rプラスミドのコピーを伝達することで、さらに薬剤耐性菌を増やしていく。
