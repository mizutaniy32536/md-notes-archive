# 108-98

### 第108回薬剤師国家試験 問98

**問題文**  
定量法に関する記述のうち、正しいのはどれか。**２つ**選べ。[3][5]

**選択肢**  
1. 試料95.67 mgを量り取った場合、「約90 mg」を量り取ったことにならない。[3][5]  
2. L-アラニンはアセチル化された後、過塩素酸と反応する。[3][5]  
3. この電位差滴定法では、指示電極にガラス電極を用いる。[1][2]  
4. 本試験より空試験の方が、0.1 mol/L過塩素酸の滴加量は少ない。[1][3]  
5. (　ア　)に入る数値は 4.455 である。[1][3]

**正解番号**  
**3, 4**[1][2]

**解説**  
- **選択肢1（誤）**: 「約」は±10%の範囲を意味し、約90 mgは81～99 mgに相当する。95.67 mgは範囲内であるため、「約90 mg」を量り取ったことになる。[1][2][5]  
- **選択肢2（誤）**: 酢酸(100)は非水滴定の溶媒であり、アラニンをアセチル化しない。無水酢酸との混同に注意。[1]  
- **選択肢3（正）**: 電位差滴定法（非水溶媒中）では、指示電極にガラス電極を用いるのが標準。[1][2]  
- **選択肢4（正）**: 空試験は試料なしのため、過塩素酸の消費量が本試験より少ない。[1][3]  
- **選択肢5（誤）**: 0.1 mol/L過塩素酸1 mLは0.1 mmolに相当し、アラニンの分子量89.09から(ア)は8.909 mg。「4.455」は誤り。[1]
