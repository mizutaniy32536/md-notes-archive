



## ◆ ブシラミンによる味覚異常患者への薬剤師対応の適切性判断

260313

**問題**
- 70歳男性。両手関節痛・腫れで整形外科受診、入院し軽度関節リウマチ診断。メトトレキサートで肝機能障害が出現し、ブシラミン錠100mg 1回1錠（1日3回、朝昼夕食後、7日分）に変更。2週間後、服薬指導時「食事が美味しくない、味がしない」との訴え、食事摂取量減少。薬剤師が血清亜鉛測定を依頼：血清亜鉛65μg/dL（基準値80-130μg/dL）。医師が酢酸亜鉛水和物を追加処方。
- 今後の薬剤師の対応として適切でないのはどれか。１つ選べ。

**選択肢**

1. 味覚異常の原因薬剤と考えられるブシラミン錠の継続の可否について医師に確認する。
​
2. 味覚障害には口内炎や口腔内乾燥を伴うこともあるため、これらの症状について患者に確認する。
​
3. 食事が摂れていないため、入院中の食事を亜鉛を多く含むメニューに変更できないか、管理栄養士と連携する。
​
4. 患者に食事を美味しく感じなくても、体力維持のためにできるだけ食事を摂るよう指導する。
​
5. 亜鉛の補充により銅の吸収が増強され銅過剰症を起こすおそれがあると医師に情報提供を行う。

**正解と理由**
- `5. 亜鉛の補充により銅の吸収が増強され銅過剰症を起こすおそれがあると医師に情報提供を行う。`
- **理由の構成**
	- **問題の整理**：ブシラミンによる血清亜鉛低下→味覚異常に対し、酢酸亜鉛水和物が追加。薬剤師の対応で「適切でない」ものを選ぶ
	- **薬学的知識・仕様・制約**：亜鉛製剤の長期投与では、亜鉛が腸管で`銅の吸収を抑制`し、`銅欠乏症`（貧血、しびれ等）を起こすリスクがある。「銅過剰症」ではなく`銅欠乏症`が正しい
	- **選択理由**：選択肢5は銅の吸収が「増強」されるとしているが、実際は「抑制」される。情報提供の内容自体が誤り

**解法の手順（思考の流れ）**
- まず各選択肢が薬剤師の対応として適切かを確認
- 選択肢1〜4は味覚異常への多職種連携・患者指導として妥当
- 選択肢5：亜鉛と銅の関係を想起→亜鉛過剰は銅「抑制」であり銅「過剰」は誤り
- これが唯一の「適切でない」対応と結論

**他の選択肢が誤りな理由**
- 1. ブシラミン錠の継続の可否について医師に確認する。
	適切。`ブシラミン`のSH基が亜鉛キレートを形成して味覚異常を引き起こしているため、薬剤変更の検討は重要

- 2. 口内炎や口腔内乾燥の有無を患者に確認する。
	適切。味覚障害は口腔内の問題（`口内炎`、`口腔乾燥`、`舌炎`等）でも起こるため、鑑別が必要

- 3. 管理栄養士と連携して亜鉛含有メニューへの変更を検討する。
	適切。栄養面からの`亜鉛補給`と多職種連携は薬剤師の役割として妥当

- 4. 体力維持のためにできるだけ食事を摂るよう指導する。
	適切。低栄養予防の基本的な患者指導として妥当

**キーポイント**
- `ブシラミン`はSH基を2個持ち、`亜鉛とキレート`を形成→血清亜鉛低下→味覚異常
- `酢酸亜鉛水和物`（ノベルジン®）の重要な副作用＝`銅欠乏症`（銅の吸収を抑制する）
- 亜鉛製剤投与中は定期的に`血清銅`をモニタリングする必要がある

**関連知識の比較**
- `D-ペニシラミン`もSH基を持つ抗リウマチ薬で、同様に亜鉛キレート→味覚異常を起こす
- 亜鉛欠乏による味覚異常の機序：亜鉛は`味蕾`の代謝回転に必須。欠乏すると味蕾細胞の新生が障害される
- `ウイルソン病`の治療でも酢酸亜鉛水和物が使われる。銅の吸収抑制が治療効果の本体

（一言メモ）
- 亜鉛と銅は`拮抗関係`。亜鉛過剰→`銅欠乏`、銅過剰ではない。これは頻出の引っかけ
- `ブシラミン`の味覚異常＝SH基2個が亜鉛をキレート。同じ機序で`D-ペニシラミン`もやる
- `酢酸亜鉛水和物`（ノベルジン®）は味覚異常とウイルソン病の両方に適応
