# 108-44

カルボキシルエステラーゼにより加水分解を受ける薬物はどれか。１つ選べ。

**正解:** １

**解説**
カルボキシルエステラーゼとは、エステル構造（O=C-O-）を加水分解する酵素のことである。本問の選択肢のうち、エステル構造を有する医薬品は選択肢１のテモカプリルである。
テモカプリルは、プロドラッグであり、経口投与後、活性代謝物のテモカプリラートとなり、これがアンギオテンシン変換酵素を阻害する。
