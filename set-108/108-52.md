# 108-52

**問題文**
吸入粉末剤に関する記述として、正しいのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-52 を参照

**選択肢**
1. 定量噴霧式である。
2. 固体粒子のエアゾールとして吸入する。
3. 噴射剤が充填されている。
4. 容器は、通例、密封容器とする。
5. ネブライザーが適用される。

**正解:** 2

**解説**
解答　２

日本薬局方において、吸入粉末剤は、吸入量が一定となるように調製された固体粒子のエアゾールとして吸入する製剤と定義されている。

１　誤
吸入エアゾール剤の記述である。

２　正
前記参照

３　誤
吸入エアゾール剤の記述である。

４　誤
吸入エアゾール剤の記述である。

５　誤
吸入液剤の記述である。
