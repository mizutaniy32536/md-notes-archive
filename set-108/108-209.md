



## ◆ PT-INRモニタリングに関与する医薬品の構造式特定

260313

**問題**
- 68歳女性。再発直腸がんに対しカペシタビン＋オキサリプラチン併用化学療法開始。かかりつけ医の処方薬の構造式を示す。
- その中でモニタリングすべき検査項目（PT-INR）に関与する医薬品はどれか。１つ選べ。
本品の性状：日本薬局方「水に極めて溶けやすく、エタノール（95）に溶けやすい」「光によって淡黄色となる」。

**正解と理由**
- `1. ワルファリンカリウム`
- **理由の構成**
	- **問題の整理**：構造式からPT-INRに関与する薬物を特定する
	- **薬学的知識・仕様・制約**：`ワルファリン`は`クマリン誘導体`で、ビタミンK依存性凝固因子（Ⅱ、Ⅶ、Ⅸ、Ⅹ）の合成を阻害する抗凝固薬。その効果の指標が`PT-INR`。ワルファリンカリウムは水に極めて溶けやすく、光により淡黄色に変色する
	- **選択理由**：PT-INRに関与する薬物はワルファリンのみ

**解法の手順（思考の流れ）**
- まずPT-INRに関与する薬物＝`抗凝固薬`＝`ワルファリン`と即座に判断
- 構造式から`クマリン骨格`を持つ薬物を特定
- 性状「水に極めて溶けやすい」「光で淡黄色」もワルファリンKの特徴に合致
- 選択肢1が該当と結論

**他の選択肢が誤りな理由**
- 2. カルベジロール
	`αβ遮断薬`。高血圧・心不全に使用。PT-INRには関与しない

- 3. ロサルタン
	`ARB`（アンジオテンシンⅡ受容体拮抗薬）。CYP2C9で代謝されるがPT-INRには直接関与しない

- 4. スピロノラクトン
	`抗アルドステロン性利尿薬`。心不全・浮腫に使用。PT-INRには関与しない

- 5. ヒドロクロロチアジド
	`サイアザイド系利尿薬`。高血圧に使用。PT-INRには関与しない

- 6. メチルジゴキシン
	`強心配糖体`。心不全・心房細動に使用。PT-INRには関与しない

**キーポイント**
- `ワルファリン`の構造：`クマリン骨格`（ベンゾピロン-2-オン）を持つ
- ワルファリンKの性状：水に極めて溶けやすい（カリウム塩のため）、`光で変色`（遮光保存が必要）
- PT-INRの治療域：通常`1.5〜2.5`（日本）、`2.0〜3.0`（欧米）

**関連知識の比較**
- `DOAC`（直接経口抗凝固薬）は原則PT-INRモニタリング不要。`リバーロキサバン`、`アピキサバン`、`エドキサバン`（Xa阻害）、`ダビガトラン`（トロンビン阻害）
- `ロサルタン`もCYP2C9基質だが、カペシタビンとの相互作用による重篤な副作用リスクはワルファリンほど高くない
- ワルファリンの`食事制限`：ビタミンK含有食品（納豆、クロレラ、青汁）の過剰摂取を避ける

（一言メモ）
- PT-INR＝`ワルファリン`の効果指標。これは反射的に出せるレベルで覚える
- ワルファリンの構造＝`クマリン骨格`。「ネズミ駆除剤」起源という歴史も覚えやすい
- DOACは日本では`DOAC`、海外では`NOAC`と呼ぶことが多い
