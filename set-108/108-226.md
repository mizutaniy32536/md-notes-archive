## ◆ 妊婦のリチウム製剤服用と胎児心奇形の相対危険度を算出するシナリオ

260311

**問題**
精神神経科の医師から医薬品情報室に、妊婦がリチウム製剤とラモトリギン製剤を服用したときの胎児における心奇形の発生について質問があった。薬剤師が論文を検索し、各製剤の服用による胎児の心奇形発生に関するコホート研究報告を見つけた。論文には下表に示した結果が掲載されていた。
心奇形発生に及ぼすリチウム製剤の服用の相対危険度はどれか。1つ選べ。

**正解と理由**
- `3. 2.4`
- **理由の構成**
	- **問題の整理**：コホート研究の結果から`リチウム製剤`服用群と非服用群の心奇形発生率を比較し、`相対危険度`を算出する
	- **薬学的知識・仕様・制約**：`相対危険度`（RR）=曝露群の発生率÷非曝露群の発生率。コホート研究で算出可能な指標
	- **選択理由**：RR = 3.6% ÷ 1.5% = 0.036 ÷ 0.015 = 2.4

**解法の手順（思考の流れ）**
- 研究デザインを確認：コホート研究 → `相対危険度`を算出する問題と判断
- 表から各群の心奇形発生率を読み取る
- リチウム製剤服用群の発生率：3.6%（0.036）
- 薬剤非服用群の発生率：1.5%（0.015）
- RR = 0.036 ÷ 0.015 = `2.4`
- 選択肢3を選択

**他の選択肢が誤りな理由**
- 1. 0.9
	RR<1は曝露が防御因子であることを意味するが、リチウムは心奇形リスクを増加させるため不適切

- 2. 1.6
	ラモトリギン製剤の相対危険度と混同した可能性がある計算結果

- 4. 3.6
	リチウム製剤服用群の心奇形発生率（%）そのものであり、`相対危険度`ではない

- 5. 4.2
	表の数値の読み間違いや計算間違いによる値

**キーポイント**
- `相対危険度`=曝露群の発生率÷非曝露群の発生率。コホート研究の基本計算
- `リチウム製剤`は`Ebstein奇形`（三尖弁奇形）のリスク因子として知られる
- RR>1 → リスク増加、RR=1 → 関連なし、RR<1 → 防御効果

**関連知識の比較**
- `リチウム`：双極性障害の気分安定薬、妊婦禁忌（`Ebstein奇形`）、TDM対象（有効血中濃度0.6-1.2mEq/L）、腎排泄型
- `ラモトリギン`：てんかん・双極性障害に使用、電位依存性Na⁺チャネル遮断、`スティーブンス・ジョンソン症候群`に注意
- 妊婦と薬物：`バルプロ酸`（神経管閉鎖障害）、`ワルファリン`（骨・軟骨異常）、`メトトレキサート`（催奇形性）も重要

（一言メモ）
- 相対危険度の計算は「割り算するだけ」。発生率を%のまま割っても小数に直して割っても同じ結果
- `リチウム`と`Ebstein奇形`のセットは国試定番。双極性障害の妊婦への薬物治療選択として問われる
- 表の数値を「発生率」と「相対危険度」で混同させるひっかけに注意（選択肢4のパターン）
