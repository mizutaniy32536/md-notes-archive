# 108-59

パーキンソン病で認められる特徴的な症状はどれか。１つ選べ。

１ 眼瞼下垂
２ 口渇
３ 動作時振戦
４ 高血圧
５ 無動

**正解:** ５

**解説**
パーキンソン病は、中脳黒質のドパミン作動性神経が変性する神経変性疾患であり、安静時振戦、無動、筋固縮、姿勢反射障害などの不随意運動障害を主徴とする。また、自律神経症状として便秘や起立性低血圧、精神症状として抑うつや認知症などもみられることがある。
