# 108-182

**問題文**
BCS（Biopharmaceutics classification system）分類とは、薬物を溶解性と膜透過性に基づいて4つに分類したものである。溶解性は低いが膜透過性が高いClass2の薬物Xを含有する経口固形製剤Aについて、以下のデータが得られている。製剤Aにおける吸収改善の機構と最も関連の深いのはどれか。1つ選べ。ただし、薬物Xとしての経口投与量は同一とする。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-182 を参照

**選択肢**
1. 塩の形成
2. 無水物化
3. 共結晶化
4. 溶媒和物
5. 固体分散体

**正解:** 5

**解説**
解答　５

溶解性は低いが膜透過性が高いClass2の薬物Xを含有する経口固形製剤Aにおける吸収改善の機構として、製剤からの薬物溶出と経口投与後の血中薬物濃度の時間推移のデータより、薬物Xから製剤Aに溶解性を改善することで、吸収改善を行うことができたと考えられる。
さらに粉末X線回折測定の結果より、製剤Aは明瞭なピークが認められないハローパターンを示すことから、非晶質であると考えられる。選択肢のうち、非晶質を含む製剤は固体分散体であるため、製剤Aは固体分散体であると判断できる。
