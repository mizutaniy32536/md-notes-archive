# 108-78

**問題文**
薬物療法の経済評価手法である費用効用分析ではアウトカム指標としてQALYを用いる。QALYに加味されている要素はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-78 を参照

**選択肢**
1. 患者の医療費
2. 介護に要する費用
3. 生活の質
4. 処方薬剤数
5. 平均寿命

**正解:** 3

**解説**
解答　３

薬物療法の経済評価手法である費用効用分析で用いられるアウトカム指標のQALYは質調整生存年ともよばれ、評価する医療技術によって期待される患者の生存年数とQOL（生活の質）をスコア化した効用値をかけ合わせた数値として表される。
