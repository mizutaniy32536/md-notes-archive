# 108-109

**第108回薬剤師国家試験 問109**

**問題文**  
下図は、ある化合物の1H-NMRスペクトル（400 MHz、CDCl₃、基準物質はテトラメチルシラン）を示したものであり、表は各シグナルの積分比を一覧にしたものである。また、この化合物はIRスペクトルにて1770 cm⁻¹付近に強い吸収を示した。この化合物の構造式として正しいのはどれか。1つ選べ。  
なお、×印のシグナルは水又はCDCl₃中に含まれるCHCl₃のプロトンに由来する。[2][3][5]

**選択肢**  
1. **正解の構造（アセチル基とメチル基を持つp-二置換ベンゼン誘導体、具体的な構造式はIRとNMRに一致）**  
2. O-CH₃を含む構造  
3. O-CH₃を含む構造  
4. 不適切な構造  
5. O-CH₃を含む構造[2][4][5]

**正解番号**  
**1**[2][3][5]

**解説**  
IRスペクトルで1770 cm⁻¹付近に強い吸収があるため、カルボニル基（C=O）が存在し、選択肢を1,3,5に絞る。1H-NMRスペクトルで4 ppm付近に1重線のピークがないため、O-CH₃構造はなく、選択肢3と5を除外。残る選択肢1は、1 ppm付近の3H（脂肪族CH₃）、6-8 ppm付近の対称的なアロマチックプロトン（p-二置換ベンゼン）、および積分比が一致する。[2][4][5]  
アとウのシグナルはp-二置換ベンゼンの特徴を示し、R=CH₃（3H）、E=3H、U=2HがNMRデータと適合。[5]  
よって、正解は1。[2][3][4][5]
