# 108-165

**問題文**
内分泌系に作用する薬物に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-165 を参照

**選択肢**
1. クロミフェンは、GnRH（性腺刺激ホルモン放出ホルモン）受容体を遮断して、排卵を誘発する。
2. テリパラチドは、副甲状腺ホルモン受容体を遮断して、骨吸収を抑制する。
3. プロピルチオウラシルは、ペルオキシダーゼを阻害して、甲状腺ホルモンの生成を抑制する。
4. ブロモクリプチンは、ドパミンD2受容体を遮断して、プロラクチン分泌を抑制する。
5. メテノロンは、アンドロゲン受容体を刺激して、タンパク質同化作用を示す。

**正解:** 3,5

**解説**
解答　３、５

１　誤
クロミフェンは、視床下部におけるエストロゲン受容体遮断により、負のフィードバックの阻害を介してGn-RH（ゴナドトロピン放出ホルモン）分泌を促進させる。その結果、脳下垂体前葉からのLH（黄体形成ホルモン）分泌を促進させることで、排卵を誘発する。

２　誤
テリパラチドは、間欠投与での副甲状腺ホルモン受容体刺激による、前駆細胞から骨芽細胞への分化促進作用や骨芽細胞のアポトーシス抑制作用を示すことで、骨芽細胞数を増加させ骨形成を促進する。

３　正
プロピルチオウラシルは、甲状腺ペルオキシダーゼを阻害することで、甲状腺ホルモンの生成を抑制する。

４　誤
ブロモクリプチンは、脳下垂体前葉のドパミンD2受容体を刺激して、プロラクチン分泌を抑制する。

５　正
メテノロンは、アンドロゲン受容体を刺激して、男性ホルモン様のタンパク質同化作用を示す。
