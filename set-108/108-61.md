# 108-61

特徴的な症状の1つとして、間欠性跛行がみられる疾患はどれか。１つ選べ。

１ 閉塞性動脈硬化症
２ 筋ジストロフィー
３ 変形性膝関節症
４ 痛風
５ 関節リウマチ

**正解:** １

**解説**
間欠性跛行とは、歩行を続けると下肢に疼痛や疲労感を感じるが、数分間の休息で軽快し、再び問題なく歩行できるようになる症状である。選択肢中の疾患のうち閉塞性動脈硬化症（ASO）は、四肢（特に下肢）の末梢動脈が動脈硬化によって慢性虚血をきたす疾患であり、下肢の慢性的な虚血による間欠性跛行が特徴的な症状である。
