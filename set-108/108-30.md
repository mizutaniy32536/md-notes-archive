# 108-30

**問題文**
スプラタストの抗アレルギー作用の機序はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-30 を参照

**選択肢**
1. ヒスタミンH1受容体遮断
2. Th2サイトカイン産生抑制
3. トロンボキサン合成酵素阻害
4. 5−リポキシゲナーゼ阻害
5. プロスタノイドTP受容体遮断

**正解:** 2

**解説**
解答　２

スプラタストは、IL−4やIL−5などのヘルパーT細胞（Th2）から放出されるサイトカインの産生を抑制し、IgE抗体の産生を抑制することで、抗アレルギー作用を示す。
