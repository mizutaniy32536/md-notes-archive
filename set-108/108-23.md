# 108-23

近年、我が国において水道の水源として年間取水量が最も多いのはどれか。1つ選べ。

1. 河川水
2. 伏流水
3. ダム水
4. 湖沼水
5. 井戸水

## 正解
**1. 河川水**

## 解説

### 日本の水道水源の内訳（年間取水量）

| 水源 | 割合 |
|---|---|
| **河川水** | 約70〜80% |
| ダム水 | 約20% |
| 湖沼水 | 約5% |
| 伏流水 | 約3% |
| 井戸水 | 約2% |

### 河川水が最も多い理由

**日本の地理的特徴：**
- 山が多く、河川が豊富
- 山から海まで距離が短く、流れが速い
- 年間降水量が多い（世界平均の約2倍）

**河川水の利点：**
- 取水量が安定している
- 水質が比較的良好
- 取水しやすい

### 各水源の特徴

#### 河川水
- 最も一般的な水源
- 取水量が豊富
- 季節変動あり

#### ダム水
- 河川水を貯めたもの
- 水量調整が可能
- 統計上は河川水に含まれることも

#### 伏流水
- 河川の地下を流れる水
- 水質が良好
- 取水量は限定的

#### 湖沼水
- 湖や沼の水
- 富栄養化のリスク
- 取水量は少ない

#### 井戸水（地下水）
- 水質が安定
- 取水量は限定的
- 地盤沈下のリスク

### ポイント
日本の水道水源 = **河川水が圧倒的**（約70〜80%）
