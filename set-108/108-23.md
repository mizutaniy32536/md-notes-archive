# 108-23

**問題文**
近年、我が国において水道の水源として年間取水量が最も多いのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-23 を参照

**選択肢**
1. 河川水
2. 伏流水
3. ダム水
4. 湖沼水
5. 井戸水

**正解:** 3

**解説**
解答　３

近年の水道水源として最も多く用いられているものはダム水で、全体の約48％を占める。次いで多いものが河川水で、全体の約25％である。
（参考資料　日本水道協会：令和元年度・水道水源の現状）
