# 108-56

**問題文**
眼窩部の激痛を特徴とする頭痛はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-56 を参照

**選択肢**
1. 筋緊張性頭痛
2. 片頭痛
3. 群発頭痛
4. くも膜下出血による頭痛
5. 脳腫瘍による頭痛

**正解:** 3

**解説**
解答　３

１　誤
筋緊張性頭痛では、一般に両側性で、圧迫感、または締めつけ感を伴う頭痛を生じる。痛みは日常的な動作によって増悪せず、悪心は伴わないが、光過敏または音過敏を呈することがある。

２　誤
片頭痛では、一般に片側性で、拍動性の頭痛を生じる。痛みは日常的な動作で悪化し、随伴症状として悪心嘔吐などの消化器症状や、光過敏・音過敏などの感覚過敏を認めることが多い。

３　正
群発頭痛では、眼窩周囲または側頭部に片側性のえぐられるような激痛が生じ、頭痛と同じ側に流涙、充血、鼻漏、鼻閉などを伴うことが多い。

４　誤
くも膜下出血による頭痛は、バットで殴られたような激しい頭痛が特徴的であり、悪心・嘔吐や意識消失を伴うことが多い。

５　誤
脳腫瘍による頭痛は、腫瘍の圧迫による頭蓋内圧上昇から引き起こされる。一般的に、夜寝ている間に頭蓋内圧は高くなりやすいため、起床時に痛みが強いのが特徴的である。
