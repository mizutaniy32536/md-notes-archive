# 108-329

**問題文**
60歳女性。腎不全のため高カロリー輸液による栄養管理を実施することになった。担当薬剤師はブドウ糖含有率50%の基本輸液500 mL、脂肪乳剤100 mL、高カロリー輸液用微量元素製剤2 mL、総合ビタミン剤5 mLを用意した。この組成の輸液に加える総窒素量10 mg/mLの総合アミノ酸輸液の量として、最も近いのはどれか。1つ選べ。ただし、この患者のNPC/N比を400、脂肪乳剤100 mLに含まれる熱量を200 kcalとする。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-329 を参照

**選択肢**
1. 200 mL
2. 300 mL
3. 400 mL
4. 500 mL
5. 600 mL

**正解:** 2

**解説**
解答　２

NPC/Nは、投与アミノ酸輸液中の窒素1gあたりの非タンパク質カロリー（kcal）であり、一般的にはNPC/Nを150〜200に設定するとタンパク質合成が効率よく行われるが、本患者のように腎不全の患者の場合は300〜500を目安とする。
本患者に投与する輸液に加える総窒素量10 mg/mLの総合アミノ酸輸液の量（X mL）は、非タンパク質カロリー／窒素量（NPC/N）から算出する。

【NPC（kcal）】
ブドウ糖含有率50%の基本輸液500 mLのカロリーは1000 kcal（50 g/100 mL×500 mL×4 kcal/g）、高カロリー輸液用微量元素製剤2 mLのカロリーは、設問文より200 kcalであるため、非タンパク質カロリー（NPC）は合計1200 kcal・・・①となる。

【窒素量（g）】
総窒素量10 mg/mL（0.01 g/mL）の総合アミノ酸輸液X mL中の総窒素量は、0.01X g（0.01 g/mL×X mL）・・・②である。

【NPC/N（kcal/ g）】
NPC/Nを400にするため、①、②より以下の式が成り立つ。

NPC/N＝1200/0.01X＝400

これを解いて、輸液に加える総窒素量10 mg/mLの総合アミノ酸輸液の量（X mL）は、X＝300 mLとなる。
