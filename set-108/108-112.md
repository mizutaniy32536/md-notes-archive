# 108-112

**問題文**
図は、ミトコンドリアと、その電子伝達系をつかさどる複合体Ⅰ、Ⅱ、Ⅲ、Ⅳ並びにATP合成酵素を示した概略図である。以下の記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-112 を参照

**選択肢**
1. NADH由来の電子は、複合体Ⅰを経由し、ユビキノンへ伝達される。
2. H＋は、複合体Ⅰ、Ⅱ及びⅣにおいてマトリックスから膜間腔へ汲み出される。
3. O2は、複合体Ⅲにおいてシトクロムcから電子を受け取りH2Oとなる。
4. ATPは、H＋が膜間腔側からマトリックス側へATP合成酵素を通過する際に生成される。
5. 内膜を通過するH＋量は、NADHから電子が供給される場合の方がFADH2からの場合に比べて少ない。

**正解:** 1,4

**解説**
解答　１、４

１　正
電子伝達系において、NADH由来の電子は、複合体Ⅰを経由し、ユビキノンへ伝達される。なお、FADH2由来の電子は、複合体Ⅱを経由し、ユビキノンへ伝達される。

２　誤
電子伝達系において、H＋のマトリックスから膜間腔への汲み出しに関与する複合体は、複合体Ⅰ、Ⅲ及びⅣである。したがって、複合体ⅡはH＋の汲み出しに関与しない。

３　誤
O2は、複合体Ⅳにおいてシトクロムcから電子を受け取りH2Oとなる。

４　正
酸化的リン酸化によるATP産生は、H＋の電気化学的勾配が駆動力となり、H＋が膜間腔側からマトリックス側へATP合成酵素を通過する際に生成される。

５　誤
内膜を通過するH＋量は、NADHから電子が供給される場合の方がFADH2からの場合に比べて多い。
