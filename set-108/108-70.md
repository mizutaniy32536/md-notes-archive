# 108-70

**問題文**
全身状態の指標であるパフォーマンスステータス（PS）の値を統計処理する際の、データ尺度はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-70 を参照

**選択肢**
1. 分類尺度
2. 間隔尺度
3. 名義尺度
4. 順序尺度
5. 比例尺度

**正解:** 4

**解説**
解答　４

パフォーマンスステータス（PS）とは、患者の日常生活での動作レベルを段階的評価する全身状態の指標であり、統計処理する際のデータ尺度としては順序尺度に分類される。
