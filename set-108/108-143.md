# 108-143

# 第108回薬剤師国家試験 問143

## 問題文

医薬品の製造販売の承認拒否事由に該当するのはどれか。2つ選べ。ただし、特例承認や緊急承認などの特別な事例は除く。[1][2]

## 選択肢

1. 申請者である企業が製造販売業の許可を受けていないとき。[1][2]

2. 申請に係る医薬品と同じ作用機序のものが、すでに10剤以上承認されているとき。[1][2]

3. 申請に係る医薬品が効能又は効果を有すると認められないとき。[1][2]

4. 申請に係る医薬品が海外での承認や審査を受けていないとき。[1][2]

5. 申請に係る医薬品の製造所における製造管理又は品質管理の方法が、「医薬品の臨床試験の実施に関する基準」に適合していると認められないとき。[1][2]

## 正解

**1、3**[1][2][6]

## 解説

**選択肢1（正解）**：申請者が**製造販売業の許可を受けていない**ことは承認拒否事由に該当します。承認申請者は申請に係る品目の種類に応じた製造販売業の許可を受けていることが必須要件です。[1][4]

**選択肢2（誤り）**：同じ作用機序の医薬品が複数承認されていることは、承認拒否事由ではありません。医薬品の承認数は拒否事由にはなりません。[1]

**選択肢3（正解）**：申請した医薬品が**効能又は効果を有すると認められない**ことは承認拒否事由に該当します。効能・効果が認められないものは承認を得られません。[1][4]

**選択肢4（誤り）**：海外での承認や審査は必須要件ではなく、医薬品は国内での審査基準を満たせば承認される可能性があります。[1]

**選択肢5（誤り）**：製造管理・品質管理の基準は「医薬品の臨床試験の実施に関する基準」ではなく、医薬品製造管理及び品質管理の基準（GMP）に適合する必要があります。[1]
