



## ◆ 塩化ナトリウム水溶液のイオン強度と活量に関するシナリオ

260313

**問題**
- 87歳男性。血清ナトリウム値108 mEq/Lの低ナトリウム血症に対し、3%塩化ナトリウム水溶液での点滴加療を開始。
- 塩化ナトリウム水溶液におけるイオン強度や活量に関する記述のうち、正しいのはどれか。２つ選べ。

**選択肢**

1. イオン強度は、Na⁺やCl⁻の電荷数に依存しない。
​
2. イオン強度が高くなると、水中のNa⁺とCl⁻の間の相互作用は強くなる。
​
3. 搬送時の血清ナトリウム値と等しいNa⁺濃度の塩化ナトリウム水溶液のイオン強度は54 mmol/Lである。
​
4. 塩化ナトリウム水溶液の活量aはa＝γ・xで表される。ただし、γは活量係数、xはモル分率である。
​
5. Na⁺、Cl⁻の活量をそれぞれa⁺、a⁻とすると、塩化ナトリウム水溶液の平均活量a±はa±＝(a⁺・a⁻)/2で表される。

**正解と理由**
- `2. イオン強度が高くなると、水中のNa⁺とCl⁻の間の相互作用は強くなる。`
- `4. 塩化ナトリウム水溶液の活量aはa＝γ・xで表される。`
- **理由の構成**
	- **問題の整理**：イオン強度と活量に関する物理化学的な正誤判定
	- **薬学的知識・仕様・制約**：`イオン強度`は電解質溶液中のイオン間相互作用の強さの指標。イオン強度が高いほどイオン間相互作用は強くなる。`活量`はa = γ・x（活量係数 × モル分率）で定義される
	- **選択理由**：選択肢2はイオン強度の定義に合致。選択肢4は活量の定義式として正しい

**解法の手順（思考の流れ）**
- 選択肢1：イオン強度 I = ½Σci・zi²。`電荷数z`に依存する→誤り
- 選択肢2：イオン強度↑＝イオン間相互作用↑→正しい
- 選択肢3：Na⁺ 108 mmol/L、Cl⁻ 108 mmol/L → I = ½(108×1² + 108×1²) = `108 mmol/L`。54ではない→誤り
- 選択肢4：活量a = γ・x → 正しい定義式
- 選択肢5：平均活量a± = `(a⁺・a⁻)^(1/2)` = √(a⁺・a⁻)。(a⁺・a⁻)/2は算術平均であり誤り→誤り

**他の選択肢が誤りな理由**
- 1. イオン強度は、Na⁺やCl⁻の電荷数に依存しない。
	イオン強度の式 I = ½Σci・zi² に`電荷数zi`が含まれるため、電荷数に依存する

- 3. 搬送時の血清ナトリウム値と等しいNa⁺濃度の塩化ナトリウム水溶液のイオン強度は54 mmol/Lである。
	Na⁺ = 108 mmol/L、Cl⁻ = 108 mmol/Lのとき、I = ½(108×1² + 108×1²) = `108 mmol/L`であり、54 mmol/Lは誤り

- 5. 平均活量a±＝(a⁺・a⁻)/2で表される。
	電解質の`平均活量`は幾何平均で定義される。NaCl（1:1電解質）の場合、a± = `√(a⁺・a⁻)`であり、(a⁺・a⁻)/2（積の半分）ではない

**キーポイント**
- `イオン強度`の式：I = ½Σci・zi²。1:1電解質ではI = 濃度と等しくなる
- `活量` = `活量係数` × `モル分率`（a = γ・x）
- `平均活量`は幾何平均：a± = (a⁺^ν⁺ × a⁻^ν⁻)^(1/ν)。NaClでは√(a⁺・a⁻)

**関連知識の比較**
- `Debye-Hückel式`はイオン強度から活量係数を算出する理論式。希薄溶液で成立する
- 1:1電解質（NaCl）と2:1電解質（CaCl₂）ではイオン強度の計算が異なる。CaCl₂では電荷の2乗が効く
- `浸透圧`の計算にもイオンの数が関わるため、イオン強度の概念と関連が深い

（一言メモ）
- 1:1電解質のイオン強度は「濃度そのまま」。NaCl 108 mmol/LならI = `108 mmol/L`
- 平均活量は「`幾何平均`」。算術平均（足して割る）ではなく、掛けてルート
- `活量` = γ × x は基本中の基本。理想溶液ではγ = 1で活量 = モル分率
