



## ◆ 高齢男性の骨粗鬆症に適切な治療薬の選択

260313

**問題**
- 85歳男性。中等度認知症、高血圧、活動量低下、腰痛で骨粗鬆症と診断。血清25-ヒドロキシビタミンD 8 ng/mL（欠乏）。肝機能正常。
- この患者の骨粗鬆症治療に用いる薬剤として、適切なのはどれか。２つ選べ。

**選択肢**

1. アレンドロン酸錠
​
2. ブシラミン錠
​
3. ラロキシフェン塩酸塩錠
​
4. L-アスパラギン酸Ca錠
​
5. エルデカルシトールカプセル

**正解と理由**
- `1. アレンドロン酸錠`
- `5. エルデカルシトールカプセル`
- **理由の構成**
	- **問題の整理**：85歳男性、ビタミンD欠乏（8 ng/mL）の骨粗鬆症。適切な治療薬を2つ選ぶ
	- **薬学的知識・仕様・制約**：`アレンドロン酸`はビスホスホネート製剤で骨吸収抑制→高齢者の骨粗鬆症に第一選択。`エルデカルシトール`は活性型ビタミンD₃誘導体で、ビタミンD欠乏の本患者に適切
	- **選択理由**：骨吸収抑制＋ビタミンD補充の組合せが本症例に最適

**解法の手順（思考の流れ）**
- まず25(OH)D = 8 ng/mL → `ビタミンD欠乏`と判断
- 骨粗鬆症の治療薬として`ビスホスホネート`（アレンドロン酸）は標準治療
- ビタミンD欠乏に対し`活性型ビタミンD`（エルデカルシトール）の補充が必要
- 選択肢1と5が該当

**他の選択肢が誤りな理由**
- 2. ブシラミン錠
	`関節リウマチ`の治療薬（抗リウマチ薬）。骨粗鬆症には適応がない

- 3. ラロキシフェン塩酸塩錠
	`SERM`（選択的エストロゲン受容体モジュレーター）。`閉経後女性`の骨粗鬆症に適応があり、`男性`には不適切

- 4. L-アスパラギン酸Ca錠
	カルシウム補充剤。血清カルシウムが正常であれば積極的な補充の優先度は低い

**キーポイント**
- `ビスホスホネート`（アレンドロン酸、リセドロン酸等）は骨粗鬆症の第一選択薬
- `エルデカルシトール`は活性型ビタミンD₃の中でも骨密度増加効果が高い
- `SERM`（ラロキシフェン、バゼドキシフェン）は閉経後女性のみに適応。`男性`には禁忌ではないが適応がない

**関連知識の比較**
- 25(OH)D₃の基準値：20 ng/mL以上が正常、20未満は不足、12未満は`欠乏`
- `デノスマブ`（プラリア®）はRANKL阻害の抗体製剤で、6ヶ月に1回皮下注。高齢者にも使いやすい
- `テリパラチド`（フォルテオ®）はPTH製剤で骨形成促進。重症骨粗鬆症に使用

（一言メモ）
- `SERM`は「閉経後`女性`限定」。男性に出てきたら除外
- ビタミンD欠乏（25(OH)D < 12 ng/mL）の患者には`活性型ビタミンD`製剤の処方が妥当
- `アレンドロン酸`の服用法：起床時に十分な水で服用→30分は横にならない。食物で吸収低下
