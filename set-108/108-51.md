# 108-51

口腔内で徐々に溶解又は崩壊させ、口腔、咽頭などの局所に適用する口腔用錠剤はどれか。１つ選べ。

１ 舌下錠
２ 付着錠
３ バッカル錠
４ トローチ剤
５ ガム剤

**正解:** ４

**解説**
日本薬局方において、各剤形は下記のように規定されている。
１　誤
舌下錠は、有効成分を舌下で速やかに溶解させ、口腔粘膜から吸収させる口腔用錠剤である。
２　誤
付着錠は、口腔粘膜に付着させて用いる口腔用錠剤である。
３　誤
バッカル錠は、有効成分を臼歯と頬の間で徐々に溶解させ、口腔粘膜から吸収させる口腔用錠剤である。
４　正
５　誤
ガム剤は、咀嚼により、有効成分を放出する口腔用錠剤である。
