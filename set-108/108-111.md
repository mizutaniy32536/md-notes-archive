# 108-111

**問題文**
生体内の物質移動に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-111 を参照

**選択肢**
1. 一酸化窒素は、促進拡散により血管内皮細胞から移動する。
2. 水素イオンは、受動輸送により胃壁細胞から分泌される。
3. 原尿中のグルコースは、Na＋との共輸送により腎臓の近位尿細管上皮細胞に取り込まれる。
4. 血中のグルコースの大部分は、エンドサイトーシスにより脂肪細胞に取り込まれる。
5. ノルアドレナリンは、エキソサイトーシスにより交感神経終末から分泌される。

**正解:** 3,5

**解説**
解答　３、５

細胞膜を介した物質輸送様式は、その物質が膜を横切って輸送される膜輸送と、膜を変形させて小胞を形成することで、高分子などを輸送する膜動輸送に分類される。膜輸送は、受動拡散と能動輸送の2種類がある。次に膜輸送と膜動輸送の特徴をまとめる。

１　誤
一酸化窒素などの気体は、単純拡散により血管内皮細胞から移動する。

２　誤
水素イオンは、H＋,K＋-ATPaseを介した能動輸送により胃壁細胞より分泌される。

３　正
原尿中のグルコースは、腎臓の近位尿細管上皮細胞において、Na＋との共輸送（SGLT）を介した能動輸送により、腎臓の近位尿細管上皮細胞に取り込まれる。

４　誤
血中のグルコースの大部分は、グルコース輸送体（GLUT）を介した促進拡散により、脂肪細胞、骨格筋細胞、肝細胞などに取り込まれる。

５　正
ノルアドレナリンは、膜動輸送であるエキソサイトーシスにより、交感神経終末から分泌される。
