# 108-156

**問題文**
抗リウマチ薬に関する記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-156 を参照

**選択肢**
1. インフリキシマブは、抗ヒトTNF−α受容体モノクローナル抗体で、TNF−αの作用を抑制する。
2. アバタセプトは、抗原提示細胞表面のCD80/CD86に結合して、CD28を介した共刺激シグナルを抑制する。
3. トファシチニブは、ヤヌスキナーゼ（JAK）を阻害して、IL−2受容体の活性化を介した作用を抑制する。
4. トシリズマブは、抗ヒトIL−1βモノクローナル抗体で、IL−1βの作用を抑制する。
5. エタネルセプトは、IL−6受容体の細胞外ドメインとヒトIgGのFc鎖との融合タンパク質で、IL−6の作用を抑制する。

**正解:** 2,3

**解説**
解答　２、３

１　誤
インフリキシマブは、キメラ型抗ヒトTNF−αモノクローナル抗体であり、TNF−αに結合することで、TNF−αの作用を抑制する。

２　正
アバタセプトは、抗原提示細胞表面のCD80/CD86に結合して、CD28を介した共刺激シグナルを抑制することで、T細胞の活性化及びサイトカイン産生を抑制する。

３　正
トファシチニブは、ヤヌスキナーゼ（JAK）を阻害することで、IL−2受容体などのサイトカイン受容体の活性化を介した細胞内情報伝達を抑制する。

４　誤
トシリズマブは、ヒト化抗ヒトIL-6受容体モノクローナル抗体であり、IL-6受容体に結合することで、IL−6とIL-6受容体の結合を阻害する。なお、抗ヒトIL−1βモノクローナル抗体で、IL−1βの作用を抑制する薬物は、カナキヌマブである。

５　誤
エタネルセプトは、完全ヒト型可溶性TNFα/LTαレセプター製剤であり、TNFα及びLTαをおとり受容体として捕捉し、TNFα及びLTαと細胞表面の受容体との結合を阻害することで、抗リウマチ作用、抗炎症作用を示す。
