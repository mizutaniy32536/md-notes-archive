# 108-35

H＋,K＋−ATPaseを不可逆的に阻害し、胃酸分泌を抑制するのはどれか。１つ選べ。

１ エソメプラゾール
２ ファモチジン
３ ピレンゼピン
４ ミソプロストール
５ テプレノン

**正解:** １

**解説**
エソメプラゾールは、H＋,K＋−ATPaseを不可逆的に阻害し、胃酸分泌を抑制する消化性潰瘍治療薬である。なお、ファモチジンはヒスタミンH2受容体遮断作用、ピレンゼピンはムスカリン受容体遮断作用、ミソプロストールはプロスタノイドEP受容体刺激作用、テプレノンはプロスタグランジン生合成促進作用をもつ消化性潰瘍治療薬である。
