



## ◆ 薬物動態の非線形性に関する判断シナリオ

260313

**問題**
- 薬物動態の非線形性に関する記述のうち、正しいのはどれか。２つ選べ。

**選択肢**

1. 薬物代謝が非線形性を示す場合、全身クリアランスは投与量の増加に伴い変化する。
​
2. 血漿タンパク結合に飽和が生じた場合、薬物の全身クリアランスは低下する。
​
3. 肝臓の薬物代謝酵素に飽和が生じた場合、薬物の分布容積は増加する。
​
4. 小腸上皮細胞の薬物代謝酵素に飽和が生じた場合、薬物の全身クリアランスは低下する。
​
5. 尿細管分泌を担う担体輸送に飽和が生じた場合、薬物の尿細管分泌クリアランスは低下する。

**正解と理由**
- `1. 薬物代謝が非線形性を示す場合、全身クリアランスは投与量の増加に伴い変化する。`
- `5. 尿細管分泌を担う担体輸送に飽和が生じた場合、薬物の尿細管分泌クリアランスは低下する。`
- **理由の構成**
	- **問題の整理**：薬物動態の非線形性（飽和現象）が各パラメータに与える影響を問う
	- **薬学的知識・仕様・制約**：選択肢1は代謝飽和によりCLが投与量依存的に変化する基本原理。選択肢5は`担体輸送`の飽和により分泌速度が頭打ちとなり、分泌クリアランスが低下する
	- **選択理由**：非線形性の基本概念（Michaelis-Menten型動態）と各臓器における飽和の影響を正確に理解していれば解答可能

**解法の手順（思考の流れ）**
- まず「非線形性」＝飽和現象と把握
- 選択肢1：代謝飽和→CL低下→投与量により変化。正しいと判断
- 選択肢2：タンパク結合飽和→遊離型増加→CL「増大」が正しい。低下は誤り
- 選択肢3：代謝酵素飽和→CLは変化するがVdは直接影響しない。誤り
- 選択肢4：小腸代謝飽和→`初回通過効果`が減少→BA増大するが「全身CL」は変化しない。誤り
- 選択肢5：担体飽和→分泌速度頭打ち→分泌CL低下。正しいと判断

**他の選択肢が誤りな理由**
- 2. 血漿タンパク結合に飽和が生じた場合、薬物の全身クリアランスは低下する。
	タンパク結合飽和により`遊離型薬物`が増加し、肝臓・腎臓への分布が増大するため、全身クリアランスは「低下」ではなく`増大`する

- 3. 肝臓の薬物代謝酵素に飽和が生じた場合、薬物の分布容積は増加する。
	代謝酵素の飽和は`全身クリアランス`の低下を引き起こすが、`分布容積`（Vd）は組織分布に依存するパラメータであり、代謝飽和の直接的影響は受けない

- 4. 小腸上皮細胞の薬物代謝酵素に飽和が生じた場合、薬物の全身クリアランスは低下する。
	小腸代謝酵素の飽和は`初回通過効果`を減少させ、`バイオアベイラビリティ`を増大させる。しかし循環血中に入った後の全身クリアランスには影響しない

**キーポイント**
- `非線形性`＝Michaelis-Menten型動態。投与量の増加で酵素・担体が飽和し、パラメータが用量依存的に変化
- タンパク結合飽和→遊離型↑→CL`増大`（よくある引っかけ問題）
- 小腸代謝飽和は`BA`に影響するが`全身CL`には影響しない（初回通過効果と全身クリアランスの区別）

**関連知識の比較**
- `フェニトイン`は代表的な非線形動態を示す薬物。投与量のわずかな増加で血中濃度が急上昇する
- `プロベネシド`は尿細管分泌の`担体輸送`を阻害し、ペニシリンの腎クリアランスを低下させる（非線形性ではなく薬物相互作用）
- Michaelis-Menten式：v = Vmax × C / (Km + C)。C >> Kmで飽和（ゼロ次速度論）に移行する

（一言メモ）
- 「タンパク結合飽和→CL低下」は典型的なひっかけ。遊離型が増えるから`CLは増大`する
- 小腸と肝臓の代謝飽和で影響を受けるパラメータが違う。小腸→`BA`、肝臓→`全身CL`
- `フェニトイン`の非線形動態は`TDM`必須の理由として頻出
