# 108-71

**問題文**
薬剤師の任務は薬剤師法第1条に規定されている。この条文の 　　　 に当てはまるのはどれか。1つ選べ。「薬剤師は、調剤、医薬品の供給その他薬事衛生をつかさどることによって、 　　　 の向上及び増進に寄与し、もって国民の健康な生活を確保するものとする。」

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-71 を参照

**選択肢**
1. 医療の質
2. 地域連携
3. 公衆衛生
4. 健康サポート
5. 薬剤師の経済的利益

**正解:** 3

**解説**
解答　３

薬剤師法第1条は下記の通りである。
「薬剤師は、調剤、医薬品の供給その他薬事衛生をつかさどることによって、公衆衛生の向上及び増進に寄与し、もって国民の健康な生活を確保するものとする。」
