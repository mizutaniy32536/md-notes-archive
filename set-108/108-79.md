# 108-79

医療に携わる者の意思決定において重要とされる生命倫理の4原則に含まれないのはどれか。1つ選べ。

1. 自律尊重
2. 無危害
3. 善行
4. 医学の進歩
5. 正義

## 正解
**4. 医学の進歩**

## 解説

生命倫理の4原則（ビーチャム＆チルドレス）：
- 自律尊重：患者の自己決定権を尊重
- 無危害：患者に危害を加えない
- 善行：患者の利益を積極的に追求
- 正義：公平性・資源配分の公正さ

「医学の進歩」は4原則に含まれない。これを原則にすると人体実験等を正当化しかねない。
