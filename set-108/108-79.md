# 108-79

医療に携わる者の意思決定において重要とされる生命倫理の4原則に含まれないのはどれか。１つ選べ。

１ 自律尊重
２ 無危害
３ 善行
４ 医学の進歩
５ 正義

**正解:** ４

**解説**
生命倫理の4原則とは、1979年にトム・L・ビーチャム（T・L・Beauchamp）とジェイムズ・F・チルドレス（J・F・Childress）が提唱した原則であり、自律尊重原則、無危害原則、善行原則、正義原則の４つを合わせたものである。
