# 108-79

**問題文**
医療に携わる者の意思決定において重要とされる生命倫理の4原則に含まれないのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-79 を参照

**選択肢**
1. 自律尊重
2. 無危害
3. 善行
4. 医学の進歩
5. 正義

**正解:** 4

**解説**
解答　４

生命倫理の4原則とは、1979年にトム・L・ビーチャム（T・L・Beauchamp）とジェイムズ・F・チルドレス（J・F・Childress）が提唱した原則であり、自律尊重原則、無危害原則、善行原則、正義原則の４つを合わせたものである。
