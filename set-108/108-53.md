# 108-53

o/w型の乳剤性基剤はどれか。１つ選べ。

１ 白色軟膏
２ 親水ワセリン
３ 親水クリーム
４ 単軟膏
５ マクロゴール軟膏

**正解:** ３

**解説**
o/w型（水中油型）の乳剤性基剤とは、外相（連続相）である水の中に、内相（不連続相）である油の粒子が分散している状態の基剤のことである。o/w型（水中油型）の乳剤性基剤の代表的なものとして、親水クリームがある。なお、白色軟膏、単軟膏は油脂性基剤、マクロゴール軟膏は水溶性基剤、親水ワセリンはw/o型（油中水型）の乳剤性基剤に分類される。
