# 108-145

**問題文**
医薬品の直接の容器・被包の表示に関する記述として、正しいのはどれか。2つ選べ。ただし、省略等の特例はないものとする。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-145 を参照

**選択肢**
1. 毒薬は、黒地に白枠、白字で品名及び「毒」の文字を記載する。
2. 一般用医薬品は、リスク区分を原則として赤枠、赤字で記載する。
3. 日本薬局方に収められている医薬品は、「薬局方」の文字を記載する。
4. 専ら他の医薬品の製造の用に供される医薬品は、「製造専用」の文字を記載する。
5. 要指導医薬品は、赤字で「要指導医薬品」の文字を記載する。

**正解:** 1,4

**解説**
解答　１、４

１　正

２　誤
一般用医薬品は、その直接の容器又は直接の被包にリスク区分（第１類医薬品、第２類医薬品、第３類医薬品などの文字）を、原則として黒枠の中に黒字で記載しなければならない。

３　誤
日本薬局方に収められている医薬品は、原則として「日本薬局方」の文字を記載する必要がある。

４　正

５　誤
要指導医薬品は、原則として黒枠の中に黒字で「要指導医薬品」の文字を記載する必要がある。
