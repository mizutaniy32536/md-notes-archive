# 108-161

利尿薬に関する記述のうち、正しいのはどれか。２つ選べ。

１ トリアムテレンは、遠位尿細管と集合管のNa＋
２ トルバプタンは、集合管のバソプレシンV2
３ インダパミドは、ヘンレ係蹄上行脚のNa＋
４ エプレレノンは、遠位尿細管と集合管のミネラルコルチコイド受容体（アルドステロン受容体）を遮断することで、Na＋
５ アゾセミドは、遠位尿細管のNa＋

**正解:** ２、４

**解説**
１　誤
トリアムテレンは、遠位尿細管と集合管のNa＋チャネルを遮断し、Na＋の再吸収を抑制することで利尿作用を示す。また、このNa＋チャネル遮断により、間接的に管腔へのK＋の排出を減少させるため、カリウム保持性の利尿作用を示す。
２　正
トルバプタンは、集合管のバソプレシンV2受容体を遮断し、水の排泄を増加させることで利尿作用を示す。
３　誤
インダパミドは、遠位尿細管のNa＋−Cl－共輸送体を阻害し、水の再吸収を抑制することで利尿作用を示す。なお、ヘンレ係蹄上行脚のNa＋−K＋−2Cl－共輸送体を阻害することで、Na＋の再吸収を抑制する利尿薬は、アゾセミドなどのループ利尿薬である。
４　正
エプレレノンは、遠位尿細管と集合管のミネラルコルチコイド受容体（アルドステロン受容体）を遮断することで、アルドステロンによるNa＋−K＋交換反応を抑制し、Na＋の再吸収を抑制するとともにK＋の排泄を減少させる。
５　誤
アゾセミドは、ヘンレ係蹄上行脚のNa＋−K＋−2Cl－共輸送体を阻害し、水の再吸収を抑制することで利尿作用を示す。（解説2参照）なお、遠位尿細管のNa＋−Cl－共輸送体を阻害することで、水の再吸収を抑制する利尿薬は、チアジド系利尿薬や非チアジド系利尿薬である。
