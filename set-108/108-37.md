# 108-37

**問題文**
キサンチンオキシダーゼを阻害することで、尿酸の生合成を低下させるのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-37 を参照

**選択肢**
1. ブコローム
2. ベンズブロマロン
3. ドチヌラド
4. トピロキソスタット
5. ラスブリカーゼ

**正解:** 4

**解説**
解答　４

トピロキソスタットは、キサンチンオキシダーゼを選択的に阻害し、尿酸の生合成を低下させることで、血中尿酸値を低下させる。なお、ブコローム、ベンズブロマロン、ドチヌラドは尿酸排泄促進作用、ラスブリカーゼは尿酸分解作用により血中尿酸値を低下させる。
