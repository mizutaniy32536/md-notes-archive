**正解と理由**
- `5. ホスファチジルセリンの構造`
- **理由の構成**
	- ホスファチジルセリン = グリセロリン脂質（グリセロール＋脂肪酸2本＋リン酸セリン）
	- アポトーシス時に膜の内葉→外葉へ翻転 → 「eat-meシグナル」

**解法の手順（思考の流れ）**
- 選択肢1：コレステロール = ステロール骨格（4環構造）→ リン脂質ではない → ✗
- 選択肢2：トリアシルグリセロール = 中性脂肪（リン酸基なし）→ ✗
- 選択肢3：リノール酸 = 不飽和脂肪酸（炭素鎖のみ）→ ✗
- 選択肢4：ホスファチジルコリン = 頭基が`コリン`（N(CH₃)₃）→ セリンではない → ✗
- 選択肢5：ホスファチジルセリン = 頭基が`セリン`（アミノ酸）→ **○**

**キーポイント**
- `ホスファチジルセリン(PS)` = グリセロール + 脂肪酸×2 + リン酸 + セリン
- 通常は膜の`内葉`に存在 → アポトーシスで`外葉`へ翻転
- マクロファージがPSを認識して貪食（eat-meシグナル）

**関連知識の比較**

| 脂質 | 構造的特徴 |
|---|---|
| ホスファチジルセリン | グリセロリン脂質、頭基=セリン |
| ホスファチジルコリン | グリセロリン脂質、頭基=コリン |
| コレステロール | ステロール（4環構造） |
| トリアシルグリセロール | 中性脂肪（リン酸基なし） |

（一言メモ）
- PS外葉翻転 = アポトーシスのマーカー。Annexin Vアッセイの原理
