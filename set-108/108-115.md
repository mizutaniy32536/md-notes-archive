# 108-115

**問題文**
原核細胞の遺伝子発現における転写と翻訳に関する記述として、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-115 を参照

**選択肢**
1. 遺伝子の転写と翻訳は同時進行が可能であり、転写が完結する前に翻訳が開始される。
2. 翻訳開始コドンの直前には、シャイン・ダルガーノ（SD）配列が存在し、リボソームの60Sサブユニットが結合する。
3. 翻訳は、mRNAの3´末端側から5´末端側の方向に進行し、ペプチド鎖はC末端側からN末端側へと合成される。
4. アミノアシルtRNAは、コドンを含むtRNAの3’末端に、mRNA上のアンチコドンに対応したアミノ酸が結合している。
5. オペロンとして隣り合う一群の遺伝子は1本のmRNA（ポリシストロニックmRNA）に転写され、複数のタンパク質に翻訳される。

**正解:** 1,5

**解説**
解答　１、５

１　正
原核細胞では、転写及び翻訳はどちらも細胞質で行われ、転写が完結する前に翻訳が開始されるため、同時進行が可能である。一方、真核細胞では、転写は核内で行われ、転写により生成した各種RNAはプロセシングを受けた後、細胞質へ移行しリボソームによる翻訳が開始されため、転写と翻訳の同時進行ができない。

２　誤
翻訳開始コドンの直前には、シャイン・ダルガーノ（SD）配列が存在し、リボソームの30Sサブユニットが結合する。 

３　誤
翻訳は、mRNAの5´末端側から3´末端側の方向に進行し、ペプチド鎖はN末端側からC末端側へと合成される。

４　誤
アミノアシルtRNAは、アンチコドンを含むtRNAの3’末端に、mRNA上のコドンに対応したアミノ酸が結合している。

５　正
原核細胞の場合、一連の代謝過程に関わる複数の遺伝子が一つのプロモーターを起点として転写され、一つのmRNAに複数のタンパク質情報がコードされていることがある。このような転写単位をオペロンという。また、複数のタンパク質をコードするmRNAをポリシストロニックmRNA（ポリシストロン性mRNA）という。
