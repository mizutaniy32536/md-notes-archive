# 108-227

（問226-227連問）

問226と同じ図には、リチウム製剤とラモトリギン製剤について、服用量別の相対危険度（RR）と95%信頼区間も示されている。

**追加データ：**
- リチウム製剤 >900 mg/日：RR ≈ 4.2（95%CI: 1.9-7.0）
- ラモトリギン製剤：RR ≈ 1.4（95%CI: 0.7-2.7）

**問227：** この図を基に、医師に説明する内容として、**誤っているもの**はどれか。**1つ選べ。**

1. リチウム製剤を900 mg/日超で服用した群では、出生100人あたり約6.32人に心奇形が発生している。
2. リチウム製剤を900 mg/日超で服用した群は、薬剤非服用群に比べて心奇形の発生が約4.2倍である。
3. リチウム製剤は、服用量の増加に伴って心奇形の発生頻度が増加する傾向がある。
4. ラモトリギン製剤は、リチウム製剤と比較して、統計学的に有意に心奇形の発生頻度が低い。
5. この研究は、リチウム製剤の催奇形性を評価する上で重要なエビデンスとなる。

## 正解
**4. ラモトリギン製剤は、リチウム製剤と比較して、統計学的に有意に心奇形の発生頻度が低い → 誤り**

## 解説

### 95%信頼区間（CI）と統計学的有意性

**95%信頼区間とは：**
- 真の値が95%の確率でこの範囲にあるという統計的推定
- 相対危険度（RR）の信頼区間が**「1」を含むかどうか**で統計学的有意性を判断

**相対危険度の意味：**
- **RR = 1** → リスクに差がない
- **RR > 1** → リスク増加
- **RR < 1** → リスク減少

**信頼区間と統計学的有意性：**

| 95%信頼区間 | 統計学的有意性 | 意味 |
|---|---|---|
| **1を含まない** | 有意（p < 0.05） | リスクに差があると言える |
| **1を含む** | 有意でない（p ≥ 0.05） | リスクに差があるとは言えない |

### 各薬剤のデータ解釈

#### リチウム製剤 >900 mg/日
- RR = 4.2
- 95%CI: **1.9-7.0**
- 信頼区間は1を含まない（1.9～7.0、全て1より大きい）
- **→ 統計学的に有意にリスク増加**

#### ラモトリギン製剤
- RR = 1.4
- 95%CI: **0.7-2.7**
- 信頼区間は1を含む（0.7～2.7、1を挟んでいる）
- **→ 統計学的に有意なリスク増加とは言えない**

---

### 各選択肢の検証

#### 選択肢1 ⭕ 正しい
「リチウム製剤を900 mg/日超で服用した群では、出生100人あたり約6.32人に心奇形が発生している。」

- 図のデータから読み取れる実測値
- 非服用群（1.5%）と比べて約4.2倍 → 1.5% × 4.2 = 6.3% ≈ 出生100人あたり6.32人

#### 選択肢2 ⭕ 正しい
「リチウム製剤を900 mg/日超で服用した群は、薬剤非服用群に比べて心奇形の発生が約4.2倍である。」

- RR = 4.2で正しい
- 非服用群を基準とした相対危険度

#### 選択肢3 ⭕ 正しい
「リチウム製剤は、服用量の増加に伴って心奇形の発生頻度が増加する傾向がある。」

- 用量依存性（dose-response relationship）が認められる
- これは因果関係を支持する重要なエビデンス

#### 選択肢4 ❌ 誤り
「ラモトリギン製剤は、リチウム製剤と比較して、統計学的に有意に心奇形の発生頻度が低い。」

**誤りのポイント①：ラモトリギンは統計学的に有意ではない**
- ラモトリギンのRR = 1.4（95%CI: 0.7-2.7）
- 信頼区間が「1」を含む
- **→ 薬剤非服用群と比べて有意な差があるとは言えない**

**誤りのポイント②：直接比較していない**
- この研究の比較：
  - リチウム vs 非服用群
  - ラモトリギン vs 非服用群
- **リチウム vs ラモトリギンの直接比較（head-to-head comparison）はしていない**
- だから「リチウム製剤と比較して」とは言えない

**正しい解釈：**
- ラモトリギンは非服用群と比べて統計学的に有意なリスク増加は認められなかった
- リチウムとの直接比較は行われていない

#### 選択肢5 ⭕ 正しい
「この研究は、リチウム製剤の催奇形性を評価する上で重要なエビデンスとなる。」

- 妊娠中のリチウム服用と心奇形の関連を示す疫学研究
- 用量依存性も確認
- 臨床判断の重要なエビデンスとなる

---

### 補足：研究デザインの理解

**この研究の構造：**
```
基準（対照群）: 薬剤非服用群
    ↓
比較群A: リチウム製剤服用群（用量別）
比較群B: ラモトリギン製剤服用群
```

**できること：**
- リチウム vs 非服用群の比較 ✅
- ラモトリギン vs 非服用群の比較 ✅

**できないこと：**
- リチウム vs ラモトリギンの直接比較 ❌

**直接比較をするには：**
- 無作為化比較試験（RCT）
- または適切に調整された間接比較

---

### 臨床への応用

**妊娠中の双極性障害治療：**
1. リスクとベネフィットの評価が必要
2. リチウムは催奇形性リスクあり（用量依存性）
3. ラモトリギンは統計学的に有意なリスク増加は認められていない
4. ただし、各症例で個別に判断が必要

**説明のポイント：**
- 絶対リスクと相対リスクの両方を説明
- 治療を中止した場合の母体のリスクも考慮
- インフォームドコンセントの重要性
