## ◆ 薬剤服用量別の相対危険度グラフから統計的有意性を判断するシナリオ

260311

**問題**
精神神経科の医師から医薬品情報室に、妊婦がリチウム製剤とラモトリギン製剤を服用したときの胎児における心奇形の発生について質問があった。薬剤師が論文を検索し、各製剤の服用による胎児の心奇形発生に関するコホート研究報告を見つけた。論文には下表に示した結果が掲載されていた。
論文には、薬剤非服用群に対するリチウム製剤又はラモトリギン製剤服用群の相対危険度（傾向スコア分析により補正した値）を服用量別に示した図が載っていた。妊娠中の薬剤服用状況と胎児の心奇形発生に関し、下図を基に薬剤師が医師に説明する内容として誤っているのはどれか。1つ選べ。

※図（服用量別の相対危険度と95%信頼区間を示したグラフ）は原本を参照
（参考値：薬剤非服用群1.50、リチウム >900mg/日で6.32。95%信頼区間はすべて1.0をまたいでいる）

**選択肢**

1. リチウム製剤 >900mg/日服用群では心奇形の発生率が高い。
​
2. リチウム製剤 >900mg/日服用群では薬剤非服用群の約4.2倍である。
​
3. リチウム製剤服用量の増加に伴い心奇形の発生率は増加傾向にある。
​
4. リチウム製剤の服用は心奇形発生の有意な危険因子である。
​
5. ラモトリギン製剤2000人服用群では心奇形が2.5人発生した。

**正解と理由**
- `4. リチウム製剤の服用は心奇形発生の有意な危険因子である。`
- **理由の構成**
	- **問題の整理**：服用量別の`相対危険度`と`95%信頼区間`を示した図から、「誤っている」説明を選ぶ
	- **薬学的知識・仕様・制約**：`95%信頼区間`が1.0をまたいでいる場合、統計学的に有意ではない。相対危険度の点推定値が高くても、信頼区間が1をまたげば「有意な危険因子」とは言えない
	- **選択理由**：図のリチウム製剤全用量群の`95%信頼区間`がいずれも1.0をまたいでおり、統計学的有意差が示されていない。よって「有意な危険因子である」は誤り

**解法の手順（思考の流れ）**
- 「誤っている」選択肢を選ぶ問題と確認
- 選択肢1：>900mg/日で相対危険度6.32 → 「発生率が高い」は数値的事実 → 正しい
- 選択肢2：6.32÷1.50≈4.2倍 → 計算通り → 正しい
- 選択肢3：用量増加に伴い相対危険度が増加傾向 → 図から読み取れる → 正しい
- 選択肢4：「有意な危険因子」→ `95%信頼区間`が1.0をまたいでいる → **統計的に有意ではない** → 誤り
- 選択肢5：2000人中の発生数は計算通り → 正しい
- 「誤り」は`4`

**他の選択肢が誤りな理由（本問では「正しい選択肢」）**
- 1. リチウム製剤 >900mg/日服用群では心奇形の発生率が高い。
	相対危険度6.32は数値として高い。「統計学的に有意」とは述べておらず、単に「発生率が高い」事実を述べているため正しい

- 2. リチウム製剤 >900mg/日服用群では薬剤非服用群の約4.2倍である。
	6.32÷1.50≈4.21。計算上約4.2倍は正しい

- 3. リチウム製剤服用量の増加に伴い心奇形の発生率は増加傾向にある。
	図から用量依存的に相対危険度が増加する傾向が読み取れるため正しい

- 5. ラモトリギン製剤2000人服用群では心奇形が2.5人発生した。
	表の数値に基づく計算として正しい

**キーポイント**
- `相対危険度`の点推定値が高くても、`95%信頼区間`が1.0をまたげば「統計学的に有意」ではない
- 「発生率が高い」（数値的事実）と「有意な危険因子」（統計的結論）は別概念
- サンプルサイズが小さいと信頼区間が広くなり、有意差が出にくい

**関連知識の比較**
- `傾向スコア分析`：観察研究で交絡因子の影響を補正する手法。RCTの代替として用いる
- `95%信頼区間`が1をまたぐ → RR=1（関連なし）を含む → 有意差なし
- `p値`<0.05 ←→ 95%信頼区間が1をまたがない、は同じ意味
- `リチウム`は治療域が狭い（0.6-1.2mEq/L）ため`TDM`対象

（一言メモ）
- 「信頼区間が1をまたぐか」の判断は表・図読解問題の超頻出テクニック。まずここを確認する習慣をつける
- 点推定値が大きくても信頼区間が広い＝「有意ではない」。これを理解しないとひっかかる
- 問226と問227は連問。226で相対危険度を計算し、227で信頼区間の解釈を問う構成
