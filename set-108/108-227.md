# 108-227

### 第108回薬剤師国家試験 問227

#### 問題文（実務）
精神神経科の医師から医薬品情報室に、妊婦が**リチウム製剤**と**ラモトリギン製剤**を服用したときの胎児における心奇形の発生について質問があった。薬剤師が論文を検索し、各製剤の服用による胎児の心奇形発生に関するコホート研究報告を見つけた。論文には下表に示した結果が掲載されていた。

（表：妊娠中の薬剤服用状況と胎児の心奇形発生。薬剤非服用群、**リチウム製剤**服用群、ラモトリギン製剤服用群の出生数と心奇形発生数が記載。）

論文には、薬剤非服用群に対する**リチウム製剤**又は**ラモトリギン製剤**服用群の**相対危険度**（傾向スコア分析により補正した値）を服用量別に示した図が載っていた。妊娠中の薬剤服用状況と胎児の心奇形発生に関し、下図を基に薬剤師が医師に説明する内容として**誤っている**のはどれか。**1つ選べ**。[3][7]

#### 全選択肢（番号付き）
1. リチウム製剤服用量が1日**900 mg**を超える群の出生**100人あたりの心奇形発生数は6.32人**である。[3]
2. 薬剤非服用群の**1.50**と、リチウム製剤服用量が1日**900 mg**を超える群の**6.32**をまとめて**「約4.2倍」**と表現できる。[1]
3. **リチウム製剤**服用群において、服用量が増加するにつれ心奇形の**発生数が増加傾向**にある。[1]
4. **リチウム製剤**服用群の相対危険度（**95%信頼区間**）は、いずれも**1.0をまたいでおり統計学的に有意**である。[1]
5. リチウム製剤服用量が1日**900 mg**を超える群では、**2000人中50人**に心奇形が発生しているため、**100人あたり2.5人**である。[1]

#### 正解番号
**4**[1][3][5]

#### 解説
この問題は、論文の図（服用量別相対危険度、95%信頼区間付き）を基にした医師への説明内容のうち、誤りを問う。選択肢1はリチウム製剤>900mg/日の発生率6.32人を正しく述べ妥当。選択肢2は非服用群1.50に対し6.32で約4.2倍（6.32/1.50≈4.21）と計算可能で妥当。選択肢3は服用量増加に伴う発生数の増加傾向が図から読み取れ妥当。**選択肢4は誤り**で、相対危険度の95%信頼区間がすべて1.0をまたいでいるため、統計学的に有意差なし（P>0.05）。選択肢5は2000人中50人で100人あたり2.5人（50/2000×100=2.5）と正しく妥当。正解は誤りの**4**。[1][3]
