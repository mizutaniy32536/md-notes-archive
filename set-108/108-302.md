## ◆ ストレス下の58歳女性が帯状疱疹を発症したシナリオ

260312

**問題**
58歳女性。管理職。ストレスが溜まっていた。背中にズキズキ痛む赤い連なった丘疹が出現。
この患者の疾患と治療薬に関する記述として、正しいのはどれか。2つ選べ。

**選択肢**

1. ストレスによる免疫の低下は発症の危険因子である。
​
2. 背中の皮疹は両側性に広がった。
​
3. プレガバリンが抗炎症作用を示すと期待される。
​
4. 原因となるウイルスは皮膚内に潜伏していた。
​
5. 痛みが強い場合には、アセトアミノフェンなどの鎮痛薬を併用してもよい。

**正解と理由**
- `1. ストレスによる免疫の低下は発症の危険因子である。`
- `5. 痛みが強い場合には、アセトアミノフェンなどの鎮痛薬を併用してもよい。`
- **理由の構成**
	- **問題の整理**：`帯状疱疹`の発症機序・皮疹の特徴・治療薬を正確に理解する
	- **薬学的知識・仕様・制約**：
		- `帯状疱疹`=`水痘帯状疱疹ウイルス`（VZV）の再活性化。免疫低下が発症の引き金
		- 疼痛管理に`アセトアミノフェン`や`NSAIDs`の併用は適切
	- **選択理由**：1（免疫低下が危険因子）と5（鎮痛薬併用は適切）が正しい

**解法の手順（思考の流れ）**
- 選択肢1：ストレス→免疫低下→VZV再活性化→帯状疱疹 → 正しい
- 選択肢2：帯状疱疹の皮疹は`片側性`（デルマトーム沿い）。両側性ではない → 誤り
- 選択肢3：`プレガバリン`=`神経障害性疼痛治療薬`（Ca²⁺チャネルα₂δ阻害）。抗炎症作用はない → 誤り
- 選択肢4：VZVが潜伏するのは`神経節`（脊髄後根神経節等）。皮膚内ではない → 誤り
- 選択肢5：帯状疱疹の疼痛に鎮痛薬併用は推奨される → 正しい

**他の選択肢が誤りな理由**
- 2. 帯状疱疹は`片側性`。神経の支配領域（デルマトーム）に沿って出現
- 3. プレガバリンは`帯状疱疹後神経痛`（PHN）に使われるが、作用機序は`抗炎症`ではなく`神経障害性疼痛抑制`
- 4. VZVの潜伏部位は`神経節`（特に脊髄後根神経節・三叉神経節）。皮膚内ではない

**キーポイント**
- `帯状疱疹`：VZV再活性化、片側性の水疱・疼痛。デルマトーム沿い
- VZV潜伏部位：`神経節`（HSVも神経節に潜伏）
- 治療：`アシクロビル`・`バラシクロビル`（抗ウイルス薬）＋鎮痛薬

**関連知識の比較**
- `帯状疱疹後神経痛`（PHN）：皮疹治癒後も持続する神経痛。プレガバリン、アミトリプチリン等で治療
- `帯状疱疹ワクチン`：50歳以上に推奨。生ワクチン（ゾスタバックス）と不活化ワクチン（シングリックス）

（一言メモ）
- 「片側性」は帯状疱疹の最大の特徴。「両側性」と出たら即×
- VZV=`神経節`に潜伏、`皮膚`ではない。この潜伏部位のひっかけは超頻出
