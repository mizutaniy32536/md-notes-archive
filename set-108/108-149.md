# 108-149

我が国の薬価基準制度の記述に関して、正しいのはどれか。2つ選べ。

1. 医療用医薬品として承認された医薬品であっても、薬価基準に収載されていないものがある。
2. 新医薬品の薬価算定は、原価計算方式を原則とする。
3. 薬価基準に定められている薬価は、国際的な統一ルールに基づいて決定される公定価格である。
4. 新医薬品の薬価算定において、医薬品の有用性や市場性の観点で加算される制度がある。
5. 新医薬品の薬価は、一度決まると後発医薬品が薬価基準に収載されるまで維持される。

## 正解
**1, 4**

## 解説

| 選択肢 | 正誤 | 理由 |
|---|---|---|
| 1. 承認済みでも薬価未収載あり | ⭕ | 承認≠薬価収載。自費診療薬など薬価基準に載っていない医薬品が存在する |
| 2. 原価計算方式が原則 | ❌ | 原則は「類似薬効比較方式」。原価計算方式は比較薬がない場合の例外的方式 |
| 3. 国際的統一ルールで決定 | ❌ | 薬価は各国が独自に決定。国際統一ルールは存在しない |
| 4. 有用性・市場性の加算制度あり | ⭕ | 画期性加算、有用性加算、市場性加算などの補正加算制度がある |
| 5. 後発品収載まで薬価維持 | ❌ | 薬価改定は毎年実施。後発品の有無に関係なく見直される |

### 薬価算定方式まとめ
| 方式 | 条件 | 内容 |
|---|---|---|
| 類似薬効比較方式（原則） | 類似薬あり | 既存の類似薬の薬価を参考に算定 |
| 原価計算方式（例外） | 類似薬なし | 原材料費等を積み上げて算定 |
