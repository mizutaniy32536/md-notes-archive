# 108-158

### 第108回薬剤師国家試験 問158

#### 問題文
高血圧症治療薬に関する記述のうち、正しいのはどれか。**2つ選べ**。[1][5]

#### 選択肢
1. アリスキレンは、レニンを阻害することで、アンジオテンシンⅠの産生を抑制する。[1]  
2. アテノロールは、α<sub>1</sub>遮断薬である。[1]  
3. シルニジピンは、L/N型カルシウム拮抗薬である。[1]  
（注: 検索結果から完全な5選択肢が断片的だが、標準的な問題形式に基づき、上記3つが主に言及。残りは類似問題の文脈から推定されるが、解説焦点はこれら。[1][2]）

#### 正解番号
**1 と 3**[1][2]

#### 解説
- **選択肢1（正）**: アリスキレンは直接レニン阻害薬（DRI）であり、レニン酵素を阻害してアンジオテンシンⅠの産生を抑制する。これは103回問156の関連知識。[1]  
- **選択肢2（誤）**: アテノロールは選択的β<sub>1</sub>アドレナリン受容体遮断薬であり、α<sub>1</sub>遮断薬ではない。[1][2]  
- **選択肢3（正）**: シルニジピンはL型/N型二重阻害型のカルシウム拮抗薬（Ca拮抗薬）である。[1][2]  

問157の文脈（50歳男性の高血圧患者、BMI28、家庭血圧140/90mmHg台）で薬物療法開始時、正解は問157の**3と5**（白衣/仮面高血圧否定不可、単剤低用量開始）だが、問158は独立した薬理記述。[2][5]
