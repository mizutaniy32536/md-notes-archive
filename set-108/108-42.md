## ◆ 胃内容排出速度を低下させる薬物を選ぶシナリオ

260308

**問題**
- 胃内容排出速度を低下させる薬物はどれか。1つ選べ。

**選択肢**

1. リボフラビン
​
2. プロパンテリン
​
3. メトクロプラミド
​
4. ドンペリドン
​
5. モサプリド

**正解と理由**
- `2. プロパンテリン`
- **理由の構成**
	- **問題の整理**：胃内容排出速度を「低下」させる薬物を選ぶ問題
	- **薬学的知識・仕様・制約**：`プロパンテリン`は`抗コリン薬`（ムスカリンM受容体遮断薬）であり、消化管平滑筋の運動を抑制して`胃内容排出速度を低下`させる。薬物動態研究において胃排出速度を遅延させる標準薬として使用される
	- **選択理由**：選択肢の中で消化管運動を抑制する薬はプロパンテリンのみであり、他はすべて促進する薬物であるため

**解法の手順（思考の流れ）**
- まず「低下させる」= 消化管運動を抑制する薬を探す
- 選択肢3・4・5（`メトクロプラミド`、`ドンペリドン`、`モサプリド`）はいずれも`消化管運動促進薬`と即座に想起
- `リボフラビン`はビタミンB₂であり、消化管運動への直接作用は持たない
- 残る`プロパンテリン`が`抗コリン薬`であり、消化管運動抑制作用を持つと結論づけた

**他の選択肢が誤りな理由**
- 1. リボフラビン
	`ビタミンB₂`であり、消化管運動を直接変化させる薬理作用は持たない。吸収にはRFVTが関与するが、胃排出速度への影響はない

- 3. メトクロプラミド
	`ドパミンD₂受容体遮断薬`であり、消化管運動を`促進`する。胃排出速度を`上昇`させるため、条件の逆。制吐薬としても使用される

- 4. ドンペリドン
	`ドパミンD₂受容体遮断薬`であり、消化管運動を`促進`する。メトクロプラミドと類似作用だが、`血液脳関門`を通過しにくいため中枢性副作用が少ない

- 5. モサプリド
	`セロトニン5-HT₄受容体作動薬`であり、アセチルコリン放出を促進して消化管運動を`促進`する。胃排出速度を上昇させる

**キーポイント**
- 胃排出速度を「低下」→ `抗コリン薬`（プロパンテリン、ブチルスコポラミンなど）
- 胃排出速度を「上昇」→ `消化管運動促進薬`（メトクロプラミド、ドンペリドン、モサプリド）
- `プロパンテリン`は薬物動態研究における胃排出遅延の標準薬として頻出

**関連知識の比較**
- 胃排出速度↓ → 小腸到達が遅れる → 多くの薬物で`Cmax低下`、`Tmax延長`。ただしAUCは変わらないことも多い
- 胃排出速度↑ → 小腸到達が速くなる → `Cmax上昇`、`Tmax短縮`
- `メトクロプラミド`はD₂遮断に加え`5-HT₃遮断`、`5-HT₄作動`の多重作用を持つ
- 消化管運動促進薬の副作用：メトクロプラミドは`BBB通過`→ `錐体外路症状`のリスクあり。ドンペリドンはBBB通過しにくいが`高プロラクチン血症`のリスク

（一言メモ）
- 「胃排出↓」と聞いたら`抗コリン薬`、「胃排出↑」と聞いたら`D₂遮断薬`か`5-HT₄作動薬`をまず想起する
- `プロパンテリン`は薬物動態の問題で「胃排出を遅らせる実験条件」として登場する定番薬
- 3・4・5はすべて「消化管運動促進」で、これを知っていれば消去法で2に絞れる
