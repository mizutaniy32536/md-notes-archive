# 108-42

**問題文**
胃内容排出速度を低下させる薬物はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-42 を参照

**選択肢**
1. リボフラビン
2. プロパンテリン
3. メトクロプラミド
4. ドンペリドン
5. モサプリド

**正解:** 2

**解説**
解答　２

胃内容排出速度（GER）を低下させる薬物は、抗コリン薬のプロパンテリンである。なお、メトクロプラミド、ドンペリドン、モサプリドは胃内容排出速度を上昇させる薬物である。
また、リボフラビンは胃内容排出速度の影響を受け、吸収量が変化する薬物である。リボフラビンは十二指腸付近のトランスポーターにより緩徐に吸収されるため、胃内容排出速度を上昇させる薬物と併用すると、トランスポーターが飽和を起こし、リボフラビンの吸収量が低下する。
