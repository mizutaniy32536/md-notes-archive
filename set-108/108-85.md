# 108-85

**問題文**
放射性医薬品の調製と保管における注意事項として、誤っているのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-85 を参照

**選択肢**
1. 調製作業はクリーンベンチ内で行う。
2. 飛散防止のためバイアル内を陰圧に保つ。
3. 汚染に備え手袋等の防護具を着用する。
4. 調製作業は放射線管理区域内で行う。
5. 放射線を安全に遮へいできる貯蔵箱に保管する。

**正解:** 1

**解説**
解答　１

１　誤っている
調整作業は、微生物等の汚染及び放射性物質による被ばく防止のため安全キャビネット内で行う。

２　正しい
バイアル内が陽圧になると、内容物が噴出してしまい放射線汚染や被曝の恐れがあるため、飛散防止のためバイアル内を陰圧に保つ。

３　正しい
放射性医薬品による汚染に備え、手袋・作業衣・腕カバー・足の甲が隠れる履物などの装備を身につける必要がある。

４　正しい
放射性医薬品は、放射線を放出する特殊性から、医療法における放射線管理区域内で厳格に管理・使用することが定められており、調製作業は放射線管理区域内で行う。

５　正しい
放射性医薬品は、鉛容器等の放射線を安全に遮へいできる貯蔵箱に保管する。
