# 108-332

「ハイリスク薬」とその医薬品が引き起こし得る「重大な健康被害」の組合せとして、正しいのはどれか。１つ選べ。

**正解:** ２

**解説**
１　誤
シクロスポリンの重大な健康被害として、腎障害、肝障害などがあるが、低血糖は報告されていない。
２　正
ジゴキシンの重大な健康被害として、不整脈、ジギタリス中毒や非閉塞性腸間膜虚血などがある。
３　誤
フェノバルビタールの重大な健康被害として、中毒性表皮壊死融解症、皮膚粘膜眼症候群、顆粒球減少などがあるが、網膜障害は報告されていない。
４　誤
ワルファリンの重大な健康被害として、出血、カルシフィラキシス（皮膚潰瘍）などがあるが、急性膵炎の報告はされていない。
５　誤
ソタロールの重大な健康被害として、心室細動、完全房室ブロック、心不全などがあるが、排尿困難は報告されていない。
