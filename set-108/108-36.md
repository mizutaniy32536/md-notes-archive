# 108-36

**問題文**
ロペラミドの止瀉作用に関わる作用点はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-36 を参照

**選択肢**
1. ドパミンD2受容体
2. オピオイドµ 受容体
3. アセチルコリンM1受容体
4. セロトニン5−HT3受容体
5. ヒスタミンH2受容体

**正解:** 2

**解説**
解答　２

ロペラミドは、腸壁内のオピオイドµ 受容体を刺激しアセチルコリンの放出を抑制することで、腸の蠕動運動を抑制による止瀉作用を示す。
