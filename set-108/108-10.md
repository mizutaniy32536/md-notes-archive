



## ◆ マグネシウムの化学的性質に関するシナリオ

260313

**問題**
- マグネシウムに関する記述のうち、誤っているのはどれか。１つ選べ。

**選択肢**

1. マグネシウムイオンは、カルシウムイオンに比べてイオン半径が小さい。
​
2. マグネシウムイオンは、ルイス酸としてATPのリン酸基と結合する。
​
3. 空気中で金属マグネシウムが燃焼すると、酸化マグネシウムとなる。
​
4. マグネシウムは、典型元素である。
​
5. 酸化マグネシウムは、酸性酸化物である。

**正解と理由**
- `5. 酸化マグネシウムは、酸性酸化物である。`
- **理由の構成**
	- **問題の整理**：マグネシウムに関する5つの記述から「誤っている」ものを選ぶ
	- **薬学的知識・仕様・制約**：`酸化マグネシウム`（MgO）は金属元素の酸化物であり、水と反応して塩基（Mg(OH)₂）を生じるため`塩基性酸化物`に分類される。酸性酸化物はCO₂やSO₃などの非金属元素の酸化物
	- **選択理由**：「酸性酸化物」という記述が誤りであり、正しくは`塩基性酸化物`

**解法の手順（思考の流れ）**
- まず各選択肢の正誤を順に確認
- 選択肢1〜4は正しい記述であることを確認
- 選択肢5で「`酸性酸化物`」に違和感を持つ。MgOは金属酸化物であり`塩基性酸化物`と判断
- これが唯一の誤りと結論づけた

**他の選択肢が誤りな理由**
- 1. マグネシウムイオンは、カルシウムイオンに比べてイオン半径が小さい。
	正しい。`Mg²⁺`の最外殻はL殻、`Ca²⁺`の最外殻はM殻であるため、Mg²⁺の方がイオン半径が小さい

- 2. マグネシウムイオンは、ルイス酸としてATPのリン酸基と結合する。
	正しい。`Mg²⁺`は空軌道を有するため`ルイス酸`として働き、ATPのリン酸基の酸素原子と配位結合する

- 3. 空気中で金属マグネシウムが燃焼すると、酸化マグネシウムとなる。
	正しい。2Mg + O₂ → 2MgOの反応が進行し、白色の`酸化マグネシウム`を生成する

- 4. マグネシウムは、典型元素である。
	正しい。Mgは2族元素であり、`典型元素`（1、2族及び12〜18族）に分類される

**キーポイント**
- 金属酸化物＝`塩基性酸化物`、非金属酸化物＝`酸性酸化物`が基本原則
- `Mg²⁺`はATP依存性酵素の多くで必須の`補因子`として機能する
- `典型元素`と`遷移元素`の分類：1、2族＋12〜18族が典型元素、3〜11族が遷移元素

**関連知識の比較**
- `酸化マグネシウム`は臨床的に制酸薬・緩下薬として使用される。胃酸を中和し、腸内で水分を保持して便を軟化させる
- `両性酸化物`は酸にも塩基にも溶ける酸化物で、`Al₂O₃`（酸化アルミニウム）や`ZnO`（酸化亜鉛）が代表例
- `ルイス酸`の概念は電子対受容体であり、`Mg²⁺`の他に`Fe³⁺`や`Zn²⁺`なども生体内でルイス酸として機能する

（一言メモ）
- 酸化物の分類は「金属＝`塩基性`、非金属＝`酸性`」が大原則。Al、Znなどの`両性酸化物`は例外として覚える
- `酸化マグネシウム`は制酸薬かつ`緩下薬`として臨床で頻用。腎機能低下時は`高Mg血症`に注意
- `ATP`は生体内で常にMg²⁺と複合体（`Mg-ATP`）として存在し、多くの酵素反応に関与する
