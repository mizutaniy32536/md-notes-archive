# 108-311

### 第108回薬剤師国家試験 問311

#### 問題文
75歳男性。糖尿病及び糖尿病性腎症で在宅療養にて腹膜透析を行い、薬物治療中である。今回、処方1及び処方2が追加された処方箋が発行された。過去の薬歴から処方1及び処方2は初めて処方されたことがわかった。薬剤師が、調剤した薬剤を持って患者宅を訪問した。

患者より、**処方1**の薬のラベルに記載されている「**生物**」について質問があった。薬剤師の説明の内容として、正しいのはどれか。**1つ選べ**。[1][2][3]

#### 全選択肢
1. 感染症のリスクがあるため、処方1の薬の使用に際し、患者からの文書による同意が必要となる。[2][3]  
2. 処方1の薬は、植物由来である。[1][2][3]  
3. 処方1の薬は、疾病の治療を目的として、人の細胞に導入され、体内で発現する遺伝子を含有するものである。[2][3]  
4. 処方1の薬は、製薬企業に感染症定期報告を義務付けるなど感染症に関する対策が強化されている。[2]

#### 正解番号
**1** [1][2][3]

#### 解説
処方1の薬（例: ダルベポエチンアルファ（遺伝子組換え）など）は**生物由来製品**であり、ラベルに「生物」と記載される。生物由来製品とは、ヒトその他の生物（**植物は除く**）に由来する原材料で製造される医薬品で、感染症リスクなどの特別な注意を要するもの（例: ワクチン、血漿製剤、遺伝子組換え蛋白製剤）。旧薬事法（現医薬品医療機器等法）で定義され、感染症防止のため製造販売業者に感染症定期報告が義務付けられるなど対策が強化されている。[1][2][3]

- **1 正**: 生物由来製品の感染症リスク（プリオン病、ウイルス感染等）対策として、**患者の文書同意**が必要。[1][2]  
- **2 誤**: 植物由来は「生物」に該当せず、記載されない。[1][3]  
- **3 誤**: これは**遺伝子治療用医薬品**（再生医療等製品）の定義で、生物由来製品とは異なる。[2][3]  
- **4 誤**: 製薬企業への報告義務はあるが、患者説明の本質ではなく選択肢の文脈で不適切（正解は患者同意）。[2]  

本問は法規・制度・倫理分野で、生物由来製品のラベル表示とリスク管理を問う。[1][2][3]
