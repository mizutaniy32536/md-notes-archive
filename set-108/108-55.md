# 108-55

皮膚表面に微細な針で小孔を形成することにより、高分子薬物の経皮吸収性を改善する手法はどれか。１つ選べ。

１ マイクロニードル
２ イオントフォレシス
３ エレクトロポレーション
４ ソノフォレシス
５ ジェットインジェクション

**正解:** １

**解説**
１　正
マイクロニードルとは、皮膚に微細針を貼付することで、角質層に細孔を形成し、薬物の経皮吸収性を改善する手法である。
２　誤
イオントフォレシスとは、皮膚に低電流・低電圧をかけることで、イオン性薬物の経皮吸収性を改善する手法である。
３　誤
エレクトロポレーションとは、皮膚に短時間高電圧をかけることで、角質層に細孔を形成し、薬物の経皮吸収性を改善する手法である。
４　誤
ソノフォレシスとは、皮膚に超音波を当てることで、薬物の経皮吸収性を改善する手法である。
５　誤
ジェットインジェクションとは、皮膚に高水圧をかけることで、角質層に細孔を形成し、薬物の経皮吸収性を改善する手法である。
