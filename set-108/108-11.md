## ◆ 人体骨格の模式図から仙骨を選ぶシナリオ
260312

**問題**
下図は、人体骨格の模式図である。1〜5のうち、仙骨はどれか。1つ選べ。

**選択肢**

1. 胸骨
​
2. 鎖骨
​
3. 尺骨
​
4. 仙骨
​
5. 坐骨

⚠️ 図が必要な問題です。実際の問題では骨格の模式図が示されます。

**正解と理由**
- `4. 仙骨`
- **理由の構成**
	- 仙骨は骨盤後面に位置する三角形の骨（仙椎が融合したもの）

**解法の手順（思考の流れ）**
- 選択肢1：胸骨 = 胸部中央の平らな骨 → ✗
- 選択肢2：鎖骨 = 肩甲骨と胸骨をつなぐS字型の骨 → ✗
- 選択肢3：尺骨 = 前腕の骨（橈骨と対） → ✗
- 選択肢4：仙骨 = 骨盤後面の三角形の骨 → **○**
- 選択肢5：坐骨 = 骨盤下部、座る時に体重を支える → ✗

**キーポイント**
- 仙骨 = 5つの仙椎が融合、骨盤後壁を構成
- 坐骨と仙骨を混同しない

（一言メモ）
- 図問題なので位置関係を確認。仙骨は脊椎の末端・骨盤の後ろ
