# 108-17

**問題文**
下図は、我が国における2017年から2021年までの業務上疾病の発生状況を示したものである。Aに該当する疾病はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-17 を参照

**選択肢**
1. 手指前腕の障害及び頸肩腕症候群
2. 熱中症
3. 振動障害
4. 騒音性難聴
5. 災害性腰痛

**正解:** 5

**解説**
解答　５

Aは2017年〜2019年まで発生数第一位であることから、災害性腰痛である。
近年、業務上疾病の発生数の第一位は災害性腰痛であったが、2020年、2021年は新型コロナウイルスの影響により、病原体による疾病が第一位となっている。
