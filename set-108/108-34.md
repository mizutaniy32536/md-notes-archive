# 108-34

**問題文**
前立腺肥大に伴う排尿障害を改善するタダラフィルの作用機序はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-34 を参照

**選択肢**
1. ホスホジエステラーゼⅤ阻害
2. アドレナリンα1受容体遮断
3. アドレナリンβ3受容体刺激
4. アセチルコリンM3受容体刺激
5. コリンエステラーゼ阻害

**正解:** 1

**解説**
解答　１

タダラフィルは、ホスホジエステラーゼⅤ阻害薬であり、cGMPの分解を抑制することで、下部尿路血管拡張作用および前立腺・尿道平滑筋弛緩作用を介した尿道抵抗軽減を示し、前立腺肥大に伴う排尿障害などを改善する。
