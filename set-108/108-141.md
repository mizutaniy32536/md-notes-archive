# 108-141

**問題文**
薬剤師の業務に関係するインターネット等の利用に関する記述のうち、正しいのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-141 を参照

**選択肢**
1. 有名人のAさんに対して調剤を行ったことについて名前と処方内容を含めてSNSに投稿する。
2. 薬局のホームページにおいて、一般用医薬品の広告をする際、利用者の口コミを表示する。
3. 医療機関では漏えいのおそれがあるため、セキュリティ対策を行ったとしてもWi−Fi（無線LAN）は利用できない。
4. 亡くなった後の患者の情報は個人情報でなくなるため、薬剤師の個人ブログで患者の情報を公表する。
5. インターネット上の医療情報は、裏付けがある情報か検討した上で利用の可否を検討する。

**正解:** 5

**解説**
解答　５

１　誤
名前と処方内容を含めてSNS（ソーシャルネットワーキングサービス）に投稿する行為は、個人情報の漏洩に該当するため行ってはならない。

２　誤
ホームページへの購入者レビューや口コミの記載は、不適切な医薬品選択につながる恐れがあるため、禁止されている。

３　誤
医療機関において、セキュリティ対策を行えば、Wi−Fi（無線LAN）を利用することができる。

４　誤
死者に関する情報は個人情報に該当しないが、医療・介護において生存する者と同様の安全管理措置を行うことが求められている。よって、薬剤師は、亡くなった後の患者情報を漏洩しないように取り扱う必要がある。

５　正
インターネット上の医療情報は、必ずしも正しいとは限らないため、裏付けがある情報かどうかを検討した上で利用する必要がある。
