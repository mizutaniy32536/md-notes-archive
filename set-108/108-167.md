# 108-167

肺がん治療に用いる薬物に関する記述のうち、正しいのはどれか。２つ選べ。

１ イリノテカンは、フリーラジカルを発生して、DNA鎖を切断する。
２ エトポシドは、トポイソメラーゼⅡを阻害して、切断されたDNA鎖の再結合を阻害する。
３ ゲフィチニブは、抗programmed cell death−1（PD−1）抗体で、T細胞による抗腫瘍作用を増強する。
４ ゲムシタビンは、DNAをアルキル化して、DNAの複製を阻害する。
５ ドセタキセルは、微小管を安定化して、有糸分裂を阻害する。

**正解:** ２、５

**解説**
１　誤
ブレオマイシンなどに関する記述である。イリノテカンは、カルボキシエステラーゼによって加水分解され、SN−38となりトポイソメラーゼⅠを阻害して、DNAの複製を阻害することで、抗悪性腫瘍作用を示す。なお、フリーラジカルを発生してDNA鎖を切断するのは、ブレオマイシンである。
２　正
エトポシドは、トポイソメラーゼⅡを阻害して、切断されたDNA鎖の再結合を阻害することで、抗悪性腫瘍作用を示す。
３　誤
ニボルマブなどに関する記述である。ゲフィチニブは、上皮増殖因子受容体（EGFR）チロシンキナーゼを選択的に阻害することで、抗悪性腫瘍作用を示す。なお、抗programmed cell death−1（PD−1）抗体で、T細胞による抗腫瘍作用を増強するのは、ニボルマブなどである。
４　誤
シクロホスファミドなどに関する記述である。ゲムシタビンは、生体内でリン酸化されて活性体の二リン酸化体と三リン酸化体になる。このうち二リン酸化体はリボヌクレオチドレダクターゼ阻害により間接的にDNA合成を阻害し、三リン酸化体はDNAポリメラーゼの作用によりDNAに組み込まれることで細胞死（アポトーシス）を誘発し、直接的にDNA合成を阻害ことで、抗悪性腫瘍作用を示す。なお、DNAをアルキル化してDNAの複製を阻害するのは、シクロホスファミドなどのアルキル化薬である。
５　正
ドセタキセルは、微小管タンパク質（チューブリン）の重合を促進することで、微小管を安定化し、有糸分裂を阻害することで、抗悪性腫瘍作用を示す。
