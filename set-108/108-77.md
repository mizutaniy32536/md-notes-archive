# 108-77

診療報酬の改定の内容に関する、厚生労働大臣の諮問先はどれか。1つ選べ。

1. 医道審議会
2. 薬事・食品衛生審議会
3. 国民健康保険団体連合会
4. 中央社会保険医療協議会
5. 社会保険診療報酬支払基金

## 正解
**4. 中央社会保険医療協議会**

## 解説

診療報酬（薬価基準含む）の改定は厚生労働大臣が中医協に諮問して決定する。

- 医道審議会 → 医師・薬剤師等の行政処分、国家試験
- 薬事・食品衛生審議会 → 新薬承認、薬局方改定
- 国保連合会 → 国保の診療報酬審査支払
- 支払基金 → 被用者保険の診療報酬審査支払

「点数を決めるのは中医協、払うのは支払基金・国保連」
