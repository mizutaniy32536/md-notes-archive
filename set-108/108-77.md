# 108-77

診療報酬の改定の内容に関する、厚生労働大臣の諮問先はどれか。１つ選べ。

１ 医道審議会
２ 薬事・食品衛生審議会
３ 国民健康保険団体連合会
４ 中央社会保険医療協議会
５ 社会保険診療報酬支払基金

**正解:** ４

**解説**
診療報酬の改定の内容に関する厚生労働大臣の諮問先は、中央社会保険医療協議会である。
１　誤
医道審議会は、厚生労働省に設置される審議会の１つであり、医師・歯科医師・薬剤師などの資格に対する行政処分などを担う。
２　誤
薬事・食品衛生審議会は、厚生労働省に設置される審議会の１つであり、医薬品の承認や毒薬・劇薬の指定、薬事食品関連の事柄の調査・審議を行う。
３　誤
国民健康保険団体連合会は、国民健康保険の医療費について、市町村からの委託を受けて審査・支払を代行する。
４　正
前記参照
５　誤
社会保険診療報酬支払基金は、健康保険、船員保険、共済組合をはじめとする職域保険の医療費および公費負担医療費について、各保険者から委任を受けて保険請求の審査・支払を代行する。
