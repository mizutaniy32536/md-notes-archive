# 108-18

食品の水分活性を表す式として正しいのはどれか。１つ選べ。 ただし、P は食品を入れて密封した容器内の水蒸気圧、P0はその温度における純水の飽和蒸気圧を表す。

**正解:** ４

**解説**
食品の微生物汚染リスクの指標である水分活性（Aw ）は、食品中に含まれる自由水の割合を表したものであり、食品を入れて密封した容器内の水蒸気圧（P ）とその温度における純水の飽和蒸気圧（P0）を用いて、次式で表される。
微生物は食品中に含まれる自由水を利用して増殖するため、水分活性が高いほど、微生物汚染のリスクが高いことを表す。また一般的に、食塩、糖、アミノ酸などを多く含む食品の場合、水分が食塩、糖、アミノ酸などに捕捉されることで自由水が少なくなり、水分活性が低くなる。
