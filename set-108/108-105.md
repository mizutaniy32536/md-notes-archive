# 108-105

**問題文**
以下の構造式で表されるダイゼインに関する記述のうち、正しいのはどれか。2つ選べ。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-105 を参照

**選択肢**
1. カッコンに含まれる成分の一つである。
2. A環の炭素原子は、シキミ酸由来である。
3. 2−フェニルクロモン骨格を有している。
4. ヒトに対してエストロゲン様作用がある。
5. マグネシウム－塩酸反応で陽性を示す。

**正解:** 1,4

**解説**
解答　１、４

ダイゼインは、カッコンなどに含まれる成分であり、イソフラボノイド骨格（3−フェニルクロモン骨格）を有している。イソフラボノイドはヒトに対して、抗酸化作用やエストロゲン様作用があることが知られている。

１　正
前記参照

２　誤
イソフラボノイドは、シキミ酸経路と酢酸-マロン酸経路の複合経路で生合成される構造であり、A環の炭素原子は酢酸-マロン酸経路由来、その他の炭素原子がシキミ酸経路由来である。

３　誤
前記参照。なお、2−フェニルクロモン骨格はフラボノイド骨格のことを指す。

４　正
前記参照

５　誤
フラボノイドの記述である。フラボノイドは、マグネシウム－塩酸反応で呈色するため陽性を示すが、イソフラボノイドは陰性を示す。
