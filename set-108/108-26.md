# 108-26

**問題文**
タキフィラキシーに関する記述として、正しいのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-26 を参照

**選択肢**
1. 短時間の反復投与による薬物に対する反応性の増強
2. 短時間の反復投与による薬物に対する反応性の減弱
3. 長時間の反復投与による薬物に対する反応性の増強
4. 長時間の反復投与による薬物に対する依存性の獲得
5. 長期薬物投与後の休薬による薬物に対する反応性の増強

**正解:** 2

**解説**
解答　２

タキフィラキシーとは、短時間の反復投与による薬物に対する反応性の減弱がみられる現象であり、チラミンなどの間接型交感神経興奮薬で起こる。チラミンを短時間に反復投与すると、シナプス小胞のノルアドレナリンが枯渇し、その昇圧作用が次第に減弱する。
