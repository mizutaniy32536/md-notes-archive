# 108-26

１ 短時間の反復投与による薬物に対する反応性の増強
２ 短時間の反復投与による薬物に対する反応性の減弱
３ 長時間の反復投与による薬物に対する反応性の増強
４ 長時間の反復投与による薬物に対する依存性の獲得
