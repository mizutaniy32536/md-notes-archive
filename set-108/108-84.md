# 108-84

**問題文**
プロトンポンプ阻害薬と併用禁忌の医薬品はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-84 を参照

**選択肢**
1. セベラマー塩酸塩錠
2. エンパグリフロジン錠
3. アタザナビル硫酸塩カプセル
4. 沈降炭酸カルシウム錠
5. レパグリニド錠

**正解:** 3

**解説**
解答　３

選択肢中で、プロトンポンプ阻害薬と併用禁忌の医薬品はアタザナビル硫酸塩である。プロトンポンプ阻害薬により胃酸分泌が低下すると、難溶性塩基性薬物であるアタザナビル硫酸塩の溶解性が低下するため、吸収率が悪くなり、血中濃度が低下することがある。
