# 108-333

**問題文**
65歳女性。体重50 kg。深在性真菌症のためボリコナゾールを点滴にて投与開始した。投与開始7日後の維持投与量は3 mg/kgで1日2回点滴静注している。血中濃度を測定したところ、平均血中濃度は1.25 µg/mLを推移していた。治療効果が得られていることから、担当医は経口投与への変更を予定しており、投与量の相談を受けた。12時間間隔での経口投与を提案する場合の1回あたりの投与量（mg）として、最も近い値はどれか。1つ選べ。ただし、ボリコナゾールの全身クリアランスは0.2 L/h/kg、本患者のバイオアベイラビリティは75%とする。

**選択肢**

2. 150
3. 200
4. 300
5. 400

**正解:** 3
