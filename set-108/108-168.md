# 108-168

薬物の消化管吸収に関する記述のうち、正しいのはどれか。１つ選べ。

１ 小腸下部から吸収された薬物は門脈を介さずに全身循環へ移行する。
２ P−糖タンパク質に認識される薬物の消化管からの吸収速度定数は、薬物の脂溶性と吸収速度定数との相関から推定される値よりも大きい。
３ 食後に薬物を投与すると胃内容排出速度が上昇するため、薬物の最高血中濃度到達時間は早くなる。
４ 脂溶性の高い薬物は、小腸吸収過程において非撹拌水層の影響を受けやすい。
５ 消化管粘膜表面のpHは消化管管腔内のpHよりも低いため、弱酸性薬物の消化管からの吸収量は、管腔内pHから予想される量よりも少ない。

**正解:** ４

**解説**
１　誤
小腸から吸収された薬物は、小腸上部・中部・下部いずれの場合も、門脈を介して全身循環血へ移行する。なお、直腸下部から吸収された薬物は、門脈を介さずに全身循環へ移行する。
２　誤
P−糖タンパク質は、ATPの加水分解エネルギーを直接利用して、細胞内から細胞外へ薬物を排出する一次性能動輸送担体であり、小腸上皮細胞や腎尿細管上皮細胞、脳毛細血管内皮細胞などに発現している。P−糖タンパク質に認識される薬物は、P—糖タンパク質の働きにより消化管側に排泄されるため、P−糖タンパク質が関与しないが脂溶性が同じ薬物に比べて、消化管側から血管側への吸収率が低下する。したがって、P−糖タンパク質に認識される薬物の消化管からの吸収速度定数は、薬物の脂溶性と吸収速度定数との相関から推定される値よりも小さくなる。
３　誤
胃内容排出速度（GER）とは、経口投与した薬物が胃から小腸へ排出される速度のことである。一般的に、食後に薬物を投与すると胃内容排出速度は低下する。そのため、食後に経口投与された薬物が単純拡散で吸収される場合、胃内容排出速度の低下により薬物の最高血中濃度到達時間は遅くなる。
４　正
薬物の小腸吸収過程では、粘膜表面に存在する非撹拌水層の拡散と消化管粘膜透過の2つの過程の影響を考慮する必要がある。小腸吸収過程において、脂溶性の高い薬物は非撹拌水層の影響を受けやすく、水溶性の高い薬物は消化管粘膜透過過程の影響を受けやすい。
５　誤
消化管粘膜表面では、プロトンが管腔側へ分泌されている。そのため、消化管粘膜表面のpHは、消化管管腔内のpHよりも低く保たれているため、弱酸性薬物の分子形分率が高くなる。（下図参照）したがって、弱酸性薬物の消化管からの吸収量は、管腔内pHから予想される量より多くなる。
