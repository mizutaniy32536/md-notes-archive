# 108-15

**問題文**
自然免疫系に認識される、グラム陰性菌に特徴的な構造はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-15 を参照

**選択肢**
1. フラジェリン
2. ペプチドグリカン
3. リポ多糖（LPS）
4. 二本鎖RNA（dsRNA）
5. β−グルカン

**正解:** 3

**解説**
解答　３

グラム陰性菌に特徴的な構造はリポ多糖（LPS）である。自然免疫系は、感染における最初の防御線として機能し、細菌やウイルスなどの外部から侵入した病原体を認識して攻撃することができる。リポ多糖は、グラム陰性菌細胞壁の主要な構成要素であり、自然免疫系に認識されることで炎症反応を引き起こす。この反応により、病原体が体内に侵入したことを身体に知らせ、獲得免疫系が病原体を攻撃するように促す連携をとっている。
