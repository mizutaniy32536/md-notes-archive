## ◆ グラム陰性菌に特徴的な自然免疫認識構造を選ぶシナリオ
260312

**問題**
自然免疫系に認識される、グラム陰性菌に特徴的な構造はどれか。1つ選べ。

**選択肢**

1. フラジェリン
​
2. ペプチドグリカン
​
3. リポ多糖 (LPS)
​
4. 二本鎖 RNA (dsRNA)
​
5. β-グルカン

**正解と理由**
- `3. リポ多糖 (LPS)`
- **理由の構成**
	- LPS = グラム陰性菌の`外膜`に存在する内毒素
	- `TLR4`により認識 → 炎症応答を誘導

**解法の手順（思考の流れ）**
- 選択肢1：フラジェリン = 細菌の`鞭毛`タンパク質。グラム陰性菌特有ではない（TLR5認識）→ ✗
- 選択肢2：ペプチドグリカン = グラム陽性菌・陰性菌`共通`（TLR2認識）→ ✗
- 選択肢3：LPS = グラム陰性菌の外膜`特有`（TLR4認識）→ **○**
- 選択肢4：dsRNA = `ウイルス`由来（TLR3認識）→ ✗
- 選択肢5：β-グルカン = `真菌`の細胞壁（Dectin-1認識）→ ✗

**キーポイント**
- `LPS（リポ多糖）` = グラム陰性菌外膜の構成成分 = 内毒素（エンドトキシン）
- グラム陽性菌にはLPSがない（外膜がないため）
- LPSのリピドA部分が毒性を発揮

**関連知識の比較**

| PAMPs | 由来 | 認識受容体 |
|---|---|---|
| LPS | **グラム陰性菌** | TLR4 |
| ペプチドグリカン | 細菌（共通） | TLR2 |
| フラジェリン | 細菌の鞭毛 | TLR5 |
| dsRNA | ウイルス | TLR3 |
| β-グルカン | 真菌 | Dectin-1 |
| CpG DNA | 細菌DNA | TLR9 |

（一言メモ）
- グラム陰性菌 = LPS = 外膜 = 内毒素。セットで覚える
- TLR4 → LPS。TLR番号とリガンドの対応は頻出
