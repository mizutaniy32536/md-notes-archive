# 108-41

**問題文**
担体介在輸送がミカエリス・メンテン（Michaelis−Menten）式に従うとき、薬物の輸送速度（v ）を表す式はどれか。1つ選べ。ただし、薬物濃度をC 、最大輸送速度をVmax、ミカエリス定数をKmとする。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-41 を参照

**選択肢**
（選択肢は画像を参照）

**正解:** 5

**解説**
解答　５

担体介在輸送がミカエリス・メンテン（Michaelis−Menten）式に従うとき、薬物の輸送速度（v ）を表す式は①式で表される。ただし、薬物濃度をC 、最大輸送速度をVmax、ミカエリス定数をKmとする。
