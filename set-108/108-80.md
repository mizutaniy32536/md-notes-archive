# 108-80

他者の価値観に配慮しつつ、自分の欲求、考え、気持ちを適切に表現することを何というか。１つ選べ。

１ ラポール
２ パターナリズム
３ アサーション
４ ミラーリング
５ コーチング

**正解:** ３

**解説**
対人関係を円滑にするためのコミュニケーションスキルのうち、他者の価値観に配慮しつつ、自分の欲求、考え、気持ちを適切に表現するスキルはアサーションとよばれる。
１　誤
ラポールとは、「調和した関係」などの意味をもつ心理学の用語であり、心の通い合った状態や互いの信頼関係が構築された状態を表す言葉である。
２　誤
パターナリズムとは、強い立場にある者が弱い立場の者の意志に反して、弱い立場の者の利益になるという理由から、その行動に介入や干渉することを表す言葉である。
３　正
前記参照
４　誤
ミラーリングとは、相手の動作・行動・しぐさなどを鏡のように真似をすることであり、信頼関係を築きたい相手や関係性を良くしたい相手に行うことが効果的とされているコミュニケーションスキルである。
５　誤
コーチングとは、相手の話を傾聴し、問いかけなどをすることで、問題解決に向けて相手自身が考え、答えを見出せるように促すコミュニケーションスキルである。
