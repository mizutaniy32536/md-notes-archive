## ◆ 薬物のリンパ系への移行に関する正しい記述を選ぶシナリオ

260308

**問題**
- 薬物のリンパ系への移行に関する記述のうち、正しいのはどれか。2つ選べ。

**選択肢**

1. 消化管からリンパ系を介して吸収された薬物は、肝初回通過効果を受けずに全身循環系に到達する。
​
2. リンパ系に移行した薬物は、血液循環系へ移行した薬物に比べてゆっくりと全身に分布する。
​
3. 毛細リンパ管内皮細胞は密着結合を形成しているため、分子量の小さい薬物の方が透過しやすい。
​
4. 皮下投与された抗体医薬品は、リンパ系へ移行しにくい。
​
5. リンパ系に移行した薬物は、胸管リンパを経て肺動脈に入り、全身循環血を介して体内に分布する。

**正解と理由**
- `1. 消化管からリンパ系を介して吸収された薬物は、肝初回通過効果を受けずに全身循環系に到達する。` と `2. リンパ系に移行した薬物は、血液循環系へ移行した薬物に比べてゆっくりと全身に分布する。`
- **理由の構成**
	- **問題の整理**：リンパ系への薬物移行に関する5つの記述から正しいものを2つ選ぶ問題
	- **薬学的知識・仕様・制約**：リンパ系は`胸管`→`左鎖骨下静脈`経由で全身循環に合流するため、`門脈`を経由しない→`肝初回通過効果`を回避できる（選択肢1）。リンパ液の流速は血流の約1/500〜1/1000であるため、薬物の分布は血液経由より`ゆっくり`となる（選択肢2）
	- **選択理由**：1と2はリンパ系の解剖学的・生理学的特性を正しく記述しているため

**解法の手順（思考の流れ）**
- 選択肢1：リンパ→胸管→左鎖骨下静脈→全身循環。門脈を通らないので肝初回通過効果なし → 正しい
- 選択肢2：リンパ流速は血流に比べて極めて遅い（約1/500）→ 分布はゆっくり → 正しい
- 選択肢3：毛細リンパ管の内皮は密着結合が「ない」→ むしろ大分子が入りやすい → 誤り
- 選択肢4：抗体医薬品（高分子）は毛細血管を通過しにくく、リンパ系経由が主な吸収経路 → 誤り
- 選択肢5：胸管は「左鎖骨下静脈」に合流するのであり、「肺動脈」ではない → 誤り

**他の選択肢が誤りな理由**
- 3. 毛細リンパ管内皮細胞は密着結合を形成しているため、分子量の小さい薬物の方が透過しやすい。
	毛細リンパ管の内皮細胞は`密着結合を形成していない`（開放型）。そのため、むしろ`高分子`（タンパク質、細菌、抗体医薬品など）も容易に侵入できる。毛細血管の内皮細胞が密着結合を持つのとは対照的

- 4. 皮下投与された抗体医薬品は、リンパ系へ移行しにくい。
	`抗体医薬品`（分子量約150 kDa）は毛細血管の内皮を通過しにくいため、`リンパ系経由`で全身循環に入る経路が主要。皮下投与→リンパ管→胸管→全身循環のルートが重要

- 5. リンパ系に移行した薬物は、胸管リンパを経て肺動脈に入り、全身循環血を介して体内に分布する。
	胸管は`左鎖骨下静脈`（左静脈角）に合流する。肺動脈ではない。左鎖骨下静脈→上大静脈→右心→肺動脈→肺→左心→全身の順で循環する

**キーポイント**
- リンパ系の合流先：胸管→`左鎖骨下静脈`（「肺動脈」とのひっかけ頻出）
- リンパ系の最大の利点：`肝初回通過効果の回避`
- リンパ流速は血流の約`1/500`→ 分布は遅い
- 毛細リンパ管は`密着結合なし`→ 高分子も侵入可能

**関連知識の比較**
- `門脈系`：消化管→門脈→肝臓→全身循環。肝初回通過効果を受ける
- `リンパ系`：消化管→リンパ管→胸管→左鎖骨下静脈→全身循環。肝初回通過効果を回避
- リンパ系経由で吸収されやすい薬物：`脂溶性`が極めて高い薬物（トリグリセリド様）、`高分子`（抗体医薬品）
- `カイロミクロン`に取り込まれた薬物はリンパ管経由で全身循環に移行する代表例

（一言メモ）
- リンパの合流先 = `左鎖骨下静脈`は必ず覚える。「胸管→肺動脈」は定番のひっかけ
- `抗体医薬品の皮下投与`→ リンパ経由が主要ルート。バイオアベイラビリティは50-80%程度で血管内投与より低い
- リンパ流速の遅さ（1/500）は、リンパ経由薬物のTmaxが長い（数日かかることも）理由
