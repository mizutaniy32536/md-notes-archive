# 108-170

# 第108回薬剤師国家試験 問170

## 問題文

薬物のリンパ系への移行に関する記述のうち、正しいのはどれか。**2つ選べ。**[1][2][3]

## 全選択肢

1. 消化管からリンパ系を介して吸収された薬物は、肝初回通過効果を受けずに全身循環系に到達する。[1]

2. リンパ系に移行した薬物は、血液循環系へ移行した薬物に比べてゆっくりと全身に分布する。[2]

3. 毛細リンパ管内皮細胞は密着結合を形成しているため、分子量の小さい薬物の方が透過しやすい。[3]

4. 皮下投与された抗体医薬品は、リンパ系へ移行しにくい。[3]

5. リンパ系に移行した薬物は、胸管リンパを経て肺動脈に入り、全身循環血を介して体内に分布する。[3]

## 正解

**1、2**[2]

## 解説

**選択肢1（○）**
消化管からリンパ系を介して吸収された薬物は、門脈を介さず、リンパ管を通り鎖骨下静脈へ流れ込み全身循環となるため、肝初回通過効果を受けません。[1]

**選択肢2（○）**
リンパ液流量は血液循環量に比べて著しく小さいため、リンパ系に移行した薬物は、血液循環系へ移行した薬物に比べてゆっくりと全身に分布します。[6]

**選択肢3（×）**
毛細リンパ管は毛細血管に比べて内皮細胞間の結合が緩いため、隙間が大きく、むしろ分子量の大きい薬物が透過できます。一般的には、分子量5000以上の薬物がリンパ管へ移行しやすく、分子量5000以下の薬物は血管へ移行する傾向があります。[1]

**選択肢4（×）**
皮下投与された抗体医薬品は分子量が大きい（約15万程度）ため、血管系よりもリンパ系へ移行しやすいです。[1]

**選択肢5（×）**
リンパ系に移行した薬物は、胸管リンパを経て静脈（肺動脈ではなく）に入り、全身循環血を介して体内に分布します。[1][6]
