# 108-123

**問題文**
下図は、我が国におけるある性感染症の報告数（全数把握）の年次推移を示したものである。この図に該当する感染症はどれか。1つ選べ。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-123 を参照

**選択肢**
1. 性器クラミジア感染症
2. 性器ヘルペス感染症
3. 尖形コンジローマ
4. 梅毒
5. 後天性免疫不全症候群

**正解:** 4

**解説**
解答　４

図に該当する感染症は梅毒である。
全数把握における梅毒罹患の報告数は、2013年頃から2018年にかけて急増しており、2019年、2020年に減少傾向にあるものの、2021年には再び増加している。次に、近年の性感染症の報告数を示す。
