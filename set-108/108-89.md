# 108-89

**問題文**
一般消費者に対する要指導医薬品の日常の販売について、正しいのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-89 を参照

**選択肢**
1. インターネットで販売できる。
2. 配置による方法で販売できる。
3. 使用する者の年齢を確認しなければならない。
4. 使用する者が同居家族の場合にも販売できる。
5. 情報提供は薬剤師又は登録販売者が行う。

**正解:** 3

**解説**
解答　３

１　誤
要指導医薬品は薬剤師が対面で情報提供・指導を行う必要があるため、インターネットで販売することはできない。

２　誤
要指導医薬品は薬剤師が対面で情報提供・指導を行う必要があるため、配置による方法で販売できない。

３　正
要指導医薬品を販売する際には、使用する者の年齢をはじめ、性別、症状など使用者の基礎情報を確認しなければならない。

４　誤
要指導医薬品は、正当な理由がない場合には使用者本人にしか販売できないため、使用する者が同居家族の場合は販売できない。

５　誤
要指導医薬品の情報提供は薬剤師が行う必要があり、登録販売者が行うことはできない。
