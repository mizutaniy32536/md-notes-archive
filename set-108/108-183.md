# 108-183

重篤な副作用と代表的な原因薬剤例を示した組合せとして、正しいのはどれか。２つ選べ。

**正解:** ３、５

**解説**
１　誤
副作用で横紋筋融解症を起こしやすい薬剤としては、HMG-CoA還元酵素阻害薬（シンバスタチンなど）、フィブラート系薬（フェノフィブラートなど）、ニューキノロン系抗菌薬（レボフロキサシンなど）があげられる。
２　誤
副作用で劇症肝炎を起こしやすい薬剤としては、ベンズブロマロンなどがあげられる。
３　正
副作用で心不全を起こしやすい薬剤としては、アントラサイクリン系抗がん剤（ドキソルビシンなど）、HER2阻害薬（トラスツズマブなど）があげられる。
４　誤
副作用で間質性肺炎を起こしやすい薬剤としては、抗がん剤（ブレオマイシンなど）、抗リウマチ薬（メトトレキサートなど）、免疫チェックポイント阻害薬（ニボルマブなど）があげられる。
５　正
副作用で糖尿病性ケトアシドーシスを起こしやすい薬剤としては、多元受容体作用抗精神病薬（オランザピンなど）、副腎皮質ステロイド性薬（プレドニゾロンなど）があげられる。
