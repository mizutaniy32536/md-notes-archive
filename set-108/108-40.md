# 108-40

**問題文**
緑内障治療薬ブリモニジンの作用機序はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-40 を参照

**選択肢**
1. プロスタノイドEP2受容体刺激
2. プロスタノイドFP受容体刺激
3. アドレナリンα2受容体刺激
4. アセチルコリンM3受容体刺激
5. Rhoキナーゼ阻害

**正解:** 3

**解説**
解答　３

ブリモニジンは、アドレナリンα2受容体刺激薬であり、眼房水産生抑制及びぶどう膜強膜流出路を介した眼房水流出促進により、眼圧を下降させる緑内障治療薬である。
