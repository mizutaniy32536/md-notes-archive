# 108-1

下図は水の状態図である。アからイへの矢印が表す相変化はどれか。１つ選べ。

１ 融解
２ 凝縮
３ 昇華
４ 凝固
５ 蒸発

**正解:** ２

**解説**
水の状態図とは、温度や圧力を変化させたときの水の状態（固体、液体、気体など）を表したものであり、水は領域アでは気体、領域イでは液体として存在する。
従って、アからイへの矢印が表す相変化は、気体から液体への相変化であり、凝縮と呼ばれる。
１　誤
融解は、固体から液体への相変化のことである。
２　正
凝縮は、気体から液体への相変化のことである。
３　誤
昇華は、固体から気体への相変化のことである。
４　誤
凝固は、液体から固体への相変化のことである。
５　誤
蒸発は、液体から気体への相変化のことである。
