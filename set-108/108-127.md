# 108-127

次の反応式は、油脂の化学的指標に関する試験法の化学反応の例を示したものである。不飽和脂肪酸を含む油脂において、**自動酸化の進行に伴い測定値が減少し続ける試験法**の反応式はどれか。**1つ選べ。**

1. 酸価の測定（遊離脂肪酸をKOHで滴定）
2. 過酸化物価の測定（過酸化物がヨウ化カリウムを酸化）
3. ヨウ素価の測定（ヨウ素が不飽和結合に付加）
4. カルボニル価の測定（カルボニル化合物を定量）

## 正解
**3. ヨウ素価の測定**

## 解説

### なぜヨウ素価が減少し続けるのか？

**ヨウ素価（Iodine Value, IV）とは：**
- 油脂100 gに付加するヨウ素のグラム数
- **不飽和結合（二重結合）の量**を表す指標
- 不飽和度が高いほどヨウ素価が高い

**反応：**
```
-CH=CH- + I₂ → -CHI-CHI-
  ↑               ↑
二重結合      ヨウ素が付加
```

**自動酸化が進むと：**
```
-CH=CH- + O₂ → 過酸化物 → さらに分解
  ↑
二重結合が酸化で壊れる
```

→ 二重結合が減る → ヨウ素が付加できる場所が減る → **ヨウ素価が減少し続ける**

---

### 油脂の自動酸化のメカニズム

**自動酸化（autoxidation）の段階：**

1. **開始反応**
   ```
   RH → R・（ラジカル生成）
   ```

2. **連鎖反応**
   ```
   R・ + O₂ → ROO・（ペルオキシラジカル）
   ROO・ + RH → ROOH（過酸化物） + R・
   ```

3. **分解反応**
   ```
   ROOH → RO・ + ・OH
   → アルデヒド、ケトン、遊離脂肪酸など
   ```

**結果：**
- 二重結合が酸化で消費される
- 過酸化物、カルボニル化合物、遊離脂肪酸が生成

---

### 各選択肢の詳細解説

#### 選択肢1 ❌ 酸価（Acid Value, AV）

**測定するもの：** 遊離脂肪酸（RCOOH）の量

**定義：** 油脂1 gを中和するのに必要なKOHのmg数

**反応：**
```
RCOOH + KOH → RCOOK + H₂O
```

**自動酸化での変化：**
- 油脂が分解 → 遊離脂肪酸が生成
- **増加し続ける**（加水分解でも増加）

#### 選択肢2 ❌ 過酸化物価（Peroxide Value, POV/PV）

**測定するもの：** 過酸化物（ROOH）の量

**定義：** 油脂1 kgに含まれる過酸化物をヨウ化カリウムと反応させたときの遊離ヨウ素のミリ当量数（meq/kg）

**反応：**
```
ROOH + 2KI + 2H⁺ → ROH + I₂ + H₂O + 2K⁺
（遊離したI₂をチオ硫酸ナトリウムで滴定）
```

**自動酸化での変化：**
- **初期：** 過酸化物が生成 → **増加**
- **後期：** 過酸化物が分解 → **減少**
- つまり「減少し続ける」わけではない

**過酸化物価の変化曲線：**
```
POV
 ↑
 |    ╱＼
 |   ╱  ＼___
 |  ╱       ＼
 |_╱___________→ 時間
   初期 中期 後期
```

#### 選択肢3 ⭕ ヨウ素価（Iodine Value, IV）

**測定するもの：** 不飽和結合（二重結合）の量

**定義：** 油脂100 gに付加するヨウ素のグラム数

**反応：**
```
-CH=CH- + I₂ → -CHI-CHI-
```

**自動酸化での変化：**
- 二重結合が酸化で壊れる
- **減少し続ける**

**ヨウ素価の変化曲線：**
```
IV
 ↑
 |＼
 | ＼
 |  ＼___
 |     ＼___
 |________＼→ 時間
   減少し続ける
```

**油脂の種類とヨウ素価（新鮮時）：**
| 油脂 | ヨウ素価 |
|---|---|
| ヤシ油 | 8-10 |
| ラード | 50-65 |
| オリーブ油 | 75-94 |
| 大豆油 | 120-140 |
| アマニ油 | 170-200 |

#### 選択肢4 ❌ カルボニル価（Carbonyl Value, CV）

**測定するもの：** カルボニル化合物（アルデヒド、ケトン）の量

**反応：** カルボニル化合物とヒドラジン誘導体の反応を利用

**自動酸化での変化：**
- 過酸化物が分解 → カルボニル化合物が生成
- **増加し続ける**

---

### 油脂の酸化度の評価

実際の食用油の品質管理では、複数の指標を組み合わせて評価する：

| 指標 | 新鮮油 | 初期酸化 | 中期酸化 | 後期酸化 |
|---|---|---|---|---|
| **ヨウ素価** | 高い | ↓ | ↓↓ | ↓↓↓ |
| **過酸化物価** | 低い | ↑↑ | ↑↑↑（ピーク） | ↓ |
| **酸価** | 低い | → | ↑ | ↑↑ |
| **カルボニル価** | 低い | → | ↑ | ↑↑ |

**判定基準例（食用油）：**
- 過酸化物価：30 meq/kg以下（JAS規格）
- 酸価：0.2以下（サラダ油）

**注意点：**
- 過酸化物価は後期で減少するため、過酸化物価だけでは判断できない
- フライ油など高温で使用した油は過酸化物が分解しているため、カルボニル価も併用
