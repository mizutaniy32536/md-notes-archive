



## ◆ SP療法に用いる薬物の電気伝導率変化と抗がん作用

260313

**問題**
- 72歳男性。胃がんStage Ⅳに対しS－1＊／シスプラチン療法（SP療法）を導入。
＊：テガフール・ギメラシル・オテラシルカリウム配合剤
- SP療法に用いられる薬物及びその代謝物を水溶液としたとき、水溶液のモル電気伝導率が時間の経過とともに増大し、抗がん作用を発現するのはどれか。１つ選べ。

**選択肢**

1. シスプラチン
​
2. テガフール
​
3. ギメラシル
​
4. オテラシルカリウム
​
5. フルオロウラシル

**正解と理由**
- `1. シスプラチン`
- **理由の構成**
	- **問題の整理**：水溶液中でモル電気伝導率が時間経過で増大し、かつ抗がん作用を発現する薬物を選ぶ
	- **薬学的知識・仕様・制約**：モル電気伝導率の増大＝水溶液中の`イオン数`が増加すること。`シスプラチン`は水溶液中でCl⁻配位子が水分子に置換される（`アクア化`）反応が進行し、イオン数が増大する。この反応により生じたアクア体が`DNA`に架橋結合して抗がん作用を発現する
	- **選択理由**：イオン数増大＋抗がん作用の両条件を満たすのはシスプラチンのみ

**解法の手順（思考の流れ）**
- まず「モル電気伝導率の増大」＝`イオン数の増加`と解釈
- 選択肢中で金属イオンを含む（イオン数変化しうる）のは`シスプラチン`と`オテラシルカリウム`
- `オテラシルカリウム`は抗がん作用ではなく消化管毒性軽減が目的
- よって両条件を満たすのは`シスプラチン`のみと結論づけた

**他の選択肢が誤りな理由**
- 2. テガフール
	`テガフール`は`5-FU`のプロドラッグで、体内で代謝されて5-FUとなる。水溶液中でイオン数が増大する反応は起こらない

- 3. ギメラシル
	`ギメラシル`はDPD（ジヒドロピリミジンデヒドロゲナーゼ）阻害薬。5-FUの分解を抑制する役割で、水溶液中でのイオン数変化は起こらない

- 4. オテラシルカリウム
	`オテラシルカリウム`はカリウム塩で解離するためイオンは存在するが、抗がん作用は持たない。消化管のオロト酸ホスホリボシルトランスフェラーゼを阻害し、5-FUの`消化管毒性`を軽減する

- 5. フルオロウラシル
	`フルオロウラシル`（5-FU）は抗がん作用を持つが、水溶液中でイオン数が時間経過で増大する反応は起こらない

**キーポイント**
- `シスプラチン`のアクア化反応：[Pt(NH₃)₂Cl₂] → [Pt(NH₃)₂(H₂O)Cl]⁺ + Cl⁻。イオン数が増えるため`モル電気伝導率`が増大
- アクア化されたシスプラチンが`DNA`のグアニンN7に結合し、`鎖内架橋`を形成して抗がん作用を発現
- シスプラチンを`生理食塩液`で希釈する理由：高Cl⁻濃度環境ではアクア化が抑制され、投与前の安定性が保たれる

**関連知識の比較**
- `オキサリプラチン`は第3世代白金製剤で、シスプラチンと異なりCl配位子の代わりに`シュウ酸配位子`を持つ。5%ブドウ糖液で希釈する（生食NG）
- `S-1`の3成分のモル比はテガフール：ギメラシル：オテラシル＝1：0.4：1
- 電気伝導率の問題は物理系で白金製剤が絡むとほぼ`シスプラチン`のアクア化反応

（一言メモ）
- `シスプラチン`のアクア化＝Cl⁻が外れて水が入る→イオン増える→`電気伝導率UP`。これが抗がん活性体
- 「生食で溶かす理由」と「電気伝導率が変化する理由」は同じ原理（Cl⁻濃度とアクア化の平衡）
- `オテラシルカリウム`はK⁺を放出するが抗がん作用なし。この「イオンは出すけど効果が違う」が引っかけ
