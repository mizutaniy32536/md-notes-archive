# 108-121

**第108回薬剤師国家試験問121の問題文は、「食品の変質のうち、HbA1c の生成と同様に、糖化反応により起こるのはどれか。2 つ選べ。」です。正解は**2, 4**です。[1][6]

### 全選択肢（番号付き）
1. 食品が腐敗するとヒスタミンが生成する。[1][6]  
2. ポテトチップスを製造するとアクリルアミドが生成する。[1][6]  
3. 砂糖水を 150 ～ 200 ℃ で加熱すると茶色く変色する。[1][6]  
4. パンを焼くと茶色く変色する。[1][6]  
5. リンゴの切り口が茶色く変色する。[1][6]

### 解説
HbA1cの生成は非酵素的糖化反応（メイラード反応）によるもので、アミノ基と糖が反応して褐色化や生成物が生じる。[1][3]  
- **1は誤り**：腐敗によるヒスタミン生成はヒスチジンの代謝反応で、糖化反応ではない。[1]  
- **2は正しい**：ポテトチップス製造時のアクリルアミド生成はメイラード反応（糖化反応）に関連する。[1][3]  
- **3は誤り**：砂糖水の加熱変色はキャラメル化反応で、メイラード反応とは異なる（アミノ酸が関与しない）。[3]  
- **4は正しい**：パンを焼く際の茶色く変色はアミノ酸＋糖によるメイラード反応の代表例。[1][3]  
- **5は誤り**：リンゴの褐変はポリフェノールの酵素的酸化（酵素的褐変）で、糖化反応ではない。[1][3]
