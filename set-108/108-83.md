# 108-83

薬剤師法に基づき薬袋への記載が必須である事項はどれか。１つ選べ。

１ 調剤した薬剤師氏名
２ 処方した医師氏名
３ 使用上の注意
４ 処方箋発行日
５ 医薬品名

**正解:** １

**解説**
薬袋に記載する事項は薬剤師法第25条同施行規則14条により、①処方箋に記載された患者氏名、②用法及び用量、③調剤年月日、④調剤した薬剤師氏名、⑤調剤した薬局などの名称及び所在地となっている。
