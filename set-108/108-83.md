# 108-83

**問題文**
薬剤師法に基づき薬袋への記載が必須である事項はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-83 を参照

**選択肢**
1. 調剤した薬剤師氏名
2. 処方した医師氏名
3. 使用上の注意
4. 処方箋発行日
5. 医薬品名

**正解:** 1

**解説**
解答　１

薬袋に記載する事項は薬剤師法第25条同施行規則14条により、①処方箋に記載された患者氏名、②用法及び用量、③調剤年月日、④調剤した薬剤師氏名、⑤調剤した薬局などの名称及び所在地となっている。
