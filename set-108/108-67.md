# 108-67

**問題文**
マイコプラズマ肺炎の治療に用いられるのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-67 を参照

**選択肢**
1. アシクロビル錠
2. イベルメクチン錠
3. テビペネムピボキシル細粒
4. アジスロマイシン錠
5. リネゾリド錠

**正解:** 4

**解説**
解答　４

マイコプラズマは、細菌とウイルスの中間の性質を示す微生物で、一般細菌とは異なり細胞壁をもたない。そのため、βラクタム系抗菌薬は全て無効であり、治療にはマクロライド系抗菌薬が第一選択として用いられる。
