# 108-90

**問題文**
膀胱炎に対して、レボフロキサシン水和物の用法及び用量を検討する上で、最も必要な患者情報はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-90 を参照

**選択肢**
1. クレアチニンクリアランス
2. クレアチンキナーゼ値
3. CRP値
4. 白血球数
5. 尿酸値

**正解:** 1

**解説**
解答　１

レボフロキサシン水和物は、通常、成人には1回500 mgを1日1回経口投与するが、腎機能低下患者が服用すると高い血中濃度が持続するため、クレアチニンクリアランス（CLcr）をもとに用法及び用量を検討する。
以下に、レボフロキサシン錠（クラビット®︎錠）の添付文書に記載された用法及び用量の目安を示す。
