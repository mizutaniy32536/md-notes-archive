# 108-135

**第108回薬剤師国家試験 問135**  
化審法＊及び化管法＊＊に関する記述のうち、正しいのはどれか。**2つ選べ**。[2][4][5][8]

＊化審法：化学物質の審査及び製造等の規制に関する法律  
＊＊化管法：特定化学物質の環境への排出量の把握等及び管理の改善の促進に関する法律[2][4]

**選択肢**  
1. 化審法の対象となる物質は、新規化学物質のみであり、化審法の公布前に製造・輸入されていた既存化学物質は、対象とはならない。[2][4][5]  
2. 化審法における監視化学物質は、難分解性、低蓄積性であり、ヒト及び生活環境動植物に対して長期毒性を有する物質である。[2][4][5]  
3. 化審法において、分解性の判定には活性汚泥が用いられる。[4][5]  
4. 化管法において、PRTR制度の対象物質は、SDS制度の対象物質に指定されている。[2]  
5. 化審法における監視化学物質は、難分解性、高蓄積性であり、ヒト及び生活環境動植物に対して長期毒性が不明な物質である。[5]  

**正解番号：3, 5**[2][4][5]

**解説**  
- **選択肢1 誤**：化審法の対象は新規化学物質だけでなく、既存化学物質も含む。一般化学物質として第二条第七項に基づき規定されている。[2][4]  
- **選択肢2 誤**：監視化学物質は難分解性・**高蓄積性**であり、長期毒性が不明な物質。低蓄積性ではない。[2][4][5]  
- **選択肢3 正**：化審法の分解性判定に活性汚泥（好気性微生物群を含む）を用いた分解度試験が行われる。[4][5]  
- **選択肢4 正誤不明**（一部ソースで正解候補だが、公式確認不足。主正解は3,5）[2]  
- **選択肢5 正**：監視化学物質の定義が正確。難分解性・高蓄積性で、ヒト健康や高次捕食動物への影響が不明な場合に大臣が指定。[2][5]  

正解は化審法の分解性判定法と監視化学物質の性質に関する記述。[2][4][5]
