# 108-135

化審法＊及び化管法＊＊に関する記述のうち、正しいのはどれか。2つ選べ。

＊化審法：化学物質の審査及び製造等の規制に関する法律
＊＊化管法：特定化学物質の環境への排出量の把握等及び管理の改善の促進に関する法律

1. 化審法の対象となる物質は、新規化学物質のみであり、化審法の公布前に製造・輸入されていた既存化学物質は、対象とはならない。
2. 化審法における監視化学物質は、難分解性、低蓄積性であり、ヒト及び生活環境動植物に対して長期毒性を有する物質である。
3. 化審法の分解性試験において、活性汚泥を用いた試験は、好気性条件下での分解性を判定するものである。
4. 化管法のPRTR制度では、対象事業者は対象化学物質の環境への排出量を自ら把握し、都道府県知事に届け出なければならない。
5. 化管法のSDS制度の対象には、PRTR制度の第一種指定化学物質及び第二種指定化学物質が含まれる。

## 正解
**3, 5**

## 解説

- 選択肢1：誤り。2009年改正で既存化学物質も一般化学物質として対象。
- 選択肢2：誤り。監視化学物質は難分解性＋高蓄積性＋長期毒性不明の物質。
- 選択肢3：正しい。活性汚泥（好気性微生物群）を用いて分解度を測定。
- 選択肢4：誤り。PRTR制度の届出先は主務大臣（経産大臣・環境大臣等）。
- 選択肢5：正しい。SDS制度は第一種（PRTR対象）と第二種の両方が対象。

化審法の物質分類：第一種特定（難分解・高蓄積・長期毒性あり）、監視（難分解・高蓄積・毒性不明）
化管法の2本柱：PRTR（排出届出→主務大臣）＋ SDS（情報提供）
