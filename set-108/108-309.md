# 108-309

### 第108回薬剤師国家試験 問309

#### 問題文（全文）
ある薬局は、地域の飲食店等からの要望を受け、毒物劇物販売業の登録を申請し、劇物である苛性ソーダ（水酸化ナトリウム）の取扱いを行っている。近所の飲食店に勤務する者が、苛性ソーダを売って欲しいと来局した。来局者の年齢と使用目的を確認したところ、年齢は19歳で、厨房を清掃するための購入であり、以前にも一度使用した経験があるとのことであった。なお、この飲食店は毒物劇物営業者ではない。[1][2][3]

この来局者に薬剤師が苛性ソーダを販売するにあたり、適切な対応はどれか。**2つ選べ**。[1][2][3]

#### 全選択肢（番号付き）
1. この薬局の毒物劇物販売業の登録が有効期限内であることを確認する。[1][2][3]  
2. この来局者に苛性ソーダの性状及び取扱いに関する情報を提供する。[1][2][3]  
3. この薬局にて、この来局者の処方箋の調剤を行ったことがあり、身元を把握しているので、譲受書の受取りに関する手続きを省略する。[1][2][3]  
4. 水剤の調剤で用いる内服液剤容器に入れて販売する。[1]  
5. この来局者の年齢では、法律上販売できない旨を説明する。[2][3]

#### 正解番号
**1, 2**[1][2]

#### 解説
毒物及び劇物取締法に基づき、劇物（苛性ソーダ）の販売にあたり、薬局は毒物劇物販売業の登録が有効であることを確認する必要がある（選択肢1）。また、譲受人（来局者）に対して性状・取扱いに関する情報を提供することが義務付けられている（選択肢2）。[1][2]

- 選択肢3は誤り。譲受書の手続きは身元確認の有無にかかわらず必須であり、省略できない。[1][2]  
- 選択肢4は誤り。劇物は原容器または適切な表示容器で販売する必要があり、内服液剤容器への移し替えは不適切。[1]  
- 選択肢5は誤り。19歳は成人であり、劇物販売に年齢制限はない。[2][3]
