# 108-309

【シナリオ（108-308と共通）】
ある薬局は、地域の飲食店等からの要望を受け、毒物劇物販売業の登録を申請し、劇物である苛性ソーダ（水酸化ナトリウム）の取扱いを行っている。近所の飲食店に勤務する者が、苛性ソーダを売って欲しいと来局した。来局者は19歳、厨房清掃目的、使用経験あり。飲食店は毒物劇物営業者ではない。

**この来局者に薬剤師が苛性ソーダを販売するにあたり、適切な対応はどれか。2つ選べ。**

1. この薬局の毒物劇物販売業の登録が有効期限内であることを確認する。
2. この来局者に苛性ソーダの性状及び取扱いに関する情報を提供する。
3. この薬局にて、この来局者の処方箋の調剤を行ったことがあり、身元を把握しているので、譲受書の受取りに関する手続きを省略する。
4. 水剤の調剤で用いる内服液剤容器に入れて販売する。
5. この来局者の年齢では、法律上販売できない旨を説明する。

## 正解
**1, 2**

## 解説

| 選択肢 | 正誤 | 理由 |
|---|---|---|
| 1. 登録の有効期限確認 | ⭕ | 販売前に登録が有効であることの確認は必須 |
| 2. 性状・取扱い情報を提供 | ⭕ | 毒物劇物販売者には情報提供義務がある |
| 3. 身元把握済みで譲受書省略 | ❌ | 譲受書の手続きは省略不可。身元確認済みでも法定手続きは必須 |
| 4. 内服液剤容器で販売 | ❌ | 飲み薬と誤認する危険。劇物を医薬品用容器で販売してはならない |
| 5. 19歳は販売不可 | ❌ | 毒物劇物の交付制限は18歳未満。19歳は販売可能 |

### 毒物劇物の交付制限対象
- 18歳未満の者
- 心身の障害により適正に取り扱えない者
- 麻薬、大麻、あへん又は覚醒剤の中毒者

> 💡 19歳＝成人（18歳以上）なので交付制限に該当しない
