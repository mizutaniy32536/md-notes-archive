



## ◆ 低ナトリウム血症患者への塩化ナトリウム投与に関するICU薬剤師の対応

260313

**問題**
- 87歳男性。畑仕事中に意識がもうろうとなり、A病院に救急搬送された。現病歴と服用している薬剤の有無は不明であった。検査の結果、血清ナトリウム値が108 mEq/Lと低ナトリウム血症を認めた。治療のため、3%塩化ナトリウム水溶液での点滴加療を開始することにした。
A病院の医療安全マニュアルには、生理食塩液に対する浸透圧比4を超える注射液を投与する場合は中心静脈より投与することと記載されている。
- ICU担当薬剤師の対応に関する記述のうち、誤っているのはどれか。１つ選べ。なお、塩化ナトリウムの式量は58.5とし、水溶液中で完全電離しているものとする。

**選択肢**

1. 処方どおりに調製すれば、3%塩化ナトリウム水溶液となるため、問題なしと判断した。
​
2. 算出された浸透圧比が4を超えていたので、中心静脈からの投与を依頼した。
​
3. 低ナトリウム血症の更なる悪化は、痙れんや昏睡を起こす可能性があると情報提供した。
​
4. 急激な血清ナトリウム値の上昇は、浸透圧性脱髄症候群を起こす可能性があると情報提供した。
​
5. 低ナトリウム血症の原因として、抗利尿ホルモン不適合分泌症候群（SIADH）も考えられると情報提供した。

**正解と理由**
- `2. 算出された浸透圧比が4を超えていたので、中心静脈からの投与を依頼した。`
- **理由の構成**
	- **問題の整理**：3%NaCl水溶液の浸透圧比を計算し、投与経路の判断が正しいかを評価
	- **薬学的知識・仕様・制約**：生理食塩液（0.9% NaCl）のオスモル濃度は308 mOsm。3% NaClのオスモル濃度は1026 mOsm。浸透圧比＝1026/308＝約`3.3`。これは4を超えていないため、`末梢静脈`からの投与でよい
	- **選択理由**：計算結果が4未満なのに「4を超えていた」と誤って判断している点が誤り

**解法の手順（思考の流れ）**
- まず`浸透圧比`の計算を行う
- 生理食塩液：0.9g/100mL → NaCl 0.9/58.5 × 1000 = 154 mmol/L → Na⁺ 154 + Cl⁻ 154 = `308 mOsm`
- 3% NaCl：3g/100mL → NaCl 30/58.5 × 1000 ≈ 513 mmol/L → `1026 mOsm`
- 浸透圧比 = 1026/308 ≈ `3.3` → 4以下 → 末梢投与OK
- 選択肢2は「4を超えていた」と誤った判断をしているため誤りと結論

**他の選択肢が誤りな理由**
- 1. 処方どおりに調製すれば、3%塩化ナトリウム水溶液となるため、問題なしと判断した。
	正しい。生理食塩液500mL（NaCl 4.5g）＋10% NaCl注150mL（NaCl 15g）＝ 650mL中19.5g → 19.5/650×100 = `3%`で計算通り

- 3. 低ナトリウム血症の更なる悪化は、痙れんや昏睡を起こす可能性があると情報提供した。
	正しい。重度の`低Na血症`（<120 mEq/L）は脳浮腫を引き起こし、`痙攣`や`昏睡`の原因となる

- 4. 急激な血清ナトリウム値の上昇は、浸透圧性脱髄症候群を起こす可能性があると情報提供した。
	正しい。急速な補正は`浸透圧性脱髄症候群`（ODS、旧称：橋中心髄鞘崩壊症）を起こすため、補正速度は24時間で`10〜12 mEq/L以下`が推奨

- 5. 低ナトリウム血症の原因として、SIADHも考えられると情報提供した。
	正しい。`SIADH`（ADH不適合分泌症候群）は希釈性低Na血症の代表的原因。高齢者の薬剤性SIADHも多い

**キーポイント**
- 浸透圧比の計算：3% NaCl ≈ `3.3`（4未満なので末梢投与可能）
- 低Na血症の補正速度は24時間で`10〜12 mEq/L以下`。急速補正は`ODS`のリスク
- `SIADH`はSSRI、カルバマゼピン、シクロホスファミドなど多くの薬剤で誘発される

**関連知識の比較**
- `高張食塩水`（3% NaCl）は重症低Na血症（痙攣・意識障害）の緊急治療に使用
- `トルバプタン`はV2受容体拮抗薬で、SIADHによる低Na血症の治療薬。水のみ排泄（水利尿）させる
- 浸透圧比4超＝`中心静脈`投与が必要。高カロリー輸液（TPN）は通常浸透圧比5〜6で中心静脈必須

（一言メモ）
- 浸透圧比の計算：「NaClのmmol/L × 2（Na⁺とCl⁻）÷ 308」で概算できる
- 低Na血症の補正は「`急ぎすぎると橋が壊れる`」（浸透圧性脱髄症候群）。24時間で10 mEq/L以下を厳守
- `SIADH`の薬剤性原因は頻出。`SSRI`、`カルバマゼピン`、`ビンクリスチン`を押さえる
