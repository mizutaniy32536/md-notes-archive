# 108-120

### 第108回薬剤師国家試験 問120

**問題文（全文）**  
HbA1cに関する記述として、正しいのはどれか。**2つ選べ**。[1][2]

**全選択肢（番号付き）**  
1. HbA1cの生成には酵素が必要である。  
2. HbA1cの生成はグルコース濃度に依存する。  
3. （検索結果に明示なし）  
4. （検索結果に明示なし）  
5. HbA1c値は、赤血球寿命の影響を受ける。[2]

**正解番号**  
**2** と **5**。[6][8]

**解説**  
HbA1cは、赤血球中のヘモグロビンA（HbA）のβ鎖N末端のアミノ基にグルコースが非酵素的に結合（糖化反応）して生成される。この反応は酵素を必要とせず（選択肢1誤り）、血中グルコース濃度が高いほど生成量が増加するため濃度依存性がある（選択肢2正しい）。また、HbA1c値は赤血球の寿命（通常120日程度）によって反映される過去1～2か月の平均血糖値を表すため、赤血球寿命の変化（例: 溶血性貧血で短縮）で影響を受ける（選択肢5正しい）。他の選択肢は糖化反応の非酵素性や非依存性を誤解した誤り。[1][2][6]
