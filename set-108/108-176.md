# 108-176

**問題文**
粘弾性の2要素モデルに関する記述のうち、正しいのはどれか。2つ選べ。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-176 を参照

**選択肢**
1. Aのスプリングは弾性体のモデル、Bのダッシュポットは粘性体のモデルである。
2. フォークトモデル全体に一定の応力を瞬時にかけてそれを保つとき、モデル全体のひずみは直線的に増加する。
3. マックスウェルモデル全体に加わる応力は、スプリングに加わる応力とダッシュポットに加わる応力の合計である。
4. マックスウェルモデル全体に一定のひずみを与えてそのまま保持するとき、モデル全体にかかる応力は減衰する。
5. マックスウェルモデル全体に一定の応力を瞬時にかけてそれを保つとき、モデル全体のひずみは一定に保たれる。

**正解:** 1,4

**解説**
解答　１、４

粘弾性の2要素モデルにおいて、Aのスプリングは弾性体のモデルであり、Bのダッシュポットは粘性体のモデルである。また、マックスウェルモデルは、スプリングとダッシュポットを直列に繋ぎ合わせたモデルであり、フォークトモデルは、スプリングとダッシュポットを並列に繋ぎ合わせたモデルである。

１　正
前記参照

２　誤
フォークトモデル全体に一定の応力を瞬時にかけてそれを保つとき、スプリングは瞬時に伸びようとするが、ダッシュポットの抵抗により、モデル全体のひずみは時間と共に徐々に増加する。

３　誤
スプリングに加わる応力とダッシュポットに加わる応力の合計で全体に加わる応力を表現できるのは、フォークトモデルである。

４　正

５　誤
マックスウェルモデル全体に一定の応力を瞬時にかけてそれを保つとき、スプリングは瞬時に伸び、ダッシュポットは徐々に伸びるため、モデル全体のひずみは一定に保たれない。
