# 108-66

**問題文**

**解説**
解答　１

メトトレキサート・ホリナート救援療法では、尿が酸性側に傾くと、メトトレキサートの結晶が尿細管に沈着し、副作用として腎障害を引き起こすおそれがあるため、炭酸水素ナトリウムやアセタゾラミドによる尿のアルカリ化を行い、メトトレキサートの尿への排泄を促す必要がある。
