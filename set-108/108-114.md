# 108-114

**問題文**
二次元電気泳動は、等電点電気泳動とSDSポリアクリルアミドゲル電気泳動（SDS−PAGE）を組み合せたもので、ある生体試料中の多種多様なタンパク質をそれぞれ固有の等電点と分子量の違いを利用して分離する方法である。図1と図2は、二次元電気泳動の概要と実験結果を示したものである。図1　等電点電気泳動：pH4〜10の連続的なpH勾配を形成させた棒状のゲルを準備する。このゲルを用いて電気泳動すると、試料中の各タンパク質はそれぞれの等電点の位置までゲル中を移動する。（注：　実際にはこの段階ではゲルは染色しないためタンパク質は見えないが、図1の下段では電気泳動後にタンパク質が等電点で分離しているイメージを示した。）＜image1:center＞図2　SDS−PAGE：等電点電気泳動により試料中のタンパク質を分離した棒状ゲルを、SDS−PAGEの分離ゲルの上に移し、一次元目の等電点電気泳動と直角の方向に電気泳動する。タンパク質は、その分子量に応じた位置まで移動する。電気泳動の終了後、泳動用のガラス板から取り出したゲルを洗浄し、クマシーブリリアントブルーでタンパク質を染色したところ、50個のスポットを検出した。図2中のスポットAは、等電点5.8、分子量56,000のタンパク質である。＜image2:center＞このようなタンパク質分析法とその実験結果に関する記述のうち、誤っているのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-114 を参照

**選択肢**
1. 等電点が5.8より小さく、かつ分子量が56,000より大きいスポットは、4つ検出されている。
2. スポットAのタンパク質では、中性溶液中での分子全体の電荷が負になる。
3. スポットBのタンパク質の等電点は、ほぼ5.8である。
4. スポットCのタンパク質は、分子量56,000より小さい。
5. タンパク質のスポットの位置は、リン酸化により、やや右上方向に移動すると推測される。

**正解:** 3

**解説**
解答　３

二次元電気泳動の実験結果は、下記のように読み取ることができる。

１　正しい
等電点が5.8より小さく、かつ分子量が56,000より大きいスポットは、4つ検出されている。

２　正しい
タンパク質は自身の等電点とpHが等しい溶液中で、分子全体の電荷がゼロになる。また、タンパク質は自身の等電点よりpHの低い溶液中では分子全体の電荷が正になり、pHの高い溶液中では分子全体の電荷が負になる。スポットAのタンパク質は等電点が5.8であるため、このタンパク質の等電点よりpHの高い中性溶液（pH7程度）中では、分子全体の電荷が負になる。

３　誤っている
スポットBのタンパク質の等電点は、約6.6である。

４　正しい
スポットCのタンパク質は、分子量56,000より小さい。

５　正しい
タンパク質がリン酸化されると、等電点はわずかに小さくなり、分子量がわずかに大きくなるため、やや右上方向に移動すると推測される。
