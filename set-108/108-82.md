# 108-82

**問題文**
成人に対する栄養輸液療法における静注用脂肪乳剤（20%イントラリポス輸液250 mL袋）の投与方法として、最も適切なのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-82 を参照

**選択肢**
1. 遮光して投与する。
2. 他の栄養輸液製剤に混合して投与する。
3. フィルターを使用して投与する。
4. シリンジポンプを使用して投与する。
5. ゆっくり（3時間以上かけて）投与する。

**正解:** 5

**解説**
解答　５

１　誤
静注用脂肪乳剤は、遮光する必要はない。

２　誤
静注用脂肪乳剤は、内容液が白く、他の栄養輸液製剤と混合すると沈殿や混濁が生じても分かりにくくなるため、他の栄養輸液製剤には混合しない。

３　誤
静注用脂肪乳剤は、フィルターに通すと目詰まりを起こすため、フィルターは使用しない。

４　誤
静注用脂肪乳剤は、ゆっくり（3時間以上かけて）点滴静注することとされているが、シリンジポンプは使用しない。なお、シリンジポンプは、厳密な投与量管理が必要な循環器系薬や昇圧薬などを点滴持続静注する際に使用される。

５　正
