# 108-19

**問題文**
保健機能食品制度において、栄養機能食品として栄養機能表示ができない食品成分はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-19 を参照

**選択肢**
1. ナトリウム
2. n−3系脂肪酸
3. カルシウム
4. パントテン酸
5. ビタミンA

**正解:** 1

**解説**
解答　１

保健機能食品制度において、栄養機能食品は、身体の健全な成長、発達、健康の維持に必要な栄養成分（ミネラル、ビタミン等）の補給・補完を目的としたものである。ナトリウムは、人体に必須の栄養素ではあるが、通常の食生活から十分な量が摂取されるため、栄養機能食品として栄養機能表示ができない成分である。
