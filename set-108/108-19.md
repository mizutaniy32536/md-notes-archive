# 108-19

保健機能食品制度において、栄養機能食品として栄養機能表示ができない食品成分はどれか。1つ選べ。

1. ナトリウム
2. n-3系脂肪酸
3. カルシウム
4. パントテン酸
5. ビタミンA

## 正解
**1. ナトリウム**

## 解説

### 栄養機能食品とは
不足しがちな栄養成分の補給を目的とした食品で、国が定めた基準を満たせば届出不要で表示できる。

### 表示対象の栄養成分
- **ビタミン**：A、B₁、B₂、B₆、B₁₂、C、D、E、K、ナイアシン、パントテン酸、ビオチン、葉酸
- **ミネラル**：カルシウム、鉄、亜鉛、銅、マグネシウム
- **n-3系脂肪酸**：EPA・DHA

### ナトリウムが対象外の理由
- ❌ **ナトリウム（塩分）は過剰摂取が問題**
- 日本人の食生活では不足ではなく摂りすぎが課題
- むしろ減塩が推奨されている

### 各選択肢の検証
1. ❌ ナトリウム → 過剰摂取が問題、表示対象外
2. ⭕ n-3系脂肪酸 → 不足しがち、表示可能
3. ⭕ カルシウム → 不足しがち、表示可能
4. ⭕ パントテン酸 → ビタミンB群、表示可能
5. ⭕ ビタミンA → 不足しがち、表示可能

### ポイント
栄養機能食品の表示対象 = **不足しがちな栄養素**のみ
