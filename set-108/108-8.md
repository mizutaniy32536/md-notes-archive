



## ◆ 生薬の基原部位を判断するシナリオ

260313

**問題**
- 基原が根皮の生薬はどれか。１つ選べ。

**選択肢**

1. ボタンピ
​
2. ケイヒ
​
3. コウボク
​
4. オウバク

**正解と理由**
- `1. ボタンピ`
- **理由の構成**
	- **問題の整理**：5つの生薬から「根皮」を基原とするものを選ぶ
	- **薬学的知識・仕様・制約**：`ボタンピ`はボタン科ボタンの`根皮`を用いる生薬。他の4つはすべて`樹皮`を基原とする
	- **選択理由**：「根皮」と「樹皮」の違いが問われている。ボタンピのみが根の皮（根皮）を使用し、残りはすべて幹や枝の皮（樹皮）

**解法の手順（思考の流れ）**
- まず「根皮」というキーワードに注目。基原が「根の皮」である生薬は限られる
- `ボタンピ`の名前に「皮」が含まれ、ボタンの根皮であることを想起
- 他の選択肢（ケイヒ、コウボク、オウバク、トチュウ）はいずれも樹皮であることを確認
- よって選択肢1が正解と結論づけた

**他の選択肢が誤りな理由**
- 2. ケイヒ
	`ケイヒ`はクスノキ科ケイの`樹皮`を用いる生薬。根皮ではない。主成分は`ケイアルデヒド`（シンナムアルデヒド）

- 3. コウボク
	`コウボク`はモクレン科ホオノキの`樹皮`を用いる生薬。根皮ではない。主成分は`マグノロール`、`ホノキオール`

- 4. オウバク
	`オウバク`はミカン科キハダの`樹皮`を用いる生薬。根皮ではない。主成分は`ベルベリン`

- 5. トチュウ
	`トチュウ`はトチュウ科トチュウの`樹皮`などを用いる生薬。根皮ではない。折ると白い糸（`グッタペルカ`）が引ける特徴がある

**キーポイント**
- `ボタンピ`は国試頻出の「根皮」を基原とする代表的生薬
- 「皮」がつく生薬でも「根皮」と「樹皮」の違いを正確に区別する
- ボタンピの主成分は`ペオノール`（消炎・鎮痛作用）

**関連知識の比較**
- 「根」を基原とする生薬の例：`カンゾウ`（マメ科、根及びストロン）、`オウギ`（マメ科、根）、`サイコ`（セリ科、根）
- 「樹皮」を基原とする生薬：`ケイヒ`、`コウボク`、`オウバク`、`トチュウ`、`キョウカツ`など
- `オウバク`の`ベルベリン`は黄色のイソキノリンアルカロイドで、止瀉薬・健胃薬として使用される

（一言メモ）
- 根皮の生薬＝`ボタンピ`は鉄板。「牡丹の根の皮」で覚える
- `オウバク`の`ベルベリン`は止瀉薬の正露丸にも含まれ、抗菌作用を持つ
- `トチュウ`の特徴は折ると出る白い糸（`グッタペルカ`様物質）。視覚的に覚えやすい
