# 108-62

遺伝子多型により、フェニトインの体内動態に最も影響を及ぼす代謝酵素はどれか。１つ選べ。

１ CYP2C9
２ NAT2
３ CYP2D6
４ UGT1A1
５ CYP2C19

> ⚠️ 連問のため、問題文は e-REC 108-60 ページを参照

**正解:** １

**解説**
フェニトインは主にCYP2C9により代謝される。そのため、CYP2C9の遺伝子多型のpoor metabolizer（PM）では、フェニトインの代謝が低下し、血中濃度が上昇することで、フェニトインによる中枢毒性などの副作用発現のリスクが増大する。
