# 108-62

**問題文**
急性膵炎で通常認められる所見はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-62 を参照

**選択肢**
1. 血中ヘモグロビン量の減少
2. 血中リパーゼ活性の低下
3. 白血球数の減少
4. 心窩部痛
5. 右腕への放散痛

**正解:** 4

**解説**
解答　４

急性膵炎は、膵酵素が何らかの原因によって膵内で異常活性化を受け、膵臓と周囲組織を自己消化する急性炎症性疾患である。典型的な症状としては、持続性で耐え難いほどの増悪する上腹部痛（心窩部痛など）がみられる。

１　誤
急性膵炎では通常、血中ヘモグロビン量の変化はみられない。

２　誤
急性膵炎では通常、血中膵酵素（リパーゼなど）活性の上昇がみられる。

３　誤
急性膵炎は急性炎症性疾患であるため、白血球数の増加がみられる。

４　正
上記参照。

５　誤
急性膵炎では通常、上背部への放散痛がみられる。
