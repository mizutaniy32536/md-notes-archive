# 108-154

抗てんかん薬に関する記述のうち、正しいのはどれか。２つ選べ。

１ ガバペンチンは、γ−アミノ酪酸GABAA
２ ラモトリギンは、K＋
３ ペランパネルは、シナプス後膜に存在するグルタミン酸AMPA受容体を遮断して、グルタミン酸による神経細胞の過剰興奮を抑制する。
４ カルバマゼピンは、電位依存性T型Ca2＋
５ レベチラセタムは、シナプス小胞タンパク質2A（SV2A）に結合して、神経伝達物質の遊離を抑制する。

**正解:** ３、５

**解説**
１　誤
ガバペンチンは、電位依存性Ca2＋チャネルのα2δサブユニットに結合し、Ca2＋の流入を抑制することで、興奮性神経伝達物質の遊離を抑制する。なお、本剤は脳内GABA量を増加させることで、GABA作動性神経伝達を増強する作用を併せもつ。
２　誤
ラモトリギンは、電位依存性Na＋チャネルを遮断することで、グルタミン酸などの興奮性神経伝達物質の遊離を抑制する。
３　正
ペランパネルは、シナプス後膜に存在するグルタミン酸AMPA受容体を選択的かつ非競合的に遮断することで、グルタミン酸による神経細胞の過剰興奮を抑制する。
４　誤
カルバマゼピンは、電位依存性Na＋チャネルを遮断し、神経細胞の過剰な興奮を抑制する。なお、電位依存性T型Ca2＋チャネルを遮断する薬物としては、エトスクシミドなどがあげられる。
５　正
