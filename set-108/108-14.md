# 108-14

**問題文**
サイクリックAMP（cAMP）の結合により活性化されるリン酸化酵素はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-14 を参照

**選択肢**
1. アデニル酸シクラーゼ
2. cAMPホスホジエステラーゼ（PDE Ⅲ）
3. プロテインキナーゼA
4. プロテインキナーゼB（Akt）
5. プロテインキナーゼC

**正解:** 3

**解説**
解答　３

サイクリックAMPは、アデニル酸シクラーゼによってATPから合成されるセカンドメッセンジャーの1つであり、細胞内のシグナル伝達において、様々な生理的プロセスを制御している。サイクリックAMPがプロテインキナーゼAのカタリティックCサブユニットに結合すると、プロテインキナーゼAの活性が促進され、様々なタンパク質をリン酸化する。それに伴い、心機能亢進などを起こす。
