## ◆ cAMP結合で活性化されるキナーゼを選ぶシナリオ
260312

**問題**
サイクリックAMP（cAMP）の結合により活性化されるリン酸化酵素はどれか。1つ選べ。

**選択肢**

1. アデニル酸シクラーゼ
​
2. cAMPホスホジエステラーゼ（PDEⅢ）
​
3. プロテインキナーゼA
​
4. プロテインキナーゼB（Akt）
​
5. プロテインキナーゼC

**正解と理由**
- `3. プロテインキナーゼA`
- **理由の構成**
	- PKAは`cAMP依存性`のセリン/スレオニンキナーゼ
	- cAMPが調節サブユニットに結合 → 触媒サブユニットが遊離 → 活性化

**解法の手順（思考の流れ）**
- 選択肢1：アデニル酸シクラーゼ(AC) = cAMPを`作る`酵素（ATP→cAMP）。リン酸化酵素ではない → ✗
- 選択肢2：PDE = cAMPを`分解する`酵素（cAMP→AMP）。リン酸化酵素ではない → ✗
- 選択肢3：PKA = cAMPが結合して活性化される`リン酸化酵素` → **○**
- 選択肢4：PKB(Akt) = PI3K経路で活性化。cAMPとは無関係 → ✗
- 選択肢5：PKC = DAG(ジアシルグリセロール)＋Ca²⁺で活性化。cAMPとは無関係 → ✗

**キーポイント**
- cAMPシグナル経路：受容体 → Gsタンパク質 → AC → cAMP → `PKA`
- PKA = 「cAMP依存性プロテインキナーゼ」が正式名称
- PKB = PI3K経路、PKC = PLC経路

**関連知識の比較**

| キナーゼ | 活性化因子 | 経路 |
|---|---|---|
| PKA | **cAMP** | Gs/AC経路 |
| PKB(Akt) | PIP3 | PI3K経路 |
| PKC | DAG + Ca²⁺ | PLC経路 |

（一言メモ）
- PKAの「A」= cAMP依存性。名前で覚える
