# 108-86

主な感染経路が飛沫感染である疾患はどれか。１つ選べ。

１ 角化型疥癬
２ クロストリディオイデス・ディフィシル（クロストリジウム・ディフィシル）感染症
３ 流行性角結膜炎
４ マイコプラズマ肺炎
５ 腸管出血性大腸菌感染症

**正解:** ４

**解説**
１　誤
角化型疥癬は、ヒゼンダニが原因で発症する皮膚感染症であり、主な感染経路は接触感染である。
２　誤
クロストリディオイデス・ディフィシル（クロストリジウム・ディフィシル）は、偽膜性大腸炎の原因菌であり、主な感染経路は接触感染である。
３　誤
流行性角結膜炎は、アデノウイルスが原因で発症する角結膜炎であり、主な感染経路は接触感染である。
４　正
マイコプラズマ肺炎は、マイコプラズマが原因で発症する代表的な非定型肺炎であり、主な感染経路は飛沫感染である。
５　誤
腸管出血性大腸菌は、感染性食中毒の原因菌の一種であり、主な感染経路は経口感染である。
