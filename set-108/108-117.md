# 108-117

リソソーム及びプロテアソームに関する記述のうち、正しいのはどれか。２つ選べ。

１ リソソームは、内部が塩基性のオルガネラで、様々な加水分解酵素を有する。
２ エンドサイトーシスによりリソソームへ運ばれたタンパク質は、ATP依存的に分解される。
３ プロテアソームは、巨大な筒状のプロテアーゼ複合体であり、ポリユビキチン化されたタンパク質を選択的に分解する。
４ プロテアソームは、キラーT細胞へ提示されるウイルス由来タンパク質の分解に関わる。
５ プロテアソームは、オートファゴソーム内に取り込まれて、オートファジーにおけるタンパク質分解を担う。

> ⚠️ 連問のため、問題文は e-REC 108-116 ページを参照

**正解:** ３、４

**解説**
１　誤
リソソームは、細胞内で不要なタンパク質などの分解を担う細胞内小器官（オルガネラ）である。リソソームは、内部を酸性に維持し、様々な加水分解酵素が働きやすい状態にしており、細胞内外から取り込まれた物質の分解などを行う。
２　誤
エンドサイトーシスによってリソソーム内に取り込まれたタンパク質は、加水分解酵素によってATP非依存的に分解される。なお、ATP依存的にタンパク質を分解するのは、プロテアソームである。
３　正
プロテアソームは、ユビキチンという小さなタンパク質が結合したポリユビキチン化された、体内に不要となったタンパク質のみを選択的に分解し、必要なタンパク質は分解しない。
４　正
ウイルス由来のタンパク質は、細胞内でプロテアソームによって分解された後、MHCクラスI分子と結合してキラーT細胞に提示される。
５　誤
オートファジーによってタンパク質分解を行うのは、プロテアソームではなくリソソームである。
オートファジーが行われる際、細胞内で不要となったタンパク質は小胞体に包まれ、オートファゴソームを形成する。その後、オートファゴソームとリソソームが融合することにより、タンパク質などの分解などを行う。
