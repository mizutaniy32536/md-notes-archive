# 108-87

**問題文**
一般用医薬品に配合される成分のうち、透析療法を受けている人が使用を避ける必要があるのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-87 を参照

**選択肢**
1. ヒマシ油
2. アミノ安息香酸エチル
3. アルジオキサ
4. オキセサゼイン
5. テプレノン

**正解:** 3

**解説**
解答　３

透析療法を受けている人は、腎機能が低下しているためアルミニウムを尿中に排泄しづらく、また、アルミニウムは透析により除去することが難しいため、体内にアルミニウムが蓄積しやすく、アルミニウム脳症やアルミニウム骨症のリスクが高い。
したがって、透析療法を受けている人は、アルジオキサなどのアルミニウムを含む医薬品の使用を避ける必要がある。
