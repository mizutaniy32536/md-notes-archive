# 108-150

**問題文**
マズローの欲求の階層説において第1層にあたる欲求はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-150 を参照

**選択肢**
1. 苦痛、不安、恐怖を避けて安定した状態を保ちたい欲求
2. 人に愛されたいという欲求
3. 周囲から価値があると認められたい欲求
4. 自身の目標を達成したい欲求
5. 食事・排泄・睡眠などの欲求

**正解:** 5

**解説**
解答　５

米国の心理学者マズローは、人間の欲求の階層構造を5層に分類した。以下にそれぞれの階層の特徴を示す。

第1層：生理的欲求………………生命維持のため食べたいや飲みたいなど根源的な欲求
第2層：安心・安定の欲求………安全や安心を確保し、苦痛や恐怖を避け安定した状態を保ちたい
第3層：愛情と所属の欲求………家族、集団において、どこかに所属している満足感がほしい
第4層：承認・自己尊重欲求……集団から存在価値を認めてもらいたい
第5層：自己実現欲求……………自分の目標を達成したい
