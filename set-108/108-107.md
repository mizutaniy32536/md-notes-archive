# 108-107

**問題文**
図1及び図2は、アスパラギン酸プロテアーゼ及びシステインプロテアーゼによる基質加水分解の初期反応過程の模式図である。以下の記述のうち、正しいのはどれか。2つ選べ。＜image1:center＞

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-107 を参照

**選択肢**
1. アスパラギン酸残基アと水の相互作用は、水の酸素原子の求電子性を高めている。
2. アスパラギン酸残基イと基質カルボニル基の相互作用は、基質カルボニル基の求核性を高めている。
3. システイン残基エのスルファニル基は、塩基として働く。
4. ヒスチジン残基ウとシステイン残基エの相互作用は、エのスルファニル基の求核性を高めている。
5. アスパラギン酸残基アとヒスチジン残基ウは、塩基として働く。

**正解:** 4,5

**解説**
解答　４、５

１　誤
アスパラギン酸残基アと水の相互作用は、水の酸素原子の求核性を高めている。

２　誤
基質カルボニル基の炭素原子はδ＋に分極しているため、求核性を持たない。よって、アスパラギン酸残基イと基質カルボニル基の相互作用は、基質カルボニル基の求核性を高めているわけではない。

３　誤
システイン残基エのスルファニル基は、水素原子（プロトン）を引き抜く反応をしていないため、塩基として働いていない。なお、システイン残基エのスルファニル基は、δ＋に分極したカルボニル炭素と反応しているため、求核剤として働く。

４　正
ヒスチジン残基ウとシステイン残基エの相互作用は、エのスルファニル基の求核性を高めている。

５　正
アスパラギン酸残基アとヒスチジン残基ウは、共に水素原子（プロトン）を引き抜く反応をしているため、塩基として働く。
