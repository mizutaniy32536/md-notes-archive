# 108-28

**問題文**
抗うつ薬デュロキセチンの作用機序はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-28 を参照

**選択肢**
1. セロトニン5−HT1A受容体遮断
2. セロトニン5−HT2A受容体遮断
3. アドレナリンα2受容体遮断
4. セロトニン及びグルタミン酸の再取り込み阻害
5. セロトニン及びノルアドレナリンの再取り込み阻害

**正解:** 5

**解説**
解答　５

デュロキセチンは、選択的セロトニン・ノルアドレナリン再取り込み阻害薬（SNRI）であり、脳内におけるセロトニン及びノルアドレナリンの再取り込み阻害作用により、抗うつ作用を示す。
