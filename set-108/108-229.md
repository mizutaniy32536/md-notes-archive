# 108-229

（108-228と連問）主治医より、イソニアジド錠、リファンピシンカプセル、エタンブトール錠、ピラジナミド錠の4剤でDOTSを開始することが薬剤師に伝えられた。薬剤の服用にあたり、薬剤師が主治医に伝えるべき内容として、正しいのはどれか。2つ選べ。

1. 肝機能障害があるため、イソニアジドの使用を避けるべきである。
2. ワルファリンの作用が減弱することがある。
3. タクロリムスの血中濃度が低下することがある。
4. 腎機能障害があるため、エタンブトールを他剤に変更すべきである。
5. タクロリムスの血中濃度が上昇することがある。

## 正解
**2, 3**

## 解説

リファンピシンはCYP450の強力な誘導薬：
1. ❌ AST 26、ALT 27 → 肝機能は正常。障害なし
2. ⭕ リファンピシンがCYP誘導→ワルファリン代謝促進→作用減弱
3. ⭕ 同じくCYP誘導→タクロリムス代謝促進→血中濃度低下
4. ❌ Cr 1.8は低下あるが、この程度で変更は不要
5. ❌ 誘導なので低下が正しい。上昇は阻害の場合
