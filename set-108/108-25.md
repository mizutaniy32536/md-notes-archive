# 108-25

病院薬剤部において高カロリー輸液を調製する際に排出される廃棄物のうち、感染性廃棄物と同等の取扱いとするのはどれか。1つ選べ。

1. 輸液バッグのゴム栓のアルコール消毒に用いたガーゼ
2. 輸液バッグへ薬液を添加するのに用いた注射筒
3. 輸液バッグへ薬液を添加するのに用いた注射針
4. 輸液バッグへ添加する薬液を取って空になったバイアル
5. 調製時に用いたディスポーザブル手袋

## 正解
**3. 輸液バッグへ薬液を添加するのに用いた注射針**

## 解説

### 感染性廃棄物とは
医療機関等から排出される廃棄物のうち、感染性病原体が含まれるか、含まれる可能性のあるもの。

### 注射針の取扱い

**重要：注射針は未使用でも感染性廃棄物と同等の取扱い**

**理由：**
- 鋭利なもの（鋭利器材）に分類される
- 針刺し事故のリスクがある
- 血液や体液に触れる可能性がある

**廃棄方法：**
- 専用の廃棄容器（針捨てボックス）に入れる
- 赤色または黄色の専用容器
- リキャップ禁止（針刺し事故防止）

### 各選択肢の廃棄区分

| 廃棄物 | 区分 | 理由 |
|---|---|---|
| アルコール消毒ガーゼ | 一般廃棄物 | 感染性なし |
| 注射筒（シリンジ） | プラスチックゴミ | 感染性なし |
| **注射針** | **感染性廃棄物扱い** | **鋭利なもの** |
| 空バイアル | ガラスゴミ | 感染性なし |
| 手袋 | 一般廃棄物 | 感染性なし |

### 感染性廃棄物の分類

**廃棄物処理法による分類：**
1. **血液等が付着したもの**
   - ガーゼ、脱脂綿、包帯等
   
2. **鋭利なもの**（鋭利器材）
   - **注射針、メス、ガラス製品等**
   - **未使用でも感染性廃棄物扱い**

3. **病原微生物に関連した試験・検査等に使用されたもの**
   - 培養容器、シャーレ等

4. **血液等が大量に付着したもの**
   - 手術で使用したリネン類等

### ポイント
- **注射針 = 鋭利なもの = 感染性廃棄物扱い**
- **未使用でも専用容器で廃棄**
- リキャップ禁止（針刺し事故防止）
