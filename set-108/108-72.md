# 108-72

**問題文**
長井長義によって世界で初めて単離されたのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-72 を参照

**選択肢**
1. モルヒネ
2. エフェドリン
3. アトロピン
4. サリチル酸
5. コカイン

**正解:** 2

**解説**
解答　２

長井長義（1845〜1929）は、麻黄から世界で初めてエフェドリンを単離し、のちに合成が可能であることを証明した人物である。日本薬学会初代会頭を40年以上務め、日本の薬学教育、薬事行政、製薬事業の基礎固めに貢献したことから、日本薬学の父と呼ばれている。
