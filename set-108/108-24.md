# 108-24

水質汚濁指標である溶存酸素量（DO）の測定法はどれか。1つ選べ。

1. インドフェノール法
2. ウインクラー法
3. エチレンジアミン四酢酸（EDTA）による滴定法
4. 硝酸銀滴定法（モール法）
5. ピリジン・ピラゾロン法

## 正解
**2. ウインクラー法**

## 解説

### 溶存酸素量（DO）とは
水中に溶けている酸素の量。水質汚濁の重要な指標。

**DOが低い** → 有機物が多く、微生物が酸素を消費 → 水質汚濁  
**DOが高い** → 水質が良好

### ウインクラー法（DO測定法）

**原理：**
1. 水サンプルにヨウ化カリウム（KI）と硫酸マンガン（MnSO₄）を加える
2. 溶存酸素がマンガンを酸化
3. ヨウ素（I₂）が遊離する
4. チオ硫酸ナトリウム（Na₂S₂O₃）で滴定
5. ヨウ素の量 = 溶存酸素の量

**別名：** ヨウ素滴定法

### 他の測定法との違い

| 測定法 | 測定対象 | 用途 |
|---|---|---|
| **ウインクラー法** | **溶存酸素量（DO）** | 水質汚濁指標 |
| インドフェノール法 | アンモニア態窒素（NH₄⁺-N） | 水質汚濁指標 |
| EDTA滴定法 | 総硬度（Ca²⁺・Mg²⁺） | 水の硬度測定 |
| 硝酸銀滴定法（モール法） | 塩化物イオン（Cl⁻） | 水質検査 |
| ピリジン・ピラゾロン法 | シアン（CN⁻） | 有害物質検出 |

### 水質汚濁指標まとめ

| 指標 | 説明 | 測定法 |
|---|---|---|
| **DO（溶存酸素量）** | 水中の酸素量 | ウインクラー法 |
| **BOD** | 生物化学的酸素要求量 | 微生物による分解 |
| **COD** | 化学的酸素要求量 | 過マンガン酸カリウム |
| **SS** | 浮遊物質量 | ろ過・重量測定 |

### ポイント
**DO（溶存酸素量）= ウインクラー法**（ヨウ素滴定）
