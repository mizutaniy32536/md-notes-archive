# 108-101

**問題文**
日本薬局方収載医薬品ジルチアゼムは、化合物Aから化合物Bを経て合成される。次の記述のうち、正しいのはどれか。2つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-101 を参照

**選択肢**
1. 酸素原子ウは、ジルチアゼム分子内には存在しない。
2. 酸素原子エは、主に酸素原子イ由来である。
3. 酸素原子オは、主に酸素原子ウ由来である。
4. 酸素原子カは、主に酸素原子ア由来である。
5. 化合物Bは、化合物Aからの転位反応による生成物である。

**正解:** 1,2

**解説**
解答　１、２

「ジルチアゼムは、化合物Aから化合物Bを経て合成される」とあることから、反応前後の部分構造の変化に着目することで、最終生成物のジルチアゼムの酸素原子がどこの酸素原子由来なのかを推察できる。

上記の反応より、−COOCH3（エステル）のOCH3と−NH2（アミノ基）のHがなくなり、−CONH−になっていることから、−NH2と残ったC=Oが反応したと推測できる。なお、この反応はエステルに対する求核置換反応である。（選択肢5）
よって、酸素原子ウは化合物B及びジルチアゼムには残っていない。（選択肢1）

上記の反応より、化合物Bの酸素原子アを含む−OHのHがなくなり、そこにCOCH3が結合していることから、−OHとCOClが反応したと推測できる。このとき、酸素原子アを含むOHのHだけ脱離し、Oはジルチアゼムに残っているため、酸素原子オはア由来である。（選択肢3）

また、酸素原子イ由来である化合物BのC=OのOは、反応を起こしていないため、酸素原子エはイ由来である。（選択肢2）

最後に、酸素原子カはCH3COClの酸素由来であるため、酸素原子カは、元々化合物Bに存在する酸素原子ア由来ではない。（選択肢4）
