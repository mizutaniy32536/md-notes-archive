# 108-47

**問題文**
肝代謝型の薬物はどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-47 を参照

**選択肢**
1. セファクロル
2. ゲンタマイシン
3. メトトレキサート
4. プロプラノロール
5. レボフロキサシン

**正解:** 4

**解説**
解答　４

肝代謝型の薬物は、プロプラノロールである。なお、セファクロル、ゲンタマイシン、メトトレキサート、レボフロキサシンは、腎排泄型の薬物である。
