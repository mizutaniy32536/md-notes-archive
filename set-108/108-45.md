# 108-45

**問題文**
イヌリンの主な腎排泄過程として最も適切なのはどれか。1つ選べ。

> ⚠️ 画像を含む問題のため、図・構造式等はe-REC 108-45 を参照

**選択肢**
1. 糸球体ろ過
2. 近位尿細管における分泌
3. 近位尿細管における再吸収
4. 遠位尿細管における分泌
5. 遠位尿細管における再吸収

**正解:** 1

**解説**
解答　１

腎排泄に関与する主な過程には、糸球体ろ過、尿細管における分泌、尿細管における再吸収の3つがあり、排泄される物質の種類（例：イヌリン、p−アミノ馬尿酸、グルコースなど）によって、どのような過程を経るかが変わる。
このうち、イヌリンは、尿細管における分泌や再吸収を受けないため、主に糸球体ろ過で排泄される。
なお、p−アミノ馬尿酸の腎排泄に関与する主な過程は、糸球体ろ過と近位尿細管における分泌であり、グルコースの腎排泄に関与する主な過程は、糸球体ろ過と近位尿細管における再吸収である。
