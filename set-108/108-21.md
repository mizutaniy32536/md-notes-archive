# 108-21

### 第108回薬剤師国家試験 問21

**問題文（全文）**  
生態系を構成する生物のうち、**独立栄養生物**はどれか。1つ選べ。[1][2][3][7]

**全選択肢（番号付き）**  
1. 一次消費者  
2. 二次消費者  
3. 高次消費者  
4. 分解者  
5. 生産者[1][2][3][7]

**正解番号**  
5[1][2][3][6]

**解説**  
**独立栄養生物**とは、無機物から有機物を同化（合成）できる生物で、光合成を行う植物などが代表例であり、生態系では**生産者**に該当する。一方、一次消費者・二次消費者・高次消費者（草食動物・肉食動物など）および分解者（菌類・一部の動物）は、他の生物由来の有機物を摂取する**従属栄養生物**である。[1][3]  
以上より、正解は**5 生産者**。[1][2][3]
