# 108-21

生態系を構成する生物のうち、独立栄養生物はどれか。1つ選べ。

1. 一次消費者
2. 二次消費者
3. 高次消費者
4. 分解者
5. 生産者

## 正解
**5. 生産者**

## 解説

### 独立栄養生物とは
無機物（CO₂、水、光など）から自分で有機物を合成できる生物

**代表例：**
- 植物（光合成）
- 藻類
- 光合成細菌
- 化学合成細菌

### 生態系の役割分担

| 役割 | 栄養形態 | 代表例 | エネルギー源 |
|---|---|---|---|
| **生産者** | **独立栄養生物** | 植物、藻類 | 光エネルギー |
| 一次消費者 | 従属栄養生物 | 草食動物 | 植物を食べる |
| 二次消費者 | 従属栄養生物 | 肉食動物 | 草食動物を食べる |
| 高次消費者 | 従属栄養生物 | 頂点捕食者 | 肉食動物を食べる |
| 分解者 | 従属栄養生物 | 細菌、菌類 | 死骸・排泄物を分解 |

### 栄養形態の分類
- **独立栄養生物（autotroph）**: 無機物から有機物を合成
  - 光合成生物：光エネルギーを利用
  - 化学合成生物：化学エネルギーを利用
  
- **従属栄養生物（heterotroph）**: 他の生物から有機物を摂取
  - 消費者（動物）
  - 分解者（細菌・菌類）

### ポイント
**独立栄養生物 = 生産者のみ**  
消費者・分解者は全て従属栄養生物
