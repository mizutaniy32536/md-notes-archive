# 108-21

生態系を構成する生物のうち、独立栄養生物はどれか。１つ選べ。

１ 一次消費者
２ 二次消費者
３ 高次消費者
４ 分解者
５ 生産者

**正解:** ５

**解説**
生態系とは、生物集団と非生物環境が相互に関係しあって、物質とエネルギーの流れを形成する系のことであり、生態系を構成する生物集団は、生産者、消費者、分解者で構成される。また生物集団は、生産者である植物や、化学合成細菌（硝化細菌など）のような独立栄養生物と、消費者である動物や、分解者である大部分の微生物のような従属栄養生物に分類される。
