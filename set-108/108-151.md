# 108-151

### 第108回薬剤師国家試験 問151

**問題文**  
完全アゴニストの濃度－反応曲線 (曲線 A) に関連する記述として、正しいのはどれか。2 つ選べ。  
ただし、各薬物は同一の受容体にのみ作用し、また、余剰受容体は存在しないものとする。[1][4]

**全選択肢**  
1. 部分アゴニストは、完全アゴニストによる最大反応には影響せず、曲線Aを低濃度側に平行移動させる。[4]  
2. 競合的アンタゴニストは、受容体に可逆的に結合し、曲線Aを高濃度側に平行移動させる。[4]  
3. 非競合的アンタゴニストは、完全アゴニストによる最大反応には影響せず、曲線Aを高濃度側に平行移動させる。[4]  
4. 逆アゴニストは、曲線Aを低濃度側に平行移動させる。[4]  
5. 完全アゴニストに化学修飾を加え、内活性は変えずに受容体に対する親和性だけを上げると、その濃度－反応曲線は、曲線Aより低濃度側に位置する。[4]

**正解番号**  
2, 5[1][9]

**解説**  
- **選択肢1 誤り**: 部分アゴニストは最大反応を低下させ、曲線Aを下方に押し下げる。完全アゴニストの最大反応に影響を与えるため、低濃度側への平行移動は起こらない。[1]  
- **選択肢2 正しい**: 競合的アンタゴニストは受容体に可逆的に結合し、完全アゴニストの濃度を高めれば反応が得られるため、曲線Aを高濃度側に平行移動させる。[1][6]  
- **選択肢3 誤り**: 非競合的アンタゴニストは最大反応を低下させ、曲線Aを下方に押し下げる。平行移動は起こらない。[1]  
- **選択肢4 誤り**: 逆アゴニストは構成活性受容体の活性を抑制する作用を持ち、曲線Aを低濃度側に平行移動させるものではない。[1]  
- **選択肢5 正しい**: 内活性（最大反応）を変えずに親和性だけを上げると、相同効力の薬物となり、低濃度で同等の反応を示すため、曲線Aより低濃度側に位置する。[1]  

用量反応曲線の変化は、競合的アンタゴニストで高濃度側平行移動、非競合的・部分アゴニストで最大反応低下が特徴。[1][6]
