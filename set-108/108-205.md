



## ◆ 弱酸性薬物の静脈血中における分子形・イオン形の存在比

260313

**問題**
- オメプラゾールは弱酸で、静脈血（pH7.4）中で解離する。
- 静脈血中でのオメプラゾールの分子形とイオン形の存在比として正しいのはどれか。１つ選べ。ただし、オメプラゾールのpKaを8.9、√10＝3.2とする。

**選択肢**

1. 分子形：イオン形＝32：1

2. 分子形：イオン形＝10：1

​
4. 分子形：イオン形＝1：10

**正解と理由**
- `1. 分子形：イオン形＝32：1`
- **理由の構成**
	- **問題の整理**：弱酸性薬物オメプラゾール（pKa 8.9）のpH 7.4における分子形/イオン形比を求める
	- **薬学的知識・仕様・制約**：Henderson-Hasselbalchの式（弱酸版）：
		- pH = pKa + log([A⁻]/[HA])
		- 7.4 = 8.9 + log([A⁻]/[HA])
		- log([A⁻]/[HA]) = 7.4 − 8.9 = `−1.5`
		- [A⁻]/[HA] = 10⁻¹·⁵ = 1/10¹·⁵ = 1/(10 × √10) = 1/(10 × 3.2) = `1/32`
		- 分子形(HA)：イオン形(A⁻) = `32：1`
	- **選択理由**：pH < pKaなので弱酸は分子形が優位。計算により32：1

**解法の手順（思考の流れ）**
- まず弱酸のH-H式を想起：pH = pKa + log([A⁻]/[HA])
- pH 7.4 < pKa 8.9 → `pH < pKa`なので分子形(HA)が優位と判断
- log([A⁻]/[HA]) = −1.5 → [A⁻]/[HA] = 10⁻¹·⁵
- 10¹·⁵ = 10¹ × 10⁰·⁵ = 10 × `√10` = 10 × 3.2 = 32
- 分子形：イオン形 = 32：1 と結論

**他の選択肢が誤りな理由**
- 2. 10：1
	pKa − pH = 1.0のとき得られる値。1.5ではなく1.0と計算間違いした場合

- 3. 1：1
	`pH = pKa`のとき分子形とイオン形は等量。pH 7.4 ≠ pKa 8.9なので不適切

- 4. 1：10
	pH > pKaの場合にイオン形が10倍になる。今回はpH < pKaなので逆

- 5. 1：32
	分子形とイオン形を逆に読んだ場合。弱酸でpH < pKaなら分子形が多い

**キーポイント**
- `Henderson-Hasselbalchの式`（弱酸）：pH = pKa + log([A⁻]/[HA])
- `pH < pKa` → 分子形(HA)優位、`pH > pKa` → イオン形(A⁻)優位
- 10の非整数乗の計算：10¹·⁵ = 10 × √10 ≈ 10 × 3.16 ≈ `32`

**関連知識の比較**
- 弱塩基の場合：pH = pKa + log([B]/[BH⁺])。pH > pKaで分子形優位（弱酸と逆の関係）
- `オメプラゾール`は胃壁細胞の酸性環境（pH 1〜2）で活性化される`プロトンポンプ阻害薬`（PPI）
- 薬物のイオン化率は`消化管吸収`に影響。分子形が多いほど脂溶性が高く、吸収されやすい（`pH分配仮説`）

（一言メモ）
- 弱酸は「pH < pKaで分子形優位」。酸性環境で酸は分子形＝これが`pH分配仮説`の基本
- 10の1.5乗 = 10 × √10 ≈ `32`。この計算パターンは国試で超頻出
- `PPI`は弱塩基性プロドラッグで、酸性環境でスルフェンアミドに変換されてH⁺/K⁺-ATPaseを不可逆阻害
